Introducció

A Swift, les funcions són un bloc fonamental del teu codi. Et permeten encapsular peces reutilitzables de lògica i poden prendre entrades (paràmetres) i produir sortides (valors de retorn). Entendre com definir i utilitzar paràmetres de funció i valors de retorn és crucial per escriure codi Swift eficient i llegible.

Paràmetres de Funció

Definició de Paràmetres

Els paràmetres són les entrades d'una funció. Els defineixes dins dels parèntesis després del nom de la funció.

func greet(name: String) {
    print("Hello, \(name)!")
}
  • name és un paràmetre de tipus String.
  • Quan crides la funció, has de proporcionar un argument que coincideixi amb el tipus del paràmetre.

Cridar Funcions amb Paràmetres

Per cridar una funció amb paràmetres, passes els arguments en el mateix ordre en què els paràmetres estan definits.

greet(name: "Alice")
// Sortida: Hello, Alice!

Paràmetres Múltiples

Una funció pot tenir múltiples paràmetres, separats per comes.

func add(a: Int, b: Int) -> Int {
    return a + b
}

let sum = add(a: 3, b: 5)
print(sum) // Sortida: 8
  • a i b són paràmetres de tipus Int.
  • La funció retorna la suma de a i b.

Etiquetes de Paràmetres i Etiquetes d'Arguments

Swift et permet definir etiquetes de paràmetres i etiquetes d'arguments per fer les teves crides de funció més llegibles.

func multiply(_ number1: Int, by number2: Int) -> Int {
    return number1 * number2
}

let product = multiply(4, by: 5)
print(product) // Sortida: 20
  • _ s'utilitza per ometre l'etiqueta d'argument per number1.
  • by és l'etiqueta d'argument per number2.

Valors de Paràmetres per Defecte

Pots proporcionar valors per defecte per als paràmetres, fent-los opcionals quan crides la funció.

func greet(name: String = "Guest") {
    print("Hello, \(name)!")
}

greet() // Sortida: Hello, Guest!
greet(name: "Bob") // Sortida: Hello, Bob!

Valors de Retorn

Valor de Retorn Únic

Les funcions poden retornar un valor utilitzant la sintaxi -> seguida del tipus de retorn.

func square(number: Int) -> Int {
    return number * number
}

let result = square(number: 4)
print(result) // Sortida: 16
  • La funció square retorna un Int.

Valors de Retorn Múltiples amb Tuples

Swift permet que les funcions retornin múltiples valors utilitzant tuples.

func minMax(numbers: [Int]) -> (min: Int, max: Int)? {
    guard let first = numbers.first else { return nil }
    var min = first
    var max = first
    for number in numbers.dropFirst() {
        if number < min {
            min = number
        } else if number > max {
            max = number
        }
    }
    return (min, max)
}

if let result = minMax(numbers: [3, 5, 1, 9, 2]) {
    print("Min: \(result.min), Max: \(result.max)")
    // Sortida: Min: 1, Max: 9
}
  • La funció minMax retorna una tuple opcional (min: Int, max: Int)?.

Tipus de Retorn Void

Si una funció no retorna un valor, pots utilitzar Void o simplement ometre el tipus de retorn.

func sayHello() {
    print("Hello, World!")
}

sayHello() // Sortida: Hello, World!

Temes Avançats

Paràmetres In-Out

Els paràmetres in-out et permeten modificar el valor d'un paràmetre dins de la funció.

func swapValues(_ a: inout Int, _ b: inout Int) {
    let temp = a
    a = b
    b = temp
}

var x = 3
var y = 5
swapValues(&x, &y)
print("x: \(x), y: \(y)") // Sortida: x: 5, y: 3
  • La paraula clau inout s'utilitza per indicar que un paràmetre pot ser modificat.
  • El símbol & s'utilitza quan passes arguments a paràmetres in-out.

Tipus de Funció com a Paràmetres i Valors de Retorn

Les funcions poden prendre altres funcions com a paràmetres i també poden retornar funcions.

func performOperation(_ operation: (Int, Int) -> Int, on a: Int, and b: Int) -> Int {
    return operation(a, b)
}

let result = performOperation(add, on: 3, and: 5)
print(result) // Sortida: 8
  • operation és un paràmetre de tipus (Int, Int) -> Int.
  • La funció performOperation pren una funció i dos enters, i retorna el resultat de l'operació.

Conclusió

Entendre els paràmetres de funció i els valors de retorn és essencial per escriure codi Swift efectiu. En dominar aquests conceptes, pots crear funcions més flexibles, reutilitzables i llegibles. Practica definint funcions amb diversos tipus de paràmetres i valors de retorn per convertir-te en un programador proficient en Swift.
