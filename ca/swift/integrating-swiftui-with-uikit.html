Introducció

Integrar SwiftUI amb UIKit permet als desenvolupadors aprofitar els punts forts de tots dos frameworks, permetent l'ús de vistes modernes de SwiftUI dins d'aplicacions existents de UIKit i viceversa. Aquesta secció et guiarà a través del procés d'incrustar vistes de SwiftUI en UIKit i integrar components de UIKit en SwiftUI.

Conceptes Clau

  • SwiftUI: Un framework declaratiu per construir interfícies d'usuari a totes les plataformes d'Apple.
  • UIKit: Un framework tradicional per construir aplicacions iOS utilitzant un enfocament imperatiu.
  • UIHostingController: Un controlador de vista de UIKit que allotja una vista de SwiftUI.
  • UIViewRepresentable: Un protocol que et permet embolicar una vista de UIKit per al seu ús en SwiftUI.

Incrustar SwiftUI en UIKit

Utilitzant UIHostingController

Per incrustar una vista de SwiftUI en una aplicació de UIKit, utilitzes UIHostingController. Aquest controlador actua com un pont entre els dos frameworks.

Exemple

import SwiftUI
import UIKit

// Defineix una vista simple de SwiftUI
struct ContentView: View {
    var body: some View {
        Text("Hola, SwiftUI!")
            .padding()
    }
}

// En el teu controlador de vista de UIKit
class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Crea una instància de la vista de SwiftUI
        let swiftUIView = ContentView()
        
        // Crea un UIHostingController amb la vista de SwiftUI
        let hostingController = UIHostingController(rootView: swiftUIView)
        
        // Afegeix el controlador d'allotjament com a controlador de vista fill
        addChild(hostingController)
        view.addSubview(hostingController.view)
        
        // Configura les restriccions de la vista del controlador d'allotjament
        hostingController.view.translatesAutoresizingMaskIntoConstraints = false
        NSLayoutConstraint.activate([
            hostingController.view.topAnchor.constraint(equalTo: view.topAnchor),
            hostingController.view.bottomAnchor.constraint(equalTo: view.bottomAnchor),
            hostingController.view.leadingAnchor.constraint(equalTo: view.leadingAnchor),
            hostingController.view.trailingAnchor.constraint(equalTo: view.trailingAnchor)
        ])
        
        // Notifica al controlador d'allotjament que ha estat mogut a un controlador de vista pare
        hostingController.didMove(toParent: self)
    }
}

Beneficis

  • Integració Sense Fissures: Integra fàcilment vistes modernes de SwiftUI en projectes existents de UIKit.
  • Reutilització: Reutilitza components de SwiftUI en diferents parts de la teva aplicació.

Incrustar UIKit en SwiftUI

Utilitzant UIViewRepresentable

Per utilitzar una vista de UIKit en SwiftUI, has de conformar-te al protocol UIViewRepresentable. Aquest protocol requereix que implementis mètodes per crear i actualitzar la vista de UIKit.

Exemple

import SwiftUI
import UIKit

// Crea una UIView personalitzada
class CustomUIView: UIView {
    override init(frame: CGRect) {
        super.init(frame: frame)
        self.backgroundColor = .blue
    }
    
    required init?(coder: NSCoder) {
        fatalError("init(coder:) has not been implementat")
    }
}

// Crea un embolcall de SwiftUI per a la UIView personalitzada
struct CustomUIViewRepresentable: UIViewRepresentable {
    func makeUIView(context: Context) -> CustomUIView {
        return CustomUIView()
    }
    
    func updateUIView(_ uiView: CustomUIView, context: Context) {
        // Actualitza la vista si cal
    }
}

// Utilitza la UIView personalitzada en una vista de SwiftUI
struct ContentView: View {
    var body: some View {
        VStack {
            Text("Hola, SwiftUI!")
            CustomUIViewRepresentable()
                .frame(height: 100)
        }
    }
}

Beneficis

  • Flexibilitat: Utilitza components de UIKit que encara no estan disponibles en SwiftUI.
  • Compatibilitat: Integra vistes de UIKit heretades en nous projectes de SwiftUI.

Resum

Integrar SwiftUI amb UIKit permet als desenvolupadors aprofitar els punts forts de tots dos frameworks, proporcionant flexibilitat i reutilització en el desenvolupament d'aplicacions. Utilitzant UIHostingController, pots incrustar vistes de SwiftUI en aplicacions de UIKit, i amb UIViewRepresentable, pots embolicar vistes de UIKit per al seu ús en SwiftUI. Aquesta integració permet una transició suau entre els dos frameworks, assegurant que puguis aprofitar les últimes tecnologies d'interfície d'usuari mentre mantens la compatibilitat amb bases de codi existents.
