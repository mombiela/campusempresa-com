Introducció als Arrays

Els arrays són estructures de dades fonamentals a Swift que et permeten emmagatzemar col·leccions ordenades de valors. Són versàtils i poden contenir elements de qualsevol tipus, incloent-hi objectes personalitzats.

Conceptes Clau

  • Definició: Un array és una col·lecció ordenada de valors.
  • Seguretat de Tipus: Els arrays a Swift són de tipus segur, és a dir, només poden emmagatzemar elements del mateix tipus.
  • Mutabilitat: Els arrays poden ser mutables (modificables) o immutables (constants).

Creació d'Arrays

Pots crear arrays utilitzant literals d'array o l'inicialitzador Array.

Literals d'Array

let numbers = [1, 2, 3, 4, 5]
let strings = ["poma", "plàtan", "cirera"]

Inicialitzador d'Array

let emptyArray = [Int]()  // Crea un array buit d'enters
let repeatedArray = Array(repetint: "Hola", count: 3)  // ["Hola", "Hola", "Hola"]

Accés i Modificació d'Arrays

Accés als Elements

Pots accedir als elements d'un array utilitzant la sintaxi de subíndex.

let fruits = ["poma", "plàtan", "cirera"]
let firstFruit = fruits[0]  // "poma"

Modificació dels Elements

Pots modificar els elements assignant un nou valor a un índex específic.

var fruits = ["poma", "plàtan", "cirera"]
fruits[1] = "nabiu"  // ["poma", "nabiu", "cirera"]

Afegir i Eliminar Elements

  • Afegir: Afegeix un element al final de l'array.
  • Inserir: Insereix un element en un índex específic.
  • Eliminar: Elimina un element en un índex específic o elimina l'últim element.
var numbers = [1, 2, 3]
numbers.append(4)  // [1, 2, 3, 4]
numbers.insert(0, at: 0)  // [0, 1, 2, 3, 4]
numbers.remove(at: 2)  // [0, 1, 3, 4]
numbers.removeLast()  // [0, 1, 3]

Iteració sobre Arrays

Pots iterar sobre arrays utilitzant bucles.

Bucle For-In

let fruits = ["poma", "plàtan", "cirera"]
for fruit in fruits {
    print(fruit)
}

Bucle Enumerat

Per obtenir tant l'índex com el valor, utilitza el mètode enumerated().

let fruits = ["poma", "plàtan", "cirera"]
for (index, fruit) in fruits.enumerated() {
    print("Element \(index + 1): \(fruit)")
}

Operacions Avançades amb Arrays

Ordenar Arrays

Pots ordenar arrays utilitzant el mètode sorted() o el mètode sort() per ordenar in situ.

let numbers = [3, 1, 4, 1, 5, 9]
let sortedNumbers = numbers.sorted()  // [1, 1, 3, 4, 5, 9]

Filtrar Arrays

Utilitza el mètode filter() per crear un nou array amb elements que compleixin una condició.

let numbers = [1, 2, 3, 4, 5]
let evenNumbers = numbers.filter { $0 % 2 == 0 }  // [2, 4]

Mapar Arrays

Utilitza el mètode map() per transformar cada element de l'array.

let numbers = [1, 2, 3, 4, 5]
let squaredNumbers = numbers.map { $0 * $0 }  // [1, 4, 9, 16, 25]

Reduir Arrays

Utilitza el mètode reduce() per combinar tots els elements en un sol valor.

let numbers = [1, 2, 3, 4, 5]
let sum = numbers.reduce(0, +)  // 15

Conclusió

Els arrays són una manera poderosa i flexible de gestionar col·leccions de dades a Swift. Des de la creació i manipulació bàsiques fins a operacions avançades com ordenar, filtrar, mapar i reduir, entendre els arrays millorarà significativament la teva capacitat per escriure codi Swift eficient i efectiu. Practica aquests conceptes amb diversos exercicis per consolidar la teva comprensió i convertir-te en un expert en l'ús d'arrays a Swift.
