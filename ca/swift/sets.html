Introducció als Conjunts

Els conjunts són col·leccions desordenades d'elements únics. Són útils quan necessites assegurar-te que un element només aparegui una vegada en una col·lecció i quan l'ordre dels elements no és important.

Conceptes Clau

  • Col·lecció Desordenada: Els conjunts no mantenen l'ordre dels elements.
  • Elements Únics: Cada element en un conjunt ha de ser únic.
  • Protocol Hashable: Els elements en un conjunt han de conformar-se al protocol Hashable.

Crear i Inicialitzar Conjunts

Pots crear conjunts utilitzant literals d'array o l'inicialitzador Set.

Utilitzant Literals d'Array

let fruits: Set = ["Apple", "Banana", "Cherry"]
print(fruits) // La sortida pot ser en qualsevol ordre, per exemple, ["Banana", "Cherry", "Apple"]

Utilitzant l'Inicialitzador Set

let numbers = Set([1, 2, 3, 4, 5])
print(numbers) // La sortida pot ser en qualsevol ordre, per exemple, [2, 4, 1, 5, 3]

Operacions Bàsiques amb Conjunts

Els conjunts suporten diverses operacions com afegir, eliminar i comprovar elements.

Afegir Elements

var animals: Set = ["Cat", "Dog"]
animals.insert("Bird")
print(animals) // La sortida pot ser en qualsevol ordre, per exemple, ["Dog", "Bird", "Cat"]

Eliminar Elements

var colors: Set = ["Red", "Green", "Blue"]
colors.remove("Green")
print(colors) // La sortida pot ser en qualsevol ordre, per exemple, ["Red", "Blue"]

Comprovar Elements

let cities: Set = ["New York", "Los Angeles", "Chicago"]
if cities.contains("Chicago") {
    print("Chicago és al conjunt.")
} else {
    print("Chicago no és al conjunt.")
}
// Sortida: Chicago és al conjunt.

Operacions amb Conjunts

Els conjunts a Swift suporten diverses operacions matemàtiques de conjunts com la unió, la intersecció i la diferència.

Unió

Combina tots els elements de tots dos conjunts.

let setA: Set = [1, 2, 3]
let setB: Set = [3, 4, 5]
let unionSet = setA.union(setB)
print(unionSet) // Sortida: [2, 4, 5, 1, 3]

Intersecció

Retorna els elements que són comuns a tots dos conjunts.

let setA: Set = [1, 2, 3]
let setB: Set = [3, 4, 5]
let intersectionSet = setA.intersection(setB)
print(intersectionSet) // Sortida: [3]

Diferència

Retorna els elements que estan en un conjunt però no en l'altre.

let setA: Set = [1, 2, 3]
let setB: Set = [3, 4, 5]
let differenceSet = setA.subtracting(setB)
print(differenceSet) // Sortida: [1, 2]

Diferència Simètrica

Retorna els elements que estan en qualsevol dels conjunts, però no en tots dos.

let setA: Set = [1, 2, 3]
let setB: Set = [3, 4, 5]
let symmetricDifferenceSet = setA.symmetricDifference(setB)
print(symmetricDifferenceSet) // Sortida: [2, 4, 5, 1]

Característiques Avançades dels Conjunts

Pertinença i Igualtat de Conjunts

Pots comprovar si un conjunt és un subconjunt, un superconjunt, o si dos conjunts són iguals.

Subconjunt

let setA: Set = [1, 2, 3]
let setB: Set = [1, 2, 3, 4, 5]
print(setA.isSubset(of: setB)) // Sortida: true

Superconjunt

let setA: Set = [1, 2, 3, 4, 5]
let setB: Set = [1, 2, 3]
print(setA.isSuperset(of: setB)) // Sortida: true

Igualtat

let setA: Set = [1, 2, 3]
let setB: Set = [3, 2, 1]
print(setA == setB) // Sortida: true

Conclusió

Els conjunts a Swift són eines poderoses per gestionar col·leccions d'elements únics. Proporcionen operacions eficients per afegir, eliminar i comprovar elements, així com per realitzar operacions matemàtiques de conjunts com la unió, la intersecció i la diferència. Entendre els conjunts i les seves operacions pot millorar enormement la teva capacitat per treballar amb col·leccions a Swift.

En dominar els conjunts, pots assegurar la unicitat de les dades i realitzar manipulacions complexes de col·leccions amb facilitat.
