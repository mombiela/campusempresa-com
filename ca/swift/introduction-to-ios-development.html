Visió General

En aquesta secció, cobrirem els conceptes bàsics del desenvolupament d'iOS utilitzant Swift. Això inclou configurar el teu entorn de desenvolupament, entendre els fonaments de Swift i crear la teva primera aplicació iOS.

Configuració del Teu Entorn de Desenvolupament

Per començar a desenvolupar aplicacions iOS, necessites configurar el teu entorn de desenvolupament.

Requisits

  • Un ordinador Mac amb macOS.
  • Xcode: l'entorn de desenvolupament integrat (IDE) d'Apple per a macOS.

Passos per Instal·lar Xcode

  1. Obre l'App Store al teu Mac.
  2. Cerca "Xcode".
  3. Fes clic al botó "Get" i després "Install".
  4. Un cop instal·lat, obre Xcode i segueix les instruccions inicials de configuració.

Conceptes Bàsics de Swift

Swift és un llenguatge de programació potent i intuïtiu per al desenvolupament d'iOS. Anem a veure alguns conceptes fonamentals.

Variables i Constants

  • Les variables s'utilitzen per emmagatzemar dades que poden canviar.
  • Les constants s'utilitzen per emmagatzemar dades que no poden canviar.
var variableName = "Hello, World!" // Això és una variable
let constantName = 42 // Això és una constant

Tipus de Dades

Swift suporta diversos tipus de dades com Int, Double, String i Bool.

var age: Int = 25
var pi: Double = 3.14159
var greeting: String = "Hello, Swift!"
var isSwiftFun: Bool = true

Flux de Control

Les instruccions de flux de control inclouen if, else, for i while.

let score = 85

if score >= 90 {
    print("Grade: A")
} else if score >= 80 {
    print("Grade: B")
} else {
    print("Grade: C")
}

Funcions

Les funcions són blocs de codi reutilitzables que realitzen tasques específiques.

func greet(name: String) -> String {
    return "Hello, \(name)!"
}

let greetingMessage = greet(name: "Alice")
print(greetingMessage) // Output: Hello, Alice!

Creant la Teva Primera Aplicació iOS

Anem a crear una aplicació iOS simple utilitzant Xcode.

Passos per Crear un Nou Projecte

  1. Obre Xcode i selecciona "Create a new Xcode project".
  2. Tria "App" sota la pestanya iOS i fes clic a "Next".
  3. Introdueix el nom del projecte i altres detalls, després fes clic a "Next".
  4. Tria una ubicació per desar el projecte i fes clic a "Create".

Entenent l'Estructura del Projecte

  • Main.storyboard: La interfície visual de la teva aplicació.
  • ViewController.swift: El codi darrere de la vista principal.
  • AppDelegate.swift: Gestiona els esdeveniments a nivell d'aplicació.

Afegint Elements d'Interfície d'Usuari

  1. Obre Main.storyboard.
  2. Arrossega una Label des de la Biblioteca d'Objectes a la vista.
  3. Arrossega un Button des de la Biblioteca d'Objectes a la vista.
  4. Configura el text de l'etiqueta a "Hello, World!" i el botó a "Press Me".

Connectant Elements d'Interfície d'Usuari al Codi

  1. Obre ViewController.swift.
  2. Arrossega amb control des de l'etiqueta fins al codi per crear un IBOutlet.
  3. Arrossega amb control des del botó fins al codi per crear un IBAction.
import UIKit

class ViewController: UIViewController {
    @IBOutlet weak var helloLabel: UILabel!
    
    @IBAction func buttonPressed(_ sender: UIButton) {
        helloLabel.text = "Button Pressed!"
    }
}

Conclusió

En aquest tema introductori, hem cobert els conceptes bàsics de la configuració del teu entorn de desenvolupament d'iOS, conceptes fonamentals de Swift i la creació d'una aplicació iOS simple. A mesura que avancis en el curs, aprofundiràs en temes més avançats i construiràs aplicacions més complexes. Bona programació!
