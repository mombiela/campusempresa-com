Introducció

A Swift, les propietats i els mètodes són conceptes fonamentals que et permeten emmagatzemar i manipular dades dins de les teves classes, estructures i enumeracions. Entendre aquests conceptes és crucial per escriure codi Swift eficient i efectiu.

Propietats

Les propietats associen valors amb una classe, estructura o enumeració particular. Es poden classificar en propietats emmagatzemades, propietats calculades i observadors de propietats.

Propietats Emmagatzemades

Les propietats emmagatzemades emmagatzemen valors constants o variables com a part d'una instància. Poden ser propietats variables (declarades amb var) o propietats constants (declarades amb let).

struct Person {
    var name: String
    let birthYear: Int
}

var person = Person(name: "Alice", birthYear: 1990)
person.name = "Bob" // Permès perquè 'name' és una propietat variable
// person.birthYear = 1991 // Error: No es pot assignar a 'birthYear' perquè és una constant 'let'

Propietats Calculades

Les propietats calculades no emmagatzemen un valor directament. En canvi, proporcionen un getter i un setter opcional per recuperar i establir altres propietats o valors indirectament.

struct Rectangle {
    var width: Double
    var height: Double
    
    var area: Double {
        return width * height
    }
}

let rect = Rectangle(width: 5.0, height: 10.0)
print("Àrea: \(rect.area)") // Sortida: Àrea: 50.0

Observadors de Propietats

Els observadors de propietats observen i responen als canvis en el valor d'una propietat. Es poden afegir a propietats emmagatzemades (excepte per a propietats lazy) i es criden cada vegada que es configura el valor d'una propietat.

class StepCounter {
    var totalSteps: Int = 0 {
        willSet(newTotalSteps) {
            print("A punt de configurar totalSteps a \(newTotalSteps)")
        }
        didSet {
            if totalSteps > oldValue {
                print("Afegits \(totalSteps - oldValue) passos")
            }
        }
    }
}

let stepCounter = StepCounter()
stepCounter.totalSteps = 200
// Sortida:
// A punt de configurar totalSteps a 200
// Afegits 200 passos

Mètodes

Els mètodes són funcions que estan associades amb un tipus particular. Poden ser mètodes d'instància o mètodes de tipus.

Mètodes d'Instància

Els mètodes d'instància són funcions que pertanyen a instàncies d'una classe, estructura o enumeració particular.

class Counter {
    var count = 0
    
    func increment() {
        count += 1
    }
    
    func increment(by amount: Int) {
        count += amount
    }
    
    func reset() {
        count = 0
    }
}

let counter = Counter()
counter.increment()
print(counter.count) // Sortida: 1
counter.increment(by: 5)
print(counter.count) // Sortida: 6
counter.reset()
print(counter.count) // Sortida: 0

Mètodes de Tipus

Els mètodes de tipus són mètodes que es criden en el tipus mateix, en lloc d'una instància del tipus. Es defineixen amb la paraula clau static per a estructures i enumeracions, i amb la paraula clau class per a classes per permetre la sobreescriptura de mètodes.

class SomeClass {
    class func someTypeMethod() {
        print("Mètode de tipus cridat")
    }
}

SomeClass.someTypeMethod() // Sortida: Mètode de tipus cridat

Conclusió

Les propietats i els mètodes són blocs de construcció essencials en la programació Swift. Les propietats et permeten emmagatzemar i calcular valors, mentre que els mètodes et permeten definir comportaments per als teus tipus. En dominar aquests conceptes, pots crear aplicacions Swift més robustes i mantenibles. Practica utilitzant propietats emmagatzemades, propietats calculades, observadors de propietats, mètodes d'instància i mètodes de tipus per aprofundir en la teva comprensió i millorar les teves habilitats de codificació.
