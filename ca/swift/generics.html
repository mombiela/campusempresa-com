Els genèrics són una característica potent a Swift que et permeten escriure codi flexible i reutilitzable. Utilitzant genèrics, pots escriure funcions i tipus que poden treballar amb qualsevol tipus, subjecte als requisits que defineixis.

Què són els Genèrics?

Els genèrics et permeten escriure funcions i tipus que poden operar sobre qualsevol tipus, en lloc d'un tipus específic. Això ajuda a crear codi més flexible i reutilitzable.

Conceptes Clau

  • Paràmetres de Tipus: Marcadors de posició per a tipus que s'utilitzen en funcions o tipus genèrics.
  • Funcions Genèriques: Funcions que poden operar sobre qualsevol tipus.
  • Tipus Genèrics: Tipus (com classes, structs i enums) que poden treballar amb qualsevol tipus.
  • Restriccions de Tipus: Restriccions que especifiquen que un paràmetre de tipus ha de conformar-se a un cert protocol o heretar d'una certa classe.

Funcions Genèriques

Una funció genèrica pot treballar amb qualsevol tipus. Aquí tens un exemple senzill:

func swapTwoValues<T>(_ a: inout T, _ b: inout T) {
    let temporaryA = a
    a = b
    b = temporaryA
}
  • T és un marcador de posició per a un paràmetre de tipus.
  • La funció swapTwoValues pot intercanviar els valors de qualsevol tipus, sempre que ambdós valors siguin del mateix tipus.

Exemple d'Ús

var a = 5
var b = 10
swapTwoValues(&a, &b)
print("a: \(a), b: \(b)") // Sortida: a: 10, b: 5

var x = "Hello"
var y = "World"
swapTwoValues(&x, &y)
print("x: \(x), y: \(y)") // Sortida: x: World, y: Hello

Tipus Genèrics

Pots definir els teus propis tipus genèrics per crear estructures de dades flexibles i reutilitzables.

Exemple: Pila Genèrica

struct Stack<Element> {
    var items = [Element]()
    
    mutating func push(_ item: Element) {
        items.append(item)
    }
    
    mutating func pop() -> Element {
        return items.removeLast()
    }
}
  • Element és un marcador de posició per a un paràmetre de tipus.
  • La Stack pot emmagatzemar elements de qualsevol tipus.

Exemple d'Ús

var intStack = Stack<Int>()
intStack.push(1)
intStack.push(2)
print(intStack.pop()) // Sortida: 2

var stringStack = Stack<String>()
stringStack.push("Hello")
stringStack.push("World")
print(stringStack.pop()) // Sortida: World

Restriccions de Tipus

Les restriccions de tipus especifiquen que un paràmetre de tipus ha de conformar-se a un cert protocol o heretar d'una certa classe.

Exemple: Restriccions de Tipus

func findIndex<T: Equatable>(of valueToFind: T, in array: [T]) -> Int? {
    for (index, value) in array.enumerated() {
        if value == valueToFind {
            return index
        }
    }
    return nil
}
  • T: Equatable especifica que el paràmetre de tipus T ha de conformar-se al protocol Equatable.
  • Això permet que la funció utilitzi l'operador == per comparar valors.

Exemple d'Ús

let numbers = [1, 2, 3, 4, 5]
if let index = findIndex(of: 3, in: numbers) {
    print("Index of 3 is \(index)") // Sortida: Index of 3 is 2
}

let strings = ["apple", "banana", "cherry"]
if let index = findIndex(of: "banana", in: strings) {
    print("Index of banana is \(index)") // Sortida: Index of banana is 1
}

Genèrics Avançats

Swift permet característiques genèriques més avançades, com tipus associats en protocols i subíndexs genèrics.

Tipus Associats en Protocols

Un tipus associat dóna un nom de marcador de posició a un tipus utilitzat com a part del protocol.

protocol Container {
    associatedtype Item
    mutating func append(_ item: Item)
    var count: Int { get }
    subscript(i: Int) -> Item { get }
}

Exemple: Ús de Tipus Associats

struct IntStack: Container {
    var items = [Int]()
    
    mutating func push(_ item: Int) {
        items.append(item)
    }
    
    mutating func pop() -> Int {
        return items.removeLast()
    }
    
    mutating func append(_ item: Int) {
        self.push(item)
    }
    
    var count: Int {
        return items.count
    }
    
    subscript(i: Int) -> Int {
        return items[i]
    }
}

Conclusió

Els genèrics són una característica fonamental a Swift que et permeten escriure codi flexible, reutilitzable i segur en termes de tipus. Entenent i utilitzant els genèrics, pots crear aplicacions més robustes i mantenibles. Ja sigui que estiguis escrivint funcions genèriques, tipus o utilitzant característiques avançades com tipus associats, els genèrics proporcionen un conjunt d'eines potent per als desenvolupadors de Swift.
