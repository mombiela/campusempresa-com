Què és Swift?

Swift és un llenguatge de programació potent i intuïtiu desenvolupat per Apple per crear aplicacions per a iOS, macOS, watchOS i tvOS. Està dissenyat per funcionar perfectament amb els frameworks Cocoa i Cocoa Touch d'Apple i està construït sobre el millor de C i Objective-C, sense les limitacions de la compatibilitat amb C.

Característiques Clau de Swift

  • Segur: Swift elimina classes senceres de codi insegur. Les variables sempre s'inicialitzen abans de ser utilitzades, els arrays i els enters es comproven per desbordament, i la memòria es gestiona automàticament.
  • Ràpid: Swift està dissenyat per ser ràpid. La sintaxi i la biblioteca estàndard s'han ajustat per fer les tasques més comunes fàcils i clares, assegurant al mateix temps que el llenguatge sigui ràpid.
  • Expressiu: Swift inclou característiques modernes que els desenvolupadors adoren. Té una inferència de tipus potent i una concordança de patrons, cosa que fa que el codi sigui més expressiu i menys propens a errors.

Configuració del Teu Entorn

Per començar a desenvolupar amb Swift, necessites configurar el teu entorn de desenvolupament:

  1. Xcode: Descarrega i instal·la Xcode des de la Mac App Store. Xcode és l'entorn de desenvolupament integrat (IDE) d'Apple per a macOS.
  2. Playgrounds: Utilitza Swift Playgrounds per a codificació interactiva i aprenentatge. Els Playgrounds et permeten escriure codi Swift i veure els resultats immediatament.

Sintaxi Bàsica

Variables i Constants

En Swift, utilitzes var per declarar variables i let per declarar constants.

var myVariable = 42
myVariable = 50
let myConstant = 42
  • Variables: Es poden canviar després de ser establertes.
  • Constants: No es poden canviar un cop establertes.

Tipus de Dades

Swift és un llenguatge segur en termes de tipus, cosa que significa que el llenguatge t'ajuda a ser clar sobre els tipus de valors amb els quals el teu codi pot treballar.

let integer: Int = 70
let double: Double = 70.0
let string: String = "Hola, Swift!"
let boolean: Bool = true

Strings i Caràcters

Les cadenes en Swift són compatibles amb Unicode i poden contenir qualsevol caràcter.

let greeting = "Hola, Món!"
let character: Character = "A"

Arrays i Diccionaris

Els arrays i els diccionaris s'utilitzen per emmagatzemar col·leccions de valors.

var shoppingList: [String] = ["Ous", "Llet"]
var occupations: [String: String] = ["Malcolm": "Capità", "Kaylee": "Mecànica"]

Flux de Control

Swift proporciona totes les estructures tradicionals de flux de control com ara bucles for i while, així com declaracions if i switch.

let individualScores = [75, 43, 103, 87, 12]
var teamScore = 0
for score in individualScores {
    if score > 50 {
        teamScore += 3
    } else {
        teamScore += 1
    }
}
print(teamScore)

Funcions

Les funcions són blocs de codi autònoms que realitzen una tasca específica.

func greet(person: String, day: String) -> String {
    return "Hola \(person), avui és \(day)."
}
print(greet(person: "John", day: "Dimecres"))

Conclusió

Swift és un llenguatge de programació modern, potent i fàcil d'aprendre que és perfecte per desenvolupar aplicacions a totes les plataformes d'Apple. Entenent la sintaxi bàsica i les característiques de Swift, pots començar a construir les teves pròpies aplicacions i aprofitar l'ecosistema ric proporcionat per Apple. A mesura que avancis en aquest curs, aprofundiràs en temes més avançats i et convertiràs en un programador proficient en Swift.
