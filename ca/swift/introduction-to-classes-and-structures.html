Visió General

A Swift, les classes i les estructures són construccions poderoses que et permeten crear tipus de dades personalitzats. Ambdues poden emmagatzemar valors i definir comportaments, però tenen algunes diferències clau. Aquesta secció et guiarà a través dels conceptes bàsics de les classes i les estructures, les seves similituds i les seves diferències.

Conceptes Clau

  • Classes i Estructures: Ambdues s'utilitzen per crear tipus de dades personalitzats.
  • Propietats i Mètodes: Ambdues poden tenir propietats (variables) i mètodes (funcions).
  • Inicialització: Ambdues poden ser inicialitzades per configurar el seu estat inicial.
  • Herència: Només les classes suporten l'herència.
  • Tipus de Referència vs. Tipus de Valor: Les classes són tipus de referència, mentre que les estructures són tipus de valor.

Classes

Definició

Una classe és un plànol per crear objectes. Les classes suporten l'herència, la qual cosa permet que una classe hereti les característiques d'una altra.

Sintaxi

class Person {
    var name: String
    var age: Int
    
    init(name: String, age: Int) {
        self.name = name
        self.age = age
    }
    
    func greet() {
        print("Hola, em dic \(name) i tinc \(age) anys.")
    }
}

Explicació

  • class Person: Defineix una nova classe anomenada Person.
  • var name: String: Una propietat per emmagatzemar el nom de la persona.
  • var age: Int: Una propietat per emmagatzemar l'edat de la persona.
  • init(name: String, age: Int): Un inicialitzador per configurar l'estat inicial de l'objecte.
  • func greet(): Un mètode per imprimir un missatge de salutació.

Exemple d'Ús

let person = Person(name: "Alice", age: 30)
person.greet() // Sortida: Hola, em dic Alice i tinc 30 anys.

Estructures

Definició

Una estructura és un tipus de valor que pot encapsular dades i funcionalitat relacionada. Les estructures no suporten l'herència.

Sintaxi

struct Rectangle {
    var width: Double
    var height: Double
    
    func area() -> Double {
        return width * height
    }
}

Explicació

  • struct Rectangle: Defineix una nova estructura anomenada Rectangle.
  • var width: Double: Una propietat per emmagatzemar l'amplada del rectangle.
  • var height: Double: Una propietat per emmagatzemar l'alçada del rectangle.
  • func area() -> Double: Un mètode per calcular l'àrea del rectangle.

Exemple d'Ús

let rectangle = Rectangle(width: 10.0, height: 5.0)
print("Àrea del rectangle: \(rectangle.area())") // Sortida: Àrea del rectangle: 50.0

Comparació: Classes vs. Estructures

| Característica | Classes | Estructures | |--------------------|---------------------------------|---------------------------------| | Herència | Sí | No | | Tipus | Tipus de Referència | Tipus de Valor | | Inicialitzadors | Inicialitzadors personalitzats | Inicialitzadors personalitzats | | Propietats | Propietats emmagatzemades i calculades | Propietats emmagatzemades i calculades | | Mètodes | Mètodes d'instància i de tipus | Mètodes d'instància i de tipus |

Conclusió

Les classes i les estructures a Swift proporcionen maneres flexibles de crear tipus de dades personalitzats. Tot i que comparteixen moltes similituds, com les propietats i els mètodes, difereixen en àrees clau com l'herència i el comportament del tipus. Entendre aquestes diferències és crucial per prendre decisions informades sobre quin constructe utilitzar en les teves aplicacions. Les classes són ideals per a escenaris on es necessiten herència i semàntica de referència, mentre que les estructures són adequades per a tipus de dades més simples i semàntica de valor.
