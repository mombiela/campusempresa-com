Introducció a les Extensions

Les extensions en Swift són una característica poderosa que et permet afegir nova funcionalitat a classes, estructures, enumeracions o protocols existents. Això es pot fer sense modificar el codi font original, fent que el teu codi sigui més modular i organitzat.

Conceptes Clau

  • Afegir Propietats Calculades: Les extensions poden afegir noves propietats calculades a tipus existents.
  • Afegir Mètodes: Pots afegir nous mètodes d'instància i mètodes de tipus.
  • Afegir Inicialitzadors: Les extensions poden afegir nous inicialitzadors a tipus existents.
  • Afegir Subscripts: Pots afegir nous subscripts a tipus existents.
  • Afegir Tipus Imbricats: Les extensions poden afegir nous tipus imbricats a classes, estructures o enumeracions existents.
  • Conformar-se a Protocols: Les extensions poden fer que un tipus existent es conformi a un protocol.

Afegir Propietats Calculades

Les extensions poden afegir propietats calculades a tipus existents. No obstant això, no poden afegir propietats emmagatzemades o observadors de propietats.

extension Double {
    var km: Double { return self * 1_000.0 }
    var m: Double { return self }
    var cm: Double { return self / 100.0 }
    var mm: Double { return self / 1_000.0 }
}

let oneMeter = 1.0.m
print("One meter is \(oneMeter) meters") // One meter is 1.0 meters

Afegir Mètodes

Les extensions poden afegir nous mètodes d'instància i mètodes de tipus a tipus existents.

extension Int {
    func repetitions(task: () -> Void) {
        for _ in 0..<self {
            task()
        }
    }
}

3.repetitions {
    print("Hello!")
}
// Prints "Hello!" three times

Afegir Inicialitzadors

Les extensions poden afegir nous inicialitzadors a tipus existents. Això és particularment útil per afegir inicialitzadors de conveniència.

struct Size {
    var width = 0.0, height = 0.0
}

extension Size {
    init(side: Double) {
        self.init(width: side, height: side)
    }
}

let square = Size(side: 10.0)
print("Square size: \(square.width) x \(square.height)") // Square size: 10.0 x 10.0

Afegir Subscripts

Les extensions poden afegir nous subscripts a tipus existents.

extension String {
    subscript(index: Int) -> Character {
        return self[self.index(self.startIndex, offsetBy: index)]
    }
}

let greeting = "Hello"
print(greeting[1]) // e

Afegir Tipus Imbricats

Les extensions poden afegir nous tipus imbricats a classes, estructures o enumeracions existents.

extension Int {
    enum Kind {
        case negative, zero, positive
    }

    var kind: Kind {
        switch self {
        case 0:
            return .zero
        case let x where x > 0:
            return .positive
        default:
            return .negative
        }
    }
}

func printIntegerKinds(_ numbers: [Int]) {
    for number in numbers {
        switch number.kind {
        case .negative:
            print("- ", terminator: "")
        case .zero:
            print("0 ", terminator: "")
        case .positive:
            print("+ ", terminator: "")
        }
    }
    print("")
}

printIntegerKinds([3, 19, -27, 0, -6, 0, 7])
// Prints "+ + - 0 - 0 + "

Conformar-se a Protocols

Les extensions poden fer que un tipus existent es conformi a un protocol, que és una manera comuna d'afegir conformitat a un protocol a un tipus que no posseeixes.

protocol TextRepresentable {
    var textualDescription: String { get }
}

extension Int: TextRepresentable {
    var textualDescription: String {
        return String(self)
    }
}

let number = 42
print(number.textualDescription) // 42

Conclusió

Les extensions són una característica versàtil en Swift que et permeten afegir nova funcionalitat a tipus existents d'una manera neta i modular. Et permeten estendre tipus que no posseeixes, afegir propietats calculades, mètodes, inicialitzadors, subscripts, tipus imbricats i fer que els tipus es conformin a protocols. Utilitzant extensions, pots mantenir el teu codi organitzat i mantenible mentre millores les capacitats dels tipus existents.
