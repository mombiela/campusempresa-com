Introducció

En Swift, entendre els tipus de dades i la inferència de tipus és crucial per escriure codi eficient i lliure d'errors. Aquesta secció cobrirà els conceptes bàsics dels tipus de dades, com Swift infereix els tipus i les millors pràctiques per utilitzar-los.

Tipus de Dades Bàsics

Swift proporciona diversos tipus de dades integrats que utilitzaràs freqüentment. Aquí tens alguns dels més comuns:

  • Int: Representa nombres enters.
  • Double: Representa nombres de coma flotant de 64 bits.
  • Float: Representa nombres de coma flotant de 32 bits.
  • Bool: Representa valors booleans (true o false).
  • String: Representa una seqüència de caràcters.
  • Character: Representa un sol caràcter.

Exemples

let age: Int = 25
let pi: Double = 3.14159
let isSwiftFun: Bool = true
let greeting: String = "Hello, Swift!"
let initial: Character = "S"

Inferència de Tipus

Swift és un llenguatge segur en termes de tipus, el que significa que realitza comprovacions de tipus quan compila el teu codi i marca qualsevol tipus no coincident com a error. Swift també pot inferir el tipus d'una variable o constant basant-se en el valor que li assignis, fent el teu codi més net i concís.

Exemples

let age = 25 // Swift infereix que això és Int
let pi = 3.14159 // Swift infereix que això és Double
let isSwiftFun = true // Swift infereix que això és Bool
let greeting = "Hello, Swift!" // Swift infereix que això és String
let initial: Character = "S" // Anotació de tipus explícita

Seguretat de Tipus i Comprovació de Tipus

Swift assegura que els tipus de valors amb els quals treballes siguin consistents. Això ajuda a prevenir errors i fa que el teu codi sigui més previsible.

Exemples

let age = 25
// age = "Twenty-five" // Això causarà un error en temps de compilació perquè 'age' s'ha inferit com a Int

Àlies de Tipus

Els àlies de tipus et permeten crear un nou nom per a un tipus existent. Això pot fer que el teu codi sigui més llegible i fàcil de gestionar.

Exemples

typealias AudioSample = UInt16
var maxAmplitudeFound = AudioSample.min // maxAmplitudeFound ara és de tipus UInt16

Optionals

Els optionals són una característica poderosa en Swift que et permet manejar l'absència d'un valor. Un optional conté un valor o nil per indicar que un valor falta.

Exemples

var optionalString: String? = "Hello"
optionalString = nil // optionalString ara pot contenir un String o ser nil

if let unwrappedString = optionalString {
    print(unwrappedString) // Això no s'executarà perquè optionalString és nil
} else {
    print("optionalString is nil")
}

Optionals Desempaquetats Implícitament

De vegades, és clar per l'estructura del programa que un optional sempre tindrà un valor després de ser inicialment assignat. En aquests casos, és útil eliminar la necessitat de comprovar i desempaquetar el valor de l'optional cada vegada que s'accedeix. Aquests tipus d'optionals es defineixen com a optionals desempaquetats implícitament.

Exemples

let assumedString: String! = "An implicitly unwrapped optional string."
let implicitString: String = assumedString // No cal el signe d'exclamació

Conclusió

Entendre els tipus de dades i la inferència de tipus en Swift és fonamental per escriure codi robust i eficient. Les característiques de seguretat de tipus i inferència de tipus de Swift t'ajuden a escriure codi més net i fiable detectant desajustos de tipus en temps de compilació. En dominar aquests conceptes, estaràs ben preparat per manejar tasques de programació més complexes en Swift.
