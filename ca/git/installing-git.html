Git és un potent sistema de control de versions que és essencial per al desenvolupament de programari modern. Aquesta secció et guiarà a través del procés d'instal·lació de Git en diversos sistemes operatius. Al final d'aquesta secció, tindràs Git instal·lat i llest per utilitzar a la teva màquina.

Instal·lant Git a Windows

  1. Descarrega l'Instal·lador de Git:

    • Visita el lloc web oficial de Git: Git per a Windows.
    • Fes clic al botó "Download" per obtenir la darrera versió de l'instal·lador de Git.

  2. Executa l'Instal·lador:

    • Localitza el fitxer .exe descarregat i fes doble clic per executar-lo.
    • Segueix els passos de l'assistent d'instal·lació. La configuració per defecte és generalment adequada per a la majoria d'usuaris.

  3. Configura Git:

    • Després de la instal·lació, obre el terminal Git Bash des del menú Inici.
    • Configura el teu nom d'usuari i correu electrònic: 
      git config --global user.name "El Teu Nom"
git config --global user.email "el.teu.email@example.com"

Instal·lant Git a macOS

  1. Utilitzant Homebrew (Recomanat):

    • Si no tens Homebrew instal·lat, pots instal·lar-lo executant la següent comanda al teu terminal: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Un cop Homebrew estigui instal·lat, instal·la Git executant: 
      brew install git

  2. Utilitzant l'Instal·lador de Git:

    • Visita el lloc web oficial de Git: Git per a macOS.
    • Descarrega i executa el paquet d'instal·lació.

  3. Configura Git:

    • Obre l'aplicació Terminal.
    • Configura el teu nom d'usuari i correu electrònic: 
      git config --global user.name "El Teu Nom"
git config --global user.email "el.teu.email@example.com"

Instal·lant Git a Linux

  1. Utilitzant el Gestor de Paquets:

    • Per a distribucions basades en Debian (p. ex., Ubuntu): 
      sudo apt update
sudo apt install git
    • Per a distribucions basades en RPM (p. ex., Fedora): 
      sudo dnf install git

  2. Configura Git:

    • Obre el teu terminal.
    • Configura el teu nom d'usuari i correu electrònic: 
      git config --global user.name "El Teu Nom"
git config --global user.email "el.teu.email@example.com"

Verificant la Instal·lació

Després d'instal·lar Git, hauries de verificar que s'ha instal·lat correctament. Obre el teu terminal o Git Bash i executa la següent comanda:

git --version

Hauries de veure una sortida similar a:

git version 2.x.x

Això confirma que Git està instal·lat i llest per utilitzar.

Resum

En aquesta secció, has après com instal·lar Git a Windows, macOS i Linux. També has configurat Git amb el teu nom d'usuari i correu electrònic, que és essencial per fer el seguiment dels canvis i contribucions. Ara que Git està instal·lat, estàs llest per començar a utilitzar-lo per al control de versions en els teus projectes.
