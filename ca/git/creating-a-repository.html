Clonant un Repositori ►

Crear un repositori és el primer pas per utilitzar Git per al control de versions. Aquesta secció et guiarà a través del procés de crear un repositori, tant localment com a GitHub, i t'explicarà els conceptes clau involucrats.

Què és un Repositori?

Un repositori (o "repo") és un espai d'emmagatzematge on viu el teu projecte. Pot ser local a una carpeta al teu ordinador o allotjat en un servidor remot com GitHub. Els repositoris fan un seguiment dels canvis als fitxers i permeten que diverses persones col·laborin en un projecte.

Crear un Repositori Local

Per crear un repositori local, segueix aquests passos:

  1. Obre el teu terminal o símbol del sistema.

  2. Navega al directori on vols crear el teu projecte.

    cd path/to/your/project

  3. Inicialitza el repositori.

    git init

    Aquesta comanda crea un nou subdirectori anomenat .git que conté tots els fitxers necessaris del teu repositori. Ara s'ha inicialitzat un nou repositori.

Exemple

Creem un repositori local per a un projecte anomenat "MyProject":

mkdir MyProject
cd MyProject
git init

Sortida:

Initialized empty Git repository in /path/to/MyProject/.git/

Crear un Repositori Remot a GitHub

Per crear un repositori remot a GitHub, segueix aquests passos:

  1. Inicia sessió al teu compte de GitHub.

  2. Fes clic a la icona "+" a la cantonada superior dreta i selecciona "New repository".

  3. Omple els detalls del repositori:

    • Nom del repositori: Tria un nom per al teu repositori.
    • Descripció: (Opcional) Afegeix una descripció per al teu repositori.
    • Públic/Privat: Tria si el teu repositori serà públic o privat.
    • Inicialitza amb un README: (Opcional) Marca aquesta casella si vols afegir un fitxer README.

  4. Fes clic a "Create repository".

Exemple

Creem un repositori remot anomenat "MyProject" a GitHub:

  1. Nom del repositori: MyProject
  2. Descripció: Un repositori de projecte d'exemple
  3. Públic/Privat: Públic
  4. Inicialitza amb un README: Marcat

Després de fer clic a "Create repository", GitHub et proporcionarà l'URL del teu nou repositori.

Connectar el Repositori Local al Repositori Remot

Un cop tinguis tant un repositori local com un de remot, necessites connectar-los. Això es fa utilitzant la comanda git remote.

  1. Afegeix l'URL del repositori remot.

    git remote add origin https://github.com/yourusername/MyProject.git

  2. Verifica l'URL remot.

    git remote -v

Exemple

git remote add origin https://github.com/yourusername/MyProject.git
git remote -v

Sortida:

origin  https://github.com/yourusername/MyProject.git (fetch)
origin  https://github.com/yourusername/MyProject.git (push)

Resum

Crear un repositori és un pas fonamental per utilitzar Git per al control de versions. Pots crear un repositori local utilitzant la comanda git init i un repositori remot a GitHub a través de la seva interfície web. Connectar el repositori local al repositori remot et permet empènyer i treure canvis, permetent la col·laboració i la còpia de seguretat.

Seguint aquests passos, ara tens una comprensió bàsica de com crear i gestionar repositoris a Git. Aquest coneixement fonamental serà essencial a mesura que progressis cap a temes més avançats de Git.
