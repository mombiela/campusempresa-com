Introducció a Git LFS

Git Large File Storage (LFS) és una extensió per a Git que et permet gestionar fitxers grans de manera eficient. Substitueix els fitxers grans per pointers de text dins de Git, mentre emmagatzema el contingut dels fitxers en un servidor remot.

  • Propòsit: Gestionar fitxers grans en un repositori Git sense inflar la mida del repositori.
  • Casos d'ús: Projectes amb fitxers multimèdia grans, conjunts de dades o qualsevol altre fitxer binari gran.

Configuració de Git LFS

Per utilitzar Git LFS, cal instal·lar-lo i configurar el teu repositori per rastrejar fitxers grans.

Instal·lació

Instal·la Git LFS utilitzant el gestor de paquets del teu sistema operatiu.

  • macOS:

    brew install git-lfs

  • Windows: Descarrega i executa l'instal·lador des de Git LFS Releases.

  • Linux:

    sudo apt-get install git-lfs

Inicialització de Git LFS en el teu repositori

Després d'instal·lar Git LFS, inicialitza'l en el teu repositori:

git lfs install

Aquesta comanda configura els hooks de Git LFS en el teu repositori.

Rastrejar fitxers grans

Especifica quins fitxers han de ser gestionats per Git LFS utilitzant la comanda git lfs track.

Exemple: Rastrejar fitxers d'imatge

Per rastrejar tots els fitxers .png en el teu repositori:

git lfs track "*.png"

Aquesta comanda crea un fitxer .gitattributes (si no existeix ja) i afegeix una entrada per als fitxers .png.

Cometre canvis

Després de rastrejar fitxers, afegeix i comet els canvis:

git add .gitattributes
git commit -m "Track PNG files with Git LFS"

Treballar amb Git LFS

Un cop els fitxers estan rastrejats per Git LFS, pots treballar amb ells com de costum. Git LFS gestiona l'emmagatzematge i la recuperació de fitxers grans en segon pla.

Afegir fitxers grans

Afegeix un fitxer gran al teu repositori:

git add largefile.zip
git commit -m "Add large file"

Empènyer a remot

Quan empenys canvis a un repositori remot, Git LFS carrega els fitxers grans al servidor LFS:

git push origin main

Gestionar fitxers de Git LFS

Git LFS proporciona diverses comandes per gestionar fitxers grans en el teu repositori.

Llistar fitxers rastrejats

Per llistar tots els fitxers rastrejats per Git LFS:

git lfs ls-files

Recuperar fitxers LFS

Per recuperar tots els fitxers LFS en la branca actual:

git lfs fetch

Comprovar l'estat de LFS

Per comprovar l'estat dels fitxers LFS:

git lfs status

Millors pràctiques

  • Rastreig selectiu: Només rastreja fitxers que siguin realment grans i binaris.
  • Neteja regular: Utilitza git lfs prune per eliminar fitxers LFS antics i no utilitzats.
  • Monitoritzar la mida del repositori: Mantingues un ull a la mida del teu repositori per assegurar-te que es manté manejable.

Conclusió

Git LFS és una eina poderosa per gestionar fitxers grans en un repositori Git. Rastrejant fitxers grans amb Git LFS, pots mantenir la mida del teu repositori sota control i millorar el rendiment. Recorda seguir les millors pràctiques per al rastreig selectiu i la neteja regular per mantenir un flux de treball eficient.
