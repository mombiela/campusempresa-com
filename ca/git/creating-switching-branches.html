Introducció

Les branques a Git et permeten divergir de la línia principal de desenvolupament i continuar treballant sense afectar la base de codi principal. Això és particularment útil per desenvolupar noves funcionalitats, corregir errors o experimentar amb noves idees.

Conceptes Clau

  • Branca: Un punter a un commit específic al repositori.
  • Branca Principal: La branca per defecte en un repositori Git, típicament anomenada main o master.
  • Branca de Funcionalitat: Una branca creada per desenvolupar una nova funcionalitat.
  • Canvi de Branques: Moure's d'una branca a una altra per treballar en diferents parts del projecte.

Creació d'una Branca

Per crear una nova branca, utilitzes el comandament git branch seguit del nom de la nova branca.

Exemple

git branch feature-branch

Aquest comandament crea una nova branca anomenada feature-branch però no hi canvia.

Explicació

  • git branch: El comandament per crear una nova branca.
  • feature-branch: El nom de la nova branca.

Canvi de Branques

Per canviar a una branca diferent, utilitzes el comandament git checkout seguit del nom de la branca a la qual vols canviar.

Exemple

git checkout feature-branch

Aquest comandament canvia el teu directori de treball a la feature-branch.

Explicació

  • git checkout: El comandament per canviar de branques.
  • feature-branch: El nom de la branca a la qual vols canviar.

Creació i Canvi en un Sol Pas

Pots crear i canviar a una nova branca en un sol pas utilitzant la bandera -b amb el comandament git checkout.

Exemple

git checkout -b new-feature

Aquest comandament crea una nova branca anomenada new-feature i hi canvia immediatament.

Explicació

  • -b: Una bandera que indica a Git que ha de crear una nova branca i canviar-hi.
  • new-feature: El nom de la nova branca.

Visualització de Branques

Per veure una llista de totes les branques al teu repositori, utilitza el comandament git branch sense arguments.

Exemple

git branch

Aquest comandament llista totes les branques i destaca la branca actual amb un asterisc (*).

Explicació

  • git branch: El comandament per llistar totes les branques.

Eliminació d'una Branca

Un cop una branca ja no és necessària, pots eliminar-la utilitzant el comandament git branch -d.

Exemple

git branch -d feature-branch

Aquest comandament elimina la feature-branch.

Explicació

  • -d: Una bandera que indica a Git que ha d'eliminar la branca especificada.
  • feature-branch: El nom de la branca a eliminar.

Resum

  • Creació d'una Branca: Utilitza git branch <branch-name>.
  • Canvi de Branques: Utilitza git checkout <branch-name>.
  • Creació i Canvi en un Sol Pas: Utilitza git checkout -b <branch-name>.
  • Visualització de Branques: Utilitza git branch.
  • Eliminació d'una Branca: Utilitza git branch -d <branch-name>.

Entendre com crear i canviar de branques a Git és fonamental per gestionar diferents línies de desenvolupament en els teus projectes. Practica aquests comandaments per convertir-te en un expert en la gestió de branques a Git.
