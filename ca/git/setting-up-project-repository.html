Configurar el Repositori del Projecte
Implementant Funcionalitats i Col·laboració ►

Configurar un repositori de projecte és una habilitat fonamental per a qualsevol desenvolupador. Aquesta secció et guiarà a través del procés d'inicialitzar un repositori Git, afegir fitxers i fer el teu primer commit. Al final d'aquesta secció, tindràs una comprensió sòlida de com configurar i gestionar un repositori Git.

Inicialitzar un Repositori Git

Per començar a utilitzar Git, necessites inicialitzar un repositori al directori del teu projecte. Aquest procés crea un directori ocult .git que fa un seguiment de tots els canvis als teus fitxers.

Passos per Inicialitzar un Repositori

  1. Navega al teu directori de projecte: 
    cd path/to/your/project
  2. Inicialitza el repositori: 
    git init
    Aquesta comanda configura els fitxers i directoris necessaris perquè Git comenci a fer un seguiment del teu projecte.

Afegir Fitxers al Repositori

Un cop el teu repositori està inicialitzat, necessites afegir-hi fitxers. Aquest procés implica preparar els fitxers per a un commit i després commitar-los al repositori.

Preparar Fitxers

Preparar fitxers significa preparar-los per a un commit. Pots preparar fitxers individualment o tots alhora.

  • Prepara un fitxer individual: 
    git add filename
  • Prepara tots els fitxers: 
    git add .

Commetre Fitxers

Després de preparar els fitxers, el següent pas és commitar-los. Un commit és una instantània del teu projecte en un moment específic.

  • Commit fitxers preparats: 
    git commit -m "Initial commit"

Crear un Fitxer .gitignore

Un fitxer .gitignore especifica quins fitxers i directoris Git ha d'ignorar. Això és útil per excloure fitxers temporals, artefactes de construcció i altres fitxers que no necessiten ser seguits.

Exemple de Fitxer .gitignore

# Ignorar directori node_modules
node_modules/

# Ignorar fitxers de registre
*.log

# Ignorar artefactes de construcció
dist/

Per crear un fitxer .gitignore:

  1. Crea el fitxer: 
    touch .gitignore
  2. Edita el fitxer: 
    nano .gitignore
  3. Afegeix patrons per ignorar (com es mostra a l'exemple anterior).

Veure l'Estat del Repositori

Per veure l'estat actual del teu repositori, utilitza la comanda git status. Aquesta comanda mostra quins fitxers estan preparats, no preparats i no seguits.

  • Comprova l'estat del repositori: 
    git status

Resum

En aquesta secció, has après com configurar un repositori Git des de zero. Ara saps com:

  • Inicialitzar un repositori Git.
  • Preparar i commitar fitxers.
  • Crear i utilitzar un fitxer .gitignore.
  • Comprovar l'estat del teu repositori.

Aquestes habilitats fonamentals són essencials per gestionar qualsevol projecte amb Git. A les properes seccions, aprofundirem en funcions i fluxos de treball més avançats de Git.
