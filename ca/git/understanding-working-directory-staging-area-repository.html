Introducció

En aquesta secció, explorarem tres conceptes fonamentals en Git: el Directori de Treball, l'Àrea d'Emmagatzematge i el Repositori. Comprendre aquests conceptes és crucial per gestionar eficaçment el teu codi i col·laborar amb altres.

Directori de Treball

El Directori de Treball és el directori a la teva màquina local on estàs treballant actualment en el teu projecte. Conté tots els fitxers i directoris que formen part del teu projecte.

Conceptes Clau

  • Fitxers No Seguits: Fitxers que estan en el teu Directori de Treball però que encara no són seguits per Git.
  • Fitxers Seguits: Fitxers que estan sent seguits per Git i poden estar en un dels tres estats: modificats, no modificats o emmagatzemats.

Exemple

$ git status
On branch main
Untracked files:
  (use "git add <file>..." to include in what will be committed)
    newfile.txt

nothing added to commit but untracked files present (use "git add" to track)

En aquest exemple, newfile.txt és un fitxer no seguit en el Directori de Treball.

Àrea d'Emmagatzematge

L'Àrea d'Emmagatzematge, també coneguda com l'índex, és un lloc on pots emmagatzemar canvis abans de confirmar-los. Et permet preparar una instantània dels canvis que vols incloure en la teva pròxima confirmació.

Conceptes Clau

  • Emmagatzemar Canvis: Afegir canvis a l'Àrea d'Emmagatzematge utilitzant git add.
  • Emmagatzematge Parcial: Emmagatzemar només parts d'un fitxer utilitzant git add -p.

Exemple

$ git add newfile.txt
$ git status
On branch main
Changes to be committed:
  (use "git restore --staged <file>..." to unstage)
    new file:   newfile.txt

En aquest exemple, newfile.txt ha estat afegit a l'Àrea d'Emmagatzematge i està llest per ser confirmat.

Repositori

El Repositori és on Git emmagatzema tota la informació sobre el projecte, incloent-hi tota la història de tots els canvis. Consta de dues parts: el repositori local a la teva màquina i el repositori remot en un servidor.

Conceptes Clau

  • Repositori Local: El directori .git en el teu projecte, que conté tots els metadades i la base de dades d'objectes.
  • Repositori Remot: Una versió del teu repositori allotjada en un servidor remot, com ara GitHub o GitLab.

Exemple

$ git commit -m "Add newfile.txt"
$ git status
On branch main
nothing to commit, working tree clean

En aquest exemple, els canvis en l'Àrea d'Emmagatzematge han estat confirmats al repositori local.

Resum

Comprendre el Directori de Treball, l'Àrea d'Emmagatzematge i el Repositori és essencial per a un control de versions efectiu amb Git. El Directori de Treball és on fas canvis, l'Àrea d'Emmagatzematge és on prepares els canvis per confirmar-los, i el Repositori és on s'emmagatzema la història del teu projecte. Dominar aquests conceptes t'ajudarà a gestionar el teu codi de manera més eficient i a col·laborar més eficaçment amb altres.
