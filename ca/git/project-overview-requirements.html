Introducció

En aquesta secció, proporcionarem una visió general del projecte en el qual treballaràs al llarg d'aquest curs de Git. Aquest projecte està dissenyat per ajudar-te a aplicar els conceptes que aprens de manera pràctica i pràctica. Al final del curs, tindràs una comprensió sòlida de Git i de com utilitzar-lo de manera efectiva en escenaris del món real.

Descripció del projecte

El projecte d'aquest curs és una aplicació web senzilla que permet als usuaris crear, llegir, actualitzar i eliminar (CRUD) notes. Aquesta aplicació es construirà utilitzant HTML, CSS i JavaScript. Al llarg del curs, utilitzaràs Git per gestionar el desenvolupament d'aquest projecte, fer un seguiment dels canvis, col·laborar amb altres i mantenir diferents versions del teu codi.

Conceptes clau

  • Control de versions: Comprendre la importància del control de versions i com Git ajuda a gestionar els canvis de codi.
  • Repositoris: Crear i gestionar repositoris de Git.
  • Commits: Fer commits per desar els canvis en el repositori.
  • Branques: Treballar amb branques per gestionar diferents versions del projecte.
  • Merging: Combinar canvis de diferents branques.
  • Col·laboració: Utilitzar Git per a la col·laboració en equip, incloent-hi la gestió de conflictes.
  • Repositoris remots: Treballar amb repositoris remots en plataformes com GitHub.

Requisits del projecte

Per completar amb èxit aquest projecte, hauràs de complir els següents requisits:

  1. Coneixements bàsics d'HTML, CSS i JavaScript: Hauries de tenir una comprensió bàsica d'aquestes tecnologies, ja que s'utilitzaran per construir l'aplicació web.
  2. Instal·lació de Git: Assegura't que Git està instal·lat al teu ordinador. Pots descarregar-lo des de git-scm.com.
  3. Compte de GitHub: Crea un compte de GitHub si encara no en tens un. Això s'utilitzarà per a la gestió del repositori remot.
  4. Editor de text: Utilitza un editor de text o un entorn de desenvolupament integrat (IDE) com Visual Studio Code, Sublime Text o Atom.

Configuració del projecte

Segueix aquests passos per configurar el projecte:

  1. Crear un nou repositori:

    git init my-notes-app
cd my-notes-app

  2. Crear fitxers inicials:

    touch index.html styles.css app.js

  3. Afegir fitxers a Git:

    git add .

  4. Fer el primer commit:

    git commit -m "Commit inicial: Configuració de l'estructura del projecte"

  5. Crear un repositori remot a GitHub:

    • Ves a GitHub i crea un nou repositori anomenat my-notes-app.
    • Segueix les instruccions per enllaçar el teu repositori local amb el repositori remot: 
      git remote add origin https://github.com/el-teu-usuari/my-notes-app.git
git push -u origin master

Resum

En aquesta secció, hem esbossat el projecte en el qual treballaràs al llarg d'aquest curs de Git. Hem discutit els conceptes clau que aprendràs, els requisits que necessites complir i els passos inicials per configurar el projecte. Seguint aquestes directrius, estaràs ben preparat per submergir-te en els aspectes pràctics de l'ús de Git per al control de versions i la col·laboració.
