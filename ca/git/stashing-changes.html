Introducció al Desament

Desar en Git et permet guardar temporalment els canvis en el teu directori de treball sense comprometre'ls. Això és particularment útil quan necessites canviar de branca o treballar en una altra cosa però vols tornar als teus canvis actuals més tard.

Conceptes Clau

  • Desament: Un desament és un estat guardat del teu directori de treball i l'índex.
  • Llista de Desaments: Una llista de tots els desaments que has guardat.
  • Aplicar un Desament: Recuperar els canvis d'un desament al teu directori de treball.
  • Eliminar un Desament: Esborrar un desament de la llista de desaments.

Comandes Bàsiques de Desament

Guardar Canvis

Per guardar els teus canvis actuals, utilitza la comanda git stash:

git stash

Aquesta comanda guardarà els teus canvis i revertirà el teu directori de treball a l'estat de l'últim commit.

Llistar Desaments

Per veure tots els desaments que has guardat, utilitza:

git stash list

Això mostrarà una llista de desaments amb els seus identificadors.

Aplicar un Desament

Per aplicar el desament més recent, utilitza:

git stash apply

Per aplicar un desament específic, utilitza:

git stash apply stash@{index}

Substitueix index pel número del desament que vols aplicar.

Eliminar un Desament

Per esborrar un desament específic, utilitza:

git stash drop stash@{index}

Per esborrar tots els desaments, utilitza:

git stash clear

Tècniques Avançades de Desament

Desar Fitxers No Controlats

Per defecte, git stash no inclou fitxers no controlats. Per desar fitxers no controlats, utilitza:

git stash -u

o

git stash --include-untracked

Crear un Desament Amb Nom

Pots donar nom als teus desaments per a una millor identificació:

git stash save "missatge significatiu"

Desar Fitxers Específics

Per desar fitxers específics, utilitza la comanda git stash push:

git stash push -m "missatge" path/to/file

Pop vs. Apply

  • Apply: Aplica el desament però el manté a la llista de desaments.
  • Pop: Aplica el desament i l'elimina de la llista de desaments.
git stash pop

Exemples

Exemple 1: Desament i Aplicació Bàsics

  1. Fes alguns canvis als teus fitxers.
  2. Guarda els canvis: 
    git stash
  3. Llista els desaments: 
    git stash list
  4. Aplica el desament: 
    git stash apply

Exemple 2: Desar Amb Fitxers No Controlats

  1. Fes canvis i afegeix nous fitxers no controlats.
  2. Guarda tots els canvis, incloent-hi els fitxers no controlats: 
    git stash -u
  3. Llista i aplica el desament: 
    git stash list
git stash apply

Exemple 3: Desament Amb Nom i Pop

  1. Fes canvis i guarda amb un nom: 
    git stash save "treball en progrés"
  2. Llista els desaments: 
    git stash list
  3. Aplica el desament: 
    git stash pop

Resum

Desar és una característica poderosa en Git que et permet guardar el teu treball temporalment i tornar-hi més tard. És especialment útil per gestionar treballs en progrés quan necessites canviar de context. Entendre com desar, llistar, aplicar i eliminar desaments millorarà el teu flux de treball i et farà més eficient en la gestió dels teus canvis de codi.
