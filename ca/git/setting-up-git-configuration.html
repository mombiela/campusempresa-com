En aquesta secció, tractarem com configurar Git per assegurar que el teu entorn Git s'adapti a les teves necessitats. Una configuració adequada és essencial per a un control de versions i col·laboració fluides.

Conceptes Clau

  • Configuració Global vs. Local: Git permet establir opcions de configuració a diferents nivells.
    • Configuració Global: S'aplica a tots els repositoris per a un usuari específic.
    • Configuració Local: S'aplica només a un repositori específic.
  • Opcions de Configuració Comunes: Informació de l'usuari, editor de text per defecte, terminacions de línia, etc.

Configuració Global

Els paràmetres de configuració global s'apliquen a tots els repositoris per a l'usuari actual. Aquests paràmetres es guarden en un fitxer situat a ~/.gitconfig.

Configuració de la Informació de l'Usuari

El primer pas per configurar Git és establir la teva informació d'usuari. Aquesta informació s'utilitza per etiquetar els teus commits.

git config --global user.name "El Teu Nom"
git config --global user.email "el.teu.email@example.com"
  • user.name: Estableix el nom que s'adjuntarà als teus commits.
  • user.email: Estableix l'adreça de correu electrònic que s'adjuntarà als teus commits.

Configuració de l'Editor de Text per Defecte

Git utilitza un editor de text per a diverses tasques, com escriure missatges de commit. Pots configurar el teu editor de text preferit.

git config --global core.editor "code --wait"
  • core.editor: Estableix l'editor de text per defecte. En aquest exemple, Visual Studio Code està configurat com l'editor per defecte.

Configuració de les Terminacions de Línia

Els diferents sistemes operatius gestionen les terminacions de línia de manera diferent. Git pot gestionar automàticament les terminacions de línia per assegurar la consistència.

git config --global core.autocrlf true
  • core.autocrlf: Converteix automàticament les terminacions de línia. Estableix a true per a Windows, input per a macOS/Linux, i false per desactivar.

Configuració Local

Els paràmetres de configuració local s'apliquen només a un repositori específic. Aquests paràmetres es guarden en un fitxer situat a .git/config dins del repositori.

Configuració de la Informació de l'Usuari Localment

Potser voldràs utilitzar informació d'usuari diferent per a diferents repositoris.

git config user.name "Usuari Local"
git config user.email "local.email@example.com"

Visualització dels Paràmetres de Configuració

Pots veure els teus paràmetres de configuració actuals en qualsevol moment.

  • Per veure els paràmetres globals:

    git config --global --list

  • Per veure els paràmetres locals:

    git config --list

Resum

Configurar Git és un pas crucial per assegurar un flux de treball fluid. Configurant la informació de l'usuari, l'editor de text per defecte i les terminacions de línia, pots adaptar Git a les teves necessitats. Recorda, els paràmetres globals s'apliquen a tots els repositoris per a un usuari, mentre que els paràmetres locals s'apliquen només a un repositori específic. Utilitza la comanda git config per gestionar aquests paràmetres de manera efectiva.

Dominant aquestes opcions de configuració, estableixes una base sòlida per a un control de versions eficient i una col·laboració en els teus projectes.
