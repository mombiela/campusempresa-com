Variables i Tipus de Dades en Delphi/Object Pascal
Operadors i Expressions ►

Introducció

En aquesta secció, explorarem els conceptes fonamentals de les variables i els tipus de dades en Delphi/Object Pascal. Entendre aquests conceptes és crucial per escriure codi eficient i lliure d'errors. Cobrirem els conceptes bàsics de declarar variables, els diferents tipus de dades disponibles en Delphi, i com utilitzar-los de manera efectiva.

Variables

Les variables s'utilitzen per emmagatzemar dades que poden ser manipulades pel programa. En Delphi, les variables han de ser declarades abans de poder ser utilitzades.

Declarar Variables

Per declarar una variable en Delphi, s'utilitza la paraula clau var seguida del nom de la variable i el seu tipus.

var
  age: Integer;
  name: String;
  isStudent: Boolean;
  • age és una variable enter.
  • name és una variable cadena.
  • isStudent és una variable booleana.

Regles de Nomenclatura de Variables

  • Ha de començar amb una lletra o un guió baix.
  • Pot contenir lletres, dígits i guions baixos.
  • No pot ser una paraula reservada.

Assignar Valors

Els valors poden ser assignats a les variables utilitzant l'operador d'assignació (:=).

age := 25;
name := 'John Doe';
isStudent := True;

Tipus de Dades

Els tipus de dades defineixen el tipus de dades que una variable pot contenir. Delphi proporciona diversos tipus de dades integrats.

Tipus Enter

S'utilitzen per emmagatzemar nombres sencers.

| Tipus | Rang | Mida (bytes) | |----------|-----------------------------|--------------| | Byte | 0 a 255 | 1 | | ShortInt | -128 a 127 | 1 | | Integer | -2,147,483,648 a 2,147,483,647 | 4 | | LongInt | -2,147,483,648 a 2,147,483,647 | 4 | | Int64 | -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807 | 8 |

Tipus de Punt Flotant

S'utilitzen per emmagatzemar nombres amb parts fraccionàries.

| Tipus | Rang | Mida (bytes) | |----------|-----------------------------|--------------| | Single | 1.5 x 10^-45 a 3.4 x 10^38 | 4 | | Double | 5.0 x 10^-324 a 1.7 x 10^308 | 8 | | Extended | 3.6 x 10^-4951 a 1.1 x 10^4932 | 10 |

Tipus Booleà

S'utilitza per emmagatzemar True o False.

var
  isActive: Boolean;
begin
  isActive := True;
end;

Tipus de Caràcter

S'utilitzen per emmagatzemar caràcters simples o cadenes.

| Tipus | Descripció | |---------|------------------------------| | Char | Caràcter simple (1 byte) | | String | Seqüència de caràcters |

var
  initial: Char;
  fullName: String;
begin
  initial := 'A';
  fullName := 'Alice Johnson';
end;

Tipus Enumerats

Tipus de dades definits per l'usuari que consisteixen en un conjunt de valors anomenats.

type
  TDays = (Sun, Mon, Tue, Wed, Thu, Fri, Sat);
var
  today: TDays;
begin
  today := Mon;
end;

Tipus Subrang

Defineixen un subconjunt d'un tipus ordinal existent.

type
  TAge = 0..120;
var
  myAge: TAge;
begin
  myAge := 30;
end;

Temes Avançats

Registres

Els registres s'utilitzen per agrupar diferents tipus de dades junts.

type
  TPerson = record
    Name: String;
    Age: Integer;
    IsStudent: Boolean;
  end;
var
  person: TPerson;
begin
  person.Name := 'John Doe';
  person.Age := 25;
  person.IsStudent := True;
end;

Arrays

Els arrays s'utilitzen per emmagatzemar múltiples valors del mateix tipus.

var
  numbers: array[1..5] of Integer;
begin
  numbers[1] := 10;
  numbers[2] := 20;
  numbers[3] := 30;
  numbers[4] := 40;
  numbers[5] := 50;
end;

Arrays Dinàmics

Els arrays dinàmics poden canviar de mida en temps d'execució.

var
  dynArray: array of Integer;
begin
  SetLength(dynArray, 3);
  dynArray[0] := 1;
  dynArray[1] := 2;
  dynArray[2] := 3;
end;

Conclusió

Entendre les variables i els tipus de dades és fonamental per programar en Delphi/Object Pascal. Aquest coneixement et permetrà emmagatzemar i manipular dades de manera efectiva. A mesura que progressis, trobaràs estructures de dades més complexes i aprendràs a utilitzar-les per resoldre problemes del món real. Practica declarant variables, assignant valors i utilitzant diferents tipus de dades per convertir-te en un programador expert en Delphi.
