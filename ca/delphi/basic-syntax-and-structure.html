Introducció a Delphi/Object Pascal

Delphi, també conegut com a Object Pascal, és un llenguatge d'alt nivell, compilat, fortament tipat que suporta programació estructurada i orientada a objectes. És conegut per la seva llegibilitat i facilitat d'ús, cosa que el fa una excel·lent elecció tant per a principiants com per a programadors experimentats.

Conceptes Clau

  • Sensibilitat a Majúscules i Minúscules: Delphi no és sensible a majúscules i minúscules.
  • Comentaris: Els comentaris d'una sola línia utilitzen //, i els comentaris de múltiples línies utilitzen {} o (* *).
  • Identificadors: Noms per a variables, tipus, funcions, etc., que han de començar amb una lletra o guió baix i poden ser seguits per lletres, dígits o guions baixos.
  • Paraules Reservades: Paraules clau que tenen un significat especial en Delphi i no poden ser utilitzades com a identificadors.

Estructura Bàsica del Programa

Un programa simple en Delphi consisteix en les següents parts:

  1. Capçalera del Programa: Declara el nom del programa.
  2. Clàusula Uses: Llista les unitats que el programa utilitza.
  3. Declaracions de Variables: Declara variables.
  4. Bloc Begin-End: Conté el codi principal.

Exemple

program HelloWorld;

uses
  SysUtils;

begin
  WriteLn('Hola, Món!');
end.

Explicació

  • program HelloWorld;: Declara el nom del programa.
  • uses SysUtils;: Inclou la unitat SysUtils, que proporciona diverses funcions d'utilitat.
  • begin ... end.: El bloc principal de codi on l'execució del programa comença i acaba.
  • WriteLn('Hola, Món!');: Mostra el text "Hola, Món!" a la consola.

Variables i Tipus de Dades

Declaració de Variables

Les variables en Delphi es declaren utilitzant la paraula clau var seguida del nom de la variable i el tipus.

var
  age: Integer;
  name: String;

Tipus de Dades Comuns

| Tipus de Dada | Descripció | |---------------|--------------------------------------| | Integer | Nombres enters | | Real | Nombres de punt flotant | | Char | Caràcter únic | | String | Seqüència de caràcters | | Boolean | Valors True o False |

Exemple

var
  age: Integer;
  price: Real;
  initial: Char;
  name: String;
  isStudent: Boolean;
begin
  age := 25;
  price := 19.99;
  initial := 'A';
  name := 'Alice';
  isStudent := True;
end.

Explicació

  • age := 25;: Assigna el valor 25 a la variable age.
  • price := 19.99;: Assigna el valor 19.99 a la variable price.
  • initial := 'A';: Assigna el caràcter 'A' a la variable initial.
  • name := 'Alice';: Assigna la cadena "Alice" a la variable name.
  • isStudent := True;: Assigna el valor booleà True a la variable isStudent.

Estructures de Control

Declaracions Condicionals

Delphi suporta les declaracions if, if-else i case per a l'execució condicional.

Exemple: if-else

var
  age: Integer;
begin
  age := 18;
  if age >= 18 then
    WriteLn('Adult')
  else
    WriteLn('Menor');
end.

Declaracions de Bucle

Delphi suporta els bucles for, while i repeat-until.

Exemple: bucle for

var
  i: Integer;
begin
  for i := 1 to 10 do
    WriteLn('Número: ', i);
end.

Explicació

  • for i := 1 to 10 do: Fa un bucle de l'1 al 10.
  • WriteLn('Número: ', i);: Mostra el valor actual de i.

Procediments i Funcions

Procediments

Els procediments són blocs de codi que realitzen una tasca però no retornen un valor.

procedure Greet;
begin
  WriteLn('Hola!');
end;

begin
  Greet;
end.

Funcions

Les funcions són similars als procediments però retornen un valor.

function Add(a, b: Integer): Integer;
begin
  Add := a + b;
end;

begin
  WriteLn('Suma: ', Add(5, 3));
end.

Explicació

  • procedure Greet;: Declara un procediment anomenat Greet.
  • function Add(a, b: Integer): Integer;: Declara una funció anomenada Add que pren dos enters i retorna un enter.
  • Add := a + b;: Retorna la suma de a i b.

Conclusió

Entendre la sintaxi i estructura bàsica de Delphi/Object Pascal és crucial per escriure programes efectius. Això inclou saber com declarar variables, utilitzar estructures de control i definir procediments i funcions. Amb aquests fonaments, pots començar a construir aplicacions més complexes i explorar característiques avançades del llenguatge.
