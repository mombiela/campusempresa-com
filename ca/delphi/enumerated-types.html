Tipus Enumerats en Delphi/Object Pascal
Cadenes i Gestió de Cadenes ►

Els tipus enumerats, o enums, són una característica poderosa en Delphi/Object Pascal que et permeten definir un conjunt de valors anomenats. Aquests valors són constants i es poden utilitzar per fer que el teu codi sigui més llegible i mantenible. Aquest tema et guiarà des dels conceptes bàsics fins a l'ús avançat dels tipus enumerats en Delphi/Object Pascal.

Introducció als Tipus Enumerats

Els tipus enumerats són tipus de dades definits per l'usuari que consisteixen en un conjunt de valors anomenats elements. Cada element en un tipus enumerat està associat amb un valor enter, començant per 0 per defecte.

Definició de Tipus Enumerats

Per definir un tipus enumerat, utilitzes la paraula clau type seguida del nom del tipus i la llista de valors possibles entre parèntesis.

type
  TColor = (Red, Green, Blue);

En aquest exemple, TColor és un tipus enumerat amb tres valors possibles: Red, Green i Blue.

Ús dels Tipus Enumerats

Un cop has definit un tipus enumerat, pots declarar variables d'aquest tipus i assignar-los qualsevol dels valors definits en l'enumeració.

var
  MyColor: TColor;
begin
  MyColor := Red;
  if MyColor = Green then
    WriteLn('The color is green.')
  else
    WriteLn('The color is not green.');
end;

Tipus Enumerats Avançats

Assignació de Valors Específics

Per defecte, el primer element en un tipus enumerat se li assigna el valor 0, el segon element el valor 1, i així successivament. No obstant això, pots assignar valors enters específics als elements.

type
  TStatus = (Pending = 1, Approved = 5, Rejected = 10);

En aquest exemple, Pending té assignat el valor 1, Approved el valor 5, i Rejected el valor 10.

Rangs de Tipus Enumerats

Pots utilitzar tipus enumerats per definir rangs, que poden ser útils en bucles i indexació d'arrays.

type
  TDay = (Mon, Tue, Wed, Thu, Fri, Sat, Sun);

var
  Day: TDay;
begin
  for Day := Mon to Fri do
    WriteLn(Day);
end;

Tipus Enumerats i Conjunts

Els tipus enumerats es poden utilitzar per definir conjunts, que són col·leccions de valors del tipus enumerat.

type
  TDays = set of TDay;

var
  Weekend: TDays;
begin
  Weekend := [Sat, Sun];
  if Sun in Weekend then
    WriteLn('Sunday is a weekend day.');
end;

Funcions de Tipus Enumerats

Delphi proporciona diverses funcions integrades per treballar amb tipus enumerats, com ara Ord, Pred, Succ i High.

  • Ord: Retorna el valor ordinal d'un element del tipus enumerat.
  • Pred: Retorna el predecessor d'un element del tipus enumerat.
  • Succ: Retorna el successor d'un element del tipus enumerat.
  • High: Retorna el valor més alt en un tipus enumerat.
begin
  WriteLn(Ord(Mon));  // Sortida: 0
  WriteLn(Pred(Wed)); // Sortida: Tue
  WriteLn(Succ(Tue)); // Sortida: Wed
  WriteLn(High(TDay)); // Sortida: Sun
end;

Conclusió

Els tipus enumerats en Delphi/Object Pascal proporcionen una manera de definir un conjunt de constants anomenades, fent que el teu codi sigui més llegible i mantenible. Pots assignar valors específics als elements, utilitzar-los en rangs i conjunts, i aprofitar les funcions integrades per manipular-los. Entendre i utilitzar els tipus enumerats de manera efectiva pot millorar molt la claredat i la funcionalitat dels teus programes en Delphi/Object Pascal.
