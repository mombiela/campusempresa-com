Gestió de Fitxers en Delphi/Object Pascal
Accés a Bases de Dades amb Delphi ►

La gestió de fitxers és una habilitat essencial per a qualsevol programador, ja que et permet llegir i escriure fitxers, permetent la persistència i manipulació de dades. Aquest tema et guiarà des dels conceptes bàsics fins als avançats de la gestió de fitxers en Delphi/Object Pascal.

Introducció a la Gestió de Fitxers

La gestió de fitxers en Delphi implica l'ús de diverses classes i procediments per crear, llegir, escriure i manipular fitxers. Delphi proporciona un conjunt ric de funcions i classes en les unitats System.SysUtils i System.Classes per treballar amb fitxers.

Conceptes Clau

  • Tipus de Fitxers: Fitxers de text, fitxers binaris.
  • Operacions de Fitxers: Crear, obrir, llegir, escriure, tancar i eliminar fitxers.
  • Modes de Fitxers: Llegir, escriure, afegir.

Operacions Bàsiques de Fitxers

Crear i Escriure en un Fitxer de Text

Per crear i escriure en un fitxer de text, pots utilitzar el tipus TextFile i els procediments associats.

program WriteToFile;

uses
  SysUtils;

var
  MyFile: TextFile;
  Text: string;
begin
  AssignFile(MyFile, 'example.txt');
  Rewrite(MyFile);  // Crear el fitxer per escriure
  
  Text := 'Hello, Delphi!';
  Writeln(MyFile, Text);  // Escriure text al fitxer
  
  CloseFile(MyFile);  // Tancar el fitxer
end.

Llegir d'un Fitxer de Text

Llegir d'un fitxer de text implica obrir el fitxer en mode de lectura i utilitzar Readln per llegir línies.

program ReadFromFile;

uses
  SysUtils;

var
  MyFile: TextFile;
  Text: string;
begin
  AssignFile(MyFile, 'example.txt');
  Reset(MyFile);  // Obrir el fitxer per llegir
  
  while not Eof(MyFile) do
  begin
    Readln(MyFile, Text);  // Llegir una línia del fitxer
    Writeln(Text);  // Mostrar la línia a la consola
  end;
  
  CloseFile(MyFile);  // Tancar el fitxer
end.

Gestió Avançada de Fitxers

Utilitzar TFileStream per a Fitxers Binaris

Per a operacions de fitxers més avançades, com la gestió de fitxers binaris, pots utilitzar la classe TFileStream.

Escriure en un Fitxer Binari

program WriteBinaryFile;

uses
  SysUtils, Classes;

var
  FileStream: TFileStream;
  Data: array[1..5] of Byte;
  I: Integer;
begin
  FileStream := TFileStream.Create('binary.dat', fmCreate);
  try
    for I := 1 to 5 do
      Data[I] := I * 10;  // Inicialitzar l'array de dades
    
    FileStream.WriteBuffer(Data, SizeOf(Data));  // Escriure dades al fitxer
  finally
    FileStream.Free;
  end;
end.

Llegir d'un Fitxer Binari

program ReadBinaryFile;

uses
  SysUtils, Classes;

var
  FileStream: TFileStream;
  Data: array[1..5] of Byte;
  I: Integer;
begin
  FileStream := TFileStream.Create('binary.dat', fmOpenRead);
  try
    FileStream.ReadBuffer(Data, SizeOf(Data));  // Llegir dades del fitxer
    
    for I := 1 to 5 do
      Writeln('Data[', I, '] = ', Data[I]);  // Mostrar les dades
  finally
    FileStream.Free;
  end;
end.

Gestionar Excepcions

Les operacions de fitxers poden fallar per diverses raons, com ara fitxers que falten o problemes de permisos. Utilitza blocs try..except per gestionar les excepcions de manera elegant.

program SafeFileHandling;

uses
  SysUtils;

var
  MyFile: TextFile;
  Text: string;
begin
  try
    AssignFile(MyFile, 'example.txt');
    Reset(MyFile);  // Intentar obrir el fitxer per llegir
    
    while not Eof(MyFile) do
    begin
      Readln(MyFile, Text);  // Llegir una línia del fitxer
      Writeln(Text);  // Mostrar la línia a la consola
    end;
    
    CloseFile(MyFile);  // Tancar el fitxer
  except
    on E: EInOutError do
      Writeln('S'ha produït un error en la gestió de fitxers: ', E.Message);
  end;
end.

Resum

En aquest tema, hem cobert els conceptes bàsics i avançats de la gestió de fitxers en Delphi/Object Pascal. Has après a:

  • Crear, escriure i llegir fitxers de text utilitzant TextFile.
  • Gestionar fitxers binaris utilitzant TFileStream.
  • Gestionar excepcions durant les operacions de fitxers.

La gestió de fitxers és una habilitat fonamental que et permetrà construir aplicacions més robustes i riques en funcionalitats. Practica aquests conceptes per convertir-te en un expert en la gestió de fitxers en Delphi.
