Introducció a les Classes i Objectes

Les classes i els objectes són conceptes fonamentals en la Programació Orientada a Objectes (OOP). En Delphi/Object Pascal, aquests conceptes et permeten crear estructures de dades i funcionalitats complexes de manera modular i reutilitzable.

Conceptes Clau

  • Classe: Un plànol per crear objectes. Defineix un tipus d'objecte segons els atributs (camps) i comportaments (mètodes) que hauria de tenir.
  • Objecte: Una instància d'una classe. Representa una implementació específica de la classe amb valors i comportaments reals.

Definició d'una Classe

Per definir una classe en Delphi/Object Pascal, utilitzes la paraula clau class. Aquí tens un exemple bàsic:

type
  TPerson = class
  private
    FName: string;
    FAge: Integer;
  public
    procedure SetName(AName: string);
    function GetName: string;
    procedure SetAge(AAge: Integer);
    function GetAge: Integer;
  end;

Explicació

  • type: Declara un nou tipus.
  • TPerson: El nom de la classe.
  • private: Secció on es declaren els camps privats. Aquests camps només són accessibles dins de la classe mateixa.
  • public: Secció on es declaren els mètodes públics. Aquests mètodes poden ser accessibles des de fora de la classe.

Creació i Ús d'Objectes

Per crear un objecte, utilitzes el mètode Create de la classe. Aquí tens un exemple de com crear i utilitzar un objecte de la classe TPerson:

var
  Person: TPerson;
begin
  Person := TPerson.Create;
  Person.SetName('John Doe');
  Person.SetAge(30);
  WriteLn('Name: ', Person.GetName);
  WriteLn('Age: ', Person.GetAge);
  Person.Free;
end;

Explicació

  • Person: Una variable de tipus TPerson.
  • TPerson.Create: Crea una instància de la classe TPerson.
  • SetName i SetAge: Mètodes per establir el nom i l'edat de la persona.
  • GetName i GetAge: Mètodes per obtenir el nom i l'edat de la persona.
  • Free: Allibera la memòria assignada per l'objecte.

Constructors i Destructors

Els constructors i destructors són mètodes especials utilitzats per inicialitzar i netejar objectes.

Exemple de Constructor

type
  TPerson = class
  private
    FName: string;
    FAge: Integer;
  public
    constructor Create(AName: string; AAge: Integer);
    destructor Destroy; override;
    procedure DisplayInfo;
  end;

constructor TPerson.Create(AName: string; AAge: Integer);
begin
  FName := AName;
  FAge := AAge;
end;

destructor TPerson.Destroy;
begin
  // Codi de neteja aquí
  inherited Destroy;
end;

procedure TPerson.DisplayInfo;
begin
  WriteLn('Name: ', FName);
  WriteLn('Age: ', FAge);
end;

Ús del Constructor

var
  Person: TPerson;
begin
  Person := TPerson.Create('Jane Doe', 25);
  Person.DisplayInfo;
  Person.Free;
end;

Explicació

  • constructor Create: Inicialitza l'objecte amb valors específics.
  • destructor Destroy: Neteja l'objecte abans que sigui destruït.
  • override: Indica que el mètode sobreescriu un mètode en la classe base.

Herència

L'herència permet que una classe hereti camps i mètodes d'una altra classe.

Exemple d'Herència

type
  TEmployee = class(TPerson)
  private
    FEmployeeID: Integer;
  public
    constructor Create(AName: string; AAge: Integer; AEmployeeID: Integer);
    procedure DisplayInfo; override;
  end;

constructor TEmployee.Create(AName: string; AAge: Integer; AEmployeeID: Integer);
begin
  inherited Create(AName, AAge);
  FEmployeeID := AEmployeeID;
end;

procedure TEmployee.DisplayInfo;
begin
  inherited DisplayInfo;
  WriteLn('Employee ID: ', FEmployeeID);
end;

Ús de la Classe Heretada

var
  Employee: TEmployee;
begin
  Employee := TEmployee.Create('John Smith', 40, 12345);
  Employee.DisplayInfo;
  Employee.Free;
end;

Explicació

  • TEmployee = class(TPerson): TEmployee hereta de TPerson.
  • inherited Create: Crida el constructor de la classe base.
  • override: Indica que el mètode sobreescriu un mètode en la classe base.

Polimorfisme

El polimorfisme permet que els mètodes siguin cridats en objectes de diferents classes a través d'una interfície comuna.

Exemple de Polimorfisme

type
  TPerson = class
  public
    procedure Speak; virtual;
  end;

  TEmployee = class(TPerson)
  public
    procedure Speak; override;
  end;

procedure TPerson.Speak;
begin
  WriteLn('Person speaking');
end;

procedure TEmployee.Speak;
begin
  WriteLn('Employee speaking');
end;

var
  Person: TPerson;
  Employee: TEmployee;
begin
  Person := TPerson.Create;
  Employee := TEmployee.Create;
  Person.Speak; // Mostra: Person speaking
  Employee.Speak; // Mostra: Employee speaking
  Person := Employee;
  Person.Speak; // Mostra: Employee speaking
  Person.Free;
  Employee.Free;
end;

Explicació

  • virtual: Indica que el mètode pot ser sobreescrit en una classe derivada.
  • override: Indica que el mètode sobreescriu un mètode en la classe base.
  • Person := Employee: Demostra el polimorfisme assignant un objecte Employee a una variable Person.

Conclusió

Les classes i els objectes són eines poderoses en Delphi/Object Pascal que et permeten crear codi modular, reutilitzable i mantenible. Comprenent i utilitzant conceptes com constructors, destructors, herència i polimorfisme, pots construir aplicacions complexes amb facilitat. Practica aquests conceptes amb diversos exemples per consolidar la teva comprensió i convertir-te en un expert en Programació Orientada a Objectes amb Delphi/Object Pascal.
