Introducció

Les estructures de dades dinàmiques són essencials per a una gestió eficient de la memòria i una manipulació flexible de les dades. En aquest tema, explorarem els punteres i les llistes enllaçades en Delphi/Object Pascal, començant pels conceptes bàsics i progressant cap a conceptes més avançats.

Punteres en Delphi/Object Pascal

Què és un Punter?

  • Un punter és una variable que conté l'adreça de memòria d'una altra variable.
  • Els punteres permeten l'assignació dinàmica de memòria i la manipulació d'estructures de dades com les llistes enllaçades.

Declaració i Ús de Punteres

var
  p: ^Integer; // Declara un punter a un enter
  num: Integer;
begin
  num := 10;
  p := @num; // p ara conté l'adreça de num
  WriteLn('Valor de num: ', num); // Mostra 10
  WriteLn('Valor apuntat per p: ', p^); // Mostra 10
end;
  • ^ s'utilitza per declarar un tipus de punter.
  • L'operador @ s'utilitza per obtenir l'adreça d'una variable.
  • p^ desreferencia el punter per accedir al valor emmagatzemat a l'adreça.

Assignació Dinàmica de Memòria

  • Utilitza New i Dispose per a l'assignació i desassignació dinàmica de memòria.
var
  p: ^Integer;
begin
  New(p); // Assigna memòria per a un enter
  p^ := 20;
  WriteLn('Valor apuntat per p: ', p^); // Mostra 20
  Dispose(p); // Desassigna la memòria
end;

Llistes Enllaçades

Què és una Llista Enllaçada?

  • Una llista enllaçada és una estructura de dades dinàmica que consisteix en nodes.
  • Cada node conté dades i un punter al següent node.

Tipus de Llistes Enllaçades

  • Llista Enllaçada Simple: Cada node apunta al següent node.
  • Llista Enllaçada Doble: Cada node apunta tant al següent com al node anterior.

Implementació d'una Llista Enllaçada Simple

Estructura del Node

type
  PNode = ^TNode;
  TNode = record
    Data: Integer;
    Next: PNode;
  end;
  • PNode és un punter a TNode.
  • TNode conté un camp de dades enter i un punter al següent node.

Creació i Manipulació d'una Llista Enllaçada

var
  Head, Temp: PNode;

procedure AddNode(var Head: PNode; Value: Integer);
var
  NewNode: PNode;
begin
  New(NewNode);
  NewNode^.Data := Value;
  NewNode^.Next := Head;
  Head := NewNode;
end;

procedure PrintList(Head: PNode);
begin
  Temp := Head;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Next;
  end;
end;

begin
  Head := nil;
  AddNode(Head, 10);
  AddNode(Head, 20);
  AddNode(Head, 30);
  PrintList(Head);
end;
  • El procediment AddNode afegeix un nou node al començament de la llista.
  • El procediment PrintList imprimeix tots els nodes de la llista.

Implementació d'una Llista Enllaçada Doble

Estructura del Node

type
  PDNode = ^TDNode;
  TDNode = record
    Data: Integer;
    Prev, Next: PDNode;
  end;
  • PDNode és un punter a TDNode.
  • TDNode conté un camp de dades enter, un punter al node anterior i un punter al següent node.

Creació i Manipulació d'una Llista Enllaçada Doble

var
  Head, Tail, Temp: PDNode;

procedure AddNode(var Head, Tail: PDNode; Value: Integer);
var
  NewNode: PDNode;
begin
  New(NewNode);
  NewNode^.Data := Value;
  NewNode^.Next := nil;
  NewNode^.Prev := Tail;
  if Tail <> nil then
    Tail^.Next := NewNode;
  Tail := NewNode;
  if Head = nil then
    Head := NewNode;
end;

procedure PrintListForward(Head: PDNode);
begin
  Temp := Head;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Next;
  end;
end;

procedure PrintListBackward(Tail: PDNode);
begin
  Temp := Tail;
  while Temp <> nil do
  begin
    WriteLn(Temp^.Data);
    Temp := Temp^.Prev;
  end;
end;

begin
  Head := nil;
  Tail := nil;
  AddNode(Head, Tail, 10);
  AddNode(Head, Tail, 20);
  AddNode(Head, Tail, 30);
  PrintListForward(Head);
  PrintListBackward(Tail);
end;
  • El procediment AddNode afegeix un nou node al final de la llista.
  • El procediment PrintListForward imprimeix tots els nodes des del cap fins a la cua.
  • El procediment PrintListBackward imprimeix tots els nodes des de la cua fins al cap.

Conclusió

Els punteres i les llistes enllaçades són eines poderoses en Delphi/Object Pascal per gestionar estructures de dades dinàmiques. Entendre com utilitzar els punteres per a l'assignació dinàmica de memòria i com implementar i manipular llistes enllaçades és crucial per a la programació avançada. Practica aquests conceptes amb diversos exercicis per consolidar la teva comprensió i millorar les teves habilitats de programació.
