Introducció a les Xarxes en Delphi/Object Pascal

  • Comprendre els conceptes bàsics de les xarxes.
  • Visió general de les capacitats de xarxa de Delphi.
  • Introducció a les biblioteques i components clau (Indy, Synapse).

Conceptes Bàsics de Xarxes

  • Adreces IP i Ports: Explicació de les adreces IP, ports i la seva importància en les xarxes.
  • Protocols: Visió general dels protocols comuns (TCP, UDP, HTTP, FTP).
  • Sòcols: Introducció als sòcols i el seu paper en la comunicació de xarxa.

Treballant amb Components Indy

  • Introducció a Indy: Visió general de la biblioteca Indy i els seus components per a xarxes.
  • Comunicació Client-Servidor: Model bàsic client-servidor utilitzant components Indy.

Exemple: Client TCP Simple

uses
  IdTCPClient, SysUtils;

var
  TCPClient: TIdTCPClient;
begin
  TCPClient := TIdTCPClient.Create(nil);
  try
    TCPClient.Host := '127.0.0.1';
    TCPClient.Port := 1234;
    TCPClient.Connect;
    TCPClient.IOHandler.WriteLn('Hola, Server!');
    Writeln(TCPClient.IOHandler.ReadLn);
  finally
    TCPClient.Free;
  end;
end.
  • Explicació: Aquest exemple demostra com crear un client TCP simple que es connecta a un servidor a 127.0.0.1 al port 1234, envia un missatge i llegeix la resposta.

Construint un Servidor HTTP Simple

  • Protocol HTTP: Conceptes bàsics de HTTP i com funciona.
  • Component Indy HTTP Server: Utilitzant TIdHTTPServer per crear un servidor HTTP simple.

Exemple: Servidor HTTP Bàsic

uses
  IdHTTPServer, IdCustomHTTPServer, SysUtils;

procedure OnCommandGet(AContext: TIdContext; ARequestInfo: TIdHTTPRequestInfo; AResponseInfo: TIdHTTPResponseInfo);
begin
  AResponseInfo.ContentText := 'Hola, Món!';
end;

var
  HTTPServer: TIdHTTPServer;
begin
  HTTPServer := TIdHTTPServer.Create(nil);
  try
    HTTPServer.DefaultPort := 8080;
    HTTPServer.OnCommandGet := OnCommandGet;
    HTTPServer.Active := True;
    Writeln('Servidor HTTP executant-se al port 8080. Prem Enter per aturar.');
    Readln;
  finally
    HTTPServer.Free;
  end;
end.
  • Explicació: Aquest exemple configura un servidor HTTP bàsic que respon amb "Hola, Món!" a qualsevol sol·licitud GET.

Tècniques Avançades de Xarxes

  • Xarxes Asíncrones: Gestionar operacions asíncrones en xarxes.
  • Multifil: Utilitzant fils per gestionar múltiples connexions simultàniament.

Exemple: Servidor TCP Multifil

uses
  IdTCPServer, IdContext, SysUtils, Classes;

procedure OnExecute(AContext: TIdContext);
var
  ReceivedText: string;
begin
  ReceivedText := AContext.Connection.IOHandler.ReadLn;
  AContext.Connection.IOHandler.WriteLn('Echo: ' + ReceivedText);
end;

var
  TCPServer: TIdTCPServer;
begin
  TCPServer := TIdTCPServer.Create(nil);
  try
    TCPServer.DefaultPort := 1234;
    TCPServer.OnExecute := OnExecute;
    TCPServer.Active := True;
    Writeln('Servidor TCP executant-se al port 1234. Prem Enter per aturar.');
    Readln;
  finally
    TCPServer.Free;
  end;
end.
  • Explicació: Aquest exemple demostra un servidor TCP multifil que retorna qualsevol missatge rebut.

Programació d'Internet amb Delphi

  • Serveis RESTful: Crear i consumir serveis web RESTful.
  • Serveis SOAP: Treballar amb serveis web basats en SOAP.
  • JSON i XML: Gestionar formats de dades JSON i XML.

Exemple: Consumint una API REST

uses
  IdHTTP, SysUtils, Classes;

var
  HTTPClient: TIdHTTP;
  Response: TStringStream;
begin
  HTTPClient := TIdHTTP.Create(nil);
  Response := TStringStream.Create;
  try
    HTTPClient.Get('https://api.example.com/data', Response);
    Writeln(Response.DataString);
  finally
    HTTPClient.Free;
    Response.Free;
  end;
end.
  • Explicació: Aquest exemple mostra com utilitzar TIdHTTP per enviar una sol·licitud GET a una API REST i imprimir la resposta.

Seguretat en Xarxes

  • SSL/TLS: Implementant comunicació segura utilitzant SSL/TLS.
  • Autenticació: Mètodes d'autenticació bàsica i basada en tokens.
  • Encriptació: Encriptant dades per a una transmissió segura.

Exemple: Client HTTPS

uses
  IdHTTP, IdSSLOpenSSL, SysUtils, Classes;

var
  HTTPClient: TIdHTTP;
  SSLHandler: TIdSSLIOHandlerSocketOpenSSL;
  Response: TStringStream;
begin
  HTTPClient := TIdHTTP.Create(nil);
  SSLHandler := TIdSSLIOHandlerSocketOpenSSL.Create(nil);
  Response := TStringStream.Create;
  try
    HTTPClient.IOHandler := SSLHandler;
    HTTPClient.Get('https://secure.example.com/data', Response);
    Writeln(Response.DataString);
  finally
    HTTPClient.Free;
    SSLHandler.Free;
    Response.Free;
  end;
end.
  • Explicació: Aquest exemple demostra com crear un client HTTPS utilitzant TIdHTTP i TIdSSLIOHandlerSocketOpenSSL per a una comunicació segura.

Conclusió

Hem explorat com utilitzar components Indy per a la comunicació client-servidor, construir servidors HTTP, gestionar operacions asíncrones i multifil, i assegurar la comunicació. En dominar aquests conceptes, estaràs ben equipat per desenvolupar aplicacions robustes en xarxa amb Delphi.
