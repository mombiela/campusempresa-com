Introducció a les Strings en Delphi/Object Pascal

Les strings són seqüències de caràcters utilitzades per representar text en programació. En Delphi/Object Pascal, les strings són un tipus de dada fonamental, i entendre com manipular-les és crucial per a una programació efectiva.

Conceptes Clau

  • Tipus de Strings: Diferents tipus de strings en Delphi, com ShortString, AnsiString, WideString i UnicodeString.
  • Declaració de Strings: Com declarar i inicialitzar strings.
  • Operacions amb Strings: Operacions bàsiques com la concatenació, la comparació i la determinació de la longitud.

Tipus de Strings en Delphi

Delphi suporta diversos tipus de strings, cadascun amb les seves característiques i casos d'ús.

ShortString

  • Una string de longitud fixa amb un màxim de 255 caràcters.
  • Útil per a la compatibilitat amb codi Pascal antic.
var
  s: ShortString;
begin
  s := 'Hola, Delphi!';
end;

AnsiString

  • Una string dinàmicament assignada que pot contenir qualsevol nombre de caràcters.
  • Adequada per manejar text ASCII.
var
  s: AnsiString;
begin
  s := 'Hola, Delphi!';
end;

WideString

  • Una string dinàmicament assignada dissenyada per contenir caràcters Unicode.
  • Útil per a la internacionalització.
var
  s: WideString;
begin
  s := 'Hola, Delphi!';
end;

UnicodeString

  • El tipus de string per defecte en les versions modernes de Delphi.
  • Suporta completament Unicode, fent-lo ideal per a la majoria d'aplicacions.
var
  s: UnicodeString;
begin
  s := 'Hola, Delphi!';
end;

Declaració i Inicialització de Strings

Declarar i inicialitzar strings en Delphi és senzill.

var
  greeting: string;
begin
  greeting := 'Hola, Món!';
end;

Operacions Bàsiques amb Strings

Concatenació

Combinar dues o més strings en una sola.

var
  s1, s2, result: string;
begin
  s1 := 'Hola, ';
  s2 := 'Món!';
  result := s1 + s2; // result és 'Hola, Món!'
end;

Comparació

Comparar dues strings per igualtat o ordre.

var
  s1, s2: string;
  isEqual: Boolean;
begin
  s1 := 'Hola';
  s2 := 'Món';
  isEqual := s1 = s2; // isEqual és Fals
end;

Determinació de la Longitud

Trobar el nombre de caràcters en una string.

var
  s: string;
  len: Integer;
begin
  s := 'Hola, Món!';
  len := Length(s); // len és 10
end;

Maneig Avançat de Strings

Funcions de Strings

Delphi proporciona un conjunt ric de funcions de strings integrades per a la manipulació avançada.

  • Copy: Extreu una subcadena d'una string.
  • Pos: Troba la posició d'una subcadena dins d'una string.
  • Delete: Elimina una part d'una string.
  • Insert: Insereix una subcadena en una string.
var
  s, sub: string;
  pos: Integer;
begin
  s := 'Hola, Delphi!';
  sub := Copy(s, 6, 6); // sub és 'Delphi'
  pos := Pos('Delphi', s); // pos és 6
  Delete(s, 6, 6); // s és 'Hola, !'
  Insert('Món', s, 6); // s és 'Hola, Món!'
end;

Formatació de Strings

Formatar strings utilitzant la funció Format.

var
  s: string;
begin
  s := Format('Hola, %s! Tens %d missatges nous.', ['Alice', 5]);
  // s és 'Hola, Alice! Tens 5 missatges nous.'
end;

Conversió de Strings

Convertir entre diferents tipus de dades i strings.

var
  i: Integer;
  s: string;
begin
  i := 123;
  s := IntToStr(i); // s és '123'
  i := StrToInt(s); // i és 123
end;

Conclusió

Les strings són una part versàtil i essencial de la programació en Delphi. Entendre els diferents tipus de strings, operacions bàsiques i tècniques de maneig avançat et permetrà manipular text de manera efectiva en les teves aplicacions. Practica aquests conceptes amb diversos exercicis per consolidar la teva comprensió i millorar la teva competència en el maneig de strings.
