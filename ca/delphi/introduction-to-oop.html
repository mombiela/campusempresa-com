Introducció a la POO en Delphi/Object Pascal
Classes i Objectes ►

Visió general

La Programació Orientada a Objectes (POO) és un paradigma que utilitza "objectes" per dissenyar aplicacions i programes informàtics. Delphi, que utilitza el llenguatge Object Pascal, és una eina poderosa per implementar conceptes de POO. Aquesta secció us introduirà als conceptes bàsics de la POO en Delphi, cobrint conceptes clau, sintaxi i exemples pràctics.

Conceptes clau de la POO

  • Classe: Un plànol per crear objectes (una estructura de dades particular), que conté mètodes i propietats.
  • Objecte: Una instància d'una classe.
  • Encapsulació: L'agrupació de dades amb els mètodes que operen sobre aquestes dades.
  • Herència: El mecanisme pel qual una classe pot heretar trets d'una altra classe.
  • Polimorfisme: La capacitat de presentar la mateixa interfície per a diferents formes subjacents (tipus de dades).

Classes i objectes

Definició d'una classe

En Delphi, una classe es defineix utilitzant la paraula clau class. Aquí teniu un exemple senzill:

type
  TPerson = class
  private
    FName: string;
    FAge: Integer;
  public
    procedure SetName(AName: string);
    function GetName: string;
    procedure SetAge(AAge: Integer);
    function GetAge: Integer;
  end;

Creació d'un objecte

Per crear un objecte, utilitzeu el mètode Create de la classe:

var
  Person: TPerson;
begin
  Person := TPerson.Create;
  Person.SetName('John Doe');
  Person.SetAge(30);
  ShowMessage(Person.GetName + ' té ' + IntToStr(Person.GetAge) + ' anys.');
  Person.Free;
end;

Encapsulació

L'encapsulació s'aconsegueix utilitzant especificadors d'accés com private, protected i public. En l'exemple anterior, FName i FAge són privats, el que significa que només poden ser accessibles dins de la classe TPerson.

Herència

L'herència permet que una classe hereti propietats i mètodes d'una altra classe. Aquí teniu un exemple:

type
  TEmployee = class(TPerson)
  private
    FEmployeeID: string;
  public
    procedure SetEmployeeID(AEmployeeID: string);
    function GetEmployeeID: string;
  end;

Polimorfisme

El polimorfisme en Delphi s'aconsegueix típicament a través de la sobrescriptura de mètodes i interfícies. Aquí teniu un exemple de sobrescriptura de mètodes:

type
  TPerson = class
  public
    procedure Speak; virtual;
  end;

  TEmployee = class(TPerson)
  public
    procedure Speak; override;
  end;

procedure TPerson.Speak;
begin
  ShowMessage('Hola, sóc una persona.');
end;

procedure TEmployee.Speak;
begin
  ShowMessage('Hola, sóc un empleat.');
end;

Exemple pràctic

Creem una aplicació senzilla que demostri aquests conceptes:

type
  TPerson = class
  private
    FName: string;
    FAge: Integer;
  public
    procedure SetName(AName: string);
    function GetName: string;
    procedure SetAge(AAge: Integer);
    function GetAge: Integer;
    procedure Speak; virtual;
  end;

  TEmployee = class(TPerson)
  private
    FEmployeeID: string;
  public
    procedure SetEmployeeID(AEmployeeID: string);
    function GetEmployeeID: string;
    procedure Speak; override;
  end;

procedure TPerson.SetName(AName: string);
begin
  FName := AName;
end;

function TPerson.GetName: string;
begin
  Result := FName;
end;

procedure TPerson.SetAge(AAge: Integer);
begin
  FAge := AAge;
end;

function TPerson.GetAge: Integer;
begin
  Result := FAge;
end;

procedure TPerson.Speak;
begin
  ShowMessage('Hola, sóc una persona.');
end;

procedure TEmployee.SetEmployeeID(AEmployeeID: string);
begin
  FEmployeeID := AEmployeeID;
end;

function TEmployee.GetEmployeeID: string;
begin
  Result := FEmployeeID;
end;

procedure TEmployee.Speak;
begin
  ShowMessage('Hola, sóc un empleat.');
end;

var
  Person: TPerson;
  Employee: TEmployee;
begin
  Person := TPerson.Create;
  Person.SetName('John Doe');
  Person.SetAge(30);
  Person.Speak;
  Person.Free;

  Employee := TEmployee.Create;
  Employee.SetName('Jane Smith');
  Employee.SetAge(28);
  Employee.SetEmployeeID('E12345');
  Employee.Speak;
  Employee.Free;
end;

Conclusió

En aquesta secció, hem introduït els conceptes fonamentals de la Programació Orientada a Objectes en Delphi/Object Pascal. Hem cobert:

  • La definició i creació de classes i objectes.
  • Encapsulació, herència i polimorfisme.
  • Exemples pràctics per il·lustrar aquests conceptes.

Entendre aquests conceptes bàsics proporcionarà una base sòlida per a temes més avançats en la programació amb Delphi. A mesura que avanceu, veureu com s'apliquen aquests principis en aplicacions del món real, fent que el vostre codi sigui més modular, reutilitzable i fàcil de mantenir.
