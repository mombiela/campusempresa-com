Sentències condicionals en Delphi/Object Pascal
Bucles: For, While i Repeat ►

Les sentències condicionals són fonamentals en la programació ja que permeten l'execució de codi basat en certes condicions. En Delphi/Object Pascal, les sentències condicionals inclouen if, case i else. Aquest tema cobrirà aquestes sentències des d'un nivell principiant fins a un nivell avançat.

Sentències If

La sentència if s'utilitza per executar un bloc de codi si una condició especificada és certa.

Sentència If bàsica

La forma més simple d'una sentència if comprova una sola condició.

var
  number: Integer;
begin
  number := 10;
  if number > 5 then
    WriteLn('El número és més gran que 5');
end.

Sentència If-Else

La sentència if-else proporciona un bloc de codi alternatiu per executar si la condició és falsa.

var
  number: Integer;
begin
  number := 3;
  if number > 5 then
    WriteLn('El número és més gran que 5')
  else
    WriteLn('El número és 5 o menys');
end.

Sentències If imbricades

Pots imbricar sentències if per comprovar múltiples condicions.

var
  number: Integer;
begin
  number := 7;
  if number > 5 then
  begin
    if number < 10 then
      WriteLn('El número està entre 6 i 9');
  end
  else
    WriteLn('El número és 5 o menys');
end.

Sentències Case

La sentència case s'utilitza per executar un dels diversos blocs de codi basats en el valor d'una expressió.

Sentència Case bàsica

Una simple sentència case comprova el valor d'una sola variable.

var
  grade: Char;
begin
  grade := 'B';
  case grade of
    'A': WriteLn('Excel·lent');
    'B': WriteLn('Bé');
    'C': WriteLn('Regular');
    'D': WriteLn('Per sota de la mitjana');
    'F': WriteLn('Suspès');
  else
    WriteLn('Nota invàlida');
  end;
end.

Sentència Case amb rangs

La sentència case també pot gestionar rangs de valors.

var
  score: Integer;
begin
  score := 85;
  case score of
    90..100: WriteLn('Nota A');
    80..89: WriteLn('Nota B');
    70..79: WriteLn('Nota C');
    60..69: WriteLn('Nota D');
    0..59: WriteLn('Nota F');
  else
    WriteLn('Puntuació invàlida');
  end;
end.

Sentències condicionals avançades

Combinació de condicions amb operadors lògics

Pots combinar múltiples condicions utilitzant operadors lògics com and, or i not.

var
  age: Integer;
  hasLicense: Boolean;
begin
  age := 18;
  hasLicense := True;
  if (age >= 18) and hasLicense then
    WriteLn('Pots conduir')
  else
    WriteLn('No pots conduir');
end.

Ús de funcions en condicions

Pots cridar funcions dins de sentències condicionals per prendre decisions basades en una lògica més complexa.

function IsEven(number: Integer): Boolean;
begin
  Result := (number mod 2) = 0;
end;

var
  number: Integer;
begin
  number := 4;
  if IsEven(number) then
    WriteLn('El número és parell')
  else
    WriteLn('El número és senar');
end.

Conclusió

Les sentències condicionals són essencials per controlar el flux d'un programa. Delphi/Object Pascal proporciona construccions poderoses com if, else i case per gestionar diverses condicions. En dominar aquestes sentències, pots escriure programes més dinàmics i responsius. Practica utilitzant aquestes construccions amb diferents escenaris per convertir-te en un expert en el seu ús.
