Introducció a les Sentències Case

Les sentències case en Delphi/Object Pascal s'utilitzen per simplificar la lògica condicional complexa. Proporcionen una manera d'executar diferents blocs de codi basats en el valor d'una variable o expressió.

Conceptes Clau

  • Estructura de la Sentència Case: L'estructura bàsica d'una sentència case.
  • Sentències Case Simples: Utilitzar sentències case amb tipus ordinals.
  • Sentències Case Imbricades: Utilitzar sentències case dins d'altres sentències case.
  • Sentències Case amb Strings: Utilitzar sentències case amb valors de cadena.
  • Case per Defecte: Gestionar valors inesperats.

Estructura de la Sentència Case

La sintaxi general d'una sentència case en Delphi/Object Pascal és la següent:

case expression of
  value1: statement1;
  value2: statement2;
  value3: statement3;
  ...
else
  default_statement;
end;
  • expression: La variable o expressió que s'està avaluant.
  • value1, value2, value3, ...: Valors possibles de l'expressió.
  • statement1, statement2, statement3, ...: Sentències a executar per a cada valor corresponent.
  • else: Opcional. S'executa si cap dels valors especificats coincideix amb l'expressió.

Sentències Case Simples

Un exemple de sentència case simple amb un tipus ordinal (enter):

var
  day: Integer;
begin
  day := 3;
  case day of
    1: WriteLn('Dilluns');
    2: WriteLn('Dimarts');
    3: WriteLn('Dimecres');
    4: WriteLn('Dijous');
    5: WriteLn('Divendres');
    6: WriteLn('Dissabte');
    7: WriteLn('Diumenge');
  else
    WriteLn('Dia invàlid');
  end;
end;

Explicació

  • La variable day s'avalua.
  • Depenent del valor de day, s'executa la sentència WriteLn corresponent.
  • Si day no està entre 1 i 7, s'executa la clàusula else, imprimint 'Dia invàlid'.

Sentències Case Imbricades

Les sentències case poden estar imbricades dins d'altres sentències case per gestionar una lògica més complexa:

var
  category, item: Integer;
begin
  category := 1;
  item := 2;
  case category of
    1: begin
      case item of
        1: WriteLn('Categoria 1, Element 1');
        2: WriteLn('Categoria 1, Element 2');
      else
        WriteLn('Element desconegut en Categoria 1');
      end;
    end;
    2: begin
      case item of
        1: WriteLn('Categoria 2, Element 1');
        2: WriteLn('Categoria 2, Element 2');
      else
        WriteLn('Element desconegut en Categoria 2');
      end;
    end;
  else
    WriteLn('Categoria desconeguda');
  end;
end;

Explicació

  • La sentència case externa avalua category.
  • Depenent del valor de category, una sentència case interna avalua item.
  • Aquesta estructura permet una gestió detallada de les combinacions de category i item.

Sentències Case amb Strings

Delphi/Object Pascal també suporta sentències case amb valors de cadena:

var
  command: string;
begin
  command := 'start';
  case command of
    'start': WriteLn('Iniciant el procés');
    'stop': WriteLn('Aturant el procés');
    'pause': WriteLn('Pausant el procés');
  else
    WriteLn('Comanda desconeguda');
  end;
end;

Explicació

  • La variable command s'avalua.
  • Depenent del valor de command, s'executa la sentència WriteLn corresponent.
  • Si command no coincideix amb cap valor especificat, s'executa la clàusula else, imprimint 'Comanda desconeguda'.

Case per Defecte

La clàusula else en una sentència case actua com un case per defecte, gestionant qualsevol valor no explícitament llistat:

var
  grade: Char;
begin
  grade := 'B';
  case grade of
    'A': WriteLn('Excel·lent');
    'B': WriteLn('Bé');
    'C': WriteLn('Mitjà');
    'D': WriteLn('Per sota de la mitjana');
    'F': WriteLn('Suspès');
  else
    WriteLn('Nota invàlida');
  end;
end;

Explicació

  • La variable grade s'avalua.
  • Depenent del valor de grade, s'executa la sentència WriteLn corresponent.
  • Si grade no coincideix amb cap valor especificat, s'executa la clàusula else, imprimint 'Nota invàlida'.

Conclusió

Les sentències case en Delphi/Object Pascal proporcionen una manera poderosa de gestionar múltiples condicions amb codi clar i llegible. Es poden utilitzar amb tipus ordinals, cadenes, i es poden imbricar per a lògica complexa. La clàusula else assegura que els valors inesperats es gestionin de manera adequada. Dominant les sentències case, pots escriure codi més eficient i mantenible.
