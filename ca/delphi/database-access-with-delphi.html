Introducció a l'Accés a Bases de Dades

Delphi, amb el seu robust llenguatge Object Pascal, proporciona eines poderoses per a l'accés i manipulació de bases de dades. Aquesta secció et guiarà a través dels conceptes bàsics de la connexió a bases de dades, l'execució de consultes i la gestió de dades en Delphi.

Conceptes Clau

  • Components de Base de Dades: Comprendre els components proporcionats per Delphi per a l'accés a bases de dades.
  • Connexions a Bases de Dades: Establir i gestionar connexions a diversos tipus de bases de dades.
  • Execució de Consultes: Executar consultes SQL per interactuar amb la base de dades.
  • Gestió de Dades: Recuperar i manipular dades dins de les aplicacions Delphi.

Components de Base de Dades en Delphi

Delphi ofereix una varietat de components per facilitar l'accés a bases de dades. Aquests components formen part de les biblioteques Data Access i Data Controls.

Components Clau

  • TDataSource: Actua com un pont entre els controls de dades i els components de dataset.
  • TQuery: S'utilitza per executar consultes SQL.
  • TTable: Proporciona accés a taules de bases de dades.
  • TDatabase: Gestiona connexions a bases de dades.
  • TDBGrid: Mostra dades en un format tabular.

Exemple: Configuració Bàsica de Components de Base de Dades

procedure TForm1.FormCreate(Sender: TObject);
begin
  // Configuració d'una connexió a la base de dades
  Database1.DatabaseName := 'MyDatabase';
  Database1.Params.Values['USER NAME'] := 'username';
  Database1.Params.Values['PASSWORD'] := 'password';
  Database1.Connected := True;

  // Configuració d'una consulta
  Query1.DatabaseName := 'MyDatabase';
  Query1.SQL.Text := 'SELECT * FROM Customers';
  Query1.Open;

  // Enllaçar la consulta a una font de dades
  DataSource1.DataSet := Query1;

  // Enllaçar la font de dades a un DBGrid
  DBGrid1.DataSource := DataSource1;
end;

Establir Connexions a Bases de Dades

Connectar-se a una base de dades és el primer pas en l'accés a bases de dades. Delphi suporta diversos sistemes de bases de dades com MySQL, SQLite, PostgreSQL, i més.

Exemple: Connexió a una Base de Dades MySQL

procedure TForm1.ConnectToMySQL;
begin
  MySQLConnection1.DatabaseName := 'MyDatabase';
  MySQLConnection1.UserName := 'username';
  MySQLConnection1.Password := 'password';
  MySQLConnection1.HostName := 'localhost';
  MySQLConnection1.Connected := True;
end;

Execució de Consultes

Un cop connectat a una base de dades, pots executar consultes SQL per recuperar o manipular dades.

Exemple: Execució d'una Consulta SELECT

procedure TForm1.ExecuteSelectQuery;
begin
  Query1.SQL.Text := 'SELECT * FROM Customers WHERE Country = :Country';
  Query1.Params.ParamByName('Country').AsString := 'USA';
  Query1.Open;
end;

Exemple: Execució d'una Consulta INSERT

procedure TForm1.ExecuteInsertQuery;
begin
  Query1.SQL.Text := 'INSERT INTO Customers (Name, Country) VALUES (:Name, :Country)';
  Query1.Params.ParamByName('Name').AsString := 'John Doe';
  Query1.Params.ParamByName('Country').AsString := 'USA';
  Query1.ExecSQL;
end;

Gestió de Dades

La gestió de dades recuperades de la base de dades implica mostrar-les a la interfície d'usuari i permetre als usuaris interactuar amb elles.

Exemple: Mostrar Dades en un DBGrid

procedure TForm1.DisplayDataInGrid;
begin
  Query1.SQL.Text := 'SELECT * FROM Customers';
  Query1.Open;
  DBGrid1.DataSource := DataSource1;
end;

Exemple: Actualitzar Dades

procedure TForm1.UpdateCustomerData;
begin
  Query1.SQL.Text := 'UPDATE Customers SET Country = :Country WHERE Name = :Name';
  Query1.Params.ParamByName('Country').AsString := 'Canada';
  Query1.Params.ParamByName('Name').AsString := 'John Doe';
  Query1.ExecSQL;
end;

Conclusió

En aquest tema, hem cobert els aspectes essencials de l'accés a bases de dades en Delphi, incloent els components clau, l'establiment de connexions, l'execució de consultes i la gestió de dades. Comprenent aquests conceptes i practicant amb els exemples proporcionats, estaràs ben preparat per construir aplicacions robustes de bases de dades utilitzant Delphi.

Recorda sempre gestionar les connexions i consultes a bases de dades amb cura per assegurar la seguretat i integritat de les teves dades. Bona programació!
