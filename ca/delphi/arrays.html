Introducció als Arrays

Els arrays són una estructura de dades fonamental en Delphi/Object Pascal, que et permeten emmagatzemar múltiples valors del mateix tipus en una sola variable. Aquesta secció cobrirà els conceptes bàsics dels arrays, incloent la seva declaració, inicialització i ús.

Conceptes Clau

  • Declaració d'Arrays: Com declarar arrays en Delphi/Object Pascal.
  • Inicialització d'Arrays: Com inicialitzar arrays amb valors.
  • Accés als Elements de l'Array: Com llegir i escriure elements de l'array.
  • Arrays Estàtics vs. Dinàmics: Entendre la diferència entre arrays estàtics i dinàmics.

Declaració d'Arrays

En Delphi/Object Pascal, els arrays es poden declarar de diverses maneres depenent del seu tipus i mida.

Arrays Estàtics

Els arrays estàtics tenen una mida fixa que es determina en temps de compilació.

var
  Numbers: array[1..5] of Integer;

En aquest exemple:

  • Numbers és un array que pot contenir 5 enters.
  • L'índex va de l'1 al 5.

Arrays Dinàmics

Els arrays dinàmics poden canviar de mida en temps d'execució.

var
  Numbers: array of Integer;

En aquest exemple:

  • Numbers és un array dinàmic d'enters.
  • La mida de l'array no es especifica en la declaració.

Inicialització d'Arrays

Després de declarar un array, pots inicialitzar-lo amb valors.

Arrays Estàtics

Els arrays estàtics es poden inicialitzar element per element o utilitzant un bucle.

var
  Numbers: array[1..5] of Integer;
begin
  Numbers[1] := 10;
  Numbers[2] := 20;
  Numbers[3] := 30;
  Numbers[4] := 40;
  Numbers[5] := 50;
end;

Arrays Dinàmics

Els arrays dinàmics necessiten ser redimensionats abans de la inicialització.

var
  Numbers: array of Integer;
begin
  SetLength(Numbers, 5); // Redimensiona l'array per contenir 5 elements
  Numbers[0] := 10;
  Numbers[1] := 20;
  Numbers[2] := 30;
  Numbers[3] := 40;
  Numbers[4] := 50;
end;

Accés als Elements de l'Array

Pots accedir i modificar els elements de l'array utilitzant el seu índex.

var
  Numbers: array[1..5] of Integer;
begin
  Numbers[1] := 10;
  WriteLn(Numbers[1]); // Mostra: 10
end;

Arrays Estàtics vs. Dinàmics

Entendre les diferències entre arrays estàtics i dinàmics és crucial per a una gestió eficient de la memòria i el rendiment.

| Característica | Arrays Estàtics | Arrays Dinàmics | |-----------------|---------------------------------|---------------------------------| | Mida | Fixa en temps de compilació | Es pot redimensionar en temps d'execució | | Assignació de Memòria | Assignada a la pila | Assignada al munt | | Flexibilitat | Menys flexible | Més flexible | | Rendiment | Accés més ràpid | Accés més lent degut a l'assignació al munt |

Operacions Avançades amb Arrays

A mesura que et facis més proficient, podràs realitzar operacions més complexes amb arrays.

Arrays Multidimensionals

Delphi/Object Pascal suporta arrays multidimensionals.

var
  Matrix: array[1..3, 1..3] of Integer;
begin
  Matrix[1, 1] := 1;
  Matrix[1, 2] := 2;
  Matrix[1, 3] := 3;
  // Continua inicialitzant altres elements
end;

Arrays Multidimensionals Dinàmics

Els arrays multidimensionals dinàmics també es poden crear i redimensionar.

var
  Matrix: array of array of Integer;
begin
  SetLength(Matrix, 3, 3); // Crea una matriu de 3x3
  Matrix[0][0] := 1;
  Matrix[0][1] := 2;
  Matrix[0][2] := 3;
  // Continua inicialitzant altres elements
end;

Array de Registres

Pots crear arrays de tipus complexos com registres.

type
  TPerson = record
    Name: string;
    Age: Integer;
  end;

var
  People: array[1..3] of TPerson;
begin
  People[1].Name := 'Alice';
  People[1].Age := 30;
  // Continua inicialitzant altres elements
end;

Conclusió

Els arrays són una estructura de dades potent i versàtil en Delphi/Object Pascal. Entendre com declarar, inicialitzar i manipular arrays és essencial per a una programació eficient. Tant si treballes amb arrays estàtics com dinàmics, dominar aquests conceptes et permetrà gestionar col·leccions de dades de manera efectiva. A mesura que avancis, podràs explorar operacions més complexes amb arrays, com arrays multidimensionals i arrays de registres, per millorar encara més les teves habilitats de programació.
