Visió General

Delphi és un potent Entorn de Desenvolupament Integrat (IDE) per al llenguatge de programació Object Pascal. És àmpliament utilitzat per desenvolupar aplicacions per a Windows, però també suporta el desenvolupament multiplataforma per a macOS, iOS, Android i Linux. Object Pascal, el llenguatge utilitzat en Delphi, és una extensió del llenguatge Pascal, dissenyada per suportar la programació orientada a objectes.

Conceptes Clau

Què és Delphi?

  • IDE: Delphi és un IDE que proporciona eines per dissenyar, codificar, depurar i desplegar aplicacions.
  • RAD: Les capacitats de Desenvolupament Ràpid d'Aplicacions (RAD) permeten una ràpida prototipació i desenvolupament.
  • Multiplataforma: Suporta múltiples plataformes incloent Windows, macOS, iOS, Android i Linux.

Què és Object Pascal?

  • Extensió de Pascal: Object Pascal amplia el llenguatge Pascal amb característiques orientades a objectes.
  • Tipatge Fort: Assegura la seguretat de tipus i redueix els errors en temps d'execució.
  • Sintaxi Llegible: Conegut per la seva sintaxi clara i llegible, fent-lo una bona elecció per a principiants.

Configuració de Delphi

  1. Descarregar i Instal·lar: Obteniu Delphi des del lloc web oficial d'Embarcadero i seguiu les instruccions d'instal·lació.
  2. Primer Projecte: Creeu un nou projecte per familiaritzar-vos amb la disposició de l'IDE i les característiques bàsiques.

Sintaxi i Estructura Bàsiques

Exemple de Hello World

program HelloWorld;

begin
  WriteLn('Hello, World!');
end.
  • program: Declara el nom del programa.
  • begin...end: Marca el començament i el final del bloc del programa.
  • WriteLn: Mostra text a la consola.

Variables i Tipus de Dades

var
  age: Integer;
  name: String;
begin
  age := 25;
  name := 'Alice';
  WriteLn('Name: ', name, ', Age: ', age);
end.
  • var: Declara variables.
  • Integer, String: Tipus de dades comuns.
  • :=: Operador d'assignació.

Estructures de Control

If-Else

var
  score: Integer;
begin
  score := 85;
  if score >= 90 then
    WriteLn('Grade: A')
  else if score >= 80 then
    WriteLn('Grade: B')
  else
    WriteLn('Grade: C');
end.

Bucle For

var
  i: Integer;
begin
  for i := 1 to 10 do
    WriteLn('Number: ', i);
end.

Programació Orientada a Objectes en Object Pascal

Classes i Objectes

type
  TPerson = class
    private
      FName: String;
      FAge: Integer;
    public
      constructor Create(Name: String; Age: Integer);
      procedure DisplayInfo;
  end;

constructor TPerson.Create(Name: String; Age: Integer);
begin
  FName := Name;
  FAge := Age;
end;

procedure TPerson.DisplayInfo;
begin
  WriteLn('Name: ', FName, ', Age: ', FAge);
end;

var
  person: TPerson;
begin
  person := TPerson.Create('Alice', 25);
  person.DisplayInfo;
end.
  • type: Defineix un nou tipus de classe.
  • private, public: Especificadors d'accés.
  • constructor: Mètode especial per inicialitzar objectes.
  • procedure: Defineix un mètode.

Conclusió

En aquest tema introductori, hem cobert els conceptes bàsics de Delphi i Object Pascal, incloent la configuració de l'entorn, la comprensió de la sintaxi bàsica i l'inici amb la programació orientada a objectes. A mesura que avanceu en el curs, aprofundireu en característiques i tècniques avançades, permetent-vos construir aplicacions robustes i eficients utilitzant Delphi i Object Pascal.
