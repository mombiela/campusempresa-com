Passatge de Paràmetres en Delphi/Object Pascal
Abast i Durada de les Variables ►

El passatge de paràmetres és un concepte fonamental en programació que implica proporcionar entrades a funcions o procediments. En Delphi/Object Pascal, entendre com es passen els paràmetres pot ajudar-te a escriure codi més eficient i efectiu. Aquest tema cobrirà els conceptes bàsics del passatge de paràmetres, diferents tipus de paràmetres i tècniques avançades.

Conceptes Bàsics del Passatge de Paràmetres

En Delphi/Object Pascal, els paràmetres es poden passar a procediments i funcions de diverses maneres. Els mètodes principals són:

  • Per Valor
  • Per Referència
  • Per Constant

Per Valor

Quan els paràmetres es passen per valor, es fa una còpia del valor del paràmetre real en memòria. Això significa que els canvis fets al paràmetre dins de la funció o procediment no afecten la variable original.

procedure IncrementByValue(a: Integer);
begin
  a := a + 1;
end;

var
  num: Integer;
begin
  num := 5;
  IncrementByValue(num);
  // num encara és 5
end;

Per Referència

Passar paràmetres per referència permet que la funció o procediment modifiqui la variable original. Això es fa utilitzant la paraula clau var.

procedure IncrementByReference(var a: Integer);
begin
  a := a + 1;
end;

var
  num: Integer;
begin
  num := 5;
  IncrementByReference(num);
  // num ara és 6
end;

Per Constant

Passar paràmetres per constant es fa utilitzant la paraula clau const. Aquest mètode és similar a passar per valor, però impedeix que la funció o procediment modifiqui el paràmetre. També pot millorar el rendiment per a tipus de dades grans.

procedure PrintValue(const a: Integer);
begin
  WriteLn(a);
end;

var
  num: Integer;
begin
  num := 5;
  PrintValue(num);
  // num encara és 5
end;

Tècniques Avançades de Passatge de Paràmetres

Paràmetres per Defecte

Delphi permet definir valors per defecte per als paràmetres. Això pot fer que les teves funcions i procediments siguin més flexibles i fàcils d'utilitzar.

procedure Greet(name: String = 'World');
begin
  WriteLn('Hello, ' + name + '!');
end;

begin
  Greet(); // Mostra: Hello, World!
  Greet('Alice'); // Mostra: Hello, Alice!
end;

Paràmetres de Matriu Oberta

Els paràmetres de matriu oberta permeten passar matrius de qualsevol mida a un procediment o funció. Això és útil quan el nombre d'elements no es coneix en temps de compilació.

procedure PrintArray(const arr: array of Integer);
var
  i: Integer;
begin
  for i := 0 to High(arr) do
    WriteLn(arr[i]);
end;

begin
  PrintArray([1, 2, 3, 4, 5]);
end;

Nombre Variable de Paràmetres

Delphi admet funcions i procediments amb un nombre variable de paràmetres utilitzant la sintaxi array of const. Això és útil per a funcions com Format.

procedure PrintValues(const Args: array of const);
var
  i: Integer;
begin
  for i := 0 to High(Args) do
    WriteLn(Args[i].VInteger);
end;

begin
  PrintValues([1, 2, 3, 4, 5]);
end;

Resum

Entendre el passatge de paràmetres en Delphi/Object Pascal és crucial per escriure codi eficient i efectiu. Aquí tens els punts clau:

  • Per Valor: Es fa una còpia del paràmetre; els canvis no afecten la variable original.
  • Per Referència: La variable original es pot modificar.
  • Per Constant: Similar a per valor però impedeix la modificació i pot millorar el rendiment.
  • Paràmetres per Defecte: Permeten especificar valors per defecte per als paràmetres.
  • Paràmetres de Matriu Oberta: Permeten passar matrius de qualsevol mida.
  • Nombre Variable de Paràmetres: Permeten que les funcions acceptin un nombre variable d'arguments.

En dominar aquestes tècniques, podràs escriure programes en Delphi/Object Pascal més flexibles i potents.
