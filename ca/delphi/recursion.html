Recursió en Delphi/Object Pascal
Arrays ►

La recursió és un concepte fonamental en ciències de la computació i programació on una funció es crida a si mateixa per resoldre instàncies més petites del mateix problema. Aquest tema cobrirà els conceptes bàsics de la recursió, proporcionarà exemples i aprofundirà en usos més avançats en Delphi/Object Pascal.

Què és la Recursió?

  • Definició: La recursió ocorre quan una funció es crida a si mateixa directament o indirectament per resoldre un problema.
  • Cas Base: La condició sota la qual la funció recursiva deixa de cridar-se a si mateixa per evitar bucles infinits.
  • Cas Recursiu: La part de la funció on ocorre la recursió, trencant el problema en instàncies més petites.

Exemple Bàsic: Càlcul del Factorial

El factorial d'un enter no negatiu n és el producte de tots els enters positius menors o iguals a n. Es denota per n!.

Funció Factorial en Delphi/Object Pascal

function Factorial(n: Integer): Integer;
begin
  if n = 0 then
    Result := 1  // Cas base: 0! és 1
  else
    Result := n * Factorial(n - 1);  // Cas recursiu
end;
  • Explicació:
    • La funció Factorial pren un enter n.
    • Si n és 0, retorna 1 (cas base).
    • Altrament, retorna n multiplicat pel factorial de n-1 (cas recursiu).

Conceptes Clau

  • Cas Base: Essencial per evitar la recursió infinita.
  • Cas Recursiu: La funció es crida a si mateixa amb un argument més petit o més simple.
  • Desbordament de Pila: Massa crides recursives poden portar a un error de desbordament de pila.

Exemple Avançat: Seqüència de Fibonacci

La seqüència de Fibonacci és una sèrie de nombres on cada nombre és la suma dels dos precedents, normalment començant amb 0 i 1.

Funció Fibonacci en Delphi/Object Pascal

function Fibonacci(n: Integer): Integer;
begin
  if n <= 1 then
    Result := n  // Cas base: Fibonacci(0) = 0, Fibonacci(1) = 1
  else
    Result := Fibonacci(n - 1) + Fibonacci(n - 2);  // Cas recursiu
end;
  • Explicació:
    • La funció Fibonacci pren un enter n.
    • Si n és 0 o 1, retorna n (cas base).
    • Altrament, retorna la suma del Fibonacci de n-1 i n-2 (cas recursiu).

Optimització de la Recursió

Memoització

La memoització és una tècnica per optimitzar funcions recursives emmagatzemant els resultats de crides de funció costoses i reutilitzant-los quan els mateixos inputs es repeteixen.

Fibonacci Memoitzat en Delphi/Object Pascal

var
  Memo: array of Integer;

function FibonacciMemo(n: Integer): Integer;
begin
  if Memo[n] <> -1 then
    Result := Memo[n]
  else
  begin
    if n <= 1 then
      Memo[n] := n
    else
      Memo[n] := FibonacciMemo(n - 1) + FibonacciMemo(n - 2);
    Result := Memo[n];
  end;
end;

procedure InitializeMemoization(MaxN: Integer);
var
  i: Integer;
begin
  SetLength(Memo, MaxN + 1);
  for i := 0 to MaxN do
    Memo[i] := -1;
end;
  • Explicació:
    • Memo és una matriu utilitzada per emmagatzemar els valors de Fibonacci prèviament calculats.
    • FibonacciMemo comprova si el resultat per n ja està calculat.
    • Si no, calcula el valor i l'emmagatzema a Memo.
    • InitializeMemoization inicialitza la matriu Memo amb -1 per indicar valors no calculats.

Exercicis

  1. Recursió Bàsica: Escriu una funció recursiva per calcular la suma dels primers n nombres naturals.
  2. Inversió de Cadena: Implementa una funció recursiva per invertir una cadena.
  3. Torre de Hanoi: Resol el problema de la Torre de Hanoi utilitzant recursió.

Resum

  • La recursió és una eina poderosa en programació, permetent que les funcions es cridin a si mateixes per resoldre problemes.
  • Entendre el cas base i el cas recursiu és crucial per escriure funcions recursives correctes.
  • Tècniques avançades com la memoització poden optimitzar solucions recursives, fent-les més eficients.
  • Practicar amb diversos problemes ajuda a solidificar la comprensió de la recursió en Delphi/Object Pascal.

En dominar la recursió, pots resoldre problemes complexos de manera més elegant i eficient en Delphi/Object Pascal.
