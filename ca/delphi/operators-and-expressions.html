Introducció

En Delphi/Object Pascal, els operadors i les expressions són conceptes fonamentals que et permeten realitzar càlculs, prendre decisions i manipular dades. Aquesta secció cobrirà els diferents tipus d'operadors disponibles en Delphi/Object Pascal, com utilitzar-los i com formar expressions.

Operadors Aritmètics

Els operadors aritmètics s'utilitzen per realitzar càlculs matemàtics.

  • Suma (+): Suma dos operands.
  • Resta (-): Resta el segon operand del primer.
  • Multiplicació (*): Multiplica dos operands.
  • Divisió (/): Divideix el primer operand pel segon, resultant en un nombre de punt flotant.
  • Divisió Entera (div): Divideix el primer operand pel segon, resultant en un enter.
  • Mòdul (mod): Retorna el residu de la divisió del primer operand pel segon.

Exemple

var
  a, b: Integer;
  c: Real;
begin
  a := 10;
  b := 3;
  c := a / b;  // c serà 3.3333
  a := a div b;  // a serà 3
  b := a mod b;  // b serà 1
end;

Operadors Relacionals

Els operadors relacionals s'utilitzen per comparar dos valors.

  • Igual a (=): Comprova si dos operands són iguals.
  • No igual a (<>): Comprova si dos operands no són iguals.
  • Més gran que (>): Comprova si el primer operand és més gran que el segon.
  • Menys que (<): Comprova si el primer operand és menys que el segon.
  • Més gran o igual a (>=): Comprova si el primer operand és més gran o igual al segon.
  • Menys que o igual a (<=): Comprova si el primer operand és menys que o igual al segon.

Exemple

var
  a, b: Integer;
  result: Boolean;
begin
  a := 10;
  b := 5;
  result := a > b;  // result serà True
  result := a = b;  // result serà False
end;

Operadors Lògics

Els operadors lògics s'utilitzen per combinar múltiples condicions.

  • AND (and): Retorna True si ambdós operands són True.
  • OR (or): Retorna True si almenys un dels operands és True.
  • NOT (not): Reverteix l'estat lògic del seu operand.

Exemple

var
  a, b: Boolean;
  result: Boolean;
begin
  a := True;
  b := False;
  result := a and b;  // result serà False
  result := a or b;   // result serà True
  result := not a;    // result serà False
end;

Operadors d'Assignació

Els operadors d'assignació s'utilitzen per assignar valors a variables.

  • Assignació Simple (:=): Assigna el valor a la dreta a la variable a l'esquerra.
  • Assignació de Suma (+=): Suma el valor a la dreta a la variable a l'esquerra i assigna el resultat a la variable a l'esquerra.
  • Assignació de Resta (-=): Resta el valor a la dreta de la variable a l'esquerra i assigna el resultat a la variable a l'esquerra.
  • Assignació de Multiplicació (*=): Multiplica la variable a l'esquerra pel valor a la dreta i assigna el resultat a la variable a l'esquerra.
  • Assignació de Divisió (/=): Divideix la variable a l'esquerra pel valor a la dreta i assigna el resultat a la variable a l'esquerra.

Exemple

var
  a: Integer;
begin
  a := 10;
  a += 5;  // a serà 15
  a -= 3;  // a serà 12
  a *= 2;  // a serà 24
  a /= 4;  // a serà 6
end;

Operadors a Nivell de Bit

Els operadors a nivell de bit s'utilitzen per realitzar operacions sobre les representacions binàries dels enters.

  • AND (and): Realitza una operació AND a nivell de bit.
  • OR (or): Realitza una operació OR a nivell de bit.
  • XOR (xor): Realitza una operació XOR a nivell de bit.
  • NOT (not): Realitza una operació NOT a nivell de bit.
  • Desplaçament a l'Esquerra (shl): Desplaça els bits a l'esquerra.
  • Desplaçament a la Dreta (shr): Desplaça els bits a la dreta.

Exemple

var
  a, b: Integer;
begin
  a := 5;  // binari: 0101
  b := 3;  // binari: 0011
  a := a and b;  // a serà 1 (binari: 0001)
  a := a or b;   // a serà 3 (binari: 0011)
  a := a xor b;  // a serà 0 (binari: 0000)
  a := not a;    // a serà -1 (binari: 1111...1111)
  a := b shl 1;  // a serà 6 (binari: 0110)
  a := b shr 1;  // a serà 1 (binari: 0001)
end;

Expressions

Les expressions són combinacions de variables, constants, operadors i crides a funcions que s'avaluen per produir un valor.

Exemple

var
  a, b, c: Integer;
  result: Boolean;
begin
  a := 10;
  b := 5;
  c := 2;
  result := (a > b) and (b > c);  // result serà True
end;

Conclusió

Entendre els operadors i les expressions és crucial per escriure codi efectiu i eficient en Delphi/Object Pascal. Dominant els operadors aritmètics, relacionals, lògics, d'assignació i a nivell de bit, pots realitzar una àmplia gamma d'operacions i crear expressions complexes per resoldre diversos problemes de programació. Practica utilitzant aquests operadors en diferents escenaris per convertir-te en un expert en el seu ús.
