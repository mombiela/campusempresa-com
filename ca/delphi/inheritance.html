Herència en Delphi/Object Pascal
Polimorfisme ►

L'herència és un concepte fonamental en la Programació Orientada a Objectes (OOP) que permet a una classe heretar propietats i mètodes d'una altra classe. Això ajuda a la reutilització del codi i a establir una relació jeràrquica entre classes.

Introducció a l'Herència

  • Definició: L'herència permet que una nova classe (classe derivada) hereti atributs i mètodes d'una classe existent (classe base).
  • Propòsit: Promou la reutilització del codi, redueix la redundància i estableix una jerarquia natural.

Conceptes Bàsics

Classe Base i Classe Derivada

  • Classe Base: La classe les propietats i mètodes de la qual són heretats.
  • Classe Derivada: La classe que hereta de la classe base.

Sintaxi

En Delphi/Object Pascal, l'herència s'implementa utilitzant la paraula clau class seguida de la classe base TObject o qualsevol altra classe definida per l'usuari.

type
  TBaseClass = class
    procedure ShowMessage; virtual;
  end;

  TDerivedClass = class(TBaseClass)
    procedure ShowMessage; override;
  end;

Mètodes Virtuals i Override

  • Mètode Virtual: Un mètode en la classe base que pot ser sobreescrit en una classe derivada.
  • Mètode Override: Un mètode en la classe derivada que sobreescriu un mètode virtual en la classe base.
procedure TBaseClass.ShowMessage;
begin
  Writeln('Missatge de la Classe Base');
end;

procedure TDerivedClass.ShowMessage;
begin
  Writeln('Missatge de la Classe Derivada');
end;

Implementació de l'Herència

Creació d'una Classe Base

Defineix una classe base amb algunes propietats i mètodes.

type
  TAnimal = class
  public
    procedure Speak; virtual;
  end;

procedure TAnimal.Speak;
begin
  Writeln('L'animal fa un so');
end;

Creació d'una Classe Derivada

Defineix una classe derivada que hereti de la classe base i sobreescrigui els seus mètodes.

type
  TDog = class(TAnimal)
  public
    procedure Speak; override;
  end;

procedure TDog.Speak;
begin
  Writeln('El gos borda');
end;

Ús de les Classes

Instancia les classes i crida els seus mètodes per veure l'herència en acció.

var
  Animal: TAnimal;
  Dog: TDog;
begin
  Animal := TAnimal.Create;
  Dog := TDog.Create;

  Animal.Speak; // Sortida: L'animal fa un so
  Dog.Speak;    // Sortida: El gos borda

  Animal.Free;
  Dog.Free;
end;

Conceptes Avançats

Classes i Mètodes Abstractes

  • Classe Abstracta: Una classe que no pot ser instanciada i està destinada a ser una classe base per a altres classes.
  • Mètode Abstracte: Un mètode que ha de ser sobreescrit en una classe derivada.
type
  TShape = class
  public
    procedure Draw; virtual; abstract;
  end;

  TCircle = class(TShape)
  public
    procedure Draw; override;
  end;

procedure TCircle.Draw;
begin
  Writeln('Dibuixant un cercle');
end;

Herència i Constructors

Els constructors en classes derivades han de cridar el constructor de la classe base per assegurar una inicialització adequada.

type
  TPerson = class
  private
    FName: string;
  public
    constructor Create(AName: string); virtual;
  end;

  TEmployee = class(TPerson)
  private
    FEmployeeID: Integer;
  public
    constructor Create(AName: string; AEmployeeID: Integer); reintroduce;
  end;

constructor TPerson.Create(AName: string);
begin
  FName := AName;
end;

constructor TEmployee.Create(AName: string; AEmployeeID: Integer);
begin
  inherited Create(AName);
  FEmployeeID := AEmployeeID;
end;

Conclusió

L'herència en Delphi/Object Pascal és una característica poderosa que promou la reutilització del codi i estableix una jerarquia clara entre classes. Entenent i utilitzant l'herència, pots crear codi més organitzat i mantenible. Recorda utilitzar mètodes virtuals i override per personalitzar el comportament en classes derivades i considera classes i mètodes abstractes per a dissenys més avançats.
