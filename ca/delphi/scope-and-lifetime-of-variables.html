Àmbit i Durada de les Variables en Delphi/Object Pascal
Recursió ►

Entendre l'àmbit i la durada de les variables és crucial per escriure codi eficient i lliure d'errors en Delphi/Object Pascal. Aquest tema cobrirà els conceptes bàsics, proporcionarà exemples i inclourà exercicis per reforçar l'aprenentatge.

Àmbit de les Variables

L'àmbit d'una variable es refereix a la regió del codi on la variable és accessible. En Delphi/Object Pascal, hi ha diversos tipus d'àmbit:

  • Àmbit Local
  • Àmbit Global
  • Àmbit de la Unitat
  • Àmbit de la Classe

Àmbit Local

Les variables declarades dins d'un procediment, funció o mètode tenen àmbit local. Només són accessibles dins d'aquest bloc de codi.

procedure ExampleProcedure;
var
  LocalVar: Integer; // Variable local
begin
  LocalVar := 10;
  WriteLn(LocalVar); // Sortida: 10
end;

Àmbit Global

Les variables declarades fora de qualsevol procediment, funció o mètode tenen àmbit global. Són accessibles des de qualsevol part del programa.

var
  GlobalVar: Integer; // Variable global

procedure ExampleProcedure;
begin
  GlobalVar := 20;
  WriteLn(GlobalVar); // Sortida: 20
end;

begin
  GlobalVar := 10;
  ExampleProcedure;
  WriteLn(GlobalVar); // Sortida: 20
end.

Àmbit de la Unitat

Les variables declarades a la secció d'interfície d'una unitat són accessibles des d'altres unitats que utilitzen aquesta unitat. Les variables a la secció d'implementació només són accessibles dins de la mateixa unitat.

unit ExampleUnit;

interface

var
  UnitVar: Integer; // Variable d'àmbit de la unitat

implementation

procedure ExampleProcedure;
begin
  UnitVar := 30;
  WriteLn(UnitVar); // Sortida: 30
end;

end.

Àmbit de la Classe

Les variables declarades dins d'una classe són accessibles a tots els mètodes d'aquesta classe. També es coneixen com a camps o variables membres.

type
  TExampleClass = class
  private
    FClassVar: Integer; // Variable d'àmbit de la classe
  public
    procedure SetVar(Value: Integer);
    procedure ShowVar;
  end;

procedure TExampleClass.SetVar(Value: Integer);
begin
  FClassVar := Value;
end;

procedure TExampleClass.ShowVar;
begin
  WriteLn(FClassVar);
end;

var
  Example: TExampleClass;

begin
  Example := TExampleClass.Create;
  Example.SetVar(40);
  Example.ShowVar; // Sortida: 40
  Example.Free;
end.

Durada de les Variables

La durada d'una variable es refereix al temps durant el qual la variable existeix en memòria. En Delphi/Object Pascal, la durada de les variables depèn del seu àmbit.

Variables Locals

Les variables locals es creen quan es crida el procediment, funció o mètode i es destrueixen quan aquest finalitza.

procedure ExampleProcedure;
var
  LocalVar: Integer;
begin
  LocalVar := 50;
  WriteLn(LocalVar); // Sortida: 50
end;

begin
  ExampleProcedure;
  // LocalVar ja no és accessible aquí
end.

Variables Globals

Les variables globals existeixen durant tota la durada del programa.

var
  GlobalVar: Integer;

begin
  GlobalVar := 60;
  WriteLn(GlobalVar); // Sortida: 60
  // GlobalVar existeix fins que el programa finalitza
end.

Variables de la Unitat

Les variables de la unitat tenen una durada que abasta la durada del programa, similar a les variables globals.

unit ExampleUnit;

interface

var
  UnitVar: Integer;

implementation

procedure ExampleProcedure;
begin
  UnitVar := 70;
  WriteLn(UnitVar); // Sortida: 70
end;

end.

Variables de la Classe

Les variables de la classe (camps) existeixen mentre l'instància de la classe existeix. Es creen quan es crea la instància i es destrueixen quan es destrueix la instància.

type
  TExampleClass = class
  private
    FClassVar: Integer;
  public
    procedure SetVar(Value: Integer);
    procedure ShowVar;
  end;

procedure TExampleClass.SetVar(Value: Integer);
begin
  FClassVar := Value;
end;

procedure TExampleClass.ShowVar;
begin
  WriteLn(FClassVar);
end;

var
  Example: TExampleClass;

begin
  Example := TExampleClass.Create;
  Example.SetVar(80);
  Example.ShowVar; // Sortida: 80
  Example.Free;
  // FClassVar ja no és accessible aquí
end.

Conclusió

Entendre l'àmbit i la durada de les variables és essencial per gestionar la memòria i assegurar que el teu codi es comporti com s'espera. Aquí tens un resum ràpid:

  • Àmbit Local: Les variables són accessibles només dins del bloc on es declaren i existeixen només durant l'execució d'aquest bloc.
  • Àmbit Global: Les variables són accessibles des de qualsevol part del programa i existeixen durant tota la durada del programa.
  • Àmbit de la Unitat: Les variables són accessibles dins de la unitat i, si es declaren a la secció d'interfície, també des d'altres unitats.
  • Àmbit de la Classe: Les variables són accessibles a tots els mètodes de la classe i existeixen mentre l'instància de la classe existeix.

Dominant aquests conceptes, podràs escriure codi en Delphi/Object Pascal més eficient i mantenible.
