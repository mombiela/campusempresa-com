Conjunts en Delphi/Object Pascal
Tipus Enumerats ►

Introducció als Conjunts

Els conjunts en Delphi/Object Pascal són col·leccions d'elements del mateix tipus, típicament utilitzats per representar un grup de valors relacionats. Són particularment útils per manejar un rang fix de valors de manera eficient.

Conceptes Clau

  • Definició: Un conjunt és una col·lecció de valors del mateix tipus.
  • Tipus d'Element: El tipus d'elements en un conjunt ha de ser un tipus ordinal (per exemple, enters, caràcters, enumeracions).
  • Rang: Els conjunts només poden contenir valors dins d'un rang específic.

Declaració de Conjunts

Per declarar un conjunt en Delphi/Object Pascal, utilitzeu la paraula clau set of seguida del tipus d'elements que contindrà el conjunt.

Exemple

type
  TDays = (Mon, Tue, Wed, Thu, Fri, Sat, Sun);
  TWorkDays = set of TDays;

En aquest exemple:

  • TDays és una enumeració que representa els dies de la setmana.
  • TWorkDays és un conjunt que pot contenir qualsevol combinació de valors de TDays.

Treballant amb Conjunts

Creació i Inicialització de Conjunts

Podeu crear i inicialitzar conjunts utilitzant constructors de conjunts, que estan tancats entre claudàtors.

Exemple

var
  WorkDays: TWorkDays;
begin
  WorkDays := [Mon, Tue, Wed, Thu, Fri];
end;

En aquest exemple, WorkDays s'inicialitza per contenir de dilluns a divendres.

Afegir i Eliminar Elements

Podeu afegir o eliminar elements d'un conjunt utilitzant els operadors + i -.

Exemple

begin
  WorkDays := WorkDays + [Sat];  // Afegir dissabte
  WorkDays := WorkDays - [Mon];  // Eliminar dilluns
end;

Comprovar la Pertinença

Per comprovar si un element forma part d'un conjunt, utilitzeu l'operador in.

Exemple

begin
  if Mon in WorkDays then
    WriteLn('Monday is a workday.')
  else
    WriteLn('Monday is not a workday.');
end;

Operacions amb Conjunts

Delphi/Object Pascal suporta diverses operacions amb conjunts, incloent unió, intersecció i diferència.

Unió

La unió de dos conjunts conté tots els elements que estan en qualsevol dels conjunts.

Exemple

var
  Weekend: TWorkDays;
begin
  Weekend := [Sat, Sun];
  WorkDays := WorkDays + Weekend;  // Unió de WorkDays i Weekend
end;

Intersecció

La intersecció de dos conjunts conté només els elements que estan en ambdós conjunts.

Exemple

var
  CommonDays: TWorkDays;
begin
  CommonDays := WorkDays * [Mon, Tue, Wed];  // Intersecció de WorkDays i [Mon, Tue, Wed]
end;

Diferència

La diferència de dos conjunts conté els elements que estan en el primer conjunt però no en el segon.

Exemple

begin
  WorkDays := WorkDays - [Fri];  // Diferència de WorkDays i [Fri]
end;

Característiques Avançades dels Conjunts

Tipus de Subrang

Els conjunts es poden definir sobre subrang de tipus ordinals per limitar el seu rang.

Exemple

type
  TSmallNumbers = 1..10;
  TSmallNumberSet = set of TSmallNumbers;

Conjunt de Caràcters

Els conjunts també es poden utilitzar amb caràcters, cosa que és útil per a tasques com comprovar la validesa d'entrades.

Exemple

var
  Vowels: set of Char;
begin
  Vowels := ['A', 'E', 'I', 'O', 'U', 'a', 'e', 'i', 'o', 'u'];
  if 'A' in Vowels then
    WriteLn('A is a vowel.');
end;

Conclusió

Els conjunts en Delphi/Object Pascal proporcionen una manera poderosa de manejar col·leccions de valors ordinals de manera eficient. Suporten diverses operacions com unió, intersecció i diferència, fent-los versàtils per a moltes tasques de programació. Entendre com declarar, inicialitzar i manipular conjunts és essencial per a una programació efectiva en Delphi/Object Pascal.
