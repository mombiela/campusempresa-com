Implementació del Projecte
Proves i Depuració ►

En aquesta secció, cobrirem el procés d'implementar un projecte en Delphi/Object Pascal des del principi fins al final. Això inclou configurar el projecte, escriure el codi, depurar i provar. Al final d'aquesta secció, hauríeu de tenir una comprensió sòlida de com crear i gestionar un projecte Delphi.

Configuració del Projecte

Abans de començar a escriure codi, heu de configurar l'entorn del vostre projecte.

  • Crear un Nou Projecte: Obriu Delphi i seleccioneu File > New > VCL Forms Application - Delphi. Això crearà un nou projecte amb un formulari per defecte.
  • Estructura del Projecte: Entendre l'estructura bàsica d'un projecte Delphi, que inclou:
    • Arxiu .dpr: L'arxiu principal del projecte.
    • Arxiu .dfm: L'arxiu del formulari, que inclou els components visuals.
    • Arxiu .pas: L'arxiu de la unitat, que conté el codi.

Exemple: Crear un Nou Projecte

program Project1;

uses
  Vcl.Forms,
  Unit1 in 'Unit1.pas' {Form1};

{$R *.res}

begin
  Application.Initialize;
  Application.MainFormOnTaskbar := True;
  Application.CreateForm(TForm1, Form1);
  Application.Run;
end.

En aquest exemple, Project1.dpr és l'arxiu principal del projecte que inicialitza l'aplicació i crea el formulari principal (TForm1).

Escriure el Codi

Un cop el projecte està configurat, podeu començar a escriure el codi. Això implica dissenyar la interfície d'usuari i implementar la lògica.

  • Dissenyar la Interfície d'Usuari: Utilitzeu el Dissenyador de Formularis per afegir components com botons, etiquetes i caixes de text.
  • Escriure Controladors d'Esdeveniments: Escriure codi per gestionar esdeveniments com clics de botons.

Exemple: Afegir un Botó i Gestionar l'Esdeveniment de Clic

  1. Afegir un Botó: Arrossegueu un component TButton des de la Paleta d'Eines fins al formulari.
  2. Escriure el Controlador d'Esdeveniments: Feu doble clic al botó per crear un controlador d'esdeveniments OnClick.
unit Unit1;

interface

uses
  Vcl.Forms, Vcl.StdCtrls, System.Classes;

type
  TForm1 = class(TForm)
    Button1: TButton;
    procedure Button1Click(Sender: TObject);
  private
    { Private declarations }
  public
    { Public declarations }
  end;

var
  Form1: TForm1;

implementation

{$R *.dfm}

procedure TForm1.Button1Click(Sender: TObject);
begin
  ShowMessage('Button clicked!');
end;

end.

En aquest exemple, quan es fa clic al botó, apareixerà un quadre de missatge amb el text "Button clicked!".

Depuració i Proves

La depuració i les proves són passos crucials en el procés de desenvolupament per assegurar que la vostra aplicació funcioni com s'espera.

  • Establir Punts de Trencament: Podeu establir punts de trencament en el vostre codi per pausar l'execució i inspeccionar variables.
  • Executar l'Aplicació: Utilitzeu Run > Run o premeu F9 per executar l'aplicació.
  • Inspeccionar Variables: Utilitzeu la llista de vigilància i les finestres de variables locals per inspeccionar els valors de les variables durant la depuració.

Exemple: Establir un Punt de Trencament

  1. Establir un Punt de Trencament: Feu clic a la vora al costat de la línia de codi on voleu establir un punt de trencament.
  2. Executar l'Aplicació: Premeu F9 per executar l'aplicació. L'execució es pausarà al punt de trencament.
procedure TForm1.Button1Click(Sender: TObject);
var
  messageText: string;
begin
  messageText := 'Button clicked!';
  ShowMessage(messageText); // Establir un punt de trencament aquí
end;

En aquest exemple, podeu inspeccionar el valor de messageText quan l'execució es pausa al punt de trencament.

Conclusió

En aquest tema, hem cobert els passos essencials per implementar un projecte en Delphi/Object Pascal:

  1. Configurar l'entorn del projecte.
  2. Escriure el codi, incloent dissenyar la interfície d'usuari i implementar controladors d'esdeveniments.
  3. Depurar i provar l'aplicació.

Seguint aquests passos, podeu crear i gestionar projectes Delphi de manera eficient, assegurant-vos que estiguin ben estructurats i funcionals. Practiqueu aquests passos amb diferents projectes per convertir-vos en experts en el desenvolupament de Delphi/Object Pascal.
