L'enrutament dinàmic és una característica potent a Vue.js que et permet crear rutes que poden canviar en funció de l'entrada de l'usuari o altres dades dinàmiques. Això és essencial per construir aplicacions que necessiten gestionar una varietat de camins i paràmetres.

Conceptes Clau

  • Segments Dinàmics: Parts de l'URL que poden canviar, sovint representades amb un dos punts (:) en el camí de la ruta.
  • Paràmetres de Ruta: Variables que es passen a través de l'URL i es poden accedir dins del component.
  • Rutes Imbricades: Rutes que estan imbricades dins d'altres rutes, permetent estructures de navegació més complexes i jeràrquiques.
  • Navegació Programàtica: Utilitzar l'API de Vue Router per navegar a diferents rutes programàticament.

Configuració de Vue Router

Primer, assegura't que tens Vue Router instal·lat en el teu projecte Vue.js. Si no, pots instal·lar-lo utilitzant npm:

npm install vue-router

Després, configura el teu router a src/router/index.js:

import Vue from 'vue';
import VueRouter from 'vue-router';
import Home from '../views/Home.vue';
import User from '../views/User.vue';

Vue.use(VueRouter);

const routes = [
  {
    path: '/',
    name: 'Home',
    component: Home
  },
  {
    path: '/user/:id',
    name: 'User',
    component: User
  }
];

const router = new VueRouter({
  mode: 'history',
  base: process.env.BASE_URL,
  routes
});

export default router;

Segments Dinàmics i Paràmetres de Ruta

Els segments dinàmics en les rutes es defineixen utilitzant un dos punts (:) seguit del nom del paràmetre. Per exemple, /user/:id defineix una ruta amb un segment dinàmic id.

Exemple

En el component User.vue, pots accedir al paràmetre de la ruta id utilitzant this.$route.params.id:

<template>
  <div>
    <h1>Perfil de l'Usuari</h1>
    <p>ID de l'Usuari: {{ userId }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      userId: this.$route.params.id
    };
  },
  watch: {
    '$route.params.id'(newId) {
      this.userId = newId;
    }
  }
};
</script>

Rutes Imbricades

Les rutes imbricades et permeten crear estructures de rutes més complexes. Per exemple, podries tenir una ruta de perfil d'usuari amb rutes imbricades per a diferents seccions del perfil.

Exemple

Defineix rutes imbricades a src/router/index.js:

const routes = [
  {
    path: '/user/:id',
    component: User,
    children: [
      {
        path: 'profile',
        component: UserProfile
      },
      {
        path: 'posts',
        component: UserPosts
      }
    ]
  }
];

En el component User.vue, utilitza <router-view> per mostrar les rutes imbricades:

<template>
  <div>
    <h1>Perfil de l'Usuari</h1>
    <router-link :to="{ name: 'UserProfile', params: { id: userId } }">Perfil</router-link>
    <router-link :to="{ name: 'UserPosts', params: { id: userId } }">Publicacions</router-link>
    <router-view></router-view>
  </div>
</template>

<script>
export default {
  data() {
    return {
      userId: this.$route.params.id
    };
  }
};
</script>

Navegació Programàtica

Pots navegar a diferents rutes programàticament utilitzant l'API de Vue Router. Això és útil per a la navegació basada en accions de l'usuari o altres condicions dinàmiques.

Exemple

En un mètode dins del teu component, utilitza this.$router.push per navegar a una ruta diferent:

<template>
  <div>
    <button @click="goToUserProfile">Anar al Perfil de l'Usuari</button>
  </div>
</template>

<script>
export default {
  methods: {
    goToUserProfile() {
      this.$router.push({ name: 'User', params: { id: 123 } });
    }
  }
};
</script>

Conclusió

L'enrutament dinàmic a Vue.js et permet crear estructures de navegació flexibles i potents en les teves aplicacions. Entenent els segments dinàmics, els paràmetres de ruta, les rutes imbricades i la navegació programàtica, pots construir interfícies complexes i fàcils d'usar. Practica aquests conceptes amb diferents escenaris per esdevenir proficient en l'enrutament dinàmic amb Vue.js.
