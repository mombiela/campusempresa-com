Introducció

La renderització de llistes és un concepte fonamental a Vue.js que et permet renderitzar dinàmicament llistes d'elements. Això és particularment útil quan es treballa amb arrays de dades que necessiten ser mostrats en un format estructurat. En aquesta secció, cobrirem els conceptes bàsics de la renderització de llistes, com utilitzar la directiva v-for, i algunes tècniques avançades per optimitzar el rendiment i manejar estructures de dades complexes.

Renderització bàsica de llistes

La directiva v-for s'utilitza per renderitzar una llista d'elements iterant sobre un array. Aquí tens un exemple senzill:

<div id="app">
  <ul>
    <li v-for="item in items" :key="item.id">{{ item.text }}</li>
  </ul>
</div>

<script>
new Vue({
  el: '#app',
  data: {
    items: [
      { id: 1, text: 'Aprendre Vue.js' },
      { id: 2, text: 'Construir un projecte' },
      { id: 3, text: 'Dominar Vue.js' }
    ]
  }
});
</script>

Explicació

  • La directiva v-for s'utilitza per iterar sobre l'array items.
  • L'atribut :key s'utilitza per identificar de manera única cada element, cosa que ajuda Vue.js a optimitzar la renderització.
  • La sintaxi {{ item.text }} s'utilitza per mostrar la propietat de text de cada element.

Renderització d'objectes

També pots utilitzar v-for per iterar sobre les propietats d'un objecte. Aquí tens un exemple:

<div id="app">
  <ul>
    <li v-for="(value, key) in user" :key="key">{{ key }}: {{ value }}</li>
  </ul>
</div>

<script>
new Vue({
  el: '#app',
  data: {
    user: {
      name: 'John Doe',
      age: 30,
      occupation: 'Desenvolupador'
    }
  }
});
</script>

Explicació

  • La directiva v-for pot iterar sobre les propietats d'un objecte.
  • La sintaxi (value, key) et permet accedir tant al nom de la propietat (key) com al seu valor (value).

Utilitzar l'índex en v-for

De vegades, pot ser necessari accedir a l'índex de l'element actual en el bucle. Pots fer-ho afegint un paràmetre addicional a la directiva v-for:

<div id="app">
  <ul>
    <li v-for="(item, index) in items" :key="item.id">{{ index }} - {{ item.text }}</li>
  </ul>
</div>

<script>
new Vue({
  el: '#app',
  data: {
    items: [
      { id: 1, text: 'Aprendre Vue.js' },
      { id: 2, text: 'Construir un projecte' },
      { id: 3, text: 'Dominar Vue.js' }
    ]
  }
});
</script>

Explicació

  • El paràmetre index proporciona l'índex actual de l'element en l'array.
  • Això pot ser útil per mostrar la posició de cada element en la llista.

Manejar estructures de dades complexes

Quan es treballa amb estructures de dades niades, pots utilitzar múltiples directives v-for per renderitzar llistes niades. Aquí tens un exemple:

<div id="app">
  <ul>
    <li v-for="category in categories" :key="category.id">
      {{ category.name }}
      <ul>
        <li v-for="item in category.items" :key="item.id">{{ item.text }}</li>
      </ul>
    </li>
  </ul>
</div>

<script>
new Vue({
  el: '#app',
  data: {
    categories: [
      {
        id: 1,
        name: 'Frontend',
        items: [
          { id: 1, text: 'Aprendre HTML' },
          { id: 2, text: 'Aprendre CSS' }
        ]
      },
      {
        id: 2,
        name: 'Backend',
        items: [
          { id: 3, text: 'Aprendre Node.js' },
          { id: 4, text: 'Aprendre Express' }
        ]
      }
    ]
  }
});
</script>

Explicació

  • Les directives v-for niades s'utilitzen per renderitzar llistes niades.
  • Cada directiva v-for té el seu propi atribut :key per identificar de manera única els elements.

Optimitzar el rendiment

Quan es renderitzen llistes grans, el rendiment pot convertir-se en un problema. Aquí tens alguns consells per optimitzar la renderització de llistes:

  • Utilitza :key Correctament: Sempre utilitza una clau única per a cada element per ajudar Vue.js a optimitzar la renderització.
  • Paginació: Divideix les llistes grans en trossos més petits i renderitza'ls segons sigui necessari.
  • Desplaçament Virtual: Utilitza biblioteques com vue-virtual-scroll-list per renderitzar només la porció visible d'una llista gran.

Conclusió

La renderització de llistes a Vue.js és una característica potent que et permet mostrar dinàmicament arrays i objectes. Comprenent com utilitzar la directiva v-for, manejar estructures de dades niades i optimitzar el rendiment, pots crear aplicacions eficients i dinàmiques. Practica aquests conceptes amb diverses estructures de dades per convertir-te en un expert en la renderització de llistes amb Vue.js.
