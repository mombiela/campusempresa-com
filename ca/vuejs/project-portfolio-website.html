Introducció

En aquest projecte, construirem una pàgina web de portafoli utilitzant Vue.js. Aquest projecte t'ajudarà a consolidar els teus coneixements de Vue.js aplicant-los a un escenari del món real. Al final d'aquest projecte, tindràs una pàgina web de portafoli completament funcional per mostrar les teves habilitats i projectes.

Configuració del Projecte

Abans de començar a construir la pàgina web de portafoli, configurem el nostre projecte.

  1. Instal·la Vue CLI: Si encara no has instal·lat Vue CLI, pots fer-ho executant:

    npm install -g @vue/cli

  2. Crea un Nou Projecte Vue: Crea un nou projecte Vue executant:

    vue create portfolio-website

  3. Navega al Directori del Projecte:

    cd portfolio-website

  4. Executa el Servidor de Desenvolupament:

    npm run serve

Estructura del Projecte

Entendre l'estructura del projecte és crucial per a un desenvolupament eficient.

  • src: Conté el codi principal de l'aplicació.
    • assets: Actius estàtics com imatges i estils.
    • components: Components reutilitzables de Vue.
    • views: Diferents vistes o pàgines de l'aplicació.
    • router: Configuració per a Vue Router.
    • store: Botiga Vuex per a la gestió d'estat (si és necessari).

Creació de Components

Crearem diversos components per a la nostra pàgina web de portafoli:

  • Header: Conté el menú de navegació.
  • Footer: Conté informació de contacte i enllaços a xarxes socials.
  • Home: La pàgina d'inici del portafoli.
  • About: Una pàgina amb informació sobre tu.
  • Projects: Una pàgina que mostra els teus projectes.
  • Contact: Una pàgina amb un formulari de contacte.

Component Header

Comencem creant el component Header.

  1. Crea Header.vue al directori components:

    <template>
  <header>
    <nav>
      <ul>
        <li><router-link to="/">Home</router-link></li>
        <li><router-link to="/about">About</router-link></li>
        <li><router-link to="/projects">Projects</router-link></li>
        <li><router-link to="/contact">Contact</router-link></li>
      </ul>
    </nav>
  </header>
</template>

<script>
export default {
  name: 'Header'
}
</script>

<style scoped>
header {
  background-color: #333;
  color: white;
  padding: 1rem;
}
nav ul {
  list-style: none;
  display: flex;
  justify-content: space-around;
}
nav ul li {
  margin: 0 1rem;
}
</style>

  2. Importa i Utilitza el Component Header a App.vue:

    <template>

    <Header />
    <router-view />

  </div>
</template>

  name: 'App',

    Header,
    Footer
  }

</script>

Enrutament

Per navegar entre diferents pàgines, necessitem configurar Vue Router.

  1. Instal·la Vue Router:

    npm install vue-router

  2. Configura Vue Router a src/router/index.js:

    import Vue from 'vue'
import VueRouter from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'
import Projects from '../views/Projects.vue'
import Contact from '../views/Contact.vue'

Vue.use(VueRouter)

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/projects', component: Projects },
  { path: '/contact', component: Contact }
]

const router = new VueRouter({
  mode: 'history',
  routes
})

export default router

  3. Importa i Utilitza el Router a src/main.js:

    import Vue from 'vue'
import App from './App.vue'
import router from './router'

Vue.config.productionTip = false

new Vue({
  router,
  render: h => h(App)
}).$mount('#app')
