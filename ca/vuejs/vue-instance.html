Introducció

La instància de Vue és el nucli de qualsevol aplicació Vue.js. És l'objecte que conté les dades, els mètodes i els hooks de cicle de vida que fan que la teva aplicació sigui dinàmica i reactiva. Entendre la instància de Vue és crucial per construir aplicacions efectives amb Vue.js.

Creant una Instància de Vue

Per crear una instància de Vue, utilitzes el constructor Vue. Aquí tens un exemple bàsic:

var vm = new Vue({
  // Objecte d'opcions
});

Exemple

Creem una instància de Vue senzilla que vincula algunes dades al DOM:

<!DOCTYPE html>
<html>
<head>
  <title>Exemple de Instància de Vue</title>
  <script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
</head>
<body>
  <div id="app">
    {{ message }}
  </div>

  <script>
    var vm = new Vue({
      el: '#app',
      data: {
        message: 'Hola, Vue!'
      }
    });
  </script>
</body>
</html>

En aquest exemple:

  • el: L'element del DOM on es muntarà la instància de Vue.
  • data: Un objecte que conté les dades per a la instància de Vue.

Dades i Mètodes

Les propietats data i methods són parts essencials de la instància de Vue.

Dades

La propietat data és un objecte que conté les dades de l'aplicació. Vue fa que aquestes dades siguin reactives, és a dir, qualsevol canvi en les dades actualitzarà automàticament el DOM.

var vm = new Vue({
  el: '#app',
  data: {
    message: 'Hola, Vue!'
  }
});

Mètodes

La propietat methods és un objecte que conté funcions. Aquestes funcions es poden utilitzar per realitzar accions o manipular dades.

var vm = new Vue({
  el: '#app',
  data: {
    message: 'Hola, Vue!'
  },
  methods: {
    reverseMessage: function() {
      this.message = this.message.split('').reverse().join('');
    }
  }
});

En aquest exemple, reverseMessage és un mètode que inverteix la cadena message.

Propietats Calculades

Les propietats calculades s'utilitzen per definir propietats que depenen d'altres propietats de dades. Es cachegen basant-se en les seves dependències i només es reavaluen quan les seves dependències canvien.

var vm = new Vue({
  el: '#app',
  data: {
    message: 'Hola, Vue!'
  },
  computed: {
    reversedMessage: function() {
      return this.message.split('').reverse().join('');
    }
  }
});

Observadors

Els observadors s'utilitzen per realitzar accions en resposta a canvis en les dades. Són especialment útils per a operacions asíncrones.

var vm = new Vue({
  el: '#app',
  data: {
    message: 'Hola, Vue!'
  },
  watch: {
    message: function(newVal, oldVal) {
      console.log('El missatge ha canviat de', oldVal, 'a', newVal);
    }
  }
});

Hooks de Cicle de Vida

Els hooks de cicle de vida són mètodes que et permeten enganxar-te a diferents etapes del cicle de vida d'una instància de Vue.

  • created: Es crida després de crear la instància.
  • mounted: Es crida després de muntar la instància al DOM.
  • updated: Es crida després de canviar les dades i tornar a renderitzar el DOM.
  • destroyed: Es crida després de destruir la instància.
var vm = new Vue({
  el: '#app',
  data: {
    message: 'Hola, Vue!'
  },
  created: function() {
    console.log('Instància creada');
  },
  mounted: function() {
    console.log('Instància muntada');
  },
  updated: function() {
    console.log('Instància actualitzada');
  },
  destroyed: function() {
    console.log('Instància destruïda');
  }
});

Resum

Entendre la instància de Vue és fonamental per dominar Vue.js. La instància de Vue és el cor de qualsevol aplicació Vue, gestionant dades, mètodes, propietats calculades, observadors i hooks de cicle de vida. Comprenent aquests conceptes, pots crear aplicacions dinàmiques i reactives amb facilitat.

  • Dades: Propietats reactives que actualitzen el DOM.
  • Mètodes: Funcions per manipular dades o realitzar accions.
  • Propietats Calculades: Propietats cachejades que depenen d'altres dades.
  • Observadors: Responen a canvis en les dades amb accions personalitzades.
  • Hooks de Cicle de Vida: Enganxa't a diferents etapes del cicle de vida de la instància.

Amb aquest coneixement, estàs ben preparat per començar a construir aplicacions potents amb Vue.js.
