Desplegar una aplicació Vue.js implica diversos passos per assegurar que la teva aplicació estigui llesta per a producció i accessible als usuaris. Aquesta secció et guiarà a través del procés de principi a fi.

Preparar la teva aplicació Vue.js per al desplegament

Abans de desplegar la teva aplicació Vue.js, has de preparar-la per a producció. Això implica optimitzar el teu codi i assegurar-te que funcioni eficientment.

  • Configuració de la compilació: Vue CLI proporciona una comanda integrada per compilar la teva aplicació per a producció.
  • Variables d'entorn: Utilitza variables d'entorn per gestionar diferents configuracions per a desenvolupament i producció.
  • Divisió de codi i càrrega mandrosa: Optimitza la teva aplicació dividint el teu codi en trossos més petits i carregant-los només quan sigui necessari.

Configuració de la compilació

Per compilar la teva aplicació Vue.js per a producció, utilitza la següent comanda:

npm run build

Aquesta comanda crea un directori dist amb la teva aplicació optimitzada. El directori dist conté tots els fitxers que necessites per desplegar la teva aplicació.

Variables d'entorn

Vue CLI et permet definir variables d'entorn en fitxers .env. Per exemple, pots crear un fitxer .env.production per a configuracions de producció:

VUE_APP_API_URL=https://api.example.com

A la teva aplicació, pots accedir a aquesta variable utilitzant process.env.VUE_APP_API_URL.

Divisió de codi i càrrega mandrosa

La divisió de codi i la càrrega mandrosa ajuden a millorar el rendiment de la teva aplicació carregant només el codi necessari quan sigui necessari. Vue Router suporta la càrrega mandrosa de manera nativa:

const Home = () => import('./views/Home.vue');
const About = () => import('./views/About.vue');

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
];

Desplegar a diferents plataformes

Hi ha diverses plataformes on pots desplegar la teva aplicació Vue.js. Aquí, cobrirem el desplegament a:

  • GitHub Pages
  • Netlify
  • Vercel
  • Heroku

Desplegar a GitHub Pages

GitHub Pages és un servei d'allotjament gratuït per a repositoris de GitHub. Per desplegar la teva aplicació Vue.js a GitHub Pages:

  1. Instal·la el paquet gh-pages:

    npm install gh-pages --save-dev

  2. Afegeix els següents scripts al teu package.json:

    "scripts": {
  "deploy": "vue-cli-service build && gh-pages -d dist"
}

  3. Executa el script de desplegament:

    npm run deploy

Desplegar a Netlify

Netlify és una plataforma popular per desplegar llocs web estàtics. Per desplegar la teva aplicació Vue.js a Netlify:

  1. Crea un fitxer netlify.toml a l'arrel del teu projecte:

    [build]
  command = "npm run build"
  publish = "dist"

  2. Puja el teu codi a un repositori Git.

  3. Inicia sessió a Netlify i crea un nou lloc des del teu repositori Git.

  4. Netlify construirà i desplegarà automàticament la teva aplicació.

Desplegar a Vercel

Vercel és una altra plataforma popular per desplegar llocs web estàtics i funcions sense servidor. Per desplegar la teva aplicació Vue.js a Vercel:

  1. Instal·la el CLI de Vercel:

    npm install -g vercel

  2. Executa la següent comanda per desplegar la teva aplicació:

    vercel

  3. Segueix les instruccions per completar el desplegament.

Desplegar a Heroku

Heroku és una plataforma al núvol que suporta diversos llenguatges de programació. Per desplegar la teva aplicació Vue.js a Heroku:

  1. Crea un fitxer static.json a l'arrel del teu projecte:

    {
  "root": "dist/",
  "clean_urls": true,
  "routes": {
    "/**": "index.html"
  }
}

  2. Crea un fitxer Procfile a l'arrel del teu projecte:

    web: npx serve -s dist

  3. Puja el teu codi a un repositori Git.

  4. Inicia sessió a Heroku i crea una nova aplicació.

  5. Connecta el teu repositori Git a Heroku i desplega la teva aplicació.

Conclusió

Desplegar una aplicació Vue.js implica preparar la teva aplicació per a producció i triar la plataforma adequada per al desplegament. Seguint els passos descrits en aquesta secció, pots assegurar-te que la teva aplicació Vue.js estigui optimitzada i accessible per als usuaris. Tant si tries GitHub Pages, Netlify, Vercel o Heroku, cada plataforma ofereix característiques i beneficis únics per satisfer les teves necessitats de desplegament.
