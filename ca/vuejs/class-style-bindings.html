Introducció

En Vue.js, els enllaços de classe i estil són característiques poderoses que et permeten aplicar dinàmicament classes CSS i estils en línia als teus components. Aquest tema cobrirà com utilitzar aquests enllaços de manera efectiva, des de l'ús bàsic fins a l'avançat.

Enllaços de Classe

Enllaç de Classe Bàsic

Vue.js et permet enllaçar classes a elements utilitzant la directiva v-bind:class o la seva forma abreujada :class.

Exemple:

<div :class="{'active': isActive}">Hola, Vue!</div>

En aquest exemple, la classe active s'aplicarà a l'element div si isActive és true.

Sintaxi d'Objecte

Pots utilitzar un objecte per aplicar condicionalment múltiples classes.

Exemple:

<div :class="{'active': isActive, 'disabled': isDisabled}">Hola, Vue!</div>

Aquí, la classe active s'aplicarà si isActive és true, i la classe disabled s'aplicarà si isDisabled és true.

Sintaxi d'Array

També pots utilitzar un array per aplicar múltiples classes.

Exemple:

<div :class="[classA, classB]">Hola, Vue!</div>

En aquest cas, tant classA com classB s'aplicaran a l'element div.

Classes Computades

Pots utilitzar propietats computades per generar dinàmicament noms de classe.

Exemple:

<div :class="computedClass">Hola, Vue!</div>

<script>
export default {
  data() {
    return {
      isActive: true,
      isDisabled: false
    };
  },
  computed: {
    computedClass() {
      return {
        active: this.isActive,
        disabled: this.isDisabled
      };
    }
  }
};
</script>

Aquest enfocament manté el teu template net i fa que la teva lògica de classes sigui més fàcil de gestionar.

Enllaços d'Estil

Enllaç d'Estil Bàsic

Vue.js et permet enllaçar estils en línia utilitzant la directiva v-bind:style o la seva forma abreujada :style.

Exemple:

<div :style="{ color: activeColor, fontSize: fontSize + 'px' }">Hola, Vue!</div>

En aquest exemple, l'element div tindrà els seus estils color i fontSize establerts dinàmicament basant-se en els valors de activeColor i fontSize.

Sintaxi d'Objecte

Pots utilitzar un objecte per aplicar múltiples estils.

Exemple:

<div :style="{ color: activeColor, fontSize: fontSize + 'px' }">Hola, Vue!</div>

Aquí, tant els estils color com fontSize s'apliquen a l'element div.

Sintaxi d'Array

També pots utilitzar un array per aplicar múltiples objectes d'estil.

Exemple:

<div :style="[baseStyles, overridingStyles]">Hola, Vue!</div>

En aquest cas, els estils de baseStyles i overridingStyles s'aplicaran a l'element div. Si hi ha estils conflictius, els últims a l'array tindran prioritat.

Estils Computats

Pots utilitzar propietats computades per generar dinàmicament estils.

Exemple:

<div :style="computedStyles">Hola, Vue!</div>

<script>
export default {
  data() {
    return {
      activeColor: 'red',
      fontSize: 14
    };
  },
  computed: {
    computedStyles() {
      return {
        color: this.activeColor,
        fontSize: this.fontSize + 'px'
      };
    }
  }
};
</script>

Aquest enfocament manté el teu template net i fa que la teva lògica d'estils sigui més fàcil de gestionar.

Resum

Els enllaços de classe i estil en Vue.js proporcionen una manera flexible i poderosa d'aplicar dinàmicament classes CSS i estils en línia als teus components. Entenent i utilitzant la sintaxi d'objecte i array, així com les propietats computades, pots crear aplicacions Vue.js més dinàmiques i mantenibles.
