Introducció

En Vue.js, les propietats i els esdeveniments personalitzats són conceptes fonamentals que permeten que els components es comuniquin entre ells. Les propietats permeten que els components pare passin dades als components fill, mentre que els esdeveniments personalitzats permeten que els components fill enviïn missatges de tornada als seus pares. Entendre aquests conceptes és crucial per construir aplicacions dinàmiques i interactives.

Propietats

Què són les Propietats?

  • Propietats són atributs personalitzats que pots registrar en un component.
  • Permeten que les dades es passin d'un component pare a un component fill.
  • Les propietats són de només lectura, és a dir, el component fill no pot modificar les dades directament.

Definint Propietats

Per definir propietats en un component Vue, utilitzes l'opció props. Aquí tens un exemple:

<template>
  <div>
    <h1>{{ title }}</h1>
    <p>{{ description }}</p>
  </div>
</template>

<script>
export default {
  props: {
    title: {
      type: String,
      required: true
    },
    description: {
      type: String,
      required: false,
      default: 'No description provided'
    }
  }
}
</script>

En aquest exemple:

  • title és una propietat requerida de tipus String.
  • description és una propietat opcional amb un valor per defecte.

Passant Propietats

Les propietats es passen als components fill en la plantilla del component pare:

<template>
  <div>
    <ChildComponent title="Hello World" description="This is a description"></ChildComponent>
  </div>
</template>

<script>
import ChildComponent from './ChildComponent.vue';

export default {
  components: {
    ChildComponent
  }
}
</script>

Validació de Propietats

Vue permet especificar el tipus de propietats i si són requerides. Això ajuda a detectar errors i fa que els teus components siguin més robustos.

props: {
  title: {
    type: String,
    required: true
  },
  count: {
    type: Number,
    required: true
  }
}

Esdeveniments Personalitzats

Què són els Esdeveniments Personalitzats?

  • Els esdeveniments personalitzats permeten que els components fill es comuniquin amb els seus components pare.
  • Són útils per notificar al pare sobre canvis o accions que ocorren en el component fill.

Emissió d'Esdeveniments Personalitzats

Per emetre un esdeveniment personalitzat des d'un component fill, utilitzes el mètode $emit:

<template>
  <button @click="notifyParent">Click Me</button>
</template>

<script>
export default {
  methods: {
    notifyParent() {
      this.$emit('custom-event', 'Hello from Child');
    }
  }
}
</script>

En aquest exemple, quan es fa clic al botó, el mètode notifyParent emet un esdeveniment personalitzat anomenat custom-event amb un missatge.

Escoltant Esdeveniments Personalitzats

El component pare escolta els esdeveniments personalitzats utilitzant la directiva v-on o la seva abreviatura @:

<template>
  <div>
    <ChildComponent @custom-event="handleCustomEvent"></ChildComponent>
  </div>
</template>

<script>
import ChildComponent from './ChildComponent.vue';

export default {
  components: {
    ChildComponent
  },
  methods: {
    handleCustomEvent(message) {
      console.log(message); // Sortida: Hello from Child
    }
  }
}
</script>

Passant Dades amb Esdeveniments Personalitzats

Els esdeveniments personalitzats poden portar dades, que poden ser accedides pel component pare:

this.$emit('custom-event', { id: 1, name: 'Item' });

El component pare pot llavors gestionar aquestes dades:

methods: {
  handleCustomEvent(data) {
    console.log(data.id); // Sortida: 1
    console.log(data.name); // Sortida: Item
  }
}

Conclusió

Les propietats i els esdeveniments personalitzats són essencials per a la comunicació entre components en Vue.js. Les propietats permeten que les dades flueixin dels components pare als components fill, mentre que els esdeveniments personalitzats permeten que els components fill enviïn missatges de tornada als seus pares. Dominant aquests conceptes, pots crear aplicacions Vue més dinàmiques i interactives. Practica utilitzant propietats i esdeveniments personalitzats en diferents escenaris per solidificar la teva comprensió i millorar les teves habilitats en Vue.js.
