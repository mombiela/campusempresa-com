Introducció

En aquest projecte, construirem una aplicació simple de Llista de Tasques utilitzant Vue.js. Aquest projecte t'ajudarà a entendre els conceptes bàsics de Vue.js, incloent la vinculació de dades, la gestió d'esdeveniments i l'arquitectura basada en components.

Conceptes Clau

  • Instància Vue: L'arrel de cada aplicació Vue.
  • Vinculació de Dades: Sincronització de dades entre el model i la vista.
  • Gestió d'Esdeveniments: Respondre a les accions de l'usuari.
  • Components: Instàncies Vue reutilitzables amb el seu propi àmbit.
  • Directives: Tokens especials en el marcat que indiquen a la biblioteca que faci alguna cosa a un element DOM.

Configuració del Projecte

  1. Instal·lar Vue CLI: 
    npm install -g @vue/cli
  2. Crear un Nou Projecte: 
    vue create todo-list
  3. Navegar al Directori del Projecte: 
    cd todo-list
  4. Executar el Servidor de Desenvolupament: 
    npm run serve

Construint l'Aplicació de Llista de Tasques

Creant la Instància Vue

Crea una nova instància Vue a src/main.js:

import Vue from 'vue';
import App from './App.vue';

Vue.config.productionTip = false;

new Vue({
  render: h => h(App),
}).$mount('#app');

Dissenyant el Layout de l'Aplicació

A src/App.vue, configura l'estructura bàsica:

<template>
  <div id="app">
    <h1>Llista de Tasques</h1>
    <AddTodo @add-todo="addTodo"></AddTodo>
    <TodoList :todos="todos" @delete-todo="deleteTodo"></TodoList>
  </div>
</template>

<script>
import AddTodo from './components/AddTodo.vue';
import TodoList from './components/TodoList.vue';

export default {
  name: 'App',
  components: {
    AddTodo,
    TodoList
  },
  data() {
    return {
      todos: []
    };
  },
  methods: {
    addTodo(todo) {
      this.todos.push(todo);
    },
    deleteTodo(index) {
      this.todos.splice(index, 1);
    }
  }
};
</script>

<style>
/* Afegeix alguns estils bàsics */
</style>

Creant el Component AddTodo

Crea un nou fitxer src/components/AddTodo.vue:

<template>
  <div>
    <input v-model="newTodo" @keyup.enter="submitTodo" placeholder="Afegeix una nova tasca" />
    <button @click="submitTodo">Afegeix</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      newTodo: ''
    };
  },
  methods: {
    submitTodo() {
      if (this.newTodo.trim()) {
        this.$emit('add-todo', this.newTodo.trim());
        this.newTodo = '';
      }
    }
  }
};
</script>

<style scoped>
/* Afegeix alguns estils bàsics */
</style>

Creant el Component TodoList

Crea un nou fitxer src/components/TodoList.vue:

<template>
  <ul>
    <li v-for="(todo, index) in todos" :key="index">
      {{ todo }}
      <button @click="$emit('delete-todo', index)">Elimina</button>
    </li>
  </ul>
</template>

<script>
export default {
  props: ['todos']
};
</script>

Conclusió

En aquest projecte, has après a crear una aplicació simple de Llista de Tasques utilitzant Vue.js. Has explorat els conceptes bàsics de les instàncies Vue, la vinculació de dades, la gestió d'esdeveniments i els components. Aquest coneixement fonamental serà crucial a mesura que avancis cap a temes més avançats en Vue.js. Continua experimentant amb el codi i intenta afegir noves funcionalitats a l'aplicació de Llista de Tasques per aprofundir en la teva comprensió.
