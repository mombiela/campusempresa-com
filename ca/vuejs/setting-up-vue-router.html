Introducció

Vue Router és el router oficial per a Vue.js, permetent als desenvolupadors crear aplicacions d'una sola pàgina amb navegació i routing dinàmic. Aquesta secció et guiarà a través del procés de configuració de Vue Router en el teu projecte Vue.js.

Instal·lació de Vue Router

Per començar a utilitzar Vue Router, cal instal·lar-lo en el teu projecte Vue.js. Pots fer-ho utilitzant npm o yarn.

Utilitzant npm

npm install vue-router

Utilitzant yarn

yarn add vue-router

Configuració bàsica

Un cop instal·lat Vue Router, cal configurar-lo en el teu projecte. Això implica crear una instància del router i definir les teves rutes.

Creació d'una instància del Router

Primer, crea un nou fitxer anomenat router.js (o router/index.js si prefereixes una estructura nidificada) en el teu directori src.

// src/router.js
import Vue from 'vue';
import VueRouter from 'vue-router';
import Home from './components/Home.vue';
import About from './components/About.vue';

Vue.use(VueRouter);

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
];

const router = new VueRouter({
  routes
});

export default router;

Integració del Router amb la teva instància de Vue

A continuació, cal integrar el router amb la teva instància principal de Vue. Obre el teu fitxer main.js i importa el router.

// src/main.js
import Vue from 'vue';
import App from './App.vue';
import router from './router';

new Vue({
  router,
  render: h => h(App)
}).$mount('#app');

Definició de Rutes

Les rutes són el nucli de Vue Router. Mapegen URLs a components.

Definició bàsica de rutes

En el fitxer router.js, defineixes les rutes com a objectes amb propietats path i component.

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
];

Rutes amb nom

Les rutes amb nom faciliten l'enllaç a rutes programàticament.

const routes = [
  { path: '/', component: Home, name: 'home' },
  { path: '/about', component: About, name: 'about' }
];

Rutes dinàmiques

Les rutes dinàmiques et permeten capturar paràmetres de la URL.

const routes = [
  { path: '/user/:id', component: User }
];

En el component User, pots accedir al paràmetre de la ruta id utilitzant this.$route.params.id.

Navegació

Vue Router proporciona diverses maneres de navegar entre rutes.

Utilitzant ``

El component <router-link> s'utilitza per crear enllaços de navegació.

<template>
  <div>
    <router-link to="/">Home</router-link>
    <router-link to="/about">About</router-link>
  </div>
</template>

Navegació programàtica

Pots navegar programàticament utilitzant el mètode this.$router.push.

methods: {
  goToAbout() {
    this.$router.push('/about');
  }
}

Rutes nidificades

Les rutes nidificades et permeten crear estructures de rutes complexes.

const routes = [
  {
    path: '/user/:id',
    component: User,
    children: [
      {
        path: 'profile',
        component: UserProfile
      },
      {
        path: 'posts',
        component: UserPosts
      }
    ]
  }
];

Guàrdies de Ruta

Les guàrdies de ruta s'utilitzen per controlar l'accés a les rutes.

Guàrdies globals

Les guàrdies globals s'apliquen a cada canvi de ruta.

router.beforeEach((to, from, next) => {
  if (to.path === '/about') {
    // Realitza alguna comprovació
    next();
  } else {
    next();
  }
});

Guàrdies per ruta

Les guàrdies per ruta es defineixen directament en la configuració de la ruta.

const routes = [
  {
    path: '/about',
    component: About,
    beforeEnter: (to, from, next) => {
      // Realitza alguna comprovació
      next();
    }
  }
];

Resum

En aquesta secció, has après com configurar Vue Router en un projecte Vue.js. Hem cobert la instal·lació, configuració bàsica, definició de rutes, navegació, rutes nidificades i guàrdies de ruta. Amb aquest coneixement, ara pots crear aplicacions dinàmiques d'una sola pàgina amb Vue.js i Vue Router.
