Introducció

En aquest projecte, construirem una plataforma de blog completament funcional utilitzant Vue.js. Aquest projecte t'ajudarà a entendre com crear una aplicació complexa desglossant-la en components manejables. Al final d'aquest projecte, tindràs una comprensió sòlida de Vue.js i de com utilitzar-lo per construir aplicacions web dinàmiques.

Conceptes Clau

  • Arquitectura basada en components
  • Vue Router per a la navegació
  • Vuex per a la gestió d'estat
  • Integració d'API
  • Gestió i validació de formularis
  • Renderització dinàmica

Configuració del Projecte

Primer, configurem el nostre projecte Vue.js utilitzant Vue CLI.

# Instal·la Vue CLI si encara no ho has fet
npm install -g @vue/cli

# Crea un nou projecte
vue create blog-platform

# Navega al directori del projecte
cd blog-platform

# Inicia el servidor de desenvolupament
npm run serve

Configuració de Vue Router

Necessitem configurar Vue Router per gestionar la navegació entre diferents pàgines.

// src/router/index.js
import Vue from 'vue'
import VueRouter from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'

Vue.use(VueRouter)

const routes = [
  {
    path: '/',
    name: 'Home',
    component: Home
  },
  {
    path: '/about',
    name: 'About',
    component: About
  }
]

const router = new VueRouter({
  mode: 'history',
  base: process.env.BASE_URL,
  routes
})

export default router

Gestió d'Estat amb Vuex

Per gestionar l'estat de la nostra aplicació, utilitzarem Vuex.

// src/store/index.js
import Vue from 'vue'
import Vuex from 'vuex'

Vue.use(Vuex)

export default new Vuex.Store({
  state: {
    posts: []
  },
  mutations: {
    setPosts(state, posts) {
      state.posts = posts
    }
  },
  actions: {
    fetchPosts({ commit }) {
      // Simula una crida a l'API
      const posts = [
        { id: 1, title: 'Primera Publicació', content: 'Aquesta és la primera publicació.' },
        { id: 2, title: 'Segona Publicació', content: 'Aquesta és la segona publicació.' }
      ]
      commit('setPosts', posts)
    }
  },
  getters: {
    allPosts: state => state.posts
  }
})

Integració d'API

Simulem la recuperació de publicacions des d'una API.

// src/views/Home.vue
<template>
  <div>
    <Header />
    <div v-for="post in posts" :key="post.id">
      <Post :post="post" />
    </div>
  </div>
</template>

<script>
import Header from '@/components/Header.vue'
import Post from '@/components/Post.vue'
import { mapGetters, mapActions } from 'vuex'

export default {
  name: 'Home',
  components: {
    Header,
    Post
  },
  computed: {
    ...mapGetters(['allPosts'])
  },
  methods: {
    ...mapActions(['fetchPosts'])
  },
  created() {
    this.fetchPosts()
  }
}
</script>

Gestió i Validació de Formularis

Afegim un formulari per crear noves publicacions del blog.

<!-- src/components/NewPost.vue -->
<template>
  <div>
    <h2>Crea Nova Publicació</h2>
    <form @submit.prevent="createPost">
      <div>
        <label for="title">Títol</label>
        <input type="text" v-model="title" required />
      </div>
      <div>
        <label for="content">Contingut</label>
        <textarea v-model="content" required></textarea>
      </div>
      <button type="submit">Crear</button>
    </form>
  </div>
</template>

<script>
export default {
  name: 'NewPost',
  data() {
    return {
      title: '',
      content: ''
    }
  },
  methods: {
    createPost() {
      // Simula una crida a l'API per crear una nova publicació
      const newPost = {
        id: Date.now(),
        title: this.title,
        content: this.content
      }
      this.$store.commit('setPosts', [...this.$store.state.posts, newPost])
      this.title = ''
      this.content = ''
    }
  }
}
</script>

<style scoped>
form {
  display: flex;
  flex-direction: column;
}
form div {
  margin-bottom: 1rem;
}
</style>
