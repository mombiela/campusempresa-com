El Projecte Sobre nosaltres Contribuir Donacions Llicència

En aquest projecte, construirem una aplicació de xat en temps real utilitzant Vue.js. Aquest projecte t'ajudarà a entendre com gestionar dades en temps real, gestionar l'estat i integrar serveis de backend. Al final d'aquest projecte, tindràs una comprensió sòlida de com crear una aplicació web dinàmica i interactiva.

Conceptes Clau

  • Configuració d'un projecte Vue.js
  • Creació i gestió de components
  • Gestió de l'estat amb Vuex
  • Integració amb un backend utilitzant Firebase
  • Implementació d'actualitzacions en temps real amb Firebase Realtime Database
  • Gestió de l'autenticació d'usuaris

Configuració del Projecte

Primer, configurem un nou projecte Vue.js utilitzant Vue CLI.

# Instal·la Vue CLI si encara no ho has fet
npm install -g @vue/cli

# Crea un nou projecte
vue create real-time-chat

# Navega al directori del projecte
cd real-time-chat

MessageInput.vue

<template>
  <div class="message-input">
    <input v-model="newMessage" @keyup.enter="handleSend" placeholder="Escriu un missatge" />
    <button @click="handleSend">Enviar</button>
  </div>
</template>

<script>
export default {
  name: 'MessageInput',
  data() {
    return {
      newMessage: ''
    };
  },
  methods: {
    handleSend() {
      if (this.newMessage.trim() !== '') {
        this.$emit('sendMessage', this.newMessage);
        this.newMessage = '';
      }
    }
  }
};
</script>

Gestió de l'Estat amb Vuex

Per gestionar l'estat de la nostra aplicació, utilitzarem Vuex. Instal·la Vuex i configura'l en el teu projecte.

npm install vuex

store/index.js

import Vue from 'vue';
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    messages: []
  },
  mutations: {
    addMessage(state, message) {
      state.messages.push(message);
    }
  },
  actions: {
    sendMessage({ commit }, message) {
      commit('addMessage', message);
    }
  },
  getters: {
    messages: state => state.messages
  }
});

Actualització de la Botiga Vuex per a Firebase

Actualitza la botiga Vuex per gestionar actualitzacions en temps real des de Firebase.

import Vue from 'vue';
import Vuex from 'vuex';
import { database } from '../firebaseConfig';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    messages: []
  },
  mutations: {
    setMessages(state, messages) {
      state.messages = messages;
    },
    addMessage(state, message) {
      state.messages.push(message);
    }
  },
  actions: {
    fetchMessages({ commit }) {
      database.ref('messages').on('value', snapshot => {
        commit('setMessages', snapshot.val() || []);
      });
    },
    sendMessage({ commit }, message) {
      database.ref('messages').push(message);
      commit('addMessage', message);
    }
  },
  getters: {
    messages: state => state.messages
  }
});

Gestió de l'Autenticació d'Usuaris

Utilitzarem Firebase Authentication per gestionar l'inici de sessió i el registre d'usuaris.

Auth.vue

<template>
  <div class="auth">
    <input v-model="email" placeholder="Email" />
    <input v-model="password" type="password" placeholder="Contrasenya" />
    <button @click="login">Iniciar Sessió</button>
    <button @click="register">Registrar-se</button>
  </div>
</template>

<script>
import { auth } from '../firebaseConfig';

export default {
  name: 'Auth',
  data() {
    return {
      email: '',
      password: ''
    };
  },
  methods: {
    async login() {
      try {
        await auth.signInWithEmailAndPassword(this.email, this.password);
        this.$router.push('/chat');
      } catch (error) {
        console.error(error);
      }
    },
    async register() {
      try {
        await auth.createUserWithEmailAndPassword(this.email, this.password);
        this.$router.push('/chat');
      } catch (error) {
        console.error(error);
      }
    }
  }
};
</script>
