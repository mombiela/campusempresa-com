El Projecte Sobre nosaltres Contribuir Donacions Llicència

Configurar l'entorn de desenvolupament és el primer pas per començar a treballar amb Vue.js. Aquesta secció et guiarà a través del procés de configuració de tot el que necessites per començar a desenvolupar aplicacions Vue.js, des d'instal·lar Node.js fins a crear el teu primer projecte Vue.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Un ordinador amb connexió a internet.
  • Coneixements bàsics de JavaScript, HTML i CSS.

Instal·lació de Node.js i npm

Vue.js requereix Node.js i npm (Node Package Manager) per gestionar dependències i executar scripts.

  1. Descarregar i Instal·lar Node.js:

    • Visita el lloc web oficial de Node.js.
    • Descarrega la versió LTS (Long Term Support) per al teu sistema operatiu.
    • Segueix les instruccions d'instal·lació per al teu SO.

  2. Verificar la Instal·lació:

    • Obre el teu terminal o símbol del sistema.
    • Executa les següents comandes per comprovar les versions instal·lades: 
      node -v
npm -v
    • Hauries de veure els números de versió de Node.js i npm.

Instal·lació de Vue CLI

Vue CLI (Command Line Interface) és una eina potent per crear i gestionar projectes Vue.js.

  1. Instal·lar Vue CLI Globalment:

    • Executa la següent comanda al teu terminal: 
      npm install -g @vue/cli
    • Aquesta comanda instal·la Vue CLI globalment al teu sistema.

  2. Verificar la Instal·lació:

    • Executa la següent comanda per comprovar la versió instal·lada: 
      vue --version
    • Hauries de veure el número de versió de Vue CLI.

Creació d'un Nou Projecte Vue

Amb Vue CLI instal·lat, pots crear un nou projecte Vue fàcilment.

  1. Crear un Nou Projecte:

    • Navega al directori on vols crear el teu projecte.
    • Executa la següent comanda: 
      vue create my-vue-app
    • Substitueix my-vue-app pel nom desitjat del teu projecte.

  2. Escollir Preset:

    • Vue CLI et demanarà que escullis un preset. Per als principiants, es recomana seleccionar el preset per defecte prement Enter.

  3. Navegar al Directori del Projecte:

    • Després de crear el projecte, navega al directori del projecte: 
      cd my-vue-app

  4. Executar el Servidor de Desenvolupament:

    • Inicia el servidor de desenvolupament executant: 
      npm run serve
    • Obre el teu navegador i ves a http://localhost:8080. Hauries de veure la pàgina de benvinguda per defecte de Vue.js.

Configuració d'un Editor de Codi

Un bon editor de codi pot millorar significativament la teva experiència de desenvolupament. Aquí tens algunes opcions populars:

  • Visual Studio Code (VS Code):

    • Descarrega i instal·la des del lloc web oficial.
    • Instal·la l'extensió de Vue.js per a ressaltat de sintaxi i altres funcions.

  • WebStorm:

    • Un IDE potent per al desenvolupament de JavaScript. Descarrega des del lloc web oficial.

  • Sublime Text:

    • Un editor lleuger amb molts plugins. Descarrega des del lloc web oficial.

Instal·lació d'Eines i Extensions Addicionals

Per millorar el teu flux de treball de desenvolupament, considera instal·lar les següents eines i extensions:

  • Vue Devtools:

    • Una extensió de navegador per depurar aplicacions Vue.js.
    • Disponible per a Chrome i Firefox.

  • ESLint:

    • Una eina per identificar i corregir problemes en el teu codi JavaScript.
    • Instal·la ESLint al teu projecte executant: 
      npm install eslint --save-dev

Conclusió

Configurar l'entorn de desenvolupament és un pas crucial per començar el teu viatge amb Vue.js. Seguint els passos descrits en aquesta secció, ara hauries de tenir un entorn de desenvolupament Vue.js completament funcional. Has instal·lat Node.js, npm, Vue CLI i has configurat un editor de codi. Ara estàs llest per començar a construir aplicacions Vue.js. En les següents seccions, aprofundirem en els conceptes i característiques principals de Vue.js.
