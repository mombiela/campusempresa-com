Introducció

En aquest projecte, construirem un lloc de comerç electrònic completament funcional utilitzant Vue.js. Aquest projecte t'ajudarà a aplicar els conceptes apresos en les lliçons anteriors i t'introduirà a funcions més avançades de Vue.js. Al final d'aquest projecte, tindràs una comprensió sòlida de com crear una aplicació web dinàmica i interactiva.

Configuració del projecte

  • Instal·lar Vue CLI: Assegura't de tenir Vue CLI instal·lat. 
    npm install -g @vue/cli
  • Crear un nou projecte: Utilitza Vue CLI per crear un nou projecte. 
    vue create ecommerce-site
  • Navegar al directori del projecte: 
    cd ecommerce-site

Estructura del projecte

  • Components: Crear components reutilitzables per a diferents parts del lloc.
  • Vistes: Configurar diferents vistes per a l'aplicació.
  • Enrutament: Implementar Vue Router per a la navegació.
  • Gestió d'estat: Utilitzar Vuex per gestionar l'estat de l'aplicació.

Components clau

Component de capçalera

  • Propòsit: Mostrar el logotip del lloc, enllaços de navegació i una icona del carret de la compra.
  • Exemple de codi: 
    <template>
  <header>
    <h1>El meu lloc de comerç electrònic</h1>
    <nav>
      <router-link to="/">Inici</router-link>
      <router-link to="/products">Productes</router-link>
      <router-link to="/cart">Carret</router-link>
    </nav>
  </header>
</template>

<script>
export default {
  name: 'HeaderComponent'
}
</script>

<style scoped>
header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
  background-color: #333;
  color: #fff;
}
nav a {
  margin: 0 1rem;
  color: #fff;
  text-decoration: none;
}
</style>

Component de llista de productes

  • Propòsit: Mostrar una llista de productes obtinguts d'una API.
  • Exemple de codi: 
    <template>
  <div>
    <h2>Productes</h2>
    <div v-if="loading">Carregant...</div>
    <div v-else>
      <div v-for="product in products" :key="product.id" class="product">
        <h3>{{ product.name }}</h3>
        <p>{{ product.description }}</p>
        <p>\${{ product.price }}</p>
        <button @click="addToCart(product)">Afegir al carret</button>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      products: [],
      loading: true
    };
  },
  created() {
    fetch('https://api.example.com/products')
      .then(response => response.json())
      .then(data => {
        this.products = data;
        this.loading = false;
      });
  },
  methods: {
    addToCart(product) {
      this.$store.commit('addToCart', product);
    }
  }
}
</script>

<style scoped>
.product {
  border: 1px solid #ccc;
  padding: 1rem;
  margin: 1rem 0;
}
</style>

Enrutament

  • Configurar Vue Router: Instal·lar i configurar Vue Router. 
    npm install vue-router
  • Definir rutes: Crear rutes per a les pàgines d'Inici, Productes i Carret. 
    import Vue from 'vue';
import Router from 'vue-router';
import Home from './views/Home.vue';
import Products from './views/Products.vue';
import Cart from './views/Cart.vue';

Vue.use(Router);

export default new Router({
  routes: [
    { path: '/', component: Home },
    { path: '/products', component: Products },
    { path: '/cart', component: Cart }
  ]
});

Gestió d'estat

  • Configurar Vuex: Instal·lar i configurar Vuex per a la gestió d'estat. 
    npm install vuex
  • Definir Store: Crear una botiga Vuex per gestionar l'estat del carret. 
    import Vue from 'vue';
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    cart: []
  },
  mutations: {
    addToCart(state, product) {
      state.cart.push(product);
    }
  }
});

Conclusió

En aquest projecte, hem cobert la creació d'un lloc de comerç electrònic utilitzant Vue.js. Hem configurat el projecte, creat components clau, implementat enrutament i gestionat l'estat amb Vuex. Aquest projecte ha proporcionat una visió general completa de com construir una aplicació web dinàmica amb Vue.js, preparant-te per a temes més avançats i aplicacions del món real.
