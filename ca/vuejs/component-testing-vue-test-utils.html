Introducció a Vue Test Utils

Vue Test Utils és la biblioteca oficial d'utilitats per a proves unitàries per a Vue.js. Proporciona mètodes per muntar i interactuar amb components de Vue, facilitant la prova del seu comportament i sortida.

Conceptes clau

  • Muntar components: Renderitzar un component en aïllament per a proves.
  • Trobar elements: Consultar el component renderitzat per trobar elements.
  • Simular esdeveniments: Desencadenar esdeveniments com clics o canvis d'entrada.
  • Afirmacions: Comprovar si el component es comporta com s'espera.

Configuració de Vue Test Utils

Abans de començar a provar, cal configurar l'entorn.

Instal·lació

Per instal·lar Vue Test Utils, pots utilitzar npm o yarn:

npm install @vue/test-utils --save-dev

o

yarn add @vue/test-utils --dev

També necessitaràs un executor de proves com Jest o Mocha. Per a aquest curs, utilitzarem Jest:

npm install jest --save-dev

Escriure la teva primera prova

Escrivim una prova senzilla per a un component de Vue.

Component d'exemple

Considera un component Counter senzill:

<template>
  <div>
    <p>{{ count }}</p>
    <button @click="increment">Increment</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      count: 0
    };
  },
  methods: {
    increment() {
      this.count++;
    }
  }
};
</script>

Escriure la prova

Crea un fitxer de prova Counter.spec.js:

import { mount } from '@vue/test-utils';
import Counter from '@/components/Counter.vue';

describe('Counter.vue', () => {
  it('increments count when button is clicked', async () => {
    const wrapper = mount(Counter);
    
    expect(wrapper.text()).toContain('0');
    
    await wrapper.find('button').trigger('click');
    
    expect(wrapper.text()).toContain('1');
  });
});

Explicació

  • Muntar: mount(Counter) renderitza el component Counter.
  • Afirmacions: expect(wrapper.text()).toContain('0') comprova el recompte inicial.
  • Simular esdeveniments: await wrapper.find('button').trigger('click') simula un clic de botó.
  • Afirmacions: expect(wrapper.text()).toContain('1') comprova el recompte després del clic.

Tècniques avançades de proves

A mesura que et familiaritzis amb les proves bàsiques, pots explorar tècniques més avançades.

Provar Props

Pots provar com es comporten els components amb diferents props.

import { mount } from '@vue/test-utils';
import Message from '@/components/Message.vue';

describe('Message.vue', () => {
  it('renders props.msg when passed', () => {
    const msg = 'Hello, Vue!';
    const wrapper = mount(Message, {
      propsData: { msg }
    });
    expect(wrapper.text()).toContain(msg);
  });
});

Provar esdeveniments emesos

Pots provar si un component emet esdeveniments correctament.

import { mount } from '@vue/test-utils';
import CustomButton from '@/components/CustomButton.vue';

describe('CustomButton.vue', () => {
  it('emits an event when clicked', async () => {
    const wrapper = mount(CustomButton);
    
    await wrapper.find('button').trigger('click');
    
    expect(wrapper.emitted()).toHaveProperty('custom-event');
  });
});

Conclusió

Les proves de components amb Vue Test Utils són una manera poderosa d'assegurar que els teus components de Vue funcionin com s'espera. Dominant els conceptes bàsics de muntar components, simular esdeveniments i fer afirmacions, pots escriure proves robustes que ajudin a mantenir la qualitat del teu codi. A mesura que avancis, explorar props i esdeveniments emesos millorarà encara més les teves capacitats de prova. Bones proves!
