Plugins de Vue.js ►

Introducció

En Vue.js, les plantilles són la manera més comuna de definir l'estructura dels teus components. No obstant això, per a casos d'ús més avançats, podries necessitar la flexibilitat de JavaScript per definir la sortida del teu component. Aquí és on entren en joc les funcions de renderització i JSX.

Què són les funcions de renderització?

Les funcions de renderització són una alternativa a les plantilles en Vue.js. Proporcionen una manera de descriure programàticament l'estructura del teu component utilitzant JavaScript.

Conceptes clau

  • Funció de renderització: Una funció que retorna un arbre de DOM Virtual (VNode).
  • VNode: Un objecte JavaScript que representa un node DOM.
  • Funció h: Una funció auxiliar utilitzada per crear VNodes.

Exemple bàsic

Aquí tens un exemple senzill d'una funció de renderització:

export default {
  render(h) {
    return h('div', { class: 'hello' }, [
      h('h1', 'Hola, Vue.js!'),
      h('p', 'Això és un paràgraf.')
    ]);
  }
};

En aquest exemple:

  • h és la funció createElement.
  • El primer argument és el nom de l'etiqueta.
  • El segon argument és un objecte que conté atributs i propietats.
  • El tercer argument és una matriu de fills.

JSX en Vue.js

JSX és una extensió de sintaxi per a JavaScript que s'assembla a HTML. Es pot utilitzar en Vue.js per escriure funcions de renderització d'una manera més llegible.

Conceptes clau

  • JSX: JavaScript XML, una extensió de sintaxi que permet escriure codi semblant a HTML en JavaScript.
  • Babel: Un compilador de JavaScript que pot transformar JSX en JavaScript regular.

Exemple bàsic

Aquí tens el mateix exemple anterior, però escrit utilitzant JSX:

export default {
  render() {
    return (
      <div class="hello">
        <h1>Hola, Vue.js!</h1>
        <p>Això és un paràgraf.</p>
      </div>
    );
  }
};

Configuració de JSX

Per utilitzar JSX en el teu projecte Vue.js, necessites configurar Babel amb els plugins adequats:

  1. Instal·la els paquets necessaris:

    npm install @babel/plugin-transform-vue-jsx @vue/babel-preset-jsx

  2. Actualitza la teva configuració de Babel (babel.config.js):

    module.exports = {
  presets: [
    '@vue/cli-plugin-babel/preset'
  ],
  plugins: [
    '@babel/plugin-transform-vue-jsx'
  ]
};

Quan utilitzar funcions de renderització i JSX

Les funcions de renderització i JSX són eines poderoses, però no sempre són necessàries. Aquí tens alguns escenaris on podrien ser útils:

  • Generació dinàmica de components: Quan necessites generar components dinàmicament basant-te en lògica complexa.
  • Optimització del rendiment: Quan necessites un control detallat sobre el procés de renderització per raons de rendiment.
  • Biblioteques de tercers: Quan s'integra amb biblioteques que requereixen funcions de renderització o JSX.

Exercicis

Exercici 1: Convertir plantilla a funció de renderització

Converteix la següent plantilla en una funció de renderització:

<template>
  <div class="container">
    <h2>{{ title }}</h2>
    <ul>
      <li v-for="item in items" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>

Solució

export default {
  props: ['title', 'items'],
  render(h) {
    return h('div', { class: 'container' }, [
      h('h2', this.title),
      h('ul', this.items.map(item => 
        h('li', { key: item.id }, item.name)
      ))
    ]);
  }
};

Exercici 2: Utilitzar JSX per crear un component

Crea un component utilitzant JSX que mostri una llista d'usuaris amb els seus noms i edats.

Solució

export default {
  props: ['users'],
  render() {
    return (
      <div class="user-list">
        <ul>
          {this.users.map(user => (
            <li key={user.id}>
              {user.name} - {user.age} anys
            </li>
          ))}
        </ul>
      </div>
    );
  }
};

Conclusió

Les funcions de renderització i JSX proporcionen alternatives poderoses a les plantilles en Vue.js, oferint una major flexibilitat i control sobre el procés de renderització del teu component. Tot i que les plantilles són suficients per a la majoria dels casos d'ús, entendre les funcions de renderització i JSX pot ser beneficiós per a escenaris avançats. Practica convertint plantilles a funcions de renderització i utilitzant JSX per sentir-te més còmode amb aquests conceptes.
