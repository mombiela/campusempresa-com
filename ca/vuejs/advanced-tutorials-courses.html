Introducció a Vue.js avançat

En aquesta secció, aprofundirem en les característiques i capacitats avançades de Vue.js. Això inclou la gestió d'estat, el routing, el renderitzat del costat del servidor i l'optimització del rendiment. Al final d'aquesta secció, seràs capaç de construir aplicacions Vue.js complexes i d'alt rendiment.

Gestió d'estat amb Vuex

Vuex és un patró de gestió d'estat + biblioteca per a aplicacions Vue.js. Serveix com una botiga centralitzada per a tots els components d'una aplicació, amb regles que asseguren que l'estat només es pot mutar de manera previsible.

  • Conceptes bàsics de Vuex:
    • Estat: La font única de veritat.
    • Getters: Propietats computades per a botigues.
    • Mutacions: Transaccions síncrones.
    • Accions: Operacions asíncrones.
    • Mòduls: Dividir la botiga en mòduls.

Exemple: Configurar Vuex

// store.js
import Vue from 'vue';
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    count: 0
  },
  mutations: {
    increment(state) {
      state.count++;
    }
  },
  actions: {
    increment({ commit }) {
      commit('increment');
    }
  },
  getters: {
    count: state => state.count
  }
});

Utilitzar Vuex en un component

<template>
  <div>
    <p>{{ count }}</p>
    <button @click="increment">Incrementar</button>
  </div>
</template>

<script>
import { mapState, mapActions } from 'vuex';

export default {
  computed: {
    ...mapState(['count'])
  },
  methods: {
    ...mapActions(['increment'])
  }
};
</script>

Routing avançat amb Vue Router

Vue Router és el router oficial per a Vue.js. S'integra profundament amb el nucli de Vue.js per facilitar la construcció d'aplicacions d'una sola pàgina amb Vue.js.

  • Routing dinàmic: Rutes que canvien segons els paràmetres.
  • Rutes niades: Rutes dins de rutes.
  • Guàrdies de navegació: Controlar l'accés a les rutes.
  • Càrrega mandrosa de rutes: Millorar el rendiment carregant rutes a demanda.

Exemple: Routing dinàmic

// router.js
import Vue from 'vue';
import Router from 'vue-router';
import User from './components/User.vue';

Vue.use(Router);

export default new Router({
  routes: [
    {
      path: '/user/:id',
      component: User
    }
  ]
});

Utilitzar rutes dinàmiques en un component

<template>
  <div>
    <h1>ID d'usuari: {{ $route.params.id }}</h1>
  </div>
</template>

<script>
export default {
  watch: {
    '$route' (to, from) {
      // reaccionar als canvis de ruta...
    }
  }
};
</script>

Renderitzat del costat del servidor (SSR) amb Nuxt.js

Nuxt.js és un framework per crear aplicacions Vue.js amb renderitzat del costat del servidor, generació de llocs estàtics i més.

  • Beneficis del SSR:
    • Millora del SEO.
    • Temps de contingut més ràpid.
    • Millor experiència d'usuari.

Exemple: Configurar Nuxt.js

# Instal·lar Nuxt.js
npx create-nuxt-app <nom-del-projecte>

Configuració bàsica de Nuxt.js

// nuxt.config.js
export default {
  head: {
    title: 'La meva aplicació Nuxt.js',
    meta: [
      { charset: 'utf-8' },
      { name: 'viewport', content: 'width=device-width, initial-scale=1' },
      { hid: 'description', name: 'description', content: 'El meu projecte Nuxt.js' }
    ]
  },
  modules: [
    '@nuxtjs/axios'
  ],
  axios: {
    // Configuració del mòdul Axios
  }
};

Optimització del rendiment

Optimitzar el rendiment de la teva aplicació Vue.js és crucial per proporcionar una experiència d'usuari fluida.

  • Càrrega mandrosa de components: Carregar components només quan sigui necessari.
  • Divisió de codi: Dividir el teu codi en fragments més petits.
  • Utilitzar Vue Devtools: Identificar colls d'ampolla de rendiment.
  • Optimització d'imatges i recursos: Reduir la mida de les imatges i altres recursos.

Exemple: Càrrega mandrosa d'un component

// Carregar mandrosament un component
const LazyComponent = () => import('./components/LazyComponent.vue');

export default {
  components: {
    LazyComponent
  }
};

Conclusió

En aquesta secció avançada del curs de Vue.js, hem cobert temes essencials com la gestió d'estat amb Vuex, el routing avançat amb Vue Router, el renderitzat del costat del servidor amb Nuxt.js i tècniques d'optimització del rendiment. Dominar aquests conceptes et permetrà construir aplicacions Vue.js robustes, escalables i d'alt rendiment. Continua practicant i explorant aquestes característiques avançades per convertir-te en un expert en el desenvolupament amb Vue.js.
