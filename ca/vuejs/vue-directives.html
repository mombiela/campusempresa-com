Les directives de Vue.js són tokens especials en el marcat que indiquen a la biblioteca que faci alguna cosa a un element del DOM. Estan prefixades amb v- per indicar que són atributs especials proporcionats per Vue.js.

Directives Clau

v-bind

  • Propòsit: Enllaçar dinàmicament un o més atributs, o una propietat de component, a una expressió.
  • Sintaxi: v-bind:attribute="expression"
<!-- Enllaçant un atribut -->
<img v-bind:src="imageSrc" alt="Imatge Dinàmica">

<!-- Forma abreujada -->
<img :src="imageSrc" alt="Imatge Dinàmica">

v-model

  • Propòsit: Crear un enllaç bidireccional en un element input, textarea o select.
  • Sintaxi: v-model="data"
<!-- Enllaç bidireccional amb un element input -->
<input v-model="message" placeholder="Escriu un missatge">

<p>El missatge és: {{ message }}</p>

v-if, v-else-if, v-else

  • Propòsit: Renderitzar elements condicionalment basant-se en la veracitat d'una expressió.
  • Sintaxi: v-if="condition", v-else-if="condition", v-else
<!-- Renderització condicional -->
<p v-if="seen">Ara em veus</p>
<p v-else>Ara no em veus</p>

v-show

  • Propòsit: Alternar la visibilitat d'un element basant-se en la veracitat d'una expressió.
  • Sintaxi: v-show="condition"
<!-- Alternar visibilitat -->
<p v-show="isVisible">Sóc visible</p>

v-for

  • Propòsit: Renderitzar una llista d'elements iterant sobre un array o objecte.
  • Sintaxi: v-for="item in items"
<!-- Iterant sobre un array -->
<ul>
  <li v-for="item in items" :key="item.id">{{ item.name }}</li>
</ul>

v-on

  • Propòsit: Adjuntar oients d'esdeveniments a elements.
  • Sintaxi: v-on:event="method"
<!-- Maneig d'esdeveniments -->
<button v-on:click="doSomething">Fes clic aquí</button>

<!-- Forma abreujada -->
<button @click="doSomething">Fes clic aquí</button>

v-pre

  • Propòsit: Saltar la compilació per aquest element i tots els seus fills.
  • Sintaxi: v-pre
<!-- Saltant la compilació -->
<span v-pre>{{ això no serà compilat }}</span>

v-cloak

  • Propòsit: Mantenir l'element i els seus fills ocults fins que la compilació de Vue estigui completa.
  • Sintaxi: v-cloak
<!-- Prevenint el flaix de contingut no compilat -->
<style>
[v-cloak] { display: none; }
</style>

<div v-cloak>
  {{ message }}
</div>

v-once

  • Propòsit: Renderitzar l'element i el component una vegada i saltar actualitzacions futures.
  • Sintaxi: v-once
<!-- Renderització única -->
<span v-once>Això mai canviarà: {{ message }}</span>

Directives Personalitzades

Vue.js també permet crear les teves pròpies directives personalitzades. Les directives personalitzades poden ser útils per encapsular manipulacions del DOM reutilitzables.

Registrant una Directiva Personalitzada

Vue.directive('focus', {
  // Quan l'element vinculat s'insereix al DOM...
  inserted: function (el) {
    // Enfocar l'element
    el.focus()
  }
})

Utilitzant una Directiva Personalitzada

<!-- Aplicant la directiva personalitzada -->
<input v-focus>

Resum

Les directives de Vue.js són eines poderoses que permeten manipular el DOM de manera declarativa. Cobreixen una àmplia gamma de funcionalitats, des de l'enllaç d'atributs i oients d'esdeveniments fins a la renderització condicional i la renderització de llistes. Entendre i utilitzar efectivament aquestes directives és crucial per construir aplicacions dinàmiques i responsives amb Vue.js. Les directives personalitzades amplien encara més les capacitats de Vue, permetent encapsular i reutilitzar manipulacions del DOM a través de la teva aplicació.
