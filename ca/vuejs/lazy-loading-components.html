Càrrega Per Mandra de Components en Vue.js
Divisió de codi ►

La càrrega per mandra és un patró de disseny comunament utilitzat en la programació informàtica per ajornar la inicialització d'un objecte fins al moment en què és necessari. En el context de Vue.js, la càrrega per mandra de components pot millorar significativament el rendiment de la teva aplicació reduint el temps de càrrega inicial. Aquest tema et guiarà des dels conceptes bàsics fins a tècniques avançades de càrrega per mandra de components en Vue.js.

Què és la Càrrega Per Mandra?

  • Definició: La càrrega per mandra és una tècnica que retarda la càrrega de recursos fins que realment es necessiten.
  • Beneficis:
    • Redueix el temps de càrrega inicial.
    • Disminueix la quantitat de dades que cal processar en l'inici.
    • Millora el rendiment general de l'aplicació.

Càrrega Per Mandra Bàsica en Vue.js

Imports Dinàmics

Vue.js suporta imports dinàmics, que és una característica clau per implementar la càrrega per mandra.

// Importar un component dinàmicament
const MyComponent = () => import('./MyComponent.vue');
  • Explicació: La funció import() retorna una Promesa que es resol amb el mòdul. Això permet que el component es carregui només quan sigui necessari.

Utilitzant Imports Dinàmics en Rutes

La càrrega per mandra és particularment útil en Vue Router per carregar components de ruta.

import Vue from 'vue';
import Router from 'vue-router';

Vue.use(Router);

const routes = [
  {
    path: '/about',
    name: 'About',
    component: () => import('./views/About.vue') // Ruta carregada per mandra
  }
];

const router = new Router({
  routes
});

export default router;
  • Explicació: En la configuració del router, la propietat component utilitza un import dinàmic per carregar el component About només quan es visita la ruta /about.

Tècniques Avançades de Càrrega Per Mandra

Agrupant Components amb Webpack

Webpack et permet agrupar múltiples components en un sol chunk. Això pot ser útil per optimitzar la càrrega de components relacionats.

const GroupedComponent = () => import(/* webpackChunkName: "grouped" */ './GroupedComponent.vue');
const AnotherGroupedComponent = () => import(/* webpackChunkName: "grouped" */ './AnotherGroupedComponent.vue');
  • Explicació: Especificant el mateix webpackChunkName, Webpack agruparà aquests components en un sol chunk, que es carregarà conjuntament.

Prefetching i Preloading

Webpack també suporta prefetching i preloading, que poden optimitzar encara més la càrrega de components.

const PrefetchedComponent = () => import(/* webpackPrefetch: true */ './PrefetchedComponent.vue');
const PreloadedComponent = () => import(/* webpackPreload: true */ './PreloadedComponent.vue');
  • Prefetching: Instruccions al navegador per obtenir el recurs quan estigui inactiu.
  • Preloading: Instruccions al navegador per obtenir el recurs tan aviat com sigui possible.

Càrrega Per Mandra amb Suspense (Vue 3)

Vue 3 introdueix el component Suspense, que proporciona una manera de gestionar components asíncrons de manera més elegant.

<template>
  <Suspense>
    <template #default>
      <AsyncComponent />
    </template>
    <template #fallback>
      <div>Loading...</div>
    </template>
  </Suspense>
</template>

<script>
const AsyncComponent = () => import('./AsyncComponent.vue');

export default {
  components: {
    AsyncComponent
  }
};
</script>
  • Explicació: El component Suspense et permet especificar un contingut de reserva mentre el component asíncron s'està carregant.

Conclusió

La càrrega per mandra és una tècnica poderosa per optimitzar el rendiment de les teves aplicacions Vue.js. Aplaçant la càrrega de components fins que siguin necessaris, pots reduir significativament el temps de càrrega inicial i millorar l'experiència de l'usuari. Ja sigui utilitzant imports dinàmics, agrupant components amb Webpack, o aprofitant el nou component Suspense en Vue 3, la càrrega per mandra proporciona una manera flexible i eficient de gestionar els recursos de la teva aplicació.
