Introducció

En aquesta secció, et guiarem a través del procés de creació de la teva primera aplicació Vue.js. Vue.js és un framework JavaScript progressiu utilitzat per construir interfícies d'usuari. Està dissenyat per ser adoptat incrementalment, el que significa que pots utilitzar tant o tan poc de Vue com necessitis.

Prerequisits

Abans de començar, assegura't de tenir el següent:

  • Coneixements bàsics d'HTML, CSS i JavaScript.
  • Node.js i npm instal·lats al teu ordinador.

Configurant l'entorn de desenvolupament

Instal·lant Vue CLI

Vue CLI (Command Line Interface) és una eina poderosa per esbossar i gestionar projectes Vue.js. Per instal·lar Vue CLI, obre el teu terminal i executa:

npm install -g @vue/cli

Creant un nou projecte Vue

Un cop Vue CLI està instal·lat, pots crear un nou projecte Vue executant:

vue create my-first-vue-app

Se't demanarà que triïs un preset. Per als principiants, es recomana seleccionar el preset per defecte prement Enter.

Navegant al directori del teu projecte

Després de crear el projecte, navega al directori del teu projecte:

cd my-first-vue-app

Executant el servidor de desenvolupament

Per iniciar el servidor de desenvolupament, executa:

npm run serve

Hauries de veure una sortida indicant que la teva aplicació s'està executant a http://localhost:8080.

Entenent l'estructura del projecte

Un projecte Vue típic creat amb Vue CLI té la següent estructura:

my-first-vue-app/
├── node_modules/
├── public/
│   ├── index.html
├── src/
│   ├── assets/
│   ├── components/
│   ├── App.vue
│   ├── main.js
├── .gitignore
├── babel.config.js
├── package.json
├── README.md
  • public/index.html: El fitxer HTML principal.
  • src/main.js: El punt d'entrada de l'aplicació.
  • src/App.vue: El component arrel.
  • src/components/: Directori per als components Vue.

Creant el teu primer component

Modificant el component App

Obre src/App.vue i reemplaça el seu contingut amb:

<template>
  <div id="app">
    <h1>{{ message }}</h1>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: 'Hello, Vue!'
    };
  }
};
</script>

<style>
#app {
  font-family: Avenir, Helvetica, Arial, sans-serif;
  text-align: center;
  color: #2c3e50;
  margin-top: 60px;
}
</style>

Aquest codi defineix un component Vue simple que mostra un missatge.

Creant un nou component

Crea un nou fitxer src/components/HelloWorld.vue i afegeix el següent contingut:

<template>
  <div>
    <h2>{{ greeting }}</h2>
  </div>
</template>

<script>
export default {
  data() {
    return {
      greeting: 'Welcome to your first Vue component!'
    };
  }
};
</script>

<style scoped>
h2 {
  color: #42b983;
}
</style>

Utilitzant el nou component

Per utilitzar el nou component a App.vue, primer importa'l i després registra'l:

<template>
  <div id="app">
    <h1>{{ message }}</h1>
    <HelloWorld />
  </div>
</template>

<script>
import HelloWorld from './components/HelloWorld.vue';

export default {
  components: {
    HelloWorld
  },
  data() {
    return {
      message: 'Hello, Vue!'
    };
  }
};
</script>

<style>
#app {
  font-family: Avenir, Helvetica, Arial, sans-serif;
  text-align: center;
  color: #2c3e50;
  margin-top: 60px;
}
</style>

Conclusió

Felicitats! Has creat amb èxit la teva primera aplicació Vue.js. Has après a configurar l'entorn de desenvolupament, crear un nou projecte Vue, entendre l'estructura del projecte, i crear i utilitzar components Vue. Aquest coneixement fonamental serà crucial mentre continues explorant característiques més avançades de Vue.js.
