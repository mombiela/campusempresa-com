Què són els Components?

Els components són una de les característiques més poderoses de Vue.js. Et permeten descompondre la teva aplicació en peces més petites i reutilitzables. Cada component encapsula la seva pròpia estructura, estil i comportament, fent que el teu codi sigui més modular i fàcil de mantenir.

Conceptes Clau

  • Reutilització: Els components es poden reutilitzar en diferents parts de la teva aplicació.
  • Encapsulació: Cada component gestiona el seu propi estat i comportament.
  • Jerarquia: Els components es poden niar dins d'altres components, formant una estructura d'arbre.

Creant un Component Bàsic

Per crear un component en Vue.js, el defineixes utilitzant el mètode Vue.component o mitjançant la propietat components dins d'una instància de Vue.

Exemple: Registre Global de Component

// Defineix un nou component anomenat 'my-component'
Vue.component('my-component', {
  template: '<div>Un component personalitzat!</div>'
})

// Crea una nova instància de Vue
new Vue({
  el: '#app'
})

En aquest exemple:

  • Definim un component anomenat my-component.
  • La propietat template conté l'estructura HTML del component.
  • El component es registra globalment, el que significa que es pot utilitzar en qualsevol lloc de la instància de Vue.

Exemple: Registre Local de Component

// Defineix un nou component
const MyComponent = {
  template: '<div>Un component personalitzat!</div>'
}

// Crea una nova instància de Vue
new Vue({
  el: '#app',
  components: {
    'my-component': MyComponent
  }
})

En aquest exemple:

  • Definim un component i l'emmagatzemem en una variable MyComponent.
  • Registrem el component localment dins de la instància de Vue utilitzant la propietat components.

Utilitzant Components

Un cop un component està registrat, el pots utilitzar en el teu HTML referenciant el seu nom.

Exemple: Utilitzant un Component

<div id="app">
  <my-component></my-component>
</div>

En aquest exemple:

  • L'etiqueta my-component s'utilitza dins de l'HTML, i Vue.js renderitzarà la plantilla del component en el seu lloc.

Passant Dades als Components

Els components poden rebre dades del seu pare utilitzant props.

Exemple: Passant Props

// Defineix un nou component
Vue.component('my-component', {
  props: ['message'],
  template: '<div>{{ message }}</div>'
})

// Crea una nova instància de Vue
new Vue({
  el: '#app'
})
<div id="app">
  <my-component message="Hola, Vue!"></my-component>
</div>

En aquest exemple:

  • La propietat props s'utilitza per declarar una prop anomenada message.
  • El component pare passa la prop message al my-component.

Comunicació entre Components

Els components sovint necessiten comunicar-se entre ells. Això es pot fer utilitzant esdeveniments personalitzats.

Exemple: Emissió d'Esdeveniments

// Defineix un nou component
Vue.component('child-component', {
  template: '<button @click="sendMessage">Fes clic aquí</button>',
  methods: {
    sendMessage() {
      this.$emit('message', 'Hola des del fill!')
    }
  }
})

// Crea una nova instància de Vue
new Vue({
  el: '#app',
  methods: {
    handleMessage(message) {
      alert(message)
    }
  }
})
<div id="app">
  <child-component @message="handleMessage"></child-component>
</div>

En aquest exemple:

  • El child-component emet un esdeveniment personalitzat anomenat message.
  • El component pare escolta l'esdeveniment message i el gestiona amb el mètode handleMessage.

Conclusió

Els components són blocs de construcció fonamentals en Vue.js que t'ajuden a crear codi modular, reutilitzable i fàcil de mantenir. Entenent com crear, utilitzar i comunicar-se entre components, pots construir aplicacions complexes de manera més eficient. A mesura que progressis, aprendràs tècniques més avançades per gestionar l'estat, hooks de cicle de vida i optimitzar el rendiment amb components.
