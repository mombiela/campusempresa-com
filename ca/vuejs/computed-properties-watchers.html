Introducció

A Vue.js, les propietats computades i els watchers són característiques poderoses que ajuden a gestionar i reaccionar als canvis de dades de manera eficient. Aquesta secció cobrirà els conceptes bàsics de les propietats computades i els watchers, les seves diferències i com utilitzar-los de manera efectiva en les teves aplicacions Vue.js.

Propietats Computades

Què són les Propietats Computades?

  • Les propietats computades són propietats que depenen d'altres propietats de dades.
  • Es cachegen basant-se en les seves dependències, és a dir, només es reavaluaran quan una de les seves dependències canviï.
  • Són útils per a càlculs complexos o transformacions de dades que necessiten ser reactives.

Ús Bàsic

Les propietats computades es defineixen dins de l'opció computed en un component Vue.

<template>
  <div>
    <p>Missatge Original: {{ message }}</p>
    <p>Missatge Reversat: {{ reversedMessage }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: 'Hello Vue.js!'
    };
  },
  computed: {
    reversedMessage() {
      return this.message.split('').reverse().join('');
    }
  }
};
</script>

Conceptes Clau

  • Cache: Les propietats computades es cachegen basant-se en les seves dependències reactives.
  • Seguiment de Dependències: Vue fa un seguiment automàtic de les dependències de les propietats computades.

Exemple: Propietat Computada amb Múltiples Dependències

<template>
  <div>
    <p>Nom Complet: {{ fullName }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      firstName: 'John',
      lastName: 'Doe'
    };
  },
  computed: {
    fullName() {
      return `${this.firstName} ${this.lastName}`;
    }
  }
};
</script>

Watchers

Què són els Watchers?

  • Els watchers són funcions que observen canvis en una propietat de dades específica i executen una callback quan les dades canvien.
  • Són útils per a realitzar operacions asíncrones o lògica complexa en resposta als canvis de dades.

Ús Bàsic

Els watchers es defineixen dins de l'opció watch en un component Vue.

<template>
  <div>
    <p>Missatge: {{ message }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: 'Hello Vue.js!'
    };
  },
  watch: {
    message(newValue, oldValue) {
      console.log(`El missatge ha canviat de ${oldValue} a ${newValue}`);
    }
  }
};
</script>

Conceptes Clau

  • Execució Immediata: Els watchers es poden configurar per a executar-se immediatament després de la seva creació.
  • Observació Profunda: Els watchers es poden configurar per a observar propietats niades.

Exemple: Watcher amb Operació Asíncrona

<template>
  <div>
    <p>Consulta de Cerca: {{ query }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      query: ''
    };
  },
  watch: {
    query(newQuery) {
      this.fetchResults(newQuery);
    }
  },
  methods: {
    fetchResults(query) {
      // Simular una crida a l'API
      setTimeout(() => {
        console.log(`Cercant resultats per a ${query}`);
      }, 500);
    }
  }
};
</script>

Comparació: Propietats Computades vs. Watchers

| Característica | Propietats Computades | Watchers | |-----------------------|----------------------------------------------|-----------------------------------------------| | Cas d'Ús | Transformació de dades, estat derivat | Operacions asíncrones, lògica complexa | | Cache | Sí | No | | Seguiment de Dependències | Automàtic | Manual | | Sintaxi | Definides a l'opció computed | Definides a l'opció watch | | Execució | Sincrònica | Pot ser asíncrona |

Conclusió

Les propietats computades i els watchers són eines essencials a Vue.js per a gestionar dades reactives. Les propietats computades són ideals per a transformacions de dades i estat derivat, mentre que els watchers són més adequats per a gestionar efectes secundaris i lògica complexa en resposta als canvis de dades. Entendre quan i com utilitzar aquestes característiques t'ajudarà a construir aplicacions Vue.js més eficients i mantenibles.
