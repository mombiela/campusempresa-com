Introducció a Vuex

Vuex és un patró de gestió d'estat + biblioteca per a aplicacions Vue.js. Serveix com a magatzem centralitzat per a tots els components d'una aplicació, amb regles que asseguren que l'estat només es pot mutar de manera previsible. Vuex s'utilitza sovint en aplicacions de mida mitjana a gran per gestionar l'estat de manera més eficient.

Configurant Vuex

Abans d'utilitzar Vuex en els teus components, has de configurar-lo en el teu projecte Vue.js.

  1. Instal·lar Vuex:

    npm install vuex

  2. Crear un Magatzem Vuex:

    // store.js
import Vue from 'vue';
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
    state: {
        count: 0
    },
    mutations: {
        increment(state) {
            state.count++;
        }
    },
    actions: {
        increment({ commit }) {
            commit('increment');
        }
    },
    getters: {
        count: state => state.count
    }
});

  3. Integrar el Magatzem amb la Instància Vue:

    // main.js
import Vue from 'vue';
import App from './App.vue';
import store from './store';

new Vue({
    store,
    render: h => h(App)
}).$mount('#app');

Accedint a l'Estat en Components

Per accedir a l'estat en els teus components Vue, pots utilitzar el helper mapState de Vuex.

  1. Utilitzant mapState:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ count }}</p>
    </div>
</template>

<script>
import { mapState } from 'vuex';

export default {
    computed: {
        ...mapState(['count'])
    }
};
</script>

  2. Accés Directe:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ $store.state.count }}</p>
    </div>
</template>

<script>
export default {
    computed: {
        count() {
            return this.$store.state.count;
        }
    }
};
</script>

Cometent Mutacions

Per canviar l'estat, necessites cometre mutacions. Això assegura que els canvis d'estat siguin rastrejables i predictibles.

  1. Utilitzant mapMutations:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ count }}</p>
        <button @click="increment">Increment</button>
    </div>
</template>

<script>
import { mapState, mapMutations } from 'vuex';

export default {
    computed: {
        ...mapState(['count'])
    },
    methods: {
        ...mapMutations(['increment'])
    }
};
</script>

  2. Comet Directe:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ count }}</p>
        <button @click="increment">Increment</button>
    </div>
</template>

<script>
export default {
    computed: {
        count() {
            return this.$store.state.count;
        }
    },
    methods: {
        increment() {
            this.$store.commit('increment');
        }
    }
};
</script>

Despatxant Accions

Les accions són similars a les mutacions, però en lloc de mutar l'estat directament, les accions cometen mutacions. Les accions poden contenir operacions asíncrones.

  1. Utilitzant mapActions:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ count }}</p>
        <button @click="increment">Increment</button>
    </div>
</template>

<script>
import { mapState, mapActions } from 'vuex';

export default {
    computed: {
        ...mapState(['count'])
    },
    methods: {
        ...mapActions(['increment'])
    }
};
</script>

  2. Despatx Directe:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ count }}</p>
        <button @click="increment">Increment</button>
    </div>
</template>

<script>
export default {
    computed: {
        count() {
            return this.$store.state.count;
        }
    },
    methods: {
        increment() {
            this.$store.dispatch('increment');
        }
    }
};
</script>

Utilitzant Getters

Els getters s'utilitzen per calcular l'estat derivat basat en l'estat del magatzem. Són similars a les propietats computades per als magatzems.

  1. Utilitzant mapGetters:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ count }}</p>
    </div>
</template>

<script>
import { mapGetters } from 'vuex';

export default {
    computed: {
        ...mapGetters(['count'])
    }
};
</script>

  2. Accés Directe:

    // MyComponent.vue
<template>
    <div>
        <p>Count: {{ count }}</p>
    </div>
</template>

<script>
export default {
    computed: {
        count() {
            return this.$store.getters.count;
        }
    }
};
</script>

Conclusió

Utilitzar Vuex en els teus components Vue.js et permet gestionar l'estat de manera més estructurada i previsible. Aprofitant l'estat, les mutacions, les accions i els getters de Vuex, pots crear aplicacions que siguin més fàcils de depurar i mantenir. Recorda utilitzar els helpers proporcionats (mapState, mapMutations, mapActions i mapGetters) per mantenir els teus components nets i concisos.
