Introducció a les directives personalitzades

Les directives personalitzades a Vue.js et permeten estendre la funcionalitat dels elements HTML. Tot i que Vue proporciona algunes directives integrades com v-if, v-for i v-bind, les directives personalitzades et donen la flexibilitat de crear els teus propis comportaments reutilitzables.

Conceptes clau

  • Directives: Tokens especials en el marcat que indiquen a la biblioteca que faci alguna cosa a un element del DOM.
  • Funcions de hook: Hooks de cicle de vida per a directives, com ara bind, inserted, update, componentUpdated i unbind.

Creació d'una directiva personalitzada bàsica

Per crear una directiva personalitzada, utilitzes el mètode Vue.directive. Aquí tens un exemple senzill d'una directiva personalitzada que canvia el color de fons d'un element.

Vue.directive('highlight', {
  bind(el, binding) {
    el.style.backgroundColor = binding.value;
  }
});

Explicació

  • Vue.directive('highlight', {...}): Registra una nova directiva anomenada highlight.
  • bind(el, binding): El hook bind es crida només una vegada, quan la directiva es vincula per primera vegada a l'element. Aquí, el és l'element al qual està vinculada la directiva, i binding conté informació sobre la directiva, com ara el seu valor.

Ús

<div v-highlight="'yellow'">Això tindrà un fons groc.</div>

Directives personalitzades avançades

Les directives personalitzades poden ser més complexes i poden utilitzar múltiples hooks de cicle de vida per gestionar el DOM.

Exemple: Directiva de tooltip

Creem una directiva de tooltip que mostra un tooltip quan l'usuari passa el ratolí per sobre d'un element.

Vue.directive('tooltip', {
  bind(el, binding) {
    let tooltip = document.createElement('div');
    tooltip.className = 'tooltip';
    tooltip.innerText = binding.value;
    el.tooltip = tooltip;
    el.addEventListener('mouseenter', () => {
      document.body.appendChild(tooltip);
      let rect = el.getBoundingClientRect();
      tooltip.style.left = `${rect.left + window.scrollX}px`;
      tooltip.style.top = `${rect.top + window.scrollY - tooltip.offsetHeight}px`;
    });
    el.addEventListener('mouseleave', () => {
      document.body.removeChild(tooltip);
    });
  },
  unbind(el) {
    if (el.tooltip) {
      el.removeEventListener('mouseenter');
      el.removeEventListener('mouseleave');
    }
  }
});

Explicació

  • bind: Crea l'element tooltip i configura els listeners d'esdeveniments per a mouseenter i mouseleave.
  • unbind: Neteja els listeners d'esdeveniments quan la directiva es desvincula de l'element.

Ús

<button v-tooltip="'Aquest és un tooltip'">Passa el ratolí per sobre de mi</button>

Funcions de hook de directiva

Vue.js proporciona diverses funcions de hook per a directives personalitzades:

| Funció de hook | Descripció | |---------------------|-----------------------------------------------------------------------------| | bind | Es crida només una vegada, quan la directiva es vincula per primera vegada a l'element. | | inserted | Es crida quan l'element vinculat ha estat inserit al seu node pare. | | update | Es crida quan el VNode del component s'actualitza, però possiblement abans que els seus fills s'actualitzin. | | componentUpdated | Es crida després que el VNode del component i els seus fills s'hagin actualitzat. | | unbind | Es crida només una vegada, quan la directiva es desvincula de l'element. |

Resum

Les directives personalitzades a Vue.js són una manera poderosa d'estendre la funcionalitat de les teves aplicacions. Et permeten encapsular comportaments reutilitzables i aplicar-los a qualsevol element del DOM. Entenent i utilitzant els hooks de cicle de vida proporcionats per Vue, pots crear directives personalitzades complexes i interactives.

Punts clau

  • Les directives personalitzades milloren els elements HTML amb comportaments reutilitzables.
  • Utilitza Vue.directive per registrar una directiva personalitzada.
  • Els hooks de cicle de vida (bind, inserted, update, componentUpdated, unbind) gestionen el comportament de la directiva.
  • Les directives personalitzades poden anar des de canvis simples d'estil fins a interaccions complexes com els tooltips.

Dominant les directives personalitzades, pots millorar significativament la modularitat i la reutilització de les teves aplicacions Vue.js.
