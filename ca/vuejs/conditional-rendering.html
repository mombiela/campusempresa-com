Renderització condicional a Vue.js
Renderització de llistes ►

La renderització condicional és un concepte fonamental a Vue.js que et permet controlar dinàmicament la visibilitat i la renderització d'elements en el DOM basant-se en certes condicions. Aquest tema cobrirà des dels conceptes bàsics fins a tècniques avançades de renderització condicional a Vue.js.

Conceptes bàsics de la renderització condicional

Directiva v-if

La directiva v-if s'utilitza per renderitzar condicionalment un bloc. El bloc només es renderitzarà si l'expressió de la directiva retorna un valor veritable.

<template>
  <div v-if="isVisible">
    Aquest contingut és visible si isVisible és veritable.
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: true
    };
  }
};
</script>

Directiva v-else

La directiva v-else es pot utilitzar per proporcionar un bloc "else" per a v-if.

<template>
  <div>
    <div v-if="isVisible">
      Aquest contingut és visible si isVisible és veritable.
    </div>
    <div v-else>
      Aquest contingut és visible si isVisible és fals.
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: false
    };
  }
};
</script>

Directiva v-else-if

La directiva v-else-if es pot utilitzar per encadenar múltiples condicions.

<template>
  <div>
    <div v-if="status === 'success'">
      L'operació ha estat exitosa!
    </div>
    <div v-else-if="status === 'error'">
      Hi ha hagut un error.
    </div>
    <div v-else>
      L'estat és desconegut.
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      status: 'error'
    };
  }
};
</script>

Renderització condicional avançada

Directiva v-show

La directiva v-show alterna la visibilitat d'un element canviant la seva propietat CSS display. A diferència de v-if, l'element sempre es renderitza i roman en el DOM.

<template>
  <div v-show="isVisible">
    Aquest contingut sempre està en el DOM però només és visible si isVisible és veritable.
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: true
    };
  }
};
</script>

v-if vs v-show

| Característica | v-if | v-show | |----------------|-------------------------------|-------------------------------| | Renderització | Renderitza/elimina condicionalment | Sempre renderitza, alterna display | | Rendiment | Cost més alt per alternar | Cost més baix per alternar | | Cas d'ús | Elements rarament alternats | Elements freqüentment alternats |

Grups condicionals

Pots utilitzar un element <template> amb v-if, v-else-if i v-else per renderitzar condicionalment múltiples elements.

<template>
  <template v-if="status === 'success'">
    <p>L'operació ha estat exitosa!</p>
    <button @click="retry">Reintentar</button>
  </template>
  <template v-else-if="status === 'error'">
    <p>Hi ha hagut un error.</p>
    <button @click="retry">Reintentar</button>
  </template>
  <template v-else>
    <p>L'estat és desconegut.</p>
  </template>
</template>

<script>
export default {
  data() {
    return {
      status: 'unknown'
    };
  },
  methods: {
    retry() {
      // Lògica de reintentar aquí
    }
  }
};
</script>

Conclusió

La renderització condicional a Vue.js proporciona eines poderoses per controlar la visibilitat i la renderització d'elements basant-se en condicions dinàmiques. Entenent i utilitzant v-if, v-else, v-else-if i v-show, pots crear interfícies d'usuari més interactives i responsives. Recorda triar entre v-if i v-show basant-te en les implicacions de rendiment i la freqüència d'alternança de la visibilitat de l'element.
