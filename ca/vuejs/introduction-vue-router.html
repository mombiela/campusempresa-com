Introducció a Vue Router
Configuració de Vue Router ►

Vue Router és el router oficial per a Vue.js, permetent als desenvolupadors crear aplicacions d'una sola pàgina amb múltiples vistes i navegació. Aquesta secció t'introduirà als conceptes bàsics de Vue Router, la seva instal·lació i com utilitzar-lo per gestionar rutes en les teves aplicacions Vue.js.

Què és Vue Router?

Vue Router és una biblioteca que s'integra amb Vue.js per proporcionar capacitats de routing. Et permet:

  • Definir rutes i mapar-les a components.
  • Navegar entre diferents vistes.
  • Gestionar routing dinàmic i rutes niades.
  • Gestionar guàrdies de navegació i transicions de rutes.

Instal·lant Vue Router

Per començar amb Vue Router, necessites instal·lar-lo en el teu projecte Vue.js. Pots fer-ho utilitzant npm o yarn.

Utilitzant npm:

npm install vue-router

Utilitzant yarn:

yarn add vue-router

Configurant Vue Router

Un cop instal·lat, necessites configurar Vue Router en la teva aplicació Vue.js. Això implica crear una instància de router i definir les teves rutes.

Configuració Bàsica

  1. Crea un fitxer router.js:
// src/router.js
import Vue from 'vue';
import Router from 'vue-router';
import Home from './components/Home.vue';
import About from './components/About.vue';

Vue.use(Router);

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
];

const router = new Router({
  routes
});

export default router;
  1. Integra el router amb la teva instància de Vue:
// src/main.js
import Vue from 'vue';
import App from './App.vue';
import router from './router';

new Vue({
  render: h => h(App),
  router
}).$mount('#app');

Definint Rutes

Les rutes es defineixen com a objectes dins d'un array. Cada objecte de ruta típicament conté:

  • path: El camí URL.
  • component: El component a renderitzar quan el camí coincideix.

Exemple:

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/contact', component: Contact }
];

Navegant Entre Rutes

Vue Router proporciona diverses maneres de navegar entre rutes:

  • Utilitzant el component <router-link>:
<template>
  <div>
    <router-link to="/">Home</router-link>
    <router-link to="/about">About</router-link>
  </div>
</template>
  • Programàticament utilitzant this.$router.push:
methods: {
  goToAbout() {
    this.$router.push('/about');
  }
}

Routing Dinàmic

El routing dinàmic et permet definir rutes amb segments dinàmics que poden coincidir amb múltiples camins.

Exemple:

const routes = [
  { path: '/user/:id', component: User }
];

En el component User, pots accedir al segment dinàmic utilitzant this.$route.params.id.

Exemple de Component:

<template>
  <div>User ID: {{ $route.params.id }}</div>
</template>

<script>
export default {
  name: 'User'
};
</script>

Rutes Niades

Les rutes niades et permeten crear rutes dins de rutes, cosa que és útil per crear dissenys complexos.

Exemple:

const routes = [
  {
    path: '/user/:id',
    component: User,
    children: [
      {
        path: 'profile',
        component: UserProfile
      },
      {
        path: 'posts',
        component: UserPosts
      }
    ]
  }
];

Guàrdies de Navegació

Les guàrdies de navegació s'utilitzen per controlar l'accés a certes rutes. Es poden definir globalment, per ruta o en el component.

Exemple de Guàrdia Global:

router.beforeEach((to, from, next) => {
  if (to.path === '/protected' && !isAuthenticated()) {
    next('/login');
  } else {
    next();
  }
});

Resum

En aquesta secció, has après:

  • Què és Vue Router i la seva finalitat.
  • Com instal·lar i configurar Vue Router en un projecte Vue.js.
  • Com definir i navegar entre rutes.
  • Els conceptes bàsics de routing dinàmic i rutes niades.
  • Com utilitzar guàrdies de navegació per controlar l'accés a les rutes.

Vue Router és una eina poderosa que millora les capacitats de les aplicacions Vue.js proporcionant una navegació i routing sense problemes. Entendre aquests conceptes bàsics t'ajudarà a construir aplicacions d'una sola pàgina més complexes i fàcils d'usar.
