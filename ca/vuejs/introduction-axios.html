Què és Axios?

Axios és una biblioteca popular de JavaScript utilitzada per fer sol·licituds HTTP des del navegador i Node.js. Es basa en promeses, cosa que permet un codi asíncron més net i manejable. En el context de Vue.js, Axios s'utilitza comunament per obtenir dades d'APIs i gestionar sol·licituds HTTP en una aplicació Vue.

Característiques clau d'Axios

  • Basat en promeses: Simplifica el codi asíncron amb async/await o .then() i .catch().
  • Interceptors: Et permet interceptar sol·licituds o respostes abans que siguin gestionades.
  • Transformació automàtica de dades JSON: Transforma automàticament les dades JSON per a tu.
  • Gestió d'errors: Proporciona una manera senzilla de gestionar errors.
  • Cancel·lació de sol·licituds: Suporta la cancel·lació de sol·licituds utilitzant l'API CancelToken.

Instal·lant Axios

Per utilitzar Axios en un projecte Vue.js, primer has d'instal·lar-lo. Pots fer-ho utilitzant npm o yarn.

npm install axios
# o
yarn add axios

Utilitzant Axios en un component Vue

Sol·licitud GET bàsica

Aquí tens com pots fer una sol·licitud GET bàsica per obtenir dades d'una API i mostrar-les en un component Vue.

<template>
  <div>
    <h1>Dades Obtingudes</h1>
    <ul>
      <li v-for="item in items" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      items: []
    };
  },
  created() {
    axios.get('https://api.example.com/items')
      .then(response => {
        this.items = response.data;
      })
      .catch(error => {
        console.error('Hi ha hagut un error!', error);
      });
  }
};
</script>

Sol·licitud POST

Per enviar dades a una API, pots utilitzar una sol·licitud POST. Aquí tens un exemple de com fer-ho en un component Vue.

<template>
  <div>
    <h1>Afegeix un Nou Element</h1>
    <form @submit.prevent="addItem">
      <input v-model="newItem" placeholder="Introdueix el nom de l'element" />
      <button type="submit">Afegeix Element</button>
    </form>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      newItem: ''
    };
  },
  methods: {
    addItem() {
      axios.post('https://api.example.com/items', { name: this.newItem })
        .then(response => {
          console.log('Element afegit:', response.data);
          this.newItem = ''; // Neteja el camp d'entrada
        })
        .catch(error => {
          console.error('Hi ha hagut un error!', error);
        });
    }
  }
};
</script>

Gestió d'Errors

La gestió d'errors en Axios és senzilla. Pots utilitzar .catch() per gestionar errors en les teves sol·licituds.

axios.get('https://api.example.com/items')
  .then(response => {
    this.items = response.data;
  })
  .catch(error => {
    console.error('Hi ha hagut un error!', error);
  });

Utilitzant Interceptors d'Axios

Els interceptors et permeten executar el teu codi o modificar la sol·licitud/resposta abans que sigui gestionada per then o catch.

Interceptor de Sol·licitud

axios.interceptors.request.use(config => {
  // Fes alguna cosa abans que la sol·licitud sigui enviada
  console.log('Interceptor de Sol·licitud:', config);
  return config;
}, error => {
  // Fes alguna cosa amb l'error de la sol·licitud
  return Promise.reject(error);
});

Interceptor de Resposta

axios.interceptors.response.use(response => {
  // Fes alguna cosa amb les dades de la resposta
  console.log('Interceptor de Resposta:', response);
  return response;
}, error => {
  // Fes alguna cosa amb l'error de la resposta
  return Promise.reject(error);
});

Cancel·lant Sol·licituds

Pots cancel·lar una sol·licitud d'Axios utilitzant l'API CancelToken.

const CancelToken = axios.CancelToken;
let cancel;

axios.get('https://api.example.com/items', {
  cancelToken: new CancelToken(function executor(c) {
    cancel = c;
  })
}).catch(thrown => {
  if (axios.isCancel(thrown)) {
    console.log('Sol·licitud cancel·lada', thrown.message);
  } else {
    // gestiona l'error
  }
});

// Cancel·la la sol·licitud
cancel('Operació cancel·lada per l'usuari.');

Conclusió

Axios és una biblioteca potent i flexible per gestionar sol·licituds HTTP en aplicacions Vue.js. Simplifica el procés de fer sol·licituds asíncrones, gestionar errors i manejar transformacions de sol·licitud/resposta. Entenent i utilitzant Axios, pots interactuar eficientment amb APIs i millorar la funcionalitat de les teves aplicacions Vue.js.
