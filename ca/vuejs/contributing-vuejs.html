Contribuir a Vue.js

Contribuir a un projecte de codi obert com Vue.js pot ser una experiència gratificant. Et permet millorar les teves habilitats, col·laborar amb altres desenvolupadors i retornar a la comunitat. Aquesta secció et guiarà a través del procés de contribuir a Vue.js, des de configurar el teu entorn de desenvolupament fins a enviar una sol·licitud d'extracció.

Configurar el Teu Entorn de Desenvolupament

Abans de poder contribuir a Vue.js, necessites configurar el teu entorn de desenvolupament.

  • Fes un Fork del Repositori: Ves al repositori de Vue.js a GitHub i fes un fork al teu propi compte.
  • Clona el Repositori: Clona el repositori forkejat a la teva màquina local. 
    git clone https://github.com/el-teu-usuari/vue.git
  • Instal·la les Dependències: Navega fins al repositori clonat i instal·la les dependències necessàries. 
    cd vue
npm install

Entendre el Codi Font

Familiaritza't amb el codi font de Vue.js per entendre la seva estructura i components.

  • Estructura de Directori: Vue.js té una estructura de directori ben organitzada. Aquí tens alguns directoris clau:

    • src: Conté el codi font de Vue.js.
    • test: Conté els casos de prova.
    • scripts: Conté els scripts de construcció i utilitat.

  • Fitxers Clau:

    • src/core: Funcionalitats bàsiques de Vue.js.
    • src/compiler: Codi relacionat amb la compilació de plantilles.
    • src/platforms: Codi específic de la plataforma (per exemple, web, weex).

Trobar Problemes per Treballar

Pots trobar problemes per treballar navegant pel repositori de Vue.js a GitHub.

  • Good First Issues: Busca problemes etiquetats com "good first issue" si ets nou en contribuir.
  • Correcció d'Errors: Identifica errors que puguis corregir.
  • Sol·licituds de Funcionalitats: Busca sol·licituds de funcionalitats que puguis implementar.

Fer Canvis

Un cop hagis identificat un problema per treballar, segueix aquests passos per fer els teus canvis.

  • Crea una Nova Branca: Crea una nova branca per als teus canvis. 
    git checkout -b feature/el-teu-nom-de-funcionalitat
  • Fes els Teus Canvis: Edita el codi per corregir el problema o implementar la funcionalitat.
  • Executa les Proves: Assegura't que totes les proves passin després de fer els teus canvis. 
    npm test

Enviar una Sol·licitud d'Extracció

Després de fer i provar els teus canvis, pots enviar una sol·licitud d'extracció.

  • Fes un Commit dels Teus Canvis: Fes un commit dels teus canvis amb un missatge descriptiu. 
    git add .
git commit -m "Fix issue #123: Descripció de la correcció"
  • Puja els Teus Canvis: Puja els teus canvis al teu repositori forkejat. 
    git push origin feature/el-teu-nom-de-funcionalitat
  • Crea una Sol·licitud d'Extracció: Ves al repositori de Vue.js i crea una sol·licitud d'extracció des del teu repositori forkejat.

Exemple de Codi: Corregir un Error Simple

Vegem un exemple senzill de com corregir un error a Vue.js.

Pas 1: Identificar l'Error

Suposem que hi ha un problema on un mètode específic a src/core/util.js no retorna el valor esperat.

Pas 2: Crear una Nova Branca

git checkout -b bugfix/fix-util-method

Pas 3: Fer els Canvis

Edita el fitxer src/core/util.js per corregir l'error.

// Abans
function faultyMethod() {
  return false;
}

// Després
function faultyMethod() {
  return true;
}

Pas 4: Executar les Proves

Assegura't que totes les proves passin.

npm test

Pas 5: Fer un Commit i Pujar els Canvis

git add src/core/util.js
git commit -m "Fix issue #456: Correct return value in faultyMethod"
git push origin bugfix/fix-util-method

Pas 6: Crear una Sol·licitud d'Extracció

Ves al repositori de Vue.js a GitHub i crea una sol·licitud d'extracció des de la teva branca.

Conclusió

Contribuir a Vue.js implica diversos passos, des de configurar el teu entorn de desenvolupament fins a enviar una sol·licitud d'extracció. Seguint aquests passos, pots contribuir efectivament al projecte i convertir-te en un membre valuós de la comunitat de Vue.js. Recorda començar amb problemes més petits i gradualment assumir tasques més complexes a mesura que et familiaritzis amb el codi font. Feliç programació!
