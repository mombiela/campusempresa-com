Introducció a l'Enllaç de Dades

L'enllaç de dades és un concepte central a Vue.js que et permet sincronitzar dades entre el model i la vista. Això significa que quan les teves dades canvien, la vista s'actualitza automàticament, i viceversa.

Conceptes Clau

  • Dades Reactives: Vue.js utilitza un sistema de dades reactives que automàticament fa un seguiment de les dependències i actualitza el DOM quan les dades canvien.
  • Enllaç de Dades Unidireccional: Les dades flueixen del model a la vista.
  • Enllaç de Dades Bidireccional: Les dades flueixen en ambdues direccions, del model a la vista i de la vista de tornada al model.

Enllaç de Dades Unidireccional

L'enllaç de dades unidireccional a Vue.js s'aconsegueix utilitzant la directiva v-bind o l'abreviatura :.

Exemple

<div id="app">
  <p>{{ message }}</p>
  <p v-bind:title="message">Passa el ratolí per sobre per veure el missatge</p>
</div>

<script>
  new Vue({
    el: '#app',
    data: {
      message: 'Hola, Vue.js!'
    }
  });
</script>

En aquest exemple:

  • La sintaxi {{ message }} es diu sintaxi "Mustache" i s'utilitza per a la interpolació de text.
  • La directiva v-bind:title="message" enllaça l'atribut title de l'element <p> a la propietat de dades message.

Enllaç de Dades Bidireccional

L'enllaç de dades bidireccional s'utilitza principalment en els camps de formulari i s'aconsegueix utilitzant la directiva v-model.

Exemple

<div id="app">
  <input v-model="message" placeholder="Edita'm">
  <p>El missatge és: {{ message }}</p>
</div>

<script>
  new Vue({
    el: '#app',
    data: {
      message: 'Hola, Vue.js!'
    }
  });
</script>

En aquest exemple:

  • La directiva v-model enllaça l'element d'entrada a la propietat de dades message.
  • Qualsevol canvi en el camp d'entrada actualitzarà la propietat message, i qualsevol canvi a la propietat message actualitzarà el camp d'entrada.

Renderització Condicional

La renderització condicional s'utilitza per renderitzar elements basats en certes condicions utilitzant les directives v-if, v-else-if i v-else.

Exemple

<div id="app">
  <button v-on:click="toggle">Alterna Missatge</button>
  <p v-if="isVisible">El missatge és visible</p>
  <p v-else>El missatge està ocult</p>
</div>

<script>
  new Vue({
    el: '#app',
    data: {
      isVisible: true
    },
    methods: {
      toggle() {
        this.isVisible = !this.isVisible;
      }
    }
  });
</script>

En aquest exemple:

  • La directiva v-if renderitza condicionalment el primer element <p> si isVisible és true.
  • La directiva v-else renderitza el segon element <p> si isVisible és false.

Renderització de Llistes

La renderització de llistes s'utilitza per renderitzar una llista d'elements utilitzant la directiva v-for.

Exemple

<div id="app">
  <ul>
    <li v-for="item in items" :key="item.id">{{ item.text }}</li>
  </ul>
</div>

<script>
  new Vue({
    el: '#app',
    data: {
      items: [
        { id: 1, text: 'Aprendre JavaScript' },
        { id: 2, text: 'Aprendre Vue.js' },
        { id: 3, text: 'Construir alguna cosa impressionant' }
      ]
    }
  });
</script>

En aquest exemple:

  • La directiva v-for itera sobre l'array items i renderitza un element <li> per a cada element.
  • La directiva :key s'utilitza per donar a cada element un identificador únic, cosa que ajuda Vue.js a optimitzar la renderització.

Resum

L'enllaç de dades a Vue.js és una característica poderosa que et permet crear aplicacions web dinàmiques i interactives amb facilitat. Entenent i utilitzant l'enllaç de dades unidireccional i bidireccional, la renderització condicional i la renderització de llistes, pots gestionar i mostrar eficientment les dades de la teva aplicació.

Punts Clau

  • Enllaç de Dades Unidireccional: Utilitza v-bind o : per enllaçar dades del model a la vista.
  • Enllaç de Dades Bidireccional: Utilitza v-model per enllaçar dades en ambdues direccions entre el model i la vista.
  • Renderització Condicional: Utilitza v-if, v-else-if i v-else per renderitzar elements basats en condicions.
  • Renderització de Llistes: Utilitza v-for per renderitzar llistes d'elements dinàmicament.

Dominant aquests conceptes, estaràs ben preparat per construir aplicacions robustes i responsives amb Vue.js.
