Introducción a los Arrays

Los arrays son una estructura de datos fundamental en la programación que te permite almacenar múltiples valores en una sola variable. En ALGOL, los arrays se utilizan para almacenar colecciones de datos del mismo tipo, que pueden ser accedidos usando índices.

Conceptos Clave

  • Declaración de Arrays: Cómo definir un array en ALGOL.
  • Inicialización de Arrays: Cómo asignar valores a un array.
  • Acceso a Arrays: Cómo recuperar y modificar valores en un array.
  • Arrays Multidimensionales: Arrays con más de una dimensión.
  • Operaciones con Arrays: Operaciones comunes realizadas en arrays.

Declaración de Arrays

En ALGOL, los arrays se declaran especificando el tipo de los elementos y el rango de índices.

Ejemplo

integer array A[1:10];

Esto declara un array A de enteros con índices que van del 1 al 10.

Explicación

  • integer: Especifica que el array almacenará valores enteros.
  • array A[1:10]: Declara un array llamado A con índices del 1 al 10.

Inicialización de Arrays

Los arrays pueden ser inicializados en el momento de la declaración o más adelante en el código.

Ejemplo

integer array B[1:5] := (1, 2, 3, 4, 5);

Esto inicializa el array B con los valores 1, 2, 3, 4 y 5.

Explicación

  • := (1, 2, 3, 4, 5): Asigna los valores del 1 al 5 al array B.

Acceso a Arrays

Puedes acceder y modificar los elementos de un array usando sus índices.

Ejemplo

integer array C[1:3];
C[1] := 10;
C[2] := 20;
C[3] := 30;
print(C[2]);  ! Imprime 20

Explicación

  • C[1] := 10;: Asigna el valor 10 al primer elemento del array C.
  • print(C[2]);: Imprime el valor del segundo elemento del array C, que es 20.

Arrays Multidimensionales

ALGOL soporta arrays multidimensionales, que son arrays de arrays.

Ejemplo

integer array D[1:3, 1:3];
D[1, 1] := 1;
D[1, 2] := 2;
D[1, 3] := 3;
D[2, 1] := 4;
D[2, 2] := 5;
D[2, 3] := 6;
D[3, 1] := 7;
D[3, 2] := 8;
D[3, 3] := 9;
print(D[2, 3]);  ! Imprime 6

Explicación

  • integer array D[1:3, 1:3];: Declara un array 3x3 D de enteros.
  • D[2, 3] := 6;: Asigna el valor 6 al elemento en la segunda fila y tercera columna del array D.
  • print(D[2, 3]);: Imprime el valor del elemento en la segunda fila y tercera columna, que es 6.

Operaciones con Arrays

Las operaciones comunes en arrays incluyen iterar a través de los elementos, buscar valores y realizar cálculos.

Ejemplo: Iterando a través de un Array

integer array E[1:5] := (10, 20, 30, 40, 50);
integer i;

for i := 1 step 1 until 5 do
    print(E[i]);

Explicación

  • for i := 1 step 1 until 5 do: Un bucle que itera del 1 al 5.
  • print(E[i]);: Imprime cada elemento del array E.

Resumen

Los arrays en ALGOL son una herramienta poderosa para gestionar colecciones de datos. Entender cómo declarar, inicializar, acceder y manipular arrays es esencial para una programación efectiva en ALGOL. Este tema cubrió:

  • Declaración e inicialización básica de arrays.
  • Acceso y modificación de elementos de arrays.
  • Trabajo con arrays multidimensionales.
  • Realización de operaciones comunes en arrays.

Al dominar estos conceptos, estarás bien equipado para manejar estructuras de datos y algoritmos más complejos en ALGOL.
