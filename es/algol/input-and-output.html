Introducción

En esta sección, exploraremos cómo manejar operaciones de entrada y salida (I/O) en ALGOL. Entender I/O es crucial para interactuar con usuarios y otros sistemas, haciendo que tus programas sean más dinámicos y útiles.

Conceptos Básicos

Entrada

  • Lectura de Datos: En ALGOL, las operaciones de entrada se utilizan para leer datos del usuario u otras fuentes de entrada.
  • Sintaxis: La instrucción read se utiliza para capturar la entrada.

Salida

  • Mostrar Datos: Las operaciones de salida se utilizan para mostrar datos al usuario u otros destinos de salida.
  • Sintaxis: La instrucción print se utiliza para mostrar la salida.

Operaciones de Entrada

Lectura de Valores Individuales

Para leer un solo valor del usuario, puedes usar la instrucción read.

begin
    integer x;
    read(x);
    print("You entered: ", x);
end
  • Explicación:
    • integer x; declara una variable entera x.
    • read(x); lee un valor entero del usuario y lo almacena en x.
    • print("You entered: ", x); muestra el valor ingresado.

Lectura de Múltiples Valores

Puedes leer múltiples valores en una sola instrucción read.

begin
    integer a, b;
    read(a, b);
    print("You entered: ", a, " and ", b);
end
  • Explicación:
    • integer a, b; declara dos variables enteras a y b.
    • read(a, b); lee dos valores enteros del usuario y los almacena en a y b.
    • print("You entered: ", a, " and ", b); muestra los valores ingresados.

Operaciones de Salida

Imprimir Valores Individuales

Para imprimir un solo valor, usa la instrucción print.

begin
    integer y;
    y := 10;
    print("The value of y is: ", y);
end
  • Explicación:
    • integer y; declara una variable entera y.
    • y := 10; asigna el valor 10 a y.
    • print("The value of y is: ", y); muestra el valor de y.

Imprimir Múltiples Valores

Puedes imprimir múltiples valores en una sola instrucción print.

begin
    integer m, n;
    m := 5;
    n := 15;
    print("Values are: ", m, " and ", n);
end
  • Explicación:
    • integer m, n; declara dos variables enteras m y n.
    • m := 5; asigna el valor 5 a m.
    • n := 15; asigna el valor 15 a n.
    • print("Values are: ", m, " and ", n); muestra los valores de m y n.

Operaciones Avanzadas de I/O

Salida Formateada

ALGOL permite salida formateada usando opciones de formato específicas.

begin
    real r;
    r := 3.14159;
    print("The value of pi is approximately: ", r:8:4);
end
  • Explicación:
    • real r; declara una variable real (de punto flotante) r.
    • r := 3.14159; asigna el valor 3.14159 a r.
    • print("The value of pi is approximately: ", r:8:4); muestra el valor de r con un ancho total de 8 caracteres y 4 decimales.

I/O de Archivos

ALGOL soporta la lectura y escritura en archivos, lo cual es útil para manejar conjuntos de datos más grandes.

begin
    integer fileVar;
    open(fileVar, "input.txt", "read");
    while not eof(fileVar) do
    begin
        integer num;
        read(fileVar, num);
        print("Read number: ", num);
    end;
    close(fileVar);
end
  • Explicación:
    • integer fileVar; declara una variable de archivo fileVar.
    • open(fileVar, "input.txt", "read"); abre el archivo input.txt para lectura.
    • while not eof(fileVar) do itera hasta que se alcance el final del archivo.
    • integer num; declara una variable entera num.
    • read(fileVar, num); lee un entero del archivo y lo almacena en num.
    • print("Read number: ", num); muestra el número leído.
    • close(fileVar); cierra el archivo.

Conclusión

En esta sección, cubrimos los aspectos básicos y algunos avanzados de las operaciones de entrada y salida en ALGOL. Aprendimos cómo leer e imprimir valores individuales y múltiples, formatear la salida y manejar I/O de archivos. Dominar estas operaciones te permitirá crear programas más interactivos y amigables para el usuario.
