Introducción a las Funciones Recursivas

Las funciones recursivas son funciones que se llaman a sí mismas para resolver un problema. Son una herramienta poderosa en la programación, que permite soluciones elegantes a problemas complejos al descomponerlos en subproblemas más simples.

Conceptos Clave

  • Caso Base: La condición bajo la cual la recursión termina.
  • Caso Recursivo: La parte de la función donde la función se llama a sí misma.
  • Desbordamiento de Pila: Un riesgo potencial cuando la recursión no se maneja adecuadamente, lo que lleva a un uso excesivo de memoria.

Estructura Básica de las Funciones Recursivas en ALGOL

En ALGOL, las funciones recursivas se definen de manera similar a otras funciones, pero incluyen una llamada a sí mismas dentro de su cuerpo.

Ejemplo: Función Factorial

El factorial de un entero no negativo n es el producto de todos los enteros positivos menores o iguales a n. Se define como:

  • factorial(0) = 1 (Caso Base)
  • factorial(n) = n * factorial(n-1) (Caso Recursivo)

Aquí se muestra cómo se puede implementar la función factorial en ALGOL:

begin
    integer procedure factorial(n);
    value n;
    integer n;
    begin
        if n = 0 then
            factorial := 1
        else
            factorial := n * factorial(n - 1)
    end;

    integer n;
    n := 5;
    print(factorial(n))
end

Explicación

  • Caso Base: if n = 0 then factorial := 1
  • Caso Recursivo: factorial := n * factorial(n - 1)

Funciones Recursivas Avanzadas

A medida que te sientas más cómodo con la recursión, puedes abordar problemas más complejos. Aquí hay algunos ejemplos:

Ejemplo: Secuencia de Fibonacci

La secuencia de Fibonacci es una serie de números donde cada número es la suma de los dos anteriores, generalmente comenzando con 0 y 1. Se define como:

  • fibonacci(0) = 0
  • fibonacci(1) = 1
  • fibonacci(n) = fibonacci(n-1) + fibonacci(n-2)

Aquí está la implementación en ALGOL:

begin
    integer procedure fibonacci(n);
    value n;
    integer n;
    begin
        if n = 0 then
            fibonacci := 0
        else if n = 1 then
            fibonacci := 1
        else
            fibonacci := fibonacci(n - 1) + fibonacci(n - 2)
    end;

    integer n;
    n := 10;
    print(fibonacci(n))
end

Explicación

  • Casos Base: if n = 0 then fibonacci := 0 y else if n = 1 then fibonacci := 1
  • Caso Recursivo: fibonacci := fibonacci(n - 1) + fibonacci(n - 2)

Recursión de Cola

La recursión de cola es una forma especial de recursión donde la llamada recursiva es la última operación en la función. Esto puede ser optimizado por el compilador para evitar el desbordamiento de pila.

Ejemplo: Factorial Recursivo de Cola

Para convertir la función factorial a una versión recursiva de cola, introducimos un acumulador para mantener los resultados intermedios.

begin
    integer procedure factorial(n, acc);
    value n, acc;
    integer n, acc;
    begin
        if n = 0 then
            factorial := acc
        else
            factorial := factorial(n - 1, n * acc)
    end;

    integer n;
    n := 5;
    print(factorial(n, 1))
end

Explicación

  • Caso Base: if n = 0 then factorial := acc
  • Caso Recursivo: factorial := factorial(n - 1, n * acc)

Conclusión

Las funciones recursivas son un concepto fundamental en la programación, que permite soluciones elegantes y eficientes a problemas complejos. Al comprender los casos base y recursivos, y estar consciente de posibles problemas como el desbordamiento de pila, puedes usar efectivamente la recursión en tus programas ALGOL. La recursión de cola es una técnica avanzada que puede ayudar a optimizar las funciones recursivas, haciéndolas más eficientes y menos propensas a errores.
