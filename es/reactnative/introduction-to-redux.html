Redux es un contenedor de estado predecible para aplicaciones JavaScript, a menudo utilizado con bibliotecas como React y React Native para gestionar el estado de la aplicación. Este tema te introducirá a Redux, sus conceptos básicos y cómo integrarlo en una aplicación React Native.

¿Qué es Redux?

Redux es una biblioteca de gestión de estado que te ayuda a manejar el estado de tu aplicación de una manera predecible. Es particularmente útil para aplicaciones con interacciones de estado complejas y flujos de datos.

Conceptos Clave de Redux

  • Store: La única fuente de verdad que contiene el estado de toda tu aplicación.
  • Actions: Objetos JavaScript simples que describen lo que ocurrió en la aplicación.
  • Reducers: Funciones que toman el estado actual y una acción, y devuelven un nuevo estado.
  • Dispatch: Un método utilizado para enviar acciones al store.
  • Selectors: Funciones que extraen piezas específicas del estado del store.

Configurando Redux en un Proyecto React Native

Para comenzar con Redux en un proyecto React Native, necesitas instalar los paquetes necesarios:

npm install redux react-redux

Creando el Store

El store es la pieza central de Redux. Contiene el estado de tu aplicación.

import { createStore } from 'redux';
import rootReducer from './reducers';

const store = createStore(rootReducer);

Definiendo Acciones

Las acciones son objetos JavaScript simples que describen lo que ocurrió en tu aplicación.

// actions.js
export const INCREMENT = 'INCREMENT';
export const DECREMENT = 'DECREMENT';

export const increment = () => ({
  type: INCREMENT,
});

export const decrement = () => ({
  type: DECREMENT,
});

Creando Reducers

Los reducers especifican cómo cambia el estado de la aplicación en respuesta a las acciones.

// counterReducer.js
import { INCREMENT, DECREMENT } from './actions';

const initialState = {
  count: 0,
};

const counterReducer = (state = initialState, action) => {
  switch (action.type) {
    case INCREMENT:
      return { ...state, count: state.count + 1 };
    case DECREMENT:
      return { ...state, count: state.count - 1 };
    default:
      return state;
  }
};

export default counterReducer;

Combinando Reducers

Si tienes múltiples reducers, puedes combinarlos usando combineReducers.

// reducers/index.js
import { combineReducers } from 'redux';
import counterReducer from './counterReducer';

const rootReducer = combineReducers({
  counter: counterReducer,
});

export default rootReducer;

Proveyendo el Store a Tu Aplicación

Usa el componente Provider de react-redux para hacer que el store esté disponible para toda tu aplicación.

// App.js
import React from 'react';
import { Provider } from 'react-redux';
import { createStore } from 'redux';
import rootReducer from './reducers';
import Counter from './Counter';

const store = createStore(rootReducer);

const App = () => (
  <Provider store={store}>
    <Counter />
  </Provider>
);

export default App;

Conectando Componentes al Store

Usa la función connect de react-redux para conectar tus componentes al store de Redux.

// Counter.js
import React from 'react';
import { connect } from 'react-redux';
import { increment, decrement } from './actions';

const Counter = ({ count, increment, decrement }) => (
  <View>
    <Text>{count}</Text>
    <Button onPress={increment} title="Incrementar" />
    <Button onPress={decrement} title="Decrementar" />
  </View>
);

const mapStateToProps = (state) => ({
  count: state.counter.count,
});

const mapDispatchToProps = {
  increment,
  decrement,
};

export default connect(mapStateToProps, mapDispatchToProps)(Counter);

Resumen

En este tema, cubrimos los conceptos básicos de Redux y cómo integrarlo en una aplicación React Native. Discutimos los conceptos clave de Redux, incluyendo el store, las acciones, los reducers y cómo conectar componentes al store. Siguiendo estos pasos, puedes gestionar el estado de tus aplicaciones React Native de una manera predecible y escalable.
