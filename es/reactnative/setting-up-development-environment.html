Configurar el entorno de desarrollo es el primer paso crucial en tu camino para convertirte en un experto en React Native. Esta sección te guiará a través del proceso de configuración de tu entorno en diferentes sistemas operativos, asegurándote de tener todas las herramientas necesarias para comenzar a construir aplicaciones React Native.

Prerrequisitos

Antes de sumergirte en la configuración, asegúrate de tener los siguientes prerrequisitos:

  • Conocimientos básicos de JavaScript.
  • Un editor de código (se recomienda Visual Studio Code).
  • Node.js instalado en tu máquina.

Instalando Node.js y npm

Node.js es un entorno de ejecución de JavaScript que te permite ejecutar JavaScript en el lado del servidor. npm (Node Package Manager) viene con Node.js y te ayuda a gestionar dependencias.

  1. Descargar e Instalar Node.js:

    • Ve al sitio web de Node.js.
    • Descarga la versión LTS para tu sistema operativo.
    • Sigue las instrucciones de instalación.

  2. Verificar la Instalación:

    node -v
npm -v

Configuración de React Native CLI

React Native CLI es una interfaz de línea de comandos que te ayuda a crear, construir y ejecutar proyectos de React Native.

  1. Instalar React Native CLI:

    npm install -g react-native-cli

  2. Verificar la Instalación:

    react-native --version

Configuración del Entorno de Desarrollo en macOS

Instalando Xcode

Xcode es necesario para el desarrollo de iOS.

  1. Descargar Xcode:

    • Abre la App Store en tu Mac.
    • Busca Xcode e instálalo.

  2. Instalar Herramientas de Línea de Comandos:

    xcode-select --install

Instalando Android Studio

Android Studio es necesario para el desarrollo de Android.

  1. Descargar Android Studio:

    • Ve al sitio web de Android Studio.
    • Descarga e instala Android Studio.

  2. Configurar Android SDK:

    • Abre Android Studio.
    • Ve a Preferences > Appearance & Behavior > System Settings > Android SDK.
    • Selecciona la pestaña SDK Platforms y asegúrate de que Android 10.0 (Q) esté marcado.
    • Selecciona la pestaña SDK Tools y asegúrate de que Android SDK Build-Tools, Android Emulator y Android SDK Platform-Tools estén marcados.

Configuración de Variables de Entorno

  1. Abrir Terminal y Editar Perfil:

    nano ~/.bash_profile

  2. Agregar las Siguientes Líneas:

    export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools

  3. Guardar y Cerrar:

    • Presiona CTRL + X, luego Y, y Enter.

  4. Aplicar Cambios:

    source ~/.bash_profile

Configuración del Entorno de Desarrollo en Windows

Instalando Android Studio

  1. Descargar Android Studio:

    • Ve al sitio web de Android Studio.
    • Descarga e instala Android Studio.

  2. Configurar Android SDK:

    • Abre Android Studio.
    • Ve a File > Settings > Appearance & Behavior > System Settings > Android SDK.
    • Selecciona la pestaña SDK Platforms y asegúrate de que Android 10.0 (Q) esté marcado.
    • Selecciona la pestaña SDK Tools y asegúrate de que Android SDK Build-Tools, Android Emulator y Android SDK Platform-Tools estén marcados.

Configuración de Variables de Entorno

  1. Abrir Variables de Entorno:

    • Haz clic derecho en Este PC > Propiedades > Configuración avanzada del sistema.
    • Haz clic en Variables de entorno.

  2. Agregar Nuevas Variables del Sistema:

    • Haz clic en Nuevo bajo Variables del sistema.
    • Agrega ANDROID_HOME con la ruta a tu Android SDK (por ejemplo, C:\Users\<Tu-NombreDeUsuario>\AppData\Local\Android\Sdk).
    • Agrega entradas PATH para platform-tools y tools.

Creando Tu Primer Proyecto de React Native

  1. Crear un Nuevo Proyecto:

    npx react-native init MyFirstProject

  2. Navegar al Directorio del Proyecto:

    cd MyFirstProject

  3. Ejecutar el Proyecto:

    • Para iOS: 
      npx react-native run-ios
    • Para Android: 
      npx react-native run-android

Conclusión

Configurar el entorno de desarrollo para React Native puede parecer desalentador al principio, pero siguiendo estos pasos te asegurarás de tener todo lo que necesitas para comenzar a construir tus aplicaciones. Ya sea que estés en macOS o Windows, tener Node.js, React Native CLI y las herramientas necesarias para el desarrollo de iOS y Android te preparará para el éxito. ¡Feliz codificación!
