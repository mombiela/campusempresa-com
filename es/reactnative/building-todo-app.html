Introducción

En este curso, construiremos una Aplicación de Tareas usando React Native. Este proyecto te ayudará a entender los fundamentos de React Native y cómo aplicarlos para crear una aplicación móvil funcional. Comenzaremos desde lo básico y gradualmente avanzaremos a temas más avanzados.

Configuración del Entorno

Antes de comenzar a codificar, necesitamos configurar nuestro entorno de desarrollo.

  • Instalar Node.js: Asegúrate de tener Node.js instalado. Puedes descargarlo desde nodejs.org.
  • Instalar Expo CLI: Expo es un marco y plataforma para aplicaciones universales de React. Instálalo usando npm: 
    npm install -g expo-cli
  • Inicializar un Nuevo Proyecto: Crea un nuevo proyecto de React Native usando Expo CLI: 
    expo init ToDoApp
cd ToDoApp
expo start

Estructura Básica de una Aplicación React Native

Entender la estructura básica de una aplicación React Native es crucial.

  • App.js: El punto de entrada principal de la aplicación.
  • Componentes: Piezas reutilizables de la interfaz de usuario.
  • Gestión de Estado: Gestión del estado de la aplicación.

Ejemplo: Estructura Básica de la Aplicación

import React from 'react';
import { StyleSheet, Text, View } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text>¡Bienvenido a la Aplicación de Tareas!</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center',
  },
});

Creando el Componente de Lista de Tareas

Crearemos un componente de Lista de Tareas que mostrará nuestras tareas.

  • Gestión de Estado: Usa el hook useState de React para gestionar la lista de tareas.
  • Renderizado de Lista: Usa FlatList para renderizar la lista de tareas.

Ejemplo: Componente de Lista de Tareas

import React, { useState } from 'react';
import { View, Text, FlatList, StyleSheet } from 'react-native';

const ToDoList = () => {
  const [tasks, setTasks] = useState([
    { key: '1', task: 'Comprar comestibles' },
    { key: '2', task: 'Pasear al perro' },
    { key: '3', task: 'Leer un libro' },
  ]);

  return (
    <View style={styles.container}>
      <FlatList
        data={tasks}
        renderItem={({ item }) => <Text style={styles.item}>{item.task}</Text>}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoList;

Añadiendo Nuevas Tareas

Necesitamos una forma de añadir nuevas tareas a nuestra lista.

  • TextInput: Usa TextInput para capturar la entrada del usuario.
  • Button: Usa Button para añadir la tarea a la lista.

Ejemplo: Añadiendo Nuevas Tareas

import React, { useState } from 'react';
import { View, TextInput, Button, FlatList, Text, StyleSheet } from 'react-native';

const ToDoApp = () => {
  const [tasks, setTasks] = useState([]);
  const [text, setText] = useState('');

  const addTask = () => {
    setTasks([...tasks, { key: Math.random().toString(), task: text }]);
    setText('');
  };

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Añadir nueva tarea"
        value={text}
        onChangeText={setText}
      />
      <Button title="Añadir Tarea" onPress={addTask} />
      <FlatList
        data={tasks}
        renderItem={({ item }) => <Text style={styles.item}>{item.task}</Text>}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingLeft: 8,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoApp;

Eliminando Tareas

Para hacer nuestra aplicación más funcional, necesitamos añadir la capacidad de eliminar tareas.

  • TouchableOpacity: Usa TouchableOpacity para hacer cada elemento de tarea clicable.
  • Función de Filtrado: Usa la función de filtro para eliminar la tarea de la lista.

Ejemplo: Eliminando Tareas

import React, { useState } from 'react';
import { View, TextInput, Button, FlatList, Text, TouchableOpacity, StyleSheet } from 'react-native';

const ToDoApp = () => {
  const [tasks, setTasks] = useState([]);
  const [text, setText] = useState('');

  const addTask = () => {
    setTasks([...tasks, { key: Math.random().toString(), task: text }]);
    setText('');
  };

  const deleteTask = (key) => {
    setTasks(tasks.filter(task => task.key !== key));
  };

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Añadir nueva tarea"
        value={text}
        onChangeText={setText}
      />
      <Button title="Añadir Tarea" onPress={addTask} />
      <FlatList
        data={tasks}
        renderItem={({ item }) => (
          <TouchableOpacity onPress={() => deleteTask(item.key)}>
            <Text style={styles.item}>{item.task}</Text>
          </TouchableOpacity>
        )}
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    paddingTop: 22,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingLeft: 8,
  },
  item: {
    padding: 10,
    fontSize: 18,
    height: 44,
  },
});

export default ToDoApp;

Estilizando la Aplicación

Para hacer nuestra aplicación visualmente atractiva, añadiremos algunos estilos.

  • StyleSheet: Usa StyleSheet para crear estilos.
  • Flexbox: Usa Flexbox para el diseño.

Ejemplo: Estilizando la Aplicación

import { View, TextInput, Button, FlatList, Text, TouchableOpacity, StyleSheet } from 'react-native';

const ToDoApp = () => {
  const [tasks, setTasks] = useState([]);
  const [text, setText] = useState('');

  const addTask = () => {

    setText('');
  };

  const deleteTask = (key) => {
    setTasks(tasks.filter(task => task.key !== key));

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Añadir nueva tarea"

        onChangeText={setText}
