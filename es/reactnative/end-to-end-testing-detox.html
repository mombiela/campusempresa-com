Pruebas de extremo a extremo con Detox
Optimización del Rendimiento ►

Las pruebas de extremo a extremo (E2E) son cruciales para asegurar que tu aplicación React Native funcione como se espera desde la perspectiva del usuario. Detox es un marco popular para pruebas E2E en React Native, proporcionando una forma robusta y confiable de automatizar tus pruebas.

Introducción a Detox

  • ¿Qué es Detox?

    • Detox es una biblioteca de pruebas de extremo a extremo para aplicaciones móviles.
    • Te permite escribir pruebas que simulan interacciones de usuario.
    • Detox funciona con plataformas iOS y Android.

  • ¿Por qué usar Detox?

    • Asegura que tu aplicación funcione correctamente en diferentes dispositivos y versiones de SO.
    • Automatiza tareas repetitivas de prueba.
    • Ayuda a detectar errores temprano en el ciclo de desarrollo.

Configuración de Detox

  • Requisitos previos:

    • Node.js y npm/yarn instalados.
    • React Native CLI instalado.
    • Xcode (para iOS) y Android Studio (para Android).

  • Instalación:

    npm install -g detox-cli
npm install detox --save-dev

  • Configuración de Detox:

    • Añade la configuración de Detox a tu package.json: 
      {
  "detox": {
    "configurations": {
      "ios.sim.debug": {
        "binaryPath": "ios/build/Build/Products/Debug-iphonesimulator/YourApp.app",
        "build": "xcodebuild -workspace ios/YourApp.xcworkspace -scheme YourApp -configuration Debug -sdk iphonesimulator -derivedDataPath ios/build",
        "type": "ios.simulator",
        "name": "iPhone 11"
      }
    }
  }
}
    • Para Android, la configuración será diferente: 
      {
  "detox": {
    "configurations": {
      "android.emu.debug": {
        "binaryPath": "android/app/build/outputs/apk/debug/app-debug.apk",
        "build": "cd android && ./gradlew assembleDebug assembleAndroidTest -DtestBuildType=debug && cd ..",
        "type": "android.emulator",
        "name": "Pixel_3a_API_30_x86"
      }
    }
  }
}

Escribiendo tu primera prueba con Detox

  • Creando un archivo de prueba:

    • Crea una carpeta e2e en la raíz de tu proyecto.
    • Añade un archivo de prueba, por ejemplo, firstTest.e2e.js.

  • Prueba de ejemplo:

    describe('Example', () => {
  beforeAll(async () => {
    await device.launchApp();
  });

  beforeEach(async () => {
    await device.reloadReactNative();
  });

  it('should have welcome screen', async () => {
    await expect(element(by.id('welcome'))).toBeVisible();
  });

  it('should show hello screen after tap', async () => {
    await element(by.id('hello_button')).tap();
    await expect(element(by.id('hello_screen'))).toBeVisible();
  });
});

  • Explicación:

    • describe: Agrupa pruebas relacionadas.
    • beforeAll: Se ejecuta una vez antes de todas las pruebas.
    • beforeEach: Se ejecuta antes de cada prueba.
    • it: Define una prueba individual.
    • element: Selecciona un elemento de la interfaz de usuario.
    • by.id: Selector para elementos por su testID.
    • tap: Simula una acción de toque.
    • expect: Aserción para verificar si un elemento es visible.

Ejecutando pruebas con Detox

  • Construyendo la aplicación:

    detox build --configuration ios.sim.debug

  • Ejecutando las pruebas:

    detox test --configuration ios.sim.debug

  • Comandos comunes:

    • detox build: Construye la aplicación para pruebas.
    • detox test: Ejecuta las pruebas.
    • detox clean-framework-cache: Limpia la caché de Detox.

Características avanzadas de Detox

  • Simulación y sustitución:

    • Usa Detox para simular solicitudes de red u otras dependencias.
    • Ejemplo usando jest para simular una solicitud de red: 
      jest.mock('axios', () => ({
  get: jest.fn(() => Promise.resolve({ data: { message: 'Hello World' } }))
}));

  • Ejecución paralela de pruebas:

    • Detox soporta la ejecución de pruebas en paralelo para acelerar el proceso de prueba.
    • Configura la ejecución paralela en tu package.json: 
      {
  "detox": {
    "test-runner": "jest",
    "runner-config": "e2e/config.json",
    "configurations": {
      "ios.sim.debug": {
        "binaryPath": "ios/build/Build/Products/Debug-iphonesimulator/YourApp.app",
        "build": "xcodebuild -workspace ios/YourApp.xcworkspace -scheme YourApp -configuration Debug -sdk iphonesimulator -derivedDataPath ios/build",
        "type": "ios.simulator",
        "name": "iPhone 11"
      }
    }
  }
}

Conclusión

Detox es una herramienta poderosa para pruebas de extremo a extremo en React Native, asegurando que tu aplicación se comporte como se espera en diferentes dispositivos y plataformas. Al configurar Detox, escribir pruebas y ejecutarlas, puedes automatizar el proceso de prueba y detectar errores temprano. Las características avanzadas como la simulación y la ejecución paralela de pruebas mejoran aún más tus capacidades de prueba. Comienza a integrar Detox en tu flujo de trabajo para mejorar la confiabilidad y calidad de tus aplicaciones React Native.
