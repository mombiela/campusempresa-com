Introducción

Manejar la entrada del usuario es un aspecto fundamental para construir aplicaciones interactivas. En React Native, el componente TextInput se utiliza para capturar la entrada de texto del usuario. Esta sección te guiará a través de los conceptos básicos del uso de TextInput, el manejo de su estado y la implementación de características avanzadas.

Uso Básico de TextInput

Para comenzar con TextInput, necesitas importarlo desde la biblioteca react-native e incluirlo en tu componente.

Ejemplo

import React, { useState } from 'react';
import { View, TextInput, Text, StyleSheet } from 'react-native';

const BasicTextInput = () => {
  const [text, setText] = useState('');

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        placeholder="Escribe aquí..."
        onChangeText={setText}
        value={text}
      />
      <Text style={styles.text}>Has escrito: {text}</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    padding: 20,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingHorizontal: 10,
  },
  text: {
    fontSize: 18,
  },
});

export default BasicTextInput;

Explicación

  • Importación de Componentes: TextInput, View, Text y StyleSheet se importan desde react-native.
  • Gestión del Estado: El hook useState se utiliza para gestionar el estado de la entrada de texto.
  • Componente TextInput:
    • style: Aplica estilos al TextInput.
    • placeholder: Muestra un texto de marcador de posición cuando la entrada está vacía.
    • onChangeText: Actualiza el estado cada vez que el texto cambia.
    • value: Vincula el estado al valor del TextInput.
  • Componente Text: Muestra el texto escrito por el usuario.

Manejo de Diferentes Tipos de Entrada

TextInput puede manejar varios tipos de entrada configurando la propiedad keyboardType.

Ejemplo

<TextInput
  style={styles.input}
  placeholder="Introduce tu correo electrónico"
  keyboardType="email-address"
  onChangeText={setEmail}
  value={email}
/>

Valores Comunes de `keyboardType`

| Valor | Descripción | |----------------|--------------------------------------| | default | Teclado por defecto | | email-address| Teclado optimizado para entrada de correo electrónico | | numeric | Teclado numérico | | phone-pad | Teclado optimizado para números de teléfono |

Gestión de Eventos de Foco y Desenfoque

Puedes manejar eventos de foco y desenfoque para mejorar la experiencia del usuario.

Ejemplo

<TextInput
  style={styles.input}
  placeholder="Enfoca para ver el efecto"
  onFocus={() => console.log('Entrada enfocada')}
  onBlur={() => console.log('Entrada desenfocada')}
/>

Explicación

  • onFocus: Se activa cuando el TextInput gana el foco.
  • onBlur: Se activa cuando el TextInput pierde el foco.

Características Avanzadas

Controlando TextInput Programáticamente

Puedes usar ref para controlar el TextInput programáticamente.

Ejemplo

import React, { useRef } from 'react';
import { View, TextInput, Button, StyleSheet } from 'react-native';

const ControlledTextInput = () => {
  const inputRef = useRef(null);

  const focusInput = () => {
    inputRef.current.focus();
  };

  return (
    <View style={styles.container}>
      <TextInput
        ref={inputRef}
        style={styles.input}
        placeholder="Entrada controlada"
      />
      <Button title="Enfocar la entrada" onPress={focusInput} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    padding: 20,
  },
  input: {
    height: 40,
    borderColor: 'gray',
    borderWidth: 1,
    marginBottom: 10,
    paddingHorizontal: 10,
  },
});

export default ControlledTextInput;

Explicación

  • Hook useRef: Se utiliza para crear una referencia al TextInput.
  • Método focus: Se llama para enfocar programáticamente el TextInput.

Conclusión

En esta sección, aprendiste cómo manejar la entrada del usuario utilizando el componente TextInput en React Native. Cubrimos el uso básico, el manejo de diferentes tipos de entrada, la gestión de eventos de foco y desenfoque, y el control programático de TextInput. Dominar estos conceptos te ayudará a construir aplicaciones más interactivas y amigables para el usuario.
