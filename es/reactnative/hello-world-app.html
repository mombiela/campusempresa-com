Introducción

En esta sección, te guiaremos a través de la creación de tu primera aplicación React Native. Esta será una aplicación simple de "Hola Mundo", que es un punto de partida tradicional para aprender cualquier nuevo lenguaje de programación o framework. Al final de esta sección, tendrás una comprensión básica de cómo configurar un entorno React Native y crear una aplicación simple.

Configuración del Entorno

Antes de comenzar a codificar, necesitamos configurar nuestro entorno de desarrollo.

Prerrequisitos

  • Node.js y npm (Node Package Manager)
  • React Native CLI o Expo CLI
  • Un editor de código (por ejemplo, Visual Studio Code)
  • Android Studio o Xcode (para ejecutar la aplicación en un emulador)

Pasos de Instalación

  1. Instalar Node.js y npm: Descarga e instala Node.js desde nodejs.org. npm está incluido con Node.js.

  2. Instalar React Native CLI: Abre tu terminal y ejecuta:

    npm install -g react-native-cli

  3. Instalar Expo CLI (Opcional): Si prefieres usar Expo, ejecuta:

    npm install -g expo-cli

  4. Configurar Android Studio o Xcode:

    • Para Android: Descarga e instala Android Studio desde developer.android.com.
    • Para iOS: Asegúrate de tener Xcode instalado desde la Mac App Store.

Creando Tu Primera Aplicación React Native

Ahora que nuestro entorno está configurado, vamos a crear nuestra primera aplicación.

Usando React Native CLI

  1. Crear un Nuevo Proyecto:

    npx react-native init HelloWorldApp

    Este comando crea un nuevo proyecto React Native llamado HelloWorldApp.

  2. Navegar al Directorio del Proyecto:

    cd HelloWorldApp

  3. Ejecutar la Aplicación:

    • Para Android: 
      npx react-native run-android
    • Para iOS: 
      npx react-native run-ios

Usando Expo CLI

  1. Crear un Nuevo Proyecto:

    expo init HelloWorldApp

    Elige una plantilla (por ejemplo, en blanco) y sigue las indicaciones.

  2. Navegar al Directorio del Proyecto:

    cd HelloWorldApp

  3. Ejecutar la Aplicación:

    expo start

    Esto abrirá una nueva pestaña en tu navegador con un código QR. Escanea el código QR con la aplicación Expo Go en tu dispositivo móvil para ejecutar la aplicación.

Escribiendo el Código "Hola Mundo"

Ahora, vamos a modificar el código predeterminado para mostrar "Hola Mundo".

Editando el Componente App

Abre App.js en tu editor de código y reemplaza su contenido con el siguiente código:

import React from 'react';
import { Text, View, StyleSheet } from 'react-native';

const App = () => {
  return (
    <View style={styles.container}>
      <Text style={styles.text}>Hola Mundo</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#F5FCFF',
  },
  text: {
    fontSize: 20,
    textAlign: 'center',
    margin: 10,
  },
});

export default App;

Explicación

  • Importaciones: Importamos React, Text, View, y StyleSheet de react-native.
  • Componente App: Este es un componente funcional que devuelve un View que contiene un elemento Text.
  • Estilos: Definimos algunos estilos básicos usando StyleSheet.create.

Ejecutando la Aplicación

Después de guardar tus cambios, tu aplicación debería recargarse automáticamente y mostrar "Hola Mundo" en la pantalla.

Conclusión

¡Felicidades! Has creado con éxito tu primera aplicación React Native. En esta sección, aprendiste a:

  • Configurar tu entorno de desarrollo.
  • Crear un nuevo proyecto React Native usando tanto React Native CLI como Expo CLI.
  • Escribir y ejecutar una simple aplicación "Hola Mundo".

Este conocimiento fundamental será crucial a medida que avancemos a temas más avanzados en el desarrollo con React Native.
