Introducción al Stack Navigator

Stack Navigator es un componente fundamental en la navegación de React Native, que te permite gestionar una pila de pantallas y navegar entre ellas. Imita el comportamiento de la pila de navegación que se encuentra en la mayoría de los sistemas operativos móviles.

Conceptos Clave

  • Pila: Una colección de pantallas donde cada nueva pantalla se coloca encima de la anterior.
  • Navegación: El proceso de moverse entre diferentes pantallas.
  • Ruta: La definición de una pantalla dentro de la pila.

Configuración del Stack Navigator

Para usar Stack Navigator, necesitas instalar los paquetes necesarios y configurar tu proyecto.

Instalación

Primero, instala los paquetes requeridos:

npm install @react-navigation/native @react-navigation/stack
npm install react-native-screens react-native-safe-area-context

Configuración Básica

A continuación, configura el contenedor de navegación y el stack navigator en tu archivo principal de la aplicación (por ejemplo, App.js).

import * as React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createStackNavigator } from '@react-navigation/stack';
import HomeScreen from './screens/HomeScreen';
import DetailsScreen from './screens/DetailsScreen';

const Stack = createStackNavigator();

export default function App() {
  return (
    <NavigationContainer>
      <Stack.Navigator initialRouteName="Home">
        <Stack.Screen name="Home" component={HomeScreen} />
        <Stack.Screen name="Details" component={DetailsScreen} />
      </Stack.Navigator>
    </NavigationContainer>
  );
}

Explicación

  • NavigationContainer: Envuelve toda tu aplicación y gestiona el árbol de navegación.
  • Stack.Navigator: Define un stack navigator.
  • Stack.Screen: Define pantallas individuales dentro de la pila.

Navegando Entre Pantallas

Para navegar entre pantallas, usa la prop navigation proporcionada por React Navigation.

Ejemplo

En HomeScreen.js:

import React from 'react';
import { Button, View, Text } from 'react-native';

export default function HomeScreen({ navigation }) {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Pantalla Principal</Text>
      <Button
        title="Ir a Detalles"
        onPress={() => navigation.navigate('Details')}
      />
    </View>
  );
}

En DetailsScreen.js:

import React from 'react';
import { View, Text } from 'react-native';

export default function DetailsScreen() {
  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Pantalla de Detalles</Text>
    </View>
  );
}

Explicación

  • navigation.navigate('Details'): Navega a la pantalla Details.

Personalizando el Stack Navigator

Puedes personalizar la apariencia y el comportamiento del stack navigator.

Personalización del Encabezado

Personaliza el encabezado para pantallas individuales o para toda la pila.

Ejemplo

En App.js:

<Stack.Navigator
  initialRouteName="Home"
  screenOptions={{
    headerStyle: {
      backgroundColor: '#f4511e',
    },
    headerTintColor: '#fff',
    headerTitleStyle: {
      fontWeight: 'bold',
    },
  }}
>
  <Stack.Screen
    name="Home"
    component={HomeScreen}
    options={{ title: 'Mi Casa' }}
  />
  <Stack.Screen
    name="Details"
    component={DetailsScreen}
    options={{ title: 'Página de Detalles' }}
  />
</Stack.Navigator>

Explicación

  • screenOptions: Aplica opciones predeterminadas a todas las pantallas en la pila.
  • options: Aplica opciones a pantallas individuales.

Pasando Parámetros Entre Pantallas

Puedes pasar parámetros a las pantallas usando la función navigate.

Ejemplo

En HomeScreen.js:

<Button
  title="Ir a Detalles"
  onPress={() => navigation.navigate('Details', { itemId: 86, otherParam: 'cualquier cosa que quieras aquí' })}
/>

En DetailsScreen.js:

export default function DetailsScreen({ route, navigation }) {
  const { itemId, otherParam } = route.params;

  return (
    <View style={{ flex: 1, justifyContent: 'center', alignItems: 'center' }}>
      <Text>Pantalla de Detalles</Text>
      <Text>itemId: {itemId}</Text>
      <Text>otherParam: {otherParam}</Text>
    </View>
  );
}

Explicación

  • route.params: Accede a los parámetros pasados a la pantalla.

Funcionalidades Avanzadas

Explora funcionalidades más avanzadas como navegadores anidados, transiciones personalizadas y deep linking.

Navegadores Anidados

Puedes anidar navegadores para crear estructuras de navegación complejas.

Ejemplo

En App.js:

const RootStack = createStackNavigator();
const MainStack = createStackNavigator();

function MainStackScreen() {
  return (
    <MainStack.Navigator>
      <MainStack.Screen name="Home" component={HomeScreen} />
      <MainStack.Screen name="Details" component={DetailsScreen} />
    </MainStack.Navigator>
  );
}

export default function App() {
  return (
    <NavigationContainer>
      <RootStack.Navigator>
        <RootStack.Screen name="Main" component={MainStackScreen} />
        <RootStack.Screen name="Profile" component={ProfileScreen} />
      </RootStack.Navigator>
    </NavigationContainer>
  );
}

Explicación

  • Navegadores Anidados: Te permite crear una jerarquía de navegadores para flujos de navegación más complejos.

Conclusión

Stack Navigator es una herramienta poderosa en React Native para gestionar la navegación entre pantallas. Al entender su configuración básica, métodos de navegación, opciones de personalización y funcionalidades avanzadas, puedes crear una experiencia de navegación fluida e intuitiva en tus aplicaciones de React Native. Practica construyendo diferentes flujos de navegación y personalizando la pila para adaptarla a las necesidades de tu aplicación.
