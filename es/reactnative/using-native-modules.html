Introducción

Los módulos nativos en React Native te permiten extender la funcionalidad de tu aplicación integrando código nativo (Java, Swift, Objective-C) con tu código JavaScript. Esto es particularmente útil cuando necesitas acceder a APIs específicas de la plataforma o realizar tareas que no son posibles solo con JavaScript.

Conceptos Clave

  • Módulos Nativos: Son módulos personalizados escritos en código nativo (Java para Android, Swift/Objective-C para iOS) que pueden ser invocados desde JavaScript.
  • Bridge: El mecanismo que permite la comunicación entre JavaScript y el código nativo.
  • React Native CLI: Una herramienta de línea de comandos que ayuda a crear y gestionar proyectos de React Native, incluyendo la vinculación de módulos nativos.

Configuración del Entorno

Antes de comenzar a trabajar con módulos nativos, asegúrate de tener lo siguiente:

  • Node.js instalado
  • React Native CLI instalado
  • Android Studio y Xcode instalados para el desarrollo en Android y iOS, respectivamente

Creación de un Módulo Nativo para Android

Paso 1: Crear un Nuevo Proyecto de React Native

npx react-native init NativeModuleExample
cd NativeModuleExample

Paso 2: Crear el Módulo Nativo

  1. Crear una nueva clase Java en android/app/src/main/java/com/nativemoduleexample/:

    package com.nativemoduleexample;

import com.facebook.react.bridge.ReactApplicationContext;
import com.facebook.react.bridge.ReactContextBaseJavaModule;
import com.facebook.react.bridge.ReactMethod;
import com.facebook.react.bridge.Promise;

public class MyNativeModule extends ReactContextBaseJavaModule {
    MyNativeModule(ReactApplicationContext context) {
        super(context);
    }

    @Override
    public String getName() {
        return "MyNativeModule";
    }

    @ReactMethod
    public void getDeviceName(Promise promise) {
        try {
            String deviceName = android.os.Build.MODEL;
            promise.resolve(deviceName);
        } catch (Exception e) {
            promise.reject("Error", e);
        }
    }
}

  2. Registrar el módulo en MainApplication.java:

    import com.nativemoduleexample.MyNativeModule;

@Override
protected List<ReactPackage> getPackages() {
    return Arrays.<ReactPackage>asList(
        new MainReactPackage(),
        new MyNativeModulePackage() // Añadir esta línea
    );
}

  3. Crear una clase de paquete:

    package com.nativemoduleexample;

import com.facebook.react.ReactPackage;
import com.facebook.react.bridge.NativeModule;
import com.facebook.react.bridge.ReactApplicationContext;
import com.facebook.react.uimanager.ViewManager;

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class MyNativeModulePackage implements ReactPackage {
    @Override
    public List<ViewManager> createViewManagers(ReactApplicationContext reactContext) {
        return Collections.emptyList();
    }

    @Override
    public List<NativeModule> createNativeModules(ReactApplicationContext reactContext) {
        List<NativeModule> modules = new ArrayList<>();
        modules.add(new MyNativeModule(reactContext));
        return modules;
    }
}

Paso 3: Usar el Módulo Nativo en JavaScript

import { NativeModules } from 'react-native';
const { MyNativeModule } = NativeModules;

MyNativeModule.getDeviceName()
  .then(deviceName => {
    console.log(deviceName);
  })
  .catch(error => {
    console.error(error);
  });

Creación de un Módulo Nativo para iOS

Paso 1: Crear un Nuevo Proyecto de React Native

npx react-native init NativeModuleExample
cd NativeModuleExample

Paso 2: Crear el Módulo Nativo

  1. Crear una nueva clase Objective-C en ios/NativeModuleExample/:

    // MyNativeModule.h
#import <React/RCTBridgeModule.h>

@interface MyNativeModule : NSObject <RCTBridgeModule>
@end

// MyNativeModule.m
#import "MyNativeModule.h"
#import <UIKit/UIKit.h>

@implementation MyNativeModule

RCT_EXPORT_MODULE();

RCT_REMAP_METHOD(getDeviceName,
                 getDeviceNameWithResolver:(RCTPromiseResolveBlock)resolve
                 rejecter:(RCTPromiseRejectBlock)reject) {
    @try {
        NSString *deviceName = [[UIDevice currentDevice] name];
        resolve(deviceName);
    } @catch (NSException *exception) {
        reject(@"Error", exception.reason, nil);
    }
}

@end

  2. Registrar el módulo en AppDelegate.m:

    #import "MyNativeModule.h"

Paso 3: Usar el Módulo Nativo en JavaScript

import { NativeModules } from 'react-native';
const { MyNativeModule } = NativeModules;

MyNativeModule.getDeviceName()
  .then(deviceName => {
    console.log(deviceName);
  })
  .catch(error => {
    console.error(error);
  });

Conclusión

Usar módulos nativos en React Native te permite aprovechar características y APIs específicas de la plataforma, mejorando las capacidades de tu aplicación. Al entender cómo crear e integrar módulos nativos, puedes construir aplicaciones más potentes y flexibles. Recuerda probar tus módulos nativos a fondo tanto en Android como en iOS para asegurar la compatibilidad y el rendimiento.
