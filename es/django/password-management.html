Gestión de Contraseñas en Django
Middleware ►

La gestión de contraseñas es un aspecto crítico de la seguridad en aplicaciones web. En esta sección, cubriremos cómo manejar las contraseñas de manera segura en Django, desde niveles principiantes hasta avanzados.

Introducción a la Gestión de Contraseñas

  • Importancia de la Seguridad de las Contraseñas: Las contraseñas son la primera línea de defensa contra el acceso no autorizado. Una gestión adecuada asegura que los datos de los usuarios permanezcan seguros.
  • Herramientas Integradas de Django: Django proporciona herramientas robustas para el hashing, validación y gestión de contraseñas.

Almacenamiento Seguro de Contraseñas

Hashing de Contraseñas

Django utiliza un algoritmo de hashing seguro para almacenar contraseñas. Esto asegura que, incluso si la base de datos es comprometida, las contraseñas reales no sean expuestas.

  • Algoritmos de Hashing: Django usa PBKDF2 por defecto, pero también soporta otros algoritmos como Argon2, BCrypt y SHA-1.
  • Configuración: Puedes configurar el algoritmo de hashing en el archivo settings.py.
# settings.py
PASSWORD_HASHERS = [
    'django.contrib.auth.hashers.PBKDF2PasswordHasher',
    'django.contrib.auth.hashers.PBKDF2SHA1PasswordHasher',
    'django.contrib.auth.hashers.Argon2PasswordHasher',
    'django.contrib.auth.hashers.BCryptSHA256PasswordHasher',
]

Creación y Almacenamiento de Contraseñas

Al crear un usuario, nunca debes almacenar la contraseña en texto plano. En su lugar, usa los métodos integrados de Django para hashear la contraseña.

from django.contrib.auth.models import User

# Crear un nuevo usuario con una contraseña hasheada
user = User.objects.create_user(username='john', password='mypassword')
user.save()

Validación de Contraseñas

Django proporciona una forma de validar contraseñas para asegurar que cumplan con ciertos criterios (por ejemplo, longitud, complejidad).

Uso de Validadores Integrados

Django viene con varios validadores de contraseñas integrados que puedes configurar en el archivo settings.py.

# settings.py
AUTH_PASSWORD_VALIDATORS = [
    {
        'NAME': 'django.contrib.auth.password_validation.UserAttributeSimilarityValidator',
    },
    {
        'NAME': 'django.contrib.auth.password_validation.MinimumLengthValidator',
        'OPTIONS': {
            'min_length': 8,
        }
    },
    {
        'NAME': 'django.contrib.auth.password_validation.CommonPasswordValidator',
    },
    {
        'NAME': 'django.contrib.auth.password_validation.NumericPasswordValidator',
    },
]

Validadores de Contraseñas Personalizados

También puedes crear validadores de contraseñas personalizados implementando los métodos validate y get_help_text.

from django.core.exceptions import ValidationError
from django.utils.translation import gettext as _

class MyCustomPasswordValidator:
    def validate(self, password, user=None):
        if not any(char.isdigit() for char in password):
            raise ValidationError(
                _("La contraseña debe contener al menos un dígito."),
                code='password_no_digit',
            )

    def get_help_text(self):
        return _("Tu contraseña debe contener al menos un dígito.")

Restablecimiento y Cambio de Contraseña

Restablecimiento de Contraseña

Django proporciona una vista integrada para el restablecimiento de contraseñas, que incluye el envío de un correo electrónico con un enlace de restablecimiento.

  • Configuración de URL: Añade las URLs de restablecimiento de contraseña a tu urls.py.
# urls.py
from django.contrib.auth import views as auth_views

urlpatterns = [
    path('password_reset/', auth_views.PasswordResetView.as_view(), name='password_reset'),
    path('password_reset/done/', auth_views.PasswordResetDoneView.as_view(), name='password_reset_done'),
    path('reset/<uidb64>/<token>/', auth_views.PasswordResetConfirmView.as_view(), name='password_reset_confirm'),
    path('reset/done/', auth_views.PasswordResetCompleteView.as_view(), name='password_reset_complete'),
]
  • Configuración de Correo Electrónico: Asegúrate de que tus configuraciones de correo electrónico estén configuradas en settings.py.
# settings.py
EMAIL_BACKEND = 'django.core.mail.backends.smtp.EmailBackend'
EMAIL_HOST = 'smtp.example.com'
EMAIL_PORT = 587
EMAIL_USE_TLS = True
EMAIL_HOST_USER = 'your-email@example.com'
EMAIL_HOST_PASSWORD = 'your-email-password'

Cambio de Contraseña

Django también proporciona vistas para que los usuarios autenticados cambien sus contraseñas.

  • Configuración de URL: Añade las URLs de cambio de contraseña a tu urls.py.
# urls.py
urlpatterns += [
    path('password_change/', auth_views.PasswordChangeView.as_view(), name='password_change'),
    path('password_change/done/', auth_views.PasswordChangeDoneView.as_view(), name='password_change_done'),
]

Temas Avanzados

Integración con Autenticación de Terceros

  • Autenticación Social: Usa bibliotecas como django-allauth o python-social-auth para integrar el inicio de sesión social (por ejemplo, Google, Facebook).
  • Autenticación de Dos Factores (2FA): Implementa 2FA usando paquetes como django-otp o django-two-factor-auth.

Políticas de Contraseñas

  • Imposición de Caducidad de Contraseñas: Implementa políticas para obligar a los usuarios a cambiar sus contraseñas periódicamente.
  • Bloqueo de Cuenta: Implementa mecanismos de bloqueo de cuenta para prevenir ataques de fuerza bruta.

Conclusión

La gestión de contraseñas es un aspecto crucial de la seguridad en aplicaciones web. Django proporciona un conjunto completo de herramientas y mejores prácticas para manejar contraseñas de manera segura. Siguiendo estas directrices, puedes asegurar que tu aplicación esté bien protegida contra amenazas de seguridad comunes.

  • Puntos Clave:
    • Siempre hashea las contraseñas antes de almacenarlas.
    • Usa los validadores integrados de Django para imponer políticas de contraseñas fuertes.
    • Implementa funcionalidades de restablecimiento y cambio de contraseñas.
    • Considera medidas de seguridad avanzadas como 2FA y mecanismos de bloqueo de cuenta.

Dominando estos conceptos, estarás bien equipado para gestionar contraseñas de manera segura en tus aplicaciones Django.
