Introducción

El registro de usuarios es una característica fundamental en las aplicaciones web, permitiendo a los usuarios crear cuentas y acceder a contenido personalizado. En esta sección, cubriremos cómo implementar el registro de usuarios en Django, desde la configuración básica hasta la personalización avanzada.

Configuración del Proyecto

Antes de comenzar con el registro de usuarios, asegúrate de tener Django instalado y un proyecto creado.

pip install django
django-admin startproject myproject
cd myproject
python manage.py startapp users

Registro Básico de Usuarios

Creación del Modelo de Usuario

Django viene con un modelo de usuario incorporado, pero puedes extenderlo si es necesario. Para simplificar, usaremos el modelo de usuario predeterminado de Django.

Creación del Formulario de Registro

Usaremos los formularios incorporados de Django para crear un formulario de registro de usuario.

# users/forms.py
from django import forms
from django.contrib.auth.models import User
from django.contrib.auth.forms import UserCreationForm

class UserRegisterForm(UserCreationForm):
    email = forms.EmailField()

    class Meta:
        model = User
        fields = ['username', 'email', 'password1', 'password2']

Creación de la Vista de Registro

Crea una vista para manejar el formulario de registro.

# users/views.py
from django.shortcuts import render, redirect
from django.contrib import messages
from .forms import UserRegisterForm

def register(request):
    if request.method == 'POST':
        form = UserRegisterForm(request.POST)
        if form.is_valid():
            form.save()
            username = form.cleaned_data.get('username')
            messages.success(request, f'¡Cuenta creada para {username}!')
            return redirect('login')
    else:
        form = UserRegisterForm()
    return render(request, 'users/register.html', {'form': form})

Creación de la Plantilla de Registro

Crea una plantilla para mostrar el formulario de registro.

<!-- users/templates/users/register.html -->
{% extends "base_generic.html" %}

{% block content %}
  <div class="container">
    <h2>Registrarse</h2>
    <form method="POST">
      {% csrf_token %}
      {{ form.as_p }}
      <button type="submit" class="btn btn-primary">Registrarse</button>
    </form>
  </div>
{% endblock %}

Adición de Patrones de URL

Agrega el patrón de URL para la vista de registro.

# users/urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('register/', views.register, name='register'),
]

Incluye las URLs de la aplicación users en las URLs del proyecto principal.

# myproject/urls.py
from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('users/', include('users.urls')),
]

Registro Avanzado de Usuarios

Personalización del Modelo de Usuario

Si necesitas almacenar información adicional, puedes extender el modelo de usuario.

# users/models.py
from django.contrib.auth.models import AbstractUser
from django.db import models

class CustomUser(AbstractUser):
    age = models.PositiveIntegerField(null=True, blank=True)

Actualiza la configuración para usar el modelo de usuario personalizado.

# myproject/settings.py
AUTH_USER_MODEL = 'users.CustomUser'

Manejo de Perfiles de Usuario

Crea un modelo de perfil para almacenar información adicional del usuario.

# users/models.py
from django.db import models
from django.contrib.auth.models import User

class Profile(models.Model):
    user = models.OneToOneField(User, on_delete=models.CASCADE)
    image = models.ImageField(default='default.jpg', upload_to='profile_pics')

    def __str__(self):
        return f'Perfil de {self.user.username}'

Señales para la Creación de Perfiles

Usa señales de Django para crear automáticamente un perfil cuando un usuario se registre.

# users/signals.py
from django.db.models.signals import post_save
from django.contrib.auth.models import User
from django.dispatch import receiver
from .models import Profile

@receiver(post_save, sender=User)
def create_profile(sender, instance, created, **kwargs):
    if created:
        Profile.objects.create(user=instance)

@receiver(post_save, sender=User)
def save_profile(sender, instance, **kwargs):
    instance.profile.save()

Registra las señales en el apps.py.

# users/apps.py
from django.apps import AppConfig

class UsersConfig(AppConfig):
    name = 'users'

    def ready(self):
        import users.signals
