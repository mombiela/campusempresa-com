Introducción

Configurar tu entorno de desarrollo es el primer paso para comenzar a trabajar con Django. Esta sección te guiará a través del proceso de instalación de las herramientas necesarias y la configuración de tu sistema para el desarrollo con Django. Al final de esta sección, tendrás un entorno de desarrollo Django completamente funcional.

Requisitos Previos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Una computadora con un sistema operativo (Windows, macOS o Linux).
  • Conocimientos básicos de programación en Python.
  • Conexión a Internet para descargar el software necesario.

Paso 1: Instalar Python

Django es un framework web de Python, por lo que necesitas tener Python instalado en tu máquina.

Windows

  1. Descarga la última versión de Python desde el sitio web oficial.
  2. Ejecuta el instalador y asegúrate de marcar la casilla que dice "Add Python to PATH".
  3. Verifica la instalación abriendo el Símbolo del sistema y escribiendo: 
    python --version

macOS

  1. Abre Terminal.
  2. Usa Homebrew para instalar Python (si no tienes Homebrew, instálalo desde aquí): 
    brew install python
  3. Verifica la instalación escribiendo: 
    python3 --version

Linux

  1. Abre Terminal.
  2. Usa el gestor de paquetes para instalar Python: 
    sudo apt-get update
sudo apt-get install python3
  3. Verifica la instalación escribiendo: 
    python3 --version

Paso 2: Instalar pip

pip es el instalador de paquetes para Python. Generalmente se incluye con Python, pero puedes instalarlo manualmente si es necesario.

Verificar la Instalación de pip

pip --version

Instalar pip (si no está instalado)

python -m ensurepip --upgrade

Paso 3: Configurar un Entorno Virtual

Un entorno virtual ayuda a gestionar las dependencias de diferentes proyectos por separado.

Crear un Entorno Virtual

  1. Navega a tu directorio de proyecto: 
    cd path/to/your/project
  2. Crea un entorno virtual: 
    python -m venv myenv

Activar el Entorno Virtual

  • Windows: 
    myenv\Scripts\activate
  • macOS/Linux: 
    source myenv/bin/activate

Desactivar el Entorno Virtual

Para desactivar el entorno virtual, simplemente escribe:

deactivate

Paso 4: Instalar Django

Con el entorno virtual activado, instala Django usando pip.

pip install django

Verifica la instalación escribiendo:

django-admin --version

Paso 5: Configurar un Proyecto Django

  1. Crea un nuevo proyecto Django: 
    django-admin startproject myproject
  2. Navega al directorio del proyecto: 
    cd myproject
  3. Ejecuta el servidor de desarrollo para asegurarte de que todo esté configurado correctamente: 
    python manage.py runserver

Deberías ver una salida indicando que el servidor está en funcionamiento. Abre un navegador web y ve a http://127.0.0.1:8000/ para ver la página de bienvenida de Django.

Conclusión

¡Felicidades! Has configurado exitosamente tu entorno de desarrollo Django. Ahora tienes Python, pip, un entorno virtual y Django instalados. También creaste un nuevo proyecto Django y ejecutaste el servidor de desarrollo. Esta configuración servirá como la base para todos tus proyectos Django. En las siguientes secciones, profundizaremos en las características de Django y comenzaremos a construir aplicaciones web.
