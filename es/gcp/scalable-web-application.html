Introducción

En este curso, exploraremos cómo construir una aplicación web escalable utilizando Google Cloud Platform (GCP). Comenzaremos con lo básico y gradualmente avanzaremos hacia conceptos más avanzados, asegurándonos de que adquieras una comprensión integral de las herramientas y servicios proporcionados por GCP.

5. Gestión de Bases de Datos

  • Elegir el servicio de base de datos adecuado:
    • Cloud SQL vs. Firestore vs. BigQuery
  • Configuración de Cloud SQL
  • Conexión de tu aplicación a Cloud SQL

Ejemplo: Conectando una Aplicación Flask a Cloud SQL

# app.py
import os
import sqlalchemy

db_user = os.environ.get('DB_USER')
db_pass = os.environ.get('DB_PASS')
db_name = os.environ.get('DB_NAME')
db_host = os.environ.get('DB_HOST')

db = sqlalchemy.create_engine(
    sqlalchemy.engine.url.URL(
        drivername='mysql+pymysql',
        username=db_user,
        password=db_pass,
        database=db_name,
        host=db_host
    )
)

@app.route('/users')
def users():
    with db.connect() as conn:
        result = conn.execute('SELECT * FROM users')
        users = [row['name'] for row in result]
    return ', '.join(users)

gcloud projects add-iam-policy-binding [PROJECT_ID] \

    --role='roles/viewer'

8. Temas Avanzados

  • Uso de Kubernetes con GKE
  • Implementación de CI/CD con Cloud Build
  • Arquitectura sin servidor con Cloud Functions
  • Redes avanzadas con VPC

Ejemplo: Desplegando una Aplicación Contenerizada con GKE

# Construir la imagen Docker
docker build -t gcr.io/[PROJECT_ID]/my-app:v1 .

# Subir la imagen a Container Registry
docker push gcr.io/[PROJECT_ID]/my-app:v1

# Crear un clúster GKE
gcloud container clusters create my-cluster

# Desplegar la aplicación en GKE
kubectl run my-app --image=gcr.io/[PROJECT_ID]/my-app:v1 --port=8080
kubectl expose deployment my-app --type=LoadBalancer --port 80 --target-port 8080

Conclusión

Construir una aplicación web escalable en GCP implica entender y utilizar varios servicios y herramientas proporcionados por la plataforma. Desde la configuración de tu entorno hasta el despliegue y escalado de tu aplicación, este curso ha cubierto los pasos esenciales y las mejores prácticas. Siguiendo estas pautas y ejemplos, estarás bien equipado para crear aplicaciones web robustas y escalables en GCP.
