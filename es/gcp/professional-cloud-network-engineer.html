Introducción

Google Cloud Platform (GCP) ofrece una amplia gama de servicios de redes que son esenciales para construir, desplegar y gestionar aplicaciones en la nube. Este curso te guiará desde los conceptos básicos de redes en GCP hasta temas avanzados, preparándote para la certificación de Ingeniero Profesional de Redes en la Nube de GCP.

Esquema del Curso

  1. Introducción a las Redes en GCP
  2. Red Privada Virtual (VPC)
  3. Conectividad Híbrida
  4. Balanceo de Carga
  5. Seguridad de la Red
  6. Monitoreo y Solución de Problemas
  7. Conceptos Avanzados de Redes

  • Descripción General de los Servicios de Redes en GCP
  • Importancia de las Redes en la Computación en la Nube

2. Red Privada Virtual (VPC)

¿Qué es una VPC?

  • Definición y Propósito
  • Componentes de una VPC: Subredes, Rutas, Cortafuegos

Creación y Configuración de VPCs

  • Pasos para Crear una VPC
  • Configuración de Subredes y Rutas
  • Ejemplo: 
    gcloud compute networks create my-vpc --subnet-mode=custom
gcloud compute networks subnets create my-subnet 
          --network=my-vpc 
          --region=us-central1 
          --range=10.0.0.0/24

Emparejamiento de VPC

  • Concepto y Casos de Uso
  • Configuración del Emparejamiento de VPC
  • Ejemplo: 
    gcloud compute networks peerings create my-peering 
          --network=my-vpc 
          --peer-network=peer-vpc

3. Conectividad Híbrida

VPN

  • VPN de Sitio a Sitio
  • Configuración de Puertas de Enlace VPN
  • Ejemplo: 
    gcloud compute vpn gateways create my-vpn-gateway 
          --network=my-vpc 
          --region=us-central1

Interconexión

  • Interconexión Dedicada vs Interconexión de Socios
  • Configuración de Interconexión
  • Ejemplo: 
    gcloud compute interconnects create my-interconnect 
          --customer-name=my-company 
          --interconnect-type=DEDICATED 
          --link-type=LINK_TYPE_ETHERNET_10G_LR 
          --location=us-central1

4. Balanceo de Carga

Tipos de Balanceadores de Carga

  • Balanceo de Carga HTTP(S)
  • Balanceo de Carga TCP/UDP
  • Balanceo de Carga Interno

Configuración de Balanceadores de Carga

  • Pasos para Configurar un Balanceador de Carga HTTP(S)
  • Ejemplo: 
    gcloud compute backend-services create my-backend-service 
          --protocol=HTTP 
          --port-name=http 
          --health-checks=my-health-check 
          --global

5. Seguridad de la Red

Cortafuegos

  • Reglas y Políticas de Cortafuegos
  • Creación y Gestión de Reglas de Cortafuegos
  • Ejemplo: 
    gcloud compute firewall-rules create allow-http 
          --network=my-vpc 
          --allow=tcp:80 
          --source-ranges=0.0.0.0/0

Gestión de Identidades y Accesos (IAM)

  • Roles de IAM para Redes
  • Configuración de Políticas de IAM
  • Ejemplo: 
    gcloud projects add-iam-policy-binding my-project 
          --member=user:example-user@gmail.com 
          --role=roles/compute.networkAdmin

6. Monitoreo y Solución de Problemas

Stackdriver Monitoring

  • Configuración de Monitoreo para Recursos de Red
  • Ejemplo: 
    gcloud monitoring policies create 
          --display-name="High Latency Alert" 
          --conditions="metric.type=\"compute.googleapis.com/instance/network/received_bytes_count\" AND metric.labels.instance_name=\"my-instance\""

Centro de Inteligencia de Redes

  • Uso de la Topología de Redes y Pruebas de Conectividad
  • Ejemplo: 
    gcloud compute network-management connectivity-tests create my-test 
          --source=my-vpc 
          --destination=peer-vpc

7. Conceptos Avanzados de Redes

Acceso Privado a Google

  • Configuración de Acceso Privado a Google
  • Ejemplo: 
    gcloud compute networks subnets update my-subnet 
          --region=us-central1 
          --enable-private-ip-google-access

Niveles de Servicio de Red

  • Nivel Estándar vs Nivel Premium
  • Configuración de Niveles de Servicio de Red
  • Ejemplo: 
    gcloud compute addresses create my-address 
          --region=us-central1 
          --network-tier=PREMIUM

Conclusión

Este curso ha cubierto los aspectos esenciales de las redes en GCP, desde conceptos básicos hasta configuraciones avanzadas. Al dominar estos temas, estarás bien preparado para gestionar y optimizar los recursos de red en GCP, y listo para tomar el examen de certificación de Ingeniero Profesional de Redes en la Nube de GCP. Continúa practicando y explorando la extensa documentación de GCP para profundizar tu comprensión y mantenerte actualizado con las últimas características y mejores prácticas.
