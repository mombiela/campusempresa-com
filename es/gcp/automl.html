Introducción a AutoML

AutoML (Machine Learning Automatizado) es un conjunto de herramientas y servicios de machine learning que automatizan el proceso de aplicar machine learning a problemas del mundo real. Google Cloud Platform (GCP) ofrece una variedad de servicios AutoML que permiten a los usuarios construir y desplegar modelos de machine learning con un mínimo de codificación y experiencia en machine learning.

Conceptos Clave

  • Ingeniería de Características Automatizada: Generar y seleccionar automáticamente las características más relevantes para un modelo.
  • Selección de Modelos: Seleccionar automáticamente la mejor arquitectura de modelo para un conjunto de datos dado.
  • Ajuste de Hiperparámetros: Ajustar automáticamente los hiperparámetros de un modelo para optimizar el rendimiento.
  • Entrenamiento de Modelos: Entrenar el modelo utilizando las características, arquitectura e hiperparámetros seleccionados.
  • Evaluación de Modelos: Evaluar el rendimiento del modelo entrenado utilizando varias métricas.
  • Despliegue de Modelos: Desplegar el modelo entrenado para hacer predicciones sobre nuevos datos.

Empezando con AutoML en GCP

Para usar AutoML en GCP, necesitas configurar un proyecto de GCP y habilitar las APIs necesarias.

Pasos para Configurar

  1. Crear un Proyecto de GCP:

    • Ve a la Consola de GCP.
    • Haz clic en el menú desplegable de proyectos y selecciona "Nuevo Proyecto".
    • Ingresa un nombre de proyecto y haz clic en "Crear".

  2. Habilitar las APIs de AutoML:

    • Navega a la sección "APIs & Services".
    • Busca "AutoML" y habilita las APIs relevantes (por ejemplo, AutoML Vision, AutoML Natural Language).

  3. Configurar la Facturación:

    • Asegúrate de que la facturación esté habilitada para tu proyecto.

  4. Instalar el SDK de Google Cloud:

    • Sigue las instrucciones aquí para instalar el SDK.

  5. Autenticar el SDK:

    gcloud auth login
gcloud config set project TU_ID_DE_PROYECTO

AutoML Vision

AutoML Vision te permite construir modelos personalizados de clasificación de imágenes.

Pasos para Crear un Modelo de AutoML Vision

  1. Subir tu Conjunto de Datos:

    • Prepara un conjunto de datos con imágenes etiquetadas.
    • Sube el conjunto de datos a un bucket de Google Cloud Storage.

  2. Crear un Conjunto de Datos en AutoML Vision:

    • Ve a la UI de AutoML Vision.
    • Haz clic en "Nuevo Conjunto de Datos" y sigue las indicaciones para importar tus imágenes desde el bucket de Cloud Storage.

  3. Entrenar el Modelo:

    • Después de crear el conjunto de datos, haz clic en "Entrenar Nuevo Modelo".
    • Elige el tipo de modelo y el presupuesto de entrenamiento.
    • Haz clic en "Iniciar Entrenamiento".

  4. Evaluar el Modelo:

    • Una vez completado el entrenamiento, revisa las métricas de evaluación proporcionadas por AutoML Vision.

  5. Desplegar el Modelo:

    • Haz clic en "Desplegar Modelo" para hacer que el modelo esté disponible para predicciones.

Código de Ejemplo para Hacer Predicciones

from google.cloud import automl_v1beta1 as automl

project_id = "TU_ID_DE_PROYECTO"
model_id = "TU_ID_DE_MODELO"
file_path = "ruta/a/tu/imagen.jpg"

client = automl.PredictionServiceClient()
model_full_id = client.model_path(project_id, "us-central1", model_id)

with open(file_path, "rb") as image_file:
    content = image_file.read()

payload = {"image": {"image_bytes": content}}
params = {}

response = client.predict(name=model_full_id, payload=payload, params=params)

for result in response.payload:
    print("Nombre de clase predicha: {}".format(result.display_name))
    print("Puntuación de clase predicha: {}".format(result.classification.score))

AutoML Natural Language

AutoML Natural Language te permite construir modelos personalizados para clasificación de texto, extracción de entidades y análisis de sentimientos.

Pasos para Crear un Modelo de AutoML Natural Language

  1. Preparar tu Conjunto de Datos:

    • Crea un archivo CSV con tus datos de texto y etiquetas.
    • Sube el archivo CSV a un bucket de Google Cloud Storage.

  2. Crear un Conjunto de Datos en AutoML Natural Language:

    • Ve a la UI de AutoML Natural Language.
    • Haz clic en "Nuevo Conjunto de Datos" y sigue las indicaciones para importar tu archivo CSV.

  3. Entrenar el Modelo:

    • Después de crear el conjunto de datos, haz clic en "Entrenar Nuevo Modelo".
    • Elige el tipo de modelo y el presupuesto de entrenamiento.
    • Haz clic en "Iniciar Entrenamiento".

  4. Evaluar el Modelo:

    • Una vez completado el entrenamiento, revisa las métricas de evaluación proporcionadas por AutoML Natural Language.

  5. Desplegar el Modelo:

    • Haz clic en "Desplegar Modelo" para hacer que el modelo esté disponible para predicciones.

Código de Ejemplo para Hacer Predicciones

from google.cloud import automl_v1beta1 as automl

project_id = "TU_ID_DE_PROYECTO"
model_id = "TU_ID_DE_MODELO"
text_content = "Tu texto para analizar"

client = automl.PredictionServiceClient()
model_full_id = client.model_path(project_id, "us-central1", model_id)

payload = {"text_snippet": {"content": text_content, "mime_type": "text/plain"}}
params = {}

response = client.predict(name=model_full_id, payload=payload, params=params)

for result in response.payload:
    print("Nombre de clase predicha: {}".format(result.display_name))
    print("Puntuación de clase predicha: {}".format(result.classification.score))

Temas Avanzados en AutoML

Entrenamiento Personalizado con AutoML Tables

AutoML Tables te permite construir modelos personalizados para datos estructurados.

Ajuste de Hiperparámetros

  • Búsqueda en Cuadrícula: Buscar exhaustivamente a través de un subconjunto especificado manualmente del espacio de hiperparámetros.
  • Búsqueda Aleatoria: Muestrear aleatoriamente el espacio de hiperparámetros.

Interpretabilidad del Modelo

  • Importancia de Características: Entender qué características tienen el mayor impacto en las predicciones del modelo.
  • Valores SHAP: Un método para explicar predicciones individuales.

Conclusión

AutoML en GCP proporciona un poderoso conjunto de herramientas para construir y desplegar modelos de machine learning con un mínimo esfuerzo. Al automatizar muchos de los pasos complejos involucrados en el machine learning, AutoML permite a los usuarios centrarse en resolver sus problemas de negocio específicos. Ya sea que estés trabajando con imágenes, texto o datos estructurados, los servicios AutoML de GCP pueden ayudarte a crear modelos de alta calidad de manera rápida y eficiente.
