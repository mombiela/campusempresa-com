Introducción a la Plataforma de IA

La Plataforma de IA de Google Cloud es un conjunto completo de herramientas y servicios diseñados para ayudar a los desarrolladores y científicos de datos a construir, desplegar y gestionar modelos de aprendizaje automático. Este curso te guiará desde los conceptos básicos hasta el uso avanzado de la Plataforma de IA.

Conceptos Clave

  • Aprendizaje Automático (ML): Un subconjunto de la inteligencia artificial que implica entrenar algoritmos para hacer predicciones o tomar decisiones basadas en datos.
  • Plataforma de IA: Un servicio gestionado en GCP que proporciona herramientas para cada paso del ciclo de vida del ML, desde la preparación de datos hasta el despliegue del modelo.

Empezando con la Plataforma de IA

Configuración de Tu Entorno

  1. Crear una Cuenta de GCP:

    • Regístrate para obtener una cuenta de Google Cloud y crea un nuevo proyecto.
    • Habilita la facturación para tu proyecto.

  2. Habilitar las API de la Plataforma de IA:

    • Navega al panel de API y Servicios.
    • Habilita la API de "Plataforma de IA".

  3. Instalar Google Cloud SDK:

    • Descarga e instala el Google Cloud SDK.
    • Inicializa el SDK con gcloud init.

Operaciones Básicas

  • Crear una Instancia de Jupyter Notebook:

    gcloud ai-platform notebooks instances create my-notebook 
          --vm-image-project=deeplearning-platform-release 
          --vm-image-family=tf2-ent-2-3-cpu 
          --location=us-central1-a

  • Acceder al Notebook:

    • Abre la página de Notebooks de la Plataforma de IA en la Consola de GCP.
    • Haz clic en tu instancia de notebook para abrir JupyterLab.

Construcción y Entrenamiento de Modelos

Preparación de Datos

  • Subir Datos a Cloud Storage:

    gsutil cp local-data.csv gs://your-bucket-name/data/

  • Cargar Datos en un Notebook:

    import pandas as pd
data = pd.read_csv('gs://your-bucket-name/data/local-data.csv')

Entrenamiento de Modelos

  • Usar TensorFlow para Entrenar Modelos:

    import tensorflow as tf
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense

# Definir un modelo simple
model = Sequential([
    Dense(64, activation='relu', input_shape=(data.shape[1],)),
    Dense(1)
])

# Compilar el modelo
model.compile(optimizer='adam', loss='mean_squared_error')

# Entrenar el modelo
model.fit(data, labels, epochs=10)

  • Enviar un Trabajo de Entrenamiento:

    gcloud ai-platform jobs submit training my_training_job 
          --module-name trainer.task 
          --package-path ./trainer 
          --region us-central1 
          --python-version 3.7 
          --runtime-version 2.3 
          --job-dir gs://your-bucket-name/job-dir

Despliegue de Modelos

Exportación de Modelos

  • Exportar un Modelo Entrenado: 
    model.save('gs://your-bucket-name/models/my_model')

Despliegue de Modelos

  • Crear un Modelo en la Plataforma de IA:

    gcloud ai-platform models create my_model

  • Desplegar una Versión del Modelo:

    gcloud ai-platform versions create v1 
          --model my_model 
          --origin gs://your-bucket-name/models/my_model 
          --runtime-version 2.3 
          --python-version 3.7

Temas Avanzados

Ajuste de Hiperparámetros

  • Usar la Plataforma de IA para el Ajuste de Hiperparámetros: 
    gcloud ai-platform jobs submit training my_hyperparameter_tuning_job 
          --module-name trainer.task 
          --package-path ./trainer 
          --region us-central1 
          --python-version 3.7 
          --runtime-version 2.3 
          --job-dir gs://your-bucket-name/job-dir 
          --config hyperparam_config.yaml

Contenedores Personalizados

  • Construir un Contenedor Personalizado para Entrenamiento:

    FROM gcr.io/deeplearning-platform-release/tf2-cpu.2-3
COPY trainer /trainer
WORKDIR /trainer
ENTRYPOINT ["python", "task.py"]

  • Enviar un Trabajo de Entrenamiento con un Contenedor Personalizado:

    gcloud ai-platform jobs submit training my_custom_container_job 
          --region us-central1 
          --master-image-uri gcr.io/your-project-id/custom-container:latest 
          --job-dir gs://your-bucket-name/job-dir

Conclusión

En este curso, hemos cubierto lo esencial del uso de la Plataforma de IA de Google Cloud, desde la configuración de tu entorno hasta el despliegue de modelos avanzados de aprendizaje automático. Al dominar estas herramientas, puedes gestionar eficientemente todo el ciclo de vida del ML, desde la preparación de datos hasta el despliegue del modelo y más allá. Continúa explorando la documentación de la Plataforma de IA y experimentando con diferentes características para profundizar tu comprensión y mejorar tus habilidades.
