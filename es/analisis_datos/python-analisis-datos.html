Introducción

El análisis de datos es una disciplina fundamental en la era de la información. Python se ha convertido en una herramienta esencial para los analistas de datos debido a su simplicidad y la potencia de sus bibliotecas. Este tema cubrirá los conceptos básicos y avanzados del uso de Python para el análisis de datos.

Instalación y Configuración

Antes de comenzar con el análisis de datos en Python, es necesario configurar el entorno de trabajo.

Instalación de Python

  • Descarga e instala Python desde python.org.
  • Verifica la instalación ejecutando python --version en la terminal.

Instalación de Bibliotecas Esenciales

Para el análisis de datos, utilizaremos varias bibliotecas populares de Python:

pip install numpy pandas matplotlib seaborn
  • NumPy: Biblioteca fundamental para el cálculo numérico.
  • Pandas: Biblioteca para la manipulación y análisis de datos.
  • Matplotlib: Biblioteca para la visualización de datos.
  • Seaborn: Biblioteca para la visualización de datos basada en Matplotlib.

Manipulación de Datos con Pandas

Pandas es una biblioteca poderosa para la manipulación y análisis de datos. Proporciona estructuras de datos como Series y DataFrames.

Series

Una Serie es una estructura de datos unidimensional similar a una lista.

import pandas as pd

# Crear una Serie
data = [1, 2, 3, 4, 5]
series = pd.Series(data)
print(series)

DataFrames

Un DataFrame es una estructura de datos bidimensional similar a una tabla.

# Crear un DataFrame
data = {
    'Nombre': ['Ana', 'Luis', 'Carlos'],
    'Edad': [28, 34, 29],
    'Ciudad': ['Madrid', 'Barcelona', 'Valencia']
}
df = pd.DataFrame(data)
print(df)

Operaciones Básicas con DataFrames

  • Selección de columnas:
print(df['Nombre'])
  • Filtrado de filas:
print(df[df['Edad'] > 30])
  • Agregar nuevas columnas:
df['Salario'] = [3000, 4000, 3500]
print(df)

Análisis Estadístico con NumPy

NumPy proporciona soporte para grandes matrices multidimensionales y funciones matemáticas de alto nivel.

Creación de Arrays

import numpy as np

# Crear un array
array = np.array([1, 2, 3, 4, 5])
print(array)

Operaciones Básicas

  • Suma:
print(np.sum(array))
  • Media:
print(np.mean(array))
  • Desviación estándar:
print(np.std(array))

Visualización de Datos

La visualización de datos es crucial para interpretar y comunicar los resultados del análisis.

Gráficos con Matplotlib

import matplotlib.pyplot as plt

# Crear un gráfico de líneas
plt.plot([1, 2, 3, 4, 5], [1, 4, 9, 16, 25])
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Gráfico de Líneas')
plt.show()

Gráficos con Seaborn

import seaborn as sns

# Crear un gráfico de dispersión
sns.scatterplot(x='Edad', y='Salario', data=df)
plt.show()

Ejercicios Prácticos

  1. Crear un DataFrame: Crea un DataFrame con datos ficticios sobre estudiantes (nombre, edad, calificación).
  2. Operaciones con DataFrames: Filtra los estudiantes con calificaciones superiores a 7.
  3. Visualización: Crea un gráfico de barras que muestre la cantidad de estudiantes por calificación.

Conclusión

Python ofrece un conjunto robusto de herramientas para el análisis de datos. Desde la manipulación de datos con Pandas, pasando por el análisis estadístico con NumPy, hasta la visualización con Matplotlib y Seaborn, Python facilita el proceso de convertir datos en información útil. Practicar con estos ejemplos y ejercicios ayudará a consolidar los conocimientos adquiridos.
