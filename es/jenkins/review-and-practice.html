Ejercicios Prácticos

Ejercicio 1: Creación de un Trabajo Básico

Crea un trabajo simple en Jenkins que imprima "¡Hola, Jenkins!" en la consola.

pipeline {
    agent any
    stages {
        stage('Hello') {
            steps {
                echo '¡Hola, Jenkins!'
            }
        }
    }
}

Ejercicio 2: Construcción de una Aplicación Java

Configura un pipeline de Jenkins para construir una aplicación Java usando Maven.

pipeline {
    agent any
    tools {
        maven 'Maven 3.6.3' // Asegúrate de que Maven esté instalado y configurado en Jenkins
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/tu-repo/tu-app-java.git'
            }
        }
        stage('Build') {
            steps {
                sh 'mvn clean install'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
    }
}

Ejercicio 3: Uso de Docker con Jenkins

Crea un pipeline de Jenkins que construya una imagen Docker y la suba a Docker Hub.

pipeline {
    agent any
    environment {
        DOCKERHUB_CREDENTIALS = credentials('dockerhub-credentials-id')
    }
    stages {
        stage('Checkout') {
            steps {
                git 'https://github.com/tu-repo/tu-app-docker.git'
            }
        }
        stage('Build Docker Image') {
            steps {
                script {
                    dockerImage = docker.build("tu-usuario-dockerhub/tu-app-docker:${env.BUILD_ID}")
                }
            }
        }
        stage('Push Docker Image') {
            steps {
                script {
                    docker.withRegistry('https://index.docker.io/v1/', 'DOCKERHUB_CREDENTIALS') {
                        dockerImage.push()
                    }
                }
            }
        }
    }
}

        }
        stage('Build') {
            steps {
                sh 'mvn clean install'
            }
        }
        stage('Test') {
            steps {
                sh 'mvn test'
            }
        }
        stage('Deploy') {
            steps {
                echo 'Desplegando aplicación...'
                // Añade aquí tus pasos de despliegue
            }
        }
    }
}

Resumen

En esta sección, revisamos los conceptos esenciales de Jenkins, incluyendo la creación de trabajos, pipelines, plugins y prácticas de CI/CD. Los ejercicios prácticos proporcionaron experiencia práctica con la creación de trabajos y pipelines en Jenkins, la integración con sistemas de control de versiones, el uso de Docker y la configuración de un pipeline completo de CI/CD. Al completar estos ejercicios, ahora deberías tener una comprensión sólida de cómo usar Jenkins para automatizar eficazmente tus procesos de desarrollo de software.
