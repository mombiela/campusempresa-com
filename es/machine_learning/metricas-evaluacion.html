Introducción

En Machine Learning, las métricas de evaluación son cruciales para medir el rendimiento de los modelos. Estas métricas nos permiten entender qué tan bien está funcionando un modelo y compararlo con otros modelos. En este tema, exploraremos las métricas más comunes utilizadas para evaluar modelos de clasificación y regresión.

Métricas para Modelos de Clasificación

Exactitud (Accuracy)

  • Definición: La proporción de predicciones correctas sobre el total de predicciones.
  • Fórmula: \[ \text{Exactitud} = \frac{\text{Número de predicciones correctas}}{\text{Número total de predicciones}} \]

Precisión (Precision)

  • Definición: La proporción de verdaderos positivos sobre el total de predicciones positivas.
  • Fórmula: \[ \text{Precisión} = \frac{\text{Verdaderos Positivos}}{\text{Verdaderos Positivos} + \text{Falsos Positivos}} \]

Recall (Sensibilidad)

  • Definición: La proporción de verdaderos positivos sobre el total de positivos reales.
  • Fórmula: \[ \text{Recall} = \frac{\text{Verdaderos Positivos}}{\text{Verdaderos Positivos} + \text{Falsos Negativos}} \]

F1-Score

  • Definición: La media armónica de la precisión y el recall.
  • Fórmula: \[ \text{F1-Score} = 2 \times \frac{\text{Precisión} \times \text{Recall}}{\text{Precisión} + \text{Recall}} \]

Ejemplo de Código

from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

# Supongamos que tenemos las siguientes etiquetas verdaderas y predicciones
y_true = [0, 1, 1, 0, 1, 0, 1, 1]
y_pred = [0, 1, 0, 0, 1, 0, 1, 1]

# Calcular las métricas
accuracy = accuracy_score(y_true, y_pred)
precision = precision_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)
f1 = f1_score(y_true, y_pred)

print(f"Exactitud: {accuracy}")
print(f"Precisión: {precision}")
print(f"Recall: {recall}")
print(f"F1-Score: {f1}")

Métricas para Modelos de Regresión

Error Absoluto Medio (MAE)

  • Definición: La media de las diferencias absolutas entre las predicciones y los valores reales.
  • Fórmula: \[ \text{MAE} = \frac{1}{n} \sum_{i=1}^{n} |y_i - \hat{y}_i| \]

Error Cuadrático Medio (MSE)

  • Definición: La media de los cuadrados de las diferencias entre las predicciones y los valores reales.
  • Fórmula: \[ \text{MSE} = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2 \]

Raíz del Error Cuadrático Medio (RMSE)

  • Definición: La raíz cuadrada del MSE.
  • Fórmula: \[ \text{RMSE} = \sqrt{\frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2} \]

Coeficiente de Determinación (R²)

  • Definición: La proporción de la varianza en la variable dependiente que es predecible a partir de las variables independientes.
  • Fórmula: \[ R^2 = 1 - \frac{\sum_{i=1}^{n} (y_i - \hat{y}i)^2}{\sum{i=1}^{n} (y_i - \bar{y})^2} \]

Ejemplo de Código

from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score
import numpy as np

# Supongamos que tenemos las siguientes etiquetas verdaderas y predicciones
y_true = np.array([3.0, -0.5, 2.0, 7.0])
y_pred = np.array([2.5, 0.0, 2.0, 8.0])

# Calcular las métricas
mae = mean_absolute_error(y_true, y_pred)
mse = mean_squared_error(y_true, y_pred)
rmse = np.sqrt(mse)
r2 = r2_score(y_true, y_pred)

print(f"MAE: {mae}")
print(f"MSE: {mse}")
print(f"RMSE: {rmse}")
print(f"R²: {r2}")

Comparación de Métricas

| Métrica | Tipo de Modelo | Interpretación | |---------------|-----------------|-------------------------------------------------------------------------------| | Exactitud | Clasificación | Proporción de predicciones correctas. | | Precisión | Clasificación | Proporción de verdaderos positivos sobre el total de predicciones positivas. | | Recall | Clasificación | Proporción de verdaderos positivos sobre el total de positivos reales. | | F1-Score | Clasificación | Media armónica de la precisión y el recall. | | MAE | Regresión | Media de las diferencias absolutas entre las predicciones y los valores reales. | | MSE | Regresión | Media de los cuadrados de las diferencias entre las predicciones y los valores reales. | | RMSE | Regresión | Raíz cuadrada del MSE. | | R² | Regresión | Proporción de la varianza explicada por el modelo. |

Conclusión

Las métricas de evaluación son esenciales para entender el rendimiento de los modelos de Machine Learning. La elección de la métrica adecuada depende del tipo de problema y de los objetivos específicos del proyecto. Es fundamental conocer y aplicar correctamente estas métricas para desarrollar modelos efectivos y tomar decisiones informadas.
