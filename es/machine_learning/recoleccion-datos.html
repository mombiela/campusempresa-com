Introducción

La recolección de datos es una fase crucial en cualquier proyecto de Machine Learning. La calidad y cantidad de los datos recolectados pueden determinar el éxito o fracaso de un modelo. En esta sección, aprenderemos sobre las diferentes fuentes de datos, métodos de recolección, y cómo preparar los datos para su uso en modelos de Machine Learning.

Fuentes de Datos

Existen diversas fuentes de datos que pueden ser utilizadas en proyectos de Machine Learning. Algunas de las más comunes incluyen:

  • Bases de datos relacionales: MySQL, PostgreSQL.
  • APIs: Twitter API, Google Maps API.
  • Archivos: CSV, JSON, Excel.
  • Web Scraping: BeautifulSoup, Scrapy.
  • Sensores y dispositivos IoT: Sensores de temperatura, acelerómetros.

Ejemplo de Conexión a una Base de Datos Relacional

import mysql.connector

# Conexión a la base de datos
conn = mysql.connector.connect(
    host="localhost",
    user="tu_usuario",
    password="tu_contraseña",
    database="nombre_de_la_base_de_datos"
)

# Creación de un cursor para ejecutar consultas
cursor = conn.cursor()

# Ejecución de una consulta
cursor.execute("SELECT * FROM tu_tabla")

# Recuperación de los datos
resultados = cursor.fetchall()

# Cierre de la conexión
conn.close()

# Mostrar los resultados
for fila in resultados:
    print(fila)

Métodos de Recolección de Datos

Dependiendo del tipo de datos y la fuente, los métodos de recolección pueden variar. Aquí se presentan algunos de los métodos más comunes:

  • Extracción de datos de bases de datos: Utilizando SQL para consultar y extraer datos.
  • Consumo de APIs: Utilizando librerías como requests en Python para hacer peticiones HTTP.
  • Web Scraping: Utilizando herramientas como BeautifulSoup o Scrapy para extraer datos de páginas web.
  • Lectura de archivos: Utilizando librerías como pandas para leer archivos CSV, Excel, etc.

Ejemplo de Consumo de una API

import requests

# URL de la API
url = "https://api.example.com/data"

# Hacer una petición GET a la API
response = requests.get(url)

# Verificar el estado de la respuesta
if response.status_code == 200:
    # Parsear la respuesta JSON
    datos = response.json()
    print(datos)
else:
    print("Error en la petición:", response.status_code)

Preparación de Datos

Una vez recolectados los datos, es necesario prepararlos antes de utilizarlos en modelos de Machine Learning. La preparación de datos incluye:

  • Limpieza de datos: Eliminación de valores nulos, duplicados, y corrección de errores.
  • Transformación de datos: Normalización, estandarización, y codificación de variables categóricas.
  • División de datos: Separación en conjuntos de entrenamiento y prueba.

Ejemplo de Limpieza y Transformación de Datos

import pandas as pd
from sklearn.preprocessing import StandardScaler, OneHotEncoder

# Cargar datos desde un archivo CSV
df = pd.read_csv("datos.csv")

# Eliminar filas con valores nulos
df = df.dropna()

# Eliminar duplicados
df = df.drop_duplicates()

# Normalizar datos numéricos
scaler = StandardScaler()
df[['columna_numerica']] = scaler.fit_transform(df[['columna_numerica']])

# Codificar variables categóricas
encoder = OneHotEncoder()
categorical_data = encoder.fit_transform(df[['columna_categorica']]).toarray()

# Añadir las columnas codificadas al DataFrame original
df = df.join(pd.DataFrame(categorical_data, columns=encoder.get_feature_names_out()))

# Mostrar los primeros registros del DataFrame preparado
print(df.head())

Conclusión

La recolección de datos es una etapa fundamental en el desarrollo de proyectos de Machine Learning. La elección de las fuentes de datos y los métodos de recolección adecuados, así como una correcta preparación de los datos, son esenciales para construir modelos precisos y eficientes. En las próximas secciones del curso, profundizaremos en técnicas avanzadas de preparación de datos y en la implementación de modelos de Machine Learning.
