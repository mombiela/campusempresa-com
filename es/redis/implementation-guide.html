Introducción a Redis

Redis es una tienda de estructuras de datos en memoria de código abierto utilizada como base de datos, caché y corredor de mensajes. Soporta varias estructuras de datos como cadenas, hashes, listas, conjuntos y más. Esta guía te llevará desde los conceptos básicos de Redis hasta técnicas avanzadas de implementación.

Configuración de Redis

Instalación

  • Linux: 
    sudo apt update
sudo apt install redis-server
  • macOS: 
    brew install redis
  • Windows: Usa el Subsistema de Windows para Linux (WSL) o Docker.

Iniciando Redis

  • Linux/macOS: 
    redis-server
  • Windows (usando WSL): 
    redis-server

Conectándose a Redis

  • Usando Redis CLI: 
    redis-cli

Comandos Básicos de Redis

Cadenas

  • Establecer un valor de cadena: 
    SET key "value"
  • Obtener un valor de cadena: 
    GET key

Listas

  • Agregar a una lista: 
    LPUSH mylist "value1"
LPUSH mylist "value2"
  • Recuperar de una lista: 
    LRANGE mylist 0 -1

Hashes

  • Establecer un campo de hash: 
    HSET myhash field1 "value1"
  • Obtener un campo de hash: 
    HGET myhash field1

Conjuntos

  • Agregar a un conjunto: 
    SADD myset "value1"
SADD myset "value2"
  • Obtener todos los miembros de un conjunto: 
    SMEMBERS myset

Conceptos Intermedios de Redis

Expiración y TTL

  • Establecer una clave con expiración: 
    SETEX key 10 "value"  # La clave expira en 10 segundos
  • Verificar TTL de una clave: 
    TTL key

Transacciones

  • Iniciar una transacción: 
    MULTI
  • Ejecutar una transacción: 
    EXEC

Pub/Sub

  • Publicar un mensaje: 
    PUBLISH channel "message"
  • Suscribirse a un canal: 
    SUBSCRIBE channel

Técnicas Avanzadas de Redis

Persistencia de Redis

  • RDB (Copia de Seguridad de la Base de Datos Redis): Instantáneas periódicas de tu conjunto de datos.
  • AOF (Archivo de Solo Añadido): Registra cada operación de escritura recibida por el servidor.

Clustering de Redis

  • Configuración de un Cluster de Redis: 
    redis-cli --cluster create 127.0.0.1:7000 127.0.0.1:7001 127.0.0.1:7002 --cluster-replicas 1

Scripting en Lua

  • Ejecutar un script en Lua: 
    EVAL "return redis.call('SET', KEYS[1], ARGV[1])" 1 key value

Conclusión

Redis es una herramienta poderosa para gestionar estructuras de datos en memoria, ofreciendo alto rendimiento y flexibilidad. Desde comandos básicos hasta clustering avanzado y scripting, dominar Redis puede mejorar significativamente el rendimiento y la escalabilidad de tu aplicación. Practica los comandos y conceptos discutidos en esta guía para volverte competente en Redis.
