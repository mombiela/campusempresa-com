Introducción a Redis

Redis es una tienda de estructuras de datos en memoria, de código abierto, utilizada como base de datos, caché y corredor de mensajes. Soporta varias estructuras de datos como cadenas, hashes, listas, conjuntos y más. Este curso te guiará desde los conceptos básicos de Redis hasta temas avanzados, asegurando que tengas una comprensión completa de sus capacidades y aplicaciones.

Conceptos Clave

  • Almacenamiento en Memoria: Redis almacena datos en memoria, proporcionando operaciones de lectura y escritura extremadamente rápidas.
  • Estructuras de Datos: Redis soporta múltiples estructuras de datos como cadenas, listas, conjuntos, conjuntos ordenados, hashes, bitmaps, hyperloglogs e índices geoespaciales.
  • Persistencia: Redis puede persistir datos en disco, permitiendo la recuperación de datos después de un reinicio.
  • Replicación: Redis soporta replicación maestro-esclavo, permitiendo redundancia de datos y alta disponibilidad.
  • Transacciones: Redis soporta transacciones, permitiendo que múltiples comandos se ejecuten en una sola operación atómica.
  • Mensajería Pub/Sub: Redis proporciona un paradigma de mensajería de publicación/suscripción, útil para aplicaciones de mensajería en tiempo real.
  • Scripting en Lua: Redis permite scripting del lado del servidor utilizando Lua, proporcionando capacidades de manipulación de datos poderosas y flexibles.

Estructura del Curso

Nivel Principiante

  1. Introducción a Redis

    • ¿Qué es Redis?
    • Casos de uso y aplicaciones
    • Instalación de Redis

  2. Comandos Básicos

    • Conectarse a Redis
    • Operaciones CRUD con cadenas
    • Ejemplo: 
      SET key "value"
GET key
DEL key

  3. Trabajando con Estructuras de Datos

    • Listas
    • Conjuntos
    • Hashes
    • Ejemplo: 
      LPUSH mylist "item1"
SADD myset "member1"
HSET myhash field1 "value1"

  4. Persistencia

    • Instantáneas RDB
    • AOF (Archivo de Solo Adición)
    • Configuración de persistencia

Nivel Intermedio

  1. Estructuras de Datos Avanzadas

    • Conjuntos ordenados
    • Bitmaps
    • HyperLogLogs
    • Ejemplo: 
      ZADD myzset 1 "one"
SETBIT mybitmap 7 1
PFADD myhyperloglog "element"

  2. Replicación y Alta Disponibilidad

    • Configuración de replicación maestro-esclavo
    • Configuración de Redis Sentinel para alta disponibilidad

  3. Transacciones y Scripting

    • Uso de MULTI/EXEC para transacciones
    • Introducción al scripting en Lua
    • Ejemplo: 
      MULTI
SET key1 "value1"
SET key2 "value2"
EXEC

Nivel Avanzado

  1. Escalando Redis

    • Particionamiento de datos con Redis Cluster
    • Configuración y gestión de un Redis Cluster

  2. Optimización del Rendimiento

    • Técnicas de optimización de memoria
    • Configuración de Redis para alto rendimiento

  3. Seguridad

    • Asegurando instancias de Redis
    • Autenticación y autorización
    • Ejemplo: 
      CONFIG SET requirepass "yourpassword"

  4. Monitoreo y Mantenimiento

    • Monitoreo del rendimiento de Redis
    • Estrategias de respaldo y restauración

Resumen

Este curso proporcionará una comprensión completa de Redis, desde operaciones básicas hasta configuraciones y optimizaciones avanzadas. Al final del curso, estarás equipado con el conocimiento para usar Redis de manera efectiva en varios escenarios, asegurando alto rendimiento y fiabilidad para tus aplicaciones.
