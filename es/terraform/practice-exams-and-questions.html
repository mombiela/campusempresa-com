Introducción

Los exámenes de práctica y las preguntas son esenciales para reforzar tu comprensión de los conceptos de Terraform y asegurarte de estar bien preparado para escenarios del mundo real. Esta sección proporciona una variedad de preguntas de práctica y exámenes simulados, que van desde niveles principiantes hasta avanzados, para ayudarte a probar tus conocimientos y habilidades.

Nivel Principiante

Conceptos Clave

  • Comandos básicos de Terraform
  • Comprensión de proveedores y recursos
  • Escribir y aplicar configuraciones simples de Terraform

Preguntas de Práctica

  1. ¿Qué es Terraform?

    • Una herramienta para construir, cambiar y versionar infraestructura de manera segura y eficiente.

  2. Escribe una configuración simple de Terraform para crear un bucket de AWS S3.

    provider "aws" {
  region = "us-west-2"
}

resource "aws_s3_bucket" "example" {
  bucket = "my-unique-bucket-name"
  acl    = "private"
}

  3. Explica el propósito del comando terraform init.

    • Inicializa un directorio de trabajo de Terraform descargando e instalando los plugins de proveedores necesarios.

Examen Simulado

  1. Opción Múltiple: ¿Qué comando se usa para aplicar los cambios necesarios para alcanzar el estado deseado de la configuración?

    • A) terraform plan
    • B) terraform apply
    • C) terraform init
    • D) terraform destroy
    • Respuesta: B) terraform apply

  2. Verdadero o Falso: El estado de Terraform se usa para mapear recursos del mundo real a tu configuración.

    • Respuesta: Verdadero

Nivel Intermedio

Conceptos Clave

  • Módulos y su uso
  • Gestión del estado
  • Fuentes de datos y salidas

Preguntas de Práctica

  1. ¿Qué es un módulo de Terraform y por qué es útil?

    • Un módulo es un contenedor para múltiples recursos que se usan juntos. Los módulos te permiten organizar y reutilizar código.

  2. Escribe una configuración de Terraform que use un módulo para crear una instancia de AWS EC2.

    module "ec2_instance" {
  source = "./modules/ec2_instance"
  instance_type = "t2.micro"
  ami           = "ami-0c55b159cbfafe1f0"
}

  3. Explica el propósito del comando terraform state.

    • El comando terraform state se usa para gestionar el archivo de estado, que lleva un registro de los recursos gestionados por Terraform.

Examen Simulado

  1. Opción Múltiple: ¿Qué comando se usa para eliminar un recurso del estado de Terraform?

    • A) terraform destroy
    • B) terraform rm
    • C) terraform state rm
    • D) terraform delete
    • Respuesta: C) terraform state rm

  2. Verdadero o Falso: Las fuentes de datos en Terraform se usan para obtener información de fuentes externas.

    • Respuesta: Verdadero

Nivel Avanzado

Conceptos Clave

  • Gestión avanzada del estado (por ejemplo, estado remoto)
  • Características de Terraform Cloud y Enterprise
  • Escribir proveedores personalizados

Preguntas de Práctica

  1. ¿Qué es el estado remoto y por qué es importante?

    • El estado remoto te permite almacenar tu archivo de estado en una ubicación remota, lo cual es esencial para la colaboración y para asegurar que el archivo de estado no se pierda.

  2. Escribe una configuración de Terraform para almacenar el estado en un bucket de AWS S3.

    terraform {
  backend "s3" {
    bucket = "my-terraform-state"
    key    = "path/to/my/key"
    region = "us-west-2"
  }
}

  3. Explica cómo escribir un proveedor personalizado en Terraform.

    • Escribir un proveedor personalizado implica usar el SDK de Plugins de Terraform para crear un nuevo proveedor que pueda gestionar recursos no soportados por los proveedores existentes. Esto incluye definir el esquema del proveedor, el esquema de recursos e implementar operaciones CRUD.

Examen Simulado

  1. Opción Múltiple: ¿Cuál de las siguientes es una característica de Terraform Enterprise?

    • A) Almacenamiento remoto del estado
    • B) Políticas como código
    • C) Integración con control de versiones
    • D) Todas las anteriores
    • Respuesta: D) Todas las anteriores

  2. Verdadero o Falso: Terraform Cloud proporciona un entorno colaborativo para que los equipos gestionen infraestructura.

    • Respuesta: Verdadero

Conclusión

Los exámenes de práctica y las preguntas son una parte vital para dominar Terraform. Al trabajar en estos ejercicios, puedes solidificar tu comprensión de los conceptos clave, mejorar tus habilidades para resolver problemas y prepararte para desafíos de infraestructura del mundo real. Ya seas un usuario principiante, intermedio o avanzado, la práctica continua te ayudará a volverte competente en el uso de Terraform para gestionar infraestructura como código.
