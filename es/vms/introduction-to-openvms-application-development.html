Visión General

OpenVMS (Open Virtual Memory System) es un sistema operativo de memoria virtual basado en multiprocesamiento y multiusuario, diseñado para su uso en procesamiento por lotes, procesamiento de transacciones y uso compartido de tiempo. Este curso te guiará a través de los conceptos básicos del desarrollo de aplicaciones en OpenVMS, desde niveles principiantes hasta avanzados.

Comenzando con OpenVMS

¿Qué es OpenVMS?

  • Multiusuario y multiprocesamiento: Soporta múltiples usuarios y procesos simultáneamente.
  • Memoria virtual: Gestión eficiente de la memoria.
  • Alta disponibilidad: Diseñado para aplicaciones críticas.
  • Seguridad: Robustas características de seguridad.

Configurando Tu Entorno

  • Accediendo a OpenVMS: Usando emuladores de terminal o SSH.
  • Comandos Básicos: 
    $ SHOW SYSTEM
$ DIRECTORY
$ TYPE filename.txt

Operaciones Básicas de Archivos

  • Creando Archivos: 
    $ CREATE filename.txt
  • Editando Archivos: Usando editores como EDT o EVE. 
    $ EDIT/TPU filename.txt
  • Eliminando Archivos: 
    $ DELETE filename.txt

Programación en OpenVMS

Lenguajes Soportados

  • C/C++
  • Fortran
  • Pascal
  • COBOL
  • BASIC

Escribiendo Tu Primer Programa en C

  • Creando un Archivo Fuente: 
    $ CREATE hello.c
  • Programa de Ejemplo en C: 
    #include <stdio.h>

int main() {
    printf("¡Hola, OpenVMS!\n");
    return 0;
}
  • Compilando y Ejecutando: 
    $ CC hello.c
$ LINK hello
$ RUN hello

Temas Avanzados

Servicios del Sistema

  • Visión General: Los servicios del sistema son rutinas invocables proporcionadas por OpenVMS para varias operaciones a nivel del sistema.
  • Ejemplo: 
    #include <lib$routines.h>
#include <starlet.h>

int main() {
    unsigned long status;
    status = lib$put_output(&"¡Hola, Servicio del Sistema OpenVMS!\n");
    return status;
}

Comunicación entre Procesos

  • Buzones: Usados para la comunicación entre procesos. 
    $ CREATE/MAILBOX MBX1
$ WRITE MBX1 "Mensaje a otro proceso"

Uso de RMS (Servicios de Gestión de Registros)

  • Visión General: RMS se usa para la gestión de archivos y registros.
  • Ejemplo: 
    #include <rms.h>

struct FAB fab;
struct RAB rab;
char record[100];

int main() {
    fab = cc$rms_fab;
    fab.fab$l_fna = "example.dat";
    fab.fab$b_fns = strlen(fab.fab$l_fna);
    fab.fab$b_fac = FAB$M_BIO | FAB$M_GET;
    fab.fab$b_shr = FAB$M_SHRGET | FAB$M_SHRPUT;

    rab = cc$rms_rab;
    rab.rab$l_fab = &fab;
    rab.rab$l_rop = RAB$M_RAH | RAB$M_RRL;
    rab.rab$l_ubf = record;
    rab.rab$w_usz = sizeof(record);

    sys$open(&fab);
    sys$connect(&rab);
    sys$get(&rab);
    printf("Registro: %s\n", record);
    sys$disconnect(&rab);
    sys$close(&fab);

    return 0;
}

Conclusión

OpenVMS es un sistema operativo poderoso y versátil adecuado para varios tipos de aplicaciones. Este curso te ha introducido a los conceptos básicos del desarrollo de aplicaciones en OpenVMS, desde la configuración de tu entorno hasta la escritura y ejecución de programas. A medida que progreses, explorarás temas más avanzados como servicios del sistema, comunicación entre procesos y RMS. Con práctica y exploración, te volverás competente en el desarrollo de aplicaciones robustas en OpenVMS.
