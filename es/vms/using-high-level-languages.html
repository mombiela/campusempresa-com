Introducción

OpenVMS es un sistema operativo versátil que soporta varios lenguajes de programación de alto nivel, incluyendo C y Fortran. Esta sección te guiará desde los conceptos básicos hasta el uso avanzado de estos lenguajes en OpenVMS, proporcionando explicaciones claras, ejemplos y ejercicios.

Comenzando con Lenguajes de Alto Nivel en OpenVMS

Configurando Tu Entorno

  • Instala los compiladores necesarios: Asegúrate de que los compiladores de C y Fortran estén instalados en tu sistema OpenVMS.
  • Configura las variables de entorno: Configura tu entorno para usar los compiladores de manera efectiva. 
    $ SET DEFAULT SYS$SYSTEM
$ DEFINE CC$COMPILER CC
$ DEFINE FORTRAN$COMPILER FORTRAN

Escribiendo Tu Primer Programa

Ejemplo en C

  1. Crea un programa simple en C: 
    #include <stdio.h>

int main() {
    printf("¡Hola, OpenVMS!\n");
    return 0;
}
  2. Guarda el archivo como hello.c.

Ejemplo en Fortran

  1. Crea un programa simple en Fortran: 
    PROGRAM HELLO
PRINT *, '¡Hola, OpenVMS!'
END PROGRAM HELLO
  2. Guarda el archivo como hello.f90.

Compilando y Ejecutando Programas

  • Compila el programa en C: 
    $ CC hello.c
$ LINK hello
$ RUN hello
  • Compila el programa en Fortran: 
    $ FORTRAN hello.f90
$ LINK hello
$ RUN hello

Conceptos Intermedios

Manejo de Entrada y Salida

Ejemplo en C

  • Leyendo entrada del usuario: 
    #include <stdio.h>

int main() {
    char name[50];
    printf("Introduce tu nombre: ");
    scanf("%s", name);
    printf("¡Hola, %s!\n", name);
    return 0;
}

Ejemplo en Fortran

  • Leyendo entrada del usuario: 
    PROGRAM GREET
CHARACTER(LEN=50) :: name
PRINT *, 'Introduce tu nombre:'
READ *, name
PRINT *, '¡Hola, ', TRIM(name), '!'
END PROGRAM GREET

Operaciones con Archivos

Ejemplo en C

  • Leyendo y escribiendo en un archivo: 
    #include <stdio.h>

int main() {
    FILE *file;
    file = fopen("example.txt", "w");
    fprintf(file, "¡Hola, OpenVMS!\n");
    fclose(file);

    file = fopen("example.txt", "r");
    char line[100];
    fgets(line, sizeof(line), file);
    printf("%s", line);
    fclose(file);
    return 0;
}

Ejemplo en Fortran

  • Leyendo y escribiendo en un archivo: 
    PROGRAM FILE_IO
INTEGER :: unit = 10
CHARACTER(LEN=100) :: line

OPEN(UNIT=unit, FILE='example.txt', STATUS='NEW')
WRITE(unit, *) '¡Hola, OpenVMS!'
CLOSE(unit)

OPEN(UNIT=unit, FILE='example.txt', STATUS='OLD')
READ(unit, *) line
PRINT *, TRIM(line)
CLOSE(unit)
END PROGRAM FILE_IO

Temas Avanzados

Interfaz entre C y Fortran

  • Llamando a Fortran desde C: 
    extern void fortran_subroutine();

int main() {
    fortran_subroutine();
    return 0;
}
  • Subrutina en Fortran: 
    SUBROUTINE FORTRAN_SUBROUTINE
PRINT *, '¡Llamado desde C!'
END SUBROUTINE FORTRAN_SUBROUTINE

Optimización de Rendimiento

  • Optimización del compilador: Usa las banderas del compilador para optimizar el rendimiento.
    • C: CC /OPTIMIZE=LEVEL=4 hello.c
    • Fortran: FORTRAN /OPTIMIZE=LEVEL=4 hello.f90

Técnicas de Depuración

  • Usando el Depurador de OpenVMS:
    • Inicia el depurador: 
      $ DEBUG hello
    • Establece puntos de interrupción y avanza paso a paso por el código: 
      DBG> SET BREAK main
DBG> GO
DBG> STEP

Conclusión

En esta sección, has aprendido cómo usar lenguajes de alto nivel como C y Fortran en OpenVMS, desde configurar tu entorno hasta escribir, compilar y ejecutar programas. También has explorado temas intermedios y avanzados, incluyendo operaciones con archivos, interfaz entre lenguajes, optimización de rendimiento y técnicas de depuración. Practicando estos conceptos, te volverás competente en el desarrollo de aplicaciones robustas en la plataforma OpenVMS.
