Las funciones léxicas en OpenVMS son funciones integradas que proporcionan una forma de recuperar información del sistema, manipular cadenas y realizar varias otras tareas. Son una parte esencial de la escritura de scripts en DCL (Digital Command Language) y pueden mejorar enormemente la funcionalidad y eficiencia de tus scripts.

Introducción a las Funciones Léxicas

  • Definición: Las funciones léxicas son funciones especiales en OpenVMS que devuelven información sobre el sistema, el entorno o realizan operaciones específicas.
  • Sintaxis: Las funciones léxicas generalmente están prefijadas con F$ y se utilizan dentro de scripts DCL.

Funciones Léxicas Básicas

Comencemos con algunas funciones léxicas básicas que se usan comúnmente:

  1. F$GETSYI: Recupera información del sistema.
  2. F$GETJPI: Recupera información del trabajo/proceso.
  3. F$TRNLNM: Traduce nombres lógicos.
  4. F$FAO: Formatea la salida.

Ejemplo: Uso de F$GETSYI

La función F$GETSYI se usa para obtener información del sistema. Aquí hay un ejemplo simple:

$ SYSTEM_NAME = F$GETSYI("NODENAME")
$ WRITE SYS$OUTPUT "Nombre del Sistema: ''SYSTEM_NAME'"
  • Explicación:
    • F$GETSYI("NODENAME"): Recupera el nombre del nodo del sistema.
    • WRITE SYS$OUTPUT "Nombre del Sistema: ''SYSTEM_NAME'": Muestra el nombre del sistema en la consola.

Funciones de Manipulación de Cadenas

Las funciones léxicas también se pueden usar para manipular cadenas. Algunas funciones comunes de manipulación de cadenas incluyen:

  1. F$EXTRACT: Extrae una subcadena de una cadena.
  2. F$LENGTH: Devuelve la longitud de una cadena.
  3. F$LOCATE: Encuentra la posición de una subcadena dentro de una cadena.
  4. F$EDIT: Edita una cadena según reglas especificadas.

Ejemplo: Uso de F$EXTRACT y F$LENGTH

$ FULL_NAME = "OpenVMS User"
$ FIRST_NAME = F$EXTRACT(0, 7, FULL_NAME)
$ NAME_LENGTH = F$LENGTH(FULL_NAME)
$ WRITE SYS$OUTPUT "Primer Nombre: ''FIRST_NAME'"
$ WRITE SYS$OUTPUT "Longitud del Nombre: ''NAME_LENGTH'"
  • Explicación:
    • F$EXTRACT(0, 7, FULL_NAME): Extrae los primeros 7 caracteres de FULL_NAME.
    • F$LENGTH(FULL_NAME): Devuelve la longitud de la cadena FULL_NAME.
    • WRITE SYS$OUTPUT "Primer Nombre: ''FIRST_NAME'": Muestra el primer nombre.
    • WRITE SYS$OUTPUT "Longitud del Nombre: ''NAME_LENGTH'": Muestra la longitud del nombre completo.

Funciones Léxicas Avanzadas

Para un uso más avanzado, las funciones léxicas pueden interactuar con procesos del sistema y nombres lógicos.

Ejemplo: Uso de F$GETJPI y F$TRNLNM

$ PROCESS_ID = F$GETJPI("","PID")
$ LOGICAL_NAME = F$TRNLNM("SYS$LOGIN")
$ WRITE SYS$OUTPUT "ID del Proceso: ''PROCESS_ID'"
$ WRITE SYS$OUTPUT "Nombre Lógico SYS$LOGIN: ''LOGICAL_NAME'"
  • Explicación:
    • F$GETJPI("","PID"): Recupera el ID del proceso del proceso actual.
    • F$TRNLNM("SYS$LOGIN"): Traduce el nombre lógico SYS$LOGIN a su equivalencia.
    • WRITE SYS$OUTPUT "ID del Proceso: ''PROCESS_ID'": Muestra el ID del proceso.
    • WRITE SYS$OUTPUT "Nombre Lógico SYS$LOGIN: ''LOGICAL_NAME'": Muestra el nombre lógico traducido.

Ejercicios Prácticos

  1. Recuperar y Mostrar el Tiempo de Actividad del Sistema:

    • Usa F$GETSYI para recuperar el tiempo de actividad del sistema y mostrarlo.
    $ UPTIME = F$GETSYI("BOOTTIME")
$ WRITE SYS$OUTPUT "Tiempo de Actividad del Sistema: ''UPTIME'"

  2. Extraer y Mostrar Información del Usuario:

    • Usa F$GETJPI para recuperar el nombre de usuario actual y mostrarlo.
    $ USERNAME = F$GETJPI("","USERNAME")
$ WRITE SYS$OUTPUT "Usuario Actual: ''USERNAME'"

  3. Traducir y Mostrar Nombres Lógicos:

    • Usa F$TRNLNM para traducir un nombre lógico de tu elección y mostrarlo.
    $ LOGICAL = F$TRNLNM("SYS$SCRATCH")
$ WRITE SYS$OUTPUT "SYS$SCRATCH: ''LOGICAL'"

Conclusión

Las funciones léxicas en OpenVMS son herramientas poderosas que pueden mejorar enormemente tus scripts DCL. Te permiten recuperar información del sistema y del proceso, manipular cadenas y trabajar con nombres lógicos. Al dominar estas funciones, puedes escribir scripts más eficientes y efectivos para automatizar tareas y gestionar tu entorno OpenVMS. Practica usando estas funciones con los ejercicios proporcionados para solidificar tu comprensión y mejorar tus habilidades de scripting.
