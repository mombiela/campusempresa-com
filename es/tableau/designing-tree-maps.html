Introducción a los Mapas de Árbol

Los mapas de árbol son una herramienta de visualización poderosa en Tableau que te permite mostrar datos jerárquicos usando rectángulos anidados. Cada rectángulo representa una categoría, y su tamaño y color pueden usarse para transmitir información adicional.

Conceptos Clave

  • Datos Jerárquicos: Datos que están organizados en una estructura similar a un árbol con relaciones de padre-hijo.
  • Rectángulos: Cada categoría o subcategoría está representada por un rectángulo.
  • Tamaño y Color: El tamaño del rectángulo puede representar una medida (por ejemplo, ventas), mientras que el color puede representar otra medida (por ejemplo, beneficio).

Creando Tu Primer Mapa de Árbol

Para crear un mapa de árbol básico en Tableau, sigue estos pasos:

  1. Conectar a los Datos: Abre Tableau y conéctate a tu fuente de datos.
  2. Arrastrar y Soltar: Arrastra una dimensión al estante de Filas y una medida al estante de Columnas.
  3. Cambiar a Mapa de Árbol: Haz clic en el panel "Show Me" y selecciona la opción de Mapa de Árbol.

Ejemplo

Supongamos que tienes un conjunto de datos de ventas con las siguientes columnas: Categoría, Subcategoría, Ventas y Beneficio.

Categoría    Subcategoría    Ventas    Beneficio
Muebles      Sillas          5000      200
Muebles      Mesas           3000      150
Oficina      Papel           4000      100
Oficina      Carpetas        2000      50

Para crear un mapa de árbol:

  1. Arrastra Categoría al estante de Filas.
  2. Arrastra Ventas al estante de Columnas.
  3. Haz clic en "Show Me" y selecciona la opción de Mapa de Árbol.

Personalizando Mapas de Árbol

Una vez que hayas creado un mapa de árbol básico, puedes personalizarlo para que se ajuste mejor a tus necesidades.

Ajustando Tamaño y Color

  • Tamaño: Arrastra una medida (por ejemplo, Ventas) al estante de Tamaño para ajustar el tamaño de los rectángulos.
  • Color: Arrastra una medida (por ejemplo, Beneficio) al estante de Color para ajustar el color de los rectángulos.

Añadiendo Etiquetas

  • Arrastra la dimensión (por ejemplo, Subcategoría) al estante de Etiqueta para añadir etiquetas a cada rectángulo.

Ejemplo

Para mejorar el ejemplo anterior:

  1. Arrastra Ventas al estante de Tamaño.
  2. Arrastra Beneficio al estante de Color.
  3. Arrastra Subcategoría al estante de Etiqueta.

Técnicas Avanzadas de Mapas de Árbol

Para usuarios avanzados, Tableau ofrece varias técnicas para mejorar aún más los mapas de árbol.

Usando Campos Calculados

Puedes crear campos calculados para añadir lógica más compleja a tus mapas de árbol.

Ejemplo

Crea un campo calculado para categorizar márgenes de beneficio:

IF [Beneficio] > 100 THEN "Alto"
ELSE "Bajo"
END

Arrastra este campo calculado al estante de Color para codificar con colores los rectángulos basados en los márgenes de beneficio.

Usando Jerarquías

Tableau te permite crear jerarquías para organizar mejor tus datos.

Ejemplo

  1. Haz clic derecho en Categoría y selecciona "Jerarquía" > "Crear Jerarquía".
  2. Añade Subcategoría a la jerarquía.
  3. Usa esta jerarquía en tu mapa de árbol para mostrar rectángulos anidados.

Conclusión

Los mapas de árbol son una forma versátil y poderosa de visualizar datos jerárquicos en Tableau. Al entender los conceptos básicos y explorar técnicas avanzadas, puedes crear mapas de árbol perspicaces y visualmente atractivos que comuniquen efectivamente la historia de tus datos. Practica creando y personalizando mapas de árbol con tus conjuntos de datos para volverte competente en esta técnica de visualización.
