Introducción a los Filtros

Los filtros en Tableau son herramientas esenciales para refinar y enfocar tus visualizaciones de datos. Te permiten controlar qué puntos de datos se muestran, facilitando el análisis de subconjuntos específicos de tus datos.

Conceptos Clave

  • Tipos de Filtros: Comprende los diferentes tipos de filtros disponibles en Tableau.

    • Filtros de Dimensión: Filtra datos basados en campos categóricos.
    • Filtros de Medida: Filtra datos basados en campos numéricos.
    • Filtros de Fecha: Filtra datos basados en campos de fecha.
    • Filtros de Fecha Relativa: Filtra datos basados en rangos de fechas relativas (por ejemplo, los últimos 7 días).
    • Filtros Top N: Filtra datos para mostrar solo los N elementos principales basados en una medida.
    • Filtros de Contexto: Crea un contexto para que otros filtros operen dentro de él.

  • Estante de Filtros: El área en Tableau donde se colocan y gestionan los filtros.

Aplicando Filtros Básicos

Filtros de Dimensión

Los filtros de dimensión te permiten incluir o excluir categorías específicas de tu visualización.

Ejemplo

Supongamos que tienes un conjunto de datos de ventas con una dimensión 'Región'. Para filtrar los datos para mostrar solo las ventas de la región 'Este':

  1. Arrastra el campo 'Región' al estante de Filtros.
  2. En el cuadro de diálogo del filtro, selecciona 'Este' y haz clic en 'OK'.

Filtros de Medida

Los filtros de medida te permiten filtrar datos basados en valores numéricos.

Ejemplo

Para filtrar los datos de ventas para mostrar solo las transacciones donde el monto de ventas sea mayor a $1000:

  1. Arrastra la medida 'Ventas' al estante de Filtros.
  2. En el cuadro de diálogo del filtro, establece el rango para comenzar desde 1000.

Filtros de Fecha

Los filtros de fecha se utilizan para filtrar datos basados en campos de fecha.

Ejemplo

Para filtrar los datos para mostrar solo las ventas del año 2022:

  1. Arrastra el campo 'Fecha de Pedido' al estante de Filtros.
  2. En el cuadro de diálogo del filtro, selecciona 'Años' y elige '2022'.

Técnicas Avanzadas de Filtrado

Filtros de Fecha Relativa

Los filtros de fecha relativa te permiten filtrar datos basados en un rango relativo a la fecha actual.

Ejemplo

Para filtrar los datos para mostrar las ventas de los últimos 30 días:

  1. Arrastra el campo 'Fecha de Pedido' al estante de Filtros.
  2. En el cuadro de diálogo del filtro, selecciona 'Fecha Relativa'.
  3. Elige 'Últimos' y establece el valor en 30 días.

Filtros Top N

Los filtros Top N se utilizan para mostrar los N elementos principales basados en una medida.

Ejemplo

Para mostrar los 10 productos principales por ventas:

  1. Arrastra el campo 'Nombre del Producto' al estante de Filtros.
  2. En el cuadro de diálogo del filtro, selecciona 'Top' y establece el valor en 10 basado en la medida 'Ventas'.

Filtros de Contexto

Los filtros de contexto crean un contexto para que otros filtros operen dentro de él, mejorando el rendimiento y refinando el alcance de otros filtros.

Ejemplo

Para establecer un filtro de contexto para una región específica y luego filtrar por los productos principales dentro de esa región:

  1. Arrastra el campo 'Región' al estante de Filtros y selecciona una región (por ejemplo, 'Oeste').
  2. Haz clic derecho en el filtro 'Región' y selecciona 'Agregar al Contexto'.
  3. Ahora, agrega un filtro 'Top N' para 'Nombre del Producto' para mostrar los 5 productos principales dentro de la región 'Oeste'.

Ejercicios Prácticos

Ejercicio 1: Filtro Básico de Dimensión

  1. Carga un conjunto de datos con múltiples dimensiones y medidas.
  2. Aplica un filtro de dimensión para mostrar datos de una categoría específica.

Ejercicio 2: Filtro de Medida

  1. Usa el mismo conjunto de datos.
  2. Aplica un filtro de medida para mostrar datos donde una medida específica exceda un cierto valor.

Ejercicio 3: Filtro de Fecha Relativa

  1. Usa un conjunto de datos con campos de fecha.
  2. Aplica un filtro de fecha relativa para mostrar datos de los últimos 7 días.

Ejercicio 4: Filtro Top N

  1. Usa un conjunto de datos con datos de ventas de productos.
  2. Aplica un filtro Top N para mostrar los 5 productos principales por ventas.

Ejercicio 5: Filtro de Contexto

  1. Usa un conjunto de datos con múltiples regiones y productos.
  2. Aplica un filtro de contexto para una región específica.
  3. Aplica un filtro Top N dentro del filtro de contexto para mostrar los productos principales en esa región.

Conclusión

Los filtros en Tableau son herramientas poderosas que te ayudan a refinar y enfocar tus visualizaciones de datos. Al comprender y aplicar diferentes tipos de filtros, puedes obtener conocimientos más profundos y crear visualizaciones más significativas. Practica el uso de filtros con varios conjuntos de datos para volverte competente en técnicas de filtrado y mejorar tus habilidades de análisis de datos.
