En esta sección, cubriremos el proceso de publicación de libros de trabajo en Tableau. Publicar libros de trabajo te permite compartir tus visualizaciones y paneles con otros, ya sea dentro de tu organización o públicamente en la web. Comenzaremos con lo básico y avanzaremos hacia temas más avanzados.

Introducción a la Publicación de Libros de Trabajo

Publicar un libro de trabajo en Tableau implica subir tu libro de trabajo a Tableau Server, Tableau Online o Tableau Public. Esto hace que tus visualizaciones sean accesibles para otros y permite la colaboración y el intercambio.

Conceptos Clave

  • Tableau Server: Una solución local para compartir libros de trabajo de Tableau dentro de una organización.
  • Tableau Online: Una solución basada en la nube para compartir libros de trabajo de Tableau.
  • Tableau Public: Una plataforma gratuita para compartir libros de trabajo de Tableau públicamente en la web.

Pasos para Publicar un Libro de Trabajo

1. Preparar tu Libro de Trabajo

Antes de publicar, asegúrate de que tu libro de trabajo esté listo:

  • Datos Limpios: Asegúrate de que tus datos estén limpios y sean precisos.
  • Visualizaciones Finalizadas: Asegúrate de que todas las visualizaciones estén completas y pulidas.
  • Verificar Permisos: Verifica que tengas los permisos necesarios para publicar.

2. Publicar en Tableau Server o Tableau Online

Para publicar un libro de trabajo en Tableau Server o Tableau Online, sigue estos pasos:

  1. Abrir Tableau Desktop: Abre el libro de trabajo que deseas publicar.

  2. Iniciar Sesión: Ve a Server > Sign In e ingresa tus credenciales de Tableau Server o Tableau Online.

  3. Publicar Libro de Trabajo: Ve a Server > Publish Workbook.

    • Proyecto: Selecciona el proyecto donde deseas publicar el libro de trabajo.
    • Nombre: Ingresa un nombre para el libro de trabajo.
    • Fuentes de Datos: Elige si deseas incrustar fuentes de datos o usar conexiones en vivo.
    • Permisos: Establece permisos para quién puede ver o editar el libro de trabajo.
    • Extractos: Decide si deseas crear extractos para un rendimiento más rápido.

  4. Publicar: Haz clic en Publish para subir el libro de trabajo.

3. Publicar en Tableau Public

Publicar en Tableau Public es similar pero con algunas diferencias:

  1. Abrir Tableau Desktop: Abre el libro de trabajo que deseas publicar.
  2. Iniciar Sesión: Ve a Server > Tableau Public > Save to Tableau Public e inicia sesión con tu cuenta de Tableau Public.
  3. Publicar Libro de Trabajo: Ingresa un nombre para el libro de trabajo y haz clic en Save.

Ejemplo

Aquí hay un ejemplo de cómo publicar un libro de trabajo en Tableau Server:

1. Abre Tableau Desktop y carga tu libro de trabajo.
2. Ve a `Server` > `Sign In`.
3. Ingresa tus credenciales de Tableau Server.
4. Ve a `Server` > `Publish Workbook`.
5. Selecciona el proyecto, nombra el libro de trabajo y establece los permisos.
6. Haz clic en `Publish`.

Gestión de Libros de Trabajo Publicados

Una vez que un libro de trabajo está publicado, puedes gestionarlo a través de Tableau Server o Tableau Online:

  • Editar: Realiza cambios en el libro de trabajo directamente en el servidor.
  • Permisos: Ajusta quién puede ver o editar el libro de trabajo.
  • Control de Versiones: Mantén un seguimiento de los cambios y vuelve a versiones anteriores si es necesario.

Ejemplo

Para editar un libro de trabajo publicado:

  1. Iniciar Sesión: Ve a Tableau Server o Tableau Online e inicia sesión.
  2. Navegar: Encuentra el libro de trabajo que deseas editar.
  3. Editar: Haz clic en el libro de trabajo y selecciona Edit.

Mejores Prácticas para Publicar Libros de Trabajo

  • Optimizar el Rendimiento: Usa extractos y optimiza consultas para asegurar un rendimiento rápido.
  • Establecer Permisos Apropiados: Asegúrate de que solo las personas adecuadas tengan acceso para ver o editar el libro de trabajo.
  • Actualizaciones Regulares: Mantén tus libros de trabajo actualizados con los últimos datos y visualizaciones.

Conclusión

Publicar libros de trabajo en Tableau es un paso crucial para compartir tus conocimientos y colaborar con otros. Ya sea que estés usando Tableau Server, Tableau Online o Tableau Public, comprender el proceso de publicación y las mejores prácticas te ayudará a compartir tu trabajo de manera efectiva. Recuerda preparar tu libro de trabajo, establecer permisos apropiados y gestionar tus libros de trabajo publicados para asegurar que sigan siendo útiles y relevantes.
