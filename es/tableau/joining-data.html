Unir datos es una habilidad crucial en Tableau que te permite combinar datos de diferentes fuentes para crear visualizaciones completas y perspicaces. Este tema te guiará desde los conceptos básicos de la unión de datos hasta técnicas más avanzadas.

Introducción a las Uniones de Datos

  • Definición: Una unión es un método para combinar datos de dos o más tablas basándose en una columna relacionada.
  • Propósito: Enriquecer tu conjunto de datos incorporando campos adicionales de otras tablas.

Tipos de Uniones

Tableau admite varios tipos de uniones, cada una con diferentes propósitos:

  • Unión Interna: Devuelve solo las filas donde hay una coincidencia en ambas tablas.
  • Unión Izquierda: Devuelve todas las filas de la tabla izquierda y las filas coincidentes de la tabla derecha. Si no se encuentra coincidencia, se devuelven valores NULL para las columnas de la tabla derecha.
  • Unión Derecha: Devuelve todas las filas de la tabla derecha y las filas coincidentes de la tabla izquierda. Si no se encuentra coincidencia, se devuelven valores NULL para las columnas de la tabla izquierda.
  • Unión Externa Completa: Devuelve todas las filas cuando hay una coincidencia en una de las tablas. Las filas sin coincidencia en una de las tablas tendrán valores NULL para las columnas de la tabla sin coincidencia.

Ejemplo de Estructuras de Tablas

| Employee_ID | Employee_Name | Department_ID | |-------------|---------------|---------------| | 1 | Alice | 101 | | 2 | Bob | 102 | | 3 | Charlie | 103 |

| Department_ID | Department_Name | |---------------|-----------------| | 101 | HR | | 102 | IT | | 104 | Finance |

Ejemplos de Uniones

Unión Interna

SELECT *
FROM Employees
INNER JOIN Departments
ON Employees.Department_ID = Departments.Department_ID;

| Employee_ID | Employee_Name | Department_ID | Department_Name | |-------------|---------------|---------------|-----------------| | 1 | Alice | 101 | HR | | 2 | Bob | 102 | IT |

Unión Izquierda

SELECT *
FROM Employees
LEFT JOIN Departments
ON Employees.Department_ID = Departments.Department_ID;

| Employee_ID | Employee_Name | Department_ID | Department_Name | |-------------|---------------|---------------|-----------------| | 1 | Alice | 101 | HR | | 2 | Bob | 102 | IT | | 3 | Charlie | 103 | NULL |

Realizar Uniones en Tableau

Guía Paso a Paso

  1. Cargar Fuentes de Datos: Importa las tablas que deseas unir en Tableau.
  2. Navegar a la Pestaña de Fuente de Datos: Ve a la pestaña de Fuente de Datos donde puedes ver las tablas.
  3. Arrastrar y Soltar: Arrastra una tabla al lienzo y luego arrastra la segunda tabla junto a ella.
  4. Configurar Unión: Tableau sugerirá automáticamente una unión basada en campos comunes. Puedes ajustar el tipo de unión y los campos si es necesario.

Ejemplo

  1. Cargar las tablas Employees y Departments.
  2. Arrastrar Employees al lienzo.
  3. Arrastrar Departments junto a Employees.
  4. Seleccionar Tipo de Unión: Elige el tipo de unión (por ejemplo, Unión Interna).
  5. Configurar Cláusulas de Unión: Asegúrate de que Employees.Department_ID coincida con Departments.Department_ID.

Técnicas Avanzadas de Uniones

  • SQL Personalizado: Usa consultas SQL personalizadas para realizar uniones más complejas.
  • Combinación de Datos: Una alternativa a la unión, útil cuando se trata de diferentes fuentes de datos que no se pueden unir directamente.
  • Unión: Combina filas de dos o más tablas con estructuras similares.

Ejemplo de SQL Personalizado

SELECT Employees.Employee_ID, Employees.Employee_Name, Departments.Department_Name
FROM Employees
LEFT JOIN Departments
ON Employees.Department_ID = Departments.Department_ID;

Combinación de Datos

  • Fuente de Datos Principal: La fuente de datos principal con la que estás trabajando.
  • Fuente de Datos Secundaria: La fuente de datos adicional que combinas con la principal.
  • Campos de Enlace: Campos utilizados para establecer una relación entre las fuentes de datos principal y secundaria.

Resumen

Unir datos en Tableau es una característica poderosa que te permite combinar información de múltiples tablas para crear conjuntos de datos más ricos. Comprender los diferentes tipos de uniones y cómo realizarlas en Tableau es esencial para crear visualizaciones completas. A medida que avances, puedes explorar SQL personalizado, combinación de datos y uniones para manejar escenarios de datos más complejos.
