Introducción a los Genéricos
Funciones Genéricas ►

Los genéricos son una característica poderosa en TypeScript que te permite crear componentes reutilizables y flexibles. Te permiten escribir funciones, clases e interfaces que funcionan con una variedad de tipos de datos mientras mantienen la seguridad de tipos. Esta sección te guiará a través de los conceptos básicos de los genéricos, su sintaxis y casos de uso prácticos.

¿Qué son los Genéricos?

Los genéricos proporcionan una forma de crear componentes que pueden trabajar con cualquier tipo de dato. Ayudan a hacer el código más flexible y reutilizable sin sacrificar la seguridad de tipos.

  • Seguridad de Tipos: Asegura que los tipos de las variables se conozcan en tiempo de compilación.
  • Reutilización: Permite que la misma función, clase o interfaz trabaje con diferentes tipos de datos.
  • Flexibilidad: Permite la creación de componentes más generales y abstractos.

Sintaxis Básica

La sintaxis básica para los genéricos implica el uso de corchetes angulares (< >) para definir un tipo genérico.

Funciones Genéricas

Una función genérica puede aceptar cualquier tipo de dato como argumento y devolver un valor del mismo tipo.

function identity<T>(arg: T): T {
    return arg;
}

let output1 = identity<string>("¡Hola, TypeScript!");
let output2 = identity<number>(42);
  • T es una variable de tipo que actúa como un marcador de posición para el tipo que se pasará a la función.
  • identity<string>("¡Hola, TypeScript!") especifica que T es string.
  • identity<number>(42) especifica que T es number.

Clases Genéricas

Las clases genéricas funcionan de manera similar a las funciones genéricas. Te permiten crear una clase que puede manejar diferentes tipos.

class GenericNumber<T> {
    zeroValue: T;
    add: (x: T, y: T) => T;
}

let myGenericNumber = new GenericNumber<number>();
myGenericNumber.zeroValue = 0;
myGenericNumber.add = (x, y) => x + y;
  • GenericNumber<T> es una clase genérica donde T es una variable de tipo.
  • myGenericNumber es una instancia de GenericNumber con T configurado como number.

Interfaces Genéricas

Las interfaces genéricas te permiten definir la forma de un objeto que puede trabajar con varios tipos.

interface GenericIdentityFn<T> {
    (arg: T): T;
}

function identity<T>(arg: T): T {
    return arg;
}

let myIdentity: GenericIdentityFn<number> = identity;
  • GenericIdentityFn<T> es una interfaz genérica.
  • myIdentity es una función que se adhiere a la interfaz GenericIdentityFn<number>.

Restricciones

A veces, necesitas imponer restricciones sobre los tipos que se pueden usar con genéricos. Esto se puede hacer usando la palabra clave extends.

interface Lengthwise {
    length: number;
}

function loggingIdentity<T extends Lengthwise>(arg: T): T {
    console.log(arg.length);
    return arg;
}

loggingIdentity({ length: 10, value: "Hola" });
  • T extends Lengthwise asegura que T tenga una propiedad length.
  • loggingIdentity ahora solo puede aceptar argumentos que tengan una propiedad length.

Uso de Múltiples Variables de Tipo

Puedes usar múltiples variables de tipo en una sola función, clase o interfaz genérica.

function swap<T, U>(tuple: [T, U]): [U, T] {
    return [tuple[1], tuple[0]];
}

let swappedTuple = swap([7, "siete"]); // [ "siete", 7 ]
  • T y U son variables de tipo.
  • La función swap intercambia los elementos de la tupla.

Casos de Uso Prácticos

Los genéricos se utilizan ampliamente en TypeScript para varios propósitos:

  • Colecciones: Arrays, listas enlazadas, pilas, colas.
  • Funciones Utilitarias: Ordenamiento, búsqueda, filtrado.
  • Estructuras de Datos: Árboles, grafos, mapas.

Ejemplo: Pila Genérica

class Stack<T> {
    private items: T[] = [];

    push(item: T): void {
        this.items.push(item);
    }

    pop(): T | undefined {
        return this.items.pop();
    }
}

let numberStack = new Stack<number>();
numberStack.push(10);
numberStack.push(20);
console.log(numberStack.pop()); // 20

let stringStack = new Stack<string>();
stringStack.push("Hola");
stringStack.push("Mundo");
console.log(stringStack.pop()); // "Mundo"
  • Stack<T> es una clase genérica.
  • numberStack y stringStack son instancias de Stack con diferentes tipos.

Conclusión

Los genéricos en TypeScript proporcionan una forma poderosa de crear componentes flexibles, reutilizables y seguros en cuanto a tipos. Al entender y utilizar los genéricos, puedes escribir código más abstracto y generalizado que funcione con una variedad de tipos de datos. Esto no solo mejora la reutilización del código, sino que también asegura la seguridad de tipos, haciendo que tus aplicaciones TypeScript sean más robustas y mantenibles.
