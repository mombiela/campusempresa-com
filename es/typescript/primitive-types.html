Tipos Primitivos en TypeScript

Introducción

Los tipos primitivos son los tipos de datos más básicos disponibles en TypeScript. Entender estos tipos es crucial ya que forman la base para estructuras de datos y operaciones más complejas. En esta sección, exploraremos los diferentes tipos primitivos disponibles en TypeScript, su uso y cómo trabajar con ellos de manera efectiva.

Tipos Primitivos Clave

Número

  • Representa tanto números enteros como de punto flotante.
  • Soporta literales hexadecimales, decimales, binarios y octales.
let decimal: number = 6;
let hex: number = 0xf00d;
let binary: number = 0b1010;
let octal: number = 0o744;

console.log(decimal); // Output: 6
console.log(hex); // Output: 61453
console.log(binary); // Output: 10
console.log(octal); // Output: 484

Cadena

  • Representa datos textuales.
  • Puede estar encerrado en comillas simples ('), comillas dobles (") o comillas invertidas (`) para literales de plantilla.
let color: string = "blue";
let sentence: string = `The color is ${color}`;

console.log(color); // Output: blue
console.log(sentence); // Output: The color is blue

Booleano

  • Representa un valor lógico: true o false.
let isDone: boolean = false;

console.log(isDone); // Output: false

Nulo e Indefinido

  • null e undefined son tipos que tienen solo un valor cada uno: null e undefined respectivamente.
  • Por defecto, null e undefined son subtipos de todos los demás tipos.
let u: undefined = undefined;
let n: null = null;

console.log(u); // Output: undefined
console.log(n); // Output: null

Símbolo

  • Representa un identificador único.
  • Creado usando la función Symbol.
let sym1 = Symbol();
let sym2 = Symbol("key");

console.log(sym1 === sym2); // Output: false

BigInt

  • Representa números enteros más grandes que el mayor número que JavaScript puede representar de manera confiable con el primitivo number.
  • Creado al agregar n al final de un entero o usando la función BigInt.
let big: bigint = 100n;

console.log(big); // Output: 100n

Comparación de Tipos Primitivos

| Tipo | Ejemplo | Descripción | |------------|-------------------|--------------------------------------| | number | let num = 42; | Representa valores numéricos | | string | let str = "Hi"; | Representa datos textuales | | boolean | let isTrue = true; | Representa valores lógicos | | null | let n = null; | Representa la ausencia intencional de cualquier valor de objeto | | undefined| let u; | Representa una variable no inicializada | | symbol | let sym = Symbol(); | Representa un identificador único | | bigint | let big = 100n; | Representa enteros grandes |

Aserciones de Tipo

TypeScript te permite anular el tipo inferido usando aserciones de tipo. Esto es útil cuando sabes más sobre el tipo de un valor que TypeScript.

let someValue: any = "this is a string";
let strLength: number = (someValue as string).length;

console.log(strLength); // Output: 16

Conclusión

Entender los tipos primitivos en TypeScript es esencial para cualquier programador. Estos tipos forman los bloques de construcción de tus aplicaciones, y saber cómo usarlos de manera efectiva te ayudará a escribir código más robusto y libre de errores. En esta sección, cubrimos los tipos primitivos básicos, su uso y cómo realizar aserciones de tipo. A medida que progreses, verás cómo estos conceptos fundamentales se integran con características más avanzadas de TypeScript.
