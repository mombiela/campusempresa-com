Introducción a los Módulos
Exportando e Importando Módulos ►

Los módulos son una parte fundamental de TypeScript y JavaScript moderno, permitiéndote organizar y gestionar tu código de manera más efectiva. Esta sección te introducirá al concepto de módulos, cómo crearlos y cómo usarlos en tus proyectos de TypeScript.

¿Qué son los Módulos?

Los módulos son piezas reutilizables de código que pueden ser importadas y exportadas entre diferentes partes de tu aplicación. Ayudan a organizar el código, evitar la contaminación del ámbito global y gestionar dependencias.

Conceptos Clave

  • Exportar: Hacer variables, funciones, clases o interfaces disponibles para otros módulos.
  • Importar: Traer variables, funciones, clases o interfaces de otros módulos al módulo actual.
  • Exportación por Defecto: Una única exportación de un módulo que puede ser importada sin especificar un nombre.
  • Exportación Nombrada: Múltiples exportaciones de un módulo que deben ser importadas usando sus nombres específicos.

Exportar e Importar

Exportar

Puedes exportar variables, funciones, clases o interfaces de un módulo usando la palabra clave export.

Ejemplo: Exportación Nombrada

// mathUtils.ts
export function add(a: number, b: number): number {
    return a + b;
}

export function subtract(a: number, b: number): number {
    return a - b;
}

Ejemplo: Exportación por Defecto

// logger.ts
export default function log(message: string): void {
    console.log(message);
}

Importar

Puedes importar variables, funciones, clases o interfaces de otros módulos usando la palabra clave import.

Ejemplo: Importando Exportaciones Nombradas

// main.ts
import { add, subtract } from './mathUtils';

console.log(add(5, 3)); // Salida: 8
console.log(subtract(5, 3)); // Salida: 2

Ejemplo: Importando Exportación por Defecto

// main.ts
import log from './logger';

log('Este es un mensaje de registro'); // Salida: Este es un mensaje de registro

Combinando Exportaciones Nombradas y por Defecto

Puedes combinar exportaciones nombradas y por defecto en un solo módulo.

// utils.ts
export function multiply(a: number, b: number): number {
    return a * b;
}

export default function divide(a: number, b: number): number {
    return a / b;
}
// main.ts
import divide, { multiply } from './utils';

console.log(multiply(6, 3)); // Salida: 18
console.log(divide(6, 3)); // Salida: 2

Re-exportar

Puedes re-exportar módulos para crear un único punto de entrada para múltiples módulos.

// mathOperations.ts
export { add, subtract } from './mathUtils';
export { multiply, default as divide } from './utils';
// main.ts
import { add, subtract, multiply, divide } from './mathOperations';

console.log(add(5, 3)); // Salida: 8
console.log(subtract(5, 3)); // Salida: 2
console.log(multiply(6, 3)); // Salida: 18
console.log(divide(6, 3)); // Salida: 2

Conclusión

Los módulos son una característica poderosa en TypeScript que te ayudan a organizar y gestionar tu código de manera más efectiva. Al entender cómo exportar e importar módulos, puedes crear aplicaciones más modulares, mantenibles y escalables. Practica creando y usando módulos en tus proyectos para familiarizarte con este concepto esencial.
