Introducción a GraphQL

  • ¿Qué es GraphQL?

    • Un lenguaje de consulta para APIs.
    • Permite a los clientes solicitar solo los datos que necesitan.
    • Desarrollado por Facebook.

  • Conceptos Clave:

    • Esquema: Define la estructura de los datos.
    • Consultas: Solicitudes para leer datos.
    • Mutaciones: Solicitudes para escribir datos.
    • Resolvers: Funciones que manejan la lógica para obtener datos.

Configuración de TypeScript con GraphQL

  • Requisitos Previos:

    • Node.js y npm instalados.
    • Comprensión básica de TypeScript.

  • Instalación de Paquetes Necesarios:

    npm install graphql express express-graphql @types/express @types/graphql typescript ts-node

  • Configuración de TypeScript:

    // tsconfig.json
{
  "compilerOptions": {
    "target": "ES6",
    "module": "commonjs",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}

Creación de un Servidor Básico de GraphQL con TypeScript

  • Configuración del Servidor Express: 
    import express from 'express';
import { graphqlHTTP } from 'express-graphql';
import { buildSchema } from 'graphql';

const app = express();

// Definir esquema de GraphQL
const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// Definir resolvers
const root = {
  hello: () => {
    return '¡Hola mundo!';
  },
};

// Configurar endpoint de GraphQL
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(4000, () => console.log('Servidor corriendo en http://localhost:4000/graphql'));

GraphQL Avanzado con TypeScript

  • Seguridad de Tipos con GraphQL:

    • Usa graphql-code-generator para generar tipos de TypeScript a partir de tu esquema de GraphQL.
    npm install @graphql-codegen/cli @graphql-codegen/typescript @graphql-codegen/typescript-resolvers

    • Configuración de Codegen:

      # codegen.yml
schema: "http://localhost:4000/graphql"
generates:
  src/generated/graphql.ts:
    plugins:
      - "typescript"
      - "typescript-resolvers"

    • Ejecutar Codegen:

      npx graphql-codegen

  • Uso de Tipos Generados:

    import { Resolvers } from './generated/graphql';

const resolvers: Resolvers = {
  Query: {
    hello: () => '¡Hola mundo!',
  },
};

Integración de GraphQL con una Base de Datos

  • Conexión a una Base de Datos:

    • Usa TypeORM para interacciones con la base de datos.
    npm install typeorm reflect-metadata sqlite3

    • Configuración de TypeORM:

      import "reflect-metadata";
import { createConnection } from "typeorm";
import { User } from "./entity/User";

createConnection().then(async connection => {
  console.log("Conectado a la base de datos");
}).catch(error => console.log(error));

    • Definición de Entidades:

      import { Entity, PrimaryGeneratedColumn, Column } from "typeorm";

@Entity()
export class User {
  @PrimaryGeneratedColumn()
  id: number;

  @Column()
  name: string;

  @Column()
  email: string;
}

    • Actualización de Resolvers para Usar la Base de Datos:

      import { getRepository } from "typeorm";
import { User } from "./entity/User";

const resolvers: Resolvers = {
  Query: {
    users: async () => {
      const userRepository = getRepository(User);
      return await userRepository.find();
    },
  },
  Mutation: {
    addUser: async (_, { name, email }) => {
      const userRepository = getRepository(User);
      const user = userRepository.create({ name, email });
      await userRepository.save(user);
      return user;
    },
  },
};
