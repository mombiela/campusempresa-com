Introducción a los Archivos de Declaración

Los archivos de declaración en TypeScript se utilizan para describir la forma del código JavaScript al compilador de TypeScript. Permiten que TypeScript entienda los tipos de objetos, funciones y otras entidades en las bibliotecas de JavaScript, habilitando la verificación de tipos y el soporte de IntelliSense.

  • Propósito: Proporcionar información de tipos sobre el código JavaScript.
  • Extensión de Archivo: .d.ts

Cuándo Usar Archivos de Declaración

  • Uso de Bibliotecas JavaScript: Al usar bibliotecas JavaScript de terceros que no tienen soporte TypeScript incorporado.
  • Interoperabilidad: Para asegurar una interoperabilidad fluida entre los códigos TypeScript y JavaScript.
  • Seguridad de Tipos: Para mejorar la seguridad de tipos y la experiencia del desarrollador con mejor soporte de herramientas.

Creación de Archivos de Declaración

Estructura Básica

Un archivo de declaración típicamente contiene declaraciones de tipos para variables, funciones, clases y módulos. Aquí hay un ejemplo básico:

// myLibrary.d.ts
declare module 'myLibrary' {
  export function greet(name: string): void;
  export const version: string;
}

Declaración de Variables y Funciones

  • Variables: Usa la palabra clave declare para declarar variables.
  • Funciones: Usa la palabra clave declare para declarar funciones con sus parámetros y tipos de retorno.
// variables.d.ts
declare const API_URL: string;
declare let isProduction: boolean;

// functions.d.ts
declare function add(a: number, b: number): number;
declare function log(message: string): void;

Declaración de Clases

Las clases pueden ser declaradas con sus propiedades y métodos:

// classes.d.ts
declare class Person {
  name: string;
  age: number;
  constructor(name: string, age: number);
  greet(): void;
}

Declaración de Módulos

Los módulos pueden ser declarados para encapsular tipos y valores:

// modules.d.ts
declare module 'myModule' {
  export interface User {
    id: number;
    name: string;
  }
  export function getUser(id: number): User;
}

Archivos de Declaración Avanzados

Fusión de Declaraciones

TypeScript permite fusionar múltiples declaraciones para extender tipos o módulos existentes:

// merging.d.ts
interface Window {
  customProperty: string;
}

declare module 'myLibrary' {
  export function newFunction(): void;
}

Declaraciones Globales

Las declaraciones globales se utilizan para definir tipos que están disponibles globalmente en todo el proyecto:

// globals.d.ts
declare global {
  interface String {
    toTitleCase(): string;
  }
}

Uso de Paquetes `@types`

Muchas bibliotecas populares de JavaScript tienen definiciones de tipos disponibles a través del ámbito @types en npm:

npm install @types/lodash

Estos pueden ser utilizados directamente sin necesidad de escribir tus propios archivos de declaración.

Ejemplos Prácticos

Ejemplo: Uso de un Archivo de Declaración para una Biblioteca JavaScript

Supongamos que estás usando una biblioteca JavaScript llamada awesomeLib:

// awesomeLib.d.ts
declare module 'awesomeLib' {
  export function awesomeFunction(param: string): number;
  export interface AwesomeInterface {
    id: number;
    name: string;
  }
}

En tu código TypeScript, ahora puedes usar awesomeLib con seguridad de tipos:

import { awesomeFunction, AwesomeInterface } from 'awesomeLib';

const result: number = awesomeFunction('test');
const obj: AwesomeInterface = { id: 1, name: 'Awesome' };

Ejemplo: Extensión de un Objeto Global

Extender el objeto Window para incluir una propiedad personalizada:

// globalExtensions.d.ts
declare global {
  interface Window {
    customProperty: string;
  }
}

// usage.ts
window.customProperty = 'Hello, World!';
console.log(window.customProperty);

Resumen

Los archivos de declaración son una característica poderosa en TypeScript que proporcionan información de tipos para el código JavaScript, mejorando la seguridad de tipos y la experiencia del desarrollador. Son esenciales al trabajar con bibliotecas JavaScript y para asegurar una interoperabilidad fluida entre TypeScript y JavaScript. Al entender cómo crear y usar archivos de declaración, puedes aprovechar todo el potencial de TypeScript en tus proyectos.
