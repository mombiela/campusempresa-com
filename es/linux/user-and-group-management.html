Introducción

La gestión de usuarios y grupos es un aspecto fundamental de la administración de sistemas Linux. Implica la creación, modificación y eliminación de cuentas de usuario y grupos, así como la gestión de permisos y controles de acceso.

Conceptos Clave

  • Usuarios: Cuentas individuales que pueden iniciar sesión en el sistema.
  • Grupos: Colecciones de usuarios que comparten permisos comunes.
  • UID (ID de Usuario): Un identificador único asignado a cada usuario.
  • GID (ID de Grupo): Un identificador único asignado a cada grupo.
  • Directorio Home: El directorio predeterminado asignado a un usuario.
  • Shell: La interfaz de línea de comandos asignada a un usuario.

Gestión de Usuarios

Creación de un Usuario

Para crear un nuevo usuario, use el comando useradd:

sudo useradd -m -s /bin/bash nombredeusuario
  • -m: Crea un directorio home para el usuario.
  • -s: Especifica el shell predeterminado del usuario.

Establecer una Contraseña

Establezca una contraseña para el nuevo usuario usando el comando passwd:

sudo passwd nombredeusuario

Modificación de un Usuario

Para modificar un usuario existente, use el comando usermod:

sudo usermod -s /bin/zsh nombredeusuario
  • -s: Cambia el shell predeterminado del usuario.

Eliminación de un Usuario

Para eliminar un usuario, use el comando userdel:

sudo userdel -r nombredeusuario
  • -r: Elimina el directorio home y el buzón de correo del usuario.

Gestión de Grupos

Creación de un Grupo

Para crear un nuevo grupo, use el comando groupadd:

sudo groupadd nombredegrupo

Añadir un Usuario a un Grupo

Para añadir un usuario a un grupo, use el comando usermod con la opción -aG:

sudo usermod -aG nombredegrupo nombredeusuario
  • -aG: Añade el usuario al grupo especificado.

Eliminar un Usuario de un Grupo

Para eliminar un usuario de un grupo, use el comando gpasswd:

sudo gpasswd -d nombredeusuario nombredegrupo

Eliminación de un Grupo

Para eliminar un grupo, use el comando groupdel:

sudo groupdel nombredegrupo

Visualización de Información de Usuarios y Grupos

Listado de Usuarios

Para listar todos los usuarios, vea el archivo /etc/passwd:

cat /etc/passwd

Listado de Grupos

Para listar todos los grupos, vea el archivo /etc/group:

cat /etc/group

Comprobación de la Membresía de un Grupo

Para comprobar a qué grupos pertenece un usuario, use el comando groups:

groups nombredeusuario

Ejemplos Prácticos

Ejemplo: Creación de un Nuevo Usuario y Añadirlo a un Grupo

  1. Crear un nuevo usuario: 
    sudo useradd -m -s /bin/bash alice
  2. Establecer una contraseña para el nuevo usuario: 
    sudo passwd alice
  3. Crear un nuevo grupo: 
    sudo groupadd developers
  4. Añadir el nuevo usuario al grupo: 
    sudo usermod -aG developers alice

Ejemplo: Modificación de Información de Usuario

  1. Cambiar el shell del usuario: 
    sudo usermod -s /bin/zsh alice

Ejemplo: Eliminación de un Usuario y un Grupo

  1. Eliminar el usuario y su directorio home: 
    sudo userdel -r alice
  2. Eliminar el grupo: 
    sudo groupdel developers

Resumen

La gestión de usuarios y grupos en Linux es esencial para mantener la seguridad y organización del sistema. Al entender cómo crear, modificar y eliminar usuarios y grupos, así como gestionar sus permisos, puede controlar efectivamente el acceso a los recursos del sistema. Practique estos comandos para volverse competente en la gestión de usuarios y grupos en un sistema Linux.
