Introducción

Una pila LAMP es una plataforma web de código abierto popular utilizada para ejecutar sitios web y servidores dinámicos. LAMP significa Linux, Apache, MySQL y PHP. Esta sección te guiará a través del proceso de configuración de una pila LAMP desde cero en un servidor Linux.

Prerrequisitos

Antes de comenzar, asegúrate de tener:

  • Un servidor Linux (se recomiendan distribuciones basadas en Ubuntu/Debian para este tutorial).
  • Conocimientos básicos de la línea de comandos de Linux.
  • Acceso root o sudo al servidor.

Paso 1: Actualiza tu Sistema

Primero, actualiza tu índice de paquetes y actualiza los paquetes de tu sistema a las versiones más recientes.

sudo apt update
sudo apt upgrade -y

Paso 2: Instalar Apache

Apache es el componente del servidor web de la pila LAMP.

Instalando Apache

sudo apt install apache2 -y

Iniciando y Habilitando Apache

Inicia el servicio de Apache y habilítalo para que se inicie al arrancar.

sudo systemctl start apache2
sudo systemctl enable apache2

Verificando la Instalación de Apache

Abre tu navegador web y navega a la dirección IP de tu servidor. Deberías ver la Página Predeterminada de Apache2 Ubuntu.

http://tu_ip_del_servidor

Paso 3: Instalar MySQL

MySQL es el sistema de gestión de bases de datos utilizado en la pila LAMP.

Instalando MySQL

sudo apt install mysql-server -y

Asegurando la Instalación de MySQL

Ejecuta el script de seguridad para configurar las opciones de seguridad de MySQL.

sudo mysql_secure_installation

Sigue las indicaciones para establecer una contraseña de root y asegurar tu instalación de MySQL.

Paso 4: Instalar PHP

PHP es el lenguaje de scripting utilizado para crear páginas web dinámicas.

Instalando PHP

sudo apt install php libapache2-mod-php php-mysql -y

Probando PHP

Crea un archivo de información PHP para probar tu instalación de PHP.

sudo nano /var/www/html/info.php

Añade el siguiente contenido al archivo:

<?php
phpinfo();
?>

Guarda y cierra el archivo. Luego, navega a http://tu_ip_del_servidor/info.php en tu navegador web. Deberías ver una página que muestra información de PHP.

Paso 5: Configurar Apache para Usar PHP

Por defecto, Apache servirá archivos index.html antes que archivos index.php. Para cambiar esto, edita el archivo dir.conf.

sudo nano /etc/apache2/mods-enabled/dir.conf

Mueve la directiva index.php al principio de la lista.

<IfModule mod_dir.c>
    DirectoryIndex index.php index.html index.cgi index.pl index.xhtml index.htm
</IfModule>

Guarda y cierra el archivo, luego reinicia Apache.

sudo systemctl restart apache2

Paso 6: Probando la Pila LAMP

Para asegurarte de que todo esté funcionando correctamente, crea un script PHP simple para conectarte a la base de datos MySQL.

Creando una Base de Datos y Usuario de Prueba

Inicia sesión en MySQL y crea una base de datos y un usuario de prueba.

sudo mysql -u root -p

Dentro del shell de MySQL, ejecuta los siguientes comandos:

CREATE DATABASE testdb;
CREATE USER 'testuser'@'localhost' IDENTIFIED BY 'password';
GRANT ALL PRIVILEGES ON testdb.* TO 'testuser'@'localhost';
FLUSH PRIVILEGES;
EXIT;

Creando un Script PHP para Probar la Conexión a la Base de Datos

Crea un nuevo archivo PHP para probar la conexión a la base de datos.

sudo nano /var/www/html/testdb.php

Añade el siguiente contenido al archivo:

<?php
$servername = "localhost";
$username = "testuser";
$password = "password";
$dbname = "testdb";

// Crear conexión
$conn = new mysqli($servername, $username, $password, $dbname);

// Verificar conexión
if ($conn->connect_error) {
    die("Conexión fallida: " . $conn->connect_error);
}
echo "Conectado exitosamente";
?>

Guarda y cierra el archivo. Navega a http://tu_ip_del_servidor/testdb.php en tu navegador web. Deberías ver "Conectado exitosamente".

Conclusión

¡Felicidades! Has configurado exitosamente una pila LAMP en tu servidor Linux. Esta pila proporciona un entorno robusto para desarrollar y desplegar aplicaciones web dinámicas. Ahora puedes proceder a desarrollar tus aplicaciones web utilizando PHP y MySQL, servidos por Apache en tu servidor Linux.
