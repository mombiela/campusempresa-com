Introducción

Apache Cordova permite a los desarrolladores construir aplicaciones móviles utilizando HTML, CSS y JavaScript. Una de las características más poderosas de Cordova es su capacidad para acceder a funcionalidades nativas del dispositivo a través de plugins. En esta sección, exploraremos cómo usar el Plugin de Cámara para capturar fotos y recuperar imágenes de la galería del dispositivo.

Configuración del Plugin de Cámara

Instalación del Plugin

Para usar el Plugin de Cámara, primero necesitas instalarlo en tu proyecto de Cordova. Abre tu terminal y navega a tu directorio de proyecto, luego ejecuta el siguiente comando:

cordova plugin add cordova-plugin-camera

Este comando añadirá el Plugin de Cámara a tu proyecto, permitiéndote acceder a la cámara del dispositivo y a la biblioteca de fotos.

Configuración de Permisos

Para Android e iOS, necesitas configurar los permisos necesarios en los archivos de configuración respectivos.

  • Android: Asegúrate de que el archivo AndroidManifest.xml incluya los siguientes permisos:
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />
  • iOS: Actualiza el archivo config.xml para incluir las siguientes preferencias:
<edit-config target="NSCameraUsageDescription" file="*-Info.plist" mode="merge">
    <string>Necesitamos acceso a tu cámara para tomar fotos.</string>
</edit-config>
<edit-config target="NSPhotoLibraryUsageDescription" file="*-Info.plist" mode="merge">
    <string>Necesitamos acceso a tu biblioteca de fotos para seleccionar fotos.</string>
</edit-config>

Captura de Fotos

Uso Básico

Para capturar una foto usando el Plugin de Cámara, puedes usar el método navigator.camera.getPicture. Aquí hay un ejemplo básico:

document.getElementById('capturePhoto').addEventListener('click', function() {
    navigator.camera.getPicture(onSuccess, onFail, {
        quality: 50,
        destinationType: Camera.DestinationType.DATA_URL
    });

    function onSuccess(imageData) {
        var image = document.getElementById('myImage');
        image.src = "data:image/jpeg;base64," + imageData;
    }

    function onFail(message) {
        alert('Falló porque: ' + message);
    }
});
  • Parámetros:
    • quality: Especifica la calidad de la imagen guardada (0-100).
    • destinationType: Determina el formato de la imagen devuelta (por ejemplo, DATA_URL devuelve una cadena codificada en base64).

Opciones Avanzadas

El método navigator.camera.getPicture acepta varias opciones para personalizar el comportamiento de la cámara:

  • sourceType: Especifica la fuente de la imagen (por ejemplo, Camera.PictureSourceType.CAMERA o Camera.PictureSourceType.PHOTOLIBRARY).
  • allowEdit: Permite una edición simple de la imagen antes de la selección.
  • targetWidth y targetHeight: Especifica el ancho y alto de la imagen.

Ejemplo con opciones avanzadas:

document.getElementById('capturePhoto').addEventListener('click', function() {
    navigator.camera.getPicture(onSuccess, onFail, {
        quality: 75,
        destinationType: Camera.DestinationType.FILE_URI,
        sourceType: Camera.PictureSourceType.CAMERA,
        allowEdit: true,
        targetWidth: 1000,
        targetHeight: 1000,
        saveToPhotoAlbum: true
    });

    function onSuccess(imageURI) {
        var image = document.getElementById('myImage');
        image.src = imageURI;
    }

    function onFail(message) {
        alert('Falló porque: ' + message);
    }
});

Recuperación de Imágenes de la Galería

Para recuperar imágenes de la biblioteca de fotos del dispositivo, puedes establecer el sourceType a Camera.PictureSourceType.PHOTOLIBRARY:

document.getElementById('getPhoto').addEventListener('click', function() {
    navigator.camera.getPicture(onSuccess, onFail, {
        quality: 50,
        destinationType: Camera.DestinationType.FILE_URI,
        sourceType: Camera.PictureSourceType.PHOTOLIBRARY
    });

    function onSuccess(imageURI) {
        var image = document.getElementById('myImage');
        image.src = imageURI;
    }

    function onFail(message) {
        alert('Falló porque: ' + message);
    }
});

Manejo de Errores

Es importante manejar los errores de manera adecuada. La función de callback onFail proporciona un mensaje de error que puede ser utilizado para informar al usuario sobre lo que salió mal.

function onFail(message) {
    alert('Falló porque: ' + message);
}

Conclusión

El Plugin de Cámara en Apache Cordova es una herramienta poderosa que te permite capturar fotos y recuperar imágenes de la galería del dispositivo. Al entender las opciones básicas y avanzadas disponibles, puedes crear una experiencia de usuario rica en tus aplicaciones móviles. Siempre recuerda manejar los errores de manera adecuada para asegurar una experiencia de usuario fluida.
