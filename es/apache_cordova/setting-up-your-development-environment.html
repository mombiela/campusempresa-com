Configurar tu entorno de desarrollo es el primer paso para comenzar a desarrollar aplicaciones usando Apache Cordova. Esta sección te guiará a través de las herramientas y configuraciones necesarias para preparar tu entorno para el desarrollo con Cordova.

Requisitos Previos

Antes de sumergirte en la configuración, asegúrate de tener los siguientes requisitos previos instalados en tu máquina:

  • Node.js y npm: Cordova está construido sobre Node.js, y npm (Node Package Manager) se usa para instalar Cordova y sus plugins.
  • Java Development Kit (JDK): Requerido para el desarrollo de Android.
  • Android Studio: Requerido para construir y ejecutar aplicaciones Android.
  • Xcode: Requerido para construir y ejecutar aplicaciones iOS (solo macOS).

Instalación de Node.js y npm

Node.js y npm son esenciales para el desarrollo con Cordova. Sigue estos pasos para instalarlos:

  1. Descargar Node.js: Visita el sitio web de Node.js y descarga la versión LTS para tu sistema operativo.
  2. Instalar Node.js: Ejecuta el instalador y sigue las instrucciones.
  3. Verificar la Instalación: Abre una terminal o símbolo del sistema y ejecuta los siguientes comandos para verificar la instalación: 
    node -v
npm -v
    Deberías ver los números de versión de Node.js y npm.

Instalación de Apache Cordova

Una vez que Node.js y npm están instalados, puedes instalar Apache Cordova usando npm:

  1. Instalar Cordova: Abre una terminal o símbolo del sistema y ejecuta:

    npm install -g cordova

    La bandera -g instala Cordova globalmente en tu sistema.

  2. Verificar la Instalación: Ejecuta el siguiente comando para verificar la instalación:

    cordova -v

    Deberías ver el número de versión de Cordova.

Configuración del Entorno de Desarrollo de Android

Para desarrollar aplicaciones Android con Cordova, necesitas configurar el entorno de desarrollo de Android:

  1. Instalar Java Development Kit (JDK):

    • Descarga el JDK desde el sitio web de Oracle.
    • Instala el JDK siguiendo las instrucciones para tu sistema operativo.
    • Configura la variable de entorno JAVA_HOME para que apunte al directorio de instalación del JDK.

  2. Instalar Android Studio:

    • Descarga Android Studio desde el sitio web oficial.
    • Instala Android Studio y sigue el asistente de configuración para instalar los componentes necesarios del SDK.

  3. Configurar el SDK de Android:

    • Abre Android Studio y ve a File > Settings > Appearance & Behavior > System Settings > Android SDK.
    • Asegúrate de que las plataformas y herramientas necesarias del SDK estén instaladas.

  4. Configurar Variables de Entorno:

    • Configura la variable de entorno ANDROID_HOME para que apunte al directorio del SDK de Android.
    • Agrega los directorios platform-tools y tools al PATH de tu sistema.

Configuración del Entorno de Desarrollo de iOS (solo macOS)

Para desarrollar aplicaciones iOS con Cordova, necesitas configurar el entorno de desarrollo de iOS:

  1. Instalar Xcode:

    • Descarga Xcode desde la Mac App Store.
    • Instala Xcode y ábrelo para completar la configuración inicial.

  2. Instalar las Herramientas de Línea de Comando de Xcode:

    • Abre una terminal y ejecuta: 
      xcode-select --install

  3. Configurar el Simulador de iOS:

    • Abre Xcode y ve a Preferences > Components.
    • Descarga e instala los simuladores de iOS necesarios.

Creación de tu Primer Proyecto Cordova

Con tu entorno de desarrollo configurado, ahora puedes crear tu primer proyecto Cordova:

  1. Crear un Nuevo Proyecto:

    • Abre una terminal o símbolo del sistema y ejecuta: 
      cordova create MyApp com.example.myapp MyApp
      Este comando crea un nuevo proyecto Cordova llamado MyApp con el nombre de paquete com.example.myapp.

  2. Agregar Plataformas:

    • Navega al directorio del proyecto: 
      cd MyApp
    • Agrega la plataforma Android: 
      cordova platform add android
    • (Opcional) Agrega la plataforma iOS (solo macOS): 
      cordova platform add ios

  3. Construir el Proyecto:

    • Construye el proyecto para Android: 
      cordova build android
    • (Opcional) Construye el proyecto para iOS (solo macOS): 
      cordova build ios

Conclusión

Configurar tu entorno de desarrollo para Apache Cordova implica instalar herramientas necesarias como Node.js, npm, JDK, Android Studio y Xcode. Una vez que estas herramientas están instaladas y configuradas, puedes crear y construir tu primer proyecto Cordova. Esta configuración forma la base para desarrollar aplicaciones móviles multiplataforma usando Cordova.
