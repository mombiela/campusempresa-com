Introducción

Las declaraciones condicionales son fundamentales en cualquier lenguaje de programación, incluido COBOL. Permiten que el programa tome decisiones y ejecute diferentes bloques de código basados en ciertas condiciones. En esta sección, cubriremos los conceptos básicos de las declaraciones condicionales en COBOL, progresando hacia conceptos más avanzados.

Declaraciones Condicionales Básicas

Declaración IF

La declaración IF es la forma más básica de declaración condicional en COBOL. Permite que el programa ejecute un bloque de código solo si una condición especificada es verdadera.

Sintaxis

IF condition
    statement-1
    statement-2
    ...
END-IF.

Ejemplo

IF AGE > 18
    DISPLAY 'You are an adult.'
END-IF.

En este ejemplo, el mensaje "You are an adult." se mostrará solo si el valor de AGE es mayor que 18.

Declaración IF-ELSE

La declaración IF-ELSE extiende la declaración IF proporcionando un bloque de código alternativo para ejecutar si la condición es falsa.

Sintaxis

IF condition
    statement-1
ELSE
    statement-2
END-IF.

Ejemplo

IF AGE > 18
    DISPLAY 'You are an adult.'
ELSE
    DISPLAY 'You are not an adult.'
END-IF.

En este ejemplo, el programa mostrará "You are an adult." si AGE es mayor que 18, de lo contrario, mostrará "You are not an adult."

Declaraciones IF Anidadas

Puedes anidar declaraciones IF dentro de otras para verificar múltiples condiciones.

Ejemplo

IF AGE > 18
    IF AGE < 65
        DISPLAY 'You are an adult.'
    ELSE
        DISPLAY 'You are a senior.'
    END-IF
ELSE
    DISPLAY 'You are not an adult.'
END-IF.

En este ejemplo, el programa verifica si AGE es mayor que 18 y menor que 65 para determinar si la persona es un adulto o un anciano.

Declaraciones Condicionales Avanzadas

Declaración EVALUATE

La declaración EVALUATE en COBOL es similar a la declaración SWITCH en otros lenguajes de programación. Permite evaluar múltiples condiciones de una manera más legible y organizada.

Sintaxis

EVALUATE TRUE
    WHEN condition-1
        statement-1
    WHEN condition-2
        statement-2
    ...
    WHEN OTHER
        statement-n
END-EVALUATE.

Ejemplo

EVALUATE TRUE
    WHEN AGE < 13
        DISPLAY 'You are a child.'
    WHEN AGE < 18
        DISPLAY 'You are a teenager.'
    WHEN AGE < 65
        DISPLAY 'You are an adult.'
    WHEN OTHER
        DISPLAY 'You are a senior.'
END-EVALUATE.

En este ejemplo, el programa evalúa el valor de AGE y muestra el mensaje apropiado basado en el rango en el que cae AGE.

Condiciones Complejas

Puedes usar operadores lógicos como AND y OR para formar condiciones complejas en declaraciones IF y EVALUATE.

Ejemplo

IF AGE > 18 AND AGE < 65
    DISPLAY 'You are an adult.'
ELSE
    DISPLAY 'You are either a minor or a senior.'
END-IF.

En este ejemplo, el programa verifica si AGE está entre 18 y 65 para determinar si la persona es un adulto.

Conclusión

Las declaraciones condicionales en COBOL son esenciales para controlar el flujo de un programa basado en diferentes condiciones. Desde las declaraciones IF básicas hasta las declaraciones EVALUATE más avanzadas, comprender estos constructos te permitirá escribir programas COBOL más dinámicos y flexibles. Practica escribiendo varias declaraciones condicionales para volverte competente en hacer que tus programas COBOL respondan a diferentes escenarios.
