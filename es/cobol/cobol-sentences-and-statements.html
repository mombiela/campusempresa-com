Introducción

COBOL (Common Business-Oriented Language) es un lenguaje de programación de alto nivel utilizado principalmente en sistemas de negocios, finanzas y administración. Entender las oraciones y declaraciones en COBOL es crucial para escribir programas COBOL efectivos. Esta sección cubrirá los conceptos básicos de las oraciones y declaraciones en COBOL, proporcionando explicaciones claras, ejemplos y ejercicios para reforzar el aprendizaje.

Oraciones en COBOL

Una oración en COBOL es una secuencia de una o más declaraciones que terminan con un punto (.). Las oraciones se utilizan para realizar tareas específicas dentro de un programa COBOL.

Conceptos Clave

  • Estructura de la Oración: Una oración consiste en una o más declaraciones y termina con un punto.
  • Propósito: Las oraciones se utilizan para agrupar declaraciones relacionadas que realizan una tarea específica.

Ejemplo

MOVE 'HELLO' TO GREETING.
DISPLAY GREETING.

En este ejemplo, hay dos declaraciones dentro de una oración. El punto después de DISPLAY GREETING marca el final de la oración.

Declaraciones en COBOL

Las declaraciones son los bloques básicos de construcción de los programas COBOL. Cada declaración realiza una operación específica, como mover datos, mostrar salida o realizar operaciones aritméticas.

Conceptos Clave

  • Declaración MOVE: Transfiere datos de un campo a otro.
  • Declaración DISPLAY: Muestra datos en la pantalla u otro dispositivo de salida.
  • Declaración COMPUTE: Realiza operaciones aritméticas y asigna el resultado a una variable.
  • Declaración IF: Se utiliza para el procesamiento condicional.
  • Declaración PERFORM: Ejecuta una serie de declaraciones repetidamente.

Declaración MOVE

La declaración MOVE transfiere datos de una variable a otra.

Ejemplo

MOVE 'HELLO' TO GREETING.

Esta declaración mueve la cadena 'HELLO' a la variable GREETING.

Declaración DISPLAY

La declaración DISPLAY muestra datos en la pantalla u otro dispositivo de salida.

Ejemplo

DISPLAY 'HELLO, WORLD'.

Esta declaración muestra la cadena 'HELLO, WORLD' en la pantalla.

Declaración COMPUTE

La declaración COMPUTE realiza operaciones aritméticas y asigna el resultado a una variable.

Ejemplo

COMPUTE TOTAL = PRICE * QUANTITY.

Esta declaración multiplica PRICE por QUANTITY y almacena el resultado en TOTAL.

Declaración IF

La declaración IF se utiliza para el procesamiento condicional.

Ejemplo

IF PRICE > 100
    DISPLAY 'EXPENSIVE ITEM'
ELSE
    DISPLAY 'AFFORDABLE ITEM'.

Esta declaración verifica si PRICE es mayor que 100. Si es verdadero, muestra 'EXPENSIVE ITEM'; de lo contrario, muestra 'AFFORDABLE ITEM'.

Declaración PERFORM

La declaración PERFORM ejecuta una serie de declaraciones repetidamente.

Ejemplo

PERFORM 10 TIMES
    DISPLAY 'HELLO, WORLD'.

Esta declaración muestra 'HELLO, WORLD' diez veces.

Resumen

En esta sección, cubrimos los conceptos básicos de las oraciones y declaraciones en COBOL. Aprendimos que una oración en COBOL es una secuencia de una o más declaraciones que terminan con un punto. También exploramos varios tipos de declaraciones en COBOL, incluyendo las declaraciones MOVE, DISPLAY, COMPUTE, IF y PERFORM, con ejemplos para ilustrar su uso.

Entender estos conceptos fundamentales es esencial para escribir programas COBOL efectivos. Practica usando estas declaraciones en diferentes escenarios para volverte competente en la programación en COBOL.
