Introducción

Power BI es una herramienta poderosa para la visualización de datos y la inteligencia empresarial. Una de sus características clave es la capacidad de ver e interactuar con informes en dispositivos móviles. Esta sección te guiará a través del proceso de acceso y uso de informes de Power BI en dispositivos móviles, desde la configuración básica hasta las funciones avanzadas.

Configuración de Power BI en Dispositivos Móviles

Para comenzar a ver informes de Power BI en tu dispositivo móvil, necesitas instalar la aplicación móvil de Power BI.

Instalación de la Aplicación Móvil de Power BI

  • Para Dispositivos iOS:

    1. Abre la App Store.
    2. Busca "Power BI".
    3. Toca "Obtener" para descargar e instalar la aplicación.

  • Para Dispositivos Android:

    1. Abre la Google Play Store.
    2. Busca "Power BI".
    3. Toca "Instalar" para descargar e instalar la aplicación.

Iniciar Sesión

  • Abre la aplicación de Power BI.
  • Ingresa las credenciales de tu cuenta de Power BI (correo electrónico y contraseña).
  • Si usas autenticación multifactor, completa los pasos adicionales de verificación.

Navegación en la Aplicación Móvil de Power BI

Una vez que hayas iniciado sesión, verás el panel principal. Aquí están los componentes clave:

  • Inicio: Muestra tus informes y paneles más frecuentemente utilizados.
  • Favoritos: Acceso rápido a tus informes favoritos.
  • Compartidos Conmigo: Informes y paneles compartidos contigo por otros.
  • Aplicaciones: Acceso a aplicaciones organizacionales que contienen colecciones de paneles e informes relacionados.

Visualización de Informes

Acceso a Informes

  • Toca un informe desde la pantalla de Inicio o navega a través de las secciones de Favoritos o Compartidos Conmigo.
  • Usa la barra de búsqueda para encontrar informes específicos.

Interacción con Informes

  • Zoom y Desplazamiento: Pellizca para acercar y alejar. Arrastra para desplazarte por el informe.
  • Filtrado de Datos: Toca elementos visuales para aplicar filtros.
  • Exploración Detallada: Toca puntos de datos para explorar vistas más detalladas.

Ejemplo: Interacción con un Informe de Ventas

1. Abre el Informe de Ventas desde la pantalla de Inicio.
2. Pellizca para acercar el gráfico de ventas por región.
3. Toca la región "América del Norte" para filtrar todo el informe por esta región.
4. Observa cómo otros elementos visuales se actualizan para reflejar los datos filtrados.

Funciones Avanzadas

Visualización Sin Conexión

  • Habilitar Modo Sin Conexión: Descarga informes para verlos sin conexión.
    1. Abre el informe que deseas ver sin conexión.
    2. Toca los puntos suspensivos (...) en la esquina superior derecha.
    3. Selecciona "Hacer disponible sin conexión".

Anotaciones y Compartir

  • Agregar Anotaciones:

    1. Abre el informe que deseas anotar.
    2. Toca el ícono del lápiz para entrar en modo de anotación.
    3. Dibuja o escribe directamente en el informe.
    4. Guarda el informe anotado.

  • Compartir Informes:

    1. Abre el informe que deseas compartir.
    2. Toca el ícono de compartir.
    3. Elige el método de compartir (correo electrónico, enlace, etc.).
    4. Ingresa los detalles del destinatario y envía.

Resumen

Ver e interactuar con informes de Power BI en dispositivos móviles es una forma conveniente de mantenerse conectado con tus datos en movimiento. Al instalar la aplicación móvil de Power BI, iniciar sesión y navegar por las diversas secciones, puedes acceder e interactuar fácilmente con tus informes. Las funciones avanzadas como la visualización sin conexión y las anotaciones mejoran aún más la experiencia móvil. Ya seas un principiante o un usuario avanzado, aprovechar Power BI en dispositivos móviles puede mejorar significativamente tu accesibilidad a los datos y los procesos de toma de decisiones.
