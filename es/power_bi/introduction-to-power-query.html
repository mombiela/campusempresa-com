Power Query es una tecnología de conexión de datos poderosa que te permite descubrir, conectar, combinar y refinar datos de una amplia variedad de fuentes. En esta sección, cubriremos los conceptos básicos de Power Query, su interfaz y cómo realizar tareas esenciales de transformación de datos.

¿Qué es Power Query?

Power Query es una tecnología de conexión de datos que forma parte del conjunto de Microsoft Power BI. Permite a los usuarios:

  • Conectar a diversas fuentes de datos.
  • Transformar y limpiar datos.
  • Cargar datos en Power BI para su análisis.

Conceptos Clave de Power Query

  • Fuentes de Datos: Power Query puede conectarse a una amplia gama de fuentes de datos, incluyendo bases de datos, servicios web, archivos de Excel y más.
  • Consultas: Una consulta es un conjunto de pasos que Power Query utiliza para conectarse a una fuente de datos, transformar los datos y cargarlos en Power BI.
  • Transformaciones: Son operaciones que puedes realizar en tus datos para limpiarlos y darles forma, como filtrar filas, eliminar columnas y combinar tablas.
  • Lenguaje M: Power Query utiliza un lenguaje de fórmulas llamado M para realizar transformaciones de datos.

Interfaz de Power Query

El Editor de Power Query es donde realizarás la mayoría de tus tareas de transformación de datos. Los componentes principales del Editor de Power Query incluyen:

  • Cinta: Contiene varias pestañas y comandos para la transformación de datos.
  • Panel de Consultas: Lista todas las consultas en tu informe de Power BI.
  • Vista Previa de Datos: Muestra una vista previa de los datos para la consulta seleccionada.
  • Configuración de la Consulta: Muestra las propiedades y los pasos aplicados para la consulta seleccionada.

Conectarse a Fuentes de Datos

Para conectarse a una fuente de datos en Power Query:

  1. Abre Power BI Desktop.
  2. Haz clic en la pestaña "Inicio".
  3. Selecciona "Obtener Datos" y elige tu fuente de datos.

Ejemplo: Conectarse a un Archivo de Excel

1. Haz clic en "Obtener Datos" > "Excel".
2. Navega hasta la ubicación de tu archivo de Excel y selecciónalo.
3. Haz clic en "Abrir".
4. En la ventana del Navegador, selecciona las hojas o tablas que deseas cargar.
5. Haz clic en "Cargar" para cargar los datos en Power BI o "Transformar Datos" para abrir el Editor de Power Query.

Transformaciones Básicas de Datos

Eliminar Columnas

Para eliminar columnas que no son necesarias:

  1. Selecciona las columnas que deseas eliminar.
  2. Haz clic derecho y elige "Eliminar Columnas".

Filtrar Filas

Para filtrar filas basadas en criterios específicos:

  1. Haz clic en la flecha desplegable en el encabezado de la columna.
  2. Selecciona los criterios de filtro (por ejemplo, igual, no igual, contiene).

Combinar Consultas

Para combinar dos consultas:

  1. Ve a la pestaña "Inicio".
  2. Haz clic en "Combinar Consultas".
  3. Selecciona las consultas que deseas combinar y las columnas coincidentes.

Ejemplo: Filtrar Filas

1. Haz clic en la flecha desplegable en la columna "País".
2. Selecciona "Filtros de Texto" > "Igual a".
3. Ingresa el nombre del país (por ejemplo, "EE.UU.").
4. Haz clic en "Aceptar".

Usando el Lenguaje M

El lenguaje M se utiliza para realizar transformaciones de datos más avanzadas. Aquí hay un ejemplo simple de un script M que filtra filas donde la columna "País" es igual a "EE.UU.":

let
    Source = Excel.Workbook(File.Contents("C:\Data\SalesData.xlsx"), null, true),
    SalesData_Sheet = Source{[Item="SalesData",Kind="Sheet"]}[Data],
    FilteredRows = Table.SelectRows(SalesData_Sheet, each [Country] = "USA")
in
    FilteredRows

Conclusión

En esta introducción a Power Query, cubrimos los conceptos básicos de qué es Power Query, sus conceptos clave y cómo realizar transformaciones de datos esenciales. Comprender estos fundamentos te ayudará a limpiar y dar forma a tus datos de manera eficiente, dejándolos listos para el análisis en Power BI. A medida que avances en el curso, aprenderás técnicas más avanzadas y mejores prácticas para usar Power Query a su máximo potencial.
