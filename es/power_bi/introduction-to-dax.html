Creando columnas calculadas y medidas ►

Resumen

Las Expresiones de Análisis de Datos (DAX) son un lenguaje de fórmulas utilizado en Microsoft Power BI, Power Pivot y SQL Server Analysis Services. Está diseñado para trabajar con datos relacionales y realizar agregaciones y cálculos dinámicos. Esta sección te guiará desde los conceptos básicos de DAX hasta conceptos más avanzados, proporcionando explicaciones claras, ejemplos y ejercicios.

¿Qué es DAX?

  • DAX es una colección de funciones, operadores y constantes que se pueden usar en fórmulas o expresiones para calcular y devolver uno o más valores.
  • Es similar a las fórmulas de Excel, pero es más potente y flexible, especialmente para trabajar con datos relacionales.

Conceptos Básicos

Sintaxis y Funciones

  • Sintaxis: Las fórmulas DAX siempre comienzan con un signo igual =.
  • Funciones: DAX tiene una rica biblioteca de funciones, incluyendo:
    • Funciones de Agregación: SUM, AVERAGE, MIN, MAX
    • Funciones Lógicas: IF, AND, OR
    • Funciones de Texto: CONCATENATE, LEFT, RIGHT
    • Funciones de Fecha y Hora: TODAY, NOW, YEAR, MONTH

Ejemplo: Cálculo Básico

Total Sales = SUM(Sales[Amount])
  • Esta fórmula calcula las ventas totales sumando la columna Amount en la tabla Sales.

Ejercicio: Crear una Medida Básica

  1. Abre Power BI Desktop.
  2. Carga un conjunto de datos de muestra.
  3. Crea una nueva medida para calcular las ventas totales usando la función SUM.

Conceptos Intermedios

Columnas Calculadas vs. Medidas

  • Columnas Calculadas: Se calculan durante la carga de datos y se almacenan en la tabla.
  • Medidas: Se calculan sobre la marcha durante la ejecución de la consulta.

| Característica | Columnas Calculadas | Medidas | |---------------------|---------------------|--------------------| | Almacenamiento | Almacenadas en la tabla | No almacenadas | | Tiempo de Cálculo | Durante la carga de datos | Durante el tiempo de consulta | | Caso de Uso | Cálculos a nivel de fila | Agregaciones y cálculos dinámicos |

Ejemplo: Columna Calculada

Sales Tax = Sales[Amount] * 0.07
  • Esta fórmula agrega una nueva columna Sales Tax a la tabla Sales, calculando un 7% de impuesto sobre cada venta.

Ejemplo: Medida

Total Sales with Tax = SUM(Sales[Amount]) + SUM(Sales[Sales Tax])
  • Esta medida calcula las ventas totales incluyendo impuestos.

Ejercicio: Crear una Columna Calculada y una Medida

  1. Crea una columna calculada para el impuesto sobre las ventas.
  2. Crea una medida para calcular las ventas totales incluyendo impuestos.

Conceptos Avanzados

Funciones de Inteligencia de Tiempo

  • Inteligencia de Tiempo: Funciones que ayudan a crear cálculos basados en fechas.
    • TOTALYTD, SAMEPERIODLASTYEAR, DATESBETWEEN

Ejemplo: Cálculo de Año a la Fecha

Total Sales YTD = TOTALYTD(SUM(Sales[Amount]), Sales[Date])
  • Esta medida calcula las ventas totales del año hasta la fecha.

Filtrado Avanzado

  • Función CALCULATE: Modifica el contexto de filtro de un cálculo.
  • Función FILTER: Devuelve una tabla que representa un subconjunto de otra tabla.

Ejemplo: CALCULATE con FILTER

High Value Sales = CALCULATE(SUM(Sales[Amount]), FILTER(Sales, Sales[Amount] > 1000))
  • Esta medida calcula las ventas totales donde el monto es mayor a 1000.

Ejercicio: Crear Medidas Avanzadas

  1. Crea una medida de ventas del año hasta la fecha.
  2. Crea una medida para ventas de alto valor usando CALCULATE y FILTER.

Conclusión

DAX es un lenguaje poderoso que mejora las capacidades analíticas de Power BI. Al comprender su sintaxis básica, funciones y características avanzadas, puedes crear informes dinámicos y perspicaces. Practica creando columnas calculadas, medidas y usando funciones de inteligencia de tiempo para volverte competente en DAX.
