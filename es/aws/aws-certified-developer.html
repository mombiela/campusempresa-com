Introducción

¡Bienvenido al curso de Desarrollador Certificado de AWS! Este curso está diseñado para llevarte de un nivel principiante a un nivel avanzado en el desarrollo con AWS. Al final de este curso, estarás bien preparado para tomar el examen de Desarrollador Certificado de AWS y aplicar tus habilidades en escenarios del mundo real.

Servicios principales de AWS

  • Amazon EC2 (Elastic Compute Cloud)
    • Lanzamiento y gestión de instancias EC2
    • Tipos de instancias EC2 y precios
    • Grupos de seguridad y pares de claves
    • Ejemplo: 
      # Lanzamiento de una instancia EC2 usando AWS CLI
aws ec2 run-instances --image-id ami-0abcdef1234567890 --count 1 --instance-type t2.micro --key-name MyKeyPair --security-groups MySecurityGroup
  • Amazon S3 (Simple Storage Service)
    • Creación y gestión de buckets S3
    • Clases de almacenamiento S3 y políticas de ciclo de vida
    • Ejemplo: 
      import boto3

# Crear un cliente S3
s3 = boto3.client('s3')

# Crear un nuevo bucket
s3.create_bucket(Bucket='my-new-bucket')
  • Amazon RDS (Relational Database Service)
    • Configuración y gestión de instancias RDS
    • Tipos de instancias RDS y precios
    • Ejemplo: 
      # Creación de una instancia RDS usando AWS CLI
aws rds create-db-instance --db-instance-identifier mydbinstance --db-instance-class db.t2.micro --engine mysql --master-username admin --master-user-password password --allocated-storage 20

Servicios avanzados de AWS

  • AWS Lambda
    • Introducción a la computación sin servidor
    • Creación y despliegue de funciones Lambda
    • Ejemplo: 
      import json

def lambda_handler(event, context):
    return {
        'statusCode': 200,
        'body': json.dumps('¡Hola desde Lambda!')
    }
  • Amazon DynamoDB
    • Creación y gestión de tablas DynamoDB
    • Modelado de datos e indexación en DynamoDB
    • Ejemplo: 
      import boto3

# Crear un cliente DynamoDB
dynamodb = boto3.client('dynamodb')

# Crear una nueva tabla
dynamodb.create_table(
    TableName='MyTable',
    KeySchema=[
        {
            'AttributeName': 'ID',
            'KeyType': 'HASH'
        }
    ],
    AttributeDefinitions=[
        {
            'AttributeName': 'ID',
            'AttributeType': 'S'
        }
    ],
    ProvisionedThroughput={
        'ReadCapacityUnits': 5,
        'WriteCapacityUnits': 5
    }
)
