Introducción

En esta sección, cubriremos los conceptos básicos de AWS Identity and Access Management (IAM), centrándonos en la creación de usuarios y grupos IAM. IAM es un servicio crítico para gestionar el acceso a los recursos de AWS de manera segura.

Conceptos Clave

  • Usuario IAM: Una entidad que creas en AWS para representar a la persona o aplicación que interactúa con los recursos de AWS.
  • Grupo IAM: Una colección de usuarios IAM. Puedes usar grupos para especificar permisos para múltiples usuarios, lo que puede facilitar la gestión de permisos.

Creación de Usuarios IAM

Guía Paso a Paso

  1. Inicia sesión en la Consola de Administración de AWS:

    • Navega a la consola IAM en https://console.aws.amazon.com/iam/.

  2. Agregar Usuario:

    • Haz clic en "Usuarios" en el panel de navegación.
    • Haz clic en el botón "Agregar usuario".

  3. Configurar Detalles del Usuario:

    • Ingresa un nombre de usuario.
    • Selecciona el tipo de acceso: Acceso programático, acceso a la Consola de Administración de AWS, o ambos.

  4. Configurar Permisos:

    • Elige cómo configurar los permisos: Adjuntar políticas existentes directamente, agregar usuario a un grupo, o copiar permisos de un usuario existente.

  5. Revisar y Crear:

    • Revisa los detalles y permisos del usuario.
    • Haz clic en "Crear usuario".

Ejemplo

aws iam create-user --user-name JohnDoe

Este comando crea un nuevo usuario IAM llamado JohnDoe.

Creación de Grupos IAM

Guía Paso a Paso

  1. Inicia sesión en la Consola de Administración de AWS:

    • Navega a la consola IAM en https://console.aws.amazon.com/iam/.

  2. Agregar Grupo:

    • Haz clic en "Grupos" en el panel de navegación.
    • Haz clic en el botón "Crear nuevo grupo".

  3. Configurar Detalles del Grupo:

    • Ingresa un nombre de grupo.

  4. Adjuntar Políticas:

    • Selecciona las políticas para adjuntar al grupo. Las políticas definen los permisos para el grupo.

  5. Revisar y Crear:

    • Revisa los detalles del grupo y las políticas adjuntas.
    • Haz clic en "Crear grupo".

Ejemplo

aws iam create-group --group-name Developers

Este comando crea un nuevo grupo IAM llamado Developers.

Agregar Usuarios a Grupos

Guía Paso a Paso

  1. Inicia sesión en la Consola de Administración de AWS:

    • Navega a la consola IAM en https://console.aws.amazon.com/iam/.

  2. Agregar Usuario a Grupo:

    • Haz clic en "Grupos" en el panel de navegación.
    • Selecciona el grupo al que deseas agregar usuarios.
    • Haz clic en el botón "Agregar usuarios al grupo".
    • Selecciona los usuarios que deseas agregar al grupo.
    • Haz clic en "Agregar usuarios".

Ejemplo

aws iam add-user-to-group --user-name JohnDoe --group-name Developers

Este comando agrega el usuario JohnDoe al grupo Developers.

Resumen

En esta sección, cubrimos los conceptos básicos de la creación de usuarios y grupos IAM en AWS. Aprendimos cómo:

  • Crear usuarios IAM.
  • Crear grupos IAM.
  • Agregar usuarios a grupos.

IAM es una herramienta poderosa para gestionar el acceso a los recursos de AWS, y entender cómo crear y gestionar usuarios y grupos es fundamental para mantener un entorno de AWS seguro.
