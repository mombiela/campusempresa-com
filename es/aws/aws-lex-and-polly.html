Introducción a AWS Lex y Polly

AWS Lex y Polly son dos servicios poderosos proporcionados por Amazon Web Services que permiten a los desarrolladores construir interfaces conversacionales y convertir texto en voz realista, respectivamente.

  • AWS Lex: Un servicio para construir interfaces conversacionales usando voz y texto.
  • AWS Polly: Un servicio que convierte texto en voz realista.

Comenzando con AWS Lex

¿Qué es AWS Lex?

AWS Lex es un servicio para construir interfaces conversacionales en cualquier aplicación usando voz y texto. Proporciona las funcionalidades de aprendizaje profundo de reconocimiento automático de voz (ASR) para convertir voz en texto, y comprensión del lenguaje natural (NLU) para reconocer la intención del texto.

Conceptos Clave en AWS Lex

  • Intenciones: Representa una acción que el usuario quiere realizar.
  • Expresiones: Frases que los usuarios pueden decir para transmitir una intención.
  • Slots: Parámetros que capturan información específica del usuario.
  • Cumplimiento: La acción tomada para satisfacer la intención.

Creando Tu Primer Bot de Lex

  1. Inicia sesión en la Consola de Administración de AWS.
  2. Navega al servicio Lex.
  3. Crea un nuevo bot:
    • Elige una plantilla o comienza desde cero.
    • Define las intenciones, expresiones y slots del bot.
  4. Construye y prueba el bot:
    • Usa la ventana de prueba incorporada para interactuar con tu bot.
  5. Despliega el bot:
    • Integra con otros servicios de AWS o plataformas de terceros.

Ejemplo: Bot Simple para Pedir Pizza

{
  "intents": [
    {
      "name": "OrderPizza",
      "sampleUtterances": [
        "Quiero pedir una pizza",
        "¿Puedo pedir una pizza?",
        "Pedir una pizza"
      ],
      "slots": [
        {
          "name": "PizzaType",
          "slotType": "PizzaType",
          "slotConstraint": "Required",
          "valueElicitationPrompt": {
            "messages": [
              {
                "contentType": "PlainText",
                "content": "¿Qué tipo de pizza te gustaría?"
              }
            ],
            "maxAttempts": 2
          }
        },
        {
          "name": "PizzaSize",
          "slotType": "PizzaSize",
          "slotConstraint": "Required",
          "valueElicitationPrompt": {
            "messages": [
              {
                "contentType": "PlainText",
                "content": "¿Qué tamaño de pizza te gustaría?"
              }
            ],
            "maxAttempts": 2
          }
        }
      ],
      "fulfillmentActivity": {
        "type": "ReturnIntent"
      }
    }
  ]
}

Comenzando con AWS Polly

¿Qué es AWS Polly?

AWS Polly es un servicio que convierte texto en voz realista, permitiéndote crear aplicaciones que hablen y construir categorías completamente nuevas de productos habilitados para voz.

Conceptos Clave en AWS Polly

  • Voces: Polly ofrece docenas de voces realistas en una variedad de idiomas.
  • Lenguaje de Marcado de Síntesis de Voz (SSML): Un estándar para controlar aspectos del habla como la pronunciación, el volumen y el tono.

Usando AWS Polly

  1. Inicia sesión en la Consola de Administración de AWS.
  2. Navega al servicio Polly.
  3. Introduce texto y selecciona una voz.
  4. Sintetiza voz:
    • Escucha la voz generada.
    • Descarga el archivo de audio.

Ejemplo: Convirtiendo Texto a Voz

import boto3

# Crea un cliente de Polly
polly = boto3.client('polly')

# Solicita la síntesis de voz
response = polly.synthesize_speech(
    Text='¡Hola, bienvenido a AWS Polly!',
    OutputFormat='mp3',
    VoiceId='Joanna'
)

# Guarda el flujo de audio devuelto por Amazon Polly
with open('speech.mp3', 'wb') as file:
    file.write(response['AudioStream'].read())

Temas Avanzados

Integrando Lex y Polly

  • Interacción de Voz: Usa Polly para proporcionar respuestas de voz en un bot de Lex.
  • SSML en Lex: Mejora las respuestas de Lex con SSML para interacciones más naturales.

Ejemplo: Usando Polly con Lex

{
  "intents": [
    {
      "name": "OrderPizza",
      "sampleUtterances": [
        "Quiero pedir una pizza"
      ],
      "slots": [
        {
          "name": "PizzaType",
          "slotType": "PizzaType",
          "slotConstraint": "Required",
          "valueElicitationPrompt": {
            "messages": [
              {
                "contentType": "SSML",
                "content": "<speak>¿Qué tipo de pizza te gustaría?</speak>"
              }
            ],
            "maxAttempts": 2
          }
        }
      ],
      "fulfillmentActivity": {
        "type": "ReturnIntent"
      }
    }
  ]
}

Conclusión

AWS Lex y Polly son herramientas poderosas para crear interfaces conversacionales y convertir texto en voz. Al entender sus conceptos clave y aprender cómo integrarlos, puedes construir aplicaciones sofisticadas habilitadas para voz que proporcionen una experiencia de usuario natural y atractiva. Ya sea que estés comenzando o buscando profundizar tu conocimiento, dominar estos servicios abrirá nuevas posibilidades en el ámbito de las interacciones de voz y texto.
