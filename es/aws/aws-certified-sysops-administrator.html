Introducción

El curso de Administrador de SysOps Certificado por AWS está diseñado para ayudarte a adquirir las habilidades y conocimientos necesarios para gestionar, desplegar y operar sistemas escalables, altamente disponibles y tolerantes a fallos en AWS. Este curso te llevará de un nivel principiante a un nivel avanzado, cubriendo todos los temas esenciales necesarios para aprobar el examen de Administrador de SysOps Certificado por AWS.

Ejemplo: Configuración de una Alarma de CloudWatch

import boto3

# Crear cliente de CloudWatch
cloudwatch = boto3.client('cloudwatch')

# Crear alarma
cloudwatch.put_metric_alarm(
    AlarmName='CPU_Utilization_Alarm',
    ComparisonOperator='GreaterThanThreshold',
    EvaluationPeriods=1,
    MetricName='CPUUtilization',
    Namespace='AWS/EC2',
    Period=60,
    Statistic='Average',
    Threshold=70.0,
    ActionsEnabled=False,
    AlarmActions=[
        'arn:aws:sns:us-east-1:123456789012:MyTopic'
    ],
    AlarmDescription='Alarma cuando la CPU del servidor excede el 70%',
    Dimensions=[
        {
          'Name': 'InstanceId',
          'Value': 'i-1234567890abcdef0'
        },
    ],
    Unit='Seconds'
)

Explicación: Este código configura una alarma de CloudWatch para monitorear la utilización de la CPU de una instancia EC2 y dispara una alarma si excede el 70%.
