Introducción

PostgreSQL es un poderoso sistema de base de datos objeto-relacional de código abierto. Es conocido por su robustez, extensibilidad y cumplimiento de estándares. Esta sección te guiará a través del proceso de instalación de PostgreSQL en varios sistemas operativos.

Instalación en Diferentes Sistemas Operativos

Windows

  1. Descargar el Instalador:

    • Visita el sitio web oficial de PostgreSQL: PostgreSQL Downloads.
    • Descarga el instalador para la última versión.

  2. Ejecutar el Instalador:

    • Haz doble clic en el archivo .exe descargado para iniciar el proceso de instalación.
    • Sigue las indicaciones en el asistente de configuración:
      • Elige el directorio de instalación.
      • Selecciona los componentes que deseas instalar (la selección predeterminada es recomendada para principiantes).
      • Establece la contraseña para el superusuario de PostgreSQL (el usuario predeterminado es postgres).
      • Elige el número de puerto (el predeterminado es 5432).
      • Selecciona la configuración regional.

  3. Completar la Instalación:

    • Haz clic en "Next" en los pasos restantes y luego en "Finish" para completar la instalación.
    • PostgreSQL se instalará y el servicio de PostgreSQL se iniciará automáticamente.

macOS

  1. Usando Homebrew:

    • Abre la aplicación Terminal.
    • Instala Homebrew si aún no lo has hecho: 
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • Instala PostgreSQL usando Homebrew: 
      brew install postgresql

  2. Inicializar e Iniciar PostgreSQL:

    • Inicializa el clúster de bases de datos: 
      initdb /usr/local/var/postgres
    • Inicia el servicio de PostgreSQL: 
      brew services start postgresql

  3. Verificar la Instalación:

    • Verifica la versión de PostgreSQL para confirmar la instalación: 
      psql --version

Linux

  1. Usando el Gestor de Paquetes (Debian/Ubuntu):

    • Actualiza la lista de paquetes: 
      sudo apt update
    • Instala PostgreSQL: 
      sudo apt install postgresql postgresql-contrib

  2. Usando el Gestor de Paquetes (Red Hat/CentOS):

    • Instala el repositorio de PostgreSQL: 
      sudo yum install -y https://download.postgresql.org/pub/repos/yum/reporpms/EL-$(rpm -E %rhel)-x86_64/pgdg-redhat-repo-latest.noarch.rpm
    • Instala PostgreSQL: 
      sudo yum install -y postgresql13-server postgresql13

  3. Inicializar e Iniciar PostgreSQL:

    • Inicializa el clúster de bases de datos (para Red Hat/CentOS): 
      sudo /usr/pgsql-13/bin/postgresql-13-setup initdb
    • Inicia el servicio de PostgreSQL: 
      sudo systemctl start postgresql
    • Habilita PostgreSQL para que se inicie al arrancar: 
      sudo systemctl enable postgresql

Pasos Posteriores a la Instalación

  1. Acceder a PostgreSQL:

    • Abre el terminal interactivo de PostgreSQL (psql): 
      sudo -i -u postgres
psql

  2. Crear una Nueva Base de Datos y Usuario:

    • Crea una nueva base de datos: 
      CREATE DATABASE mydatabase;
    • Crea un nuevo usuario con una contraseña: 
      CREATE USER myuser WITH PASSWORD 'mypassword';
    • Concede privilegios al nuevo usuario: 
      GRANT ALL PRIVILEGES ON DATABASE mydatabase TO myuser;

Conclusión

Instalar PostgreSQL es un proceso sencillo que varía ligeramente dependiendo de tu sistema operativo. Siguiendo los pasos descritos en esta sección, deberías tener PostgreSQL funcionando en tu máquina. Esta configuración servirá como la base para el resto del curso, donde aprenderás a gestionar bases de datos, realizar consultas y utilizar las funciones avanzadas de PostgreSQL.
