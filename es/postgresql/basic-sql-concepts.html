Introducción a SQL y PostgreSQL

  • SQL (Structured Query Language): Un lenguaje estándar para gestionar y manipular bases de datos.
  • PostgreSQL: Un sistema de gestión de bases de datos relacional (RDBMS) avanzado y de código abierto conocido por su robustez y cumplimiento de estándares.

Configuración de PostgreSQL

  • Instalación: Instrucciones para instalar PostgreSQL en varios sistemas operativos.
  • Conectarse a PostgreSQL: Usando la herramienta de línea de comandos psql o herramientas gráficas como pgAdmin.

Creación de Bases de Datos y Tablas

  • Crear una Base de Datos:

    CREATE DATABASE mydatabase;
    • Este comando crea una nueva base de datos llamada mydatabase.

  • Crear una Tabla:

    CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100) UNIQUE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
    • id SERIAL PRIMARY KEY: Clave primaria auto-incremental.
    • name VARCHAR(100): Un campo de caracteres variables con una longitud máxima de 100.
    • email VARCHAR(100) UNIQUE: Un campo único para almacenar direcciones de correo electrónico.
    • created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP: Un campo de marca de tiempo con un valor predeterminado de la hora actual.

Manipulación Básica de Datos

  • Insertar Datos:

    INSERT INTO users (name, email) VALUES ('John Doe', 'john.doe@example.com');
    • Este comando inserta una nueva fila en la tabla users.

  • Actualizar Datos:

    UPDATE users SET email = 'john.new@example.com' WHERE name = 'John Doe';
    • Este comando actualiza el correo electrónico del usuario llamado 'John Doe'.

  • Eliminar Datos:

    DELETE FROM users WHERE name = 'John Doe';
    • Este comando elimina al usuario llamado 'John Doe' de la tabla users.

Consultar Datos

  • Instrucción SELECT Básica:

    SELECT * FROM users;
    • Este comando recupera todas las columnas de la tabla users.

  • Filtrar Resultados con WHERE:

    SELECT * FROM users WHERE email = 'john.doe@example.com';
    • Este comando recupera filas donde el correo electrónico es 'john.doe@example.com'.

  • Ordenar Resultados con ORDER BY:

    SELECT * FROM users ORDER BY created_at DESC;
    • Este comando recupera todos los usuarios ordenados por el campo created_at en orden descendente.

Agregar Datos

  • COUNT, AVG, MAX, MIN, SUM: 
    SELECT COUNT(*) FROM users;
SELECT AVG(salary) FROM employees;
SELECT MAX(age) FROM users;
SELECT MIN(age) FROM users;
SELECT SUM(salary) FROM employees;
    • Estos comandos realizan varias funciones agregadas en las columnas especificadas.

Unir Tablas

  • INNER JOIN:

    SELECT users.name, orders.amount 
FROM users 
INNER JOIN orders ON users.id = orders.user_id;
    • Este comando recupera datos de las tablas users y orders donde hay una coincidencia de user_id.

  • LEFT JOIN:

    SELECT users.name, orders.amount 
FROM users 
LEFT JOIN orders ON users.id = orders.user_id;
    • Este comando recupera todos los usuarios y sus pedidos, incluyendo usuarios sin pedidos.

Conclusión

En este tema, cubrimos los conceptos fundamentales de SQL y PostgreSQL, incluyendo la creación de bases de datos y tablas, manipulación básica de datos, consultas, agregación de datos y unión de tablas. Estos conceptos básicos forman la base para operaciones SQL más avanzadas y tareas de gestión de bases de datos. Practica estos comandos para familiarizarte con la sintaxis de SQL y las capacidades de PostgreSQL.
