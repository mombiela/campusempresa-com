Introducción

Las funciones de fecha y hora son esenciales para gestionar y manipular datos temporales en PostgreSQL. Esta sección cubrirá varias funciones y operaciones que puedes realizar en tipos de datos de fecha y hora, desde niveles básicos hasta avanzados.

Funciones Básicas de Fecha y Hora

Fecha y Hora Actual

  • CURRENT_DATE: Devuelve la fecha actual.
  • CURRENT_TIME: Devuelve la hora actual.
  • CURRENT_TIMESTAMP: Devuelve la fecha y hora actual (timestamp).
SELECT CURRENT_DATE;       -- Devuelve la fecha actual
SELECT CURRENT_TIME;       -- Devuelve la hora actual
SELECT CURRENT_TIMESTAMP;  -- Devuelve la fecha y hora actual

Extrayendo Partes de una Fecha/Hora

  • EXTRACT(field FROM source): Extrae una parte específica de un valor de fecha/hora.
SELECT EXTRACT(YEAR FROM CURRENT_DATE);  -- Extrae el año de la fecha actual
SELECT EXTRACT(MONTH FROM CURRENT_DATE); -- Extrae el mes de la fecha actual
SELECT EXTRACT(DAY FROM CURRENT_DATE);   -- Extrae el día de la fecha actual

Aritmética de Fecha y Hora

  • Adición/Sustracción: Puedes sumar o restar intervalos a/de fechas.
SELECT CURRENT_DATE + INTERVAL '1 day';  -- Suma un día a la fecha actual
SELECT CURRENT_DATE - INTERVAL '1 month'; -- Resta un mes de la fecha actual

Funciones Intermedias de Fecha y Hora

Formateando Fecha y Hora

  • TO_CHAR: Convierte un valor de fecha/hora a una cadena con un formato especificado.
SELECT TO_CHAR(CURRENT_TIMESTAMP, 'YYYY-MM-DD HH24:MI:SS'); -- Formatea el timestamp actual

Parseando Fecha y Hora

  • TO_DATE: Convierte una cadena a una fecha.
  • TO_TIMESTAMP: Convierte una cadena a un timestamp.
SELECT TO_DATE('2023-10-01', 'YYYY-MM-DD'); -- Convierte una cadena a una fecha
SELECT TO_TIMESTAMP('2023-10-01 12:30:00', 'YYYY-MM-DD HH24:MI:SS'); -- Convierte una cadena a un timestamp

Cálculo de Edad

  • AGE: Calcula la edad entre dos fechas.
SELECT AGE('2023-10-01', '2000-01-01'); -- Calcula la edad entre dos fechas

Funciones Avanzadas de Fecha y Hora

Zonas Horarias

  • AT TIME ZONE: Convierte un timestamp a una zona horaria diferente.
SELECT '2023-10-01 12:00:00'::timestamp AT TIME ZONE 'UTC'; -- Convierte a la zona horaria UTC
SELECT '2023-10-01 12:00:00'::timestamp AT TIME ZONE 'America/New_York'; -- Convierte a la zona horaria de Nueva York

Aritmética de Intervalos

  • JUSTIFY_DAYS: Ajusta intervalos para ser más legibles.
  • JUSTIFY_HOURS: Ajusta intervalos para ser más legibles.
SELECT JUSTIFY_DAYS(INTERVAL '30 days'); -- Ajusta el intervalo para ser más legible
SELECT JUSTIFY_HOURS(INTERVAL '25 hours'); -- Ajusta el intervalo para ser más legible

Agregación de Fecha y Hora

  • Agrupando por Fecha/Hora: Útil para resumir datos en períodos de tiempo.
SELECT DATE_TRUNC('month', order_date) AS month, COUNT(*) 
FROM orders 
GROUP BY DATE_TRUNC('month', order_date); -- Agrupa pedidos por mes

Resumen

En esta sección, cubrimos una amplia gama de funciones de fecha y hora en PostgreSQL, desde la recuperación básica de la fecha y hora actual hasta la manipulación y el formateo avanzados. Entender estas funciones te permitirá manejar datos temporales de manera más efectiva, ya sea que estés realizando aritmética simple de fechas o conversiones complejas de zonas horarias. Practica estas funciones con datos del mundo real para solidificar tu comprensión y mejorar tus habilidades en PostgreSQL.
