Actualización y Eliminación de Datos
Joins y Subconsultas ►

En esta sección, cubriremos cómo actualizar y eliminar datos en PostgreSQL. Estas operaciones son fundamentales para mantener y gestionar tu base de datos de manera efectiva.

Actualización de Datos

La actualización de datos en PostgreSQL se realiza utilizando la instrucción UPDATE. Esta instrucción te permite modificar registros existentes en una tabla.

Sintaxis Básica

La sintaxis básica para la instrucción UPDATE es la siguiente:

UPDATE nombre_tabla
SET columna1 = valor1, columna2 = valor2, ...
WHERE condición;
  • nombre_tabla: El nombre de la tabla donde deseas actualizar los datos.
  • SET columna1 = valor1, columna2 = valor2, ...: Especifica las columnas y sus nuevos valores.
  • WHERE condición: Especifica qué filas deben actualizarse. Si se omite, todas las filas de la tabla se actualizarán.

Ejemplo

Consideremos una tabla empleados con la siguiente estructura:

| id | nombre | salario | |-----|----------|---------| | 1 | Alicia | 50000 | | 2 | Bob | 60000 | | 3 | Carlos | 55000 |

Para actualizar el salario de Bob a 65000, usarías la siguiente consulta:

UPDATE empleados
SET salario = 65000
WHERE nombre = 'Bob';

Actualización de Múltiples Columnas

Puedes actualizar múltiples columnas en una sola instrucción UPDATE. Por ejemplo, para actualizar tanto el nombre como el salario del empleado con id 3:

UPDATE empleados
SET nombre = 'Carlos', salario = 58000
WHERE id = 3;

Uso de Subconsultas en Actualizaciones

También puedes usar subconsultas para actualizar datos basados en valores de otras tablas. Por ejemplo, si tienes otra tabla bonos:

| id_empleado | bono | |-------------|------| | 1 | 5000 | | 2 | 7000 | | 3 | 6000 |

Para actualizar el salario en la tabla empleados sumando el bono correspondiente de la tabla bonos:

UPDATE empleados
SET salario = salario + (SELECT bono FROM bonos WHERE bonos.id_empleado = empleados.id);

Eliminación de Datos

La eliminación de datos en PostgreSQL se realiza utilizando la instrucción DELETE. Esta instrucción te permite eliminar registros existentes de una tabla.

Sintaxis Básica

La sintaxis básica para la instrucción DELETE es la siguiente:

DELETE FROM nombre_tabla
WHERE condición;
  • nombre_tabla: El nombre de la tabla de la que deseas eliminar datos.
  • WHERE condición: Especifica qué filas deben eliminarse. Si se omite, todas las filas de la tabla se eliminarán.

Ejemplo

Para eliminar al empleado llamado 'Carlos' de la tabla empleados:

DELETE FROM empleados
WHERE nombre = 'Carlos';

Eliminación de Todas las Filas

Para eliminar todas las filas de una tabla, puedes omitir la cláusula WHERE:

DELETE FROM empleados;

Uso de Subconsultas en Eliminaciones

También puedes usar subconsultas para eliminar datos basados en valores de otras tablas. Por ejemplo, para eliminar empleados que tienen un bono mayor a 6000:

DELETE FROM empleados
WHERE id IN (SELECT id_empleado FROM bonos WHERE bono > 6000);

Conclusión

La actualización y eliminación de datos son operaciones esenciales para gestionar una base de datos PostgreSQL. La instrucción UPDATE te permite modificar registros existentes, mientras que la instrucción DELETE te permite eliminar registros. Ambas instrucciones pueden usarse con condiciones y subconsultas para realizar operaciones más complejas. Entender estos comandos te ayudará a mantener la integridad y precisión de tu base de datos.
