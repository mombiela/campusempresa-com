Introducción

En esta sección, delinearemos los criterios de evaluación para el curso de programación en Assembly. Los criterios están diseñados para evaluar tu comprensión y competencia en el lenguaje Assembly desde niveles principiantes hasta avanzados. Las evaluaciones incluirán cuestionarios, tareas de codificación, proyectos y exámenes.

Nivel Principiante

Conceptos Clave

  • Sintaxis y Estructura Básica: Comprensión de la sintaxis y estructura básica del lenguaje Assembly.
  • Registros y Memoria: Conocimiento de los diferentes tipos de registros y modos de direccionamiento de memoria.
  • Instrucciones Básicas: Familiaridad con instrucciones básicas como MOV, ADD, SUB y JMP.
  • Programas Simples: Capacidad para escribir programas simples en Assembly que realicen operaciones aritméticas básicas y movimiento de datos.

Ejemplo de Evaluación

section .data
    num1 db 10
    num2 db 20
    result db 0

section .text
    global _start

_start:
    mov al, [num1]
    add al, [num2]
    mov [result], al

    ; Salir del programa
    mov eax, 1
    int 0x80
  • Tarea: Modificar el código anterior para restar num2 de num1 y almacenar el resultado en result.

Criterios de Evaluación

  • Uso correcto de la sintaxis de Assembly.
  • Manipulación adecuada de los registros.
  • Implementación precisa de las operaciones aritméticas.

Nivel Intermedio

Conceptos Clave

  • Flujo de Control: Dominio de las instrucciones de flujo de control como bucles y saltos condicionales.
  • Subrutinas y Macros: Comprensión e implementación de subrutinas y macros.
  • Operaciones con la Pila: Competencia en el uso de la pila para llamadas a funciones y variables locales.
  • Estructuras de Datos: Capacidad para implementar y manipular estructuras de datos básicas como arreglos y cadenas.

Ejemplo de Evaluación

section .data
    array db 1, 2, 3, 4, 5
    length db 5
    sum db 0

section .text
    global _start

_start:
    mov ecx, [length]
    mov esi, array
    xor eax, eax

sum_loop:
    add al, [esi]
    inc esi
    loop sum_loop

    mov [sum], al

    ; Salir del programa
    mov eax, 1
    int 0x80
  • Tarea: Modificar el código anterior para calcular el producto de los elementos en el arreglo en lugar de la suma.

Criterios de Evaluación

  • Implementación correcta de bucles y flujo de control.
  • Uso adecuado de la pila y subrutinas.
  • Manipulación precisa de estructuras de datos.

Nivel Avanzado

Conceptos Clave

  • Técnicas de Optimización: Competencia en optimizar el código Assembly para mejorar el rendimiento.
  • Llamadas al Sistema e Interrupciones: Comprensión e implementación de llamadas al sistema e interrupciones.
  • Estructuras de Datos Avanzadas: Capacidad para implementar estructuras de datos complejas como listas enlazadas y árboles.
  • Interfaz con Lenguajes de Alto Nivel: Conocimiento de cómo interconectar el código Assembly con lenguajes de alto nivel como C.

Ejemplo de Evaluación

section .data
    message db 'Hello, World!', 0

section .text
    global _start

_start:
    ; Escribir mensaje en stdout
    mov eax, 4
    mov ebx, 1
    mov ecx, message
    mov edx, 13
    int 0x80

    ; Salir del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80
  • Tarea: Optimizar el código anterior para minimizar el número de instrucciones utilizadas.

Criterios de Evaluación

  • Uso efectivo de técnicas de optimización.
  • Implementación correcta de llamadas al sistema e interrupciones.
  • Competencia en el manejo de estructuras de datos avanzadas.
  • Capacidad para interconectar el código Assembly con lenguajes de alto nivel.

Conclusión

Los criterios de evaluación para este curso de Assembly están diseñados para evaluar progresivamente tus habilidades desde la sintaxis y estructura básica hasta la optimización avanzada y programación a nivel de sistema. Al final del curso, deberías ser competente en escribir código Assembly eficiente y efectivo, capaz de resolver problemas complejos e interconectar con otros lenguajes de programación.
