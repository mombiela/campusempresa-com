Introducción a RISC-V

  • ¿Qué es RISC-V?

    • RISC-V es un conjunto de instrucciones estándar abierto (ISA) basado en los principios de la computación de conjunto de instrucciones reducidas (RISC).
    • Está diseñado para ser simple, eficiente y extensible, lo que lo hace adecuado para una amplia gama de aplicaciones, desde pequeños sistemas embebidos hasta grandes supercomputadoras.

  • ¿Por qué aprender RISC-V?

    • Código abierto y gratuito.
    • Creciente adopción en la industria.
    • Valor educativo en la comprensión de la arquitectura de computadoras y la programación de bajo nivel.

Conceptos Básicos

  • Registros

    • RISC-V tiene 32 registros de propósito general (x0 a x31).
    • x0 está cableado a cero.
    • x1 a x31 se utilizan para varios propósitos, incluyendo argumentos de funciones, valores de retorno y almacenamiento temporal.

  • Formatos de Instrucción

    • Las instrucciones de RISC-V tienen 32 bits de longitud y siguen un formato fijo.
    • Los formatos comunes incluyen R-type, I-type, S-type, B-type, U-type y J-type.

| Formato | Descripción | Ejemplo de Instrucción | |---------|-------------|------------------------| | R-type | Operaciones Registro-Registro | add x1, x2, x3 | | I-type | Operaciones Inmediatas | addi x1, x2, 10 | | S-type | Operaciones de Almacenamiento | sw x1, 0(x2) | | B-type | Operaciones de Rama | beq x1, x2, label | | U-type | Operaciones Inmediatas Superiores | lui x1, 0x10000 | | J-type | Operaciones de Salto | jal x1, label |

Instrucciones Básicas

  • Instrucciones Aritméticas

    add x1, x2, x3    # x1 = x2 + x3
sub x1, x2, x3    # x1 = x2 - x3
    • add y sub realizan sumas y restas en registros.

  • Instrucciones Inmediatas

    addi x1, x2, 10   # x1 = x2 + 10
    • addi suma un valor inmediato a un registro.

  • Instrucciones de Carga y Almacenamiento

    lw x1, 0(x2)      # Cargar palabra desde la dirección de memoria en x2 a x1
sw x1, 0(x2)      # Almacenar palabra desde x1 a la dirección de memoria en x2
    • lw y sw se usan para cargar y almacenar datos entre registros y memoria.

  • Instrucciones de Rama

    beq x1, x2, label # Rama a label si x1 == x2
bne x1, x2, label # Rama a label si x1 != x2
    • beq y bne se usan para ramificaciones condicionales.

Instrucciones Avanzadas

  • Instrucciones de Desplazamiento

    sll x1, x2, x3    # x1 = x2 << x3 (desplazamiento lógico a la izquierda)
srl x1, x2, x3    # x1 = x2 >> x3 (desplazamiento lógico a la derecha)
    • sll y srl realizan desplazamientos bit a bit.

  • Instrucciones de Salto

    jal x1, label     # Salta a label y guarda la dirección de retorno en x1
jalr x1, 0(x2)    # Salta a la dirección en x2 y guarda la dirección de retorno en x1
    • jal y jalr se usan para saltar a direcciones y guardar direcciones de retorno.

Programa de Ejemplo

# Programa simple para sumar dos números y almacenar el resultado en memoria
.data
result: .word 0

.text
.globl _start
_start:
  li x5, 10        # Cargar inmediato 10 en x5
  li x6, 20        # Cargar inmediato 20 en x6
  add x7, x5, x6   # Sumar x5 y x6, almacenar resultado en x7
  la x8, result    # Cargar dirección de result en x8
  sw x7, 0(x8)     # Almacenar x7 en memoria en la dirección en x8

  # Salir del programa (asumiendo que el entorno soporta ecall para salir)
  li x10, 10       # Cargar inmediato 10 en x10 (código de salida)
  ecall            # Hacer llamada al sistema para salir
  • Explicación:
    • El programa suma dos números (10 y 20) y almacena el resultado (30) en memoria.
    • li se usa para cargar valores inmediatos en registros.
    • add realiza la suma.
    • la carga la dirección de la variable result.
    • sw almacena el resultado en memoria.
    • ecall se usa para salir del programa.

Conclusión

  • Resumen
    • RISC-V es un ISA versátil y abierto que está ganando popularidad.
    • Entender el ensamblador RISC-V proporciona conocimientos sobre la programación de bajo nivel y la arquitectura de computadoras.
    • Las instrucciones básicas cubren aritmética, operaciones inmediatas, carga/almacenamiento y ramificación.
    • Las instrucciones avanzadas incluyen desplazamientos y saltos.
    • Los ejemplos prácticos ayudan a solidificar la comprensión de la programación en ensamblador RISC-V.

Al dominar el ensamblador RISC-V, adquieres habilidades valiosas que son aplicables en varios dominios, desde sistemas embebidos hasta computación de alto rendimiento.
