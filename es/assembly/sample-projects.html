Introducción

En esta sección, exploraremos una variedad de proyectos de ejemplo que te ayudarán a aplicar los conceptos que has aprendido en programación en Assembly. Estos proyectos van desde niveles principiantes hasta avanzados, permitiéndote desarrollar gradualmente tus habilidades y confianza.

Proyectos para Principiantes

1. Hola Mundo

El clásico programa "Hola Mundo" es un excelente punto de partida para principiantes. Te introduce a la sintaxis básica de Assembly y al proceso de escribir, ensamblar y ejecutar un programa simple.

section .data
    hello db 'Hola, Mundo!',0

section .text
    global _start

_start:
    ; Escribir la cadena en stdout
    mov eax, 4          ; número de syscall para sys_write
    mov ebx, 1          ; el descriptor de archivo 1 es stdout
    mov ecx, hello      ; puntero a la cadena
    mov edx, 13         ; longitud de la cadena
    int 0x80            ; llamar al kernel

    ; Salir del programa
    mov eax, 1          ; número de syscall para sys_exit
    xor ebx, ebx        ; código de retorno 0
    int 0x80            ; llamar al kernel
  • Conceptos Clave:
    • Sección de datos y sección de texto.
    • Llamadas al sistema para escribir en stdout y salir del programa.
    • Uso básico de registros.

2. Calculadora Simple

Crea una calculadora simple que pueda realizar operaciones aritméticas básicas como suma, resta, multiplicación y división.

section .data
    num1 db 10
    num2 db 5
    result db 0

section .text
    global _start

_start:
    ; Suma
    mov al, [num1]
    add al, [num2]
    mov [result], al

    ; Salir del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80
  • Conceptos Clave:
    • Operaciones aritméticas básicas.
    • Manipulación de registros.
    • Almacenamiento y recuperación de datos de la memoria.

Proyectos Intermedios

1. Inversión de Cadena

Escribe un programa que invierta una cadena dada.

section .data
    input db 'Assembly',0
    length equ $ - input
    output db length dup(0)

section .text
    global _start

_start:
    mov esi, input
    mov edi, output + length - 1

reverse_loop:
    lodsb
    stosb
    dec edi
    cmp esi, input + length
    jne reverse_loop

    ; Salir del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80
  • Conceptos Clave:
    • Manipulación de cadenas.
    • Constructos de bucle.
    • Uso de las instrucciones lodsb y stosb.

2. Secuencia de Fibonacci

Genera los primeros N números de la secuencia de Fibonacci.

section .data
    N equ 10
    fib db N dup(0)

section .text
    global _start

_start:
    mov byte [fib], 0
    mov byte [fib+1], 1

    mov ecx, N-2
    mov esi, fib
    add esi, 2

fib_loop:
    mov al, [esi-1]
    add al, [esi-2]
    mov [esi], al
    inc esi
    loop fib_loop

    ; Salir del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80
  • Conceptos Clave:
    • Bucle con la instrucción loop.
    • Manipulación de arrays.
    • Implementación básica de algoritmos.

Proyectos Avanzados

1. Núcleo de Sistema Operativo Simple

Desarrolla un núcleo simple que pueda arrancar y mostrar un mensaje.

section .text
    global start

start:
    mov ax, 0x07C0
    mov ds, ax
    mov es, ax

    mov si, msg
    call print_string

hang:
    jmp hang

print_string:
    mov ah, 0x0E
.next_char:
    lodsb
    cmp al, 0
    je .done
    int 0x10
    jmp .next_char
.done:
    ret

section .data
msg db '¡Hola desde el kernel!', 0
  • Conceptos Clave:
    • Fundamentos del cargador de arranque.
    • Interrupciones del BIOS.
    • Desarrollo de núcleos.

2. Gestión de Memoria

Implementa un gestor de memoria simple que pueda asignar y liberar bloques de memoria.

section .bss
    heap resb 1024

section .text
    global _start

_start:
    ; Inicializar el puntero del heap
    mov esi, heap

    ; Asignar memoria
    mov eax, 100
    call malloc

    ; Liberar memoria
    call free

    ; Salir del programa
    mov eax, 1
    xor ebx, ebx
    int 0x80

malloc:
    ; Implementación de malloc
    ret

free:
    ; Implementación de free
    ret
  • Conceptos Clave:
    • Asignación dinámica de memoria.
    • Aritmética de punteros.
    • Algoritmos de gestión de memoria.

Conclusión

Al trabajar en estos proyectos de ejemplo, obtendrás experiencia práctica con la programación en Assembly. Cada proyecto se basa en el anterior, aumentando gradualmente en complejidad y ayudándote a desarrollar una comprensión más profunda de los conceptos de programación de bajo nivel. Recuerda experimentar y modificar el código para mejorar aún más tu experiencia de aprendizaje.
