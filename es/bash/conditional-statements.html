Introducción

Las declaraciones condicionales son fundamentales en cualquier lenguaje de programación, ya que te permiten ejecutar diferentes bloques de código basados en ciertas condiciones. En Bash, las declaraciones condicionales ayudan a controlar el flujo de tus scripts, haciéndolos más dinámicos y receptivos a diferentes entradas o estados.

Conceptos clave

  • Declaraciones if-else: Estructura condicional básica.
  • Declaraciones elif: Añadir múltiples condiciones.
  • Declaraciones case: Manejar múltiples condiciones de manera más eficiente.
  • Comando test: Usado para evaluar condiciones.
  • Operadores lógicos: Combinar múltiples condiciones.

Declaraciones if-else

La declaración if-else te permite ejecutar un bloque de código si una condición es verdadera, y otro bloque si es falsa.

Sintaxis

if [ condition ]; then
    # Código a ejecutar si la condición es verdadera
else
    # Código a ejecutar si la condición es falsa
fi

Ejemplo

#!/bin/bash

read -p "Enter a number: " number

if [ $number -gt 10 ]; then
    echo "The number is greater than 10."
else
    echo "The number is 10 or less."
fi

En este ejemplo, el script lee un número del usuario y verifica si es mayor que 10.

Declaraciones elif

La declaración elif te permite verificar múltiples condiciones secuencialmente.

Sintaxis

if [ condition1 ]; then
    # Código a ejecutar si condition1 es verdadera
elif [ condition2 ]; then
    # Código a ejecutar si condition2 es verdadera
else
    # Código a ejecutar si ninguna de las condiciones es verdadera
fi

Ejemplo

#!/bin/bash

read -p "Enter a number: " number

if [ $number -gt 10 ]; then
    echo "The number is greater than 10."
elif [ $number -eq 10 ]; then
    echo "The number is exactly 10."
else
    echo "The number is less than 10."
fi

Este script verifica si el número es mayor, igual o menor que 10.

Declaraciones case

La declaración case es útil para manejar múltiples condiciones de manera más eficiente, especialmente cuando tienes muchos valores posibles que verificar.

Sintaxis

case $variable in
    pattern1)
        # Código a ejecutar si variable coincide con pattern1
        ;;
    pattern2)
        # Código a ejecutar si variable coincide con pattern2
        ;;
    *)
        # Código a ejecutar si variable no coincide con ningún patrón
        ;;
esac

Ejemplo

#!/bin/bash

read -p "Enter a character: " char

case $char in
    [a-z])
        echo "You entered a lowercase letter."
        ;;
    [A-Z])
        echo "You entered an uppercase letter."
        ;;
    [0-9])
        echo "You entered a digit."
        ;;
    *)
        echo "You entered a special character."
        ;;
esac

Este script categoriza el carácter de entrada como una letra minúscula, letra mayúscula, dígito o carácter especial.

Comando test

El comando test (o la sintaxis [ ]) se usa para evaluar condiciones.

Expresiones comunes de test

  • Comparaciones de cadenas:
    • [ "$str1" = "$str2" ]: Verdadero si las cadenas son iguales.
    • [ "$str1" != "$str2" ]: Verdadero si las cadenas no son iguales.
  • Comparaciones numéricas:
    • [ $num1 -eq $num2 ]: Verdadero si los números son iguales.
    • [ $num1 -ne $num2 ]: Verdadero si los números no son iguales.
    • [ $num1 -gt $num2 ]: Verdadero si num1 es mayor que num2.
    • [ $num1 -lt $num2 ]: Verdadero si num1 es menor que num2.
  • Pruebas de archivos:
    • [ -e $file ]: Verdadero si el archivo existe.
    • [ -d $directory ]: Verdadero si el directorio existe.
    • [ -f $file ]: Verdadero si el archivo es un archivo regular.

Ejemplo

#!/bin/bash

read -p "Enter a filename: " filename

if [ -e $filename ]; then
    echo "The file exists."
else
    echo "The file does not exist."
fi

Este script verifica si un archivo existe.

Operadores lógicos

Los operadores lógicos te permiten combinar múltiples condiciones.

Operadores lógicos comunes

  • AND (&&): Ambas condiciones deben ser verdaderas.
  • OR (||): Al menos una condición debe ser verdadera.
  • NOT (!): Niega la condición.

Ejemplo

#!/bin/bash

read -p "Enter a number: " number

if [ $number -gt 0 ] && [ $number -lt 100 ]; then
    echo "The number is between 1 and 99."
else
    echo "The number is not in the range 1-99."
fi

Este script verifica si el número está entre 1 y 99.

Conclusión

Las declaraciones condicionales en Bash son esenciales para controlar el flujo de tus scripts. Al dominar if-else, elif, case y el comando test, junto con los operadores lógicos, puedes crear scripts más dinámicos y receptivos. Practica estos conceptos con varios ejemplos para volverte competente en la escritura de scripts Bash que manejen diferentes condiciones de manera efectiva.
