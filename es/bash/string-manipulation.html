La manipulación de cadenas es una habilidad fundamental en la escritura de scripts Bash. Este tema te guiará desde operaciones básicas hasta avanzadas con cadenas, proporcionando explicaciones claras, ejemplos y ejercicios para solidificar tu comprensión.

Operaciones Básicas con Cadenas

Concatenación

La concatenación es el proceso de unir dos o más cadenas.

str1="Hola"
str2="Mundo"
resultado="$str1 $str2"
echo $resultado  # Salida: Hola Mundo

Longitud de la Cadena

Puedes encontrar la longitud de una cadena usando la sintaxis ${#string}.

str="Hola"
longitud=${#str}
echo $longitud  # Salida: 4

Extracción de Subcadena

Extrae una subcadena usando la sintaxis ${string:posición:longitud}.

str="Hola Mundo"
substr=${str:5:5}
echo $substr  # Salida: Mundo

Operaciones Intermedias con Cadenas

Reemplazo de Cadenas

Reemplaza una subcadena dentro de una cadena usando la sintaxis ${string/substring/replacement}.

str="Hola Mundo"
nueva_str=${str/Mundo/Bash}
echo $nueva_str  # Salida: Hola Bash

Conversión de Mayúsculas y Minúsculas

Convierte cadenas a mayúsculas o minúsculas usando tr.

str="Hola Mundo"
mayusculas=$(echo $str | tr '[:lower:]' '[:upper:]')
echo $mayusculas  # Salida: HOLA MUNDO

minusculas=$(echo $str | tr '[:upper:]' '[:lower:]')
echo $minusculas  # Salida: hola mundo

División de Cadenas

Divide una cadena en un array basado en un delimitador.

str="manzana,banana,cereza"
IFS=',' read -r -a array <<< "$str"
echo ${array[0]}  # Salida: manzana
echo ${array[1]}  # Salida: banana
echo ${array[2]}  # Salida: cereza

Operaciones Avanzadas con Cadenas

Expresiones Regulares

Usa expresiones regulares para coincidencias y manipulaciones complejas de cadenas.

str="Hola123"
if [[ $str =~ [0-9]+ ]]; then
  echo "La cadena contiene números"
else
  echo "La cadena no contiene números"
fi
# Salida: La cadena contiene números

Comparación de Cadenas

Compara cadenas usando expresiones condicionales.

str1="Hola"
str2="Mundo"

if [ "$str1" == "$str2" ]; then
  echo "Las cadenas son iguales"
else
  echo "Las cadenas no son iguales"
fi
# Salida: Las cadenas no son iguales

Reemplazo Avanzado de Cadenas

Usa la expansión de parámetros para reemplazos más complejos.

str="Hola Mundo Mundo"
nueva_str=${str//Mundo/Bash}
echo $nueva_str  # Salida: Hola Bash Bash

Resumen

La manipulación de cadenas en Bash es una herramienta poderosa para cualquier programador. Desde operaciones básicas como la concatenación y el cálculo de longitud hasta técnicas más avanzadas como expresiones regulares y reemplazos complejos, dominar estas habilidades mejorará enormemente tus capacidades de scripting. Practica los ejemplos y ejercicios proporcionados para volverte competente en el manejo de cadenas en Bash.
