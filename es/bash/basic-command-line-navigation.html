Introducción

En esta sección, cubriremos los comandos fundamentales para navegar por el sistema de archivos usando la línea de comandos de Bash. Entender estos comandos es esencial para gestionar eficientemente archivos y directorios en un sistema operativo similar a Unix.

Conceptos Clave

  • Directorio Actual: El directorio en el que estás trabajando actualmente.
  • Directorio Home: El directorio predeterminado para tu cuenta de usuario.
  • Ruta Absoluta: La ruta completa desde el directorio raíz hasta un archivo o directorio específico.
  • Ruta Relativa: La ruta relativa al directorio actual.

Navegación por Directorios

Ver el Directorio Actual

Para ver el directorio en el que te encuentras actualmente, usa el comando pwd (print working directory).

pwd

Cambiar de Directorio

Para cambiar tu directorio actual, usa el comando cd (change directory).

  • Para moverte a un directorio específico:

    cd /ruta/al/directorio

  • Para moverte a tu directorio home:

    cd ~

  • Para subir un nivel de directorio:

    cd ..

  • Para moverte al directorio anterior:

    cd -

Listar el Contenido del Directorio

Para listar los archivos y directorios en tu directorio actual, usa el comando ls.

  • Listado básico:

    ls

  • Listado detallado:

    ls -l

  • Incluyendo archivos ocultos:

    ls -a

  • Combinando opciones:

    ls -la

Crear y Eliminar Directorios

Para crear y eliminar directorios, usa los comandos mkdir y rmdir, respectivamente.

  • Crear un directorio:

    mkdir nuevo_directorio

  • Eliminar un directorio vacío:

    rmdir nuevo_directorio

Ejemplos y Ejercicios

Ejemplo 1: Navegar a un Directorio

  1. Abre tu terminal.
  2. Usa pwd para verificar tu directorio actual.
  3. Cambia al directorio /tmp: 
    cd /tmp
  4. Verifica tu nueva ubicación con pwd.

Ejemplo 2: Listar Archivos

  1. Lista el contenido de tu directorio home: 
    ls ~
  2. Lista todos los archivos, incluyendo los ocultos, en el directorio /etc: 
    ls -a /etc

Ejercicio 1: Práctica de Navegación por Directorios

  1. Navega a tu directorio home.
  2. Crea un nuevo directorio llamado practica.
  3. Cambia al directorio practica.
  4. Crea otro directorio dentro de practica llamado subdir.
  5. Lista el contenido del directorio practica para verificar que subdir fue creado.
  6. Elimina el directorio subdir.
  7. Navega de vuelta a tu directorio home.

Conclusión

Dominar la navegación básica por la línea de comandos es una habilidad crucial para cualquier programador que trabaje en un entorno similar a Unix. Al entender y practicar estos comandos, podrás gestionar eficientemente tus archivos y directorios, haciendo tu flujo de trabajo más productivo. Continúa practicando estos comandos hasta que te sientas cómodo navegando por el sistema de archivos sin dudar.
