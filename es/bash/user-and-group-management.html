Introducción

La gestión de usuarios y grupos es un aspecto fundamental de la administración de sistemas en sistemas operativos tipo Unix, incluidos aquellos que utilizan Bash. Una gestión adecuada asegura que los usuarios tengan los permisos y el acceso apropiados a los recursos, mejorando tanto la seguridad como la eficiencia.

Conceptos Clave

  • Usuario: Una cuenta individual que puede iniciar sesión y realizar tareas en el sistema.
  • Grupo: Una colección de usuarios que pueden compartir permisos y acceso a recursos.
  • UID (Identificador de Usuario): Un identificador único asignado a cada usuario.
  • GID (Identificador de Grupo): Un identificador único asignado a cada grupo.
  • Directorio Home: Un directorio personal para cada usuario donde pueden almacenar archivos.
  • Shell: La interfaz de línea de comandos utilizada por el usuario.

Gestión de Usuarios

Añadir un Usuario

Para añadir un nuevo usuario, utiliza el comando useradd. Este comando crea una nueva cuenta de usuario y configura el directorio home y el shell.

sudo useradd -m -s /bin/bash newuser
  • -m: Crea un directorio home para el usuario.
  • -s: Especifica el shell para el usuario.

Establecer una Contraseña

Después de crear un usuario, necesitas establecer una contraseña usando el comando passwd.

sudo passwd newuser

Se te pedirá que ingreses y confirmes la nueva contraseña.

Modificar un Usuario

Para modificar un usuario existente, utiliza el comando usermod. Por ejemplo, para cambiar el shell del usuario:

sudo usermod -s /bin/zsh newuser

Eliminar un Usuario

Para eliminar un usuario y su directorio home, utiliza el comando userdel con la opción -r.

sudo userdel -r newuser

Gestión de Grupos

Añadir un Grupo

Para añadir un nuevo grupo, utiliza el comando groupadd.

sudo groupadd newgroup

Añadir un Usuario a un Grupo

Para añadir un usuario a un grupo, utiliza el comando usermod con la opción -aG.

sudo usermod -aG newgroup newuser
  • -aG: Añade el usuario al grupo especificado sin eliminarlo de otros grupos.

Ver la Membresía de un Grupo

Para ver los grupos a los que pertenece un usuario, utiliza el comando groups.

groups newuser

Eliminar un Grupo

Para eliminar un grupo, utiliza el comando groupdel.

sudo groupdel newgroup

Permisos y Propiedad de Archivos

Cambiar la Propiedad

Para cambiar la propiedad de un archivo o directorio, utiliza el comando chown.

sudo chown newuser:newgroup filename
  • newuser:newgroup: Especifica el nuevo propietario y grupo.

Cambiar Permisos

Para cambiar los permisos de un archivo o directorio, utiliza el comando chmod.

chmod 755 filename
  • 755: Establece los permisos (rwxr-xr-x).

Entender los Permisos

Los permisos están representados por un número de tres dígitos, donde cada dígito puede ser un valor de 0 a 7:

  • r (leer) = 4
  • w (escribir) = 2
  • x (ejecutar) = 1

Por ejemplo, 755 significa:

  • Propietario: rwx (4+2+1 = 7)
  • Grupo: r-x (4+0+1 = 5)
  • Otros: r-x (4+0+1 = 5)

Ejercicios Prácticos

Ejercicio 1: Crear un Nuevo Usuario

  1. Crea un nuevo usuario llamado student.
  2. Establece una contraseña para student.
  3. Verifica la creación listando todos los usuarios.

Ejercicio 2: Gestionar Grupos

  1. Crea un nuevo grupo llamado developers.
  2. Añade el usuario student al grupo developers.
  3. Verifica la membresía del grupo.

Ejercicio 3: Permisos de Archivos

  1. Crea un archivo llamado project.txt.
  2. Cambia la propiedad de project.txt a student:developers.
  3. Establece los permisos de project.txt a 644.

Conclusión

La gestión de usuarios y grupos en Bash es esencial para mantener un sistema seguro y eficiente. Al entender cómo añadir, modificar y eliminar usuarios y grupos, así como gestionar los permisos y la propiedad de archivos, puedes controlar efectivamente el acceso a los recursos del sistema. Practica estos comandos y conceptos para volverte competente en la gestión de usuarios y grupos en un entorno tipo Unix.
