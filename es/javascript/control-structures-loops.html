Introducción a los Bucles

Los bucles son estructuras de control fundamentales en JavaScript que te permiten ejecutar un bloque de código múltiples veces. Son esenciales para tareas que requieren repetición, como iterar sobre arrays o realizar acciones hasta que se cumpla una cierta condición.

Tipos de Bucles en JavaScript

JavaScript proporciona varios tipos de bucles para manejar diferentes escenarios:

  • Bucle for
  • Bucle while
  • Bucle do...while
  • Bucle for...in
  • Bucle for...of

Bucle `for`

El bucle for es el bucle más comúnmente utilizado en JavaScript. Es ideal para escenarios donde sabes de antemano cuántas veces quieres ejecutar una declaración o un bloque de declaraciones.

Sintaxis:

for (inicialización; condición; incremento) {
  // bloque de código a ejecutar
}

Ejemplo:

for (let i = 0; i < 5; i++) {
  console.log("Número de iteración: " + i);
}

Explicación:

  • Inicialización: let i = 0 establece el punto de partida.
  • Condición: i < 5 verifica si el bucle debe continuar.
  • Incremento: i++ actualiza el contador después de cada iteración.

Bucle `while`

El bucle while se utiliza cuando el número de iteraciones no se conoce de antemano. Continúa ejecutándose mientras la condición especificada sea true.

Sintaxis:

while (condición) {
  // bloque de código a ejecutar
}

Ejemplo:

let i = 0;
while (i < 5) {
  console.log("Número de iteración: " + i);
  i++;
}

Explicación:

  • El bucle continuará ejecutándose mientras i < 5 sea true.
  • i++ incrementa el contador para eventualmente terminar el bucle.

Bucle `do...while`

El bucle do...while es similar al bucle while, pero garantiza que el bloque de código se ejecutará al menos una vez antes de que se pruebe la condición.

Sintaxis:

do {
  // bloque de código a ejecutar
} while (condición);

Ejemplo:

let i = 0;
do {
  console.log("Número de iteración: " + i);
  i++;
} while (i < 5);

Explicación:

  • El bloque de código dentro de la declaración do se ejecuta primero.
  • La condición i < 5 se verifica después de la primera ejecución.

Bucle `for...in`

El bucle for...in se utiliza para iterar sobre las propiedades de un objeto.

Sintaxis:

for (clave in objeto) {
  // bloque de código a ejecutar
}

Ejemplo:

const persona = {nombre: "John", apellido: "Doe", edad: 25};
for (let clave in persona) {
  console.log(clave + ": " + persona[clave]);
}

Explicación:

  • El bucle itera sobre cada propiedad en el objeto persona.
  • clave representa el nombre de la propiedad, y persona[clave] accede al valor de la propiedad.

Bucle `for...of`

El bucle for...of se utiliza para iterar sobre objetos iterables como arrays, cadenas, mapas y conjuntos.

Sintaxis:

for (variable of iterable) {
  // bloque de código a ejecutar
}

Ejemplo:

const numeros = [1, 2, 3, 4, 5];
for (let numero of numeros) {
  console.log(numero);
}

Explicación:

  • El bucle itera sobre cada elemento en el array numeros.
  • numero representa el elemento actual en cada iteración.

Comparación de Estructuras de Bucle

| Tipo de Bucle | Caso de Uso | Garantiza Ejecución al Menos Una Vez | |---------------|-------------------------------------------------|--------------------------------------| | for | Número conocido de iteraciones | No | | while | Número desconocido de iteraciones | No | | do...while | Número desconocido de iteraciones | Sí | | for...in | Iterar sobre propiedades de objetos | No | | for...of | Iterar sobre objetos iterables (arrays, etc.) | No |

Ejercicios

  1. Bucle for Básico: Escribe un bucle for que imprima los números del 1 al 10.

    for (let i = 1; i <= 10; i++) {
  console.log(i);
}

  2. Bucle while: Escribe un bucle while que imprima los números del 10 al 1.

    let i = 10;
while (i > 0) {
  console.log(i);
  i--;
}

  3. Bucle do...while: Escribe un bucle do...while que imprima los números del 1 al 5.

    let i = 1;
do {
  console.log(i);
  i++;
} while (i <= 5);

  4. Bucle for...in: Escribe un bucle for...in para imprimir todas las propiedades y valores del siguiente objeto:

    const coche = {marca: "Toyota", modelo: "Corolla", año: 2020};
for (let clave in coche) {
  console.log(clave + ": " + coche[clave]);
}

  5. Bucle for...of: Escribe un bucle for...of para imprimir cada carácter de la cadena "JavaScript".

    const str = "JavaScript";
for (let char of str) {
  console.log(char);
}

Conclusión

Los bucles son herramientas poderosas en JavaScript que te permiten realizar tareas repetitivas de manera eficiente. Comprender los diferentes tipos de bucles y sus casos de uso te permitirá escribir código más efectivo y conciso. Practica usando cada tipo de bucle para familiarizarte con su sintaxis y funcionalidad.
