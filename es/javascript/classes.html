Introducción a las Clases

Las clases en JavaScript son un plano para crear objetos. Encapsulan datos con código para trabajar en esos datos. Aunque JavaScript es un lenguaje basado en prototipos, las clases proporcionan una sintaxis más familiar para crear objetos y manejar la herencia.

Conceptos Clave

  • Declaración de Clase: La sintaxis para definir una clase.
  • Método Constructor: Un método especial para crear e inicializar un objeto creado con una clase.
  • Métodos: Funciones definidas dentro de una clase.
  • Herencia: Mecanismo para crear una nueva clase basada en una clase existente.

Declaración de Clase

Una clase se define usando la palabra clave class seguida del nombre de la clase.

class Person {
  // Método constructor
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  // Método
  greet() {
    console.log(`Hello, my name is ${this.name} and I am ${this.age} years old.`);
  }
}

// Creando una instancia de la clase
const person1 = new Person('Alice', 30);
person1.greet(); // Output: Hello, my name is Alice and I am 30 years old.

Explicación

  • Declaración de Clase: class Person { ... }
  • Método Constructor: constructor(name, age) { ... }
  • Método: greet() { ... }
  • Creación de Instancia: const person1 = new Person('Alice', 30);

Método Constructor

El método constructor es un método especial para crear e inicializar objetos. Se llama automáticamente cuando se crea una nueva instancia de la clase.

class Car {
  constructor(brand, model) {
    this.brand = brand;
    this.model = model;
  }

  displayInfo() {
    console.log(`This car is a ${this.brand} ${this.model}.`);
  }
}

const car1 = new Car('Toyota', 'Corolla');
car1.displayInfo(); // Output: This car is a Toyota Corolla.

Explicación

  • Parámetros del Constructor: brand, model
  • Propiedades de la Instancia: this.brand, this.model

Métodos

Los métodos son funciones definidas dentro de una clase. Pueden ser usados para realizar acciones sobre los datos almacenados en la clase.

class Rectangle {
  constructor(width, height) {
    this.width = width;
    this.height = height;
  }

  // Método para calcular el área
  calculateArea() {
    return this.width * this.height;
  }
}

const rectangle1 = new Rectangle(5, 10);
console.log(rectangle1.calculateArea()); // Output: 50

Explicación

  • Definición del Método: calculateArea() { ... }
  • Invocación del Método: rectangle1.calculateArea()

Herencia

La herencia permite que una clase herede propiedades y métodos de otra clase. Esto se hace usando la palabra clave extends.

class Animal {
  constructor(name) {
    this.name = name;
  }

  speak() {
    console.log(`${this.name} makes a sound.`);
  }
}

class Dog extends Animal {
  speak() {
    console.log(`${this.name} barks.`);
  }
}

const dog1 = new Dog('Rex');
dog1.speak(); // Output: Rex barks.

Explicación

  • Clase Base: class Animal { ... }
  • Clase Derivada: class Dog extends Animal { ... }
  • Método Sobrescrito: speak() { ... }

Resumen

  • Clases en JavaScript proporcionan un plano para crear objetos.
  • Los métodos constructores inicializan nuevos objetos.
  • Los métodos son funciones definidas dentro de una clase para realizar acciones sobre los datos de la clase.
  • La herencia permite que una clase herede propiedades y métodos de otra clase.

Al entender estos conceptos, puedes crear código más estructurado y reutilizable en JavaScript. Practica creando clases, usando constructores, definiendo métodos e implementando herencia para solidificar tu comprensión.
