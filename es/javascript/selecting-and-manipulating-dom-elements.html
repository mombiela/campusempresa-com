Introducción

En esta sección, exploraremos cómo seleccionar y manipular elementos del DOM (Modelo de Objeto de Documento) utilizando JavaScript. Esta es una habilidad fundamental para el desarrollo web, que te permite interactuar y modificar dinámicamente las páginas web.

Selección de Elementos del DOM

Usando `getElementById`

  • Descripción: Selecciona un solo elemento por su atributo id.
  • Sintaxis: document.getElementById('id')
  • Ejemplo: 
    const element = document.getElementById('myElement');
console.log(element); // Muestra el elemento con id 'myElement'

Usando `getElementsByClassName`

  • Descripción: Selecciona todos los elementos con un nombre de clase dado.
  • Sintaxis: document.getElementsByClassName('className')
  • Ejemplo: 
    const elements = document.getElementsByClassName('myClass');
console.log(elements); // Muestra una HTMLCollection en vivo de elementos con la clase 'myClass'

Usando `getElementsByTagName`

  • Descripción: Selecciona todos los elementos con un nombre de etiqueta dado.
  • Sintaxis: document.getElementsByTagName('tagName')
  • Ejemplo: 
    const elements = document.getElementsByTagName('div');
console.log(elements); // Muestra una HTMLCollection en vivo de todos los elementos <div>

Usando `querySelector`

  • Descripción: Selecciona el primer elemento que coincide con un selector CSS.
  • Sintaxis: document.querySelector('selector')
  • Ejemplo: 
    const element = document.querySelector('.myClass');
console.log(element); // Muestra el primer elemento con la clase 'myClass'

Usando `querySelectorAll`

  • Descripción: Selecciona todos los elementos que coinciden con un selector CSS.
  • Sintaxis: document.querySelectorAll('selector')
  • Ejemplo: 
    const elements = document.querySelectorAll('.myClass');
console.log(elements); // Muestra una NodeList de todos los elementos con la clase 'myClass'

Manipulación de Elementos del DOM

Cambiando el Contenido

  • HTML Interno: Modifica el contenido HTML dentro de un elemento.

    const element = document.getElementById('myElement');
element.innerHTML = '<p>Nuevo contenido</p>';

  • Contenido de Texto: Modifica el contenido de texto dentro de un elemento.

    const element = document.querySelector('.myClass');
element.textContent = 'Nuevo contenido de texto';

Cambiando Atributos

  • Establecer Atributo: Cambia o añade un atributo a un elemento.

    const element = document.querySelector('img');
element.setAttribute('src', 'newImage.jpg');

  • Eliminar Atributo: Elimina un atributo de un elemento.

    const element = document.querySelector('img');
element.removeAttribute('alt');

Cambiando Estilos

  • Estilos en Línea: Modifica directamente el estilo de un elemento. 
    const element = document.getElementById('myElement');
element.style.color = 'red';
element.style.backgroundColor = 'yellow';

Añadiendo y Eliminando Clases

  • Añadir Clase: Añade una clase a un elemento.

    const element = document.querySelector('.myClass');
element.classList.add('newClass');

  • Eliminar Clase: Elimina una clase de un elemento.

    const element = document.querySelector('.myClass');
element.classList.remove('oldClass');

  • Alternar Clase: Alterna una clase en un elemento.

    const element = document.querySelector('.myClass');
element.classList.toggle('active');

Manejo de Eventos

  • Añadir Manejador de Eventos: Adjunta un manejador de eventos a un elemento.

    const button = document.querySelector('button');
button.addEventListener('click', function() {
    alert('¡Botón clickeado!');
});

  • Eliminar Manejador de Eventos: Elimina un manejador de eventos de un elemento.

    const button = document.querySelector('button');
function handleClick() {
    alert('¡Botón clickeado!');
}
button.addEventListener('click', handleClick);
button.removeEventListener('click', handleClick);

Conclusión

Seleccionar y manipular elementos del DOM es una parte crucial del desarrollo web. Al dominar estas técnicas, puedes crear páginas web dinámicas e interactivas. Practica usando los diferentes métodos para seleccionar elementos y manipular su contenido, atributos y estilos para volverte competente en la manipulación del DOM con JavaScript.
