¿Qué es JavaScript?
Configuración de tu Entorno de Desarrollo ►

JavaScript es un lenguaje de programación versátil y poderoso utilizado principalmente para el desarrollo web. Permite a los desarrolladores crear páginas web dinámicas e interactivas manipulando el Modelo de Objeto del Documento (DOM), manejando eventos y comunicándose con servidores.

Conceptos Clave de JavaScript

  • Scripting del Lado del Cliente: JavaScript se ejecuta en el navegador del usuario, lo que lo hace ideal para crear páginas web interactivas.
  • Lenguaje Interpretado: A diferencia de los lenguajes compilados, el código JavaScript se ejecuta línea por línea por el navegador.
  • Orientado a Eventos: JavaScript puede responder a acciones del usuario como clics, envíos de formularios y eventos de teclado.
  • Orientado a Objetos: JavaScript soporta principios de programación orientada a objetos, permitiendo la creación de código reutilizable.
  • Multiplataforma: JavaScript se ejecuta en cualquier dispositivo con un navegador web, lo que lo hace altamente versátil.

Sintaxis y Estructura Básica

La sintaxis de JavaScript es el conjunto de reglas que definen un programa JavaScript correctamente estructurado. Aquí hay algunos elementos básicos:

Variables

Las variables se utilizan para almacenar valores de datos. En JavaScript, puedes declarar variables usando var, let o const.

var name = "John"; // Declara una variable llamada 'name' con el valor "John"
let age = 30;      // Declara una variable de ámbito de bloque llamada 'age' con el valor 30
const pi = 3.14;   // Declara una constante llamada 'pi' con el valor 3.14

Tipos de Datos

JavaScript soporta varios tipos de datos, incluyendo:

  • String: Datos textuales, por ejemplo, "¡Hola, Mundo!"
  • Number: Datos numéricos, por ejemplo, 42 o 3.14
  • Boolean: Datos lógicos, por ejemplo, true o false
  • Array: Colección ordenada de valores, por ejemplo, [1, 2, 3]
  • Object: Colección de pares clave-valor, por ejemplo, {name: "John", age: 30}

Funciones

Las funciones son bloques de código diseñados para realizar una tarea particular. Se ejecutan cuando son "llamadas" o "invocadas".

function greet(name) {
    return "¡Hola, " + name + "!";
}

console.log(greet("Alice")); // Salida: ¡Hola, Alice!

Estructuras de Control

JavaScript incluye estructuras de control como condicionales y bucles para controlar el flujo del programa.

Condicionales

let score = 85;

if (score >= 90) {
    console.log("A");
} else if (score >= 80) {
    console.log("B");
} else {
    console.log("C");
}

Bucles

for (let i = 0; i < 5; i++) {
    console.log(i); // Salida: 0, 1, 2, 3, 4
}

Comparación de `var`, `let` y `const`

| Característica | var | let | const | |------------------|-------------------------|-------------------------|-------------------------| | Ámbito | Ámbito de función | Ámbito de bloque | Ámbito de bloque | | Reasignación | Permitida | Permitida | No permitida | | Elevación | Sí, inicializado como undefined | Sí, pero no inicializado | Sí, pero no inicializado |

Conclusión

JavaScript es un lenguaje esencial para el desarrollo web, permitiendo la creación de experiencias de usuario dinámicas e interactivas. Entender su sintaxis básica, tipos de datos y estructuras de control es el primer paso hacia el dominio de JavaScript. A medida que avances en este curso, profundizarás en temas más avanzados, construyendo una base sólida en programación con JavaScript.
