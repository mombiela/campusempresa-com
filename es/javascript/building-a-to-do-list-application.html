Introducción

En este curso, construiremos una aplicación de Lista de Tareas utilizando JavaScript. Este proyecto te ayudará a entender conceptos clave de JavaScript y cómo aplicarlos en un escenario del mundo real. Al final de este curso, tendrás una aplicación de Lista de Tareas completamente funcional y una comprensión más profunda de JavaScript.

Prerrequisitos

  • Comprensión básica de HTML y CSS.
  • Conocimiento básico de la sintaxis de JavaScript y conceptos de programación.

Configurando el Proyecto

Creando la Estructura HTML

Comenzaremos creando la estructura HTML básica para nuestra aplicación de Lista de Tareas.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Lista de Tareas</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div id="app">
        <h1>Lista de Tareas</h1>
        <input type="text" id="new-task" placeholder="Agregar una nueva tarea">
        <button id="add-task">Agregar Tarea</button>
        <ul id="task-list"></ul>
    </div>
    <script src="app.js"></script>
</body>
</html>

Agregando Funcionalidad con JavaScript

Seleccionando Elementos

Necesitamos seleccionar los elementos HTML con los que interactuaremos usando JavaScript.

// app.js
document.addEventListener('DOMContentLoaded', () => {
    const newTaskInput = document.getElementById('new-task');
    const addTaskButton = document.getElementById('add-task');
    const taskList = document.getElementById('task-list');
});

Agregando una Nueva Tarea

Agregaremos funcionalidad para añadir una nueva tarea a la lista.

document.addEventListener('DOMContentLoaded', () => {
    const newTaskInput = document.getElementById('new-task');
    const addTaskButton = document.getElementById('add-task');
    const taskList = document.getElementById('task-list');

    addTaskButton.addEventListener('click', () => {
        const taskText = newTaskInput.value.trim();
        if (taskText !== '') {
            const li = document.createElement('li');
            li.textContent = taskText;
            const deleteButton = document.createElement('button');
            deleteButton.textContent = 'Eliminar';
            li.appendChild(deleteButton);
            taskList.appendChild(li);
            newTaskInput.value = '';
        }
    });
});

Eliminando una Tarea

Agregaremos funcionalidad para eliminar una tarea de la lista.

document.addEventListener('DOMContentLoaded', () => {
    const newTaskInput = document.getElementById('new-task');
    const addTaskButton = document.getElementById('add-task');
    const taskList = document.getElementById('task-list');

    addTaskButton.addEventListener('click', () => {
        const taskText = newTaskInput.value.trim();
        if (taskText !== '') {
            const li = document.createElement('li');
            li.textContent = taskText;
            const deleteButton = document.createElement('button');
            deleteButton.textContent = 'Eliminar';
            li.appendChild(deleteButton);
            taskList.appendChild(li);
            newTaskInput.value = '';
        }
    });

    taskList.addEventListener('click', (e) => {
        if (e.target.tagName === 'BUTTON') {
            const li = e.target.parentElement;
            taskList.removeChild(li);
        }
    });
});

Funcionalidades Avanzadas

Persistiendo Datos con Almacenamiento Local

Agregaremos funcionalidad para guardar tareas en el almacenamiento local del navegador para que persistan incluso después de que la página se actualice.

document.addEventListener('DOMContentLoaded', () => {
    const newTaskInput = document.getElementById('new-task');
    const addTaskButton = document.getElementById('add-task');
    const taskList = document.getElementById('task-list');

    const saveTasks = () => {
        const tasks = [];
        taskList.querySelectorAll('li').forEach(li => {
            tasks.push(li.firstChild.textContent);
        });
        localStorage.setItem('tasks', JSON.stringify(tasks));
    };

    const loadTasks = () => {
        const tasks = JSON.parse(localStorage.getItem('tasks')) || [];
        tasks.forEach(taskText => {
            const li = document.createElement('li');
            li.textContent = taskText;
            const deleteButton = document.createElement('button');
            deleteButton.textContent = 'Eliminar';
            li.appendChild(deleteButton);
            taskList.appendChild(li);
        });
    };

    addTaskButton.addEventListener('click', () => {
        const taskText = newTaskInput.value.trim();
        if (taskText !== '') {
            const li = document.createElement('li');
            li.textContent = taskText;
            const deleteButton = document.createElement('button');
            deleteButton.textContent = 'Eliminar';
            li.appendChild(deleteButton);
            taskList.appendChild(li);
            newTaskInput.value = '';
            saveTasks();
        }
    });

    taskList.addEventListener('click', (e) => {
        if (e.target.tagName === 'BUTTON') {
            const li = e.target.parentElement;
            taskList.removeChild(li);
            saveTasks();
        }
    });

    loadTasks();
});
