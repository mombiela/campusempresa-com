Introducción

Los operadores son bloques de construcción fundamentales en JavaScript que te permiten realizar varias operaciones en variables y valores. Entender cómo usar los operadores de manera efectiva es crucial para escribir código eficiente y efectivo.

Tipos de Operadores

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas.

  • Adición (+)

    let a = 5;
let b = 3;
let sum = a + b; // sum es 8

  • Sustracción (-)

    let difference = a - b; // difference es 2

  • Multiplicación (*)

    let product = a * b; // product es 15

  • División (/)

    let quotient = a / b; // quotient es 1.6667

  • Módulo (%)

    let remainder = a % b; // remainder es 2

  • Exponenciación (**)

    let power = a ** b; // power es 125

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación (=)

    let x = 10;

  • Asignación de Adición (+=)

    x += 5; // x es ahora 15

  • Asignación de Sustracción (-=)

    x -= 3; // x es ahora 12

  • Asignación de Multiplicación (*=)

    x *= 2; // x es ahora 24

  • Asignación de División (/=)

    x /= 4; // x es ahora 6

  • Asignación de Módulo (%=)

    x %= 5; // x es ahora 1

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores y devolver un resultado booleano.

  • Igual (==)

    let isEqual = (a == b); // false

  • Igual Estricto (===)

    let isStrictEqual = (a === b); // false

  • No Igual (!=)

    let isNotEqual = (a != b); // true

  • No Igual Estricto (!==)

    let isStrictNotEqual = (a !== b); // true

  • Mayor Que (>)

    let isGreaterThan = (a > b); // true

  • Menor Que (<)

    let isLessThan = (a < b); // false

  • Mayor o Igual Que (>=)

    let isGreaterThanOrEqual = (a >= b); // true

  • Menor o Igual Que (<=)

    let isLessThanOrEqual = (a <= b); // false

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones booleanas.

  • AND Lógico (&&)

    let andResult = (a > 0 && b > 0); // true

  • OR Lógico (||)

    let orResult = (a > 0 || b < 0); // true

  • NOT Lógico (!)

    let notResult = !(a > 0); // false

Operadores a Nivel de Bits

Los operadores a nivel de bits se utilizan para realizar operaciones a nivel de bits.

  • AND (&)

    let bitwiseAnd = a & b; // 1 (binario: 0101 & 0011 = 0001)

  • OR (|)

    let bitwiseOr = a | b; // 7 (binario: 0101 | 0011 = 0111)

  • XOR (^)

    let bitwiseXor = a ^ b; // 6 (binario: 0101 ^ 0011 = 0110)

  • NOT (~)

    let bitwiseNot = ~a; // -6 (binario: ~0101 = 1010)

  • Desplazamiento a la Izquierda (<<)

    let leftShift = a << 1; // 10 (binario: 0101 << 1 = 1010)

  • Desplazamiento a la Derecha (>>)

    let rightShift = a >> 1; // 2 (binario: 0101 >> 1 = 0010)

Operador Ternario

El operador ternario es una abreviatura para la declaración if-else.

  • Sintaxis: condition ? expr1 : expr2 
    let age = 18;
let canVote = (age >= 18) ? 'Yes' : 'No'; // canVote es 'Yes'

Operadores de Tipo

Los operadores de tipo se utilizan para determinar el tipo de una variable.

  • typeof

    let typeOfA = typeof a; // 'number'

  • instanceof

    let isInstanceOf = a instanceof Number; // false

Conclusión

Los operadores son herramientas esenciales en JavaScript que te permiten manipular datos y controlar el flujo de tus programas. Al dominar los operadores aritméticos, de asignación, de comparación, lógicos, a nivel de bits, ternarios y de tipo, puedes escribir código más eficiente y efectivo. Practica usando estos operadores en diferentes escenarios para volverte competente en su uso.
