Introducción

BigQuery es un almacén de datos totalmente gestionado y sin servidor que permite un análisis escalable de petabytes de datos. Para interactuar con BigQuery de manera programática, puedes usar APIs y bibliotecas cliente. Este tema te guiará a través del uso de las APIs y bibliotecas cliente de BigQuery desde niveles principiantes hasta avanzados.

Conceptos Clave

  • API (Interfaz de Programación de Aplicaciones): Un conjunto de reglas que permite que diferentes entidades de software se comuniquen entre sí.
  • Biblioteca Cliente: Una colección de código preescrito que los desarrolladores pueden usar para interactuar con las APIs más fácilmente.
  • API de BigQuery: Proporciona acceso programático a las características de BigQuery.
  • Autenticación: El proceso de verificar la identidad de un usuario o aplicación.

Configuración

Antes de que puedas usar la API de BigQuery o las bibliotecas cliente, necesitas configurar tu entorno.

Requisitos Previos

  • Una cuenta de Google Cloud Platform (GCP).
  • Un proyecto de GCP con BigQuery habilitado.
  • Credenciales de cuenta de servicio para la autenticación.

Autenticación

  1. Crear una Cuenta de Servicio:

    • Ve a la Consola de GCP.
    • Navega a IAM & Admin > Cuentas de Servicio.
    • Haz clic en "Crear Cuenta de Servicio" y sigue las indicaciones.
    • Descarga el archivo de clave JSON.

  2. Establecer la Variable de Entorno:

    export GOOGLE_APPLICATION_CREDENTIALS="/ruta/a/tu/archivo-de-cuenta-de-servicio.json"

Uso de la API de BigQuery

La API de BigQuery te permite interactuar con BigQuery de manera programática. A continuación se muestran los pasos para realizar operaciones básicas usando la API.

Realización de Solicitudes API

Puedes usar herramientas como curl o Postman para hacer solicitudes HTTP a la API de BigQuery.

Ejemplo: Listar Conjuntos de Datos

curl -X GET \
  -H "Authorization: Bearer $(gcloud auth application-default print-access-token)" \
  "https://bigquery.googleapis.com/bigquery/v2/projects/tu-id-de-proyecto/datasets"

Análisis de Respuestas API

Las respuestas de la API suelen estar en formato JSON. Puedes usar herramientas como jq para analizar respuestas JSON.

Ejemplo: Analizar Respuesta JSON

curl -X GET \
  -H "Authorization: Bearer $(gcloud auth application-default print-access-token)" \
  "https://bigquery.googleapis.com/bigquery/v2/projects/tu-id-de-proyecto/datasets" | jq .

Uso de Bibliotecas Cliente

Las bibliotecas cliente proporcionan una forma más conveniente de interactuar con BigQuery. Google proporciona bibliotecas cliente para varios lenguajes de programación.

Biblioteca Cliente de Python

Python es uno de los lenguajes más populares para el análisis de datos. A continuación se muestran los pasos para usar la biblioteca cliente de BigQuery para Python.

Instalación

pip install google-cloud-bigquery

Operaciones Básicas

  1. Inicializar el Cliente:

    from google.cloud import bigquery

client = bigquery.Client()

  2. Listar Conjuntos de Datos:

    datasets = list(client.list_datasets())
for dataset in datasets:
    print(dataset.dataset_id)

  3. Ejecutar una Consulta:

    query = """
SELECT name, SUM(number) as total
FROM `bigquery-public-data.usa_names.usa_1910_2013`
WHERE state = 'TX'
GROUP BY name
ORDER BY total DESC
LIMIT 10
"""
query_job = client.query(query)

for row in query_job:
    print(f"name: {row['name']}, total: {row['total']}")

Biblioteca Cliente de JavaScript

JavaScript se usa ampliamente para aplicaciones web. A continuación se muestran los pasos para usar la biblioteca cliente de BigQuery para JavaScript.

Instalación

npm install --save @google-cloud/bigquery

Operaciones Básicas

  1. Inicializar el Cliente:

    const {BigQuery} = require('@google-cloud/bigquery');
const bigquery = new BigQuery();

  2. Listar Conjuntos de Datos:

    async function listDatasets() {
  const [datasets] = await bigquery.getDatasets();
  datasets.forEach(dataset => console.log(dataset.id));
}

listDatasets();

  3. Ejecutar una Consulta:

    async function runQuery() {
  const query = `
  SELECT name, SUM(number) as total
  FROM \`bigquery-public-data.usa_names.usa_1910_2013\`
  WHERE state = 'TX'
  GROUP BY name
  ORDER BY total DESC
  LIMIT 10
  `;
  const [rows] = await bigquery.query(query);
  rows.forEach(row => console.log(`name: ${row.name}, total: ${row.total}`));
}

runQuery();

Conclusión

Usar APIs y bibliotecas cliente para interactuar con BigQuery permite realizar operaciones de datos poderosas y flexibles. Ya sea que estés usando Python, JavaScript u otro lenguaje, entender cómo autenticar, hacer solicitudes y manejar respuestas es crucial. Al dominar estas habilidades, puedes aprovechar todo el potencial de BigQuery para tus necesidades de análisis de datos.
