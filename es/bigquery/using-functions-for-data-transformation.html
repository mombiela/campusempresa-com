Introducción

En esta sección, exploraremos cómo usar funciones en BigQuery para transformar datos. La transformación de datos es un paso crucial en el análisis y preparación de datos, permitiéndote limpiar, manipular y preparar tus datos para un análisis o reporte posterior.

Conceptos Clave

  • Funciones: Operaciones predefinidas que se pueden aplicar a los datos para realizar tareas específicas.
  • Transformación de Datos: El proceso de convertir datos de un formato o estructura a otro.
  • BigQuery SQL: El dialecto SQL utilizado en Google BigQuery para consultar y manipular datos.

Funciones Comunes en BigQuery

BigQuery proporciona una amplia gama de funciones que se pueden usar para la transformación de datos. Aquí están algunas de las más comúnmente utilizadas:

Funciones de Cadena

  • UPPER(): Convierte una cadena a mayúsculas.
  • LOWER(): Convierte una cadena a minúsculas.
  • CONCAT(): Concatena dos o más cadenas.
  • SUBSTR(): Extrae una subcadena de una cadena.
SELECT 
  UPPER('hello world') AS upper_case,
  LOWER('HELLO WORLD') AS lower_case,
  CONCAT('Hello', ' ', 'World') AS concatenated_string,
  SUBSTR('Hello World', 1, 5) AS substring;

Funciones Numéricas

  • ROUND(): Redondea un número a un número especificado de decimales.
  • CEIL(): Redondea un número hacia arriba al entero más cercano.
  • FLOOR(): Redondea un número hacia abajo al entero más cercano.
  • ABS(): Devuelve el valor absoluto de un número.
SELECT 
  ROUND(3.14159, 2) AS rounded_value,
  CEIL(3.14159) AS ceiling_value,
  FLOOR(3.14159) AS floor_value,
  ABS(-42) AS absolute_value;

Funciones de Fecha

  • CURRENT_DATE(): Devuelve la fecha actual.
  • DATE_ADD(): Añade un intervalo de tiempo especificado a una fecha.
  • DATE_SUB(): Resta un intervalo de tiempo especificado de una fecha.
  • FORMAT_DATE(): Formatea una fecha según un formato especificado.
SELECT 
  CURRENT_DATE() AS current_date,
  DATE_ADD(CURRENT_DATE(), INTERVAL 7 DAY) AS next_week,
  DATE_SUB(CURRENT_DATE(), INTERVAL 1 MONTH) AS last_month,
  FORMAT_DATE('%Y-%m-%d', CURRENT_DATE()) AS formatted_date;

Funciones Condicionales

  • IF(): Devuelve un valor basado en una condición.
  • CASE: Evalúa una lista de condiciones y devuelve un resultado correspondiente.
SELECT 
  IF(1 > 0, 'True', 'False') AS if_example,
  CASE 
    WHEN 1 > 0 THEN 'Positive'
    WHEN 1 < 0 THEN 'Negative'
    ELSE 'Zero'
  END AS case_example;

Transformación Avanzada de Datos

A medida que te sientas más cómodo con las funciones básicas, puedes comenzar a combinarlas para realizar transformaciones más complejas.

Combinando Funciones

Puedes anidar funciones dentro de otras para realizar múltiples transformaciones en una sola consulta.

SELECT 
  UPPER(CONCAT('Hello', ' ', 'World')) AS upper_concatenated_string,
  ROUND(ABS(-3.14159), 2) AS rounded_absolute_value;

Uso de Funciones de Ventana

Las funciones de ventana te permiten realizar cálculos a través de un conjunto de filas de la tabla que están relacionadas con la fila actual.

SELECT 
  name,
  score,
  RANK() OVER (ORDER BY score DESC) AS rank
FROM 
  students;

Pivotando Datos

Pivotar datos es una transformación común que implica convertir filas en columnas.

SELECT 
  name,
  MAX(CASE WHEN subject = 'Math' THEN score END) AS math_score,
  MAX(CASE WHEN subject = 'Science' THEN score END) AS science_score
FROM 
  student_scores
GROUP BY 
  name;

Conclusión

Usar funciones para la transformación de datos en BigQuery es una forma poderosa de manipular y preparar tus datos para el análisis. Al dominar estas funciones, puedes limpiar, formatear y transformar tus datos de manera eficiente. Comienza con funciones básicas y gradualmente avanza a transformaciones más complejas a medida que te sientas más cómodo con BigQuery SQL.
