Introducción

Los bucles e iteraciones son conceptos fundamentales en la programación que te permiten ejecutar un bloque de código múltiples veces. En Scala, hay varias formas de realizar bucles e iteraciones, cada una con sus propios casos de uso y ventajas. Esta sección cubrirá los constructos de bucles básicos y avanzados disponibles en Scala.

Constructos Básicos de Bucle

Bucle `while`

El bucle while ejecuta repetidamente un bloque de código mientras una condición dada sea verdadera.

Sintaxis

while (condition) {
  // bloque de código a ejecutar
}

Ejemplo

var i = 0
while (i < 5) {
  println(s"i = $i")
  i += 1
}

En este ejemplo, el bucle imprimirá el valor de i de 0 a 4.

Bucle `do-while`

El bucle do-while es similar al bucle while, pero garantiza que el bloque de código se ejecute al menos una vez.

Sintaxis

do {
  // bloque de código a ejecutar
} while (condition)

Ejemplo

var i = 0
do {
  println(s"i = $i")
  i += 1
} while (i < 5)

Este bucle también imprimirá el valor de i de 0 a 4, pero el bloque de código se ejecuta antes de que se verifique la condición.

Bucles `for`

Bucle `for` Básico

El bucle for en Scala es más poderoso y flexible en comparación con los bucles for tradicionales en otros lenguajes.

Sintaxis

for (i <- range) {
  // bloque de código a ejecutar
}

Ejemplo

for (i <- 0 until 5) {
  println(s"i = $i")
}

Este bucle imprimirá el valor de i de 0 a 4.

Bucle `for` Anidado

Puedes anidar bucles for para iterar sobre múltiples rangos.

Ejemplo

for (i <- 1 to 3; j <- 1 to 3) {
  println(s"i = $i, j = $j")
}

Esto imprimirá todas las combinaciones de i y j de 1 a 3.

Bucle `for` con Condiciones (Guardas)

Puedes añadir condiciones a los bucles for para filtrar las iteraciones.

Ejemplo

for (i <- 1 to 10 if i % 2 == 0) {
  println(s"Número par: $i")
}

Este bucle imprimirá solo los números pares entre 1 y 10.

Constructos Avanzados de Bucle

Comprensiones `for`

Las comprensiones for son una característica poderosa en Scala que te permiten generar y transformar colecciones.

Ejemplo

val numbers = List(1, 2, 3, 4, 5)
val doubled = for (n <- numbers) yield n * 2
println(doubled)  // Salida: List(2, 4, 6, 8, 10)

En este ejemplo, la comprensión for crea una nueva lista donde cada elemento se duplica.

Uso de `foreach`

El método foreach se utiliza para aplicar una función a cada elemento de una colección.

Ejemplo

val numbers = List(1, 2, 3, 4, 5)
numbers.foreach(n => println(n))

Esto imprimirá cada número en la lista.

Uso de `map`

El método map transforma cada elemento de una colección utilizando una función dada.

Ejemplo

val numbers = List(1, 2, 3, 4, 5)
val squared = numbers.map(n => n * n)
println(squared)  // Salida: List(1, 4, 9, 16, 25)

Este ejemplo crea una nueva lista donde cada elemento se eleva al cuadrado.

Conclusión

Los bucles y las iteraciones son herramientas esenciales en el kit de cualquier programador. Scala ofrece una variedad de constructos de bucles, desde los básicos while y for hasta las avanzadas comprensiones for y métodos de colección como foreach y map. Entender estos constructos te ayudará a escribir código más eficiente y legible. Practica usando estos bucles en diferentes escenarios para volverte competente en la programación en Scala.
