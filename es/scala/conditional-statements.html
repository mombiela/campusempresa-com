Introducción

Las declaraciones condicionales son fundamentales en la programación, permitiendo a los desarrolladores ejecutar diferentes bloques de código basados en ciertas condiciones. En Scala, las declaraciones condicionales son similares a las de otros lenguajes de programación, pero vienen con algunas características y sintaxis únicas. Esta sección cubrirá los conceptos básicos de las declaraciones condicionales en Scala, progresando hacia conceptos más avanzados.

Declaraciones condicionales básicas

Declaración `if`

La declaración if se usa para ejecutar un bloque de código solo si una condición especificada es verdadera.

Sintaxis

if (condition) {
  // código a ejecutar si la condición es verdadera
}

Ejemplo

val x = 10
if (x > 5) {
  println("x es mayor que 5")
}

Declaración `if-else`

La declaración if-else te permite ejecutar un bloque de código si la condición es verdadera y otro bloque si es falsa.

Sintaxis

if (condition) {
  // código a ejecutar si la condición es verdadera
} else {
  // código a ejecutar si la condición es falsa
}

Ejemplo

val x = 3
if (x > 5) {
  println("x es mayor que 5")
} else {
  println("x no es mayor que 5")
}

Escalera `if-else if-else`

Cuando tienes múltiples condiciones que verificar, puedes usar una escalera if-else if-else.

Sintaxis

if (condition1) {
  // código a ejecutar si condition1 es verdadera
} else if (condition2) {
  // código a ejecutar si condition2 es verdadera
} else {
  // código a ejecutar si ambas condiciones son falsas
}

Ejemplo

val x = 7
if (x > 10) {
  println("x es mayor que 10")
} else if (x > 5) {
  println("x es mayor que 5 pero menor o igual a 10")
} else {
  println("x es 5 o menos")
}

Declaraciones condicionales avanzadas

Expresión `if` tipo ternario

Scala no tiene un operador ternario como algunos otros lenguajes (por ejemplo, condition ? trueExpr : falseExpr), pero puedes lograr una funcionalidad similar usando una expresión if.

Sintaxis

val result = if (condition) trueExpr else falseExpr

Ejemplo

val x = 8
val result = if (x > 5) "Mayor" else "Menor"
println(result) // Salida: Mayor

Coincidencia de patrones

La coincidencia de patrones es una característica poderosa en Scala que puede usarse como una alternativa a declaraciones condicionales complejas.

Sintaxis

value match {
  case pattern1 => // código a ejecutar si value coincide con pattern1
  case pattern2 => // código a ejecutar si value coincide con pattern2
  case _ => // código a ejecutar si value no coincide con ninguno de los patrones anteriores
}

Ejemplo

val x = 3
x match {
  case 1 => println("x es 1")
  case 2 => println("x es 2")
  case 3 => println("x es 3")
  case _ => println("x es otra cosa")
}

Condiciones de guardia en coincidencia de patrones

Las condiciones de guardia pueden usarse en la coincidencia de patrones para agregar condiciones adicionales a un caso.

Sintaxis

value match {
  case pattern if condition => // código a ejecutar si pattern coincide y condition es verdadera
  case _ => // código a ejecutar si ningún otro caso coincide
}

Ejemplo

val x = 10
x match {
  case y if y > 5 => println("x es mayor que 5")
  case _ => println("x es 5 o menos")
}

Conclusión

Las declaraciones condicionales en Scala proporcionan una forma robusta de controlar el flujo de tu programa. Desde las declaraciones if básicas hasta la coincidencia de patrones más avanzada, Scala ofrece una variedad de herramientas para manejar diferentes escenarios de manera efectiva. Comprender y dominar estos conceptos mejorará significativamente tu capacidad para escribir código Scala eficiente y legible.
