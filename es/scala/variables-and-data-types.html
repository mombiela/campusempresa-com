Introducción

En Scala, las variables y los tipos de datos son conceptos fundamentales que forman la base de cualquier programa. Entender cómo declarar variables y usar diferentes tipos de datos es esencial para escribir código Scala efectivo y eficiente. Esta sección cubrirá los conceptos básicos de las variables, los diferentes tipos de datos disponibles en Scala y cómo usarlos.

Variables

Las variables en Scala pueden definirse usando dos palabras clave: var y val.

  • var: Define una variable mutable, lo que significa que su valor puede cambiar después de ser inicialmente establecido.
  • val: Define una variable inmutable, lo que significa que su valor no puede cambiar una vez que se establece.

Ejemplos

// Variable mutable
var mutableVar: Int = 10
mutableVar = 20 // Esto está permitido

// Variable inmutable
val immutableVal: Int = 10
// immutableVal = 20 // Esto causará un error de compilación

Conceptos Clave

  • Mutable vs Inmutable: Usa var para variables que necesitan cambiar y val para constantes.
  • Inferencia de Tipos: Scala a menudo puede inferir el tipo de una variable, por lo que las declaraciones de tipo explícitas son opcionales.
var inferredVar = 10 // Scala infiere esto como un Int
val inferredVal = "¡Hola, Scala!" // Scala infiere esto como un String

Tipos de Datos

Scala tiene un conjunto rico de tipos de datos, que pueden categorizarse en:

  • Tipos de Datos Primitivos
  • Tipos de Datos Compuestos

Tipos de Datos Primitivos

Los tipos de datos primitivos son los bloques básicos para la manipulación de datos. Aquí están los tipos de datos primitivos comunes en Scala:

| Tipo de Dato | Descripción | Ejemplo | |--------------|-------------|---------| | Int | Entero con signo de 32 bits | val age: Int = 25 | | Long | Entero con signo de 64 bits | val distance: Long = 123456789L | | Float | Punto flotante de 32 bits | val pi: Float = 3.14f | | Double | Punto flotante de 64 bits | val e: Double = 2.71828 | | Char | Carácter Unicode de 16 bits | val initial: Char = 'A' | | String | Secuencia de caracteres | val name: String = "Scala" | | Boolean | Valor verdadero o falso | val isScalaFun: Boolean = true | | Unit | Representa ningún valor | val result: Unit = () |

Ejemplos

val age: Int = 25
val distance: Long = 123456789L
val pi: Float = 3.14f
val e: Double = 2.71828
val initial: Char = 'A'
val name: String = "Scala"
val isScalaFun: Boolean = true
val result: Unit = ()

Tipos de Datos Compuestos

Los tipos de datos compuestos se construyen a partir de tipos de datos primitivos. Los tipos de datos compuestos más comunes en Scala son:

  • Tuplas: Agrupan múltiples valores de diferentes tipos.
  • Arrays: Secuencias de tamaño fijo de elementos del mismo tipo.
  • Listas: Secuencias inmutables de elementos del mismo tipo.
  • Mapas: Colecciones de pares clave-valor.

Ejemplos

// Tupla
val person: (String, Int) = ("Alice", 30)

// Array
val numbers: Array[Int] = Array(1, 2, 3, 4, 5)

// Lista
val fruits: List[String] = List("Apple", "Banana", "Cherry")

// Mapa
val ages: Map[String, Int] = Map("Alice" -> 30, "Bob" -> 25)

Conclusión

Entender las variables y los tipos de datos es crucial para cualquier programador de Scala. Las variables pueden ser mutables o inmutables, y Scala proporciona una variedad de tipos de datos primitivos y compuestos para manejar diferentes tipos de datos. Al dominar estos conceptos, estarás bien equipado para escribir programas robustos y eficientes en Scala.
