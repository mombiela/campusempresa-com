La coincidencia de patrones es una característica poderosa en Scala que te permite comparar valores contra patrones y descomponer estructuras de datos. Es similar a las declaraciones switch-case en otros lenguajes, pero mucho más poderosa y expresiva.

Introducción a la Coincidencia de Patrones

  • Definición: La coincidencia de patrones es un mecanismo para verificar un valor contra un patrón. Es una versión más poderosa de la declaración switch que se encuentra en muchos otros lenguajes.
  • Casos de Uso: Se usa comúnmente para:
    • Descomponer estructuras de datos
    • Manejar diferentes tipos de datos
    • Implementar flujo de control basado en la coincidencia de patrones

Sintaxis Básica

La sintaxis básica de la coincidencia de patrones en Scala usa la palabra clave match seguida de declaraciones case.

val number = 3

number match {
  case 1 => println("One")
  case 2 => println("Two")
  case 3 => println("Three")
  case _ => println("Other number")
}
  • Explicación:
    • number match { ... }: La palabra clave match se usa para iniciar la coincidencia de patrones.
    • case 1 => println("One"): Cada palabra clave case introduce un patrón para comparar contra el valor.
    • case _ => println("Other number"): El guion bajo _ es un patrón comodín que coincide con cualquier valor.

Coincidencia de Tipos

La coincidencia de patrones también se puede usar para coincidir con tipos, lo cual es útil para manejar diferentes tipos de datos de manera segura.

def process(value: Any): String = value match {
  case i: Int => s"Integer: $i"
  case s: String => s"String: $s"
  case _ => "Unknown type"
}

println(process(42))        // Output: Integer: 42
println(process("Scala"))   // Output: String: Scala
println(process(3.14))      // Output: Unknown type
  • Explicación:
    • case i: Int => ...: Coincide si value es de tipo Int.
    • case s: String => ...: Coincide si value es de tipo String.
    • case _ => ...: Coincide con cualquier otro tipo.

Descomposición de Estructuras de Datos

La coincidencia de patrones es particularmente poderosa cuando se trabaja con estructuras de datos complejas como tuplas, listas y clases de caso.

Tuplas

val tuple = (1, "Scala")

tuple match {
  case (1, lang) => println(s"Language: $lang")
  case (2, lang) => println(s"Second language: $lang")
  case _ => println("Other tuple")
}
  • Explicación:
    • case (1, lang) => ...: Coincide con una tupla donde el primer elemento es 1 y asigna el segundo elemento a lang.

Listas

val list = List(1, 2, 3)

list match {
  case List(1, 2, 3) => println("Exact match")
  case List(1, _*) => println("Starts with 1")
  case _ => println("Other list")
}
  • Explicación:
    • case List(1, 2, 3) => ...: Coincide con una lista con exactamente tres elementos: 1, 2 y 3.
    • case List(1, _*) => ...: Coincide con una lista que comienza con 1 seguida de cualquier número de elementos.

Clases de Caso

case class Person(name: String, age: Int)

val person = Person("Alice", 25)

person match {
  case Person("Alice", age) => println(s"Alice is $age years old")
  case Person(name, age) => println(s"$name is $age years old")
  case _ => println("Unknown person")
}
  • Explicación:
    • case Person("Alice", age) => ...: Coincide con un objeto Person con el nombre "Alice" y asigna la edad a age.
    • case Person(name, age) => ...: Coincide con cualquier objeto Person y asigna el nombre y la edad a name y age.

Coincidencia de Patrones Avanzada

Patrones Anidados

La coincidencia de patrones puede anidarse para manejar estructuras de datos más complejas.

val nestedList = List(List(1, 2), List(3, 4))

nestedList match {
  case List(List(1, 2), List(a, b)) => println(s"Nested match: $a, $b")
  case _ => println("Other nested list")
}
  • Explicación:
    • case List(List(1, 2), List(a, b)) => ...: Coincide con una lista que contiene dos listas, donde la primera lista es exactamente List(1, 2) y la segunda lista tiene dos elementos que se asignan a a y b.

Guardias

Las guardias añaden condiciones adicionales a los patrones.

val number = 5

number match {
  case x if x % 2 == 0 => println("Even number")
  case x if x % 2 != 0 => println("Odd number")
  case _ => println("Unknown number")
}
  • Explicación:
    • case x if x % 2 == 0 => ...: Coincide si x es un número par.
    • case x if x % 2 != 0 => ...: Coincide si x es un número impar.

Resumen

La coincidencia de patrones en Scala es una característica versátil y poderosa que te permite:

  • Coincidir valores contra patrones.
  • Descomponer estructuras de datos complejas.
  • Manejar diferentes tipos de datos de manera segura.
  • Implementar flujo de control basado en la coincidencia de patrones.

Al dominar la coincidencia de patrones, puedes escribir código Scala más expresivo y conciso, lo que facilita el manejo de una amplia gama de escenarios de programación.
