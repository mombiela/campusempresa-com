Introducción

La herencia y el polimorfismo son conceptos fundamentales en la programación orientada a objetos (POO). Permiten la creación de código flexible y reutilizable. En Scala, estos conceptos se implementan de una manera que aprovecha tanto los paradigmas funcionales como los orientados a objetos.

Herencia

La herencia permite que una clase herede propiedades y métodos de otra clase. La clase que hereda se llama subclase, y la clase de la que se hereda se llama superclase.

Conceptos Clave

  • Superclase: La clase cuyas propiedades y métodos son heredados.
  • Subclase: La clase que hereda propiedades y métodos de la superclase.
  • Palabra clave extends: Se usa para denotar la herencia.

Ejemplo

// Superclase
class Animal {
  def eat(): Unit = {
    println("Este animal está comiendo")
  }
}

// Subclase
class Dog extends Animal {
  def bark(): Unit = {
    println("El perro está ladrando")
  }
}

// Uso
val dog = new Dog
dog.eat()  // Método heredado
dog.bark() // Método específico de la subclase

En este ejemplo:

  • Animal es la superclase.
  • Dog es la subclase que hereda de Animal.
  • La clase Dog puede usar el método eat de la clase Animal.

Polimorfismo

El polimorfismo permite que los métodos hagan cosas diferentes según el objeto sobre el que actúan, aunque compartan el mismo nombre. En Scala, el polimorfismo se logra mediante la sobrescritura de métodos y el uso de traits.

Conceptos Clave

  • Sobrescritura de Métodos: Una subclase proporciona una implementación específica de un método que ya está definido en su superclase.
  • Traits: Los traits son similares a las interfaces en Java y se pueden usar para lograr polimorfismo.

Ejemplo de Sobrescritura de Métodos

class Animal {
  def sound(): Unit = {
    println("Algún sonido genérico de animal")
  }
}

class Dog extends Animal {
  override def sound(): Unit = {
    println("Ladrido")
  }
}

class Cat extends Animal {
  override def sound(): Unit = {
    println("Maullido")
  }
}

// Uso
val animals: List[Animal] = List(new Dog, new Cat)
animals.foreach(_.sound())

En este ejemplo:

  • El método sound se sobrescribe tanto en las clases Dog como Cat.
  • Cuando se llama a sound en cada animal, se ejecuta la implementación específica para ese animal.

Ejemplo de Traits

trait Speaker {
  def speak(): Unit
}

class Human extends Speaker {
  def speak(): Unit = {
    println("¡Hola!")
  }
}

class Robot extends Speaker {
  def speak(): Unit = {
    println("Bip bop")
  }
}

// Uso
val speakers: List[Speaker] = List(new Human, new Robot)
speakers.foreach(_.speak())

En este ejemplo:

  • Speaker es un trait con un método abstracto speak.
  • Tanto las clases Human como Robot implementan el método speak.
  • El método speak se llama polimórficamente en cada instancia de la lista.

Conclusión

La herencia y el polimorfismo son características poderosas en Scala que permiten la reutilización y flexibilidad del código. Al comprender cómo usar estos conceptos, puedes crear aplicaciones más mantenibles y escalables. Recuerda:

  • Usa la herencia para compartir código común entre clases.
  • Usa el polimorfismo para permitir que los métodos operen de manera diferente según el objeto sobre el que actúan.
  • Aprovecha los traits para lograr polimorfismo y herencia múltiple en Scala.
