Introducción

En Google Data Studio, los parámetros son una herramienta poderosa que permite a los usuarios interactuar con los datos de manera dinámica. Los parámetros pueden ser utilizados para filtrar datos, cambiar visualizaciones y personalizar informes según las necesidades del usuario final.

Conceptos Clave

  • Parámetro: Una variable que puede ser utilizada para cambiar dinámicamente el contenido de un informe.
  • Control de Parámetro: Un componente de la interfaz de usuario que permite a los usuarios establecer el valor de un parámetro.
  • Campo Calculado: Un campo que utiliza el valor del parámetro para realizar cálculos o filtrar datos.

Creación de Parámetros

Para crear un parámetro en Google Data Studio, sigue estos pasos:

  1. Abre tu informe en Google Data Studio.
  2. Haz clic en "Recursos" en la barra de menú.
  3. Selecciona "Administrar parámetros".
  4. Haz clic en "Agregar un parámetro".
  5. Define el nombre, tipo de datos y valor predeterminado del parámetro.

Ejemplo de Creación de Parámetro

Nombre: Rango de Fechas
Tipo de Datos: Fecha
Valor Predeterminado: 2023-01-01

Uso de Parámetros en Campos Calculados

Una vez que has creado un parámetro, puedes utilizarlo en campos calculados para filtrar o transformar datos.

Ejemplo de Campo Calculado

Supongamos que tienes un parámetro llamado Rango de Fechas. Puedes crear un campo calculado para filtrar los datos según este parámetro:

CASE
  WHEN Fecha >= @Rango_de_Fechas THEN "Dentro del Rango"
  ELSE "Fuera del Rango"
END

En este ejemplo, el campo calculado clasifica las fechas como "Dentro del Rango" o "Fuera del Rango" según el valor del parámetro Rango de Fechas.

Controles de Parámetro

Para permitir que los usuarios cambien el valor de un parámetro, debes agregar un control de parámetro a tu informe.

Pasos para Agregar un Control de Parámetro

  1. Selecciona el informe donde deseas agregar el control.
  2. Haz clic en "Agregar un control" en la barra de herramientas.
  3. Selecciona "Control de Parámetro".
  4. Configura el control para que esté vinculado al parámetro que has creado.

Ejemplo de Control de Parámetro

Si has creado un parámetro Rango de Fechas, puedes agregar un control de parámetro de tipo "Selector de Fecha" para permitir a los usuarios seleccionar una fecha.

Ejemplos Prácticos

Ejemplo 1: Filtrado Dinámico de Datos

Supongamos que tienes un informe de ventas y deseas permitir a los usuarios filtrar las ventas por un rango de fechas específico.

  1. Crea un parámetro Fecha Inicio y otro Fecha Fin.
  2. Utiliza estos parámetros en un campo calculado para filtrar las ventas:
CASE
  WHEN Fecha >= @Fecha_Inicio AND Fecha <= @Fecha_Fin THEN Ventas
  ELSE NULL
END
  1. Agrega controles de parámetro para Fecha Inicio y Fecha Fin en tu informe.

Ejemplo 2: Cambio de Visualización

Puedes utilizar parámetros para cambiar dinámicamente el tipo de visualización en tu informe.

  1. Crea un parámetro Tipo de Visualización con valores posibles como "Gráfico de Barras" y "Gráfico de Líneas".
  2. Utiliza este parámetro en un campo calculado para cambiar la visualización:
CASE
  WHEN @Tipo_de_Visualización = "Gráfico de Barras" THEN "Bar"
  WHEN @Tipo_de_Visualización = "Gráfico de Líneas" THEN "Line"
  ELSE "Bar"
END
  1. Agrega un control de parámetro para Tipo de Visualización en tu informe.

Conclusión

El uso de parámetros en Google Data Studio permite a los usuarios crear informes interactivos y personalizados. Al comprender cómo crear y utilizar parámetros, puedes ofrecer a los usuarios finales una experiencia más dinámica y flexible. Practica creando y utilizando parámetros en tus propios informes para aprovechar al máximo esta funcionalidad.
