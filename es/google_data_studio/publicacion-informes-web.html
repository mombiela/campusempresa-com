Introducción

En esta sección, aprenderás cómo publicar tus informes de Google Data Studio en la web. Publicar informes permite compartir tus análisis y visualizaciones con un público más amplio, facilitando la toma de decisiones basada en datos.

Pasos para publicar un informe

Para publicar un informe en Google Data Studio, sigue estos pasos:

  1. Abre tu informe en Google Data Studio:

    • Navega a Google Data Studio.
    • Selecciona el informe que deseas publicar.

  2. Configura los permisos de acceso:

    • Haz clic en el botón de "Compartir" en la esquina superior derecha.
    • Configura los permisos de acceso según tus necesidades:
      • Privado: Solo las personas específicas pueden ver el informe.
      • Cualquiera con el enlace: Cualquier persona con el enlace puede ver el informe.
      • Público en la web: Cualquier persona en la web puede encontrar y ver el informe.

  3. Publica el informe:

    • Haz clic en "Archivo" en el menú superior.
    • Selecciona "Incorporar informe".
    • Copia el código de inserción proporcionado.

  4. Incorpora el informe en tu sitio web:

    • Abre el editor de tu sitio web.
    • Pega el código de inserción en el lugar donde deseas que aparezca el informe.

Ejemplo de código de inserción

A continuación, se muestra un ejemplo de cómo se vería el código de inserción de un informe de Google Data Studio:

<iframe width="600" height="450" src="https://datastudio.google.com/embed/reporting/1A2B3C4D5E6F7G8H9I0J/page/1M" frameborder="0" style="border:0" allowfullscreen></iframe>

Explicación del código

  • <iframe>: Este es el elemento HTML utilizado para incrustar contenido de otra fuente en una página web.
  • width y height: Estas propiedades definen el tamaño del iframe.
  • src: Esta propiedad contiene la URL del informe de Google Data Studio que deseas incrustar.
  • frameborder: Esta propiedad define si el iframe tendrá un borde. Un valor de "0" significa que no habrá borde.
  • style: Aquí puedes agregar estilos CSS adicionales. En este caso, se establece border:0 para asegurarse de que no haya borde.
  • allowfullscreen: Esta propiedad permite que el iframe se vea en pantalla completa.

Consideraciones de seguridad

Al publicar informes en la web, es importante tener en cuenta las siguientes consideraciones de seguridad:

  • Datos sensibles: Asegúrate de que el informe no contenga datos sensibles o confidenciales antes de hacerlo público.
  • Permisos de acceso: Configura los permisos de acceso adecuadamente para controlar quién puede ver el informe.
  • Actualizaciones: Ten en cuenta que cualquier cambio en el informe se reflejará automáticamente en la versión publicada.

Conclusión

Publicar informes en la web con Google Data Studio es una excelente manera de compartir tus análisis y visualizaciones con un público más amplio. Siguiendo los pasos descritos, puedes configurar los permisos de acceso y utilizar el código de inserción para incorporar tus informes en cualquier sitio web. Recuerda siempre considerar la seguridad y privacidad de los datos antes de hacer un informe público.
