Introducción

En esta sección del curso sobre Google Data Studio, aprenderás cómo conectar y utilizar diversas fuentes de datos externas. Google Data Studio permite integrar datos de múltiples orígenes, lo que facilita la creación de informes y dashboards completos y personalizados.

Fuentes de Datos Compatibles

Google Data Studio es compatible con una amplia variedad de fuentes de datos. Algunas de las más comunes incluyen:

  • Google Sheets
  • Google Analytics
  • BigQuery
  • MySQL
  • PostgreSQL
  • CSV Uploads
  • Facebook Ads
  • YouTube Analytics
  • Salesforce

Conectando Google Sheets

Google Sheets es una de las fuentes de datos más populares debido a su simplicidad y accesibilidad. Para conectar Google Sheets a Google Data Studio, sigue estos pasos:

  1. Abrir Google Data Studio: Inicia sesión en tu cuenta de Google Data Studio.
  2. Crear una Fuente de Datos: Haz clic en el botón "Crear" y selecciona "Fuente de datos".
  3. Seleccionar Google Sheets: En la lista de conectores, selecciona "Google Sheets".
  4. Autorizar el Acceso: Autoriza a Google Data Studio para acceder a tus hojas de cálculo de Google.
  5. Seleccionar el Archivo y la Hoja: Elige el archivo de Google Sheets y la hoja específica que deseas conectar.
  6. Configurar los Campos: Revisa y configura los campos de datos según sea necesario.
# Ejemplo de configuración de campos
{
  "fieldName": "Fecha",
  "fieldType": "DATE"
},
{
  "fieldName": "Ventas",
  "fieldType": "NUMBER"
}

Conectando BigQuery

BigQuery es una poderosa herramienta de análisis de datos de Google Cloud. Para conectar BigQuery a Google Data Studio:

  1. Abrir Google Data Studio: Inicia sesión en tu cuenta de Google Data Studio.
  2. Crear una Fuente de Datos: Haz clic en el botón "Crear" y selecciona "Fuente de datos".
  3. Seleccionar BigQuery: En la lista de conectores, selecciona "BigQuery".
  4. Autorizar el Acceso: Autoriza a Google Data Studio para acceder a tus proyectos de BigQuery.
  5. Seleccionar el Proyecto y el Conjunto de Datos: Elige el proyecto y el conjunto de datos que deseas conectar.
  6. Configurar los Campos: Revisa y configura los campos de datos según sea necesario.
# Ejemplo de consulta SQL en BigQuery
SELECT
  fecha,
  ventas
FROM
  `mi_proyecto.mi_dataset.mi_tabla`
WHERE
  fecha BETWEEN '2023-01-01' AND '2023-12-31'

Conectando MySQL

Para conectar una base de datos MySQL a Google Data Studio:

  1. Abrir Google Data Studio: Inicia sesión en tu cuenta de Google Data Studio.
  2. Crear una Fuente de Datos: Haz clic en el botón "Crear" y selecciona "Fuente de datos".
  3. Seleccionar MySQL: En la lista de conectores, selecciona "MySQL".
  4. Configurar la Conexión: Introduce los detalles de la conexión, como la dirección del servidor, el puerto, el nombre de usuario y la contraseña.
  5. Seleccionar la Base de Datos y la Tabla: Elige la base de datos y la tabla que deseas conectar.
  6. Configurar los Campos: Revisa y configura los campos de datos según sea necesario.
# Ejemplo de consulta SQL en MySQL
SELECT
  fecha,
  ventas
FROM
  mi_tabla
WHERE
  fecha BETWEEN '2023-01-01' AND '2023-12-31'

Ejercicios Prácticos

  1. Conectar una Hoja de Google Sheets: Crea una nueva fuente de datos en Google Data Studio utilizando una hoja de Google Sheets que contenga datos de ventas mensuales.
  2. Conectar a BigQuery: Configura una conexión a BigQuery y realiza una consulta para obtener datos de ventas anuales.
  3. Conectar a MySQL: Establece una conexión a una base de datos MySQL y extrae datos de una tabla específica.

Conclusión

Conectar diversas fuentes de datos a Google Data Studio te permite crear informes y dashboards más completos y personalizados. Ya sea que utilices Google Sheets, BigQuery, MySQL u otras fuentes, el proceso de conexión es relativamente sencillo y te brinda la flexibilidad de trabajar con los datos que necesitas. Practica conectando diferentes fuentes de datos para familiarizarte con el proceso y maximizar el potencial de tus informes en Google Data Studio.
