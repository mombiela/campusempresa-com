Introducción

En esta sección, aprenderás las operaciones básicas en SQL que son fundamentales para interactuar con bases de datos relacionales. Estas operaciones incluyen la creación, lectura, actualización y eliminación de datos, comúnmente conocidas por sus siglas en inglés como CRUD (Create, Read, Update, Delete).

Creación de Datos (INSERT)

La operación INSERT se utiliza para agregar nuevos registros a una tabla en la base de datos.

Sintaxis Básica

INSERT INTO nombre_tabla (columna1, columna2, columna3, ...)
VALUES (valor1, valor2, valor3, ...);

Ejemplo

Supongamos que tenemos una tabla llamada empleados con las columnas id, nombre, edad y departamento.

INSERT INTO empleados (id, nombre, edad, departamento)
VALUES (1, 'Juan Pérez', 30, 'Recursos Humanos');

Lectura de Datos (SELECT)

La operación SELECT se utiliza para consultar y recuperar datos de una tabla.

Sintaxis Básica

SELECT columna1, columna2, ...
FROM nombre_tabla
WHERE condición;

Ejemplo

Para seleccionar todos los empleados del departamento de "Recursos Humanos":

SELECT nombre, edad
FROM empleados
WHERE departamento = 'Recursos Humanos';

Actualización de Datos (UPDATE)

La operación UPDATE se utiliza para modificar los datos existentes en una tabla.

Sintaxis Básica

UPDATE nombre_tabla
SET columna1 = valor1, columna2 = valor2, ...
WHERE condición;

Ejemplo

Para actualizar la edad de "Juan Pérez" a 31:

UPDATE empleados
SET edad = 31
WHERE nombre = 'Juan Pérez';

Eliminación de Datos (DELETE)

La operación DELETE se utiliza para eliminar registros de una tabla.

Sintaxis Básica

DELETE FROM nombre_tabla
WHERE condición;

Ejemplo

Para eliminar a "Juan Pérez" de la tabla empleados:

DELETE FROM empleados
WHERE nombre = 'Juan Pérez';

Comparación de Operaciones CRUD

| Operación | Descripción | Sintaxis Básica | |-----------|-------------|-----------------| | INSERT | Agrega nuevos registros a una tabla | INSERT INTO nombre_tabla (col1, col2, ...) VALUES (val1, val2, ...); | | SELECT | Recupera datos de una tabla | SELECT col1, col2, ... FROM nombre_tabla WHERE condición; | | UPDATE | Modifica datos existentes en una tabla | UPDATE nombre_tabla SET col1 = val1, col2 = val2, ... WHERE condición; | | DELETE | Elimina registros de una tabla | DELETE FROM nombre_tabla WHERE condición; |

Conclusión

Las operaciones básicas en SQL son esenciales para cualquier profesional de TI que trabaje con bases de datos. Dominar INSERT, SELECT, UPDATE y DELETE te permitirá gestionar y manipular datos de manera efectiva. Practica estos comandos con diferentes escenarios y condiciones para fortalecer tu comprensión y habilidades en SQL.
