Escribiendo Tu Primer Script Perl CGI

  • Creación de un simple script Perl CGI.
  • Explicación de la estructura del script.
  • Ejecución del script en un servidor web.
#!/usr/bin/perl
use strict;
use warnings;
use CGI;

my $cgi = CGI->new;
print $cgi->header;
print $cgi->start_html('Hello World');
print $cgi->h1('Hello World from Perl CGI!');
print $cgi->end_html;

Manejo de Datos de Formularios

  • Creación de formularios HTML.
  • Recuperación de datos de formularios usando Perl CGI.
  • Validación y procesamiento de la entrada del formulario.
#!/usr/bin/perl
use strict;
use warnings;
use CGI;

my $cgi = CGI->new;
print $cgi->header;
print $cgi->start_html('Form Example');
print $cgi->start_form;
print $cgi->textfield(-name=>'name');
print $cgi->submit;
print $cgi->end_form;

if ($cgi->param) {
    my $name = $cgi->param('name');
    print $cgi->h1("Hello, $name!");
}
print $cgi->end_html;

Integración con Bases de Datos

  • Conexión a una base de datos usando DBI.
  • Realización de operaciones CRUD.
  • Uso de ORM (Mapeo Objeto-Relacional) con DBIx::Class.
use DBI;

my $dbh = DBI->connect("DBI:mysql:database=testdb;host=localhost", "user", "password", {'RaiseError' => 1});

my $sth = $dbh->prepare("SELECT * FROM users");
$sth->execute();

while (my @row = $sth->fetchrow_array) {
    print "User: $row[0], Email: $row[1]\n";
}

$sth->finish;
$dbh->disconnect;
