Introducción

Perl, conocido por sus capacidades de procesamiento de texto, es un lenguaje de programación versátil y poderoso. Entender su sintaxis y estructura básica es esencial para escribir código eficiente y legible. Esta sección cubrirá los elementos fundamentales de la sintaxis de Perl, incluyendo variables, operadores, estructuras de control y operaciones básicas de entrada/salida.

Variables

Las variables en Perl se utilizan para almacenar datos. Hay tres tipos principales de variables:

  • Escalares: Almacenan valores únicos (números, cadenas).
  • Arreglos: Almacenan listas ordenadas de escalares.
  • Hashes: Almacenan conjuntos de pares clave-valor.

Escalares

Los escalares se prefijan con un signo de dólar ($).

# Variables escalares
my $name = "John";
my $age = 30;
my $pi = 3.14;

Arreglos

Los arreglos se prefijan con un signo de arroba (@).

# Variables de arreglo
my @colors = ("red", "green", "blue");
print $colors[0]; # Salida: red

Hashes

Los hashes se prefijan con un signo de porcentaje (%).

# Variables de hash
my %fruit_colors = (
    apple  => "red",
    banana => "yellow",
    grape  => "purple"
);
print $fruit_colors{"apple"}; # Salida: red

Operadores

Perl soporta varios operadores para realizar operaciones en variables.

Operadores Aritméticos

  • Adición: +
  • Sustracción: -
  • Multiplicación: *
  • División: /
  • Módulo: %
my $sum = 5 + 10; # 15
my $product = 4 * 3; # 12

Operadores de Cadenas

  • Concatenación: .
my $greeting = "Hello, " . "world!"; # Hello, world!

Operadores de Comparación

  • Numéricos: ==, !=, <, >, <=, >=
  • Cadenas: eq, ne, lt, gt, le, ge
if (5 == 5) {
    print "Los números son iguales\n";
}

if ("apple" eq "apple") {
    print "Las cadenas son iguales\n";
}

Estructuras de Control

Las estructuras de control gestionan el flujo del programa.

Declaraciones Condicionales

  • if, elsif, else
my $number = 10;

if ($number > 10) {
    print "El número es mayor que 10\n";
} elsif ($number == 10) {
    print "El número es igual a 10\n";
} else {
    print "El número es menor que 10\n";
}

Declaraciones de Bucle

  • while
  • for
  • foreach
# Bucle while
my $i = 0;
while ($i < 5) {
    print "i = $i\n";
    $i++;
}

# Bucle for
for (my $j = 0; $j < 5; $j++) {
    print "j = $j\n";
}

# Bucle foreach
my @numbers = (1, 2, 3, 4, 5);
foreach my $num (@numbers) {
    print "num = $num\n";
}

Entrada y Salida

Las operaciones básicas de entrada y salida son esenciales para interactuar con usuarios y archivos.

Entrada Estándar

Usando el manejador de archivos <STDIN> para leer la entrada del usuario.

print "Introduce tu nombre: ";
my $name = <STDIN>;
chomp($name); # Eliminar el carácter de nueva línea
print "¡Hola, $name!\n";

Salida Estándar

Usando la función print para mostrar la salida.

print "Este es un mensaje simple\n";

Conclusión

Entender la sintaxis y estructura básica de Perl es el primer paso para dominar el lenguaje. Esta sección cubrió lo esencial, incluyendo variables, operadores, estructuras de control y operaciones básicas de entrada/salida. Con estos fundamentos, puedes comenzar a escribir scripts simples en Perl y gradualmente avanzar a tareas más complejas. Continúa practicando y explorando para profundizar tu comprensión y competencia en Perl.
