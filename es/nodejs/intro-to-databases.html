Resumen

En esta sección, exploraremos los fundamentos de las bases de datos y cómo interactuar con ellas usando Node.js. Las bases de datos son esenciales para almacenar, recuperar y gestionar datos en aplicaciones web. Cubriremos tanto bases de datos SQL como NoSQL, sus diferencias y cómo integrarlas con Node.js.

Conceptos Clave

  • Base de Datos: Una colección estructurada de datos.
  • SQL (Structured Query Language): Un lenguaje utilizado para gestionar y manipular bases de datos relacionales.
  • NoSQL (Not Only SQL): Una categoría de sistemas de gestión de bases de datos que no utilizan SQL como su lenguaje de consulta principal.
  • Operaciones CRUD: Las cuatro operaciones básicas para gestionar datos - Crear, Leer, Actualizar, Eliminar.

Tipos de Bases de Datos

Bases de Datos SQL

  • Ejemplos: MySQL, PostgreSQL, SQLite
  • Características:
    • Usan tablas para almacenar datos.
    • Los datos están estructurados en filas y columnas.
    • Soporte para transacciones ACID (Atomicidad, Consistencia, Aislamiento, Durabilidad).

Bases de Datos NoSQL

  • Ejemplos: MongoDB, CouchDB, Redis
  • Características:
    • Usan varios modelos de datos como documento, clave-valor, gráfico o familia de columnas.
    • Sin esquema o esquema flexible.
    • Diseñadas para escalabilidad y rendimiento.

Configuración de una Base de Datos con Node.js

Instalación de Paquetes Requeridos

Para interactuar con bases de datos en Node.js, necesitas instalar paquetes específicos. Aquí hay algunos comunes:

  • Para MySQL:

    npm install mysql

  • Para MongoDB:

    npm install mongodb

Conectando a una Base de Datos

Ejemplo de MySQL

const mysql = require('mysql');

const connection = mysql.createConnection({
  host: 'localhost',
  user: 'root',
  password: 'password',
  database: 'test_db'
});

connection.connect((err) => {
  if (err) {
    console.error('Error connecting: ' + err.stack);
    return;
  }
  console.log('Connected as id ' + connection.threadId);
});
  • Explicación:
    • mysql.createConnection: Crea una conexión a la base de datos MySQL.
    • connection.connect: Establece la conexión.

Ejemplo de MongoDB

const { MongoClient } = require('mongodb');

const url = 'mongodb://localhost:27017';
const dbName = 'test_db';

MongoClient.connect(url, { useNewUrlParser: true, useUnifiedTopology: true }, (err, client) => {
  if (err) {
    console.error('Error connecting: ' + err.stack);
    return;
  }
  console.log('Connected successfully to server');
  const db = client.db(dbName);
  client.close();
});
  • Explicación:
    • MongoClient.connect: Conecta al servidor de MongoDB.
    • client.db: Accede a la base de datos específica.

Realizando Operaciones CRUD

Crear

MySQL

const sql = 'INSERT INTO users (name, age) VALUES (?, ?)';
const values = ['John Doe', 30];

connection.query(sql, values, (err, result) => {
  if (err) throw err;
  console.log('Record inserted, ID: ' + result.insertId);
});

MongoDB

const collection = db.collection('users');
collection.insertOne({ name: 'John Doe', age: 30 }, (err, result) => {
  if (err) throw err;
  console.log('Record inserted, ID: ' + result.insertedId);
});

Leer

MySQL

const sql = 'SELECT * FROM users WHERE age > ?';
const age = 25;

connection.query(sql, [age], (err, results) => {
  if (err) throw err;
  console.log(results);
});

MongoDB

const collection = db.collection('users');
collection.find({ age: { $gt: 25 } }).toArray((err, docs) => {
  if (err) throw err;
  console.log(docs);
});

Actualizar

MySQL

const sql = 'UPDATE users SET age = ? WHERE name = ?';
const values = [35, 'John Doe'];

connection.query(sql, values, (err, result) => {
  if (err) throw err;
  console.log('Record updated, affected rows: ' + result.affectedRows);
});

MongoDB

const collection = db.collection('users');
collection.updateOne({ name: 'John Doe' }, { $set: { age: 35 } }, (err, result) => {
  if (err) throw err;
  console.log('Record updated, modified count: ' + result.modifiedCount);
});

Eliminar

MySQL

const sql = 'DELETE FROM users WHERE name = ?';
const name = 'John Doe';

connection.query(sql, [name], (err, result) => {
  if (err) throw err;
  console.log('Record deleted, affected rows: ' + result.affectedRows);
});

MongoDB

const collection = db.collection('users');
collection.deleteOne({ name: 'John Doe' }, (err, result) => {
  if (err) throw err;
  console.log('Record deleted, deleted count: ' + result.deletedCount);
});

Conclusión

En esta sección, cubrimos los conceptos básicos de las bases de datos, incluyendo las diferencias entre bases de datos SQL y NoSQL. También demostramos cómo configurar y conectar ambos tipos de bases de datos usando Node.js, y cómo realizar operaciones CRUD. Entender estos fundamentos es crucial para construir aplicaciones web robustas y escalables.
