Introducción a Node.js

  • ¿Qué es Node.js?
    • Node.js es un entorno de ejecución de JavaScript construido sobre el motor de JavaScript V8 de Chrome.
    • Permite ejecutar JavaScript en el lado del servidor.
  • ¿Por qué usar Node.js para el desarrollo backend?
    • Arquitectura no bloqueante y basada en eventos.
    • Alto rendimiento para tareas dependientes de I/O.
    • Gran ecosistema con npm (Node Package Manager).

Configuración de tu Entorno Node.js

  • Instalación de Node.js y npm
    • Descarga e instala desde el sitio web oficial de Node.js.
    • Verifica la instalación: 
      node -v
npm -v
  • Configuración de un proyecto
    • Inicializa un nuevo proyecto Node.js: 
      mkdir my-node-app
cd my-node-app
npm init -y

Conceptos Básicos

  • Módulos y require
    • Creación y uso de módulos: 
      // math.js
function add(a, b) {
  return a + b;
}
module.exports = add;

// app.js
const add = require('./math');
console.log(add(2, 3)); // Output: 5
  • Programación asíncrona
    • Callbacks: 
      const fs = require('fs');
fs.readFile('example.txt', 'utf8', (err, data) => {
  if (err) throw err;
  console.log(data);
});
    • Promesas: 
      const fs = require('fs').promises;
fs.readFile('example.txt', 'utf8')
  .then(data => console.log(data))
  .catch(err => console.error(err));
    • Async/Await: 
      const fs = require('fs').promises;
async function readFile() {
  try {
    const data = await fs.readFile('example.txt', 'utf8');
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}
readFile();

Construcción de un Servidor Web Simple

  • Uso del módulo HTTP
    • Creación de un servidor básico: 
      const http = require('http');
const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});
server.listen(3000, '127.0.0.1', () => {
  console.log('Server running at http://127.0.0.1:3000/');
});

Framework Express.js

  • Introducción a Express.js
    • Simplifica la creación de servidores y el enrutamiento.
    • Instalación: 
      npm install express
  • Uso básico
    • Creación de un servidor Express: 
      const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello, World!');
});

app.listen(port, () => {
  console.log(`Server running at http://localhost:${port}/`);
});
  • Middleware
    • Uso de middleware para el procesamiento de solicitudes: 
      app.use((req, res, next) => {
  console.log('Time:', Date.now());
  next();
});

Trabajo con Bases de Datos

  • Conexión a una base de datos MongoDB usando Mongoose
    • Instalación: 
      npm install mongoose
    • Uso básico: 
      const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost/mydatabase', { useNewUrlParser: true, useUnifiedTopology: true });

const db = mongoose.connection;
db.on('error', console.error.bind(console, 'connection error:'));
db.once('open', () => {
  console.log('Connected to the database');
});
  • Definición de esquemas y modelos
    • Ejemplo de esquema y modelo: 
      const userSchema = new mongoose.Schema({
  name: String,
  age: Number,
  email: String
});

const User = mongoose.model('User', userSchema);

const newUser = new User({ name: 'John Doe', age: 30, email: 'john@example.com' });
newUser.save((err) => {
  if (err) return console.error(err);
  console.log('User saved successfully');
});

Temas Avanzados

  • Autenticación con JWT (JSON Web Tokens)
    • Instalación: 
      npm install jsonwebtoken
    • Generación y verificación de tokens: 
      const jwt = require('jsonwebtoken');
const secretKey = 'your-secret-key';

// Generación de un token
const token = jwt.sign({ username: 'john_doe' }, secretKey, { expiresIn: '1h' });
console.log('Token:', token);

// Verificación de un token
jwt.verify(token, secretKey, (err, decoded) => {
  if (err) {
    console.error('Token verification failed:', err);
  } else {
    console.log('Decoded token:', decoded);
  }
});
  • Comunicación en tiempo real con Socket.io
    • Instalación: 
      npm install socket.io
    • Uso básico: 
      const http = require('http');
const socketIo = require('socket.io');
const server = http.createServer();
const io = socketIo(server);

io.on('connection', (socket) => {
  console.log('A user connected');
  socket.on('disconnect', () => {
    console.log('User disconnected');
  });
});

server.listen(3000, () => {
  console.log('Server running at http://localhost:3000/');
});
