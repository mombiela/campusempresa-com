El enrutamiento es un concepto fundamental en Express, un popular framework de aplicaciones web para Node.js. Se refiere a cómo los endpoints (URIs) de una aplicación responden a las solicitudes del cliente. Este tema te guiará desde los conceptos básicos del enrutamiento hasta conceptos más avanzados.

Introducción al Enrutamiento

El enrutamiento en Express implica definir cómo tu aplicación responde a varias solicitudes HTTP. Cada ruta puede tener una o más funciones manejadoras, que se ejecutan cuando la ruta coincide.

Enrutamiento Básico

  • Definición de Ruta: Una ruta se define por un método HTTP, un camino y una función manejadora.
  • Métodos HTTP: Los métodos comunes incluyen GET, POST, PUT, DELETE, etc.

Ejemplo: Ruta Básica GET

const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('¡Hola Mundo!');
});

app.listen(3000, () => {
  console.log('El servidor está corriendo en el puerto 3000');
});

En este ejemplo:

  • app.get('/') define una ruta que escucha solicitudes GET en la URL raíz (/).
  • La función manejadora envía una respuesta "¡Hola Mundo!".

Parámetros de Ruta

Los parámetros de ruta son segmentos de URL nombrados que se utilizan para capturar valores en posiciones específicas en la URL.

Ejemplo: Parámetros de Ruta

app.get('/users/:userId', (req, res) => {
  const userId = req.params.userId;
  res.send(`ID de Usuario: ${userId}`);
});

En este ejemplo:

  • :userId es un parámetro de ruta.
  • req.params.userId captura el valor de userId de la URL.

Parámetros de Consulta

Los parámetros de consulta se utilizan para pasar datos al servidor como parte de la URL.

Ejemplo: Parámetros de Consulta

app.get('/search', (req, res) => {
  const query = req.query.q;
  res.send(`Consulta de búsqueda: ${query}`);
});

En este ejemplo:

  • req.query.q captura el valor del parámetro de consulta q de la URL.

Manejo de Diferentes Métodos HTTP

Express te permite manejar diferentes métodos HTTP para la misma ruta.

Ejemplo: Manejo de Diferentes Métodos

app.route('/book')
  .get((req, res) => {
    res.send('Obtener un libro aleatorio');
  })
  .post((req, res) => {
    res.send('Agregar un libro');
  })
  .put((req, res) => {
    res.send('Actualizar el libro');
  });

En este ejemplo:

  • app.route('/book') encadena múltiples manejadores para diferentes métodos HTTP en la misma ruta.

Middleware en Rutas

Las funciones middleware se pueden utilizar para manejar solicitudes antes de que lleguen al manejador de la ruta.

Ejemplo: Middleware en Rutas

const logRequest = (req, res, next) => {
  console.log(`${req.method} ${req.url}`);
  next();
};

app.use(logRequest);

app.get('/', (req, res) => {
  res.send('¡Hola Mundo!');
});

En este ejemplo:

  • logRequest es una función middleware que registra el método y la URL de la solicitud.
  • app.use(logRequest) aplica este middleware a todas las rutas.

Enrutamiento Avanzado

Grupos de Rutas

Los grupos de rutas te permiten aplicar middleware a un grupo de rutas.

Ejemplo: Grupos de Rutas

const express = require('express');
const app = express();
const router = express.Router();

router.use((req, res, next) => {
  console.log('URL de la solicitud:', req.originalUrl);
  next();
});

router.get('/user/:id', (req, res) => {
  res.send(`ID de Usuario: ${req.params.id}`);
});

router.get('/user/:id/edit', (req, res) => {
  res.send(`Editar ID de Usuario: ${req.params.id}`);
});

app.use('/admin', router);

app.listen(3000, () => {
  console.log('El servidor está corriendo en el puerto 3000');
});

En este ejemplo:

  • router.use aplica middleware a todas las rutas dentro del enrutador.
  • app.use('/admin', router) monta el enrutador en la ruta /admin.

Resumen

El enrutamiento en Express es una característica poderosa que te permite definir cómo tu aplicación responde a varias solicitudes HTTP. Los conceptos clave incluyen:

  • Enrutamiento básico con métodos HTTP.
  • Uso de parámetros de ruta y parámetros de consulta.
  • Manejo de diferentes métodos HTTP en la misma ruta.
  • Aplicación de middleware a las rutas.
  • Agrupación de rutas con enrutadores.

Al dominar estos conceptos, puedes crear aplicaciones web robustas y escalables utilizando Express.
