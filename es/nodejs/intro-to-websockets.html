WebSockets proporcionan un canal de comunicación full-dúplex sobre una única conexión de larga duración. Esto es particularmente útil para aplicaciones en tiempo real, como aplicaciones de chat, notificaciones en vivo y juegos en línea. En esta sección, exploraremos WebSockets en el contexto de Node.js, desde lo básico hasta conceptos más avanzados.

¿Qué son los WebSockets?

WebSockets son un protocolo que permite la comunicación bidireccional entre un cliente y un servidor. A diferencia de HTTP, que se basa en solicitudes y respuestas, los WebSockets permiten conexiones persistentes, permitiendo que los datos se envíen y reciban en cualquier momento.

Conceptos Clave

  • Comunicación Full-Dúplex: Tanto el cliente como el servidor pueden enviar y recibir mensajes simultáneamente.
  • Conexión Persistente: La conexión permanece abierta, reduciendo la sobrecarga de establecer múltiples conexiones HTTP.
  • Baja Latencia: Ideal para aplicaciones en tiempo real debido a la latencia reducida.

Configuración de WebSockets en Node.js

Para usar WebSockets en Node.js, utilizaremos la biblioteca ws, una implementación popular de WebSocket.

Instalación de la Biblioteca `ws`

Primero, instala la biblioteca ws usando npm:

npm install ws

Creación de un Servidor WebSocket Básico

Vamos a crear un servidor WebSocket simple que escuche conexiones y devuelva cualquier mensaje que reciba.

const WebSocket = require('ws');

const server = new WebSocket.Server({ port: 8080 });

server.on('connection', (socket) => {
  console.log('Cliente conectado');

  socket.on('message', (message) => {
    console.log(`Recibido: ${message}`);
    socket.send(`Echo: ${message}`);
  });

  socket.on('close', () => {
    console.log('Cliente desconectado');
  });
});

console.log('El servidor WebSocket está funcionando en ws://localhost:8080');

Explicación

  • WebSocket.Server: Crea un nuevo servidor WebSocket que escucha en el puerto 8080.
  • server.on('connection'): Oyente de eventos para nuevas conexiones de clientes.
  • socket.on('message'): Oyente de eventos para mensajes del cliente.
  • socket.send(): Envía un mensaje de vuelta al cliente.
  • socket.on('close'): Oyente de eventos para cuando el cliente se desconecta.

Creación de un Cliente WebSocket

Ahora, vamos a crear un cliente WebSocket simple que se conecte a nuestro servidor y envíe un mensaje.

Código del Cliente

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Cliente WebSocket</title>
</head>
<body>
  <script>
    const socket = new WebSocket('ws://localhost:8080');

    socket.onopen = () => {
      console.log('Conectado al servidor');
      socket.send('¡Hola Servidor!');
    };

    socket.onmessage = (event) => {
      console.log(`Recibido del servidor: ${event.data}`);
    };

    socket.onclose = () => {
      console.log('Desconectado del servidor');
    };
  </script>
</body>
</html>

Explicación

  • new WebSocket('ws://localhost:8080'): Crea una nueva conexión WebSocket al servidor.
  • socket.onopen: Oyente de eventos para cuando la conexión se establece.
  • socket.send(): Envía un mensaje al servidor.
  • socket.onmessage: Oyente de eventos para mensajes del servidor.
  • socket.onclose: Oyente de eventos para cuando la conexión se cierra.

Características Avanzadas de WebSocket

Difusión de Mensajes

Para difundir un mensaje a todos los clientes conectados, itera sobre los clientes y envía el mensaje.

server.on('connection', (socket) => {
  socket.on('message', (message) => {
    server.clients.forEach((client) => {
      if (client.readyState === WebSocket.OPEN) {
        client.send(`Difusión: ${message}`);
      }
    });
  });
});

Manejo de Datos Binarios

Los WebSockets también pueden manejar datos binarios. Aquí hay un ejemplo de envío y recepción de datos binarios.

// Lado del servidor
socket.on('message', (data) => {
  if (data instanceof Buffer) {
    console.log('Datos binarios recibidos');
    // Procesar datos binarios
  }
});

// Lado del cliente
const buffer = new ArrayBuffer(8);
const view = new DataView(buffer);
view.setInt32(0, 12345678, true);
socket.send(buffer);
