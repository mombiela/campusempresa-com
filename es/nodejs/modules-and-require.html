Introducción a los Módulos

Los módulos son una parte fundamental de Node.js, permitiéndote organizar tu código en piezas reutilizables. Esta sección cubrirá los conceptos básicos de los módulos y cómo usarlos efectivamente en tus aplicaciones Node.js.

¿Qué es un Módulo?

  • Un módulo es una pieza de código reutilizable que puede ser exportada e importada en otras partes de tu aplicación.
  • Node.js utiliza el sistema de módulos CommonJS, donde cada archivo es tratado como un módulo separado.

¿Por Qué Usar Módulos?

  • Organización del Código: Divide tu código en piezas más pequeñas y manejables.
  • Reutilización: Usa el mismo código en múltiples partes de tu aplicación.
  • Mantenibilidad: Más fácil de mantener y actualizar tu código.

Usando require() para Importar Módulos

La función require() se usa para importar módulos en Node.js.

Uso Básico

Para importar un módulo, usas la función require() seguida de la ruta al módulo.

// Importando un módulo incorporado
const http = require('http');

// Importando un módulo local
const myModule = require('./myModule');

Módulos Incorporados

Node.js viene con varios módulos incorporados que puedes usar sin instalar nada.

  • http: Se usa para crear servidores y clientes HTTP.
  • fs: Proporciona una API para interactuar con el sistema de archivos.
  • path: Utilidades para trabajar con rutas de archivos y directorios.

Ejemplo de uso del módulo http:

const http = require('http');

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello, World!\n');
});

server.listen(3000, '127.0.0.1', () => {
  console.log('Server running at http://127.0.0.1:3000/');
});

Módulos Locales

Puedes crear tus propios módulos e importarlos usando require().

Ejemplo de un módulo local (math.js):

// math.js
function add(a, b) {
  return a + b;
}

function subtract(a, b) {
  return a - b;
}

module.exports = {
  add,
  subtract
};

Importando y usando el módulo local:

// app.js
const math = require('./math');

console.log(math.add(2, 3)); // Salida: 5
console.log(math.subtract(5, 2)); // Salida: 3

Exportando Módulos

Para hacer funciones, objetos o valores disponibles a otros módulos, necesitas exportarlos.

module.exports vs exports

  • module.exports: El objeto que realmente se devuelve como resultado de una llamada a require().
  • exports: Una abreviatura para module.exports.

Ejemplo usando module.exports:

// logger.js
module.exports = function(message) {
  console.log(message);
};

Ejemplo usando exports:

// calculator.js
exports.add = function(a, b) {
  return a + b;
};

exports.subtract = function(a, b) {
  return a - b;
};

Conceptos Avanzados de Módulos

A medida que te sientas más cómodo con los módulos, puedes explorar conceptos más avanzados.

Dependencias Circulares

Las dependencias circulares ocurren cuando dos o más módulos dependen entre sí. Node.js maneja esto devolviendo un módulo parcialmente construido.

Ejemplo:

// a.js
const b = require('./b');
console.log('En a.js, b.done =', b.done);
exports.done = true;

// b.js
const a = require('./a');
console.log('En b.js, a.done =', a.done);
exports.done = true;

Caché

Los módulos se almacenan en caché después de la primera vez que se cargan. Esto significa que las llamadas subsecuentes a require() devolverán la versión en caché.

Ejemplo:

// counter.js
let count = 0;

module.exports = {
  increment: () => count++,
  getCount: () => count
};

// app.js
const counter = require('./counter');
counter.increment();
console.log(counter.getCount()); // Salida: 1

const counterAgain = require('./counter');
console.log(counterAgain.getCount()); // Salida: 1 (en caché)

Conclusión

Entender los módulos y la función require() es crucial para desarrollar aplicaciones Node.js eficientes y mantenibles. Al organizar tu código en módulos, puedes mejorar la legibilidad, reutilización y mantenibilidad del código. A medida que avances, encontrarás escenarios más complejos como dependencias circulares y caché, pero los conceptos fundamentales permanecen iguales. ¡Feliz codificación!
