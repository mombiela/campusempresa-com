Introducción a los Structs

Los structs en Rust son una forma de crear tipos de datos personalizados que agrupan valores relacionados. Son similares a las clases en otros lenguajes de programación pero sin métodos. Los structs son esenciales para organizar datos complejos y hacer que tu código sea más legible y mantenible.

Definiendo un Struct

Para definir un struct en Rust, usas la palabra clave struct seguida del nombre del struct y sus campos.

Definición Básica de un Struct

Aquí hay un ejemplo simple de cómo definir un struct:

struct User {
    username: String,
    email: String,
    sign_in_count: u64,
    active: bool,
}

En este ejemplo:

  • User es el nombre del struct.
  • username, email, sign_in_count y active son los campos del struct.
  • Cada campo tiene un nombre y un tipo.

Creando Instancias de Structs

Una vez que has definido un struct, puedes crear instancias del mismo.

Creando una Instancia

Aquí se muestra cómo crear una instancia del struct User:

fn main() {
    let user1 = User {
        username: String::from("johndoe"),
        email: String::from("johndoe@example.com"),
        sign_in_count: 1,
        active: true,
    };

    println!("Username: {}", user1.username);
}

En este ejemplo:

  • user1 es una instancia del struct User.
  • La función String::from se usa para crear valores String para los campos username y email.
  • La macro println! se usa para imprimir el campo username de user1.

Actualizando Structs

Puedes crear una nueva instancia de un struct basada en una instancia existente, usando la sintaxis de actualización.

Sintaxis de Actualización

Aquí hay un ejemplo de cómo actualizar un struct:

fn main() {
    let user1 = User {
        username: String::from("johndoe"),
        email: String::from("johndoe@example.com"),
        sign_in_count: 1,
        active: true,
    };

    let user2 = User {
        email: String::from("janedoe@example.com"),
        ..user1
    };

    println!("Username: {}", user2.username);
    println!("Email: {}", user2.email);
}

En este ejemplo:

  • user2 se crea basado en user1.
  • La sintaxis ..user1 copia los campos restantes de user1 a user2.

Tuple Structs

Rust también soporta tuple structs, que son similares a las tuplas regulares pero tienen tipos nombrados.

Definiendo y Usando Tuple Structs

Aquí hay un ejemplo de un tuple struct:

struct Color(i32, i32, i32);

fn main() {
    let black = Color(0, 0, 0);

    println!("Black color: ({}, {}, {})", black.0, black.1, black.2);
}

En este ejemplo:

  • Color es un tuple struct con tres campos i32.
  • black es una instancia del tuple struct Color.
  • Los campos de black se acceden usando la notación de punto con índices.

Structs de Tipo Unidad

Los structs de tipo unidad son structs sin ningún campo. Son útiles para implementar traits sin almacenar datos.

Definiendo y Usando Structs de Tipo Unidad

Aquí hay un ejemplo de un struct de tipo unidad:

struct AlwaysEqual;

fn main() {
    let subject = AlwaysEqual;

    println!("This struct has no fields!");
}

En este ejemplo:

  • AlwaysEqual es un struct de tipo unidad.
  • subject es una instancia de AlwaysEqual.

Resumen

Los structs en Rust son una forma poderosa de agrupar datos relacionados. Vienen en tres sabores: structs regulares, tuple structs y structs de tipo unidad. Entender cómo definir y usar structs es fundamental para organizar y gestionar datos complejos en programas Rust.

  • Los structs regulares tienen campos nombrados.
  • Los tuple structs tienen campos sin nombre.
  • Los structs de tipo unidad no tienen campos.

Al dominar los structs, puedes escribir código Rust más organizado y mantenible.
