Enums y Coincidencia de Patrones en Rust
Vectores ►

Introducción a los Enums

Los enums, abreviatura de "enumeraciones", son una característica poderosa en Rust que te permite definir un tipo enumerando sus posibles valores. Los enums son particularmente útiles para representar un valor que podría ser uno de varios tipos diferentes.

Definiendo Enums

Para definir un enum, usa la palabra clave enum seguida del nombre del enum y sus posibles variantes.

enum Direction {
    North,
    South,
    East,
    West,
}
  • Direction es el nombre del enum.
  • North, South, East, West son las variantes del enum.

Usando Enums

Puedes crear instancias de un enum especificando el nombre del enum seguido de :: y el nombre de la variante.

let direction = Direction::North;

Variantes de Enum con Datos

Los enums también pueden contener datos, lo que los hace aún más versátiles.

enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(i32, i32, i32),
}
  • Quit: No tiene datos asociados.
  • Move: Contiene una estructura anónima con campos x y y.
  • Write: Contiene un String.
  • ChangeColor: Contiene una tupla con tres valores i32.

Coincidencia de Patrones

La coincidencia de patrones es un mecanismo para verificar un valor contra un patrón. Es una construcción de control de flujo poderosa que te permite desestructurar y coincidir con tipos de datos complejos como enums.

La Expresión `match`

La expresión match se usa para comparar un valor contra una serie de patrones y ejecutar código basado en qué patrón coincide.

fn main() {
    let direction = Direction::North;

    match direction {
        Direction::North => println!("¡Rumbo al Norte!"),
        Direction::South => println!("¡Rumbo al Sur!"),
        Direction::East => println!("¡Rumbo al Este!"),
        Direction::West => println!("¡Rumbo al Oeste!"),
    }
}
  • La expresión match toma un valor y lo compara contra cada patrón en turno.
  • Cuando un patrón coincide, se ejecuta el bloque de código correspondiente.

Coincidiendo con Enums que Contienen Datos

Al coincidir con variantes de enum que contienen datos, puedes vincular los datos a variables dentro del patrón.

fn process_message(msg: Message) {
    match msg {
        Message::Quit => println!("Mensaje de salida recibido."),
        Message::Move { x, y } => println!("Mover a coordenadas: ({}, {})", x, y),
        Message::Write(texto) => println!("Mensaje: {}", texto),
        Message::ChangeColor(r, g, b) => println!("Cambiar color a RGB: ({}, {}, {})", r, g, b),
    }
}
  • Message::Move { x, y }: Desestructura la estructura anónima y vincula x y y a los valores respectivos.
  • Message::Write(texto): Vincula texto al String contenido en la variante.
  • Message::ChangeColor(r, g, b): Vincula r, g y b a los valores i32 respectivos en la tupla.

Coincidencia de Patrones Avanzada

La coincidencia de patrones en Rust puede ser muy sofisticada, permitiendo condiciones complejas y desestructuración.

Coincidiendo con Múltiples Patrones

Puedes coincidir con múltiples patrones usando el operador |.

fn main() {
    let direction = Direction::North;

    match direction {
        Direction::North | Direction::South => println!("Dirección vertical"),
        Direction::East | Direction::West => println!("Dirección horizontal"),
    }
}

Coincidiendo con Rangos

Puedes coincidir con rangos de valores usando ..=.

fn main() {
    let number = 5;

    match number {
        1..=5 => println!("El número está entre 1 y 5"),
        _ => println!("El número está fuera de rango"),
    }
}

Desestructurando Enums Complejos

Puedes desestructurar estructuras de datos anidadas dentro de enums.

enum ComplexMessage {
    Move { x: i32, y: i32, z: i32 },
    ChangeColor { red: i32, green: i32, blue: i32 },
}

fn process_complex_message(msg: ComplexMessage) {
    match msg {
        ComplexMessage::Move { x, y, z } => println!("Mover a coordenadas: ({}, {}, {})", x, y, z),
        ComplexMessage::ChangeColor { red, green, blue } => println!("Cambiar color a RGB: ({}, {}, {})", red, green, blue),
    }
}

Conclusión

Los enums y la coincidencia de patrones son características esenciales en Rust que proporcionan formas poderosas de representar y manipular datos. Los enums te permiten definir un tipo con múltiples variantes, opcionalmente conteniendo datos. La coincidencia de patrones te permite desestructurar y manejar estas variantes de una manera limpia y legible.

  • Enums: Definen tipos con múltiples variantes.
  • Coincidencia de Patrones: Desestructura y maneja tipos de datos complejos.

Al dominar estos conceptos, puedes escribir código en Rust más expresivo y robusto.
