Vectores en Rust
Cadenas ►

Los vectores son una de las estructuras de datos más comúnmente usadas en Rust. Proporcionan un array dinámico que puede crecer o reducirse en tamaño. Este tema cubrirá todo, desde los conceptos básicos de la creación y uso de vectores hasta operaciones más avanzadas y consideraciones de rendimiento.

Introducción a los Vectores

Los vectores en Rust son proporcionados por el tipo Vec<T>, donde T representa el tipo de elementos almacenados en el vector.

Creando un Vector

  • Vector Vacío: Puedes crear un vector vacío usando Vec::new().
  • Vector con Valores Iniciales: Puedes crear un vector con valores iniciales usando la macro vec!.
// Creando un vector vacío
let mut v: Vec<i32> = Vec::new();

// Creando un vector con valores iniciales
let v = vec![1, 2, 3, 4, 5];

Añadiendo Elementos a un Vector

  • Usando el Método push: Puedes añadir elementos al final del vector usando el método push.
let mut v = Vec::new();
v.push(1);
v.push(2);
v.push(3);

Accediendo a Elementos en un Vector

  • Indexación: Puedes acceder a los elementos usando indexación, pero ten cuidado ya que puede causar un pánico si el índice está fuera de los límites.
  • Usando el Método get: El método get devuelve un Option<&T> que es más seguro.
let v = vec![1, 2, 3, 4, 5];

// Accediendo a elementos usando indexación
let tercero = &v[2];
println!("El tercer elemento es {}", tercero);

// Accediendo a elementos usando el método `get`
match v.get(2) {
    Some(tercero) => println!("El tercer elemento es {}", tercero),
    None => println!("No hay un tercer elemento."),
}

Iterando Sobre un Vector

  • Usando un Bucle for: Puedes iterar sobre los elementos de un vector usando un bucle for.
  • Usando Iteradores: Rust proporciona métodos de iteradores poderosos para operaciones más complejas.
let v = vec![1, 2, 3, 4, 5];

// Iterando usando un bucle `for`
for i in &v {
    println!("{}", i);
}

// Iterando usando un iterador
for i in v.iter() {
    println!("{}", i);
}

Operaciones Avanzadas con Vectores

Eliminando Elementos

  • Usando el Método pop: Elimina el último elemento y lo devuelve.
  • Usando el Método remove: Elimina el elemento en un índice especificado.
let mut v = vec![1, 2, 3, 4, 5];

// Eliminando el último elemento
let ultimo = v.pop();
println!("Eliminado el último elemento: {:?}", ultimo);

// Eliminando un elemento en el índice 2
let tercero = v.remove(2);
println!("Eliminado el tercer elemento: {}", tercero);

Dividiendo Vectores

  • Slices: Puedes crear slices de vectores para trabajar con un subconjunto de elementos.
let v = vec![1, 2, 3, 4, 5];
let slice = &v[1..4]; // Esto crea un slice que contiene los elementos 2, 3 y 4

for i in slice {
    println!("{}", i);
}

Capacidad y Rendimiento del Vector

  • Capacidad: Los vectores tienen una capacidad, que es la cantidad de espacio asignado para los elementos. La capacidad puede ser diferente de la longitud.
  • Consideraciones de Rendimiento: Entender cómo crecen los vectores puede ayudarte a escribir código más eficiente.
let mut v = Vec::with_capacity(10);
println!("Capacidad: {}", v.capacity());

for i in 0..10 {
    v.push(i);
}
println!("Capacidad después de añadir 10 elementos: {}", v.capacity());

v.push(11);
println!("Capacidad después de añadir un elemento más: {}", v.capacity());

Conclusión

Los vectores son una estructura de datos versátil y poderosa en Rust. Permiten un redimensionamiento dinámico y proporcionan una gama de métodos para manipular datos eficientemente. Al entender cómo crear, acceder y modificar vectores, así como manejar operaciones más avanzadas, puedes aprovechar los vectores para escribir programas en Rust más efectivos. Recuerda siempre considerar las implicaciones de rendimiento, especialmente cuando trabajes con grandes conjuntos de datos o aplicaciones críticas en términos de rendimiento.
