Tu Primer Programa en Ada ►

Configurar el entorno de desarrollo es un primer paso crucial para aprender Ada. Esta sección te guiará a través del proceso de instalación de las herramientas necesarias y la configuración de tu sistema para comenzar a escribir y ejecutar programas en Ada.

Instalación del Compilador Ada

Para escribir y ejecutar programas en Ada, necesitas un compilador. El compilador Ada más utilizado es GNAT, que forma parte del GNU Compiler Collection (GCC).

Pasos para Instalar GNAT

  1. Descargar GNAT:

    • Visita el sitio web de AdaCore para descargar la Edición Comunitaria de GNAT.
    • Elige la versión apropiada para tu sistema operativo (Windows, macOS o Linux).

  2. Instalar GNAT:

    • Windows: Ejecuta el instalador y sigue las instrucciones en pantalla.
    • macOS: Abre el archivo .dmg descargado y arrastra la carpeta GNAT a tu directorio de Aplicaciones.
    • Linux: Extrae el tarball descargado y sigue las instrucciones en el archivo README.

  3. Verificar la Instalación:

    • Abre una terminal o símbolo del sistema.
    • Escribe gnat --version y presiona Enter. Deberías ver la información de la versión de GNAT.
$ gnat --version
GNAT 11.2.0

Configuración de un Entorno de Desarrollo Integrado (IDE)

Aunque puedes escribir código Ada en cualquier editor de texto, usar un Entorno de Desarrollo Integrado (IDE) puede mejorar significativamente tu productividad. Dos IDEs populares para Ada son GNAT Studio y Visual Studio Code.

GNAT Studio

  1. Descargar GNAT Studio:

    • GNAT Studio está incluido con la Edición Comunitaria de GNAT. Si ya has instalado GNAT, también deberías tener GNAT Studio.

  2. Lanzar GNAT Studio:

    • Abre GNAT Studio desde tu directorio de Aplicaciones o escribiendo gnatstudio en tu terminal/símbolo del sistema.

  3. Configurar GNAT Studio:

    • Configura tu proyecto creando un nuevo archivo de proyecto (.gpr).
    • Configura los ajustes del proyecto como directorios de fuentes y opciones del compilador.

Visual Studio Code

  1. Instalar Visual Studio Code:

    • Descarga e instala Visual Studio Code desde el sitio web oficial.

  2. Instalar Extensiones de Ada:

    • Abre Visual Studio Code.
    • Ve a la vista de Extensiones haciendo clic en el ícono de Extensiones en la Barra de Actividades en el lado de la ventana.
    • Busca "Ada" e instala la extensión AdaCore Ada Language Support.

  3. Configurar Visual Studio Code:

    • Crea un nuevo espacio de trabajo y añade tu directorio de proyecto Ada.
    • Configura tareas y ajustes de lanzamiento para compilar y ejecutar programas en Ada.

Escribiendo Tu Primer Programa en Ada

Una vez que tu entorno de desarrollo esté configurado, puedes escribir y ejecutar tu primer programa en Ada.

Ejemplo: Hola Mundo

  1. Crear un nuevo archivo:

    • Abre tu IDE y crea un nuevo archivo llamado hello.adb.

  2. Escribir el código:

    • Introduce el siguiente código en hello.adb:
with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("¡Hola, Mundo!");
end Hello;
  1. Compilar el programa:
    • Abre una terminal o usa el comando de compilación en tu IDE.
    • Navega al directorio que contiene hello.adb.
    • Ejecuta el siguiente comando para compilar el programa:
$ gnatmake hello.adb
  1. Ejecutar el programa:
    • Ejecuta el programa compilado escribiendo:
$ ./hello
¡Hola, Mundo!

Conclusión

Configurar el entorno de desarrollo para Ada implica instalar el compilador GNAT, elegir y configurar un IDE, y escribir tu primer programa. Siguiendo estos pasos, estarás listo para comenzar tu viaje en la programación con Ada. Recuerda, un entorno bien configurado puede mejorar enormemente tu experiencia de codificación y productividad.
