Introducción a los Arrays

Los arrays son una estructura de datos fundamental en Ada que te permite almacenar y manipular colecciones de elementos. Cada elemento en un array se identifica mediante un índice, y todos los elementos deben ser del mismo tipo.

Conceptos Clave

  • Indexación: Los arrays en Ada están indexados, lo que significa que cada elemento se accede usando un índice.
  • Límites: Los arrays tienen límites definidos, especificando el rango de índices válidos.
  • Tipos: Todos los elementos en un array deben ser del mismo tipo.
  • Arrays Estáticos y Dinámicos: Los arrays pueden ser estáticos (tamaño fijo) o dinámicos (el tamaño puede cambiar).

Declaración de Arrays

En Ada, los arrays se declaran especificando el tipo de elementos y el rango de índices.

Arrays Estáticos

Los arrays estáticos tienen un tamaño fijo definido en tiempo de compilación.

-- Declaración de un array estático con índices del 1 al 5
type Index_Type is range 1 .. 5;
type Array_Type is array (Index_Type) of Integer;

-- Inicialización del array
My_Array : Array_Type := (1, 2, 3, 4, 5);

Arrays Dinámicos

Los arrays dinámicos pueden cambiar de tamaño durante la ejecución usando el tipo Unbounded_Array del paquete Ada.Containers.

with Ada.Containers.Indefinite_Vectors;
procedure Dynamic_Array_Example is
   package Integer_Vectors is new Ada.Containers.Indefinite_Vectors (Index_Type => Natural, Element_Type => Integer);
   My_Vector : Integer_Vectors.Vector;
begin
   -- Añadiendo elementos al array dinámico
   Integer_Vectors.Append (My_Vector, 10);
   Integer_Vectors.Append (My_Vector, 20);
end Dynamic_Array_Example;

Acceso a Elementos del Array

Los elementos en un array se acceden usando sus índices.

-- Accediendo a elementos en un array estático
My_Array(1) := 10;
Put_Line (Integer'Image (My_Array(1))); -- Salida: 10

Iteración Sobre Arrays

Puedes iterar sobre arrays usando bucles.

Usando un Bucle `for`

-- Iterando sobre un array estático
for Index in My_Array'Range loop
   Put_Line (Integer'Image (My_Array(Index)));
end loop;

Usando Bucle `while`

-- Iterando sobre un array dinámico
Index : Natural := My_Vector.First_Index;
while Index /= My_Vector.No_Element loop
   Put_Line (Integer'Image (My_Vector.Element (Index)));
   Index := My_Vector.Next_Index (Index);
end loop;

Arrays Multidimensionales

Ada soporta arrays multidimensionales, que son arrays de arrays.

-- Declaración de un array 2D
type Matrix_Type is array (1 .. 3, 1 .. 3) of Integer;
Matrix : Matrix_Type := ((1, 2, 3), (4, 5, 6), (7, 8, 9));

-- Accediendo a elementos en un array 2D
Put_Line (Integer'Image (Matrix(2, 3))); -- Salida: 6

Atributos de los Arrays

Ada proporciona varios atributos para trabajar con arrays.

  • 'First: Devuelve el primer índice del array.
  • 'Last: Devuelve el último índice del array.
  • 'Length: Devuelve el número de elementos en el array.
  • 'Range: Devuelve el rango de índices.
Put_Line (Integer'Image (My_Array'First));  -- Salida: 1
Put_Line (Integer'Image (My_Array'Last));   -- Salida: 5
Put_Line (Integer'Image (My_Array'Length)); -- Salida: 5

Resumen

Los arrays en Ada son estructuras versátiles y poderosas que te permiten almacenar y manipular colecciones de elementos de manera eficiente. Entender cómo declarar, acceder e iterar sobre arrays es crucial para programar efectivamente en Ada. Ya sea que trabajes con arrays estáticos o dinámicos, Ada proporciona características y atributos robustos para gestionarlos eficazmente.
