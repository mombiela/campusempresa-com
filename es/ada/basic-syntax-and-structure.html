Introducción

Ada es un lenguaje de programación de alto nivel, estructurado, tipado estáticamente, imperativo y orientado a objetos. Está diseñado para aplicaciones grandes y de larga duración donde la fiabilidad y la eficiencia son esenciales. Esta sección cubrirá la sintaxis y estructura básicas de los programas en Ada, proporcionando una base para un aprendizaje más avanzado.

Conceptos Clave

  • Estructura del Programa: Entender el diseño básico de un programa en Ada.
  • Comentarios: Cómo escribir comentarios en Ada.
  • Identificadores: Reglas para nombrar variables, constantes y otras entidades.
  • Tipos de Datos: Tipos de datos básicos disponibles en Ada.
  • Estructuras de Control: Mecanismos básicos de control de flujo como bucles y condicionales.

Estructura del Programa

Un programa en Ada consiste en uno o más paquetes, procedimientos y funciones. La estructura básica de un programa en Ada es la siguiente:

with Ada.Text_IO; use Ada.Text_IO;

procedure Hello is
begin
   Put_Line("¡Hola, Mundo!");
end Hello;
  • with Ada.Text_IO; use Ada.Text_IO;: Estas líneas importan el paquete Ada.Text_IO, que proporciona funcionalidades básicas de entrada/salida.
  • procedure Hello is: Esta línea declara un procedimiento llamado Hello.
  • begin ... end Hello;: Este bloque contiene las sentencias ejecutables del procedimiento.

Comentarios

Los comentarios en Ada se escriben usando -- para comentarios de una sola línea. No hay delimitadores de comentarios de varias líneas en Ada.

-- Este es un comentario de una sola línea
Put_Line("¡Hola, Mundo!"); -- Esto también es un comentario

Identificadores

Los identificadores en Ada son nombres dados a varios elementos del programa como variables, constantes, procedimientos, etc. Las reglas para nombrar identificadores son:

  • Deben comenzar con una letra.
  • Pueden contener letras, dígitos y guiones bajos.
  • No distinguen entre mayúsculas y minúsculas.
Mi_Variable : Integer := 10;
otraVariable : Integer := 20;

Tipos de Datos

Ada soporta varios tipos de datos básicos, incluyendo:

  • Integer: Números enteros.
  • Float: Números de punto flotante.
  • Boolean: Valores Verdadero o Falso.
  • Character: Caracteres individuales.
  • String: Secuencias de caracteres.

Ejemplo:

Mi_Entero : Integer := 42;
Mi_Flotante : Float := 3.14;
Mi_Booleano : Boolean := True;
Mi_Caracter : Character := 'A';
Mi_Cadena : String := "¡Hola, Ada!";

Estructuras de Control

Ada proporciona estructuras de control estándar para la toma de decisiones y la iteración.

Sentencias If

if Mi_Entero > 10 then
   Put_Line("Mayor que 10");
elsif Mi_Entero = 10 then
   Put_Line("Igual a 10");
else
   Put_Line("Menor que 10");
end if;

Sentencias Case

case Mi_Entero is
   when 1 => Put_Line("Uno");
   when 2 => Put_Line("Dos");
   when others => Put_Line("Otro");
end case;

Bucles

Bucle While

while Mi_Entero < 50 loop
   Mi_Entero := Mi_Entero + 1;
end loop;

Bucle For

for I in 1 .. 10 loop
   Put_Line(Integer'Image(I));
end loop;

Resumen

En esta sección, cubrimos la sintaxis y estructura básicas de los programas en Ada. Discutimos el diseño general del programa, cómo escribir comentarios, reglas para nombrar identificadores, tipos de datos básicos y estructuras de control como sentencias if, sentencias case y bucles. Entender estos fundamentos es crucial para escribir y comprender programas más complejos en Ada.
