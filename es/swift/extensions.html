Introducción a las Extensiones

Las extensiones en Swift son una característica poderosa que te permite agregar nueva funcionalidad a clases, estructuras, enumeraciones o protocolos existentes. Esto se puede hacer sin modificar el código fuente original, haciendo que tu código sea más modular y organizado.

Conceptos Clave

  • Agregar Propiedades Computadas: Las extensiones pueden agregar nuevas propiedades computadas a tipos existentes.
  • Agregar Métodos: Puedes agregar nuevos métodos de instancia y métodos de tipo.
  • Agregar Inicializadores: Las extensiones pueden agregar nuevos inicializadores a tipos existentes.
  • Agregar Subíndices: Puedes agregar nuevos subíndices a tipos existentes.
  • Agregar Tipos Anidados: Las extensiones pueden agregar nuevos tipos anidados a clases, estructuras o enumeraciones existentes.
  • Conformar a Protocolos: Las extensiones pueden hacer que un tipo existente conforme a un protocolo.

Agregar Propiedades Computadas

Las extensiones pueden agregar propiedades computadas a tipos existentes. Sin embargo, no pueden agregar propiedades almacenadas u observadores de propiedades.

extension Double {
    var km: Double { return self * 1_000.0 }
    var m: Double { return self }
    var cm: Double { return self / 100.0 }
    var mm: Double { return self / 1_000.0 }
}

let oneMeter = 1.0.m
print("One meter is \(oneMeter) meters") // One meter is 1.0 meters

Agregar Métodos

Las extensiones pueden agregar nuevos métodos de instancia y métodos de tipo a tipos existentes.

extension Int {
    func repetitions(task: () -> Void) {
        for _ in 0..<self {
            task()
        }
    }
}

3.repetitions {
    print("Hello!")
}
// Prints "Hello!" three times

Agregar Inicializadores

Las extensiones pueden agregar nuevos inicializadores a tipos existentes. Esto es particularmente útil para agregar inicializadores de conveniencia.

struct Size {
    var width = 0.0, height = 0.0
}

extension Size {
    init(side: Double) {
        self.init(width: side, height: side)
    }
}

let square = Size(side: 10.0)
print("Square size: \(square.width) x \(square.height)") // Square size: 10.0 x 10.0

Agregar Subíndices

Las extensiones pueden agregar nuevos subíndices a tipos existentes.

extension String {
    subscript(index: Int) -> Character {
        return self[self.index(self.startIndex, offsetBy: index)]
    }
}

let greeting = "Hello"
print(greeting[1]) // e

Agregar Tipos Anidados

Las extensiones pueden agregar nuevos tipos anidados a clases, estructuras o enumeraciones existentes.

extension Int {
    enum Kind {
        case negative, zero, positive
    }

    var kind: Kind {
        switch self {
        case 0:
            return .zero
        case let x where x > 0:
            return .positive
        default:
            return .negative
        }
    }
}

func printIntegerKinds(_ numbers: [Int]) {
    for number in numbers {
        switch number.kind {
        case .negative:
            print("- ", terminator: "")
        case .zero:
            print("0 ", terminator: "")
        case .positive:
            print("+ ", terminator: "")
        }
    }
    print("")
}

printIntegerKinds([3, 19, -27, 0, -6, 0, 7])
// Prints "+ + - 0 - 0 + "

Conformar a Protocolos

Las extensiones pueden hacer que un tipo existente conforme a un protocolo, lo cual es una forma común de agregar conformidad a un protocolo a un tipo que no posees.

protocol TextRepresentable {
    var textualDescription: String { get }
}

extension Int: TextRepresentable {
    var textualDescription: String {
        return String(self)
    }
}

let number = 42
print(number.textualDescription) // 42

Conclusión

Las extensiones son una característica versátil en Swift que te permite agregar nueva funcionalidad a tipos existentes de una manera limpia y modular. Te permiten extender tipos que no posees, agregar propiedades computadas, métodos, inicializadores, subíndices, tipos anidados y hacer que los tipos conformen a protocolos. Al usar extensiones, puedes mantener tu código organizado y mantenible mientras mejoras las capacidades de los tipos existentes.
