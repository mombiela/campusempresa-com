Introducción

En Swift, los operadores son símbolos o frases especiales que se utilizan para verificar, cambiar o combinar valores. Swift soporta la mayoría de los operadores estándar de C y mejora varias capacidades para eliminar errores comunes de codificación. Esta sección cubrirá los operadores básicos en Swift, incluyendo operadores aritméticos, de comparación, lógicos y de asignación.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

  • Adición (+): Suma dos valores.
  • Sustracción (-): Resta un valor de otro.
  • Multiplicación (*): Multiplica dos valores.
  • División (/): Divide un valor por otro.
  • Resto (%): Devuelve el resto de dividir un valor por otro.

Ejemplos

let a = 10
let b = 3

let sum = a + b        // sum es 13
let difference = a - b // difference es 7
let product = a * b    // product es 30
let quotient = a / b   // quotient es 3
let remainder = a % b  // remainder es 1

Operadores de Comparación

Los operadores de comparación se utilizan para comparar dos valores.

  • Igual a (==): Verifica si dos valores son iguales.
  • No igual a (!=): Verifica si dos valores no son iguales.
  • Mayor que (>): Verifica si el valor de la izquierda es mayor que el valor de la derecha.
  • Menor que (<): Verifica si el valor de la izquierda es menor que el valor de la derecha.
  • Mayor o igual a (>=): Verifica si el valor de la izquierda es mayor o igual al valor de la derecha.
  • Menor o igual a (<=): Verifica si el valor de la izquierda es menor o igual al valor de la derecha.

Ejemplos

let x = 5
let y = 10

let isEqual = x == y       // isEqual es false
let isNotEqual = x != y    // isNotEqual es true
let isGreater = x > y      // isGreater es false
let isLess = x < y         // isLess es true
let isGreaterOrEqual = x >= y // isGreaterOrEqual es false
let isLessOrEqual = x <= y // isLessOrEqual es true

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples condiciones booleanas.

  • AND lógico (&&): Devuelve true si ambas condiciones son verdaderas.
  • OR lógico (||): Devuelve true si al menos una condición es verdadera.
  • NOT lógico (!): Invierte el valor booleano.

Ejemplos

let isTrue = true
let isFalse = false

let andResult = isTrue && isFalse  // andResult es false
let orResult = isTrue || isFalse   // orResult es true
let notResult = !isTrue            // notResult es false

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación (=): Asigna el valor de la derecha a la variable de la izquierda.
  • Operadores de Asignación Compuesta: Combinan una operación aritmética con la asignación.

Operadores de Asignación Compuesta

  • Asignación de Adición (+=): Suma y asigna el valor.
  • Asignación de Sustracción (-=): Resta y asigna el valor.
  • Asignación de Multiplicación (*=): Multiplica y asigna el valor.
  • Asignación de División (/=): Divide y asigna el valor.
  • Asignación de Resto (%=): Calcula el resto y asigna el valor.

Ejemplos

var number = 10

number += 5  // number ahora es 15
number -= 3  // number ahora es 12
number *= 2  // number ahora es 24
number /= 4  // number ahora es 6
number %= 5  // number ahora es 1

Conclusión

Entender los operadores básicos es fundamental para cualquier programador de Swift. Estos operadores te permiten realizar tareas esenciales como cálculos aritméticos, comparaciones, operaciones lógicas y asignaciones de valores. Dominar estos operadores proporcionará una base sólida para conceptos de programación más avanzados. Practica usando estos operadores en diferentes escenarios para familiarizarte con su sintaxis y funcionalidad.
