Introducción a los Arrays

Los arrays son estructuras de datos fundamentales en Swift que te permiten almacenar colecciones ordenadas de valores. Son versátiles y pueden contener elementos de cualquier tipo, incluidos objetos personalizados.

Conceptos Clave

  • Definición: Un array es una colección ordenada de valores.
  • Seguridad de Tipo: Los arrays en Swift son seguros en cuanto a tipos, lo que significa que solo pueden almacenar elementos del mismo tipo.
  • Mutabilidad: Los arrays pueden ser mutables (modificables) o inmutables (constantes).

Creación de Arrays

Puedes crear arrays usando literales de array o el inicializador Array.

Literales de Array

let numbers = [1, 2, 3, 4, 5]
let strings = ["manzana", "plátano", "cereza"]

Inicializador de Array

let emptyArray = [Int]()  // Crea un array vacío de enteros
let repeatedArray = Array(repeating: "Hola", count: 3)  // ["Hola", "Hola", "Hola"]

Acceso y Modificación de Arrays

Acceso a Elementos

Puedes acceder a los elementos en un array usando la sintaxis de subíndice.

let fruits = ["manzana", "plátano", "cereza"]
let firstFruit = fruits[0]  // "manzana"

Modificación de Elementos

Puedes modificar elementos asignando un nuevo valor a un índice específico.

var fruits = ["manzana", "plátano", "cereza"]
fruits[1] = "arándano"  // ["manzana", "arándano", "cereza"]

Adición y Eliminación de Elementos

  • Añadir: Agrega un elemento al final del array.
  • Insertar: Inserta un elemento en un índice específico.
  • Eliminar: Elimina un elemento en un índice específico o elimina el último elemento.
var numbers = [1, 2, 3]
numbers.append(4)  // [1, 2, 3, 4]
numbers.insert(0, at: 0)  // [0, 1, 2, 3, 4]
numbers.remove(at: 2)  // [0, 1, 3, 4]
numbers.removeLast()  // [0, 1, 3]

Iteración Sobre Arrays

Puedes iterar sobre arrays usando bucles.

Bucle For-In

let fruits = ["manzana", "plátano", "cereza"]
for fruit in fruits {
    print(fruit)
}

Bucle Enumerado

Para obtener tanto el índice como el valor, usa el método enumerated().

let fruits = ["manzana", "plátano", "cereza"]
for (index, fruit) in fruits.enumerated() {
    print("Elemento \(index + 1): \(fruit)")
}

Operaciones Avanzadas con Arrays

Ordenar Arrays

Puedes ordenar arrays usando el método sorted() o el método sort() para ordenar en su lugar.

let numbers = [3, 1, 4, 1, 5, 9]
let sortedNumbers = numbers.sorted()  // [1, 1, 3, 4, 5, 9]

Filtrar Arrays

Usa el método filter() para crear un nuevo array con elementos que cumplan una condición.

let numbers = [1, 2, 3, 4, 5]
let evenNumbers = numbers.filter { $0 % 2 == 0 }  // [2, 4]

Mapear Arrays

Usa el método map() para transformar cada elemento en el array.

let numbers = [1, 2, 3, 4, 5]
let squaredNumbers = numbers.map { $0 * $0 }  // [1, 4, 9, 16, 25]

Reducir Arrays

Usa el método reduce() para combinar todos los elementos en un solo valor.

let numbers = [1, 2, 3, 4, 5]
let sum = numbers.reduce(0, +)  // 15

Conclusión

Los arrays son una forma poderosa y flexible de gestionar colecciones de datos en Swift. Desde la creación y manipulación básica hasta operaciones avanzadas como ordenar, filtrar, mapear y reducir, entender los arrays mejorará significativamente tu capacidad para escribir código Swift eficiente y efectivo. Practica estos conceptos con varios ejercicios para solidificar tu comprensión y volverte competente en el uso de arrays en Swift.
