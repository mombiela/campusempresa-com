¿Qué es Swift?

Swift es un lenguaje de programación poderoso e intuitivo desarrollado por Apple para construir aplicaciones para iOS, macOS, watchOS y tvOS. Está diseñado para trabajar sin problemas con los frameworks Cocoa y Cocoa Touch de Apple y está construido sobre lo mejor de C y Objective-C, sin las limitaciones de la compatibilidad con C.

Características Clave de Swift

  • Seguro: Swift elimina clases enteras de código inseguro. Las variables siempre se inicializan antes de usarse, los arrays y enteros se verifican para evitar desbordamientos, y la memoria se gestiona automáticamente.
  • Rápido: Swift está diseñado para ser rápido. La sintaxis y la biblioteca estándar se han ajustado para que las tareas más comunes sean fáciles y claras, asegurando al mismo tiempo que el lenguaje en sí sea rápido.
  • Expresivo: Swift incluye características modernas que los desarrolladores adoran. Tiene una poderosa inferencia de tipos y coincidencia de patrones, lo que hace que el código sea más expresivo y menos propenso a errores.

Configuración de Tu Entorno

Para comenzar a desarrollar con Swift, necesitas configurar tu entorno de desarrollo:

  1. Xcode: Descarga e instala Xcode desde la Mac App Store. Xcode es el entorno de desarrollo integrado (IDE) de Apple para macOS.
  2. Playgrounds: Usa Swift Playgrounds para codificación interactiva y aprendizaje. Los Playgrounds te permiten escribir código Swift y ver los resultados de inmediato.

Sintaxis Básica

Variables y Constantes

En Swift, usas var para declarar variables y let para declarar constantes.

var myVariable = 42
myVariable = 50
let myConstant = 42
  • Variables: Pueden cambiar después de ser establecidas.
  • Constantes: No pueden cambiar una vez que se establecen.

Tipos de Datos

Swift es un lenguaje seguro en cuanto a tipos, lo que significa que el lenguaje te ayuda a ser claro sobre los tipos de valores con los que tu código puede trabajar.

let integer: Int = 70
let double: Double = 70.0
let string: String = "¡Hola, Swift!"
let boolean: Bool = true

Cadenas y Caracteres

Las cadenas en Swift son compatibles con Unicode y pueden contener cualquier carácter.

let greeting = "¡Hola, Mundo!"
let character: Character = "A"

Arrays y Diccionarios

Los arrays y diccionarios se utilizan para almacenar colecciones de valores.

var shoppingList: [String] = ["Huevos", "Leche"]
var occupations: [String: String] = ["Malcolm": "Capitán", "Kaylee": "Mecánica"]

Control de Flujo

Swift proporciona todas las estructuras tradicionales de control de flujo, como los bucles for y while, así como las declaraciones if y switch.

let individualScores = [75, 43, 103, 87, 12]
var teamScore = 0
for score in individualScores {
    if score > 50 {
        teamScore += 3
    } else {
        teamScore += 1
    }
}
print(teamScore)

Funciones

Las funciones son fragmentos de código autónomos que realizan una tarea específica.

func greet(person: String, day: String) -> String {
    return "Hola \(person), hoy es \(day)."
}
print(greet(person: "John", day: "miércoles"))

Conclusión

Swift es un lenguaje de programación moderno, poderoso y fácil de aprender que es perfecto para desarrollar aplicaciones en todas las plataformas de Apple. Al comprender la sintaxis básica y las características de Swift, puedes comenzar a construir tus propias aplicaciones y aprovechar el rico ecosistema proporcionado por Apple. A medida que avances en este curso, profundizarás en temas más avanzados y te volverás competente en la programación con Swift.
