Herencia en Swift
Polimorfismo y Conversión de Tipos ►

La herencia es un concepto fundamental en la programación orientada a objetos que permite a una clase heredar propiedades y métodos de otra clase. Esto ayuda a reutilizar el código y a crear una relación jerárquica entre las clases.

Conceptos Clave

  • Superclase: La clase cuyas propiedades y métodos son heredados.
  • Subclase: La clase que hereda propiedades y métodos de la superclase.
  • Sobrescritura: El proceso de proporcionar una nueva implementación para un método o propiedad heredada.
  • Clases Finales: Clases que no pueden ser heredadas.

Creando una Superclase

Para crear una superclase en Swift, defines una clase con propiedades y métodos que pueden ser heredados por otras clases.

class Vehicle {
    var currentSpeed = 0.0
    
    func description() -> String {
        return "Traveling at \(currentSpeed) miles per hour"
    }
}

En este ejemplo, Vehicle es una superclase con una propiedad currentSpeed y un método description().

Creando una Subclase

Una subclase hereda propiedades y métodos de su superclase. Creas una subclase especificando la superclase después del nombre de la subclase, separada por dos puntos.

class Bicycle: Vehicle {
    var hasBasket = false
}

Aquí, Bicycle es una subclase de Vehicle. Hereda la propiedad currentSpeed y el método description() de Vehicle.

Sobrescritura de Métodos

Una subclase puede sobrescribir métodos y propiedades de su superclase para proporcionar una implementación específica.

class Bicycle: Vehicle {
    var hasBasket = false
    
    override func description() -> String {
        return "Traveling at \(currentSpeed) miles per hour on a bicycle"
    }
}

En este ejemplo, el método description() es sobrescrito en la clase Bicycle para proporcionar una descripción más específica.

Uso de Métodos y Propiedades de la Superclase

Una subclase puede usar la palabra clave super para acceder a métodos y propiedades de su superclase.

class Tandem: Bicycle {
    var currentNumberOfPassengers = 0
    
    override func description() -> String {
        return super.description() + " with \(currentNumberOfPassengers) passengers"
    }
}

Aquí, la clase Tandem, que es una subclase de Bicycle, usa super.description() para llamar al método description() de su superclase y luego añade información adicional.

Clases Finales

Si quieres evitar que una clase sea heredada, puedes marcarla como final.

final class Train: Vehicle {
    var numberOfCarriages = 0
}

La clase Train no puede ser subclasificada porque está marcada como final.

Ejercicios

  1. Crear una Superclase y Subclase:

    • Define una superclase Animal con una propiedad name y un método makeSound().
    • Crea una subclase Dog que sobrescriba el método makeSound() para imprimir "¡Guau!".

  2. Sobrescribir Métodos:

    • Crea una superclase Shape con un método area().
    • Crea subclases Circle y Rectangle que sobrescriban el método area() para calcular el área de la forma.

  3. Usar Métodos de la Superclase:

    • Crea una superclase Employee con un método work().
    • Crea una subclase Manager que sobrescriba el método work() y use super.work() para llamar al método de la superclase.

Resumen

La herencia te permite crear una nueva clase basada en una clase existente, promoviendo la reutilización del código y estableciendo una relación jerárquica entre las clases. Los conceptos clave incluyen superclases, subclases, sobrescritura de métodos y clases finales. Al entender y utilizar la herencia, puedes crear un código más organizado y mantenible en Swift.
