Introducción a los Diccionarios

Los diccionarios en Swift son colecciones que almacenan asociaciones entre claves del mismo tipo y valores del mismo tipo. Cada valor está asociado con una clave única, que actúa como un identificador para ese valor dentro del diccionario.

  • Par Clave-Valor: Cada entrada en un diccionario es un par clave-valor.
  • Claves Únicas: Las claves deben ser únicas dentro de un diccionario.
  • Colección Desordenada: Los diccionarios son colecciones desordenadas, lo que significa que el orden de los pares clave-valor no está garantizado.

Creación e Inicialización de Diccionarios

Puedes crear diccionarios usando literales de diccionario o el inicializador del tipo Dictionary.

Usando Literales de Diccionario

var studentGrades: [String: String] = ["John": "A", "Jane": "B", "Jim": "C"]

En este ejemplo:

  • El diccionario studentGrades tiene claves String y valores String.
  • Las claves son nombres de estudiantes y los valores son sus calificaciones.

Usando el Inicializador de Diccionario

var emptyDictionary = [Int: String]()

Esto crea un diccionario vacío con claves Int y valores String.

Acceso y Modificación de Diccionarios

Puedes acceder y modificar diccionarios usando subíndices y varios métodos.

Acceso a Valores

let johnGrade = studentGrades["John"]
print(johnGrade) // Optional("A")
  • Acceder a un valor devuelve un opcional porque la clave podría no existir.

Modificación de Valores

studentGrades["Jane"] = "A+"
  • Esto actualiza el valor para la clave "Jane" a "A+".

Adición de Nuevos Pares Clave-Valor

studentGrades["Jake"] = "B"
  • Esto añade un nuevo par clave-valor al diccionario.

Eliminación de Pares Clave-Valor

studentGrades["Jim"] = nil
  • Esto elimina el par clave-valor para "Jim".

Iteración Sobre Diccionarios

Puedes iterar sobre los pares clave-valor en un diccionario usando un bucle for-in.

for (student, grade) in studentGrades {
    print("\(student): \(grade)")
}
  • Esto imprime cada par clave-valor en el diccionario.

Métodos Comunes de Diccionario

Swift proporciona varios métodos útiles para trabajar con diccionarios.

`count`

Devuelve el número de pares clave-valor en el diccionario.

print(studentGrades.count) // 3

`isEmpty`

Verifica si el diccionario está vacío.

print(studentGrades.isEmpty) // false

`removeValue(forKey:)`

Elimina el valor para la clave especificada y lo devuelve.

if let removedGrade = studentGrades.removeValue(forKey: "Jane") {
    print("Removed grade: \(removedGrade)")
}

`keys` y `values`

Propiedades que devuelven colecciones de las claves y valores del diccionario.

let students = studentGrades.keys
let grades = studentGrades.values

Características Avanzadas de Diccionarios

Los diccionarios en Swift ofrecen características más avanzadas para desarrolladores experimentados.

Fusión de Diccionarios

Puedes fusionar dos diccionarios usando el método merge(_:uniquingKeysWith:).

var additionalGrades = ["Alice": "A", "Bob": "B"]
studentGrades.merge(additionalGrades) { (current, _) in current }
  • Esto fusiona additionalGrades en studentGrades, manteniendo los valores actuales para claves duplicadas.

Filtrado de Diccionarios

Puedes filtrar diccionarios usando el método filter.

let filteredGrades = studentGrades.filter { $0.value == "A" }
print(filteredGrades) // ["John": "A", "Alice": "A"]
  • Esto crea un nuevo diccionario con solo los pares clave-valor donde el valor es "A".

Transformación de Diccionarios

Puedes transformar diccionarios usando el método mapValues.

let uppercasedGrades = studentGrades.mapValues { $0.uppercased() }
print(uppercasedGrades) // ["John": "A", "Jane": "A+", "Jake": "B", "Alice": "A", "Bob": "B"]
  • Esto crea un nuevo diccionario con los valores transformados a mayúsculas.

Conclusión

Los diccionarios en Swift son colecciones poderosas y flexibles que te permiten almacenar y gestionar pares clave-valor de manera eficiente. Ya seas un principiante o un desarrollador avanzado, entender cómo crear, acceder, modificar y utilizar diccionarios es esencial para una programación efectiva en Swift. Al dominar estos conceptos, puedes manejar estructuras de datos complejas y realizar varias operaciones con facilidad.
