Los genéricos son una característica poderosa en Swift que te permiten escribir código flexible y reutilizable. Al usar genéricos, puedes escribir funciones y tipos que pueden trabajar con cualquier tipo, sujeto a los requisitos que definas.

¿Qué son los Genéricos?

Los genéricos te permiten escribir funciones y tipos que pueden operar sobre cualquier tipo, en lugar de un tipo específico. Esto ayuda a crear código más flexible y reutilizable.

Conceptos Clave

  • Parámetros de Tipo: Marcadores de posición para tipos que se usan en funciones o tipos genéricos.
  • Funciones Genéricas: Funciones que pueden operar sobre cualquier tipo.
  • Tipos Genéricos: Tipos (como clases, estructuras y enumeraciones) que pueden trabajar con cualquier tipo.
  • Restricciones de Tipo: Restricciones que especifican que un parámetro de tipo debe conformarse a un cierto protocolo o heredar de una cierta clase.

Funciones Genéricas

Una función genérica puede trabajar con cualquier tipo. Aquí hay un ejemplo simple:

func swapTwoValues<T>(_ a: inout T, _ b: inout T) {
    let temporaryA = a
    a = b
    b = temporaryA
}
  • T es un parámetro de tipo marcador de posición.
  • La función swapTwoValues puede intercambiar los valores de cualquier tipo, siempre y cuando ambos valores sean del mismo tipo.

Ejemplo de Uso

var a = 5
var b = 10
swapTwoValues(&a, &b)
print("a: \(a), b: \(b)") // Salida: a: 10, b: 5

var x = "Hola"
var y = "Mundo"
swapTwoValues(&x, &y)
print("x: \(x), y: \(y)") // Salida: x: Mundo, y: Hola

Tipos Genéricos

Puedes definir tus propios tipos genéricos para crear estructuras de datos flexibles y reutilizables.

Ejemplo: Pila Genérica

struct Stack<Element> {
    var items = [Element]()
    
    mutating func push(_ item: Element) {
        items.append(item)
    }
    
    mutating func pop() -> Element {
        return items.removeLast()
    }
}
  • Element es un parámetro de tipo marcador de posición.
  • La Stack puede almacenar elementos de cualquier tipo.

Ejemplo de Uso

var intStack = Stack<Int>()
intStack.push(1)
intStack.push(2)
print(intStack.pop()) // Salida: 2

var stringStack = Stack<String>()
stringStack.push("Hola")
stringStack.push("Mundo")
print(stringStack.pop()) // Salida: Mundo

Restricciones de Tipo

Las restricciones de tipo especifican que un parámetro de tipo debe conformarse a un cierto protocolo o heredar de una cierta clase.

Ejemplo: Restricciones de Tipo

func findIndex<T: Equatable>(of valueToFind: T, in array: [T]) -> Int? {
    for (index, value) in array.enumerated() {
        if value == valueToFind {
            return index
        }
    }
    return nil
}
  • T: Equatable especifica que el parámetro de tipo T debe conformarse al protocolo Equatable.
  • Esto permite que la función use el operador == para comparar valores.

Ejemplo de Uso

let numbers = [1, 2, 3, 4, 5]
if let index = findIndex(of: 3, in: numbers) {
    print("El índice de 3 es \(index)") // Salida: El índice de 3 es 2
}

let strings = ["manzana", "banana", "cereza"]
if let index = findIndex(of: "banana", in: strings) {
    print("El índice de banana es \(index)") // Salida: El índice de banana es 1
}

Genéricos Avanzados

Swift permite características genéricas más avanzadas, como tipos asociados en protocolos y subíndices genéricos.

Tipos Asociados en Protocolos

Un tipo asociado da un nombre de marcador de posición a un tipo usado como parte del protocolo.

protocol Container {
    associatedtype Item
    mutating func append(_ item: Item)
    var count: Int { get }
    subscript(i: Int) -> Item { get }
}

Ejemplo: Uso de Tipos Asociados

struct IntStack: Container {
    var items = [Int]()
    
    mutating func push(_ item: Int) {
        items.append(item)
    }
    
    mutating func pop() -> Int {
        return items.removeLast()
    }
    
    mutating func append(_ item: Int) {
        self.push(item)
    }
    
    var count: Int {
        return items.count
    }
    
    subscript(i: Int) -> Int {
        return items[i]
    }
}

Conclusión

Los genéricos son una característica fundamental en Swift que te permiten escribir código flexible, reutilizable y seguro en cuanto a tipos. Al entender y utilizar los genéricos, puedes crear aplicaciones más robustas y mantenibles. Ya sea que estés escribiendo funciones genéricas, tipos o usando características avanzadas como tipos asociados, los genéricos proporcionan un conjunto de herramientas poderoso para los desarrolladores de Swift.
