Introducción a los Atributos HTML

Los atributos HTML proporcionan información adicional sobre los elementos HTML. Siempre se incluyen en la etiqueta de apertura y generalmente vienen en pares de nombre/valor como name="value".

Conceptos Clave

  • Los atributos siempre se especifican en la etiqueta de inicio.
  • Los atributos generalmente vienen en pares de nombre/valor como name="value".
  • Los atributos comunes incluyen id, class, style, title y href.

Atributos HTML Comunes

El Atributo `id`

El atributo id especifica un id único para un elemento HTML. El valor del atributo id debe ser único dentro del documento HTML.

<p id="unique-paragraph">Este es un párrafo con un id único.</p>

El Atributo `class`

El atributo class se utiliza para definir una clase para los elementos HTML. Múltiples elementos pueden compartir la misma clase.

<p class="intro">Este es un párrafo introductorio.</p>
<p class="intro">Este es otro párrafo introductorio.</p>

El Atributo `style`

El atributo style se utiliza para agregar CSS en línea a un elemento.

<p style="color: blue; font-size: 20px;">Este es un párrafo con estilo.</p>

El Atributo `title`

El atributo title proporciona información adicional sobre un elemento. La información a menudo se muestra como una herramienta emergente cuando el mouse se mueve sobre el elemento.

<p title="Este es un tooltip">Pasa el ratón sobre este párrafo para ver el tooltip.</p>

El Atributo `href`

El atributo href especifica la URL de la página a la que va el enlace.

<a href="https://www.example.com">Visita Example.com</a>

Atributos HTML Avanzados

Los Atributos `data-*`

Los atributos data-* se utilizan para almacenar datos personalizados privados para la página o aplicación.

<div data-user-id="12345" data-role="admin">Información del Usuario</div>

El Atributo `src`

El atributo src especifica la URL de un recurso externo, más comúnmente utilizado con imágenes, scripts e iframes.

<img src="image.jpg" alt="Descripción de la imagen">

El Atributo `alt`

El atributo alt proporciona texto alternativo para una imagen si la imagen no puede mostrarse.

<img src="image.jpg" alt="Descripción de la imagen">

El Atributo `disabled`

El atributo disabled puede aplicarse a elementos de formulario para deshabilitarlos.

<input type="text" disabled>

El Atributo `readonly`

El atributo readonly puede aplicarse a campos de entrada para hacerlos de solo lectura.

<input type="text" value="No puedes cambiar esto" readonly>

Comparación de Atributos Comunes vs. Avanzados

| Atributo | Descripción | Común/Avanzado | |------------|--------------------------------------------------|-----------------| | id | Identificador único para un elemento | Común | | class | Nombre(s) de clase para un elemento | Común | | style | Estilos CSS en línea | Común | | title | Texto de tooltip | Común | | href | URL para un enlace | Común | | data-* | Atributos de datos personalizados | Avanzado | | src | URL de un recurso externo | Avanzado | | alt | Texto alternativo para una imagen | Avanzado | | disabled | Deshabilita un elemento de formulario | Avanzado | | readonly | Hace un campo de entrada de solo lectura | Avanzado |

Conclusión

Los atributos HTML son esenciales para agregar información y funcionalidad adicional a los elementos HTML. Comprender tanto los atributos comunes como los avanzados te permite crear páginas web más dinámicas e interactivas. Practica usando estos atributos en tus documentos HTML para volverte más competente en el desarrollo web.
