Introducción

El texto preformateado en HTML se utiliza para mostrar el texto exactamente como está escrito en el archivo HTML, preservando espacios, saltos de línea y otros caracteres de espacio en blanco. Esto es particularmente útil para mostrar fragmentos de código, poesía o cualquier texto donde el formato sea crucial.

La etiqueta `
`

El elemento HTML principal utilizado para el texto preformateado es la etiqueta <pre>.

  • Sintaxis: 
    <pre>
  Tu texto preformateado va aquí.
</pre>
  • Atributos: La etiqueta <pre> admite atributos globales como id, class, style, etc.

Ejemplo

<pre>
  Este es un ejemplo de texto preformateado.
  Observa cómo se conservan los espacios y los saltos de línea.
</pre>

Cuando se renderiza en un navegador, el texto aparecerá exactamente como está escrito dentro de las etiquetas <pre>, incluyendo todos los espacios y saltos de línea.

Combinando `
` con otras etiquetas

Puedes combinar la etiqueta <pre> con otras etiquetas HTML para mejorar la presentación del texto preformateado.

Usando `` dentro de `
`

La etiqueta <code> se usa a menudo dentro de <pre> para denotar fragmentos de código.

  • Sintaxis: 
    <pre>
  <code>
    // Este es un fragmento de código
    function helloWorld() {
      console.log("¡Hola, Mundo!");
    }
  </code>
</pre>

Ejemplo

<pre>
  <code>
    // Esta es una función de JavaScript
    function greet(name) {
      console.log("¡Hola, " + name + "!");
    }
  </code>
</pre>

Estilizando texto preformateado

Puedes estilizar el texto preformateado usando CSS para hacerlo más legible o para que coincida con el diseño de tu sitio web.

Ejemplo

<style>
  pre {
    background-color: #f4f4f4;
    padding: 10px;
    border: 1px solid #ccc;
    overflow: auto;
  }
  code {
    color: #d63384;
  }
</style>

<pre>
  <code>
    // Fragmento de código estilizado
    const add = (a, b) => a + b;
  </code>
</pre>

Ejercicios prácticos

Ejercicio 1: Texto preformateado básico

Crea un archivo HTML simple que use la etiqueta <pre> para mostrar un poema con un formato específico.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo de Texto Preformateado</title>
</head>
<body>
  <pre>
    Las rosas son rojas,
    Las violetas son azules,
    El azúcar es dulce,
    Y tú también.
  </pre>
</body>
</html>

Ejercicio 2: Mostrando fragmentos de código

Crea un archivo HTML que use tanto las etiquetas <pre> como <code> para mostrar un fragmento de código en Python.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo de Fragmento de Código</title>
</head>
<body>
  <pre>
    <code>
      def greet(name):
          print(f"¡Hola, {name}!")
    </code>
  </pre>
</body>
</html>

Ejercicio 3: Estilizando texto preformateado

Crea un archivo HTML que estilice el texto preformateado usando CSS.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Texto Preformateado Estilizado</title>
  <style>
    pre {
      background-color: #f0f0f0;
      padding: 15px;
      border-radius: 5px;
    }
    code {
      color: #2a9d8f;
    }
  </style>
</head>
<body>
  <pre>
    <code>
      // Función de JavaScript
      function add(a, b) {
        return a + b;
      }
    </code>
  </pre>
</body>
</html>

Conclusión

El texto preformateado es una característica poderosa en HTML que te permite mostrar el texto exactamente como está escrito, preservando todos los espacios en blanco y saltos de línea. La etiqueta <pre> es esencial para este propósito, y se puede combinar con otras etiquetas como <code> para mostrar fragmentos de código de manera efectiva. Estilizar el texto preformateado con CSS puede mejorar su legibilidad e integración con el diseño de tu sitio web. Al entender y practicar estos conceptos, puedes usar efectivamente el texto preformateado en tus proyectos web.
