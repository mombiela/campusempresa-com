Introducción a SVG

Scalable Vector Graphics (SVG) es un formato basado en XML para crear imágenes vectoriales. A diferencia de las imágenes rasterizadas (por ejemplo, JPEG, PNG), las imágenes SVG pueden escalarse a cualquier tamaño sin perder calidad, lo que las hace ideales para gráficos web.

Conceptos Clave

  • Gráficos Vectoriales: Imágenes definidas por puntos, líneas, curvas y formas, que pueden escalarse sin pérdida de calidad.
  • Basado en XML: Los archivos SVG están escritos en XML, lo que los hace legibles tanto para humanos como para máquinas.
  • Interactivo y Animable: Los elementos SVG pueden ser estilizados, animados y hechos interactivos usando CSS y JavaScript.

Estructura Básica de SVG

Los elementos SVG pueden ser incrustados directamente en documentos HTML usando la etiqueta <svg>.

Ejemplo: Círculo SVG Simple

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejemplo SVG</title>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
    </svg>
</body>
</html>

Explicación

  • <svg width="100" height="100">: Define el contenedor SVG con un ancho y alto de 100 unidades.
  • <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />: Dibuja un círculo con:
    • cx y cy: Coordenadas del centro (50, 50).
    • r: Radio (40).
    • stroke: Color del borde (negro).
    • stroke-width: Grosor del borde (3 unidades).
    • fill: Color de relleno (rojo).

Elementos Comunes de SVG

  • <rect>: Dibuja rectángulos.
  • <circle>: Dibuja círculos.
  • <ellipse>: Dibuja elipses.
  • <line>: Dibuja líneas rectas.
  • <polyline>: Dibuja una serie de líneas conectadas.
  • <polygon>: Dibuja una forma cerrada con múltiples lados.
  • <path>: Dibuja formas complejas usando una serie de comandos.

Ejemplo: Varios Elementos SVG

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Elementos SVG</title>
</head>
<body>
    <svg width="200" height="200">
        <rect x="10" y="10" width="50" height="50" stroke="black" stroke-width="2" fill="blue" />
        <circle cx="100" cy="50" r="30" stroke="black" stroke-width="2" fill="green" />
        <ellipse cx="150" cy="150" rx="30" ry="20" stroke="black" stroke-width="2" fill="yellow" />
        <line x1="10" y1="150" x2="50" y2="190" stroke="black" stroke-width="2" />
        <polyline points="60,150 80,170 100,150" stroke="black" stroke-width="2" fill="none" />
        <polygon points="110,150 130,170 150,150" stroke="black" stroke-width="2" fill="purple" />
        <path d="M 160 150 Q 180 170 200 150" stroke="black" stroke-width="2" fill="none" />
    </svg>
</body>
</html>

Estilizando SVG con CSS

Los elementos SVG pueden ser estilizados usando CSS, al igual que los elementos HTML.

Ejemplo: Estilizando SVG con CSS

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>SVG Estilizado</title>
    <style>
        .my-circle {
            fill: orange;
            stroke: blue;
            stroke-width: 4;
        }
    </style>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" class="my-circle" />
    </svg>
</body>
</html>

Explicación

  • La clase .my-circle está definida en la sección <style>.
  • El atributo class="my-circle" se añade al elemento <circle> para aplicar los estilos.

Animando SVG

Los elementos SVG pueden ser animados usando la etiqueta <animate> o animaciones CSS.

Ejemplo: Animación SVG Simple

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Animación SVG</title>
</head>
<body>
    <svg width="100" height="100">
        <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red">
            <animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite" />
        </circle>
    </svg>
</body>
</html>

Explicación

  • <animate attributeName="cx" from="50" to="150" dur="2s" repeatCount="indefinite" />: Anima el atributo cx del círculo de 50 a 150 en 2 segundos, repitiendo indefinidamente.

Conclusión

SVG es una herramienta poderosa para crear gráficos escalables, interactivos y animados en HTML. Al comprender la estructura básica, los elementos comunes, las técnicas de estilizado y animación, puedes mejorar tus proyectos web con gráficos vectoriales de alta calidad.
