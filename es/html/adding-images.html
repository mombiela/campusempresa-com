Introducción

Añadir imágenes a una página web es una habilidad fundamental en HTML. Las imágenes pueden mejorar el atractivo visual de tu sitio web y proporcionar un contexto importante a tu contenido. En esta sección, cubriremos los conceptos básicos de añadir imágenes, incluyendo atributos, mejores prácticas y técnicas avanzadas.

Etiqueta Básica de Imagen

La etiqueta HTML <img> se utiliza para incrustar imágenes en una página web. Esta etiqueta es auto-cerrada, lo que significa que no tiene una etiqueta de cierre.

Sintaxis

<img src="image-url" alt="description">
  • src: Especifica la ruta al archivo de imagen.
  • alt: Proporciona texto alternativo para la imagen si no puede ser mostrada.

Ejemplo

<img src="https://example.com/image.jpg" alt="Imagen de Ejemplo">

Explicación

  • src="https://example.com/image.jpg": Este atributo especifica la URL de la imagen.
  • alt="Imagen de Ejemplo": Este atributo proporciona una descripción textual de la imagen, lo cual es útil para la accesibilidad y SEO.

Atributos de Imagen

Hay varios atributos que puedes usar con la etiqueta <img> para controlar la apariencia y el comportamiento de las imágenes.

Atributos Comunes

  • width y height: Definen las dimensiones de la imagen.
  • title: Proporciona información adicional sobre la imagen, a menudo mostrada como un tooltip.
  • loading: Especifica el comportamiento de carga de la imagen (por ejemplo, lazy para carga diferida).

Ejemplo con Atributos

<img src="https://example.com/image.jpg" alt="Imagen de Ejemplo" width="300" height="200" title="Una imagen de muestra" loading="lazy">

Explicación

  • width="300" y height="200": Estos atributos establecen el ancho y la altura de la imagen en 300 píxeles y 200 píxeles, respectivamente.
  • title="Una imagen de muestra": Este atributo añade un tooltip que aparece cuando el usuario pasa el cursor sobre la imagen.
  • loading="lazy": Este atributo difiere la carga de la imagen hasta que sea necesaria, lo que puede mejorar los tiempos de carga de la página.

Imágenes Responsivas

Las imágenes responsivas ajustan su tamaño y resolución según el dispositivo y el tamaño de la pantalla. Esto asegura que las imágenes se vean bien en todos los dispositivos.

Usando el Atributo `srcset`

El atributo srcset te permite especificar múltiples fuentes de imagen para diferentes tamaños de pantalla y resoluciones.

Ejemplo

<img src="small.jpg" srcset="small.jpg 500w, medium.jpg 1000w, large.jpg 1500w" sizes="(max-width: 600px) 480px, (max-width: 1200px) 800px, 1000px" alt="Imagen Responsiva">

Explicación

  • srcset="small.jpg 500w, medium.jpg 1000w, large.jpg 1500w": Este atributo proporciona diferentes archivos de imagen para diferentes resoluciones.
    • small.jpg 500w: Usa small.jpg para pantallas de hasta 500 píxeles de ancho.
    • medium.jpg 1000w: Usa medium.jpg para pantallas de hasta 1000 píxeles de ancho.
    • large.jpg 1500w: Usa large.jpg para pantallas de hasta 1500 píxeles de ancho.
  • sizes="(max-width: 600px) 480px, (max-width: 1200px) 800px, 1000px": Este atributo define los tamaños de las imágenes basados en el ancho de la pantalla.

Técnicas Avanzadas

Usando el Elemento ``

El elemento <picture> proporciona más control sobre qué imagen mostrar basándose en diferentes condiciones, como el tamaño de la pantalla o el tipo de dispositivo.

Ejemplo

<picture>
  <source media="(max-width: 600px)" srcset="small.jpg">
  <source media="(max-width: 1200px)" srcset="medium.jpg">
  <img src="large.jpg" alt="Imagen Responsiva Avanzada">
</picture>

Explicación

  • <source media="(max-width: 600px)" srcset="small.jpg">: Usa small.jpg para pantallas de hasta 600 píxeles de ancho.
  • <source media="(max-width: 1200px)" srcset="medium.jpg">: Usa medium.jpg para pantallas de hasta 1200 píxeles de ancho.
  • <img src="large.jpg" alt="Imagen Responsiva Avanzada">: Imagen de respaldo si ninguna de las condiciones se cumple.

Conclusión

Añadir imágenes a tus páginas web es una habilidad crucial en HTML. Al entender la etiqueta básica <img>, usar los atributos de manera efectiva e implementar imágenes responsivas, puedes asegurarte de que tus imágenes mejoren la experiencia del usuario en tu sitio web. Técnicas avanzadas como el elemento <picture> proporcionan aún más control sobre cómo se muestran las imágenes, haciendo que tu sitio sea adaptable a varios dispositivos y tamaños de pantalla.
