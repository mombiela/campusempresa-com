Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar documentos en la World Wide Web. Estructura el contenido web y permite a los navegadores mostrar texto, imágenes y otros elementos en un formato legible.

Conceptos Clave de HTML

  • Elementos: Los bloques de construcción de HTML, definidos por etiquetas.
  • Etiquetas: Palabras clave rodeadas por corchetes angulares, por ejemplo, <tagname>.
  • Atributos: Proporcionan información adicional sobre los elementos, generalmente en forma de pares nombre/valor.
  • Modelo de Objeto de Documento (DOM): Una estructura en forma de árbol que representa el documento HTML.

Estructura Básica de un Documento HTML

Un documento HTML tiene una estructura específica que incluye varios elementos clave:

<!DOCTYPE html>
<html>
<head>
    <title>Título de la Página</title>
</head>
<body>
    <h1>Este es un Encabezado</h1>
    <p>Este es un párrafo.</p>
</body>
</html>

Explicación de la Estructura Básica

  • <!DOCTYPE html>: Declara el tipo de documento y la versión de HTML.
  • <html>: El elemento raíz de una página HTML.
  • <head>: Contiene meta-información sobre el documento HTML, como el título y enlaces a hojas de estilo.
  • <title>: Establece el título de la página web, mostrado en la barra de título o pestaña del navegador.
  • <body>: Contiene el contenido del documento HTML, como texto, imágenes y otros medios.

Elementos Comunes de HTML

Aquí hay algunos elementos HTML comúnmente utilizados:

  • Encabezados: Definidos con las etiquetas <h1> a <h6>.
  • Párrafos: Definidos con la etiqueta <p>.
  • Enlaces: Definidos con la etiqueta <a>.
  • Imágenes: Definidas con la etiqueta <img>.
  • Listas: Listas ordenadas (<ol>) y listas desordenadas (<ul>).

Ejemplos de Elementos Comunes de HTML

Encabezados

<h1>Este es un Encabezado 1</h1>
<h2>Este es un Encabezado 2</h2>

Párrafos

<p>Este es un párrafo de texto.</p>

Enlaces

<a href="https://www.example.com">Este es un enlace</a>

Imágenes

<img src="image.jpg" alt="Descripción de la imagen">

Listas

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
</ol>

Atributos en HTML

Los atributos proporcionan información adicional sobre los elementos HTML. Siempre se incluyen en la etiqueta de apertura y generalmente vienen en pares nombre/valor como name="value".

Ejemplo de Atributos

<a href="https://www.example.com" target="_blank">Visitar Ejemplo</a>
<img src="image.jpg" alt="Descripción de la imagen" width="500" height="600">
  • href: Especifica la URL de la página a la que va el enlace.
  • target: Especifica dónde abrir el documento enlazado.
  • src: Especifica la ruta a la imagen.
  • alt: Proporciona texto alternativo para la imagen.
  • width y height: Especifican las dimensiones de la imagen.

Conclusión

HTML es el lenguaje fundamental para crear páginas web. Entender su estructura básica, elementos y atributos es esencial para cualquiera que desee construir o diseñar sitios web. A medida que avances en este curso, aprenderás conceptos más avanzados de HTML y cómo integrarlos con CSS y JavaScript para crear páginas web dinámicas y visualmente atractivas.
