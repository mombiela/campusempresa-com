Citas y Referencias en HTML
Texto Preformateado ►

Las citas y referencias son esenciales para referenciar fuentes externas, proporcionar contexto y añadir credibilidad a tu contenido web. En HTML, hay etiquetas específicas diseñadas para manejar estas tareas de manera efectiva.

Citas en Bloque

Las citas en bloque se utilizan para citas más largas que normalmente se muestran como un bloque de texto separado, a menudo con sangría.

La Etiqueta `
`

  • La etiqueta <blockquote> se utiliza para definir una sección que se cita de otra fuente.
  • Normalmente se muestra como un bloque de texto con sangría.

Ejemplo:

<blockquote>
  "El único límite para nuestra realización del mañana son nuestras dudas de hoy."
  - Franklin D. Roosevelt
</blockquote>

En este ejemplo, la cita de Franklin D. Roosevelt se muestra como una cita en bloque.

El Atributo `cite`

  • El atributo cite se puede usar dentro de la etiqueta <blockquote> para proporcionar una URL a la fuente de la cita.

Ejemplo:

<blockquote cite="https://www.example.com/roosevelt-quote">
  "El único límite para nuestra realización del mañana son nuestras dudas de hoy."
  - Franklin D. Roosevelt
</blockquote>

Aquí, el atributo cite especifica la URL de la fuente de la cita.

Citas en Línea

Las citas en línea se utilizan para citas más cortas que forman parte de una oración o párrafo más grande.

La Etiqueta ``

  • La etiqueta <q> se utiliza para citas cortas en línea.
  • Los navegadores normalmente renderizan el contenido de la etiqueta <q> con comillas.

Ejemplo:

<p>Franklin D. Roosevelt una vez dijo, <q>El único límite para nuestra realización del mañana son nuestras dudas de hoy.</q></p>

En este ejemplo, la cita es parte de una oración más grande y está encerrada entre comillas.

Referencias

Las referencias se utilizan para citar la fuente de una cita, una pieza de información o una obra.

La Etiqueta ``

  • La etiqueta <cite> se utiliza para definir el título de una obra, como un libro, un sitio web o un artículo de investigación.
  • Normalmente se renderiza en cursiva por los navegadores.

Ejemplo:

<p>Una de las citas más famosas de Franklin D. Roosevelt se puede encontrar en <cite>The Public Papers and Addresses of Franklin D. Roosevelt</cite>.</p>

En este ejemplo, el título de la obra está encerrado en la etiqueta <cite> y se renderiza en cursiva.

Combinando Citas y Referencias

Puedes combinar citas en bloque y referencias para proporcionar una referencia completa.

Ejemplo:

<blockquote cite="https://www.example.com/roosevelt-quote">
  "El único límite para nuestra realización del mañana son nuestras dudas de hoy."
  - Franklin D. Roosevelt
  <footer>Fuente: <cite>The Public Papers and Addresses of Franklin D. Roosevelt</cite></footer>
</blockquote>

En este ejemplo, la cita en bloque incluye una referencia dentro de una etiqueta <footer>, proporcionando una referencia completa a la fuente.

Resumen

  • Usa la etiqueta <blockquote> para citas más largas a nivel de bloque.
  • Usa el atributo cite dentro de <blockquote> para proporcionar una URL de la fuente.
  • Usa la etiqueta <q> para citas más cortas en línea.
  • Usa la etiqueta <cite> para referenciar el título de una obra.
  • Combina citas y referencias para proporcionar referencias completas.

Entender cómo usar correctamente las citas y referencias en HTML ayuda a asegurar que tu contenido web sea creíble, bien referenciado y fácil de leer.
