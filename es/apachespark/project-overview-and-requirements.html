Introducción

En esta sección, proporcionaremos una descripción general del proyecto en el que trabajarás a lo largo de este curso de Apache Spark. Este proyecto te ayudará a aplicar los conceptos y técnicas que aprendas de una manera práctica y práctica. También describiremos los requisitos y prerrequisitos necesarios para completar con éxito el proyecto.

Descripción General del Proyecto

El proyecto de este curso implica la construcción de una canalización de procesamiento de datos utilizando Apache Spark. La canalización ingerirá, procesará y analizará un gran conjunto de datos para extraer información significativa. Al final del proyecto, tendrás una comprensión sólida de cómo usar Spark para el procesamiento y análisis de grandes volúmenes de datos.

Objetivos Clave

  • Ingesta de Datos: Cargar datos de varias fuentes en Spark.
  • Procesamiento de Datos: Limpiar, transformar y agregar datos utilizando transformaciones y acciones de Spark.
  • Análisis de Datos: Realizar análisis exploratorio de datos (EDA) y generar información.
  • Optimización: Optimizar trabajos de Spark para un mejor rendimiento.
  • Visualización: Visualizar los resultados utilizando herramientas apropiadas.

Requisitos

Prerrequisitos

Antes de comenzar el proyecto, asegúrate de tener los siguientes prerrequisitos:

  • Conocimientos Básicos de Programación: Familiaridad con Python o Scala.
  • Comprensión de Conceptos de Big Data: Conocimientos básicos de tecnologías y conceptos de big data.
  • Entorno de Desarrollo: Una instalación funcional de Apache Spark y un IDE compatible (por ejemplo, IntelliJ IDEA, PyCharm).

Software y Herramientas

  • Apache Spark: El motor principal para el procesamiento de datos a gran escala.
  • Hadoop (Opcional): Para HDFS si planeas usarlo como capa de almacenamiento.
  • Jupyter Notebook: Para análisis de datos interactivo y visualización.
  • Python/Scala: Lenguajes de programación para escribir aplicaciones Spark.
  • Herramientas de Visualización: Bibliotecas como Matplotlib, Seaborn o Plotly para la visualización de datos.

Conjunto de Datos

Para este proyecto, utilizaremos un conjunto de datos disponible públicamente. Puedes elegir entre varios conjuntos de datos disponibles en plataformas como Kaggle, UCI Machine Learning Repository o data.gov. Asegúrate de que el conjunto de datos sea lo suficientemente grande como para demostrar las capacidades de Spark pero manejable dentro del alcance de este curso.

Configuración del Entorno

Para configurar tu entorno de desarrollo, sigue estos pasos:

  1. Instalar Apache Spark:

    # Para MacOS usando Homebrew
brew install apache-spark

# Para Ubuntu
sudo apt-get update
sudo apt-get install -y spark

  2. Instalar Jupyter Notebook:

    pip install notebook

  3. Instalar Bibliotecas Requeridas:

    pip install pyspark matplotlib seaborn

  4. Verificar Instalación:

    import pyspark
print(pyspark.__version__)
