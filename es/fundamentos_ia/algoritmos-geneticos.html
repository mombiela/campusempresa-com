Introducción a los Algoritmos Genéticos

Los algoritmos genéticos (AG) son una técnica de optimización inspirada en el proceso de selección natural de Charles Darwin. Utilizan conceptos de genética y evolución para resolver problemas complejos de búsqueda y optimización.

Conceptos Clave

  • Población: Conjunto de posibles soluciones al problema.
  • Cromosoma: Representación de una solución individual.
  • Gen: Elemento de un cromosoma que representa una característica de la solución.
  • Fitness: Medida de la calidad de una solución.
  • Selección: Proceso de elegir los mejores cromosomas para reproducirse.
  • Cruzamiento (Crossover): Combinación de dos cromosomas para producir descendencia.
  • Mutación: Alteración aleatoria de uno o más genes en un cromosoma.
  • Generación: Iteración del algoritmo que produce una nueva población.

Funcionamiento de los Algoritmos Genéticos

  1. Inicialización: Crear una población inicial de cromosomas aleatorios.
  2. Evaluación: Calcular el fitness de cada cromosoma en la población.
  3. Selección: Elegir cromosomas para reproducirse basándose en su fitness.
  4. Cruzamiento: Combinar pares de cromosomas seleccionados para crear nuevos cromosomas.
  5. Mutación: Aplicar mutaciones aleatorias a algunos cromosomas.
  6. Reemplazo: Formar una nueva población reemplazando la antigua con los nuevos cromosomas.
  7. Iteración: Repetir los pasos 2-6 hasta que se cumpla un criterio de parada (por ejemplo, un número fijo de generaciones o una solución óptima).

Ejemplo de Código en Python

A continuación, se presenta un ejemplo básico de un algoritmo genético en Python para maximizar una función simple.

import random

# Función de fitness
def fitness_function(x):
    return x**2

# Crear una población inicial
def create_population(size, lower_bound, upper_bound):
    return [random.uniform(lower_bound, upper_bound) for _ in range(size)]

# Selección por torneo
def selection(population, fitnesses):
    selected = []
    for _ in range(len(population)):
        i, j = random.sample(range(len(population)), 2)
        selected.append(population[i] if fitnesses[i] > fitnesses[j] else population[j])
    return selected

# Cruzamiento de un punto
def crossover(parent1, parent2):
    point = random.randint(1, len(parent1) - 1)
    return parent1[:point] + parent2[point:]

# Mutación
def mutate(chromosome, mutation_rate, lower_bound, upper_bound):
    if random.random() < mutation_rate:
        return random.uniform(lower_bound, upper_bound)
    return chromosome

# Algoritmo genético
def genetic_algorithm(pop_size, lower_bound, upper_bound, generations, mutation_rate):
    population = create_population(pop_size, lower_bound, upper_bound)
    for generation in range(generations):
        fitnesses = [fitness_function(ind) for ind in population]
        population = selection(population, fitnesses)
        next_population = []
        for i in range(0, len(population), 2):
            parent1, parent2 = population[i], population[i+1]
            child1, child2 = crossover(parent1, parent2), crossover(parent2, parent1)
            next_population.extend([mutate(child1, mutation_rate, lower_bound, upper_bound),
                                    mutate(child2, mutation_rate, lower_bound, upper_bound)])
        population = next_population
    best_solution = max(population, key=fitness_function)
    return best_solution

# Parámetros
pop_size = 10
lower_bound = -10
upper_bound = 10
generations = 20
mutation_rate = 0.1

# Ejecutar el algoritmo genético
best_solution = genetic_algorithm(pop_size, lower_bound, upper_bound, generations, mutation_rate)
print(f"La mejor solución encontrada es: {best_solution} con un fitness de {fitness_function(best_solution)}")

Conclusión

Los algoritmos genéticos son una poderosa herramienta de optimización que puede aplicarse a una amplia variedad de problemas. Su capacidad para explorar grandes espacios de búsqueda y encontrar soluciones óptimas o cercanas a lo óptimo los hace especialmente útiles en el campo de la inteligencia artificial. A través de la selección, el cruzamiento y la mutación, los algoritmos genéticos imitan el proceso de evolución natural para mejorar iterativamente las soluciones a un problema dado.
