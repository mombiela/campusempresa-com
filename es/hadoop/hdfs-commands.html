Introducción a HDFS

Hadoop Distributed File System (HDFS) es el sistema de almacenamiento principal utilizado por las aplicaciones de Hadoop. Proporciona acceso de alto rendimiento a los datos de la aplicación y está diseñado para ser altamente tolerante a fallos. Entender los comandos de HDFS es crucial para gestionar e interactuar con el sistema de archivos de Hadoop.

Comandos Básicos de HDFS

Estos comandos son esenciales para que los principiantes comiencen con HDFS.

Listar Archivos y Directorios

  • hdfs dfs -ls [ruta]: Lista archivos y directorios en la ruta especificada.
hdfs dfs -ls /user/hadoop

Este comando lista todos los archivos y directorios bajo /user/hadoop.

Crear Directorios

  • hdfs dfs -mkdir [ruta]: Crea un nuevo directorio en la ruta especificada.
hdfs dfs -mkdir /user/hadoop/newdir

Este comando crea un nuevo directorio llamado newdir bajo /user/hadoop.

Copiar Archivos a HDFS

  • hdfs dfs -put [ruta_local] [ruta_hdfs]: Copia archivos desde el sistema de archivos local a HDFS.
hdfs dfs -put /local/path/to/file.txt /user/hadoop/

Este comando copia file.txt desde el sistema de archivos local al directorio HDFS /user/hadoop/.

Ver Contenido de Archivos

  • hdfs dfs -cat [ruta]: Muestra el contenido de un archivo.
hdfs dfs -cat /user/hadoop/file.txt

Este comando muestra el contenido de file.txt ubicado en /user/hadoop/.

Eliminar Archivos y Directorios

  • hdfs dfs -rm [ruta]: Elimina un archivo.
hdfs dfs -rm /user/hadoop/file.txt

Este comando elimina file.txt de /user/hadoop/.

  • hdfs dfs -rm -r [ruta]: Elimina un directorio y su contenido de forma recursiva.
hdfs dfs -rm -r /user/hadoop/newdir

Este comando elimina el directorio newdir y todo su contenido de /user/hadoop/.

Comandos Intermedios de HDFS

Estos comandos proporcionan más control y son útiles para gestionar HDFS de manera más efectiva.

Mover y Renombrar Archivos

  • hdfs dfs -mv [ruta_origen] [ruta_destino]: Mueve o renombra archivos y directorios.
hdfs dfs -mv /user/hadoop/file.txt /user/hadoop/renamed_file.txt

Este comando renombra file.txt a renamed_file.txt.

Copiar Archivos dentro de HDFS

  • hdfs dfs -cp [ruta_origen] [ruta_destino]: Copia archivos de una ubicación de HDFS a otra.
hdfs dfs -cp /user/hadoop/file.txt /user/hadoop/copy_of_file.txt

Este comando copia file.txt a copy_of_file.txt dentro del mismo directorio.

Verificar el Estado de un Archivo

  • hdfs dfs -stat [formato] [ruta]: Muestra el estado de un archivo o directorio.
hdfs dfs -stat %b /user/hadoop/file.txt

Este comando muestra el tamaño en bytes de file.txt.

Cambiar Permisos de Archivos

  • hdfs dfs -chmod [permisos] [ruta]: Cambia los permisos de un archivo o directorio.
hdfs dfs -chmod 755 /user/hadoop/file.txt

Este comando establece los permisos de file.txt a 755.

Comandos Avanzados de HDFS

Estos comandos son para usuarios avanzados que necesitan realizar operaciones complejas y optimizaciones.

Verificar el Uso del Disco

  • hdfs dfs -du [ruta]: Muestra el uso del disco de archivos y directorios.
hdfs dfs -du /user/hadoop

Este comando muestra el uso del disco de todos los archivos y directorios bajo /user/hadoop.

Balancear el Clúster de HDFS

  • hdfs balancer: Balancea los datos entre los DataNodes en el clúster de HDFS.
hdfs balancer

Este comando inicia el proceso de balanceo para asegurar una distribución uniforme de los datos entre los DataNodes.

Operaciones de Modo Seguro

  • hdfs dfsadmin -safemode [enter|leave|get]: Gestiona el estado de modo seguro de HDFS.
hdfs dfsadmin -safemode enter

Este comando pone a HDFS en modo seguro, donde no se permiten modificaciones.

Gestión de Instantáneas

  • hdfs dfs -createSnapshot [ruta] [nombreInstantánea]: Crea una instantánea de un directorio.
hdfs dfs -createSnapshot /user/hadoop mySnapshot

Este comando crea una instantánea llamada mySnapshot del directorio /user/hadoop.

  • hdfs dfs -deleteSnapshot [ruta] [nombreInstantánea]: Elimina una instantánea.
hdfs dfs -deleteSnapshot /user/hadoop mySnapshot

Este comando elimina la instantánea llamada mySnapshot.

Conclusión

Entender y usar efectivamente los comandos de HDFS es crucial para gestionar datos dentro de un ecosistema Hadoop. Desde operaciones básicas de archivos hasta la gestión avanzada del clúster, estos comandos proporcionan las herramientas necesarias para interactuar con HDFS de manera eficiente. Practica estos comandos regularmente para volverte competente en el manejo de HDFS y aprovechar todo su potencial.
