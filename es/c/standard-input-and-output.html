Introducción

La Entrada y Salida (I/O) estándar son conceptos fundamentales en la programación en C. Permiten que los programas interactúen con los usuarios y otros programas leyendo entradas y escribiendo salidas. Esta sección cubrirá los conceptos básicos de la I/O estándar, incluyendo funciones como printf, scanf, getchar y putchar.

Salida Básica con `printf`

La función printf se usa para mostrar texto en la consola. Es parte de la biblioteca de I/O estándar, que se incluye añadiendo #include <stdio.h> al principio de tu programa.

Sintaxis

#include <stdio.h>

int main() {
    printf("¡Hola, Mundo!\n");
    return 0;
}
  • #include <stdio.h>: Incluye la biblioteca de I/O estándar.
  • printf("¡Hola, Mundo!\n");: Imprime "¡Hola, Mundo!" seguido de un carácter de nueva línea.

Especificadores de Formato

printf usa especificadores de formato para imprimir diferentes tipos de datos:

  • %d: Entero
  • %f: Número de punto flotante
  • %c: Carácter
  • %s: Cadena

Ejemplo

#include <stdio.h>

int main() {
    int edad = 25;
    float altura = 5.9;
    char inicial = 'A';
    char nombre[] = "Alicia";

    printf("Nombre: %s\n", nombre);
    printf("Inicial: %c\n", inicial);
    printf("Edad: %d\n", edad);
    printf("Altura: %.1f\n", altura);

    return 0;
}

Entrada Básica con `scanf`

La función scanf se usa para leer entradas formateadas desde la entrada estándar (usualmente el teclado).

Sintaxis

#include <stdio.h>

int main() {
    int edad;
    printf("Introduce tu edad: ");
    scanf("%d", &edad);
    printf("Tu edad es %d\n", edad);
    return 0;
}
  • scanf("%d", &edad);: Lee un entero del usuario y lo almacena en la variable edad.

Especificadores de Formato

scanf usa los mismos especificadores de formato que printf para leer diferentes tipos de datos:

  • %d: Entero
  • %f: Número de punto flotante
  • %c: Carácter
  • %s: Cadena

Ejemplo

#include <stdio.h>

int main() {
    int edad;
    float altura;
    char inicial;
    char nombre[50];

    printf("Introduce tu nombre: ");
    scanf("%s", nombre);
    printf("Introduce tu inicial: ");
    scanf(" %c", &inicial); // Nota el espacio antes de %c para consumir cualquier carácter de nueva línea sobrante
    printf("Introduce tu edad: ");
    scanf("%d", &edad);
    printf("Introduce tu altura: ");
    scanf("%f", &altura);

    printf("Nombre: %s\n", nombre);
    printf("Inicial: %c\n", inicial);
    printf("Edad: %d\n", edad);
    printf("Altura: %.2f\n", altura);

    return 0;
}

Entrada y Salida de Caracteres

Para la entrada y salida de un solo carácter, se usan las funciones getchar y putchar.

Ejemplo de `getchar`

#include <stdio.h>

int main() {
    char ch;
    printf("Introduce un carácter: ");
    ch = getchar();
    printf("Has introducido: %c\n", ch);
    return 0;
}

Ejemplo de `putchar`

#include <stdio.h>

int main() {
    char ch = 'A';
    printf("El carácter es: ");
    putchar(ch);
    putchar('\n');
    return 0;
}

Funciones Avanzadas de I/O

Para operaciones de I/O más avanzadas, C proporciona funciones como fgets y fputs para la entrada y salida de cadenas.

Ejemplo de `fgets`

#include <stdio.h>

int main() {
    char str[50];
    printf("Introduce una cadena: ");
    fgets(str, sizeof(str), stdin);
    printf("Has introducido: %s", str);
    return 0;
}

Ejemplo de `fputs`

#include <stdio.h>

int main() {
    char str[] = "¡Hola, Mundo!";
    printf("La cadena es: ");
    fputs(str, stdout);
    putchar('\n');
    return 0;
}

Resumen

  • printf: Usado para salida formateada.
  • scanf: Usado para entrada formateada.
  • getchar: Lee un solo carácter de la entrada estándar.
  • putchar: Escribe un solo carácter en la salida estándar.
  • fgets: Lee una cadena de la entrada estándar.
  • fputs: Escribe una cadena en la salida estándar.

Entender estas funciones básicas de I/O es crucial para interactuar con usuarios y otros programas. Practica usando estas funciones para volverte competente en el manejo de entrada y salida en C.
