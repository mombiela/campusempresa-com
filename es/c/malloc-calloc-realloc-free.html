La gestión dinámica de memoria es un aspecto crucial de la programación en C. Te permite asignar memoria en tiempo de ejecución, lo cual es esencial para crear programas flexibles y eficientes. En esta sección, cubriremos las siguientes funciones: malloc, calloc, realloc y free.

malloc

malloc significa "asignación de memoria". Asigna un número especificado de bytes y devuelve un puntero al primer byte de la memoria asignada. La memoria no está inicializada, lo que significa que contiene valores basura.

Sintaxis

void* malloc(size_t size);

Ejemplo

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    int n = 5;

    // Asignando memoria para 5 enteros
    ptr = (int*) malloc(n * sizeof(int));

    if (ptr == NULL) {
        printf("Fallo en la asignación de memoria\n");
        return 1;
    }

    // Inicializando y mostrando la memoria asignada
    for (int i = 0; i < n; i++) {
        ptr[i] = i + 1;
        printf("%d ", ptr[i]);
    }

    // Liberando la memoria asignada
    free(ptr);

    return 0;
}

calloc

calloc significa "asignación contigua". Asigna memoria para un array de elementos, inicializa la memoria a cero y luego devuelve un puntero a la memoria.

Sintaxis

void* calloc(size_t num, size_t size);

Ejemplo

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    int n = 5;

    // Asignando memoria para 5 enteros e inicializando a cero
    ptr = (int*) calloc(n, sizeof(int));

    if (ptr == NULL) {
        printf("Fallo en la asignación de memoria\n");
        return 1;
    }

    // Mostrando la memoria asignada
    for (int i = 0; i < n; i++) {
        printf("%d ", ptr[i]);
    }

    // Liberando la memoria asignada
    free(ptr);

    return 0;
}

realloc

realloc significa "reasignación". Cambia el tamaño de la memoria previamente asignada. Si el nuevo tamaño es mayor, la nueva memoria no está inicializada. Si el nuevo tamaño es menor, la memoria excedente se libera.

Sintaxis

void* realloc(void* ptr, size_t size);

Ejemplo

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    int n = 5;

    // Asignando memoria para 5 enteros
    ptr = (int*) malloc(n * sizeof(int));

    if (ptr == NULL) {
        printf("Fallo en la asignación de memoria\n");
        return 1;
    }

    // Inicializando la memoria asignada
    for (int i = 0; i < n; i++) {
        ptr[i] = i + 1;
    }

    // Reasignando memoria para 10 enteros
    ptr = (int*) realloc(ptr, 10 * sizeof(int));

    if (ptr == NULL) {
        printf("Fallo en la reasignación de memoria\n");
        return 1;
    }

    // Inicializando la nueva memoria
    for (int i = n; i < 10; i++) {
        ptr[i] = i + 1;
    }

    // Mostrando la memoria reasignada
    for (int i = 0; i < 10; i++) {
        printf("%d ", ptr[i]);
    }

    // Liberando la memoria asignada
    free(ptr);

    return 0;
}

free

free desasigna la memoria previamente asignada por malloc, calloc o realloc. No cambia el valor del puntero en sí, pero la memoria a la que apunta se libera.

Sintaxis

void free(void* ptr);

Ejemplo

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    int n = 5;

    // Asignando memoria para 5 enteros
    ptr = (int*) malloc(n * sizeof(int));

    if (ptr == NULL) {
        printf("Fallo en la asignación de memoria\n");
        return 1;
    }

    // Inicializando y mostrando la memoria asignada
    for (int i = 0; i < n; i++) {
        ptr[i] = i + 1;
        printf("%d ", ptr[i]);
    }

    // Liberando la memoria asignada
    free(ptr);

    return 0;
}

Resumen

  • malloc: Asigna memoria no inicializada.
  • calloc: Asigna memoria inicializada a cero.
  • realloc: Cambia el tamaño de la memoria previamente asignada.
  • free: Desasigna la memoria previamente asignada.

Entender estas funciones es esencial para una gestión efectiva de la memoria en C. El uso adecuado de la asignación dinámica de memoria puede llevar a programas más flexibles y eficientes, pero también requiere un manejo cuidadoso para evitar fugas de memoria y otros problemas.
