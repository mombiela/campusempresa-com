Introducción

En la programación en C, manejar archivos es una habilidad crucial para leer datos de archivos y escribir datos en archivos. Este tema cubrirá los conceptos básicos de las operaciones con archivos, incluyendo abrir, leer, escribir y cerrar archivos. Al final de esta sección, serás capaz de realizar operaciones básicas de E/S de archivos en C.

Operaciones con Archivos en C

Conceptos Clave

  • Puntero de Archivo: Un puntero de tipo FILE utilizado para realizar un seguimiento del archivo que se está accediendo.
  • Abrir un Archivo: Usar fopen() para abrir un archivo.
  • Cerrar un Archivo: Usar fclose() para cerrar un archivo.
  • Leer de un Archivo: Usar funciones como fgetc(), fgets() y fread().
  • Escribir en un Archivo: Usar funciones como fputc(), fputs() y fwrite().

Abrir y Cerrar Archivos

Para trabajar con archivos en C, primero necesitas abrirlos. Esto se hace usando la función fopen(), que devuelve un puntero de archivo. Después de realizar las operaciones necesarias, debes cerrar el archivo usando fclose().

Sintaxis

FILE *fopen(const char *filename, const char *mode);
int fclose(FILE *stream);

Modos para fopen()

| Modo | Descripción | |------|-------------| | "r" | Abrir para lectura. El archivo debe existir. | | "w" | Abrir para escritura. Crea un nuevo archivo si no existe o trunca el archivo si existe. | | "a" | Abrir para agregar. Los datos se añaden al final del archivo. | | "r+" | Abrir para lectura y escritura. El archivo debe existir. | | "w+" | Abrir para lectura y escritura. Crea un nuevo archivo si no existe o trunca el archivo si existe. | | "a+" | Abrir para lectura y agregar. Crea un nuevo archivo si no existe. |

Ejemplo

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("¡Error al abrir el archivo!\n");
        return 1;
    }
    // Realizar operaciones con el archivo
    fclose(file);
    return 0;
}

Leer de un Archivo

Existen varias funciones para leer datos de un archivo, cada una adecuada para diferentes tipos de datos y casos de uso.

fgetc()

Lee un solo carácter del archivo.

int fgetc(FILE *stream);

fgets()

Lee una cadena del archivo.

char *fgets(char *str, int n, FILE *stream);

fread()

Lee un bloque de datos del archivo.

size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream);

Ejemplo

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "r");
    if (file == NULL) {
        printf("¡Error al abrir el archivo!\n");
        return 1;
    }
    
    char ch;
    while ((ch = fgetc(file)) != EOF) {
        putchar(ch);
    }
    
    fclose(file);
    return 0;
}

Escribir en un Archivo

Similar a la lectura, existen varias funciones para escribir datos en un archivo.

fputc()

Escribe un solo carácter en el archivo.

int fputc(int char, FILE *stream);

fputs()

Escribe una cadena en el archivo.

int fputs(const char *str, FILE *stream);

fwrite()

Escribe un bloque de datos en el archivo.

size_t fwrite(const void *ptr, size_t size, size_t nmemb, FILE *stream);

Ejemplo

#include <stdio.h>

int main() {
    FILE *file = fopen("example.txt", "w");
    if (file == NULL) {
        printf("¡Error al abrir el archivo!\n");
        return 1;
    }
    
    fputs("¡Hola, Mundo!\n", file);
    
    fclose(file);
    return 0;
}

Conclusión

Entender cómo leer y escribir en archivos en C es esencial para muchas tareas de programación. Este tema cubrió los conceptos básicos de las operaciones con archivos, incluyendo abrir, leer, escribir y cerrar archivos. Practicando estos conceptos, serás capaz de manejar operaciones de E/S de archivos de manera efectiva en tus programas en C.
