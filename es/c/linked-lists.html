Introducción a las listas enlazadas

Las listas enlazadas son una estructura de datos fundamental en la informática, proporcionando una forma de almacenar una colección de elementos. A diferencia de los arreglos, las listas enlazadas no requieren ubicaciones de memoria contiguas, lo que las hace flexibles y eficientes para ciertas operaciones.

Conceptos clave

  • Nodo: La unidad básica de una lista enlazada, que contiene datos y una referencia (o puntero) al siguiente nodo.
  • Cabeza: El primer nodo en una lista enlazada.
  • Cola: El último nodo en una lista enlazada, que apunta a NULL.
  • NULL: Un valor especial que indica el final de la lista.

Tipos de listas enlazadas

Lista enlazada simple

Una lista enlazada simple es el tipo más simple de lista enlazada, donde cada nodo apunta al siguiente nodo en la secuencia.

Lista doblemente enlazada

Una lista doblemente enlazada permite la travesía en ambas direcciones, ya que cada nodo contiene punteros tanto al siguiente como al nodo anterior.

Lista enlazada circular

En una lista enlazada circular, el último nodo apunta de nuevo al primer nodo, formando una estructura circular.

Implementación de una lista enlazada simple en C

Definición de la estructura del nodo

#include <stdio.h>
#include <stdlib.h>

// Define la estructura del nodo
struct Node {
    int data;
    struct Node* next;
};
  • data: Almacena el valor del nodo.
  • next: Puntero al siguiente nodo en la lista.

Creación de un nuevo nodo

struct Node* createNode(int data) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = data;
    newNode->next = NULL;
    return newNode;
}
  • malloc: Asigna memoria para el nuevo nodo.
  • data: Inicializa los datos del nodo.
  • next: Establece el puntero siguiente a NULL.

Insertar un nodo al principio

void insertAtBeginning(struct Node** head, int data) {
    struct Node* newNode = createNode(data);
    newNode->next = *head;
    *head = newNode;
}
  • head: Doble puntero al nodo cabeza para permitir la modificación.
  • newNode->next: Apunta a la cabeza actual.
  • *head: Actualiza la cabeza al nuevo nodo.

Recorrer la lista

void traverseList(struct Node* head) {
    struct Node* current = head;
    while (current != NULL) {
        printf("%d -> ", current->data);
        current = current->next;
    }
    printf("NULL\n");
}
  • current: Puntero utilizado para recorrer la lista.
  • printf: Imprime los datos de cada nodo seguidos de una flecha.

Eliminar un nodo

void deleteNode(struct Node** head, int key) {
    struct Node* temp = *head;
    struct Node* prev = NULL;

    // Si el nodo cabeza contiene la clave
    if (temp != NULL && temp->data == key) {
        *head = temp->next;
        free(temp);
        return;
    }

    // Buscar la clave
    while (temp != NULL && temp->data != key) {
        prev = temp;
        temp = temp->next;
    }

    // Si la clave no estaba presente en la lista
    if (temp == NULL) return;

    // Desvincular el nodo y liberar memoria
    prev->next = temp->next;
    free(temp);
}
  • temp: Puntero temporal para recorrer la lista.
  • prev: Puntero al nodo anterior para actualizar el enlace.
  • free: Desasigna la memoria del nodo eliminado.

Operaciones avanzadas en listas enlazadas

Invertir una lista enlazada

void reverseList(struct Node** head) {
    struct Node* prev = NULL;
    struct Node* current = *head;
    struct Node* next = NULL;

    while (current != NULL) {
        next = current->next;
        current->next = prev;
        prev = current;
        current = next;
    }
    *head = prev;
}
  • prev: Rastrea el nodo anterior.
  • current: Rastrea el nodo actual.
  • next: Almacena temporalmente el siguiente nodo.

Detectar un ciclo en una lista enlazada

int detectCycle(struct Node* head) {
    struct Node* slow = head;
    struct Node* fast = head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;

        if (slow == fast) {
            return 1; // Ciclo detectado
        }
    }
    return 0; // No hay ciclo
}
  • slow: Se mueve un paso a la vez.
  • fast: Se mueve dos pasos a la vez.
  • Detección de ciclo: Si slow y fast se encuentran, existe un ciclo.

Conclusión

Las listas enlazadas son una estructura de datos versátil que puede usarse para gestionar eficientemente colecciones de datos. Comprender las operaciones básicas, como la inserción, eliminación, recorrido y operaciones avanzadas como invertir y detectar ciclos, es crucial para aprovechar las listas enlazadas en diversas aplicaciones. Al dominar estos conceptos, los programadores pueden mejorar sus habilidades para resolver problemas y optimizar su código para un mejor rendimiento.
