Introducción a las Cadenas en C

En C, las cadenas son arreglos de caracteres terminados por un carácter nulo ('\0'). Entender cómo manipular cadenas es crucial para muchas tareas de programación, desde el procesamiento simple de texto hasta el análisis complejo de datos.

Conceptos Clave

  • Declaración de Cadenas: Las cadenas se declaran como arreglos de caracteres.
  • Terminador Nulo: El final de una cadena está marcado por un carácter nulo ('\0').
  • Inicialización de Cadenas: Las cadenas pueden ser inicializadas en el momento de la declaración.

Ejemplo

char str1[] = "¡Hola, Mundo!";
char str2[20] = "¡Hola, C!";
char str3[20]; // Cadena no inicializada

Funciones Estándar de Cadenas

La Biblioteca Estándar de C proporciona un conjunto de funciones para la manipulación de cadenas, declaradas en el archivo de cabecera <string.h>.

Funciones de Cadenas Comúnmente Usadas

  • strlen(): Devuelve la longitud de una cadena.
  • strcpy(): Copia una cadena a otra.
  • strncpy(): Copia un número especificado de caracteres de una cadena a otra.
  • strcat(): Concatenar dos cadenas.
  • strncat(): Concatenar un número especificado de caracteres de una cadena a otra.
  • strcmp(): Compara dos cadenas.
  • strncmp(): Compara un número especificado de caracteres de dos cadenas.
  • strchr(): Encuentra la primera aparición de un carácter en una cadena.
  • strstr(): Encuentra la primera aparición de una subcadena en una cadena.

Ejemplo de Uso

#include <stdio.h>
#include <string.h>

int main() {
    char str1[20] = "Hola";
    char str2[20] = "Mundo";
    char str3[40];

    // Longitud de la Cadena
    printf("Longitud de str1: %lu\n", strlen(str1));

    // Copia de Cadenas
    strcpy(str3, str1);
    printf("str3 después de strcpy: %s\n", str3);

    // Concatenación de Cadenas
    strcat(str3, " ");
    strcat(str3, str2);
    printf("str3 después de strcat: %s\n", str3);

    // Comparación de Cadenas
    if (strcmp(str1, str2) == 0) {
        printf("str1 y str2 son iguales\n");
    } else {
        printf("str1 y str2 no son iguales\n");
    }

    return 0;
}

Funciones Avanzadas de Cadenas

Para manipulaciones de cadenas más complejas, C proporciona funciones adicionales.

Funciones de Gestión de Memoria

  • strdup(): Duplica una cadena asignando memoria dinámicamente.
  • strtok(): Tokeniza una cadena basada en delimitadores.

Ejemplo de Uso

#include <stdio.h>
#include <string.h>
#include <stdlib.h>

int main() {
    char str[] = "¡Hola, Mundo!";
    char *token;
    char *dup_str;

    // Duplicación de Cadenas
    dup_str = strdup(str);
    if (dup_str != NULL) {
        printf("Cadena duplicada: %s\n", dup_str);
        free(dup_str); // Liberar la memoria asignada
    }

    // Tokenización de Cadenas
    token = strtok(str, ", ");
    while (token != NULL) {
        printf("Token: %s\n", token);
        token = strtok(NULL, ", ");
    }

    return 0;
}

Resumen

Las funciones de cadenas en C son herramientas esenciales para manejar y manipular texto. Desde operaciones básicas como copiar y concatenar hasta tareas más avanzadas como la tokenización y la gestión dinámica de memoria, dominar estas funciones mejorará significativamente tus capacidades de programación. Recuerda siempre incluir el archivo de cabecera <string.h> y gestionar la memoria con cuidado cuando trabajes con cadenas dinámicas.
