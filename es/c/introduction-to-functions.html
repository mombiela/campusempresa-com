Prototipos y Definiciones de Funciones ►

¿Qué es una Función?

En la programación en C, una función es un bloque de código que realiza una tarea específica. Las funciones ayudan a organizar el código, haciéndolo más legible, reutilizable y más fácil de depurar.

Conceptos Clave

  • Declaración de Función (Prototipo): Especifica el nombre de la función, el tipo de retorno y los parámetros.
  • Definición de Función: Contiene el cuerpo real de la función.
  • Llamada de Función: Ejecuta la función.

¿Por qué Usar Funciones?

  • Modularidad: Descompone problemas complejos en tareas más pequeñas y manejables.
  • Reutilización: Las funciones pueden ser reutilizadas en diferentes partes del programa.
  • Mantenibilidad: Más fácil de actualizar y mantener el código.

Sintaxis de Función

Declaración de Función

Una función debe ser declarada antes de ser usada. Esto también se conoce como un prototipo de función.

return_type function_name(parameter_list);

Definición de Función

La definición de la función incluye el código real que se ejecutará cuando se llame a la función.

return_type function_name(parameter_list) {
    // cuerpo de la función
}

Llamada de Función

Para ejecutar una función, simplemente se llama por su nombre seguido de paréntesis.

function_name(arguments);

Ejemplo: Función Básica

Vamos a crear una función simple que sume dos enteros.

Declaración de Función

int add(int a, int b);

Definición de Función

int add(int a, int b) {
    return a + b;
}

Llamada de Función

#include <stdio.h>

// Declaración de función
int add(int a, int b);

int main() {
    int result;
    result = add(5, 3); // Llamada de función
    printf("La suma es: %d\n", result);
    return 0;
}

// Definición de función
int add(int a, int b) {
    return a + b;
}

Tipos de Funciones

Funciones Integradas

C proporciona muchas funciones integradas como printf(), scanf(), strlen(), etc.

Funciones Definidas por el Usuario

Estas son funciones que creas para realizar tareas específicas.

Tipos de Retorno y Parámetros

Tipos de Retorno

Una función puede devolver un valor de cualquier tipo de dato. Si una función no devuelve un valor, su tipo de retorno es void.

Parámetros

Las funciones pueden tomar cero o más parámetros. Los parámetros permiten pasar datos a la función.

Ejemplo: Función sin Parámetros y sin Valor de Retorno

#include <stdio.h>

void greet() {
    printf("¡Hola, Mundo!\n");
}

int main() {
    greet(); // Llamada de función
    return 0;
}

Alcance y Vida Útil de las Variables

Variables Locales

Las variables declaradas dentro de una función son locales a esa función y no pueden ser accedidas fuera de ella.

Variables Globales

Las variables declaradas fuera de todas las funciones son globales y pueden ser accedidas por cualquier función en el programa.

Ejemplo: Variables Locales vs Globales

#include <stdio.h>

int globalVar = 10; // Variable global

void display() {
    int localVar = 20; // Variable local
    printf("Variable global: %d\n", globalVar);
    printf("Variable local: %d\n", localVar);
}

int main() {
    display();
    return 0;
}

Recursión

Una función que se llama a sí misma se conoce como una función recursiva. La recursión puede ser una herramienta poderosa para resolver problemas que pueden descomponerse en problemas más pequeños y similares.

Ejemplo: Factorial Usando Recursión

#include <stdio.h>

int factorial(int n) {
    if (n == 0) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int number = 5;
    printf("El factorial de %d es %d\n", number, factorial(number));
    return 0;
}

Resumen

  • Las funciones son esenciales para escribir código modular, reutilizable y mantenible.
  • Una función consta de una declaración, definición y llamada.
  • Las funciones pueden tener diferentes tipos de retorno y parámetros.
  • Entender el alcance y la vida útil de las variables es crucial al trabajar con funciones.
  • La recursión es una técnica donde una función se llama a sí misma para resolver un problema.

Al dominar las funciones, podrás escribir programas en C más eficientes y organizados.
