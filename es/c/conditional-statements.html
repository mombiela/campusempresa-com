Las sentencias condicionales son fundamentales en la programación ya que permiten la ejecución de código basado en ciertas condiciones. En C, las sentencias condicionales ayudan a controlar el flujo del programa, haciendo posible ejecutar diferentes bloques de código basados en criterios específicos.

Conceptos Clave

  • sentencia if: Ejecuta un bloque de código si una condición especificada es verdadera.
  • sentencia else: Ejecuta un bloque de código si la misma condición es falsa.
  • sentencia else if: Especifica una nueva condición para probar si la primera condición es falsa.
  • sentencia switch: Selecciona uno de muchos bloques de código para ejecutar basado en el valor de una expresión.

Sentencia if

La sentencia if es la forma más simple de sentencia condicional. Evalúa una condición y ejecuta el siguiente bloque de código si la condición es verdadera.

Sintaxis

if (condition) {
    // Código a ejecutar si la condición es verdadera
}

Ejemplo

#include <stdio.h>

int main() {
    int number = 10;

    if (number > 0) {
        printf("El número es positivo.\n");
    }

    return 0;
}

En este ejemplo, el programa verifica si number es mayor que 0. Dado que number es 10, la condición es verdadera y se imprime el mensaje "El número es positivo.".

Sentencia else

La sentencia else sigue a una sentencia if y ejecuta un bloque de código si la condición en la sentencia if es falsa.

Sintaxis

if (condition) {
    // Código a ejecutar si la condición es verdadera
} else {
    // Código a ejecutar si la condición es falsa
}

Ejemplo

#include <stdio.h>

int main() {
    int number = -5;

    if (number > 0) {
        printf("El número es positivo.\n");
    } else {
        printf("El número no es positivo.\n");
    }

    return 0;
}

Aquí, el programa verifica si number es mayor que 0. Dado que number es -5, la condición es falsa y se imprime el mensaje "El número no es positivo.".

Sentencia else if

La sentencia else if te permite verificar múltiples condiciones. Si la primera condición es falsa, verifica la siguiente condición, y así sucesivamente.

Sintaxis

if (condition1) {
    // Código a ejecutar si condition1 es verdadera
} else if (condition2) {
    // Código a ejecutar si condition1 es falsa y condition2 es verdadera
} else {
    // Código a ejecutar si tanto condition1 como condition2 son falsas
}

Ejemplo

#include <stdio.h>

int main() {
    int number = 0;

    if (number > 0) {
        printf("El número es positivo.\n");
    } else if (number < 0) {
        printf("El número es negativo.\n");
    } else {
        printf("El número es cero.\n");
    }

    return 0;
}

En este ejemplo, el programa verifica si number es mayor que 0, menor que 0, o igual a 0, e imprime el mensaje correspondiente.

Sentencia switch

La sentencia switch se usa para seleccionar uno de muchos bloques de código para ejecutar. A menudo se usa cuando tienes múltiples condiciones que dependen del valor de una sola variable.

Sintaxis

switch (expression) {
    case value1:
        // Código a ejecutar si expression es igual a value1
        break;
    case value2:
        // Código a ejecutar si expression es igual a value2
        break;
    // Puedes tener cualquier número de sentencias case
    default:
        // Código a ejecutar si expression no coincide con ningún case
}

Ejemplo

#include <stdio.h>

int main() {
    int day = 3;

    switch (day) {
        case 1:
            printf("Lunes\n");
            break;
        case 2:
            printf("Martes\n");
            break;
        case 3:
            printf("Miércoles\n");
            break;
        case 4:
            printf("Jueves\n");
            break;
        case 5:
            printf("Viernes\n");
            break;
        case 6:
            printf("Sábado\n");
            break;
        case 7:
            printf("Domingo\n");
            break;
        default:
            printf("Día inválido\n");
    }

    return 0;
}

En este ejemplo, el programa usa una sentencia switch para imprimir el nombre del día basado en el valor de day. Dado que day es 3, imprime "Miércoles".

Resumen

Las sentencias condicionales en C son esenciales para controlar el flujo de un programa. Las sentencias if, else, y else if permiten la verificación de condiciones simples y complejas, mientras que la sentencia switch proporciona una forma limpia de manejar múltiples condiciones basadas en una sola variable. Entender y usar estas sentencias efectivamente te permitirá escribir programas más dinámicos y receptivos.
