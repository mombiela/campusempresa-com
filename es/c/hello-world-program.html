Introducción

El programa "Hola Mundo" es tradicionalmente el primer programa escrito por principiantes que están aprendiendo un nuevo lenguaje de programación. Sirve como una introducción simple a la sintaxis y estructura del lenguaje. En esta sección, cubriremos cómo escribir, compilar y ejecutar un programa "Hola Mundo" en C.

Escribiendo el Programa

Para escribir un programa "Hola Mundo" en C, necesitas entender la estructura básica de un programa en C. Aquí está el código para un simple programa "Hola Mundo":

#include <stdio.h>

int main() {
    printf("Hola, Mundo!\n");
    return 0;
}

Explicación

  • #include <stdio.h>: Esta línea incluye la biblioteca de Entrada y Salida Estándar, que es necesaria para usar la función printf.
  • int main() { ... }: Esta es la función principal donde comienza la ejecución del programa.
  • printf("Hola, Mundo!\n");: Esta línea imprime la cadena "Hola, Mundo!" en la consola. El \n es un carácter de nueva línea que mueve el cursor a la siguiente línea.
  • return 0;: Esta línea termina la función principal y devuelve 0 al sistema operativo, indicando que el programa terminó exitosamente.

Compilando el Programa

Para ejecutar un programa en C, primero necesitas compilarlo. Aquí están los pasos para compilar y ejecutar el programa "Hola Mundo":

  1. Guardar el Programa: Guarda el código en un archivo con una extensión .c, por ejemplo, hola.c.

  2. Abrir Terminal/Command Prompt: Navega al directorio donde guardaste el archivo.

  3. Compilar el Programa: Usa un compilador de C como gcc para compilar el programa. Ejecuta el siguiente comando:

    gcc hola.c -o hola

    Este comando compila hola.c y crea un archivo ejecutable llamado hola.

  4. Ejecutar el Programa: Ejecuta el programa compilado ejecutando:

    ./hola

Ejemplo

$ gcc hola.c -o hola
$ ./hola
Hola, Mundo!

Conceptos Clave

  • Directivas del Preprocesador: Las líneas que comienzan con # son directivas del preprocesador. #include <stdio.h> le dice al compilador que incluya la biblioteca de Entrada y Salida Estándar.
  • Función Principal: El punto de entrada de un programa en C. Cada programa en C debe tener una función main.
  • Salida Estándar: printf se usa para imprimir texto en la consola.
  • Sentencia de Retorno: return 0; indica que el programa se ha ejecutado exitosamente.

Errores Comunes

  • Falta de Punto y Coma: Cada declaración en C debe terminar con un punto y coma (;). Olvidar esto resultará en un error de compilación.
  • Sensibilidad a Mayúsculas y Minúsculas: C distingue entre mayúsculas y minúsculas. main es diferente de Main.
  • Extensión del Archivo: Asegúrate de que tu archivo esté guardado con una extensión .c.

Conclusión

El programa "Hola Mundo" es un punto de partida fundamental para aprender programación en C. Introduce la estructura básica de un programa en C, incluyendo directivas del preprocesador, la función principal y la salida estándar. Al entender y practicar este simple programa, sientas las bases para conceptos de programación en C más complejos.
