Introducción

Las pilas y colas son estructuras de datos fundamentales en la informática. Se utilizan para almacenar y gestionar datos en un orden particular. Comprender estas estructuras es crucial para resolver varios problemas computacionales de manera eficiente.

Pilas

Conceptos Clave

  • LIFO (Last In, First Out): El último elemento añadido a la pila es el primero en ser eliminado.
  • Push: Añadir un elemento a la cima de la pila.
  • Pop: Eliminar el elemento superior de la pila.
  • Peek/Top: Ver el elemento superior sin eliminarlo.

Operaciones de Pila en C

Definición de Estructura

#define MAX 100

typedef struct {
    int data[MAX];
    int top;
} Stack;

Inicializar Pila

void initialize(Stack *s) {
    s->top = -1;
}

Operación Push

void push(Stack *s, int value) {
    if (s->top == MAX - 1) {
        printf("Stack overflow\n");
        return;
    }
    s->data[++(s->top)] = value;
}

Operación Pop

int pop(Stack *s) {
    if (s->top == -1) {
        printf("Stack underflow\n");
        return -1; // Devolver un valor de error
    }
    return s->data[(s->top)--];
}

Operación Peek

int peek(Stack *s) {
    if (s->top == -1) {
        printf("Stack is empty\n");
        return -1; // Devolver un valor de error
    }
    return s->data[s->top];
}

Ejemplo de Uso

#include <stdio.h>

int main() {
    Stack s;
    initialize(&s);

    push(&s, 10);
    push(&s, 20);
    push(&s, 30);

    printf("Top element is %d\n", peek(&s));
    printf("Popped element is %d\n", pop(&s));
    printf("Top element is %d\n", peek(&s));

    return 0;
}

Colas

Conceptos Clave

  • FIFO (First In, First Out): El primer elemento añadido a la cola es el primero en ser eliminado.
  • Enqueue: Añadir un elemento al final de la cola.
  • Dequeue: Eliminar el elemento frontal de la cola.
  • Front: Ver el elemento frontal sin eliminarlo.
  • Rear: Ver el elemento trasero sin eliminarlo.

Operaciones de Cola en C

Definición de Estructura

#define MAX 100

typedef struct {
    int data[MAX];
    int front;
    int rear;
} Queue;

Inicializar Cola

void initialize(Queue *q) {
    q->front = -1;
    q->rear = -1;
}

Operación Enqueue

void enqueue(Queue *q, int value) {
    if (q->rear == MAX - 1) {
        printf("Queue overflow\n");
        return;
    }
    if (q->front == -1) q->front = 0;
    q->data[++(q->rear)] = value;
}

Operación Dequeue

int dequeue(Queue *q) {
    if (q->front == -1 || q->front > q->rear) {
        printf("Queue underflow\n");
        return -1; // Devolver un valor de error
    }
    return q->data[(q->front)++];
}

Operación Front

int front(Queue *q) {
    if (q->front == -1 || q->front > q->rear) {
        printf("Queue is empty\n");
        return -1; // Devolver un valor de error
    }
    return q->data[q->front];
}

Ejemplo de Uso

#include <stdio.h>

int main() {
    Queue q;
    initialize(&q);

    enqueue(&q, 10);
    enqueue(&q, 20);
    enqueue(&q, 30);

    printf("Front element is %d\n", front(&q));
    printf("Dequeued element is %d\n", dequeue(&q));
    printf("Front element is %d\n", front(&q));

    return 0;
}

Comparación de Pilas y Colas

| Característica | Pila (LIFO) | Cola (FIFO) | |----------------|---------------------|---------------------| | Inserción | Push (cima) | Enqueue (final) | | Eliminación | Pop (cima) | Dequeue (frontal) | | Acceso | Peek/Top (cima) | Front (frontal) |

Conclusión

Las pilas y colas son estructuras de datos esenciales que ayudan a gestionar datos en órdenes específicos. Las pilas siguen el principio LIFO, lo que las hace adecuadas para tareas como invertir datos y algoritmos de retroceso. Las colas siguen el principio FIFO, lo que las hace ideales para programar tareas y gestionar buffers. Comprender cómo implementar y usar estas estructuras en C es fundamental para una programación y resolución de problemas eficientes.
