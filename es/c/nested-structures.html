Introducción a las Estructuras Anidadas

Las estructuras anidadas en C te permiten crear tipos de datos complejos al incrustar una estructura dentro de otra. Esto es particularmente útil cuando necesitas representar datos jerárquicos o compuestos.

Conceptos Clave

  • Estructura: Un tipo de dato definido por el usuario que agrupa variables relacionadas.
  • Estructura Anidada: Una estructura que contiene otra estructura como miembro.

Definiendo Estructuras Anidadas

Para definir una estructura anidada, simplemente declaras una estructura dentro de otra. Aquí hay un ejemplo básico:

#include <stdio.h>

// Definir la estructura interna
struct Fecha {
    int dia;
    int mes;
    int año;
};

// Definir la estructura externa
struct Estudiante {
    char nombre[50];
    int edad;
    struct Fecha fechaNacimiento; // Estructura anidada
};

int main() {
    // Declarar e inicializar una variable Estudiante
    struct Estudiante estudiante1 = {"Juan Pérez", 20, {15, 8, 2000}};
    
    // Acceder a los miembros de la estructura anidada
    printf("Nombre: %s\n", estudiante1.nombre);
    printf("Edad: %d\n", estudiante1.edad);
    printf("Fecha de Nacimiento: %02d/%02d/%d\n", estudiante1.fechaNacimiento.dia, estudiante1.fechaNacimiento.mes, estudiante1.fechaNacimiento.año);
    
    return 0;
}

Explicación

  • Estructura Interna (Fecha): Contiene dia, mes y año.
  • Estructura Externa (Estudiante): Contiene nombre, edad y una estructura Fecha anidada llamada fechaNacimiento.
  • Inicialización: La estructura Estudiante se inicializa con valores, incluyendo la estructura Fecha anidada.
  • Acceso a Miembros: Usa el operador punto (.) para acceder a los miembros de ambas estructuras, la externa y la anidada.

Casos Prácticos

Las estructuras anidadas son útiles en varios escenarios, tales como:

  • Registros de Base de Datos: Representar un registro con múltiples campos, algunos de los cuales son en sí mismos tipos complejos.
  • Datos Jerárquicos: Modelar datos jerárquicos como organigramas o sistemas de archivos.

Ejemplo: Registros de Empleados

Considera un ejemplo donde necesitas almacenar información de empleados, incluyendo su dirección.

#include <stdio.h>

// Definir la estructura Dirección
struct Direccion {
    char calle[100];
    char ciudad[50];
    char estado[50];
    int codigoPostal;
};

// Definir la estructura Empleado
struct Empleado {
    char nombre[50];
    int id;
    struct Direccion direccion; // Estructura anidada
};

int main() {
    // Declarar e inicializar una variable Empleado
    struct Empleado emp1 = {"Alicia Pérez", 101, {"123 Calle Principal", "Springfield", "IL", 62701}};
    
    // Acceder a los miembros de la estructura anidada
    printf("Nombre: %s\n", emp1.nombre);
    printf("ID: %d\n", emp1.id);
    printf("Dirección: %s, %s, %s, %d\n", emp1.direccion.calle, emp1.direccion.ciudad, emp1.direccion.estado, emp1.direccion.codigoPostal);
    
    return 0;
}

Explicación

  • Estructura Interna (Direccion): Contiene calle, ciudad, estado y codigoPostal.
  • Estructura Externa (Empleado): Contiene nombre, id y una estructura Direccion anidada llamada direccion.
  • Inicialización y Acceso: Similar al ejemplo anterior, pero con datos más complejos.

Ejercicios

  1. Crea una estructura anidada para un Libro que incluya una estructura Autor.

    • Autor debe tener nombre y añoNacimiento.
    • Libro debe tener titulo, precio y un Autor.

  2. Escribe un programa para inicializar e imprimir los detalles de un Libro.

#include <stdio.h>

// Definir la estructura Autor
struct Autor {
    char nombre[50];
    int añoNacimiento;
};

// Definir la estructura Libro
struct Libro {
    char titulo[100];
    float precio;
    struct Autor autor; // Estructura anidada
};

int main() {
    // Declarar e inicializar una variable Libro
    struct Libro libro1 = {"El Gran Gatsby", 10.99, {"F. Scott Fitzgerald", 1896}};
    
    // Acceder a los miembros de la estructura anidada
    printf("Título: %s\n", libro1.titulo);
    printf("Precio: $%.2f\n", libro1.precio);
    printf("Autor: %s\n", libro1.autor.nombre);
    printf("Año de Nacimiento del Autor: %d\n", libro1.autor.añoNacimiento);
    
    return 0;
}

Resumen

Las estructuras anidadas en C proporcionan una forma poderosa de modelar datos complejos al incrustar una estructura dentro de otra. Esto permite un código más organizado y legible, especialmente cuando se trata de datos jerárquicos o compuestos. Al entender y utilizar estructuras anidadas, puedes crear programas más sofisticados y mantenibles.
