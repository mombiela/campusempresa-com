Configurar tu entorno de desarrollo es el primer paso crucial en el desarrollo de aplicaciones con DirectX. Esta sección te guiará a través de las herramientas y configuraciones necesarias para comenzar, asegurando que tengas una base sólida tanto para la programación de DirectX para principiantes como para avanzados.

Herramientas y Requisitos de Software

Antes de sumergirte en la programación de DirectX, necesitas instalar y configurar varias herramientas y paquetes de software. Aquí tienes una lista de lo que necesitarás:

  • Sistema Operativo Windows: DirectX es una colección de APIs para manejar tareas relacionadas con multimedia, especialmente programación de juegos y video, en plataformas de Microsoft.
  • Visual Studio: Un entorno de desarrollo integrado (IDE) de Microsoft. La edición Community de Visual Studio es gratuita y suficiente para nuestros propósitos.
  • DirectX SDK: Aunque el SDK de DirectX ahora es parte del SDK de Windows, algunas muestras y herramientas heredadas aún pueden requerir el SDK independiente de DirectX.

Instalación de Visual Studio

  1. Descargar Visual Studio: Visita la página de descarga de Visual Studio y descarga la edición Community.
  2. Instalar Visual Studio:
    • Ejecuta el instalador y selecciona la carga de trabajo "Desarrollo de escritorio con C++".
    • Asegúrate de que la opción "Desarrollo de juegos con C++" también esté seleccionada, ya que incluye herramientas y bibliotecas para el desarrollo de DirectX.
  3. Configurar Visual Studio:
    • Abre Visual Studio y ve a Herramientas > Opciones.
    • Navega a Proyectos y soluciones > Directorios de VC++ y asegúrate de que las rutas para archivos de inclusión, archivos de biblioteca y archivos ejecutables estén configuradas correctamente.

Configuración del SDK de DirectX

Aunque la mayoría de los componentes de DirectX están incluidos en el SDK de Windows, algunas herramientas y muestras heredadas pueden requerir el SDK independiente de DirectX.

  1. Descargar el SDK de DirectX: Puedes encontrar el SDK de DirectX (junio de 2010) en el sitio web de Microsoft.
  2. Instalar el SDK de DirectX:
    • Ejecuta el instalador y sigue las instrucciones en pantalla.
    • Nota la ruta de instalación, ya que necesitarás referenciarla en Visual Studio.

Configuración de Visual Studio para el Desarrollo con DirectX

  1. Incluir Rutas del SDK de DirectX:

    • Abre Visual Studio y ve a Herramientas > Opciones.
    • Navega a Proyectos y soluciones > Directorios de VC++.
    • Agrega las rutas de inclusión y biblioteca del SDK de DirectX a los directorios respectivos.

  2. Crear un Nuevo Proyecto:

    • Ve a Archivo > Nuevo > Proyecto.
    • Selecciona Visual C++ y luego Proyecto Win32.
    • Nombra tu proyecto y haz clic en OK.
    • En el Asistente de Aplicaciones Win32, selecciona Configuración de la Aplicación y elige Proyecto Vacío.

  3. Agregar Bibliotecas de DirectX:

    • Haz clic derecho en tu proyecto en el Explorador de Soluciones y selecciona Propiedades.
    • Ve a Propiedades de Configuración > Enlazador > Entrada.
    • Agrega las bibliotecas necesarias de DirectX (por ejemplo, d3d11.lib, d3dx11.lib, dxgi.lib) al campo Dependencias Adicionales.

Escribiendo tu Primer Programa en DirectX

Ahora que tu entorno está configurado, escribamos un programa simple en DirectX para asegurarnos de que todo esté funcionando correctamente.

#include <windows.h>
#include <d3d11.h>

// Enlazar bibliotecas necesarias
#pragma comment (lib, "d3d11.lib")

// Declaraciones globales
IDXGISwapChain *swapchain;             // el puntero a la interfaz de la cadena de intercambio
ID3D11Device *dev;                     // el puntero a nuestra interfaz de dispositivo Direct3D
ID3D11DeviceContext *devcon;           // el puntero a nuestro contexto de dispositivo Direct3D

// Prototipos de funciones
void InitD3D(HWND hWnd);    // configura e inicializa Direct3D
void CleanD3D(void);        // cierra Direct3D y libera memoria

// El punto de entrada para cualquier programa de Windows
int WINAPI WinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, LPSTR lpCmdLine, int nCmdShow)
{
    HWND hWnd;
    WNDCLASSEX wc;

    ZeroMemory(&wc, sizeof(WNDCLASSEX));

    wc.cbSize = sizeof(WNDCLASSEX);
    wc.style = CS_HREDRAW | CS_VREDRAW;
    wc.lpfnWndProc = WindowProc;
    wc.hInstance = hInstance;
    wc.hCursor = LoadCursor(NULL, IDC_ARROW);
    wc.lpszClassName = L"WindowClass";

    RegisterClassEx(&wc);

    hWnd = CreateWindowEx(NULL,
                          L"WindowClass",
                          L"Nuestro Primer Programa Direct3D",
                          WS_OVERLAPPEDWINDOW,
                          300, 300,
                          800, 600,
                          NULL,
                          NULL,
                          hInstance,
                          NULL);

    ShowWindow(hWnd, nCmdShow);

    // configurar e inicializar Direct3D
    InitD3D(hWnd);

    // entrar en el bucle principal:

    MSG msg;

    while(TRUE)
    {
        while(PeekMessage(&msg, NULL, 0, 0, PM_REMOVE))
        {
            TranslateMessage(&msg);
            DispatchMessage(&msg);
        }

        if(msg.message == WM_QUIT)
            break;
    }

    // limpiar Direct3D y COM
    CleanD3D();

    return msg.wParam;
}

// este es el manejador de mensajes principal para el programa
LRESULT CALLBACK WindowProc(HWND hWnd, UINT message, WPARAM wParam, LPARAM lParam)
{
    switch(message)
    {
        case WM_DESTROY:
            {
                PostQuitMessage(0);
                return 0;
            } break;
    }

    return DefWindowProc (hWnd, message, wParam, lParam);
}

// esta función inicializa y prepara Direct3D para su uso
void InitD3D(HWND hWnd)
{
    // crear una estructura para contener información sobre la cadena de intercambio
    DXGI_SWAP_CHAIN_DESC scd;

    // limpiar la estructura para su uso
    ZeroMemory(&scd, sizeof(DXGI_SWAP_CHAIN_DESC));

    // llenar la estructura de descripción de la cadena de intercambio
    scd.BufferCount = 1;                                    // un búfer trasero
    scd.BufferDesc.Format = DXGI_FORMAT_R8G8B8A8_UNORM;     // usar color de 32 bits
    scd.BufferUsage = DXGI_USAGE_RENDER_TARGET_OUTPUT;      // cómo se va a usar la cadena de intercambio
    scd.OutputWindow = hWnd;                                // la ventana a usar
    scd.SampleDesc.Count = 4;                               // cuántas muestras múltiples
    scd.Windowed = TRUE;                                    // modo ventana/pantalla completa

    // crear un dispositivo, contexto de dispositivo y cadena de intercambio usando la información en la estructura scd
    D3D11CreateDeviceAndSwapChain(NULL,
                                  D3D_DRIVER_TYPE_HARDWARE,
                                  NULL,
                                  NULL,
                                  NULL,
                                  NULL,
                                  D3D11_SDK_VERSION,
                                  &scd,
                                  &swapchain,
                                  &dev,
                                  NULL,
                                  &devcon);
}

// esta es la función que limpia Direct3D y COM
void CleanD3D(void)
{
    // cerrar y liberar todos los objetos COM existentes
    swapchain->Release();
    dev->Release();
    devcon->Release();
}

Explicación del Código

  • Encabezados y Bibliotecas: Incluimos los encabezados necesarios y enlazamos la biblioteca de Direct3D.
  • Declaraciones Globales: Declaramos punteros globales para la cadena de intercambio, el dispositivo y el contexto del dispositivo.
  • Función WinMain: Este es el punto de entrada para la aplicación de Windows. Configura la ventana y entra en el bucle principal.
  • Función WindowProc: Esta maneja los mensajes enviados a la ventana.
  • Función InitD3D: Esta inicializa Direct3D, configurando la cadena de intercambio, el dispositivo y el contexto del dispositivo.
  • Función CleanD3D: Esta libera los recursos de Direct3D cuando el programa termina.

Conclusión

Configurar tu entorno de desarrollo para la programación de DirectX implica instalar y configurar Visual Studio, configurar el SDK de DirectX y escribir un programa básico en DirectX para asegurarte de que todo esté funcionando correctamente. Con tu entorno listo, ahora puedes pasar a temas más avanzados y comenzar a desarrollar aplicaciones sofisticadas con DirectX.
