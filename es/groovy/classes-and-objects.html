Introducción a Clases y Objetos

En Groovy, como en muchos lenguajes de programación orientados a objetos, las clases y los objetos son conceptos fundamentales. Una clase es un plano para crear objetos, proporcionando valores iniciales para el estado (variables miembro o propiedades) e implementaciones de comportamiento (funciones miembro o métodos). Un objeto es una instancia de una clase.

Definiendo una Clase

Para definir una clase en Groovy, se usa la palabra clave class seguida del nombre de la clase. Aquí hay un ejemplo simple:

class Person {
    String name
    int age

    void displayInfo() {
        println "Name: $name, Age: $age"
    }
}
  • class Person: Define una nueva clase llamada Person.
  • String name: Declara una propiedad name de tipo String.
  • int age: Declara una propiedad age de tipo int.
  • void displayInfo(): Declara un método que imprime la información de la persona.

Creando Objetos

Para crear un objeto (o instancia) de una clase, se usa la palabra clave new seguida del nombre de la clase y paréntesis. Aquí se muestra cómo se puede crear un objeto de la clase Person:

def person = new Person()
person.name = "John Doe"
person.age = 30
person.displayInfo()
  • new Person(): Crea una nueva instancia de la clase Person.
  • person.name = "John Doe": Establece la propiedad name del objeto person.
  • person.age = 30: Establece la propiedad age del objeto person.
  • person.displayInfo(): Llama al método displayInfo en el objeto person.

Constructores

Los constructores son métodos especiales utilizados para inicializar objetos. En Groovy, se pueden definir constructores para establecer valores iniciales para las propiedades. Aquí hay un ejemplo:

class Person {
    String name
    int age

    Person(String name, int age) {
        this.name = name
        this.age = age
    }

    void displayInfo() {
        println "Name: $name, Age: $age"
    }
}

def person = new Person("Jane Doe", 25)
person.displayInfo()
  • Person(String name, int age): Constructor que inicializa las propiedades name y age.
  • this.name = name: Asigna el parámetro del constructor name a la propiedad name.
  • this.age = age: Asigna el parámetro del constructor age a la propiedad age.

Métodos

Los métodos definen el comportamiento de los objetos. Son funciones definidas dentro de una clase. Aquí hay un ejemplo de una clase con múltiples métodos:

class Calculator {
    int add(int a, int b) {
        return a + b
    }

    int subtract(int a, int b) {
        return a - b
    }
}

def calc = new Calculator()
println calc.add(5, 3)       // Output: 8
println calc.subtract(5, 3)  // Output: 2
  • int add(int a, int b): Método que suma dos enteros.
  • int subtract(int a, int b): Método que resta un entero de otro.

Herencia

La herencia permite que una clase herede propiedades y métodos de otra clase. En Groovy, se usa la palabra clave extends para lograr la herencia. Aquí hay un ejemplo:

class Animal {
    String name

    void eat() {
        println "$name is eating"
    }
}

class Dog extends Animal {
    void bark() {
        println "$name is barking"
    }
}

def dog = new Dog()
dog.name = "Buddy"
dog.eat()   // Output: Buddy is eating
dog.bark()  // Output: Buddy is barking
  • class Dog extends Animal: La clase Dog hereda de la clase Animal.
  • dog.eat(): Llama al método heredado eat de la clase Animal.
  • dog.bark(): Llama al método bark definido en la clase Dog.

Polimorfismo

El polimorfismo permite que los objetos sean tratados como instancias de su clase padre en lugar de su clase real. Esto es útil para escribir código más genérico y reutilizable. Aquí hay un ejemplo:

class Animal {
    void makeSound() {
        println "Some generic animal sound"
    }
}

class Dog extends Animal {
    void makeSound() {
        println "Bark"
    }
}

class Cat extends Animal {
    void makeSound() {
        println "Meow"
    }
}

def animals = [new Dog(), new Cat()]
animals.each { it.makeSound() }
// Output:
// Bark
// Meow
  • void makeSound(): Método sobrescrito en las clases Dog y Cat.
  • animals.each { it.makeSound() }: Llama al método makeSound en cada animal, demostrando polimorfismo.

Resumen

En este tema, cubrimos los conceptos básicos de clases y objetos en Groovy, incluyendo:

  • Definir clases y crear objetos.
  • Usar constructores para inicializar objetos.
  • Definir y usar métodos.
  • Implementar herencia para crear una jerarquía de clases.
  • Utilizar polimorfismo para escribir código flexible y reutilizable.

Entender estos conceptos es crucial para dominar la programación orientada a objetos en Groovy. Practica creando tus propias clases y objetos para reforzar estos conceptos.
