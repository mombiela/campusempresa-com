Rangos en Groovy
Closures ►

Los rangos son una característica poderosa y versátil en Groovy, que te permite crear secuencias de valores con facilidad. Se utilizan comúnmente para iteración, segmentación de colecciones y más. Este tema cubrirá los conceptos básicos de los rangos, su uso y características avanzadas.

¿Qué es un Rango?

Un rango en Groovy es una secuencia de valores definida por un valor inicial y un valor final. Los rangos pueden ser inclusivos o exclusivos, y se pueden usar con números, caracteres e incluso objetos personalizados.

Conceptos Clave

  • Rangos Inclusivos: Incluyen tanto el valor inicial como el final.
  • Rangos Exclusivos: Excluyen el valor final.
  • Valores de Paso: Definen el incremento entre valores en el rango.
  • Rangos Inversos: Definen rangos en orden descendente.

Creando Rangos

Rangos Inclusivos

Un rango inclusivo incluye tanto el valor inicial como el final. Puedes crear un rango inclusivo usando el operador ...

def inclusiveRange = 1..5
println inclusiveRange // Salida: [1, 2, 3, 4, 5]

Rangos Exclusivos

Un rango exclusivo excluye el valor final. Puedes crear un rango exclusivo usando el operador ..<.

def exclusiveRange = 1..<5
println exclusiveRange // Salida: [1, 2, 3, 4]

Rangos de Caracteres

Los rangos también se pueden crear con caracteres.

def charRange = 'a'..'e'
println charRange // Salida: [a, b, c, d, e]

Rangos Inversos

Los rangos se pueden definir en orden inverso.

def reverseRange = 5..1
println reverseRange // Salida: [5, 4, 3, 2, 1]

Usando Rangos

Iteración

Los rangos se utilizan a menudo en bucles para iteración.

for (i in 1..5) {
    println i
}
// Salida: 
// 1
// 2
// 3
// 4
// 5

Segmentación de Colecciones

Los rangos se pueden usar para segmentar listas y arreglos.

def list = [10, 20, 30, 40, 50]
def sublist = list[1..3]
println sublist // Salida: [20, 30, 40]

Valores de Paso

Puedes definir el valor de paso para un rango usando el método step.

def steppedRange = 1..10
println steppedRange.step(2) // Salida: [1, 3, 5, 7, 9]

Características Avanzadas

Rangos de Objetos Personalizados

Puedes crear rangos con objetos personalizados implementando los métodos next y previous.

class CustomRange {
    int value

    CustomRange next() {
        return new CustomRange(value: value + 1)
    }

    CustomRange previous() {
        return new CustomRange(value: value - 1)
    }

    String toString() {
        return value.toString()
    }
}

def customRange = new CustomRange(value: 1)..new CustomRange(value: 5)
println customRange // Salida: [1, 2, 3, 4, 5]

Comprobación de Pertenencia

Puedes comprobar si un valor está dentro de un rango usando el método contains o la palabra clave in.

def range = 1..5
println range.contains(3) // Salida: true
println 3 in range        // Salida: true
println 6 in range        // Salida: false

Conclusión

Los rangos en Groovy son una herramienta poderosa para crear secuencias de valores, iterar y segmentar colecciones. Pueden ser inclusivos o exclusivos, y soportan objetos personalizados y valores de paso. Entender cómo usar los rangos efectivamente puede mejorar en gran medida tus habilidades de programación en Groovy. Practica creando y usando rangos en diferentes contextos para sentirte más cómodo con esta característica.
