Introducción a la Entrada/Salida de Archivos

La entrada/salida de archivos (Input/Output) es un aspecto fundamental de la programación que te permite leer y escribir archivos. En F#, puedes manejar operaciones de archivos utilizando el espacio de nombres System.IO, que proporciona varias clases y métodos para trabajar con archivos y directorios.

Leyendo Archivos

Leyendo el Archivo Completo

Para leer el contenido completo de un archivo, puedes usar el método File.ReadAllText.

open System.IO

let readFile path =
    let content = File.ReadAllText(path)
    printfn "Contenido del Archivo:\n%s" content

// Ejemplo de uso
readFile "example.txt"

Leyendo Archivo Línea por Línea

Para leer un archivo línea por línea, puedes usar el método File.ReadLines.

open System.IO

let readFileLines path =
    let lines = File.ReadLines(path)
    for line in lines do
        printfn "%s" line

// Ejemplo de uso
readFileLines "example.txt"

Escribiendo Archivos

Escribiendo Contenido Completo

Para escribir una cadena a un archivo, puedes usar el método File.WriteAllText.

open System.IO

let writeFile path content =
    File.WriteAllText(path, content)
    printfn "Contenido escrito en %s" path

// Ejemplo de uso
writeFile "example.txt" "¡Hola, Mundo F#!"

Escribiendo Línea por Línea

Para escribir múltiples líneas a un archivo, puedes usar el método File.WriteAllLines.

open System.IO

let writeFileLines path lines =
    File.WriteAllLines(path, lines)
    printfn "Líneas escritas en %s" path

// Ejemplo de uso
writeFileLines "example.txt" [| "Línea 1"; "Línea 2"; "Línea 3" |]

Añadiendo a Archivos

Para añadir contenido a un archivo existente, puedes usar los métodos File.AppendAllText o File.AppendAllLines.

Añadiendo Texto

open System.IO

let appendToFile path content =
    File.AppendAllText(path, content)
    printfn "Contenido añadido a %s" path

// Ejemplo de uso
appendToFile "example.txt" "\nLínea añadida"

Añadiendo Líneas

open System.IO

let appendLinesToFile path lines =
    File.AppendAllLines(path, lines)
    printfn "Líneas añadidas a %s" path

// Ejemplo de uso
appendLinesToFile "example.txt" [| "Línea Añadida 1"; "Línea Añadida 2" |]

Trabajando con Archivos Binarios

Para operaciones de archivos binarios, puedes usar FileStream junto con BinaryReader y BinaryWriter.

Escribiendo Datos Binarios

open System.IO

let writeBinaryFile path data =
    use fs = new FileStream(path, FileMode.Create, FileAccess.Write)
    use bw = new BinaryWriter(fs)
    bw.Write(data)
    printfn "Datos binarios escritos en %s" path

// Ejemplo de uso
writeBinaryFile "example.bin" 12345

Leyendo Datos Binarios

open System.IO

let readBinaryFile path =
    use fs = new FileStream(path, FileMode.Open, FileAccess.Read)
    use br = new BinaryReader(fs)
    let data = br.ReadInt32()
    printfn "Datos binarios leídos de %s: %d" path data

// Ejemplo de uso
readBinaryFile "example.bin"

Manejo de Excepciones

Las operaciones de archivos pueden fallar por varias razones (por ejemplo, archivo no encontrado, acceso denegado). Es importante manejar las excepciones adecuadamente.

open System.IO

let safeReadFile path =
    try
        let content = File.ReadAllText(path)
        printfn "Contenido del Archivo:\n%s" content
    with
    | :? FileNotFoundException ->
        printfn "Archivo no encontrado: %s" path
    | :? UnauthorizedAccessException ->
        printfn "Acceso denegado al archivo: %s" path
    | ex ->
        printfn "Ocurrió un error: %s" (ex.Message)

// Ejemplo de uso
safeReadFile "nonexistent.txt"

Resumen

En esta sección, cubrimos los conceptos básicos de la entrada/salida de archivos en F#. Aprendiste cómo:

  • Leer archivos completamente o línea por línea.
  • Escribir y añadir texto o líneas a archivos.
  • Trabajar con archivos binarios usando FileStream, BinaryReader y BinaryWriter.
  • Manejar excepciones que pueden ocurrir durante las operaciones de archivos.

Entender la entrada/salida de archivos es crucial para muchas aplicaciones, y dominar estas técnicas te ayudará a manejar el almacenamiento y la recuperación de datos de manera eficiente en tus programas en F#.
