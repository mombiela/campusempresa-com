Coincidencia de Patrones en F#
Funciones de Orden Superior ►

La coincidencia de patrones es una característica poderosa en F# que te permite verificar un valor contra un patrón y descomponer datos de una manera concisa y legible. Este tema te guiará desde lo básico hasta el uso avanzado de la coincidencia de patrones en F#.

Introducción a la Coincidencia de Patrones

La coincidencia de patrones en F# se utiliza para hacer coincidir valores contra patrones y ejecutar código basado en la coincidencia. Es similar a las declaraciones switch-case en otros lenguajes, pero más poderosa y expresiva.

Coincidencia de Patrones Básica

En su forma más simple, la coincidencia de patrones se puede usar para hacer coincidir valores literales.

let describeNumber x =
    match x with
    | 1 -> "One"
    | 2 -> "Two"
    | 3 -> "Three"
    | _ -> "Unknown"

printfn "%s" (describeNumber 2)  // Salida: Two
  • match x with inicia la coincidencia de patrones.
  • | separa diferentes patrones.
  • _ es un patrón comodín que coincide con cualquier valor no coincidente con los patrones anteriores.

Coincidencia de Patrones con Tuplas

La coincidencia de patrones también se puede usar con tuplas para descomponer valores.

let addTuples (a, b) (c, d) =
    match (a, b, c, d) with
    | (0, 0, 0, 0) -> "All zeros"
    | (x, y, 0, 0) -> sprintf "First tuple: (%d, %d)" x y
    | (0, 0, x, y) -> sprintf "Second tuple: (%d, %d)" x y
    | _ -> "Mixed values"

printfn "%s" (addTuples (1, 2) (0, 0))  // Salida: First tuple: (1, 2)
  • Las tuplas se descomponen en la coincidencia de patrones.
  • Se pueden hacer coincidir patrones específicos para manejar diferentes casos.

Coincidencia de Patrones con Listas

Las listas también se pueden hacer coincidir para manejar diferentes estructuras.

let rec sumList lst =
    match lst with
    | [] -> 0
    | head :: tail -> head + sumList tail

printfn "%d" (sumList [1; 2; 3; 4])  // Salida: 10
  • [] coincide con una lista vacía.
  • head :: tail coincide con una lista no vacía, donde head es el primer elemento y tail es el resto de la lista.

Coincidencia de Patrones con Registros y Uniones Discriminadas

La coincidencia de patrones es particularmente poderosa cuando se usa con registros y uniones discriminadas.

Registros

type Person = { Name: string; Age: int }

let describePerson person =
    match person with
    | { Name = "Alice"; Age = 30 } -> "Alice is 30 years old"
    | { Name = name; Age = age } -> sprintf "%s is %d years old" name age

let alice = { Name = "Alice"; Age = 30 }
printfn "%s" (describePerson alice)  // Salida: Alice is 30 years old

Uniones Discriminadas

type Shape =
    | Circle of radius: float
    | Rectangle of width: float * height: float

let area shape =
    match shape with
    | Circle radius -> System.Math.PI * radius * radius
    | Rectangle (width, height) -> width * height

let myCircle = Circle 5.0
printfn "%f" (area myCircle)  // Salida: 78.539816

Coincidencia de Patrones Avanzada

Patrones Activos

Los patrones activos te permiten definir lógica de coincidencia de patrones personalizada.

let (|Even|Odd|) x =
    if x % 2 == 0 then Even else Odd

let describeNumber x =
    match x with
    | Even -> "Even"
    | Odd -> "Odd"

printfn "%s" (describeNumber 4)  // Salida: Even

Patrones Activos Parciales

Los patrones activos parciales pueden devolver un tipo de opción, permitiendo coincidencias más complejas.

let (|Positive|_|) x =
    if x > 0 then Some Positive else None

let describeNumber x =
    match x with
    | Positive -> "Positive"
    | _ -> "Non-positive"

printfn "%s" (describeNumber 5)  // Salida: Positive

Conclusión

La coincidencia de patrones en F# es una característica versátil y poderosa que simplifica el manejo de estructuras de datos complejas. Desde hacer coincidir valores simples hasta descomponer tuplas, listas, registros y uniones discriminadas, la coincidencia de patrones mejora la legibilidad y mantenibilidad del código. Las técnicas avanzadas como los patrones activos extienden aún más sus capacidades, haciendo de F# un lenguaje robusto para la programación funcional.

Al dominar la coincidencia de patrones, puedes escribir código más expresivo y conciso, haciendo tu programación en F# más efectiva y agradable.
