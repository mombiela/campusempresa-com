Introducción a la Recursión

La recursión es un concepto fundamental en la informática y la programación funcional. Implica que una función se llame a sí misma para resolver instancias más pequeñas del mismo problema. En F#, la recursión es una herramienta poderosa que permite soluciones elegantes y concisas para muchos problemas.

Conceptos Clave

  • Caso Base: La condición bajo la cual la función recursiva deja de llamarse a sí misma.
  • Caso Recursivo: La parte de la función donde la función se llama a sí misma con una entrada más pequeña o más simple.

Recursión Básica

Ejemplo: Función Factorial

El factorial de un número entero no negativo n es el producto de todos los enteros positivos menores o iguales a n. Se puede definir recursivamente como:

  • factorial(0) = 1 (Caso Base)
  • factorial(n) = n * factorial(n - 1) (Caso Recursivo)
let rec factorial n =
    if n = 0 then 1
    else n * factorial (n - 1)

// Ejemplo de uso:
let result = factorial 5 // result es 120

Explicación

  • Caso Base: Cuando n es 0, la función devuelve 1.
  • Caso Recursivo: La función se llama a sí misma con n - 1 y multiplica el resultado por n.

Recursión Intermedia

Ejemplo: Secuencia de Fibonacci

La secuencia de Fibonacci es una serie de números donde cada número es la suma de los dos anteriores. Se puede definir recursivamente como:

  • fibonacci(0) = 0 (Caso Base)
  • fibonacci(1) = 1 (Caso Base)
  • fibonacci(n) = fibonacci(n - 1) + fibonacci(n - 2) (Caso Recursivo)
let rec fibonacci n =
    match n with
    | 0 -> 0
    | 1 -> 1
    | _ -> fibonacci (n - 1) + fibonacci (n - 2)

// Ejemplo de uso:
let result = fibonacci 6 // result es 8

Explicación

  • Casos Base: Cuando n es 0 o 1, la función devuelve 0 o 1, respectivamente.
  • Caso Recursivo: La función se llama a sí misma con n - 1 y n - 2 y suma los resultados.

Recursión Avanzada

Ejemplo: Recursión de Cola

La recursión de cola es una forma especial de recursión donde la llamada recursiva es la última operación en la función. Esto permite que el compilador optimice la recursión, evitando el desbordamiento de pila y mejorando el rendimiento.

Factorial Recursivo de Cola

let factorialTailRecursive n =
    let rec loop acc n =
        if n = 0 then acc
        else loop (acc * n) (n - 1)
    loop 1 n

// Ejemplo de uso:
let result = factorialTailRecursive 5 // result es 120

Explicación

  • Acumulador: El parámetro acc acumula el resultado a medida que avanza la recursión.
  • Llamada de Cola: La llamada recursiva a loop es la última operación, permitiendo la optimización.

Ejemplo: Fibonacci Recursivo de Cola

let fibonacciTailRecursive n =
    let rec loop a b n =
        if n = 0 then a
        else loop b (a + b) (n - 1)
    loop 0 1 n

// Ejemplo de uso:
let result = fibonacciTailRecursive 6 // result es 8

Explicación

  • Parámetros: a y b contienen los dos números de Fibonacci anteriores.
  • Llamada de Cola: La llamada recursiva a loop es la última operación, permitiendo la optimización.

Conclusión

La recursión es una técnica poderosa en F# que permite soluciones elegantes a problemas complejos. Entender el caso base y el caso recursivo es crucial para escribir funciones recursivas correctas. La recursión de cola es una forma avanzada de recursión que optimiza el rendimiento y previene el desbordamiento de pila. Al dominar la recursión, puedes resolver una amplia gama de problemas de manera más efectiva en F#.
