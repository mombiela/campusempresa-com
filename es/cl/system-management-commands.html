Introducción

Los Comandos de Gestión del Sistema en Lenguaje de Control (CL) son esenciales para gestionar y controlar el sistema operativo IBM i. Este tema te guiará desde los conceptos básicos hasta el uso avanzado de estos comandos, proporcionándote el conocimiento para gestionar eficazmente los recursos del sistema, los usuarios y los trabajos.

Comandos Básicos de Gestión del Sistema

Comandos de Visualización

  • DSPJOB (Mostrar Trabajo): Muestra información sobre un trabajo.
  • DSPUSRPRF (Mostrar Perfil de Usuario): Muestra información sobre un perfil de usuario.
  • DSPMSG (Mostrar Mensaje): Muestra mensajes de una cola de mensajes.

Ejemplo: Mostrar Información del Trabajo

DSPJOB JOB(123456/USER/JOBNAME)

Este comando muestra información detallada sobre el trabajo con el número de trabajo, usuario y nombre de trabajo especificados.

Comandos de Trabajo

  • WRKACTJOB (Trabajar con Trabajos Activos): Muestra una lista de trabajos activos.
  • WRKSYSSTS (Trabajar con el Estado del Sistema): Muestra el estado actual del sistema.
  • WRKUSRPRF (Trabajar con Perfiles de Usuario): Muestra y permite la gestión de perfiles de usuario.

Ejemplo: Trabajar con Trabajos Activos

WRKACTJOB

Este comando abre una pantalla que muestra todos los trabajos activos, permitiéndote gestionarlos.

Comandos Intermedios de Gestión del Sistema

Comandos de Control

  • ENDJOB (Finalizar Trabajo): Finaliza un trabajo especificado.
  • ENDSYS (Finalizar Sistema): Finaliza el sistema de manera controlada.
  • ENDSBS (Finalizar Subsistema): Finaliza un subsistema especificado.

Ejemplo: Finalizar un Trabajo

ENDJOB JOB(123456/USER/JOBNAME) OPTION(*IMMED)

Este comando finaliza inmediatamente el trabajo especificado.

Comandos de Gestión de Usuarios

  • CRTUSRPRF (Crear Perfil de Usuario): Crea un nuevo perfil de usuario.
  • CHGUSRPRF (Cambiar Perfil de Usuario): Cambia los atributos de un perfil de usuario existente.
  • DLTUSRPRF (Eliminar Perfil de Usuario): Elimina un perfil de usuario.

Ejemplo: Crear un Perfil de Usuario

CRTUSRPRF USRPRF(NEWUSER) PASSWORD('password') USRCLS(*USER)

Este comando crea un nuevo perfil de usuario llamado NEWUSER con la contraseña y clase de usuario especificadas.

Comandos Avanzados de Gestión del Sistema

Comandos de Respaldo y Restauración

  • SAVLIB (Guardar Biblioteca): Guarda una biblioteca en un dispositivo especificado.
  • RSTLIB (Restaurar Biblioteca): Restaura una biblioteca desde un dispositivo especificado.
  • SAVSYS (Guardar Sistema): Guarda todo el sistema.

Ejemplo: Guardar una Biblioteca

SAVLIB LIB(MYLIB) DEV(TAP01)

Este comando guarda la biblioteca MYLIB en el dispositivo de cinta TAP01.

Comandos de Gestión del Rendimiento

  • WRKSYSACT (Trabajar con Actividad del Sistema): Muestra y gestiona la actividad del sistema.
  • WRKDSKSTS (Trabajar con el Estado del Disco): Muestra y gestiona el estado del disco.
  • WRKSHRPOOL (Trabajar con Pools Compartidos): Gestiona los pools de memoria compartida.

Ejemplo: Trabajar con la Actividad del Sistema

WRKSYSACT

Este comando abre una pantalla que muestra la actividad del sistema, permitiéndote monitorizar y gestionar el rendimiento.

Resumen

Los Comandos de Gestión del Sistema en CL son herramientas poderosas para gestionar el sistema operativo IBM i. Desde mostrar información de trabajos hasta gestionar perfiles de usuario y el rendimiento del sistema, estos comandos proporcionan un control integral sobre las operaciones del sistema. Al dominar estos comandos, puedes asegurar una gestión eficiente y efectiva del sistema.

Puntos Clave

  • Comandos básicos como DSPJOB y WRKACTJOB son esenciales para monitorizar el estado del sistema.
  • Comandos intermedios como ENDJOB y CRTUSRPRF permiten un control más directo sobre trabajos y perfiles de usuario.
  • Comandos avanzados como SAVLIB y WRKSYSACT son cruciales para el respaldo, la restauración y la gestión del rendimiento del sistema.

Al practicar estos comandos y entender sus aplicaciones, estarás bien equipado para gestionar un sistema IBM i de manera efectiva.
