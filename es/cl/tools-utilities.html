Introducción

El Lenguaje de Control (CL) es un poderoso lenguaje de scripting utilizado en los sistemas iSeries (AS/400) de IBM. Esta sección cubrirá varias herramientas y utilidades que pueden mejorar tu productividad y eficiencia al trabajar con CL. Exploraremos tanto herramientas integradas como utilidades de terceros que pueden asistir en el desarrollo, depuración y mantenimiento de programas CL.

Herramientas Integradas

SEU (Source Entry Utility)

SEU es un editor de texto para crear y modificar código fuente CL.

  • Características Clave:

    • Resaltado de sintaxis para comandos CL.
    • Numeración de líneas para fácil referencia.
    • Verificación básica de errores.

  • Ejemplo:

    STRSEU SRCFILE(MYLIB/QCLSRC) SRCMBR(MYCLPGM)

PDM (Programming Development Manager)

PDM es una interfaz basada en menús para gestionar código fuente y objetos.

  • Características Clave:

    • Navegar y editar miembros de fuente.
    • Compilar y enviar trabajos.
    • Gestionar bibliotecas y objetos.

  • Ejemplo:

    STRPDM

SDA (Screen Design Aid)

SDA se utiliza para diseñar archivos de pantalla (DSPF) para programas interactivos CL.

  • Características Clave:

    • Interfaz gráfica para diseñar pantallas.
    • Validación y formato a nivel de campo.
    • Integración con programas CL.

  • Ejemplo:

    STRSDA FILE(MYLIB/MYDSPF)

DFU (Data File Utility)

DFU permite una entrada y modificación de datos rápida y fácil en archivos de base de datos.

  • Características Clave:

    • Acceso directo a registros de base de datos.
    • Interfaz simple para manipulación de datos.
    • Soporta varios tipos de datos.

  • Ejemplo:

    STRDFU FILE(MYLIB/MYFILE)

    • Integración con control de versiones.

Conclusión

Entender y utilizar las diversas herramientas y utilidades disponibles para CL puede mejorar significativamente tu productividad y eficiencia. Las herramientas integradas como SEU, PDM, SDA y DFU proporcionan funcionalidades esenciales para el desarrollo y mantenimiento, mientras que las utilidades de terceros como RDi, TurnOver y Aldon ofrecen características avanzadas para proyectos más grandes y colaboración en equipo. Al dominar estas herramientas, puedes optimizar tu flujo de trabajo y mejorar la calidad de tus programas CL.
