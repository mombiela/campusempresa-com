Introducción al Lenguaje de Control (CL)

El Lenguaje de Control (CL) es un lenguaje de scripting utilizado en los sistemas de rango medio de IBM, como el AS/400 (ahora conocido como IBM i). Se utiliza principalmente para tareas de administración del sistema, control de trabajos y automatización de tareas repetitivas. Este curso te llevará de un nivel principiante a un nivel avanzado en CL, cubriendo conceptos esenciales, ejemplos prácticos y técnicas avanzadas.

Conceptos Clave

  • Comandos: Los bloques de construcción básicos de CL, utilizados para realizar tareas específicas.
  • Variables: Utilizadas para almacenar datos que pueden ser manipulados dentro de un programa CL.
  • Estructuras de Control: Incluye bucles y declaraciones condicionales para controlar el flujo del programa.
  • Manejo de Archivos: Técnicas para leer y escribir en archivos.
  • Manejo de Errores: Métodos para gestionar y responder a errores en programas CL.
  • Subrutinas: Bloques de código reutilizables que pueden ser llamados desde múltiples lugares dentro de un programa.

Nivel Principiante

Comandos Básicos

Los comandos CL se utilizan para realizar una amplia gama de tareas. Aquí hay algunos comandos básicos:

DSPJOB
  • DSPJOB: Muestra el estado de un trabajo.
WRKACTJOB
  • WRKACTJOB: Muestra una lista de trabajos activos.

Variables

Las variables en CL se definen usando el comando DCL:

DCL VAR(&MYVAR) TYPE(*CHAR) LEN(10)
  • DCL: Declara una variable.
  • VAR: Especifica el nombre de la variable.
  • TYPE: Especifica el tipo de datos (por ejemplo, *CHAR para carácter).
  • LEN: Especifica la longitud de la variable.

Estructuras de Control

Declaraciones Condicionales

IF COND(&MYVAR *EQ 'HELLO') THEN(DO)
    /* Comandos a ejecutar si la condición es verdadera */
ENDDO
  • IF: Inicia una declaración condicional.
  • COND: Especifica la condición a evaluar.
  • THEN: Especifica los comandos a ejecutar si la condición es verdadera.
  • DO/ENDDO: Marca el inicio y el fin de un bloque de comandos.

Bucles

DOFOR VAR(&I) FROM(1) TO(10)
    /* Comandos a ejecutar en el bucle */
ENDDO
  • DOFOR: Inicia un bucle que itera un número especificado de veces.
  • VAR: Especifica la variable del bucle.
  • FROM/TO: Especifica el rango para la variable del bucle.

Nivel Intermedio

Manejo de Archivos

Lectura de un Archivo

DCLF FILE(MYFILE)
RCVF
  • DCLF: Declara un archivo para su uso en el programa.
  • RCVF: Lee un registro del archivo en las variables declaradas.

Escritura en un Archivo

SNDF FILE(MYFILE) RCDFMT(MYRCDFMT)
  • SNDF: Envía un registro al archivo.
  • FILE: Especifica el nombre del archivo.
  • RCDFMT: Especifica el formato del registro.

Manejo de Errores

MONMSG MSGID(CPF0000) EXEC(DO)
    /* Comandos a ejecutar si ocurre un error */
ENDDO
  • MONMSG: Monitorea mensajes específicos (errores).
  • MSGID: Especifica el ID del mensaje a monitorear.
  • EXEC: Especifica los comandos a ejecutar si se recibe el mensaje.

Nivel Avanzado

Subrutinas

Las subrutinas te permiten crear bloques de código reutilizables:

CALLSUBR SUBR(MYSUBR)
SUBR MYSUBR
    /* Comandos en la subrutina */
ENDSUBR
  • CALLSUBR: Llama a una subrutina.
  • SUBR: Define el inicio de una subrutina.
  • ENDSUBR: Marca el fin de una subrutina.

Manejo Avanzado de Archivos

Uso de Estructuras de Datos

DCLF FILE(MYFILE)
DCL DS MYDS
RCVF
  • DS: Declara una estructura de datos.
  • MYDS: Especifica el nombre de la estructura de datos.

Conclusión

Este curso ha proporcionado una visión general completa del Lenguaje de Control (CL) desde comandos básicos y variables hasta manejo avanzado de archivos y subrutinas. Al dominar estos conceptos, estarás bien equipado para automatizar tareas, gestionar trabajos y manejar archivos de manera eficiente en los sistemas de rango medio de IBM. Continúa practicando y explorando temas más avanzados para profundizar tu comprensión y competencia en CL.
