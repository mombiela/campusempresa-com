Introducción

La gestión de archivos es un aspecto crucial al trabajar con el Lenguaje de Control (CL) en sistemas IBM i. Esta sección cubrirá los comandos esenciales para gestionar archivos, desde operaciones básicas hasta técnicas avanzadas. Al final de este tema, podrás crear, eliminar y manipular archivos usando comandos CL de manera efectiva.

Comandos Básicos de Gestión de Archivos

Creación de Archivos

Para crear archivos en CL, puedes usar el comando CRTPF (Crear Archivo Físico). Este comando te permite definir la estructura y los atributos del archivo.

CRTPF FILE(MYLIB/MYFILE) RCDLEN(100)
  • FILE: Especifica la biblioteca y el nombre del archivo.
  • RCDLEN: Define la longitud del registro.

Eliminación de Archivos

El comando DLTF (Eliminar Archivo) se usa para eliminar archivos del sistema.

DLTF FILE(MYLIB/MYFILE)
  • FILE: Especifica la biblioteca y el nombre del archivo a eliminar.

Copia de Archivos

Para copiar archivos, puedes usar el comando CPYF (Copiar Archivo). Este comando te permite copiar datos de un archivo a otro.

CPYF FROMFILE(MYLIB/SRCFILE) TOFILE(MYLIB/DESTFILE) MBROPT(*REPLACE)
  • FROMFILE: Especifica el archivo fuente.
  • TOFILE: Especifica el archivo de destino.
  • MBROPT: Define la opción de miembro, como *REPLACE para reemplazar los datos existentes.

Comandos Intermedios de Gestión de Archivos

Renombrar Archivos

El comando RNMOBJ (Renombrar Objeto) se usa para renombrar archivos.

RNMOBJ OBJ(MYLIB/OLDNAME) OBJTYPE(*FILE) NEWOBJ(NEWNAME)
  • OBJ: Especifica la biblioteca y el nombre actual del archivo.
  • OBJTYPE: Define el tipo de objeto, como *FILE.
  • NEWOBJ: Especifica el nuevo nombre del archivo.

Mostrar Información del Archivo

Para mostrar información sobre un archivo, usa el comando DSPFD (Mostrar Descripción del Archivo).

DSPFD FILE(MYLIB/MYFILE)
  • FILE: Especifica la biblioteca y el nombre del archivo.

Cambiar Atributos del Archivo

El comando CHGPF (Cambiar Archivo Físico) te permite cambiar los atributos de un archivo.

CHGPF FILE(MYLIB/MYFILE) SIZE(*NOMAX)
  • FILE: Especifica la biblioteca y el nombre del archivo.
  • SIZE: Define el atributo de tamaño del archivo, como *NOMAX para sin tamaño máximo.

Comandos Avanzados de Gestión de Archivos

Fusionar Archivos

El comando CPYF también se puede usar para fusionar archivos especificando la opción MBROPT(*ADD).

CPYF FROMFILE(MYLIB/SRCFILE) TOFILE(MYLIB/DESTFILE) MBROPT(*ADD)
  • MBROPT(*ADD): Agrega los registros del archivo fuente al archivo de destino.

Creación de Archivos Lógicos

Los archivos lógicos se pueden crear usando el comando CRTLF (Crear Archivo Lógico). Los archivos lógicos proporcionan una forma de ver y acceder a los datos en archivos físicos.

CRTLF FILE(MYLIB/MYLOGFILE) SRCFILE(MYLIB/QDDSSRC) SRCMBR(MYLOGMBR)
  • FILE: Especifica la biblioteca y el nombre del archivo lógico.
  • SRCFILE: Especifica el archivo fuente que contiene las DDS (Especificaciones de Descripción de Datos).
  • SRCMBR: Especifica el nombre del miembro fuente.

Reorganización de Archivos

El comando RGZPFM (Reorganizar Miembro de Archivo Físico) se usa para reorganizar un archivo físico para recuperar espacio y mejorar el rendimiento.

RGZPFM FILE(MYLIB/MYFILE)
  • FILE: Especifica la biblioteca y el nombre del archivo.

Resumen

En este tema, cubrimos los comandos esenciales de gestión de archivos en CL, desde operaciones básicas como crear y eliminar archivos hasta técnicas más avanzadas como fusionar archivos y crear archivos lógicos. Entender estos comandos te permitirá gestionar archivos de manera efectiva en un entorno IBM i. Practica usando estos comandos para volverte competente en la gestión de archivos con CL.
