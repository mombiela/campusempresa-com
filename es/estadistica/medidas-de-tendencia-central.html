Introducción

Las medidas de tendencia central son estadísticas que describen el centro de un conjunto de datos. Estas medidas son fundamentales en estadística porque proporcionan una idea de dónde se encuentra el "centro" de los datos. Las tres medidas de tendencia central más comunes son la media, la mediana y la moda.

Media

La media, también conocida como promedio, es la suma de todos los valores dividida por el número de valores. Es una medida muy utilizada debido a su simplicidad y facilidad de cálculo.

  • Fórmula: \[ \text{Media} = \frac{\sum_{i=1}^{n} x_i}{n} \] donde \( x_i \) son los valores del conjunto de datos y \( n \) es el número total de valores.

  • Ejemplo:

    # Calcular la media de una lista de números en Python
datos = [10, 20, 30, 40, 50]
media = sum(datos) / len(datos)
print(f"La media es: {media}")

Mediana

La mediana es el valor que separa la mitad superior de la mitad inferior de un conjunto de datos ordenados. Si el número de valores es impar, la mediana es el valor del medio. Si es par, la mediana es el promedio de los dos valores del medio.

  • Pasos para calcular la mediana:

    1. Ordenar los datos.
    2. Si el número de valores es impar, la mediana es el valor del medio.
    3. Si el número de valores es par, la mediana es el promedio de los dos valores del medio.

  • Ejemplo:

    # Calcular la mediana de una lista de números en Python
datos = [10, 20, 30, 40, 50]
datos.sort()
n = len(datos)
if n % 2 == 0:
    mediana = (datos[n//2 - 1] + datos[n//2]) / 2
else:
    mediana = datos[n//2]
print(f"La mediana es: {mediana}")

Moda

La moda es el valor que aparece con mayor frecuencia en un conjunto de datos. Un conjunto de datos puede tener una moda, más de una moda (bimodal o multimodal) o no tener ninguna moda si todos los valores son únicos.

  • Ejemplo: 
    from collections import Counter

# Calcular la moda de una lista de números en Python
datos = [10, 20, 20, 30, 40, 50, 50, 50]
frecuencia = Counter(datos)
moda = [k for k, v in frecuencia.items() if v == max(frecuencia.values())]
print(f"La moda es: {moda}")

Comparación de Medidas de Tendencia Central

| Medida | Ventajas | Desventajas | |---------|-----------------------------------------------|----------------------------------------------| | Media | Fácil de calcular y entender | Sensible a valores atípicos (outliers) | | Mediana | No afectada por valores atípicos | No utiliza toda la información del conjunto | | Moda | Útil para datos categóricos | Puede no ser única o no existir |

Conclusión

Las medidas de tendencia central son herramientas esenciales en estadística para resumir y describir conjuntos de datos. La media, la mediana y la moda ofrecen diferentes perspectivas sobre los datos y son útiles en distintas situaciones. Comprender cómo y cuándo utilizar cada una de estas medidas es crucial para el análisis de datos efectivo.
