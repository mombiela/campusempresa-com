¿Qué es la Estadística?

La estadística es una rama de las matemáticas que se ocupa de la recolección, análisis, interpretación, presentación y organización de datos. Es una herramienta fundamental en muchas disciplinas, incluyendo la ciencia, la ingeniería, la economía y las ciencias sociales.

Conceptos Clave

  • Población y Muestra:

    • Población: Conjunto completo de elementos que se estudian.
    • Muestra: Subconjunto de la población que se selecciona para el estudio.

  • Variables:

    • Cualitativas: Describen cualidades o categorías (e.g., color, género).
    • Cuantitativas: Describen cantidades y pueden ser discretas o continuas (e.g., edad, altura).

  • Parámetros y Estadísticos:

    • Parámetro: Medida descriptiva de una población (e.g., media poblacional).
    • Estadístico: Medida descriptiva de una muestra (e.g., media muestral).

  • Tipos de Datos:

    • Nominal: Categorías sin orden (e.g., colores).
    • Ordinal: Categorías con orden (e.g., niveles de satisfacción).
    • Intervalo: Diferencias significativas entre valores (e.g., temperatura en Celsius).
    • Razón: Igual que intervalo, pero con un punto cero absoluto (e.g., peso).

Métodos de Recolección de Datos

  • Encuestas: Recopilación de datos a través de cuestionarios.
  • Experimentos: Realización de pruebas controladas para observar efectos.
  • Observación: Registro de datos sin intervención directa.
  • Fuentes Secundarias: Uso de datos ya recopilados por otras entidades.

Análisis Descriptivo

El análisis descriptivo se centra en resumir y describir las características de un conjunto de datos.

Medidas de Tendencia Central

  • Media (Promedio): Suma de todos los valores dividida por el número de valores.
  • Mediana: Valor central cuando los datos están ordenados.
  • Moda: Valor que aparece con mayor frecuencia.

Medidas de Dispersión

  • Rango: Diferencia entre el valor máximo y el mínimo.
  • Varianza: Promedio de las diferencias al cuadrado respecto a la media.
  • Desviación Estándar: Raíz cuadrada de la varianza.

Ejemplo Práctico

Supongamos que tenemos los siguientes datos de edades de un grupo de personas: 23, 29, 20, 32, 25, 27, 22.

Cálculo de la Media

edades = [23, 29, 20, 32, 25, 27, 22]
media = sum(edades) / len(edades)
print(f"La media es: {media}")

Cálculo de la Mediana

edades.sort()
n = len(edades)
if n % 2 == 0:
    mediana = (edades[n//2 - 1] + edades[n//2]) / 2
else:
    mediana = edades[n//2]
print(f"La mediana es: {mediana}")

Cálculo de la Moda

from collections import Counter
moda = Counter(edades).most_common(1)[0][0]
print(f"La moda es: {moda}")

Conclusión

La estadística es una herramienta poderosa para analizar datos y tomar decisiones informadas. En esta introducción, hemos cubierto los conceptos básicos y métodos de recolección de datos, así como las principales medidas de tendencia central y dispersión. Estos fundamentos son esenciales para cualquier análisis estadístico más avanzado que se realice en el futuro.
