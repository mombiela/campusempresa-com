Introducción

En esta sección, exploraremos los fundamentos de la clasificación de imágenes utilizando PyTorch. Este proyecto te guiará desde la configuración de tu entorno hasta la construcción y el entrenamiento de un modelo de aprendizaje profundo para clasificar imágenes. Al final de este proyecto, tendrás una comprensión sólida de cómo implementar modelos de clasificación de imágenes usando PyTorch.

Configuración del Entorno

Antes de comenzar a codificar, necesitamos configurar nuestro entorno de desarrollo.

Instalando PyTorch

Para instalar PyTorch, puedes usar pip. Abre tu terminal y ejecuta el siguiente comando:

pip install torch torchvision

Importando Librerías Necesarias

Necesitaremos varias librerías para este proyecto. Vamos a importarlas:

import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms

Preparación de Datos

La preparación de datos es un paso crucial en cualquier proyecto de aprendizaje automático. Usaremos el conjunto de datos CIFAR-10 para este proyecto.

Cargando el Conjunto de Datos CIFAR-10

PyTorch proporciona una manera conveniente de cargar el conjunto de datos CIFAR-10 usando torchvision.datasets.

transform = transforms.Compose(
    [transforms.ToTensor(),
     transforms.Normalize((0.5, 0.5, 0.5), (0.5, 0.5, 0.5))])

trainset = torchvision.datasets.CIFAR10(root='./data', train=True,
                                        download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=4,
                                          shuffle=True, num_workers=2)

testset = torchvision.datasets.CIFAR10(root='./data', train=False,
                                       download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=4,
                                         shuffle=False, num_workers=2)

classes = ('plane', 'car', 'bird', 'cat',
           'deer', 'dog', 'frog', 'horse', 'ship', 'truck')

Construcción del Modelo

Construiremos una Red Neuronal Convolucional (CNN) para nuestra tarea de clasificación de imágenes.

Definiendo la CNN

Aquí hay un modelo CNN simple:

class Net(nn.Module):
    def __init__(self):
        super(Net, self).__init__()
        self.conv1 = nn.Conv2d(3, 6, 5)
        self.pool = nn.MaxPool2d(2, 2)
        self.conv2 = nn.Conv2d(6, 16, 5)
        self.fc1 = nn.Linear(16 * 5 * 5, 120)
        self.fc2 = nn.Linear(120, 84)
        self.fc3 = nn.Linear(84, 10)

    def forward(self, x):
        x = self.pool(F.relu(self.conv1(x)))
        x = self.pool(F.relu(self.conv2(x)))
        x = x.view(-1, 16 * 5 * 5)
        x = F.relu(self.fc1(x))
        x = F.relu(self.fc2(x))
        x = self.fc3(x)
        return x

net = Net()

Entrenamiento del Modelo

Ahora que tenemos nuestro modelo, necesitamos entrenarlo.

Definiendo la Función de Pérdida y el Optimizador

Usaremos la Pérdida de Entropía Cruzada y el Descenso de Gradiente Estocástico (SGD) para esta tarea.

criterion = nn.CrossEntropyLoss()
optimizer = optim.SGD(net.parameters(), lr=0.001, momentum=0.9)

Bucle de Entrenamiento

Aquí está el código para entrenar el modelo:

for epoch in range(2):  # recorrer el conjunto de datos varias veces

    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        # obtener las entradas; data es una lista de [entradas, etiquetas]
        inputs, labels = data

        # poner a cero los gradientes de los parámetros
        optimizer.zero_grad()

        # hacia adelante + hacia atrás + optimizar
        outputs = net(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()

        # imprimir estadísticas
        running_loss += loss.item()
        if i % 2000 == 1999:    # imprimir cada 2000 mini-lotes
            print(f'[{epoch + 1}, {i + 1}] pérdida: {running_loss / 2000:.3f}')
            running_loss = 0.0

print('Entrenamiento Finalizado')

Evaluación del Modelo

Después del entrenamiento, necesitamos evaluar el rendimiento del modelo en los datos de prueba.

Probando el Modelo

Aquí está el código para probar el modelo:

correct = 0
total = 0
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f'Precisión de la red en las 10000 imágenes de prueba: {100 * correct / total}%')

Rendimiento por Clase

También podemos verificar el rendimiento del modelo para cada clase.

class_correct = list(0. for i in range(10))
class_total = list(0. for i in range(10))
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = net(images)
        _, predicted = torch.max(outputs, 1)
        c = (predicted == labels).squeeze()
        for i in range(4):
            label = labels[i]
            class_correct[label] += c[i].item()
            class_total[label] += 1

for i in range(10):
    print(f'Precisión de {classes[i]} : {100 * class_correct[i] / class_total[i]}%')
