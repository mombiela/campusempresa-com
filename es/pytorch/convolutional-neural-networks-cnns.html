Introducción a las CNNs

Las Redes Neuronales Convolucionales (CNNs) son una clase de redes neuronales profundas que son particularmente efectivas para analizar datos visuales. Se utilizan ampliamente en el reconocimiento de imágenes y videos, sistemas de recomendación y procesamiento del lenguaje natural.

Conceptos Clave

  • Capas Convolucionales: Capas que aplican una operación de convolución a la entrada, pasando el resultado a la siguiente capa.
  • Capas de Pooling: Capas que reducen las dimensiones espaciales (ancho y alto) del volumen de entrada.
  • Capas Completamente Conectadas: Capas donde cada neurona está conectada a cada neurona de la capa anterior.
  • Funciones de Activación: Funciones aplicadas a la salida de cada neurona para introducir no linealidad (por ejemplo, ReLU).
  • Dropout: Una técnica para prevenir el sobreajuste al establecer aleatoriamente una fracción de las unidades de entrada a cero en cada actualización durante el entrenamiento.

Configuración de PyTorch

Antes de profundizar en las CNNs, asegúrate de tener PyTorch instalado. Puedes instalarlo usando pip:

pip install torch torchvision

Construyendo una CNN Simple

Comencemos construyendo una CNN simple para la clasificación de imágenes usando el conjunto de datos MNIST.

Paso 1: Importar Bibliotecas

import torch
import torch.nn as nn
import torch.optim as optim
import torchvision
import torchvision.transforms as transforms

Paso 2: Cargar y Preprocesar Datos

transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
trainloader = torch.utils.data.DataLoader(trainset, batch_size=32, shuffle=True)

testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)
testloader = torch.utils.data.DataLoader(testset, batch_size=32, shuffle=False)

Paso 3: Definir el Modelo CNN

class SimpleCNN(nn.Module):
    def __init__(self):
        super(SimpleCNN, self).__init__()
        self.conv1 = nn.Conv2d(1, 32, 3, 1)
        self.conv2 = nn.Conv2d(32, 64, 3, 1)
        self.fc1 = nn.Linear(9216, 128)
        self.fc2 = nn.Linear(128, 10)
        self.dropout1 = nn.Dropout(0.25)
        self.dropout2 = nn.Dropout(0.5)

    def forward(self, x):
        x = self.conv1(x)
        x = nn.ReLU()(x)
        x = self.conv2(x)
        x = nn.ReLU()(x)
        x = nn.MaxPool2d(2)(x)
        x = self.dropout1(x)
        x = torch.flatten(x, 1)
        x = self.fc1(x)
        x = nn.ReLU()(x)
        x = self.dropout2(x)
        x = self.fc2(x)
        return nn.LogSoftmax(dim=1)(x)

Paso 4: Entrenar el Modelo

model = SimpleCNN()
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

for epoch in range(10):  # recorrer el conjunto de datos varias veces
    running_loss = 0.0
    for i, data in enumerate(trainloader, 0):
        inputs, labels = data
        optimizer.zero_grad()
        outputs = model(inputs)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()
        running_loss += loss.item()
        if i % 100 == 99:    # imprimir cada 100 mini-lotes
            print(f'[Epoch {epoch + 1}, Batch {i + 1}] loss: {running_loss / 100:.3f}')
            running_loss = 0.0

print('Finished Training')

Paso 5: Evaluar el Modelo

correct = 0
total = 0
with torch.no_grad():
    for data in testloader:
        images, labels = data
        outputs = model(images)
        _, predicted = torch.max(outputs.data, 1)
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

print(f'Accuracy of the network on the 10000 test images: {100 * correct / total:.2f}%')

Temas Avanzados en CNNs

Una vez que te sientas cómodo construyendo una CNN simple, puedes explorar temas más avanzados.

Transferencia de Aprendizaje

La transferencia de aprendizaje implica usar un modelo preentrenado y ajustarlo para una tarea específica.

import torchvision.models as models

# Cargar un modelo ResNet preentrenado
model = models.resnet18(pretrained=True)

# Congelar todas las capas
for param in model.parameters():
    param.requires_grad = False

# Reemplazar la última capa completamente conectada
model.fc = nn.Linear(model.fc.in_features, 10)

# Ahora solo se actualizarán los parámetros de la última capa
optimizer = optim.Adam(model.fc.parameters(), lr=0.001)

Aumento de Datos

Las técnicas de aumento de datos pueden ayudar a mejorar la robustez de tu modelo al aumentar artificialmente el tamaño de tu conjunto de datos de entrenamiento.

transform = transforms.Compose([
    transforms.RandomHorizontalFlip(),
    transforms.RandomRotation(10),
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

Ajuste de Hiperparámetros

Experimenta con diferentes hiperparámetros como la tasa de aprendizaje, el tamaño del lote y el número de épocas para optimizar el rendimiento de tu modelo.

Conclusión

En este tema, cubrimos los conceptos básicos de las Redes Neuronales Convolucionales (CNNs) y cómo implementarlas usando PyTorch. Comenzamos con una CNN simple para la clasificación de imágenes y luego exploramos temas avanzados como la transferencia de aprendizaje, el aumento de datos y el ajuste de hiperparámetros. Al dominar estos conceptos, estarás bien equipado para abordar una amplia gama de tareas que involucren datos visuales.
