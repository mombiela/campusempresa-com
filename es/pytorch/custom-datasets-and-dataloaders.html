Conjuntos de Datos Personalizados y DataLoaders
Funciones de Pérdida Personalizadas ►

En esta sección, exploraremos cómo crear conjuntos de datos personalizados y dataloaders en PyTorch. Esto es crucial para manejar datos que no se ajustan a los conjuntos de datos estándar proporcionados por PyTorch. Al final de esta sección, podrás crear tus propios conjuntos de datos y cargarlos eficientemente usando el DataLoader de PyTorch.

Conceptos Clave

  • Clase Dataset: Una clase personalizada que hereda de torch.utils.data.Dataset e implementa los métodos __len__ y __getitem__.
  • Clase DataLoader: Una clase que proporciona un iterable sobre un conjunto de datos con soporte para batching, shuffling y carga de datos multiproceso.

Creando un Conjunto de Datos Personalizado

Para crear un conjunto de datos personalizado, necesitas subclasificar torch.utils.data.Dataset y sobrescribir dos métodos:

  1. __len__: Devuelve el tamaño del conjunto de datos.
  2. __getitem__: Recupera una muestra del conjunto de datos en el índice dado.

Ejemplo: Conjunto de Datos de Imágenes Personalizado

Vamos a crear un conjunto de datos personalizado para cargar imágenes y sus etiquetas correspondientes.

import os
from PIL import Image
import torch
from torch.utils.data import Dataset

class CustomImageDataset(Dataset):
    def __init__(self, img_dir, transform=None):
        self.img_dir = img_dir
        self.transform = transform
        self.img_labels = [(img_name, int(img_name.split('_')[1])) for img_name in os.listdir(img_dir)]

    def __len__(self):
        return len(self.img_labels)

    def __getitem__(self, idx):
        img_name, label = self.img_labels[idx]
        img_path = os.path.join(self.img_dir, img_name)
        image = Image.open(img_path)
        
        if self.transform:
            image = self.transform(image)
        
        return image, label
  • Explicación:
    • __init__: Inicializa el conjunto de datos con el directorio de imágenes y una transformación opcional.
    • __len__: Devuelve el número de imágenes en el directorio.
    • __getitem__: Carga una imagen y su etiqueta basada en el índice.

Usando DataLoader

La clase DataLoader proporciona una forma eficiente de iterar sobre el conjunto de datos. Soporta batching, shuffling y carga de datos en paralelo.

Ejemplo: Usando DataLoader con Conjunto de Datos Personalizado

from torchvision import transforms
from torch.utils.data import DataLoader

# Define una transformación para convertir imágenes a tensores
transform = transforms.Compose([
    transforms.Resize((128, 128)),
    transforms.ToTensor()
])

# Crea una instancia del conjunto de datos personalizado
dataset = CustomImageDataset(img_dir='path/to/images', transform=transform)

# Crea un DataLoader
dataloader = DataLoader(dataset, batch_size=4, shuffle=True, num_workers=2)

# Itera sobre el DataLoader
for images, labels in dataloader:
    print(images.size(), labels)
  • Explicación:
    • transform: Una serie de transformaciones aplicadas a cada imagen.
    • DataLoader: Crea un iterable sobre el conjunto de datos con batching, shuffling y carga en paralelo.
    • for images, labels in dataloader: Itera sobre el conjunto de datos en lotes.

Uso Avanzado

Manejo de Diferentes Tipos de Datos

Puedes crear conjuntos de datos personalizados para varios tipos de datos como texto, audio o incluso datos multimodales complejos.

Ejemplo: Conjunto de Datos de Texto Personalizado

class CustomTextDataset(Dataset):
    def __init__(self, text_file, transform=None):
        with open(text_file, 'r') as file:
            self.text_data = file.readlines()
        self.transform = transform

    def __len__(self):
        return len(self.text_data)

    def __getitem__(self, idx):
        text = self.text_data[idx].strip()
        
        if self.transform:
            text = self.transform(text)
        
        return text
  • Explicación:
    • __init__: Lee datos de texto de un archivo.
    • __len__: Devuelve el número de líneas en el archivo de texto.
    • __getitem__: Recupera una línea de texto y aplica cualquier transformación.

Conclusión

Crear conjuntos de datos personalizados y dataloaders en PyTorch te permite manejar una amplia variedad de tipos y formatos de datos. Al subclasificar torch.utils.data.Dataset y usar DataLoader, puedes cargar y preprocesar tus datos de manera eficiente, dejándolos listos para entrenar modelos de aprendizaje automático.

  • Puntos Clave:
    • Subclasifica torch.utils.data.Dataset para crear conjuntos de datos personalizados.
    • Implementa los métodos __len__ y __getitem__.
    • Usa DataLoader para una carga de datos eficiente con soporte para batching, shuffling y procesamiento en paralelo.

Con estas herramientas, puedes manejar cualquier conjunto de datos, sin importar cuán complejo o único sea, y prepararlo para tus tareas de aprendizaje automático.
