Introducción a las Intenciones

Las intenciones son un concepto fundamental en el desarrollo de Android, utilizado para la comunicación entre componentes como actividades, servicios y receptores de difusión. Pueden usarse para iniciar una nueva actividad, enviar datos a otra actividad o incluso comunicarse con otras aplicaciones.

Conceptos Clave

  • Intención: Un objeto de mensajería que puedes usar para solicitar una acción de otro componente de la aplicación.
  • Intención Explícita: Especifica el componente a iniciar por nombre.
  • Intención Implícita: No nombra un componente específico, sino que declara una acción general a realizar.

Intenciones Explícitas

Las intenciones explícitas se utilizan para iniciar un componente específico dentro de tu aplicación. Esto es útil cuando conoces la actividad o servicio exacto que deseas iniciar.

Ejemplo

Intent intent = new Intent(this, SecondActivity.class);
intent.putExtra("EXTRA_MESSAGE", "¡Hola, Segunda Actividad!");
startActivity(intent);
  • Intent(this, SecondActivity.class): Crea una intención explícita para iniciar SecondActivity.
  • putExtra("EXTRA_MESSAGE", "¡Hola, Segunda Actividad!"): Añade datos extra a la intención.
  • startActivity(intent): Inicia la actividad especificada en la intención.

Intenciones Implícitas

Las intenciones implícitas no especifican un componente; en su lugar, declaran una acción a realizar. El sistema Android empareja la intención con el componente apropiado.

Ejemplo

Intent intent = new Intent(Intent.ACTION_VIEW);
intent.setData(Uri.parse("http://www.example.com"));
startActivity(intent);
  • Intent(Intent.ACTION_VIEW): Crea una intención implícita con la acción ACTION_VIEW.
  • setData(Uri.parse("http://www.example.com")): Establece los datos para la intención.
  • startActivity(intent): Inicia una actividad que pueda manejar la acción ACTION_VIEW para la URI dada.

Filtros de Intenciones

Los filtros de intenciones son declaraciones en el archivo AndroidManifest.xml que especifican los tipos de intenciones a los que un componente puede responder. Se utilizan para registrar actividades, servicios y receptores de difusión para acciones, tipos de datos y categorías específicos.

Conceptos Clave

  • Acción: La acción general a realizar (por ejemplo, ACTION_VIEW, ACTION_SEND).
  • Datos: El tipo de datos y la URI que la intención puede manejar.
  • Categoría: Información adicional sobre la acción (por ejemplo, CATEGORY_DEFAULT, CATEGORY_BROWSABLE).

Ejemplo de un Filtro de Intención

<activity android:name=".SecondActivity">
    <intent-filter>
        <action android:name="android.intent.action.VIEW" />
        <category android:name="android.intent.category.DEFAULT" />
        <data android:scheme="http" android:host="www.example.com" />
    </intent-filter>
</activity>
  • <action android:name="android.intent.action.VIEW" />: Especifica la acción VIEW.
  • <category android:name="android.intent.category.DEFAULT" />: Especifica la categoría predeterminada.
  • <data android:scheme="http" android:host="www.example.com" />: Especifica el esquema de datos y el host.

Ejercicios Prácticos

Ejercicio 1: Crear una Intención Explícita

  1. Crea una nueva actividad llamada DetailActivity.
  2. En tu actividad principal, crea una intención explícita para iniciar DetailActivity.
  3. Pasa un mensaje de cadena a DetailActivity usando putExtra.
  4. Muestra el mensaje en DetailActivity.

Ejercicio 2: Crear una Intención Implícita

  1. Crea una intención implícita para abrir una página web.
  2. Usa Intent.ACTION_VIEW y establece los datos a una URL.
  3. Prueba la intención para asegurarte de que abre el navegador web con la URL especificada.

Ejercicio 3: Definir un Filtro de Intención

  1. Modifica tu AndroidManifest.xml para añadir un filtro de intención para DetailActivity.
  2. Configura el filtro de intención para manejar la acción ACTION_VIEW y un esquema personalizado myapp.
  3. Prueba el filtro de intención creando una intención implícita con el esquema personalizado.

Conclusión

Las intenciones y los filtros de intenciones son herramientas poderosas en el desarrollo de Android, que permiten una comunicación fluida entre componentes y aplicaciones. Al comprender y utilizar intenciones explícitas e implícitas, así como definir correctamente los filtros de intenciones, puedes crear aplicaciones de Android más dinámicas e interactivas. Practica con los ejercicios proporcionados para solidificar tu comprensión y mejorar tus habilidades.
