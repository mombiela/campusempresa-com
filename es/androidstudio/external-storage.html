El almacenamiento externo en Android se refiere al almacenamiento que es accesible para todas las aplicaciones y el usuario. Puede ser utilizado para almacenar archivos grandes y datos que necesitan ser compartidos entre diferentes aplicaciones o con el usuario. Esta sección te guiará a través de los conceptos básicos del almacenamiento externo, cómo leer y escribir archivos, y las mejores prácticas para gestionar el almacenamiento externo en tus aplicaciones Android.

Conceptos Clave

  • Tipos de Almacenamiento Externo:

    • Almacenamiento Externo Primario: El almacenamiento externo principal, usualmente el almacenamiento compartido interno del dispositivo.
    • Almacenamiento Externo Secundario: Opciones de almacenamiento adicionales como tarjetas SD o unidades USB.

  • Permisos:

    • READ_EXTERNAL_STORAGE: Requerido para leer archivos del almacenamiento externo.
    • WRITE_EXTERNAL_STORAGE: Requerido para escribir archivos en el almacenamiento externo.

  • Directorios de Almacenamiento:

    • Environment.getExternalStorageDirectory(): Devuelve el directorio de almacenamiento externo primario.
    • Environment.getExternalStoragePublicDirectory(String type): Devuelve un directorio público para almacenar archivos de un tipo específico (por ejemplo, imágenes, música).

Lectura y Escritura de Archivos

Solicitud de Permisos

Antes de acceder al almacenamiento externo, necesitas solicitar los permisos necesarios en tu archivo AndroidManifest.xml:

<manifest xmlns:android="http://schemas.android.com/apk/res/android"
    package="com.example.myapp">

    <uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"/>
    <uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE"/>

    <application
        ... >
        ...
    </application>
</manifest>

Comprobación de Permisos en Tiempo de Ejecución

Desde Android 6.0 (nivel de API 23) en adelante, necesitas solicitar permisos en tiempo de ejecución:

if (ContextCompat.checkSelfPermission(this, Manifest.permission.WRITE_EXTERNAL_STORAGE)
        != PackageManager.PERMISSION_GRANTED) {
    ActivityCompat.requestPermissions(this, new String[]{Manifest.permission.WRITE_EXTERNAL_STORAGE}, REQUEST_CODE);
}

Escritura en el Almacenamiento Externo

Para escribir un archivo en el almacenamiento externo, sigue estos pasos:

  1. Verifica si el almacenamiento está disponible:
String state = Environment.getExternalStorageState();
if (!Environment.MEDIA_MOUNTED.equals(state)) {
    // Manejar error: el almacenamiento externo no está disponible
}
  1. Obtén el directorio de almacenamiento externo:
File externalStorageDir = Environment.getExternalStorageDirectory();
File myFile = new File(externalStorageDir, "myfile.txt");
  1. Escribe en el archivo:
try (FileOutputStream fos = new FileOutputStream(myFile)) {
    fos.write("¡Hola, Mundo!".getBytes());
} catch (IOException e) {
    e.printStackTrace();
}

Lectura desde el Almacenamiento Externo

Para leer un archivo desde el almacenamiento externo, sigue estos pasos:

  1. Verifica si el almacenamiento está disponible:
String state = Environment.getExternalStorageState();
if (!Environment.MEDIA_MOUNTED.equals(state) && !Environment.MEDIA_MOUNTED_READ_ONLY.equals(state)) {
    // Manejar error: el almacenamiento externo no está disponible o no es legible
}
  1. Obtén el directorio de almacenamiento externo:
File externalStorageDir = Environment.getExternalStorageDirectory();
File myFile = new File(externalStorageDir, "myfile.txt");
  1. Lee desde el archivo:
try (FileInputStream fis = new FileInputStream(myFile)) {
    int content;
    while ((content = fis.read()) != -1) {
        System.out.print((char) content);
    }
} catch (IOException e) {
    e.printStackTrace();
}

Mejores Prácticas

  • Verifica la Disponibilidad del Almacenamiento: Siempre verifica si el almacenamiento externo está disponible antes de leer o escribir.
  • Maneja los Permisos: Maneja correctamente las solicitudes de permisos y sus respuestas.
  • Usa Directorios Públicos: Almacena archivos en directorios públicos cuando necesiten ser accesibles por otras aplicaciones o el usuario.
  • Gestiona el Almacenamiento Eficientemente: Limpia los archivos innecesarios para evitar llenar el almacenamiento.

Resumen

El almacenamiento externo es una característica poderosa en Android que permite a las aplicaciones almacenar y compartir archivos grandes. Al comprender los tipos de almacenamiento externo, manejar correctamente los permisos y seguir las mejores prácticas, puedes gestionar eficazmente el almacenamiento externo en tus aplicaciones Android. Siempre asegúrate de verificar la disponibilidad del almacenamiento y manejar los permisos adecuadamente para proporcionar una experiencia de usuario sin problemas.
