Pasando Datos Entre Actividades ►

Navegar entre actividades es un concepto fundamental en el desarrollo de Android. Este tema te guiará a través de los conceptos básicos de la navegación de actividades, desde la creación de intents hasta el paso de datos entre actividades. Al final de esta sección, serás capaz de crear transiciones fluidas entre diferentes pantallas en tu aplicación de Android.

Introducción a las Actividades

  • Actividad: Una sola cosa enfocada que el usuario puede hacer. Casi todas las actividades interactúan con el usuario, por lo que la clase Activity se encarga de crear una ventana para ti en la que puedes colocar tu interfaz de usuario con setContentView(View).
  • Intent: Una descripción abstracta de una operación a realizar. Se puede usar con startActivity para lanzar una actividad.

Crear y Comenzar una Actividad

Para navegar de una actividad a otra, necesitas crear un intent y usarlo para iniciar la nueva actividad.

Ejemplo: Iniciar una Nueva Actividad

  1. Crear una nueva Actividad: Supongamos que tienes una actividad principal (MainActivity) y quieres navegar a una segunda actividad (SecondActivity).
// MainActivity.java
package com.example.myfirstapp;

import android.content.Intent;
import android.os.Bundle;
import android.view.View;
import androidx.appcompat.app.AppCompatActivity;

public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
    }

    public void navigateToSecondActivity(View view) {
        Intent intent = new Intent(this, SecondActivity.class);
        startActivity(intent);
    }
}
  1. Definir la nueva Actividad en el Manifest: Asegúrate de que SecondActivity esté declarada en tu AndroidManifest.xml.
<!-- AndroidManifest.xml -->
<application
    ... >
    <activity android:name=".MainActivity">
        <intent-filter>
            <action android:name="android.intent.action.MAIN" />
            <category android:name="android.intent.category.LAUNCHER" />
        </intent-filter>
    </activity>
    <activity android:name=".SecondActivity"></activity>
</application>
  1. Crear la nueva clase de Actividad: Implementa la SecondActivity.
// SecondActivity.java
package com.example.myfirstapp;

import android.os.Bundle;
import androidx.appcompat.app.AppCompatActivity;

public class SecondActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_second);
    }
}
  1. Crear el layout para la nueva Actividad: Define la interfaz de usuario para SecondActivity en activity_second.xml.
<!-- activity_second.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="¡Bienvenido a la Segunda Actividad!" />
</LinearLayout>

Pasar Datos Entre Actividades

A menudo, necesitas pasar datos de una actividad a otra. Esto se puede hacer usando extras de Intent.

Ejemplo: Pasar Datos con Intent Extras

  1. Enviar Datos desde MainActivity:
// MainActivity.java
public void navigateToSecondActivity(View view) {
    Intent intent = new Intent(this, SecondActivity.class);
    intent.putExtra("EXTRA_MESSAGE", "¡Hola, Segunda Actividad!");
    startActivity(intent);
}
  1. Recibir Datos en SecondActivity:
// SecondActivity.java
@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_second);

    Intent intent = getIntent();
    String message = intent.getStringExtra("EXTRA_MESSAGE");

    TextView textView = findViewById(R.id.textView);
    textView.setText(message);
}
  1. Actualizar el layout para incluir un TextView:
<!-- activity_second.xml -->
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    android:padding="16dp">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="¡Bienvenido a la Segunda Actividad!" />
</LinearLayout>

Navegar de Vuelta a la Actividad Anterior

Android maneja la navegación hacia atrás por ti, pero también puedes finalizar explícitamente una actividad.

Ejemplo: Finalizar una Actividad

// SecondActivity.java
public void finishActivity(View view) {
    finish();
}

Conclusión

Navegar entre actividades es una parte crucial del desarrollo de Android. Al entender cómo crear intents, iniciar actividades y pasar datos entre ellas, puedes crear aplicaciones más dinámicas e interactivas. Practica estos conceptos creando múltiples actividades y navegando entre ellas, pasando diferentes tipos de datos para familiarizarte con el proceso.
