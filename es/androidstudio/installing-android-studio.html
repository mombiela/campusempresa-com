Introducción

Android Studio es el Entorno de Desarrollo Integrado (IDE) oficial para el desarrollo de aplicaciones Android, basado en IntelliJ IDEA. Esta sección te guiará a través del proceso de instalación de Android Studio en tu computadora, asegurándote de tener todas las herramientas necesarias para comenzar a desarrollar aplicaciones Android.

Requisitos del Sistema

Antes de instalar Android Studio, asegúrate de que tu sistema cumpla con los siguientes requisitos:

  • Windows:

    • Microsoft® Windows® 7/8/10 (32 o 64 bits)
    • 4 GB de RAM mínimo, 8 GB de RAM recomendados
    • 2 GB de espacio disponible en disco mínimo, 4 GB recomendados (500 MB para el IDE + 1.5 GB para el SDK de Android y la imagen del sistema del emulador)
    • Resolución de pantalla mínima de 1280 x 800

  • macOS:

    • macOS 10.10 (Yosemite) o superior
    • 4 GB de RAM mínimo, 8 GB de RAM recomendados
    • 2 GB de espacio disponible en disco mínimo, 4 GB recomendados (500 MB para el IDE + 1.5 GB para el SDK de Android y la imagen del sistema del emulador)
    • Resolución de pantalla mínima de 1280 x 800

  • Linux:

    • Escritorio GNOME o KDE
    • Probado en Ubuntu® 14.04 LTS, Trusty Tahr (distribución de 64 bits capaz de ejecutar aplicaciones de 32 bits)
    • 4 GB de RAM mínimo, 8 GB de RAM recomendados
    • 2 GB de espacio disponible en disco mínimo, 4 GB recomendados (500 MB para el IDE + 1.5 GB para el SDK de Android y la imagen del sistema del emulador)
    • Resolución de pantalla mínima de 1280 x 800

Descargando Android Studio

  1. Visita el Sitio Web Oficial:

    • Ve a la página de descarga de Android Studio.

  2. Selecciona tu Sistema Operativo:

    • Elige la versión apropiada para tu sistema operativo (Windows, macOS o Linux).

  3. Descarga el Instalador:

    • Haz clic en el enlace de descarga para obtener el archivo del instalador.

Instalando Android Studio

Windows

  1. Ejecuta el Instalador:

    • Localiza el archivo .exe descargado y haz doble clic para ejecutarlo.

  2. Sigue el Asistente de Configuración:

    • Haz clic en "Next" para continuar a través del asistente de configuración.
    • Elige la ubicación de instalación y los componentes (Android Studio, Android SDK, Android Virtual Device).

  3. Completa la Instalación:

    • Haz clic en "Finish" una vez que la instalación esté completa.

macOS

  1. Abre el Archivo DMG:

    • Localiza el archivo .dmg descargado y haz doble clic para abrirlo.

  2. Arrastra y Suelta:

    • Arrastra y suelta el icono de Android Studio en la carpeta de Aplicaciones.

  3. Lanza Android Studio:

    • Abre la carpeta de Aplicaciones y haz doble clic en Android Studio para lanzarlo.

Linux

  1. Extrae el Archivo Tar:

    • Abre una terminal y navega al directorio donde se encuentra el archivo .tar.gz descargado.
    • Ejecuta el siguiente comando para extraer el archivo: 
      tar -xvf android-studio-ide-*.tar.gz

  2. Mueve al Directorio de Instalación:

    • Mueve la carpeta extraída al directorio de instalación deseado: 
      sudo mv android-studio /opt/

  3. Lanza Android Studio:

    • Navega al directorio bin y ejecuta el script studio.sh: 
      cd /opt/android-studio/bin
./studio.sh

Configuración Inicial

  1. Pantalla de Bienvenida:

    • Cuando lances Android Studio por primera vez, verás una pantalla de bienvenida. Haz clic en "Next" para comenzar el asistente de configuración.

  2. Instala Componentes del SDK:

    • El asistente de configuración te guiará a través del proceso de descarga e instalación de los componentes necesarios del SDK. Asegúrate de tener una conexión a internet estable.

  3. Configura Ajustes:

    • Elige los ajustes predeterminados a menos que tengas preferencias específicas. Haz clic en "Finish" para completar la configuración.

Conclusión

Siguiendo estos pasos, has instalado exitosamente Android Studio en tu computadora. Ahora estás listo para comenzar a desarrollar aplicaciones Android. En las siguientes secciones, exploraremos la interfaz de Android Studio, crearemos tu primer proyecto y profundizaremos en funciones más avanzadas.
