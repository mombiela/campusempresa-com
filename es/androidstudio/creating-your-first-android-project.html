Introducción

En esta sección, te guiaremos a través del proceso de creación de tu primer proyecto Android usando Android Studio. Este es un paso crucial para cualquier desarrollador Android, ya que sienta las bases para todo el trabajo de desarrollo futuro.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Android Studio instalado en tu computadora.
  • Comprensión básica de Java o Kotlin (los lenguajes principales utilizados para el desarrollo de Android).
  • Un dispositivo Android funcional o un emulador configurado en Android Studio.

Guía Paso a Paso para Crear Tu Primer Proyecto Android

Paso 1: Abre Android Studio

  • Lanza Android Studio desde el menú de aplicaciones o el acceso directo en el escritorio.

Paso 2: Inicia un Nuevo Proyecto de Android Studio

  • Haz clic en "Start a new Android Studio project" desde la pantalla de bienvenida.

Paso 3: Configura Tu Nuevo Proyecto

  • Name: Ingresa un nombre para tu proyecto (por ejemplo, "MyFirstApp").
  • Package name: Esto suele estar en el formato com.example.myfirstapp. Debe ser único para evitar conflictos.
  • Save location: Elige un directorio donde deseas guardar tu proyecto.
  • Language: Selecciona Java o Kotlin. Para este ejemplo, usaremos Kotlin.
  • Minimum API level: Elige la versión mínima de Android que tu aplicación soportará. Una elección común es API 21: Android 5.0 (Lollipop).

Paso 4: Selecciona el Factor de Forma y la Plantilla

  • Phone and Tablet: Asegúrate de que esté marcado.
  • Template: Selecciona "Empty Activity" para comenzar con una plantilla básica.

Paso 5: Configura la Actividad

  • Activity Name: El valor predeterminado es "MainActivity". Puedes dejarlo así.
  • Layout Name: El valor predeterminado es "activity_main". Este es el archivo XML donde diseñarás tu interfaz de usuario.
  • Generate Layout File: Asegúrate de que esté marcado.
  • Backwards Compatibility: Asegúrate de que "Use a legacy support library" esté marcado si necesitas soportar versiones anteriores de Android.

Paso 6: Finaliza y Construye el Proyecto

  • Haz clic en "Finish" para crear tu proyecto. Android Studio configurará el proyecto y construirá los archivos necesarios.

Entendiendo la Estructura del Proyecto

Una vez que tu proyecto esté creado, verás varios directorios y archivos. Aquí están los componentes clave:

  • app/src/main/java/com.example.myfirstapp/MainActivity.kt: Este es el archivo principal de actividad donde escribirás tu código Kotlin.
  • app/src/main/res/layout/activity_main.xml: Este es el archivo de diseño donde diseñarás tu interfaz de usuario.
  • app/src/main/AndroidManifest.xml: Este archivo contiene información esencial sobre tu aplicación, como sus componentes y permisos.

Escribiendo Tu Primer Código

Vamos a agregar un simple mensaje de "Hello, World!" a tu aplicación.

MainActivity.kt

package com.example.myfirstapp

import androidx.appcompat.app.AppCompatActivity
import android.os.Bundle
import android.widget.TextView

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        // Encuentra el TextView en el diseño
        val textView: TextView = findViewById(R.id.textView)
        // Establece el texto a "Hello, World!"
        textView.text = "Hello, World!"
    }
}

activity_main.xml

<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello, World!"
        android:layout_centerInParent="true"/>
</RelativeLayout>

Ejecutando Tu Aplicación

  • Conecta tu dispositivo Android o inicia el emulador.
  • Haz clic en el botón "Run" (icono de reproducción verde) en Android Studio.
  • Selecciona tu dispositivo o emulador de la lista.
  • Android Studio construirá e instalará la aplicación en tu dispositivo/emulador. Deberías ver "Hello, World!" en la pantalla.

Conclusión

¡Felicidades! Has creado y ejecutado con éxito tu primer proyecto Android. Este paso fundamental es crucial a medida que continúas desarrollando aplicaciones más complejas. En las siguientes secciones, profundizaremos en los componentes de Android, el diseño de la interfaz de usuario y las características avanzadas para mejorar tus habilidades en el desarrollo de aplicaciones.
