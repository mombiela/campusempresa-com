Visión General

Delphi es un potente Entorno de Desarrollo Integrado (IDE) para el lenguaje de programación Object Pascal. Es ampliamente utilizado para desarrollar aplicaciones de Windows, pero también soporta el desarrollo multiplataforma para macOS, iOS, Android y Linux. Object Pascal, el lenguaje utilizado en Delphi, es una extensión del lenguaje Pascal, diseñado para soportar la programación orientada a objetos.

Conceptos Clave

¿Qué es Delphi?

  • IDE: Delphi es un IDE que proporciona herramientas para diseñar, codificar, depurar y desplegar aplicaciones.
  • RAD: Las capacidades de Desarrollo Rápido de Aplicaciones (RAD) permiten una rápida creación de prototipos y desarrollo.
  • Multiplataforma: Soporta múltiples plataformas, incluyendo Windows, macOS, iOS, Android y Linux.

¿Qué es Object Pascal?

  • Extensión de Pascal: Object Pascal extiende el lenguaje Pascal con características orientadas a objetos.
  • Tipado Fuerte: Asegura la seguridad de tipos y reduce los errores en tiempo de ejecución.
  • Sintaxis Legible: Conocido por su sintaxis clara y legible, lo que lo convierte en una buena opción para principiantes.

Configuración de Delphi

  1. Descargar e Instalar: Obtén Delphi desde el sitio web oficial de Embarcadero y sigue las instrucciones de instalación.
  2. Primer Proyecto: Crea un nuevo proyecto para familiarizarte con el diseño del IDE y las características básicas.

Sintaxis y Estructura Básica

Ejemplo de Hola Mundo

program HelloWorld;

begin
  WriteLn('¡Hola, Mundo!');
end.
  • program: Declara el nombre del programa.
  • begin...end: Marca el inicio y el fin del bloque del programa.
  • WriteLn: Muestra texto en la consola.

Variables y Tipos de Datos

var
  age: Integer;
  name: String;
begin
  age := 25;
  name := 'Alice';
  WriteLn('Nombre: ', name, ', Edad: ', age);
end.
  • var: Declara variables.
  • Integer, String: Tipos de datos comunes.
  • :=: Operador de asignación.

Estructuras de Control

If-Else

var
  score: Integer;
begin
  score := 85;
  if score >= 90 then
    WriteLn('Calificación: A')
  else if score >= 80 then
    WriteLn('Calificación: B')
  else
    WriteLn('Calificación: C');
end.

Bucle For

var
  i: Integer;
begin
  for i := 1 to 10 do
    WriteLn('Número: ', i);
end.

Programación Orientada a Objetos en Object Pascal

Clases y Objetos

type
  TPerson = class
    private
      FName: String;
      FAge: Integer;
    public
      constructor Create(Name: String; Age: Integer);
      procedure DisplayInfo;
  end;

constructor TPerson.Create(Name: String; Age: Integer);
begin
  FName := Name;
  FAge := Age;
end;

procedure TPerson.DisplayInfo;
begin
  WriteLn('Nombre: ', FName, ', Edad: ', FAge);
end;

var
  person: TPerson;
begin
  person := TPerson.Create('Alice', 25);
  person.DisplayInfo;
end.
  • type: Define un nuevo tipo de clase.
  • private, public: Especificadores de acceso.
  • constructor: Método especial para inicializar objetos.
  • procedure: Define un método.

Conclusión

En este tema introductorio, cubrimos los conceptos básicos de Delphi y Object Pascal, incluyendo la configuración del entorno, la comprensión de la sintaxis básica y el inicio con la programación orientada a objetos. A medida que avances en el curso, profundizarás en características y técnicas avanzadas, lo que te permitirá construir aplicaciones robustas y eficientes utilizando Delphi y Object Pascal.
