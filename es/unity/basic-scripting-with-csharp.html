Introducción

Bienvenido al curso de "Scripting Básico con C#" para Unity. Este curso está diseñado para llevarte de un nivel principiante a un nivel avanzado en scripting con C# dentro del entorno de Unity. Al final de este curso, serás capaz de crear juegos interactivos y dinámicos utilizando scripts en C#.

Comenzando con C# en Unity

Configurando tu Entorno

  • Instalar Unity Hub y Unity Editor: Asegúrate de tener instalada la última versión de Unity.
  • Crear un Nuevo Proyecto: Abre Unity Hub, haz clic en "New Project" y selecciona la plantilla "3D".
  • Familiarízate con la Interfaz de Unity: Entiende las ventanas de Scene, Game, Hierarchy, Project e Inspector.

Tu Primer Script

  • Creando un Script:

    1. Haz clic derecho en la ventana de Project.
    2. Selecciona Create > C# Script.
    3. Nombra tu script HelloWorld.

  • Editando el Script:

    using UnityEngine;

public class HelloWorld : MonoBehaviour
{
    // Start se llama antes de la primera actualización del frame
    void Start()
    {
        Debug.Log("Hello, World!");
    }

    // Update se llama una vez por frame
    void Update()
    {

    }
}

  • Adjuntando el Script a un GameObject:

    1. Crea un nuevo GameObject en la ventana de Hierarchy.
    2. Arrastra y suelta el script HelloWorld sobre el GameObject.

Conceptos Clave

  • MonoBehaviour: La clase base de la cual deriva cada script de Unity.
  • Métodos Start() y Update():
    • Start(): Se llama antes de la primera actualización del frame.
    • Update(): Se llama una vez por frame.

Variables y Tipos de Datos

Declarando Variables

  • Sintaxis: 
    int score = 0;
float speed = 5.5f;
string playerName = "Player1";
bool isGameOver = false;

Tipos de Datos Comunes

  • int: Números enteros.
  • float: Números de punto flotante.
  • string: Texto.
  • bool: Valores booleanos (true/false).

Ejemplo: Usando Variables en Unity

using UnityEngine;

public class Player : MonoBehaviour
{
    public int score = 0;
    public float speed = 5.5f;
    public string playerName = "Player1";
    public bool isGameOver = false;

    void Start()
    {
        Debug.Log("Player Name: " + playerName);
        Debug.Log("Initial Score: " + score);
    }

    void Update()
    {
        if (isGameOver)
        {
            Debug.Log("Game Over!");
        }
    }
}

Estructuras de Control

Sentencias Condicionales

  • if-else: 
    if (score >= 100)
{
    Debug.Log("You win!");
}
else
{
    Debug.Log("Keep trying!");
}

Bucles

  • Bucle for:

    for (int i = 0; i < 10; i++)
{
    Debug.Log("Count: " + i);
}

  • Bucle while:

    int count = 0;
while (count < 10)
{
    Debug.Log("Count: " + count);
    count++;
}

Funciones y Métodos

Creando Funciones

  • Sintaxis: 
    void PrintMessage(string message)
{
    Debug.Log(message);
}

Llamando Funciones

  • Ejemplo: 
    void Start()
{
    PrintMessage("Hello from a function!");
}

Conceptos Específicos de Unity

GameObject y Componentes

  • GameObject: Los objetos fundamentales en las escenas de Unity.
  • Componentes: Scripts, física, renderizado y otras funcionalidades adjuntas a los GameObjects.

Componente Transform

  • Posición, Rotación, Escala: 
    void Update()
{
    transform.position = new Vector3(0, 1, 0);
    transform.rotation = Quaternion.Euler(0, 45, 0);
    transform.localScale = new Vector3(1, 2, 1);
}

Manejo de Entradas

  • Entrada de Teclado: 
    void Update()
{
    if (Input.GetKeyDown(KeyCode.Space))
    {
        Debug.Log("Space key was pressed.");
    }
}

Conclusión

En este tema, cubrimos los conceptos básicos de scripting con C# en Unity. Comenzamos con la configuración del entorno y la creación de tu primer script. Luego exploramos variables, estructuras de control, funciones y conceptos específicos de Unity como GameObjects y componentes. Al entender estos fundamentos, ahora estás equipado para comenzar a construir proyectos de Unity más complejos e interactivos. Continúa practicando y experimentando con estos conceptos para profundizar tu comprensión y mejorar tus habilidades. ¡Feliz codificación!
