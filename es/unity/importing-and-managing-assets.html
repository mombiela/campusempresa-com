Introducción

En Unity, los activos son los componentes básicos de tu juego. Incluyen modelos 3D, texturas, archivos de audio, scripts y más. Importar y gestionar adecuadamente estos activos es crucial para un desarrollo de juegos eficiente.

Importación de Activos

Importación desde la Tienda de Activos de Unity

  • Tienda de Activos de Unity: Un mercado donde puedes encontrar activos gratuitos y de pago.
    1. Abre la ventana de la Tienda de Activos: Ventana > Tienda de Activos.
    2. Busca el activo deseado.
    3. Haz clic en el activo y selecciona Descargar o Comprar.
    4. Una vez descargado, haz clic en Importar para llevar el activo a tu proyecto.

Importación de Activos Personalizados

  • Arrastrar y Soltar: Simplemente arrastra archivos desde tu explorador de archivos a la carpeta Assets en Unity.
  • Usando el Diálogo de Importación:
    1. Haz clic derecho en la carpeta Assets en la ventana del Proyecto.
    2. Selecciona Importar Nuevo Activo....
    3. Elige el archivo que deseas importar y haz clic en Importar.

Tipos de Archivos Soportados

  • Modelos 3D: .fbx, .obj, .dae, etc.
  • Texturas: .png, .jpg, .tga, etc.
  • Audio: .mp3, .wav, .ogg, etc.
  • Scripts: .cs (scripts en C#).

Gestión de Activos

Organización de la Carpeta de Activos

  • Estructura de Carpetas: Crea una estructura de carpetas lógica para mantener tus activos organizados.
    • Ejemplo: 
      Assets/
├── Models/
├── Textures/
├── Audio/
├── Scripts/
├── Prefabs/
└── Scenes/
  • Convenciones de Nombres: Usa convenciones de nombres claras y consistentes para tus activos.

Etiquetas de Activos

  • Uso de Etiquetas: Las etiquetas te ayudan a categorizar y encontrar activos rápidamente.
    1. Selecciona el activo en la ventana del Proyecto.
    2. En el Inspector, haz clic en el campo Etiquetas de Activos.
    3. Añade o crea nuevas etiquetas.

Paquetes de Activos

  • Creación de Paquetes de Activos: Los Paquetes de Activos te permiten empaquetar activos juntos para una carga eficiente.
    1. Selecciona los activos que deseas empaquetar.
    2. En el Inspector, establece el campo AssetBundle a un nombre de paquete nuevo o existente.
    3. Usa el método BuildPipeline.BuildAssetBundles para crear los paquetes.

Configuración de Importación

  • Configuración de Importación de Modelos:
    • Factor de Escala: Ajusta el tamaño del modelo importado.
    • Compresión de Malla: Optimiza la malla para el rendimiento.
    • Lectura/Escritura Habilitada: Permite que la malla sea modificada en tiempo de ejecución.
  • Configuración de Importación de Texturas:
    • Tipo de Textura: Elige el tipo apropiado (por ejemplo, Predeterminado, Sprite, Mapa normal).
    • Tamaño Máximo: Establece la resolución máxima.
    • Compresión: Elige el método de compresión para equilibrar calidad y rendimiento.
  • Configuración de Importación de Audio:
    • Tipo de Carga: Elige entre Descomprimir al Cargar, Comprimido en Memoria o Streaming.
    • Formato de Compresión: Selecciona el formato (por ejemplo, PCM, Vorbis).
// Ejemplo de configuración de importación para una textura
using UnityEditor;
using UnityEngine;

public class TextureImportSettings : AssetPostprocessor
{
    void OnPreprocessTexture()
    {
        TextureImporter importer = (TextureImporter)assetImporter;
        importer.textureType = TextureImporterType.Default;
        importer.maxTextureSize = 1024;
        importer.textureCompression = TextureImporterCompression.Compressed;
    }
}

Conclusión

Importar y gestionar activos eficientemente es esencial para un desarrollo de juegos fluido en Unity. Al comprender cómo importar activos de varias fuentes, organizarlos efectivamente y configurar sus ajustes de importación, puedes asegurarte de que tu proyecto se mantenga manejable y funcione bien. Practica estas técnicas para volverte competente en el manejo de activos en Unity.
