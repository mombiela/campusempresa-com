Introducción

Los Editores e Inspectores Personalizados en Unity te permiten crear interfaces personalizadas para tus scripts y componentes dentro del Editor de Unity. Esto puede mejorar enormemente tu flujo de trabajo al proporcionar herramientas más intuitivas y poderosas para el desarrollo de juegos.

¿Por qué Usar Editores e Inspectores Personalizados?

  • Mejora del Flujo de Trabajo: Adapta el Editor de Unity para que se ajuste mejor a tus necesidades específicas.
  • Mejor Usabilidad: Crea interfaces más amigables para scripts complejos.
  • Depuración y Pruebas: Más fácil de visualizar y manipular datos durante el desarrollo.

Empezando con Editores Personalizados

Creando un Editor Personalizado Básico

Para crear un editor personalizado, necesitas:

  1. Crear un nuevo script en la carpeta Editor.
  2. Extender la clase Editor.
  3. Usar el atributo CustomEditor para vincular el editor a un componente específico.
// ExampleComponent.cs
using UnityEngine;

public class ExampleComponent : MonoBehaviour
{
    public float exampleFloat;
    public string exampleString;
}

// ExampleComponentEditor.cs
using UnityEditor;
using UnityEngine;

[CustomEditor(typeof(ExampleComponent))]
public class ExampleComponentEditor : Editor
{
    public override void OnInspectorGUI()
    {
        ExampleComponent exampleComponent = (ExampleComponent)target;

        exampleComponent.exampleFloat = EditorGUILayout.FloatField("Example Float", exampleComponent.exampleFloat);
        exampleComponent.exampleString = EditorGUILayout.TextField("Example String", exampleComponent.exampleString);

        if (GUILayout.Button("Press Me"))
        {
            Debug.Log("Button Pressed!");
        }

        // Asegúrate de que cualquier cambio se guarde
        if (GUI.changed)
        {
            EditorUtility.SetDirty(exampleComponent);
        }
    }
}

Conceptos Clave

  • EditorGUILayout: Proporciona un conjunto de controles que se pueden usar para crear campos de inspector personalizados.
  • GUILayout: Se usa para controles basados en diseño como botones y etiquetas.
  • EditorUtility.SetDirty: Marca el objeto objetivo como sucio, asegurando que los cambios se guarden.

Editores Personalizados Avanzados

Dibujadores de Propiedades Personalizadas

Los Dibujadores de Propiedades Personalizadas te permiten crear UI personalizada para campos individuales en tus scripts.

// CustomProperty.cs
using UnityEngine;

public class CustomProperty : MonoBehaviour
{
    public Range range;
}

[System.Serializable]
public class Range
{
    public float min;
    public float max;
}

// RangeDrawer.cs
using UnityEditor;
using UnityEngine;

[CustomPropertyDrawer(typeof(Range))]
public class RangeDrawer : PropertyDrawer
{
    public override void OnGUI(Rect position, SerializedProperty property, GUIContent label)
    {
        EditorGUI.BeginProperty(position, label, property);

        SerializedProperty min = property.FindPropertyRelative("min");
        SerializedProperty max = property.FindPropertyRelative("max");

        float minValue = min.floatValue;
        float maxValue = max.floatValue;

        EditorGUI.MinMaxSlider(position, label, ref minValue, ref maxValue, 0, 100);

        min.floatValue = minValue;
        max.floatValue = maxValue;

        EditorGUI.EndProperty();
    }
}

Ventanas del Editor

Crear ventanas del editor personalizadas puede proporcionar herramientas más complejas e independientes dentro de Unity.

// CustomEditorWindow.cs
using UnityEditor;
using UnityEngine;

public class CustomEditorWindow : EditorWindow
{
    [MenuItem("Window/Custom Editor Window")]
    public static void ShowWindow()
    {
        GetWindow<CustomEditorWindow>("Custom Editor Window");
    }

    private void OnGUI()
    {
        GUILayout.Label("This is a custom editor window", EditorStyles.boldLabel);

        if (GUILayout.Button("Do Something"))
        {
            Debug.Log("Button in custom window pressed!");
        }
    }
}

Mejores Prácticas

  • Manténlo Simple: Evita complicar demasiado la UI; mantenla intuitiva.
  • Rendimiento: Asegúrate de que tus editores personalizados sean eficientes y no degraden el rendimiento.
  • Consistencia: Mantén una apariencia y sensación consistentes con el resto del Editor de Unity.

Conclusión

Los Editores e Inspectores Personalizados en Unity son herramientas poderosas que pueden mejorar significativamente tu flujo de trabajo de desarrollo. Al crear interfaces personalizadas adaptadas a tus necesidades, puedes hacer que tu proceso de desarrollo sea más eficiente y agradable. Comienza con editores personalizados básicos y explora gradualmente características avanzadas como dibujadores de propiedades personalizadas y ventanas del editor para aprovechar al máximo las capacidades de la escritura de scripts del editor de Unity.
