Introducción a GraphQL

  • ¿Qué es GraphQL?

    • Un lenguaje de consulta para tu API.
    • Proporciona una descripción completa y comprensible de los datos en tu API.
    • Permite a los clientes solicitar exactamente lo que necesitan y nada más.

  • ¿Por qué usar GraphQL?

    • Obtención eficiente de datos.
    • Esquema fuertemente tipado.
    • Datos en tiempo real con suscripciones.
    • Mejor experiencia para desarrolladores.

Configuración del Entorno

  • Requisitos Previos

    • Conocimientos básicos de JavaScript/TypeScript.
    • Familiaridad con Node.js y Express.
    • Comprensión de frameworks de front-end como React.

  • Herramientas y Bibliotecas

    • Node.js
    • Express.js
    • Apollo Server
    • GraphQL

Creando el Backend con GraphQL

  • Configuración del Servidor

    mkdir graphql-fullstack
cd graphql-fullstack
npm init -y
npm install express apollo-server-express graphql

  • Creando el Servidor Express

    const express = require('express');
const { ApolloServer, gql } = require('apollo-server-express');

const app = express();

const typeDefs = gql`
  type Query {
    hello: String
  }
`;

const resolvers = {
  Query: {
    hello: () => '¡Hola mundo!',
  },
};

const server = new ApolloServer({ typeDefs, resolvers });
server.applyMiddleware({ app });

app.listen({ port: 4000 }, () =>
  console.log(`Servidor listo en http://localhost:4000${server.graphqlPath}`)
);

  • Definiendo el Esquema

    type Query {
  hello: String
  users: [User]
}

type User {
  id: ID!
  name: String!
  email: String!
}

  • Creando Resolvers

    const resolvers = {
  Query: {
    hello: () => '¡Hola mundo!',
    users: () => [
      { id: 1, name: 'John Doe', email: 'john@example.com' },
      { id: 2, name: 'Jane Doe', email: 'jane@example.com' },
    ],
  },
};

Conectando a una Base de Datos

  • Usando MongoDB con Mongoose

    npm install mongoose

  • Configuración de Modelos Mongoose

    const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost:27017/graphql', {
  useNewUrlParser: true,
  useUnifiedTopology: true,
});

const UserSchema = new mongoose.Schema({
  name: String,
  email: String,
});

const User = mongoose.model('User', UserSchema);

  • Actualizando Resolvers para Usar la Base de Datos

    const resolvers = {
  Query: {
    hello: () => '¡Hola mundo!',
    users: async () => await User.find(),
  },
};

Construyendo el Frontend con React y Apollo Client

  • Configuración de la Aplicación React

    npx create-react-app client
cd client
npm install @apollo/client graphql

  • Creando Apollo Client

    import React from 'react';
import { ApolloClient, InMemoryCache, ApolloProvider } from '@apollo/client';
import App from './App';

const client = new ApolloClient({
  uri: 'http://localhost:4000/graphql',
  cache: new InMemoryCache(),
});

const Root = () => (
  <ApolloProvider client={client}>
    <App />
  </ApolloProvider>
);

export default Root;

  • Obteniendo Datos con Apollo Client

    import React from 'react';
import { useQuery, gql } from '@apollo/client';

const GET_USERS = gql`
  query GetUsers {
    users {
      id
      name
      email
    }
  }
`;

const Users = () => {
  const { loading, error, data } = useQuery(GET_USERS);

  if (loading) return <p>Cargando...</p>;
  if (error) return <p>Error :(</p>;

  return (
    <ul>
      {data.users.map(({ id, name, email }) => (
        <li key={id}>
          {name} - {email}
        </li>
      ))}
    </ul>
  );
};

export default Users;

Características Avanzadas

  • Mutaciones

    type Mutation {
  addUser(name: String!, email: String!): User
}

    const resolvers = {
  Mutation: {
    addUser: async (_, { name, email }) => {
      const user = new User({ name, email });
      await user.save();
      return user;
    },
  },
};

    import React, { useState } from 'react';
import { useMutation, gql } from '@apollo/client';

const ADD_USER = gql`
  mutation AddUser($name: String!, $email: String!) {
    addUser(name: $name, email: $email) {
      id
      name
      email
    }
  }
`;

const AddUser = () => {
  let input;
  const [addUser, { data }] = useMutation(ADD_USER);

  return (
    <div>
      <form
        onSubmit={e => {
          e.preventDefault();
          addUser({ variables: { name: input.value, email: input.value } });
          input.value = '';
        }}
      >
        <input
          ref={node => {
            input = node;
          }}
        />
        <button type="submit">Agregar Usuario</button>
      </form>
    </div>
  );
};

export default AddUser;

  • Suscripciones

    type Subscription {
  userAdded: User
}

    const { PubSub } = require('apollo-server');
const pubsub = new PubSub();

const resolvers = {
  Subscription: {
    userAdded: {
      subscribe: () => pubsub.asyncIterator(['USER_ADDED']),
    },
  },
  Mutation: {
    addUser: async (_, { name, email }) => {
      const user = new User({ name, email });
      await user.save();
      pubsub.publish('USER_ADDED', { userAdded: user });
      return user;
    },
  },
};

    import React from 'react';
import { useSubscription, gql } from '@apollo/client';

const USER_ADDED = gql`
  subscription OnUserAdded {
    userAdded {
      id
      name
      email
    }
  }
`;

const UserAdded = () => {
  const { data, loading } = useSubscription(USER_ADDED);

  if (loading) return <p>Cargando...</p>;

  return (
    <div>
      <p>Nuevo Usuario Agregado:</p>
      <p>{data.userAdded.name} - {data.userAdded.email}</p>
    </div>
  );
};

export default UserAdded;

Conclusión

Construir una aplicación full-stack con GraphQL implica configurar un servidor backend con un esquema y resolvers bien definidos, conectarse a una base de datos y crear una aplicación frontend que interactúe con la API GraphQL. Al aprovechar el poder de GraphQL, los desarrolladores pueden crear aplicaciones eficientes, escalables y flexibles que proporcionan una excelente experiencia de usuario. Este curso ha cubierto desde los conceptos básicos hasta las características avanzadas, proporcionando una base sólida para construir aplicaciones full-stack robustas con GraphQL.
