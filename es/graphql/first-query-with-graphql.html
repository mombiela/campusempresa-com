En esta sección, aprenderemos a escribir y ejecutar nuestra primera consulta utilizando GraphQL. Cubriremos los conceptos básicos de la consulta de datos, entendiendo la estructura de una consulta GraphQL y cómo interpretar los resultados.

Conceptos Clave

  • Consulta GraphQL: Una solicitud de datos que especifica exactamente la información que necesitas.
  • Campos: Las piezas específicas de datos que solicitas en una consulta.
  • Argumentos: Parámetros que puedes pasar a los campos para filtrar o modificar los datos devueltos.
  • Respuesta: Los datos devueltos por el servidor en respuesta a tu consulta.

Escribiendo Tu Primera Consulta

Para comenzar con GraphQL, necesitas entender la estructura básica de una consulta. Una consulta GraphQL se escribe en una sintaxis específica que te permite especificar exactamente los datos que necesitas.

Estructura Básica de una Consulta

Una consulta GraphQL simple podría verse así:

{
  user(id: "1") {
    id
    name
    email
  }
}
  • user(id: "1"): Este es el campo que estamos consultando. Estamos pidiendo un usuario con el ID "1".
  • id, name, email: Estas son las piezas específicas de datos que queremos recuperar para el usuario.

Explicación

  • Llaves {}: Encierran toda la consulta.
  • Nombre del Campo user: El tipo de datos que estamos consultando.
  • Argumentos (id: "1"): Parámetros para filtrar los datos.
  • Sub-campos id, name, email: Puntos de datos específicos que queremos recuperar.

Ejemplo de Consulta

Escribamos una consulta para obtener una lista de libros con sus títulos y autores:

{
  books {
    title
    author {
      name
    }
  }
}
  • books: El campo que estamos consultando para obtener una lista de libros.
  • title: El título de cada libro.
  • author: Un campo anidado para obtener los detalles del autor.
  • name: El nombre del autor.

Ejecutando la Consulta

Para ejecutar una consulta GraphQL, normalmente utilizas un cliente GraphQL o un entorno de desarrollo integrado (IDE) como GraphiQL o Apollo Studio. Aquí te explicamos cómo hacerlo:

  1. Abre GraphiQL o Apollo Studio.
  2. Ingresa la Consulta: Copia y pega tu consulta en el editor.
  3. Ejecuta la Consulta: Haz clic en el botón "Run" para ejecutar la consulta.

Ejemplo de Respuesta

Cuando ejecutas la consulta anterior, podrías obtener una respuesta como esta:

{
  "data": {
    "books": [
      {
        "title": "El Gran Gatsby",
        "author": {
          "name": "F. Scott Fitzgerald"
        }
      },
      {
        "title": "1984",
        "author": {
          "name": "George Orwell"
        }
      }
    ]
  }
}
  • data: El campo raíz de la respuesta.
  • books: La lista de libros devueltos por la consulta.
  • title y author.name: Los puntos de datos específicos que solicitamos.

Conclusión

En esta sección, cubrimos los conceptos básicos de escribir y ejecutar una consulta GraphQL. Aprendimos sobre la estructura de una consulta, cómo especificar los datos que necesitamos y cómo interpretar la respuesta. Al dominar estos conceptos básicos, ahora estás listo para explorar características más avanzadas de GraphQL, como mutaciones y suscripciones.
