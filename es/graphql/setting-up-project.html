En esta sección, cubriremos cómo configurar un proyecto GraphQL desde cero. Esto incluye la instalación de herramientas necesarias, la inicialización del proyecto y la configuración de la estructura básica. Al final de esta sección, tendrás un servidor GraphQL funcional sobre el cual podrás construir en lecciones posteriores.

Requisitos Previos

Antes de comenzar, asegúrate de tener lo siguiente instalado en tu máquina:

  • Node.js (versión 12.x o posterior)
  • npm (Node Package Manager) o yarn

Puedes verificar si Node.js y npm están instalados ejecutando los siguientes comandos en tu terminal:

node -v
npm -v

Inicializando el Proyecto

  1. Crea un nuevo directorio para tu proyecto:

    mkdir graphql-project
cd graphql-project

  2. Inicializa un nuevo proyecto de Node.js:

    npm init -y

    Este comando creará un archivo package.json con configuraciones predeterminadas.

Instalando Dependencias

Necesitaremos varios paquetes para configurar nuestro servidor GraphQL. Los paquetes principales son express para nuestro servidor, graphql para la funcionalidad de GraphQL y express-graphql para integrar GraphQL con Express.

  1. Instala los paquetes necesarios: 
    npm install express graphql express-graphql

Configurando el Servidor

  1. Crea un nuevo archivo llamado server.js en el directorio raíz:

    touch server.js

  2. Abre server.js y agrega el siguiente código:

    const express = require('express');
const { graphqlHTTP } = require('express-graphql');
const { buildSchema } = require('graphql');

// Construye un esquema, usando el lenguaje de esquemas de GraphQL
const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

// El root proporciona una función de resolución para cada endpoint de la API
const root = {
  hello: () => {
    return '¡Hola mundo!';
  },
};

const app = express();
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(4000, () => console.log('Ahora navega a localhost:4000/graphql'));

Explicación del Código

  • Importando Módulos:

    const express = require('express');
const { graphqlHTTP } = require('express-graphql');
const { buildSchema } = require('graphql');

    Importamos los módulos necesarios para configurar nuestro servidor e integración con GraphQL.

  • Definiendo el Esquema:

    const schema = buildSchema(`
  type Query {
    hello: String
  }
`);

    Definimos un esquema simple con una única consulta hello que devuelve un String.

  • Creando el Resolver Root:

    const root = {
  hello: () => {
    return '¡Hola mundo!';
  },
};

    La función de resolución para la consulta hello devuelve la cadena "¡Hola mundo!".

  • Configurando el Servidor Express:

    const app = express();
app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true,
}));

app.listen(4000, () => console.log('Ahora navega a localhost:4000/graphql'));

    Configuramos un servidor Express y configuramos el endpoint /graphql para usar express-graphql con nuestro esquema y resolver root. También habilitamos la interfaz GraphiQL para probar nuestras consultas GraphQL.

Ejecutando el Servidor

  1. Inicia el servidor ejecutando:

    node server.js

  2. Abre tu navegador y navega a:

    http://localhost:4000/graphql

    Deberías ver la interfaz de GraphiQL donde puedes probar tus consultas GraphQL.

Conclusión

En esta sección, hemos configurado exitosamente un servidor GraphQL básico usando Node.js, Express y el middleware express-graphql. Creamos un esquema y un resolver simples, y configuramos nuestro servidor para usar GraphiQL para pruebas. Esta configuración fundamental nos permitirá construir APIs GraphQL más complejas en las siguientes secciones.
