Introducción

Los tipos de datos son fundamentales para cualquier lenguaje de programación, y Go no es una excepción. Entender los tipos de datos es crucial para escribir código eficiente y libre de errores. En esta sección, cubriremos los tipos de datos básicos y avanzados en Go, su uso y cómo trabajar con ellos.

Tipos de Datos Básicos

Enteros

Go soporta tanto enteros con signo como sin signo en varios tamaños:

  • int y uint: Tamaño dependiente de la plataforma (32 o 64 bits)
  • int8, int16, int32, int64: Enteros con signo de 8, 16, 32 y 64 bits
  • uint8, uint16, uint32, uint64: Enteros sin signo de 8, 16, 32 y 64 bits
package main

import "fmt"

func main() {
    var a int = 10
    var b int8 = -128
    var c uint16 = 65535

    fmt.Println(a, b, c)
}

Números de Punto Flotante

Go soporta números de punto flotante:

  • float32: Números de punto flotante de 32 bits
  • float64: Números de punto flotante de 64 bits
package main

import "fmt"

func main() {
    var a float32 = 3.14
    var b float64 = 2.718281828459045

    fmt.Println(a, b)
}

Booleanos

El tipo bool representa un valor booleano, que puede ser true o false.

package main

import "fmt"

func main() {
    var isGoFun bool = true
    fmt.Println(isGoFun)
}

Cadenas

Las cadenas en Go son secuencias de bytes. Son inmutables, lo que significa que una vez creadas, no pueden ser cambiadas.

package main

import "fmt"

func main() {
    var greeting string = "¡Hola, Go!"
    fmt.Println(greeting)
}

Tipos de Datos Compuestos

Arreglos

Los arreglos son secuencias de tamaño fijo de elementos del mismo tipo.

package main

import "fmt"

func main() {
    var arr [5]int = [5]int{1, 2, 3, 4, 5}
    fmt.Println(arr)
}

Slices

Los slices son más flexibles que los arreglos. Son de tamaño dinámico y proporcionan operaciones más poderosas y convenientes.

package main

import "fmt"

func main() {
    var slice []int = []int{1, 2, 3, 4, 5}
    fmt.Println(slice)
}

Mapas

Los mapas son colecciones de pares clave-valor. Las claves deben ser únicas y de un tipo comparable.

package main

import "fmt"

func main() {
    var m map[string]int = map[string]int{"uno": 1, "dos": 2}
    fmt.Println(m)
}

Estructuras

Las estructuras son colecciones de campos. Se utilizan para agrupar datos.

package main

import "fmt"

type Persona struct {
    Nombre string
    Edad   int
}

func main() {
    var p Persona = Persona{Nombre: "Alicia", Edad: 30}
    fmt.Println(p)
}

Tipos de Datos Avanzados

Punteros

Los punteros contienen la dirección de memoria de una variable. Son útiles para la manipulación eficiente de datos y para evitar la copia de estructuras grandes.

package main

import "fmt"

func main() {
    var a int = 10
    var p *int = &a

    fmt.Println("Valor de a:", a)
    fmt.Println("Dirección de a:", p)
    fmt.Println("Valor en la dirección p:", *p)
}

Interfaces

Las interfaces definen un conjunto de firmas de métodos. Cualquier tipo que implemente estos métodos satisface la interfaz.

package main

import "fmt"

type Hablador interface {
    Hablar() string
}

type Perro struct{}

func (d Perro) Hablar() string {
    return "¡Guau!"
}

func main() {
    var h Hablador = Perro{}
    fmt.Println(h.Hablar())
}

Conclusión

Entender los tipos de datos en Go es esencial para escribir programas efectivos. Cubrimos tipos de datos básicos como enteros, números de punto flotante, booleanos y cadenas, así como tipos de datos compuestos como arreglos, slices, mapas y estructuras. También tocamos tipos de datos avanzados como punteros e interfaces. Dominar estos conceptos te ayudará a escribir código en Go más eficiente y robusto.
