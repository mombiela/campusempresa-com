Introducción a la Entrada/Salida de Archivos

La entrada/salida de archivos (File I/O) es un aspecto fundamental de la programación que permite a un programa leer y escribir en archivos. En Go, el paquete os proporciona una interfaz independiente de la plataforma para la funcionalidad del sistema operativo, incluida la entrada/salida de archivos.

Lectura de Archivos

Apertura de un Archivo

Para leer un archivo, primero necesitas abrirlo usando la función os.Open. Esta función devuelve un descriptor de archivo y un error.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.Open("example.txt")
    if err != nil {
        fmt.Println("Error abriendo el archivo:", err)
        return
    }
    defer file.Close()
}

Lectura del Contenido del Archivo

Una vez que el archivo está abierto, puedes leer su contenido usando varios métodos de los paquetes bufio y io/ioutil.

Usando bufio.Scanner

El bufio.Scanner es útil para leer un archivo línea por línea.

package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Open("example.txt")
    if err != nil {
        fmt.Println("Error abriendo el archivo:", err)
        return
    }
    defer file.Close()

    scanner := bufio.NewScanner(file)
    for scanner.Scan() {
        fmt.Println(scanner.Text())
    }

    if err := scanner.Err(); err != nil {
        fmt.Println("Error leyendo el archivo:", err)
    }
}

Usando ioutil.ReadFile

La función ioutil.ReadFile lee todo el contenido del archivo en memoria.

package main

import (
    "fmt"
    "io/ioutil"
)

func main() {
    content, err := ioutil.ReadFile("example.txt")
    if err != nil {
        fmt.Println("Error leyendo el archivo:", err)
        return
    }
    fmt.Println(string(content))
}

Escritura de Archivos

Creación o Apertura de un Archivo para Escritura

Para escribir en un archivo, puedes usar os.Create o os.OpenFile. La función os.Create crea un archivo si no existe o lo trunca si ya existe.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("example.txt")
    if err != nil {
        fmt.Println("Error creando el archivo:", err)
        return
    }
    defer file.Close()
}

Escritura de Contenido en un Archivo

Puedes escribir en un archivo usando los métodos Write o WriteString.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("example.txt")
    if err != nil {
        fmt.Println("Error creando el archivo:", err)
        return
    }
    defer file.Close()

    _, err = file.WriteString("¡Hola, Mundo!\n")
    if err != nil {
        fmt.Println("Error escribiendo en el archivo:", err)
    }
}

Usando `bufio.Writer`

Para escritura con búfer, puedes usar bufio.Writer.

package main

import (
    "bufio"
    "fmt"
    "os"
)

func main() {
    file, err := os.Create("example.txt")
    if err != nil {
        fmt.Println("Error creando el archivo:", err)
        return
    }
    defer file.Close()

    writer := bufio.NewWriter(file)
    _, err = writer.WriteString("¡Hola, Mundo con Búfer!\n")
    if err != nil {
        fmt.Println("Error escribiendo en el archivo:", err)
    }
    writer.Flush()
}

Operaciones Avanzadas de Archivos

Permisos de Archivos

Al crear o abrir archivos, puedes especificar permisos de archivo usando os.OpenFile.

package main

import (
    "fmt"
    "os"
)

func main() {
    file, err := os.OpenFile("example.txt", os.O_CREATE|os.O_WRONLY, 0644)
    if err != nil {
        fmt.Println("Error abriendo el archivo:", err)
        return
    }
    defer file.Close()

    _, err = file.WriteString("¡Hola con permisos!\n")
    if err != nil {
        fmt.Println("Error escribiendo en el archivo:", err)
    }
}

Información del Archivo

Puedes recuperar información del archivo usando la función os.Stat.

package main

import (
    "fmt"
    "os"
)

func main() {
    fileInfo, err := os.Stat("example.txt")
    if err != nil {
        fmt.Println("Error obteniendo información del archivo:", err)
        return
    }
    fmt.Printf("Nombre del Archivo: %s\n", fileInfo.Name())
    fmt.Printf("Tamaño: %d bytes\n", fileInfo.Size())
    fmt.Printf("Permisos: %s\n", fileInfo.Mode())
    fmt.Printf("Última Modificación: %s\n", fileInfo.ModTime())
}

Conclusión

La entrada/salida de archivos es una habilidad crucial para cualquier programador. En Go, los paquetes os, bufio y io/ioutil proporcionan herramientas poderosas para trabajar con archivos. Al entender cómo leer y escribir en archivos, manejar errores y gestionar permisos de archivos, puedes crear aplicaciones robustas que interactúan eficazmente con el sistema de archivos.
