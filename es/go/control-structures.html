Las estructuras de control son fundamentales en la programación, permitiéndote dictar el flujo de tu programa basado en condiciones y bucles. En Go, las estructuras de control son directas y fáciles de entender. Esta sección cubrirá las estructuras de control esenciales en Go, incluyendo declaraciones if, bucles for, declaraciones switch y declaraciones select.

Declaraciones If

La declaración if te permite ejecutar un bloque de código solo si una condición especificada es verdadera.

Declaración If Básica

package main

import "fmt"

func main() {
    x := 10
    if x > 5 {
        fmt.Println("x es mayor que 5")
    }
}
  • Explicación: El código verifica si x es mayor que 5. Si es verdadero, imprime "x es mayor que 5".

Declaración If-Else

package main

import "fmt"

func main() {
    x := 3
    if x > 5 {
        fmt.Println("x es mayor que 5")
    } else {
        fmt.Println("x no es mayor que 5")
    }
}
  • Explicación: Si x no es mayor que 5, el bloque else se ejecuta, imprimiendo "x no es mayor que 5".

Declaración If-Else If-Else

package main

import "fmt"

func main() {
    x := 7
    if x > 10 {
        fmt.Println("x es mayor que 10")
    } else if x > 5 {
        fmt.Println("x es mayor que 5 pero menor o igual a 10")
    } else {
        fmt.Println("x es 5 o menos")
    }
}
  • Explicación: Esta estructura permite verificar múltiples condiciones en secuencia.

Bucles For

El bucle for es el único constructo de bucle en Go, pero es lo suficientemente versátil para manejar todas las necesidades de bucles.

Bucle For Básico

package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }
}
  • Explicación: Este bucle imprime números del 0 al 4.

Bucle For con Condición

package main

import "fmt"

func main() {
    x := 0
    for x < 5 {
        fmt.Println(x)
        x++
    }
}
  • Explicación: Este bucle continúa mientras x sea menor que 5.

Bucle Infinito

package main

import "fmt"

func main() {
    for {
        fmt.Println("Bucle infinito")
        break // Para prevenir un bucle infinito real
    }
}
  • Explicación: Se puede crear un bucle infinito usando for sin ninguna condición. La declaración break se usa para salir del bucle.

Declaraciones Switch

La declaración switch proporciona una forma de ejecutar un bloque de código entre muchos basándose en el valor de una variable.

Declaración Switch Básica

package main

import "fmt"

func main() {
    day := "Tuesday"
    switch day {
    case "Monday":
        fmt.Println("Inicio de la semana laboral")
    case "Tuesday":
        fmt.Println("Segundo día de la semana laboral")
    default:
        fmt.Println("Otro día")
    }
}
  • Explicación: El código imprime un mensaje basado en el valor de day.

Switch con Múltiples Casos

package main

import "fmt"

func main() {
    day := "Saturday"
    switch day {
    case "Saturday", "Sunday":
        fmt.Println("Fin de semana")
    default:
        fmt.Println("Día laborable")
    }
}
  • Explicación: Se pueden combinar múltiples casos para manejar el mismo bloque de código.

Declaraciones Select

La declaración select se usa con canales para manejar múltiples operaciones de canal.

Declaración Select Básica

package main

import (
    "fmt"
    "time"
)

func main() {
    ch1 := make(chan string)
    ch2 := make(chan string)

    go func() {
        time.Sleep(1 * time.Second)
        ch1 <- "uno"
    }()

    go func() {
        time.Sleep(2 * time.Second)
        ch2 <- "dos"
    }()

    select {
    case msg1 := <-ch1:
        fmt.Println("Recibido", msg1)
    case msg2 := <-ch2:
        fmt.Println("Recibido", msg2)
    }
}
  • Explicación: La declaración select espera a que uno de los canales envíe un mensaje y luego ejecuta el caso correspondiente.

Conclusión

Las estructuras de control en Go son esenciales para gestionar el flujo de tu programa. Al dominar las declaraciones if, los bucles for, las declaraciones switch y las declaraciones select, puedes escribir código más eficiente y legible. Practica usando estos constructos en varios escenarios para volverte competente en el control del flujo de tus programas en Go.
