Introducción

En Go, los arrays y los slices son estructuras de datos fundamentales que te permiten almacenar y gestionar colecciones de elementos. Entender cómo usarlos eficazmente es crucial para escribir código en Go eficiente e idiomático.

Arrays

Conceptos Clave

  • Tamaño Fijo: Los arrays tienen un tamaño fijo que se define en el momento de la declaración.
  • Elementos Homogéneos: Todos los elementos en un array deben ser del mismo tipo.
  • Inicialización con Valor Cero: Los arrays se inicializan con valores cero para su tipo.

Declaración de Arrays

Para declarar un array en Go, especificas el tamaño y el tipo de sus elementos.

var arr [5]int // Un array de 5 enteros

Inicialización de Arrays

Puedes inicializar un array en el momento de la declaración.

var arr = [5]int{1, 2, 3, 4, 5}

También puedes dejar que Go infiera el tamaño del array.

var arr = [...]int{1, 2, 3, 4, 5}

Acceso a Elementos del Array

Los elementos del array se acceden usando índices basados en cero.

fmt.Println(arr[0]) // Salida: 1

Modificación de Elementos del Array

Puedes modificar elementos asignando nuevos valores a índices específicos.

arr[0] = 10
fmt.Println(arr[0]) // Salida: 10

Iteración sobre Arrays

Puedes usar un bucle for para iterar sobre los elementos del array.

for i := 0; i < len(arr); i++ {
    fmt.Println(arr[i])
}

Alternativamente, puedes usar un bucle range.

for index, value := range arr {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}

Slices

Conceptos Clave

  • Tamaño Dinámico: Los slices tienen tamaño dinámico, lo que los hace más flexibles que los arrays.
  • Tipo de Referencia: Los slices son referencias a arrays.
  • Inicialización con Valor Cero: El valor cero de un slice es nil.

Declaración de Slices

Puedes declarar un slice sin especificar su tamaño.

var slice []int

Creación de Slices

Puedes crear un slice usando la función make.

slice := make([]int, 5) // Crea un slice de longitud 5

También puedes crear un slice a partir de un array existente.

arr := [5]int{1, 2, 3, 4, 5}
slice := arr[1:4] // Crea un slice [2, 3, 4]

Añadir Elementos a Slices

Puedes añadir elementos a un slice usando la función append.

slice := []int{1, 2, 3}
slice = append(slice, 4, 5)
fmt.Println(slice) // Salida: [1, 2, 3, 4, 5]

Copiar Slices

Puedes copiar elementos de un slice a otro usando la función copy.

src := []int{1, 2, 3}
dst := make([]int, len(src))
copy(dst, src)
fmt.Println(dst) // Salida: [1, 2, 3]

Dividir Slices

Puedes crear un nuevo slice a partir de un slice existente.

slice := []int{1, 2, 3, 4, 5}
newSlice := slice[1:4] // Crea un slice [2, 3, 4]

Iteración sobre Slices

Puedes iterar sobre slices igual que sobre arrays.

for index, value := range slice {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}

Arrays vs. Slices

| Característica | Arrays | Slices | |------------------|---------------------------------|---------------------------------| | Tamaño | Fijo | Dinámico | | Inicialización | Valor cero | Valor cero (nil) | | Flexibilidad | Menos flexible | Más flexible | | Sobrecarga de Memoria | Menos | Más (debido al redimensionamiento dinámico) | | Caso de Uso | Cuando el tamaño es conocido y fijo | Cuando el tamaño puede variar |

Conclusión

Los arrays y los slices son herramientas esenciales en Go para gestionar colecciones de datos. Los arrays proporcionan una forma eficiente de almacenar elementos con tamaño fijo, mientras que los slices ofrecen flexibilidad y redimensionamiento dinámico. Entender las diferencias y los casos de uso apropiados para cada uno te ayudará a escribir código en Go más eficiente e idiomático. Practica usando arrays y slices con varios ejemplos para volverte competente en su uso.
