Introducción

En Go, las variables y constantes son conceptos fundamentales que te permiten almacenar y gestionar datos. Entender cómo declararlas, inicializarlas y usarlas es crucial para escribir programas efectivos en Go. Esta sección cubrirá los conceptos básicos de variables y constantes, sus diferencias y las mejores prácticas para usarlas.

Variables

Declaración de Variables

En Go, las variables pueden ser declaradas usando la palabra clave var o la notación abreviada. Aquí están las diferentes maneras de declarar variables:

  • Usando la palabra clave var:
var name string
var age int
  • Notación abreviada:
name := "John"
age := 30

Inicialización de Variables

Las variables pueden ser inicializadas en el momento de su declaración:

  • Con la palabra clave var:
var name string = "John"
var age int = 30
  • Usando notación abreviada:
name := "John"
age := 30

Valores Cero

En Go, las variables que se declaran sin un valor inicial se les asigna un "valor cero" predeterminado basado en su tipo:

  • Valores cero para tipos básicos:

| Tipo | Valor Cero | |---------|------------| | int | 0 | | float64 | 0.0 | | string | "" | | bool | false |

Ejemplo:

var number int
var price float64
var message string
var isAvailable bool

fmt.Println(number)      // Salida: 0
fmt.Println(price)       // Salida: 0.0
fmt.Println(message)     // Salida: ""
fmt.Println(isAvailable) // Salida: false

Declaraciones Múltiples de Variables

Go te permite declarar múltiples variables en una sola línea:

var x, y, z int
x, y, z := 1, 2, 3

Constantes

Declaración de Constantes

Las constantes se declaran usando la palabra clave const y deben ser asignadas un valor en el momento de su declaración. Las constantes no pueden ser cambiadas una vez que se establecen.

const Pi = 3.14
const Greeting = "Hello, World!"

Constantes Tipadas y No Tipadas

Las constantes pueden ser tipadas o no tipadas:

  • Constantes tipadas:
const Pi float64 = 3.14
  • Constantes no tipadas:
const Pi = 3.14

Constantes Enumeradas

Go soporta constantes enumeradas usando el identificador iota, lo cual simplifica la creación de una secuencia de constantes relacionadas.

const (
    Red = iota
    Green
    Blue
)

fmt.Println(Red, Green, Blue) // Salida: 0 1 2

Mejores Prácticas

  • Usa nombres significativos: Elige nombres de variables y constantes que describan claramente su propósito.
  • Prefiere constantes sobre variables: Usa constantes para valores que no cambian para hacer tu código más legible y mantenible.
  • Agrupa constantes relacionadas: Usa iota para crear constantes relacionadas y mejorar la organización del código.

Conclusión

Entender las variables y constantes es esencial para escribir programas efectivos en Go. Las variables te permiten almacenar y manipular datos, mientras que las constantes proporcionan una manera de definir valores fijos que no cambian. Siguiendo las mejores prácticas y usando los métodos de declaración apropiados, puedes escribir código en Go claro y mantenible.
