Introducción

En esta sección del curso, exploraremos los dispositivos de red, que son componentes esenciales en cualquier infraestructura de red. Estos dispositivos permiten la comunicación y el intercambio de datos entre diferentes equipos y redes. Comprender su funcionamiento y características es crucial para cualquier profesional de TI.

Tipos de Dispositivos de Red

Routers

  • Función Principal: Dirigen el tráfico de datos entre diferentes redes.
  • Características:
    • Capacidad para conectar múltiples redes.
    • Utilizan tablas de enrutamiento para determinar la mejor ruta para los datos.
    • Pueden realizar funciones de seguridad como firewalls.
  • Ejemplo de Configuración Básica: 
    # Configuración de una interfaz en un router Cisco
Router> enable
Router# configure terminal
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# exit
Router# write memory

Switches

  • Función Principal: Conectan dispositivos dentro de la misma red local (LAN).
  • Características:
    • Operan en la capa 2 del modelo OSI (capa de enlace de datos).
    • Utilizan direcciones MAC para enviar datos al dispositivo correcto.
    • Pueden ser gestionados o no gestionados.
  • Ejemplo de Configuración Básica: 
    # Configuración de una VLAN en un switch Cisco
Switch> enable
Switch# configure terminal
Switch(config)# vlan 10
Switch(config-vlan)# name Ventas
Switch(config-vlan)# exit
Switch(config)# interface FastEthernet0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport access vlan 10
Switch(config-if)# exit
Switch(config)# exit
Switch# write memory

Access Points (AP)

  • Función Principal: Permiten la conexión inalámbrica de dispositivos a una red cableada.
  • Características:
    • Operan en la capa 2 del modelo OSI.
    • Pueden ser autónomos o controlados por un controlador central.
    • Soportan diferentes estándares Wi-Fi (802.11a/b/g/n/ac).
  • Ejemplo de Configuración Básica: 
    # Configuración de un SSID en un AP Cisco
AP> enable
AP# configure terminal
AP(config)# interface Dot11Radio0
AP(config-if)# ssid MiRedInalambrica
AP(config-if-ssid)# authentication open
AP(config-if-ssid)# authentication key-management wpa
AP(config-if-ssid)# wpa-psk ascii 0 MiClaveSecreta
AP(config-if-ssid)# exit
AP(config-if)# exit
AP(config)# exit
AP# write memory

Firewalls

  • Función Principal: Proveen seguridad a la red controlando el tráfico entrante y saliente.
  • Características:
    • Pueden ser hardware o software.
    • Implementan políticas de seguridad basadas en reglas.
    • Pueden realizar inspección de paquetes y filtrado de contenido.
  • Ejemplo de Configuración Básica: 
    # Configuración de una regla de firewall en un dispositivo Cisco ASA
ASA> enable
ASA# configure terminal
ASA(config)# access-list OUTSIDE_IN extended permit tcp any host 192.168.1.10 eq 80
ASA(config)# access-group OUTSIDE_IN in interface outside
ASA(config)# exit
ASA# write memory

Comparación de Dispositivos de Red

| Dispositivo | Capa OSI | Función Principal | Ejemplo de Uso | |---------------|----------------|-----------------------------------------------|---------------------------------------| | Router | Capa 3 (Red) | Dirigir tráfico entre diferentes redes | Conectar una red doméstica a Internet | | Switch | Capa 2 (Enlace)| Conectar dispositivos dentro de una misma LAN | Conectar PCs en una oficina | | Access Point | Capa 2 (Enlace)| Proveer conectividad inalámbrica | Conectar laptops y smartphones | | Firewall | Capa 3 (Red) | Proveer seguridad mediante control de tráfico | Proteger una red empresarial |

Conclusión

Los dispositivos de red son fundamentales para la construcción y mantenimiento de cualquier infraestructura de red. Desde routers y switches hasta access points y firewalls, cada dispositivo tiene un rol específico y características que lo hacen adecuado para diferentes tareas. Comprender cómo configurar y gestionar estos dispositivos es esencial para cualquier profesional de TI que trabaje en el campo de las redes.
