Introducción

Los formularios e inputs son componentes fundamentales en cualquier aplicación, permitiendo a los usuarios interactuar con la app proporcionando datos. En Ionic, crear formularios y manejar la entrada del usuario es sencillo, gracias a su rico conjunto de componentes de UI y las potentes capacidades de manejo de formularios de Angular.

Elementos Básicos del Formulario

Ionic proporciona una variedad de elementos de formulario que pueden ser utilizados para capturar la entrada del usuario. Estos elementos están estilizados para coincidir con el aspecto y la sensación de tu aplicación Ionic.

Inputs de Texto

Los inputs de texto se utilizan para capturar datos de texto de una sola línea del usuario.

<ion-item>
  <ion-label position="floating">Nombre de usuario</ion-label>
  <ion-input type="text"></ion-input>
</ion-item>
  • <ion-item>: Un contenedor para elementos de formulario.
  • <ion-label>: Una etiqueta para el campo de entrada.
  • <ion-input>: El campo de entrada en sí.

Inputs de Contraseña

Los inputs de contraseña se utilizan para capturar información sensible como contraseñas.

<ion-item>
  <ion-label position="floating">Contraseña</ion-label>
  <ion-input type="password"></ion-input>
</ion-item>

Inputs de Correo Electrónico

Los inputs de correo electrónico son específicamente para capturar direcciones de correo electrónico.

<ion-item>
  <ion-label position="floating">Correo Electrónico</ion-label>
  <ion-input type="email"></ion-input>
</ion-item>

Inputs de Número

Los inputs de número se utilizan para capturar datos numéricos.

<ion-item>
  <ion-label position="floating">Edad</ion-label>
  <ion-input type="number"></ion-input>
</ion-item>

Elementos Avanzados del Formulario

Ionic también proporciona elementos de formulario más avanzados para capturar diferentes tipos de datos.

Inputs de Selección

Los inputs de selección permiten a los usuarios elegir de una lista de opciones.

<ion-item>
  <ion-label>Fruta Favorita</ion-label>
  <ion-select>
    <ion-select-option value="apple">Manzana</ion-select-option>
    <ion-select-option value="banana">Banana</ion-select-option>
    <ion-select-option value="cherry">Cereza</ion-select-option>
  </ion-select>
</ion-item>

Inputs de Fecha

Los inputs de fecha permiten a los usuarios seleccionar una fecha.

<ion-item>
  <ion-label>Fecha de Nacimiento</ion-label>
  <ion-datetime display-format="MM/DD/YYYY"></ion-datetime>
</ion-item>

Inputs de Conmutador

Los inputs de conmutador se utilizan para elecciones binarias, como interruptores de encendido/apagado.

<ion-item>
  <ion-label>Habilitar Notificaciones</ion-label>
  <ion-toggle></ion-toggle>
</ion-item>

Validación de Formularios

La validación de formularios es crucial para asegurar que los datos ingresados por el usuario sean correctos y completos. Angular proporciona herramientas poderosas para la validación de formularios, que pueden integrarse fácilmente con los formularios de Ionic.

Formularios Basados en Plantillas

Los formularios basados en plantillas son simples y adecuados para casos de uso básicos.

<form #myForm="ngForm">
  <ion-item>
    <ion-label position="floating">Nombre de usuario</ion-label>
    <ion-input type="text" name="username" ngModel required></ion-input>
  </ion-item>
  <div *ngIf="myForm.controls.username?.invalid && myForm.controls.username?.touched">
    El nombre de usuario es obligatorio.
  </div>
  <ion-button type="submit" [disabled]="myForm.invalid">Enviar</ion-button>
</form>

Formularios Reactivos

Los formularios reactivos proporcionan más control y son adecuados para escenarios complejos.

import { Component } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-form-example',
  templateUrl: './form-example.component.html'
})
export class FormExampleComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      username: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]]
    });
  }

  onSubmit() {
    if (this.myForm.valid) {
      console.log(this.myForm.value);
    }
  }
}
<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <ion-item>
    <ion-label position="floating">Nombre de usuario</ion-label>
    <ion-input formControlName="username"></ion-input>
  </ion-item>
  <div *ngIf="myForm.get('username').invalid && myForm.get('username').touched">
    El nombre de usuario es obligatorio.
  </div>

  <ion-item>
    <ion-label position="floating">Correo Electrónico</ion-label>
    <ion-input formControlName="email"></ion-input>
  </ion-item>
  <div *ngIf="myForm.get('email').invalid && myForm.get('email').touched">
    Se requiere un correo electrónico válido.
  </div>

  <ion-button type="submit" [disabled]="myForm.invalid">Enviar</ion-button>
</form>

Manejo del Envío del Formulario

El manejo del envío del formulario implica capturar los datos ingresados por el usuario y procesarlos, como enviarlos a un servidor.

<form (ngSubmit)="onSubmit(myForm)" #myForm="ngForm">
  <ion-item>
    <ion-label position="floating">Nombre de usuario</ion-label>
    <ion-input type="text" name="username" ngModel required></ion-input>
  </ion-item>
  <ion-button type="submit" [disabled]="myForm.invalid">Enviar</ion-button>
</form>
import { Component } from '@angular/core';

@Component({
  selector: 'app-form-example',
  templateUrl: './form-example.component.html'
})
export class FormExampleComponent {
  onSubmit(form) {
    if (form.valid) {
      console.log(form.value);
    }
  }
}

Conclusión

Los formularios e inputs son esenciales para capturar datos del usuario en cualquier aplicación. Ionic proporciona un conjunto completo de elementos de formulario que son fáciles de usar e integrar con las capacidades de manejo de formularios de Angular. Ya sea que estés construyendo formularios simples con enfoques basados en plantillas o formularios complejos con formularios reactivos, Ionic tiene las herramientas que necesitas para crear formularios robustos y fáciles de usar.
