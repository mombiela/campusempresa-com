Introducción

En esta sección, exploraremos cómo crear y gestionar listas y tarjetas en Ionic. Las listas y tarjetas son componentes fundamentales para mostrar colecciones de datos de manera estructurada y visualmente atractiva. Al final de esta sección, podrás crear listas y tarjetas dinámicas, personalizar su apariencia y manejar interacciones de usuario.

Listas en Ionic

Conceptos Clave

  • IonList: El contenedor principal para una lista.
  • IonItem: Representa cada elemento dentro de una lista.
  • IonLabel: Se usa para mostrar texto dentro de un elemento.
  • IonAvatar: Se usa para mostrar un avatar o imagen dentro de un elemento.
  • IonIcon: Se usa para mostrar íconos dentro de un elemento.

Ejemplo Básico de Lista

Aquí tienes un ejemplo básico de una lista en Ionic:

<ion-list>
  <ion-item>
    <ion-label>Elemento 1</ion-label>
  </ion-item>
  <ion-item>
    <ion-label>Elemento 2</ion-label>
  </ion-item>
  <ion-item>
    <ion-label>Elemento 3</ion-label>
  </ion-item>
</ion-list>

En este ejemplo:

  • <ion-list> es el contenedor para la lista.
  • <ion-item> representa cada elemento en la lista.
  • <ion-label> se usa para mostrar el texto de cada elemento.

Lista con Avatares e Íconos

Puedes mejorar los elementos de tu lista añadiendo avatares e íconos:

<ion-list>
  <ion-item>
    <ion-avatar slot="start">
      <img src="avatar1.png" />
    </ion-avatar>
    <ion-label>Elemento 1</ion-label>
    <ion-icon name="star" slot="end"></ion-icon>
  </ion-item>
  <ion-item>
    <ion-avatar slot="start">
      <img src="avatar2.png" />
    </ion-avatar>
    <ion-label>Elemento 2</ion-label>
    <ion-icon name="heart" slot="end"></ion-icon>
  </ion-item>
</ion-list>

En este ejemplo:

  • <ion-avatar> se usa para mostrar una imagen al inicio de cada elemento.
  • <ion-icon> se usa para mostrar un ícono al final de cada elemento.

Tarjetas en Ionic

Conceptos Clave

  • IonCard: El contenedor principal para una tarjeta.
  • IonCardHeader: Se usa para definir la sección de encabezado de una tarjeta.
  • IonCardTitle: Se usa para definir el título dentro del encabezado de la tarjeta.
  • IonCardSubtitle: Se usa para definir el subtítulo dentro del encabezado de la tarjeta.
  • IonCardContent: Se usa para definir el contenido principal de la tarjeta.

Ejemplo Básico de Tarjeta

Aquí tienes un ejemplo básico de una tarjeta en Ionic:

<ion-card>
  <ion-card-header>
    <ion-card-title>Título de la Tarjeta</ion-card-title>
    <ion-card-subtitle>Subtítulo de la Tarjeta</ion-card-subtitle>
  </ion-card-header>
  <ion-card-content>
    Este es el contenido de la tarjeta.
  </ion-card-content>
</ion-card>

En este ejemplo:

  • <ion-card> es el contenedor para la tarjeta.
  • <ion-card-header> contiene el título y el subtítulo.
  • <ion-card-content> contiene el contenido principal de la tarjeta.

Tarjeta con Imágenes y Botones

Puedes mejorar tus tarjetas añadiendo imágenes y botones:

<ion-card>
  <img src="card-image.png" />
  <ion-card-header>
    <ion-card-title>Título de la Tarjeta</ion-card-title>
    <ion-card-subtitle>Subtítulo de la Tarjeta</ion-card-subtitle>
  </ion-card-header>
  <ion-card-content>
    Este es el contenido de la tarjeta.
    <ion-button expand="full">Botón</ion-button>
  </ion-card-content>
</ion-card>

En este ejemplo:

  • Se añade una imagen en la parte superior de la tarjeta.
  • Se añade un botón dentro del contenido de la tarjeta.

Conclusión

Las listas y tarjetas son componentes esenciales en Ionic para mostrar datos de manera estructurada y visualmente atractiva. Al entender cómo usar IonList, IonItem, IonCard y sus componentes relacionados, puedes crear interfaces de usuario dinámicas e interactivas. Practica creando listas y tarjetas con diferentes configuraciones para volverte competente en el uso de estos componentes en tus aplicaciones Ionic.
