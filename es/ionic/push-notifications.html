Notificaciones Push en Ionic

Las notificaciones push son una herramienta poderosa para involucrar a los usuarios al entregar información oportuna y relevante directamente a sus dispositivos. En este tema, cubriremos lo esencial para implementar notificaciones push en una aplicación Ionic, desde niveles principiantes hasta avanzados.

Introducción a las Notificaciones Push

  • Definición: Las notificaciones push son mensajes enviados desde un servidor al dispositivo de un usuario, incluso cuando la aplicación no está activa.
  • Casos de Uso: Alertas, recordatorios, promociones, actualizaciones, etc.
  • Plataformas: Android, iOS y Web.

Configuración de Notificaciones Push en Ionic

Requisitos Previos

  • Ionic CLI: Asegúrate de tener la última versión del Ionic CLI instalada.
  • Cuenta de Firebase: Se utilizará Firebase Cloud Messaging (FCM) para enviar notificaciones.
  • Capacitor o Cordova: Ionic soporta tanto Capacitor como Cordova para funcionalidad nativa.

Instalación de Plugins Necesarios

  1. Capacitor:

    npm install @capacitor/push-notifications
npx cap sync

  2. Cordova:

    ionic cordova plugin add phonegap-plugin-push
npm install @ionic-native/push

Configuración de Firebase

  1. Crear un Proyecto en Firebase:

    • Ve a la Consola de Firebase.
    • Crea un nuevo proyecto o usa uno existente.

  2. Agregar tu Aplicación a Firebase:

    • Registra tu aplicación (iOS y Android).
    • Descarga los archivos google-services.json (para Android) y GoogleService-Info.plist (para iOS) y colócalos en los directorios apropiados.

Inicialización de Notificaciones Push

Ejemplo con Capacitor

  1. Importar e Inicializar:

    import { PushNotifications } from '@capacitor/push-notifications';

PushNotifications.requestPermissions().then(result => {
  if (result.receive === 'granted') {
    PushNotifications.register();
  } else {
    // Mostrar algún error
  }
});

  2. Manejo de Notificaciones:

    PushNotifications.addListener('pushNotificationReceived', (notification) => {
  console.log('Notificación recibida: ', notification);
});

PushNotifications.addListener('pushNotificationActionPerformed', (notification) => {
  console.log('Acción de notificación realizada: ', notification);
});

Ejemplo con Cordova

  1. Importar e Inicializar: 
    import { Push, PushObject, PushOptions } from '@ionic-native/push/ngx';

constructor(private push: Push) {}

initializePushNotifications() {
  const options: PushOptions = {
    android: {},
    ios: {
      alert: 'true',
      badge: true,
      sound: 'false'
    }
  };

  const pushObject: PushObject = this.push.init(options);

  pushObject.on('notification').subscribe((notification: any) => {
    console.log('Notificación recibida', notification);
  });

  pushObject.on('registration').subscribe((registration: any) => {
    console.log('Dispositivo registrado', registration);
  });

  pushObject.on('error').subscribe(error => console.error('Error con el plugin Push', error));
}

Temas Avanzados

Personalización de Notificaciones

  • Estructura del Payload: Personaliza el payload para incluir datos adicionales. 
    {
  "to": "device_token",
  "notification": {
    "title": "Nuevo Mensaje",
    "body": "Has recibido un nuevo mensaje.",
    "icon": "myicon"
  },
  "data": {
    "additionalData": "value"
  }
}

Manejo de Notificaciones en Segundo Plano

  • Capacitor:

    PushNotifications.addListener('pushNotificationReceived', (notification) => {
  if (notification.foreground) {
    // Manejar notificación en primer plano
  } else {
    // Manejar notificación en segundo plano
  }
});

  • Cordova:

    pushObject.on('notification').subscribe((notification: any) => {
  if (notification.additionalData.foreground) {
    // Manejar notificación en primer plano
  } else {
    // Manejar notificación en segundo plano
  }
});

Programación de Notificaciones

  • Notificaciones Locales: Usa el plugin Local Notifications para programar notificaciones. 
    import { LocalNotifications } from '@capacitor/local-notifications';

LocalNotifications.schedule({
  notifications: [
    {
      title: "Notificación Programada",
      body: "Esta es una notificación programada.",
      id: 1,
      schedule: { at: new Date(Date.now() + 1000 * 60 * 5) }, // 5 minutos a partir de ahora
      sound: null,
      attachments: null,
      actionTypeId: "",
      extra: null
    }
  ]
});

Conclusión

Las notificaciones push son una característica crucial para las aplicaciones móviles modernas, permitiendo la comunicación en tiempo real con los usuarios. Siguiendo esta guía, deberías poder configurar y personalizar notificaciones push en tu aplicación Ionic, manejarlas tanto en estados de primer plano como de segundo plano, e incluso programar notificaciones locales. A medida que avances, considera explorar casos de uso más complejos e integrar analíticas para medir la efectividad de tus notificaciones.
