Resumen

La tematización en Ionic te permite personalizar el aspecto y la sensación de tu aplicación para que coincida con tu marca o preferencias de diseño. Esta sección te guiará a través de los conceptos básicos de la tematización en Ionic, desde la comprensión de los conceptos básicos hasta la implementación de temas personalizados.

Conceptos Clave

  • Variables CSS: Ionic utiliza variables CSS (propiedades personalizadas) para definir colores de tema y otros estilos.
  • Estilos Globales: Estilos que se aplican a toda la aplicación.
  • Estilos de Componentes: Estilos que se aplican a componentes específicos.
  • Modo Oscuro: Un tema que utiliza colores más oscuros para una mejor legibilidad en entornos con poca luz.

Variables CSS

Las variables CSS son una forma poderosa de gestionar y aplicar estilos en toda tu aplicación Ionic. Te permiten definir valores reutilizables que se pueden aplicar a varias propiedades CSS.

Ejemplo

:root {
  --ion-color-primary: #3880ff;
  --ion-color-secondary: #0cd1e8;
  --ion-color-tertiary: #7044ff;
}

En este ejemplo, --ion-color-primary, --ion-color-secondary y --ion-color-tertiary son variables CSS que definen los colores primario, secundario y terciario de tu tema.

Estilos Globales

Los estilos globales se definen en el archivo src/theme/variables.scss. Aquí es donde puedes establecer los estilos predeterminados para toda tu aplicación.

Ejemplo

// src/theme/variables.scss
:root {
  --ion-font-family: 'Open Sans', sans-serif;
  --ion-color-primary: #3880ff;
  --ion-color-secondary: #0cd1e8;
  --ion-color-tertiary: #7044ff;
}

En este ejemplo, establecemos la familia de fuentes predeterminada y los colores para toda la aplicación.

Estilos de Componentes

Los estilos de componentes están limitados a componentes individuales. Puedes definir estos estilos en la hoja de estilos del componente.

Ejemplo

// src/pages/home/home.page.scss
.page-home {
  ion-content {
    --background: #f0f0f0;
  }

  .custom-button {
    --background: var(--ion-color-primary);
    --color: white;
  }
}

En este ejemplo, establecemos un color de fondo personalizado para el ion-content y un estilo personalizado para un botón dentro de la página home.

Modo Oscuro

Ionic soporta el modo oscuro, que se puede habilitar estableciendo una clase CSS en el elemento body.

Ejemplo

<!-- index.html -->
<body class="dark-theme">
  <app-root></app-root>
</body>

Variables del Modo Oscuro

Puedes definir variables específicas para el modo oscuro en tu archivo variables.scss.

// src/theme/variables.scss
body.dark-theme {
  --ion-background-color: #222428;
  --ion-text-color: #ffffff;
}

En este ejemplo, establecemos los colores de fondo y de texto para el modo oscuro.

Resumen

La tematización en Ionic es una forma poderosa de personalizar la apariencia de tu aplicación. Utilizando variables CSS, estilos globales, estilos de componentes y el modo oscuro, puedes crear una experiencia de usuario cohesiva y visualmente atractiva. Recuerda:

  • Utilizar variables CSS para estilos reutilizables y mantenibles.
  • Definir estilos globales en variables.scss para una consistencia en toda la aplicación.
  • Limitar los estilos a componentes individuales cuando sea necesario.
  • Implementar el modo oscuro para mejorar la legibilidad en entornos con poca luz.

Con estas herramientas y técnicas, puedes crear un aspecto único y pulido para tus aplicaciones Ionic.
