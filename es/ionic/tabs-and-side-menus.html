Introducción

En esta sección, exploraremos cómo crear y gestionar pestañas y menús laterales en aplicaciones Ionic. Las pestañas y los menús laterales son componentes esenciales para construir aplicaciones móviles intuitivas y navegables. Al final de esta sección, serás capaz de implementar estas características en tus proyectos Ionic.

Pestañas en Ionic

¿Qué son las Pestañas?

Las pestañas son un patrón de navegación común que permite a los usuarios cambiar entre diferentes vistas o páginas dentro de una aplicación. Cada pestaña representa una sección diferente de la aplicación.

Creando un Diseño Básico de Pestañas

Para crear un diseño básico de pestañas en Ionic, puedes usar el componente ion-tabs junto con los componentes ion-tab-bar y ion-tab-button.

Ejemplo:

<ion-tabs>
  <ion-tab-bar slot="bottom">
    <ion-tab-button tab="home">
      <ion-icon name="home"></ion-icon>
      <ion-label>Home</ion-label>
    </ion-tab-button>

    <ion-tab-button tab="settings">
      <ion-icon name="settings"></ion-icon>
      <ion-label>Settings</ion-label>
    </ion-tab-button>
  </ion-tab-bar>
</ion-tabs>
  • ion-tabs: El contenedor para la navegación de pestañas.
  • ion-tab-bar: La barra que contiene los botones de las pestañas.
  • ion-tab-button: Cada botón representa una pestaña.

Enlazando Pestañas a Páginas

Para enlazar pestañas a páginas específicas, necesitas definir rutas en tu archivo app-routing.module.ts.

Ejemplo:

const routes: Routes = [
  {
    path: 'tabs',
    component: TabsPage,
    children: [
      {
        path: 'home',
        loadChildren: () => import('./home/home.module').then(m => m.HomePageModule)
      },
      {
        path: 'settings',
        loadChildren: () => import('./settings/settings.module').then(m => m.SettingsPageModule)
      }
    ]
  },
  {
    path: '',
    redirectTo: '/tabs/home',
    pathMatch: 'full'
  }
];
  • Define rutas hijas bajo la ruta tabs.
  • Usa loadChildren para cargar los módulos de cada pestaña de forma perezosa.

Menús Laterales en Ionic

¿Qué es un Menú Lateral?

Un menú lateral (también conocido como drawer o cajón de navegación) es un panel que se desliza desde el lado de la pantalla, proporcionando opciones de navegación.

Creando un Menú Lateral Básico

Para crear un menú lateral, puedes usar el componente ion-menu junto con ion-menu-toggle y ion-content.

Ejemplo:

<ion-menu side="start" contentId="main-content">
  <ion-header>
    <ion-toolbar>
      <ion-title>Menu</ion-title>
    </ion-toolbar>
  </ion-header>
  <ion-content>
    <ion-list>
      <ion-menu-toggle auto-hide="false">
        <ion-item routerLink="/home">
          <ion-icon slot="start" name="home"></ion-icon>
          <ion-label>Home</ion-label>
        </ion-item>
        <ion-item routerLink="/settings">
          <ion-icon slot="start" name="settings"></ion-icon>
          <ion-label>Settings</ion-label>
        </ion-item>
      </ion-menu-toggle>
    </ion-list>
  </ion-content>
</ion-menu>

<ion-router-outlet id="main-content"></ion-router-outlet>
  • ion-menu: El contenedor para el menú lateral.
  • side="start": Especifica que el menú debe aparecer desde el lado izquierdo.
  • contentId: Vincula el menú al área de contenido principal.
  • ion-menu-toggle: Alterna el menú abierto y cerrado.
  • ion-router-outlet: El área de contenido principal donde se mostrarán las páginas.

Combinando Pestañas y Menús Laterales

Puedes combinar pestañas y menús laterales para crear una estructura de navegación más compleja.

Ejemplo:

<ion-menu side="start" contentId="main-content">
  <ion-header>
    <ion-toolbar>
      <ion-title>Menu</ion-title>
    </ion-toolbar>
  </ion-header>
  <ion-content>
    <ion-list>
      <ion-menu-toggle auto-hide="false">
        <ion-item routerLink="/tabs/home">
          <ion-icon slot="start" name="home"></ion-icon>
          <ion-label>Home</ion-label>
        </ion-item>
        <ion-item routerLink="/tabs/settings">
          <ion-icon slot="start" name="settings"></ion-icon>
          <ion-label>Settings</ion-label>
        </ion-item>
      </ion-menu-toggle>
    </ion-list>
  </ion-content>
</ion-menu>

<ion-router-outlet id="main-content"></ion-router-outlet>

<ion-tabs>
  <ion-tab-bar slot="bottom">
    <ion-tab-button tab="home">
      <ion-icon name="home"></ion-icon>
      <ion-label>Home</ion-label>
    </ion-tab-button>
    <ion-tab-button tab="settings">
      <ion-icon name="settings"></ion-icon>
      <ion-label>Settings</ion-label>
    </ion-tab-button>
  </ion-tab-bar>
</ion-tabs>
  • Este ejemplo combina tanto pestañas como un menú lateral, proporcionando múltiples opciones de navegación.

Conclusión

Las pestañas y los menús laterales son componentes de navegación poderosos en Ionic que ayudan a crear una interfaz fácil de usar. Al entender cómo implementar y combinar estos componentes, puedes construir aplicaciones móviles complejas e intuitivas. Practica creando diferentes diseños y explora opciones adicionales de personalización para mejorar tus proyectos Ionic.
