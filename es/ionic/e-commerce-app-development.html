Introducción

En este curso, aprenderás a desarrollar una aplicación de comercio electrónico completamente funcional utilizando Ionic. Cubriremos todo, desde la configuración de tu entorno de desarrollo hasta el despliegue de tu aplicación. Al final de este curso, tendrás una comprensión completa de cómo construir, probar y desplegar una aplicación de comercio electrónico.

Configuración del Entorno de Desarrollo

  • Instalar Node.js y npm
  • Instalar Ionic CLI
  • Crear un nuevo proyecto Ionic
# Instalar Ionic CLI globalmente
npm install -g @ionic/cli

# Crear un nuevo proyecto Ionic
ionic start ecommerce-app blank

Gestión del Estado con NgRx

  • Introducción a NgRx
  • Configuración de NgRx en tu proyecto Ionic
  • Gestión del estado de productos
  • Gestión del estado del carrito
// Ejemplo de acciones de NgRx para productos
export const loadProducts = createAction('[Product] Load Products');
export const loadProductsSuccess = createAction('[Product] Load Products Success', props<{ products: Product[] }>());
export const loadProductsFailure = createAction('[Product] Load Products Failure', props<{ error: any }>());

// Ejemplo de reductor de NgRx para productos
const productReducer = createReducer(
  initialState,
  on(loadProductsSuccess, (state, { products }) => ({ ...state, products })),
  on(loadProductsFailure, (state, { error }) => ({ ...state, error }))
);

  let service: ProductService;

  beforeEach(() => {
    TestBed.configureTestingModule({});
    service = TestBed.inject(ProductService);
  });

  it('should be created', () => {
    expect(service).toBeTruthy();
  });

  it('should fetch products', () => {
    service.getProducts().subscribe(products => {
      expect(products.length).toBeGreaterThan(0);
    });
  });
});
