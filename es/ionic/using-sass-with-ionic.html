Introducción a Sass

Sass (Syntactically Awesome Stylesheets) es un preprocesador de CSS que añade poder y elegancia al lenguaje básico. Te permite usar variables, reglas anidadas, mixins, funciones y más, todo con una sintaxis completamente compatible con CSS.

Conceptos Clave de Sass

  • Variables: Almacena valores que puedes reutilizar a lo largo de tu hoja de estilos.
  • Anidación: Anida tus selectores CSS de una manera que siga la misma jerarquía visual de tu HTML.
  • Parciales e Importación: Divide tu CSS en piezas más pequeñas y reutilizables.
  • Mixins: Crea fragmentos de código reutilizables.
  • Herencia: Comparte un conjunto de propiedades CSS de un selector a otro.

Configuración de Sass en un Proyecto Ionic

Instalando Sass

Para usar Sass con Ionic, necesitas instalar las herramientas necesarias. Ionic CLI viene con soporte integrado para Sass.

  1. Crea un nuevo proyecto Ionic:

    ionic start myApp tabs
cd myApp

  2. Instala Sass:

    npm install sass --save-dev

Configurando Sass

Ionic usa una carpeta scss para los archivos Sass por defecto. Puedes configurar tu proyecto para usar Sass creando archivos .scss.

  1. Crea una carpeta scss:

    mkdir src/theme/scss

  2. Crea un archivo principal de Sass:

    touch src/theme/scss/variables.scss

  3. Importa el archivo principal de Sass en tus estilos globales:

    // src/global.scss
@import 'theme/scss/variables';

Usando Funcionalidades de Sass en Ionic

Variables

Las variables en Sass te permiten almacenar valores que puedes reutilizar a lo largo de tu hoja de estilos.

// src/theme/scss/variables.scss
$primary-color: #3880ff;
$secondary-color: #3dc2ff;

body {
  background-color: $primary-color;
  color: $secondary-color;
}

Anidación

La anidación te permite anidar tus selectores CSS de una manera que siga la misma jerarquía visual de tu HTML.

// src/theme/scss/variables.scss
nav {
  ul {
    margin: 0;
    padding: 0;
    list-style: none;

    li {
      display: inline-block;
      margin-right: 5px;

      a {
        text-decoration: none;
        color: $primary-color;

        &:hover {
          color: $secondary-color;
        }
      }
    }
  }
}

Parciales e Importación

Los parciales te permiten dividir tu CSS en piezas más pequeñas y reutilizables. Luego puedes importar estos parciales en tu hoja de estilos principal.

// src/theme/scss/_buttons.scss
.button {
  background-color: $primary-color;
  border: none;
  color: white;
  padding: 10px 20px;
  text-align: center;
  text-decoration: none;
  display: inline-block;
  font-size: 16px;
  margin: 4px 2px;
  cursor: pointer;
}

// src/theme/scss/variables.scss
@import 'buttons';

Mixins

Los mixins te permiten crear fragmentos de código reutilizables.

// src/theme/scss/_mixins.scss
@mixin border-radius($radius) {
  -webkit-border-radius: $radius;
  -moz-border-radius: $radius;
  -ms-border-radius: $radius;
  -o-border-radius: $radius;
  border-radius: $radius;
}

// src/theme/scss/variables.scss
@import 'mixins';

.box { 
  @include border-radius(10px); 
}

Herencia

La herencia te permite compartir un conjunto de propiedades CSS de un selector a otro.

// src/theme/scss/_base.scss
.message {
  padding: 10px;
  border: 1px solid #ccc;
  color: #333;
}

// src/theme/scss/variables.scss
@import 'base';

.success {
  @extend .message;
  border-color: green;
}

.error {
  @extend .message;
  border-color: red;
}

Conclusión

Usar Sass con Ionic mejora tu capacidad para gestionar y mantener tus hojas de estilo de manera eficiente. Al aprovechar las funcionalidades de Sass como variables, anidación, parciales, mixins y herencia, puedes crear CSS más organizado y reutilizable. Esto no solo mejora tu flujo de trabajo de desarrollo, sino que también hace que tu base de código sea más escalable y fácil de mantener.
