Configurar tu entorno de desarrollo es el primer paso para volverte competente con Ionic. Esta sección te guiará a través de los pasos necesarios para preparar tu entorno para el desarrollo con Ionic, desde la instalación del software requerido hasta la configuración de tu primer proyecto.

Requisitos Previos

Antes de sumergirte en la configuración, asegúrate de tener los siguientes requisitos previos:

  • Conocimientos básicos de HTML, CSS y JavaScript.
  • Node.js y npm instalados en tu máquina.

Instalación de Node.js y npm

Ionic requiere Node.js y npm (Node Package Manager) para gestionar dependencias y ejecutar scripts. Sigue estos pasos para instalarlos:

  1. Descargar Node.js:

    • Visita el sitio web de Node.js.
    • Descarga la versión LTS (Long Term Support) para tu sistema operativo.

  2. Instalar Node.js:

    • Ejecuta el instalador descargado y sigue las instrucciones en pantalla.
    • Verifica la instalación ejecutando los siguientes comandos en tu terminal: 
      node -v
npm -v
    • Deberías ver los números de versión tanto de Node.js como de npm.

Instalación de Ionic CLI

La Interfaz de Línea de Comandos (CLI) de Ionic es una herramienta poderosa que te ayuda a crear, construir y ejecutar aplicaciones Ionic. Para instalar la CLI de Ionic, usa npm:

npm install -g @ionic/cli

Verifica la instalación ejecutando:

ionic --version

Deberías ver el número de versión de la CLI de Ionic.

Configuración de tu Primer Proyecto Ionic

Con la CLI de Ionic instalada, ahora puedes crear tu primer proyecto Ionic. Sigue estos pasos:

  1. Crear un nuevo proyecto:

    ionic start myFirstApp blank
    • myFirstApp es el nombre de tu proyecto.
    • blank es la plantilla de inicio. Puedes elegir otras plantillas como tabs o sidemenu.

  2. Navegar al directorio de tu proyecto:

    cd myFirstApp

  3. Servir la aplicación:

    ionic serve
    • Este comando iniciará un servidor de desarrollo local y abrirá tu aplicación en un navegador web.
    • Deberías ver la aplicación predeterminada de Ionic ejecutándose en tu navegador.

Configuración de tu Entorno de Desarrollo

Para mejorar tu experiencia de desarrollo, considera configurar las siguientes herramientas:

  • Visual Studio Code (VS Code):

    • Un editor de código popular con excelente soporte para JavaScript y TypeScript.
    • Descarga e instala desde el sitio web de VS Code.

  • Extensiones de Ionic para VS Code:

    • Instala extensiones como "Ionic" y "Angular Language Service" para una mejor autocompletación y depuración de código.

  • Git:

    • Sistema de control de versiones para gestionar tu código.
    • Descarga e instala desde el sitio web de Git.

Ejemplo: Creación de un Componente Simple de Ionic

Vamos a crear un componente simple para entender los conceptos básicos del desarrollo con Ionic.

  1. Generar un nuevo componente:

    ionic generate component my-component

  2. Editar el componente:

    • Abre src/app/my-component/my-component.component.html y añade el siguiente código: 
      <ion-header>
  <ion-toolbar>
    <ion-title>
      My Component
    </ion-title>
  </ion-toolbar>
</ion-header>

<ion-content>
  <p>Hello, this is my first Ionic component!</p>
</ion-content>

  3. Usar el componente en una página:

    • Abre src/app/home/home.page.html y añade el siguiente código: 
      <app-my-component></app-my-component>

  4. Servir la aplicación:

    ionic serve
    • Deberías ver el nuevo componente renderizado en la página de inicio.

Conclusión

Configurar tu entorno de desarrollo es un paso crucial en tu viaje de desarrollo con Ionic. Siguiendo los pasos descritos en esta sección, ahora deberías tener un entorno de desarrollo Ionic completamente funcional. También has creado tu primer proyecto Ionic y un componente simple. En las siguientes secciones, profundizaremos en las características y capacidades de Ionic, construyendo sobre esta base.
