Generadores en Dart
Manejo de Excepciones ►

Los generadores son una característica poderosa en Dart que te permiten producir una secuencia de valores de manera perezosa, lo que significa que los valores se generan bajo demanda en lugar de todos a la vez. Esto puede ser particularmente útil cuando se trabaja con grandes conjuntos de datos o flujos de datos.

Conceptos Clave

  • Generadores: Funciones que producen una secuencia de valores.
  • Generadores Sincrónicos: Usan sync* y yield para producir valores.
  • Generadores Asincrónicos: Usan async* y yield para producir valores de manera asincrónica.
  • yield: Palabra clave utilizada para producir un valor en una función generadora.
  • *yield`: Palabra clave utilizada para delegar parte del proceso de generación a otro generador.

Generadores Sincrónicos

Los generadores sincrónicos se utilizan para producir valores en una secuencia que se puede iterar de manera sincrónica.

Ejemplo

Iterable<int> syncGenerator(int n) sync* {
  for (int i = 0; i < n; i++) {
    yield i;
  }
}

void main() {
  var numbers = syncGenerator(5);
  for (var number in numbers) {
    print(number);
  }
}

Explicación

  • La función syncGenerator es un generador sincrónico que produce enteros desde 0 hasta n-1.
  • La palabra clave sync* indica que esta función es un generador sincrónico.
  • La palabra clave yield se usa para producir cada valor en la secuencia.
  • En la función main, llamamos a syncGenerator e iteramos sobre los valores producidos usando un bucle for.

Generadores Asincrónicos

Los generadores asincrónicos se utilizan para producir valores en una secuencia que se puede iterar de manera asincrónica, lo cual es útil para manejar flujos de datos o realizar operaciones asincrónicas.

Ejemplo

Stream<int> asyncGenerator(int n) async* {
  for (int i = 0; i < n; i++) {
    await Future.delayed(Duration(seconds: 1)); // Simula operación asincrónica
    yield i;
  }
}

void main() async {
  var numbers = asyncGenerator(5);
  await for (var number in numbers) {
    print(number);
  }
}

Explicación

  • La función asyncGenerator es un generador asincrónico que produce enteros desde 0 hasta n-1.
  • La palabra clave async* indica que esta función es un generador asincrónico.
  • La palabra clave yield se usa para producir cada valor en la secuencia.
  • La palabra clave await se usa para simular una operación asincrónica (por ejemplo, una demora).
  • En la función main, llamamos a asyncGenerator e iteramos sobre los valores producidos usando un bucle await for.

Palabra Clave yield*

La palabra clave yield* se usa para delegar parte del proceso de generación a otro generador.

Ejemplo

Iterable<int> subGenerator(int n) sync* {
  for (int i = 0; i < n; i++) {
    yield i;
  }
}

Iterable<int> mainGenerator(int n) sync* {
  yield* subGenerator(n);
  yield n;
}

void main() {
  var numbers = mainGenerator(5);
  for (var number in numbers) {
    print(number);
  }
}

Explicación

  • La función subGenerator produce enteros desde 0 hasta n-1.
  • La función mainGenerator usa yield* para delegar la generación de valores a subGenerator y luego produce un valor adicional n.
  • En la función main, llamamos a mainGenerator e iteramos sobre los valores producidos usando un bucle for.

Conclusión

Los generadores en Dart proporcionan una forma flexible y eficiente de producir secuencias de valores, ya sea de manera sincrónica o asincrónica. Al usar sync* y yield para generadores sincrónicos, y async* y yield para generadores asincrónicos, puedes manejar una amplia gama de escenarios de generación de datos. La palabra clave yield* mejora aún más esta capacidad al permitir la delegación a otros generadores, haciendo que tu código sea más modular y fácil de gestionar. Entender y utilizar generadores puede mejorar enormemente el rendimiento y la legibilidad de tus programas en Dart, especialmente cuando se trata de grandes conjuntos de datos o flujos de datos asincrónicos.
