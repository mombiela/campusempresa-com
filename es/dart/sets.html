Introducción a los Conjuntos

Los conjuntos son una colección de elementos únicos. A diferencia de las listas, los conjuntos no permiten elementos duplicados. En Dart, los conjuntos están representados por la clase Set.

Conceptos Clave

  • Unicidad: Cada elemento en un conjunto debe ser único.
  • Desordenado: Los elementos en un conjunto no se almacenan en ningún orden particular.
  • Literales de Conjunto: Los conjuntos se pueden crear usando literales de conjunto, similares a los literales de lista pero con llaves.

Creación de Conjuntos

Puedes crear un conjunto en Dart usando el constructor Set o literales de conjunto.

Usando el Constructor Set

void main() {
  Set<int> numbers = Set();
  numbers.add(1);
  numbers.add(2);
  numbers.add(3);
  print(numbers); // Salida: {1, 2, 3}
}

Usando Literales de Conjunto

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  print(fruits); // Salida: {apple, banana, orange}
}

Operaciones Básicas en Conjuntos

Añadir Elementos

Puedes añadir elementos a un conjunto usando el método add.

void main() {
  Set<int> numbers = {1, 2, 3};
  numbers.add(4);
  print(numbers); // Salida: {1, 2, 3, 4}
}

Eliminar Elementos

Puedes eliminar elementos usando el método remove.

void main() {
  Set<int> numbers = {1, 2, 3, 4};
  numbers.remove(3);
  print(numbers); // Salida: {1, 2, 4}
}

Comprobar Elementos

Puedes comprobar si un conjunto contiene un elemento usando el método contains.

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  print(fruits.contains('banana')); // Salida: true
  print(fruits.contains('grape')); // Salida: false
}

Operaciones Avanzadas en Conjuntos

Unión

La unión de dos conjuntos es un conjunto que contiene todos los elementos de ambos conjuntos.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> unionSet = setA.union(setB);
  print(unionSet); // Salida: {1, 2, 3, 4, 5}
}

Intersección

La intersección de dos conjuntos es un conjunto que contiene solo los elementos que están presentes en ambos conjuntos.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> intersectionSet = setA.intersection(setB);
  print(intersectionSet); // Salida: {3}
}

Diferencia

La diferencia entre dos conjuntos es un conjunto que contiene elementos que están en el primer conjunto pero no en el segundo conjunto.

void main() {
  Set<int> setA = {1, 2, 3};
  Set<int> setB = {3, 4, 5};
  Set<int> differenceSet = setA.difference(setB);
  print(differenceSet); // Salida: {1, 2}
}

Iterar Sobre Conjuntos

Puedes iterar sobre un conjunto usando un bucle for o un método forEach.

Usando un Bucle For

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  for (var fruit in fruits) {
    print(fruit);
  }
}

Usando el Método forEach

void main() {
  Set<String> fruits = {'apple', 'banana', 'orange'};
  fruits.forEach((fruit) => print(fruit));
}

Conclusión

Los conjuntos en Dart son una forma poderosa de gestionar colecciones de elementos únicos. Proporcionan varios métodos para añadir, eliminar y comprobar elementos, así como realizar operaciones avanzadas como unión, intersección y diferencia. Entender los conjuntos y sus operaciones puede ayudarte a escribir código Dart más eficiente y efectivo.

Al dominar los conjuntos, puedes manejar colecciones únicas de datos con facilidad, asegurando que tus aplicaciones sean tanto eficientes como lógicamente sólidas.
