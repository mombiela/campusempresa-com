¿Qué es Flutter?

Flutter es un kit de herramientas de desarrollo de software de interfaz de usuario de código abierto creado por Google. Se utiliza para desarrollar aplicaciones para Android, iOS, Linux, Mac, Windows, Google Fuchsia y la web desde una única base de código.

Características Clave de Flutter

  • Base de Código Única: Escribe una vez, ejecuta en cualquier lugar.
  • Desarrollo Rápido: Hot reload te permite ver los cambios al instante.
  • Interfaz de Usuario Expresiva y Flexible: Conjunto rico de widgets personalizables.
  • Rendimiento Nativo: Se compila a código ARM nativo para una ejecución rápida.

Configuración de Flutter

Antes de comenzar a desarrollar con Flutter, necesitas configurar tu entorno de desarrollo.

Pasos de Instalación

  1. Descargar Flutter SDK: Visita el sitio web oficial de Flutter y descarga el SDK para tu sistema operativo.
  2. Extraer el SDK: Descomprime el archivo descargado y colócalo en la ubicación deseada.
  3. Actualizar la Ruta: Añade el directorio bin de Flutter a la variable PATH de tu sistema.
  4. Ejecutar Flutter Doctor: Abre una terminal y ejecuta flutter doctor para verificar si necesitas instalar alguna dependencia.
$ flutter doctor

Ejemplo: Configuración en Windows

  1. Descarga el SDK de Flutter desde el sitio web oficial.
  2. Extrae el archivo zip y colócalo en C:\src\flutter.
  3. Añade C:\src\flutter\bin a tu variable de entorno PATH.
  4. Abre el Símbolo del Sistema y ejecuta flutter doctor.

Creando Tu Primera Aplicación Flutter

Vamos a crear una aplicación Flutter simple para entender los conceptos básicos.

Pasos para Crear una Aplicación Flutter

  1. Crear un Nuevo Proyecto: Usa la CLI de Flutter para crear un nuevo proyecto. 
    $ flutter create my_first_app
  2. Navegar al Directorio del Proyecto: 
    $ cd my_first_app
  3. Ejecutar la Aplicación: Usa el siguiente comando para ejecutar tu aplicación en un emulador o dispositivo conectado. 
    $ flutter run

Entendiendo la Estructura del Proyecto

  • lib/main.dart: El punto de entrada principal de la aplicación.
  • pubspec.yaml: Archivo de configuración del proyecto, incluyendo dependencias.

Ejemplo: Aplicación Flutter Básica

Aquí tienes una aplicación Flutter simple que muestra "¡Hola, Mundo!".

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: Text('Bienvenido a Flutter'),
        ),
        body: Center(
          child: Text('¡Hola, Mundo!'),
        ),
      ),
    );
  }
}

Explicación

  • main(): El punto de entrada de la aplicación.
  • MyApp: Un widget sin estado que representa la aplicación.
  • MaterialApp: Un widget de conveniencia que envuelve varios widgets comúnmente requeridos para aplicaciones de diseño material.
  • Scaffold: Implementa la estructura visual básica del diseño material.
  • AppBar: Una barra de aplicaciones de diseño material.
  • Center: Centra su widget hijo.
  • Text: Un widget que muestra una cadena de texto.

Conclusión

En esta sección, presentamos Flutter, configuramos el entorno de desarrollo y creamos una aplicación Flutter básica. La capacidad de Flutter para usar una única base de código para múltiples plataformas, combinada con su interfaz de usuario expresiva y su rápido ciclo de desarrollo, lo convierte en una herramienta poderosa para el desarrollo de aplicaciones modernas. A medida que avances en este curso, profundizarás en las capacidades de Flutter y aprenderás a construir aplicaciones complejas y de alto rendimiento.
