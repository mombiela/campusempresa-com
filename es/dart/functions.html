Introducción a las Funciones

Las funciones son bloques de construcción fundamentales en la programación en Dart. Te permiten encapsular un bloque de código que realiza una tarea específica, el cual puede ser reutilizado a lo largo de tu programa.

Conceptos Clave

  • Definición de Función: Cómo definir una función.
  • Invocación de Función: Cómo llamar a una función.
  • Parámetros y Argumentos: Cómo pasar datos a las funciones.
  • Valores de Retorno: Cómo las funciones pueden devolver datos.
  • Funciones Anónimas: Funciones sin nombre.
  • Funciones Flecha: Una forma concisa de escribir funciones.
  • Funciones de Orden Superior: Funciones que toman otras funciones como parámetros o las devuelven.

Definición e Invocación de Función

Para definir una función en Dart, usas la palabra clave void si la función no devuelve un valor, o especificas un tipo de retorno si lo hace.

// Función que no devuelve un valor
void greet() {
  print('¡Hola, Mundo!');
}

// Función que devuelve un valor
int add(int a, int b) {
  return a + b;
}

Para llamar a una función, simplemente usa su nombre seguido de paréntesis.

void main() {
  greet(); // Salida: ¡Hola, Mundo!
  int sum = add(3, 4);
  print(sum); // Salida: 7
}

Parámetros y Argumentos

Las funciones pueden tomar parámetros, los cuales se especifican en la definición de la función. Al llamar a la función, proporcionas argumentos.

void printMessage(String message) {
  print(message);
}

void main() {
  printMessage('¡Hola, Dart!'); // Salida: ¡Hola, Dart!
}

Parámetros Opcionales

Dart soporta parámetros opcionales, los cuales pueden ser posicionales o nombrados.

Parámetros Opcionales Posicionales

Usa corchetes [] para definir parámetros opcionales posicionales.

void greet([String name = 'Invitado']) {
  print('¡Hola, $name!');
}

void main() {
  greet(); // Salida: ¡Hola, Invitado!
  greet('Alicia'); // Salida: ¡Hola, Alicia!
}

Parámetros Opcionales Nombrados

Usa llaves {} para definir parámetros opcionales nombrados.

void greet({String name = 'Invitado'}) {
  print('¡Hola, $name!');
}

void main() {
  greet(); // Salida: ¡Hola, Invitado!
  greet(name: 'Alicia'); // Salida: ¡Hola, Alicia!
}

Valores de Retorno

Las funciones pueden devolver valores usando la palabra clave return.

double multiply(double a, double b) {
  return a * b;
}

void main() {
  double result = multiply(2.5, 4.0);
  print(result); // Salida: 10.0
}

Funciones Anónimas

Las funciones anónimas, también conocidas como funciones lambda o en línea, son funciones sin nombre. A menudo se usan como argumentos para funciones de orden superior.

void main() {
  var list = ['manzana', 'banana', 'cereza'];
  list.forEach((item) {
    print(item);
  });
}

Funciones Flecha

Las funciones flecha proporcionan una sintaxis concisa para funciones que contienen una sola expresión.

int square(int num) => num * num;

void main() {
  print(square(4)); // Salida: 16
}

Funciones de Orden Superior

Las funciones de orden superior son funciones que toman otras funciones como parámetros o las devuelven.

void printElement(int element) {
  print(element);
}

void main() {
  var list = [1, 2, 3, 4];
  list.forEach(printElement); // Salida: 1 2 3 4
}

Conclusión

Las funciones en Dart son herramientas versátiles y poderosas que te ayudan a organizar y reutilizar el código de manera eficiente. Desde definiciones básicas de funciones hasta conceptos avanzados como funciones de orden superior, entender cómo trabajar con funciones es crucial para una programación efectiva en Dart. Practica definiendo, invocando y utilizando diferentes tipos de funciones para volverte competente en Dart.
