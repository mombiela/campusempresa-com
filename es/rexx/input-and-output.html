Introducción

Las operaciones de entrada y salida (I/O) son fundamentales en cualquier lenguaje de programación, permitiendo que los programas interactúen con los usuarios y otros sistemas. En REXX, las operaciones de I/O son directas y fáciles de aprender, lo que lo convierte en un excelente lenguaje tanto para principiantes como para usuarios avanzados.

Entrada Básica

Usando la Instrucción `PULL`

La instrucción PULL se utiliza para leer la entrada del usuario. Espera a que el usuario escriba algo y luego almacena la entrada en una variable.

/* Ejemplo de uso de PULL para obtener entrada del usuario */
say "Por favor, ingrese su nombre:"
pull nombre
say "¡Hola, " nombre "!"
  • Explicación:
    • say "Por favor, ingrese su nombre:" muestra un mensaje al usuario.
    • pull nombre espera a que el usuario escriba algo y lo almacena en la variable nombre.
    • say "¡Hola, " nombre "!" muestra un saludo usando la entrada.

Salida Básica

Usando la Instrucción `SAY`

La instrucción SAY se utiliza para mostrar salida al usuario.

/* Ejemplo de uso de SAY para mostrar salida */
say "¡Bienvenido al curso de programación REXX!"
  • Explicación:
    • say "¡Bienvenido al curso de programación REXX!" imprime el mensaje en la pantalla.

Entrada y Salida de Archivos

Leyendo desde un Archivo

Para leer desde un archivo, se utiliza la función LINEIN. Esta función lee una línea de un archivo.

/* Ejemplo de lectura desde un archivo */
nombreArchivo = "ejemplo.txt"
archivo = open(nombreArchivo, "r")
if archivo = "" then do
  say "Error: No se pudo abrir el archivo."
  exit
end

linea = linein(archivo)
do while linea \= ""
  say linea
  linea = linein(archivo)
end

call close archivo
  • Explicación:
    • nombreArchivo = "ejemplo.txt" especifica el archivo a leer.
    • archivo = open(nombreArchivo, "r") abre el archivo para lectura.
    • if archivo = "" then do verifica si el archivo se abrió correctamente.
    • linea = linein(archivo) lee una línea del archivo.
    • do while linea \= "" recorre el archivo hasta el final.
    • say linea imprime cada línea en la pantalla.
    • call close archivo cierra el archivo.

Escribiendo en un Archivo

Para escribir en un archivo, se utiliza la función LINEOUT. Esta función escribe una línea en un archivo.

/* Ejemplo de escritura en un archivo */
nombreArchivo = "salida.txt"
archivo = open(nombreArchivo, "w")
if archivo = "" then do
  say "Error: No se pudo abrir el archivo."
  exit
end

linea = "Esta es una línea de texto."
call lineout archivo, linea

call close archivo
  • Explicación:
    • nombreArchivo = "salida.txt" especifica el archivo para escribir.
    • archivo = open(nombreArchivo, "w") abre el archivo para escritura.
    • if archivo = "" then do verifica si el archivo se abrió correctamente.
    • linea = "Esta es una línea de texto." especifica la línea a escribir.
    • call lineout archivo, linea escribe la línea en el archivo.
    • call close archivo cierra el archivo.

Operaciones Avanzadas de I/O

Usando Streams

REXX proporciona funciones de stream para operaciones de archivo más avanzadas. La función STREAM se puede usar para consultar y manipular streams de archivos.

/* Ejemplo de uso de la función STREAM */
nombreArchivo = "ejemplo.txt"
archivo = open(nombreArchivo, "r")
if archivo = "" then do
  say "Error: No se pudo abrir el archivo."
  exit
end

posicion = stream(archivo, "seek", "end")
say "El tamaño del archivo es:" posicion "bytes."

call close archivo
  • Explicación:
    • posicion = stream(archivo, "seek", "end") mueve el puntero del archivo al final y devuelve el tamaño del archivo.
    • say "El tamaño del archivo es:" posicion "bytes." imprime el tamaño del archivo.

Conclusión

Entender las operaciones de entrada y salida en REXX es crucial para crear programas interactivos y útiles. Al dominar la I/O básica con PULL y SAY, la I/O de archivos con LINEIN y LINEOUT, y las operaciones avanzadas con STREAM, puedes manejar una amplia gama de tareas en tus programas REXX. Practica estos conceptos con varios ejemplos para volverte competente en las operaciones de I/O en REXX.
