Introducción a REXX

REXX (Restructured Extended Executor) es un lenguaje de programación que es fácil de aprender y usar. Está diseñado tanto para principiantes como para programadores experimentados. Esta sección cubrirá la sintaxis y estructura básica de los programas REXX, proporcionando una base para temas más avanzados.

Conceptos Clave

  • Insensibilidad a Mayúsculas y Minúsculas: REXX no distingue entre mayúsculas y minúsculas, lo que significa que say, SAY y Say se tratan de la misma manera.
  • Espacios en Blanco: Los espacios en blanco generalmente se ignoran en REXX, haciendo el código más legible.
  • Comentarios: Los comentarios se utilizan para explicar el código y se ignoran durante la ejecución.

Estructura Básica del Programa

Un programa REXX típicamente consiste en una secuencia de instrucciones que se ejecutan en orden. Aquí hay un ejemplo simple:

/* Este es un comentario */
say "¡Hola, Mundo!"

Comentarios

  • Los comentarios de una sola línea comienzan con /* y terminan con */.
  • Los comentarios pueden abarcar varias líneas.
/* Este es un 
   comentario de varias líneas */

Variables

  • Las variables en REXX son de tipo dinámico y no necesitan ser declaradas antes de su uso.
  • Los nombres de las variables pueden contener letras, números y caracteres especiales como . y _.
nombre = "John Doe"
edad = 30
say "Nombre:" nombre
say "Edad:" edad

Operadores

  • Operadores Aritméticos: +, -, *, /, // (división entera), ** (exponenciación)
  • Operadores de Comparación: =, <>, >, <, >=, <=
  • Operadores Lógicos: & (AND), | (OR), \ (NOT)
a = 10
b = 20
c = a + b
say "Suma:" c

Estructuras de Control

If-Else

La declaración if-else se utiliza para la ejecución condicional.

edad = 18
if edad >= 18 then
    say "Eres un adulto."
else
    say "Eres un menor."

Select-When

La estructura select-when se utiliza para múltiples condiciones.

calificación = "B"
select
    when calificación = "A" then say "¡Excelente!"
    when calificación = "B" then say "¡Buen trabajo!"
    when calificación = "C" then say "Aprobaste."
    otherwise say "Mejor suerte la próxima vez."
end

Bucles

  • Do-Loop: Utilizado para iteración.
do i = 1 to 5
    say "Iteración:" i
end
  • Do While/Until: Utilizado para bucles condicionales.
i = 1
do while i <= 5
    say "Iteración del Bucle While:" i
    i = i + 1
end

i = 1
do until i > 5
    say "Iteración del Bucle Until:" i
    i = i + 1
end

Funciones Integradas

REXX proporciona varias funciones integradas para tareas comunes.

Funciones de Cadena

  • Length: Devuelve la longitud de una cadena.
str = "Hola"
say length(str)  /* Salida: 5 */
  • Substr: Extrae una subcadena.
str = "¡Hola, Mundo!"
say substr(str, 1, 5)  /* Salida: Hola */

Funciones Numéricas

  • Abs: Devuelve el valor absoluto.
num = -10
say abs(num)  /* Salida: 10 */
  • Max/Min: Devuelve el valor máximo o mínimo.
say max(10, 20, 30)  /* Salida: 30 */
say min(10, 20, 30)  /* Salida: 10 */

Conclusión

Entender la sintaxis y estructura básica de REXX es crucial para escribir programas efectivos. Esta sección cubrió lo esencial, incluyendo comentarios, variables, operadores, estructuras de control y funciones integradas. Con esta base, ahora estás preparado para profundizar en temas y características más avanzadas del lenguaje de programación REXX.
