La manipulación de archivos es una habilidad esencial para cualquier programador. En esta sección, exploraremos cómo manejar archivos usando REXX, desde operaciones básicas hasta técnicas más avanzadas.

Introducción a la Manipulación de Archivos

La manipulación de archivos involucra varias operaciones como leer archivos, escribir archivos y gestionar atributos de archivos. En REXX, estas operaciones se realizan utilizando funciones y comandos integrados.

Conceptos Clave

  • Manejo de Archivos: Abrir, leer, escribir y cerrar archivos.
  • Funciones de E/S de Archivos: Funciones proporcionadas por REXX para entrada y salida de archivos.
  • Manejo de Errores: Gestionar errores que ocurren durante las operaciones de archivos.

Operaciones Básicas de Archivos

Abrir y Cerrar Archivos

Para realizar cualquier operación en un archivo, primero necesitas abrirlo. En REXX, los archivos se abren típicamente usando la función OPEN y se cierran usando la función CLOSE.

Ejemplo: Abrir y Cerrar un Archivo

/* Abrir un archivo para lectura */
file = 'example.txt'
fileHandle = open(file, 'r')

/* Realizar operaciones en el archivo */

/* Cerrar el archivo */
call close(fileHandle)

Leer Archivos

Leer un archivo se puede hacer línea por línea o en bloques. La función LINEIN se usa comúnmente para leer líneas de un archivo.

Ejemplo: Leer un Archivo Línea por Línea

file = 'example.txt'
fileHandle = open(file, 'r')

do while lines(fileHandle) > 0
    line = linein(fileHandle)
    say line
end

call close(fileHandle)

Escribir en Archivos

Escribir en un archivo se logra usando la función LINEOUT. Puedes escribir líneas en un archivo una a la vez.

Ejemplo: Escribir en un Archivo

file = 'output.txt'
fileHandle = open(file, 'w')

call lineout fileHandle, 'Esta es la primera línea.'
call lineout fileHandle, 'Esta es la segunda línea.'

call close(fileHandle)

Operaciones Avanzadas de Archivos

Añadir a Archivos

Añadir datos a un archivo existente se puede hacer usando la función OPEN con el modo de añadir (a).

Ejemplo: Añadir a un Archivo

file = 'output.txt'
fileHandle = open(file, 'a')

call lineout fileHandle, 'Esta línea se añade.'

call close(fileHandle)

Manejo de Errores en Operaciones de Archivos

El manejo adecuado de errores asegura que tu programa pueda manejar de manera elegante problemas como archivos faltantes o errores de permisos.

Ejemplo: Manejo de Errores

file = 'nonexistent.txt'
fileHandle = open(file, 'r')

if fileHandle = '' then do
    say 'Error: No se puede abrir el archivo.'
    exit
end

/* Realizar operaciones en el archivo */

call close(fileHandle)

Funciones de Gestión de Archivos

REXX proporciona varias funciones para gestionar archivos, como DELETE para eliminar un archivo y EXISTS para verificar si un archivo existe.

Ejemplo: Eliminar un Archivo

file = 'output.txt'

if exists(file) then do
    call delete file
    say 'Archivo eliminado exitosamente.'
end
else do
    say 'El archivo no existe.'
end

Resumen

En esta sección, cubrimos los conceptos básicos y algunas técnicas avanzadas para la manipulación de archivos en REXX. Aprendiste cómo:

  • Abrir y cerrar archivos.
  • Leer y escribir archivos.
  • Añadir datos a archivos.
  • Manejar errores durante las operaciones de archivos.
  • Gestionar archivos usando funciones integradas.

Entender estos conceptos te permitirá gestionar archivos de manera efectiva en tus programas REXX, haciendo tus scripts más robustos y versátiles.
