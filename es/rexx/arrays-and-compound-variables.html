Introducción

En esta sección, exploraremos los arrays y las variables compuestas en REXX. Los arrays y las variables compuestas son esenciales para manejar colecciones de datos de manera eficiente. Cubriremos los conceptos básicos de los arrays, cómo crearlos y manipularlos, y profundizaremos en temas más avanzados como arrays multidimensionales y arrays dinámicos.

Conceptos Clave

  • Arrays: Una colección de elementos que se pueden acceder usando un índice.
  • Variables Compuestas: Variables que pueden contener múltiples valores, a menudo usadas para simular arrays en REXX.
  • Indexación: El método de acceder a elementos individuales dentro de un array o variable compuesta.

Arrays Simples en REXX

REXX no tiene soporte incorporado para arrays tradicionales. En su lugar, usa variables compuestas para simular arrays.

Creación y Uso de Variables Compuestas

Las variables compuestas en REXX se crean usando un nombre base seguido de uno o más subíndices.

/* Array simple usando variables compuestas */
array.1 = "Manzana"
array.2 = "Banana"
array.3 = "Cereza"

say array.1  /* Salida: Manzana */
say array.2  /* Salida: Banana */
say array.3  /* Salida: Cereza */

Iterando a Través de Arrays

Puedes iterar a través de arrays usando un bucle DO.

/* Iterar a través del array */
do i = 1 to 3
    say array.i
end

Arrays Multidimensionales

REXX te permite crear arrays multidimensionales usando múltiples subíndices.

Ejemplo de un Array Bidimensional

/* Array bidimensional */
matrix.1.1 = "A1"
matrix.1.2 = "A2"
matrix.2.1 = "B1"
matrix.2.2 = "B2"

say matrix.1.1  /* Salida: A1 */
say matrix.2.2  /* Salida: B2 */

Iterando a Través de Arrays Multidimensionales

/* Iterar a través de un array bidimensional */
do i = 1 to 2
    do j = 1 to 2
        say matrix.i.j
    end
end

Arrays Dinámicos

REXX te permite agregar elementos dinámicamente a los arrays.

Agregando Elementos Dinámicamente

/* Agregando elementos dinámicamente */
array.4 = "Dátil"
array.5 = "Elderberry"

do i = 1 to 5
    say array.i
end

Ejemplo Práctico: Almacenando y Recuperando Datos

Vamos a crear un programa simple que almacene y recupere nombres de estudiantes y sus calificaciones.

/* Programa para almacenar y recuperar calificaciones de estudiantes */
students.1 = "Alicia"
grades.1 = 85
students.2 = "Bob"
grades.2 = 90
students.3 = "Carlos"
grades.3 = 78

/* Recuperar y mostrar nombres de estudiantes y calificaciones */
do i = 1 to 3
    say students.i "obtuvo" grades.i
end

Conclusión

En esta sección, hemos cubierto los conceptos básicos de arrays y variables compuestas en REXX. Aprendimos cómo crear y manipular arrays simples y multidimensionales usando variables compuestas. También exploramos cómo agregar elementos dinámicamente a los arrays e iterar a través de ellos. Entender estos conceptos es crucial para manejar colecciones de datos de manera eficiente en programas REXX. Practica estos ejemplos e intenta crear tus propios arrays para familiarizarte más con estos conceptos.
