Introducción

La manipulación de cadenas es una habilidad fundamental en la programación en REXX. Este tema cubrirá varias técnicas y funciones para manejar cadenas, desde operaciones básicas hasta manipulaciones avanzadas.

Operaciones Básicas de Cadenas

Concatenación

  • Concepto: Combinar dos o más cadenas en una sola.
  • Ejemplo: 
    string1 = "Hola"
string2 = "Mundo"
combinedString = string1 || " " || string2
say combinedString  /* Salida: Hola Mundo */

Extracción de Subcadenas

  • Concepto: Extraer una parte de una cadena.
  • Ejemplo: 
    fullString = "Hola Mundo"
subString = substr(fullString, 1, 5)
say subString  /* Salida: Hola */

Longitud de la Cadena

  • Concepto: Encontrar la longitud de una cadena.
  • Ejemplo: 
    myString = "Hola"
length = length(myString)
say length  /* Salida: 4 */

Operaciones Intermedias de Cadenas

Recortar Cadenas

  • Concepto: Eliminar espacios al principio y/o al final.
  • Ejemplo: 
    myString = "  Hola Mundo  "
trimmedString = strip(myString)
say trimmedString  /* Salida: Hola Mundo */

Cambio de Mayúsculas y Minúsculas

  • Concepto: Convertir cadenas a mayúsculas o minúsculas.
  • Ejemplo: 
    myString = "Hola Mundo"
upperCaseString = translate(myString)
lowerCaseString = translate(myString, "abcdefghijklmnopqrstuvwxyz", "ABCDEFGHIJKLMNOPQRSTUVWXYZ")
say upperCaseString  /* Salida: HOLA MUNDO */
say lowerCaseString  /* Salida: hola mundo */

Reemplazo de Subcadenas

  • Concepto: Reemplazar partes de una cadena con otra cadena.
  • Ejemplo: 
    myString = "Hola Mundo"
newString = overlay("REXX", myString, 6)
say newString  /* Salida: Hola REXX */

Operaciones Avanzadas de Cadenas

Coincidencia de Patrones y Sustitución

  • Concepto: Encontrar y reemplazar patrones dentro de cadenas.
  • Ejemplo: 
    myString = "El rápido zorro marrón salta sobre el perro perezoso"
newString = changestr("zorro", "gato", myString)
say newString  /* Salida: El rápido gato marrón salta sobre el perro perezoso */

División de Cadenas

  • Concepto: Dividir una cadena en múltiples partes basadas en un delimitador.
  • Ejemplo: 
    myString = "manzana,plátano,cereza"
parse var myString fruit1 "," fruit2 "," fruit3
say fruit1  /* Salida: manzana */
say fruit2  /* Salida: plátano */
say fruit3  /* Salida: cereza */

Formateo de Cadenas

  • Concepto: Formatear cadenas a una estructura o disposición específica.
  • Ejemplo: 
    name = "Juan"
age = 30
formattedString = "Nombre: " || name || ", Edad: " || age
say formattedString  /* Salida: Nombre: Juan, Edad: 30 */

Resumen

La manipulación de cadenas en REXX implica una variedad de operaciones, desde la concatenación básica y la extracción de subcadenas hasta la coincidencia de patrones avanzada y el formateo de cadenas. Dominar estas técnicas es esencial para una programación efectiva en REXX. Practica estos conceptos con diferentes ejemplos para volverte competente en el manejo de cadenas en REXX.
