Introducción

En esta sección, exploraremos cómo trabajar con listas y cuadrículas en Flutter. Las listas y cuadrículas son componentes esenciales para mostrar colecciones de datos de manera estructurada y visualmente atractiva. Al final de esta sección, serás capaz de crear listas y cuadrículas dinámicas, personalizar su apariencia y manejar interacciones del usuario.

Listas en Flutter

ListView Básico

El widget ListView es el widget más comúnmente utilizado para mostrar una lista de elementos en Flutter. Puede ser utilizado para crear listas tanto estáticas como dinámicas.

ListView Estático

Un ListView estático es útil cuando tienes un número fijo de elementos para mostrar.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Static ListView')),
        body: ListView(
          children: <Widget>[
            ListTile(
              leading: Icon(Icons.map),
              title: Text('Map'),
            ),
            ListTile(
              leading: Icon(Icons.photo_album),
              title: Text('Album'),
            ),
            ListTile(
              leading: Icon(Icons.phone),
              title: Text('Phone'),
            ),
          ],
        ),
      ),
    );
  }
}

ListView Dinámico

Un ListView dinámico es útil cuando el número de elementos no es fijo y puede cambiar durante el tiempo de ejecución.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  final List<String> items = List<String>.generate(100, (i) => "Item $i");

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Dynamic ListView')),
        body: ListView.builder(
          itemCount: items.length,
          itemBuilder: (context, index) {
            return ListTile(
              title: Text('${items[index]}'),
            );
          },
        ),
      ),
    );
  }
}

Personalizando Elementos de la Lista

Puedes personalizar la apariencia de los elementos de la lista utilizando varios widgets y propiedades.

ListTile(
  leading: Icon(Icons.map),
  title: Text('Map'),
  subtitle: Text('Navigate the world'),
  trailing: Icon(Icons.navigate_next),
  onTap: () {
    // Manejar evento de toque
  },
);

Cuadrículas en Flutter

GridView Básico

El widget GridView se utiliza para mostrar elementos en un diseño de cuadrícula. Puede ser utilizado para crear cuadrículas tanto estáticas como dinámicas.

GridView Estático

Un GridView estático es útil cuando tienes un número fijo de elementos para mostrar.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Static GridView')),
        body: GridView.count(
          crossAxisCount: 2,
          children: <Widget>[
            Container(
              color: Colors.red,
              child: Center(child: Text('Item 1')),
            ),
            Container(
              color: Colors.blue,
              child: Center(child: Text('Item 2')),
            ),
            Container(
              color: Colors.green,
              child: Center(child: Text('Item 3')),
            ),
            Container(
              color: Colors.yellow,
              child: Center(child: Text('Item 4')),
            ),
          ],
        ),
      ),
    );
  }
}

GridView Dinámico

Un GridView dinámico es útil cuando el número de elementos no es fijo y puede cambiar durante el tiempo de ejecución.

import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  final List<String> items = List<String>.generate(100, (i) => "Item $i");

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Dynamic GridView')),
        body: GridView.builder(
          gridDelegate: SliverGridDelegateWithFixedCrossAxisCount(
            crossAxisCount: 2,
          ),
          itemCount: items.length,
          itemBuilder: (context, index) {
            return Card(
              child: Center(child: Text('${items[index]}')),
            );
          },
        ),
      ),
    );
  }
}

Personalizando Elementos de la Cuadrícula

Puedes personalizar la apariencia de los elementos de la cuadrícula utilizando varios widgets y propiedades.

Card(
  child: Column(
    mainAxisAlignment: MainAxisAlignment.center,
    children: <Widget>[
      Icon(Icons.map, size: 50),
      Text('Map'),
    ],
  ),
);

Conclusión

En esta sección, cubrimos los conceptos básicos de trabajar con listas y cuadrículas en Flutter. Aprendimos cómo crear listas y cuadrículas tanto estáticas como dinámicas, y cómo personalizar su apariencia. Las listas y cuadrículas son herramientas poderosas para mostrar datos de manera estructurada y visualmente atractiva, y dominarlas mejorará enormemente tus habilidades de desarrollo en Flutter.
