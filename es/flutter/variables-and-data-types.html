Introducción

En esta sección, cubriremos los conceptos fundamentales de variables y tipos de datos en Flutter. Entender estos conceptos básicos es crucial para construir aplicaciones Flutter robustas y eficientes. Comenzaremos con lo básico de las variables y luego pasaremos a los diferentes tipos de datos disponibles en Dart, el lenguaje de programación utilizado en Flutter.

Variables

Las variables se utilizan para almacenar datos que pueden ser referenciados y manipulados en un programa. En Dart, las variables se declaran usando las palabras clave var, final o const.

Declaración de Variables

  • var: El tipo de la variable es inferido por el compilador.
  • final: Una variable cuyo valor no puede ser cambiado una vez que se establece.
  • const: Una constante en tiempo de compilación.
void main() {
  var name = 'Flutter'; // Tipo String inferido
  final age = 5; // Tipo int inferido, no puede ser cambiado
  const pi = 3.14; // Tipo double, constante en tiempo de compilación

  print(name); // Salida: Flutter
  print(age); // Salida: 5
  print(pi); // Salida: 3.14
}

Tipos de Datos

Dart soporta varios tipos de datos, que pueden ser categorizados ampliamente en los siguientes:

  • Números
  • Cadenas
  • Booleanos
  • Listas
  • Mapas

Números

Dart tiene dos tipos de números: int y double.

void main() {
  int age = 30;
  double height = 5.9;

  print(age); // Salida: 30
  print(height); // Salida: 5.9
}

Cadenas

Las cadenas son una secuencia de caracteres. Pueden ser creadas usando comillas simples o dobles.

void main() {
  String greeting = 'Hello, Flutter!';
  String name = "John Doe";

  print(greeting); // Salida: Hello, Flutter!
  print(name); // Salida: John Doe
}

Booleanos

Los booleanos representan valores verdaderos o falsos.

void main() {
  bool isFlutterAwesome = true;
  bool isLearningFun = true;

  print(isFlutterAwesome); // Salida: true
  print(isLearningFun); // Salida: true
}

Listas

Las listas son colecciones ordenadas de objetos. Son similares a los arreglos en otros lenguajes de programación.

void main() {
  List<String> fruits = ['Apple', 'Banana', 'Mango'];
  
  print(fruits); // Salida: [Apple, Banana, Mango]
  print(fruits[0]); // Salida: Apple
}

Mapas

Los mapas son colecciones de pares clave-valor. Cada clave en un mapa debe ser única.

void main() {
  Map<String, String> capitals = {
    'USA': 'Washington, D.C.',
    'India': 'New Delhi',
    'Japan': 'Tokyo'
  };

  print(capitals); // Salida: {USA: Washington, D.C., India: New Delhi, Japan: Tokyo}
  print(capitals['India']); // Salida: New Delhi
}

Resumen

En esta sección, cubrimos lo básico de las variables y tipos de datos en Flutter. Aprendimos cómo declarar variables usando var, final y const. También exploramos los diferentes tipos de datos disponibles en Dart, incluyendo números, cadenas, booleanos, listas y mapas. Entender estos fundamentos es esencial para construir cualquier aplicación Flutter. En la próxima sección, profundizaremos en temas más avanzados y cómo usar efectivamente estos tipos de datos en aplicaciones del mundo real.
