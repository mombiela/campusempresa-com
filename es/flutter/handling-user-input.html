Introducción

El manejo de la entrada del usuario es un aspecto fundamental de cualquier aplicación interactiva. En Flutter, la entrada del usuario puede provenir de varias fuentes, como campos de texto, botones y gestos. Esta sección te guiará desde las técnicas básicas hasta las avanzadas para gestionar la entrada del usuario en Flutter.

Entrada de Texto

Widget TextField

El widget TextField es la forma principal de obtener entrada de texto de los usuarios en Flutter.

  • Uso Básico:

    TextField(
  decoration: InputDecoration(
    border: OutlineInputBorder(),
    labelText: 'Introduce tu nombre',
  ),
)

  • Manejo de la Entrada: Para manejar la entrada, puedes usar un TextEditingController.

    final TextEditingController _controller = TextEditingController();

TextField(
  controller: _controller,
  decoration: InputDecoration(
    border: OutlineInputBorder(),
    labelText: 'Introduce tu nombre',
  ),
)

// Acceder al valor de entrada
String inputValue = _controller.text;

Validación de Formularios

Para formularios más complejos, Flutter proporciona el widget Form y los widgets FormField.

  • Ejemplo Básico de Formulario: 
    final _formKey = GlobalKey<FormState>();

Form(
  key: _formKey,
  child: Column(
    children: <Widget>[
      TextFormField(
        decoration: InputDecoration(labelText: 'Introduce tu nombre'),
        validator: (value) {
          if (value == null || value.isEmpty) {
            return 'Por favor, introduce algún texto';
          }
          return null;
        },
      ),
      ElevatedButton(
        onPressed: () {
          if (_formKey.currentState!.validate()) {
            // Procesar datos
          }
        },
        child: Text('Enviar'),
      ),
    ],
  ),
)

Entrada de Botones

ElevatedButton

El widget ElevatedButton se usa para crear un botón que puede desencadenar acciones.

  • Uso Básico: 
    ElevatedButton(
  onPressed: () {
    print('Botón presionado');
  },
  child: Text('Presióname'),
)

Manejo de Estados del Botón

Puedes gestionar el estado de un botón para habilitarlo o deshabilitarlo según ciertas condiciones.

  • Ejemplo: 
    bool _isButtonDisabled = true;

ElevatedButton(
  onPressed: _isButtonDisabled ? null : () {
    print('Botón presionado');
  },
  child: Text('Presióname'),
)

Detección de Gestos

GestureDetector

El widget GestureDetector te permite detectar varios gestos como toques, deslizamientos y pulsaciones largas.

  • Detección Básica de Toques: 
    GestureDetector(
  onTap: () {
    print('Tocado');
  },
  child: Container(
    color: Colors.blue,
    width: 100,
    height: 100,
  ),
)

Gestos Avanzados

Puedes detectar gestos más complejos como doble toque, pulsación larga y arrastre.

  • Ejemplo: 
    GestureDetector(
  onDoubleTap: () {
    print('Doble Tocado');
  },
  onLongPress: () {
    print('Pulsación Larga');
  },
  onPanUpdate: (details) {
    print('Arrastrado: ${details.delta}');
  },
  child: Container(
    color: Colors.blue,
    width: 100,
    height: 100,
  ),
)

Conclusión

El manejo de la entrada del usuario en Flutter es sencillo con el uso de widgets como TextField, ElevatedButton y GestureDetector. Al entender estas técnicas básicas y avanzadas, puedes crear aplicaciones interactivas y receptivas. Practica estos conceptos con varios ejemplos para obtener un sólido dominio de la gestión de la entrada del usuario en Flutter.
