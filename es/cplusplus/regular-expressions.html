Las expresiones regulares (regex) son herramientas poderosas para la coincidencia de patrones y el procesamiento de texto. En C++, la biblioteca <regex> proporciona soporte para trabajar con expresiones regulares. Este tema te guiará desde los conceptos básicos hasta el uso avanzado de expresiones regulares en C++.

Introducción a las Expresiones Regulares

  • Definición: Las expresiones regulares son secuencias de caracteres que definen patrones de búsqueda, principalmente para su uso en la coincidencia de patrones con cadenas.
  • Casos de Uso: Validación de entrada, búsqueda dentro de cadenas, reemplazo de subcadenas y análisis de texto.

Conceptos Básicos de las Expresiones Regulares

Incluir la Biblioteca Regex

Para usar expresiones regulares en C++, necesitas incluir el encabezado <regex>:

#include <regex>

Sintaxis Básica

  • Caracteres Literales: Coinciden consigo mismos.
  • Metacaracteres: Caracteres con significados especiales (por ejemplo, . para cualquier carácter, ^ para el inicio de una cadena, $ para el final de una cadena).

Ejemplo Simple

#include <iostream>
#include <regex>

int main() {
    std::string text = "¡Hola, Mundo!";
    std::regex pattern("Mundo");

    if (std::regex_search(text, pattern)) {
        std::cout << "¡Patrón encontrado!" << std::endl;
    } else {
        std::cout << "Patrón no encontrado." << std::endl;
    }

    return 0;
}

Explicación

  • std::regex: Representa una expresión regular compilada.
  • std::regex_search: Busca el patrón dentro del texto.

Patrones Comunes de Regex

  • Clases de Caracteres: [abc] coincide con cualquier carácter único a, b o c.
  • Clases de Caracteres Predefinidas: \d para dígitos, \w para caracteres de palabra, \s para espacios en blanco.
  • Cuantificadores: * (0 o más), + (1 o más), ? (0 o 1), {n} (exactamente n), {n,} (n o más), {n,m} (entre n y m).

Ejemplo con Cuantificadores

#include <iostream>
#include <regex>

int main() {
    std::string text = "El número es 12345.";
    std::regex pattern("\\d+");

    if (std::regex_search(text, pattern)) {
        std::cout << "¡Número encontrado!" << std::endl;
    } else {
        std::cout << "Número no encontrado." << std::endl;
    }

    return 0;
}

Expresiones Regulares Avanzadas

Grupos de Captura

  • Definición: Los paréntesis () se utilizan para crear grupos de captura.
  • Caso de Uso: Extraer partes específicas de una coincidencia.

Ejemplo con Grupos de Captura

#include <iostream>
#include <regex>

int main() {
    std::string text = "Mi correo es ejemplo@ejemplo.com.";
    std::regex pattern("(\\w+)@(\\w+\\.\\w+)");

    std::smatch matches;
    if (std::regex_search(text, matches, pattern)) {
        std::cout << "Nombre de usuario: " << matches[1] << std::endl;
        std::cout << "Dominio: " << matches[2] << std::endl;
    } else {
        std::cout << "No se encontró coincidencia." << std::endl;
    }

    return 0;
}

Explicación

  • std::smatch: Contiene los resultados de la búsqueda.
  • matches[n]: Accede al enésimo grupo de captura.

Reemplazo con Regex

  • Función: std::regex_replace reemplaza las ocurrencias del patrón con una cadena especificada.

Ejemplo de Reemplazo con Regex

#include <iostream>
#include <regex>

int main() {
    std::string text = "Tengo 2 manzanas y 3 naranjas.";
    std::regex pattern("\\d");
    std::string replacement = "#";

    std::string result = std::regex_replace(text, pattern, replacement);
    std::cout << "Resultado: " << result << std::endl;

    return 0;
}

Explicación

  • std::regex_replace: Reemplaza todas las ocurrencias del patrón con la cadena de reemplazo.

Consideraciones de Rendimiento

  • Compilación: Compilar una regex puede ser costoso. Reutiliza objetos regex compilados cuando sea posible.
  • Complejidad: Ten cuidado con los patrones complejos, ya que pueden llevar a problemas de rendimiento.

Conclusión

Las expresiones regulares son una herramienta versátil en C++ para el procesamiento de texto. Desde la coincidencia básica de patrones hasta la manipulación avanzada de texto, entender las regex puede mejorar significativamente tu capacidad para manejar cadenas de manera efectiva. Practica con diferentes patrones y casos de uso para volverte competente en el uso de expresiones regulares en tus programas en C++.
