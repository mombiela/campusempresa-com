Las plantillas son una característica poderosa en C++ que te permiten escribir código genérico y reutilizable. Permiten que las funciones y clases operen con tipos genéricos, que pueden ser especificados cuando la función o clase es instanciada. Esta sección te guiará desde los conceptos básicos de las plantillas hasta usos más avanzados.

Introducción a las Plantillas

Las plantillas te permiten definir funciones y clases con un marcador de posición para el tipo de dato. Esto hace que tu código sea más flexible y reutilizable.

Plantillas de Funciones

Las plantillas de funciones te permiten crear una función que puede operar en diferentes tipos de datos sin reescribir toda la función para cada tipo.

Ejemplo:

#include <iostream>
using namespace std;

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    cout << add(3, 4) << endl;       // Muestra 7
    cout << add(3.5, 4.5) << endl;   // Muestra 8.0
    return 0;
}
  • template <typename T>: Esta línea declara una plantilla con un marcador de posición de tipo T.
  • T add(T a, T b): Esta función toma dos parámetros de tipo T y devuelve un valor de tipo T.

Plantillas de Clases

Las plantillas de clases te permiten crear una clase que puede manejar cualquier tipo de dato.

Ejemplo:

#include <iostream>
using namespace std;

template <typename T>
class Box {
private:
    T value;
public:
    Box(T val) : value(val) {}
    T getValue() { return value; }
};

int main() {
    Box<int> intBox(123);
    Box<double> doubleBox(456.78);

    cout << intBox.getValue() << endl;       // Muestra 123
    cout << doubleBox.getValue() << endl;    // Muestra 456.78
    return 0;
}
  • template <typename T>: Esta línea declara una plantilla con un marcador de posición de tipo T.
  • Box<T>: Esta clase ahora puede manejar cualquier tipo de dato especificado cuando se crea un objeto.

Plantillas Avanzadas

Especialización de Plantillas

La especialización de plantillas te permite definir una implementación diferente de una plantilla para un tipo de dato específico.

Ejemplo:

#include <iostream>
using namespace std;

template <typename T>
class Box {
public:
    T value;
    Box(T val) : value(val) {}
    void print() {
        cout << "Plantilla genérica: " << value << endl;
    }
};

// Especialización para int
template <>
class Box<int> {
public:
    int value;
    Box(int val) : value(val) {}
    void print() {
        cout << "Plantilla especializada para int: " << value << endl;
    }
};

int main() {
    Box<double> doubleBox(3.14);
    Box<int> intBox(123);

    doubleBox.print();  // Muestra: Plantilla genérica: 3.14
    intBox.print();     // Muestra: Plantilla especializada para int: 123

    return 0;
}
  • template <> class Box<int>: Esta línea declara una versión especializada de la clase Box para el tipo int.
  • La clase especializada puede tener una implementación diferente de la plantilla genérica.

Parámetros de Plantillas

Las plantillas pueden tener parámetros que no son de tipo, lo que te permite pasar valores a las plantillas.

Ejemplo:

#include <iostream>
using namespace std;

template <typename T, int size>
class Array {
private:
    T arr[size];
public:
    void set(int index, T value) {
        if (index >= 0 && index < size) {
            arr[index] = value;
        }
    }
    T get(int index) {
        if (index >= 0 && index < size) {
            return arr[index];
        }
        return T(); // Devuelve el valor por defecto de T
    }
};

int main() {
    Array<int, 5> intArray;
    intArray.set(0, 10);
    intArray.set(1, 20);

    cout << intArray.get(0) << endl;  // Muestra 10
    cout << intArray.get(1) << endl;  // Muestra 20

    return 0;
}
  • template <typename T, int size>: Esta línea declara una plantilla con un parámetro de tipo T y un parámetro que no es de tipo size.
  • Array<int, 5>: Esto crea un objeto Array que puede contener 5 enteros.

Conclusión

Las plantillas en C++ proporcionan una forma poderosa de escribir código genérico y reutilizable. Te permiten crear funciones y clases que pueden operar con cualquier tipo de dato, reduciendo la redundancia y aumentando la flexibilidad. Al entender y utilizar las plantillas, puedes escribir programas en C++ más eficientes y mantenibles.
