Las declaraciones condicionales son fundamentales en la programación ya que te permiten ejecutar diferentes bloques de código basados en ciertas condiciones. En C++, las declaraciones condicionales principales son if, else if, else y switch.

Declaración If

La declaración if se usa para ejecutar un bloque de código si una condición especificada es verdadera.

Sintaxis

if (condition) {
    // código a ejecutar si la condición es verdadera
}

Ejemplo

#include <iostream>
using namespace std;

int main() {
    int number = 10;

    if (number > 5) {
        cout << "El número es mayor que 5" << endl;
    }

    return 0;
}

En este ejemplo, se imprimirá el mensaje "El número es mayor que 5" porque la condición number > 5 es verdadera.

Declaración Else

La declaración else se usa para ejecutar un bloque de código si la condición en la declaración if es falsa.

Sintaxis

if (condition) {
    // código a ejecutar si la condición es verdadera
} else {
    // código a ejecutar si la condición es falsa
}

Ejemplo

#include <iostream>
using namespace std;

int main() {
    int number = 3;

    if (number > 5) {
        cout << "El número es mayor que 5" << endl;
    } else {
        cout << "El número no es mayor que 5" << endl;
    }

    return 0;
}

En este ejemplo, se imprimirá el mensaje "El número no es mayor que 5" porque la condición number > 5 es falsa.

Declaración Else If

La declaración else if se usa para especificar una nueva condición a probar si la primera condición es falsa.

Sintaxis

if (condition1) {
    // código a ejecutar si la condición1 es verdadera
} else if (condition2) {
    // código a ejecutar si la condición2 es verdadera
} else {
    // código a ejecutar si todas las condiciones son falsas
}

Ejemplo

#include <iostream>
using namespace std;

int main() {
    int number = 7;

    if (number > 10) {
        cout << "El número es mayor que 10" << endl;
    } else if (number > 5) {
        cout << "El número es mayor que 5 pero menor o igual a 10" << endl;
    } else {
        cout << "El número es 5 o menos" << endl;
    }

    return 0;
}

En este ejemplo, se imprimirá el mensaje "El número es mayor que 5 pero menor o igual a 10" porque la condición number > 5 es verdadera.

Declaración Switch

La declaración switch se usa para ejecutar un bloque de código de múltiples opciones basadas en el valor de una variable.

Sintaxis

switch (expression) {
    case constant1:
        // código a ejecutar si la expresión es igual a constant1
        break;
    case constant2:
        // código a ejecutar si la expresión es igual a constant2
        break;
    // puedes tener cualquier número de declaraciones case
    default:
        // código a ejecutar si la expresión no coincide con ningún case
}

Ejemplo

#include <iostream>
using namespace std;

int main() {
    int day = 3;

    switch (day) {
        case 1:
            cout << "Lunes" << endl;
            break;
        case 2:
            cout << "Martes" << endl;
            break;
        case 3:
            cout << "Miércoles" << endl;
            break;
        case 4:
            cout << "Jueves" << endl;
            break;
        case 5:
            cout << "Viernes" << endl;
            break;
        case 6:
            cout << "Sábado" << endl;
            break;
        case 7:
            cout << "Domingo" << endl;
            break;
        default:
            cout << "Día inválido" << endl;
    }

    return 0;
}

En este ejemplo, se imprimirá el mensaje "Miércoles" porque el valor de day es 3.

Resumen

  • Declaración If: Ejecuta un bloque de código si una condición especificada es verdadera.
  • Declaración Else: Ejecuta un bloque de código si la condición en la declaración if es falsa.
  • Declaración Else If: Especifica una nueva condición a probar si la primera condición es falsa.
  • Declaración Switch: Ejecuta un bloque de código de múltiples opciones basadas en el valor de una variable.

Entender y usar las declaraciones condicionales de manera efectiva te permite controlar el flujo de tu programa y tomar decisiones basadas en diferentes condiciones. Practica escribiendo y combinando estas declaraciones para volverte competente en manejar varios escenarios en tus programas en C++.
