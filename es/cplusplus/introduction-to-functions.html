Parámetros de Función y Tipos de Retorno ►

¿Qué es una Función?

  • Una función es un bloque de código reutilizable que realiza una tarea específica.
  • Las funciones ayudan a descomponer problemas complejos en piezas más pequeñas y manejables.
  • Promueven la reutilización del código y mejoran la legibilidad.

Sintaxis de una Función

  • Una función en C++ consiste en un tipo de retorno, un nombre de función y una lista de parámetros encerrados entre paréntesis.
  • El cuerpo de la función está encerrado en llaves {}.

Ejemplo:

#include <iostream>

// Declaración de la función
int add(int a, int b);

int main() {
    int result = add(5, 3); // Llamada a la función
    std::cout << "La suma es: " << result << std::endl;
    return 0;
}

// Definición de la función
int add(int a, int b) {
    return a + b;
}

Declaración y Definición de Funciones

  • Declaración de Función (Prototipo): Informa al compilador sobre el nombre de la función, el tipo de retorno y los parámetros. Termina con un punto y coma.
  • Definición de Función: Contiene el cuerpo real de la función.

Ejemplo:

// Declaración
int multiply(int, int);

// Definición
int multiply(int x, int y) {
    return x * y;
}

Tipo de Retorno

  • El tipo de retorno especifica el tipo de valor que la función devolverá.
  • Si una función no devuelve ningún valor, su tipo de retorno es void.

Ejemplo:

void printMessage() {
    std::cout << "¡Hola, Mundo!" << std::endl;
}

Parámetros y Argumentos

  • Parámetros: Variables listadas en la declaración/definición de la función.
  • Argumentos: Valores pasados a la función cuando se llama.

Ejemplo:

void greet(std::string name) {
    std::cout << "Hola, " << name << "!" << std::endl;
}

int main() {
    greet("Alice"); // Argumento: "Alice"
    return 0;
}

Parámetros por Defecto

  • C++ permite que las funciones tengan valores de parámetros por defecto.
  • Si no se proporciona ningún argumento, se utiliza el valor por defecto.

Ejemplo:

void display(int x = 10) {
    std::cout << "Valor: " << x << std::endl;
}

int main() {
    display();    // Usa el valor por defecto 10
    display(5);   // Usa el valor proporcionado 5
    return 0;
}

Sobrecarga de Funciones

  • La sobrecarga de funciones permite que múltiples funciones tengan el mismo nombre con diferentes parámetros.
  • El compilador las diferencia por sus listas de parámetros.

Ejemplo:

void print(int i) {
    std::cout << "Entero: " << i << std::endl;
}

void print(double f) {
    std::cout << "Doble: " << f << std::endl;
}

void print(std::string s) {
    std::cout << "Cadena: " << s << std::endl;
}

int main() {
    print(10);
    print(3.14);
    print("Hola");
    return 0;
}

Funciones Inline

  • Las funciones inline se definen con la palabra clave inline.
  • Sugerir al compilador que inserte el cuerpo de la función donde se llama, para reducir la sobrecarga de una llamada a función.

Ejemplo:

inline int square(int x) {
    return x * x;
}

int main() {
    std::cout << "Cuadrado de 5: " << square(5) << std::endl;
    return 0;
}

Recursión

  • Una función que se llama a sí misma se conoce como función recursiva.
  • La recursión es útil para tareas que pueden definirse en términos de subtareas similares.

Ejemplo:

int factorial(int n) {
    if (n <= 1) return 1;
    else return n * factorial(n - 1);
}

int main() {
    std::cout << "Factorial de 5: " << factorial(5) << std::endl;
    return 0;
}

Resumen

  • Las funciones son esenciales para descomponer problemas complejos y promover la reutilización del código.
  • Comprender la declaración de funciones, definición, parámetros, tipos de retorno y sobrecarga es crucial.
  • Las funciones inline y la recursión son conceptos avanzados que ofrecen beneficios de rendimiento y soluciones elegantes a ciertos problemas.
  • Practica escribir y usar funciones para volverte competente en estructurar tus programas en C++ de manera efectiva.
