El multihilo es una característica poderosa en C++ que permite a un programa realizar múltiples tareas de manera concurrente. Este tema te guiará desde los conceptos básicos del multihilo hasta conceptos avanzados, proporcionando explicaciones claras, ejemplos y ejercicios.

Introducción al Multihilo

El multihilo implica ejecutar múltiples hilos simultáneamente dentro de un solo programa. Cada hilo se ejecuta de manera independiente, permitiendo la ejecución paralela de tareas.

Conceptos Clave

  • Hilo: La unidad más pequeña de un proceso que puede ser programada para su ejecución.
  • Concurrencia: La capacidad de ejecutar múltiples hilos en períodos superpuestos.
  • Paralelismo: La ejecución simultánea de múltiples hilos.

Creación de Hilos

En C++, la biblioteca <thread> proporciona las herramientas necesarias para crear y gestionar hilos.

Creación Básica de Hilos

Para crear un hilo, puedes usar la clase std::thread.

#include <iostream>
#include <thread>

void printMessage() {
    std::cout << "¡Hola desde el hilo!" << std::endl;
}

int main() {
    std::thread t(printMessage); // Crear un nuevo hilo que ejecuta printMessage
    t.join(); // Esperar a que el hilo termine
    return 0;
}

Explicación

  • std::thread t(printMessage): Crea un nuevo hilo que ejecuta la función printMessage.
  • t.join(): Espera a que el hilo t termine la ejecución antes de continuar.

Pasar Argumentos a los Hilos

Puedes pasar argumentos a la función del hilo usando el constructor std::thread.

#include <iostream>
#include <thread>

void printNumber(int n) {
    std::cout << "Número: " << n << std::endl;
}

int main() {
    std::thread t(printNumber, 42); // Pasar 42 como argumento a printNumber
    t.join();
    return 0;
}

Explicación

  • std::thread t(printNumber, 42): Crea un nuevo hilo que ejecuta printNumber con 42 como argumento.

Gestión de Hilos

Desacoplar Hilos

Un hilo desacoplado se ejecuta independientemente del hilo principal.

#include <iostream>
#include <thread>

void printMessage() {
    std::cout << "¡Hola desde el hilo desacoplado!" << std::endl;
}

int main() {
    std::thread t(printMessage);
    t.detach(); // Desacoplar el hilo
    std::this_thread::sleep_for(std::chrono::seconds(1)); // Asegurar que el hilo principal espere
    return 0;
}

Explicación

  • t.detach(): El hilo t se ejecuta independientemente del hilo principal.
  • std::this_thread::sleep_for: Asegura que el hilo principal espere lo suficiente para que el hilo desacoplado se ejecute.

Sincronización

Cuando múltiples hilos acceden a recursos compartidos, la sincronización es necesaria para evitar conflictos.

Mutex

Un std::mutex se usa para proteger datos compartidos.

#include <iostream>
#include <thread>
#include <mutex>

std::mutex mtx;

void printMessage(const std::string& msg) {
    std::lock_guard<std::mutex> lock(mtx);
    std::cout << msg << std::endl;
}

int main() {
    std::thread t1(printMessage, "¡Hola desde el hilo 1!");
    std::thread t2(printMessage, "¡Hola desde el hilo 2!");
    t1.join();
    t2.join();
    return 0;
}

Explicación

  • std::mutex mtx: Declara un mutex.
  • std::lock_guardstd::mutex lock(mtx): Bloquea el mutex durante la duración de la función printMessage.

Temas Avanzados

Variables de Condición

Las variables de condición permiten que los hilos esperen a que se cumplan ciertas condiciones.

#include <iostream>
#include <thread>
#include <mutex>
#include <condition_variable>

std::mutex mtx;
std::condition_variable cv;
bool ready = false;

void printMessage() {
    std::unique_lock<std::mutex> lock(mtx);
    cv.wait(lock, []{ return ready; });
    std::cout << "¡El hilo está en ejecución!" << std::endl;
}

int main() {
    std::thread t(printMessage);
    {
        std::lock_guard<std::mutex> lock(mtx);
        ready = true;
    }
    cv.notify_one();
    t.join();
    return 0;
}

Explicación

  • std::condition_variable cv: Declara una variable de condición.
  • cv.wait(lock, []{ return ready; }): Espera hasta que ready sea true.
  • cv.notify_one(): Notifica a un hilo en espera.

Conclusión

El multihilo en C++ permite la ejecución concurrente de tareas, mejorando el rendimiento y la capacidad de respuesta. Al entender la creación, gestión y sincronización de hilos, puedes utilizar eficazmente el multihilo en tus aplicaciones.
