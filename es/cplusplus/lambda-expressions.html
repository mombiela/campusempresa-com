Las expresiones lambda, también conocidas como funciones lambda o cierres, son una característica poderosa en C++ que te permite definir funciones anónimas directamente dentro de tu código. Son particularmente útiles para funciones cortas y desechables que no se reutilizan en otros lugares.

Introducción a las Expresiones Lambda

  • Definición: Una expresión lambda es una función anónima que puede capturar variables de su ámbito circundante.
  • Sintaxis: La sintaxis básica de una expresión lambda es: 
    [captura](parámetros) -> tipo_de_retorno {
    // cuerpo de la función
};

Sintaxis Básica y Uso

  • Cláusula de Captura: La cláusula de captura (los corchetes []) se usa para especificar qué variables del ámbito circundante son accesibles dentro de la lambda.
  • Parámetros: La lista de parámetros es similar a la de una función regular.
  • Tipo de Retorno: El tipo de retorno es opcional y a menudo puede ser inferido por el compilador.
  • Cuerpo de la Función: El cuerpo de la función contiene el código que se ejecutará cuando se llame a la lambda.

Ejemplo: Expresión Lambda Básica

#include <iostream>

int main() {
    auto add = [](int a, int b) -> int {
        return a + b;
    };

    std::cout << "Suma: " << add(3, 4) << std::endl; // Salida: Suma: 7
    return 0;
}

En este ejemplo:

  • auto add declara una lambda que toma dos enteros y devuelve su suma.
  • La lambda se llama con los argumentos 3 y 4, y el resultado se imprime.

Capturando Variables

Las expresiones lambda pueden capturar variables de su ámbito circundante de varias maneras:

  • Por Valor: Captura una copia de la variable.
  • Por Referencia: Captura una referencia a la variable.
  • Captura Implícita: Captura automáticamente las variables usadas en la lambda por valor o por referencia.

Ejemplo: Capturando por Valor y Referencia

#include <iostream>

int main() {
    int x = 10;
    int y = 20;

    auto captureByValue = [x, y]() {
        return x + y;
    };

    auto captureByReference = [&x, &y]() {
        return x + y;
    };

    x = 15;
    y = 25;

    std::cout << "Captura por valor: " << captureByValue() << std::endl; // Salida: 30
    std::cout << "Captura por referencia: " << captureByReference() << std::endl; // Salida: 40

    return 0;
}

En este ejemplo:

  • captureByValue captura x y y por valor, por lo que los cambios en x y y después de que se define la lambda no afectan el resultado.
  • captureByReference captura x y y por referencia, por lo que los cambios en x y y se reflejan en el resultado de la lambda.

Características Avanzadas

Lambdas Mutables

Por defecto, las lambdas que capturan variables por valor no están permitidas a modificar esas variables. Para permitir la modificación, puedes usar la palabra clave mutable.

#include <iostream>

int main() {
    int x = 10;

    auto mutableLambda = [x]() mutable {
        x += 5;
        return x;
    };

    std::cout << "Lambda mutable: " << mutableLambda() << std::endl; // Salida: 15
    std::cout << "x original: " << x << std::endl; // Salida: 10

    return 0;
}

Lambdas Genéricas

C++14 introdujo lambdas genéricas, que te permiten usar auto en la lista de parámetros para crear una lambda que puede aceptar argumentos de cualquier tipo.

#include <iostream>

int main() {
    auto genericLambda = [](auto a, auto b) {
        return a + b;
    };

    std::cout << "Suma de enteros: " << genericLambda(3, 4) << std::endl; // Salida: 7
    std::cout << "Suma de dobles: " << genericLambda(3.5, 4.5) << std::endl; // Salida: 8

    return 0;
}

Aplicaciones Prácticas

Usando Lambdas con Algoritmos Estándar

Las lambdas se usan a menudo con algoritmos estándar como std::sort, std::for_each y std::transform.

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};

    std::for_each(numbers.begin(), numbers.end(), [](int &n) {
        n *= 2;
    });

    for (int n : numbers) {
        std::cout << n << " "; // Salida: 2 4 6 8 10
    }

    return 0;
}

En este ejemplo:

  • std::for_each aplica la lambda a cada elemento en el vector, duplicando cada valor.

Conclusión

Las expresiones lambda en C++ proporcionan una manera concisa y poderosa de definir funciones anónimas. Son particularmente útiles para funciones cortas y de una sola vez y pueden capturar variables de su ámbito circundante. Entender cómo usar las lambdas de manera efectiva puede mejorar en gran medida tu capacidad para escribir código C++ limpio y eficiente.
