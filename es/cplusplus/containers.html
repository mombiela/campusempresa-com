Introducción a los Contenedores

Los contenedores son componentes fundamentales en C++ que te permiten almacenar y gestionar colecciones de datos. Son parte de la Biblioteca de Plantillas Estándar (STL) y proporcionan varias formas de organizar y manipular datos de manera eficiente.

Conceptos Clave

  • Contenedores de Secuencia: Almacenan datos de manera lineal.
  • Contenedores Asociativos: Almacenan datos de una manera que permite una recuperación rápida usando claves.
  • Adaptadores de Contenedores: Proporcionan una interfaz diferente para los contenedores secuenciales.

Contenedores de Secuencia

Los contenedores de secuencia mantienen el orden de los elementos.

Vector

Un array dinámico que puede redimensionarse automáticamente cuando se añade o elimina un elemento.

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_back(6); // Añadir un elemento al final

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Lista

Una lista doblemente enlazada que permite una inserción y eliminación eficiente de elementos.

#include <iostream>
#include <list>

int main() {
    std::list<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_back(6); // Añadir un elemento al final

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Deque

Una cola de doble extremo que permite la inserción y eliminación en ambos extremos.

#include <iostream>
#include <deque>

int main() {
    std::deque<int> numbers = {1, 2, 3, 4, 5};
    numbers.push_front(0); // Añadir un elemento al frente

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Contenedores Asociativos

Los contenedores asociativos almacenan elementos formados por una combinación de un valor clave y un valor mapeado, y permiten una recuperación rápida de elementos basados en las claves.

Mapa

Una colección de pares clave-valor con claves únicas.

#include <iostream>
#include <map>

int main() {
    std::map<int, std::string> students;
    students[1] = "Alice";
    students[2] = "Bob";

    for (const auto& pair : students) {
        std::cout << pair.first << ": " << pair.second << std::endl;
    }
    return 0;
}

Conjunto

Una colección de claves únicas.

#include <iostream>
#include <set>

int main() {
    std::set<int> numbers = {1, 2, 3, 4, 5};
    numbers.insert(6); // Añadir un elemento

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Adaptadores de Contenedores

Los adaptadores de contenedores proporcionan una interfaz diferente para los contenedores secuenciales.

Pila

Una estructura de datos LIFO (Último en Entrar, Primero en Salir).

#include <iostream>
#include <stack>

int main() {
    std::stack<int> numbers;
    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.top() << " ";
        numbers.pop();
    }
    return 0;
}

Cola

Una estructura de datos FIFO (Primero en Entrar, Primero en Salir).

#include <iostream>
#include <queue>

int main() {
    std::queue<int> numbers;
    numbers.push(1);
    numbers.push(2);
    numbers.push(3);

    while (!numbers.empty()) {
        std::cout << numbers.front() << " ";
        numbers.pop();
    }
    return 0;
}

Cola de Prioridad

Una estructura de datos donde los elementos se procesan en base a la prioridad.

#include <iostream>
#include <queue>

int main() {
    std::priority_queue<int> numbers;
    numbers.push(3);
    numbers.push(1);
    numbers.push(2);

    while (!numbers.empty()) {
        std::cout << numbers.top() << " ";
        numbers.pop();
    }
    return 0;
}

Resumen

En esta sección, cubrimos los diferentes tipos de contenedores disponibles en C++:

  • Contenedores de Secuencia: Vector, Lista, Deque.
  • Contenedores Asociativos: Mapa, Conjunto.
  • Adaptadores de Contenedores: Pila, Cola, Cola de Prioridad.

Entender estos contenedores y sus casos de uso apropiados es crucial para una gestión y manipulación eficiente de datos en C++. Cada tipo de contenedor ofrece ventajas únicas y es adecuado para escenarios específicos, lo que los convierte en herramientas poderosas en el conjunto de herramientas de un programador de C++.
