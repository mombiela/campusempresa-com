Introducción

La declaración switch en C++ es una declaración de control que permite que una variable sea probada para igualdad contra una lista de valores. Cada valor se llama un caso, y la variable que se está cambiando se verifica para cada caso.

Sintaxis Básica

La sintaxis básica de una declaración switch es la siguiente:

switch (expression) {
    case constant1:
        // código a ejecutar si la expresión es igual a constant1;
        break;
    case constant2:
        // código a ejecutar si la expresión es igual a constant2;
        break;
    // puedes tener cualquier número de declaraciones case
    default:
        // código a ejecutar si la expresión no coincide con ningún caso
}
  • expression: La variable o valor que se está comparando.
  • case: Cada caso es seguido por un valor constante y dos puntos.
  • break: Esta declaración termina el caso switch. Si se omite, el programa continuará ejecutando el siguiente caso.
  • default: Esto es opcional y puede ser usado para ejecutar un bloque de código si ninguno de los casos coincide.

Ejemplo: Caso Básico de Switch

#include <iostream>
using namespace std;

int main() {
    int day = 4;

    switch (day) {
        case 1:
            cout << "Monday";
            break;
        case 2:
            cout << "Tuesday";
            break;
        case 3:
            cout << "Wednesday";
            break;
        case 4:
            cout << "Thursday";
            break;
        case 5:
            cout << "Friday";
            break;
        case 6:
            cout << "Saturday";
            break;
        case 7:
            cout << "Sunday";
            break;
        default:
            cout << "Invalid day";
    }

    return 0;
}

Explicación

  • La variable day se compara con cada caso.
  • Cuando day es igual a 4, el programa imprime "Thursday".
  • La declaración break evita que el programa ejecute el siguiente caso.

Uso Avanzado

Múltiples Casos para un Solo Bloque

Puedes agrupar múltiples casos juntos si deben ejecutar el mismo bloque de código.

#include <iostream>
using namespace std;

int main() {
    char grade = 'B';

    switch (grade) {
        case 'A':
        case 'B':
        case 'C':
            cout << "You passed.";
            break;
        case 'D':
        case 'F':
            cout << "You failed.";
            break;
        default:
            cout << "Invalid grade.";
    }

    return 0;
}

Explicación

  • Los casos A, B y C están agrupados para ejecutar el mismo bloque de código.
  • De manera similar, los casos D y F están agrupados.

Declaraciones Switch Anidadas

Las declaraciones switch pueden ser anidadas dentro de otras.

#include <iostream>
using namespace std;

int main() {
    int num1 = 1, num2 = 2;

    switch (num1) {
        case 1:
            switch (num2) {
                case 2:
                    cout << "num1 is 1 and num2 is 2";
                    break;
                default:
                    cout << "num1 is 1 but num2 is not 2";
            }
            break;
        default:
            cout << "num1 is not 1";
    }

    return 0;
}

Explicación

  • El switch externo verifica el valor de num1.
  • Si num1 es 1, el switch interno verifica el valor de num2.

Mejores Prácticas

  • Siempre incluye un caso default para manejar valores inesperados.
  • Usa declaraciones break para evitar la caída a menos que sea intencional.
  • Mantén los casos del switch simples y evita la lógica compleja dentro de ellos.

Conclusión

La declaración switch es una herramienta poderosa para controlar el flujo de un programa basado en el valor de una variable. Es particularmente útil cuando tienes múltiples condiciones que dependen de la misma variable. Al entender y utilizar la declaración switch, puedes escribir código más legible y eficiente.
