Resumen

El proyecto final está diseñado para consolidar tus conocimientos y habilidades adquiridas a lo largo de este curso de C++. Este proyecto te desafiará a aplicar varios conceptos, desde la sintaxis básica hasta técnicas de programación avanzadas, de manera cohesiva y práctica. Al final de este proyecto, deberías tener una comprensión integral de C++ y ser capaz de desarrollar una aplicación completamente funcional.

Descripción del Proyecto

Crearás un juego simple basado en texto usando C++. El juego involucrará a un jugador navegando a través de una serie de habitaciones, recolectando objetos y resolviendo acertijos para lograr un objetivo específico. Este proyecto requerirá que uses:

  • Sintaxis básica y estructuras de control
  • Funciones y programación modular
  • Programación Orientada a Objetos (POO)
  • Operaciones de E/S de archivos
  • Gestión dinámica de memoria
  • Manejo de errores

Requisitos

Estructura Básica del Juego

  1. Menú Principal:

    • Mostrar opciones para iniciar un nuevo juego, cargar un juego guardado o salir.
    • Ejemplo: 
      void displayMainMenu() {
    std::cout << "1. Iniciar Nuevo Juego\n";
    std::cout << "2. Cargar Juego\n";
    std::cout << "3. Salir\n";
    std::cout << "Ingrese su elección: ";
}

  2. Bucle del Juego:

    • Ejecutar continuamente hasta que el jugador gane o elija salir.
    • Ejemplo: 
      bool gameRunning = true;
while (gameRunning) {
    // Lógica del juego aquí
    if (playerWins || playerExits) {
        gameRunning = false;
    }
}

Clases de Jugador y Habitación

  1. Clase Jugador:

    • Atributos: nombre, salud, inventario.
    • Métodos: moverse, recoger objeto, usar objeto.
    • Ejemplo: 
      class Player {
public:
    std::string name;
    int health;
    std::vector<std::string> inventory;

    void move();
    void pickUpItem(const std::string& item);
    void useItem(const std::string& item);
};

  2. Clase Habitación:

    • Atributos: descripción, objetos, salidas.
    • Métodos: agregar objeto, remover objeto, describir habitación.
    • Ejemplo: 
      class Room {
public:
    std::string description;
    std::vector<std::string> items;
    std::map<std::string, Room*> exits;

    void addItem(const std::string& item);
    void removeItem(const std::string& item);
    void describeRoom();
};

Mecánicas del Juego

  1. Navegación:

    • Permitir al jugador moverse entre habitaciones.
    • Ejemplo: 
      void Player::move(Room* currentRoom, const std::string& direction) {
    if (currentRoom->exits.find(direction) != currentRoom->exits.end()) {
        currentRoom = currentRoom->exits[direction];
        std::cout << "Te moviste hacia el " << direction << ".\n";
    } else {
        std::cout << "No puedes ir por ahí.\n";
    }
}

  2. Gestión de Objetos:

    • Recoger y usar objetos.
    • Ejemplo: 
      void Player::pickUpItem(const std::string& item) {
    inventory.push_back(item);
    std::cout << "Recogiste un " << item << ".\n";
}

void Player::useItem(const std::string& item) {
    auto it = std::find(inventory.begin(), inventory.end(), item);
    if (it != inventory.end()) {
        inventory.erase(it);
        std::cout << "Usaste el " << item << ".\n";
    } else {
        std::cout << "No tienes un " << item << ".\n";
    }
}

Guardar y Cargar el Estado del Juego

  1. E/S de Archivos:
    • Guardar el estado actual del juego en un archivo.
    • Cargar un estado de juego guardado desde un archivo.
    • Ejemplo: 
      void saveGame(const Player& player, const Room& room) {
    std::ofstream outFile("savegame.txt");
    outFile << player.name << "\n" << player.health << "\n";
    for (const auto& item : player.inventory) {
        outFile << item << "\n";
    }
    outFile.close();
}

void loadGame(Player& player, Room& room) {
    std::ifstream inFile("savegame.txt");
    inFile >> player.name >> player.health;
    std::string item;
    while (inFile >> item) {
        player.inventory.push_back(item);
    }
    inFile.close();
}
