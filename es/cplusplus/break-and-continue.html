Introducción

En esta sección, exploraremos las sentencias break y continue en C++. Estas sentencias de control de flujo se utilizan para alterar el flujo normal de los bucles, proporcionando más control sobre cómo y cuándo se ejecutan los bucles.

Sentencia Break

La sentencia break se utiliza para salir de un bucle prematuramente. Cuando se encuentra una sentencia break dentro de un bucle, el bucle se termina inmediatamente y el control del programa se reanuda en la siguiente sentencia después del bucle.

Conceptos Clave

  • Terminación Inmediata: La sentencia break hace que el bucle termine inmediatamente.
  • Bucles Aplicables: Puede ser utilizada en bucles for, while y do-while.
  • Bucles Anidados: Cuando se usa en bucles anidados, break solo sale del bucle más interno.

Ejemplo

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i == 5) {
            break; // Salir del bucle cuando i es 5
        }
        cout << i << " ";
    }
    cout << "Bucle terminado." << endl;
    return 0;
}

Explicación: En este ejemplo, el bucle imprimirá números del 0 al 4. Cuando i es igual a 5, se ejecuta la sentencia break, terminando el bucle.

Sentencia Continue

La sentencia continue salta la iteración actual del bucle y procede con la siguiente iteración. A diferencia de break, no termina el bucle, sino que simplemente salta el código restante en la iteración actual.

Conceptos Clave

  • Saltar Iteración: La sentencia continue salta el resto del código en la iteración actual del bucle.
  • Bucles Aplicables: Puede ser utilizada en bucles for, while y do-while.
  • Bucles Anidados: Cuando se usa en bucles anidados, continue solo afecta al bucle más interno.

Ejemplo

#include <iostream>
using namespace std;

int main() {
    for (int i = 0; i < 10; i++) {
        if (i % 2 == 0) {
            continue; // Saltar el resto del cuerpo del bucle para números pares
        }
        cout << i << " ";
    }
    cout << "Bucle completado." << endl;
    return 0;
}

Explicación: En este ejemplo, el bucle imprimirá solo números impares del 1 al 9. Cuando i es par, se ejecuta la sentencia continue, saltando la sentencia cout para esa iteración.

Comparación de Break y Continue

| Característica | Break | Continue | |-------------------|--------------------------------|--------------------------------| | Efecto | Termina el bucle | Salta la iteración actual | | Terminación del Bucle | Sí | No | | Efecto en Bucles Anidados | Afecta solo al bucle más interno | Afecta solo al bucle más interno |

Ejercicios

  1. Ejercicio 1: Escribe un programa que use un bucle for para imprimir números del 1 al 20. Usa la sentencia break para salir del bucle cuando el número sea 15.
  2. Ejercicio 2: Escribe un programa que use un bucle while para imprimir números del 1 al 10. Usa la sentencia continue para saltar la impresión del número 5.

Solución del Ejercicio 1

#include <iostream>
using namespace std;

int main() {
    for (int i = 1; i <= 20; i++) {
        if (i == 15) {
            break; // Salir del bucle cuando i es 15
        }
        cout << i << " ";
    }
    cout << "Bucle terminado en 15." << endl;
    return 0;
}

Solución del Ejercicio 2

#include <iostream>
using namespace std;

int main() {
    int i = 1;
    while (i <= 10) {
        if (i == 5) {
            i++;
            continue; // Saltar el resto del cuerpo del bucle cuando i es 5
        }
        cout << i << " ";
        i++;
    }
    cout << "Bucle completado, se saltó el 5." << endl;
    return 0;
}

Conclusión

Las sentencias break y continue son herramientas poderosas para controlar el flujo de los bucles en C++. La sentencia break te permite salir de un bucle prematuramente, mientras que la sentencia continue te permite saltar la iteración actual y proceder con la siguiente. Entender cómo y cuándo usar estas sentencias puede ayudarte a escribir código más eficiente y legible.
