Introducción

En C#, las variables y constantes son conceptos fundamentales que te permiten almacenar y manipular datos. Entender cómo declarar, inicializar y usar variables y constantes es esencial para cualquier programador de C#. Esta sección te guiará a través de estos conceptos desde un nivel principiante hasta uno avanzado.

Variables

¿Qué es una Variable?

  • Una variable es una ubicación de almacenamiento en memoria con un tipo específico que determina qué tipo de datos puede contener.
  • Las variables tienen nombres (identificadores) que se utilizan para referenciar los datos almacenados.

Declaración e Inicialización de Variables

  • Declaración: Especificar el tipo y el nombre de la variable.
  • Inicialización: Asignar un valor inicial a la variable.

Sintaxis

tipo nombreVariable = valorInicial;

Ejemplo

int edad = 25; // Declara una variable entera llamada 'edad' y la inicializa con el valor 25.
string nombre = "Juan"; // Declara una variable de cadena llamada 'nombre' y la inicializa con el valor "Juan".

Tipos de Variables

  • Tipos Primitivos: int, float, double, char, bool, etc.
  • Tipos de Referencia: string, arrays, clases, etc.

Ejemplo

int numero = 10; // Tipo entero
float pi = 3.14f; // Tipo de punto flotante
char letra = 'A'; // Tipo carácter
bool esVerdadero = true; // Tipo booleano
string saludo = "¡Hola, Mundo!"; // Tipo cadena

Alcance de las Variables

  • Variables Locales: Declaradas dentro de un método y accesibles solo dentro de ese método.
  • Variables Globales: Declaradas fuera de los métodos y accesibles en toda la clase.

Ejemplo

class Programa
{
    int varGlobal = 10; // Variable global

    void Metodo()
    {
        int varLocal = 5; // Variable local
        Console.WriteLine(varLocal);
    }
}

Constantes

¿Qué es una Constante?

  • Una constante es una variable cuyo valor no puede cambiar una vez que se asigna.
  • Las constantes se declaran usando la palabra clave const.

Declaración de Constantes

  • Sintaxis: const tipo nombreConstante = valor;

Ejemplo

const double PI = 3.14159; // Declara una constante doble llamada 'PI' con el valor 3.14159.
const int DIAS_EN_SEMANA = 7; // Declara una constante entera llamada 'DIAS_EN_SEMANA' con el valor 7.

¿Por Qué Usar Constantes?

  • Las constantes proporcionan una manera de usar nombres significativos para valores fijos, haciendo el código más legible y mantenible.
  • Previenen la modificación accidental de valores que deben permanecer constantes a lo largo del programa.

Comparación: Variables vs. Constantes

| Característica | Variables | Constantes | |----------------|------------------------------------|-------------------------------------| | Mutabilidad | Pueden cambiar después de la inicialización | No pueden cambiar después de la inicialización | | Declaración | tipo nombreVariable = valor; | const tipo nombreConstante = valor; | | Uso | Usadas para valores que pueden cambiar | Usadas para valores fijos | | Ejemplo | int edad = 25; | const int DIAS_EN_SEMANA = 7; |

Temas Avanzados

Campos Readonly

  • Similares a las constantes pero pueden asignarse un valor ya sea en el momento de la declaración o en el constructor.
  • Declarados usando la palabra clave readonly.

Ejemplo

class Circulo
{
    public readonly double Radio;

    public Circulo(double radio)
    {
        Radio = radio; // Puede asignarse en el constructor
    }
}

Variables y Constantes Estáticas

  • Variables Estáticas: Compartidas entre todas las instancias de una clase.
  • Constantes Estáticas: Constantes que son compartidas entre todas las instancias de una clase.

Ejemplo

class ConstantesMatematicas
{
    public const double PI = 3.14159; // Constante estática
    public static int contador = 0; // Variable estática
}

Conclusión

Entender las variables y constantes es crucial para una programación efectiva en C#. Las variables te permiten almacenar y manipular datos, mientras que las constantes proporcionan una manera de usar valores fijos que mejoran la legibilidad y mantenibilidad del código. Al dominar estos conceptos, puedes escribir programas en C# más eficientes y confiables.
