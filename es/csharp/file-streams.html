Introducción a los Flujos de Archivos

Los flujos de archivos en C# se utilizan para leer y escribir en archivos. Proporcionan una forma de manejar operaciones de archivos de manera eficiente y segura. Entender los flujos de archivos es crucial para tareas como almacenamiento de datos, gestión de configuración y registro de eventos.

Conceptos Básicos

  • Stream: Una secuencia de bytes que se puede leer o escribir.
  • FileStream: Una clase en el espacio de nombres System.IO utilizada para leer y escribir en archivos.
  • StreamReader: Una clase para leer caracteres de un flujo de bytes en una codificación particular.
  • StreamWriter: Una clase para escribir caracteres en un flujo en una codificación particular.

Trabajando con FileStream

Leyendo desde un Archivo

Para leer desde un archivo, puedes usar la clase FileStream junto con StreamReader.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        
        using (FileStream fs = new FileStream(path, FileMode.Open, FileAccess.Read))
        using (StreamReader sr = new StreamReader(fs))
        {
            string content = sr.ReadToEnd();
            Console.WriteLine(content);
        }
    }
}
  • FileMode.Open: Abre el archivo si existe.
  • FileAccess.Read: Concede acceso de lectura al archivo.
  • ReadToEnd(): Lee todos los caracteres desde la posición actual hasta el final del flujo.

Escribiendo en un Archivo

Para escribir en un archivo, puedes usar la clase FileStream junto con StreamWriter.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = "Hello, World!";
        
        using (FileStream fs = new FileStream(path, FileMode.Create, FileAccess.Write))
        using (StreamWriter sw = new StreamWriter(fs))
        {
            sw.Write(content);
        }
    }
}
  • FileMode.Create: Crea un nuevo archivo o sobrescribe uno existente.
  • FileAccess.Write: Concede acceso de escritura al archivo.
  • Write(): Escribe una cadena en el flujo.

Operaciones Avanzadas con Flujos de Archivos

Operaciones Asíncronas de Archivos

Las operaciones asíncronas pueden mejorar el rendimiento al no bloquear el hilo principal.

Lectura Asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        
        using (FileStream fs = new FileStream(path, FileMode.Open, FileAccess.Read, FileShare.Read, 4096, true))
        using (StreamReader sr = new StreamReader(fs))
        {
            string content = await sr.ReadToEndAsync();
            Console.WriteLine(content);
        }
    }
}
  • ReadToEndAsync(): Lee asíncronamente todos los caracteres desde la posición actual hasta el final del flujo.

Escritura Asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = "Hello, Async World!";
        
        using (FileStream fs = new FileStream(path, FileMode.Create, FileAccess.Write, FileShare.None, 4096, true))
        using (StreamWriter sw = new StreamWriter(fs))
        {
            await sw.WriteAsync(content);
        }
    }
}
  • WriteAsync(): Escribe asíncronamente una cadena en el flujo.

Manejo de Datos Binarios

Para datos binarios, usa BinaryReader y BinaryWriter.

Leyendo Datos Binarios

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.bin";
        
        using (FileStream fs = new FileStream(path, FileMode.Open, FileAccess.Read))
        using (BinaryReader br = new BinaryReader(fs))
        {
            int number = br.ReadInt32();
            Console.WriteLine(number);
        }
    }
}
  • ReadInt32(): Lee un entero de 4 bytes con signo del flujo.

Escribiendo Datos Binarios

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.bin";
        int number = 12345;
        
        using (FileStream fs = new FileStream(path, FileMode.Create, FileAccess.Write))
        using (BinaryWriter bw = new BinaryWriter(fs))
        {
            bw.Write(number);
        }
    }
}
  • Write(int): Escribe un entero de 4 bytes con signo en el flujo.

Resumen

Los flujos de archivos en C# proporcionan una forma poderosa de manejar operaciones de archivos, desde la lectura y escritura básica hasta el manejo más avanzado de datos asíncronos y binarios. Al dominar los flujos de archivos, puedes gestionar de manera eficiente el almacenamiento y la recuperación de datos en tus aplicaciones.
