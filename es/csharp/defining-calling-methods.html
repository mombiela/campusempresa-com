Introducción

Los métodos son bloques de construcción fundamentales en C#. Te permiten encapsular código en bloques reutilizables, haciendo tus programas más modulares y fáciles de mantener. En esta sección, cubriremos cómo definir y llamar métodos, desde conceptos básicos hasta avanzados.

Definiendo Métodos

Definición Básica de Método

Un método en C# se define dentro de una clase y consiste en una firma y un cuerpo. La firma incluye el modificador de acceso, el tipo de retorno, el nombre del método y los parámetros (si los hay).

public class Program
{
    // Definición del método
    public void SayHello()
    {
        Console.WriteLine("¡Hola, Mundo!");
    }
}
  • Modificador de Acceso: Determina la visibilidad del método (por ejemplo, public, private).
  • Tipo de Retorno: Especifica el tipo de valor que el método devuelve (por ejemplo, void si no se devuelve ningún valor).
  • Nombre del Método: Un identificador único para el método.
  • Parámetros: Opcional; se utilizan para pasar datos al método.

Método con Parámetros

Los métodos pueden tomar parámetros para realizar operaciones sobre datos de entrada.

public class Program
{
    public void GreetUser(string name)
    {
        Console.WriteLine($"¡Hola, {name}!");
    }
}
  • Parámetros: Definidos dentro de los paréntesis y separados por comas si hay múltiples.

Método con Valor de Retorno

Los métodos pueden devolver valores al llamador.

public class Program
{
    public int Add(int a, int b)
    {
        return a + b;
    }
}
  • Sentencia Return: Utilizada para devolver un valor desde el método.

Llamando Métodos

Llamando un Método sin Parámetros

Para llamar un método, usas el nombre del método seguido de paréntesis.

public class Program
{
    public void SayHello()
    {
        Console.WriteLine("¡Hola, Mundo!");
    }

    public static void Main()
    {
        Program program = new Program();
        program.SayHello(); // Llamada al método
    }
}

Llamando un Método con Parámetros

Al llamar un método con parámetros, pasas los argumentos dentro de los paréntesis.

public class Program
{
    public void GreetUser(string name)
    {
        Console.WriteLine($"¡Hola, {name}!");
    }

    public static void Main()
    {
        Program program = new Program();
        program.GreetUser("Alice"); // Llamada al método con argumento
    }
}

Llamando un Método con Valor de Retorno

Al llamar un método que devuelve un valor, puedes almacenar el resultado en una variable.

public class Program
{
    public int Add(int a, int b)
    {
        return a + b;
    }

    public static void Main()
    {
        Program program = new Program();
        int result = program.Add(5, 3); // Llamada al método con valor de retorno
        Console.WriteLine(result); // Salida: 8
    }
}

Conceptos Avanzados

Sobrecarga de Métodos

La sobrecarga de métodos te permite definir múltiples métodos con el mismo nombre pero diferentes parámetros.

public class Program
{
    public void Display(int number)
    {
        Console.WriteLine($"Número: {number}");
    }

    public void Display(string text)
    {
        Console.WriteLine($"Texto: {text}");
    }

    public static void Main()
    {
        Program program = new Program();
        program.Display(10); // Llama al primer método Display
        program.Display("Hola"); // Llama al segundo método Display
    }
}

Parámetros Opcionales

C# te permite definir métodos con parámetros opcionales, proporcionando valores predeterminados si no se suministran argumentos.

public class Program
{
    public void DisplayMessage(string message = "Mensaje Predeterminado")
    {
        Console.WriteLine(message);
    }

    public static void Main()
    {
        Program program = new Program();
        program.DisplayMessage(); // Salida: Mensaje Predeterminado
        program.DisplayMessage("Mensaje Personalizado"); // Salida: Mensaje Personalizado
    }
}

Argumentos Nombrados

Los argumentos nombrados te permiten especificar el nombre del parámetro al llamar un método, mejorando la legibilidad.

public class Program
{
    public void DisplayInfo(string name, int age)
    {
        Console.WriteLine($"Nombre: {name}, Edad: {age}");
    }

    public static void Main()
    {
        Program program = new Program();
        program.DisplayInfo(name: "Alice", age: 30); // Argumentos nombrados
    }
}

Resumen

En esta sección, cubrimos los conceptos básicos de definir y llamar métodos en C#. Exploramos:

  • Definiciones y llamadas básicas de métodos.
  • Métodos con parámetros y valores de retorno.
  • Conceptos avanzados como sobrecarga de métodos, parámetros opcionales y argumentos nombrados.

Entender cómo definir y llamar métodos es crucial para escribir código limpio, eficiente y mantenible. Practica estos conceptos para volverte competente en el uso de métodos en C#.
