¿Qué es LINQ?

Language Integrated Query (LINQ) es una característica poderosa en C# que permite a los desarrolladores escribir consultas directamente dentro del lenguaje C#. LINQ proporciona una forma consistente de consultar varias fuentes de datos, como colecciones, bases de datos, XML y más.

  • Sintaxis Consistente: LINQ utiliza una sintaxis uniforme para consultar diferentes fuentes de datos.
  • Seguridad de Tipos: Las consultas se verifican en tiempo de compilación, reduciendo los errores en tiempo de ejecución.
  • Soporte de IntelliSense: Visual Studio proporciona soporte de IntelliSense para consultas LINQ, facilitando la escritura y depuración.

Sintaxis Básica de LINQ

Las consultas LINQ se escriben utilizando sintaxis de consulta o sintaxis de método. Ambas logran el mismo resultado pero tienen enfoques sintácticos diferentes.

Sintaxis de Consulta

La sintaxis de consulta se asemeja a SQL y es más fácil de entender para aquellos familiarizados con SQL.

int[] numbers = { 2, 3, 4, 5, 6 };

var evenNumbers = from num in numbers
                  where num % 2 == 0
                  select num;

foreach (var num in evenNumbers)
{
    Console.WriteLine(num);
}

Sintaxis de Método

La sintaxis de método utiliza métodos de extensión y expresiones lambda.

int[] numbers = { 2, 3, 4, 5, 6 };

var evenNumbers = numbers.Where(num => num % 2 == 0);

foreach (var num in evenNumbers)
{
    Console.WriteLine(num);
}

Operadores LINQ

LINQ proporciona una variedad de operadores para realizar diferentes operaciones en los datos. Aquí hay algunos operadores clave:

  • Where: Filtra elementos basados en una condición.
  • Select: Proyecta cada elemento en una nueva forma.
  • OrderBy: Ordena elementos en orden ascendente.
  • GroupBy: Agrupa elementos que comparten un atributo común.
  • Join: Une dos secuencias basadas en una clave.

Ejemplo: Usando Múltiples Operadores

string[] names = { "John", "Jane", "Jake", "Jessie", "James" };

var filteredNames = names
    .Where(name => name.StartsWith("J"))
    .OrderBy(name => name)
    .Select(name => name.ToUpper());

foreach (var name in filteredNames)
{
    Console.WriteLine(name);
}

LINQ con Diferentes Fuentes de Datos

LINQ a Objetos

LINQ se puede usar para consultar colecciones en memoria como arreglos, listas, etc.

List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };

var oddNumbers = from num in numbers
                 where num % 2 != 0
                 select num;

foreach (var num in oddNumbers)
{
    Console.WriteLine(num);
}

LINQ a SQL

LINQ se puede usar para consultar bases de datos. Esto requiere configurar un contexto de datos.

using (var context = new DataContext())
{
    var customers = from customer in context.Customers
                    where customer.City == "Seattle"
                    select customer;

    foreach (var customer in customers)
    {
        Console.WriteLine(customer.Name);
    }
}

LINQ a XML

LINQ se puede usar para consultar documentos XML.

XDocument doc = XDocument.Load("books.xml");

var books = from book in doc.Descendants("book")
            where (int)book.Element("price") > 20
            select book.Element("title").Value;

foreach (var title in books)
{
    Console.WriteLine(title);
}

Ejercicios

  1. Filtrado Básico: Escribe una consulta LINQ para filtrar los números pares de un arreglo de enteros.
  2. Proyección: Usa LINQ para seleccionar los nombres de los estudiantes que obtuvieron más de 80 de una lista de objetos estudiante.
  3. Ordenamiento: Escribe una consulta LINQ para ordenar una lista de productos por su precio en orden descendente.
  4. Agrupamiento: Agrupa una lista de empleados por su departamento y muestra los nombres de los departamentos junto con el conteo de empleados en cada departamento.
  5. Unión: Usa LINQ para unir dos listas: una que contiene IDs de clientes y otra que contiene órdenes, y muestra los IDs de los clientes junto con los detalles de sus órdenes.

Resumen

LINQ es una característica versátil y poderosa en C# que simplifica la consulta de datos en diferentes fuentes de datos. Al proporcionar una sintaxis consistente y aprovechar el poder del lenguaje C#, LINQ mejora la productividad y la legibilidad del código. Ya sea que estés trabajando con colecciones en memoria, bases de datos o documentos XML, LINQ ofrece un enfoque unificado para la manipulación de datos.
