Los genéricos son una característica poderosa en C# que te permite definir clases, métodos y estructuras de datos con un marcador de posición para el tipo de datos que almacenan o usan. Esto permite la reutilización de código, la seguridad de tipos y la optimización del rendimiento.

Introducción a los Genéricos

Los genéricos te permiten escribir una clase o método que puede trabajar con cualquier tipo de dato. En lugar de especificar un tipo de dato específico, usas un marcador de posición, que se reemplaza con el tipo de dato real cuando la clase o método se instancia o se llama.

Conceptos Clave

  • Parámetro de Tipo: Un marcador de posición para un tipo de dato, especificado usando corchetes angulares (<T>).
  • Clase Genérica: Una clase que puede operar con cualquier tipo de dato.
  • Método Genérico: Un método que puede operar con cualquier tipo de dato.
  • Seguridad de Tipos: Asegura que el tipo de dato sea consistente, reduciendo errores en tiempo de ejecución.
  • Reutilización de Código: Permite que el mismo código funcione con diferentes tipos de datos.

Clases Genéricas

Una clase genérica se define con un parámetro de tipo. Aquí hay un ejemplo de una clase genérica simple:

public class GenericClass<T>
{
    private T data;

    public GenericClass(T value)
    {
        data = value;
    }

    public T GetData()
    {
        return data;
    }
}

Ejemplo de Uso

GenericClass<int> intInstance = new GenericClass<int>(10);
Console.WriteLine(intInstance.GetData()); // Salida: 10

GenericClass<string> stringInstance = new GenericClass<string>("Hola");
Console.WriteLine(stringInstance.GetData()); // Salida: Hola

Métodos Genéricos

Un método genérico se define con un parámetro de tipo. Aquí hay un ejemplo:

public class GenericMethods
{
    public void PrintData<T>(T data)
    {
        Console.WriteLine(data);
    }
}

Ejemplo de Uso

GenericMethods gm = new GenericMethods();
gm.PrintData<int>(100); // Salida: 100
gm.PrintData<string>("Hola Mundo"); // Salida: Hola Mundo

Restricciones en los Parámetros de Tipo

Puedes imponer restricciones en los parámetros de tipo para restringir los tipos que se pueden usar. Aquí hay algunas restricciones comunes:

  • where T : struct: T debe ser un tipo de valor.
  • where T : class: T debe ser un tipo de referencia.
  • where T : new(): T debe tener un constructor sin parámetros.
  • where T : BaseClass: T debe heredar de BaseClass.
  • where T : Interface: T debe implementar la interfaz especificada.

Ejemplo con Restricciones

public class GenericClassWithConstraints<T> where T : class, new()
{
    private T data;

    public GenericClassWithConstraints()
    {
        data = new T();
    }

    public T GetData()
    {
        return data;
    }
}

Colecciones Genéricas

C# proporciona varias colecciones genéricas en el espacio de nombres System.Collections.Generic, como List<T>, Dictionary<TKey, TValue>, y Queue<T>.

Ejemplo con List

List<int> numbers = new List<int>();
numbers.Add(1);
numbers.Add(2);
numbers.Add(3);

foreach (int number in numbers)
{
    Console.WriteLine(number); // Salida: 1 2 3
}

Ejemplo con Dictionary

Dictionary<int, string> dict = new Dictionary<int, string>();
dict.Add(1, "Uno");
dict.Add(2, "Dos");

foreach (var kvp in dict)
{
    Console.WriteLine($"Clave: {kvp.Key}, Valor: {kvp.Value}");
    // Salida: Clave: 1, Valor: Uno
    //         Clave: 2, Valor: Dos
}

Genéricos Avanzados

Covarianza y Contravarianza

La covarianza y la contravarianza te permiten usar un tipo más derivado de lo originalmente especificado (covarianza) o un tipo más genérico (contravarianza).

  • Covarianza: Te permite usar un tipo derivado donde se espera un tipo base.
  • Contravarianza: Te permite usar un tipo base donde se espera un tipo derivado.

Ejemplo de Covarianza

IEnumerable<string> strings = new List<string>();
IEnumerable<object> objects = strings; // Covarianza

Ejemplo de Contravarianza

Action<object> actObject = (obj) => Console.WriteLine(obj);
Action<string> actString = actObject; // Contravarianza

Resumen

Los genéricos en C# proporcionan una forma de crear componentes de código flexibles y reutilizables que pueden trabajar con cualquier tipo de dato mientras mantienen la seguridad de tipos. Al entender y utilizar clases, métodos y colecciones genéricas, puedes escribir código más eficiente y mantenible. Las restricciones en los parámetros de tipo y conceptos avanzados como la covarianza y la contravarianza mejoran aún más el poder de los genéricos, convirtiéndolos en una herramienta esencial en el conjunto de herramientas de un programador de C#.
