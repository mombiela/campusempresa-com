Introducción a las Listas Enlazadas

Las listas enlazadas son una estructura de datos fundamental en la informática, proporcionando una forma de almacenar una colección de elementos en un orden lineal. A diferencia de los arreglos, las listas enlazadas no requieren una asignación de memoria contigua, lo que las hace más flexibles para ciertos tipos de operaciones.

Conceptos Clave

  • Nodo: La unidad básica de una lista enlazada, que contiene datos y una referencia (o puntero) al siguiente nodo.
  • Cabeza: El primer nodo en una lista enlazada.
  • Cola: El último nodo en una lista enlazada, que apunta a null.
  • Lista Enlazada Simple: Una lista enlazada donde cada nodo apunta al siguiente nodo.
  • Lista Enlazada Doble: Una lista enlazada donde cada nodo apunta tanto al siguiente como al nodo anterior.

Lista Enlazada Simple

Estructura de un Nodo

En C#, un nodo en una lista enlazada simple puede ser representado como una clase con dos propiedades: una para los datos y otra para la referencia al siguiente nodo.

public class Node
{
    public int Data { get; set; }
    public Node Next { get; set; }

    public Node(int data)
    {
        Data = data;
        Next = null;
    }
}

Creando una Lista Enlazada Simple

Para crear una lista enlazada simple, necesitamos una clase que gestione los nodos y proporcione métodos para manipular la lista.

public class SinglyLinkedList
{
    private Node head;

    public SinglyLinkedList()
    {
        head = null;
    }

    // Método para agregar un nodo al final
    public void AddLast(int data)
    {
        Node newNode = new Node(data);
        if (head == null)
        {
            head = newNode;
        }
        else
        {
            Node current = head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
        }
    }

    // Método para mostrar la lista
    public void Display()
    {
        Node current = head;
        while (current != null)
        {
            Console.Write(current.Data + " -> ");
            current = current.Next;
        }
        Console.WriteLine("null");
    }
}

Ejemplo de Uso

class Program
{
    static void Main(string[] args)
    {
        SinglyLinkedList list = new SinglyLinkedList();
        list.AddLast(1);
        list.AddLast(2);
        list.AddLast(3);
        list.Display(); // Salida: 1 -> 2 -> 3 -> null
    }
}

Lista Enlazada Doble

Estructura de un Nodo

Un nodo en una lista enlazada doble contiene referencias tanto al siguiente como al nodo anterior.

public class DoublyNode
{
    public int Data { get; set; }
    public DoublyNode Next { get; set; }
    public DoublyNode Previous { get; set; }

    public DoublyNode(int data)
    {
        Data = data;
        Next = null;
        Previous = null;
    }
}

Creando una Lista Enlazada Doble

Una clase de lista enlazada doble gestiona los nodos y proporciona métodos para manipular la lista.

public class DoublyLinkedList
{
    private DoublyNode head;

    public DoublyLinkedList()
    {
        head = null;
    }

    // Método para agregar un nodo al final
    public void AddLast(int data)
    {
        DoublyNode newNode = new DoublyNode(data);
        if (head == null)
        {
            head = newNode;
        }
        else
        {
            DoublyNode current = head;
            while (current.Next != null)
            {
                current = current.Next;
            }
            current.Next = newNode;
            newNode.Previous = current;
        }
    }

    // Método para mostrar la lista
    public void Display()
    {
        DoublyNode current = head;
        while (current != null)
        {
            Console.Write(current.Data + " <-> ");
            current = current.Next;
        }
        Console.WriteLine("null");
    }
}

Ejemplo de Uso

class Program
{
    static void Main(string[] args)
    {
        DoublyLinkedList list = new DoublyLinkedList();
        list.AddLast(1);
        list.AddLast(2);
        list.AddLast(3);
        list.Display(); // Salida: 1 <-> 2 <-> 3 <-> null
    }
}

Operaciones Avanzadas

Inserción al Comienzo

Agregar un nodo al comienzo de la lista requiere actualizar la cabeza y la referencia siguiente del nuevo nodo.

public void AddFirst(int data)
{
    Node newNode = new Node(data);
    newNode.Next = head;
    head = newNode;
}

Eliminación de un Nodo

Eliminar un nodo implica actualizar las referencias de los nodos anterior y siguiente para omitir el nodo que se va a eliminar.

public void Delete(int data)
{
    if (head == null) return;

    if (head.Data == data)
    {
        head = head.Next;
        return;
    }

    Node current = head;
    while (current.Next != null && current.Next.Data != data)
    {
        current = current.Next;
    }

    if (current.Next != null)
    {
        current.Next = current.Next.Next;
    }
}

Invertir una Lista Enlazada

Invertir una lista enlazada implica cambiar la dirección de los punteros.

public void Reverse()
{
    Node previous = null;
    Node current = head;
    Node next = null;

    while (current != null)
    {
        next = current.Next;
        current.Next = previous;
        previous = current;
        current = next;
    }

    head = previous;
}

Conclusión

Las listas enlazadas son una estructura de datos versátil que puede ser utilizada para gestionar eficientemente colecciones de datos. Entender cómo implementar y manipular listas enlazadas en C# es una habilidad valiosa para cualquier programador. Al dominar tanto las listas enlazadas simples como las dobles, así como las operaciones avanzadas como la inserción, eliminación e inversión, puedes manejar una amplia gama de desafíos de programación.
