Conectarse a una base de datos es una habilidad fundamental para cualquier desarrollador de C#. Este tema te guiará a través del proceso desde niveles principiantes hasta avanzados, cubriendo los conceptos básicos de las conexiones a bases de datos, la ejecución de consultas y el manejo eficiente de datos.

Introducción a las Bases de Datos

Antes de sumergirse en el código, es esencial entender qué es una base de datos y por qué se utiliza.

  • Base de Datos: Una colección estructurada de datos que puede ser fácilmente accesible, gestionada y actualizada.
  • SQL (Lenguaje de Consulta Estructurada): Un lenguaje estándar para gestionar y manipular bases de datos.

Configuración de tu Entorno

Para conectarte a una base de datos en C#, necesitarás:

  1. Un Sistema de Gestión de Bases de Datos (DBMS): Ejemplos incluyen SQL Server, MySQL, PostgreSQL, etc.
  2. .NET Framework o .NET Core: Asegúrate de tener la última versión instalada.
  3. Biblioteca de Conexión a la Base de Datos: Para SQL Server, puedes usar System.Data.SqlClient.

Conexión Básica a una Base de Datos SQL Server

Paso 1: Instalar Paquetes Necesarios

Para .NET Core, puedes instalar el paquete System.Data.SqlClient a través de NuGet:

dotnet add package System.Data.SqlClient

Paso 2: Establecer una Conexión

Aquí tienes un ejemplo simple de cómo conectarte a una base de datos SQL Server:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            try
            {
                connection.Open();
                Console.WriteLine("¡Conexión exitosa!");
            }
            catch (Exception ex)
            {
                Console.WriteLine($"Ocurrió un error: {ex.Message}");
            }
        }
    }
}
  • Explicación:
    • SqlConnection: Representa una conexión a una base de datos SQL Server.
    • connectionString: Contiene la información necesaria para conectarse a la base de datos.
    • connection.Open(): Abre la conexión a la base de datos.
    • La declaración using: Asegura que la conexión se cierre y se elimine correctamente.

Ejecución de Comandos SQL

Paso 1: Ejecutar una Consulta Simple

Para ejecutar una consulta, puedes usar la clase SqlCommand:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            
            string query = "SELECT COUNT(*) FROM tu_tabla";
            SqlCommand command = new SqlCommand(query, connection);
            
            int count = (int)command.ExecuteScalar();
            Console.WriteLine($"Número de filas: {count}");
        }
    }
}
  • Explicación:
    • SqlCommand: Representa una declaración SQL para ejecutar contra una base de datos SQL Server.
    • ExecuteScalar(): Ejecuta la consulta y devuelve la primera columna de la primera fila en el conjunto de resultados.

Paso 2: Leer Datos con `SqlDataReader`

Para consultas más complejas, puedes usar SqlDataReader para leer datos fila por fila:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            
            string query = "SELECT * FROM tu_tabla";
            SqlCommand command = new SqlCommand(query, connection);
            
            using (SqlDataReader reader = command.ExecuteReader())
            {
                while (reader.Read())
                {
                    Console.WriteLine($"ID: {reader["ID"]}, Nombre: {reader["Nombre"]}");
                }
            }
        }
    }
}
  • Explicación:
    • ExecuteReader(): Ejecuta la consulta y devuelve un SqlDataReader para leer los resultados.
    • reader.Read(): Avanza al siguiente registro en el conjunto de resultados.

Temas Avanzados

Uso de Parámetros para Prevenir Inyecciones SQL

Para prevenir ataques de inyección SQL, siempre usa parámetros en tus consultas:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            
            string query = "SELECT * FROM tu_tabla WHERE Nombre = @nombre";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@nombre", "John Doe");
            
            using (SqlDataReader reader = command.ExecuteReader())
            {
                while (reader.Read())
                {
                    Console.WriteLine($"ID: {reader["ID"]}, Nombre: {reader["Nombre"]}");
                }
            }
        }
    }
}
  • Explicación:
    • command.Parameters.AddWithValue("@nombre", "John Doe"): Añade un parámetro a la consulta SQL para prevenir inyecciones SQL.

Manejo de Transacciones

Las transacciones aseguran que una serie de operaciones o todas tengan éxito o todas fallen, manteniendo la integridad de los datos:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "Server=tu_servidor;Database=tu_base_de_datos;User Id=tu_usuario;Password=tu_contraseña;";
        
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            connection.Open();
            SqlTransaction transaction = connection.BeginTransaction();
            
            try
            {
                SqlCommand command1 = new SqlCommand("INSERT INTO tu_tabla (Nombre) VALUES ('John Doe')", connection, transaction);
                command1.ExecuteNonQuery();
                
                SqlCommand command2 = new SqlCommand("INSERT INTO tu_tabla (Nombre) VALUES ('Jane Doe')", connection, transaction);
                command2.ExecuteNonQuery();
                
                transaction.Commit();
                Console.WriteLine("Transacción confirmada.");
            }
            catch (Exception ex)
            {
                transaction.Rollback();
                Console.WriteLine($"Transacción revertida: {ex.Message}");
            }
        }
    }
}
  • Explicación:
    • BeginTransaction(): Inicia una nueva transacción.
    • transaction.Commit(): Confirma la transacción.
    • transaction.Rollback(): Revierte la transacción en caso de un error.

Conclusión

Conectarse a una base de datos en C# implica entender los conceptos básicos de los sistemas de gestión de bases de datos, configurar tu entorno y usar las clases y métodos apropiados para ejecutar consultas y manejar datos. Siguiendo las mejores prácticas, como usar parámetros para prevenir inyecciones SQL y manejar transacciones adecuadamente, puedes asegurar que tus aplicaciones sean seguras y confiables.
