Los diccionarios son una estructura de datos fundamental en C# que te permite almacenar pares clave-valor. Son altamente eficientes para búsquedas, inserciones y eliminaciones. Este tema te guiará desde los conceptos básicos hasta el uso avanzado de diccionarios en C#.

Introducción a los Diccionarios

  • Definición: Un diccionario es una colección de pares clave-valor donde cada clave es única.
  • Espacio de nombres: System.Collections.Generic
  • Clase: Dictionary<TKey, TValue>

Creando un Diccionario

Para crear un diccionario, necesitas especificar los tipos para las claves y los valores.

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        // Creando un diccionario con claves de tipo string y valores de tipo int
        Dictionary<string, int> ages = new Dictionary<string, int>();
    }
}

Añadiendo Elementos

Puedes añadir elementos a un diccionario usando el método Add o el indexador.

ages.Add("Alice", 30);
ages["Bob"] = 25;

Accediendo a Elementos

Accede a los elementos por su clave usando el indexador.

int aliceAge = ages["Alice"];
Console.WriteLine($"La edad de Alice es {aliceAge}");

Eliminando Elementos

Elimina elementos usando el método Remove.

ages.Remove("Alice");

Comprobando Claves y Valores

Usa ContainsKey y ContainsValue para comprobar la existencia de claves y valores.

bool hasAlice = ages.ContainsKey("Alice");
bool hasAge30 = ages.ContainsValue(30);

Iterando sobre un Diccionario

Puedes iterar sobre un diccionario usando un bucle foreach.

foreach (KeyValuePair<string, int> kvp in ages)
{
    Console.WriteLine($"Clave: {kvp.Key}, Valor: {kvp.Value}");
}

Operaciones Avanzadas con Diccionarios

Inicializando con Valores

Puedes inicializar un diccionario con valores usando la sintaxis del inicializador de colecciones.

Dictionary<string, int> ages = new Dictionary<string, int>
{
    { "Alice", 30 },
    { "Bob", 25 },
    { "Charlie", 35 }
};

Actualizando Valores

Puedes actualizar valores asignando un nuevo valor a una clave existente.

ages["Bob"] = 26;

Fusionando Diccionarios

Puedes fusionar dos diccionarios usando el método Union de LINQ.

using System.Linq;

Dictionary<string, int> moreAges = new Dictionary<string, int>
{
    { "David", 40 },
    { "Eve", 22 }
};

var mergedAges = ages.Union(moreAges).ToDictionary(kvp => kvp.Key, kvp => kvp.Value);

Manejo de Claves Faltantes

Usa TryGetValue para manejar de manera segura las claves faltantes.

if (ages.TryGetValue("Alice", out int aliceAge))
{
    Console.WriteLine($"La edad de Alice es {aliceAge}");
}
else
{
    Console.WriteLine("Alice no está en el diccionario.");
}

Consideraciones de Rendimiento

  • Complejidad Temporal: Promedio O(1) para búsquedas, inserciones y eliminaciones.
  • Uso de Memoria: Puede ser mayor en comparación con otras colecciones debido a la necesidad de almacenar claves y valores.

Casos de Uso Comunes

  • Configuración de Ajustes: Almacenar configuraciones de la aplicación donde las claves son nombres de ajustes y los valores son valores de ajustes.
  • Caché: Almacenar datos frecuentemente accedidos para una recuperación rápida.
  • Contar Ocurrencias: Contar el número de ocurrencias de elementos en una colección.

Conclusión

Los diccionarios son una estructura de datos versátil y eficiente en C#. Proporcionan búsquedas, inserciones y eliminaciones rápidas, lo que los hace ideales para una amplia gama de aplicaciones. Al entender cómo crear, manipular e iterar sobre diccionarios, puedes aprovechar su poder en tus programas en C#.
