Introducción

Las pilas y colas son estructuras de datos fundamentales en la informática. Se utilizan para almacenar y gestionar datos en un orden específico, y entenderlas es crucial para resolver diversos problemas de programación. En esta sección, exploraremos cómo implementar y usar pilas y colas en C#.

Pilas

¿Qué es una Pila?

Una pila es una colección de elementos que sigue el principio de Último en Entrar, Primero en Salir (LIFO, por sus siglas en inglés). Esto significa que el último elemento añadido a la pila será el primero en ser removido.

Conceptos Clave

  • Push: Añadir un elemento a la cima de la pila.
  • Pop: Remover el elemento de la cima de la pila.
  • Peek: Ver el elemento de la cima sin removerlo.
  • IsEmpty: Verificar si la pila está vacía.

Implementando una Pila en C#

C# proporciona una clase Stack<T> incorporada en el espacio de nombres System.Collections.Generic. Así es como puedes usarla:

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Stack<int> stack = new Stack<int>();

        // Añadir elementos a la pila
        stack.Push(1);
        stack.Push(2);
        stack.Push(3);

        // Ver el elemento de la cima
        Console.WriteLine("Elemento de la cima: " + stack.Peek()); // Salida: 3

        // Remover elementos de la pila
        Console.WriteLine("Elemento removido: " + stack.Pop()); // Salida: 3
        Console.WriteLine("Elemento removido: " + stack.Pop()); // Salida: 2

        // Verificar si la pila está vacía
        Console.WriteLine("¿Está la pila vacía? " + stack.Count == 0); // Salida: Falso
    }
}

Implementación Personalizada de una Pila

Para una comprensión más profunda, implementemos una pila desde cero:

public class CustomStack<T>
{
    private List<T> elements = new List<T>();

    public void Push(T item)
    {
        elements.Add(item);
    }

    public T Pop()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La pila está vacía");

        T item = elements[elements.Count - 1];
        elements.RemoveAt(elements.Count - 1);
        return item;
    }

    public T Peek()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La pila está vacía");

        return elements[elements.Count - 1];
    }

    public bool IsEmpty()
    {
        return elements.Count == 0;
    }
}

Colas

¿Qué es una Cola?

Una cola es una colección de elementos que sigue el principio de Primero en Entrar, Primero en Salir (FIFO, por sus siglas en inglés). Esto significa que el primer elemento añadido a la cola será el primero en ser removido.

Conceptos Clave

  • Enqueue: Añadir un elemento al final de la cola.
  • Dequeue: Remover el elemento del frente de la cola.
  • Peek: Ver el elemento del frente sin removerlo.
  • IsEmpty: Verificar si la cola está vacía.

Implementando una Cola en C#

C# proporciona una clase Queue<T> incorporada en el espacio de nombres System.Collections.Generic. Así es como puedes usarla:

using System;
using System.Collections.Generic;

class Program
{
    static void Main()
    {
        Queue<int> queue = new Queue<int>();

        // Añadir elementos a la cola
        queue.Enqueue(1);
        queue.Enqueue(2);
        queue.Enqueue(3);

        // Ver el elemento del frente
        Console.WriteLine("Elemento del frente: " + queue.Peek()); // Salida: 1

        // Remover elementos de la cola
        Console.WriteLine("Elemento removido: " + queue.Dequeue()); // Salida: 1
        Console.WriteLine("Elemento removido: " + queue.Dequeue()); // Salida: 2

        // Verificar si la cola está vacía
        Console.WriteLine("¿Está la cola vacía? " + queue.Count == 0); // Salida: Falso
    }
}

Implementación Personalizada de una Cola

Para una comprensión más profunda, implementemos una cola desde cero:

public class CustomQueue<T>
{
    private List<T> elements = new List<T>();

    public void Enqueue(T item)
    {
        elements.Add(item);
    }

    public T Dequeue()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La cola está vacía");

        T item = elements[0];
        elements.RemoveAt(0);
        return item;
    }

    public T Peek()
    {
        if (elements.Count == 0)
            throw new InvalidOperationException("La cola está vacía");

        return elements[0];
    }

    public bool IsEmpty()
    {
        return elements.Count == 0;
    }
}

Resumen

En esta sección, cubrimos los conceptos básicos de pilas y colas, incluyendo sus operaciones clave y cómo implementarlas en C#. Exploramos tanto las clases incorporadas Stack<T> y Queue<T> como implementaciones personalizadas para profundizar nuestra comprensión. Dominar estas estructuras de datos es esencial para una gestión eficiente de datos y resolución de problemas en programación.
