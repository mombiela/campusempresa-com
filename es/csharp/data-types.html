Introducción a los Tipos de Datos

Los tipos de datos son fundamentales para cualquier lenguaje de programación, y C# no es una excepción. Definen el tipo de datos que se pueden almacenar y manipular dentro de un programa. Entender los tipos de datos es crucial para escribir código eficiente y libre de errores.

Tipos de Datos Primitivos

C# proporciona varios tipos de datos integrados, también conocidos como tipos de datos primitivos. Estos son los tipos más básicos y se utilizan para almacenar valores simples.

Tipos Enteros

  • int: Representa un entero con signo de 32 bits.
  • long: Representa un entero con signo de 64 bits.
  • short: Representa un entero con signo de 16 bits.
  • byte: Representa un entero sin signo de 8 bits.
int age = 25;
long distance = 9876543210L;
short temperature = -5;
byte level = 255;

Tipos de Punto Flotante

  • float: Representa un número de punto flotante de precisión simple.
  • double: Representa un número de punto flotante de doble precisión.
  • decimal: Representa un valor decimal preciso de 128 bits, adecuado para cálculos financieros.
float pi = 3.14F;
double e = 2.7182818284;
decimal price = 19.99M;

Tipo Booleano

  • bool: Representa un valor booleano que puede ser true o false.
bool isActive = true;
bool isCompleted = false;

Tipo de Carácter

  • char: Representa un solo carácter Unicode de 16 bits.
char grade = 'A';
char symbol = '#';

Tipo de Cadena

  • string: Representa una secuencia de caracteres.
string greeting = "¡Hola, Mundo!";
string name = "Alicia";

Tipos de Datos Complejos

C# también soporta tipos de datos más complejos que pueden almacenar múltiples valores o estructuras más complejas.

Arreglos

Los arreglos son colecciones de elementos del mismo tipo.

int[] numbers = { 1, 2, 3, 4, 5 };
string[] names = { "Alicia", "Bob", "Carlos" };

Listas

Las listas son similares a los arreglos pero son más flexibles y forman parte del espacio de nombres System.Collections.Generic.

using System.Collections.Generic;

List<int> scores = new List<int> { 90, 85, 100 };
scores.Add(95);

Diccionarios

Los diccionarios almacenan pares clave-valor y forman parte del espacio de nombres System.Collections.Generic.

using System.Collections.Generic;

Dictionary<string, int> ages = new Dictionary<string, int>();
ages["Alicia"] = 30;
ages["Bob"] = 25;

Tipos Nulos

Los tipos nulos permiten que los tipos de valor se asignen a null.

int? nullableInt = null;
nullableInt = 5;

Conversión de Tipos

La conversión de tipos es el proceso de convertir un tipo de datos a otro. C# soporta conversiones tanto implícitas como explícitas.

Conversión Implícita

Las conversiones implícitas son manejadas automáticamente por el compilador.

int num = 123;
double doubleNum = num; // Conversión implícita de int a double

Conversión Explícita

Las conversiones explícitas requieren un operador de conversión.

double doubleNum = 123.45;
int num = (int)doubleNum; // Conversión explícita de double a int

Resumen

Entender los tipos de datos es esencial para cualquier programador de C#. Este tema cubrió:

  • Tipos de datos primitivos como int, float, bool, char y string.
  • Tipos de datos complejos como arreglos, listas y diccionarios.
  • Tipos nulos y conversión de tipos.

Al dominar estos conceptos, estarás bien equipado para manejar datos de manera efectiva en tus programas de C#.
