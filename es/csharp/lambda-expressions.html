Las expresiones lambda son una característica poderosa en C# que te permiten escribir expresiones o funciones en línea. Son particularmente útiles para escribir código conciso y legible, especialmente cuando se trabaja con colecciones y LINQ (Language Integrated Query). Este tema te guiará desde los conceptos básicos hasta el uso avanzado de las expresiones lambda en C#.

¿Qué son las Expresiones Lambda?

  • Definición: Una expresión lambda es una función anónima que puedes usar para crear delegados o tipos de árbol de expresiones.
  • Sintaxis: La sintaxis básica de una expresión lambda es (parámetros de entrada) => expresión o bloque de instrucciones.

Sintaxis Básica

(parámetros) => expresión
(parámetros) => { instrucciones }

Ejemplo

Func<int, int> square = x => x * x;
Console.WriteLine(square(5)); // Salida: 25

¿Por Qué Usar Expresiones Lambda?

  • Concisión: Las expresiones lambda te permiten escribir menos código.
  • Legibilidad: Pueden hacer tu código más legible al eliminar código repetitivo.
  • Flexibilidad: Son versátiles y pueden ser usadas en varios escenarios, como consultas LINQ, manejo de eventos, y más.

Uso Básico

Parámetro Único

Func<int, int> increment = x => x + 1;
Console.WriteLine(increment(5)); // Salida: 6

Múltiples Parámetros

Func<int, int, int> add = (x, y) => x + y;
Console.WriteLine(add(3, 4)); // Salida: 7

Sin Parámetros

Action greet = () => Console.WriteLine("¡Hola, Mundo!");
greet(); // Salida: ¡Hola, Mundo!

Uso Avanzado

Uso de Expresiones Lambda con LINQ

Las expresiones lambda se utilizan extensamente con LINQ para realizar operaciones en colecciones.

List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };
var evenNumbers = numbers.Where(n => n % 2 == 0).ToList();
evenNumbers.ForEach(n => Console.WriteLine(n)); // Salida: 2 4

Árboles de Expresiones

Las expresiones lambda pueden ser usadas para crear árboles de expresiones, que son estructuras de datos que representan código.

Expression<Func<int, bool>> isEven = x => x % 2 == 0;
Console.WriteLine(isEven); // Salida: x => (x % 2 == 0)

Captura de Variables

Las expresiones lambda pueden capturar variables de su ámbito circundante.

int factor = 2;
Func<int, int> multiply = x => x * factor;
Console.WriteLine(multiply(3)); // Salida: 6

Casos de Uso Comunes

Manejo de Eventos

Las expresiones lambda pueden ser usadas para manejar eventos de manera concisa.

button.Click += (sender, e) => { Console.WriteLine("¡Botón clicado!"); };

Ordenar Colecciones

Puedes usar expresiones lambda para definir lógica de ordenamiento personalizada.

List<string> names = new List<string> { "Alice", "Bob", "Charlie" };
names.Sort((a, b) => a.Length.CompareTo(b.Length));
names.ForEach(name => Console.WriteLine(name)); // Salida: Bob Alice Charlie

Conclusión

Las expresiones lambda son una característica versátil y poderosa en C#. Te permiten escribir código conciso y legible, especialmente cuando trabajas con colecciones y LINQ. Al entender y utilizar las expresiones lambda, puedes mejorar significativamente la eficiencia y legibilidad de tu código.

Resumen

  • Las expresiones lambda son funciones anónimas que pueden ser usadas para crear delegados o tipos de árbol de expresiones.
  • Proporcionan una manera concisa y legible de escribir expresiones o funciones en línea.
  • Las expresiones lambda se utilizan extensamente con LINQ, manejo de eventos y ordenamiento personalizado.
  • Entender las expresiones lambda puede mejorar enormemente tus habilidades de programación en C#.

Al dominar las expresiones lambda, estarás bien equipado para escribir código en C# más eficiente y mantenible.
