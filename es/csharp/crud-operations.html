CRUD significa Crear, Leer, Actualizar y Eliminar. Estas son las cuatro operaciones básicas para gestionar datos en una base de datos. En esta sección, exploraremos cómo realizar operaciones CRUD en C# utilizando varios enfoques, incluyendo ADO.NET y Entity Framework.

Introducción a las Operaciones CRUD

  • Crear: Añadir nuevos registros a la base de datos.
  • Leer: Recuperar registros existentes de la base de datos.
  • Actualizar: Modificar registros existentes en la base de datos.
  • Eliminar: Eliminar registros de la base de datos.

Configuración del Entorno

Antes de sumergirnos en las operaciones CRUD, asegúrate de tener la siguiente configuración:

  • Visual Studio instalado.
  • SQL Server u otro sistema de gestión de bases de datos.
  • Una comprensión básica de la sintaxis de C#.

Usando ADO.NET para Operaciones CRUD

Operación de Crear

Para insertar un nuevo registro en la base de datos:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "tu_cadena_de_conexión_aquí";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "INSERT INTO Students (Name, Age) VALUES (@Name, @Age)";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 21);

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Verificar si la inserción fue exitosa
            if (result > 0)
                Console.WriteLine("Registro insertado exitosamente.");
            else
                Console.WriteLine("Error al insertar el registro.");
        }
    }
}

Operación de Leer

Para recuperar registros de la base de datos:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "tu_cadena_de_conexión_aquí";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "SELECT * FROM Students";
            SqlCommand command = new SqlCommand(query, connection);

            connection.Open();
            SqlDataReader reader = command.ExecuteReader();

            while (reader.Read())
            {
                Console.WriteLine($"ID: {reader["ID"]}, Nombre: {reader["Name"]}, Edad: {reader["Age"]}");
            }
        }
    }
}

Operación de Actualizar

Para actualizar un registro existente en la base de datos:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "tu_cadena_de_conexión_aquí";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "UPDATE Students SET Age = @Age WHERE Name = @Name";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");
            command.Parameters.AddWithValue("@Age", 22);

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Verificar si la actualización fue exitosa
            if (result > 0)
                Console.WriteLine("Registro actualizado exitosamente.");
            else
                Console.WriteLine("Error al actualizar el registro.");
        }
    }
}

Operación de Eliminar

Para eliminar un registro de la base de datos:

using System;
using System.Data.SqlClient;

class Program
{
    static void Main()
    {
        string connectionString = "tu_cadena_de_conexión_aquí";
        using (SqlConnection connection = new SqlConnection(connectionString))
        {
            string query = "DELETE FROM Students WHERE Name = @Name";
            SqlCommand command = new SqlCommand(query, connection);
            command.Parameters.AddWithValue("@Name", "John Doe");

            connection.Open();
            int result = command.ExecuteNonQuery();

            // Verificar si la eliminación fue exitosa
            if (result > 0)
                Console.WriteLine("Registro eliminado exitosamente.");
            else
                Console.WriteLine("Error al eliminar el registro.");
        }
    }
}

Usando Entity Framework para Operaciones CRUD

Entity Framework (EF) es un ORM (Mapeador Objeto-Relacional) que simplifica el acceso a datos en aplicaciones .NET.

Configuración de Entity Framework

  1. Instalar Entity Framework a través del Administrador de Paquetes NuGet:

    Install-Package EntityFramework

  2. Crear una clase de modelo y una clase de contexto:

using System.Data.Entity;

public class Student
{
    public int ID { get; set; }
    public string Name { get; set; }
    public int Age { get; set; }
}

public class SchoolContext : DbContext
{
    public DbSet<Student> Students { get; set; }
}

Operación de Crear

Para insertar un nuevo registro usando Entity Framework:

using System;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = new Student { Name = "Jane Doe", Age = 20 };
            context.Students.Add(student);
            context.SaveChanges();
            Console.WriteLine("Registro insertado exitosamente.");
        }
    }
}

Operación de Leer

Para recuperar registros usando Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var students = context.Students.ToList();
            foreach (var student in students)
            {
                Console.WriteLine($"ID: {student.ID}, Nombre: {student.Name}, Edad: {student.Age}");
            }
        }
    }
}

Operación de Actualizar

Para actualizar un registro existente usando Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = context.Students.FirstOrDefault(s => s.Name == "Jane Doe");
            if (student != null)
            {
                student.Age = 21;
                context.SaveChanges();
                Console.WriteLine("Registro actualizado exitosamente.");
            }
        }
    }
}

Operación de Eliminar

Para eliminar un registro usando Entity Framework:

using System;
using System.Linq;

class Program
{
    static void Main()
    {
        using (var context = new SchoolContext())
        {
            var student = context.Students.FirstOrDefault(s => s.Name == "Jane Doe");
            if (student != null)
            {
                context.Students.Remove(student);
                context.SaveChanges();
                Console.WriteLine("Registro eliminado exitosamente.");
            }
        }
    }
}

Conclusión

En esta sección, cubrimos los conceptos básicos de las operaciones CRUD en C# utilizando tanto ADO.NET como Entity Framework. Aprendimos cómo crear, leer, actualizar y eliminar registros en una base de datos. Entender estas operaciones es fundamental para cualquier aplicación que interactúe con una base de datos. A medida que avances, puedes explorar temas más avanzados como la gestión de transacciones, procedimientos almacenados y optimización del rendimiento.
