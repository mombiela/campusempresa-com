Introducción

Las declaraciones switch en C# proporcionan una forma de ejecutar diferentes bloques de código basados en el valor de una variable o expresión. Son una alternativa al uso de múltiples declaraciones if-else y pueden hacer que tu código sea más limpio y legible.

Sintaxis Básica

La sintaxis básica de una declaración switch en C# es la siguiente:

switch (expression)
{
    case value1:
        // Código a ejecutar si expression == value1
        break;
    case value2:
        // Código a ejecutar si expression == value2
        break;
    // Casos adicionales...
    default:
        // Código a ejecutar si ninguno de los casos anteriores coincide
        break;
}

Ejemplo

Aquí tienes un ejemplo simple para ilustrar el uso de una declaración switch:

int day = 3;

switch (day)
{
    case 1:
        Console.WriteLine("Lunes");
        break;
    case 2:
        Console.WriteLine("Martes");
        break;
    case 3:
        Console.WriteLine("Miércoles");
        break;
    case 4:
        Console.WriteLine("Jueves");
        break;
    case 5:
        Console.WriteLine("Viernes");
        break;
    case 6:
        Console.WriteLine("Sábado");
        break;
    case 7:
        Console.WriteLine("Domingo");
        break;
    default:
        Console.WriteLine("Día inválido");
        break;
}

Conceptos Clave

  • Expresión: El valor o variable que se está evaluando.
  • Caso: Cada valor posible de la expresión que deseas manejar.
  • Break: Termina el bloque de caso actual.
  • Default: Opcional; se ejecuta si ninguno de los casos coincide con la expresión.

Uso Avanzado

Coincidencia de Patrones

C# 7.0 introdujo la coincidencia de patrones en las declaraciones switch, permitiendo condiciones más complejas.

object obj = "Hola";

switch (obj)
{
    case int i:
        Console.WriteLine($"Entero: {i}");
        break;
    case string s:
        Console.WriteLine($"Cadena: {s}");
        break;
    default:
        Console.WriteLine("Tipo desconocido");
        break;
}

Expresiones Switch

C# 8.0 introdujo las expresiones switch, que proporcionan una sintaxis más concisa.

int number = 3;
string result = number switch
{
    1 => "Uno",
    2 => "Dos",
    3 => "Tres",
    _ => "Desconocido"
};

Console.WriteLine(result); // Salida: Tres

Errores Comunes

  • Olvidar la declaración break: Esto puede llevar a un comportamiento de caída, donde se ejecutan múltiples casos.
  • No manejar todos los valores posibles: Siempre incluye un caso default para manejar valores inesperados.

Ejercicios

  1. Declaración Switch Básica: Escribe una declaración switch que imprima el nombre de un mes basado en una entrada entera (1-12).
  2. Coincidencia de Patrones: Crea una declaración switch que maneje diferentes tipos de objetos (por ejemplo, int, string, bool) e imprima un mensaje para cada tipo.
  3. Expresión Switch: Usa una expresión switch para convertir una calificación (A, B, C, D, F) en una descripción correspondiente (Excelente, Bueno, Promedio, Pobre, Fallo).

Conclusión

Las declaraciones switch son una herramienta poderosa en C# para manejar múltiples condiciones de una manera limpia y legible. Al entender la sintaxis básica y las características avanzadas como la coincidencia de patrones y las expresiones switch, puedes escribir código más eficiente y mantenible. Siempre recuerda manejar todos los casos posibles y usar el caso default para cubrir valores inesperados.
