Introducción

Los operadores son símbolos especiales en C# que realizan operaciones sobre variables y valores. Son fundamentales para la programación ya que permiten manipular datos y variables de diversas maneras. Esta sección cubrirá los diferentes tipos de operadores disponibles en C#, desde aritmética básica hasta operadores más avanzados.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División
  • % : Módulo (resto de una división)

Ejemplo

int a = 10;
int b = 3;

int suma = a + b;        // 13
int diferencia = a - b;  // 7
int producto = a * b;    // 30
int cociente = a / b;    // 3
int resto = a % b;       // 1

Console.WriteLine($"Suma: {suma}, Diferencia: {diferencia}, Producto: {producto}, Cociente: {cociente}, Resto: {resto}");

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores.

  • == : Igual a
  • != : No igual a
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual a
  • <= : Menor o igual a

Ejemplo

int a = 10;
int b = 3;

bool esIgual = (a == b);       // false
bool noEsIgual = (a != b);     // true
bool esMayor = (a > b);        // true
bool esMenor = (a < b);        // false
bool esMayorOIgual = (a >= b); // true
bool esMenorOIgual = (a <= b); // false

Console.WriteLine($"Igual: {esIgual}, No Igual: {noEsIgual}, Mayor: {esMayor}, Menor: {esMenor}, Mayor o Igual: {esMayorOIgual}, Menor o Igual: {esMenorOIgual}");

Operadores Lógicos

Los operadores lógicos se utilizan para combinar declaraciones condicionales.

  • && : AND lógico
  • || : OR lógico
  • ! : NOT lógico

Ejemplo

bool a = true;
bool b = false;

bool resultadoAnd = a && b; // false
bool resultadoOr = a || b;  // true
bool resultadoNot = !a;     // false

Console.WriteLine($"AND: {resultadoAnd}, OR: {resultadoOr}, NOT: {resultadoNot}");

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • = : Asignación simple
  • += : Sumar y asignar
  • -= : Restar y asignar
  • *= : Multiplicar y asignar
  • /= : Dividir y asignar
  • %= : Módulo y asignar

Ejemplo

int a = 10;

a += 5; // a = a + 5;  => 15
a -= 3; // a = a - 3;  => 12
a *= 2; // a = a * 2;  => 24
a /= 4; // a = a / 4;  => 6
a %= 4; // a = a % 4;  => 2

Console.WriteLine($"Valor final de a: {a}");

Operadores a Nivel de Bits

Los operadores a nivel de bits se utilizan para realizar operaciones en representaciones binarias de números.

  • & : AND a nivel de bits
  • | : OR a nivel de bits
  • ^ : XOR a nivel de bits
  • ~ : NOT a nivel de bits
  • << : Desplazamiento a la izquierda
  • >> : Desplazamiento a la derecha

Ejemplo

int a = 5;  // 0101 en binario
int b = 3;  // 0011 en binario

int resultadoAnd = a & b; // 0001 => 1
int resultadoOr = a | b;  // 0111 => 7
int resultadoXor = a ^ b; // 0110 => 6
int resultadoNot = ~a;    // 1010 => -6 (en forma de complemento a 2)
int desplazamientoIzquierda = a << 1; // 1010 => 10
int desplazamientoDerecha = a >> 1;   // 0010 => 2

Console.WriteLine($"AND: {resultadoAnd}, OR: {resultadoOr}, XOR: {resultadoXor}, NOT: {resultadoNot}, Desplazamiento Izquierda: {desplazamientoIzquierda}, Desplazamiento Derecha: {desplazamientoDerecha}");

Operador Condicional (Ternario)

El operador condicional es una forma abreviada de una declaración if-else.

  • ?: : Condicional

Ejemplo

int a = 10;
int b = 20;

string resultado = (a > b) ? "a es mayor" : "b es mayor";

Console.WriteLine(resultado); // Salida: b es mayor

Operadores de Fusión Nula

Los operadores de fusión nula se utilizan para proporcionar un valor predeterminado para tipos anulables.

  • ?? : Fusión nula
  • ??= : Asignación de fusión nula

Ejemplo

int? a = null;
int b = a ?? 5; // Si a es nulo, b será 5

int? c = null;
c ??= 10; // Si c es nulo, asignar 10 a c

Console.WriteLine($"b: {b}, c: {c}"); // Salida: b: 5, c: 10

Resumen

Los operadores en C# son esenciales para realizar diversas operaciones sobre datos. Desde aritmética básica hasta operaciones más complejas a nivel de bits, entender cómo usar estos operadores de manera efectiva es crucial para cualquier programador de C#. Practica usando estos operadores en diferentes escenarios para volverte competente en la manipulación de datos y la escritura de código eficiente.
