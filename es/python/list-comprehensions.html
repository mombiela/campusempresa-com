Introducción a las Comprensiones de Listas

Las comprensiones de listas proporcionan una forma concisa de crear listas en Python. A menudo son más legibles y rápidas que la creación de listas basada en bucles for tradicionales.

Conceptos Clave

  • Sintaxis: [expresión para elemento en iterable si condición]
  • Expresión: El valor de salida para cada elemento en la lista.
  • Iterable: Cualquier iterable de Python (por ejemplo, lista, rango, cadena).
  • Condición (opcional): Un filtro que selecciona qué elementos incluir.

Ejemplos Básicos

Comencemos con un ejemplo simple para entender la sintaxis básica.

Ejemplo 1: Creando una Lista de Cuadrados

cuadrados = [x**2 para x en range(10)]
print(cuadrados)
  • Explicación: Esta comprensión de lista itera sobre cada número x en el rango de 0 a 9, calcula x**2, y recoge estos valores en una nueva lista.

Ejemplo 2: Filtrando con Condiciones

cuadrados_pares = [x**2 para x en range(10) si x % 2 == 0]
print(cuadrados_pares)
  • Explicación: Este ejemplo añade una condición para filtrar solo los números pares antes de calcular sus cuadrados.

Comprensiones de Listas Avanzadas

A medida que te sientas más cómodo con las comprensiones de listas, puedes comenzar a usar expresiones más complejas y comprensiones anidadas.

Comprensiones de Listas Anidadas

Las comprensiones de listas anidadas pueden usarse para aplanar una lista de listas o para crear una matriz.

Ejemplo 3: Aplanando una Lista de Listas

matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
aplanada = [num para fila en matriz para num en fila]
print(aplanada)
  • Explicación: Esta comprensión itera sobre cada fila en la matriz, y luego sobre cada número en la fila, recogiendo todos los números en una sola lista.

Usando Funciones en Comprensiones de Listas

También puedes llamar funciones dentro de una comprensión de lista.

Ejemplo 4: Aplicando una Función

def cuadrado(x):
    return x**2

cuadrados = [cuadrado(x) para x en range(10)]
print(cuadrados)
  • Explicación: Este ejemplo define una función cuadrado y luego la usa dentro de la comprensión de lista para calcular los cuadrados de los números del 0 al 9.

Combinando Múltiples Condiciones

Puedes combinar múltiples condiciones usando operadores lógicos.

Ejemplo 5: Múltiples Condiciones

filtrados = [x para x en range(20) si x % 2 == 0 y x % 3 == 0]
print(filtrados)
  • Explicación: Esta comprensión filtra números que son divisibles por 2 y 3.

Consideraciones de Rendimiento

Las comprensiones de listas son generalmente más rápidas que los bucles for tradicionales porque están optimizadas para el rendimiento. Sin embargo, la legibilidad no debe sacrificarse por pequeñas ganancias de rendimiento.

Comparando el Rendimiento

Aquí hay una comparación simple entre un bucle for y una comprensión de lista.

Ejemplo 6: Bucle For vs Comprensión de Lista

import time

# Usando un bucle for
inicio = time.time()
cuadrados_bucle = []
para x en range(1000000):
    cuadrados_bucle.append(x**2)
fin = time.time()
print("Tiempo del bucle for:", fin - inicio)

# Usando una comprensión de lista
inicio = time.time()
cuadrados_comp = [x**2 para x en range(1000000)]
fin = time.time()
print("Tiempo de la comprensión de lista:", fin - inicio)
  • Explicación: Este código mide el tiempo tomado para crear una lista de cuadrados usando tanto un bucle for como una comprensión de lista.

Conclusión

Las comprensiones de listas son una característica poderosa en Python que puede hacer tu código más conciso y legible. Son adecuadas para una amplia gama de tareas, desde la creación simple de listas hasta operaciones más complejas que involucran bucles anidados y condiciones. A medida que practiques, las encontrarás una herramienta invaluable en tu kit de herramientas de programación en Python.
