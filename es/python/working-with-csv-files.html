Introducción

Los archivos CSV (Comma-Separated Values) son un formato común para almacenar datos tabulares. Son archivos de texto simples donde cada línea representa una fila de datos, y cada valor en la fila está separado por una coma. Python proporciona varias formas de trabajar con archivos CSV, lo que facilita la lectura y escritura de estos archivos.

Leyendo Archivos CSV

Usando el Módulo `csv`

El módulo csv en la biblioteca estándar de Python proporciona funcionalidad tanto para leer como para escribir archivos CSV.

Leyendo un Archivo CSV

Para leer un archivo CSV, puedes usar el objeto csv.reader.

import csv

with open('example.csv', mode='r') as file:
    csv_reader = csv.reader(file)
    for row in csv_reader:
        print(row)
  • Explicación:
    • open('example.csv', mode='r'): Abre el archivo en modo lectura.
    • csv.reader(file): Crea un objeto lector de CSV.
    • for row in csv_reader: Itera sobre cada fila en el archivo CSV.

Leyendo un Archivo CSV con Encabezados

Si tu archivo CSV tiene encabezados, puedes usar el objeto csv.DictReader para leer el archivo en un diccionario.

import csv

with open('example_with_headers.csv', mode='r') as file:
    csv_reader = csv.DictReader(file)
    for row in csv_reader:
        print(row)
  • Explicación:
    • csv.DictReader(file): Lee el archivo CSV en un diccionario donde las claves son los encabezados de las columnas.

Escribiendo Archivos CSV

Usando el Módulo `csv`

Para escribir en un archivo CSV, puedes usar el objeto csv.writer.

Escribiendo en un Archivo CSV

import csv

data = [
    ['Name', 'Age', 'City'],
    ['Alice', 30, 'New York'],
    ['Bob', 25, 'Los Angeles'],
    ['Charlie', 35, 'Chicago']
]

with open('output.csv', mode='w', newline='') as file:
    csv_writer = csv.writer(file)
    csv_writer.writerows(data)
  • Explicación:
    • open('output.csv', mode='w', newline=''): Abre el archivo en modo escritura.
    • csv.writer(file): Crea un objeto escritor de CSV.
    • csv_writer.writerows(data): Escribe múltiples filas en el archivo CSV.

Escribiendo un Archivo CSV con Encabezados

Si deseas escribir un archivo CSV con encabezados, puedes usar el objeto csv.DictWriter.

import csv

data = [
    {'Name': 'Alice', 'Age': 30, 'City': 'New York'},
    {'Name': 'Bob', 'Age': 25, 'City': 'Los Angeles'},
    {'Name': 'Charlie', 'Age': 35, 'City': 'Chicago'}
]

with open('output_with_headers.csv', mode='w', newline='') as file:
    fieldnames = ['Name', 'Age', 'City']
    csv_writer = csv.DictWriter(file, fieldnames=fieldnames)
    csv_writer.writeheader()
    csv_writer.writerows(data)
  • Explicación:
    • csv.DictWriter(file, fieldnames=fieldnames): Crea un objeto escritor de CSV con los nombres de campo especificados.
    • csv_writer.writeheader(): Escribe la fila de encabezado en el archivo CSV.

Manejo Avanzado de CSV

Usando `pandas` para Archivos CSV

Para un manejo más avanzado de CSV, se recomienda altamente la biblioteca pandas. Proporciona potentes capacidades de manipulación de datos.

Leyendo un Archivo CSV con pandas

import pandas as pd

df = pd.read_csv('example.csv')
print(df)
  • Explicación:
    • pd.read_csv('example.csv'): Lee el archivo CSV en un DataFrame.

Escribiendo en un Archivo CSV con pandas

import pandas as pd

data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [30, 25, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}

df = pd.DataFrame(data)
df.to_csv('output.csv', index=False)
  • Explicación:
    • pd.DataFrame(data): Crea un DataFrame a partir de los datos.
    • df.to_csv('output.csv', index=False): Escribe el DataFrame en un archivo CSV sin el índice.

Conclusión

Trabajar con archivos CSV en Python es sencillo con el módulo csv incorporado para tareas básicas y la biblioteca pandas para operaciones más avanzadas. Entender cómo leer y escribir archivos CSV es esencial para tareas de manipulación y análisis de datos. Al dominar estas técnicas, puedes manejar archivos CSV de manera eficiente en tus proyectos de Python.
