Introducción

En esta sección, cubriremos los conceptos fundamentales de las operaciones de entrada y salida en Python. Entender cómo manejar la entrada y salida es crucial para interactuar con los usuarios y otros sistemas.

Entrada en Python

Las operaciones de entrada permiten que tu programa tome datos del usuario u otras fuentes.

Usando la función `input()`

  • La función input() lee una línea de la entrada (usualmente del usuario), la convierte en una cadena y la devuelve.

Ejemplo:

name = input("Introduce tu nombre: ")
print("¡Hola, " + name + "!")
  • En este ejemplo, el programa solicita al usuario que introduzca su nombre y luego lo saluda.

Convirtiendo la Entrada

  • Dado que input() devuelve una cadena, puede que necesites convertirla a otros tipos de datos como enteros o flotantes.

Ejemplo:

age = input("Introduce tu edad: ")
age = int(age)  # Convierte la entrada a un entero
print("Tienes " + str(age) + " años.")
  • Aquí, la entrada del usuario se convierte de una cadena a un entero antes de ser usada.

Salida en Python

Las operaciones de salida permiten que tu programa muestre datos al usuario u otros sistemas.

Usando la función `print()`

  • La función print() muestra datos en la salida estándar (usualmente la consola).

Ejemplo:

print("¡Hola, Mundo!")
  • Este simple ejemplo imprime "¡Hola, Mundo!" en la consola.

Formateando la Salida

  • Puedes formatear cadenas usando varios métodos para hacer la salida más legible.

Usando Concatenación:

name = "Alicia"
print("¡Hola, " + name + "!")

Usando el método str.format():

name = "Alicia"
print("¡Hola, {}!".format(name))

Usando f-Strings (Python 3.6+):

name = "Alicia"
print(f"¡Hola, {name}!")

Imprimiendo Múltiples Elementos

  • Puedes imprimir múltiples elementos separándolos con comas.

Ejemplo:

name = "Alicia"
age = 30
print("Nombre:", name, "Edad:", age)
  • Esto imprime "Nombre: Alicia Edad: 30" con espacios entre los elementos.

Entrada y Salida Avanzada

Leyendo de Archivos

  • Puedes leer la entrada de archivos usando la función open() y métodos de archivo.

Ejemplo:

with open('ejemplo.txt', 'r') as file:
    content = file.read()
    print(content)
  • Esto lee todo el contenido de ejemplo.txt y lo imprime.

Escribiendo en Archivos

  • Puedes escribir la salida en archivos usando la función open() con modo de escritura.

Ejemplo:

with open('salida.txt', 'w') as file:
    file.write("¡Hola, Mundo!")
  • Esto escribe "¡Hola, Mundo!" en salida.txt.

Manejando Excepciones

  • Es importante manejar excepciones que puedan ocurrir durante las operaciones de entrada y salida.

Ejemplo:

try:
    with open('ejemplo.txt', 'r') as file:
        content = file.read()
        print(content)
except FileNotFoundError:
    print("El archivo no fue encontrado.")
  • Esto maneja el caso en que ejemplo.txt no existe.

Resumen

  • Entrada: Usa input() para obtener la entrada del usuario y convertirla al tipo de dato requerido.
  • Salida: Usa print() para mostrar datos. Formatea la salida usando concatenación, str.format(), o f-strings.
  • Operaciones con Archivos: Usa open() para leer y escribir en archivos. Maneja excepciones para gestionar errores de manera elegante.

Entender las operaciones básicas de entrada y salida es esencial para crear programas interactivos y amigables para el usuario en Python. Practica estos conceptos para volverte competente en el manejo de datos en tus aplicaciones.
