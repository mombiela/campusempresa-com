Introducción a Matplotlib

Matplotlib es una biblioteca completa para crear visualizaciones estáticas, animadas e interactivas en Python. Es ampliamente utilizada para la visualización de datos debido a su flexibilidad y facilidad de uso.

Conceptos Clave

  • Figura: La ventana o página completa en la que se dibuja todo.
  • Ejes: El área donde se trazan los datos, que incluye los ejes x e y (y posiblemente z).
  • Gráfico: La representación real de los datos (por ejemplo, línea, barra, dispersión).

Instalación

Para instalar Matplotlib, usa el siguiente comando:

pip install matplotlib

Trazado Básico

Creando un Gráfico de Línea Simple

import matplotlib.pyplot as plt

# Datos de muestra
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Creando el gráfico
plt.plot(x, y)

# Añadiendo título y etiquetas
plt.title('Gráfico de Línea Simple')
plt.xlabel('Eje X')
plt.ylabel('Eje Y')

# Mostrando el gráfico
plt.show()
  • plt.plot(): Traza los datos.
  • plt.title(): Añade un título al gráfico.
  • plt.xlabel() y plt.ylabel(): Etiquetan los ejes x e y.
  • plt.show(): Muestra el gráfico.

Personalización de Gráficos

Añadiendo Estilos y Colores

plt.plot(x, y, color='green', linestyle='--', marker='o')
plt.title('Gráfico de Línea Estilizado')
plt.xlabel('Eje X')
plt.ylabel('Eje Y')
plt.show()
  • color: Cambia el color de la línea.
  • linestyle: Cambia el estilo de la línea (por ejemplo, discontinua).
  • marker: Añade marcadores a los puntos de datos.

Añadiendo Leyendas

plt.plot(x, y, label='Números Primos')
plt.title('Gráfico de Línea con Leyenda')
plt.xlabel('Eje X')
plt.ylabel('Eje Y')
plt.legend()
plt.show()
  • label: Añade una etiqueta al gráfico.
  • plt.legend(): Muestra la leyenda en el gráfico.

Trazado Avanzado

Subgráficos

fig, axs = plt.subplots(2, 2)

# Primer subgráfico
axs[0, 0].plot(x, y)
axs[0, 0].set_title('Subgráfico 1')

# Segundo subgráfico
axs[0, 1].plot(x, [i**2 for i in x])
axs[0, 1].set_title('Subgráfico 2')

# Tercer subgráfico
axs[1, 0].plot(x, [i**3 for i in x])
axs[1, 0].set_title('Subgráfico 3')

# Cuarto subgráfico
axs[1, 1].plot(x, [i**0.5 for i in x])
axs[1, 1].set_title('Subgráfico 4')

plt.tight_layout()
plt.show()
  • plt.subplots(): Crea una cuadrícula de subgráficos.
  • axs[fila, col]: Accede a un subgráfico específico.

Gráficos de Barras

categories = ['A', 'B', 'C', 'D']
values = [4, 7, 1, 8]

plt.bar(categories, values)
plt.title('Gráfico de Barras')
plt.xlabel('Categorías')
plt.ylabel('Valores')
plt.show()
  • plt.bar(): Crea un gráfico de barras.

Gráficos de Dispersión

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]
sizes = [20, 50, 80, 200, 500]

plt.scatter(x, y, s=sizes, c='red', alpha=0.5)
plt.title('Gráfico de Dispersión')
plt.xlabel('Eje X')
plt.ylabel('Eje Y')
plt.show()
  • plt.scatter(): Crea un gráfico de dispersión.
  • s: Establece el tamaño de los marcadores.
  • c: Establece el color de los marcadores.
  • alpha: Establece la transparencia de los marcadores.

Conclusión

Matplotlib es una herramienta poderosa para la visualización de datos en Python. Permite la creación de una amplia gama de gráficos y diagramas, desde gráficos de líneas simples hasta subgráficos complejos y gráficos de dispersión. Al dominar Matplotlib, puedes comunicar eficazmente tus ideas a través de visualizaciones atractivas.
