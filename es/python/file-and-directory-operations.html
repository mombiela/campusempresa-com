Introducción

Las operaciones con archivos y directorios son habilidades esenciales para cualquier programador. En Python, los módulos os y shutil proporcionan una amplia gama de funciones para interactuar con el sistema de archivos. Esta sección cubrirá los conceptos básicos de las operaciones con archivos y directorios, desde la lectura y escritura de archivos hasta tareas más avanzadas como recorrer directorios y manejar permisos de archivos.

Trabajando con Archivos

Abrir y Cerrar Archivos

  • Abrir un archivo: Usa la función open() para abrir un archivo.
  • Cerrar un archivo: Siempre cierra el archivo usando el método close() para liberar recursos del sistema.
# Abrir un archivo para lectura
file = open('example.txt', 'r')

# Realizar operaciones con el archivo

# Cerrar el archivo
file.close()

Leer Archivos

  • Leer todo el archivo: Usa el método read().
  • Leer línea por línea: Usa los métodos readline() o readlines().
# Leer todo el archivo
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)

# Leer el archivo línea por línea
with open('example.txt', 'r') as file:
    for line in file:
        print(line.strip())

Escribir en Archivos

  • Escribir texto en un archivo: Usa el método write().
  • Añadir texto a un archivo: Abre el archivo en modo de adición ('a').
# Escribir en un archivo
with open('example.txt', 'w') as file:
    file.write('¡Hola, Mundo!\n')

# Añadir a un archivo
with open('example.txt', 'a') as file:
    file.write('Añadiendo una nueva línea.\n')

Trabajando con Directorios

Crear y Eliminar Directorios

  • Crear un directorio: Usa os.mkdir() o os.makedirs() para directorios anidados.
  • Eliminar un directorio: Usa os.rmdir() o shutil.rmtree() para directorios no vacíos.
import os
import shutil

# Crear un directorio
os.mkdir('new_directory')

# Crear directorios anidados
os.makedirs('parent_directory/child_directory')

# Eliminar un directorio vacío
os.rmdir('new_directory')

# Eliminar un directorio no vacío
shutil.rmtree('parent_directory')

Listar Contenidos del Directorio

  • Listar archivos y directorios: Usa os.listdir() o os.scandir() para obtener información más detallada.
# Listar archivos y directorios
contents = os.listdir('.')
print(contents)

# Listar con información detallada
with os.scandir('.') as entries:
    for entry in entries:
        print(entry.name, entry.is_file(), entry.is_dir())

Operaciones Avanzadas con Archivos y Directorios

Recorrer Directorios

  • Recorrer directorios: Usa os.walk() para generar nombres de archivos en un árbol de directorios.
# Recorrer directorios
for dirpath, dirnames, filenames in os.walk('.'):
    print(f'Directorio encontrado: {dirpath}')
    para filename en filenames:
        print(f'\t{filename}')

Permisos de Archivos

  • Cambiar permisos de archivos: Usa os.chmod() para cambiar el modo de un archivo.
  • Verificar permisos de archivos: Usa os.access() para verificar los permisos de acceso del archivo.
# Cambiar permisos de archivos
os.chmod('example.txt', 0o644)  # Lectura y escritura para el propietario, lectura para otros

# Verificar permisos de archivos
if os.access('example.txt', os.R_OK):
    print('El archivo es legible')
else:
    print('El archivo no es legible')

Resumen

En esta sección, cubrimos los conceptos básicos y algunos temas avanzados relacionados con las operaciones con archivos y directorios en Python. Aprendiste cómo abrir, leer, escribir y cerrar archivos, así como cómo crear, eliminar y listar directorios. También tocamos operaciones más avanzadas como recorrer directorios y manejar permisos de archivos. Dominar estas habilidades te permitirá gestionar eficazmente archivos y directorios en tus aplicaciones Python.
