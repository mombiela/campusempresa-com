Introducción a los Métodos Mágicos

Los métodos mágicos en Python son métodos especiales que comienzan y terminan con dobles guiones bajos (__). También se conocen como métodos dunder. Estos métodos te permiten definir el comportamiento de tus objetos para operaciones integradas como adición, sustracción, comparación y más.

Conceptos Clave

  • Métodos Mágicos: Métodos especiales con dobles guiones bajos antes y después de sus nombres.
  • Sobrecarga de Operadores: Usar métodos mágicos para definir comportamientos personalizados para operadores.
  • Operaciones Integradas: Los métodos mágicos pueden personalizar cómo se comportan los objetos con funciones y operaciones integradas.

Métodos Mágicos Comunes

Inicialización y Representación

  • __init__(self, ...): Inicializa una nueva instancia de una clase.
  • __str__(self): Devuelve una representación en cadena del objeto, utilizada por print() y str().
  • __repr__(self): Devuelve una representación en cadena "oficial" del objeto, utilizada por repr().
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __str__(self):
        return f"{self.name}, {self.age} años"

    def __repr__(self):
        return f"Person(name={self.name}, age={self.age})"

p = Person("Alice", 30)
print(str(p))  # Salida: Alice, 30 años
print(repr(p)) # Salida: Person(name=Alice, age=30)

Operaciones Aritméticas

  • __add__(self, other): Define el comportamiento para el operador +.
  • __sub__(self, other): Define el comportamiento para el operador -.
  • __mul__(self, other): Define el comportamiento para el operador *.
  • __truediv__(self, other): Define el comportamiento para el operador /.
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y

    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)

    def __str__(self):
        return f"Vector({self.x}, {self.y})"

v1 = Vector(2, 3)
v2 = Vector(4, 5)
v3 = v1 + v2
print(v3)  # Salida: Vector(6, 8)

Operaciones de Comparación

  • __eq__(self, other): Define el comportamiento para el operador ==.
  • __lt__(self, other): Define el comportamiento para el operador <.
  • __le__(self, other): Define el comportamiento para el operador <=.
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def __eq__(self, other):
        return self.age == other.age

    def __lt__(self, other):
        return self.age < other.age

p1 = Person("Alice", 30)
p2 = Person("Bob", 25)
print(p1 == p2)  # Salida: False
print(p1 < p2)   # Salida: False

Métodos de Contenedor

  • __len__(self): Define el comportamiento para la función len().
  • __getitem__(self, key): Define el comportamiento para la indexación.
  • __setitem__(self, key, value): Define el comportamiento para la asignación de elementos.
  • __delitem__(self, key): Define el comportamiento para la eliminación de elementos.
class CustomList:
    def __init__(self):
        self.items = []

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __setitem__(self, index, value):
        self.items[index] = value

    def __delitem__(self, index):
        del self.items[index]

cl = CustomList()
cl.items = [1, 2, 3]
print(len(cl))  # Salida: 3
print(cl[1])    # Salida: 2
cl[1] = 5
print(cl[1])    # Salida: 5
del cl[1]
print(cl.items) # Salida: [1, 3]

Métodos Mágicos Avanzados

Acceso a Atributos

  • __getattr__(self, name): Se llama cuando no se encuentra un atributo.
  • __setattr__(self, name, value): Se llama cuando se intenta asignar un atributo.
  • __delattr__(self, name): Se llama cuando se intenta eliminar un atributo.
class DynamicAttributes:
    def __init__(self):
        self.attributes = {}

    def __getattr__(self, name):
        return self.attributes.get(name, None)

    def __setattr__(self, name, value):
        if name == "attributes":
            super().__setattr__(name, value)
        else:
            self.attributes[name] = value

    def __delattr__(self, name):
        if name in self.attributes:
            del self.attributes[name]

da = DynamicAttributes()
da.new_attr = "value"
print(da.new_attr)  # Salida: value
del da.new_attr
print(da.new_attr)  # Salida: None

Gestión de Contexto

  • __enter__(self): Se llama al entrar en un contexto (usando la declaración with).
  • __exit__(self, exc_type, exc_value, traceback): Se llama al salir de un contexto.
class ManagedResource:
    def __enter__(self):
        print("Entrando en el contexto")
        return self

    def __exit__(self, exc_type, exc_value, traceback):
        print("Saliendo del contexto")

with ManagedResource():
    print("Dentro del contexto")
# Salida:
# Entrando en el contexto
# Dentro del contexto
# Saliendo del contexto

Conclusión

Los métodos mágicos en Python proporcionan una forma poderosa de personalizar el comportamiento de los objetos. Te permiten definir cómo los objetos interactúan con operaciones, operadores y funciones integradas. Al comprender y utilizar los métodos mágicos, puedes crear clases más intuitivas y flexibles que se integren perfectamente con la sintaxis y las características de Python.
