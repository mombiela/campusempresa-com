Introducción a los Módulos

Los módulos en Python son archivos que contienen código Python. Este código puede definir funciones, clases y variables. Un módulo también puede incluir código ejecutable.

Creación de un Módulo

Para crear un módulo, simplemente guarda el código que deseas en un archivo con la extensión .py.

# my_module.py

def greet(name):
    return f"Hello, {name}!"

def add(a, b):
    return a + b

Importación de un Módulo

Puedes usar la declaración import para traer un módulo a tu script.

# main.py

import my_module

print(my_module.greet("Alice"))  # Salida: Hello, Alice!
print(my_module.add(5, 3))       # Salida: 8

Uso de `from` para Importar Funciones Específicas

Puedes importar funciones o variables específicas de un módulo usando la palabra clave from.

# main.py

from my_module import greet

print(greet("Bob"))  # Salida: Hello, Bob!

Alias para Módulos

Los módulos pueden recibir un alias usando la palabra clave as.

# main.py

import my_module as mm

print(mm.greet("Charlie"))  # Salida: Hello, Charlie!

Módulos Incorporados

Python viene con una biblioteca estándar de módulos que puedes usar directamente.

Módulos Incorporados Comúnmente Usados

  • math: Proporciona funciones matemáticas.
  • datetime: Suministra clases para manipular fechas y horas.
  • os: Proporciona una forma de usar funcionalidades dependientes del sistema operativo.
import math
import datetime
import os

print(math.sqrt(16))  # Salida: 4.0
print(datetime.datetime.now())  # Salida: Fecha y hora actual
print(os.getcwd())  # Salida: Directorio de trabajo actual

Introducción a los Paquetes

Un paquete es una forma de organizar módulos relacionados en una jerarquía de directorios. Un paquete es un directorio que contiene un archivo especial llamado __init__.py y puede contener múltiples módulos y sub-paquetes.

Creación de un Paquete

  1. Crea un directorio para el paquete.
  2. Agrega un archivo __init__.py al directorio.
  3. Agrega módulos al directorio.
my_package/
    __init__.py
    module1.py
    module2.py

Importación desde un Paquete

Puedes importar módulos de un paquete usando notación de puntos.

# my_package/module1.py

def func1():
    return "Function 1"

# my_package/module2.py

def func2():
    return "Function 2"

# main.py

from my_package import module1, module2

print(module1.func1())  # Salida: Function 1
print(module2.func2())  # Salida: Function 2

Importación de Sub-paquetes

Los paquetes pueden contener sub-paquetes, que son paquetes dentro de paquetes.

my_package/
    __init__.py
    sub_package/
        __init__.py
        sub_module.py
# my_package/sub_package/sub_module.py

def sub_func():
    return "Sub Function"

# main.py

from my_package.sub_package import sub_module

print(sub_module.sub_func())  # Salida: Sub Function

Conceptos Avanzados de Módulos y Paquetes

Importaciones Relativas

Las importaciones relativas usan un punto (.) para indicar los paquetes actuales y padres involucrados.

# my_package/module1.py

from .module2 import func2

def combined_func():
    return func2() + " & Function 1"

La Variable `__all__`

La variable __all__ puede ser usada en un módulo para definir una lista de objetos públicos de ese módulo.

# my_package/module1.py

__all__ = ['func1']

def func1():
    return "Function 1"

def _hidden_func():
    return "Hidden Function"

El Archivo `__init__.py`

El archivo __init__.py puede ser usado para inicializar un paquete y también puede ejecutar código de inicialización para el paquete.

# my_package/__init__.py

print("Initializing my_package")

from .module1 import func1
from .module2 import func2

Conclusión

Los módulos y paquetes son conceptos fundamentales en Python que ayudan a organizar y gestionar el código de manera eficiente. Al entender cómo crear, importar y usar módulos y paquetes, puedes escribir código más modular, mantenible y reutilizable. Ya sea que estés trabajando en un pequeño script o en una gran aplicación, aprovechar estos conceptos mejorará significativamente tus habilidades de programación.
