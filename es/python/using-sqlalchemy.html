Introducción a SQLAlchemy

SQLAlchemy es una poderosa herramienta de SQL y una biblioteca de mapeo objeto-relacional (ORM) para Python. Proporciona un conjunto completo de patrones de persistencia a nivel empresarial, diseñados para un acceso a la base de datos eficiente y de alto rendimiento.

  • SQLAlchemy Core: Proporciona una capa de abstracción de SQL centrada en el esquema.
  • SQLAlchemy ORM: Proporciona una interfaz centrada en ORM de nivel superior.

Instalando SQLAlchemy

Para comenzar con SQLAlchemy, necesitas instalarlo usando pip.

pip install sqlalchemy

SQLAlchemy Core

SQLAlchemy Core proporciona un lenguaje de expresión SQL y un Mapeador Objeto-Relacional (ORM). Comenzaremos con el Core para entender los conceptos básicos de SQLAlchemy.

Conectándose a una Base de Datos

Primero, necesitas crear un motor, que es el punto de partida para cualquier aplicación SQLAlchemy.

from sqlalchemy import create_engine

# Reemplaza 'sqlite:///example.db' con tu URL de base de datos
engine = create_engine('sqlite:///example.db')

Creando Tablas

Puedes definir tablas usando la clase Table y metadatos.

from sqlalchemy import Table, Column, Integer, String, MetaData

metadata = MetaData()

users = Table('users', metadata,
    Column('id', Integer, primary_key=True),
    Column('name', String),
    Column('age', Integer)
)

metadata.create_all(engine)  # Crea la tabla

Insertando Datos

Puedes insertar datos en la tabla usando la función insert.

from sqlalchemy import insert

ins = users.insert().values(name='John Doe', age=30)
conn = engine.connect()
result = conn.execute(ins)

Consultando Datos

Puedes consultar datos usando la función select.

from sqlalchemy import select

s = select([users])
result = conn.execute(s)

for row in result:
    print(row)

SQLAlchemy ORM

El ORM te permite mapear clases de Python a tablas de base de datos, proporcionando una abstracción de nivel superior.

Definiendo un Modelo

Defines un modelo creando una clase que hereda de Base.

from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy import Column, Integer, String

Base = declarative_base()

class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, primary_key=True)
    name = Column(String)
    age = Column(Integer)

Creando una Sesión

Se requiere una sesión para interactuar con la base de datos usando el ORM.

from sqlalchemy.orm import sessionmaker

Session = sessionmaker(bind=engine)
session = Session()

Agregando y Confirmando Datos

Puedes agregar y confirmar datos usando la sesión.

new_user = User(name='Jane Doe', age=25)
session.add(new_user)
session.commit()

Consultando Datos

Puedes consultar datos usando la sesión.

users = session.query(User).all()
for user in users:
    print(user.name, user.age)

Temas Avanzados

Relaciones

SQLAlchemy soporta la definición de relaciones entre tablas.

from sqlalchemy import ForeignKey
from sqlalchemy.orm import relationship

class Address(Base):
    __tablename__ = 'addresses'
    id = Column(Integer, primary_key=True)
    user_id = Column(Integer, ForeignKey('users.id'))
    email_address = Column(String)
    user = relationship("User", back_populates="addresses")

User.addresses = relationship("Address", order_by=Address.id, back_populates="user")

Migraciones con Alembic

Alembic es una herramienta ligera de migración de bases de datos para usar con SQLAlchemy.

  • Instalar Alembic: pip install alembic
  • Inicializar Alembic: alembic init alembic
  • Crear una Migración: alembic revision --autogenerate -m "Initial migration"
  • Aplicar Migraciones: alembic upgrade head

Conclusión

SQLAlchemy es una biblioteca versátil y poderosa para interacciones con bases de datos en Python. Ya sea que estés usando el Core para expresiones SQL directas o el ORM para abstracciones de nivel superior, SQLAlchemy proporciona las herramientas que necesitas para una gestión de bases de datos eficiente y efectiva. Al dominar SQLAlchemy, puedes manejar operaciones complejas de bases de datos con facilidad, convirtiéndolo en una habilidad esencial para cualquier desarrollador de Python.
