Introducción

Python es un lenguaje de programación versátil y ampliamente utilizado, conocido por su legibilidad y simplicidad. Esta sección cubrirá la sintaxis fundamental y los tipos de datos básicos en Python, proporcionando una base sólida para principiantes.

Sintaxis de Python

La sintaxis de Python se refiere al conjunto de reglas que definen cómo se escribe e interpreta un programa en Python.

Conceptos Clave

  • Indentación: Python usa la indentación para definir bloques de código. La indentación consistente es crucial.
  • Comentarios: Usa # para comentarios de una sola línea y comillas triples (''' o """) para comentarios de varias líneas.
  • Variables: Las variables no requieren declaración explícita y pueden cambiar de tipo dinámicamente.

Ejemplos de Código

Indentación

if True:
    print("Esto está indentado")
    if True:
        print("Esto está más indentado")

Comentarios

# Este es un comentario de una sola línea

"""
Este es un
comentario de varias líneas
"""

Variables

x = 5       # x es un entero
x = "Hola"  # Ahora x es una cadena

Tipos de Datos Básicos

Python soporta varios tipos de datos básicos que son esenciales para cualquier tarea de programación.

Tipos de Datos Clave

  • Enteros: Números enteros, por ejemplo, 1, 42, -7
  • Flotantes: Números decimales, por ejemplo, 3.14, 0.001, -2.5
  • Cadenas: Texto, por ejemplo, "¡Hola, Mundo!", 'Python'
  • Booleanos: Valores Verdadero o Falso, por ejemplo, True, False

Ejemplos de Código

Enteros y Flotantes

a = 10      # Entero
b = 3.14    # Flotante
c = a + b   # El resultado es un flotante: 13.14

Cadenas

saludo = "Hola"
nombre = 'Alicia'
mensaje = saludo + ", " + nombre + "!"  # Concatenación
print(mensaje)  # Salida: ¡Hola, Alicia!

Booleanos

python_es_divertido = True
el_cielo_es_azul = False

print(python_es_divertido)  # Salida: True
print(el_cielo_es_azul)     # Salida: False

Resumen

Entender la sintaxis de Python y los tipos de datos básicos es esencial para escribir programas efectivos en Python. Puntos clave a recordar:

  • Python usa la indentación para definir bloques de código.
  • Los comentarios se crean usando # para una sola línea y comillas triples para varias líneas.
  • Las variables pueden cambiar de tipo dinámicamente.
  • Los tipos de datos básicos incluyen enteros, flotantes, cadenas y booleanos.

Al dominar estos fundamentos, estarás bien preparado para abordar conceptos y tareas de programación más complejos en Python.
