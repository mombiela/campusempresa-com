Introducción a Django

Django es un framework web de Python de alto nivel que fomenta el desarrollo rápido y un diseño limpio y pragmático. Está construido por desarrolladores experimentados para encargarse de gran parte de las molestias del desarrollo web, para que puedas centrarte en escribir tu aplicación sin necesidad de reinventar la rueda.

  • Características Clave:
    • Desarrollo rápido
    • Seguro
    • Escalable
    • Versátil

Configuración de Django

Antes de sumergirte en Django, necesitas configurar tu entorno.

  • Instalación:

    pip install django

  • Creando un Proyecto:

    django-admin startproject myproject

  • Ejecutando el Servidor de Desarrollo:

    cd myproject
python manage.py runserver

Estructura del Proyecto Django

Entender la estructura de un proyecto Django es crucial para un desarrollo efectivo.

  • Componentes Clave:
    • manage.py: Una utilidad de línea de comandos que te permite interactuar con tu proyecto.
    • settings.py: Configuración para tu proyecto Django.
    • urls.py: Declaraciones de URL para tu proyecto.
    • wsgi.py: Un punto de entrada para servidores web compatibles con WSGI para servir tu proyecto.

Creando una App Django

Los proyectos Django están compuestos por múltiples aplicaciones, cada una con un propósito específico.

  • Creando una App:

    python manage.py startapp myapp

  • Registrando la App: Añade tu app a la lista INSTALLED_APPS en settings.py:

    INSTALLED_APPS = [
    ...
    'myapp',
]

Modelos y Bases de Datos

Los modelos de Django definen la estructura de tus tablas de base de datos.

  • Creando un Modelo:

    from django.db import models

class Article(models.Model):
    title = models.CharField(max_length=100)
    content = models.TextField()
    published_date = models.DateTimeField(auto_now_add=True)

  • Aplicando Migraciones:

    python manage.py makemigrations
python manage.py migrate

Vistas y Plantillas

Las vistas manejan la lógica de tu aplicación, mientras que las plantillas manejan la presentación.

  • Creando una Vista:

    from django.shortcuts import render
from .models import Article

def index(request):
    articles = Article.objects.all()
    return render(request, 'index.html', {'articles': articles})

  • Creando una Plantilla:

    <!-- templates/index.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Lista de Artículos</title>
</head>
<body>
    <h1>Artículos</h1>
    <ul>
        {% for article in articles %}
            <li>{{ article.title }} - {{ article.published_date }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Enrutamiento de URL

El enrutamiento de URL mapea URLs a vistas.

  • Definiendo URLs: 
    from django.urls import path
from . import views

urlpatterns = [
    path('', views.index, name='index'),
]

Formularios e Input de Usuario

Los formularios de Django manejan la entrada y validación de usuario.

  • Creando un Formulario:

    from django import forms

class ArticleForm(forms.ModelForm):
    class Meta:
        model = Article
        fields = ['title', 'content']

  • Manejando el Envío del Formulario:

    from django.shortcuts import render, redirect
from .forms import ArticleForm

def create_article(request):
    if request.method == 'POST':
        form = ArticleForm(request.POST)
        if form.is_valid():
            form.save()
            return redirect('index')
    else:
        form = ArticleForm()
    return render(request, 'create_article.html', {'form': form})

  • Plantilla para el Formulario:

    <!-- templates/create_article.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Crear Artículo</title>
</head>
<body>
    <h1>Crear Artículo</h1>
    <form method="post">
        {% csrf_token %}
        {{ form.as_p }}
        <button type="submit">Guardar</button>
    </form>
</body>
</html>

Interfaz de Administración

Django proporciona una interfaz de administración incorporada para gestionar tus modelos.

  • Habilitando la Interfaz de Administración:

    from django.contrib import admin
from .models import Article

admin.site.register(Article)

  • Accediendo a la Interfaz de Administración:

    • Ejecuta el servidor de desarrollo y navega a /admin.
    • Inicia sesión usando las credenciales del superusuario que creaste.

Conclusión

Django es un framework poderoso que simplifica el desarrollo web proporcionando un conjunto robusto de herramientas y características. Al entender sus componentes principales—modelos, vistas, plantillas y enrutamiento de URL—puedes construir aplicaciones web escalables y mantenibles de manera eficiente. Continúa explorando la extensa documentación y los recursos comunitarios de Django para profundizar en tu conocimiento y habilidades.
