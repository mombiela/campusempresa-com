Introducción

En Python, las funciones son un bloque fundamental que te permite encapsular código en bloques reutilizables. Entender cómo pasar argumentos a las funciones es crucial para escribir código flexible y mantenible. Esta sección cubrirá los diferentes tipos de argumentos de función en Python, desde conceptos básicos hasta avanzados.

Tipos de Argumentos de Función

1. Argumentos Posicionales

Los argumentos posicionales son el tipo más sencillo de argumentos. Se pasan a la función en el mismo orden en que se definen.

def greet(name, age):
    print(f"Hola, {name}! Tienes {age} años.")

greet("Alice", 30)
  • Explicación: En el ejemplo anterior, name y age son argumentos posicionales. Al llamar a greet("Alice", 30), "Alice" se asigna a name y 30 a age.

2. Argumentos de Palabra Clave

Los argumentos de palabra clave se pasan a la función especificando explícitamente el nombre del parámetro junto con su valor.

def greet(name, age):
    print(f"Hola, {name}! Tienes {age} años.")

greet(name="Bob", age=25)
  • Explicación: Aquí, name="Bob" y age=25 son argumentos de palabra clave. El orden de los argumentos no importa cuando se usan argumentos de palabra clave.

3. Argumentos por Defecto

Los argumentos por defecto te permiten definir valores predeterminados para los parámetros. Si no se proporciona un valor para un parámetro con un valor predeterminado, se usa el valor predeterminado.

def greet(name, age=20):
    print(f"Hola, {name}! Tienes {age} años.")

greet("Charlie")
greet("Diana", 22)
  • Explicación: En este ejemplo, age tiene un valor predeterminado de 20. Al llamar a greet("Charlie"), se usa el valor predeterminado. Al llamar a greet("Diana", 22), el valor proporcionado 22 sobrescribe el predeterminado.

4. Argumentos de Longitud Variable

Python permite que las funciones acepten un número arbitrario de argumentos usando *args para argumentos no clave y **kwargs para argumentos de palabra clave.

*args

*args te permite pasar un número variable de argumentos no clave a una función.

def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3, 4))  # Salida: 10
  • Explicación: El parámetro *args recoge todos los argumentos posicionales adicionales en una tupla. En este caso, sum_all(1, 2, 3, 4) suma todos los números proporcionados.

**kwargs

**kwargs te permite pasar un número variable de argumentos de palabra clave a una función.

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Eve", age=28, city="New York")
  • Explicación: El parámetro **kwargs recoge todos los argumentos de palabra clave adicionales en un diccionario. La función luego itera sobre este diccionario para imprimir cada par clave-valor.

Conceptos Avanzados

1. Combinando Diferentes Tipos de Argumentos

Puedes combinar argumentos posicionales, de palabra clave, por defecto y de longitud variable en una sola función. El orden de los parámetros debe ser: posicionales, *args, por defecto, **kwargs.

def complex_function(a, b, *args, c=10, **kwargs):
    print(f"a: {a}, b: {b}, c: {c}")
    print(f"args: {args}")
    print(f"kwargs: {kwargs}")

complex_function(1, 2, 3, 4, 5, d=6, e=7)
  • Explicación: En este ejemplo, a y b son argumentos posicionales, *args recoge argumentos posicionales adicionales, c es un argumento por defecto, y **kwargs recoge argumentos de palabra clave adicionales.

2. Desempaquetando Argumentos

Puedes desempaquetar listas o diccionarios en argumentos de función usando * y ** respectivamente.

def greet(name, age):
    print(f"Hola, {name}! Tienes {age} años.")

args = ["Frank", 35]
kwargs = {"name": "Grace", "age": 40}

greet(*args)
greet(**kwargs)
  • Explicación: La sintaxis *args desempaqueta la lista en argumentos posicionales, mientras que **kwargs desempaqueta el diccionario en argumentos de palabra clave.

Conclusión

Entender los argumentos de función en Python es esencial para escribir código flexible y reutilizable. Esta sección cubrió argumentos posicionales, de palabra clave, por defecto y de longitud variable, junto con conceptos avanzados como combinar diferentes tipos de argumentos y desempaquetar argumentos. Dominar estos conceptos mejorará significativamente tu capacidad para escribir código Python eficiente y limpio.
