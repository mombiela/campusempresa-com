¿Qué es una Base de Datos?

Una base de datos es una colección organizada de datos, generalmente almacenados y accesados electrónicamente desde un sistema informático. Las bases de datos se utilizan para almacenar, recuperar y gestionar datos de manera eficiente.

Conceptos Clave

  • Datos: Hechos y cifras en bruto.
  • Sistema de Gestión de Bases de Datos (DBMS): Software que interactúa con los usuarios finales, aplicaciones y la propia base de datos para capturar y analizar datos.
  • SQL (Lenguaje de Consulta Estructurado): Un lenguaje estándar para gestionar y manipular bases de datos.

Tipos de Bases de Datos

  • Bases de Datos Relacionales: Utilizan tablas para almacenar datos. Ejemplos incluyen MySQL, PostgreSQL y SQLite.
  • Bases de Datos NoSQL: Diseñadas para modelos de datos específicos y tienen esquemas flexibles. Ejemplos incluyen MongoDB, Cassandra y Redis.

Bases de Datos Relacionales vs. NoSQL

| Característica | Bases de Datos Relacionales | Bases de Datos NoSQL | |--------------------|-----------------------------|----------------------| | Estructura de Datos| Tablas | Documentos, Clave-Valor, Grafos, etc. | | Esquema | Fijo | Flexible | | Lenguaje de Consulta| SQL | Varía (por ejemplo, MongoDB usa MQL) | | Transacciones | Cumplimiento ACID | Cumplimiento BASE | | Escalabilidad | Vertical | Horizontal |

Configuración de una Base de Datos en Python

Python proporciona varias bibliotecas para interactuar con bases de datos. Para bases de datos relacionales, SQLite es un buen punto de partida porque es liviano y viene incorporado con Python.

Ejemplo de SQLite

  1. Conectarse a una Base de Datos: Usa la biblioteca sqlite3 para conectarte a una base de datos SQLite.
  2. Crear una Tabla: Define una tabla para almacenar datos.
  3. Insertar Datos: Añade datos a la tabla.
  4. Consultar Datos: Recupera datos de la tabla.
import sqlite3

# Conectar a la base de datos SQLite (o crearla si no existe)
conn = sqlite3.connect('example.db')

# Crear un objeto cursor para interactuar con la base de datos
cursor = conn.cursor()

# Crear una tabla
cursor.execute('''
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY,
    name TEXT NOT NULL,
    age INTEGER NOT NULL
)
''')

# Insertar datos en la tabla
cursor.execute('''
INSERT INTO users (name, age) VALUES ('Alice', 30)
''')

# Confirmar la transacción
conn.commit()

# Consultar datos de la tabla
cursor.execute('SELECT * FROM users')
rows = cursor.fetchall()

for row in rows:
    print(row)

# Cerrar la conexión
conn.close()

Explicación

  • Conectarse a la Base de Datos: sqlite3.connect('example.db') se conecta a un archivo de base de datos llamado example.db. Si el archivo no existe, se creará.
  • Crear un Cursor: conn.cursor() crea un objeto cursor para ejecutar comandos SQL.
  • Crear una Tabla: cursor.execute('CREATE TABLE IF NOT EXISTS users (...)') crea una tabla llamada users con las columnas id, name y age.
  • Insertar Datos: cursor.execute('INSERT INTO users (name, age) VALUES (...)') inserta un nuevo registro en la tabla users.
  • Confirmar la Transacción: conn.commit() guarda los cambios en la base de datos.
  • Consultar Datos: cursor.execute('SELECT * FROM users') recupera todos los registros de la tabla users.
  • Obtener Resultados: cursor.fetchall() obtiene todas las filas de la consulta ejecutada.
  • Cerrar la Conexión: conn.close() cierra la conexión a la base de datos.

Operaciones Avanzadas en Bases de Datos

A medida que te sientas más cómodo con las operaciones básicas de bases de datos, puedes explorar temas más avanzados como:

  • Joins: Combinar filas de dos o más tablas basadas en una columna relacionada.
  • Índices: Mejorar la velocidad de las operaciones de recuperación de datos.
  • Transacciones: Asegurar la integridad de los datos con operaciones de confirmación y reversión.
  • ORM (Mapeo Objeto-Relacional): Usar bibliotecas como SQLAlchemy para interactuar con bases de datos usando objetos de Python.

Ejemplo de Uso de SQLAlchemy

from sqlalchemy import create_engine, Column, Integer, String
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker

# Crear un motor
engine = create_engine('sqlite:///example.db')

# Crear una clase base
Base = declarative_base()

# Definir una clase User
class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, primary_key=True)
    name = Column(String)
    age = Column(Integer)

# Crear la tabla users
Base.metadata.create_all(engine)

# Crear una sesión
Session = sessionmaker(bind=engine)
session = Session()

# Añadir un nuevo usuario
new_user = User(name='Bob', age=25)
session.add(new_user)
session.commit()

# Consultar la base de datos
users = session.query(User).all()
for user in users:
    print(user.name, user.age)

# Cerrar la sesión
session.close()

Explicación

  • Crear un Motor: create_engine('sqlite:///example.db') configura la conexión a la base de datos SQLite.
  • Definir una Clase Base: declarative_base() crea una clase base para definiciones de clases declarativas.
  • Definir una Clase User: La clase User representa la tabla users.
  • Crear la Tabla: Base.metadata.create_all(engine) crea la tabla en la base de datos.
  • Crear una Sesión: sessionmaker(bind=engine) crea una fábrica de sesiones.
  • Añadir un Nuevo Usuario: session.add(new_user) añade un nuevo usuario a la sesión.
  • Confirmar la Transacción: session.commit() guarda los cambios en la base de datos.
  • Consultar la Base de Datos: session.query(User).all() recupera todos los usuarios de la base de datos.
  • Cerrar la Sesión: session.close() cierra la sesión.

Conclusión

Las bases de datos son esenciales para almacenar y gestionar datos en aplicaciones. Python proporciona bibliotecas poderosas como sqlite3 y SQLAlchemy para interactuar con bases de datos de manera eficiente. Al comprender los conceptos básicos de las operaciones de bases de datos y explorar temas avanzados, puedes construir aplicaciones robustas y escalables.
