La Biblioteca Estándar de Python es una colección de módulos y paquetes que vienen con Python, proporcionando una amplia gama de funcionalidades listas para usar. Esta descripción general te introducirá a algunos de los módulos más comúnmente utilizados, ayudándote a entender su propósito y cómo usarlos de manera efectiva.

Funciones Incorporadas

Python viene con un conjunto de funciones incorporadas que siempre están disponibles. Estas funciones realizan operaciones básicas y son esenciales para la programación diaria.

  • print(): Muestra datos en la consola.
  • len(): Devuelve la longitud de un objeto.
  • type(): Devuelve el tipo de un objeto.
  • int(), float(), str(): Funciones de conversión de tipos.

Ejemplo

print("¡Hola, Mundo!")  # Muestra: ¡Hola, Mundo!
print(len("Hola"))      # Muestra: 4
print(type(42))         # Muestra: <class 'int'>
print(int("123"))       # Muestra: 123

Módulo `math`

El módulo math proporciona funciones y constantes matemáticas.

  • Funciones Comunes:
    • math.sqrt(x): Devuelve la raíz cuadrada de x.
    • math.pow(x, y): Devuelve x elevado a la potencia de y.
    • math.sin(x), math.cos(x), math.tan(x): Funciones trigonométricas.
  • Constantes:
    • math.pi: El valor de π.
    • math.e: El valor de e (número de Euler).

Ejemplo

import math

print(math.sqrt(16))    # Muestra: 4.0
print(math.pow(2, 3))   # Muestra: 8.0
print(math.pi)          # Muestra: 3.141592653589793

Módulo `datetime`

El módulo datetime proporciona clases para manipular fechas y horas.

  • Clases:
    • datetime.date: Representa una fecha (año, mes, día).
    • datetime.time: Representa una hora (hora, minuto, segundo, microsegundo).
    • datetime.datetime: Combina fecha y hora.
  • Métodos:
    • datetime.datetime.now(): Devuelve la fecha y hora actuales.
    • datetime.datetime.strptime(date_string, format): Convierte una cadena en un objeto datetime.
    • datetime.datetime.strftime(format): Formatea un objeto datetime en una cadena.

Ejemplo

from datetime import datetime

now = datetime.now()
print(now)  # Muestra: fecha y hora actuales

date_string = "2023-10-01"
date_object = datetime.strptime(date_string, "%Y-%m-%d")
print(date_object)  # Muestra: 2023-10-01 00:00:00

formatted_date = now.strftime("%Y-%m-%d %H:%M:%S")
print(formatted_date)  # Muestra: fecha y hora actuales en el formato especificado

Módulo `os`

El módulo os proporciona una forma de usar la funcionalidad dependiente del sistema operativo.

  • Funciones Comunes:
    • os.getcwd(): Devuelve el directorio de trabajo actual.
    • os.listdir(path): Devuelve una lista de entradas en el directorio dado por path.
    • os.mkdir(path): Crea un directorio llamado path.
    • os.remove(path): Elimina el archivo path.

Ejemplo

import os

print(os.getcwd())  # Muestra: directorio de trabajo actual

os.mkdir("new_directory")
print(os.listdir("."))  # Muestra: lista de archivos y directorios en el directorio actual

os.remove("new_directory")

Módulo `sys`

El módulo sys proporciona acceso a algunas variables usadas o mantenidas por el intérprete y a funciones que interactúan fuertemente con el intérprete.

  • Funciones Comunes:
    • sys.argv: Una lista de argumentos de línea de comandos pasados al script.
    • sys.exit([arg]): Sale de Python.
    • sys.path: Una lista de cadenas que especifica la ruta de búsqueda para módulos.

Ejemplo

import sys

print(sys.argv)  # Muestra: lista de argumentos de línea de comandos

sys.exit()  # Sale del script

Módulo `json`

El módulo json proporciona una forma fácil de codificar y decodificar datos en formato JSON.

  • Funciones Comunes:
    • json.dumps(obj): Serializa obj a una cadena en formato JSON.
    • json.loads(s): Deserializa s (una cadena en formato JSON) a un objeto de Python.

Ejemplo

import json

data = {"name": "Alice", "age": 25}
json_string = json.dumps(data)
print(json_string)  # Muestra: {"name": "Alice", "age": 25}

parsed_data = json.loads(json_string)
print(parsed_data)  # Muestra: {'name': 'Alice', 'age': 25}

Conclusión

La Biblioteca Estándar de Python es una herramienta poderosa que proporciona una amplia gama de funcionalidades, desde operaciones básicas hasta manipulaciones complejas de datos. Familiarizarte con estos módulos mejorará enormemente tu capacidad para escribir código Python eficiente y efectivo. A medida que avances de niveles principiantes a avanzados, te encontrarás confiando más en estos módulos incorporados para resolver problemas y construir aplicaciones.
