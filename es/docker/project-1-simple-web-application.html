Proyecto 2: Aplicación Multi-Contenedor ►

En este proyecto, crearemos una aplicación web simple y la contenedorizaremos usando Docker. Este proyecto te ayudará a entender los conceptos básicos de Docker, incluyendo la creación de Dockerfiles, la construcción de imágenes y la ejecución de contenedores.

Objetivos

Al final de este proyecto, tú:

  • Entenderás cómo crear un Dockerfile.
  • Aprenderás cómo construir una imagen Docker.
  • Ejecutarás un contenedor Docker.
  • Expondrás una aplicación web contenedorizada al mundo exterior.

Prerrequisitos

Antes de comenzar este proyecto, asegúrate de tener:

  • Conocimientos básicos de desarrollo web (HTML, CSS, JavaScript).
  • Comprensión básica de los conceptos de Docker.
  • Docker instalado en tu máquina.

Paso 1: Crear una Aplicación Web Simple

Primero, vamos a crear una aplicación web simple usando Node.js.

  1. Inicializa un proyecto de Node.js:

    mkdir simple-web-app
cd simple-web-app
npm init -y

  2. Instala Express.js:

    npm install express

  3. Crea app.js:

    const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
    res.send('¡Hola, Mundo!');
});

app.listen(port, () => {
    console.log(`App running at http://localhost:${port}`);
});

  4. Actualiza package.json para incluir un script de inicio:

    "scripts": {
    "start": "node app.js"
}

Paso 2: Crear un Dockerfile

Un Dockerfile es un documento de texto que contiene todos los comandos para ensamblar una imagen.

  1. Crea un Dockerfile en la raíz de tu proyecto:

    # Usa la imagen oficial de Node.js como imagen base
FROM node:14

# Establece el directorio de trabajo
WORKDIR /usr/src/app

# Copia package.json y package-lock.json
COPY package*.json ./

# Instala las dependencias
RUN npm install

# Copia el resto del código de la aplicación
COPY . .

# Expone el puerto en el que se ejecuta la aplicación
EXPOSE 3000

# Comando para ejecutar la aplicación
CMD ["npm", "start"]

Paso 3: Construir la Imagen Docker

Para construir la imagen Docker, usa el comando docker build.

  1. Construye la imagen:

    docker build -t simple-web-app .
    • -t simple-web-app: Etiqueta la imagen con el nombre simple-web-app.
    • .: Especifica el directorio actual como el contexto de construcción.

Paso 4: Ejecutar el Contenedor Docker

Una vez que la imagen está construida, puedes ejecutarla usando el comando docker run.

  1. Ejecuta el contenedor:

    docker run -p 3000:3000 simple-web-app
    • -p 3000:3000: Mapea el puerto 3000 en tu host al puerto 3000 en el contenedor.

  2. Accede a la aplicación:

    Abre tu navegador y navega a http://localhost:3000. Deberías ver "¡Hola, Mundo!".

Conclusión

En este proyecto, aprendiste a:

  • Crear una aplicación web simple usando Node.js y Express.
  • Escribir un Dockerfile para contenedizar la aplicación.
  • Construir una imagen Docker a partir del Dockerfile.
  • Ejecutar un contenedor Docker a partir de la imagen y exponerlo al mundo exterior.

Este conocimiento fundamental servirá como un trampolín para temas más avanzados de Docker, como construcciones multi-etapa, Docker Compose y la orquestación de contenedores con Kubernetes.
