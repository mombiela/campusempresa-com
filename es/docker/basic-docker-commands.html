Introducción

Docker es una plataforma poderosa para desarrollar, enviar y ejecutar aplicaciones dentro de contenedores. Entender los comandos básicos de Docker es esencial para cualquiera que esté comenzando con Docker. Esta sección cubrirá los comandos fundamentales que necesitas para empezar con Docker.

Conceptos Clave

  • Imagen de Docker: Un paquete de software ligero, autónomo y ejecutable que incluye todo lo necesario para ejecutar una pieza de software.
  • Contenedor de Docker: Una instancia en tiempo de ejecución de una imagen de Docker.
  • Docker Hub: Un repositorio basado en la nube donde los usuarios de Docker pueden crear, probar, almacenar y distribuir imágenes de contenedores.

Instalación de Docker

Antes de sumergirte en los comandos de Docker, asegúrate de que Docker esté instalado en tu sistema. Puedes descargar e instalar Docker desde el sitio web oficial de Docker.

Comandos Básicos de Docker

1. `docker --version`

  • Propósito: Verificar la versión instalada de Docker.
  • Uso: 
    docker --version
  • Ejemplo: 
    $ docker --version
Docker version 20.10.7, build f0df350

2. `docker pull`

  • Propósito: Descargar una imagen de Docker desde Docker Hub.
  • Uso: 
    docker pull <image_name>
  • Ejemplo: 
    $ docker pull nginx
Using default tag: latest
latest: Pulling from library/nginx
...

3. `docker images`

  • Propósito: Listar todas las imágenes de Docker en tu máquina local.
  • Uso: 
    docker images
  • Ejemplo: 
    $ docker images
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
nginx               latest              4bb46517cac3        2 weeks ago         133MB

4. `docker run`

  • Propósito: Crear y arrancar un nuevo contenedor desde una imagen.
  • Uso: 
    docker run <image_name>
  • Ejemplo: 
    $ docker run nginx

5. `docker ps`

  • Propósito: Listar todos los contenedores en ejecución.
  • Uso: 
    docker ps
  • Ejemplo: 
    $ docker ps
CONTAINER ID        IMAGE               COMMAND                  CREATED             STATUS              PORTS               NAMES
1a2b3c4d5e6f        nginx               "nginx -g 'daemon of…"   5 seconds ago       Up 4 seconds        80/tcp              loving_turing

6. `docker stop`

  • Propósito: Detener un contenedor en ejecución.
  • Uso: 
    docker stop <container_id>
  • Ejemplo: 
    $ docker stop 1a2b3c4d5e6f

7. `docker rm`

  • Propósito: Eliminar un contenedor detenido.
  • Uso: 
    docker rm <container_id>
  • Ejemplo: 
    $ docker rm 1a2b3c4d5e6f

8. `docker rmi`

  • Propósito: Eliminar una imagen de Docker.
  • Uso: 
    docker rmi <image_id>
  • Ejemplo: 
    $ docker rmi 4bb46517cac3

Resumen

En esta sección, cubrimos los comandos básicos de Docker esenciales para principiantes. Estos comandos incluyen verificar la versión de Docker, descargar imágenes, ejecutar contenedores, listar imágenes y contenedores, detener y eliminar contenedores, y eliminar imágenes. Dominar estos comandos proporcionará una base sólida para trabajar con Docker. A medida que avances, aprenderás comandos y conceptos más avanzados para gestionar entornos Docker de manera efectiva.
