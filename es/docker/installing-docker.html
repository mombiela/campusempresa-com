Introducción

Docker es una plataforma que permite a los desarrolladores automatizar el despliegue de aplicaciones dentro de contenedores ligeros y portátiles. Esta sección te guiará a través del proceso de instalación de Docker en varios sistemas operativos.

Requisitos previos

Antes de instalar Docker, asegúrate de que tu sistema cumpla con los siguientes requisitos:

  • Un sistema operativo de 64 bits.
  • Al menos 4GB de RAM.
  • Privilegios administrativos en tu máquina.

Instalación de Docker en Windows

Guía paso a paso

  1. Descargar Docker Desktop para Windows:

    • Visita el Docker Hub y descarga el instalador de Docker Desktop.

  2. Ejecutar el instalador:

    • Haz doble clic en el archivo .exe descargado para ejecutar el instalador.
    • Sigue las instrucciones en pantalla para completar la instalación.

  3. Iniciar Docker Desktop:

    • Una vez instalado, Docker Desktop se iniciará automáticamente.
    • Es posible que necesites cerrar sesión y volver a iniciarla para completar la instalación.

  4. Verificar la instalación:

    • Abre una ventana de Command Prompt o PowerShell.
    • Ejecuta el siguiente comando para verificar que Docker está instalado correctamente: 
      docker --version
    • Deberías ver una salida similar a Docker version 20.10.7, build f0df350.

Instalación de Docker en macOS

Guía paso a paso

  1. Descargar Docker Desktop para Mac:

    • Visita el Docker Hub y descarga el instalador de Docker Desktop.

  2. Ejecutar el instalador:

    • Haz doble clic en el archivo .dmg descargado para abrir el instalador.
    • Arrastra el icono de Docker a la carpeta de Aplicaciones.

  3. Iniciar Docker Desktop:

    • Abre Docker desde la carpeta de Aplicaciones.
    • Docker Desktop se iniciará y es posible que necesites autorizar Docker con tu contraseña del sistema.

  4. Verificar la instalación:

    • Abre una ventana de Terminal.
    • Ejecuta el siguiente comando para verificar que Docker está instalado correctamente: 
      docker --version
    • Deberías ver una salida similar a Docker version 20.10.7, build f0df350.

Instalación de Docker en Linux

Guía paso a paso

  1. Actualizar el índice de paquetes:

    • Abre una terminal y ejecuta: 
      sudo apt-get update

  2. Instalar paquetes requeridos:

    • Ejecuta el siguiente comando para instalar los paquetes necesarios: 
      sudo apt-get install apt-transport-https ca-certificates curl software-properties-common

  3. Agregar la clave GPG oficial de Docker:

    • Ejecuta el siguiente comando: 
      curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

  4. Configurar el repositorio estable:

    • Agrega el repositorio de Docker a las fuentes de APT: 
      sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

  5. Instalar Docker CE:

    • Actualiza nuevamente el índice de paquetes: 
      sudo apt-get update
    • Instala Docker: 
      sudo apt-get install docker-ce

  6. Verificar la instalación:

    • Ejecuta el siguiente comando para verificar que Docker está instalado correctamente: 
      docker --version
    • Deberías ver una salida similar a Docker version 20.10.7, build f0df350.

Pasos posteriores a la instalación

Gestionar Docker como un usuario sin privilegios de root

Por defecto, los comandos de Docker solo pueden ser ejecutados por el usuario root o por un usuario en el grupo docker. Para evitar usar sudo con cada comando de Docker, agrega tu usuario al grupo docker:

  1. Crear el grupo Docker:

    sudo groupadd docker

  2. Agregar tu usuario al grupo Docker:

    sudo usermod -aG docker $USER

  3. Aplicar la nueva membresía del grupo:

    • Cierra sesión y vuelve a iniciarla para que se reevalúe tu membresía de grupo.
    • Alternativamente, puedes ejecutar: 
      newgrp docker

  4. Verificar la membresía del grupo:

    • Ejecuta un comando de Docker sin sudo para verificar: 
      docker run hello-world

Conclusión

Instalar Docker es un proceso sencillo que varía ligeramente dependiendo de tu sistema operativo. Siguiendo los pasos descritos en esta sección, ahora deberías tener Docker instalado y funcionando en tu máquina. Esta configuración te permitirá comenzar a construir y desplegar aplicaciones en contenedores de manera eficiente.
