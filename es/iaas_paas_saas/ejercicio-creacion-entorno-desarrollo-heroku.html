Introducción

En este ejercicio, aprenderás a crear un entorno de desarrollo en Heroku, una plataforma como servicio (PaaS) que permite a los desarrolladores construir, ejecutar y operar aplicaciones completamente en la nube. Heroku simplifica el proceso de despliegue y gestión de aplicaciones, permitiendo a los desarrolladores centrarse en la escritura de código.

Objetivos del Ejercicio

  • Configurar una cuenta en Heroku.
  • Instalar y configurar la CLI de Heroku.
  • Crear y desplegar una aplicación básica en Heroku.
  • Gestionar el entorno de desarrollo en Heroku.

Requisitos Previos

Antes de comenzar, asegúrate de tener:

  • Una cuenta en Heroku (puedes registrarte en Heroku).
  • Git instalado en tu máquina.
  • Node.js y npm instalados (para el ejemplo de aplicación).

Pasos para Crear un Entorno de Desarrollo en Heroku

1. Configuración de la Cuenta en Heroku

  1. Registro en Heroku:

    • Visita Heroku.
    • Haz clic en "Sign Up" y sigue las instrucciones para crear una cuenta.

  2. Verificación de la Cuenta:

    • Verifica tu cuenta a través del correo electrónico enviado por Heroku.

2. Instalación de la CLI de Heroku

  1. Descarga e Instalación:

    • Descarga la CLI de Heroku desde Heroku CLI.
    • Sigue las instrucciones de instalación para tu sistema operativo.

  2. Verificación de la Instalación:

    • Abre una terminal y ejecuta el siguiente comando para verificar la instalación: 
      heroku --version
    • Deberías ver la versión de la CLI de Heroku instalada.

3. Creación y Despliegue de una Aplicación Básica

  1. Inicio de Sesión en Heroku:

    • En la terminal, ejecuta: 
      heroku login
    • Sigue las instrucciones para iniciar sesión en tu cuenta de Heroku.

  2. Creación de una Aplicación Node.js:

    • Crea un nuevo directorio para tu aplicación y navega a él: 
      mkdir my-heroku-app
cd my-heroku-app
    • Inicializa un nuevo proyecto Node.js: 
      npm init -y
    • Instala Express.js: 
      npm install express
    • Crea un archivo index.js con el siguiente contenido: 
      const express = require('express');
const app = express();
const PORT = process.env.PORT || 3000;

app.get('/', (req, res) => {
  res.send('Hello, Heroku!');
});

app.listen(PORT, () => {
  console.log(`Server is running on port ${PORT}`);
});

  3. Preparación para el Despliegue:

    • Crea un archivo Procfile en el directorio raíz con el siguiente contenido: 
      web: node index.js
    • Inicializa un repositorio Git y realiza un commit: 
      git init
git add .
git commit -m "Initial commit"

  4. Despliegue en Heroku:

    • Crea una nueva aplicación en Heroku: 
      heroku create
    • Despliega la aplicación: 
      git push heroku master

  5. Verificación del Despliegue:

    • Una vez completado el despliegue, abre la aplicación en tu navegador: 
      heroku open

4. Gestión del Entorno de Desarrollo

  1. Ver Logs de la Aplicación:

    • Para ver los logs de tu aplicación, ejecuta: 
      heroku logs --tail

  2. Escalado de Dynos:

    • Para escalar el número de dynos (instancias de tu aplicación), usa: 
      heroku ps:scale web=1

  3. Configuración de Variables de Entorno:

    • Para configurar variables de entorno, usa: 
      heroku config:set MY_VAR=value

Conclusión

En este ejercicio, has aprendido a configurar un entorno de desarrollo en Heroku, desde la creación de una cuenta hasta el despliegue y gestión de una aplicación básica. Heroku facilita el proceso de despliegue y gestión de aplicaciones, permitiendo a los desarrolladores centrarse en la escritura de código y en la mejora de sus aplicaciones.
