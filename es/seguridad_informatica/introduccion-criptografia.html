¿Qué es la Criptografía?

La criptografía es la ciencia y el arte de crear y analizar protocolos que previenen que terceros o el público en general lean mensajes privados. La criptografía moderna se basa en teorías matemáticas y en la práctica de la computación.

Objetivos de la Criptografía

  • Confidencialidad: Garantizar que la información solo sea accesible para las partes autorizadas.
  • Integridad: Asegurar que la información no ha sido alterada de manera no autorizada.
  • Autenticación: Verificar la identidad de las partes involucradas en la comunicación.
  • No repudio: Prevenir que una parte niegue haber realizado una acción.

Tipos de Criptografía

Criptografía Simétrica

En la criptografía simétrica, el mismo clave se utiliza tanto para cifrar como para descifrar la información.

  • Ventajas:
    • Rápida y eficiente.
    • Menos recursos computacionales.
  • Desventajas:
    • Distribución segura de claves.
    • No escalable para grandes sistemas.

Ejemplo de Criptografía Simétrica: Cifrado AES

from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generar una clave aleatoria
key = get_random_bytes(16)

# Crear un objeto de cifrado AES
cipher = AES.new(key, AES.MODE_EAX)

# Mensaje a cifrar
data = b"Mensaje secreto"

# Cifrar el mensaje
ciphertext, tag = cipher.encrypt_and_digest(data)

print("Texto cifrado:", ciphertext)

Criptografía Asimétrica

En la criptografía asimétrica, se utilizan dos claves diferentes: una clave pública para cifrar y una clave privada para descifrar.

  • Ventajas:
    • No requiere distribución segura de claves.
    • Escalable para grandes sistemas.
  • Desventajas:
    • Más lenta y requiere más recursos computacionales.
    • Complejidad en la gestión de claves.

Ejemplo de Criptografía Asimétrica: Cifrado RSA

from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# Generar un par de claves RSA
key = RSA.generate(2048)
public_key = key.publickey()

# Crear un objeto de cifrado RSA con la clave pública
cipher = PKCS1_OAEP.new(public_key)

# Mensaje a cifrar
data = b"Mensaje secreto"

# Cifrar el mensaje
ciphertext = cipher.encrypt(data)

print("Texto cifrado:", ciphertext)

Algoritmos Criptográficos Comunes

| Algoritmo | Tipo | Uso Principal | |-----------|------|---------------| | AES | Simétrico | Cifrado de datos | | DES | Simétrico | Cifrado de datos (obsoleto) | | RSA | Asimétrico | Cifrado y firma digital | | ECC | Asimétrico | Cifrado y firma digital | | SHA-256 | Hash | Integridad de datos |

Aplicaciones de la Criptografía

  • Seguridad en la comunicación: Proteger la información transmitida a través de redes.
  • Firmas digitales: Verificar la autenticidad y la integridad de los mensajes.
  • Autenticación: Confirmar la identidad de usuarios y dispositivos.
  • Almacenamiento seguro: Proteger datos almacenados contra accesos no autorizados.

Conclusión

La criptografía es una herramienta esencial en la seguridad informática que permite proteger la confidencialidad, integridad, autenticación y no repudio de la información. Comprender los diferentes tipos de criptografía y sus aplicaciones es fundamental para cualquier profesional de TI que desee asegurar sistemas y datos de manera efectiva.
