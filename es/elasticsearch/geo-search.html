La Búsqueda Geográfica en Elasticsearch te permite realizar consultas basadas en datos geográficos. Esto es particularmente útil para aplicaciones que necesitan filtrar o ordenar datos por ubicación, como encontrar restaurantes cercanos, tiendas u otros servicios basados en la ubicación.

Conceptos Clave

  • Campo Geo-Point: Un tipo de campo en Elasticsearch utilizado para almacenar puntos geográficos (latitud y longitud).
  • Campo Geo-Shape: Un tipo de campo utilizado para almacenar formas complejas como polígonos, líneas y círculos.
  • Consultas Geográficas: Consultas que te permiten filtrar u ordenar datos basados en criterios geográficos.

Configuración de Campos Geo-Point

Para realizar una búsqueda geográfica, necesitas definir un campo geo-point en el mapeo de tu índice.

PUT /my_index
{
  "mappings": {
    "properties": {
      "location": {
        "type": "geo_point"
      }
    }
  }
}

Indexación de Documentos con Geo-Points

Al indexar documentos, puedes incluir datos geo-point en forma de latitud y longitud.

POST /my_index/_doc/1
{
  "name": "Central Park",
  "location": {
    "lat": 40.785091,
    "lon": -73.968285
  }
}

Consultas Geográficas Básicas

Consulta de Distancia Geográfica

Encuentra documentos dentro de una cierta distancia desde un punto específico.

GET /my_index/_search
{
  "query": {
    "geo_distance": {
      "distance": "10km",
      "location": {
        "lat": 40.785091,
        "lon": -73.968285
      }
    }
  }
}

Consulta de Caja de Delimitación Geográfica

Encuentra documentos dentro de un área rectangular especificada.

GET /my_index/_search
{
  "query": {
    "geo_bounding_box": {
      "location": {
        "top_left": {
          "lat": 40.915255,
          "lon": -74.25909
        },
        "bottom_right": {
          "lat": 40.496044,
          "lon": -73.700272
        }
      }
    }
  }
}

Consulta de Polígono Geográfico

Encuentra documentos dentro de un polígono especificado.

GET /my_index/_search
{
  "query": {
    "geo_polygon": {
      "location": {
        "points": [
          {"lat": 40.915255, "lon": -74.25909},
          {"lat": 40.496044, "lon": -74.25909},
          {"lat": 40.496044, "lon": -73.700272},
          {"lat": 40.915255, "lon": -73.700272}
        ]
      }
    }
  }
}

Técnicas Avanzadas de Búsqueda Geográfica

Ordenar por Distancia

Ordena los resultados de búsqueda por su distancia desde un punto específico.

GET /my_index/_search
{
  "sort": [
    {
      "_geo_distance": {
        "location": {
          "lat": 40.785091,
          "lon": -73.968285
        },
        "order": "asc",
        "unit": "km"
      }
    }
  ],
  "query": {
    "match_all": {}
  }
}

Consultas de Geo-Shape

Para consultas geográficas más complejas, puedes usar campos y consultas geo-shape.

Definición de un Campo Geo-Shape

PUT /my_index
{
  "mappings": {
    "properties": {
      "location": {
        "type": "geo_shape"
      }
    }
  }
}

Indexación de Documentos con Geo-Shapes

POST /my_index/_doc/1
{
  "name": "Central Park",
  "location": {
    "type": "polygon",
    "coordinates": [
      [
        [-73.981, 40.768],
        [-73.958, 40.768],
        [-73.958, 40.785],
        [-73.981, 40.785],
        [-73.981, 40.768]
      ]
    ]
  }
}

Consulta de Geo-Shape

Encuentra documentos que intersecten con una forma dada.

GET /my_index/_search
{
  "query": {
    "geo_shape": {
      "location": {
        "shape": {
          "type": "polygon",
          "coordinates": [
            [
              [-73.981, 40.768],
              [-73.958, 40.768],
              [-73.958, 40.785],
              [-73.981, 40.785],
              [-73.981, 40.768]
            ]
          ]
        },
        "relation": "intersects"
      }
    }
  }
}

Conclusión

La Búsqueda Geográfica en Elasticsearch es una característica poderosa que te permite realizar consultas geográficas complejas. Al entender cómo definir campos geo-point y geo-shape, y cómo usar varias consultas geográficas, puedes construir funcionalidades robustas de búsqueda basada en la ubicación en tus aplicaciones. Ya sea que estés filtrando resultados dentro de una cierta distancia, caja de delimitación o polígono, Elasticsearch proporciona las herramientas que necesitas para manejar datos geográficos de manera eficiente.
