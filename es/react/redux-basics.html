Introducción a Redux

Redux es un contenedor de estado predecible para aplicaciones JavaScript, a menudo utilizado con React. Ayuda a gestionar el estado de tu aplicación de una manera más estructurada y predecible.

Conceptos Clave

  • Estado: La única fuente de verdad para tu aplicación.
  • Acciones: Objetos simples de JavaScript que describen lo que ocurrió.
  • Reducers: Funciones que determinan cómo cambia el estado en respuesta a las acciones.
  • Store: El objeto que reúne acciones y reducers, manteniendo el estado de la aplicación.

Configuración de Redux

Para comenzar a usar Redux en una aplicación React, necesitas instalar los paquetes necesarios.

npm install redux react-redux

Creación de una Store de Redux

La store es la pieza central de una aplicación Redux. Mantiene el estado y permite que se actualice a través de acciones.

import { createStore } from 'redux';

// Un reducer simple
const initialState = { count: 0 };
const counterReducer = (state = initialState, action) => {
  switch (action.type) {
    case 'INCREMENT':
      return { count: state.count + 1 };
    case 'DECREMENT':
      return { count: state.count - 1 };
    default:
      return state;
  }
};

// Crear la store
const store = createStore(counterReducer);

Conectando Redux a React

Para conectar Redux con React, utilizas el componente Provider de react-redux.

import React from 'react';
import ReactDOM from 'react-dom';
import { Provider } from 'react-redux';
import App from './App'; // Tu componente principal App

ReactDOM.render(
  <Provider store={store}>
    <App />
  </Provider>,
  document.getElementById('root')
);

Despachando Acciones

Las acciones se despachan a la store para actualizar el estado.

// Creadores de acciones
const increment = () => ({ type: 'INCREMENT' });
const decrement = () => ({ type: 'DECREMENT' });

// Despachando acciones
store.dispatch(increment());
store.dispatch(decrement());

Usando `connect` para Acceder al Estado y Despachar en Componentes

La función connect de react-redux se utiliza para conectar componentes React a la store de Redux.

import React from 'react';
import { connect } from 'react-redux';

// Un componente simple
const Counter = ({ count, increment, decrement }) => (
  <div>
    <h1>{count}</h1>
    <button onClick={increment}>Incrementar</button>
    <button onClick={decrement}>Decrementar</button>
  </div>
);

// Mapear estado y despacho a props
const mapStateToProps = state => ({ count: state.count });
const mapDispatchToProps = dispatch => ({
  increment: () => dispatch(increment()),
  decrement: () => dispatch(decrement())
});

// Conectar el componente a la store de Redux
export default connect(mapStateToProps, mapDispatchToProps)(Counter);

Resumen

Redux ayuda a gestionar el estado de tu aplicación de una manera predecible. Al entender los conceptos básicos—estado, acciones, reducers y store—puedes usar Redux efectivamente en tus aplicaciones React. Configurar Redux implica crear una store, conectarla a tu aplicación React usando el Provider, despachar acciones para actualizar el estado, y usar connect para acceder al estado y despachar acciones en tus componentes. Con estos conceptos básicos, puedes comenzar a construir aplicaciones más complejas y escalables.
