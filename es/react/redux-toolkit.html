Redux Toolkit es la forma oficial y recomendada de escribir lógica Redux. Proporciona un conjunto de herramientas y mejores prácticas que simplifican el proceso de escribir aplicaciones Redux. Esta sección te guiará desde los conceptos básicos hasta los avanzados de Redux Toolkit en el contexto de una aplicación React.

Introducción a Redux Toolkit

Redux Toolkit está diseñado para ayudarte a escribir lógica Redux de manera más fácil y eficiente. Incluye utilidades para crear y gestionar slices de estado, escribir reducers y manejar lógica asincrónica.

Conceptos Clave

  • Slices: Un slice es una colección de lógica de reducer de Redux y acciones para una característica específica de tu aplicación.
  • Reducers: Funciones que determinan cambios en el estado de una aplicación.
  • Actions: Objetos simples que representan una intención de cambiar el estado.
  • Store: El objeto que une acciones y reducers.

Configuración de Redux Toolkit

Instalación

Para comenzar con Redux Toolkit, necesitas instalar los paquetes necesarios:

npm install @reduxjs/toolkit react-redux

Creación de una Tienda Redux

La función configureStore de Redux Toolkit simplifica el proceso de configurar la tienda.

import { configureStore } from '@reduxjs/toolkit';

const store = configureStore({
  reducer: {
    // Añade tus reducers aquí
  },
});

export default store;

Creación de Slices

Un slice contiene la lógica del reducer y las acciones para una parte específica del estado.

Ejemplo: Counter Slice

import { createSlice } from '@reduxjs/toolkit';

const counterSlice = createSlice({
  name: 'counter',
  initialState: 0,
  reducers: {
    increment: state => state + 1,
    decrement: state => state - 1,
    incrementByAmount: (state, action) => state + action.payload,
  },
});

export const { increment, decrement, incrementByAmount } = counterSlice.actions;
export default counterSlice.reducer;

Usando el Slice en la Tienda

import { configureStore } from '@reduxjs/toolkit';
import counterReducer from './counterSlice';

const store = configureStore({
  reducer: {
    counter: counterReducer,
  },
});

export default store;

Conectando Redux a React

Configuración del Provider

Para usar Redux en una aplicación React, envuelve tu aplicación con el componente Provider de react-redux.

import React from 'react';
import ReactDOM from 'react-dom';
import { Provider } from 'react-redux';
import store from './store';
import App from './App';

ReactDOM.render(
  <Provider store={store}>
    <App />
  </Provider>,
  document.getElementById('root')
);

Usando el Estado y las Acciones de Redux en Componentes

Usa los hooks useSelector y useDispatch de react-redux para interactuar con la tienda Redux en tus componentes.

import React from 'react';
import { useSelector, useDispatch } from 'react-redux';
import { increment, decrement, incrementByAmount } from './counterSlice';

const Counter = () => {
  const count = useSelector(state => state.counter);
  const dispatch = useDispatch();

  return (
    <div>
      <div>{count}</div>
      <button onClick={() => dispatch(increment())}>Incrementar</button>
      <button onClick={() => dispatch(decrement())}>Decrementar</button>
      <button onClick={() => dispatch(incrementByAmount(5))}>Incrementar por 5</button>
    </div>
  );
};

export default Counter;

Características Avanzadas de Redux Toolkit

Lógica Asincrónica con createAsyncThunk

createAsyncThunk simplifica el proceso de manejar acciones asincrónicas.

import { createAsyncThunk, createSlice } from '@reduxjs/toolkit';
import axios from 'axios';

export const fetchUser = createAsyncThunk('user/fetchUser', async (userId) => {
  const response = await axios.get(`/api/user/${userId}`);
  return response.data;
});

const userSlice = createSlice({
  name: 'user',
  initialState: { user: null, status: 'idle' },
  reducers: {},
  extraReducers: (builder) => {
    builder
      .addCase(fetchUser.pending, (state) => {
        state.status = 'loading';
      })
      .addCase(fetchUser.fulfilled, (state, action) => {
        state.status = 'succeeded';
        state.user = action.payload;
      })
      .addCase(fetchUser.rejected, (state) => {
        state.status = 'failed';
      });
  },
});

export default userSlice.reducer;

Middleware

El middleware puede ser añadido a la tienda Redux para manejar efectos secundarios y otra lógica personalizada.

import { configureStore, getDefaultMiddleware } from '@reduxjs/toolkit';
import logger from 'redux-logger';
import counterReducer from './counterSlice';

const store = configureStore({
  reducer: {
    counter: counterReducer,
  },
  middleware: (getDefaultMiddleware) => getDefaultMiddleware().concat(logger),
});

export default store;

Conclusión

Redux Toolkit simplifica el proceso de escribir lógica Redux proporcionando un conjunto de herramientas y mejores prácticas. Usando slices, configureStore y createAsyncThunk, puedes gestionar el estado y manejar acciones asincrónicas de manera más eficiente. Integrar Redux con React usando los hooks useSelector y useDispatch permite una experiencia de gestión de estado sin problemas. A medida que avances, puedes aprovechar el middleware y otras características avanzadas para construir aplicaciones robustas.
