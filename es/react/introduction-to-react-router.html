React Router es una biblioteca poderosa que permite el enrutamiento dinámico en una aplicación React. Te permite crear aplicaciones de una sola página con navegación sin que la página se recargue mientras el usuario navega. Este tema cubrirá los conceptos básicos de React Router, incluyendo su configuración, definición de rutas y el uso de varios componentes proporcionados por la biblioteca.

Configuración de React Router

Para comenzar con React Router, necesitas instalar la biblioteca en tu proyecto React. Puedes hacerlo usando npm o yarn.

# Usando npm
npm install react-router-dom

# Usando yarn
yarn add react-router-dom

Una vez instalada, necesitas envolver tu aplicación con el componente BrowserRouter para habilitar el enrutamiento.

import React from 'react';
import ReactDOM from 'react-dom';
import { BrowserRouter } from 'react-router-dom';
import App from './App';

ReactDOM.render(
  <BrowserRouter>
    <App />
  </BrowserRouter>,
  document.getElementById('root')
);

Definiendo Rutas

Las rutas se definen usando el componente Route. Cada componente Route especifica una ruta y el componente que debe renderizarse cuando la ruta coincide con la URL.

import React from 'react';
import { Route, Switch } from 'react-router-dom';
import Home from './Home';
import About from './About';
import Contact from './Contact';

function App() {
  return (
    <Switch>
      <Route exact path="/" component={Home} />
      <Route path="/about" component={About} />
      <Route path="/contact" component={Contact} />
    </Switch>
  );
}

export default App;

Conceptos Clave

  • BrowserRouter: Una implementación de enrutador que utiliza la API de historial HTML5 para mantener tu UI sincronizada con la URL.
  • Route: Renderiza un componente de UI cuando su ruta coincide con la URL actual.
  • Switch: Renderiza el primer Route hijo que coincide con la URL actual. Esto es útil para asegurar que solo una ruta se renderice a la vez.

Navegación

Para navegar entre rutas, puedes usar el componente Link. Este componente es similar a una etiqueta de anclaje HTML pero evita que la página se recargue.

import React from 'react';
import { Link } from 'react-router-dom';

function Navigation() {
  return (
    <nav>
      <ul>
        <li><Link to="/">Home</Link></li>
        <li><Link to="/about">About</Link></li>
        <li><Link to="/contact">Contact</Link></li>
      </ul>
    </nav>
  );
}

export default Navigation;

Rutas Anidadas

React Router te permite definir rutas anidadas, lo cual puede ser útil para crear diseños complejos.

import React from 'react';
import { Route, Switch, Link, useRouteMatch } from 'react-router-dom';

function Topics() {
  let { path, url } = useRouteMatch();

  return (
    <div>
      <h2>Topics</h2>
      <ul>
        <li><Link to={`${url}/components`}>Components</Link></li>
        <li><Link to={`${url}/props-v-state`}>Props v. State</Link></li>
      </ul>

      <Switch>
        <Route exact path={path}>
          <h3>Please select a topic.</h3>
        </Route>
        <Route path={`${path}/:topicId`}>
          <Topic />
        </Route>
      </Switch>
    </div>
  );
}

function Topic() {
  let { topicId } = useParams();
  return <h3>Requested topic ID: {topicId}</h3>;
}

Redirecciones y Navegación Programática

A veces necesitas redirigir a los usuarios o navegar programáticamente. React Router proporciona el componente Redirect y el hook useHistory para estos propósitos.

Componente Redirect

import React from 'react';
import { Redirect } from 'react-router-dom';

function OldPage() {
  return <Redirect to="/new-page" />;
}

Hook useHistory

import React from 'react';
import { useHistory } from 'react-router-dom';

function HomePage() {
  let history = useHistory();

  function handleClick() {
    history.push('/about');
  }

  return (
    <div>
      <button onClick={handleClick}>Go to About Page</button>
    </div>
  );
}

Resumen

React Router es una herramienta esencial para construir aplicaciones de una sola página con React. Te permite:

  • Definir rutas usando el componente Route.
  • Navegar entre rutas usando el componente Link.
  • Manejar rutas anidadas para diseños complejos.
  • Redirigir y navegar programáticamente usando el componente Redirect y el hook useHistory.

Al dominar React Router, puedes crear aplicaciones web dinámicas y receptivas que proporcionan una experiencia de usuario fluida.
