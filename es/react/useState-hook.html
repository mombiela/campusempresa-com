Introducción al Hook useState

El hook useState es uno de los hooks fundamentales en React que te permite agregar estado a los componentes funcionales. Antes de los hooks, la gestión del estado solo era posible en componentes de clase. Con useState, ahora puedes gestionar el estado en componentes funcionales, haciéndolos más poderosos y versátiles.

Uso Básico de useState

Para usar el hook useState, necesitas importarlo de la biblioteca de React. El hook useState devuelve un array con dos elementos: el estado actual y una función para actualizar ese estado.

Ejemplo

import React, { useState } from 'react';

function Counter() {
  // Declarar una variable de estado llamada 'count' y una función para actualizarla
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}

export default Counter;

Explicación

  • useState(0): Inicializa la variable de estado count con un valor de 0.
  • const [count, setCount]: Desestructura el array devuelto por useState en count (la variable de estado) y setCount (la función para actualizar count).
  • setCount(count + 1): Actualiza la variable de estado count incrementando su valor en 1.

Gestión de Múltiples Variables de Estado

Puedes usar múltiples hooks useState para gestionar diferentes variables de estado dentro del mismo componente.

Ejemplo

import React, { useState } from 'react';

function UserProfile() {
  const [name, setName] = useState('');
  const [age, setAge] = useState(0);

  return (
    <div>
      <input
        type="text"
        placeholder="Enter your name"
        value={name}
        onChange={(e) => setName(e.target.value)}
      />
      <input
        type="number"
        placeholder="Enter your age"
        value={age}
        onChange={(e) => setAge(Number(e.target.value))}
      />
      <p>Name: {name}</p>
      <p>Age: {age}</p>
    </div>
  );
}

export default UserProfile;

Explicación

  • const [name, setName] = useState(''): Inicializa la variable de estado name con una cadena vacía.
  • const [age, setAge] = useState(0): Inicializa la variable de estado age con un valor de 0.
  • setName(e.target.value): Actualiza la variable de estado name con el valor del campo de entrada.
  • setAge(Number(e.target.value)): Actualiza la variable de estado age con el valor numérico del campo de entrada.

Actualizaciones Funcionales

Cuando actualizas el estado basado en el estado anterior, se recomienda usar una actualización funcional para asegurarte de tener el valor de estado más reciente.

Ejemplo

import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(prevCount => prevCount + 1)}>
        Click me
      </button>
    </div>
  );
}

export default Counter;

Explicación

  • setCount(prevCount => prevCount + 1): Usa una actualización funcional para incrementar la variable de estado count en 1, asegurando que la actualización se base en el valor de estado más reciente.

Inicialización Perezosa

Puedes inicializar el estado de manera perezosa pasando una función a useState. Esta función solo se ejecutará durante el renderizado inicial.

Ejemplo

import React, { useState } from 'react';

function ExpensiveComponent() {
  const [value, setValue] = useState(() => {
    // Simular una computación costosa
    const initialValue = computeExpensiveValue();
    return initialValue;
  });

  return (
    <div>
      <p>Value: {value}</p>
      <button onClick={() => setValue(value + 1)}>
        Increment
      </button>
    </div>
  );
}

function computeExpensiveValue() {
  // Simular una computación costosa
  return 42;
}

export default ExpensiveComponent;

Explicación

  • useState(() => computeExpensiveValue()): Inicializa la variable de estado value llamando a computeExpensiveValue solo durante el renderizado inicial, evitando computaciones innecesarias en renderizados posteriores.

Conclusión

El hook useState es una herramienta poderosa para gestionar el estado en componentes funcionales. Proporciona una API simple e intuitiva para inicializar, actualizar y gestionar el estado. Al entender su uso básico, la gestión de múltiples variables de estado, las actualizaciones funcionales y la inicialización perezosa, puedes gestionar eficazmente el estado en tus aplicaciones React.
