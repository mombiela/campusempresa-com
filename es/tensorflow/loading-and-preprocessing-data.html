En esta sección, cubriremos cómo cargar y preprocesar datos usando TensorFlow. Este es un paso crucial en cualquier pipeline de aprendizaje automático, ya que la calidad y el formato de tus datos pueden afectar significativamente el rendimiento de tus modelos.

Conceptos Clave

  • Carga de Datos: Importar datos de varias fuentes como archivos CSV, imágenes o bases de datos.
  • Preprocesamiento de Datos: Transformar datos en bruto en un formato adecuado para entrenar modelos de aprendizaje automático. Esto incluye normalización, aumento y división de datos en conjuntos de entrenamiento, validación y prueba.

Carga de Datos

Carga de Datos desde Archivos CSV

TensorFlow proporciona varias utilidades para cargar datos desde archivos CSV. La API tf.data es particularmente útil para este propósito.

import tensorflow as tf

# Definir la ruta del archivo
file_path = 'ruta/a/tus/datos.csv'

# Crear un dataset desde el archivo CSV
dataset = tf.data.experimental.make_csv_dataset(
    file_path,
    batch_size=32,  # Número de muestras por lote
    label_name='target_column',  # La columna que se usará como etiquetas
    na_value="?",  # Valor que se considerará como faltante
    num_epochs=1,  # Número de veces para iterar sobre el dataset
    ignore_errors=True  # Ignorar errores durante la carga
)

# Inspeccionar el dataset
for features, label in dataset.take(1):
    print("Características: ", features)
    print("Etiqueta: ", label)

Carga de Datos de Imágenes

TensorFlow también proporciona utilidades para cargar y preprocesar datos de imágenes. La función tf.keras.preprocessing.image_dataset_from_directory es muy útil para esto.

import tensorflow as tf

# Definir la ruta del directorio
directory_path = 'ruta/a/tus/imagenes'

# Crear un dataset desde el directorio
dataset = tf.keras.preprocessing.image_dataset_from_directory(
    directory_path,
    image_size=(256, 256),  # Redimensionar imágenes a 256x256
    batch_size=32,  # Número de muestras por lote
    label_mode='int'  # Las etiquetas se devuelven como enteros
)

# Inspeccionar el dataset
for images, labels in dataset.take(1):
    print("Imágenes: ", images.shape)
    print("Etiquetas: ", labels)

Preprocesamiento de Datos

Normalización

La normalización es un paso común de preprocesamiento que escala las características a un rango estándar, usualmente [0, 1] o [-1, 1].

def normalize(image, label):
    image = tf.cast(image, tf.float32) / 255.0
    return image, label

# Aplicar normalización al dataset
normalized_dataset = dataset.map(normalize)

# Inspeccionar el dataset normalizado
for images, labels in normalized_dataset.take(1):
    print("Imágenes Normalizadas: ", images[0])
    print("Etiquetas: ", labels)

Aumento de Datos

El aumento de datos es una técnica para aumentar artificialmente el tamaño del conjunto de entrenamiento creando versiones modificadas de las imágenes en el conjunto de datos.

data_augmentation = tf.keras.Sequential([
    tf.keras.layers.RandomFlip("horizontal_and_vertical"),
    tf.keras.layers.RandomRotation(0.2),
])

# Aplicar aumento de datos al dataset
augmented_dataset = normalized_dataset.map(lambda x, y: (data_augmentation(x, training=True), y))

# Inspeccionar el dataset aumentado
for images, labels in augmented_dataset.take(1):
    print("Imágenes Aumentadas: ", images[0])
    print("Etiquetas: ", labels)

División de Datos

Dividir el dataset en conjuntos de entrenamiento, validación y prueba es esencial para evaluar el rendimiento de tu modelo.

# Definir los tamaños de división
train_size = 0.7
val_size = 0.15
test_size = 0.15

# Calcular el número de muestras
total_samples = len(list(dataset))
train_samples = int(train_size * total_samples)
val_samples = int(val_size * total_samples)

# Dividir el dataset
train_dataset = dataset.take(train_samples)
val_dataset = dataset.skip(train_samples).take(val_samples)
test_dataset = dataset.skip(train_samples + val_samples)

# Inspeccionar las divisiones
print(f"Muestras de entrenamiento: {len(list(train_dataset))}")
print(f"Muestras de validación: {len(list(val_dataset))}")
print(f"Muestras de prueba: {len(list(test_dataset))}")

Conclusión

En esta sección, cubrimos los pasos esenciales para cargar y preprocesar datos en TensorFlow. Aprendimos cómo:

  • Cargar datos desde archivos CSV y directorios de imágenes.
  • Normalizar y aumentar datos.
  • Dividir datos en conjuntos de entrenamiento, validación y prueba.

Estos pasos son fundamentales para preparar tus datos para tareas de aprendizaje automático. Una adecuada carga y preprocesamiento de datos puede mejorar significativamente el rendimiento y la fiabilidad de tus modelos.
