TensorFlow.js es una poderosa biblioteca para el aprendizaje automático en JavaScript. Te permite construir y entrenar modelos directamente en el navegador o en Node.js. Esta sección te guiará desde los conceptos básicos hasta los avanzados de TensorFlow.js, proporcionándote el conocimiento para aprovechar esta tecnología en tus aplicaciones web.

Introducción a TensorFlow.js

  • ¿Qué es TensorFlow.js?

    • Una biblioteca de JavaScript para entrenar y desplegar modelos de aprendizaje automático en el navegador y en Node.js.
    • Permite aplicaciones de aprendizaje automático en tiempo real e interactivas.

  • ¿Por qué usar TensorFlow.js?

    • Aprovecha el poder del aprendizaje automático en aplicaciones web.
    • Utiliza la GPU para cálculos directamente en el navegador.
    • Construye y entrena modelos en JavaScript, haciéndolo accesible para desarrolladores web.

Configuración de TensorFlow.js

  • Instalando TensorFlow.js

    • Usando npm: 
      npm install @tensorflow/tfjs
    • Usando un CDN: 
      <script src="https://cdn.jsdelivr.net/npm/@tensorflow/tfjs"></script>

  • Ejemplo de Configuración Básica

    <!DOCTYPE html>
<html>
<head>
  <title>Ejemplo de TensorFlow.js</title>
  <script src="https://cdn.jsdelivr.net/npm/@tensorflow/tfjs"></script>
</head>
<body>
  <script>
    // Tu código de TensorFlow.js irá aquí
    console.log(tf.version.tfjs);
  </script>
</body>
</html>

Conceptos Básicos

  • Tensores

    • La unidad central de datos en TensorFlow.js.
    • Arreglos multidimensionales con un tipo uniforme.
    • Ejemplo: 
      const tensor = tf.tensor([1, 2, 3, 4]);
tensor.print();  // Salida: [1, 2, 3, 4]

  • Operaciones en Tensores

    • Operaciones básicas como suma, multiplicación, etc.
    • Ejemplo: 
      const a = tf.tensor([1, 2, 3, 4]);
const b = tf.tensor([5, 6, 7, 8]);
const c = tf.add(a, b);
c.print();  // Salida: [6, 8, 10, 12]

Construcción y Entrenamiento de Modelos

  • Modelo Secuencial

    • Una pila lineal de capas.
    • Ejemplo: 
      const model = tf.sequential();
model.add(tf.layers.dense({units: 32, inputShape: [50]}));
model.add(tf.layers.dense({units: 1}));

  • Compilando el Modelo

    • Especificar el optimizador, la función de pérdida y las métricas.
    • Ejemplo: 
      model.compile({
  optimizer: 'sgd',
  loss: 'meanSquaredError',
  metrics: ['mse']
});

  • Entrenando el Modelo

    • Usando el método fit.
    • Ejemplo: 
      const xs = tf.tensor2d([[1], [2], [3], [4]], [4, 1]);
const ys = tf.tensor2d([[1], [3], [5], [7]], [4, 1]);

model.fit(xs, ys, {
  epochs: 10
}).then(() => {
  // Usar el modelo para inferencia
  model.predict(tf.tensor2d([[5]], [1, 1])).print();  // Salida: [9]
});

Temas Avanzados

  • Aprendizaje por Transferencia

    • Usando modelos preentrenados y ajustándolos para tareas específicas.
    • Ejemplo con MobileNet: 
      const mobilenet = await tf.loadLayersModel('https://tfhub.dev/google/tfjs-model/imagenet/mobilenet_v2_100_224/classification/3/default/1', {fromTFHub: true});
const activation = mobilenet.predict(tf.zeros([1, 224, 224, 3]));

  • Capas y Modelos Personalizados

    • Creando capas personalizadas extendiendo tf.layers.Layer.
    • Ejemplo: 
      class CustomLayer extends tf.layers.Layer {
  constructor() {
    super({});
  }

  computeOutputShape(inputShape) {
    return inputShape;
  }

  call(input) {
    return input.mul(tf.scalar(2));
  }

  static get className() {
    return 'CustomLayer';
  }
}
tf.serialization.registerClass(CustomLayer);

const model = tf.sequential();
model.add(new CustomLayer());

  • Usando TensorFlow.js con Node.js

    • Ejecutando TensorFlow.js en el lado del servidor.
    • Ejemplo: 
      const tf = require('@tensorflow/tfjs-node');

const tensor = tf.tensor([1, 2, 3, 4]);
tensor.print();  // Salida: [1, 2, 3, 4]

Conclusión

TensorFlow.js trae el poder del aprendizaje automático al ecosistema de JavaScript, permitiendo a los desarrolladores construir, entrenar y desplegar modelos en el navegador y en Node.js. Al entender los conceptos básicos, configurar el entorno y explorar temas avanzados, puedes crear aplicaciones de aprendizaje automático sofisticadas que funcionen eficientemente en la web. Ya seas un principiante o un usuario avanzado, TensorFlow.js ofrece un conjunto de herramientas versátiles y poderosas para tus proyectos de aprendizaje automático.
