Introducción a las Variables CSS

Las Variables CSS, también conocidas como Propiedades Personalizadas, te permiten almacenar valores que puedes reutilizar a lo largo de tu hoja de estilos. Son una característica poderosa que puede hacer que tu CSS sea más mantenible y flexible.

Conceptos Clave

  • Definición: Las Variables CSS son entidades definidas por los autores de CSS que contienen valores específicos para ser reutilizados a lo largo de un documento.
  • Sintaxis: Las Variables CSS se definen usando la notación de propiedad personalizada (--nombre-propiedad) y se accede a ellas usando la función var().
  • Ámbito: Las variables pueden tener un ámbito global (disponibles en todo el documento) o local (disponibles dentro de un selector específico).

Definiendo y Usando Variables CSS

Definiendo Variables

Las Variables CSS se definen dentro de una regla CSS. Generalmente se definen en el selector :root para un ámbito global.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
  --padding: 10px;
}

Usando Variables

Para usar una Variable CSS, usas la función var().

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
  padding: var(--padding);
}

Ejemplo

Aquí hay un ejemplo simple que demuestra la definición y el uso de Variables CSS:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejemplo de Variables CSS</title>
  <style>
    :root {
      --main-bg-color: #3498db;
      --main-text-color: #ffffff;
      --padding: 10px;
    }

    body {
      background-color: var(--main-bg-color);
      color: var(--main-text-color);
      padding: var(--padding);
    }

    .container {
      background-color: var(--main-text-color);
      color: var(--main-bg-color);
      padding: var(--padding);
    }
  </style>
</head>
<body>
  <div class="container">
    <h1>¡Hola, Mundo!</h1>
    <p>Este es un ejemplo de Variables CSS.</p>
  </div>
</body>
</html>

Uso Avanzado de Variables CSS

Valores de Respaldo

Las Variables CSS pueden tener valores de respaldo, que se usan si la variable no está definida.

body {
  background-color: var(--main-bg-color, #2980b9); /* Respaldo a #2980b9 si --main-bg-color no está definida */
}

Actualizaciones Dinámicas

Las Variables CSS pueden ser actualizadas dinámicamente usando JavaScript.

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Variables CSS Dinámicas</title>
  <style>
    :root {
      --main-bg-color: #3498db;
    }

    body {
      background-color: var(--main-bg-color);
    }
  </style>
</head>
<body>
  <button onclick="changeColor()">Cambiar Color de Fondo</button>

  <script>
    function changeColor() {
      document.documentElement.style.setProperty('--main-bg-color', '#e74c3c');
    }
  </script>
</body>
</html>

Tematización con Variables CSS

Las Variables CSS pueden ser usadas para crear temas para tu aplicación.

:root {
  --main-bg-color: #3498db;
  --main-text-color: #ffffff;
}

.dark-theme {
  --main-bg-color: #2c3e50;
  --main-text-color: #ecf0f1;
}

body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}

Para aplicar el tema oscuro:

<body class="dark-theme">
  <!-- Contenido aquí -->
</body>

Conclusión

Las Variables CSS son una herramienta poderosa para hacer tu CSS más mantenible y flexible. Te permiten definir valores una vez y reutilizarlos a lo largo de tu hoja de estilos, haciendo más fácil gestionar y actualizar tus estilos. Al entender y utilizar las Variables CSS, puedes crear diseños web más dinámicos y adaptables.
