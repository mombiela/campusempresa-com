Introducción a CSS

  • ¿Qué es CSS?
    • CSS significa Hojas de Estilo en Cascada.
    • Se utiliza para diseñar y maquetar páginas web.
  • Cómo Funciona CSS
    • CSS se utiliza para controlar la presentación de los elementos HTML.
    • CSS se puede incluir en HTML de tres maneras: Inline, Interno y Externo.

Ejemplo: CSS Inline

<p style="color: blue;">Este es un párrafo azul.</p>

Ejemplo: CSS Interno

<!DOCTYPE html>
<html>
<head>
<style>
  p {
    color: blue;
  }
</style>
</head>
<body>
  <p>Este es un párrafo azul.</p>
</body>
</html>

Ejemplo: CSS Externo

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
  <p>Este es un párrafo azul.</p>
</body>
</html>
/* styles.css */
p {
  color: blue;
}

Conceptos Básicos de CSS

  • Selectores
    • Selector de Elemento
    • Selector de Clase
    • Selector de ID
  • Propiedades y Valores
    • Color
    • Fuente
    • Margen
    • Relleno

Ejemplo: Selectores

<!DOCTYPE html>
<html>
<head>
<style>
  /* Selector de Elemento */
  p {
    color: red;
  }
  /* Selector de Clase */
  .blue-text {
    color: blue;
  }
  /* Selector de ID */
  #green-text {
    color: green;
  }
</style>
</head>
<body>
  <p>Este es un párrafo rojo.</p>
  <p class="blue-text">Este es un párrafo azul.</p>
  <p id="green-text">Este es un párrafo verde.</p>
</body>
</html>

Conceptos Intermedios de CSS

  • Modelo de Caja
    • Contenido
    • Relleno
    • Borde
    • Margen
  • Posicionamiento
    • Estático
    • Relativo
    • Absoluto
    • Fijo
    • Pegajoso

Ejemplo: Modelo de Caja

<!DOCTYPE html>
<html>
<head>
<style>
  .box {
    width: 100px;
    height: 100px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
    background-color: lightblue;
  }
</style>
</head>
<body>
  <div class="box">Ejemplo de Modelo de Caja</div>
</body>
</html>

Ejemplo: Posicionamiento

<!DOCTYPE html>
<html>
<head>
<style>
  .relative {
    position: relative;
    top: 20px;
    left: 20px;
    background-color: yellow;
  }
  .absolute {
    position: absolute;
    top: 50px;
    left: 50px;
    background-color: green;
  }
</style>
</head>
<body>
  <div class="relative">Posicionamiento Relativo</div>
  <div class="absolute">Posicionamiento Absoluto</div>
</body>
</html>

Conceptos Avanzados de CSS

  • Flexbox
    • Contenedor Flex
    • Ítems Flex
    • Propiedades Flex (justify-content, align-items, etc.)
  • Diseño de Cuadrícula
    • Contenedor de Cuadrícula
    • Ítems de Cuadrícula
    • Propiedades de Cuadrícula (grid-template-columns, grid-template-rows, etc.)

Ejemplo: Flexbox

<!DOCTYPE html>
<html>
<head>
<style>
  .container {
    display: flex;
    justify-content: space-between;
  }
  .item {
    background-color: lightcoral;
    padding: 20px;
    margin: 10px;
  }
</style>
</head>
<body>
  <div class="container">
    <div class="item">Ítem 1</div>
    <div class="item">Ítem 2</div>
    <div class="item">Ítem 3</div>
  </div>
</body>
</html>

Ejemplo: Diseño de Cuadrícula

<!DOCTYPE html>
<html>
<head>
<style>
  .grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
    gap: 10px;
  }
  .grid-item {
    background-color: lightgreen;
    padding: 20px;
    text-align: center;
  }
</style>
</head>
<body>
  <div class="grid-container">
    <div class="grid-item">1</div>
    <div class="grid-item">2</div>
    <div class="grid-item">3</div>
    <div class="grid-item">4</div>
    <div class="grid-item">5</div>
    <div class="grid-item">6</div>
  </div>
</body>
</html>

Construyendo un Sitio Web Completo

  • Estructura del Proyecto
    • Archivo HTML
    • Archivo CSS
    • Recursos (imágenes, fuentes, etc.)
  • Estilizando el Diseño
    • Encabezado
    • Navegación
    • Contenido Principal
    • Barra Lateral
    • Pie de Página
  • Diseño Responsivo
    • Consultas de Medios
    • Enfoque Mobile-First

Ejemplo: Estructura del Proyecto

/project
  /assets
    /images
      logo.png
  index.html
  styles.css

Ejemplo: Sitio Web Completo

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" type="text/css" href="styles.css">
</head>
<body>
  <header>
    <img src="assets/images/logo.png" alt="Logo">
    <nav>
      <ul>
        <li><a href="#home">Inicio</a></li>
        <li><a href="#about">Acerca de</a></li>
        <li><a href="#contact">Contacto</a></li>
      </ul>
    </nav>
  </header>
  <main>
    <section>
      <h1>Bienvenido a Nuestro Sitio Web</h1>
      <p>Esta es el área de contenido principal.</p>
    </section>
    <aside>
      <h2>Barra Lateral</h2>
      <p>Este es el contenido de la barra lateral.</p>
    </aside>
  </main>
  <footer>
    <p>&copy; 2023 Nuestro Sitio Web</p>
  </footer>
</body>
</html>
/* styles.css */
body {
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

header {
  background-color: #333;
  color: white;
  padding: 10px 0;
  text-align: center;
}

header img {
  width: 50px;
}

nav ul {
  list-style-type: none;
  padding: 0;
}

nav ul li {
  display: inline;
  margin: 0 10px;
}

nav ul li a {
  color: white;
  text-decoration: none;
}

main {
  display: flex;
  justify-content: space-between;
  padding: 20px;
}

section {
  flex: 3;
}

aside {
  flex: 1;
  background-color: #f4f4f4;
  padding: 10px;
}

footer {
  background-color: #333;
  color: white;
  text-align: center;
  padding: 10px 0;
  position: fixed;
  width: 100%;
  bottom: 0;
}

Conclusión

En este tema, cubrimos lo esencial de CSS desde conceptos básicos hasta avanzados. Comenzamos con una introducción a CSS y sus varios métodos de inclusión en HTML. Luego exploramos conceptos básicos como selectores y propiedades, seguidos por conceptos intermedios como el modelo de caja y el posicionamiento. Finalmente, profundizamos en conceptos avanzados como Flexbox y Diseño de Cuadrícula, y demostramos cómo construir un sitio web completo con una estructura de proyecto. Al dominar estos conceptos, estarás bien equipado para crear sitios web visualmente atractivos y responsivos.
