Las transiciones y animaciones en CSS son herramientas poderosas que permiten a los desarrolladores crear experiencias web dinámicas y atractivas. Este tema cubrirá los conceptos básicos de las transiciones y animaciones en CSS, avanzando hacia técnicas más avanzadas.

Introducción a las Transiciones

Las transiciones en CSS permiten cambiar los valores de las propiedades de manera suave (durante una duración determinada).

Conceptos Clave

  • Propiedades de Transición: transition-property, transition-duration, transition-timing-function, transition-delay.
  • Propiedad Abreviada: transition.

Ejemplo Básico

/* HTML */
<button class="btn">¡Pasa el ratón por aquí!</button>

/* CSS */
.btn {
  background-color: blue;
  color: white;
  padding: 10px 20px;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

.btn:hover {
  background-color: green;
}

En este ejemplo:

  • La propiedad transition se usa para definir el efecto de transición.
  • Cuando se pasa el ratón sobre el botón, el color de fondo cambia de azul a verde en 0.3 segundos.

Propiedades de Transición Explicadas

  • transition-property: Especifica la propiedad CSS a la que se aplicará la transición.
  • transition-duration: Especifica la duración de la transición.
  • transition-timing-function: Especifica la curva de velocidad de la transición.
  • transition-delay: Especifica un retraso antes de que comience la transición.

Funciones de Tiempo de Transición

| Función de Tiempo | Descripción | |-------------------|-------------| | ease | Predeterminado. Comienza lento, luego rápido, luego termina lento. | | linear | Misma velocidad de principio a fin. | | ease-in | Comienza lento. | | ease-out | Termina lento. | | ease-in-out | Comienza y termina lento. |

Introducción a las Animaciones

Las animaciones en CSS hacen posible animar transiciones de una configuración de estilo CSS a otra.

Conceptos Clave

  • Propiedades de Animación: animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction, animation-fill-mode, animation-play-state.
  • Keyframes: Definen los estilos para el elemento en varios puntos durante la secuencia de animación.

Ejemplo Básico

/* HTML */
<div class="box"></div>

/* CSS */
.box {
  width: 100px;
  height: 100px;
  background-color: red;
  position: relative;
  animation: move 2s infinite;
}

@keyframes move {
  0% { left: 0; }
  50% { left: 200px; }
  100% { left: 0; }
}

En este ejemplo:

  • La propiedad animation se usa para definir la animación.
  • La regla @keyframes especifica la secuencia de animación.

Propiedades de Animación Explicadas

  • animation-name: Especifica el nombre de la animación @keyframes.
  • animation-duration: Especifica la duración de la animación.
  • animation-timing-function: Especifica la curva de velocidad de la animación.
  • animation-delay: Especifica un retraso antes de que comience la animación.
  • animation-iteration-count: Especifica el número de veces que se debe reproducir la animación.
  • animation-direction: Especifica si la animación debe reproducirse en reversa en ciclos alternos.
  • animation-fill-mode: Especifica cómo una animación CSS debe aplicar estilos a su objetivo antes y después de su ejecución.
  • animation-play-state: Especifica si la animación está en ejecución o en pausa.

Ejemplo Avanzado

/* HTML */
<div class="circle"></div>

/* CSS */
.circle {
  width: 50px;
  height: 50px;
  background-color: blue;
  border-radius: 50%;
  position: relative;
  animation: bounce 2s infinite alternate ease-in-out;
}

@keyframes bounce {
  0% { top: 0; }
  100% { top: 200px; }
}

En este ejemplo:

  • La propiedad animation incluye múltiples valores: bounce (nombre), 2s (duración), infinite (recuento de iteraciones), alternate (dirección) y ease-in-out (función de tiempo).
  • La regla @keyframes define el efecto de rebote.

Conclusión

Las transiciones y animaciones en CSS son esenciales para crear páginas web interactivas y visualmente atractivas. Al comprender y utilizar estas propiedades, puedes mejorar la experiencia del usuario y dar vida a tus diseños web. Practica experimentando con diferentes propiedades y keyframes para ver los efectos que producen.
