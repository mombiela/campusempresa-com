Introducción a CSS para Impresión

CSS para impresión es un área especializada de CSS que se enfoca en estilizar contenido web para medios impresos. Mientras que la mayoría de CSS está diseñado para la visualización en pantalla, las hojas de estilo para impresión aseguran que tu contenido se vea bien cuando se imprime en papel.

Conceptos Clave

  • Consultas de Medios para Impresión: Usa consultas de medios para aplicar estilos específicamente para impresión.
  • Saltos de Página: Controla dónde se divide el contenido entre páginas.
  • Ocultar Elementos: Oculta elementos que son innecesarios o irrelevantes en la impresión.
  • Estilización para la Legibilidad: Ajusta fuentes, colores y diseño para una mejor legibilidad en papel.

Consultas de Medios para Impresión

Las consultas de medios te permiten aplicar reglas CSS basadas en el tipo de medio. Para estilos de impresión, usa la regla @media print.

@media print {
  body {
    font-size: 12pt;
    line-height: 1.5;
  }
}

En este ejemplo:

  • La regla @media print se dirige a medios de impresión.
  • El tamaño del texto del cuerpo se establece en 12pt, y la altura de línea se incrementa a 1.5 para una mejor legibilidad.

Saltos de Página

Los saltos de página son cruciales para asegurar que el contenido se divida lógicamente a través de las páginas. Usa las propiedades break-before, break-after y break-inside.

@media print {
  h1 {
    break-before: page;
  }
  p {
    break-inside: avoid;
  }
}

En este ejemplo:

  • break-before: page asegura que cada h1 comience en una nueva página.
  • break-inside: avoid evita que los párrafos se dividan entre páginas.

Ocultar Elementos

Algunos elementos, como barras de navegación o contenido interactivo, son innecesarios en la impresión. Usa la propiedad display: none para ocultar estos elementos.

@media print {
  .navbar, .footer, .interactive-content {
    display: none;
  }
}

En este ejemplo:

  • Las clases .navbar, .footer y .interactive-content se ocultan en la versión impresa.

Estilización para la Legibilidad

El contenido impreso debe ser fácil de leer. Ajusta fuentes, colores y diseño para mejorar la legibilidad.

@media print {
  body {
    font-family: serif;
    color: black;
    background: white;
  }
  a {
    color: black;
    text-decoration: none;
  }
}

En este ejemplo:

  • La fuente del cuerpo se cambia a una fuente serif, que generalmente es más fácil de leer en papel.
  • El color del texto se establece en negro, y el fondo se establece en blanco.
  • Los enlaces se estilizan para que parezcan texto regular, ya que no son clicables en la impresión.

Resumen

CSS para impresión es esencial para asegurar que el contenido web se presente efectivamente en papel. Usando consultas de medios, controlando los saltos de página, ocultando elementos innecesarios y estilizando para la legibilidad, puedes crear páginas web amigables para la impresión. Estas técnicas aseguran que tu contenido mantenga su integridad y legibilidad, ya sea visto en una pantalla o impreso.
