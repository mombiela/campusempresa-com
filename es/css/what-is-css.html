Introducción a CSS

CSS, o Hojas de Estilo en Cascada, es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. CSS es una tecnología fundamental de la World Wide Web, junto con HTML y JavaScript. Te permite controlar el diseño, los colores, las fuentes y la apariencia visual general de las páginas web.

Conceptos Clave de CSS

  • Selectores: Patrones utilizados para seleccionar los elementos que deseas estilizar.
  • Propiedades: Aspectos de los elementos que deseas cambiar (por ejemplo, color, tamaño de fuente).
  • Valores: Configuraciones específicas para las propiedades (por ejemplo, rojo, 16px).
  • Conjuntos de Reglas: Combinaciones de selectores, propiedades y valores.

Ejemplo de un Conjunto de Reglas CSS Simple

/* Este es un comentario en CSS */
p {
  color: blue; /* Esto establece el color del texto de todos los elementos <p> a azul */
  font-size: 16px; /* Esto establece el tamaño de la fuente de todos los elementos <p> a 16 píxeles */
}

En este ejemplo:

  • p es el selector, que apunta a todos los elementos <p>.
  • color y font-size son propiedades.
  • blue y 16px son valores.

Cómo Funciona CSS

CSS funciona asociando reglas con elementos HTML. Estas reglas gobiernan cómo debe mostrarse el contenido de los elementos especificados. CSS puede aplicarse de tres maneras diferentes:

  1. CSS en Línea: Directamente dentro de un elemento HTML usando el atributo style.
  2. CSS Interno: Dentro de una etiqueta <style> en la sección <head> de un documento HTML.
  3. CSS Externo: En un archivo .css externo vinculado al documento HTML.

Ejemplo de CSS en Línea

<p style="color: blue; font-size: 16px;">Este es un párrafo con CSS en línea.</p>

Ejemplo de CSS Interno

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejemplo de CSS Interno</title>
  <style>
    p {
      color: blue;
      font-size: 16px;
    }
  </style>
</head>
<body>
  <p>Este es un párrafo con CSS interno.</p>
</body>
</html>

Ejemplo de CSS Externo

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ejemplo de CSS Externo</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>Este es un párrafo con CSS externo.</p>
</body>
</html>

En el archivo styles.css:

p {
  color: blue;
  font-size: 16px;
}

Ventajas de Usar CSS

  • Separación de Preocupaciones: CSS separa el contenido (HTML) de la presentación (CSS), lo que facilita el mantenimiento y la actualización.
  • Reutilización: Las reglas CSS pueden reutilizarse en múltiples páginas HTML.
  • Consistencia: Asegura una apariencia y sensación consistentes en todo un sitio web.
  • Eficiencia: Reduce la cantidad de duplicación de código y hace que el sitio web cargue más rápido.

Conclusión

CSS es una tecnología esencial para el desarrollo web, que permite a los desarrolladores crear sitios web visualmente atractivos y fáciles de usar. Al comprender los conceptos básicos de CSS, incluidos los selectores, propiedades y valores, y cómo aplicar CSS a través de métodos en línea, internos y externos, puedes comenzar a controlar la presentación de tus páginas web de manera efectiva. A medida que avances en este curso, aprenderás técnicas más avanzadas y mejores prácticas para dominar CSS.
