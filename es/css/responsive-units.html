El diseño responsivo es un aspecto crucial del desarrollo web moderno. Garantiza que las páginas web se vean bien y funcionen correctamente en una variedad de dispositivos y tamaños de pantalla. En CSS, las unidades responsivas ayudan a lograr esta flexibilidad. Esta sección cubrirá los diferentes tipos de unidades responsivas, su uso y ejemplos prácticos.

Conceptos Clave

  • Unidades Absolutas: Unidades fijas que no cambian con el tamaño del viewport.
    • Ejemplos: px (píxeles), pt (puntos), cm (centímetros), in (pulgadas)
  • Unidades Relativas: Unidades que son relativas a otra propiedad de longitud.
    • Ejemplos: em, rem, %, vw, vh, vmin, vmax

Unidades Absolutas

Las unidades absolutas son fijas y no se escalan con el viewport. Rara vez se usan en diseño responsivo porque no se adaptan a diferentes tamaños de pantalla.

Ejemplo

div {
  width: 300px;
  height: 200px;
}

En este ejemplo, el div siempre tendrá 300 píxeles de ancho y 200 píxeles de alto, independientemente del tamaño de la pantalla.

Unidades Relativas

Las unidades relativas son más flexibles y se usan comúnmente en diseño responsivo. Se adaptan según el contexto en el que se usan.

`em` y `rem`

  • em: Relativo al tamaño de fuente del elemento.
  • rem: Relativo al tamaño de fuente del elemento raíz (<html>).

Ejemplo

html {
  font-size: 16px;
}

div {
  font-size: 2em; /* 32px */
  padding: 1em; /* 16px */
}

p {
  font-size: 1.5rem; /* 24px */
}

En este ejemplo:

  • El tamaño de fuente del div es 2 veces el tamaño de fuente de su elemento padre.
  • El tamaño de fuente del p es 1.5 veces el tamaño de fuente del elemento raíz.

Porcentaje (`%`)

Las unidades de porcentaje son relativas a las dimensiones del elemento padre.

Ejemplo

div {
  width: 50%;
  height: 50%;
}

En este ejemplo, el div ocupará el 50% del ancho y el 50% de la altura de su elemento padre.

Unidades de Viewport (`vw`, `vh`, `vmin`, `vmax`)

  • vw: 1% del ancho del viewport.
  • vh: 1% de la altura del viewport.
  • vmin: 1% de la dimensión menor del viewport (ancho o altura).
  • vmax: 1% de la dimensión mayor del viewport (ancho o altura).

Ejemplo

div {
  width: 50vw; /* 50% del ancho del viewport */
  height: 50vh; /* 50% de la altura del viewport */
}

En este ejemplo, el div siempre será la mitad del ancho y la altura del viewport.

Ejercicios Prácticos

  1. Ejercicio 1: Crea un div responsivo que siempre sea el 50% del ancho y la altura del viewport.
  2. Ejercicio 2: Usa unidades em y rem para establecer el tamaño de fuente de un párrafo dentro de un div de manera que se escale con el tamaño de fuente raíz.
  3. Ejercicio 3: Crea un diseño donde un div ocupe el 75% del ancho del elemento padre y el 50% de su altura.

Soluciones de Ejemplo

Solución del Ejercicio 1:

div {
  width: 50vw;
  height: 50vh;
}

Solución del Ejercicio 2:

html {
  font-size: 16px;
}

div {
  font-size: 2em; /* 32px */
}

p {
  font-size: 1.5rem; /* 24px */
}

Solución del Ejercicio 3:

.parent {
  width: 100%;
  height: 100%;
}

.child {
  width: 75%;
  height: 50%;
}

Conclusión

Entender y usar unidades responsivas en CSS es esencial para crear diseños web flexibles y adaptativos. Al dominar unidades como em, rem, %, vw y vh, puedes asegurarte de que tus páginas web se vean geniales en cualquier dispositivo. Practica usando estas unidades en varios escenarios para volverte competente en diseño responsivo.
