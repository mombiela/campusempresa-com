Introducción

En esta sección, revisaremos y recapitularemos los conceptos clave, herramientas y técnicas cubiertas a lo largo del curso de Ansible. Esto ayudará a reforzar tu comprensión y asegurará que estés bien preparado para aplicar Ansible en escenarios del mundo real.

Conceptos Clave

Fundamentos de Ansible

  • ¿Qué es Ansible?
    • Ansible es una herramienta de automatización de código abierto utilizada para la gestión de configuraciones, despliegue de aplicaciones y automatización de tareas.
  • Características Clave:
    • Arquitectura sin agentes
    • Sintaxis simple de YAML
    • Idempotencia

Inventario

  • Archivos de Inventario:
    • Definen los hosts y grupos de hosts sobre los cuales operan los comandos, módulos y playbooks de Ansible.
    • Ejemplo: 
      [webservers]
web1.ejemplo.com
web2.ejemplo.com

[dbservers]
db1.ejemplo.com
db2.ejemplo.com
  • Inventario Dinámico:
    • Permite la generación dinámica de inventarios basados en fuentes externas como proveedores de nube.

Módulos

  • Módulos Básicos:
    • Preinstalados con Ansible y cubren una amplia gama de funcionalidades.
    • Ejemplo: 
      - name: Instalar Nginx
  ansible.builtin.yum:
    name: nginx
    state: present
  • Módulos Personalizados:
    • Módulos definidos por el usuario para extender las capacidades de Ansible.

Playbooks

  • Estructura:
    • Escritos en YAML y definen una serie de tareas a ejecutar en un conjunto de hosts.
    • Ejemplo: 
      - hosts: webservers
  tasks:
    - name: Instalar Nginx
      ansible.builtin.yum:
        name: nginx
        state: present
  • Handlers:
    • Tareas especiales que se ejecutan solo cuando son notificadas por otras tareas.
  • Variables:
    • Utilizadas para almacenar valores que pueden ser reutilizados a lo largo de los playbooks.

Roles

  • Propósito:
    • Organizar los playbooks en componentes reutilizables.
  • Estructura:
    • Típicamente incluyen directorios como tasks, handlers, files, templates y vars.

Ansible Galaxy

  • ¿Qué es Ansible Galaxy?
    • Un repositorio para roles de Ansible, permitiendo a los usuarios compartir y descargar roles.
  • Usando Ansible Galaxy:
    • Ejemplo: 
      ansible-galaxy install nombre_usuario.nombre_rol

Temas Avanzados

Ansible Vault

  • Propósito:
    • Almacenar de manera segura datos sensibles como contraseñas y claves.
  • Uso:
    • Encriptar un archivo: 
      ansible-vault encrypt secretos.yml
    • Desencriptar un archivo: 
      ansible-vault decrypt secretos.yml

Ansible Tower

  • ¿Qué es Ansible Tower?
    • Una solución basada en web que hace Ansible aún más poderoso al proporcionar una GUI, API REST y otras características empresariales.
  • Características Clave:
    • Control de acceso basado en roles
    • Programación de trabajos
    • Actualizaciones de estado de trabajos en tiempo real

Plugins Personalizados

  • Tipos de Plugins:
    • Plugins de Acción, Callback, Conexión, Lookup y Filtro.
  • Creando un Plugin Personalizado:
    • Ejemplo de un simple plugin de filtro: 
      # filter_plugins/my_filter.py
def to_uppercase(value):
    return value.upper()

class FilterModule(object):
    def filters(self):
        return {
            'to_uppercase': to_uppercase
        }

Resumen

En este curso, hemos cubierto una amplia gama de temas desde los fundamentos de Ansible hasta características avanzadas y mejores prácticas. Aquí hay un breve resumen:

  • Fundamentos de Ansible: Entender qué es Ansible y sus características clave.
  • Gestión de Inventarios: Cómo definir y gestionar inventarios.
  • Módulos y Playbooks: Escribir y ejecutar tareas usando módulos y playbooks.
  • Roles y Ansible Galaxy: Organizar playbooks y usar roles compartidos.
  • Características Avanzadas: Usar Ansible Vault para seguridad, Ansible Tower para características empresariales y crear plugins personalizados.

Al dominar estos temas, ahora estás equipado para usar Ansible de manera efectiva para automatizar tareas de TI y gestionar despliegues complejos. Sigue practicando y explorando casos de uso más avanzados para mejorar aún más tus habilidades.
