Sintaxis de Métodos en Objective-C
Paso de Parámetros ►

Introducción

Objective-C es un lenguaje poderoso que extiende C con capacidades orientadas a objetos. Entender la sintaxis de los métodos es crucial para escribir código efectivo en Objective-C. Esta sección cubrirá los conceptos básicos de la sintaxis de métodos, incluyendo declaración, implementación y uso.

Declaración de Métodos

En Objective-C, los métodos se declaran en la sección de interfaz de una clase. Las declaraciones de métodos especifican el nombre del método, el tipo de retorno y los parámetros.

Métodos de Instancia

Los métodos de instancia se llaman en instancias de una clase.

@interface MyClass : NSObject
- (void)sayHello;
- (int)addNumber:(int)a toNumber:(int)b;
@end
  • - indica un método de instancia.
  • (void) especifica el tipo de retorno.
  • sayHello es el nombre del método.
  • addNumber:toNumber: es un método con dos parámetros.

Métodos de Clase

Los métodos de clase se llaman en la clase misma.

@interface MyClass : NSObject
+ (void)classMethodExample;
@end
  • + indica un método de clase.

Implementación de Métodos

Los métodos se implementan en la sección de implementación de una clase.

Métodos de Instancia

@implementation MyClass
- (void)sayHello {
    NSLog(@"Hello, World!");
}

- (int)addNumber:(int)a toNumber:(int)b {
    return a + b;
}
@end
  • El cuerpo del método está encerrado en {}.
  • NSLog se usa para registrar la salida en la consola.

Métodos de Clase

@implementation MyClass
+ (void)classMethodExample {
    NSLog(@"This is a class method.");
}
@end

Uso de Métodos

Los métodos se llaman usando la siguiente sintaxis:

Llamando Métodos de Instancia

MyClass *myObject = [[MyClass alloc] init];
[myObject sayHello];
int result = [myObject addNumber:5 toNumber:10];
NSLog(@"Result: %d", result);
  • [[MyClass alloc] init] crea una instancia de MyClass.
  • [myObject sayHello] llama al método sayHello en myObject.
  • [myObject addNumber:5 toNumber:10] llama al método addNumber:toNumber: y almacena el resultado.

Llamando Métodos de Clase

[MyClass classMethodExample];
  • [MyClass classMethodExample] llama al método de clase classMethodExample en MyClass.

Convenciones de Nombres de Métodos

Objective-C usa nombres de métodos descriptivos que incluyen los nombres de los parámetros.

Ejemplo

- (void)setFirstName:(NSString *)firstName lastName:(NSString *)lastName;
  • El nombre del método es setFirstName:lastName:.
  • Esta convención de nombres mejora la legibilidad del código.

Resumen

Entender la sintaxis de métodos en Objective-C es esencial para escribir código claro y efectivo. Esta sección cubrió:

  • Declaración de métodos tanto para métodos de instancia como de clase.
  • Implementación de métodos.
  • Cómo llamar métodos.
  • Convenciones de nombres para métodos.

Al dominar estos conceptos, estarás bien equipado para escribir y entender métodos en Objective-C, allanando el camino para temas más avanzados en el lenguaje.
