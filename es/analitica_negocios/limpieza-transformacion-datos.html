Introducción

La limpieza y transformación de datos son pasos críticos en el proceso de analítica de negocios. Estos procesos aseguran que los datos sean precisos, completos y estén en un formato adecuado para el análisis. En esta sección, aprenderás técnicas y herramientas para limpiar y transformar datos de manera efectiva.

Importancia de la Limpieza de Datos

La limpieza de datos es esencial para:

  • Eliminar errores y duplicados.
  • Manejar valores faltantes.
  • Corregir inconsistencias.
  • Mejorar la calidad de los datos para análisis precisos.

Técnicas de Limpieza de Datos

Eliminación de Duplicados

Los duplicados pueden distorsionar los resultados del análisis. Para eliminarlos:

import pandas as pd

# Crear un DataFrame de ejemplo
data = {'ID': [1, 2, 2, 3], 'Nombre': ['Ana', 'Luis', 'Luis', 'Carlos']}
df = pd.DataFrame(data)

# Eliminar duplicados
df_cleaned = df.drop_duplicates()
print(df_cleaned)

Manejo de Valores Faltantes

Los valores faltantes pueden ser tratados de varias maneras:

  • Eliminación de filas o columnas con valores faltantes.
  • Imputación de valores faltantes con la media, mediana o moda.
  • Uso de técnicas avanzadas como la imputación múltiple.

Ejemplo de imputación con la media:

import pandas as pd
import numpy as np

# Crear un DataFrame de ejemplo con valores faltantes
data = {'ID': [1, 2, 3], 'Edad': [25, np.nan, 30]}
df = pd.DataFrame(data)

# Imputar valores faltantes con la media
df['Edad'].fillna(df['Edad'].mean(), inplace=True)
print(df)

Corrección de Inconsistencias

Las inconsistencias pueden surgir de diferentes formatos o errores tipográficos. Ejemplo de corrección de inconsistencias:

import pandas as pd

# Crear un DataFrame de ejemplo con inconsistencias
data = {'ID': [1, 2, 3], 'Genero': ['M', 'F', 'm']}
df = pd.DataFrame(data)

# Corregir inconsistencias
df['Genero'] = df['Genero'].str.upper()
print(df)

Transformación de Datos

La transformación de datos implica convertir los datos en un formato adecuado para el análisis. Esto puede incluir:

  • Normalización y estandarización.
  • Creación de nuevas variables.
  • Agrupación y agregación de datos.

Normalización y Estandarización

Estas técnicas son útiles para poner los datos en una escala común.

from sklearn.preprocessing import StandardScaler, MinMaxScaler
import pandas as pd

# Crear un DataFrame de ejemplo
data = {'ID': [1, 2, 3], 'Ingreso': [50000, 60000, 55000]}
df = pd.DataFrame(data)

# Normalización
scaler = MinMaxScaler()
df['Ingreso_Normalizado'] = scaler.fit_transform(df[['Ingreso']])

# Estandarización
scaler = StandardScaler()
df['Ingreso_Estandarizado'] = scaler.fit_transform(df[['Ingreso']])
print(df)

Creación de Nuevas Variables

A veces es útil crear nuevas variables a partir de las existentes.

import pandas as pd

# Crear un DataFrame de ejemplo
data = {'ID': [1, 2, 3], 'Ingreso': [50000, 60000, 55000]}
df = pd.DataFrame(data)

# Crear una nueva variable 'Ingreso_Anual' a partir de 'Ingreso'
df['Ingreso_Anual'] = df['Ingreso'] * 12
print(df)

Agrupación y Agregación de Datos

Estas técnicas son útiles para resumir los datos.

import pandas as pd

# Crear un DataFrame de ejemplo
data = {'ID': [1, 2, 3, 4], 'Departamento': ['Ventas', 'Ventas', 'IT', 'IT'], 'Ingreso': [50000, 60000, 55000, 70000]}
df = pd.DataFrame(data)

# Agrupar y agregar datos
df_grouped = df.groupby('Departamento').agg({'Ingreso': 'mean'})
print(df_grouped)

Conclusión

La limpieza y transformación de datos son pasos fundamentales en la analítica de negocios. A través de técnicas como la eliminación de duplicados, manejo de valores faltantes, corrección de inconsistencias, normalización, creación de nuevas variables y agrupación de datos, puedes asegurar que tus datos estén en la mejor forma posible para el análisis. Practicar estas técnicas te permitirá mejorar la calidad de tus análisis y obtener insights más precisos y valiosos.
