Algoritmos de Flujo Máximo
Árboles de Expansión Mínima ►

Introducción

Los algoritmos de flujo máximo son fundamentales en la teoría de grafos y tienen aplicaciones en diversas áreas como la optimización de redes, la planificación de proyectos y la logística. Este tema cubre los conceptos básicos, los algoritmos más importantes y ejemplos prácticos para entender su funcionamiento.

Conceptos Básicos

  • Grafo de Flujo: Un grafo dirigido donde cada arista tiene una capacidad que representa el máximo flujo que puede pasar a través de ella.
  • Flujo: La cantidad de "material" que pasa a través de una arista.
  • Fuente (s): El nodo donde se origina el flujo.
  • Sumidero (t): El nodo donde el flujo se consume.
  • Corte: Una partición del grafo en dos subconjuntos que separa la fuente del sumidero.
  • Capacidad de Corte: La suma de las capacidades de las aristas que cruzan el corte.

Algoritmos Principales

Algoritmo de Ford-Fulkerson

El algoritmo de Ford-Fulkerson es uno de los métodos más conocidos para encontrar el flujo máximo en un grafo de flujo. Utiliza el concepto de caminos aumentantes.

Pasos del Algoritmo

  1. Inicialización: Comenzar con un flujo de 0.
  2. Búsqueda de Caminos Aumentantes: Encontrar un camino desde la fuente al sumidero donde el flujo puede ser incrementado.
  3. Aumentar el Flujo: Incrementar el flujo a lo largo del camino encontrado.
  4. Repetir: Repetir los pasos 2 y 3 hasta que no haya más caminos aumentantes.

Ejemplo de Código

def bfs(C, F, source, sink):
    queue = [source]
    paths = {source: []}
    if source == sink:
        return paths[source]
    while queue:
        u = queue.pop(0)
        for v in range(len(C)):
            if C[u][v] - F[u][v] > 0 and v not in paths:
                paths[v] = paths[u] + [(u, v)]
                if v == sink:
                    return paths[v]
                queue.append(v)
    return None

def ford_fulkerson(C, source, sink):
    n = len(C)
    F = [[0] * n for _ in range(n)]
    path = bfs(C, F, source, sink)
    while path:
        flow = min(C[u][v] - F[u][v] for u, v in path)
        for u, v in path:
            F[u][v] += flow
            F[v][u] -= flow
        path = bfs(C, F, source, sink)
    return sum(F[source][i] for i in range(n))

# Ejemplo de uso
C = [[0, 16, 13, 0, 0, 0],
     [0, 0, 10, 12, 0, 0],
     [0, 4, 0, 0, 14, 0],
     [0, 0, 9, 0, 0, 20],
     [0, 0, 0, 7, 0, 4],
     [0, 0, 0, 0, 0, 0]]
source = 0
sink = 5
print(f"El flujo máximo es: {ford_fulkerson(C, source, sink)}")

Algoritmo de Edmonds-Karp

El algoritmo de Edmonds-Karp es una implementación específica del algoritmo de Ford-Fulkerson que utiliza la búsqueda en anchura (BFS) para encontrar caminos aumentantes.

Diferencias Clave

  • Utiliza BFS en lugar de DFS para encontrar caminos aumentantes.
  • Garantiza un tiempo de ejecución polinomial de O(VE^2), donde V es el número de vértices y E es el número de aristas.

Ejemplo de Código

El código del algoritmo de Edmonds-Karp es similar al de Ford-Fulkerson, pero con BFS garantizado para la búsqueda de caminos aumentantes.

Comparación de Algoritmos

| Algoritmo | Método de Búsqueda | Complejidad Temporal | |-------------------|--------------------|----------------------| | Ford-Fulkerson | DFS/BFS | O(max_flow * E) | | Edmonds-Karp | BFS | O(VE^2) |

Aplicaciones Prácticas

  • Redes de Transporte: Optimización de rutas y flujos de tráfico.
  • Planificación de Proyectos: Asignación de recursos y tareas.
  • Logística: Optimización de flujos en cadenas de suministro.

Conclusión

Los algoritmos de flujo máximo son herramientas poderosas para resolver problemas de optimización en grafos. Comprender los conceptos básicos y los algoritmos principales como Ford-Fulkerson y Edmonds-Karp es esencial para aplicar estas técnicas en problemas del mundo real. Practicar con ejemplos y ejercicios ayudará a solidificar este conocimiento y a desarrollar habilidades para abordar problemas complejos de flujo en redes.
