Introducción

Un Árbol de Expansión Mínima (MST, por sus siglas en inglés) es un subgrafo de un grafo no dirigido y ponderado que conecta todos los vértices con el menor peso total posible. Los MST son fundamentales en diversas aplicaciones, como el diseño de redes, la optimización de rutas y la teoría de grafos.

Conceptos Clave

  • Grafo No Dirigido: Un conjunto de nodos conectados por aristas, donde las aristas no tienen una dirección específica.
  • Peso de la Arista: Un valor asignado a cada arista que representa el costo o la distancia entre dos nodos.
  • Subgrafo: Un grafo formado por un subconjunto de los nodos y aristas del grafo original.
  • Conectividad: Un grafo es conectado si existe un camino entre cualquier par de nodos.
  • Árbol: Un grafo conectado sin ciclos.

Algoritmos Principales

Algoritmo de Kruskal

El algoritmo de Kruskal es un enfoque de "greedy" que construye el MST seleccionando las aristas de menor peso y asegurando que no se formen ciclos.

Pasos del Algoritmo de Kruskal

  1. Ordenar todas las aristas del grafo en orden ascendente según su peso.
  2. Inicializar un subgrafo vacío.
  3. Iterar sobre las aristas ordenadas y agregar la arista al subgrafo si no forma un ciclo.
  4. Repetir hasta que el subgrafo tenga \(V-1\) aristas (donde \(V\) es el número de vértices).

Ejemplo de Código en Python

class Grafo:
    def __init__(self, vertices):
        self.V = vertices
        self.aristas = []

    def agregar_arista(self, u, v, w):
        self.aristas.append([u, v, w])

    def encontrar(self, parent, i):
        if parent[i] == i:
            return i
        return self.encontrar(parent, parent[i])

    def union(self, parent, rank, x, y):
        raiz_x = self.encontrar(parent, x)
        raiz_y = self.encontrar(parent, y)
        if rank[raiz_x] < rank[raiz_y]:
            parent[raiz_x] = raiz_y
        elif rank[raiz_x] > rank[raiz_y]:
            parent[raiz_y] = raiz_x
        else:
            parent[raiz_y] = raiz_x
            rank[raiz_x] += 1

    def kruskal(self):
        resultado = []
        i, e = 0, 0
        self.aristas = sorted(self.aristas, key=lambda item: item[2])
        parent = []
        rank = []
        for nodo in range(self.V):
            parent.append(nodo)
            rank.append(0)
        while e < self.V - 1:
            u, v, w = self.aristas[i]
            i = i + 1
            x = self.encontrar(parent, u)
            y = self.encontrar(parent, v)
            if x != y:
                e = e + 1
                resultado.append([u, v, w])
                self.union(parent, rank, x, y)
        return resultado

# Ejemplo de uso
g = Grafo(4)
g.agregar_arista(0, 1, 10)
g.agregar_arista(0, 2, 6)
g.agregar_arista(0, 3, 5)
g.agregar_arista(1, 3, 15)
g.agregar_arista(2, 3, 4)

mst = g.kruskal()
print("Aristas en el MST:")
for u, v, peso in mst:
    print(f"{u} -- {v} == {peso}")

Algoritmo de Prim

El algoritmo de Prim también es un enfoque de "greedy" que construye el MST seleccionando el vértice más cercano no incluido en el MST.

Pasos del Algoritmo de Prim

  1. Inicializar un conjunto vacío para los vértices incluidos en el MST.
  2. Inicializar un conjunto con un vértice arbitrario.
  3. Repetir hasta que todos los vértices estén en el MST:
    • Seleccionar la arista de menor peso que conecta un vértice en el MST con un vértice fuera del MST.
    • Agregar el vértice y la arista al MST.

Ejemplo de Código en Python

import heapq

def prim(grafo, inicio):
    mst = []
    visitados = set()
    min_heap = [(0, inicio, None)]
    while min_heap:
        peso, u, prev = heapq.heappop(min_heap)
        if u not in visitados:
            visitados.add(u)
            if prev is not None:
                mst.append((prev, u, peso))
            for v, peso in grafo[u]:
                if v not in visitados:
                    heapq.heappush(min_heap, (peso, v, u))
    return mst

# Ejemplo de uso
grafo = {
    0: [(1, 10), (2, 6), (3, 5)],
    1: [(0, 10), (3, 15)],
    2: [(0, 6), (3, 4)],
    3: [(0, 5), (1, 15), (2, 4)]
}

mst = prim(grafo, 0)
print("Aristas en el MST:")
for u, v, peso in mst:
    print(f"{u} -- {v} == {peso}")

Comparación de Algoritmos

| Algoritmo | Complejidad Temporal | Mejor para | |-----------|-----------------------|------------| | Kruskal | \(O(E \log E)\) | Grafos dispersos | | Prim | \(O(E + V \log V)\) | Grafos densos |

  • Kruskal: Es más eficiente en grafos dispersos debido a su enfoque en las aristas.
  • Prim: Es más eficiente en grafos densos debido a su enfoque en los vértices.

Conclusión

Los Árboles de Expansión Mínima son una herramienta esencial en la teoría de grafos y tienen aplicaciones prácticas en diversas áreas. Los algoritmos de Kruskal y Prim son dos métodos eficientes para encontrar el MST, cada uno con sus ventajas dependiendo de la estructura del grafo. Entender estos algoritmos y sus implementaciones es crucial para resolver problemas de optimización en redes y otras aplicaciones.
