La programación lineal es una técnica matemática utilizada para optimizar un objetivo lineal sujeto a un conjunto de restricciones lineales. Es una herramienta fundamental en la investigación de operaciones y se aplica en diversas áreas como la economía, la ingeniería, la logística y la gestión de recursos.

Conceptos Clave

  • Función Objetivo: Es la función lineal que se desea maximizar o minimizar.
  • Variables de Decisión: Son las variables que afectan la función objetivo y están sujetas a restricciones.
  • Restricciones: Son las limitaciones o condiciones que deben cumplirse, expresadas como ecuaciones o inecuaciones lineales.
  • Región Factible: Es el conjunto de todas las posibles soluciones que satisfacen las restricciones.
  • Solución Óptima: Es la solución dentro de la región factible que maximiza o minimiza la función objetivo.

Formulación de un Problema de Programación Lineal

Para formular un problema de programación lineal, se deben seguir los siguientes pasos:

  1. Definir las Variables de Decisión: Identificar las variables que afectan el resultado.
  2. Establecer la Función Objetivo: Determinar la función lineal que se desea optimizar.
  3. Formular las Restricciones: Especificar las limitaciones que deben cumplirse.

Ejemplo de Formulación

Supongamos que una empresa fabrica dos productos, A y B. Cada producto requiere diferentes cantidades de dos recursos, R1 y R2. La empresa desea maximizar sus ganancias.

  • Variables de Decisión:

    • \( x_1 \): Cantidad de producto A a fabricar.
    • \( x_2 \): Cantidad de producto B a fabricar.

  • Función Objetivo:

    • Maximizar \( Z = 40x_1 + 30x_2 \) (donde 40 y 30 son las ganancias por unidad de A y B, respectivamente).

  • Restricciones:

    • \( 2x_1 + 3x_2 \leq 120 \) (limitación de recurso R1).
    • \( x_1 + 2x_2 \leq 80 \) (limitación de recurso R2).
    • \( x_1, x_2 \geq 0 \) (no se pueden fabricar cantidades negativas).

Métodos de Solución

Método Gráfico

El método gráfico es útil para problemas con dos variables de decisión. Consiste en:

  1. Graficar las restricciones en un plano cartesiano.
  2. Identificar la región factible.
  3. Evaluar la función objetivo en los vértices de la región factible para encontrar la solución óptima.

Método Simplex

El método Simplex es un algoritmo iterativo utilizado para resolver problemas de programación lineal con más de dos variables. Los pasos básicos son:

  1. Convertir el problema en su forma estándar.
  2. Construir la tabla Simplex inicial.
  3. Iterar para mejorar la solución hasta alcanzar la óptima.

Ejemplo de Código: Método Simplex en Python

A continuación, se presenta un ejemplo de cómo resolver un problema de programación lineal utilizando la biblioteca scipy.optimize en Python.

from scipy.optimize import linprog

# Coeficientes de la función objetivo
c = [-40, -30]

# Coeficientes de las restricciones
A = [[2, 3], [1, 2]]
b = [120, 80]

# Resolver el problema de programación lineal
res = linprog(c, A_ub=A, b_ub=b, bounds=(0, None), method='simplex')

# Mostrar los resultados
print('Cantidad óptima de producto A:', res.x[0])
print('Cantidad óptima de producto B:', res.x[1])
print('Ganancia máxima:', -res.fun)

Conclusión

La programación lineal es una herramienta poderosa para la optimización en diversos campos. Comprender sus conceptos básicos y métodos de solución, como el método gráfico y el método Simplex, permite abordar problemas complejos de manera eficiente. La implementación en Python facilita la resolución de estos problemas mediante bibliotecas especializadas como scipy.optimize.
