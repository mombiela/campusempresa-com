Introducción

El clustering es una técnica de aprendizaje no supervisado que se utiliza para agrupar datos en subconjuntos o "clusters" de tal manera que los datos en el mismo cluster sean más similares entre sí que con los datos de otros clusters. Esta técnica es fundamental en diversas aplicaciones como segmentación de mercado, reconocimiento de patrones, análisis de imágenes y más.

Conceptos Clave

  • Cluster: Un grupo de datos que son similares entre sí.
  • Centroid: El centro de un cluster, utilizado en algoritmos como K-means.
  • Distancia: Medida de similitud entre dos puntos de datos, comúnmente utilizando la distancia euclidiana.
  • Algoritmo de Clustering: Método para agrupar datos en clusters.

Algoritmos de Clustering

K-means

K-means es uno de los algoritmos de clustering más populares y fáciles de entender. Funciona de la siguiente manera:

  1. Selecciona K puntos aleatorios como centroides iniciales.
  2. Asigna cada punto de datos al centroide más cercano.
  3. Recalcula los centroides como el promedio de todos los puntos asignados a cada centroide.
  4. Repite los pasos 2 y 3 hasta que los centroides no cambien significativamente.

Ejemplo de Código

from sklearn.cluster import KMeans
import numpy as np

# Datos de ejemplo
X = np.array([[1, 2], [1, 4], [1, 0], 
              [10, 2], [10, 4], [10, 0]])

# Crear el modelo K-means
kmeans = KMeans(n_clusters=2, random_state=0).fit(X)

# Predicciones
labels = kmeans.labels_
centroids = kmeans.cluster_centers_

print("Labels:", labels)
print("Centroids:", centroids)

DBSCAN

DBSCAN (Density-Based Spatial Clustering of Applications with Noise) es un algoritmo que agrupa puntos que están densamente empaquetados y marca como ruido los puntos que están en regiones de baja densidad.

  1. Selecciona un punto aleatorio.
  2. Encuentra todos los puntos en su vecindad (dentro de un radio ε).
  3. Si hay suficientes puntos en la vecindad, se forma un cluster.
  4. Repite para todos los puntos en el cluster.
  5. Marca los puntos que no pertenecen a ningún cluster como ruido.

Ejemplo de Código

from sklearn.cluster import DBSCAN
import numpy as np

# Datos de ejemplo
X = np.array([[1, 2], [2, 2], [2, 3], 
              [8, 7], [8, 8], [25, 80]])

# Crear el modelo DBSCAN
dbscan = DBSCAN(eps=3, min_samples=2).fit(X)

# Predicciones
labels = dbscan.labels_

print("Labels:", labels)

Comparación de Algoritmos

| Algoritmo | Ventajas | Desventajas | |-----------|----------|-------------| | K-means | Fácil de implementar, eficiente para grandes datasets | Sensible a la elección de K, no maneja bien clusters de formas irregulares | | DBSCAN | No requiere especificar el número de clusters, maneja bien el ruido y clusters de formas irregulares | Sensible a la elección de parámetros (ε y min_samples), no funciona bien con datos de alta dimensión |

Conclusión

Los algoritmos de clustering son herramientas poderosas en el análisis de datos. K-means y DBSCAN son dos de los algoritmos más utilizados, cada uno con sus propias ventajas y desventajas. La elección del algoritmo adecuado depende de la naturaleza de los datos y del problema específico que se desea resolver. Es importante comprender bien los conceptos y las técnicas para aplicar el clustering de manera efectiva en diferentes contextos.
