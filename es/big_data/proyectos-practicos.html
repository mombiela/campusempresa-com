Introducción

En esta sección del curso, nos enfocaremos en proyectos prácticos que te permitirán aplicar los conceptos y técnicas aprendidas en Big Data. Estos proyectos están diseñados para reforzar tu comprensión y habilidades en el manejo de grandes volúmenes de datos, utilizando herramientas y tecnologías populares en la industria.

Proyecto 1: Análisis de Sentimientos en Redes Sociales

Descripción del Proyecto

El objetivo de este proyecto es analizar los sentimientos expresados en publicaciones de redes sociales sobre un tema específico. Utilizaremos técnicas de procesamiento de lenguaje natural (NLP) y herramientas de Big Data para recopilar, procesar y analizar los datos.

Pasos del Proyecto

  1. Recopilación de Datos:

    • Utilizar la API de Twitter para recopilar tweets relacionados con un tema específico.
    • Almacenar los datos en un sistema de almacenamiento distribuido como Hadoop HDFS o Amazon S3.

  2. Procesamiento de Datos:

    • Limpiar y preprocesar los datos utilizando Apache Spark.
    • Tokenizar y normalizar el texto para facilitar el análisis.

  3. Análisis de Sentimientos:

    • Aplicar un modelo de análisis de sentimientos preentrenado (por ejemplo, VADER o TextBlob).
    • Clasificar los tweets en categorías de sentimientos (positivo, negativo, neutral).

  4. Visualización de Resultados:

    • Utilizar herramientas de visualización como Tableau o Matplotlib para presentar los resultados del análisis.

Ejemplo de Código

from pyspark.sql import SparkSession
from textblob import TextBlob

# Crear una sesión de Spark
spark = SparkSession.builder.appName("TwitterSentimentAnalysis").getOrCreate()

# Cargar datos de tweets desde HDFS
tweets_df = spark.read.json("hdfs://path/to/tweets.json")

# Función para analizar el sentimiento de un tweet
def analyze_sentiment(tweet):
    analysis = TextBlob(tweet)
    if analysis.sentiment.polarity > 0:
        return 'positive'
    elif analysis.sentiment.polarity == 0:
        return 'neutral'
    else:
        return 'negative'

# Aplicar la función de análisis de sentimientos a los tweets
tweets_df = tweets_df.withColumn("sentiment", analyze_sentiment(tweets_df.text))

# Mostrar los resultados
tweets_df.show()

Proyecto 2: Recomendador de Productos

Descripción del Proyecto

En este proyecto, desarrollaremos un sistema de recomendación de productos utilizando técnicas de filtrado colaborativo. El objetivo es recomendar productos a los usuarios basándonos en sus preferencias y comportamientos pasados.

Pasos del Proyecto

  1. Recopilación de Datos:

    • Obtener datos de transacciones de usuarios de una tienda en línea.
    • Almacenar los datos en un sistema de almacenamiento distribuido.

  2. Procesamiento de Datos:

    • Limpiar y preprocesar los datos utilizando Apache Spark.
    • Crear una matriz de usuario-producto que represente las interacciones.

  3. Construcción del Modelo de Recomendación:

    • Utilizar la biblioteca MLlib de Spark para construir un modelo de filtrado colaborativo.
    • Entrenar el modelo con los datos de transacciones.

  4. Generación de Recomendaciones:

    • Utilizar el modelo entrenado para generar recomendaciones personalizadas para cada usuario.
    • Evaluar la precisión del modelo utilizando métricas como RMSE.

Ejemplo de Código

from pyspark.sql import SparkSession
from pyspark.ml.recommendation import ALS

# Crear una sesión de Spark
spark = SparkSession.builder.appName("ProductRecommender").getOrCreate()

# Cargar datos de transacciones desde HDFS
transactions_df = spark.read.csv("hdfs://path/to/transactions.csv", header=True, inferSchema=True)

# Crear una matriz de usuario-producto
user_product_df = transactions_df.select("user_id", "product_id", "rating")

# Construir el modelo de recomendación utilizando ALS
als = ALS(userCol="user_id", itemCol="product_id", ratingCol="rating", coldStartStrategy="drop")
model = als.fit(user_product_df)

# Generar recomendaciones para todos los usuarios
user_recommendations = model.recommendForAllUsers(10)

# Mostrar las recomendaciones
user_recommendations.show()
