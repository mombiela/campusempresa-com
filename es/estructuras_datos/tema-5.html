Introducción

Las listas doblemente enlazadas son una estructura de datos fundamental en la programación y la informática. A diferencia de las listas simplemente enlazadas, en las que cada nodo apunta solo al siguiente nodo, en una lista doblemente enlazada, cada nodo contiene dos punteros: uno al nodo siguiente y otro al nodo anterior. Esto permite una navegación bidireccional a través de la lista, lo que puede ser útil en diversas aplicaciones.

Conceptos Clave

  • Nodo: La unidad básica de una lista doblemente enlazada. Cada nodo contiene un valor y dos punteros: uno al siguiente nodo y otro al nodo anterior.
  • Cabeza (Head): El primer nodo de la lista.
  • Cola (Tail): El último nodo de la lista.
  • Puntero Siguiente (Next): Un puntero que apunta al siguiente nodo en la lista.
  • Puntero Anterior (Prev): Un puntero que apunta al nodo anterior en la lista.

Estructura de un Nodo

En una lista doblemente enlazada, cada nodo tiene la siguiente estructura:

class Nodo:
    def __init__(self, valor):
        self.valor = valor
        self.siguiente = None  # Puntero al siguiente nodo
        self.anterior = None   # Puntero al nodo anterior

Operaciones Básicas

Inserción

La inserción en una lista doblemente enlazada puede realizarse en varias posiciones: al inicio, al final o en una posición específica.

Inserción al Inicio

def insertar_al_inicio(self, valor):
    nuevo_nodo = Nodo(valor)
    if self.cabeza is None:
        self.cabeza = self.cola = nuevo_nodo
    else:
        nuevo_nodo.siguiente = self.cabeza
        self.cabeza.anterior = nuevo_nodo
        self.cabeza = nuevo_nodo

Inserción al Final

def insertar_al_final(self, valor):
    nuevo_nodo = Nodo(valor)
    if self.cola is None:
        self.cabeza = self.cola = nuevo_nodo
    else:
        nuevo_nodo.anterior = self.cola
        self.cola.siguiente = nuevo_nodo
        self.cola = nuevo_nodo

Eliminación

La eliminación también puede realizarse en varias posiciones: al inicio, al final o en una posición específica.

Eliminación al Inicio

def eliminar_al_inicio(self):
    if self.cabeza is None:
        return
    if self.cabeza == self.cola:
        self.cabeza = self.cola = None
    else:
        self.cabeza = self.cabeza.siguiente
        self.cabeza.anterior = None

Eliminación al Final

def eliminar_al_final(self):
    if self.cola is None:
        return
    if self.cabeza == self.cola:
        self.cabeza = self.cola = None
    else:
        self.cola = self.cola.anterior
        self.cola.siguiente = None

Comparación con Listas Simplemente Enlazadas

| Característica | Lista Simplemente Enlazada | Lista Doblemente Enlazada | |-------------------------------|----------------------------|---------------------------| | Punteros por nodo | 1 | 2 | | Navegación | Unidireccional | Bidireccional | | Complejidad de inserción/eliminación | O(n) en el peor caso | O(1) en el mejor caso | | Uso de memoria | Menor | Mayor |

        self.anterior = None

            self.cola.siguiente = None

Conclusión

Las listas doblemente enlazadas son una estructura de datos versátil que permite una navegación bidireccional y operaciones eficientes de inserción y eliminación. Aunque consumen más memoria que las listas simplemente enlazadas debido a los punteros adicionales, su flexibilidad y eficiencia en ciertas operaciones las hacen una opción valiosa en muchas aplicaciones.
