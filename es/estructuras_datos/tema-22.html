Introducción

Un Árbol Binario de Búsqueda (ABB) es una estructura de datos jerárquica que facilita la búsqueda, inserción y eliminación de elementos. Cada nodo en un ABB tiene como máximo dos hijos, y se organiza de manera que para cualquier nodo dado, todos los valores en el subárbol izquierdo son menores y todos los valores en el subárbol derecho son mayores.

Conceptos Clave

  • Nodo: Elemento del árbol que contiene un valor y referencias a sus nodos hijos.
  • Raíz: Nodo superior del árbol.
  • Hijo Izquierdo: Nodo cuyo valor es menor que el nodo padre.
  • Hijo Derecho: Nodo cuyo valor es mayor que el nodo padre.
  • Hoja: Nodo que no tiene hijos.
  • Altura del Árbol: Longitud del camino más largo desde la raíz hasta una hoja.

Operaciones Básicas

Búsqueda

La búsqueda en un ABB se realiza comparando el valor buscado con el valor del nodo actual y moviéndose a la izquierda o derecha según corresponda.

class Nodo:
    def __init__(self, valor):
        self.valor = valor
        self.izquierdo = None
        self.derecho = None

def buscar(nodo, valor):
    if nodo is None or nodo.valor == valor:
        return nodo
    if valor < nodo.valor:
        return buscar(nodo.izquierdo, valor)
    else:
        return buscar(nodo.derecho, valor)

Inserción

Para insertar un nuevo valor, se sigue el mismo principio de comparación utilizado en la búsqueda hasta encontrar una posición vacía.

def insertar(nodo, valor):
    if nodo is None:
        return Nodo(valor)
    if valor < nodo.valor:
        nodo.izquierdo = insertar(nodo.izquierdo, valor)
    else:
        nodo.derecho = insertar(nodo.derecho, valor)
    return nodo

Eliminación

Eliminar un nodo en un ABB puede ser más complejo, ya que hay tres casos a considerar:

  1. El nodo es una hoja.
  2. El nodo tiene un solo hijo.
  3. El nodo tiene dos hijos.
def eliminar(nodo, valor):
    if nodo is None:
        return nodo
    if valor < nodo.valor:
        nodo.izquierdo = eliminar(nodo.izquierdo, valor)
    elif valor > nodo.valor:
        nodo.derecho = eliminar(nodo.derecho, valor)
    else:
        if nodo.izquierdo is None:
            return nodo.derecho
        elif nodo.derecho is None:
            return nodo.izquierdo
        temp = encontrar_minimo(nodo.derecho)
        nodo.valor = temp.valor
        nodo.derecho = eliminar(nodo.derecho, temp.valor)
    return nodo

def encontrar_minimo(nodo):
    actual = nodo
    while actual.izquierdo is not None:
        actual = actual.izquierdo
    return actual

Ejemplo Práctico

A continuación, se muestra un ejemplo de cómo construir un ABB y realizar operaciones básicas.

# Crear el árbol
raiz = Nodo(50)
insertar(raiz, 30)
insertar(raiz, 20)
insertar(raiz, 40)
insertar(raiz, 70)
insertar(raiz, 60)
insertar(raiz, 80)

# Buscar un valor
resultado = buscar(raiz, 40)
print("Valor encontrado:", resultado.valor if resultado else "No encontrado")

# Eliminar un valor
raiz = eliminar(raiz, 20)

Conclusión

Los Árboles Binarios de Búsqueda son una estructura fundamental en la informática que permite realizar operaciones eficientes de búsqueda, inserción y eliminación. Comprender su funcionamiento y cómo implementarlos es esencial para cualquier profesional de TI que trabaje con estructuras de datos y algoritmos.
