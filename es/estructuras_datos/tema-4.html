Introducción a las Listas Enlazadas

Las listas enlazadas son una estructura de datos fundamental que consiste en una secuencia de elementos, donde cada elemento apunta al siguiente. A diferencia de los arrays, las listas enlazadas no requieren un tamaño fijo y permiten una inserción y eliminación de elementos más eficiente.

  • Nodo: La unidad básica de una lista enlazada que contiene un valor y una referencia (o enlace) al siguiente nodo.
  • Cabeza (Head): El primer nodo de la lista enlazada.
  • Cola (Tail): El último nodo de la lista enlazada, que apunta a null o None.

Tipos de Listas Enlazadas

Lista Enlazada Simple

En una lista enlazada simple, cada nodo apunta al siguiente nodo en la secuencia.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        last_node = self.head
        while last_node.next:
            last_node = last_node.next
        last_node.next = new_node

    def print_list(self):
        current_node = self.head
        while current_node:
            print(current_node.data, end=" -> ")
            current_node = current_node.next
        print("None")

# Ejemplo de uso
ll = LinkedList()
ll.append(1)
ll.append(2)
ll.append(3)
ll.print_list()  # Output: 1 -> 2 -> 3 -> None

Lista Enlazada Doble

En una lista enlazada doble, cada nodo tiene dos referencias: una al siguiente nodo y otra al nodo anterior.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None
        self.prev = None

class DoublyLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            return
        last_node = self.head
        while last_node.next:
            last_node = last_node.next
        last_node.next = new_node
        new_node.prev = last_node

    def print_list(self):
        current_node = self.head
        while current_node:
            print(current_node.data, end=" <-> ")
            current_node = current_node.next
        print("None")

# Ejemplo de uso
dll = DoublyLinkedList()
dll.append(1)
dll.append(2)
dll.append(3)
dll.print_list()  # Output: 1 <-> 2 <-> 3 <-> None

Lista Enlazada Circular

En una lista enlazada circular, el último nodo apunta de nuevo al primer nodo, formando un ciclo.

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class CircularLinkedList:
    def __init__(self):
        self.head = None

    def append(self, data):
        new_node = Node(data)
        if not self.head:
            self.head = new_node
            new_node.next = self.head
            return
        last_node = self.head
        while last_node.next != self.head:
            last_node = last_node.next
        last_node.next = new_node
        new_node.next = self.head

    def print_list(self):
        current_node = self.head
        if not self.head:
            return
        while True:
            print(current_node.data, end=" -> ")
            current_node = current_node.next
            if current_node == self.head:
                break
        print("Head")

# Ejemplo de uso
cll = CircularLinkedList()
cll.append(1)
cll.append(2)
cll.append(3)
cll.print_list()  # Output: 1 -> 2 -> 3 -> Head

Operaciones Comunes

  • Inserción: Añadir un nuevo nodo a la lista.
  • Eliminación: Quitar un nodo existente de la lista.
  • Búsqueda: Encontrar un nodo con un valor específico.
  • Recorrido: Visitar cada nodo de la lista para realizar alguna operación.

Comparación de Listas Enlazadas

| Tipo de Lista | Ventajas | Desventajas | |---------------------|---------------------------------------------|----------------------------------------------| | Simple | Fácil de implementar, menos memoria usada | No se puede recorrer hacia atrás | | Doble | Recorrido bidireccional | Más memoria usada debido a referencias extra | | Circular | Recorrido continuo sin fin | Complejidad adicional en manejo de enlaces |

Conclusión

Las listas enlazadas son una estructura de datos versátil y eficiente para muchas aplicaciones. Comprender los diferentes tipos de listas enlazadas y sus operaciones básicas es crucial para cualquier profesional de TI. Practicar con ejemplos de código y ejercicios ayudará a consolidar este conocimiento y a aplicarlo en situaciones del mundo real.
