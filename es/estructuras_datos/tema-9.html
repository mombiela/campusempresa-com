Introducción

Una pila es una estructura de datos lineal que sigue el principio LIFO (Last In, First Out), lo que significa que el último elemento en entrar es el primero en salir. Este concepto es fundamental en la informática y se utiliza en diversas aplicaciones como la gestión de memoria, la evaluación de expresiones y la implementación de algoritmos.

Características de una Pila

  • LIFO (Last In, First Out): El último elemento insertado es el primero en ser removido.
  • Operaciones básicas:
    • Push: Añadir un elemento a la pila.
    • Pop: Remover el elemento superior de la pila.
    • Peek/Top: Obtener el elemento superior sin removerlo.
    • isEmpty: Verificar si la pila está vacía.
    • Size: Obtener el número de elementos en la pila.

Implementación de una Pila

Las pilas pueden ser implementadas de varias maneras, siendo las más comunes:

  • Usando Arrays: Una pila de tamaño fijo.
  • Usando Listas Enlazadas: Una pila de tamaño dinámico.

Implementación usando Arrays

class StackArray:
    def __init__(self, capacity):
        self.capacity = capacity
        self.stack = [None] * capacity
        self.top = -1

    def push(self, item):
        if self.top == self.capacity - 1:
            raise Exception("Stack Overflow")
        self.top += 1
        self.stack[self.top] = item

    def pop(self):
        if self.top == -1:
            raise Exception("Stack Underflow")
        item = self.stack[self.top]
        self.stack[self.top] = None
        self.top -= 1
        return item

    def peek(self):
        if self.top == -1:
            raise Exception("Stack is empty")
        return self.stack[self.top]

    def isEmpty(self):
        return self.top == -1

    def size(self):
        return self.top + 1

Implementación usando Listas Enlazadas

class Node:
    def __init__(self, value):
        self.value = value
        self.next = None

class StackLinkedList:
    def __init__(self):
        self.head = None
        self.count = 0

    def push(self, item):
        new_node = Node(item)
        new_node.next = self.head
        self.head = new_node
        self.count += 1

    def pop(self):
        if self.head is None:
            raise Exception("Stack Underflow")
        item = self.head.value
        self.head = self.head.next
        self.count -= 1
        return item

    def peek(self):
        if self.head is None:
            raise Exception("Stack is empty")
        return self.head.value

    def isEmpty(self):
        return self.head is None

    def size(self):
        return self.count

Comparación entre Implementaciones

Ejemplo Práctico

A continuación, se muestra un ejemplo práctico de cómo usar una pila para verificar si una secuencia de paréntesis está balanceada.

def is_balanced(expression):
    stack = StackArray(len(expression))
    for char in expression:
        if char in "({[":
            stack.push(char)
        elif char in ")}]":
            if stack.isEmpty():
                return False
            top_char = stack.pop()
            if not matches(top_char, char):
                return False
    return stack.isEmpty()

def matches(opening, closing):
    opens = "({["
    closes = ")}]"
    return opens.index(opening) == closes.index(closing)

# Ejemplo de uso
expression = "{[()]}"
print(is_balanced(expression))  # Output: True

Conclusión

Las pilas son una estructura de datos fundamental en la informática, con aplicaciones en diversas áreas como la gestión de memoria y la evaluación de expresiones. Comprender cómo implementar y utilizar pilas es esencial para cualquier profesional de TI. Las implementaciones pueden variar según las necesidades específicas, ya sea usando arrays para un tamaño fijo o listas enlazadas para un tamaño dinámico.
