Introducción

En esta sección, nos enfocaremos en ejercicios prácticos que te ayudarán a comprender y aplicar los conceptos de árboles en estructuras de datos. Los árboles son una estructura jerárquica que se utiliza en diversas aplicaciones, desde la representación de jerarquías hasta la optimización de búsquedas y la organización de datos.

Conceptos Clave

  • Nodo: Elemento básico de un árbol que contiene un valor o dato.
  • Raíz: El nodo superior de un árbol.
  • Hijo: Nodo descendiente de otro nodo.
  • Padre: Nodo que tiene uno o más nodos hijos.
  • Hoja: Nodo que no tiene hijos.
  • Altura: Longitud del camino más largo desde la raíz hasta una hoja.
  • Profundidad: Longitud del camino desde la raíz hasta un nodo específico.
  • Árbol Binario: Árbol en el que cada nodo tiene como máximo dos hijos.
  • Árbol de Búsqueda Binaria (BST): Árbol binario en el que para cada nodo, los valores del subárbol izquierdo son menores y los del subárbol derecho son mayores.

Ejercicio 1: Creación de un Árbol Binario

Descripción

En este ejercicio, crearás un árbol binario simple y realizarás operaciones básicas como inserción y recorrido.

Código de Ejemplo

class Nodo:
    def __init__(self, valor):
        self.valor = valor
        self.izquierdo = None
        self.derecho = None

class ArbolBinario:
    def __init__(self):
        self.raiz = None

    def insertar(self, valor):
        if self.raiz is None:
            self.raiz = Nodo(valor)
        else:
            self._insertar_recursivo(valor, self.raiz)

    def _insertar_recursivo(self, valor, nodo_actual):
        if valor < nodo_actual.valor:
            if nodo_actual.izquierdo is None:
                nodo_actual.izquierdo = Nodo(valor)
            else:
                self._insertar_recursivo(valor, nodo_actual.izquierdo)
        else:
            if nodo_actual.derecho is None:
                nodo_actual.derecho = Nodo(valor)
            else:
                self._insertar_recursivo(valor, nodo_actual.derecho)

    def recorrido_inorden(self, nodo_actual):
        if nodo_actual is not None:
            self.recorrido_inorden(nodo_actual.izquierdo)
            print(nodo_actual.valor, end=' ')
            self.recorrido_inorden(nodo_actual.derecho)

# Uso del árbol
arbol = ArbolBinario()
valores = [7, 3, 9, 1, 5, 8, 10]
for valor in valores:
    arbol.insertar(valor)

print("Recorrido Inorden:")
arbol.recorrido_inorden(arbol.raiz)

Explicación

  • Clase Nodo: Define un nodo con un valor y punteros a sus hijos izquierdo y derecho.
  • Clase ArbolBinario: Define el árbol binario con métodos para insertar valores y recorrer el árbol en orden.
  • Método insertar: Inserta un valor en el árbol, ubicándolo en la posición correcta.
  • Método recorrido_inorden: Realiza un recorrido en orden del árbol, imprimiendo los valores en orden ascendente.

Ejercicio 2: Búsqueda en un Árbol de Búsqueda Binaria (BST)

Descripción

En este ejercicio, implementarás un método para buscar un valor en un BST.

Código de Ejemplo

class ArbolBinarioBusqueda(ArbolBinario):
    def buscar(self, valor):
        return self._buscar_recursivo(valor, self.raiz)

    def _buscar_recursivo(self, valor, nodo_actual):
        if nodo_actual is None:
            return False
        elif valor == nodo_actual.valor:
            return True
        elif valor < nodo_actual.valor:
            return self._buscar_recursivo(valor, nodo_actual.izquierdo)
        else:
            return self._buscar_recursivo(valor, nodo_actual.derecho)

# Uso del árbol de búsqueda
arbol_bst = ArbolBinarioBusqueda()
valores = [7, 3, 9, 1, 5, 8, 10]
for valor in valores:
    arbol_bst.insertar(valor)

print("Buscar 5:", arbol_bst.buscar(5))  # Debería imprimir True
print("Buscar 6:", arbol_bst.buscar(6))  # Debería imprimir False

Explicación

  • Clase ArbolBinarioBusqueda: Hereda de ArbolBinario y añade un método para buscar valores.
  • Método buscar: Llama a un método recursivo para buscar un valor en el árbol.
  • Método _buscar_recursivo: Realiza la búsqueda recursiva, comparando el valor buscado con el valor del nodo actual y moviéndose a la izquierda o derecha según corresponda.

Conclusión
