Introducción

En esta sección del curso sobre Estructuras de Datos, nos enfocaremos en las listas, una de las estructuras de datos más fundamentales y versátiles en la programación. Las listas permiten almacenar colecciones de elementos, que pueden ser de cualquier tipo de dato, y ofrecen una variedad de operaciones útiles para manipular estos elementos.

Conceptos Clave

  • Definición de Lista: Una lista es una colección ordenada de elementos que pueden ser accedidos por su índice.
  • Operaciones Básicas: Inserción, eliminación, acceso y búsqueda de elementos.
  • Tipos de Listas: Listas enlazadas, listas dobles, listas circulares.

Ejercicios Prácticos

Ejercicio 1: Creación y Acceso a Elementos

Objetivo: Crear una lista y acceder a sus elementos.

Instrucciones:

  1. Crea una lista con los números del 1 al 5.
  2. Accede al primer y último elemento de la lista.
  3. Imprime todos los elementos de la lista.

Código de Ejemplo:

# Creación de la lista
mi_lista = [1, 2, 3, 4, 5]

# Acceso al primer y último elemento
primer_elemento = mi_lista[0]
ultimo_elemento = mi_lista[-1]

# Imprimir los elementos
print("Primer elemento:", primer_elemento)
print("Último elemento:", ultimo_elemento)
print("Todos los elementos:", mi_lista)

Ejercicio 2: Modificación de Elementos

Objetivo: Modificar elementos dentro de una lista.

Instrucciones:

  1. Cambia el segundo elemento de la lista a 10.
  2. Añade el número 6 al final de la lista.
  3. Elimina el tercer elemento de la lista.

Código de Ejemplo:

# Lista inicial
mi_lista = [1, 2, 3, 4, 5]

# Modificación del segundo elemento
mi_lista[1] = 10

# Añadir un nuevo elemento al final
mi_lista.append(6)

# Eliminar el tercer elemento
del mi_lista[2]

# Imprimir la lista modificada
print("Lista modificada:", mi_lista)

Ejercicio 3: Iteración y Suma de Elementos

Objetivo: Iterar sobre una lista y calcular la suma de sus elementos.

Instrucciones:

  1. Itera sobre cada elemento de la lista.
  2. Calcula la suma de todos los elementos.

Código de Ejemplo:

# Lista inicial
mi_lista = [1, 2, 3, 4, 5]

# Inicializar suma
suma = 0

# Iterar y sumar elementos
for elemento in mi_lista:
    suma += elemento

# Imprimir la suma
print("Suma de los elementos:", suma)

Ejercicio 4: Listas Anidadas

Objetivo: Trabajar con listas dentro de listas (listas anidadas).

Instrucciones:

  1. Crea una lista que contenga tres listas, cada una con tres números.
  2. Accede al segundo elemento de la tercera lista.
  3. Imprime todos los elementos de la lista anidada.

Código de Ejemplo:

# Creación de la lista anidada
lista_anidada = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Acceso al segundo elemento de la tercera lista
elemento = lista_anidada[2][1]

# Imprimir el elemento
print("Segundo elemento de la tercera lista:", elemento)

# Imprimir todos los elementos de la lista anidada
for sublista in lista_anidada:
    for item in sublista:
        print(item, end=' ')
    print()

Conclusión

Las listas son una estructura de datos esencial en la programación, ofreciendo flexibilidad y eficiencia para almacenar y manipular colecciones de elementos. A través de estos ejercicios prácticos, hemos explorado cómo crear, modificar, iterar y trabajar con listas anidadas. Comprender y dominar el uso de listas es fundamental para cualquier profesional de TI, ya que son ampliamente utilizadas en diversas aplicaciones y algoritmos.
