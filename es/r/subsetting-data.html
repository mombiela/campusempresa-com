Crear subconjuntos de datos es una habilidad fundamental en R que te permite extraer elementos específicos de estructuras de datos como vectores, matrices, data frames y listas. Este tema te guiará desde técnicas básicas hasta avanzadas de creación de subconjuntos.

Subconjuntos Básicos

Vectores

Los vectores son las estructuras de datos más simples en R. Puedes crear subconjuntos de vectores usando índices, vectores lógicos o nombres.

  • Usando Índices: Extrae elementos especificando sus posiciones.

    vec <- c(10, 20, 30, 40, 50)
vec[1]  # Extrae el primer elemento: 10
vec[3:5]  # Extrae elementos del 3º al 5º: 30, 40, 50

  • Usando Vectores Lógicos: Extrae elementos que cumplen una condición.

    vec <- c(10, 20, 30, 40, 50)
vec[vec > 25]  # Extrae elementos mayores a 25: 30, 40, 50

  • Usando Nombres: Extrae elementos por sus nombres.

    vec <- c(a = 10, b = 20, c = 30)
vec["b"]  # Extrae el elemento llamado 'b': 20

Matrices

Las matrices son arreglos bidimensionales. Puedes crear subconjuntos de matrices usando índices de filas y columnas.

  • Usando Índices: Extrae elementos especificando posiciones de filas y columnas. 
    mat <- matrix(1:9, nrow = 3)
mat[1, 2]  # Extrae el elemento en la 1ª fila, 2ª columna: 2
mat[1, ]  # Extrae toda la 1ª fila: 1, 2, 3
mat[, 2]  # Extrae toda la 2ª columna: 2, 5, 8

Data Frames

Los data frames son similares a las matrices pero pueden contener diferentes tipos de datos. Puedes crear subconjuntos de data frames usando índices de filas y columnas, nombres de columnas o vectores lógicos.

  • Usando Índices: Extrae elementos especificando posiciones de filas y columnas.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df[1, 2]  # Extrae el elemento en la 1ª fila, 2ª columna: 4
df[1, ]  # Extrae toda la 1ª fila
df[, 2]  # Extrae toda la 2ª columna

  • Usando Nombres de Columnas: Extrae columnas por sus nombres.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df["b"]  # Extrae la columna llamada 'b'
df[c("a", "c")]  # Extrae las columnas 'a' y 'c'

  • Usando Vectores Lógicos: Extrae filas que cumplen una condición.

    df <- data.frame(a = 1:3, b = 4:6, c = 7:9)
df[df$a > 1, ]  # Extrae filas donde la columna 'a' es mayor a 1

Subconjuntos Intermedios

Listas

Las listas pueden contener diferentes tipos de datos y estructuras. Puedes crear subconjuntos de listas usando índices o nombres.

  • Usando Índices: Extrae elementos especificando sus posiciones.

    lst <- list(a = 1:3, b = "hola", c = TRUE)
lst[[1]]  # Extrae el primer elemento: 1, 2, 3
lst[1]  # Extrae el primer elemento como una lista

  • Usando Nombres: Extrae elementos por sus nombres.

    lst <- list(a = 1:3, b = "hola", c = TRUE)
lst[["b"]]  # Extrae el elemento llamado 'b': "hola"
lst["b"]  # Extrae el elemento llamado 'b' como una lista

Data Frames Avanzados

Combinar múltiples técnicas de creación de subconjuntos puede ser poderoso.

  • Combinando Condiciones: Usa múltiples condiciones para crear subconjuntos de data frames.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df[df$a > 2 & df$b < 9, ]  # Extrae filas donde 'a' > 2 y 'b' < 9

  • Usando la Función subset: Una manera más legible de crear subconjuntos de data frames.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
subset(df, a > 2 & b < 9)  # Extrae filas donde 'a' > 2 y 'b' < 9

Subconjuntos Avanzados

Usando `dplyr` para Data Frames

El paquete dplyr proporciona una manera más legible y eficiente de crear subconjuntos de data frames.

  • Usando filter: Extrae filas que cumplen una condición.

    library(dplyr)
df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% filter(a > 2 & b < 9)  # Extrae filas donde 'a' > 2 y 'b' < 9

  • Usando select: Extrae columnas específicas.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% select(a, c)  # Extrae las columnas 'a' y 'c'

  • Combinando filter y select: Combina múltiples operaciones.

    df <- data.frame(a = 1:5, b = 6:10, c = 11:15)
df %>% filter(a > 2) %>% select(a, c)  # Extrae filas donde 'a' > 2 y columnas 'a' y 'c'

Usando `data.table` para Conjuntos de Datos Grandes

El paquete data.table está optimizado para conjuntos de datos grandes y proporciona una creación de subconjuntos eficiente.

  • Subconjuntos Básicos: Extrae filas y columnas.

    library(data.table)
dt <- data.table(a = 1:5, b = 6:10, c = 11:15)
dt[a > 2, .(a, c)]  # Extrae filas donde 'a' > 2 y columnas 'a' y 'c'

  • Subconjuntos Avanzados: Usa condiciones y operaciones complejas.

    dt <- data.table(a = 1:5, b = 6:10, c = 11:15)
dt[a > 2 & b < 9, .(sum_a = sum(a), mean_c = mean(c))]  # Extrae filas donde 'a' > 2 y 'b' < 9, luego calcula la suma de 'a' y la media de 'c'

Conclusión

Crear subconjuntos de datos es una habilidad crucial en R que te permite manipular y analizar datos de manera eficiente. Comenzando con técnicas básicas con vectores y matrices, puedes avanzar a operaciones más complejas con data frames y listas. Utilizar paquetes como dplyr y data.table puede mejorar aún más tu capacidad para manejar conjuntos de datos grandes y complejos. Practica estas técnicas para volverte competente en la manipulación de datos en R.
