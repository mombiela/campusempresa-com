Introducción a plotly

Plotly es una poderosa biblioteca para crear visualizaciones interactivas en R. Permite a los usuarios crear una amplia gama de gráficos y diagramas que se pueden incrustar en aplicaciones web, informes y paneles de control.

  • Conceptos Clave:
    • Visualizaciones Interactivas vs. Estáticas
    • Visión general de la biblioteca plotly
    • Instalación y configuración
# Instalar y cargar plotly
install.packages("plotly")
library(plotly)

Gráficos Básicos de Plotly

Aprende a crear gráficos básicos usando plotly, como gráficos de dispersión, gráficos de líneas y gráficos de barras.

  • Gráfico de Dispersión:
# Gráfico de Dispersión Básico
data <- data.frame(x = rnorm(50), y = rnorm(50))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers')
  • Gráfico de Líneas:
# Gráfico de Líneas Básico
data <- data.frame(x = 1:10, y = cumsum(rnorm(10)))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'lines')
  • Gráfico de Barras:
# Gráfico de Barras Básico
data <- data.frame(x = c('A', 'B', 'C'), y = c(10, 20, 30))
plot_ly(data, x = ~x, y = ~y, type = 'bar')

Personalización de Visualizaciones

Aprende a personalizar tus visualizaciones modificando colores, agregando títulos y ajustando ejes.

  • Personalización de Colores:
# Personalización de Colores en Gráfico de Dispersión
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers', marker = list(color = 'red'))
  • Agregar Títulos y Etiquetas:
# Agregar Títulos y Etiquetas
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers') %>%
  layout(title = 'Ejemplo de Gráfico de Dispersión', xaxis = list(title = 'Eje X'), yaxis = list(title = 'Eje Y'))
  • Ajuste de Ejes:
# Ajuste de Ejes
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers') %>%
  layout(xaxis = list(range = c(-3, 3)), yaxis = list(range = c(-3, 3)))

Características Avanzadas de Plotly

Explora características avanzadas como subgráficos, gráficos 3D y animaciones.

  • Subgráficos:
# Creación de Subgráficos
subplot(
  plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers'),
  plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'lines')
)
  • Gráficos 3D:
# Gráfico de Dispersión 3D
data <- data.frame(x = rnorm(50), y = rnorm(50), z = rnorm(50))
plot_ly(data, x = ~x, y = ~y, z = ~z, type = 'scatter3d', mode = 'markers')
  • Animaciones:
# Gráfico de Dispersión Animado
data <- data.frame(x = rnorm(50), y = rnorm(50), frame = rep(1:5, each = 10))
plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers', frame = ~frame)

Integración de Plotly con Shiny

Aprende cómo integrar visualizaciones de plotly en aplicaciones Shiny para aplicaciones web interactivas.

  • Aplicación Shiny Básica con Plotly:
# Aplicación Shiny con Plotly
library(shiny)
library(plotly)

ui <- fluidPage(
  plotlyOutput("plot")
)

server <- function(input, output) {
  output$plot <- renderPlotly({
    plot_ly(data, x = ~x, y = ~y, type = 'scatter', mode = 'markers')
  })
}

shinyApp(ui, server)

Conclusión

Plotly es una herramienta versátil y poderosa para crear visualizaciones interactivas en R. Al dominar los conceptos básicos y explorar características avanzadas, puedes crear visualizaciones atractivas y dinámicas para una variedad de aplicaciones.

  • Resumen:
    • Plotly ofrece una amplia gama de tipos de gráficos y opciones de personalización.
    • Las características avanzadas como subgráficos, gráficos 3D y animaciones pueden mejorar tus visualizaciones.
    • La integración con Shiny permite la creación de aplicaciones web interactivas.

Siguiendo este curso, adquirirás las habilidades necesarias para crear y personalizar visualizaciones interactivas usando plotly en R, desde gráficos básicos hasta aplicaciones interactivas avanzadas.
