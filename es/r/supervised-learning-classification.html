Introducción al Aprendizaje Supervisado

El aprendizaje supervisado es un tipo de aprendizaje automático donde el modelo se entrena con datos etiquetados. El objetivo es aprender una correspondencia de entradas a salidas basándose en las etiquetas proporcionadas.

  • Datos Etiquetados: Datos que incluyen tanto características de entrada como las etiquetas de salida correspondientes.
  • Fase de Entrenamiento: El modelo aprende de los datos etiquetados.
  • Fase de Predicción: El modelo predice etiquetas para nuevos datos no vistos.

¿Qué es la Clasificación?

La clasificación es un tipo de aprendizaje supervisado donde la variable de salida es categórica. El objetivo es predecir la categoría o clase de nuevas observaciones basándose en observaciones pasadas.

  • Clasificación Binaria: Dos posibles clases (por ejemplo, spam o no spam).
  • Clasificación Multiclase: Más de dos clases (por ejemplo, clasificar tipos de animales).

Conceptos Clave en Clasificación

  • Características: Las variables de entrada utilizadas para hacer predicciones.
  • Etiquetas: Las categorías de salida que queremos predecir.
  • Conjunto de Entrenamiento: El subconjunto de datos utilizado para entrenar el modelo.
  • Conjunto de Prueba: El subconjunto de datos utilizado para evaluar el rendimiento del modelo.

Algoritmos Comunes de Clasificación

  • Regresión Logística
  • k-Nearest Neighbors (k-NN)
  • Árboles de Decisión
  • Bosques Aleatorios
  • Máquinas de Vectores de Soporte (SVM)
  • Naive Bayes

Implementación de Clasificación en R

Regresión Logística

La regresión logística se utiliza para problemas de clasificación binaria. Modela la probabilidad de que una entrada dada pertenezca a una clase particular.

# Cargar la biblioteca necesaria
library(caret)

# Cargar el conjunto de datos
data(iris)
# Usar solo dos clases para clasificación binaria
iris_binary <- subset(iris, Species != "setosa")

# Dividir los datos en conjuntos de entrenamiento y prueba
set.seed(123)
trainIndex <- createDataPartition(iris_binary$Species, p = .8, 
                                  list = FALSE, 
                                  times = 1)
irisTrain <- iris_binary[ trainIndex,]
irisTest  <- iris_binary[-trainIndex,]

# Entrenar el modelo de regresión logística
model <- train(Species ~ ., data = irisTrain, method = "glm", family = "binomial")

# Hacer predicciones
predictions <- predict(model, irisTest)

# Evaluar el modelo
confusionMatrix(predictions, irisTest$Species)

k-Nearest Neighbors (k-NN)

k-NN es un algoritmo simple basado en instancias que clasifica una nueva muestra basándose en la clase mayoritaria entre sus k vecinos más cercanos.

# Cargar la biblioteca necesaria
library(class)

# Preparar los datos
trainData <- irisTrain[, -5]
trainLabels <- irisTrain$Species
testData <- irisTest[, -5]
testLabels <- irisTest$Species

# Entrenar y predecir usando k-NN
k <- 3
predictions <- knn(trainData, testData, trainLabels, k = k)

# Evaluar el modelo
confusionMatrix(predictions, testLabels)

Árboles de Decisión

Los árboles de decisión clasifican datos dividiéndolos en subconjuntos basados en el valor de las características de entrada.

# Cargar la biblioteca necesaria
library(rpart)

# Entrenar el modelo de árbol de decisión
model <- rpart(Species ~ ., data = irisTrain, method = "class")

# Hacer predicciones
predictions <- predict(model, irisTest, type = "class")

# Evaluar el modelo
confusionMatrix(predictions, irisTest$Species)

Métricas de Evaluación del Modelo

  • Precisión: La proporción de instancias clasificadas correctamente.
  • Precisión: La proporción de instancias verdaderamente positivas entre las instancias clasificadas como positivas.
  • Recall (Sensibilidad): La proporción de instancias verdaderamente positivas entre todas las instancias realmente positivas.
  • F1 Score: La media armónica de precisión y recall.

| Métrica | Fórmula | Interpretación | |------------|----------------------------------------------|-----------------------------------------------------| | Precisión | (TP + TN) / (TP + TN + FP + FN) | Corrección general del modelo | | Precisión | TP / (TP + FP) | Corrección de las predicciones positivas | | Recall | TP / (TP + FN) | Capacidad para encontrar todas las instancias positivas | | F1 Score | 2 * (Precisión * Recall) / (Precisión + Recall) | Equilibrio entre precisión y recall |

Conclusión

El aprendizaje supervisado, específicamente la clasificación, es una herramienta poderosa para el modelado predictivo. Al comprender e implementar varios algoritmos de clasificación en R, puedes construir modelos para predecir resultados categóricos. Recuerda evaluar tus modelos utilizando métricas apropiadas para asegurar su efectividad.
