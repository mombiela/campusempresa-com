Introducción

La importación y exportación de datos son habilidades fundamentales en la programación en R, permitiéndote leer datos de diversas fuentes y guardar tus resultados. Esta sección te guiará desde las técnicas básicas hasta las avanzadas de importación y exportación de datos en R.

Importación de Datos

Lectura de Datos desde Archivos CSV

Los archivos CSV (Valores Separados por Comas) son uno de los formatos más comunes para el almacenamiento de datos.

  • Importación Básica de CSV:

    data <- read.csv("ruta/a/tu/archivo.csv")
    • read.csv() es una función que lee un archivo en formato de tabla y crea un data frame a partir de él.
    • "ruta/a/tu/archivo.csv" es la ruta a tu archivo CSV.

  • Especificando Delimitadores:

    data <- read.csv("ruta/a/tu/archivo.csv", sep = ";")
    • El parámetro sep te permite especificar el delimitador usado en tu archivo.

Lectura de Datos desde Archivos de Excel

Los archivos de Excel son ampliamente utilizados en el análisis de datos. R proporciona varios paquetes para leer archivos de Excel, como readxl y openxlsx.

  • Usando el Paquete readxl: 
    library(readxl)
data <- read_excel("ruta/a/tu/archivo.xlsx", sheet = "Sheet1")
    • La función read_excel() lee un archivo de Excel.
    • El parámetro sheet especifica la hoja de la cual leer.

Lectura de Datos desde Bases de Datos

R puede conectarse a varias bases de datos usando paquetes como DBI y RSQLite.

  • Conexión a una Base de Datos SQLite: 
    library(DBI)
con <- dbConnect(RSQLite::SQLite(), "ruta/a/tu/base_de_datos.sqlite")
data <- dbGetQuery(con, "SELECT * FROM tu_tabla")
dbDisconnect(con)
    • dbConnect() establece una conexión con la base de datos.
    • dbGetQuery() ejecuta una consulta SQL y devuelve el resultado como un data frame.
    • dbDisconnect() cierra la conexión.

Lectura de Datos desde la Web

R también puede leer datos directamente desde la web usando funciones como read.csv() para archivos CSV alojados en línea.

  • Lectura de CSV desde URL: 
    url <- "http://example.com/data.csv"
data <- read.csv(url)

Exportación de Datos

Escritura de Datos en Archivos CSV

Exportar datos a CSV es sencillo con la función write.csv().

  • Exportación Básica de CSV: 
    write.csv(data, "ruta/a/tu/salida.csv")
    • write.csv() escribe un data frame en un archivo CSV.
    • "ruta/a/tu/salida.csv" es la ruta donde se guardará el archivo.

Escritura de Datos en Archivos de Excel

Para escribir datos en archivos de Excel, puedes usar los paquetes writexl o openxlsx.

  • Usando el Paquete writexl: 
    library(writexl)
write_xlsx(data, "ruta/a/tu/salida.xlsx")
    • write_xlsx() escribe un data frame en un archivo de Excel.

Escritura de Datos en Bases de Datos

También puedes exportar datos a bases de datos usando el paquete DBI.

  • Escritura en una Base de Datos SQLite: 
    library(DBI)
con <- dbConnect(RSQLite::SQLite(), "ruta/a/tu/base_de_datos.sqlite")
dbWriteTable(con, "tu_tabla", data)
dbDisconnect(con)
    • dbWriteTable() escribe un data frame en una tabla de la base de datos.

Escritura de Datos en la Web

Aunque R no soporta directamente la escritura de datos en ubicaciones web, puedes usar APIs u otros servicios web para subir datos.

Resumen

En esta sección, cubrimos las técnicas esenciales para la importación y exportación de datos en R:

  • Importación de Datos:

    • Lectura desde archivos CSV usando read.csv().
    • Lectura desde archivos de Excel usando el paquete readxl.
    • Lectura desde bases de datos usando DBI y RSQLite.
    • Lectura desde la web usando URLs.

  • Exportación de Datos:

    • Escritura en archivos CSV usando write.csv().
    • Escritura en archivos de Excel usando el paquete writexl.
    • Escritura en bases de datos usando DBI y RSQLite.

Dominar estas habilidades te permitirá manejar datos de diversas fuentes y formatos, haciendo tu flujo de trabajo de análisis de datos más eficiente y versátil.
