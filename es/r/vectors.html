Vectores en R
Matrices ►

Los vectores son una de las estructuras de datos más fundamentales en R. Son esenciales para almacenar y manipular datos. Esta sección cubrirá todo, desde los conceptos básicos de los vectores hasta operaciones avanzadas.

Introducción a los Vectores

  • Definición: Un vector es una secuencia de elementos de datos del mismo tipo básico. En R, los vectores pueden ser numéricos, de caracteres, lógicos o complejos.
  • Creación: Los vectores se pueden crear usando la función c().

Creación de Vectores

# Vector numérico
numeric_vector <- c(1, 2, 3, 4, 5)
print(numeric_vector)

# Vector de caracteres
char_vector <- c("manzana", "plátano", "cereza")
print(char_vector)

# Vector lógico
logical_vector <- c(TRUE, FALSE, TRUE)
print(logical_vector)

Acceso a Elementos

  • Indexación: Los elementos en un vector se pueden acceder usando corchetes [].
# Acceso al primer elemento de un vector numérico
first_element <- numeric_vector[1]
print(first_element)

# Acceso a múltiples elementos
subset_vector <- numeric_vector[1:3]
print(subset_vector)

Operaciones con Vectores

  • Operaciones Aritméticas: Puedes realizar operaciones aritméticas en vectores elemento por elemento.
# Sumar dos vectores
vector1 <- c(1, 2, 3)
vector2 <- c(4, 5, 6)
sum_vector <- vector1 + vector2
print(sum_vector)

# Multiplicar un vector por un escalar
scalar_mult <- vector1 * 2
print(scalar_mult)
  • Operaciones Lógicas: Las operaciones lógicas también se pueden realizar elemento por elemento.
# Comparación lógica
logical_result <- vector1 > 2
print(logical_result)

Operaciones Avanzadas con Vectores

Funciones de Vectores

  • Funciones Comunes: R proporciona varias funciones integradas para trabajar con vectores.
# Suma de elementos
sum_vector <- sum(numeric_vector)
print(sum_vector)

# Media de elementos
mean_vector <- mean(numeric_vector)
print(mean_vector)

# Ordenar un vector
sorted_vector <- sort(numeric_vector)
print(sorted_vector)

Reciclaje de Vectores

  • Regla de Reciclaje: Al realizar operaciones en vectores de diferentes longitudes, R recicla el vector más corto para que coincida con la longitud del más largo.
# Ejemplo de reciclaje
short_vector <- c(1, 2)
long_vector <- c(10, 20, 30, 40)
recycled_sum <- short_vector + long_vector
print(recycled_sum)

Vectores Nombrados

  • Nombrar Elementos: Puedes asignar nombres a los elementos de un vector para una mejor legibilidad.
# Creación de un vector nombrado
named_vector <- c(a = 1, b = 2, c = 3)
print(named_vector)

# Acceso a elementos por nombre
element_b <- named_vector["b"]
print(element_b)

Conclusión

Los vectores son una estructura de datos versátil y esencial en R. Permiten un almacenamiento y manipulación eficientes de datos. Entender cómo crear, acceder y realizar operaciones en vectores es crucial para cualquier programador de R. A medida que avances, dominar las funciones de vectores, el reciclaje y la nomenclatura te permitirá escribir código más eficiente y legible.
