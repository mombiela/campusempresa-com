Entender los tipos de datos es fundamental para programar en R. Esta sección cubrirá los principales tipos de datos en R, cómo usarlos y su importancia en el análisis de datos.

Tipos de Datos Básicos

R tiene varios tipos de datos básicos que son esenciales para cualquier programador de R. Estos incluyen:

  • Numérico
  • Entero
  • Carácter
  • Lógico
  • Complejo

Numérico

Los tipos de datos numéricos se utilizan para representar números reales.

# Ejemplo de Numérico
num <- 42.5
print(num)  # Salida: 42.5

Entero

Los enteros son números enteros. En R, puedes especificar un entero añadiendo una L al número.

# Ejemplo de Entero
int <- 42L
print(int)  # Salida: 42

Carácter

Los tipos de datos de carácter se utilizan para representar valores de texto o cadenas.

# Ejemplo de Carácter
char <- "¡Hola, R!"
print(char)  # Salida: "¡Hola, R!"

Lógico

Los tipos de datos lógicos representan valores booleanos: TRUE o FALSE.

# Ejemplo de Lógico
logi <- TRUE
print(logi)  # Salida: TRUE

Complejo

Los números complejos tienen una parte real y una parte imaginaria.

# Ejemplo de Complejo
comp <- 3 + 2i
print(comp)  # Salida: 3+2i

Estructuras de Datos

R también proporciona varias estructuras de datos para almacenar colecciones de datos. Estas incluyen:

  • Vectores
  • Matrices
  • Arreglos
  • Data Frames
  • Listas

Vectores

Los vectores son la estructura de datos más básica en R y pueden contener elementos del mismo tipo.

# Ejemplo de Vector
vec <- c(1, 2, 3, 4, 5)
print(vec)  # Salida: 1 2 3 4 5

Matrices

Las matrices son estructuras de datos homogéneas bidimensionales.

# Ejemplo de Matriz
mat <- matrix(1:9, nrow = 3, ncol = 3)
print(mat)
# Salida:
#      [,1] [,2] [,3]
# [1,]    1    4    7
# [2,]    2    5    8
# [3,]    3    6    9

Arreglos

Los arreglos son estructuras de datos homogéneas multidimensionales.

# Ejemplo de Arreglo
arr <- array(1:8, dim = c(2, 2, 2))
print(arr)
# Salida:
# , , 1
#      [,1] [,2]
# [1,]    1    3
# [2,]    2    4
#
# , , 2
#      [,1] [,2]
# [1,]    5    7
# [2,]    6    8

Data Frames

Los data frames son estructuras de datos heterogéneas bidimensionales, similares a tablas en una base de datos o una hoja de cálculo de Excel.

# Ejemplo de Data Frame
df <- data.frame(
  Nombre = c("John", "Jane", "Doe"),
  Edad = c(23, 25, 31),
  Género = c("Masculino", "Femenino", "Masculino")
)
print(df)
# Salida:
#   Nombre Edad    Género
# 1   John   23 Masculino
# 2   Jane   25  Femenino
# 3    Doe   31 Masculino

Listas

Las listas pueden contener elementos de diferentes tipos y estructuras.

# Ejemplo de Lista
lst <- list(
  Nombre = "John",
  Edad = 23,
  Puntuaciones = c(90, 85, 88)
)
print(lst)
# Salida:
# $Nombre
# [1] "John"
#
# $Edad
# [1] 23
#
# $Puntuaciones
# [1] 90 85 88

Resumen

En esta sección, cubrimos los tipos de datos básicos en R, incluyendo numérico, entero, carácter, lógico y complejo. También exploramos varias estructuras de datos como vectores, matrices, arreglos, data frames y listas. Entender estos tipos de datos y estructuras es crucial para una manipulación y análisis de datos efectivos en R. A medida que avances en tu viaje de programación en R, estos conceptos formarán la base para temas y técnicas más avanzados.
