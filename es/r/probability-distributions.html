Introducción

Las distribuciones de probabilidad son fundamentales en estadística y análisis de datos. En R, hay numerosas funciones y paquetes disponibles para trabajar con varias distribuciones de probabilidad. Esta sección te guiará desde los conceptos básicos hasta los avanzados de las distribuciones de probabilidad en R.

Conceptos Clave

  • Distribución de Probabilidad: Una función que describe la probabilidad de obtener los valores posibles que una variable aleatoria puede tomar.
  • Variable Aleatoria: Una variable cuyos valores dependen de los resultados de un fenómeno aleatorio.
  • Distribuciones Discretas vs. Continuas: Las distribuciones discretas tratan con resultados contables, mientras que las distribuciones continuas tratan con resultados que pueden tomar cualquier valor dentro de un rango.

Trabajando con Distribuciones en R

Distribución Binomial

La distribución binomial modela el número de éxitos en un número fijo de ensayos de Bernoulli independientes.

  • Funciones:
    • dbinom(x, size, prob): Función de masa de probabilidad.
    • pbinom(q, size, prob): Función de distribución acumulativa.
    • qbinom(p, size, prob): Función de cuantiles.
    • rbinom(n, size, prob): Generación aleatoria.
# Ejemplo: Distribución Binomial
# Parámetros: size = 10 ensayos, prob = 0.5 probabilidad de éxito
size <- 10
prob <- 0.5

# Probabilidad de exactamente 5 éxitos
dbinom(5, size, prob)

# Probabilidad acumulada de 5 o menos éxitos
pbinom(5, size, prob)

# Cuantil para 0.5 de probabilidad
qbinom(0.5, size, prob)

# Generar 10 muestras aleatorias
rbinom(10, size, prob)

Distribución Normal

La distribución normal es una distribución de probabilidad continua caracterizada por su curva en forma de campana.

  • Funciones:
    • dnorm(x, mean, sd): Función de densidad de probabilidad.
    • pnorm(q, mean, sd): Función de distribución acumulativa.
    • qnorm(p, mean, sd): Función de cuantiles.
    • rnorm(n, mean, sd): Generación aleatoria.
# Ejemplo: Distribución Normal
# Parámetros: mean = 0, sd = 1
mean <- 0
sd <- 1

# Densidad en x = 0
dnorm(0, mean, sd)

# Probabilidad acumulada hasta x = 1
pnorm(1, mean, sd)

# Cuantil para 0.95 de probabilidad
qnorm(0.95, mean, sd)

# Generar 10 muestras aleatorias
rnorm(10, mean, sd)

Distribución de Poisson

La distribución de Poisson modela el número de eventos que ocurren dentro de un intervalo fijo de tiempo o espacio.

  • Funciones:
    • dpois(x, lambda): Función de masa de probabilidad.
    • ppois(q, lambda): Función de distribución acumulativa.
    • qpois(p, lambda): Función de cuantiles.
    • rpois(n, lambda): Generación aleatoria.
# Ejemplo: Distribución de Poisson
# Parámetro: lambda = 3 (tasa promedio de ocurrencia)
lambda <- 3

# Probabilidad de exactamente 2 eventos
dpois(2, lambda)

# Probabilidad acumulada de 2 o menos eventos
ppois(2, lambda)

# Cuantil para 0.8 de probabilidad
qpois(0.8, lambda)

# Generar 10 muestras aleatorias
rpois(10, lambda)

Temas Avanzados

Ajuste de Distribuciones a Datos

R proporciona funciones para ajustar distribuciones a datos, permitiendo la estimación de parámetros y pruebas de bondad de ajuste.

  • Ejemplo: Ajuste de una distribución normal a datos
# Generar datos de muestra
data <- rnorm(100, mean = 5, sd = 2)

# Ajustar una distribución normal
fit <- fitdistr(data, "normal")

# Mostrar los parámetros del ajuste
fit

Distribuciones Personalizadas

Puedes definir distribuciones de probabilidad personalizadas usando funciones de R.

  • Ejemplo: Función de densidad de probabilidad personalizada
# Función de densidad personalizada
custom_density <- function(x) {
  ifelse(x >= 0 & x <= 1, 2 * x, 0)
}

# Integrar para verificar si es una función de densidad válida
integrate(custom_density, -Inf, Inf)

Resumen

En esta sección, cubrimos los conceptos básicos de las distribuciones de probabilidad en R, incluyendo distribuciones discretas y continuas comunes. Exploramos cómo usar funciones de R para trabajar con estas distribuciones y tocamos temas avanzados como el ajuste de distribuciones a datos y la creación de distribuciones personalizadas. Entender estos conceptos es crucial para el análisis estadístico y las aplicaciones de ciencia de datos.
