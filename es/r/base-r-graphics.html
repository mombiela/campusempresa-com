Introducción a los Gráficos Base R

Los gráficos base R proporcionan un sistema poderoso y flexible para crear una amplia variedad de gráficos y visualizaciones directamente en R. Esta sección te introducirá a los conceptos y funciones fundamentales utilizados en los gráficos base R.

  • Conceptos Clave:
    • Dispositivos Gráficos: El medio donde se muestra el gráfico (por ejemplo, pantalla, archivo).
    • Funciones de Gráficos: Funciones utilizadas para crear diferentes tipos de gráficos.
    • Personalización de Gráficos: Modificar la apariencia de los gráficos usando varios parámetros.

Funciones Básicas de Gráficos

Base R ofrece varias funciones para crear gráficos básicos. Estas funciones son fáciles de usar y proporcionan un buen punto de partida para principiantes.

  • Creación de un Gráfico de Dispersión Simple:

    # Generar algunos datos de muestra
x <- 1:10
y <- x^2

# Crear un gráfico de dispersión
plot(x, y, main="Gráfico de Dispersión", xlab="Eje X", ylab="Eje Y")

    Explicación:

    • plot(x, y): Crea un gráfico de dispersión con valores x y y.
    • main, xlab, ylab: Añaden un título y etiquetas a los ejes.

  • Gráfico de Líneas:

    # Crear un gráfico de líneas
plot(x, y, type="l", main="Gráfico de Líneas", xlab="Eje X", ylab="Eje Y")

    Explicación:

    • type="l": Especifica que el gráfico debe ser un gráfico de líneas.

  • Gráfico de Barras:

    # Crear un gráfico de barras
barplot(y, names.arg=x, main="Gráfico de Barras", xlab="Eje X", ylab="Eje Y")

    Explicación:

    • barplot(y): Crea un gráfico de barras con valores y.
    • names.arg=x: Etiqueta las barras con valores x.

Personalización de Gráficos

Personalizar gráficos te permite mejorar la legibilidad y la estética de tus visualizaciones.

  • Cambiar Colores y Símbolos de Gráficos:

    # Gráfico de dispersión con colores y símbolos personalizados
plot(x, y, col="blue", pch=19, main="Gráfico de Dispersión Personalizado", xlab="Eje X", ylab="Eje Y")

    Explicación:

    • col: Cambia el color de los puntos.
    • pch: Cambia el símbolo de trazado.

  • Añadir Líneas de Cuadrícula:

    # Gráfico de dispersión con líneas de cuadrícula
plot(x, y, main="Gráfico de Dispersión con Cuadrícula", xlab="Eje X", ylab="Eje Y")
grid()

    Explicación:

    • grid(): Añade líneas de cuadrícula al gráfico.

  • Añadir Leyendas:

    # Gráfico de dispersión con una leyenda
plot(x, y, col="blue", pch=19, main="Gráfico de Dispersión con Leyenda", xlab="Eje X", ylab="Eje Y")
legend("topleft", legend="Puntos de Datos", col="blue", pch=19)

    Explicación:

    • legend(): Añade una leyenda al gráfico. La posición ("topleft") y otros parámetros especifican la apariencia de la leyenda.

Técnicas Avanzadas de Gráficos

Para visualizaciones más complejas, base R proporciona funciones y técnicas adicionales.

  • Múltiples Gráficos en una Figura:

    # Configurar el área de trazado para tener 2 filas y 1 columna
par(mfrow=c(2,1))

# Primer gráfico
plot(x, y, main="Gráfico 1")

# Segundo gráfico
plot(y, x, main="Gráfico 2")

    Explicación:

    • par(mfrow=c(2,1)): Divide el área de trazado en una cuadrícula de 2x1.

  • Superposición de Gráficos:

    # Crear el primer gráfico
plot(x, y, type="l", col="blue", main="Superposición de Gráficos", xlab="Eje X", ylab="Eje Y")

# Añadir un segundo gráfico encima
lines(x, y/2, col="red")

    Explicación:

    • lines(): Añade un gráfico de líneas a un gráfico existente.

Guardar Gráficos

Puedes guardar tus gráficos en varios formatos de archivo para su uso en informes o presentaciones.

  • Guardar en un Archivo PNG:

    # Abrir un dispositivo PNG
png("plot.png")

# Crear un gráfico
plot(x, y, main="Gráfico Guardado", xlab="Eje X", ylab="Eje Y")

# Cerrar el dispositivo
dev.off()

    Explicación:

    • png("plot.png"): Abre un dispositivo PNG para guardar el gráfico.
    • dev.off(): Cierra el dispositivo y escribe el archivo.

  • Guardar en un Archivo PDF:

    # Abrir un dispositivo PDF
pdf("plot.pdf")

# Crear un gráfico
plot(x, y, main="Gráfico Guardado", xlab="Eje X", ylab="Eje Y")

# Cerrar el dispositivo
dev.off()

    Explicación:

    • pdf("plot.pdf"): Abre un dispositivo PDF para guardar el gráfico.

Conclusión

Los gráficos base R proporcionan un conjunto robusto de herramientas para crear una amplia gama de visualizaciones. Desde gráficos básicos hasta personalizaciones avanzadas y opciones de guardado, entender estas herramientas es esencial para una visualización de datos efectiva en R. Al dominar estas técnicas, puedes crear gráficos informativos y visualmente atractivos que mejoren tu análisis de datos y presentación.
