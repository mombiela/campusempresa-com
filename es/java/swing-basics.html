Introducción a Swing

Swing es una parte de las Clases Fundamentales de Java (JFC) utilizada para crear aplicaciones basadas en ventanas. Está construida sobre el Kit de Herramientas de Ventana Abstracta (AWT) y proporciona un conjunto más rico de componentes GUI.

  • Componentes Livianos: A diferencia de AWT, los componentes de Swing son livianos, lo que significa que están escritos completamente en Java y no dependen de código nativo.
  • Apariencia y Comportamiento Enchufables: Swing te permite cambiar la apariencia de tu aplicación sin alterar el código subyacente.
  • Arquitectura MVC: Swing sigue la arquitectura Modelo-Vista-Controlador, que separa los datos (modelo), la interfaz de usuario (vista) y la lógica de control (controlador).

Configuración de una Aplicación Swing

Para crear una aplicación básica de Swing, debes seguir estos pasos:

  1. Importar Paquetes de Swing: Importa los paquetes necesarios de Swing y AWT.
  2. Crear un JFrame: La ventana principal de tu aplicación.
  3. Agregar Componentes: Añade varios componentes de Swing como botones, etiquetas, campos de texto, etc.
  4. Establecer el Diseño: Define el gestor de diseño para organizar los componentes.
  5. Mostrar el Marco: Haz visible el marco.

Ejemplo: Una Aplicación Simple de Swing

import javax.swing.JFrame;
import javax.swing.JLabel;

public class HelloWorldSwing {
    public static void main(String[] args) {
        // Crear un nuevo JFrame
        JFrame frame = new JFrame("Hello World Swing");
        
        // Establecer la operación de cierre predeterminada
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        
        // Añadir un JLabel al marco
        JLabel label = new JLabel("Hello, World!");
        frame.getContentPane().add(label);
        
        // Establecer el tamaño del marco
        frame.setSize(300, 100);
        
        // Hacer visible el marco
        frame.setVisible(true);
    }
}
  • Explicación:
    • JFrame: La ventana principal.
    • JLabel: Una etiqueta de texto simple.
    • setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE): Asegura que la aplicación se cierre cuando la ventana se cierra.
    • setSize(300, 100): Establece el tamaño de la ventana.
    • setVisible(true): Hace visible la ventana.

Componentes Comunes de Swing

Swing proporciona una variedad de componentes para construir una interfaz de usuario.

  • JButton: Un botón que puede desencadenar una acción.
  • JLabel: Un área de visualización para una cadena de texto corta o una imagen.
  • JTextField: Un campo de texto de una sola línea para la entrada del usuario.
  • JTextArea: Un área de varias líneas para mostrar/editar texto.
  • JCheckBox: Una caja que puede estar marcada o desmarcada.
  • JRadioButton: Un botón en un grupo donde solo uno puede ser seleccionado.
  • JComboBox: Una lista desplegable de elementos.

Ejemplo: Añadiendo Más Componentes

import javax.swing.*;

public class SwingComponentsExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Swing Components Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 200);
        
        // Crear componentes
        JButton button = new JButton("Click Me");
        JLabel label = new JLabel("Label:");
        JTextField textField = new JTextField(20);
        JCheckBox checkBox = new JCheckBox("Check Me");
        JRadioButton radioButton = new JRadioButton("Select Me");
        JComboBox<String> comboBox = new JComboBox<>(new String[]{"Option 1", "Option 2", "Option 3"});
        
        // Añadir componentes al marco
        JPanel panel = new JPanel();
        panel.add(label);
        panel.add(textField);
        panel.add(button);
        panel.add(checkBox);
        panel.add(radioButton);
        panel.add(comboBox);
        
        frame.getContentPane().add(panel);
        frame.setVisible(true);
    }
}
  • Explicación:
    • JPanel: Un contenedor para agrupar componentes.
    • JButton, JLabel, JTextField, JCheckBox, JRadioButton, JComboBox: Varios componentes de Swing añadidos al panel.

Manejo de Eventos en Swing

Swing utiliza el modelo de eventos de delegación para manejar eventos. Esto implica tres componentes:

  • Fuente del Evento: El componente que genera el evento (por ejemplo, un botón).
  • Objeto del Evento: Encapsula la información del evento.
  • Oyente del Evento: Una interfaz que recibe y procesa el evento.

Ejemplo: Manejo de Clics de Botón

import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class ButtonClickExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Button Click Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(300, 200);
        
        JButton button = new JButton("Click Me");
        JLabel label = new JLabel("Button not clicked");
        
        // Añadir ActionListener al botón
        button.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                label.setText("Button clicked!");
            }
        });
        
        JPanel panel = new JPanel();
        panel.add(button);
        panel.add(label);
        
        frame.getContentPane().add(panel);
        frame.setVisible(true);
    }
}
  • Explicación:
    • ActionListener: Una interfaz para manejar eventos de acción.
    • actionPerformed(ActionEvent e): Método llamado cuando se hace clic en el botón.
    • label.setText("Button clicked!"): Actualiza el texto de la etiqueta cuando se hace clic en el botón.

Gestores de Diseño

Los gestores de diseño se utilizan para organizar componentes en un contenedor. Algunos gestores de diseño comunes incluyen:

  • FlowLayout: Organiza los componentes en un flujo de izquierda a derecha.
  • BorderLayout: Organiza los componentes en cinco regiones: Norte, Sur, Este, Oeste y Centro.
  • GridLayout: Organiza los componentes en una cuadrícula de celdas.
  • BoxLayout: Organiza los componentes vertical u horizontalmente.

Ejemplo: Usando BorderLayout

import javax.swing.*;
import java.awt.*;

public class BorderLayoutExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("BorderLayout Example");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setSize(400, 300);
        
        JButton northButton = new JButton("North");
        JButton southButton = new JButton("South");
        JButton eastButton = new JButton("East");
        JButton westButton = new JButton("West");
        JButton centerButton = new JButton("Center");
        
        frame.setLayout(new BorderLayout());
        frame.add(northButton, BorderLayout.NORTH);
        frame.add(southButton, BorderLayout.SOUTH);
        frame.add(eastButton, BorderLayout.EAST);
        frame.add(westButton, BorderLayout.WEST);
        frame.add(centerButton, BorderLayout.CENTER);
        
        frame.setVisible(true);
    }
}
  • Explicación:
    • BorderLayout: Divide el contenedor en cinco regiones.
    • frame.add(component, BorderLayout.REGION): Añade componentes a regiones específicas.

Conclusión

Swing es una poderosa herramienta para crear aplicaciones GUI en Java. Al entender los conceptos básicos de los componentes de Swing, el manejo de eventos y los gestores de diseño, puedes construir aplicaciones interactivas y amigables para el usuario. A medida que progreses, puedes explorar características más avanzadas y personalizaciones para mejorar aún más tus aplicaciones.
