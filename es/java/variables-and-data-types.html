Introducción

En Java, las variables y los tipos de datos son conceptos fundamentales que forman la base de toda la programación. Entender cómo declarar, inicializar y usar variables, así como conocer los diferentes tipos de datos disponibles, es crucial para escribir programas efectivos en Java.

Variables

Las variables son contenedores para almacenar valores de datos. En Java, cada variable debe ser declarada con un tipo de dato específico, que determina el tipo de datos que puede contener.

Declaración de Variables

Para declarar una variable en Java, necesitas especificar el tipo de dato seguido por el nombre de la variable.

int miNumero;

Inicialización de Variables

Puedes inicializar una variable en el momento de la declaración o más tarde en el código.

int miNumero = 10; // Declaración e inicialización
miNumero = 20;     // Reasignación

Reglas para Nombrar Variables

  • Los nombres de las variables son sensibles a mayúsculas y minúsculas.
  • Deben comenzar con una letra, guion bajo (_) o signo de dólar ($).
  • Los caracteres subsiguientes pueden ser letras, dígitos, guiones bajos o signos de dólar.
  • Las palabras reservadas (como int, class, etc.) no pueden ser usadas como nombres de variables.

Tipos de Datos

Java es un lenguaje fuertemente tipado, lo que significa que cada variable debe tener un tipo declarado. Los tipos de datos en Java se dividen en dos categorías: tipos de datos primitivos y tipos de datos de referencia.

Tipos de Datos Primitivos

Java tiene 8 tipos de datos primitivos:

| Tipo de Dato | Tamaño | Descripción | Valores de Ejemplo | |--------------|---------|------------------------------------|-----------------------------| | byte | 8-bit | Almacena números enteros de -128 a 127 | -128, 0, 127 | | short | 16-bit | Almacena números enteros de -32,768 a 32,767 | -32768, 0, 32767 | | int | 32-bit | Almacena números enteros de -2^31 a 2^31-1 | -2147483648, 0, 2147483647 | | long | 64-bit | Almacena números enteros de -2^63 a 2^63-1 | -9223372036854775808, 0, 9223372036854775807 | | float | 32-bit | Almacena números fraccionarios, suficiente para 6 a 7 dígitos decimales | 3.14f, -0.001f | | double | 64-bit | Almacena números fraccionarios, suficiente para 15 dígitos decimales | 3.141592653589793, -0.0000001 | | boolean | 1-bit | Almacena valores verdaderos o falsos | true, false | | char | 16-bit | Almacena un solo carácter/letra o valores ASCII | 'a', 'Z', '1' |

Tipos de Datos de Referencia

Los tipos de datos de referencia son objetos y arreglos. Se crean usando constructores de clases.

String miCadena = "¡Hola, Mundo!";
int[] miArreglo = {1, 2, 3, 4, 5};

Conversión de Tipos

Java soporta tanto la conversión implícita como la explícita de tipos.

Conversión Implícita de Tipos

También conocida como conversión de ampliación, ocurre automáticamente cuando un tipo de dato más pequeño se asigna a un tipo de dato más grande.

int miInt = 9;
double miDouble = miInt; // Conversión automática: int a double

Conversión Explícita de Tipos

También conocida como conversión de reducción, requiere un casting explícito.

double miDouble = 9.78;
int miInt = (int) miDouble; // Casting manual: double a int

Ejemplos y Ejercicios

Ejemplo 1: Declaración e Inicialización Básica de Variables

public class Main {
    public static void main(String[] args) {
        int edad = 25;
        double salario = 50000.50;
        boolean estaEmpleado = true;
        char calificacion = 'A';
        
        System.out.println("Edad: " + edad);
        System.out.println("Salario: " + salario);
        System.out.println("Empleado: " + estaEmpleado);
        System.out.println("Calificación: " + calificacion);
    }
}

Ejercicio 1: Declarar e Inicializar Variables

  1. Declara una variable de tipo int llamada año e inicialízala con el año actual.
  2. Declara una variable de tipo float llamada temperatura e inicialízala con un valor de tu elección.
  3. Declara una variable de tipo boolean llamada esJavaDivertido e inicialízala con true.

Ejemplo 2: Conversión de Tipos

public class Main {
    public static void main(String[] args) {
        int miInt = 9;
        double miDouble = miInt; // Conversión automática: int a double
        System.out.println("miInt: " + miInt);
        System.out.println("miDouble: " + miDouble);
        
        double otroDouble = 9.78;
        int otroInt = (int) otroDouble; // Casting manual: double a int
        System.out.println("otroDouble: " + otroDouble);
        System.out.println("otroInt: " + otroInt);
    }
}

Ejercicio 2: Conversión de Tipos

  1. Declara una variable de tipo double llamada pi e inicialízala con 3.14159.
  2. Convierte pi a un int y almacénalo en una variable llamada piInt.
  3. Imprime tanto pi como piInt.

Conclusión

Entender las variables y los tipos de datos es esencial para cualquier programador de Java. Las variables almacenan datos, y los tipos de datos definen el tipo de datos que una variable puede contener. El sistema de tipos fuertes de Java asegura que las variables se usen correctamente, reduciendo errores y mejorando la fiabilidad del código. Practica declarando, inicializando y convirtiendo variables para volverte competente en el manejo de datos en Java.
