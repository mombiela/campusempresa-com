Las declaraciones de control de flujo en Java te permiten dictar el orden en que se ejecutan las declaraciones. Esta sección cubrirá las declaraciones esenciales de control de flujo, incluyendo declaraciones condicionales, bucles y declaraciones de ramificación.

Declaraciones Condicionales

Las declaraciones condicionales permiten que tu programa tome decisiones basadas en ciertas condiciones.

Declaración if

La declaración if ejecuta un bloque de código si una condición especificada es verdadera.

int number = 10;
if (number > 0) {
    System.out.println("El número es positivo.");
}

Declaración if-else

La declaración if-else proporciona un bloque de código alternativo para ejecutar si la condición es falsa.

int number = -10;
if (number > 0) {
    System.out.println("El número es positivo.");
} else {
    System.out.println("El número no es positivo.");
}

Escalera if-else-if

La escalera if-else-if te permite probar múltiples condiciones secuencialmente.

int number = 0;
if (number > 0) {
    System.out.println("El número es positivo.");
} else if (number < 0) {
    System.out.println("El número es negativo.");
} else {
    System.out.println("El número es cero.");
}

Declaración switch

La declaración switch te permite ejecutar un bloque de código de muchos basados en el valor de una expresión.

int day = 3;
switch (day) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    case 3:
        System.out.println("Miércoles");
        break;
    default:
        System.out.println("Día inválido");
        break;
}

Declaraciones de Bucle

Las declaraciones de bucle te permiten ejecutar un bloque de código múltiples veces.

Bucle for

El bucle for se usa cuando se conoce el número de iteraciones.

for (int i = 0; i < 5; i++) {
    System.out.println("Iteración: " + i);
}

Bucle while

El bucle while se usa cuando el número de iteraciones no se conoce y depende de una condición.

int i = 0;
while (i < 5) {
    System.out.println("Iteración: " + i);
    i++;
}

Bucle do-while

El bucle do-while es similar al bucle while, pero garantiza que el bloque de código se ejecutará al menos una vez.

int i = 0;
do {
    System.out.println("Iteración: " + i);
    i++;
} while (i < 5);

Declaraciones de Ramificación

Las declaraciones de ramificación te permiten alterar el flujo de control en tu programa.

Declaración break

La declaración break se usa para salir de un bucle o declaración switch prematuramente.

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break;
    }
    System.out.println("Iteración: " + i);
}

Declaración continue

La declaración continue salta la iteración actual de un bucle y procede a la siguiente iteración.

for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue;
    }
    System.out.println("Iteración: " + i);
}

Declaración return

La declaración return sale del método actual y opcionalmente devuelve un valor.

public int add(int a, int b) {
    return a + b;
}

Conclusión

Las declaraciones de control de flujo son fundamentales para programar en Java. Te permiten controlar el flujo de ejecución de tu programa basado en condiciones y bucles. Entender y dominar estas declaraciones te permitirá escribir programas Java más complejos y eficientes. Practica usando estas declaraciones en diferentes escenarios para obtener un sólido dominio de su funcionalidad.
