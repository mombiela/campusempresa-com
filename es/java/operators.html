Introducción

Los operadores son símbolos especiales en Java que realizan operaciones específicas en uno, dos o tres operandos y luego devuelven un resultado. Son fundamentales para escribir código efectivo y eficiente. Esta sección cubrirá los diversos tipos de operadores disponibles en Java, desde aritmética básica hasta operadores lógicos y de bits más avanzados.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

  • + : Suma
  • - : Resta
  • * : Multiplicación
  • / : División
  • % : Módulo (resto)

Ejemplo

int a = 10;
int b = 5;

int sum = a + b; // 15
int difference = a - b; // 5
int product = a * b; // 50
int quotient = a / b; // 2
int remainder = a % b; // 0

System.out.println("Suma: " + sum);
System.out.println("Diferencia: " + difference);
System.out.println("Producto: " + product);
System.out.println("Cociente: " + quotient);
System.out.println("Resto: " + remainder);

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores. Devuelven un resultado booleano.

  • == : Igual a
  • != : No igual a
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual a
  • <= : Menor o igual a

Ejemplo

int a = 10;
int b = 5;

boolean isEqual = (a == b); // false
boolean isNotEqual = (a != b); // true
boolean isGreater = (a > b); // true
boolean isLess = (a < b); // false
boolean isGreaterOrEqual = (a >= b); // true
boolean isLessOrEqual = (a <= b); // false

System.out.println("Es Igual: " + isEqual);
System.out.println("No es Igual: " + isNotEqual);
System.out.println("Es Mayor: " + isGreater);
System.out.println("Es Menor: " + isLess);
System.out.println("Es Mayor o Igual: " + isGreaterOrEqual);
System.out.println("Es Menor o Igual: " + isLessOrEqual);

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones booleanas.

  • && : AND lógico
  • || : OR lógico
  • ! : NOT lógico

Ejemplo

boolean a = true;
boolean b = false;

boolean andResult = a && b; // false
boolean orResult = a || b; // true
boolean notResult = !a; // false

System.out.println("Resultado AND: " + andResult);
System.out.println("Resultado OR: " + orResult);
System.out.println("Resultado NOT: " + notResult);

Operadores de Bits

Los operadores de bits realizan operaciones en bits individuales de tipos enteros.

  • & : AND de bits
  • | : OR de bits
  • ^ : XOR de bits
  • ~ : Complemento de bits
  • << : Desplazamiento a la izquierda
  • >> : Desplazamiento a la derecha
  • >>> : Desplazamiento a la derecha sin signo

Ejemplo

int a = 5; // 0101 en binario
int b = 3; // 0011 en binario

int andResult = a & b; // 0001 en binario, que es 1
int orResult = a | b; // 0111 en binario, que es 7
int xorResult = a ^ b; // 0110 en binario, que es 6
int complementResult = ~a; // 1010 en binario, que es -6 (complemento a 2)
int leftShiftResult = a << 1; // 1010 en binario, que es 10
int rightShiftResult = a >> 1; // 0010 en binario, que es 2
int unsignedRightShiftResult = a >>> 1; // 0010 en binario, que es 2

System.out.println("Resultado AND: " + andResult);
System.out.println("Resultado OR: " + orResult);
System.out.println("Resultado XOR: " + xorResult);
System.out.println("Resultado Complemento: " + complementResult);
System.out.println("Resultado Desplazamiento a la Izquierda: " + leftShiftResult);
System.out.println("Resultado Desplazamiento a la Derecha: " + rightShiftResult);
System.out.println("Resultado Desplazamiento a la Derecha sin Signo: " + unsignedRightShiftResult);

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • = : Asignación simple
  • += : Sumar y asignar
  • -= : Restar y asignar
  • *= : Multiplicar y asignar
  • /= : Dividir y asignar
  • %= : Módulo y asignar

Ejemplo

int a = 10;

a += 5; // a = a + 5, así que a ahora es 15
a -= 3; // a = a - 3, así que a ahora es 12
a *= 2; // a = a * 2, así que a ahora es 24
a /= 4; // a = a / 4, así que a ahora es 6
a %= 5; // a = a % 5, así que a ahora es 1

System.out.println("Valor final de a: " + a);

Operadores Unarios

Los operadores unarios operan en un solo operando.

  • + : Más unario (promueve el operando)
  • - : Menos unario (niega el operando)
  • ++ : Operador de incremento (aumenta el valor en 1)
  • -- : Operador de decremento (disminuye el valor en 1)
  • ! : Operador de complemento lógico

Ejemplo

int a = 10;
int b = -a; // b ahora es -10

a++; // a ahora es 11
a--; // a ahora es 10 de nuevo

boolean c = true;
boolean d = !c; // d ahora es false

System.out.println("Valor de b: " + b);
System.out.println("Valor de a después de incrementar y decrementar: " + a);
System.out.println("Valor de d: " + d);

Operador Ternario

El operador ternario es una abreviatura para la declaración if-else y se utiliza para evaluar expresiones booleanas.

  • ? : : Operador ternario

Ejemplo

int a = 10;
int b = 5;

int max = (a > b) ? a : b; // max es 10 porque a es mayor que b

System.out.println("Valor máximo: " + max);

Conclusión

Los operadores son herramientas esenciales en la programación en Java, permitiendo a los desarrolladores realizar una amplia gama de operaciones en datos. Entender cómo usar estos operadores de manera efectiva puede mejorar en gran medida tu capacidad para escribir código eficiente y legible. Desde aritmética básica hasta operaciones avanzadas de bits, dominar estos operadores es un paso crucial para convertirse en un programador competente en Java.
