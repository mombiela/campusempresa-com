Introducción

Los bucles son constructos fundamentales en la programación que te permiten ejecutar un bloque de código múltiples veces. En RPG (Report Program Generator), los bucles ayudan a automatizar tareas repetitivas, haciendo tu código más eficiente y fácil de manejar. Esta sección cubrirá los tres tipos principales de bucles en RPG: DO, FOR y WHILE.

Bucle DO

El bucle DO en RPG se utiliza para ejecutar un bloque de código un número específico de veces. Es particularmente útil cuando sabes de antemano cuántas iteraciones se requieren.

Sintaxis

DCL VAR(&COUNTER) TYPE(*INT) VALUE(1)

DO &COUNTER = 1 TO 10
    /* Tu código aquí */
    CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1)
ENDDO

Explicación

  • DCL VAR(&COUNTER) TYPE(*INT) VALUE(1): Declara una variable &COUNTER de tipo entero y la inicializa a 1.
  • DO &COUNTER = 1 TO 10: Inicia el bucle, estableciendo &COUNTER en 1 y ejecutándose hasta que &COUNTER sea 10.
  • CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1): Incrementa el contador en 1.
  • ENDDO: Termina el bucle.

Ejemplo

DCL VAR(&I) TYPE(*INT) VALUE(1)

DO &I = 1 TO 5
    SNDPGMMSG MSG('Iteración ' *CAT &I)
    CHGVAR VAR(&I) VALUE(&I + 1)
ENDDO

Este ejemplo envía un mensaje de programa para cada iteración de 1 a 5.

Bucle FOR

El bucle FOR es otra forma de ejecutar un bloque de código múltiples veces, a menudo utilizado cuando se conoce el número de iteraciones.

Sintaxis

FOR &COUNTER = 1 TO 10
    /* Tu código aquí */
ENDFOR

Explicación

  • FOR &COUNTER = 1 TO 10: Inicia el bucle, estableciendo &COUNTER en 1 y ejecutándose hasta que &COUNTER sea 10.
  • ENDFOR: Termina el bucle.

Ejemplo

DCL VAR(&I) TYPE(*INT) VALUE(1)

FOR &I = 1 TO 5
    SNDPGMMSG MSG('Iteración ' *CAT &I)
ENDFOR

Este ejemplo envía un mensaje de programa para cada iteración de 1 a 5, similar al ejemplo del bucle DO pero usando la sintaxis del bucle FOR.

Bucle WHILE

El bucle WHILE se utiliza para ejecutar un bloque de código mientras una condición especificada sea verdadera. Es particularmente útil cuando el número de iteraciones no se conoce de antemano.

Sintaxis

DCL VAR(&COUNTER) TYPE(*INT) VALUE(1)

WHILE &COUNTER <= 10
    /* Tu código aquí */
    CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1)
ENDWHILE

Explicación

  • DCL VAR(&COUNTER) TYPE(*INT) VALUE(1): Declara una variable &COUNTER de tipo entero y la inicializa a 1.
  • WHILE &COUNTER <= 10: Inicia el bucle, ejecutándose mientras &COUNTER sea menor o igual a 10.
  • CHGVAR VAR(&COUNTER) VALUE(&COUNTER + 1): Incrementa el contador en 1.
  • ENDWHILE: Termina el bucle.

Ejemplo

DCL VAR(&I) TYPE(*INT) VALUE(1)

WHILE &I <= 5
    SNDPGMMSG MSG('Iteración ' *CAT &I)
    CHGVAR VAR(&I) VALUE(&I + 1)
ENDWHILE

Este ejemplo envía un mensaje de programa para cada iteración de 1 a 5, similar a los ejemplos anteriores pero usando la sintaxis del bucle WHILE.

Comparación de Bucles

| Tipo de Bucle | Caso de Uso | Complejidad de Sintaxis | Flexibilidad | |---------------|-------------|-------------------------|--------------| | DO | Número conocido de iteraciones | Media | Media | | FOR | Número conocido de iteraciones | Simple | Baja | | WHILE | Número desconocido de iteraciones | Media | Alta |

Conclusión

Entender los bucles es crucial para una programación eficiente en RPG. Los bucles DO, FOR y WHILE tienen cada uno sus propios casos de uso y ventajas:

  • Bucle DO: Mejor cuando conoces el número de iteraciones y necesitas más control sobre la variable del bucle.
  • Bucle FOR: Sintaxis más simple para un número conocido de iteraciones.
  • Bucle WHILE: Más flexible, ideal para situaciones donde el número de iteraciones no se conoce de antemano.

Al dominar estos bucles, puedes escribir código RPG más eficiente y legible. Practica usando cada tipo de bucle en diferentes escenarios para volverte competente en su uso.
