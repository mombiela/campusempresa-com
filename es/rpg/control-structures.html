Introducción

Las estructuras de control son fundamentales en la programación ya que permiten alterar el flujo de ejecución basado en ciertas condiciones. En RPG (Report Program Generator), las estructuras de control como IF, ELSE y SELECT se utilizan para tomar decisiones dentro del código.

Declaración IF

La declaración IF se utiliza para ejecutar un bloque de código solo si una condición especificada es verdadera.

Sintaxis

IF condición;
   // Código a ejecutar si la condición es verdadera
ENDIF;

Ejemplo

DCL-S edad INT(3);
edad = 20;

IF edad >= 18;
   DSPLY 'Eres un adulto.';
ENDIF;
  • Explicación: En este ejemplo, el programa verifica si la variable edad es mayor o igual a 18. Si la condición es verdadera, muestra el mensaje "Eres un adulto."

Declaración ELSE

La declaración ELSE se utiliza junto con la declaración IF para ejecutar un bloque de código si la condición es falsa.

Sintaxis

IF condición;
   // Código a ejecutar si la condición es verdadera
ELSE;
   // Código a ejecutar si la condición es falsa
ENDIF;

Ejemplo

DCL-S edad INT(3);
edad = 16;

IF edad >= 18;
   DSPLY 'Eres un adulto.';
ELSE;
   DSPLY 'Eres un menor.';
ENDIF;
  • Explicación: Aquí, el programa verifica si edad es mayor o igual a 18. Si es verdadero, muestra "Eres un adulto." Si es falso, muestra "Eres un menor."

Declaración ELSEIF

La declaración ELSEIF permite verificar múltiples condiciones en secuencia.

Sintaxis

IF condición1;
   // Código a ejecutar si la condición1 es verdadera
ELSEIF condición2;
   // Código a ejecutar si la condición2 es verdadera
ELSE;
   // Código a ejecutar si ninguna de las condiciones es verdadera
ENDIF;

Ejemplo

DCL-S puntuación INT(3);
puntuación = 75;

IF puntuación >= 90;
   DSPLY 'Calificación: A';
ELSEIF puntuación >= 80;
   DSPLY 'Calificación: B';
ELSEIF puntuación >= 70;
   DSPLY 'Calificación: C';
ELSE;
   DSPLY 'Calificación: F';
ENDIF;
  • Explicación: El programa evalúa la puntuación y asigna una calificación basada en el valor. Se verifican múltiples condiciones en secuencia.

Declaración SELECT

La declaración SELECT se utiliza para ejecutar un bloque de código de múltiples bloques posibles basado en el valor de una variable.

Sintaxis

SELECT;
   WHEN condición1;
      // Código a ejecutar si la condición1 es verdadera
   WHEN condición2;
      // Código a ejecutar si la condición2 es verdadera
   OTHER;
      // Código a ejecutar si ninguna de las condiciones es verdadera
ENDSL;

Ejemplo

DCL-S día CHAR(10);
día = 'Lunes';

SELECT;
   WHEN día = 'Lunes';
      DSPLY 'Inicio de la semana laboral.';
   WHEN día = 'Viernes';
      DSPLY 'Fin de la semana laboral.';
   OTHER;
      DSPLY 'Día entre semana.';
ENDSL;
  • Explicación: El programa verifica el valor de la variable día y muestra un mensaje basado en el día de la semana.

Comparación de IF y SELECT

| Característica | IF/ELSEIF/ELSE | SELECT/WHEN/OTHER | |------------------|---------------------------|-----------------------------| | Caso de Uso | Condiciones simples | Múltiples condiciones basadas en una sola variable | | Legibilidad | Puede volverse complejo | Más legible para múltiples condiciones | | Flexibilidad | Alta | Limitada a una sola variable |

Ejercicios

  1. Escribe un programa RPG que verifique si un número es positivo, negativo o cero usando declaraciones IF, ELSEIF y ELSE.
  2. Crea un programa RPG que asigne una calificación de letra (A, B, C, D, F) basada en una puntuación numérica usando la declaración SELECT.

Conclusión

Entender las estructuras de control como IF, ELSE y SELECT es crucial para tomar decisiones en tus programas RPG. Estas estructuras te permiten controlar el flujo de tu programa basado en varias condiciones, haciendo tu código más dinámico y receptivo. Practica usando estas estructuras de control para volverte competente en escribir lógica condicional en RPG.
