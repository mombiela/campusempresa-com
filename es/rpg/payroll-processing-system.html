Introducción

En este estudio de caso, exploraremos el desarrollo de un Sistema de Procesamiento de Nómina utilizando RPG (Report Program Generator). Este sistema manejará datos de empleados, calculará salarios, generará recibos de pago y gestionará registros de nómina. El estudio de caso se dividirá en secciones que cubren diferentes aspectos del sistema, desde conceptos básicos hasta características avanzadas.

Conceptos Clave

  • Gestión de Datos de Empleados: Almacenamiento y recuperación de información de empleados.
  • Cálculo de Salarios: Cálculo de salarios brutos y netos basados en varios factores.
  • Generación de Recibos de Pago: Creación de recibos de pago detallados para los empleados.
  • Gestión de Registros de Nómina: Mantenimiento y actualización de registros de nómina.

Gestión de Datos de Empleados

Para comenzar, necesitamos crear una base de datos para almacenar la información de los empleados. Esto incluye campos como ID de Empleado, Nombre, Departamento, Salario Básico y más.

Ejemplo: Creación de Base de Datos de Empleados

**free
ctl-opt dftactgrp(*no) actgrp('EMPDB');

dcl-f EmployeeFile keyed;

dcl-ds EmployeeData qualified;
  EmployeeID char(10);
  Name char(50);
  Department char(20);
  BasicSalary packed(7:2);
end-ds;

dcl-proc CreateEmployee;
  dcl-pi *n;
    EmployeeID char(10);
    Name char(50);
    Department char(20);
    BasicSalary packed(7:2);
  end-pi;

  EmployeeData.EmployeeID = EmployeeID;
  EmployeeData.Name = Name;
  EmployeeData.Department = Department;
  EmployeeData.BasicSalary = BasicSalary;

  write EmployeeFile EmployeeData;
end-proc;

En este ejemplo, definimos una estructura de datos EmployeeData y un procedimiento CreateEmployee para agregar nuevos registros de empleados al EmployeeFile.

Cálculo de Salarios

A continuación, necesitamos calcular los salarios. Esto implica calcular el salario bruto, las deducciones y el salario neto.

Ejemplo: Cálculo de Salarios

**free
dcl-proc CalculateSalary;
  dcl-pi *n packed(7:2);
    BasicSalary packed(7:2);
    HRA packed(7:2);
    OtherAllowances packed(7:2);
    Deductions packed(7:2);
  end-pi;

  dcl-s GrossSalary packed(7:2);
  dcl-s NetSalary packed(7:2);

  GrossSalary = BasicSalary + HRA + OtherAllowances;
  NetSalary = GrossSalary - Deductions;

  return NetSalary;
end-proc;

Este procedimiento CalculateSalary toma el salario básico, HRA, otras asignaciones y deducciones como entrada y devuelve el salario neto.

Generación de Recibos de Pago

Generar recibos de pago implica crear una salida formateada que incluya todos los detalles del salario para un empleado.

Ejemplo: Generación de Recibos de Pago

**free
dcl-proc GeneratePaySlip;
  dcl-pi *n;
    EmployeeID char(10);
  end-pi;

  dcl-ds EmployeeData qualified;
    EmployeeID char(10);
    Name char(50);
    Department char(20);
    BasicSalary packed(7:2);
  end-ds;

  chain EmployeeID EmployeeFile EmployeeData;

  if %found(EmployeeFile);
    dsply ('Recibo de Pago para ' + EmployeeData.Name);
    dsply ('Departamento: ' + EmployeeData.Department);
    dsply ('Salario Básico: ' + %char(EmployeeData.BasicSalary));
    // Detalles adicionales como HRA, asignaciones, deducciones, salario neto
  else;
    dsply ('Empleado no encontrado');
  endif;
end-proc;

Este procedimiento GeneratePaySlip recupera los datos del empleado y muestra un recibo de pago formateado.

Gestión de Registros de Nómina

Finalmente, necesitamos gestionar los registros de nómina, incluyendo la actualización y eliminación de registros según sea necesario.

Ejemplo: Actualización de Registro de Empleado

**free
dcl-proc UpdateEmployee;
  dcl-pi *n;
    EmployeeID char(10);
    NewBasicSalary packed(7:2);
  end-pi;

  chain EmployeeID EmployeeFile EmployeeData;

  if %found(EmployeeFile);
    EmployeeData.BasicSalary = NewBasicSalary;
    update EmployeeFile EmployeeData;
  else;
    dsply ('Empleado no encontrado');
  endif;
end-proc;

Este procedimiento UpdateEmployee actualiza el salario básico de un empleado.

Conclusión

En este estudio de caso, cubrimos los componentes esenciales de un Sistema de Procesamiento de Nómina utilizando RPG. Comenzamos con la gestión de datos de empleados, pasamos al cálculo de salarios, generación de recibos de pago y, finalmente, la gestión de registros de nómina. Siguiendo estos ejemplos, deberías tener una base sólida para construir y expandir tu propio sistema de procesamiento de nómina en RPG.
