Introducción

En RPG (Report Program Generator), entender las variables y los tipos de datos es fundamental. Esta sección cubrirá los conceptos básicos de las variables, cómo declararlas y los diversos tipos de datos disponibles en RPG. Al final de esta sección, podrás usar variables de manera efectiva y entender los diferentes tipos de datos en RPG.

Variables

Las variables se utilizan para almacenar datos que pueden ser manipulados y recuperados a lo largo de un programa. En RPG, las variables deben ser declaradas antes de que puedan ser usadas.

Declaración de Variables

En RPG, las variables se declaran usando la instrucción DCL-S (Declarar Independiente). Aquí está la sintaxis básica:

DCL-S nombreVariable tipoDeDato;

Ejemplo

DCL-S miNumero INT(10);
DCL-S miCadena CHAR(50);

En este ejemplo:

  • miNumero se declara como un entero con una longitud de 10.
  • miCadena se declara como una cadena de caracteres con una longitud de 50.

Tipos de Datos

RPG soporta varios tipos de datos, cada uno con diferentes propósitos. Los tipos de datos más comúnmente usados son:

  • Carácter (CHAR)
  • Numérico (INT, DEC, UNS)
  • Fecha (DATE)
  • Hora (TIME)
  • Marca de Tiempo (TIMESTAMP)

Tipo de Dato Carácter

Los tipos de datos carácter se utilizan para almacenar datos alfanuméricos.

DCL-S nombreCliente CHAR(50);

Tipos de Datos Numéricos

Los tipos de datos numéricos se utilizan para almacenar números. Hay diferentes tipos de datos numéricos en RPG:

  • Entero (INT): Usado para números enteros.
  • Decimal (DEC): Usado para números con puntos decimales.
  • Sin Signo (UNS): Usado para números enteros no negativos.
DCL-S edad INT(3);
DCL-S salario DEC(7:2);
DCL-S cantidadProducto UNS(5);

Tipos de Datos de Fecha, Hora y Marca de Tiempo

Estos tipos de datos se utilizan para almacenar información de fecha y hora.

DCL-S fechaNacimiento DATE;
DCL-S horaInicio TIME;
DCL-S ultimaActualizacion TIMESTAMP;

Ejemplos y Ejercicios

Ejemplo 1: Declaración e Inicialización Básica de Variables

DCL-S nombreEmpleado CHAR(50) INZ('John Doe');
DCL-S edadEmpleado INT(3) INZ(30);
DCL-S fechaContratacion DATE INZ(D'2023-01-01');

En este ejemplo:

  • nombreEmpleado se inicializa con el valor 'John Doe'.
  • edadEmpleado se inicializa con el valor 30.
  • fechaContratacion se inicializa con la fecha '2023-01-01'.

Ejercicio 1: Declarar e Inicializar Variables

Declara e inicializa las siguientes variables:

  • nombreProducto como una cadena de caracteres con una longitud de 100 e inicialízala con 'Laptop'.
  • precioProducto como un decimal con 7 dígitos y 2 lugares decimales, inicialízalo con 999.99.
  • fechaFabricacion como una fecha, inicialízala con '2022-12-15'.

Ejemplo 2: Uso de Variables en Expresiones

DCL-S cantidad INT(5) INZ(10);
DCL-S precio DEC(7:2) INZ(15.75);
DCL-S costoTotal DEC(9:2);

costoTotal = cantidad * precio;

En este ejemplo:

  • cantidad se multiplica por precio y el resultado se almacena en costoTotal.

Ejercicio 2: Calcular Precio Total

Declara e inicializa las siguientes variables:

  • precioUnitario como un decimal con 7 dígitos y 2 lugares decimales, inicialízalo con 20.50.
  • cantidadOrdenada como un entero con 5 dígitos, inicialízalo con 25.
  • Calcula el precio total y almacénalo en una variable llamada precioTotal.

Resumen

En esta sección, cubrimos los conceptos básicos de variables y tipos de datos en RPG. Aprendimos cómo declarar variables, inicializarlas y usar diferentes tipos de datos como carácter, numérico, fecha, hora y marca de tiempo. Entender estos conceptos es crucial para escribir programas RPG efectivos. Asegúrate de practicar los ejercicios para reforzar tu comprensión.
