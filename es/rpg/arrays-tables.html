Introducción a Arreglos y Tablas en RPG

Los arreglos y tablas son estructuras de datos fundamentales en RPG (Report Program Generator) que te permiten almacenar y manipular colecciones de datos de manera eficiente. Entender cómo usar estas estructuras es crucial para escribir programas RPG efectivos y optimizados.

Arreglos en RPG

Los arreglos se utilizan para almacenar múltiples valores del mismo tipo en una sola variable. Cada valor en un arreglo se accede usando un índice.

Definición de Arreglos

Para definir un arreglo en RPG, usas la especificación D.

Dcl-S miArreglo Char(10) Dim(5);
  • Dcl-S: Declara una variable independiente.
  • miArreglo: El nombre del arreglo.
  • Char(10): Especifica que cada elemento en el arreglo es una cadena de caracteres de longitud 10.
  • Dim(5): Especifica que el arreglo tiene 5 elementos.

Acceso a Elementos del Arreglo

Puedes acceder y modificar elementos en un arreglo usando su índice.

miArreglo(1) = 'Primero';
miArreglo(2) = 'Segundo';
miArreglo(3) = 'Tercero';
miArreglo(4) = 'Cuarto';
miArreglo(5) = 'Quinto';

Iteración a Través de Arreglos

Puedes usar un bucle para iterar a través de los elementos de un arreglo.

For i = 1 to %Elem(miArreglo);
    Dsply miArreglo(i);
EndFor;
  • For i = 1 to %Elem(miArreglo);: Bucle desde 1 hasta el número de elementos en el arreglo.
  • Dsply miArreglo(i);: Muestra cada elemento del arreglo.

Tablas en RPG

Las tablas, también conocidas como estructuras de datos, se utilizan para almacenar tipos de datos complejos y pueden incluir múltiples campos de diferentes tipos.

Definición de Tablas

Para definir una tabla, usas la especificación D con una estructura de datos.

Dcl-DS miTabla Dim(5);
    nombre Char(10);
    edad  Int(3);
End-DS;
  • Dcl-DS: Declara una estructura de datos.
  • miTabla: El nombre de la tabla.
  • Dim(5): Especifica que la tabla tiene 5 elementos.
  • nombre Char(10);: Un campo en la tabla que almacena una cadena de caracteres de longitud 10.
  • edad Int(3);: Un campo en la tabla que almacena un entero de longitud 3.

Acceso a Elementos de la Tabla

Puedes acceder y modificar elementos en una tabla usando su índice y nombre de campo.

miTabla(1).nombre = 'Alicia';
miTabla(1).edad = 30;
miTabla(2).nombre = 'Bob';
miTabla(2).edad = 25;

Iteración a Través de Tablas

Puedes usar un bucle para iterar a través de los elementos de una tabla.

For i = 1 to %Elem(miTabla);
    Dsply miTabla(i).nombre + ' tiene ' + %Char(miTabla(i).edad) + ' años.';
EndFor;

Comparación de Arreglos y Tablas

| Característica | Arreglos | Tablas | |---------------|---------------------------------|---------------------------------| | Definición | Elementos de un solo tipo | Múltiples campos de diferentes tipos | | Acceso | Por índice | Por índice y nombre de campo | | Caso de Uso | Listas simples | Estructuras de datos complejas |

Conclusión

Los arreglos y tablas son herramientas poderosas en la programación RPG. Los arreglos son ideales para almacenar listas de tipos de datos simples, mientras que las tablas son más adecuadas para estructuras de datos complejas con múltiples campos. Dominar estas estructuras de datos mejorará tu capacidad para escribir programas RPG eficientes y organizados.

Al entender cómo definir, acceder e iterar a través de arreglos y tablas, puedes manejar una amplia gama de tareas de programación de manera más efectiva. Practica usando estas estructuras en diferentes escenarios para volverte competente en su aplicación.
