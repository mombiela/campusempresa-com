Introducción

Los directorios de enlace son un concepto crucial en la programación RPG (Report Program Generator), especialmente cuando se trata de programación modular y programas de servicio. Ayudan a gestionar y organizar los diversos módulos y programas de servicio que una aplicación RPG podría utilizar.

¿Qué es un Directorio de Enlace?

Un directorio de enlace es una colección de referencias a módulos y programas de servicio. Cuando creas o actualizas un programa o programa de servicio, el directorio de enlace ayuda al enlazador (el componente que vincula los módulos) a localizar los módulos y programas de servicio necesarios.

Conceptos Clave

  • Módulos: Piezas individuales de código compilado que pueden vincularse para formar un programa.
  • Programas de Servicio: Colecciones de módulos que pueden ser utilizados por múltiples programas.
  • Enlazador: El componente que vincula módulos y programas de servicio para crear un programa ejecutable.

¿Por qué Usar Directorios de Enlace?

Los directorios de enlace simplifican el proceso de gestión de dependencias entre diferentes partes de una aplicación. Permiten:

  • Reutilizar código de manera eficiente.
  • Simplificar el mantenimiento de aplicaciones grandes.
  • Mejorar la modularidad y organización de tu base de código.

Creación y Uso de Directorios de Enlace

Creación de un Directorio de Enlace

Puedes crear un directorio de enlace usando el comando CRTBNDDIR (Crear Directorio de Enlace) en el sistema operativo IBM i.

CRTBNDDIR BNDDIR(MYLIB/MYBNDDIR)

Este comando crea un directorio de enlace llamado MYBNDDIR en la biblioteca MYLIB.

Añadir Entradas a un Directorio de Enlace

Una vez que has creado un directorio de enlace, puedes añadir entradas a él usando el comando ADDBNDDIRE (Añadir Entrada de Directorio de Enlace).

ADDBNDDIRE BNDDIR(MYLIB/MYBNDDIR) OBJ((MYLIB/MYMODULE *MODULE) (MYLIB/MYSRVPGM *SRVPGM))

Este comando añade un módulo llamado MYMODULE y un programa de servicio llamado MYSRVPGM al directorio de enlace MYBNDDIR.

Usar un Directorio de Enlace en un Programa

Al compilar un programa RPG, puedes especificar el directorio de enlace a usar con la palabra clave BNDDIR en la especificación H o mediante los comandos CRTRPGMOD y CRTPGM.

Usando la Especificación H

H BNDDIR('MYLIB/MYBNDDIR')

Usando los Comandos CRTRPGMOD y CRTPGM

CRTRPGMOD MODULE(MYLIB/MYMODULE) SRCFILE(MYLIB/QRPGLESRC)
CRTPGM PGM(MYLIB/MYPROGRAM) BNDDIR(MYLIB/MYBNDDIR)

Ejemplo: Programa RPG Modular con Directorio de Enlace

Paso 1: Crear Módulos

Crea dos módulos simples, ModuleA y ModuleB.

ModuleA (ModuleA.rpgle)

**FREE
Dcl-Proc ModuleA;
  Dcl-Pi *N;
    InputValue Int(10);
  End-Pi;

  Dsply ('ModuleA: ' + %Char(InputValue));
End-Proc;

ModuleB (ModuleB.rpgle)

**FREE
Dcl-Proc ModuleB;
  Dcl-Pi *N;
    InputValue Int(10);
  End-Pi;

  Dsply ('ModuleB: ' + %Char(InputValue));
End-Proc;

Paso 2: Crear un Directorio de Enlace

Crea un directorio de enlace y añade los módulos a él.

CRTBNDDIR BNDDIR(MYLIB/MYBNDDIR)
ADDBNDDIRE BNDDIR(MYLIB/MYBNDDIR) OBJ((MYLIB/MODULEA *MODULE) (MYLIB/MODULEB *MODULE))

Paso 3: Crear un Programa Principal

Crea un programa principal que use los módulos.

MainProgram (MainProgram.rpgle)

**FREE
H BNDDIR('MYLIB/MYBNDDIR')

Dcl-Proc Main;
  Dcl-Pi *N;
  End-Pi;

  ModuleA(10);
  ModuleB(20);
End-Proc;

Paso 4: Compilar y Ejecutar

Compila los módulos y el programa principal.

CRTRPGMOD MODULE(MYLIB/MODULEA) SRCFILE(MYLIB/QRPGLESRC)
CRTRPGMOD MODULE(MYLIB/MODULEB) SRCFILE(MYLIB/QRPGLESRC)
CRTRPGMOD MODULE(MYLIB/MAINPROGRAM) SRCFILE(MYLIB/QRPGLESRC)
CRTPGM PGM(MYLIB/MAINPROGRAM) MODULE(MYLIB/MAINPROGRAM) BNDDIR(MYLIB/MYBNDDIR)

Ejecuta el programa principal para ver el resultado.

CALL PGM(MYLIB/MAINPROGRAM)

Conclusión

Los directorios de enlace son una herramienta poderosa en la programación RPG que ayudan a gestionar y organizar módulos y programas de servicio. Al usar directorios de enlace, puedes mejorar la modularidad, mantenibilidad y reutilización de tu código. Este tema cubrió los conceptos básicos de creación, gestión y uso de directorios de enlace, junto con un ejemplo práctico para ilustrar su uso.
