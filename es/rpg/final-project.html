Introducción

En este proyecto final, aplicarás todo lo que has aprendido a lo largo del curso para crear una aplicación integral de RPG (Juego de Rol). Este proyecto te ayudará a consolidar tus conocimientos y habilidades en desarrollo de juegos, programación orientada a objetos, estructuras de datos y más.

Resumen del Proyecto

La aplicación de RPG incluirá las siguientes características:

  • Creación y gestión de personajes
  • Sistema de inventario
  • Mecánicas de combate
  • Misiones y progresión de la historia
  • Funcionalidad para guardar y cargar el juego

Creación y Gestión de Personajes

Conceptos Clave

  • Clases y Objetos
  • Herencia y Polimorfismo
  • Encapsulación

Ejemplo

class Character:
    def __init__(self, name, health, strength, defense):
        self.name = name
        self.health = health
        self.strength = strength
        self.defense = defense

    def attack(self, target):
        damage = self.strength - target.defense
        target.health -= damage
        return damage

class Warrior(Character):
    def __init__(self, name):
        super().__init__(name, health=100, strength=20, defense=10)

class Mage(Character):
    def __init__(self, name):
        super().__init__(name, health=70, strength=30, defense=5)

# Ejemplo de uso
hero = Warrior("Aragorn")
villain = Mage("Saruman")
damage_dealt = hero.attack(villain)
print(f"{hero.name} causó {damage_dealt} de daño a {villain.name}.")

Sistema de Inventario

Conceptos Clave

  • Listas y Diccionarios
  • Métodos y Funciones

Ejemplo

class Inventory:
    def __init__(self):
        self.items = {}

    def add_item(self, item_name, quantity=1):
        if item_name in self.items:
            self.items[item_name] += quantity
        else:
            self.items[item_name] = quantity

    def remove_item(self, item_name, quantity=1):
        if item_name in self.items:
            self.items[item_name] -= quantity
            if self.items[item_name] <= 0:
                del self.items[item_name]

# Ejemplo de uso
inventory = Inventory()
inventory.add_item("Poción de Salud", 3)
inventory.add_item("Poción de Maná")
inventory.remove_item("Poción de Salud")
print(inventory.items)

Mecánicas de Combate

Conceptos Clave

  • Sentencias Condicionales
  • Bucles

Ejemplo

def combat_round(attacker, defender):
    damage = attacker.attack(defender)
    print(f"{attacker.name} ataca a {defender.name} por {damage} de daño.")
    if defender.health <= 0:
        print(f"¡{defender.name} ha sido derrotado!")
        return True
    return False

# Ejemplo de uso
hero = Warrior("Aragorn")
villain = Mage("Saruman")
while villain.health > 0 and hero.health > 0:
    if combat_round(hero, villain):
        break
    if combat_round(villain, hero):
        break

Misiones y Progresión de la Historia

Conceptos Clave

  • Clases y Objetos
  • Listas y Diccionarios

Ejemplo

class Quest:
    def __init__(self, name, description, is_completed=False):
        self.name = name
        self.description = description
        self.is_completed = is_completed

    def complete_quest(self):
        self.is_completed = True

# Ejemplo de uso
quest1 = Quest("Salvar el Pueblo", "Derrota a los goblins que atacan el pueblo.")
print(f"Misión: {quest1.name} - {quest1.description}")
quest1.complete_quest()
print(f"Misión Completada: {quest1.is_completed}")

Funcionalidad para Guardar y Cargar el Juego

Conceptos Clave

  • Entrada/Salida de Archivos
  • Serialización y Deserialización

Ejemplo

import json

class Game:
    def __init__(self, character, inventory):
        self.character = character
        self.inventory = inventory

    def save_game(self, filename):
        game_data = {
            'character': self.character.__dict__,
            'inventory': self.inventory.items
        }
        with open(filename, 'w') as file:
            json.dump(game_data, file)

    def load_game(self, filename):
        with open(filename, 'r') as file:
            game_data = json.load(file)
            self.character.__dict__.update(game_data['character'])
            self.inventory.items = game_data['inventory']

# Ejemplo de uso
hero = Warrior("Aragorn")
inventory = Inventory()
game = Game(hero, inventory)
game.save_game("savefile.json")
game.load_game("savefile.json")
