Introducción

En la visualización de datos, es crucial entender los diferentes tipos de datos y cómo se pueden representar de manera efectiva. Este conocimiento permite elegir las visualizaciones adecuadas para comunicar información de manera clara y precisa.

Tipos de Datos

Datos Categóricos

  • Definición: Datos que se dividen en categorías distintas y no tienen un orden inherente.
  • Ejemplos:
    • Colores (rojo, azul, verde)
    • Tipos de productos (electrónica, ropa, alimentos)
  • Visualizaciones Comunes:
    • Gráficos de barras
    • Gráficos de pastel

Datos Ordinales

  • Definición: Datos categóricos que tienen un orden específico.
  • Ejemplos:
    • Clasificaciones (primero, segundo, tercero)
    • Niveles de satisfacción (bajo, medio, alto)
  • Visualizaciones Comunes:
    • Gráficos de barras ordenados
    • Diagramas de caja

Datos Numéricos

  • Definición: Datos que representan cantidades y pueden ser medidos.
  • Subtipos:
    • Datos Discretos: Valores contables, como el número de estudiantes en una clase.
    • Datos Continuos: Valores medibles, como la altura o el peso.
  • Ejemplos:
    • Edad (numérico continuo)
    • Número de hijos (numérico discreto)
  • Visualizaciones Comunes:
    • Histogramas
    • Gráficos de dispersión

Representación de Datos

Gráficos de Barras

  • Uso: Ideal para comparar cantidades entre diferentes categorías.
  • Ejemplo de Código: 
    import matplotlib.pyplot as plt

categories = ['Electrónica', 'Ropa', 'Alimentos']
values = [150, 120, 300]

plt.bar(categories, values)
plt.xlabel('Categorías')
plt.ylabel('Ventas')
plt.title('Ventas por Categoría')
plt.show()

Gráficos de Pastel

  • Uso: Útil para mostrar la proporción de partes dentro de un todo.
  • Ejemplo de Código: 
    import matplotlib.pyplot as plt

labels = ['Electrónica', 'Ropa', 'Alimentos']
sizes = [150, 120, 300]

plt.pie(sizes, labels=labels, autopct='%1.1f%%')
plt.title('Distribución de Ventas por Categoría')
plt.show()

Histogramas

  • Uso: Ideal para mostrar la distribución de datos numéricos continuos.
  • Ejemplo de Código: 
    import matplotlib.pyplot as plt

data = [22, 45, 30, 35, 40, 50, 55, 60, 65, 70]

plt.hist(data, bins=5)
plt.xlabel('Edad')
plt.ylabel('Frecuencia')
plt.title('Distribución de Edades')
plt.show()

Gráficos de Dispersión

  • Uso: Útil para mostrar la relación entre dos variables numéricas.
  • Ejemplo de Código: 
    import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

plt.scatter(x, y)
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Gráfico de Dispersión de X vs Y')
plt.show()

Conclusión

Comprender los diferentes tipos de datos y sus representaciones es fundamental para la visualización efectiva de la información. Elegir la visualización correcta puede hacer una gran diferencia en cómo se interpreta y se entiende la información presentada. Practicar con diferentes tipos de gráficos y datos ayudará a desarrollar una intuición para seleccionar la mejor representación en cada caso.
