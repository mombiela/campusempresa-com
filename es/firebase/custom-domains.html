Firebase Hosting te permite desplegar aplicaciones web de manera rápida y segura. Una de las características más potentes que ofrece es la capacidad de usar dominios personalizados para tus aplicaciones web. Esta sección te guiará a través del proceso de configuración y gestión de dominios personalizados en Firebase, desde niveles principiantes hasta avanzados.

Introducción a los dominios personalizados

Los dominios personalizados te permiten usar tu propio nombre de dominio (por ejemplo, www.tudominio.com) en lugar del subdominio predeterminado de Firebase (por ejemplo, tuapp.firebaseapp.com). Esto puede mejorar la identidad de tu marca y hacer que tu aplicación sea más accesible.

Conceptos clave

  • Dominio personalizado: Tu propio nombre de dominio que deseas usar con tu sitio alojado en Firebase.
  • Registros DNS: Configuraciones que indican a internet dónde encontrar tu dominio.
  • Certificados SSL: Certificados de seguridad que encriptan los datos entre tus usuarios y tu sitio.

Configuración de un dominio personalizado

Paso 1: Comprar un dominio

Antes de poder configurar un dominio personalizado, necesitas comprar uno de un registrador de dominios como Google Domains, GoDaddy o Namecheap.

Paso 2: Agregar tu dominio a Firebase

  1. Ve a la Consola de Firebase.
  2. Selecciona tu proyecto.
  3. En el menú de la izquierda, haz clic en Hosting.
  4. Haz clic en Agregar dominio personalizado.
  5. Ingresa tu nombre de dominio y haz clic en Continuar.

Paso 3: Verificar la propiedad del dominio

Firebase te proporcionará un registro TXT que necesitas agregar a tu configuración DNS para verificar que eres el propietario del dominio.

Ejemplo:

Tipo: TXT
Nombre: @
Valor: firebase=tu-código-de-verificación

Paso 4: Actualizar los registros DNS

Después de verificar tu dominio, necesitas actualizar tus registros DNS para apuntar a Firebase Hosting.

Ejemplo:

Tipo: A
Nombre: @
Valor: 151.101.1.195

Paso 5: Desplegar tu sitio

Despliega tu sitio usando Firebase CLI:

firebase deploy

Gestión de certificados SSL

Firebase provisiona automáticamente certificados SSL para tu dominio personalizado, asegurando que tu sitio se sirva a través de HTTPS.

Puntos clave

  • SSL automático: Firebase maneja los certificados SSL por ti.
  • Renovación: Los certificados SSL se renuevan automáticamente por Firebase.
  • SSL personalizado: Los usuarios avanzados pueden cargar sus propios certificados SSL si es necesario.

Configuración avanzada

Redirecciones y reescrituras

Puedes configurar redirecciones y reescrituras en tu archivo firebase.json para gestionar cómo se manejan las URLs.

Ejemplo:

{
  "hosting": {
    "redirects": [
      {
        "source": "/old-page",
        "destination": "/new-page",
        "type": 301
      }
    ],
    "rewrites": [
      {
        "source": "/api/**",
        "function": "api"
      }
    ]
  }
}

Múltiples sitios

Firebase Hosting soporta múltiples sitios dentro de un solo proyecto de Firebase. Esto es útil para gestionar diferentes entornos (por ejemplo, staging y producción) o diferentes partes de una aplicación grande.

Ejemplo:

{
  "hosting": [
    {
      "target": "project1",
      "public": "public1",
      "ignore": ["firebase.json", "**/.*", "**/node_modules/**"]
    },
    {
      "target": "project2",
      "public": "public2",
      "ignore": ["firebase.json", "**/.*", "**/node_modules/**"]
    }
  ]
}

Solución de problemas

Problemas comunes

  • Propagación DNS: Los cambios en los registros DNS pueden tardar hasta 48 horas en propagarse.
  • Errores SSL: Asegúrate de que tu dominio esté correctamente verificado y que los registros DNS estén configurados adecuadamente.
  • Problemas de despliegue: Revisa la salida de Firebase CLI en busca de errores y asegúrate de que tu firebase.json esté correctamente configurado.

Conclusión

Configurar un dominio personalizado en Firebase Hosting mejora el profesionalismo y la accesibilidad de tu aplicación web. Siguiendo los pasos descritos en esta sección, puedes configurar y gestionar fácilmente dominios personalizados, asegurando una experiencia segura y con marca para tus usuarios. Ya seas un principiante o un usuario avanzado, Firebase proporciona las herramientas y la flexibilidad necesarias para gestionar eficazmente el alojamiento de tu aplicación web.
