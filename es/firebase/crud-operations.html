Introducción a Firebase

Firebase es una plataforma desarrollada por Google para crear aplicaciones móviles y web. Proporciona una variedad de herramientas y servicios para ayudar a los desarrolladores a construir aplicaciones de alta calidad, incluyendo una base de datos NoSQL en tiempo real, autenticación, análisis y más.

¿Qué son las Operaciones CRUD?

CRUD significa Crear, Leer, Actualizar y Eliminar. Estas son las cuatro operaciones básicas que se pueden realizar en los datos de una base de datos. Entender cómo realizar estas operaciones es fundamental para trabajar con cualquier base de datos, incluyendo Firebase.

Configuración de Firebase

Antes de sumergirnos en las operaciones CRUD, configuremos Firebase para tu proyecto.

  1. Crear un Proyecto de Firebase:

    • Ve a la Consola de Firebase.
    • Haz clic en "Agregar proyecto" y sigue las instrucciones para crear un nuevo proyecto.

  2. Agregar Firebase a tu Aplicación:

    • Para aplicaciones web, agrega el SDK de Firebase a tu proyecto.
    • Para aplicaciones Android/iOS, sigue las instrucciones específicas proporcionadas en la Consola de Firebase.

  3. Inicializar Firebase en tu Aplicación:

    // Importa los módulos de Firebase que necesitas en tu aplicación.
import { initializeApp } from "firebase/app";
import { getFirestore } from "firebase/firestore";

// TODO: Reemplaza lo siguiente con la configuración del proyecto de Firebase de tu aplicación
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_AUTH_DOMAIN",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_STORAGE_BUCKET",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};

// Inicializa Firebase
const app = initializeApp(firebaseConfig);
const db = getFirestore(app);

Operación de Crear

Crear datos en Firebase implica agregar nuevos documentos a una colección.

Ejemplo: Agregar un Nuevo Documento

import { collection, addDoc } from "firebase/firestore"; 

async function addUser() {
  try {
    const docRef = await addDoc(collection(db, "users"), {
      first: "Ada",
      last: "Lovelace",
      born: 1815
    });
    console.log("Documento escrito con ID: ", docRef.id);
  } catch (e) {
    console.error("Error al agregar el documento: ", e);
  }
}

addUser();
  • Explicación:
    • collection(db, "users"): Especifica la colección donde se agregará el documento.
    • addDoc: Agrega un nuevo documento con los datos especificados.

Operación de Leer

Leer datos de Firebase implica recuperar documentos de una colección.

Ejemplo: Obtener Todos los Documentos en una Colección

import { collection, getDocs } from "firebase/firestore"; 

async function getUsers() {
  const querySnapshot = await getDocs(collection(db, "users"));
  querySnapshot.forEach((doc) => {
    console.log(`${doc.id} => ${doc.data()}`);
  });
}

getUsers();
  • Explicación:
    • getDocs(collection(db, "users")): Recupera todos los documentos de la colección "users".
    • querySnapshot.forEach: Itera sobre cada documento en la colección.

Operación de Actualizar

Actualizar datos en Firebase implica modificar un documento existente.

Ejemplo: Actualizar un Documento

import { doc, updateDoc } from "firebase/firestore"; 

async function updateUser(userId) {
  const userRef = doc(db, "users", userId);

  // Establece el campo "last" del usuario
  await updateDoc(userRef, {
    last: "Byron",
    born: 1812
  });
}

updateUser("USER_DOCUMENT_ID");
  • Explicación:
    • doc(db, "users", userId): Especifica el documento a actualizar.
    • updateDoc: Actualiza los campos especificados del documento.

Operación de Eliminar

Eliminar datos en Firebase implica remover un documento de una colección.

Ejemplo: Eliminar un Documento

import { doc, deleteDoc } from "firebase/firestore"; 

async function deleteUser(userId) {
  await deleteDoc(doc(db, "users", userId));
}

deleteUser("USER_DOCUMENT_ID");
  • Explicación:
    • doc(db, "users", userId): Especifica el documento a eliminar.
    • deleteDoc: Elimina el documento especificado.

Conclusión

Las operaciones CRUD son fundamentales para gestionar datos en cualquier base de datos, incluyendo Firebase. Al dominar estas operaciones, puedes crear, leer, actualizar y eliminar datos de manera efectiva en tus aplicaciones de Firebase. Practica estas operaciones para volverte competente en el manejo de Firebase Firestore y explora características más avanzadas a medida que te sientas más cómodo con los conceptos básicos.
