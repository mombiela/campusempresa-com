Introducción a Firebase Hosting

Firebase Hosting es un servicio de alojamiento de contenido web de grado de producción para desarrolladores. Está diseñado para alojar tu aplicación web, contenido estático y dinámico, y microservicios.

Características Clave

  • Alojamiento rápido y seguro: CDN global, SSL por defecto y caché potente.
  • Despliegue con un solo clic: Despliega tu aplicación con un solo comando.
  • Dominios personalizados: Conecta fácilmente tu dominio personalizado.

Configurando Firebase Hosting

Prerrequisitos

  • Node.js y npm instalados en tu máquina.
  • Firebase CLI instalado globalmente a través de npm.
  • Un proyecto de Firebase creado en la Consola de Firebase.

Instalando Firebase CLI

Para instalar Firebase CLI, ejecuta el siguiente comando en tu terminal:

npm install -g firebase-tools

Inicializando Firebase en Tu Proyecto

Navega a tu directorio de proyecto y ejecuta:

firebase init
  • Selecciona "Hosting" de la lista de características de Firebase.
  • Elige un proyecto de Firebase existente o crea uno nuevo.
  • Especifica el directorio público (por ejemplo, build o public).
  • Configura como una aplicación de una sola página si es aplicable (por ejemplo, para aplicaciones React o Angular).

Ejemplo: Inicializando Firebase Hosting

cd your-project-directory
firebase init

Sigue las indicaciones:

  • Selecciona "Hosting".
  • Elige tu proyecto de Firebase.
  • Establece el directorio público (por ejemplo, build).
  • Configura como una aplicación de una sola página si usas un framework como React.

Desplegando Tu Aplicación Web

Construyendo Tu Aplicación Web

Antes de desplegar, asegúrate de que tu aplicación web esté construida y lista para producción. Por ejemplo, si estás usando React:

npm run build

Desplegando en Firebase Hosting

Ejecuta el siguiente comando para desplegar tu aplicación:

firebase deploy

Este comando:

  • Subirá tus archivos a Firebase Hosting.
  • Proporcionará una URL donde tu aplicación está alojada.

Ejemplo: Desplegando una Aplicación React

  1. Construye tu aplicación: 
    npm run build
  2. Despliega en Firebase: 
    firebase deploy

Configuración Avanzada

Dominios Personalizados

Puedes añadir un dominio personalizado a tu proyecto de Firebase:

  1. Ve a la Consola de Firebase.
  2. Navega a Hosting.
  3. Haz clic en "Añadir Dominio Personalizado" y sigue las instrucciones.

Redirecciones y Reescrituras

Puedes configurar redirecciones y reescrituras en tu archivo firebase.json:

{
  "hosting": {
    "public": "build",
    "rewrites": [
      {
        "source": "**",
        "destination": "/index.html"
      }
    ],
    "redirects": [
      {
        "source": "/old-page",
        "destination": "/new-page",
        "type": 301
      }
    ]
  }
}

Resumen

En este tema, aprendiste cómo desplegar una aplicación web usando Firebase Hosting. Cubrimos:

  • Configuración de Firebase Hosting.
  • Construcción y despliegue de tu aplicación web.
  • Configuraciones avanzadas como dominios personalizados, redirecciones y reescrituras.

Firebase Hosting proporciona una forma rápida, segura y fácil de desplegar tus aplicaciones web, lo que lo convierte en una excelente opción para desarrolladores de todos los niveles.
