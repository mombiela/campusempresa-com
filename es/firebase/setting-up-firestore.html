Introducción

Firestore es una base de datos flexible y escalable para el desarrollo móvil, web y de servidores de Firebase y Google Cloud Platform. En esta sección, cubriremos cómo configurar Firestore para tu proyecto, desde la configuración inicial hasta la integración en tu aplicación.

Requisitos previos

Antes de configurar Firestore, asegúrate de tener:

  • Una cuenta de Google.
  • Un proyecto de Firebase creado.
  • Conocimientos básicos de la Consola de Firebase.

Guía paso a paso para configurar Firestore

1. Crear un proyecto de Firebase

  1. Ve a la Consola de Firebase.
  2. Haz clic en "Agregar proyecto" y sigue las instrucciones en pantalla para crear un nuevo proyecto.

2. Agregar Firestore a tu proyecto de Firebase

  1. En la Consola de Firebase, selecciona tu proyecto.
  2. En el menú de la izquierda, haz clic en "Firestore Database".
  3. Haz clic en "Crear base de datos".
  4. Elige un modo de inicio para tus Reglas de Seguridad de Firestore:
    • Modo de prueba: Permite que cualquiera con la referencia de tu base de datos lea y escriba en tu base de datos. Adecuado para desarrollo.
    • Modo bloqueado: Niega todas las lecturas y escrituras desde clientes móviles y web. Adecuado para producción.
  5. Selecciona una ubicación de Cloud Firestore y haz clic en "Habilitar".

3. Integrar Firestore en tu aplicación

Para usar Firestore en tu aplicación, necesitas agregar el SDK de Firebase a tu proyecto.

Para aplicaciones web

  1. Agrega el SDK de Firebase a tu proyecto: 
    <!-- Agregar SDKs de Firebase -->
<script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-firestore.js"></script>
  2. Inicializa Firebase y Firestore: 
    // Importa las funciones que necesitas de los SDKs que necesitas
import { initializeApp } from "https://www.gstatic.com/firebasejs/9.0.0/firebase-app.js";
import { getFirestore } from "https://www.gstatic.com/firebasejs/9.0.0/firebase-firestore.js";

// La configuración de Firebase de tu aplicación web
const firebaseConfig = {
    apiKey: "TU_API_KEY",
    authDomain: "TU_AUTH_DOMAIN",
    projectId: "TU_PROJECT_ID",
    storageBucket: "TU_STORAGE_BUCKET",
    messagingSenderId: "TU_MESSAGING_SENDER_ID",
    appId: "TU_APP_ID"
};

// Inicializa Firebase
const app = initializeApp(firebaseConfig);

// Inicializa Firestore
const db = getFirestore(app);

Para aplicaciones Android

  1. Agrega el SDK de Firebase a tu proyecto: 
    // Agrega la dependencia para el SDK de Firebase
implementation 'com.google.firebase:firebase-firestore:23.0.3'
  2. Inicializa Firebase y Firestore en tu MainActivity: 
    // Importa las bibliotecas necesarias de Firebase
import com.google.firebase.firestore.FirebaseFirestore;

// Inicializa Firestore
FirebaseFirestore db = FirebaseFirestore.getInstance();

Para aplicaciones iOS

  1. Agrega el SDK de Firebase a tu proyecto usando CocoaPods: 
    pod 'Firebase/Firestore'
  2. Inicializa Firebase y Firestore en tu AppDelegate: 
    import Firebase

@UIApplicationMain
class AppDelegate: UIResponder, UIApplicationDelegate {

    var window: UIWindow?

    func application(_ application: UIApplication,
                     didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey: Any]?) -> Bool {
        FirebaseApp.configure()

        // Inicializa Firestore
        let db = Firestore.firestore()

        return true
    }
}

Conclusión

Configurar Firestore implica crear un proyecto de Firebase, habilitar Firestore en la Consola de Firebase e integrar Firestore en tu aplicación utilizando los SDKs apropiados. Siguiendo estos pasos, puedes comenzar a usar Firestore para almacenar y gestionar los datos de tu aplicación de manera eficiente. En las siguientes secciones, exploraremos cómo realizar operaciones CRUD y gestionar datos en Firestore.
