Introducción a Kubernetes

Kubernetes, a menudo abreviado como K8s, es una plataforma de código abierto diseñada para automatizar el despliegue, escalado y operación de contenedores de aplicaciones. Agrupa contenedores que conforman una aplicación en unidades lógicas para facilitar su gestión y descubrimiento. Este curso te guiará desde los conceptos básicos de Kubernetes hasta conceptos avanzados, asegurando que tengas una comprensión integral de la plataforma.

Conceptos Clave

  • Contenedores: Paquetes ejecutables, autónomos y ligeros que incluyen todo lo necesario para ejecutar un software.
  • Pods: El objeto más pequeño y simple de Kubernetes. Un pod representa un conjunto de contenedores en ejecución en tu clúster.
  • Nodos: Máquinas de trabajo en Kubernetes, que pueden ser virtuales o físicas.
  • Clústeres: Un conjunto de nodos que ejecutan aplicaciones en contenedores gestionadas por Kubernetes.
  • Servicios: Una forma abstracta de exponer una aplicación que se ejecuta en un conjunto de Pods como un servicio de red.
  • Despliegues: Un controlador que proporciona actualizaciones declarativas a las aplicaciones.

Estructura del Curso

Nivel Principiante

Entendiendo los Contenedores y los Conceptos Básicos de Kubernetes

  • Introducción a los Contenedores:

    • Explicación de Docker y la contenedorización.
    • Comandos básicos de Docker para crear y gestionar contenedores.
    • Ejemplo: 
      # Descargando una imagen de Docker
docker pull nginx

# Ejecutando un contenedor de Docker
docker run -d -p 80:80 nginx

  • Arquitectura de Kubernetes:

    • Visión general de los componentes de Kubernetes: Nodo Maestro, Nodos de Trabajo, etc.
    • Explicación de Pods, Nodos y Clústeres.
    • Ejemplo: 
      # Definición simple de un Pod
apiVersion: v1
kind: Pod
metadata:
  name: nginx-pod
spec:
  containers:
  - name: nginx
    image: nginx:latest

Nivel Intermedio

Gestión de Recursos de Kubernetes

  • Despliegues y Servicios:

    • Creación y gestión de Despliegues.
    • Exposición de Despliegues usando Servicios.
    • Ejemplo: 
      # Definición de un Despliegue
apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:latest

# Definición de un Servicio
apiVersion: v1
kind: Service
metadata:
  name: nginx-service
spec:
  selector:
    app: nginx
  ports:
  - protocol: TCP
    port: 80
    targetPort: 80

  • ConfigMaps y Secrets:

    • Gestión de datos de configuración con ConfigMaps.
    • Almacenamiento de información sensible con Secrets.
    • Ejemplo: 
      # Definición de ConfigMap
apiVersion: v1
kind: ConfigMap
metadata:
  name: example-config
data:
  key1: value1
  key2: value2

# Definición de Secret
apiVersion: v1
kind: Secret
metadata:
  name: example-secret
type: Opaque
data:
  username: YWRtaW4=  # codificado en base64
  password: MWYyZDFlMmU2N2Rm  # codificado en base64

Nivel Avanzado

Características Avanzadas de Kubernetes

  • StatefulSets y DaemonSets:

    • Gestión de aplicaciones con estado con StatefulSets.
    • Ejecución de una copia de un Pod en todos los nodos con DaemonSets.
    • Ejemplo: 
      # Definición de StatefulSet
apiVersion: apps/v1
kind: StatefulSet
metadata:
  name: web
spec:
  serviceName: "nginx"
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:latest

# Definición de DaemonSet
apiVersion: apps/v1
kind: DaemonSet
metadata:
  name: fluentd
spec:
  selector:
    matchLabels:
      name: fluentd
  template:
    metadata:
      labels:
        name: fluentd
    spec:
      containers:
      - name: fluentd
        image: fluent/fluentd:latest

  • Helm Charts:

    • Introducción a Helm y sus beneficios.
    • Creación y despliegue de aplicaciones usando Helm charts.
    • Ejemplo: 
      # Instalando un Helm chart
helm install my-nginx stable/nginx

Conclusión

Al final de este curso, tendrás una sólida comprensión de Kubernetes, desde conceptos básicos hasta características avanzadas. Serás capaz de desplegar, gestionar y escalar aplicaciones en contenedores de manera eficiente usando Kubernetes. Este conocimiento te permitirá manejar escenarios y desafíos del mundo real en la gestión de aplicaciones en contenedores.
