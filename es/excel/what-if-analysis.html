Introducción al análisis de hipótesis

El análisis de hipótesis en Excel te permite experimentar con datos para ver cómo los cambios afectan los resultados. Esto es particularmente útil para la toma de decisiones y la previsión. Las tres herramientas principales para el análisis de hipótesis en Excel son:

  • Buscar objetivo
  • Tablas de datos
  • Administrador de escenarios

Buscar objetivo

Buscar objetivo es una herramienta simple que te permite encontrar el valor de entrada necesario para lograr un objetivo específico. Funciona ajustando una variable para alcanzar un valor objetivo.

Cómo usar Buscar objetivo

  1. Configura tus datos: Asegúrate de tener una fórmula que calcule el resultado que te interesa.
  2. Abre Buscar objetivo: Ve a la pestaña Datos, haz clic en Análisis de hipótesis y selecciona Buscar objetivo.
  3. Configura Buscar objetivo:
    • Definir celda: La celda que contiene la fórmula.
    • A valor: El resultado deseado que quieres lograr.
    • Cambiando la celda: La celda que Excel cambiará para lograr el resultado deseado.

Ejemplo

Supongamos que quieres saber cuánto necesitas vender para alcanzar un ingreso de $10,000.

| A | B | |-----------|---------| | Unidades vendidas | 100 | | Precio | $50 | | Ingreso | =A2*B2 |

Para encontrar las unidades vendidas necesarias para alcanzar $10,000 en ingresos:

  1. Definir celda: B3 (Ingreso)
  2. A valor: 10000
  3. Cambiando la celda: A2 (Unidades vendidas)

Tablas de datos

Las tablas de datos te permiten ver cómo los cambios en una o dos variables afectan los resultados de una fórmula. Hay dos tipos de tablas de datos:

  • Tabla de datos de una variable
  • Tabla de datos de dos variables

Tabla de datos de una variable

Una tabla de datos de una variable muestra cómo cambiar una variable impacta el resultado.

Cómo crear una tabla de datos de una variable

  1. Configura tus datos: Crea una fórmula que dependa de una variable.
  2. Crea una estructura de tabla: Lista los valores de la variable en una columna o fila.
  3. Inserta la tabla de datos:
    • Selecciona el rango que incluye la fórmula y los valores de la variable.
    • Ve a la pestaña Datos, haz clic en Análisis de hipótesis y selecciona Tabla de datos.
    • Para una columna de valores de variable, ingresa la referencia de celda para la variable en Celda de entrada de columna.

Ejemplo

Calcula el ingreso para diferentes unidades vendidas.

| A | B | C | |-----------|---------|---------| | Unidades vendidas | Ingreso | | | 50 | | | | 100 | | | | 150 | | | | 200 | | | | Fórmula | =A2*50 | |

  1. Selecciona el rango A1:B5.
  2. Ve a Datos > Análisis de hipótesis > Tabla de datos.
  3. Ingresa A2 como la Celda de entrada de columna.

Tabla de datos de dos variables

Una tabla de datos de dos variables muestra cómo cambiar dos variables simultáneamente afecta el resultado.

Cómo crear una tabla de datos de dos variables

  1. Configura tus datos: Crea una fórmula que dependa de dos variables.
  2. Crea una estructura de tabla: Lista los valores de una variable en una fila y los de la otra en una columna.
  3. Inserta la tabla de datos:
    • Selecciona el rango que incluye la fórmula y los valores de las variables.
    • Ve a la pestaña Datos, haz clic en Análisis de hipótesis y selecciona Tabla de datos.
    • Ingresa las referencias de celda para las celdas de entrada de fila y columna.

Ejemplo

Calcula el ingreso para diferentes unidades vendidas y precios.

| A | B | C | D | E | |-----------|---------|---------|---------|---------| | Unidades vendidas | 40 | 50 | 60 | 70 | | 100 | | | | | | 150 | | | | | | 200 | | | | | | Fórmula | =A2*B1 | | | |

  1. Selecciona el rango A1:E4.
  2. Ve a Datos > Análisis de hipótesis > Tabla de datos.
  3. Ingresa A2 como la Celda de entrada de fila y B1 como la Celda de entrada de columna.

Administrador de escenarios

El Administrador de escenarios te permite crear y guardar diferentes conjuntos de valores de entrada, llamados escenarios, y alternar entre ellos para ver cómo afectan tus resultados.

Cómo usar el Administrador de escenarios

  1. Configura tus datos: Identifica las celdas que cambiarán en cada escenario.
  2. Abre el Administrador de escenarios: Ve a la pestaña Datos, haz clic en Análisis de hipótesis y selecciona Administrador de escenarios.
  3. Crea escenarios:
    • Haz clic en Agregar para crear un nuevo escenario.
    • Nombra el escenario y selecciona las celdas que cambiarán.
    • Ingresa los valores para cada escenario.
  4. Ver escenarios: Selecciona un escenario y haz clic en Mostrar para ver los resultados.

Ejemplo

Supongamos que tienes diferentes estrategias de ventas con unidades vendidas y precios variables.

| A | B | C | |-----------|---------|---------| | Unidades vendidas | 100 | | | Precio | $50 | | | Ingreso | =A2*B2 | |

  1. Ve a Datos > Análisis de hipótesis > Administrador de escenarios.
  2. Haz clic en Agregar para crear un escenario para la Estrategia 1:
    • Nombre: Estrategia 1
    • Celdas cambiantes: A2, B2
    • Valores: 100, 50
  3. Agrega otro escenario para la Estrategia 2:
    • Nombre: Estrategia 2
    • Celdas cambiantes: A2, B2
    • Valores: 150, 45
  4. Selecciona un escenario y haz clic en Mostrar para ver el impacto en los ingresos.

Conclusión

El análisis de hipótesis en Excel es una herramienta poderosa para explorar diferentes escenarios y tomar decisiones informadas. Al usar Buscar objetivo, Tablas de datos y el Administrador de escenarios, puedes analizar cómo los cambios en las variables afectan tus resultados y planificar en consecuencia. Practica usando estas herramientas con tus propios datos para volverte competente en el análisis de hipótesis.
