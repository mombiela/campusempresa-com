Introducción

Las funciones de búsqueda en Excel son herramientas poderosas que te permiten buscar datos específicos dentro de un rango y devolver valores correspondientes. Esta sección cubrirá tres funciones de búsqueda esenciales: VLOOKUP, HLOOKUP y XLOOKUP. Al final de este tema, comprenderás cómo usar estas funciones para agilizar tus tareas de análisis de datos.

VLOOKUP

¿Qué es VLOOKUP?

  • VLOOKUP significa "Búsqueda Vertical."
  • Busca un valor en la primera columna de un rango y devuelve un valor en la misma fila desde una columna especificada.

Sintaxis

VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
  • lookup_value: El valor que deseas buscar.
  • table_array: El rango de celdas que contiene los datos.
  • col_index_num: El número de columna en la tabla desde la cual recuperar el valor.
  • [range_lookup]: Opcional; TRUE para una coincidencia aproximada, FALSE para una coincidencia exacta.

Ejemplo

Supongamos que tienes la siguiente tabla:

| A | B | C | |---------|---------|---------| | ID | Nombre | Salario | | 1 | John | 50000 | | 2 | Jane | 60000 | | 3 | Alice | 55000 |

Para encontrar el salario del empleado con ID 2:

=VLOOKUP(2, A2:C4, 3, FALSE)

Esta fórmula busca 2 en la columna A y devuelve el valor de la tercera columna (Salario), que es 60000.

HLOOKUP

¿Qué es HLOOKUP?

  • HLOOKUP significa "Búsqueda Horizontal."
  • Busca un valor en la primera fila de un rango y devuelve un valor en la misma columna desde una fila especificada.

Sintaxis

HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
  • lookup_value: El valor que deseas buscar.
  • table_array: El rango de celdas que contiene los datos.
  • row_index_num: El número de fila en la tabla desde la cual recuperar el valor.
  • [range_lookup]: Opcional; TRUE para una coincidencia aproximada, FALSE para una coincidencia exacta.

Ejemplo

Supongamos que tienes la siguiente tabla:

| A | B | C | |---------|---------|---------| | ID | 1 | 2 | | Nombre | John | Jane | | Salario | 50000 | 60000 |

Para encontrar el nombre del empleado con ID 2:

=HLOOKUP(2, A1:C3, 2, FALSE)

Esta fórmula busca 2 en la fila 1 y devuelve el valor de la segunda fila (Nombre), que es Jane.

XLOOKUP

¿Qué es XLOOKUP?

  • XLOOKUP es una función más flexible y poderosa introducida en Excel 2019 y Office 365.
  • Puede buscar tanto vertical como horizontalmente y ofrece más opciones para manejar errores y valores faltantes.

Sintaxis

XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])
  • lookup_value: El valor que deseas buscar.
  • lookup_array: El rango de celdas a buscar.
  • return_array: El rango de celdas desde el cual devolver el valor.
  • [if_not_found]: Opcional; el valor a devolver si no se encuentra ninguna coincidencia.
  • [match_mode]: Opcional; 0 para una coincidencia exacta, -1 para coincidencia exacta o siguiente elemento menor, 1 para coincidencia exacta o siguiente elemento mayor, 2 para coincidencia con comodines.
  • [search_mode]: Opcional; 1 para buscar de primero a último, -1 para buscar de último a primero.

Ejemplo

Supongamos que tienes la siguiente tabla:

| A | B | C | |---------|---------|---------| | ID | Nombre | Salario | | 1 | John | 50000 | | 2 | Jane | 60000 | | 3 | Alice | 55000 |

Para encontrar el salario del empleado con ID 2:

=XLOOKUP(2, A2:A4, C2:C4, "No Encontrado")

Esta fórmula busca 2 en la columna A y devuelve el valor correspondiente de la columna C (Salario), que es 60000.

Conclusión

Las funciones de búsqueda en Excel, incluyendo VLOOKUP, HLOOKUP y XLOOKUP, son herramientas esenciales para el análisis de datos. VLOOKUP y HLOOKUP son útiles para búsquedas verticales y horizontales simples, respectivamente, mientras que XLOOKUP ofrece más flexibilidad y características avanzadas. Al dominar estas funciones, puedes buscar y recuperar datos de manera eficiente, haciendo tus tareas en Excel más eficientes y efectivas.
