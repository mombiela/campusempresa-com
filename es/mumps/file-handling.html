Introducción al Manejo de Archivos

El manejo de archivos en MUMPS (también conocido como M) implica leer y escribir archivos. Esto es esencial para el almacenamiento, recuperación y manipulación de datos. En esta sección, cubriremos los conceptos básicos del manejo de archivos, incluyendo abrir, leer, escribir y cerrar archivos.

Operaciones Básicas de Archivos

Abrir un Archivo

Para trabajar con un archivo, primero necesitas abrirlo. En MUMPS, utilizas el comando OPEN para abrir un archivo.

OPEN "nombrearchivo":("R"):1
IF '$TEST WRITE "Error al abrir el archivo",! QUIT
  • nombrearchivo: El nombre del archivo que deseas abrir.
  • ("R"): El modo en el que se abre el archivo. "R" significa modo de lectura.
  • 1: El número de dispositivo. Esto es opcional pero recomendado para mayor claridad.

Cerrar un Archivo

Después de terminar con un archivo, debes cerrarlo usando el comando CLOSE.

CLOSE "nombrearchivo"

Leer de un Archivo

Para leer de un archivo, utilizas el comando READ. Primero debes abrir el archivo en modo de lectura.

OPEN "nombrearchivo":("R"):1
IF '$TEST WRITE "Error al abrir el archivo",! QUIT

USE "nombrearchivo"
READ linea
WRITE "Línea leída: ", linea,!

CLOSE "nombrearchivo"
  • USE "nombrearchivo": Dirige las operaciones de E/S subsecuentes al archivo especificado.
  • READ linea: Lee una línea del archivo en la variable linea.

Escribir en un Archivo

Para escribir en un archivo, utilizas el comando WRITE. Primero debes abrir el archivo en modo de escritura.

OPEN "nombrearchivo":("W"):1
IF '$TEST WRITE "Error al abrir el archivo",! QUIT

USE "nombrearchivo"
WRITE "Esta es una línea de texto",!

CLOSE "nombrearchivo"
  • ("W"): El modo en el que se abre el archivo. "W" significa modo de escritura.
  • WRITE "Esta es una línea de texto": Escribe el texto especificado en el archivo.

Operaciones Avanzadas de Archivos

Añadir a un Archivo

Para añadir datos a un archivo existente, abres el archivo en modo de adición usando ("A").

OPEN "nombrearchivo":("A"):1
IF '$TEST WRITE "Error al abrir el archivo",! QUIT

USE "nombrearchivo"
WRITE "Añadiendo esta línea",!

CLOSE "nombrearchivo"
  • ("A"): El modo en el que se abre el archivo. "A" significa modo de adición.

Manejo de Errores

El manejo adecuado de errores es crucial para operaciones de archivos robustas. Usa la variable $TEST para verificar si las operaciones de archivos tienen éxito.

OPEN "nombrearchivo":("R"):1
IF '$TEST DO  QUIT
. WRITE "Error al abrir el archivo",!
. ; Código adicional de manejo de errores

USE "nombrearchivo"
READ linea
IF '$TEST DO  QUIT
. WRITE "Error al leer el archivo",!
. ; Código adicional de manejo de errores

CLOSE "nombrearchivo"

Trabajar con Múltiples Archivos

Puedes trabajar con múltiples archivos simultáneamente usando diferentes números de dispositivo.

OPEN "archivo1":("R"):1
IF '$TEST WRITE "Error al abrir archivo1",! QUIT

OPEN "archivo2":("W"):2
IF '$TEST WRITE "Error al abrir archivo2",! QUIT

USE "archivo1"
READ linea
USE "archivo2"
WRITE linea,!

CLOSE "archivo1"
CLOSE "archivo2"

Resumen

En este tema, cubrimos lo esencial del manejo de archivos en MUMPS, incluyendo:

  • Abrir y cerrar archivos.
  • Leer y escribir archivos.
  • Añadir datos a archivos.
  • Manejar errores durante las operaciones de archivos.
  • Trabajar con múltiples archivos simultáneamente.

Entender estos conceptos es crucial para una gestión efectiva de datos en MUMPS. Practica estas operaciones con diferentes archivos para volverte competente en el manejo de archivos.
