Introducción

En esta sección, cubriremos los conceptos básicos de variables y tipos de datos en MUMPS (M). Comprender estos conceptos fundamentales es crucial para escribir código MUMPS efectivo y eficiente. Comenzaremos con una introducción a las variables, seguida de una exploración de los diferentes tipos de datos disponibles en MUMPS.

Variables

Las variables en MUMPS se utilizan para almacenar datos que pueden ser manipulados y recuperados durante la ejecución de un programa. A diferencia de muchos otros lenguajes de programación, MUMPS no requiere la declaración explícita de variables antes de su uso.

Conceptos Clave

  • Declaración Implícita: Las variables se crean cuando se les asigna un valor por primera vez.
  • Sensibilidad a Mayúsculas y Minúsculas: Los nombres de las variables en MUMPS distinguen entre mayúsculas y minúsculas.
  • Convenciones de Nomenclatura: Los nombres de las variables deben comenzar con una letra y pueden ser seguidos por letras, números o el carácter de subrayado.

Ejemplos

SET name="John Doe"
SET age=30
SET _score=95

En los ejemplos anteriores:

  • name es una variable que almacena un valor de cadena.
  • age es una variable que almacena un valor entero.
  • _score es una variable que almacena un valor entero.

Tipos de Datos

MUMPS soporta varios tipos de datos, pero principalmente maneja cadenas y números. El lenguaje está diseñado para manejar datos dinámicamente, por lo que el tipo de datos que una variable contiene puede cambiar durante la ejecución.

Tipos de Datos Clave

  • Cadena: Una secuencia de caracteres.
  • Número: Puede ser un entero o un número de punto flotante.

Tipo de Dato Cadena

Las cadenas en MUMPS están encerradas en comillas dobles. Pueden contener cualquier carácter, incluidos números y caracteres especiales.

Ejemplos

SET greeting="Hello, World!"
SET address="123 Main St."

Tipo de Dato Número

Los números en MUMPS pueden ser enteros o números de punto flotante. No están encerrados en comillas.

Ejemplos

SET count=100
SET price=19.99

Tipado Dinámico

Las variables en MUMPS pueden cambiar de tipo dinámicamente según el valor asignado.

Ejemplo

SET var="123"  ; var es una cadena
SET var=123    ; var ahora es un número

Operaciones en Variables

MUMPS permite varias operaciones en variables, incluidas operaciones aritméticas, concatenación de cadenas y operaciones lógicas.

Operaciones Aritméticas

Puedes realizar operaciones aritméticas estándar en variables numéricas.

Ejemplos

SET a=10
SET b=5
SET sum=a+b       ; sum es 15
SET difference=a-b ; difference es 5
SET product=a*b   ; product es 50
SET quotient=a/b  ; quotient es 2

Concatenación de Cadenas

Las cadenas pueden ser concatenadas usando el operador de subrayado (_).

Ejemplo

SET firstName="John"
SET lastName="Doe"
SET fullName=firstName_" "_lastName  ; fullName es "John Doe"

Operaciones Lógicas

Las operaciones lógicas pueden ser realizadas usando operadores lógicos estándar.

Ejemplos

SET x=1
SET y=0
SET andResult=x&y  ; andResult es 0 (falso)
SET orResult=x!y   ; orResult es 1 (verdadero)

Resumen

En esta sección, hemos cubierto los conceptos básicos de variables y tipos de datos en MUMPS. Aprendimos que las variables en MUMPS se declaran implícitamente y pueden cambiar de tipo dinámicamente. También exploramos los principales tipos de datos en MUMPS: cadenas y números, y vimos cómo realizar varias operaciones en estas variables.

Comprender estos conceptos fundamentales es esencial para progresar a temas más avanzados en la programación MUMPS. Practica creando y manipulando variables para sentirte más cómodo con estos conceptos.
