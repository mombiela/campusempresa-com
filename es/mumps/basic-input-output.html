Introducción

En esta sección, cubriremos los conceptos básicos de las operaciones de entrada y salida en MUMPS (M). Entender cómo manejar la entrada y salida es crucial para interactuar con los usuarios y gestionar datos dentro de tus programas.

Conceptos Clave

  • Entrada: Leer datos del usuario u otra fuente.
  • Salida: Mostrar datos al usuario u otro destino.

Operaciones de Entrada

Usando el Comando `READ`

El comando READ se utiliza para tomar entrada del usuario.

Sintaxis

READ variable

Ejemplo

READ "Ingrese su nombre: ", name
WRITE "¡Hola, ", name, "!"

En este ejemplo:

  • El comando READ solicita al usuario que ingrese su nombre.
  • El comando WRITE luego muestra un mensaje de saludo usando la entrada.

Leyendo Múltiples Variables

Puedes leer múltiples variables en un solo comando READ.

Ejemplo

READ "Ingrese su primer nombre: ", firstName, " Ingrese su apellido: ", lastName
WRITE "¡Hola, ", firstName, " ", lastName, "!"

En este ejemplo:

  • Se solicita al usuario que ingrese su primer nombre y apellido.
  • El comando WRITE muestra un mensaje de saludo usando ambas entradas.

Operaciones de Salida

Usando el Comando `WRITE`

El comando WRITE se utiliza para mostrar datos al usuario.

Sintaxis

WRITE expression

Ejemplo

SET greeting = "¡Hola, Mundo!"
WRITE greeting

En este ejemplo:

  • El comando SET asigna la cadena "¡Hola, Mundo!" a la variable greeting.
  • El comando WRITE muestra el valor de greeting.

Formateando la Salida

Puedes formatear la salida usando caracteres especiales y expresiones.

Ejemplo

SET age = 25
WRITE "Tienes ", age, " años."

En este ejemplo:

  • El comando WRITE muestra un mensaje formateado que incluye el valor de age.

Combinando Entrada y Salida

Programa de Ejemplo

Aquí hay un programa simple que combina operaciones de entrada y salida:

READ "Ingrese su edad: ", age
IF age<18 WRITE "Eres menor de edad."
ELSE  WRITE "Eres mayor de edad."

En este ejemplo:

  • El comando READ solicita al usuario que ingrese su edad.
  • La declaración IF verifica si la edad es menor de 18.
  • Dependiendo de la condición, el comando WRITE muestra un mensaje apropiado.

Conclusión

En esta sección, hemos cubierto los conceptos básicos de las operaciones de entrada y salida en MUMPS. Aprendiste cómo usar el comando READ para tomar entrada del usuario y el comando WRITE para mostrar salida. También exploramos cómo formatear la salida y combinar operaciones de entrada y salida en un programa simple. Dominar estos conceptos básicos te permitirá crear programas interactivos en MUMPS que puedan comunicarse efectivamente con los usuarios.
