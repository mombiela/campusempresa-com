Resumen

SQLAlchemy es una poderosa herramienta de SQL y una biblioteca de Mapeo Objeto-Relacional (ORM) para Python. Proporciona un conjunto completo de patrones de persistencia a nivel empresarial bien conocidos, diseñados para un acceso a la base de datos eficiente y de alto rendimiento. En esta sección, cubriremos los conceptos básicos de SQLAlchemy y cómo integrarlo con una aplicación Flask.

Conceptos Clave

  • SQLAlchemy Core: La capa fundamental que proporciona un lenguaje de expresión SQL y un enfoque centrado en el esquema para la interacción con la base de datos.
  • SQLAlchemy ORM: Una capa opcional que proporciona una interfaz orientada a objetos de alto nivel para las operaciones de la base de datos.
  • Engine: El punto de partida para cualquier aplicación SQLAlchemy, que representa la conexión a la base de datos.
  • Session: Un espacio de trabajo para todos los objetos cargados desde la base de datos durante una transacción.
  • Model: Una clase de Python que se mapea a una tabla de la base de datos.

Configuración de SQLAlchemy con Flask

Instalación

Para comenzar, necesitas instalar Flask y SQLAlchemy. Puedes hacerlo usando pip:

pip install Flask SQLAlchemy

Configuración

Primero, configura tu aplicación Flask para usar SQLAlchemy. Aquí tienes un ejemplo básico:

from flask import Flask
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///example.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False

db = SQLAlchemy(app)
  • SQLALCHEMY_DATABASE_URI: Especifica la URL de la base de datos.
  • SQLALCHEMY_TRACK_MODIFICATIONS: Desactiva una característica que señala a la aplicación cada vez que se va a realizar un cambio en la base de datos, lo cual puede ser útil para la depuración pero no se recomienda para producción.

Definiendo Modelos

Los modelos son clases de Python que representan tablas de la base de datos. Aquí tienes un ejemplo de un modelo simple:

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

    def __repr__(self):
        return f'<User {self.username}>'
  • db.Column: Define una columna en la tabla de la base de datos.
  • db.Integer, db.String: Especifican el tipo de datos de la columna.
  • primary_key=True: Indica que esta columna es la clave primaria.
  • unique=True: Asegura que todos los valores en esta columna sean únicos.
  • nullable=False: Asegura que esta columna no pueda ser nula.

Creando la Base de Datos

Para crear la base de datos y las tablas, necesitas ejecutar los siguientes comandos en una consola de Python:

from your_application import db
db.create_all()

Esto creará las tablas definidas por tus modelos en la base de datos especificada por SQLALCHEMY_DATABASE_URI.

Operaciones CRUD

Crear

Para agregar un nuevo registro a la base de datos:

new_user = User(username='john_doe', email='john@example.com')
db.session.add(new_user)
db.session.commit()

Leer

Para consultar registros de la base de datos:

users = User.query.all()  # Obtener todos los usuarios
user = User.query.filter_by(username='john_doe').first()  # Obtener un usuario específico

Actualizar

Para actualizar un registro existente:

user = User.query.filter_by(username='john_doe').first()
user.email = 'john_doe@example.com'
db.session.commit()

Eliminar

Para eliminar un registro:

user = User.query.filter_by(username='john_doe').first()
db.session.delete(user)
db.session.commit()

Conclusión

En esta sección, presentamos SQLAlchemy y demostramos cómo integrarlo con una aplicación Flask. Cubrimos los conceptos básicos de la configuración de SQLAlchemy, la definición de modelos y la realización de operaciones CRUD. SQLAlchemy es una herramienta poderosa que puede simplificar en gran medida las interacciones con la base de datos en tus aplicaciones Flask. A medida que avances, aprenderás características más avanzadas y mejores prácticas para usar SQLAlchemy de manera efectiva.
