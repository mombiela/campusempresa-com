¿Qué es Flask?
Configuración de tu Entorno de Desarrollo ►

Flask es un micro framework web escrito en Python. Está diseñado para ser ligero y modular, lo que facilita escalar hacia aplicaciones complejas o reducirse a aplicaciones simples. Flask es conocido por su simplicidad, flexibilidad y control detallado.

Características Clave de Flask

  • Ligero y Modular: Flask es minimalista y no viene con muchas características integradas, permitiendo a los desarrolladores añadir solo lo que necesitan.
  • Servidor de Desarrollo Integrado: Flask incluye un servidor integrado para desarrollo y pruebas.
  • Despacho de Solicitudes RESTful: Flask soporta el despacho de solicitudes RESTful, facilitando la creación de APIs.
  • Plantillas Jinja2: Flask utiliza Jinja2 para plantillas, permitiendo la generación dinámica de HTML.
  • Extensible: Flask puede ser extendido con varios plugins y extensiones para añadir funcionalidad.
  • Pruebas Unitarias Integradas: Flask soporta pruebas unitarias, facilitando la prueba de aplicaciones.

¿Por Qué Usar Flask?

  • Simplicidad: Flask es fácil de aprender y usar, lo que lo convierte en una excelente opción para principiantes.
  • Flexibilidad: Flask permite a los desarrolladores elegir los componentes que desean usar, proporcionando un mayor control sobre la aplicación.
  • Comunidad y Documentación: Flask tiene una gran comunidad y una extensa documentación, lo cual puede ser muy útil para la resolución de problemas y el aprendizaje.

Aplicación Básica de Flask

Comencemos con un ejemplo simple para demostrar cómo funciona Flask. Este ejemplo creará una aplicación web básica que muestra "¡Hola, Mundo!" cuando se accede.

Guía Paso a Paso

  1. Instalar Flask: Primero, necesitas instalar Flask. Puedes hacerlo usando pip:

    pip install Flask

  2. Crear una Aplicación Flask: Crea un nuevo archivo Python, por ejemplo, app.py, y añade el siguiente código:

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return '¡Hola, Mundo!'

if __name__ == '__main__':
    app.run(debug=True)

  3. Ejecutar la Aplicación: Ejecuta la aplicación ejecutando el siguiente comando en tu terminal:

    python app.py

  4. Acceder a la Aplicación: Abre tu navegador web y ve a http://127.0.0.1:5000/. Deberías ver "¡Hola, Mundo!" mostrado en la página.

Explicación del Código

  • Importando Flask: La línea from flask import Flask importa la clase Flask del módulo Flask.
  • Creando una Instancia de la Aplicación: app = Flask(__name__) crea una instancia de la clase Flask. La variable __name__ se pasa al constructor de Flask para ayudar a Flask a determinar la ruta raíz de la aplicación.
  • Definiendo una Ruta: El decorador @app.route('/') define una ruta para la URL raíz (/). La función hello_world se llama cuando se accede a esta URL, y devuelve la cadena "¡Hola, Mundo!".
  • Ejecutando la Aplicación: El bloque if __name__ == '__main__': asegura que la aplicación se ejecute solo si el script se ejecuta directamente (no importado como un módulo). app.run(debug=True) inicia el servidor de desarrollo con el modo de depuración habilitado.

Conclusión

Flask es un framework web poderoso pero simple para Python que permite a los desarrolladores crear aplicaciones web con una configuración mínima. Su naturaleza ligera y flexibilidad lo convierten en una excelente opción tanto para principiantes como para desarrolladores experimentados. En esta sección, cubrimos los conceptos básicos de lo que es Flask, sus características clave y cómo crear una aplicación Flask simple. A medida que avancemos en este curso, profundizaremos en temas y características más avanzadas de Flask, construyendo sobre este conocimiento fundamental.
