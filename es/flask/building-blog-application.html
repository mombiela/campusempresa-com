Introducción

En este curso, construiremos una aplicación de blog completa usando Flask, un popular micro framework web para Python. Este curso está diseñado para llevarte de un nivel principiante a un nivel avanzado, cubriendo todos los aspectos esenciales del desarrollo con Flask.

Configuración del Entorno

  • Instalar Flask: Usa pip para instalar Flask. 
    pip install Flask
  • Estructura del Proyecto: Organiza tus archivos del proyecto. 
    blog/
├── app.py
├── templates/
│   ├── base.html
│   ├── index.html
│   ├── post.html
├── static/
│   ├── style.css
└── models.py

Creando la Aplicación Flask

  • Aplicación Flask Básica: Crea una aplicación Flask simple. 
    from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Hola, Blog!"

if __name__ == '__main__':
    app.run(debug=True)

Plantillas y Archivos Estáticos

  • Plantillas HTML: Usa el motor de plantillas Jinja2. 
    <!-- templates/base.html -->
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{% block title %}Blog{% endblock %}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">
</head>
<body>
    {% block content %}{% endblock %}
</body>
</html>
  • Archivos Estáticos: Sirve archivos CSS y JavaScript. 
    /* static/style.css */
body {
    font-family: Arial, sans-serif;
}

Rutas y Vistas

  • Rutas Dinámicas: Crea rutas con segmentos dinámicos. 
    @app.route('/post/<int:post_id>')
def post(post_id):
    return f"Post {post_id}"

Modelos e Integración de Base de Datos

  • Configuración de SQLAlchemy: Integra SQLAlchemy para la gestión de bases de datos.

    from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///blog.db'
db = SQLAlchemy(app)

  • Creando Modelos: Define modelos para las publicaciones del blog.

    # models.py
from app import db

class Post(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(100), nullable=False)
    content = db.Column(db.Text, nullable=False)
    date_posted = db.Column(db.DateTime, nullable=False, default=datetime.utcnow)

Formularios e Ingreso de Usuario

  • WTForms: Usa Flask-WTF para el manejo de formularios. 
    from flask_wtf import FlaskForm
from wtforms import StringField, TextAreaField, SubmitField
from wtforms.validators import DataRequired

class PostForm(FlaskForm):
    title = StringField('Title', validators=[DataRequired()])
    content = TextAreaField('Content', validators=[DataRequired()])
    submit = SubmitField('Post')

Autenticación de Usuario

  • Flask-Login: Implementa la autenticación de usuario. 
    from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

login_manager = LoginManager()
login_manager.init_app(app)

class User(UserMixin, db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(150), unique=True, nullable=False)
    password = db.Column(db.String(150), nullable=False)

Funciones Avanzadas

  • Paginación: Implementa la paginación para las publicaciones del blog.

    @app.route('/posts')
def posts():
    page = request.args.get('page', 1, type=int)
    posts = Post.query.paginate(page=page, per_page=5)
    return render_template('posts.html', posts=posts)

  • Funcionalidad de Búsqueda: Añade funcionalidad de búsqueda al blog.

    @app.route('/search')
def search():
    query = request.args.get('query')
    results = Post.query.filter(Post.title.contains(query) | Post.content.contains(query)).all()
    return render_template('search_results.html', results=results)

Despliegue

  • Desplegando en Heroku: Pasos para desplegar tu aplicación Flask en Heroku. 
    heroku create
git push heroku master
heroku open

Conclusión

En este curso, hemos construido una aplicación de blog completamente funcional usando Flask. Cubrimos los conceptos básicos de la configuración de un entorno Flask, la creación de rutas y vistas, la integración de una base de datos, el manejo de la entrada del usuario, la implementación de la autenticación y la adición de funciones avanzadas como la paginación y la búsqueda. Finalmente, desplegamos nuestra aplicación en Heroku. Esta guía completa debería darte una base sólida para construir y expandir tus propias aplicaciones Flask.
