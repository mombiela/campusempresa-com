Configuración de tu Entorno de Desarrollo
Creando tu Primera Aplicación Flask ►

Configurar tu entorno de desarrollo es el primer paso para comenzar a construir aplicaciones web con Flask. Esta sección te guiará a través del proceso de instalación de las herramientas necesarias y la configuración de tu entorno para el desarrollo con Flask.

Requisitos Previos

Antes de comenzar, asegúrate de tener los siguientes requisitos previos instalados en tu máquina:

  • Python: Flask es un framework web de Python, por lo que necesitas tener Python instalado. Flask soporta Python 3.6 y versiones superiores.
  • pip: El instalador de paquetes de Python, que se utiliza para instalar Flask y otras dependencias.

Instalando Python

Si no tienes Python instalado, sigue estos pasos:

  1. Descargar Python: Visita el sitio web oficial de Python y descarga la última versión de Python.
  2. Instalar Python: Ejecuta el instalador y sigue las instrucciones. Asegúrate de marcar la opción para agregar Python a tu PATH del sistema durante la instalación.

Configurando un Entorno Virtual

Un entorno virtual es un directorio autónomo que contiene una instalación de Python para una versión particular de Python, además de varios paquetes adicionales. Usar un entorno virtual asegura que las dependencias de tu proyecto estén aisladas de otros proyectos.

  1. Crear un Entorno Virtual:

    python -m venv venv

    Este comando crea un directorio llamado venv en tu carpeta de proyecto.

  2. Activar el Entorno Virtual:

    • En Windows: 
      venv\Scripts\activate
    • En macOS y Linux: 
      source venv/bin/activate

  3. Desactivar el Entorno Virtual: Para desactivar el entorno virtual, simplemente ejecuta:

    deactivate

Instalando Flask

Con tu entorno virtual activado, ahora puedes instalar Flask usando pip:

pip install Flask

Para verificar la instalación, puedes ejecutar:

python -m flask --version

Creando tu Primera Aplicación Flask

Ahora que Flask está instalado, vamos a crear una simple aplicación Flask para asegurarnos de que todo está configurado correctamente.

  1. Crea un nuevo archivo llamado app.py en tu directorio de proyecto.

  2. Agrega el siguiente código a app.py:

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

  3. Ejecuta la aplicación Flask:

    python app.py

    Deberías ver una salida similar a esta:

    * Serving Flask app "app" (lazy loading)
* Environment: production
  WARNING: This is a development server. Do not use it in a production deployment.
  Use a production WSGI server instead.
* Debug mode: on
* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

  4. Abre tu navegador web y navega a http://127.0.0.1:5000/. Deberías ver "Hello, World!" mostrado en la página.

Resumen

En esta sección, has aprendido a:

  • Instalar Python y pip.
  • Configurar un entorno virtual para gestionar las dependencias de tu proyecto.
  • Instalar Flask usando pip.
  • Crear y ejecutar una simple aplicación Flask.

Con tu entorno de desarrollo configurado, ahora estás listo para profundizar en Flask y comenzar a construir aplicaciones web más complejas.
