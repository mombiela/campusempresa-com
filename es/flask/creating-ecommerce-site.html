Introducción

En este curso, aprenderemos a crear un sitio de comercio electrónico completamente funcional utilizando Flask, un marco web ligero para Python. Este curso está diseñado para llevarte de un nivel principiante a un nivel avanzado, cubriendo todos los aspectos esenciales del desarrollo web con Flask.

Configuración del Entorno

Antes de comenzar a construir nuestro sitio de comercio electrónico, necesitamos configurar nuestro entorno de desarrollo.

  • Instalar Python: Asegúrate de tener Python instalado en tu máquina.
  • Instalar Flask: Usa pip para instalar Flask. 
    pip install Flask
  • Crear un Directorio de Proyecto: Configura un directorio para tu proyecto. 
    mkdir ecommerce_site
cd ecommerce_site

Aplicación Básica de Flask

Comencemos con una aplicación simple de Flask para entender los conceptos básicos.

  • Crear una Aplicación Básica de Flask: Crea un archivo llamado app.py.

    from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Bienvenido al Sitio de Comercio Electrónico!"

if __name__ == '__main__':
    app.run(debug=True)

  • Ejecutar la Aplicación: Ejecuta la aplicación.

    python app.py

Plantillas y Archivos Estáticos

Flask utiliza plantillas Jinja2 para renderizar HTML. Crearemos plantillas para nuestro sitio de comercio electrónico.

  • Crear Directorio de Plantillas: Crea un directorio llamado templates.

    mkdir templates

  • Crear una Plantilla Base: Crea un archivo llamado base.html en el directorio templates.

    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{% block title %}Sitio de Comercio Electrónico{% endblock %}</title>
</head>
<body>
    <header>
        <h1>Sitio de Comercio Electrónico</h1>
    </header>
    <main>
        {% block content %}{% endblock %}
    </main>
</body>
</html>

  • Crear una Plantilla de Inicio: Crea un archivo llamado home.html en el directorio templates.

    {% extends "base.html" %}

{% block title %}Inicio{% endblock %}

{% block content %}
<h2>¡Bienvenido al Sitio de Comercio Electrónico!</h2>
{% endblock %}

  • Actualizar app.py para Usar Plantillas:

    from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def home():
    return render_template('home.html')

if __name__ == '__main__':
    app.run(debug=True)

Integración de Base de Datos

Usaremos SQLite para nuestra base de datos. Flask-SQLAlchemy es una extensión que simplifica la integración de bases de datos.

  • Instalar Flask-SQLAlchemy:

    pip install Flask-SQLAlchemy

  • Configurar la Base de Datos:

    from flask import Flask, render_template
from flask_sqlalchemy import SQLAlchemy

app = Flask(__name__)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///ecommerce.db'
db = SQLAlchemy(app)

@app.route('/')
def home():
    return render_template('home.html')

if __name__ == '__main__':
    app.run(debug=True)

  • Crear Modelos: Define modelos para productos y usuarios.

    class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

class Product(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), nullable=False)
    price = db.Column(db.Float, nullable=False)

  • Crear la Base de Datos:

    from app import db
db.create_all()

Autenticación de Usuarios

Implementa la funcionalidad de registro e inicio de sesión de usuarios.

  • Instalar Flask-Login:

    pip install Flask-Login

  • Configurar la Autenticación de Usuarios:

    from flask_login import LoginManager, UserMixin, login_user, login_required, logout_user, current_user

login_manager = LoginManager()
login_manager.init_app(app)

class User(UserMixin, db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)
    password = db.Column(db.String(120), nullable=False)

@login_manager.user_loader
def load_user(user_id):
    return User.query.get(int(user_id))

  • Crear Formularios de Registro e Inicio de Sesión:

    from flask_wtf import FlaskForm
from wtforms import StringField, PasswordField, SubmitField
from wtforms.validators import DataRequired, Email, EqualTo

class RegistrationForm(FlaskForm):
    username = StringField('Nombre de Usuario', validators=[DataRequired()])
    email = StringField('Correo Electrónico', validators=[DataRequired(), Email()])
    password = PasswordField('Contraseña', validators=[DataRequired()])
    confirm_password = PasswordField('Confirmar Contraseña', validators=[DataRequired(), EqualTo('password')])
    submit = SubmitField('Registrarse')

class LoginForm(FlaskForm):
    email = StringField('Correo Electrónico', validators=[DataRequired(), Email()])
    password = PasswordField('Contraseña', validators=[DataRequired()])
    submit = SubmitField('Iniciar Sesión')

  • Crear Rutas de Registro e Inicio de Sesión:

    from flask import Flask, render_template, redirect, url_for, flash
from flask_login import login_user, login_required, logout_user, current_user
from werkzeug.security import generate_password_hash, check_password_hash

@app.route('/register', methods=['GET', 'POST'])
def register():
    form = RegistrationForm()
    if form.validate_on_submit():
        hashed_password = generate_password_hash(form.password.data, method='sha256')
        new_user = User(username=form.username.data, email=form.email.data, password=hashed_password)
        db.session.add(new_user)
        db.session.commit()
        flash('¡Tu cuenta ha sido creada!', 'success')
        return redirect(url_for('login'))
    return render_template('register.html', form=form)

@app.route('/login', methods=['GET', 'POST'])
def login():
    form = LoginForm()
    if form.validate_on_submit():
        user = User.query.filter_by(email=form.email.data).first()
        if user and check_password_hash(user.password, form.password.data):
            login_user(user)
            return redirect(url_for('home'))
        else:
            flash('Inicio de sesión fallido. Por favor, verifica el correo electrónico y la contraseña', 'danger')
    return render_template('login.html', form=form)

@app.route('/logout')
@login_required
def logout():
    logout_user()
    return redirect(url_for('home'))

Gestión de Productos

Implementa la funcionalidad para agregar, ver y gestionar productos.

  • Crear Formulario de Producto:

    class ProductForm(FlaskForm):
    name = StringField('Nombre del Producto', validators=[DataRequired()])
    price = StringField('Precio', validators=[DataRequired()])
    submit = SubmitField('Agregar Producto')

  • Crear Rutas de Producto:

    @app.route('/product/new', methods=['GET', 'POST'])
@login_required
def new_product():
    form = ProductForm()
    if form.validate_on_submit():
        product = Product(name=form.name.data, price=form.price.data)
        db.session.add(product)
        db.session.commit()
        flash('¡El producto ha sido agregado!', 'success')
        return redirect(url_for('home'))
    return render_template('create_product.html', form=form)

@app.route('/products')
def products():
    products = Product.query.all()
    return render_template('products.html', products=products)
