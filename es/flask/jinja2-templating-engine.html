Introducción a Jinja2

Jinja2 es un potente motor de plantillas para Python, utilizado extensamente con el framework web Flask para crear páginas web dinámicas. Te permite incrustar expresiones similares a Python en tus archivos HTML, facilitando la generación dinámica de HTML.

Conceptos Clave

  • Plantillas: Archivos HTML con sintaxis Jinja2 incrustada.
  • Variables: Marcadores de posición para contenido dinámico.
  • Estructuras de Control: Lógica para bucles y condicionales.
  • Filtros: Funciones para modificar variables.
  • Macros: Bloques de código reutilizables.

Configuración de Jinja2 en Flask

Para usar Jinja2 con Flask, necesitas configurar correctamente la estructura de tu proyecto.

Estructura del Proyecto

mi_app_flask/
│
├── app.py
├── templates/
│   ├── base.html
│   └── index.html
└── static/
    └── style.css

Aplicación Básica de Flask

from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html', title='Página Principal')

if __name__ == '__main__':
    app.run(debug=True)

Uso de Variables en Plantillas

Las variables en Jinja2 están encerradas en dobles llaves {{ }}.

Ejemplo

<!-- index.html -->
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>{{ title }}</title>
</head>
<body>
    <h1>Bienvenido a {{ title }}</h1>
</body>
</html>

Explicación

  • {{ title }}: Esto será reemplazado por el valor de la variable title pasada desde la ruta de Flask.

Estructuras de Control

Jinja2 soporta estructuras de control como bucles y condicionales.

Bucles

<!-- index.html -->
<ul>
    {% for item in items %}
        <li>{{ item }}</li>
    {% endfor %}
</ul>

Condicionales

<!-- index.html -->
{% if user %}
    <p>Bienvenido, {{ user }}!</p>
{% else %}
    <p>Bienvenido, Invitado!</p>
{% endif %}

Explicación

  • {% for item in items %}: Itera sobre cada elemento en la lista items.
  • {% if user %}: Verifica si la variable user está definida.

Filtros

Los filtros se utilizan para modificar variables. Se aplican usando el símbolo de tubería |.

Ejemplo

<!-- index.html -->
<p>{{ name | upper }}</p>
<p>{{ date | dateformat('%Y-%m-%d') }}</p>

Explicación

  • {{ name | upper }}: Convierte la variable name a mayúsculas.
  • {{ date | dateformat('%Y-%m-%d') }}: Formatea la variable date.

Macros

Los macros son bloques de código reutilizables que pueden ser definidos y llamados dentro de las plantillas.

Ejemplo

<!-- macros.html -->
{% macro render_item(item) %}
    <li>{{ item }}</li>
{% endmacro %}
<!-- index.html -->
{% from 'macros.html' import render_item %}
<ul>
    {% for item in items %}
        {{ render_item(item) }}
    {% endfor %}
</ul>

Explicación

  • {% macro render_item(item) %}: Define un macro llamado render_item.
  • {% from 'macros.html' import render_item %}: Importa el macro render_item desde macros.html.

Conclusión

Jinja2 es una herramienta esencial para crear páginas web dinámicas en Flask. Al entender sus conceptos clave—plantillas, variables, estructuras de control, filtros y macros—puedes construir aplicaciones web más interactivas y amigables para el usuario. Practica usando estas características para volverte competente en Jinja2 y mejorar tus proyectos de Flask.
