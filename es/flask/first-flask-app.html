Introducción

Flask es un micro framework web escrito en Python. Está diseñado para ser simple y fácil de usar, lo que lo convierte en una excelente opción para principiantes que desean comenzar con el desarrollo web. En esta sección, te guiaremos a través de la creación de tu primera aplicación Flask.

Configurando Tu Entorno

Antes de comenzar a codificar, necesitamos configurar nuestro entorno de desarrollo.

Instalando Flask

Para instalar Flask, necesitas tener Python y pip (el instalador de paquetes de Python) instalados en tu sistema. Puedes instalar Flask usando pip con el siguiente comando:

pip install Flask

Creando un Directorio de Proyecto

Crea un directorio para tu proyecto Flask. Esto te ayudará a mantener tus archivos de proyecto organizados.

mkdir my_flask_app
cd my_flask_app

Escribiendo Tu Primera Aplicación Flask

Ahora que tenemos nuestro entorno configurado, escribamos nuestra primera aplicación Flask.

Creando el Archivo de la Aplicación

Crea un nuevo archivo Python llamado app.py en tu directorio de proyecto. Este archivo contendrá el código para tu aplicación Flask.

# app.py
from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "¡Hola, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Explicación del Código

  • Importando Flask: Comenzamos importando la clase Flask del módulo flask.
  • Creando una Instancia de Flask: Creamos una instancia de la clase Flask. Esta instancia será nuestra aplicación WSGI.
  • Definiendo una Ruta: Usamos el decorador @app.route('/') para definir una ruta para la URL raíz (/). Cuando un usuario visite esta URL, se llamará a la función home.
  • Devolviendo una Respuesta: La función home devuelve una cadena simple "¡Hola, Flask!" que se mostrará en el navegador web del usuario.
  • Ejecutando la Aplicación: Finalmente, verificamos si el script se ejecuta directamente (no importado como un módulo) y llamamos a app.run(debug=True) para iniciar el servidor de desarrollo. El argumento debug=True habilita el modo de depuración, que proporciona mensajes de error útiles y recarga automáticamente el servidor cuando el código cambia.

Ejecutando Tu Aplicación Flask

Para ejecutar tu aplicación Flask, navega a tu directorio de proyecto y ejecuta el siguiente comando:

python app.py

Deberías ver una salida indicando que el servidor está en funcionamiento. Abre tu navegador web y ve a http://127.0.0.1:5000/. Deberías ver el mensaje "¡Hola, Flask!".

Resumen

En esta sección, cubrimos los conceptos básicos para crear una aplicación Flask simple. Configuramos nuestro entorno de desarrollo, escribimos una aplicación Flask básica y la ejecutamos en un servidor local. Este conocimiento fundamental servirá como base para temas más avanzados de Flask.
