Las cargas de archivos son una característica común en las aplicaciones web, permitiendo a los usuarios cargar archivos como imágenes, documentos y otros medios. Flask proporciona una manera sencilla de manejar cargas de archivos. Este tema te guiará desde lo básico hasta el manejo más avanzado de cargas de archivos en Flask.

Introducción a las Cargas de Archivos

  • ¿Qué son las cargas de archivos?
    • Las cargas de archivos permiten a los usuarios enviar archivos desde su máquina local a un servidor.
  • ¿Por qué son importantes las cargas de archivos?
    • Habilitan funcionalidades como actualizaciones de fotos de perfil, envío de documentos y compartición de medios.

Configurando Tu Aplicación Flask

Antes de manejar cargas de archivos, asegúrate de tener Flask instalado y configurado. Si no, puedes instalar Flask usando pip:

pip install Flask

Crea una aplicación básica de Flask:

from flask import Flask, render_template, request

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)

Manejo Básico de Cargas de Archivos

Configurando la Aplicación

Para manejar cargas de archivos, necesitas configurar tu aplicación Flask para especificar la carpeta de carga y las extensiones de archivo permitidas.

import os

app.config['UPLOAD_FOLDER'] = 'uploads/'
app.config['ALLOWED_EXTENSIONS'] = {'png', 'jpg', 'jpeg', 'gif'}

def allowed_file(filename):
    return '.' in filename and filename.rsplit('.', 1)[1].lower() in app.config['ALLOWED_EXTENSIONS']

Creando el Formulario de Carga

Crea un formulario HTML para permitir a los usuarios cargar archivos. Guarda esto como templates/index.html:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>File Upload</title>
  </head>
  <body>
    <h1>Upload a File</h1>
    <form method="post" enctype="multipart/form-data" action="/upload">
      <input type="file" name="file">
      <input type="submit" value="Upload">
    </form>
  </body>
</html>

Manejo de la Carga de Archivos

Añade una ruta para manejar la carga de archivos en tu aplicación Flask:

@app.route('/upload', methods=['POST'])
def upload_file():
    if 'file' not in request.files:
        return 'No file part'
    file = request.files['file']
    if file.filename == '':
        return 'No selected file'
    if file and allowed_file(file.filename):
        filename = secure_filename(file.filename)
        file.save(os.path.join(app.config['UPLOAD_FOLDER'], filename))
        return 'File successfully uploaded'
    return 'File type not allowed'

Explicación

  • request.files: Accede al archivo cargado.
  • secure_filename: Asegura que el nombre del archivo sea seguro para usar.
  • file.save: Guarda el archivo en la carpeta de carga especificada.

Manejo Avanzado de Cargas de Archivos

Validando el Tamaño del Archivo

Para prevenir que los usuarios carguen archivos excesivamente grandes, puedes validar el tamaño del archivo:

app.config['MAX_CONTENT_LENGTH'] = 16 * 1024 * 1024  # 16 MB

@app.errorhandler(413)
def request_entity_too_large(error):
    return 'File Too Large', 413

Almacenando Archivos en una Base de Datos

En lugar de guardar archivos en el sistema de archivos, puedes almacenarlos en una base de datos. Aquí hay un ejemplo usando SQLAlchemy:

from flask_sqlalchemy import SQLAlchemy

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///files.db'
db = SQLAlchemy(app)

class File(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(80), nullable=False)
    data = db.Column(db.LargeBinary, nullable=False)

@app.route('/upload', methods=['POST'])
def upload_file():
    file = request.files['file']
    if file and allowed_file(file.filename):
        new_file = File(name=secure_filename(file.filename), data=file.read())
        db.session.add(new_file)
        db.session.commit()
        return 'File successfully uploaded'
    return 'File type not allowed'

Sirviendo Archivos Cargados

Para servir archivos cargados, puedes crear una ruta para enviar el archivo al cliente:

from flask import send_from_directory

@app.route('/uploads/<filename>')
def uploaded_file(filename):
    return send_from_directory(app.config['UPLOAD_FOLDER'], filename)

Conclusión

Manejar cargas de archivos en Flask implica configurar tu aplicación, crear un formulario de carga y escribir rutas para manejar el proceso de carga de archivos. Las técnicas avanzadas incluyen validar el tamaño del archivo y almacenar archivos en una base de datos. Con estas habilidades, puedes implementar una funcionalidad robusta de carga de archivos en tus aplicaciones Flask.

Siguiendo esta guía, ahora deberías tener una comprensión sólida de cómo manejar cargas de archivos en Flask, desde configuraciones básicas hasta implementaciones más avanzadas.
