Introducción

En esta sección, cubriremos los aspectos esenciales del movimiento y controles del personaje en Unreal Engine. Este tema es crucial para crear una jugabilidad interactiva y atractiva. Comenzaremos con el movimiento básico y gradualmente avanzaremos hacia mecanismos de control más avanzados.

Movimiento Básico del Personaje

Configuración de un Personaje

  • Crear un Nuevo Blueprint de Personaje: Comienza creando un nuevo Blueprint de Personaje en Unreal Engine.
  • Agregar Componentes: Agrega los componentes necesarios como un Componente Cápsula para la colisión, un Mesh Esquelético para el modelo del personaje y un Componente de Movimiento de Personaje para manejar el movimiento.
// Ejemplo: Configuración de un personaje básico en C++
#include "GameFramework/Character.h"
#include "MyCharacter.generated.h"

UCLASS()
class AMyCharacter : public ACharacter
{
    GENERATED_BODY()

public:
    AMyCharacter();
};

Configuración del Movimiento

  • Componente de Movimiento del Personaje: Este componente maneja toda la lógica del movimiento.
    • Velocidad de Caminata: Ajusta la propiedad MaxWalkSpeed para establecer la velocidad de caminata del personaje.
    • Salto: Habilita el salto configurando JumpZVelocity.
// Ejemplo: Configuración del movimiento en C++
AMyCharacter::AMyCharacter()
{
    // Establecer velocidad de caminata por defecto
    GetCharacterMovement()->MaxWalkSpeed = 600.0f;

    // Habilitar salto
    GetCharacterMovement()->JumpZVelocity = 600.0f;
}

Configuración de Entrada

  • Configuración del Proyecto: Ve a Editar > Configuración del Proyecto > Entrada para configurar los mapeos de entrada.
    • Mapeos de Eje: Para entradas continuas como el movimiento (por ejemplo, MoveForward, MoveRight).
    • Mapeos de Acción: Para acciones discretas como saltar (por ejemplo, Jump).
// Ejemplo: Configuración de enlaces de entrada en C++
void AMyCharacter::SetupPlayerInputComponent(UInputComponent* PlayerInputComponent)
{
    Super::SetupPlayerInputComponent(PlayerInputComponent);

    // Enlazar entradas de movimiento
    PlayerInputComponent->BindAxis("MoveForward", this, &AMyCharacter::MoveForward);
    PlayerInputComponent->BindAxis("MoveRight", this, &AMyCharacter::MoveRight);

    // Enlazar entrada de salto
    PlayerInputComponent->BindAction("Jump", IE_Pressed, this, &ACharacter::Jump);
    PlayerInputComponent->BindAction("Jump", IE_Released, this, &ACharacter::StopJumping);
}

Implementación de Funciones de Movimiento

  • MoveForward y MoveRight: Implementa estas funciones para manejar el movimiento del personaje basado en la entrada.
// Ejemplo: Implementación de funciones de movimiento en C++
void AMyCharacter::MoveForward(float Value)
{
    if (Value != 0.0f)
    {
        AddMovementInput(GetActorForwardVector(), Value);
    }
}

void AMyCharacter::MoveRight(float Value)
{
    if (Value != 0.0f)
    {
        AddMovementInput(GetActorRightVector(), Value);
    }
}

Controles Intermedios del Personaje

Control de Cámara

  • Componente Spring Arm: Adjunta un Componente Spring Arm al personaje para controlar la posición y rotación de la cámara.
  • Componente Cámara: Adjunta un Componente Cámara al Spring Arm.
// Ejemplo: Configuración del control de cámara en C++
AMyCharacter::AMyCharacter()
{
    // Crear un Componente Spring Arm
    SpringArm = CreateDefaultSubobject<USpringArmComponent>(TEXT("SpringArm"));
    SpringArm->SetupAttachment(RootComponent);
    SpringArm->TargetArmLength = 300.0f;

    // Crear un Componente Cámara
    Camera = CreateDefaultSubobject<UCameraComponent>(TEXT("Camera"));
    Camera->SetupAttachment(SpringArm);
}

Mecánicas Avanzadas de Movimiento

  • Agacharse: Habilita el agacharse configurando CanCrouch a verdadero y enlazando entradas de agacharse.
  • Correr: Implementa correr modificando dinámicamente MaxWalkSpeed.
// Ejemplo: Implementación de agacharse y correr en C++
void AMyCharacter::SetupPlayerInputComponent(UInputComponent* PlayerInputComponent)
{
    Super::SetupPlayerInputComponent(PlayerInputComponent);

    // Enlazar entradas de agacharse
    PlayerInputComponent->BindAction("Crouch", IE_Pressed, this, &AMyCharacter::StartCrouch);
    PlayerInputComponent->BindAction("Crouch", IE_Released, this, &AMyCharacter::StopCrouch);

    // Enlazar entradas de correr
    PlayerInputComponent->BindAction("Sprint", IE_Pressed, this, &AMyCharacter::StartSprint);
    PlayerInputComponent->BindAction("Sprint", IE_Released, this, &AMyCharacter::StopSprint);
}

void AMyCharacter::StartCrouch()
{
    Crouch();
}

void AMyCharacter::StopCrouch()
{
    UnCrouch();
}

void AMyCharacter::StartSprint()
{
    GetCharacterMovement()->MaxWalkSpeed = 1200.0f;
}

void AMyCharacter::StopSprint()
{
    GetCharacterMovement()->MaxWalkSpeed = 600.0f;
}

Controles Avanzados del Personaje

Modos de Movimiento Personalizados

  • Volar: Implementa volar configurando el modo de movimiento a MOVE_Flying.
  • Nadar: Maneja nadar configurando el modo de movimiento a MOVE_Swimming.
// Ejemplo: Implementación de modos de movimiento personalizados en C++
void AMyCharacter::StartFlying()
{
    GetCharacterMovement()->SetMovementMode(MOVE_Flying);
}

void AMyCharacter::StopFlying()
{
    GetCharacterMovement()->SetMovementMode(MOVE_Walking);
}

Integración de Animaciones

  • Blueprints de Animación: Usa Blueprints de Animación para mezclar animaciones basadas en estados de movimiento.
  • Máquinas de Estado: Implementa máquinas de estado para manejar transiciones entre diferentes animaciones de movimiento.
// Ejemplo: Configuración básica de Blueprint de Animación
// En el Blueprint de Animación, crea una máquina de estado con estados como Idle, Walk, Run, Jump, etc.
// Usa Espacios de Mezcla para transiciones suaves entre animaciones basadas en velocidad y dirección.

Redes y Replicación

  • Replicación de Movimiento: Asegúrate de que el movimiento del personaje se replique a través de la red para juegos multijugador.
  • Validación del Lado del Servidor: Valida las entradas de movimiento en el servidor para prevenir trampas.
// Ejemplo: Habilitación de replicación de movimiento en C++
AMyCharacter::AMyCharacter()
{
    // Habilitar replicación
    bReplicates = true;
    bReplicateMovement = true;
}
