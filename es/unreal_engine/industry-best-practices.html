Mejores Prácticas de la Industria

Introducción

En esta sección, cubriremos las mejores prácticas de la industria para trabajar con Unreal Engine. Estas prácticas son esenciales para crear proyectos eficientes, mantenibles y escalables. Ya seas un principiante o un usuario avanzado, adherirse a estas directrices te ayudará a producir trabajos de alta calidad y colaborar eficazmente con otros.

Organización del Proyecto

  • Estructura de Carpetas: Organiza los archivos de tu proyecto en una estructura de carpetas clara y lógica. Esto facilita encontrar activos y gestionar tu proyecto.
    • Ejemplo: 
      /Content
  /Blueprints
  /Materials
  /Meshes
  /Textures
  /UI
  • Convenciones de Nombres: Usa convenciones de nombres consistentes para tus activos. Esto ayuda a identificar rápidamente el tipo y propósito de cada activo.
    • Ejemplo: 
      BP_PlayerCharacter
M_Wood
SM_Tree
T_Grass

Estándares de Codificación

  • Comentarios y Documentación: Siempre comenta tu código y proporciona documentación. Esto es crucial para mantener y entender la base de código.

    • Ejemplo: 
      // Esta función maneja el movimiento del jugador
void APlayerCharacter::MoveForward(float Value)
{
    if (Controller && Value != 0.0f)
    {
        // Descubre cuál es la dirección hacia adelante
        const FRotator Rotation = Controller->GetControlRotation();
        const FRotator YawRotation(0, Rotation.Yaw, 0);

        // Obtén el vector hacia adelante
        const FVector Direction = FRotationMatrix(YawRotation).GetUnitAxis(EAxis::X);
        AddMovementInput(Direction, Value);
    }
}

  • Estilo de Codificación Consistente: Sigue un estilo de codificación consistente en todo tu proyecto. Esto incluye la indentación, la colocación de llaves y las convenciones de nombres para variables y funciones.

    • Ejemplo: 
      // Bueno
void APlayerCharacter::Jump()
{
    Super::Jump();
}

// Malo
void APlayerCharacter::jump() {
    super::jump();
}

Optimización del Rendimiento

  • Nivel de Detalle (LOD): Usa LODs para tus mallas para reducir el número de polígonos renderizados a distancia.
  • Transmisión de Texturas: Habilita la transmisión de texturas para gestionar el uso de memoria de manera eficiente.
  • Herramientas de Perfilado: Usa las herramientas de perfilado de Unreal Engine para identificar y abordar cuellos de botella en el rendimiento.
    • Ejemplo:
      • Stat Unit: Muestra el tiempo de fotograma, el tiempo del hilo del juego, el tiempo del hilo de dibujo y el tiempo de la GPU.
      • Stat FPS: Muestra los fotogramas por segundo.

Gestión de Activos

  • Control de Versiones: Usa sistemas de control de versiones como Git o Perforce para gestionar los archivos de tu proyecto. Esto es esencial para la colaboración y el versionado.
  • Referenciación de Activos: Evita referencias directas a activos cuando sea posible. Usa referencias suaves o punteros de activos para reducir el uso de memoria y los tiempos de carga.

Mejores Prácticas de Blueprint

  • Blueprints Modulares: Crea Blueprints modulares que puedan ser reutilizados en diferentes partes de tu proyecto.
  • Arquitectura Basada en Eventos: Usa eventos y delegados para manejar interacciones entre Blueprints.
    • Ejemplo: 
      // Despachador de Eventos en Blueprint
Event OnPlayerHealthChanged

Pruebas y Depuración

  • Pruebas Automatizadas: Implementa pruebas automatizadas para asegurar que tu juego funcione correctamente.
  • Herramientas de Depuración: Usa las herramientas de depuración de Unreal Engine para solucionar problemas.
    • Ejemplo:
      • Puntos de Interrupción: Establece puntos de interrupción en tu código para pausar la ejecución e inspeccionar variables.
      • Salida de Registro: Usa UE_LOG para imprimir mensajes en el registro de salida con fines de depuración. 
        UE_LOG(LogTemp, Warning, TEXT("Player Health: %f"), PlayerHealth);

Conclusión

Adherirse a las mejores prácticas de la industria en el desarrollo de Unreal Engine es crucial para crear proyectos de alta calidad, mantenibles y eficientes. Al organizar tu proyecto, seguir estándares de codificación, optimizar el rendimiento, gestionar activos de manera efectiva y utilizar herramientas de prueba y depuración, puedes asegurarte de que tu trabajo cumpla con los estándares profesionales y esté listo para la colaboración y la escalabilidad.
