Introducción a la Criptografía

La criptografía es una disciplina fundamental en la seguridad informática que se enfoca en proteger la información mediante técnicas de cifrado y descifrado. En el contexto de OWASP (Open Web Application Security Project), la criptografía es crucial para asegurar la integridad, confidencialidad y autenticidad de los datos en aplicaciones web.

Conceptos Clave

  • Cifrado y Descifrado: Proceso de convertir datos legibles en un formato codificado (cifrado) y viceversa (descifrado).
  • Clave Criptográfica: Información utilizada en el proceso de cifrado y descifrado.
  • Algoritmos Simétricos: Utilizan la misma clave para cifrar y descifrar datos (e.g., AES).
  • Algoritmos Asimétricos: Utilizan un par de claves (pública y privada) para cifrar y descifrar datos (e.g., RSA).
  • Hashing: Proceso de convertir datos en una cadena de longitud fija que representa los datos originales.
  • Firma Digital: Método para verificar la autenticidad e integridad de un mensaje o documento.

Algoritmos Criptográficos

Algoritmos Simétricos

  • AES (Advanced Encryption Standard):
    • Utiliza claves de 128, 192 o 256 bits.
    • Es ampliamente utilizado debido a su eficiencia y seguridad.
from Crypto.Cipher import AES
from Crypto.Random import get_random_bytes

# Generar una clave aleatoria
key = get_random_bytes(16)  # AES-128

# Crear un objeto de cifrado
cipher = AES.new(key, AES.MODE_EAX)

# Cifrar datos
plaintext = b'Este es un mensaje secreto'
ciphertext, tag = cipher.encrypt_and_digest(plaintext)

print(f'Cifrado: {ciphertext}')

Algoritmos Asimétricos

  • RSA (Rivest-Shamir-Adleman):
    • Utiliza un par de claves (pública y privada).
    • Es comúnmente utilizado para el intercambio seguro de claves y firmas digitales.
from Crypto.PublicKey import RSA
from Crypto.Cipher import PKCS1_OAEP

# Generar un par de claves RSA
key = RSA.generate(2048)
public_key = key.publickey()

# Crear un objeto de cifrado con la clave pública
cipher = PKCS1_OAEP.new(public_key)

# Cifrar datos
plaintext = b'Este es un mensaje secreto'
ciphertext = cipher.encrypt(plaintext)

print(f'Cifrado: {ciphertext}')

Hashing

  • SHA-256 (Secure Hash Algorithm 256-bit):
    • Genera un hash de 256 bits.
    • Es ampliamente utilizado para verificar la integridad de los datos.
import hashlib

# Crear un objeto de hash
hash_object = hashlib.sha256()

# Actualizar el objeto de hash con los datos
hash_object.update(b'Este es un mensaje secreto')

# Obtener el hash
hash_digest = hash_object.hexdigest()

print(f'Hash: {hash_digest}')

Firma Digital

  • DSA (Digital Signature Algorithm):
    • Utilizado para firmar digitalmente y verificar la autenticidad de los mensajes.
from Crypto.Signature import DSS
from Crypto.Hash import SHA256
from Crypto.PublicKey import DSA

# Generar un par de claves DSA
key = DSA.generate(2048)

# Crear un objeto de hash
hash_object = SHA256.new(b'Este es un mensaje secreto')

# Firmar el hash
signer = DSS.new(key, 'fips-186-3')
signature = signer.sign(hash_object)

print(f'Firma: {signature}')

Buenas Prácticas en Criptografía

  • Uso de Algoritmos Seguros: Utilizar algoritmos criptográficos que sean considerados seguros por la comunidad de seguridad.
  • Gestión de Claves: Asegurar que las claves criptográficas sean gestionadas y almacenadas de manera segura.
  • Actualización Regular: Mantenerse actualizado con las últimas recomendaciones y actualizaciones de seguridad en criptografía.
  • Evitar Algoritmos Obsoletos: No utilizar algoritmos que hayan sido comprometidos o considerados inseguros (e.g., DES, MD5).

Conclusión

La criptografía es una herramienta esencial en la seguridad de aplicaciones web, proporcionando mecanismos para proteger la confidencialidad, integridad y autenticidad de los datos. Comprender y aplicar correctamente los conceptos y algoritmos criptográficos es crucial para cualquier profesional de TI que trabaje en el desarrollo y seguridad de aplicaciones. A través de este tema, hemos explorado los fundamentos de la criptografía, los principales algoritmos utilizados y las buenas prácticas para su implementación en el contexto de OWASP.
