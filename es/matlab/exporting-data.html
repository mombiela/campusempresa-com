Exportación de Datos
Trabajo con Diferentes Formatos de Archivos ►

La exportación de datos es una habilidad crucial en MATLAB, que te permite guardar tu trabajo y compartir resultados con otros. Este tema te guiará a través de varios métodos para exportar datos, desde técnicas básicas hasta avanzadas.

1. Exportación Básica de Datos

1.1 Exportación a Archivos de Texto

Los archivos de texto son un formato simple y ampliamente utilizado para exportar datos.

  • Función save: Guarda variables del espacio de trabajo en un archivo.
  • Función dlmwrite: Escribe datos de matriz en un archivo de texto con un delimitador especificado.
  • Función fprintf: Escribe datos formateados en un archivo de texto.

Ejemplo: Usando save

% Crear algunos datos
A = rand(5);
B = magic(5);

% Guardar datos en un archivo .mat
save('myData.mat', 'A', 'B');

Ejemplo: Usando dlmwrite

% Crear algunos datos
C = rand(5);

% Guardar datos en un archivo de texto con delimitador de coma
dlmwrite('myData.txt', C, 'delimiter', ',');

Ejemplo: Usando fprintf

% Crear algunos datos
D = rand(5);

% Abrir un archivo para escribir
fileID = fopen('myDataFormatted.txt', 'w');

% Escribir datos con salida formateada
fprintf(fileID, '%6.2f %6.2f %6.2f %6.2f %6.2f\n', D');

% Cerrar el archivo
fclose(fileID);

1.2 Exportación a Archivos de Excel

Los archivos de Excel son útiles para el análisis de datos y para compartir con personas que no son programadores.

  • Función xlswrite: Escribe datos en un archivo de Excel.

Ejemplo: Usando xlswrite

% Crear algunos datos
E = rand(5);

% Guardar datos en un archivo de Excel
xlswrite('myData.xlsx', E);

2. Exportación Intermedia de Datos

2.1 Exportación a Archivos CSV

Los archivos CSV (Valores Separados por Comas) son un formato común para el intercambio de datos.

  • Función csvwrite: Escribe datos de matriz en un archivo CSV.
  • Función writematrix: Escribe datos de matriz en un archivo, soportando varios formatos.

Ejemplo: Usando csvwrite

% Crear algunos datos
F = rand(5);

% Guardar datos en un archivo CSV
csvwrite('myData.csv', F);

Ejemplo: Usando writematrix

% Crear algunos datos
G = rand(5);

% Guardar datos en un archivo CSV
writematrix(G, 'myData.csv');

2.2 Exportación a Otros Formatos de Hojas de Cálculo

MATLAB soporta varios formatos de hojas de cálculo más allá de Excel.

  • Función writetable: Escribe datos de tabla en un archivo, soportando diferentes formatos como Excel, CSV y texto.

Ejemplo: Usando writetable

% Crear una tabla
T = table(rand(5,1), rand(5,1), 'VariableNames', {'Column1', 'Column2'});

% Guardar datos de tabla en un archivo de Excel
writetable(T, 'myTableData.xlsx');

% Guardar datos de tabla en un archivo CSV
writetable(T, 'myTableData.csv');

3. Exportación Avanzada de Datos

3.1 Exportación a Formatos Personalizados

Para aplicaciones especializadas, puede que necesites exportar datos en formatos personalizados.

  • Función fwrite: Escribe datos binarios en un archivo.
  • Función jsonencode: Codifica datos de MATLAB en formato JSON.
  • Función xmlwrite: Escribe datos en un archivo XML.

Ejemplo: Usando fwrite

% Crear algunos datos
H = rand(5);

% Abrir un archivo para escribir
fileID = fopen('myData.bin', 'w');

% Escribir datos binarios
fwrite(fileID, H, 'double');

% Cerrar el archivo
fclose(fileID);

Ejemplo: Usando jsonencode

% Crear una estructura
dataStruct = struct('Field1', rand(5,1), 'Field2', magic(5));

% Codificar estructura a JSON
jsonData = jsonencode(dataStruct);

% Escribir datos JSON en un archivo
fid = fopen('myData.json', 'w');
fwrite(fid, jsonData, 'char');
fclose(fid);

Ejemplo: Usando xmlwrite

% Crear un documento XML
docNode = com.mathworks.xml.XMLUtils.createDocument('root_element');
docRootNode = docNode.getDocumentElement;
docRootNode.appendChild(docNode.createElement('child_element'));

% Escribir datos XML en un archivo
xmlwrite('myData.xml', docNode);

Conclusión

La exportación de datos en MATLAB es un proceso versátil que puede adaptarse a diversas necesidades, desde archivos de texto simples hasta formatos personalizados complejos. Al dominar estas técnicas, puedes asegurarte de que tus datos se guarden, compartan y utilicen eficazmente en diferentes entornos.
