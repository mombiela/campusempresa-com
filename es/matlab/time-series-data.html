Introducción a los Datos de Series Temporales

Los datos de series temporales son una secuencia de puntos de datos recopilados o registrados en intervalos de tiempo específicos. Se utilizan ampliamente en diversos campos como finanzas, economía, estudios ambientales e ingeniería.

Conceptos Clave

  • Serie Temporal: Una serie de puntos de datos indexados en orden temporal.
  • Marca de Tiempo: El momento específico en el que se registra un punto de datos.
  • Frecuencia: El intervalo en el que se registran los puntos de datos (por ejemplo, diario, mensual).
  • Tendencia: El movimiento a largo plazo en una serie temporal.
  • Estacionalidad: Patrones o ciclos regulares en una serie temporal.

Trabajando con Datos de Series Temporales en MATLAB

Creación de Datos de Series Temporales

MATLAB proporciona varias formas de crear y manipular datos de series temporales. Las funciones datetime y timetable son particularmente útiles.

Ejemplo: Creación de una Serie Temporal Simple

% Crear un array datetime para las marcas de tiempo
timeStamps = datetime(2023,1,1):caldays(1):datetime(2023,1,10);

% Crear un array de datos correspondiente
data = rand(1, 10); % Datos aleatorios para ilustración

% Combinar en un timetable
tsData = timetable(timeStamps', data');
disp(tsData);

Explicación:

  • datetime(2023,1,1):caldays(1):datetime(2023,1,10) crea un rango de fechas desde el 1 de enero de 2023 hasta el 10 de enero de 2023.
  • rand(1, 10) genera puntos de datos aleatorios.
  • timetable(timeStamps', data') crea un timetable con las marcas de tiempo y los datos.

Graficar Datos de Series Temporales

Visualizar datos de series temporales es crucial para el análisis.

Ejemplo: Graficar Datos de Series Temporales

% Graficar los datos de la serie temporal
plot(tsData.timeStamps, tsData.data);
xlabel('Fecha');
ylabel('Valor');
title('Datos de Series Temporales');
grid on;

Explicación:

  • plot(tsData.timeStamps, tsData.data) grafica los puntos de datos contra sus marcas de tiempo.
  • xlabel, ylabel y title añaden etiquetas y un título al gráfico.
  • grid on añade una cuadrícula al gráfico para una mejor legibilidad.

Análisis Avanzado de Series Temporales

Descomposición de Series Temporales

Descomponer una serie temporal en sus componentes (tendencia, estacionalidad y residuos) ayuda a entender los patrones subyacentes.

Ejemplo: Descomposición de Series Temporales

% Generar una serie temporal de muestra con tendencia y estacionalidad
timeStamps = datetime(2023,1,1):caldays(1):datetime(2023,12,31);
data = 0.05*(1:length(timeStamps))' + 2*sin(2*pi*(1:length(timeStamps))'/365) + randn(length(timeStamps),1);

% Crear un timetable
tsData = timetable(timeStamps', data);

% Descomponer la serie temporal
[trend, seasonal, residual] = decompose(tsData.data, 'model', 'additive');

% Graficar los componentes
figure;
subplot(3,1,1);
plot(tsData.timeStamps, trend);
title('Tendencia');
subplot(3,1,2);
plot(tsData.timeStamps, seasonal);
title('Estacionalidad');
subplot(3,1,3);
plot(tsData.timeStamps, residual);
title('Residuos');

Explicación:

  • La función decompose se utiliza para descomponer la serie temporal en componentes de tendencia, estacionalidad y residuos.
  • La función subplot se utiliza para crear múltiples gráficos en una sola figura para una mejor visualización.

Pronóstico de Series Temporales

Pronosticar valores futuros basados en datos históricos es una aplicación común del análisis de series temporales.

Ejemplo: Pronóstico con el Modelo ARIMA

% Cargar datos de muestra
load(fullfile(matlabroot,'examples','econ','Data_Airline.mat'))

% Ajustar un modelo ARIMA
Mdl = arima(1,1,1);
EstMdl = estimate(Mdl, Data);

% Pronosticar valores futuros
numPeriods = 12; % Pronosticar para los próximos 12 períodos
[forecast, forecastMSE] = forecast(EstMdl, numPeriods, 'Y0', Data);

% Graficar el pronóstico
figure;
plot(Data, 'Color', [0, 0.4470, 0.7410]);
hold on;
plot(length(Data)+1:length(Data)+numPeriods, forecast, 'r', 'LineWidth', 2);
title('Pronóstico ARIMA');
xlabel('Tiempo');
ylabel('Valor');
legend('Datos Históricos', 'Pronóstico');
grid on;

Explicación:

  • arima(1,1,1) especifica un modelo ARIMA con un término autorregresivo, un término de diferenciación y un término de media móvil.
  • estimate ajusta el modelo ARIMA a los datos.
  • forecast predice valores futuros basados en el modelo ajustado.

Conclusión

El análisis de series temporales en MATLAB implica crear, visualizar, descomponer y pronosticar datos de series temporales. Al comprender estos conceptos y técnicas fundamentales, puedes analizar e interpretar eficazmente los datos de series temporales para diversas aplicaciones.
