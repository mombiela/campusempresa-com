Introducción

En esta sección, exploraremos cómo trabajar con varios formatos de archivo en MATLAB. Esto incluye leer y escribir en diferentes tipos de archivos como archivos de texto, archivos CSV, archivos de Excel y archivos MAT. Al final de esta sección, podrás manejar datos almacenados en diferentes formatos de manera eficiente.

Archivos de Texto

Los archivos de texto son una de las formas más simples de almacenamiento de datos. MATLAB proporciona varias funciones para leer y escribir en archivos de texto.

Leyendo desde Archivos de Texto

Para leer datos de un archivo de texto, puedes usar las funciones fopen, fgets, fscanf y fclose.

% Abrir el archivo
fileID = fopen('data.txt', 'r');

% Leer cada línea del archivo
line = fgets(fileID);
while ischar(line)
    disp(line);
    line = fgets(fileID);
end

% Cerrar el archivo
fclose(fileID);

Escribiendo en Archivos de Texto

Para escribir datos en un archivo de texto, puedes usar las funciones fopen, fprintf y fclose.

% Abrir el archivo
fileID = fopen('output.txt', 'w');

% Escribir datos en el archivo
fprintf(fileID, '¡Hola, Mundo!\n');
fprintf(fileID, 'Esto es una prueba.\n');

% Cerrar el archivo
fclose(fileID);

Archivos CSV

Los archivos CSV (Valores Separados por Comas) se usan comúnmente para almacenar datos tabulares. MATLAB proporciona funciones integradas para manejar archivos CSV fácilmente.

Leyendo desde Archivos CSV

Para leer datos de un archivo CSV, puedes usar las funciones readtable o csvread.

% Usando readtable
data = readtable('data.csv');

% Usando csvread
data = csvread('data.csv', 1, 0); % Saltando la fila del encabezado

Escribiendo en Archivos CSV

Para escribir datos en un archivo CSV, puedes usar las funciones writetable o csvwrite.

% Usando writetable
data = table([1; 2; 3], [4; 5; 6], 'VariableNames', {'A', 'B'});
writetable(data, 'output.csv');

% Usando csvwrite
csvwrite('output.csv', [1 4; 2 5; 3 6]);

Archivos de Excel

Los archivos de Excel se usan ampliamente en el análisis de datos y la elaboración de informes. MATLAB proporciona funciones para leer y escribir en archivos de Excel.

Leyendo desde Archivos de Excel

Para leer datos de un archivo de Excel, puedes usar las funciones readtable o xlsread.

% Usando readtable
data = readtable('data.xlsx');

% Usando xlsread
[num, txt, raw] = xlsread('data.xlsx');

Escribiendo en Archivos de Excel

Para escribir datos en un archivo de Excel, puedes usar las funciones writetable o xlswrite.

% Usando writetable
data = table([1; 2; 3], [4; 5; 6], 'VariableNames', {'A', 'B'});
writetable(data, 'output.xlsx');

% Usando xlswrite
xlswrite('output.xlsx', [1 4; 2 5; 3 6]);

Archivos MAT

Los archivos MAT son el formato nativo de MATLAB para almacenar variables. Son particularmente útiles para guardar y cargar grandes conjuntos de datos y variables complejas.

Guardando en Archivos MAT

Para guardar variables en un archivo MAT, puedes usar la función save.

% Definir algunas variables
A = [1, 2, 3];
B = {'Hola', 'Mundo'};

% Guardar variables en un archivo MAT
save('data.mat', 'A', 'B');

Cargando desde Archivos MAT

Para cargar variables desde un archivo MAT, puedes usar la función load.

% Cargar variables desde un archivo MAT
load('data.mat');

% Mostrar las variables cargadas
disp(A);
disp(B);

Conclusión

En esta sección, cubrimos cómo trabajar con diferentes formatos de archivo en MATLAB, incluyendo archivos de texto, archivos CSV, archivos de Excel y archivos MAT. Entender cómo leer y escribir en estos formatos de archivo es esencial para el análisis y la manipulación de datos en MATLAB. Practica usando estas funciones para volverte competente en el manejo de varios tipos de archivos de datos.
