Introducción a las Estructuras

Las estructuras en MATLAB son tipos de datos que agrupan datos relacionados utilizando contenedores de datos llamados campos. Cada campo puede contener cualquier tipo de dato, incluidas matrices, cadenas o incluso otras estructuras. Esto hace que las estructuras sean una forma versátil de gestionar datos complejos.

Conceptos Clave

  • Nombres de Campos: Identificadores para los datos almacenados en una estructura.
  • Notación de Punto: Se utiliza para acceder a los datos en los campos de una estructura.
  • Estructuras Anidadas: Estructuras que contienen otras estructuras como campos.

Creación de Estructuras

Para crear una estructura, puedes usar la función struct o la asignación directa utilizando la notación de punto.

Usando la Función `struct`

student = struct('Name', 'John Doe', 'Age', 21, 'GPA', 3.5);

En este ejemplo:

  • Name, Age y GPA son nombres de campos.
  • 'John Doe', 21 y 3.5 son los valores correspondientes.

Usando la Notación de Punto

student.Name = 'John Doe';
student.Age = 21;
student.GPA = 3.5;

Este método logra el mismo resultado que usar la función struct.

Acceso y Modificación de Datos en Estructuras

Puedes acceder y modificar los datos en una estructura utilizando la notación de punto.

Acceso a Datos

name = student.Name;  % Recupera 'John Doe'
age = student.Age;    % Recupera 21
gpa = student.GPA;    % Recupera 3.5

Modificación de Datos

student.GPA = 3.8;  % Actualiza el GPA a 3.8

Estructuras Anidadas

Las estructuras pueden contener otras estructuras, lo que permite una organización de datos más compleja.

Creación de Estructuras Anidadas

student.Address = struct('Street', '123 Main St', 'City', 'Springfield', 'Zip', '12345');

Acceso a Datos en Estructuras Anidadas

city = student.Address.City;  % Recupera 'Springfield'

Arreglos de Estructuras

Puedes crear arreglos de estructuras para gestionar múltiples registros.

Creación de un Arreglo de Estructuras

students(1) = struct('Name', 'John Doe', 'Age', 21, 'GPA', 3.5);
students(2) = struct('Name', 'Jane Smith', 'Age', 22, 'GPA', 3.7);

Acceso a Datos en un Arreglo de Estructuras

name1 = students(1).Name;  % Recupera 'John Doe'
gpa2 = students(2).GPA;    % Recupera 3.7

Operaciones Avanzadas con Estructuras

MATLAB proporciona funciones para manipular estructuras de manera más eficiente.

Fieldnames y Structfun

  • fieldnames: Devuelve un arreglo de celdas con los nombres de los campos.
  • structfun: Aplica una función a cada campo de una estructura.

Ejemplo

fields = fieldnames(student);  % Devuelve {'Name', 'Age', 'GPA', 'Address'}
gpaSum = structfun(@(x) sum([students.GPA]), students);  % Suma todos los GPAs

Resumen

Las estructuras en MATLAB son herramientas poderosas para organizar y gestionar datos complejos. Permiten un almacenamiento de datos flexible con campos nombrados, soporte para estructuras anidadas y la capacidad de crear arreglos de estructuras. Entender cómo crear, acceder y manipular estructuras es esencial para una gestión eficiente de datos en MATLAB.
