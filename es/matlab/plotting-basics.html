Introducción

La creación de gráficos es un aspecto fundamental de la visualización de datos en MATLAB. Esta sección te guiará a través de los conceptos básicos para crear gráficos, personalizarlos y comprender los diferentes tipos de gráficos disponibles en MATLAB. Al final de esta sección, serás capaz de crear y personalizar gráficos simples y entender cómo usar las funciones de gráficos de MATLAB de manera efectiva.

Creación de Gráficos Simples

Para crear un gráfico básico en MATLAB, puedes usar la función plot. Esta función toma puntos de datos y los grafica en un gráfico 2D.

Ejemplo: Gráfico de Línea Básico

% Definir los puntos de datos
x = 0:0.1:10;
y = sin(x);

% Crear el gráfico
plot(x, y);

% Añadir etiquetas y título
xlabel('Eje X');
ylabel('Eje Y');
title('Gráfico de Línea Básico');
  • x = 0:0.1:10; crea un vector de valores x desde 0 hasta 10 con un paso de 0.1.
  • y = sin(x); calcula el seno de cada valor x.
  • plot(x, y); crea el gráfico.
  • xlabel, ylabel y title añaden etiquetas y un título al gráfico.

Personalización de Gráficos

MATLAB proporciona varias opciones para personalizar gráficos, incluyendo cambiar estilos de línea, colores, marcadores y añadir líneas de cuadrícula.

Ejemplo: Personalización de Estilo de Línea y Color

% Definir los puntos de datos
x = 0:0.1:10;
y = cos(x);

% Crear el gráfico con estilo de línea y color personalizados
plot(x, y, '--r', 'LineWidth', 2);

% Añadir etiquetas y título
xlabel('Eje X');
ylabel('Eje Y');
title('Gráfico de Línea Personalizado');
grid on; % Añadir líneas de cuadrícula
  • '--r' especifica una línea roja discontinua.
  • 'LineWidth', 2 establece el ancho de la línea a 2.
  • grid on; añade líneas de cuadrícula al gráfico.

Múltiples Gráficos en una Figura

Puedes graficar múltiples conjuntos de datos en la misma figura usando los comandos hold on y hold off.

Ejemplo: Múltiples Gráficos

% Definir los puntos de datos
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

% Crear el primer gráfico
plot(x, y1, '-b', 'LineWidth', 2);
hold on; % Mantener el gráfico actual

% Crear el segundo gráfico
plot(x, y2, '--r', 'LineWidth', 2);

% Añadir etiquetas y título
xlabel('Eje X');
ylabel('Eje Y');
title('Múltiples Gráficos');
legend('sin(x)', 'cos(x)'); % Añadir una leyenda
hold off; % Liberar el gráfico
  • hold on; permite que se dibujen múltiples gráficos en la misma figura.
  • legend añade una leyenda para diferenciar entre los gráficos.

Subgráficos

Los subgráficos te permiten crear múltiples gráficos en una sola figura, dispuestos en una cuadrícula.

Ejemplo: Subgráficos

% Definir los puntos de datos
x = 0:0.1:10;
y1 = sin(x);
y2 = cos(x);

% Crear una cuadrícula de subgráficos de 2x1
subplot(2, 1, 1); % Primer subgráfico
plot(x, y1, '-b');
title('Función Seno');

subplot(2, 1, 2); % Segundo subgráfico
plot(x, y2, '--r');
title('Función Coseno');
  • subplot(2, 1, 1); crea el primer subgráfico en una cuadrícula de 2x1.
  • subplot(2, 1, 2); crea el segundo subgráfico.

Técnicas Avanzadas de Gráficos

MATLAB ofrece técnicas avanzadas de gráficos como gráficos 3D, gráficos de contorno y gráficos de superficie.

Ejemplo: Gráfico 3D

% Definir los puntos de datos
[x, y] = meshgrid(-5:0.5:5, -5:0.5:5);
z = sin(sqrt(x.^2 + y.^2));

% Crear el gráfico 3D
surf(x, y, z);

% Añadir etiquetas y título
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');
title('Gráfico de Superficie 3D');
  • meshgrid genera una cuadrícula de valores x e y.
  • surf crea un gráfico de superficie 3D.
  • zlabel añade una etiqueta al eje z.

Conclusión

En esta sección, aprendiste los conceptos básicos de la creación de gráficos en MATLAB, incluyendo la creación de gráficos simples, su personalización, la representación de múltiples conjuntos de datos, el uso de subgráficos y la creación de gráficos 3D avanzados. Dominar estos conceptos básicos te permitirá visualizar datos de manera efectiva y mejorar tus habilidades de programación en MATLAB.
