Bucles: for y while

Introducción

Los bucles son constructos fundamentales en la programación que te permiten ejecutar un bloque de código múltiples veces. En MATLAB, los dos tipos principales de bucles son los bucles for y los bucles while. Esta sección te guiará a través de los conceptos básicos de estos bucles, proporcionará ejemplos y ofrecerá ejercicios para practicar.

Bucles for

Sintaxis Básica

Un bucle for en MATLAB repite un grupo de sentencias un número fijo y predeterminado de veces. La sintaxis es la siguiente:

for index = startValue:endValue
    % Sentencias a ejecutar
end

Ejemplo

Comencemos con un ejemplo simple donde imprimimos números del 1 al 5:

for i = 1:5
    disp(i)
end

Explicación

  • i = 1:5 inicializa la variable del bucle i para que varíe de 1 a 5.
  • disp(i) muestra el valor actual de i en cada iteración.

Bucles for anidados

También puedes anidar bucles for para manejar tareas más complejas. Por ejemplo, crear una tabla de multiplicar:

for i = 1:5
    for j = 1:5
        fprintf('%d * %d = %d\n', i, j, i*j);
    end
end

Explicación

  • El bucle externo corre de 1 a 5.
  • El bucle interno también corre de 1 a 5.
  • fprintf imprime el producto de i y j.

Bucles while

Sintaxis Básica

Un bucle while en MATLAB repite un grupo de sentencias un número indefinido de veces mientras la condición sea verdadera. La sintaxis es:

while condition
    % Sentencias a ejecutar
end

Ejemplo

Veamos un ejemplo donde imprimimos números del 1 al 5 usando un bucle while:

i = 1;
while i <= 5
    disp(i)
    i = i + 1;
end

Explicación

  • i = 1 inicializa la variable del bucle i.
  • while i <= 5 verifica si i es menor o igual a 5.
  • disp(i) muestra el valor actual de i.
  • i = i + 1 incrementa i en 1 en cada iteración.

Bucles Infinitos

Ten cuidado con los bucles while ya que pueden crear bucles infinitos si la condición nunca se vuelve falsa. Por ejemplo:

i = 1;
while true
    disp(i)
    i = i + 1;
    if i > 5
        break;
    end
end

Explicación

  • while true crea un bucle infinito.
  • if i > 5 verifica si i es mayor que 5.
  • break sale del bucle cuando i es mayor que 5.

Comparación de bucles for y while

| Característica | Bucle for | Bucle while | |----------------|-----------------------------------|------------------------------------| | Caso de Uso | Número conocido de iteraciones | Número desconocido de iteraciones | | Inicialización | Dentro de la definición del bucle | Fuera del bucle | | Condición | Definida por el rango | Definida por una condición lógica | | Peligro Común | Ninguno | Riesgo de bucles infinitos |

Ejercicios

Ejercicio 1: Suma de Números

Escribe un bucle for para calcular la suma de los números del 1 al 10.

sum = 0;
for i = 1:10
    sum = sum + i;
end
disp(sum)

Ejercicio 2: Cálculo del Factorial

Escribe un bucle while para calcular el factorial de un número dado n.

n = 5; % Puedes cambiar este valor para probar con diferentes números
factorial = 1;
i = 1;
while i <= n
    factorial = factorial * i;
    i = i + 1;
end
disp(factorial)

Conclusión

Entender y usar los bucles de manera efectiva es crucial para una programación eficiente en MATLAB. Los bucles for son ideales cuando el número de iteraciones se conoce de antemano, mientras que los bucles while son más adecuados para situaciones donde el número de iteraciones depende de una condición. Practica usando ambos tipos de bucles para volverte competente en su aplicación.
