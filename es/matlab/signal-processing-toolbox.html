Introducción

La Caja de Herramientas de Procesamiento de Señales en MATLAB es un conjunto integral de funciones y aplicaciones diseñadas para analizar, preprocesar y extraer características de señales. Esta caja de herramientas es esencial para ingenieros y científicos que trabajan en campos como comunicaciones, procesamiento de audio y análisis de señales biomédicas.

Comenzando con la Caja de Herramientas de Procesamiento de Señales

Instalación

Para usar la Caja de Herramientas de Procesamiento de Señales, asegúrate de que esté instalada y licenciada. Puedes verificar esto escribiendo el siguiente comando en MATLAB:

ver

Busca "Signal Processing Toolbox" en la lista de cajas de herramientas instaladas.

Conceptos Básicos

  • Señales: Una señal es una función que transmite información sobre un fenómeno. En MATLAB, las señales se representan típicamente como vectores o matrices.
  • Muestreo: El proceso de convertir una señal continua en una señal discreta tomando muestras a intervalos regulares.
  • Dominio de Frecuencia: Análisis de señales en términos de sus componentes de frecuencia usando la Transformada de Fourier.

Creación y Visualización de Señales

Creación de Señales

Puedes crear señales básicas como ondas sinusoidales, ondas cuadradas y ruido aleatorio usando funciones integradas de MATLAB.

fs = 1000; % Frecuencia de muestreo
t = 0:1/fs:1-1/fs; % Vector de tiempo
f = 5; % Frecuencia de la onda sinusoidal
x = sin(2*pi*f*t); % Onda sinusoidal

Visualización de Señales

La visualización es crucial para entender las características de la señal. Usa la función plot para visualizar señales.

plot(t, x);
title('Onda Sinusoidal');
xlabel('Tiempo (s)');
ylabel('Amplitud');

Análisis de Señales

Análisis en el Dominio del Tiempo

  • Media y Varianza: Medidas estadísticas básicas.
  • Autocorrelación: Medida de similitud entre una señal y una versión retrasada de sí misma.
mean_x = mean(x);
var_x = var(x);
autocorr_x = xcorr(x);

Análisis en el Dominio de la Frecuencia

  • Transformada de Fourier: Convierte una señal en el dominio del tiempo en sus componentes de frecuencia.
X = fft(x);
f = (0:length(X)-1)*fs/length(X); % Vector de frecuencia
plot(f, abs(X));
title('Espectro de Magnitud');
xlabel('Frecuencia (Hz)');
ylabel('Magnitud');

Filtrado de Señales

Tipos de Filtros

  • Filtro Pasa-bajos: Permite que señales con una frecuencia menor que una cierta frecuencia de corte pasen a través.
  • Filtro Pasa-altos: Permite que señales con una frecuencia mayor que una cierta frecuencia de corte pasen a través.
  • Filtro Pasa-banda: Permite que señales dentro de un cierto rango de frecuencias pasen a través.

Diseño de Filtros

Usa la función designfilt para crear filtros.

d = designfilt('lowpassfir', 'PassbandFrequency', 0.25, 'StopbandFrequency', 0.35, 'PassbandRipple', 1, 'StopbandAttenuation', 60);

Aplicación de Filtros

Usa la función filter para aplicar el filtro diseñado a una señal.

y = filter(d, x);
plot(t, y);
title('Señal Filtrada');
xlabel('Tiempo (s)');
ylabel('Amplitud');

Temas Avanzados

Transformada Wavelet

La transformada wavelet se usa para el análisis tiempo-frecuencia de señales no estacionarias.

[c, l] = wavedec(x, 4, 'db1');

Espectrograma

Un espectrograma proporciona una representación visual del espectro de frecuencias en una señal a medida que varía con el tiempo.

spectrogram(x, 128, 120, 128, fs, 'yaxis');
title('Espectrograma');

Conclusión

La Caja de Herramientas de Procesamiento de Señales en MATLAB ofrece una amplia gama de funciones para la creación, visualización, análisis y filtrado de señales. Al dominar estas herramientas, puedes procesar y analizar eficazmente señales para diversas aplicaciones. Este tema cubrió los conceptos básicos e introdujo algunas técnicas avanzadas, proporcionando una base sólida para una mayor exploración y aplicación en escenarios del mundo real.
