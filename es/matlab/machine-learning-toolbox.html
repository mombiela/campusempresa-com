Introducción a la Caja de Herramientas de Aprendizaje Automático

La Caja de Herramientas de Aprendizaje Automático en MATLAB proporciona herramientas y funciones para realizar diversas tareas de aprendizaje automático. Esta sección cubrirá los conceptos básicos de la caja de herramientas, incluida la instalación y una visión general de sus capacidades.

  • Instalación: Asegúrese de que la Caja de Herramientas de Aprendizaje Automático esté instalada a través del Explorador de Complementos de MATLAB.
  • Capacidades: Preprocesamiento de datos, clasificación, regresión, agrupamiento y evaluación de modelos.

Preprocesamiento de Datos

El preprocesamiento de datos es un paso crucial en el aprendizaje automático. Esta sección cubrirá técnicas para preparar datos para el análisis.

  • Importación de Datos: Use readtable, readmatrix y otras funciones para importar datos.
  • Limpieza de Datos: Maneje valores faltantes, valores atípicos y normalización de datos.
  • Ingeniería de Características: Cree nuevas características a partir de datos existentes.

Ejemplo: Importación y Limpieza de Datos

% Importar datos
data = readtable('data.csv');

% Mostrar las primeras filas de la tabla
head(data)

% Manejar valores faltantes llenando con la media
data = fillmissing(data, 'constant', mean(data, 'omitnan'));

% Normalizar datos
data = normalize(data);

Clasificación

La clasificación es el proceso de predecir la clase de puntos de datos dados. Esta sección cubrirá varios algoritmos de clasificación disponibles en MATLAB.

  • Algoritmos: Árboles de Decisión, Máquinas de Vectores de Soporte (SVM), k-Vecinos Más Cercanos (k-NN) y Redes Neuronales.
  • Entrenamiento: Use fitctree, fitcsvm, fitcknn y fitcnet para entrenar modelos.
  • Evaluación: Use matrices de confusión y validación cruzada para evaluar modelos.

Ejemplo: Entrenamiento de un Árbol de Decisión

% Cargar datos de muestra
load fisheriris

% Dividir datos en conjuntos de entrenamiento y prueba
cv = cvpartition(species, 'HoldOut', 0.3);
trainData = meas(training(cv), :);
trainLabels = species(training(cv));
testData = meas(test(cv), :);
testLabels = species(test(cv));

% Entrenar un clasificador de árbol de decisión
treeModel = fitctree(trainData, trainLabels);

% Predecir en datos de prueba
predictions = predict(treeModel, testData);

% Evaluar el modelo
confMat = confusionmat(testLabels, predictions);
disp('Matriz de Confusión:');
disp(confMat);

Regresión

El análisis de regresión se utiliza para predecir una variable de salida continua basada en una o más características de entrada. Esta sección cubrirá varias técnicas de regresión.

  • Algoritmos: Regresión Lineal, Regresión Polinómica, Regresión de Vectores de Soporte (SVR).
  • Entrenamiento: Use fitlm, fitrsvm y otras funciones para entrenar modelos de regresión.
  • Evaluación: Use métricas como el Error Cuadrático Medio (MSE) y el R-cuadrado.

Ejemplo: Regresión Lineal

% Cargar datos de muestra
load carsmall

% Preparar datos
X = [Weight, Horsepower];
y = MPG;

% Entrenar un modelo de regresión lineal
lm = fitlm(X, y);

% Predecir en nuevos datos
newData = [3500, 200]; % Ejemplo de nuevos datos
predictedMPG = predict(lm, newData);

% Mostrar el resultado
disp(['MPG Predicho: ', num2str(predictedMPG)]);

Agrupamiento

El agrupamiento es la tarea de agrupar un conjunto de objetos de tal manera que los objetos en el mismo grupo sean más similares entre sí que aquellos en otros grupos.

  • Algoritmos: k-Medias, Agrupamiento Jerárquico, Modelos de Mezcla Gaussiana (GMM).
  • Entrenamiento: Use kmeans, linkage y fitgmdist para el agrupamiento.
  • Evaluación: Use gráficos de silueta e índices de validez de clústeres.

Ejemplo: Agrupamiento k-Medias

% Cargar datos de muestra
load fisheriris

% Realizar agrupamiento k-medias
numClusters = 3;
[idx, C] = kmeans(meas, numClusters);

% Graficar los clústeres
figure;
gscatter(meas(:,1), meas(:,2), idx);
hold on;
plot(C(:,1), C(:,2), 'kx', 'MarkerSize', 15, 'LineWidth', 3);
title('Agrupamiento k-Medias');
xlabel('Longitud del Sépalo');
ylabel('Ancho del Sépalo');
hold off;

Evaluación de Modelos y Ajuste de Hiperparámetros

Evaluar y ajustar modelos de aprendizaje automático es esencial para mejorar su rendimiento. Esta sección cubrirá varias métricas de evaluación y técnicas para el ajuste de hiperparámetros.

  • Métricas de Evaluación: Precisión, Precisión, Recall, F1-Score, MSE, R-cuadrado.
  • Validación Cruzada: Use crossval y cvpartition para validación cruzada.
  • Ajuste de Hiperparámetros: Use bayesopt, fitctree con opciones de hiperparámetros.

Ejemplo: Validación Cruzada y Ajuste de Hiperparámetros

% Cargar datos de muestra
load fisheriris

% Definir el modelo y los hiperparámetros
treeTemplate = templateTree('MaxNumSplits', 1:10);

% Realizar validación cruzada
cvModel = fitcensemble(meas, species, 'Method', 'Bag', 'Learners', treeTemplate, 'CrossVal', 'on');

% Mostrar resultados de la validación cruzada
cvLoss = kfoldLoss(cvModel);
disp(['Pérdida de Validación Cruzada: ', num2str(cvLoss)]);

Conclusión

La Caja de Herramientas de Aprendizaje Automático en MATLAB proporciona un conjunto completo de herramientas para el preprocesamiento de datos, clasificación, regresión, agrupamiento y evaluación de modelos. Al comprender y utilizar estas herramientas, puede construir y evaluar eficazmente modelos de aprendizaje automático para diversas aplicaciones. Practique con los ejemplos proporcionados y explore más para profundizar su comprensión y competencia en el aprendizaje automático con MATLAB.
