Introducción

MATLAB (Matrix Laboratory) es un lenguaje de alto rendimiento para la computación técnica. Integra computación, visualización y programación en un entorno fácil de usar. Esta sección cubrirá los comandos y la sintaxis básicos necesarios para comenzar con MATLAB.

Comandos Básicos

Entender los comandos básicos es esencial para navegar en MATLAB y realizar operaciones simples.

  • clc: Limpia la ventana de comandos.
  • clear: Elimina variables del espacio de trabajo.
  • close all: Cierra todas las ventanas de figuras.
  • who: Lista las variables actuales en el espacio de trabajo.
  • whos: Lista las variables actuales junto con sus detalles.
  • help: Proporciona ayuda para las funciones de MATLAB.
  • doc: Abre la documentación para las funciones de MATLAB.

Ejemplo

clc;          % Limpia la ventana de comandos
clear;        % Elimina todas las variables del espacio de trabajo
close all;    % Cierra todas las ventanas de figuras

Variables y Tipos de Datos

Las variables en MATLAB se utilizan para almacenar datos, y MATLAB admite varios tipos de datos.

  • Tipos Numéricos: double, single, int8, int16, int32, int64, uint8, uint16, uint32, uint64
  • Caracteres y Cadenas: char, string
  • Lógico: true, false
  • Arreglos y Matrices: Fundamental en MATLAB

Ejemplo

a = 5;                % Entero
b = 3.14;             % Número de punto flotante
c = 'Hola';           % Arreglo de caracteres (cadena)
d = true;             % Lógico verdadero
e = [1, 2, 3; 4, 5, 6]; % Matriz 2x3

Operaciones Básicas

MATLAB admite una variedad de operaciones básicas, incluidas operaciones aritméticas, relacionales y lógicas.

Operaciones Aritméticas

  • Adición: +
  • Sustracción: -
  • Multiplicación: *
  • División: /
  • Exponenciación: ^

Ejemplo

x = 10;
y = 3;
sum = x + y;          % Adición
difference = x - y;   % Sustracción
product = x * y;      % Multiplicación
quotient = x / y;     % División
power = x ^ y;        % Exponenciación

Operaciones Relacionales

  • Igual a: ==
  • No igual a: ~=
  • Mayor que: >
  • Menor que: <
  • Mayor o igual que: >=
  • Menor o igual que: <=

Ejemplo

a = 5;
b = 10;
isEqual = (a == b);          % Falso
isNotEqual = (a ~= b);       % Verdadero
isGreater = (a > b);         % Falso
isLess = (a < b);            % Verdadero
isGreaterOrEqual = (a >= b); % Falso
isLessOrEqual = (a <= b);    % Verdadero

Operaciones Lógicas

  • AND: &
  • OR: |
  • NOT: ~

Ejemplo

p = true;
q = false;
andResult = p & q;   % Falso
orResult = p | q;    % Verdadero
notResult = ~p;      % Falso

Flujo de Control

Las declaraciones de flujo de control te permiten controlar la ejecución del código basado en condiciones.

Declaraciones If-Else

x = 10;
if x > 5
    disp('x es mayor que 5');
else
    disp('x no es mayor que 5');
end

Declaraciones Switch-Case

choice = 'A';
switch choice
    case 'A'
        disp('La elección es A');
    case 'B'
        disp('La elección es B');
    otherwise
        disp('Elección desconocida');
end

Bucles

Bucle For

for i = 1:5
    disp(['Iteración: ', num2str(i)]);
end

Bucle While

i = 1;
while i <= 5
    disp(['Iteración: ', num2str(i)]);
    i = i + 1;
end

Funciones

Las funciones en MATLAB te permiten encapsular código para reutilización y mejor organización.

Definiendo una Función

function result = addNumbers(a, b)
    result = a + b;
end

Llamando a una Función

sum = addNumbers(3, 4);  % Llama a la función y almacena el resultado en 'sum'
disp(sum);               % Muestra 7

Conclusión

Esta sección ha introducido los comandos y la sintaxis básicos en MATLAB, incluyendo la declaración de variables, operaciones básicas, flujo de control y funciones. Dominar estos conceptos básicos es crucial para avanzar a temas más avanzados en MATLAB. Practica estos comandos e intenta escribir scripts simples para solidificar tu comprensión.
