Introducción

En esta sección, cubriremos los conceptos básicos de variables y tipos de datos en MATLAB. Comprender estos conceptos es fundamental para escribir código MATLAB eficiente y efectivo. Comenzaremos con los conceptos básicos de las variables, luego pasaremos a los diferentes tipos de datos y finalmente exploraremos temas más avanzados.

Variables

Las variables se utilizan para almacenar datos que pueden ser usados y manipulados a lo largo de tu código MATLAB.

Creación de Variables

  • Para crear una variable, simplemente asigna un valor a ella usando el operador =.
  • Los nombres de las variables deben comenzar con una letra, seguida de letras, números o guiones bajos.
x = 5;          % Crea una variable x con valor 5
y = 3.14;       % Crea una variable y con valor 3.14
nombre = 'John';  % Crea una variable nombre con la cadena 'John'

Convenciones de Nombres de Variables

  • Usa nombres descriptivos para las variables.
  • Evita usar palabras reservadas de MATLAB (por ejemplo, if, else, end).
suma_total = 100;  % Nombre de variable descriptivo

Tipos de Datos

MATLAB soporta varios tipos de datos, cada uno adecuado para diferentes tipos de datos y operaciones.

Tipos de Datos Numéricos

  • Double: Tipo de dato numérico predeterminado en MATLAB.
  • Integer: MATLAB soporta varios tipos de enteros (por ejemplo, int8, int16, int32, int64).
a = 10;        % Double
b = int32(10); % Entero de 32 bits

Tipos de Datos de Caracteres y Cadenas

  • Arreglos de Caracteres: Se usan comillas simples para crear arreglos de caracteres.
  • Arreglos de Cadenas: Se usan comillas dobles para crear arreglos de cadenas.
arregloChar = 'Hola';  % Arreglo de caracteres
arregloStr = "Hola";   % Arreglo de cadenas

Tipos de Datos Lógicos

  • Los tipos de datos lógicos se usan para valores verdadero/falso.
esVerdadero = true;   % Lógico verdadero
esFalso = false;      % Lógico falso

Números Complejos

  • MATLAB soporta números complejos con partes reales e imaginarias.
z = 3 + 4i;  % Número complejo

Arreglos y Matrices

  • Los arreglos y matrices son fundamentales en MATLAB, que significa "Laboratorio de Matrices".
arreglo = [1, 2, 3, 4];       % Arreglo 1x4
matriz = [1, 2; 3, 4];        % Matriz 2x2

Arreglos de Celdas y Estructuras

  • Arreglos de Celdas: Pueden contener diferentes tipos de datos en cada celda.
  • Estructuras: Se usan para agrupar datos relacionados usando campos con nombre.
arregloCelda = {1, 'texto', [2, 3]};  % Arreglo de celdas
estructuraVar = struct('campo1', 1, 'campo2', 'texto');  % Estructura

Conversión de Tipos

MATLAB te permite convertir entre diferentes tipos de datos usando funciones integradas.

num = 5.67;
numInt = int32(num);  % Convertir double a int32
numStr = num2str(num);  % Convertir número a cadena

Resumen

En esta sección, cubrimos los conceptos básicos de variables y tipos de datos en MATLAB. Aprendimos cómo crear variables, exploramos diferentes tipos de datos incluyendo numéricos, caracteres, lógicos, números complejos, arreglos, matrices, arreglos de celdas y estructuras. También discutimos la conversión de tipos. Comprender estos conceptos es esencial para una programación efectiva en MATLAB y servirá como base para temas más avanzados.

Al dominar las variables y los tipos de datos, estarás bien equipado para manejar datos de manera eficiente y escribir código MATLAB robusto.
