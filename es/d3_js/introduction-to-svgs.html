¿Qué es SVG?

SVG (Gráficos Vectoriales Escalables) es un formato basado en XML para describir gráficos vectoriales bidimensionales. A diferencia de las imágenes rasterizadas (por ejemplo, JPEG, PNG), los SVGs pueden escalarse a cualquier tamaño sin perder calidad, lo que los hace ideales para el diseño web responsivo y la visualización de datos.

Características clave de SVG:

  • Escalabilidad: Las imágenes SVG pueden escalarse a cualquier tamaño sin perder calidad.
  • Basado en texto: Los archivos SVG son basados en texto, lo que los hace fáciles de editar y manipular.
  • Interactividad: Los SVGs soportan interactividad y animación.
  • Accesibilidad: Los SVGs pueden hacerse accesibles a lectores de pantalla y otras tecnologías asistivas.

Estructura básica de un SVG

Un archivo SVG es esencialmente un documento XML con una estructura específica. Aquí hay un ejemplo simple de un archivo SVG:

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

Explicación:

  • <svg>: El elemento raíz de un documento SVG.
  • width y height: Definen las dimensiones del lienzo SVG.
  • xmlns: Especifica el espacio de nombres XML para SVG.
  • <circle>: Un elemento de forma que dibuja un círculo.
    • cx y cy: Definen las coordenadas del centro del círculo.
    • r: Define el radio del círculo.
    • stroke: Define el color del borde del círculo.
    • stroke-width: Define el ancho del borde del círculo.
    • fill: Define el color de relleno del círculo.

Elementos comunes de SVG

Aquí hay algunos elementos SVG comúnmente usados:

  • : Dibuja rectángulos.
  • : Dibuja círculos.
  • : Dibuja elipses.
  • : Dibuja líneas rectas.
  • : Dibuja una serie de líneas rectas conectadas.
  • : Dibuja una forma cerrada con múltiples lados.
  • : Dibuja formas complejas usando una serie de comandos.

Ejemplo:

<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <rect x="10" y="10" width="50" height="50" stroke="blue" fill="yellow" />
  <ellipse cx="100" cy="100" rx="30" ry="20" stroke="green" fill="purple" />
  <line x1="150" y1="10" x2="150" y2="50" stroke="red" />
  <polyline points="10,150 50,180 90,150" stroke="black" fill="none" />
  <polygon points="120,150 160,180 200,150" stroke="black" fill="orange" />
  <path d="M10 200 C 20 100, 40 100, 50 200" stroke="brown" fill="none" />
</svg>

Estilizando SVGs

Los elementos SVG pueden ser estilizados usando CSS. Puedes aplicar estilos directamente dentro del archivo SVG o a través de CSS externo.

Estilo en línea:

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" style="stroke: black; stroke-width: 3; fill: red;" />
</svg>

CSS externo:

<style>
  .my-circle {
    stroke: black;
    stroke-width: 3;
    fill: red;
  }
</style>

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" class="my-circle" />
</svg>

Usando SVGs con D3.js

D3.js (Documentos Basados en Datos) es una poderosa biblioteca de JavaScript para crear visualizaciones de datos dinámicas e interactivas usando estándares web como SVG.

Ejemplo: Creando un SVG con D3.js

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo de SVG con D3.js</title>
  <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
  <script>
    // Crear un contenedor SVG
    const svg = d3.select("body")
                  .append("svg")
                  .attr("width", 200)
                  .attr("height", 200);

    // Añadir un círculo al SVG
    svg.append("circle")
       .attr("cx", 100)
       .attr("cy", 100)
       .attr("r", 50)
       .attr("stroke", "black")
       .attr("stroke-width", 3)
       .attr("fill", "red");
  </script>
</body>
</html>

Explicación:

  • d3.select("body").append("svg"): Selecciona el elemento body y le añade un elemento SVG.
  • .attr("width", 200).attr("height", 200): Establece el ancho y la altura del SVG.
  • svg.append("circle"): Añade un elemento círculo al SVG.
  • .attr("cx", 100).attr("cy", 100).attr("r", 50): Establece las coordenadas del centro del círculo y su radio.
  • .attr("stroke", "black").attr("stroke-width", 3).attr("fill", "red"): Establece el color del borde del círculo, el ancho del borde y el color de relleno.

Conclusión

Los SVGs son una herramienta versátil y poderosa para crear gráficos escalables e interactivos en la web. Entender los conceptos básicos de SVG es crucial para aprovechar todo el potencial de D3.js en la visualización de datos. Al combinar SVG con D3.js, puedes crear visualizaciones dinámicas basadas en datos que son tanto visualmente atractivas como altamente interactivas.
