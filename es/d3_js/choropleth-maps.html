Introducción a los Mapas Coropléticos

Los mapas coropléticos son un tipo de mapa temático donde las áreas se sombrean o se modelan en proporción al valor de una variable representada. Estos mapas son útiles para visualizar datos a través de regiones geográficas.

Conceptos Clave

  • Datos Geográficos: Datos que incluyen ubicaciones geográficas, como países, estados o condados.
  • Escalas de Color: Un rango de colores utilizados para representar diferentes valores en el mapa.
  • GeoJSON: Un formato para codificar una variedad de estructuras de datos geográficos utilizando JavaScript Object Notation (JSON).

Configuración de Tu Entorno

Antes de crear un mapa coroplético, asegúrate de tener las herramientas y bibliotecas necesarias instaladas.

Prerrequisitos

  • Comprensión básica de HTML, CSS y JavaScript.
  • Familiaridad con los conceptos básicos de D3.js.

Instalación

Para comenzar, incluye D3.js en tu proyecto. Puedes hacerlo agregando la siguiente etiqueta de script a tu archivo HTML:

<script src="https://d3js.org/d3.v6.min.js"></script>

Creando un Mapa Coroplético Básico

Vamos a crear un mapa coroplético simple usando D3.js.

Paso 1: Preparar la Estructura HTML

Crea un archivo HTML con una estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mapa Coroplético</title>
    <style>
        .map {
            width: 800px;
            height: 600px;
        }
    </style>
</head>
<body>
    <div class="map" id="map"></div>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <script src="script.js"></script>
</body>
</html>

Paso 2: Cargar Datos GeoJSON

En tu archivo script.js, carga los datos GeoJSON para las regiones geográficas que deseas mostrar. Para este ejemplo, usaremos un archivo GeoJSON de muestra para los estados de EE.UU.

d3.json('path/to/us-states.json').then(function(data) {
    console.log(data);
});

Paso 3: Crear un Elemento SVG

Crea un elemento SVG para contener el mapa:

const width = 800;
const height = 600;

const svg = d3.select("#map")
    .append("svg")
    .attr("width", width)
    .attr("height", height);

Paso 4: Definir una Proyección y un Generador de Rutas

Define una proyección para convertir coordenadas geográficas a coordenadas de pantalla y un generador de rutas para dibujar el mapa:

const projection = d3.geoAlbersUsa()
    .translate([width / 2, height / 2])
    .scale(1000);

const path = d3.geoPath()
    .projection(projection);

Paso 5: Dibujar el Mapa

Usa los datos GeoJSON para dibujar el mapa:

d3.json('path/to/us-states.json').then(function(data) {
    svg.selectAll("path")
        .data(data.features)
        .enter()
        .append("path")
        .attr("d", path)
        .attr("fill", "#ccc")
        .attr("stroke", "#333");
});

Añadiendo Datos al Mapa

Para hacer el mapa más informativo, añadiremos datos y usaremos escalas de color para representarlos.

Paso 1: Preparar los Datos

Supongamos que tenemos un conjunto de datos que incluye valores para cada estado:

const data = [
    { state: 'California', value: 100 },
    { state: 'Texas', value: 80 },
    // Añadir más datos aquí
];

Paso 2: Crear una Escala de Color

Crea una escala de color para representar los valores de los datos:

const colorScale = d3.scaleQuantize()
    .domain([0, 100])
    .range(d3.schemeBlues[9]);

Paso 3: Fusionar Datos con GeoJSON

Fusiona los datos con las características GeoJSON:

d3.json('path/to/us-states.json').then(function(geoData) {
    data.forEach(d => {
        const state = geoData.features.find(f => f.properties.name === d.state);
        if (state) {
            state.properties.value = d.value;
        }
    });

    svg.selectAll("path")
        .data(geoData.features)
        .enter()
        .append("path")
        .attr("d", path)
        .attr("fill", d => colorScale(d.properties.value))
        .attr("stroke", "#333");
});

Características Avanzadas

Mejora tu mapa coroplético con características interactivas y técnicas adicionales de visualización de datos.

Tooltips

Añade tooltips para mostrar los valores de los datos al pasar el ratón sobre las regiones:

const tooltip = d3.select("body").append("div")
    .attr("class", "tooltip")
    .style("opacity", 0);

svg.selectAll("path")
    .on("mouseover", function(event, d) {
        tooltip.transition()
            .duration(200)
            .style("opacity", .9);
        tooltip.html(d.properties.name + "<br/>" + d.properties.value)
            .style("left", (event.pageX + 5) + "px")
            .style("top", (event.pageY - 28) + "px");
    })
    .on("mouseout", function(d) {
        tooltip.transition()
            .duration(500)
            .style("opacity", 0);
    });

Leyendas

Añade una leyenda para explicar la escala de color:

const legend = svg.append("g")
    .attr("class", "legend")
    .attr("transform", "translate(20,20)");

const legendScale = d3.scaleLinear()
    .domain([0, 100])
    .range([0, 300]);

const legendAxis = d3.axisBottom(legendScale)
    .ticks(5);

legend.selectAll("rect")
    .data(colorScale.range().map(color => {
        const d = colorScale.invertExtent(color);
        if (!d[0]) d[0] = legendScale.domain()[0];
        if (!d[1]) d[1] = legendScale.domain()[1];
        return d;
    }))
    .enter().append("rect")
    .attr("height", 20)
    .attr("x", d => legendScale(d[0]))
    .attr("width", d => legendScale(d[1]) - legendScale(d[0]))
    .attr("fill", d => colorScale(d[0]));

legend.append("g")
    .attr("transform", "translate(0,20)")
    .call(legendAxis);

Conclusión

Los mapas coropléticos son una herramienta poderosa para visualizar datos geográficos. Siguiendo los pasos descritos en esta guía, puedes crear mapas informativos e interactivos usando D3.js. A medida que te sientas más cómodo con estas técnicas, puedes explorar características adicionales y personalizaciones para mejorar aún más tus mapas.
