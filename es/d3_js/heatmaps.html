Los mapas de calor son una herramienta de visualización poderosa utilizada para representar datos donde los valores individuales se representan como colores. En esta sección, exploraremos cómo crear mapas de calor usando D3.js, desde conceptos básicos hasta técnicas avanzadas.

Introducción a los Mapas de Calor

  • Definición: Un mapa de calor es una representación gráfica de datos donde los valores individuales se representan mediante colores.
  • Casos de Uso: Comúnmente utilizados en análisis de datos para visualizar la magnitud de un fenómeno, como el tráfico de un sitio web, datos del mercado de valores o patrones climáticos.

Configurando Tu Entorno

Antes de sumergirnos en la creación de mapas de calor, asegúrate de tener lo siguiente:

  • Biblioteca D3.js: Incluye D3.js en tu proyecto añadiendo la siguiente etiqueta de script en tu archivo HTML: 
    <script src="https://d3js.org/d3.v6.min.js"></script>
  • Estructura Básica de HTML: Crea una estructura básica de HTML para contener tu mapa de calor. 
    <!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mapa de Calor con D3.js</title>
</head>
<body>
    <div id="heatmap"></div>
    <script src="script.js"></script>
</body>
</html>

Creando un Mapa de Calor Básico

  1. Preparar los Datos: Los mapas de calor requieren datos en un formato de matriz. Aquí hay un conjunto de datos de ejemplo:

    const data = [
    [30, 20, 10],
    [60, 50, 40],
    [90, 80, 70]
];

  2. Configurar el Contenedor SVG: Crea un contenedor SVG para contener el mapa de calor.

    const width = 300;
const height = 300;
const svg = d3.select("#heatmap")
              .append("svg")
              .attr("width", width)
              .attr("height", height);

  3. Definir la Escala de Colores: Usa una escala de colores para mapear los valores de los datos a colores.

    const colorScale = d3.scaleSequential(d3.interpolateBlues)
                     .domain([0, 100]);

  4. Renderizar el Mapa de Calor: Usa D3.js para renderizar el mapa de calor.

    const cellSize = 100; // Tamaño de cada celda

data.forEach((row, i) => {
    row.forEach((value, j) => {
        svg.append("rect")
           .attr("x", j * cellSize)
           .attr("y", i * cellSize)
           .attr("width", cellSize)
           .attr("height", cellSize)
           .attr("fill", colorScale(value));
    });
});

Mejorando el Mapa de Calor

  • Agregar Etiquetas: Agrega etiquetas a las celdas del mapa de calor para una mejor legibilidad.

    data.forEach((row, i) => {
    row.forEach((value, j) => {
        svg.append("text")
           .attr("x", j * cellSize + cellSize / 2)
           .attr("y", i * cellSize + cellSize / 2)
           .attr("dy", ".35em")
           .attr("text-anchor", "middle")
           .text(value)
           .attr("fill", "black");
    });
});

  • Características Interactivas: Agrega interactividad como tooltips para mostrar los valores de los datos al pasar el cursor.

    const tooltip = d3.select("body").append("div")
                  .style("position", "absolute")
                  .style("background", "#f9f9f9")
                  .style("padding", "5px")
                  .style("border", "1px solid #d3d3d3")
                  .style("border-radius", "5px")
                  .style("visibility", "hidden");

svg.selectAll("rect")
   .on("mouseover", function(event, d) {
       tooltip.style("visibility", "visible")
              .text(d3.select(this).data());
   })
   .on("mousemove", function(event) {
       tooltip.style("top", (event.pageY - 10) + "px")
              .style("left", (event.pageX + 10) + "px");
   })
   .on("mouseout", function() {
       tooltip.style("visibility", "hidden");
   });

Técnicas Avanzadas

  • Carga Dinámica de Datos: Carga datos dinámicamente desde fuentes externas como archivos CSV o JSON.

    d3.csv("data.csv").then(data => {
    // Procesa y renderiza el mapa de calor usando los datos cargados

  • Escalas de Colores Personalizadas: Crea escalas de colores personalizadas para tener más control sobre la apariencia del mapa de calor.

    const customColorScale = d3.scaleLinear()
                           .domain([0, 50, 100])
                           .range(["green", "yellow", "red"]);

  • Mapas de Calor Responsivos: Haz que el mapa de calor sea responsivo a diferentes tamaños de pantalla.

    const svg = d3.select("#heatmap")
              .append("svg")
              .attr("viewBox", `0 0 ${width} ${height}`)
              .attr("preserveAspectRatio", "xMidYMid meet");

Conclusión

Los mapas de calor son una forma versátil y visualmente atractiva de representar datos. Usando D3.js, puedes crear mapas de calor dinámicos e interactivos que proporcionan valiosos conocimientos sobre tus datos. Desde configurar un mapa de calor básico hasta implementar características avanzadas, esta guía ha cubierto los pasos esenciales para ayudarte a comenzar con los mapas de calor en D3.js. Experimenta con diferentes conjuntos de datos y opciones de personalización para crear mapas de calor que mejor se adapten a tus necesidades.
