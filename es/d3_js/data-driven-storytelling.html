Introducción a la narración basada en datos

La narración basada en datos es la práctica de usar datos para contar historias convincentes. Implica el uso de técnicas de visualización de datos para presentar datos de una manera que sea tanto informativa como atractiva.

  • Importancia de la narración basada en datos:
    • Hace que los datos complejos sean comprensibles.
    • Involucra a la audiencia.
    • Apoya la toma de decisiones basada en datos.

Empezando con D3.js

D3.js (Documentos Basados en Datos) es una biblioteca de JavaScript para crear visualizaciones de datos dinámicas e interactivas en navegadores web.

  • Conceptos clave:
    • Selecciones: Seleccionar elementos del DOM.
    • Vinculación de datos: Vincular datos a elementos del DOM.
    • Entrar, Actualizar, Salir: Manejar cambios basados en datos en el DOM.

Ejemplo básico: Creando un gráfico de barras simple

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Gráfico de Barras Simple</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <style>
        .bar {
            fill: steelblue;
        }
    </style>
</head>
<body>
    <script>
        const data = [4, 8, 15, 16, 23, 42];

        const width = 420;
        const barHeight = 20;

        const x = d3.scaleLinear()
            .domain([0, d3.max(data)])
            .range([0, width]);

        const chart = d3.select("body")
            .append("svg")
            .attr("width", width)
            .attr("height", barHeight * data.length);

        const bar = chart.selectAll("g")
            .data(data)
            .enter().append("g")
            .attr("transform", (d, i) => `translate(0,${i * barHeight})`);

        bar.append("rect")
            .attr("width", x)
            .attr("height", barHeight - 1)
            .attr("class", "bar");

        bar.append("text")
            .attr("x", d => x(d) - 3)
            .attr("y", barHeight / 2)
            .attr("dy", ".35em")
            .text(d => d);
    </script>
</body>
</html>

Técnicas intermedias

A medida que te sientas más cómodo con D3.js, puedes comenzar a explorar visualizaciones e interacciones más complejas.

  • Escalas y Ejes:

    • Las escalas mapean valores de datos a valores visuales.
    • Los ejes proporcionan marcas de referencia para las escalas.

  • Transiciones:

    • Animar cambios en el DOM.

Ejemplo: Añadiendo transiciones al gráfico de barras

bar.select("rect")
    .transition()
    .duration(750)
    .attr("width", x);

Técnicas avanzadas

Las técnicas avanzadas de D3.js implican la creación de visualizaciones altamente interactivas y complejas.

  • Interactividad:

    • Añadir tooltips.
    • Manejar eventos de usuario (clic, hover).

  • Diseños complejos:

    • Gráficos dirigidos por fuerza.
    • Diseños jerárquicos (por ejemplo, mapas de árbol, gráficos de rayos de sol).

Ejemplo: Añadiendo tooltips

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Gráfico de Barras con Tooltips</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <style>
        .bar {
            fill: steelblue;
        }
        .tooltip {
            position: absolute;
            text-align: center;
            width: 60px;
            height: 28px;
            padding: 2px;
            font: 12px sans-serif;
            background: lightsteelblue;
            border: 0px;
            border-radius: 8px;
            pointer-events: none;
        }
    </style>
</head>
<body>
    <script>
        const data = [4, 8, 15, 16, 23, 42];

        const width = 420;
        const barHeight = 20;

        const x = d3.scaleLinear()
            .domain([0, d3.max(data)])
            .range([0, width]);

        const chart = d3.select("body")
            .append("svg")
            .attr("width", width)
            .attr("height", barHeight * data.length);

        const bar = chart.selectAll("g")
            .data(data)
            .enter().append("g")
            .attr("transform", (d, i) => `translate(0,${i * barHeight})`);

        bar.append("rect")
            .attr("width", x)
            .attr("height", barHeight - 1)
            .attr("class", "bar");

        bar.append("text")
            .attr("x", d => x(d) - 3)
            .attr("y", barHeight / 2)
            .attr("dy", ".35em")
            .text(d => d);

        const tooltip = d3.select("body").append("div")
            .attr("class", "tooltip")
            .style("opacity", 0);

        bar.on("mouseover", function(event, d) {
            tooltip.transition()
                .duration(200)
                .style("opacity", .9);
            tooltip.html(d)
                .style("left", (event.pageX) + "px")
                .style("top", (event.pageY - 28) + "px");
        })
        .on("mouseout", function(d) {
            tooltip.transition()
                .duration(500)
                .style("opacity", 0);
        });
    </script>
</body>
</html>

Conclusión

La narración basada en datos con D3.js te permite crear visualizaciones poderosas que pueden comunicar datos complejos de una manera atractiva y comprensible. Al dominar los conceptos básicos, las técnicas intermedias y las características avanzadas de D3.js, puedes construir visualizaciones interactivas y dinámicas que cuenten historias convincentes con tus datos.

  • Resumen:
    • Comienza con visualizaciones básicas.
    • Aprende a usar escalas, ejes y transiciones.
    • Añade interactividad y explora diseños complejos.
    • Practica y experimenta para mejorar tus habilidades.
