Introducción

Construir tableros interactivos es una forma poderosa de visualizar datos y obtener información. D3.js (Documentos Impulsados por Datos) es una biblioteca de JavaScript que te permite crear visualizaciones de datos dinámicas e interactivas en navegadores web. Este tema te guiará a través del proceso de construcción de tableros usando D3.js, comenzando desde lo básico y progresando a técnicas avanzadas.

Comenzando con D3.js

Configuración de tu Entorno

  • Instala D3.js vía CDN o npm.
  • Crea una plantilla HTML básica para alojar tu tablero.
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Tablero D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
    <style>
        /* Estilo básico para el tablero */
        .chart {
            display: inline-block;
            margin: 10px;
        }
    </style>
</head>
<body>
    <div id="dashboard"></div>
    <script src="app.js"></script>
</body>
</html>

Conceptos Básicos de D3.js

  • Selecciones: Selecciona elementos del DOM y vincula datos a ellos.
  • Vinculación de Datos: Usa .data() para vincular datos a elementos.
  • Enter, Update, Exit: Maneja cambios impulsados por datos en el DOM.
// app.js
const data = [10, 20, 30, 40, 50];

d3.select("#dashboard")
  .selectAll("div")
  .data(data)
  .enter()
  .append("div")
  .style("width", d => `${d * 10}px`)
  .style("height", "20px")
  .style("background-color", "steelblue")
  .style("margin", "5px");

Creación de Gráficos Básicos

Gráfico de Barras

  • Preparación de Datos: Formatea tus datos.
  • Escalas: Mapea valores de datos a dimensiones visuales.
  • Ejes: Crea y estiliza ejes.
const svg = d3.select("#dashboard")
              .append("svg")
              .attr("width", 500)
              .attr("height", 300);

const xScale = d3.scaleBand()
                 .domain(data.map((d, i) => i))
                 .range([0, 500])
                 .padding(0.1);

const yScale = d3.scaleLinear()
                 .domain([0, d3.max(data)])
                 .range([300, 0]);

svg.selectAll(".bar")
   .data(data)
   .enter()
   .append("rect")
   .attr("class", "bar")
   .attr("x", (d, i) => xScale(i))
   .attr("y", d => yScale(d))
   .attr("width", xScale.bandwidth())
   .attr("height", d => 300 - yScale(d))
   .attr("fill", "steelblue");

Gráfico de Líneas

  • Generador de Líneas: Usa d3.line() para crear una ruta.
  • Vinculación de Datos: Vincula datos al elemento de ruta.
const lineData = [
  { x: 0, y: 30 },
  { x: 1, y: 50 },
  { x: 2, y: 80 },
  { x: 3, y: 40 },
  { x: 4, y: 60 }
];

const line = d3.line()
               .x(d => xScale(d.x))
               .y(d => yScale(d.y));

svg.append("path")
   .datum(lineData)
   .attr("fill", "none")
   .attr("stroke", "steelblue")
   .attr("stroke-width", 2)
   .attr("d", line);

Técnicas Intermedias

Añadiendo Interactividad

  • Listeners de Eventos: Añade eventos de ratón a elementos.
  • Transiciones: Anima cambios en los datos.
svg.selectAll(".bar")
   .on("mouseover", function(event, d) {
       d3.select(this).attr("fill", "orange");
   })
   .on("mouseout", function(event, d) {
       d3.select(this).attr("fill", "steelblue");
   });

svg.selectAll(".bar")
   .transition()
   .duration(1000)
   .attr("height", d => 300 - yScale(d))
   .attr("y", d => yScale(d));

Diseño Responsivo

  • ViewBox de SVG: Haz tus gráficos responsivos.
  • Escalado Dinámico: Ajusta las escalas según el tamaño de la ventana.
const svg = d3.select("#dashboard")
              .append("svg")
              .attr("viewBox", "0 0 500 300")
              .attr("preserveAspectRatio", "xMidYMid meet");

Técnicas Avanzadas

Integración con Otras Bibliotecas

  • Crossfilter: Maneja grandes conjuntos de datos.
  • DC.js: Simplifica la creación de gráficos.
// Ejemplo de integración con Crossfilter
const cf = crossfilter(data);
const dimension = cf.dimension(d => d.value);
const group = dimension.group().reduceCount();

dc.barChart("#chart")
  .dimension(dimension)
  .group(group)
  .x(d3.scaleLinear().domain([0, 100]));

Visualizaciones Personalizadas

  • Formas Personalizadas: Crea elementos visuales únicos.
  • Interacciones Complejas: Implementa arrastrar y soltar, zoom y paneo.
const drag = d3.drag()
               .on("start", function(event, d) {
                   d3.select(this).raise().attr("stroke", "black");
               })
               .on("drag", function(event, d) {
                   d3.select(this).attr("cx", d.x = event.x).attr("cy", d.y = event.y);
               })
               .on("end", function(event, d) {
                   d3.select(this).attr("stroke", null);
               });

svg.selectAll("circle")
   .data(data)
   .enter()
   .append("circle")
   .attr("cx", d => xScale(d.x))
   .attr("cy", d => yScale(d.y))
   .attr("r", 5)
   .call(drag);

Conclusión

Construir tableros con D3.js implica entender los conceptos básicos de la vinculación de datos, crear varios tipos de gráficos y añadir interactividad y capacidad de respuesta. A medida que progreses hacia técnicas más avanzadas, puedes integrar otras bibliotecas y crear visualizaciones personalizadas para satisfacer necesidades específicas. La práctica y la experimentación son clave para dominar D3.js y crear tableros interactivos y efectivos.
