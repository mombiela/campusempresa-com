Introducción

Los gráficos de líneas son un tipo fundamental de visualización de datos que se utilizan para mostrar información como una serie de puntos de datos conectados por segmentos de línea recta. D3.js (Documentos Basados en Datos) es una poderosa biblioteca de JavaScript para crear visualizaciones de datos dinámicas e interactivas en navegadores web. Esta sección te guiará a través del proceso de creación de gráficos de líneas utilizando D3.js, desde niveles básicos hasta avanzados.

Configuración de Tu Entorno

Antes de crear un gráfico de líneas, asegúrate de tener la configuración necesaria:

  • Incluye la biblioteca D3.js en tu archivo HTML.
  • Prepara un contenedor HTML (como un elemento SVG) donde se renderizará el gráfico.

Incluyendo D3.js

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Gráfico de Líneas con D3.js</title>
    <script src="https://d3js.org/d3.v7.min.js"></script>
</head>
<body>
    <svg width="800" height="400"></svg>
</body>
</html>

Gráfico de Líneas Básico

Preparación de Datos

Comienza preparando tus datos. Para simplificar, usaremos un array de objetos donde cada objeto representa un punto de datos.

const data = [
    { date: new Date(2023, 0, 1), value: 30 },
    { date: new Date(2023, 1, 1), value: 40 },
    { date: new Date(2023, 2, 1), value: 35 },
    { date: new Date(2023, 3, 1), value: 50 },
    { date: new Date(2023, 4, 1), value: 55 }
];

Creación de Escalas

Las escalas son funciones que mapean valores de datos a valores de píxeles. Crearemos escalas para los ejes x e y.

const svg = d3.select("svg");
const margin = { top: 20, right: 30, bottom: 30, left: 40 };
const width = +svg.attr("width") - margin.left - margin.right;
const height = +svg.attr("height") - margin.top - margin.bottom;

const x = d3.scaleTime()
    .domain(d3.extent(data, d => d.date))
    .range([0, width]);

const y = d3.scaleLinear()
    .domain([0, d3.max(data, d => d.value)])
    .range([height, 0]);

Dibujando la Línea

Usa la función d3.line para crear un generador de líneas.

const line = d3.line()
    .x(d => x(d.date))
    .y(d => y(d.value));

const g = svg.append("g")
    .attr("transform", `translate(${margin.left},${margin.top})`);

g.append("path")
    .datum(data)
    .attr("fill", "none")
    .attr("stroke", "steelblue")
    .attr("stroke-width", 1.5)
    .attr("d", line);

Añadiendo Ejes

Añade ejes x e y al gráfico.

g.append("g")
    .attr("transform", `translate(0,${height})`)
    .call(d3.axisBottom(x));

g.append("g")
    .call(d3.axisLeft(y));

Gráfico de Líneas Intermedio

Añadiendo Interactividad

Mejora el gráfico añadiendo tooltips que muestren los valores de los datos al pasar el ratón sobre los puntos.

const tooltip = d3.select("body").append("div")
    .attr("class", "tooltip")
    .style("opacity", 0);

g.selectAll("dot")
    .data(data)
    .enter().append("circle")
    .attr("cx", d => x(d.date))
    .attr("cy", d => y(d.value))
    .attr("r", 5)
    .attr("fill", "red")
    .on("mouseover", (event, d) => {
        tooltip.transition()
            .duration(200)
            .style("opacity", .9);
        tooltip.html(`Fecha: ${d.date}<br/>Valor: ${d.value}`)
            .style("left", (event.pageX + 5) + "px")
            .style("top", (event.pageY - 28) + "px");
    })
    .on("mouseout", d => {
        tooltip.transition()
            .duration(500)
            .style("opacity", 0);
    });

Estilizando el Gráfico

Mejora el atractivo visual añadiendo estilos CSS.

.tooltip {
    position: absolute;
    text-align: center;
    width: 60px;
    height: 28px;
    padding: 2px;
    font: 12px sans-serif;
    background: lightsteelblue;
    border: 0px;
    border-radius: 8px;
    pointer-events: none;
}

Gráfico de Líneas Avanzado

Manejo de Datos en Tiempo Real

Actualiza el gráfico dinámicamente con datos en tiempo real.

function updateChart(newData) {
    // Actualiza las escalas
    x.domain(d3.extent(newData, d => d.date));
    y.domain([0, d3.max(newData, d => d.value)]);

    // Actualiza la línea
    g.select("path")
        .datum(newData)
        .attr("d", line);

    // Actualiza los puntos
    const circles = g.selectAll("circle")
        .data(newData);

    circles.enter().append("circle")
        .attr("r", 5)
        .attr("fill", "red")
        .merge(circles)
        .attr("cx", d => x(d.date))
        .attr("cy", d => y(d.value));

    circles.exit().remove();

    // Actualiza los ejes
    g.select(".x.axis")
        .call(d3.axisBottom(x));

    g.select(".y.axis")
        .call(d3.axisLeft(y));
}

Añadiendo Zoom y Pan

Habilita el zoom y el desplazamiento para una mejor exploración de los datos.

const zoom = d3.zoom()
    .scaleExtent([1, 10])
    .translateExtent([[-width, -height], [2 * width, 2 * height]])
    .on("zoom", zoomed);

svg.call(zoom);

function zoomed(event) {
    const transform = event.transform;
    const newX = transform.rescaleX(x);
    const newY = transform.rescaleY(y);

    g.select(".x.axis").call(d3.axisBottom(newX));
    g.select(".y.axis").call(d3.axisLeft(newY));

    g.select("path").attr("d", line.x(d => newX(d.date)).y(d => newY(d.value)));
    g.selectAll("circle")
        .attr("cx", d => newX(d.date))
        .attr("cy", d => newY(d.value));
}

Conclusión

Crear gráficos de líneas con D3.js implica varios pasos, desde configurar el entorno y preparar los datos hasta dibujar el gráfico y añadir interactividad. Al dominar estas técnicas, puedes crear visualizaciones dinámicas e interactivas que comuniquen eficazmente los conocimientos de los datos. A medida que avances, puedes incorporar actualizaciones de datos en tiempo real y habilitar el zoom y el desplazamiento para mejorar la experiencia del usuario.
