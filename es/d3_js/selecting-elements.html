Selección de Elementos
Vinculando Datos a Elementos ►

Seleccionar elementos es un concepto fundamental en D3.js, ya que te permite vincular datos al DOM y manipular elementos basados en esos datos. Este tema te guiará desde técnicas básicas hasta avanzadas para seleccionar elementos en D3.js.

Selección Básica

Seleccionar un Solo Elemento

  • Usa d3.select() para seleccionar un solo elemento.
  • Este método selecciona el primer elemento coincidente en el DOM.
// Selecciona el primer elemento <p> y cambia su color de texto a rojo
d3.select("p").style("color", "red");

Seleccionar Múltiples Elementos

  • Usa d3.selectAll() para seleccionar múltiples elementos.
  • Este método selecciona todos los elementos coincidentes en el DOM.
// Selecciona todos los elementos <p> y cambia su color de texto a azul
d3.selectAll("p").style("color", "blue");

Selección Intermedia

Encadenar Métodos

  • D3.js permite el encadenamiento de métodos, lo que significa que puedes aplicar múltiples métodos en una sola declaración.
// Selecciona todos los elementos <p>, cambia su color de texto a verde y establece su tamaño de fuente
d3.selectAll("p")
  .style("color", "green")
  .style("font-size", "20px");

Usar Selectores

  • Puedes usar selectores CSS para hacer selecciones más complejas.
// Selecciona todos los elementos <p> con la clase 'highlight' y cambia su color de fondo
d3.selectAll("p.highlight").style("background-color", "yellow");

Selección Avanzada

Seleccionar Elementos Anidados

  • Usa selecciones anidadas para seleccionar elementos dentro de un elemento padre específico.
// Selecciona todos los elementos <li> dentro de un <ul> con el id 'list' y cambia su color de texto
d3.select("#list").selectAll("li").style("color", "purple");

Usar Funciones para Selección Dinámica

  • Puedes usar funciones para establecer dinámicamente estilos o atributos basados en datos.
// Selecciona todos los elementos <p> y establece su color de texto basado en su índice
d3.selectAll("p").style("color", function(d, i) {
  return i % 2 === 0 ? "red" : "blue";
});

Combinar Datos con Selecciones

  • Usa data() para vincular datos a elementos y enter() para manejar nuevos puntos de datos.
// Array de datos
var data = [10, 20, 30, 40, 50];

// Selecciona todos los elementos <div> y vincula datos
var divs = d3.select("body").selectAll("div")
  .data(data);

// Selección de entrada para crear nuevos elementos <div> para nuevos puntos de datos
divs.enter().append("div")
  .style("width", function(d) { return d + "px"; })
  .style("height", "20px")
  .style("background-color", "teal")
  .text(function(d) { return d; });

Conclusión

Seleccionar elementos es una habilidad crucial en D3.js, permitiéndote vincular datos al DOM y manipular elementos dinámicamente. Comenzando con selecciones básicas usando d3.select() y d3.selectAll(), puedes progresar a técnicas más avanzadas como selecciones anidadas, estilos dinámicos y combinar datos con selecciones. Dominar estas técnicas te proporcionará una base sólida para crear visualizaciones complejas e interactivas con D3.js.
