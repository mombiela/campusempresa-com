La vinculación de datos es un concepto fundamental en D3.js, que permite la conexión entre los datos y los elementos en el DOM. Este tema te guiará desde las técnicas básicas hasta las avanzadas de vinculación de datos en D3.js.

¿Qué es la Vinculación de Datos?

La vinculación de datos en D3.js se refiere al proceso de conectar datos a elementos del DOM. Esto te permite crear, actualizar y manipular dinámicamente elementos basados en los datos.

Conceptos Clave

  • Selección: El proceso de seleccionar elementos del DOM.
  • Unión de Datos: El proceso de vincular datos a los elementos seleccionados.
  • Entrar, Actualizar, Salir: Las tres fases de la vinculación de datos en D3.js.

Vinculación de Datos Básica

Selección de Elementos

Para vincular datos a elementos, primero necesitas seleccionar los elementos. D3.js utiliza selectores CSS para seleccionar elementos.

d3.select("body") // Selecciona el elemento body
d3.selectAll("p") // Selecciona todos los elementos de párrafo

Vinculación de Datos

Una vez que has seleccionado los elementos, puedes vincular datos a ellos usando el método .data().

const data = [10, 20, 30, 40, 50];

d3.selectAll("p")
  .data(data)
  .text(d => d); // Establece el texto de cada párrafo al valor de datos correspondiente

Entrar, Actualizar, Salir

Al vincular datos, D3.js categoriza los elementos en tres grupos:

  • Entrar: Elementos que necesitan ser creados.
  • Actualizar: Elementos que necesitan ser actualizados.
  • Salir: Elementos que necesitan ser eliminados.
const data = [10, 20, 30, 40, 50];

const p = d3.select("body").selectAll("p")
  .data(data);

// Fase de Entrar
p.enter().append("p")
  .text(d => d);

// Fase de Actualizar
p.text(d => d);

// Fase de Salir
p.exit().remove();

Vinculación de Datos Avanzada

Vinculación de Datos a Elementos Complejos

Puedes vincular datos a elementos más complejos, como formas SVG.

const data = [10, 20, 30, 40, 50];

const svg = d3.select("body").append("svg")
  .attr("width", 500)
  .attr("height", 100);

const circles = svg.selectAll("circle")
  .data(data);

// Fase de Entrar
circles.enter().append("circle")
  .attr("cx", (d, i) => i * 50 + 25)
  .attr("cy", 50)
  .attr("r", d => d / 2);

// Fase de Actualizar
circles.attr("r", d => d / 2);

// Fase de Salir
circles.exit().remove();

Funciones Clave

Al trabajar con datos más complejos, puede que necesites usar funciones clave para asegurar que los datos se vinculen correctamente a los elementos.

const data = [
  { id: 1, value: 10 },
  { id: 2, value: 20 },
  { id: 3, value: 30 }
];

const circles = svg.selectAll("circle")
  .data(data, d => d.id); // Usando el id como la función clave

// Fase de Entrar
circles.enter().append("circle")
  .attr("cx", (d, i) => i * 50 + 25)
  .attr("cy", 50)
  .attr("r", d => d.value / 2);

// Fase de Actualizar
circles.attr("r", d => d.value / 2);

// Fase de Salir
circles.exit().remove();

Conclusión

La vinculación de datos es una característica poderosa de D3.js que te permite crear, actualizar y eliminar dinámicamente elementos basados en tus datos. Entender las fases de entrar, actualizar y salir es crucial para usar D3.js de manera efectiva. Al dominar estos conceptos, puedes crear visualizaciones complejas e interactivas que respondan a los cambios en tus datos.
