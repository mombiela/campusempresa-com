Introducción a D3.js

D3.js (Documentos Basados en Datos) es una poderosa biblioteca de JavaScript utilizada para crear visualizaciones de datos dinámicas e interactivas en navegadores web. Este curso te llevará desde niveles principiantes hasta avanzados, cubriendo los conceptos fundamentales y las técnicas avanzadas necesarias para dominar D3.js.

Conceptos Clave

  • Vinculación de Datos: Aprende a vincular datos a elementos del DOM.
  • Selecciones: Entiende cómo seleccionar y manipular elementos del DOM.
  • Escalas: Usa escalas para mapear valores de datos a propiedades visuales.
  • Ejes: Crea y personaliza ejes para tus visualizaciones.
  • Formas: Dibuja formas básicas como líneas, rectángulos y círculos.
  • Transiciones: Añade animaciones y transiciones a tus visualizaciones.
  • Interactividad: Haz que tus visualizaciones sean interactivas con escuchadores de eventos.
  • Diseños: Usa diseños de D3 para visualizaciones complejas como gráficos de pastel, gráficos dirigidos por fuerza y mapas de árbol.

Código de Ejemplo

Aquí tienes un ejemplo básico de cómo crear un gráfico de barras simple con D3.js:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Gráfico de Barras Simple</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
    <style>
        .bar {
            fill: steelblue;
        }
    </style>
</head>
<body>
    <script>
        // Datos de ejemplo
        const data = [30, 86, 168, 281, 303, 365];

        // Crear el contenedor SVG
        const svg = d3.select("body").append("svg")
            .attr("width", 500)
            .attr("height", 300);

        // Crear las barras
        svg.selectAll("rect")
            .data(data)
            .enter().append("rect")
            .attr("class", "bar")
            .attr("x", (d, i) => i * 50)
            .attr("y", d => 300 - d)
            .attr("width", 40)
            .attr("height", d => d);
    </script>
</body>
</html>

Explicación

  • Datos: Un array de números que representa los datos a visualizar.
  • Contenedor SVG: Se crea un elemento SVG y se añade al cuerpo del documento HTML.
  • Barras: Se crean rectángulos para cada punto de datos. El atributo x posiciona las barras horizontalmente, el atributo y las posiciona verticalmente, y el atributo height establece su altura basada en los valores de los datos.

Conclusión

Al final de este curso, tendrás una comprensión completa de D3.js y serás capaz de crear una amplia gama de visualizaciones de datos. Comenzarás con lo básico y avanzarás gradualmente hacia temas más avanzados, asegurando una base sólida y la capacidad de abordar proyectos complejos. ¡Feliz codificación!
