Introducción a las Escalas

En D3.js, las escalas son funciones que mapean desde un dominio de entrada a un rango de salida. Son esenciales para crear visualizaciones porque te permiten transformar valores de datos en propiedades visuales como posición, longitud y color.

Conceptos Clave

  • Dominio: El conjunto de valores de entrada.
  • Rango: El conjunto de valores de salida.
  • Tipos de Escalas: Diferentes tipos de escalas para diferentes tipos de datos (por ejemplo, lineal, ordinal, de tiempo).

Tipos de Escalas

Escalas Lineales

Las escalas lineales se utilizan para datos cuantitativos continuos. Mapean un dominio de entrada continuo a un rango de salida continuo.

// Crear una escala lineal
const linearScale = d3.scaleLinear()
    .domain([0, 100])  // Dominio de entrada
    .range([0, 500]);  // Rango de salida

// Usar la escala
console.log(linearScale(50));  // Salida: 250

Escalas Ordinales

Las escalas ordinales se utilizan para datos categóricos. Mapean un dominio discreto a un rango discreto.

// Crear una escala ordinal
const ordinalScale = d3.scaleOrdinal()
    .domain(['A', 'B', 'C'])  // Dominio de entrada
    .range([0, 100, 200]);    // Rango de salida

// Usar la escala
console.log(ordinalScale('B'));  // Salida: 100

Escalas de Tiempo

Las escalas de tiempo se utilizan para datos basados en el tiempo. Mapean un dominio de tiempo continuo a un rango de salida continuo.

// Crear una escala de tiempo
const timeScale = d3.scaleTime()
    .domain([new Date(2020, 0, 1), new Date(2020, 11, 31)])  // Dominio de entrada
    .range([0, 500]);  // Rango de salida

// Usar la escala
console.log(timeScale(new Date(2020, 5, 15)));  // Salida: 250

Personalización de Escalas

Clamping

El clamping restringe el rango de salida al rango especificado, incluso si el valor de entrada está fuera del dominio.

// Crear una escala lineal con clamping
const clampedScale = d3.scaleLinear()
    .domain([0, 100])
    .range([0, 500])
    .clamp(true);

// Usar la escala
console.log(clampedScale(150));  // Salida: 500 (clamped)

Nice

El método nice extiende el dominio a valores redondeados, lo que puede hacer que las etiquetas del eje sean más legibles.

// Crear una escala lineal y aplicar nice
const niceScale = d3.scaleLinear()
    .domain([0.123, 99.987])
    .range([0, 500])
    .nice();

// Usar la escala
console.log(niceScale.domain());  // Salida: [0, 100]

Ejemplos Prácticos

Creando un Gráfico de Barras

Usando escalas para crear un gráfico de barras simple.

// Datos de ejemplo
const data = [10, 20, 30, 40, 50];

// Crear escalas
const xScale = d3.scaleBand()
    .domain(data.map((d, i) => i))
    .range([0, 500])
    .padding(0.1);

const yScale = d3.scaleLinear()
    .domain([0, d3.max(data)])
    .range([400, 0]);

// Crear contenedor SVG
const svg = d3.select('body').append('svg')
    .attr('width', 600)
    .attr('height', 500);

// Crear barras
svg.selectAll('rect')
    .data(data)
    .enter().append('rect')
    .attr('x', (d, i) => xScale(i))
    .attr('y', d => yScale(d))
    .attr('width', xScale.bandwidth())
    .attr('height', d => 400 - yScale(d))
    .attr('fill', 'steelblue');

Conclusión

Entender las escalas en D3.js es crucial para una visualización de datos efectiva. Las escalas te permiten mapear valores de datos a propiedades visuales, haciendo que tus gráficos y diagramas sean precisos y significativos. Al dominar diferentes tipos de escalas y sus opciones de personalización, puedes crear una amplia gama de visualizaciones que comuniquen efectivamente tus conocimientos de datos.
