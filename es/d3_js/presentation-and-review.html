Presentación y Revisión

Introducción

En esta sección, presentaremos y revisaremos los conceptos y técnicas clave cubiertos en el curso de D3.js. Esto ayudará a consolidar tu comprensión y asegurará que estés bien preparado para aplicar D3.js en proyectos del mundo real.

Conceptos Clave

Vamos a repasar los conceptos y técnicas fundamentales que hemos aprendido a lo largo del curso:

  • Selecciones: Cómo seleccionar elementos del DOM usando D3.js.
  • Vinculación de Datos: Asociar datos con elementos del DOM.
  • Patrón Enter, Update, Exit: Gestionar cambios impulsados por datos en el DOM.
  • Escalas y Ejes: Mapear valores de datos a propiedades visuales.
  • Formas y Caminos: Crear y manipular formas SVG.
  • Transiciones y Animaciones: Añadir comportamiento dinámico a las visualizaciones.
  • Interactividad: Hacer visualizaciones interactivas con eventos.
  • Diseños: Usar diseños de D3.js para visualizaciones complejas como gráficos de fuerza dirigida y mapas de árbol.

Selecciones

Las selecciones son la base de D3.js, permitiéndote seleccionar y manipular elementos del DOM.

// Seleccionando todos los elementos de párrafo
d3.selectAll("p")
  .style("color", "blue");

Explicación:

  • d3.selectAll("p"): Selecciona todos los elementos <p>.
  • .style("color", "blue"): Cambia el color del texto a azul.

Vinculación de Datos

La vinculación de datos es el proceso de asociar datos con elementos del DOM.

// Vinculando datos a elementos de párrafo
const data = [10, 20, 30, 40, 50];
d3.selectAll("p")
  .data(data)
  .text(d => `Valor: ${d}`);

Explicación:

  • data: Un array de valores.
  • .data(data): Vincula el array de datos a los elementos seleccionados.
  • .text(d => Valor: ${d}): Establece el contenido de texto de cada párrafo al valor de datos correspondiente.

Patrón Enter, Update, Exit

Este patrón es crucial para gestionar datos dinámicos.

const data = [10, 20, 30];
const p = d3.select("body").selectAll("p")
  .data(data);

// Selección Enter
p.enter().append("p")
  .text(d => `Nuevo Valor: ${d}`);

// Selección Update
p.text(d => `Valor Actualizado: ${d}`);

// Selección Exit
p.exit().remove();

Explicación:

  • p.enter(): Maneja nuevos elementos de datos.
  • p.text(): Actualiza elementos existentes.
  • p.exit(): Elimina elementos que ya no tienen datos correspondientes.

Escalas y Ejes

Las escalas mapean valores de datos a propiedades visuales, y los ejes proporcionan marcas de referencia.

const scale = d3.scaleLinear()
  .domain([0, 100])
  .range([0, 500]);

const axis = d3.axisBottom(scale);

d3.select("svg")
  .append("g")
  .attr("transform", "translate(0, 50)")
  .call(axis);

Explicación:

  • d3.scaleLinear(): Crea una escala lineal.
  • .domain([0, 100]): Establece el dominio de entrada.
  • .range([0, 500]): Establece el rango de salida.
  • d3.axisBottom(scale): Crea un eje inferior usando la escala.
  • .call(axis): Renderiza el eje.

Formas y Caminos

Crear y manipular formas SVG es una parte central de D3.js.

const line = d3.line()
  .x(d => d.x)
  .y(d => d.y);

const data = [{x: 0, y: 0}, {x: 50, y: 50}, {x: 100, y: 0}];

d3.select("svg")
  .append("path")
  .datum(data)
  .attr("d", line)
  .attr("stroke", "black")
  .attr("fill", "none");

Explicación:

  • d3.line(): Crea un generador de líneas.
  • .x(d => d.x): Establece el accesor de coordenadas x.
  • .y(d => d.y): Establece el accesor de coordenadas y.
  • .datum(data): Vincula los datos al elemento path.
  • .attr("d", line): Genera los datos del camino.
  • .attr("stroke", "black"): Establece el color del trazo.
  • .attr("fill", "none"): Elimina el relleno.

Transiciones y Animaciones

Las transiciones añaden comportamiento dinámico a las visualizaciones.

d3.select("circle")
  .transition()
  .duration(1000)
  .attr("r", 50);

Explicación:

  • .transition(): Inicia una transición.
  • .duration(1000): Establece la duración a 1000 milisegundos.
  • .attr("r", 50): Cambia el radio a 50.

Interactividad

La interactividad hace que las visualizaciones sean más atractivas.

d3.select("circle")
  .on("mouseover", function() {
    d3.select(this).attr("fill", "red");
  })
  .on("mouseout", function() {
    d3.select(this).attr("fill", "blue");
  });

Explicación:

  • .on("mouseover", function() {...}): Añade un listener de evento mouseover.
  • .on("mouseout", function() {...}): Añade un listener de evento mouseout.
  • d3.select(this).attr("fill", "red"): Cambia el color de relleno a rojo en mouseover.
  • d3.select(this).attr("fill", "blue"): Cambia el color de relleno de nuevo a azul en mouseout.

Diseños

Los diseños ayudan a crear visualizaciones complejas.

const data = [10, 20, 30, 40, 50];
const pie = d3.pie()(data);

const arc = d3.arc()
  .innerRadius(0)
  .outerRadius(100);

d3.select("svg")
  .selectAll("path")
  .data(pie)
  .enter()
  .append("path")
  .attr("d", arc)
  .attr("fill", (d, i) => d3.schemeCategory10[i]);

Explicación:

  • d3.pie(): Crea un diseño de pastel.
  • d3.arc(): Crea un generador de arcos.
  • .innerRadius(0): Establece el radio interno.
  • .outerRadius(100): Establece el radio externo.
  • .data(pie): Vincula los datos del pastel a los elementos path.
  • .attr("d", arc): Genera los datos del camino del arco.
  • .attr("fill", (d, i) => d3.schemeCategory10[i]): Establece el color de relleno usando un esquema de colores.

Conclusión

En esta sección, revisamos los conceptos y técnicas clave de D3.js, incluyendo selecciones, vinculación de datos, el patrón enter-update-exit, escalas y ejes, formas y caminos, transiciones y animaciones, interactividad y diseños. Al dominar estos conceptos, estás bien equipado para crear visualizaciones de datos poderosas y dinámicas con D3.js. Sigue practicando y experimentando con diferentes conjuntos de datos y tipos de visualización para mejorar aún más tus habilidades.
