En esta sección, exploraremos los conceptos de escalas lineales y ordinales en D3.js. Las escalas son una parte fundamental de D3.js, ya que te permiten mapear valores de datos a dimensiones visuales. Entender cómo usar las escalas de manera efectiva es crucial para crear visualizaciones significativas.

Escalas Lineales

Las escalas lineales se utilizan para mapear dominios de entrada continuos a rangos de salida continuos. Son particularmente útiles cuando se trata de datos numéricos que necesitan ser representados proporcionalmente.

Conceptos Clave

  • Dominio: El conjunto de valores de entrada.
  • Rango: El conjunto de valores de salida.
  • Mapeo: El proceso de convertir un valor del dominio al rango.

Ejemplo

Vamos a crear una escala lineal simple que mapea un dominio de [0, 100] a un rango de [0, 500].

// Crear una escala lineal
const linearScale = d3.scaleLinear()
    .domain([0, 100])
    .range([0, 500]);

// Probar la escala
console.log(linearScale(0));   // Salida: 0
console.log(linearScale(50));  // Salida: 250
console.log(linearScale(100)); // Salida: 500

Explicación

  • d3.scaleLinear(): Crea una nueva escala lineal.
  • domain([0, 100]): Establece el dominio de entrada para que vaya de 0 a 100.
  • range([0, 500]): Establece el rango de salida para que vaya de 0 a 500.

Escalas Ordinales

Las escalas ordinales se utilizan para mapear dominios de entrada discretos a rangos de salida discretos. Son útiles cuando se trata de datos categóricos.

Conceptos Clave

  • Dominio: El conjunto de valores de entrada discretos.
  • Rango: El conjunto de valores de salida discretos.
  • Mapeo: El proceso de convertir un valor del dominio al rango.

Ejemplo

Vamos a crear una escala ordinal que mapea un dominio de ["A", "B", "C"] a un rango de [0, 100, 200].

// Crear una escala ordinal
const ordinalScale = d3.scaleOrdinal()
    .domain(["A", "B", "C"])
    .range([0, 100, 200]);

// Probar la escala
console.log(ordinalScale("A")); // Salida: 0
console.log(ordinalScale("B")); // Salida: 100
console.log(ordinalScale("C")); // Salida: 200

Explicación

  • d3.scaleOrdinal(): Crea una nueva escala ordinal.
  • domain(["A", "B", "C"]): Establece el dominio de entrada a los valores discretos "A", "B" y "C".
  • range([0, 100, 200]): Establece el rango de salida a los valores discretos 0, 100 y 200.

Comparación de Escalas Lineales y Ordinales

| Característica | Escala Lineal | Escala Ordinal | |-----------------|------------------------------------|-------------------------------------| | Dominio de Entrada | Continuo (e.g., [0, 100]) | Discreto (e.g., ["A", "B", "C"]) | | Rango de Salida | Continuo (e.g., [0, 500]) | Discreto (e.g., [0, 100, 200]) | | Caso de Uso | Datos numéricos | Datos categóricos | | Mapeo | Proporcional | No proporcional |

Conclusión

Entender las escalas lineales y ordinales es esencial para crear visualizaciones efectivas en D3.js. Las escalas lineales son ideales para datos numéricos, proporcionando un mapeo proporcional de entrada a salida. Las escalas ordinales, por otro lado, son perfectas para datos categóricos, mapeando valores de entrada discretos a valores de salida discretos.

Al dominar estas escalas, puedes asegurarte de que tus datos se representen con precisión en tus visualizaciones, haciéndolas más informativas y atractivas.
