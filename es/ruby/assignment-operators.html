Los operadores de asignación se utilizan para asignar valores a variables. En Ruby, los operadores de asignación no solo incluyen el operador básico =, sino también una variedad de operadores compuestos que realizan una operación y asignación en un solo paso.

Operador de Asignación Básico

El operador de asignación más fundamental es el operador =.

x = 10
y = "¡Hola, Ruby!"
  • x = 10 asigna el valor entero 10 a la variable x.
  • y = "¡Hola, Ruby!" asigna la cadena "¡Hola, Ruby!" a la variable y.

Operadores de Asignación Compuestos

Los operadores de asignación compuestos combinan una operación aritmética con la asignación. Estos operadores son una forma abreviada de realizar una operación y luego asignar el resultado a una variable.

Asignación de Suma (`+=`)

x = 5
x += 3  # Equivalente a x = x + 3
puts x  # Salida: 8

Asignación de Resta (`-=`)

x = 5
x -= 2  # Equivalente a x = x - 2
puts x  # Salida: 3

Asignación de Multiplicación (`*=`)

x = 5
x *= 4  # Equivalente a x = x * 4
puts x  # Salida: 20

Asignación de División (`/=`)

x = 20
x /= 4  # Equivalente a x = x / 4
puts x  # Salida: 5

Asignación de Módulo (`%=`)

x = 20
x %= 3  # Equivalente a x = x % 3
puts x  # Salida: 2

Asignación de Exponenciación (`**=`)

x = 2
x **= 3  # Equivalente a x = x ** 3
puts x  # Salida: 8

Operadores de Asignación Bit a Bit

Ruby también soporta operadores de asignación bit a bit, que realizan operaciones bit a bit y luego asignan el resultado a la variable.

Asignación AND Bit a Bit (`&=`)

x = 6  # Binario: 110
x &= 3  # Binario: 011, Resultado: 010 (2 en decimal)
puts x  # Salida: 2

Asignación OR Bit a Bit (`|=`)

x = 6  # Binario: 110
x |= 3  # Binario: 011, Resultado: 111 (7 en decimal)
puts x  # Salida: 7

Asignación XOR Bit a Bit (`^=`)

x = 6  # Binario: 110
x ^= 3  # Binario: 011, Resultado: 101 (5 en decimal)
puts x  # Salida: 5

Asignación de Desplazamiento a la Izquierda Bit a Bit (`<<=`)

x = 3  # Binario: 11
x <<= 2  # Binario: 1100 (12 en decimal)
puts x  # Salida: 12

Asignación de Desplazamiento a la Derecha Bit a Bit (`>>=`)

x = 12  # Binario: 1100
x >>= 2  # Binario: 11 (3 en decimal)
puts x  # Salida: 3

Conclusión

Los operadores de asignación en Ruby son herramientas esenciales para gestionar y manipular valores de variables de manera eficiente. Entender tanto los operadores de asignación básicos como los compuestos te permite escribir código más conciso y legible. Practica usando estos operadores en diferentes escenarios para familiarizarte con su funcionalidad y aplicaciones.
