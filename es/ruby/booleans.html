Introducción a los Booleanos

Los booleanos son un tipo de dato fundamental en Ruby, que representan valores verdaderos o falsos. Son esenciales para tomar decisiones en tu código a través de declaraciones condicionales y bucles.

  • True: Representa un valor verdadero.
  • False: Representa un valor falso.

En Ruby, los booleanos son instancias de las clases TrueClass y FalseClass.

Literales Booleanos

En Ruby, los literales booleanos son true y false.

is_ruby_fun = true
is_sky_green = false

puts is_ruby_fun  # Output: true
puts is_sky_green # Output: false

Expresiones Booleanas

Las expresiones booleanas se evalúan como true o false. A menudo involucran operadores de comparación.

Operadores de Comparación

  • == : Igual a
  • != : No igual a
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual a
  • <= : Menor o igual a
a = 10
b = 20

puts a == b  # Output: false
puts a != b  # Output: true
puts a > b   # Output: false
puts a < b   # Output: true
puts a >= 10 # Output: true
puts b <= 15 # Output: false

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones booleanas.

  • && : AND lógico
  • || : OR lógico
  • ! : NOT lógico
x = true
y = false

puts x && y  # Output: false
puts x || y  # Output: true
puts !x      # Output: false
puts !y      # Output: true

Declaraciones Condicionales

Las declaraciones condicionales usan expresiones booleanas para controlar el flujo del programa.

Declaración if

age = 18

if age >= 18
  puts "Eres un adulto."
else
  puts "Eres un menor."
end

Declaración unless

La declaración unless es lo opuesto a if.

age = 15

unless age >= 18
  puts "Eres un menor."
else
  puts "Eres un adulto."
end

Operador Ternario

El operador ternario es una forma abreviada de if-else.

age = 20
message = age >= 18 ? "Eres un adulto." : "Eres un menor."
puts message  # Output: Eres un adulto.

Métodos Booleanos

Ruby proporciona varios métodos que devuelven valores booleanos.

.nil?

Verifica si un objeto es nil.

name = nil
puts name.nil?  # Output: true

name = "Alice"
puts name.nil?  # Output: false

.empty?

Verifica si una cadena o un array está vacío.

str = ""
puts str.empty?  # Output: true

arr = []
puts arr.empty?  # Output: true

.include?

Verifica si una cadena o un array incluye un elemento específico.

str = "¡Hola, mundo!"
puts str.include?("mundo")  # Output: true

arr = [1, 2, 3, 4, 5]
puts arr.include?(3)  # Output: true

Resumen

Los booleanos son una parte crucial de la programación en Ruby, permitiendo la toma de decisiones y el control del flujo. Entender los literales booleanos, las expresiones y los operadores lógicos es esencial para escribir código Ruby efectivo. Las declaraciones condicionales como if, unless y el operador ternario te permiten ejecutar código basado en condiciones booleanas. Además, Ruby proporciona varios métodos que devuelven valores booleanos, ayudándote a escribir código más expresivo y legible.
