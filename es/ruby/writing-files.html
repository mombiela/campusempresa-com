En esta sección, cubriremos cómo escribir archivos en Ruby, comenzando desde lo básico y avanzando hacia técnicas más avanzadas. Escribir archivos es una habilidad esencial para cualquier programador, ya que te permite almacenar datos de manera persistente.

Escritura Básica de Archivos

Abrir un Archivo para Escribir

Para escribir en un archivo en Ruby, primero necesitas abrirlo. Puedes hacer esto usando el método File.open.

File.open('example.txt', 'w') do |file|
  file.puts 'Hello, world!'
end
  • 'example.txt': El nombre del archivo al que quieres escribir.
  • 'w': El modo en el que quieres abrir el archivo. 'w' significa modo de escritura. Si el archivo no existe, se creará. Si existe, se truncará (es decir, todo el contenido existente será eliminado).

Escribir Datos en un Archivo

Puedes escribir datos en un archivo usando métodos como puts, print y write.

File.open('example.txt', 'w') do |file|
  file.puts 'Hello, world!'  # Añade una nueva línea al final
  file.print 'This is a test.'  # No añade una nueva línea
  file.write 'Writing to a file in Ruby.'  # No añade una nueva línea
end

Modos de Archivo

Ruby soporta varios modos de archivo, cada uno con un propósito diferente. Aquí hay algunos comunes:

| Modo | Descripción | |------|-------------| | 'r' | Modo solo lectura. El archivo debe existir. | | 'w' | Modo solo escritura. Crea un nuevo archivo o trunca uno existente. | | 'a' | Modo de anexado. Escribe datos al final del archivo. | | 'r+' | Modo de lectura y escritura. El archivo debe existir. | | 'w+' | Modo de lectura y escritura. Crea un nuevo archivo o trunca uno existente. | | 'a+' | Modo de lectura y escritura. Escribe datos al final del archivo. |

Escritura Avanzada de Archivos

Escribir Datos Binarios

A veces puede que necesites escribir datos binarios en un archivo. Puedes hacerlo abriendo el archivo en modo binario.

File.open('binary_file.bin', 'wb') do |file|
  file.write [255, 0, 127].pack('C*')
end
  • 'wb': Modo de escritura binaria.
  • [255, 0, 127].pack('C*'): Convierte un array de enteros en una cadena binaria.

Manejo de Excepciones

Cuando trabajas con archivos, es importante manejar excepciones para evitar posibles errores.

begin
  File.open('example.txt', 'w') do |file|
    file.puts 'Hello, world!'
  end
rescue Errno::ENOENT => e
  puts "Archivo no encontrado: #{e.message}"
rescue IOError => e
  puts "Error de E/S: #{e.message}"
ensure
  file.close unless file.nil?
end
  • begin...rescue...ensure: Un bloque para manejar excepciones.
  • Errno::ENOENT: Se lanza cuando el archivo no se encuentra.
  • IOError: Se lanza para errores generales de E/S.
  • ensure: Asegura que el archivo se cierre incluso si ocurre un error.

Resumen

Escribir archivos en Ruby es sencillo pero poderoso. Al entender los diferentes modos de archivo y métodos disponibles, puedes escribir datos en archivos de manera eficiente. Recuerda manejar excepciones para hacer tu código robusto y libre de errores.

  • Usa File.open con el modo apropiado para escribir en archivos.
  • Elige el método correcto (puts, print, write) según tus necesidades.
  • Entiende y usa diferentes modos de archivo ('w', 'a', 'r+', etc.).
  • Maneja excepciones para asegurar que tu programa pueda lidiar con situaciones inesperadas de manera elegante.

Al dominar estos conceptos, estarás bien equipado para manejar tareas de escritura de archivos en Ruby, desde archivos de texto simples hasta datos binarios complejos.
