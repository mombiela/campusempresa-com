Introducción a las declaraciones condicionales

Las declaraciones condicionales son fundamentales en la programación, ya que te permiten ejecutar ciertos bloques de código basados en condiciones específicas. En Ruby, las declaraciones condicionales ayudan a controlar el flujo de tu programa tomando decisiones.

Declaraciones condicionales básicas

Declaración `if`

La declaración if es la forma más básica de declaración condicional en Ruby. Ejecuta un bloque de código si una condición especificada es verdadera.

if condition
  # código a ejecutar si la condición es verdadera
end

Ejemplo:

age = 18

if age >= 18
  puts "Eres un adulto."
end

Declaración `else`

La declaración else se puede usar junto con if para ejecutar un bloque de código si la condición es falsa.

if condition
  # código a ejecutar si la condición es verdadera
else
  # código a ejecutar si la condición es falsa
end

Ejemplo:

age = 16

if age >= 18
  puts "Eres un adulto."
else
  puts "Eres un menor."
end

Declaración `elsif`

La declaración elsif te permite verificar múltiples condiciones.

if condition1
  # código a ejecutar si la condición1 es verdadera
elsif condition2
  # código a ejecutar si la condición2 es verdadera
else
  # código a ejecutar si ninguna de las condiciones anteriores es verdadera
end

Ejemplo:

age = 20

if age < 13
  puts "Eres un niño."
elsif age < 18
  puts "Eres un adolescente."
else
  puts "Eres un adulto."
end

Declaraciones condicionales avanzadas

Declaración `unless`

La declaración unless es lo opuesto a if. Ejecuta un bloque de código si la condición es falsa.

unless condition
  # código a ejecutar si la condición es falsa
end

Ejemplo:

age = 15

unless age >= 18
  puts "No eres un adulto."
end

Operador ternario

El operador ternario es una forma abreviada de las declaraciones if-else. Es útil para condiciones simples.

condition ? true_expression : false_expression

Ejemplo:

age = 20
message = age >= 18 ? "Eres un adulto." : "Eres un menor."
puts message

Declaración `case`

La declaración case se usa para comparar una variable contra múltiples valores. Es similar a la declaración switch en otros lenguajes.

case variable
when value1
  # código a ejecutar si variable == value1
when value2
  # código a ejecutar si variable == value2
else
  # código a ejecutar si la variable no coincide con ningún valor
end

Ejemplo:

day = "Saturday"

case day
when "Monday"
  puts "Inicio de la semana laboral."
when "Friday"
  puts "Fin de la semana laboral."
when "Saturday", "Sunday"
  puts "¡Fin de semana!"
else
  puts "Días entre semana."
end

Combinando condiciones

Operadores lógicos

Los operadores lógicos se pueden usar para combinar múltiples condiciones en una sola declaración if, elsif o unless.

  • && (y): Ambas condiciones deben ser verdaderas.
  • || (o): Al menos una condición debe ser verdadera.
  • ! (no): Invierte la condición.

Ejemplo:

age = 25
has_id = true

if age >= 18 && has_id
  puts "Puedes entrar al club."
else
  puts "No puedes entrar al club."
end

Ejercicios

  1. Escribe un programa en Ruby que verifique si un número es positivo, negativo o cero.
  2. Crea un script en Ruby que categorice el grupo de edad de una persona (niño, adolescente, adulto, anciano) basado en su edad.
  3. Implementa un programa en Ruby que use una declaración case para imprimir el día de la semana basado en un número de entrada (1 para lunes, 2 para martes, etc.).

Resumen

Las declaraciones condicionales en Ruby son esenciales para controlar el flujo de tu programa. Comenzando con las declaraciones básicas if, else y elsif, puedes manejar condiciones simples. Para escenarios más complejos, Ruby ofrece unless, operadores ternarios y declaraciones case. Combinar condiciones con operadores lógicos permite un control aún más preciso. Practica estos conceptos para volverte competente en la gestión de la lógica del programa en Ruby.
