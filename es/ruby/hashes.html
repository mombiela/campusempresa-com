Introducción a los Hashes

Los hashes, también conocidos como diccionarios, mapas o arreglos asociativos, son una estructura de datos fundamental en Ruby. Almacenan pares clave-valor, lo que te permite recuperar valores basados en sus claves correspondientes.

Conceptos Clave

  • Pares Clave-Valor: Cada elemento en un hash es un par que consiste en una clave única y un valor.
  • Colección Desordenada: A diferencia de los arreglos, los hashes no mantienen ningún orden específico para sus elementos.
  • Claves y Valores Flexibles: Las claves y los valores pueden ser de cualquier tipo de dato.

Creando Hashes

Puedes crear hashes usando diferentes sintaxis en Ruby.

Usando Notación Literal

# Creando un hash vacío
empty_hash = {}

# Creando un hash con pares clave-valor iniciales
person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

Usando la Clase `Hash`

# Creando un hash vacío
empty_hash = Hash.new

# Creando un hash con pares clave-valor iniciales
person = Hash.new
person["name"] = "Alice"
person["age"] = 30
person["city"] = "New York"

Accediendo y Modificando Hashes

Accediendo a Valores

Puedes acceder a los valores en un hash usando sus claves.

person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

# Accediendo a valores
puts person["name"]  # Salida: Alice
puts person["age"]   # Salida: 30

Modificando Valores

Puedes modificar valores existentes o agregar nuevos pares clave-valor.

person["age"] = 31  # Modificando un valor existente
person["country"] = "USA"  # Agregando un nuevo par clave-valor

puts person
# Salida: {"name"=>"Alice", "age"=>31, "city"=>"New York", "country"=>"USA"}

Métodos Comunes de Hash

Ruby proporciona varios métodos útiles para trabajar con hashes.

`keys` y `values`

Recupera todas las claves o valores de un hash.

person = {
  "name" => "Alice",
  "age" => 30,
  "city" => "New York"
}

puts person.keys   # Salida: ["name", "age", "city"]
puts person.values # Salida: ["Alice", 30, "New York"]

`each`

Itera sobre cada par clave-valor en un hash.

person.each do |key, value|
  puts "#{key}: #{value}"
end
# Salida:
# name: Alice
# age: 30
# city: New York

`delete`

Elimina un par clave-valor de un hash.

person.delete("age")
puts person
# Salida: {"name"=>"Alice", "city"=>"New York"}

`merge`

Combina dos hashes en uno.

person = {
  "name" => "Alice",
  "age" => 30
}

address = {
  "city" => "New York",
  "country" => "USA"
}

person.merge!(address)
puts person
# Salida: {"name"=>"Alice", "age"=>30, "city"=>"New York", "country"=>"USA"}

Técnicas Avanzadas de Hash

Valores Predeterminados

Puedes establecer un valor predeterminado para un hash, que se devolverá al acceder a una clave inexistente.

hash_with_default = Hash.new("Not Found")
puts hash_with_default["missing_key"]  # Salida: Not Found

Símbolos como Claves

Usar símbolos como claves puede ser más eficiente y legible.

person = {
  name: "Alice",
  age: 30,
  city: "New York"
}

puts person[:name]  # Salida: Alice

Hashes Anidados

Los hashes pueden contener otros hashes, permitiendo estructuras de datos más complejas.

person = {
  name: "Alice",
  age: 30,
  address: {
    city: "New York",
    country: "USA"
  }
}

puts person[:address][:city]  # Salida: New York

Conclusión

Los hashes son una estructura de datos versátil y poderosa en Ruby, adecuada para una amplia gama de aplicaciones. Entender cómo crear, acceder y manipular hashes mejorará significativamente tu capacidad para escribir código Ruby eficiente y efectivo. Practica usando los métodos y técnicas discutidos en esta sección para volverte competente en el trabajo con hashes.
