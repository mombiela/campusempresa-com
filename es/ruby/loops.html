Introducción a los Bucles

Los bucles son fundamentales en la programación ya que te permiten ejecutar un bloque de código múltiples veces. En Ruby, hay varios tipos de bucles, cada uno con sus propios casos de uso y sintaxis.

Tipos de Bucles

Bucle `while`

El bucle while continúa ejecutándose mientras su condición sea verdadera.

i = 0
while i < 5 do
  puts "i es #{i}"
  i += 1
end
  • Explicación:
    • i = 0: Inicializa la variable i.
    • while i < 5 do: El bucle se ejecutará mientras i sea menor que 5.
    • puts "i es #{i}": Imprime el valor actual de i.
    • i += 1: Incrementa i en 1.

Bucle `until`

El bucle until es lo opuesto al bucle while. Continúa ejecutándose hasta que su condición se vuelva verdadera.

i = 0
until i == 5 do
  puts "i es #{i}"
  i += 1
end
  • Explicación:
    • i = 0: Inicializa la variable i.
    • until i == 5 do: El bucle se ejecutará hasta que i sea igual a 5.
    • puts "i es #{i}": Imprime el valor actual de i.
    • i += 1: Incrementa i en 1.

Bucle `for`

El bucle for itera sobre un rango o un array.

for i in 0..4 do
  puts "i es #{i}"
end
  • Explicación:
    • for i in 0..4 do: El bucle iterará sobre el rango de 0 a 4.
    • puts "i es #{i}": Imprime el valor actual de i.

Bucle `times`

El bucle times ejecuta un bloque de código un número especificado de veces.

5.times do |i|
  puts "i es #{i}"
end
  • Explicación:
    • 5.times do |i|: El bucle se ejecutará 5 veces, con i tomando valores de 0 a 4.
    • puts "i es #{i}": Imprime el valor actual de i.

Bucle `each`

El bucle each itera sobre elementos de un array o un hash.

[1, 2, 3, 4, 5].each do |i|
  puts "i es #{i}"
end
  • Explicación:
    • [1, 2, 3, 4, 5].each do |i|: El bucle iterará sobre cada elemento del array.
    • puts "i es #{i}": Imprime el valor actual de i.

Control de Bucles: `break`, `next`, y `redo`

  • break: Sale del bucle inmediatamente.
  • next: Salta a la siguiente iteración.
  • redo: Repite la iteración actual.
for i in 0..5 do
  if i == 3
    next
  elsif i == 4
    break
  end
  puts "i es #{i}"
end
  • Explicación:
    • if i == 3: Cuando i es 3, next salta a la siguiente iteración.
    • elsif i == 4: Cuando i es 4, break sale del bucle.
    • puts "i es #{i}": Imprime el valor actual de i.

Técnicas Avanzadas de Bucles

Bucles Anidados

Los bucles pueden anidarse unos dentro de otros para manejar escenarios más complejos.

for i in 1..3 do
  for j in 1..3 do
    puts "i es #{i}, j es #{j}"
  end
end
  • Explicación:
    • El bucle externo itera sobre i de 1 a 3.
    • El bucle interno itera sobre j de 1 a 3.
    • puts "i es #{i}, j es #{j}": Imprime los valores actuales de i y j.

Iteración con Enumeradores

Los enumeradores proporcionan una forma de crear patrones de iteración personalizados.

enum = (1..5).each
loop do
  puts enum.next
end
  • Explicación:
    • enum = (1..5).each: Crea un enumerador para el rango de 1 a 5.
    • loop do: Inicia un bucle infinito.
    • puts enum.next: Imprime el siguiente elemento del enumerador.

Conclusión

Los bucles son una herramienta poderosa en Ruby, permitiéndote ejecutar código múltiples veces de manera eficiente. Entender los diferentes tipos de bucles y sus casos de uso te permitirá escribir código más efectivo y conciso. Practica usando estos bucles en varios escenarios para volverte competente en su aplicación.
