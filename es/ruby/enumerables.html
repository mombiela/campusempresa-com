Enumerables en Ruby
Clases y Objetos ►

Los Enumerables son una parte fundamental de Ruby, proporcionando un conjunto poderoso de métodos para recorrer, buscar, ordenar y manipular colecciones. Este tema cubrirá lo esencial de los Enumerables, desde conceptos básicos hasta usos avanzados.

Introducción a los Enumerables

  • Definición: Un Enumerable es un módulo que proporciona un conjunto de métodos para recorrer, buscar, ordenar y manipular colecciones.
  • Clases Comunes: Arrays, Hashes y Ranges son clases comunes que incluyen el módulo Enumerable.

Métodos Básicos

each

El método each es la forma más básica de iterar sobre una colección.

[1, 2, 3].each do |number|
  puts number
end
  • Explicación: Este código itera sobre el array [1, 2, 3] y imprime cada número.

map

El método map crea un nuevo array que contiene los valores devueltos por el bloque.

squares = [1, 2, 3].map do |number|
  number * 2
end
puts squares.inspect
  • Explicación: Este código multiplica cada número en el array por 2 y devuelve un nuevo array [2, 4, 6].

select

El método select devuelve un nuevo array que contiene todos los elementos para los cuales el bloque dado devuelve un valor verdadero.

evens = [1, 2, 3, 4].select do |number|
  number.even?
end
puts evens.inspect
  • Explicación: Este código selecciona números pares del array, resultando en [2, 4].

Métodos Intermedios

`reduce`

El método reduce combina todos los elementos de un enumerable aplicando una operación binaria, especificada por un bloque o un símbolo.

sum = [1, 2, 3, 4].reduce(0) do |accumulator, number|
  accumulator + number
end
puts sum
  • Explicación: Este código calcula la suma de los elementos del array, resultando en 10.

`find`

El método find devuelve el primer elemento para el cual el bloque devuelve un valor verdadero.

first_even = [1, 2, 3, 4].find do |number|
  number.even?
end
puts first_even
  • Explicación: Este código encuentra el primer número par en el array, que es 2.

`all?` y `any?`

  • all?: Devuelve verdadero si el bloque nunca devuelve falso o nil para todos los elementos.
  • any?: Devuelve verdadero si el bloque alguna vez devuelve un valor distinto de falso o nil para cualquier elemento.
all_even = [2, 4, 6].all? { |number| number.even? }
any_even = [1, 3, 4].any? { |number| number.even? }

puts all_even  # true
puts any_even  # true

Métodos Avanzados

`group_by`

El método group_by agrupa la colección por el resultado del bloque.

grouped = [1, 2, 3, 4, 5, 6].group_by do |number|
  number.even?
end
puts grouped.inspect
  • Explicación: Este código agrupa los números en pares e impares, resultando en {false=>[1, 3, 5], true=>[2, 4, 6]}.

`partition`

El método partition divide la colección en dos arrays, el primero contiene elementos para los cuales el bloque evalúa a verdadero, el segundo contiene el resto.

evens, odds = [1, 2, 3, 4, 5, 6].partition do |number|
  number.even?
end
puts evens.inspect  # [2, 4, 6]
puts odds.inspect   # [1, 3, 5]

`lazy`

El método lazy permite la enumeración perezosa, lo cual puede ser útil para manejar colecciones grandes o infinitas.

infinite = (1..Float::INFINITY).lazy
first_ten = infinite.first(10)
puts first_ten.inspect
  • Explicación: Este código crea un enumerador perezoso para un rango infinito y obtiene los primeros 10 elementos.

Conclusión

Los Enumerables en Ruby proporcionan un rico conjunto de métodos para trabajar con colecciones. Desde la iteración básica con each hasta la agrupación avanzada con group_by, entender estos métodos puede mejorar en gran medida tu capacidad para escribir código Ruby eficiente y legible. Practica usando estos métodos para volverte competente en la manipulación de colecciones en Ruby.
