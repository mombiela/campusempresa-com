Introducción

Ruby es un lenguaje de programación dinámico y de código abierto con un enfoque en la simplicidad y la productividad. Su sintaxis elegante es natural de leer y fácil de escribir. En esta sección, cubriremos la sintaxis básica de Ruby, que es esencial para escribir y entender el código Ruby.

Comentarios

  • Los comentarios de una sola línea comienzan con #.
  • Los comentarios de varias líneas están encerrados dentro de =begin y =end.
# Este es un comentario de una sola línea

=begin
Este es un
comentario de varias líneas
=end

Variables

  • Las variables en Ruby son de tipo dinámico.
  • Los nombres de las variables deben comenzar con una letra minúscula o un guion bajo.
name = "Alice"
age = 30
is_student = true

Tipos de Datos

Ruby admite varios tipos de datos básicos:

  • Números (Integer, Float)
  • Cadenas
  • Símbolos
  • Arrays
  • Hashes
  • Booleanos
# Integer
num = 42

# Float
pi = 3.14

# String
greeting = "¡Hola, Mundo!"

# Symbol
status = :active

# Array
numbers = [1, 2, 3, 4, 5]

# Hash
person = { name: "Alice", age: 30 }

# Boolean
is_ruby_fun = true

Operadores

Operadores Aritméticos

  • + Adición
  • - Sustracción
  • * Multiplicación
  • / División
  • % Módulo
sum = 5 + 3       # 8
difference = 5 - 3 # 2
product = 5 * 3    # 15
quotient = 5 / 3   # 1
remainder = 5 % 3  # 2

Operadores de Comparación

  • == Igual a
  • != No igual a
  • > Mayor que
  • < Menor que
  • >= Mayor o igual que
  • <= Menor o igual que
a = 5
b = 3

puts a == b  # false
puts a != b  # true
puts a > b   # true
puts a < b   # false
puts a >= b  # true
puts a <= b  # false

Operadores Lógicos

  • && AND lógico
  • || OR lógico
  • ! NOT lógico
x = true
y = false

puts x && y  # false
puts x || y  # true
puts !x      # false

Estructuras de Control

Sentencias Condicionales

  • if, elsif, else
age = 18

if age < 18
  puts "Eres menor de edad."
elsif age == 18
  puts "Acabas de convertirte en adulto."
else
  puts "Eres adulto."
end

Bucles

  • Bucle while
  • Bucle for
  • Método each para iterar sobre colecciones
# bucle while
i = 0
while i < 5
  puts i
  i += 1
end

# bucle for
for i in 0..4
  puts i
end

# método each
[0, 1, 2, 3, 4].each do |i|
  puts i
end

Métodos

  • Define métodos usando la palabra clave def.
  • Los métodos pueden tomar argumentos y devolver valores.
def greet(name)
  return "¡Hola, #{name}!"
end

puts greet("Alice")  # Salida: ¡Hola, Alice!

Conclusión

En esta sección, cubrimos la sintaxis básica de Ruby, incluyendo comentarios, variables, tipos de datos, operadores, estructuras de control y métodos. Entender estos fundamentos es crucial para escribir y comprender el código Ruby. A medida que avances en el curso, estos conceptos básicos servirán como la base para temas más avanzados.
