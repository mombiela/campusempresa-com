Introducción a Ruby

Ruby es un lenguaje de programación dinámico y de código abierto con un enfoque en la simplicidad y la productividad. Tiene una sintaxis elegante que es natural de leer y fácil de escribir. Ruby fue creado a mediados de la década de 1990 por Yukihiro "Matz" Matsumoto en Japón.

Características Clave de Ruby

  • Orientado a Objetos: Todo en Ruby es un objeto, incluidos los tipos de datos primitivos.
  • Tipado Dinámico: Las variables en Ruby no están vinculadas a un tipo de dato específico.
  • Recolección de Basura: Ruby gestiona automáticamente la asignación y liberación de memoria.
  • Biblioteca Estándar Rica: Ruby viene con una biblioteca estándar completa que facilita la realización de una amplia gama de tareas.
  • Comunidad y Ecosistema: Ruby tiene una comunidad vibrante y un rico ecosistema de bibliotecas y frameworks, como Ruby on Rails.

Sintaxis Básica

Veamos algo de sintaxis básica de Ruby para familiarizarnos con el lenguaje.

Ejemplo de Hola Mundo

puts '¡Hola, Mundo!'
  • puts es un método que muestra texto en la consola.
  • '¡Hola, Mundo!' es un literal de cadena.

Variables y Tipos de Datos

name = "Alicia"  # Cadena
age = 30         # Entero
height = 5.5     # Flotante
is_student = true  # Booleano

puts name       # Muestra: Alicia
puts age        # Muestra: 30
puts height     # Muestra: 5.5
puts is_student # Muestra: true
  • Las variables son de tipo dinámico.
  • Los tipos de datos comunes incluyen Cadena, Entero, Flotante y Booleano.

Estructuras de Control

Declaración If-Else

age = 18

if age >= 18
  puts "Eres un adulto."
else
  puts "Eres un menor."
end
  • if y else se usan para la ejecución condicional.
  • La condición se evalúa y se ejecuta el bloque de código correspondiente.

Bucles

Bucle While
count = 0

while count < 5
  puts "El conteo es #{count}"
  count += 1
end
  • El bucle while se ejecuta mientras la condición sea verdadera.
  • #{} se usa para la interpolación de cadenas.
Bucle Each
numbers = [1, 2, 3, 4, 5]

numbers.each do |number|
  puts number
end
  • El método each itera sobre los elementos en una colección, como un array.
  • |number| es una variable de bloque que toma el valor de cada elemento en el array.

Programación Orientada a Objetos en Ruby

Ruby es un lenguaje puramente orientado a objetos, lo que significa que todo es un objeto. Vamos a crear una clase simple para ilustrar esto.

Definiendo una Clase

class Person
  attr_accessor :name, :age

  def initialize(name, age)
    @name = name
    @age = age
  end

  def introduce
    puts "Hola, mi nombre es #{@name} y tengo #{@age} años."
  end
end

person = Person.new("Alicia", 30)
person.introduce
  • La palabra clave class se usa para definir una clase.
  • attr_accessor crea métodos getter y setter para las variables de instancia.
  • El método initialize es el constructor.
  • @name y @age son variables de instancia.
  • introduce es un método de instancia.

Resumen

Ruby es un lenguaje de programación versátil y amigable para principiantes, conocido por su simplicidad y productividad. Soporta la programación orientada a objetos y tiene una rica biblioteca estándar. Al entender su sintaxis básica, estructuras de control y características orientadas a objetos, puedes comenzar a construir aplicaciones poderosas. A medida que progreses, descubrirás características más avanzadas y bibliotecas que hacen de Ruby una herramienta poderosa para los desarrolladores.
