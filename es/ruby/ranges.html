Introducción a los Rangos

Los rangos en Ruby representan una secuencia de valores, que pueden ser números, letras u objetos. Se utilizan comúnmente para la iteración y la definición de intervalos.

Conceptos Clave

  • Operadores de Rango: .. (inclusivo) y ... (exclusivo).
  • Métodos de Rango: Métodos para manipular y consultar rangos.
  • Casos de Uso: Iteración, verificación de condiciones y segmentación.

Creación de Rangos

Los rangos pueden ser creados usando los operadores .. y ....

Rangos Inclusivos

Un rango inclusivo incluye el valor final.

inclusive_range = 1..5
puts inclusive_range.to_a  # Salida: [1, 2, 3, 4, 5]

Rangos Exclusivos

Un rango exclusivo excluye el valor final.

exclusive_range = 1...5
puts exclusive_range.to_a  # Salida: [1, 2, 3, 4]

Métodos de Rango

Ruby proporciona varios métodos para trabajar con rangos.

Método `to_a`

Convierte un rango en un array.

range = 1..3
array = range.to_a
puts array.inspect  # Salida: [1, 2, 3]

Método `include?`

Verifica si un valor está dentro del rango.

range = 1..5
puts range.include?(3)  # Salida: true
puts range.include?(6)  # Salida: false

Método `each`

Itera sobre cada elemento en el rango.

(1..3).each do |number|
  puts number
end
# Salida:
# 1
# 2
# 3

Características Avanzadas de los Rangos

Los rangos en Ruby pueden ser utilizados de maneras más avanzadas, como con objetos personalizados y en declaraciones condicionales.

Objetos Personalizados en Rangos

Puedes definir rangos con objetos personalizados si implementan el operador <=> (nave espacial).

class Custom
  attr_reader :value

  def initialize(value)
    @value = value
  end

  def <=>(other)
    @value <=> other.value
  end
end

range = Custom.new(1)..Custom.new(3)
puts range.include?(Custom.new(2))  # Salida: true

Rangos en Declaraciones Condicionales

Los rangos pueden ser utilizados en declaraciones condicionales para verificar si un valor cae dentro de un rango específico.

age = 25

case age
when 0..12
  puts "Child"
when 13..19
  puts "Teenager"
when 20..64
  puts "Adult"
else
  puts "Senior"
end
# Salida: Adult

Resumen

Los rangos en Ruby son una herramienta poderosa para representar secuencias e intervalos. Pueden ser creados usando operadores inclusivos (..) y exclusivos (...), y ofrecen varios métodos para manipulación e iteración. Entender los rangos es esencial para una programación efectiva en Ruby, especialmente para tareas que involucran iteración, verificación de condiciones y manejo de objetos personalizados.
