Introducción

En Ruby, los métodos pueden tomar argumentos para proporcionar datos de entrada para sus operaciones. Entender cómo usar los argumentos de método de manera efectiva es crucial para escribir código flexible y reutilizable. Esta sección cubrirá los conceptos básicos de los argumentos de método, incluyendo los diferentes tipos de argumentos y cómo usarlos.

Argumentos Básicos de Método

Definiendo Métodos con Argumentos

Para definir un método que tome argumentos, especificas los nombres de los argumentos entre paréntesis después del nombre del método.

def greet(name)
  puts "Hello, #{name}!"
end

greet("Alice")  # Output: Hello, Alice!
  • Concepto Clave: El name dentro de los paréntesis es un parámetro, y "Alice" es un argumento pasado al método.

Múltiples Argumentos

Puedes definir métodos que tomen múltiples argumentos separándolos con comas.

def add(a, b)
  a + b
end

puts add(3, 4)  # Output: 7
  • Concepto Clave: a y b son parámetros, y 3 y 4 son argumentos pasados al método.

Argumentos por Defecto

Ruby te permite establecer valores por defecto para los argumentos de método. Si no se proporciona un argumento, se usa el valor por defecto.

def greet(name = "World")
  puts "Hello, #{name}!"
end

greet()         # Output: Hello, World!
greet("Alice")  # Output: Hello, Alice!
  • Concepto Clave: name = "World" establece un valor por defecto para el parámetro name.

Argumentos de Palabra Clave

Los argumentos de palabra clave son una forma de definir métodos con parámetros nombrados, haciendo el código más legible y flexible.

def print_address(street:, city:, zip:)
  puts "#{street}, #{city}, #{zip}"
end

print_address(street: "123 Main St", city: "Springfield", zip: "12345")
  • Concepto Clave: street:, city:, y zip: son argumentos de palabra clave, y deben ser proporcionados al llamar al método.

Argumentos de Palabra Clave por Defecto

También puedes establecer valores por defecto para los argumentos de palabra clave.

def print_address(street:, city:, zip: "00000")
  puts "#{street}, #{city}, #{zip}"
end

print_address(street: "123 Main St", city: "Springfield")  # Output: 123 Main St, Springfield, 00000
  • Concepto Clave: zip: "00000" establece un valor por defecto para el argumento de palabra clave zip.

Argumentos de Longitud Variable

Ruby permite que los métodos acepten un número variable de argumentos usando el operador splat (*).

def sum(*numbers)
  numbers.reduce(0) { |total, num| total + num }
end

puts sum(1, 2, 3)  # Output: 6
puts sum(4, 5)     # Output: 9
  • Concepto Clave: *numbers recoge todos los argumentos en un array llamado numbers.

Argumentos de Bloque

Los métodos en Ruby también pueden tomar bloques como argumentos, que son fragmentos de código que pueden ser ejecutados dentro del método.

def repeat(n)
  n.times { yield }
end

repeat(3) { puts "Hello!" }
# Output:
# Hello!
# Hello!
# Hello!
  • Concepto Clave: yield se usa para llamar al bloque pasado al método.

Conclusión

Entender los argumentos de método en Ruby es esencial para escribir código claro y flexible. Esta sección cubrió:

  • Argumentos básicos de método y múltiples argumentos.
  • Argumentos por defecto y argumentos de palabra clave.
  • Argumentos de longitud variable usando el operador splat.
  • Argumentos de bloque para pasar fragmentos de código.

Al dominar estos conceptos, puedes crear métodos más dinámicos y reutilizables en tus programas Ruby. Practica usando estos diferentes tipos de argumentos para volverte competente en escribir métodos versátiles en Ruby.
