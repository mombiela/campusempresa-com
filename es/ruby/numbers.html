Introducción

Los números son una parte fundamental de cualquier lenguaje de programación, y Ruby no es una excepción. En esta sección, cubriremos los conceptos básicos de los números en Ruby, incluyendo diferentes tipos de números, operaciones aritméticas básicas y algunos temas avanzados como trabajar con números grandes y números complejos.

Tipos de Números

Ruby soporta varios tipos de números, cada uno con sus propias características y casos de uso.

  • Integer: Números enteros sin punto decimal.
  • Float: Números con punto decimal.
  • Rational: Números representados como fracciones.
  • Complex: Números con una parte real y una parte imaginaria.

Integer

Los enteros son números enteros, tanto positivos como negativos, incluyendo el cero.

a = 42
b = -7
c = 0

Float

Los floats son números que contienen un punto decimal.

a = 3.14
b = -0.001
c = 2.0

Rational

Los números racionales se representan como fracciones.

a = Rational(2, 3)  # 2/3
b = Rational(7, 4)  # 7/4

Complex

Los números complejos tienen una parte real y una parte imaginaria.

a = Complex(2, 3)  # 2 + 3i
b = Complex(1, -4) # 1 - 4i

Operaciones Aritméticas Básicas

Ruby soporta operaciones aritméticas básicas como suma, resta, multiplicación y división.

# Suma
sum = 5 + 3  # 8

# Resta
difference = 10 - 4  # 6

# Multiplicación
product = 7 * 6  # 42

# División
quotient = 20 / 4  # 5

# Módulo (resto)
remainder = 10 % 3  # 1

# Exponenciación
power = 2 ** 3  # 8

Temas Avanzados

Trabajando con Números Grandes

Ruby puede manejar números muy grandes con facilidad. No necesitas preocuparte por el desbordamiento de enteros.

large_number = 123456789012345678901234567890
puts large_number * large_number

Convertir Entre Tipos de Números

Puedes convertir entre diferentes tipos de números usando los métodos integrados de Ruby.

# Entero a Float
int_num = 10
float_num = int_num.to_f  # 10.0

# Float a Entero
float_num = 3.14
int_num = float_num.to_i  # 3

# Entero a Rational
int_num = 5
rational_num = int_num.to_r  # 5/1

# Float a Rational
float_num = 2.5
rational_num = float_num.to_r  # 5/2

# Entero a Complex
int_num = 4
complex_num = int_num.to_c  # 4+0i

Operaciones con Números Complejos

Ruby proporciona métodos para realizar operaciones aritméticas en números complejos.

a = Complex(2, 3)
b = Complex(1, -4)

# Suma
sum = a + b  # 3 - 1i

# Resta
difference = a - b  # 1 + 7i

# Multiplicación
product = a * b  # 14 - 5i

# División
quotient = a / b  # -0.6470588235294118 + 0.7058823529411765i

Resumen

En esta sección, cubrimos los conceptos básicos de los números en Ruby, incluyendo diferentes tipos de números (Integer, Float, Rational, Complex), operaciones aritméticas básicas y algunos temas avanzados como trabajar con números grandes y convertir entre tipos de números. Entender estos conceptos es crucial para realizar operaciones matemáticas y manejar datos numéricos en tus programas Ruby.
