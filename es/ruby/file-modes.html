Introducción

El manejo de archivos es un aspecto crucial de la programación, permitiéndote leer y escribir en archivos. En Ruby, los modos de archivo determinan cómo se abre un archivo y qué operaciones se pueden realizar en él. Esta sección cubrirá los diversos modos de archivo disponibles en Ruby, proporcionando explicaciones claras, ejemplos y ejercicios para ayudarte a entender y usarlos eficazmente.

Conceptos Clave

  • Modos de Archivo: Diferentes formas de abrir un archivo, determinando las operaciones que se pueden realizar.
  • Modos de Archivo Comunes: Modos como leer, escribir, añadir y sus combinaciones.
  • Métodos de la Clase File: Métodos proporcionados por la clase File de Ruby para manejar archivos.

Modos de Archivo Comunes

Ruby proporciona varios modos de archivo que puedes usar para abrir archivos. Aquí están los más comúnmente utilizados:

| Modo | Descripción | |------|-------------| | r | Modo de solo lectura. El archivo debe existir. | | w | Modo de solo escritura. Crea un nuevo archivo o trunca un archivo existente a longitud cero. | | a | Modo de añadir. Solo escritura. Crea un nuevo archivo si no existe. | | r+ | Modo de lectura y escritura. El archivo debe existir. | | w+ | Modo de lectura y escritura. Crea un nuevo archivo o trunca un archivo existente a longitud cero. | | a+ | Modo de lectura y escritura. Crea un nuevo archivo si no existe. Los datos se añaden al final del archivo. |

Ejemplos

Modo de Solo Lectura (`r`)

# Abrir un archivo en modo de solo lectura
file = File.open("example.txt", "r")

# Leer el contenido del archivo
content = file.read
puts content

# Cerrar el archivo
file.close

Explicación:

  • El archivo se abre en modo de solo lectura usando "r".
  • El método read lee todo el contenido del archivo.
  • El método close cierra el archivo.

Modo de Solo Escritura (`w`)

# Abrir un archivo en modo de solo escritura
file = File.open("example.txt", "w")

# Escribir contenido en el archivo
file.write("¡Hola, Mundo!")

# Cerrar el archivo
file.close

Explicación:

  • El archivo se abre en modo de solo escritura usando "w".
  • El método write escribe el contenido especificado en el archivo.
  • El método close cierra el archivo.

Modo de Añadir (`a`)

# Abrir un archivo en modo de añadir
file = File.open("example.txt", "a")

# Añadir contenido al archivo
file.write("\nTexto añadido.")

# Cerrar el archivo
file.close

Explicación:

  • El archivo se abre en modo de añadir usando "a".
  • El método write añade el contenido especificado al final del archivo.
  • El método close cierra el archivo.

Modo de Lectura y Escritura (`r+`)

# Abrir un archivo en modo de lectura y escritura
file = File.open("example.txt", "r+")

# Leer el contenido actual
content = file.read
puts "Contenido actual: #{content}"

# Escribir nuevo contenido
file.write("\nNuevo contenido añadido.")

# Cerrar el archivo
file.close

Explicación:

  • El archivo se abre en modo de lectura y escritura usando "r+".
  • El método read lee el contenido actual del archivo.
  • El método write escribe nuevo contenido en el archivo.
  • El método close cierra el archivo.

Ejercicios

  1. Leyendo un Archivo: Crea un archivo llamado data.txt con algo de texto en él. Escribe un script en Ruby para leer e imprimir el contenido del archivo.
  2. Escribiendo en un Archivo: Escribe un script en Ruby para crear un nuevo archivo llamado output.txt y escribir el texto "Manejo de Archivos en Ruby" en él.
  3. Añadiendo a un Archivo: Modifica el script del Ejercicio 2 para añadir el texto "Añadiendo más texto." a output.txt.
  4. Operaciones de Lectura y Escritura: Escribe un script en Ruby para abrir un archivo existente en modo de lectura y escritura, leer su contenido, y luego añadir el texto "Fin del archivo." a él.

Conclusión

Entender los modos de archivo en Ruby es esencial para un manejo efectivo de archivos. Al dominar los diferentes modos, puedes leer, escribir y añadir a archivos según sea necesario. Practica los ejercicios proporcionados para solidificar tu comprensión y volverte competente en el manejo de archivos en Ruby.
