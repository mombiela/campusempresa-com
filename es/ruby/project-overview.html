Introducción

En este curso, te guiaremos a través del viaje de aprender Ruby, desde niveles principiantes hasta avanzados. Ruby es un lenguaje de programación dinámico y de código abierto con un enfoque en la simplicidad y la productividad. Tiene una sintaxis elegante que es natural de leer y fácil de escribir. Al final de este curso, serás competente en Ruby y capaz de construir tus propios proyectos.

Estructura del Curso

El curso está dividido en tres secciones principales:

  1. Nivel Principiante
  2. Nivel Intermedio
  3. Nivel Avanzado

Cada sección se construye sobre la anterior, asegurando una transición suave de conceptos básicos a complejos.

Nivel Principiante

Conceptos Clave

  • Introducción a Ruby: Entender qué es Ruby y su historia.
  • Configuración del Entorno: Instalar Ruby y configurar un entorno de desarrollo.
  • Sintaxis Básica: Aprender la sintaxis básica y la estructura del código Ruby.
  • Variables y Tipos de Datos: Entender los diferentes tipos de datos y cómo usar variables.
  • Estructuras de Control: Usar declaraciones if-else, bucles e iteradores.

Código de Ejemplo

# Ejemplo de Sintaxis Básica de Ruby
puts "¡Hola, Mundo!"

# Variables y Tipos de Datos
nombre = "Alicia"
edad = 30
es_estudiante = true

# Estructuras de Control
if edad > 18
  puts "#{nombre} es un adulto."
else
  puts "#{nombre} no es un adulto."
end

# Ejemplo de Bucle
3.times do |i|
  puts "Este es el bucle número #{i + 1}"
end

Nivel Intermedio

Conceptos Clave

  • Métodos: Definir y usar métodos.
  • Clases y Objetos: Entender la programación orientada a objetos en Ruby.
  • Módulos y Mixins: Usar módulos para organizar el código y mixins para compartir funcionalidad.
  • Entrada/Salida de Archivos: Leer y escribir en archivos.
  • Manejo de Errores: Usar excepciones para manejar errores de manera elegante.

Código de Ejemplo

# Métodos
def saludar(nombre)
  "¡Hola, #{nombre}!"
end

puts saludar("Bob")

# Clases y Objetos
class Persona
  attr_accessor :nombre, :edad

  def initialize(nombre, edad)
    @nombre = nombre
    @edad = edad
  end

  def presentarse
    "Hola, soy #{@nombre} y tengo #{@edad} años."
  end
end

persona = Persona.new("Alicia", 30)
puts persona.presentarse

# Módulos y Mixins
module Saludable
  def saludar
    "¡Hola!"
  end
end

class Robot
  include Saludable
end

robot = Robot.new
puts robot.saludar

# Entrada/Salida de Archivos
File.open("ejemplo.txt", "w") do |file|
  file.puts "Este es un archivo de ejemplo."
end

# Manejo de Errores
begin
  resultado = 10 / 0
rescue ZeroDivisionError
  puts "¡No puedes dividir por cero!"
end

Nivel Avanzado

Conceptos Clave

  • Metaprogramación: Escribir código que escribe código.
  • Concurrencia y Paralelismo: Usar hilos y procesos.
  • Programación de Redes: Construir aplicaciones en red.
  • Pruebas: Escribir pruebas para asegurar la calidad del código.
  • Optimización del Rendimiento: Técnicas para hacer que el código Ruby se ejecute más rápido.

Código de Ejemplo

# Metaprogramación
class MetodoDinamico
  define_method(:saludar) do |nombre|
    "¡Hola, #{nombre}!"
  end
end

dinamico = MetodoDinamico.new
puts dinamico.saludar("Charlie")

# Concurrencia y Paralelismo
hilos = []
5.times do |i|
  hilos << Thread.new do
    puts "El hilo #{i} está corriendo"
  end
end
hilos.each(&:join)

# Programación de Redes
require 'socket'

servidor = TCPServer.new(2345)
loop do
  cliente = servidor.accept
  cliente.puts "¡Hola desde el servidor!"
  cliente.close
end

# Pruebas
require 'minitest/autorun'

class PruebaEjemplo < Minitest::Test
  def test_suma
    assert_equal 4, 2 + 2
  end
end

# Optimización del Rendimiento
require 'benchmark'

array = (1..1000000).to_a
Benchmark.bm do |x|
  x.report { array.map { |i| i * 2 } }
  x.report { array.each { |i| i * 2 } }
end

Conclusión

Al progresar a través de los niveles principiante, intermedio y avanzado, obtendrás una comprensión integral de la programación en Ruby. Cada sección incluye ejemplos prácticos y ejercicios para reforzar tu aprendizaje. ¡Feliz codificación!
