Introducción a Clases y Objetos

En Ruby, todo es un objeto. Los objetos son instancias de clases, que definen el plano para los objetos. Entender las clases y los objetos es fundamental para dominar Ruby.

Conceptos Clave

  • Clase: Un plano para crear objetos.
  • Objeto: Una instancia de una clase.
  • Variables de Instancia: Variables que pertenecen a una instancia de una clase.
  • Métodos: Funciones definidas dentro de una clase para realizar acciones en los objetos.

Definiendo una Clase

Para definir una clase en Ruby, usa la palabra clave class seguida del nombre de la clase. Los nombres de las clases se escriben en CamelCase.

class Person
  # El cuerpo de la clase va aquí
end

Ejemplo: Creando una Clase Simple

Vamos a crear una clase Person simple con variables de instancia y métodos.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_details
    puts "Nombre: #{@name}, Edad: #{@age}"
  end
end

# Creando un objeto de la clase Person
person1 = Person.new("Alice", 30)
person1.display_details  # Salida: Nombre: Alice, Edad: 30

Explicación

  • Método initialize: Un método especial llamado cuando se crea un nuevo objeto. Inicializa las variables de instancia del objeto.
  • @name y @age: Variables de instancia que almacenan el estado del objeto.
  • Método display_details: Un método para mostrar los detalles del objeto.

Variables y Métodos de Instancia

Las variables de instancia están precedidas por @ y se utilizan para almacenar el estado de un objeto. Los métodos se utilizan para definir el comportamiento de los objetos.

Ejemplo: Agregando Más Métodos

Vamos a agregar más métodos a nuestra clase Person.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_details
    puts "Nombre: #{@name}, Edad: #{@age}"
  end

  def birthday
    @age += 1
  end
end

person1 = Person.new("Alice", 30)
person1.display_details  # Salida: Nombre: Alice, Edad: 30
person1.birthday
person1.display_details  # Salida: Nombre: Alice, Edad: 31

Explicación

  • Método birthday: Aumenta la edad de la persona en 1.

Métodos y Variables de Clase

Los métodos y variables de clase están asociados con la clase en sí misma en lugar de con cualquier instancia de la clase.

Ejemplo: Métodos y Variables de Clase

Vamos a agregar una variable de clase y un método de clase a nuestra clase Person.

class Person
  @@count = 0  # Variable de clase

  def initialize(name, age)
    @name = name
    @age = age
    @@count += 1
  end

  def display_details
    puts "Nombre: #{@name}, Edad: #{@age}"
  end

  def self.total_count
    @@count
  end
end

person1 = Person.new("Alice", 30)
person2 = Person.new("Bob", 25)
puts Person.total_count  # Salida: 2

Explicación

  • @@count: Una variable de clase que lleva la cuenta del número de objetos Person creados.
  • self.total_count: Un método de clase que devuelve el valor de la variable de clase @@count.

Herencia

La herencia permite que una clase herede características de otra clase.

Ejemplo: Herencia

Vamos a crear una clase Student que hereda de la clase Person.

class Student < Person
  def initialize(name, age, grade)
    super(name, age)
    @grade = grade
  end

  def display_details
    super
    puts "Grado: #{@grade}"
  end
end

student1 = Student.new("Charlie", 20, "A")
student1.display_details  # Salida: Nombre: Charlie, Edad: 20, Grado: A

Explicación

  • < Person: Indica que Student hereda de Person.
  • super: Llama al método initialize de la clase padre para inicializar name y age.

Conclusión

Entender las clases y los objetos es crucial para la programación en Ruby. Las clases definen el plano para los objetos, y los objetos son instancias de clases. Las variables de instancia almacenan el estado de un objeto, mientras que los métodos definen el comportamiento. Los métodos y variables de clase están asociados con la clase en sí misma. La herencia permite que una clase herede características de otra clase, promoviendo la reutilización y organización del código.

Al dominar estos conceptos, puedes crear programas complejos y eficientes en Ruby.
