Introducción a las Variables

Las variables son fundamentales en cualquier lenguaje de programación, y Ruby no es una excepción. Te permiten almacenar y manipular datos a lo largo de tu programa. En esta sección, cubriremos los conceptos básicos de las variables en Ruby, incluyendo cómo crearlas, los diferentes tipos de variables y las mejores prácticas.

Creación de Variables

En Ruby, creas una variable simplemente asignando un valor a un nombre. Ruby es de tipado dinámico, lo que significa que no necesitas declarar el tipo de la variable.

name = "Alice"
age = 30
is_student = true
  • name es una variable que almacena una cadena.
  • age es una variable que almacena un entero.
  • is_student es una variable que almacena un booleano.

Tipos de Variables

Ruby tiene varios tipos de variables, cada una con su propio alcance y convenciones de nomenclatura.

Variables Locales

Las variables locales son el tipo más común de variable. Solo son accesibles dentro del contexto donde se definen, como dentro de un método o un bloque.

def greet
  greeting = "¡Hola, Mundo!"
  puts greeting
end

greet
# Salida: ¡Hola, Mundo!
  • Las variables locales comienzan con una letra minúscula o un guion bajo (_).

Variables de Instancia

Las variables de instancia se utilizan en la programación orientada a objetos. Son accesibles dentro de la instancia de una clase.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def display_info
    puts "Nombre: #{@name}, Edad: #{@age}"
  end
end

person = Person.new("Alice", 30)
person.display_info
# Salida: Nombre: Alice, Edad: 30
  • Las variables de instancia comienzan con un símbolo @.

Variables de Clase

Las variables de clase se comparten entre todas las instancias de una clase. Se utilizan cuando quieres almacenar información que es común a todos los objetos de una clase.

class Person
  @@species = "Homo sapiens"

  def self.species
    @@species
  end
end

puts Person.species
# Salida: Homo sapiens
  • Las variables de clase comienzan con @@.

Variables Globales

Las variables globales son accesibles desde cualquier parte del programa Ruby. Deben usarse con moderación ya que pueden dificultar la depuración.

$global_var = "Soy una variable global"

def display_global
  puts $global_var
end

display_global
# Salida: Soy una variable global
  • Las variables globales comienzan con un símbolo $.

Constantes

Las constantes son variables que, una vez asignadas, no deben cambiarse. Se utilizan para almacenar valores que permanecen constantes a lo largo del programa.

PI = 3.14159
puts PI
# Salida: 3.14159
  • Las constantes comienzan con una letra mayúscula.

Alcance de las Variables

El alcance de una variable determina dónde puede ser accesible dentro del programa. Entender el alcance es crucial para escribir código limpio y mantenible.

  • Alcance Local: Variables definidas dentro de un método o bloque.
  • Alcance de Instancia: Variables accesibles dentro de una instancia de una clase.
  • Alcance de Clase: Variables compartidas entre todas las instancias de una clase.
  • Alcance Global: Variables accesibles desde cualquier parte del programa.

Mejores Prácticas

  • Usa nombres de variables significativos para hacer tu código más legible.
  • Evita usar variables globales a menos que sea absolutamente necesario.
  • Sigue las convenciones de nomenclatura de Ruby para los diferentes tipos de variables.
  • Mantén el alcance de las variables lo más limitado posible para evitar efectos secundarios no deseados.

Conclusión

Las variables son un concepto central en Ruby, permitiéndote almacenar y manipular datos de manera eficiente. Al entender los diferentes tipos de variables y sus alcances, puedes escribir código más efectivo y mantenible. Recuerda seguir las mejores prácticas para asegurar que tu código sea limpio y fácil de entender.
