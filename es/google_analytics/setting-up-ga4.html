Google Analytics 4 (GA4) es la última versión de Google Analytics, que ofrece características y capacidades mejoradas para rastrear y analizar las interacciones de los usuarios en sitios web y aplicaciones. Esta sección te guiará a través del proceso de configuración de GA4, desde la creación de una nueva propiedad hasta la configuración de ajustes clave.

Creación de una Propiedad GA4

Para comenzar a usar GA4, necesitas crear una nueva propiedad GA4 en tu cuenta de Google Analytics.

  1. Inicia sesión en Google Analytics:

    • Ve a Google Analytics e inicia sesión con tu cuenta de Google.

  2. Crea una nueva propiedad:

    • Haz clic en el ícono de engranaje "Admin" en la parte inferior izquierda de la pantalla.
    • En la columna "Cuenta", selecciona la cuenta a la que deseas agregar la propiedad.
    • En la columna "Propiedad", haz clic en "Crear Propiedad".

  3. Configura los ajustes de la propiedad:

    • Ingresa un nombre para la propiedad (por ejemplo, "Mi Sitio Web GA4").
    • Selecciona tu zona horaria de informes y moneda.
    • Haz clic en "Siguiente" para proceder al siguiente paso.

  4. Configura los flujos de datos:

    • Elige la plataforma desde la que deseas recopilar datos (Web, iOS, Android).
    • Para un sitio web, ingresa la URL y el nombre del flujo.
    • Haz clic en "Crear Flujo" para generar un ID de Medición.

Instalación del Código de Seguimiento de GA4

Una vez que hayas creado una propiedad GA4, necesitas instalar el código de seguimiento en tu sitio web para comenzar a recopilar datos.

  1. Copia el código de seguimiento de GA4:

    • En la página "Detalles del Flujo Web", encuentra la sección "Instrucciones de Etiquetado".
    • Haz clic en "Etiqueta del Sitio Global (gtag.js)" para expandir las instrucciones.
    • Copia el fragmento de código JavaScript proporcionado.

  2. Agrega el código de seguimiento a tu sitio web:

    • Pega el fragmento de código copiado en la sección <head> del HTML de tu sitio web.
    • Asegúrate de que el código esté colocado en cada página que desees rastrear.
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Sitio Web</title>
    <!-- Código de Seguimiento GA4 -->
    <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
    <script>
      window.dataLayer = window.dataLayer || [];
      function gtag(){dataLayer.push(arguments);}
      gtag('js', new Date());
      gtag('config', 'G-XXXXXXXXXX');
    </script>
</head>
<body>
    <!-- Contenido de la página -->
</body>
</html>

Configuración de Ajustes Clave

Después de instalar el código de seguimiento, debes configurar ajustes clave para asegurar una recopilación y reporte de datos precisos.

Configuración de Retención de Datos

  1. Navega a la configuración de retención de datos:

    • Ve a "Admin" > "Configuración de Datos" > "Retención de Datos".

  2. Configura el período de retención de datos:

    • Elige el período de retención deseado para los datos de usuario y eventos (por ejemplo, 14 meses).
    • Habilita "Restablecer en nueva actividad" si deseas restablecer el período de retención con cada nueva actividad del usuario.

Seguimiento de Eventos

GA4 utiliza un modelo basado en eventos para rastrear las interacciones de los usuarios. Puedes configurar eventos personalizados para capturar acciones específicas en tu sitio web.

  1. Crea eventos personalizados:

    • Ve a "Admin" > "Eventos" > "Crear Evento".
    • Define el nombre del evento y los parámetros (por ejemplo, button_click, page_view).

  2. Implementa el código de seguimiento de eventos:

    • Usa la función gtag para enviar eventos personalizados a GA4.
// Ejemplo de seguimiento de un evento de clic en un botón
document.getElementById('myButton').addEventListener('click', function() {
    gtag('event', 'button_click', {
        'event_category': 'engagement',
        'event_label': 'Mi Botón'
    });
});

Verificación de la Recopilación de Datos

Después de configurar GA4 y ajustar los ajustes clave, es importante verificar que los datos se estén recopilando correctamente.

  1. Usa el informe en tiempo real:

    • Ve a "Informes" > "Tiempo Real" para ver datos en vivo de tu sitio web.
    • Verifica que las interacciones de los usuarios se estén rastreando como se espera.

  2. Revisa DebugView:

    • Habilita DebugView para ver información detallada sobre eventos y parámetros.
    • Usa la extensión de Chrome Google Tag Assistant para depurar y validar tu configuración de seguimiento.

Conclusión

Configurar GA4 implica crear una nueva propiedad, instalar el código de seguimiento, configurar ajustes clave y verificar la recopilación de datos. Siguiendo estos pasos, puedes asegurarte de que tu configuración de GA4 sea precisa y esté lista para proporcionar información valiosa sobre el comportamiento de los usuarios en tu sitio web. A medida que te familiarices más con GA4, puedes explorar características avanzadas y personalizaciones para mejorar aún más tus capacidades analíticas.
