Introducción

Google Analytics es una herramienta poderosa para rastrear y analizar el tráfico de sitios web. Entender su terminología y conceptos básicos es crucial para aprovechar todo su potencial. Esta sección cubrirá los términos e ideas fundamentales que necesitas para comenzar.

Conceptos Clave

Usuarios

  • Definición: Individuos que visitan tu sitio web.
  • Tipos:
    • Nuevos Usuarios: Visitantes por primera vez.
    • Usuarios Recurrentes: Visitantes que han estado en tu sitio antes.

Sesiones

  • Definición: Una sesión es un grupo de interacciones de usuario con tu sitio web que tienen lugar dentro de un marco de tiempo determinado.
  • Duración: Por defecto, una sesión termina después de 30 minutos de inactividad o cuando el reloj marca la medianoche.

Vistas de Página

  • Definición: Se registra una vista de página cada vez que una página en tu sitio web se carga o recarga.
  • Vistas de Página Únicas: El número de sesiones durante las cuales se vio la página especificada al menos una vez.

Tasa de Rebote

  • Definición: El porcentaje de sesiones de una sola página en las que no hubo interacción con la página.
  • Importancia: Una alta tasa de rebote puede indicar que la página de destino no es relevante para tus visitantes.

Adquisición

  • Definición: Cómo los usuarios llegan a tu sitio web.
  • Canales:
    • Búsqueda Orgánica: Tráfico de motores de búsqueda.
    • Directo: Tráfico de usuarios que escribieron tu URL directamente en su navegador.
    • Referencia: Tráfico de otros sitios web.
    • Social: Tráfico de plataformas de redes sociales.

Comportamiento

  • Definición: Cómo los usuarios interactúan con tu sitio web.
  • Métricas:
    • Páginas/Sesión: El número promedio de páginas vistas durante una sesión.
    • Duración Promedio de la Sesión: La duración promedio de una sesión.

Conversión

  • Definición: Cuando un usuario completa un objetivo deseado en tu sitio web.
  • Tipos:
    • Micro Conversiones: Acciones más pequeñas e incrementales (por ejemplo, suscribirse a un boletín).
    • Macro Conversiones: Objetivos principales (por ejemplo, realizar una compra).

Ejemplo: Entendiendo la Tasa de Rebote

Considera un sitio web con los siguientes datos:

  • Total de Sesiones: 1,000
  • Sesiones de Una Sola Página: 300

Para calcular la tasa de rebote:

Tasa de Rebote = (Sesiones de Una Sola Página / Total de Sesiones) * 100
Tasa de Rebote = (300 / 1,000) * 100 = 30%

Esto significa que el 30% de las sesiones fueron sesiones de una sola página sin interacción adicional.

Resumen

Entender estos términos y conceptos básicos es esencial para usar Google Analytics de manera efectiva. Familiarizándote con usuarios, sesiones, vistas de página, tasa de rebote, adquisición, comportamiento y conversiones, puedes comenzar a analizar y mejorar el rendimiento de tu sitio web. A medida que avances en este curso, estos conceptos fundamentales se ampliarán para proporcionar conocimientos más profundos y capacidades analíticas más avanzadas.
