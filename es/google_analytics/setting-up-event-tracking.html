Introducción al Seguimiento de Eventos

El seguimiento de eventos en Google Analytics te permite medir las interacciones de los usuarios con los elementos de tu sitio web que no desencadenan automáticamente una vista de página. Estas interacciones pueden incluir clics en botones, descargas, reproducciones de videos y más.

Conceptos Clave

  • Eventos: Interacciones de los usuarios con el contenido que se pueden rastrear independientemente de una página web o una carga de pantalla.
  • Categoría de Evento: El nombre que proporcionas para el grupo de objetos que deseas rastrear.
  • Acción de Evento: El tipo de interacción (por ejemplo, 'clic').
  • Etiqueta de Evento: Información adicional sobre el evento (por ejemplo, el nombre del botón en el que se hizo clic).
  • Valor de Evento: Un valor numérico asociado con el evento (por ejemplo, el tiempo dedicado a ver un video).

Configuración del Seguimiento Básico de Eventos

Para configurar el seguimiento de eventos, necesitas modificar el código de tu sitio web para enviar datos de eventos a Google Analytics.

Usando Google Tag Manager (GTM)

Google Tag Manager simplifica el proceso de configuración del seguimiento de eventos sin necesidad de modificar directamente el código de tu sitio web.

Pasos:

  1. Crear una Nueva Etiqueta:

    • Ve a tu cuenta de GTM.
    • Haz clic en "Tags" y luego en "New".
    • Elige "Tag Configuration" y selecciona "Google Analytics: Universal Analytics".
    • Configura el "Track Type" a "Event".

  2. Configurar la Etiqueta:

    • Rellena los Parámetros de Seguimiento de Eventos:
      • Categoría: por ejemplo, 'Button'
      • Acción: por ejemplo, 'Click'
      • Etiqueta: por ejemplo, 'Subscribe Button'
      • Valor: Opcional, por ejemplo, '1'

  3. Configurar el Disparador:

    • Haz clic en "Triggering" y luego en "New".
    • Elige "Click - All Elements" o un tipo de disparador específico.
    • Configura el disparador para que se active en la interacción deseada (por ejemplo, clics en un botón específico).

  4. Publicar el Contenedor:

    • Envía y publica tu contenedor para hacer los cambios en vivo.

Código de Ejemplo para Implementación Manual

Si no estás usando GTM, puedes agregar manualmente el código de seguimiento de eventos a tu sitio web.

<a href="#" onclick="ga('send', 'event', 'Button', 'Click', 'Subscribe Button');">Subscribe</a>

Explicación:

  • ga('send', 'event', 'Button', 'Click', 'Subscribe Button'); envía un evento a Google Analytics.
    • 'Button' es la Categoría de Evento.
    • 'Click' es la Acción de Evento.
    • 'Subscribe Button' es la Etiqueta de Evento.

Seguimiento Avanzado de Eventos

El seguimiento avanzado de eventos implica interacciones más complejas y parámetros adicionales.

Seguimiento de Envíos de Formularios

Para rastrear envíos de formularios, puedes usar el siguiente código:

<form id="contactForm" onsubmit="ga('send', 'event', 'Form', 'Submit', 'Contact Form');">
  <!-- campos del formulario -->
  <button type="submit">Submit</button>
</form>

Seguimiento de Reproducciones de Videos

Para rastrear reproducciones de videos, puedes usar la API de YouTube o métodos similares para otras plataformas de video.

Ejemplo con la API de YouTube:

  1. Cargar la API de IFrame de YouTube:

    <script src="https://www.youtube.com/iframe_api"></script>

  2. Inicializar el Reproductor y Rastrear Eventos:

    <div id="player"></div>
<script>
  var tag = document.createElement('script');
  tag.src = "https://www.youtube.com/iframe_api";
  var firstScriptTag = document.getElementsByTagName('script')[0];
  firstScriptTag.parentNode.insertBefore(tag, firstScriptTag);

  var player;
  function onYouTubeIframeAPIReady() {
    player = new YT.Player('player', {
      height: '390',
      width: '640',
      videoId: 'M7lc1UVf-VE',
      events: {
        'onStateChange': onPlayerStateChange
      }
    });
  }

  function onPlayerStateChange(event) {
    if (event.data == YT.PlayerState.PLAYING) {
      ga('send', 'event', 'Video', 'Play', 'Demo Video');
    }
  }
</script>

Resumen

Configurar el seguimiento de eventos en Google Analytics te permite medir interacciones específicas de los usuarios en tu sitio web. Ya sea usando Google Tag Manager para una configuración simplificada o agregando manualmente el código de seguimiento, puedes capturar datos valiosos sobre cómo los usuarios interactúan con tu contenido. El seguimiento avanzado puede incluir envíos de formularios y reproducciones de videos, proporcionando una visión más profunda del comportamiento del usuario. Al comprender e implementar el seguimiento de eventos, puedes tomar decisiones basadas en datos para mejorar el rendimiento y la experiencia del usuario en tu sitio web.
