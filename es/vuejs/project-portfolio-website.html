Introducción

En este proyecto, construiremos un sitio web de portafolio utilizando Vue.js. Este proyecto te ayudará a consolidar tu conocimiento de Vue.js aplicándolo a un escenario del mundo real. Al final de este proyecto, tendrás un sitio web de portafolio completamente funcional para mostrar tus habilidades y proyectos.

Configuración del Proyecto

Antes de comenzar a construir el sitio web de portafolio, configuremos nuestro proyecto.

  1. Instalar Vue CLI: Si aún no has instalado Vue CLI, puedes hacerlo ejecutando:

    npm install -g @vue/cli

  2. Crear un Nuevo Proyecto Vue: Crea un nuevo proyecto Vue ejecutando:

    vue create portfolio-website

  3. Navegar al Directorio del Proyecto:

    cd portfolio-website

  4. Ejecutar el Servidor de Desarrollo:

    npm run serve

Estructura del Proyecto

Entender la estructura del proyecto es crucial para un desarrollo eficiente.

  • src: Contiene el código principal de la aplicación.
    • assets: Activos estáticos como imágenes y estilos.
    • components: Componentes reutilizables de Vue.
    • views: Diferentes vistas o páginas de la aplicación.
    • router: Configuración para Vue Router.
    • store: Tienda Vuex para la gestión del estado (si es necesario).

Creación de Componentes

Crearemos varios componentes para nuestro sitio web de portafolio:

  • Header: Contiene el menú de navegación.
  • Footer: Contiene información de contacto y enlaces a redes sociales.
  • Home: La página de inicio del portafolio.
  • About: Una página con información sobre ti.
  • Projects: Una página que muestra tus proyectos.
  • Contact: Una página con un formulario de contacto.

Componente Header

Comencemos creando el componente Header.

  1. Crear Header.vue en el directorio components:

    <template>
  <header>
    <nav>
      <ul>

        <li><router-link to="/about">About</router-link></li>
        <li><router-link to="/projects">Projects</router-link></li>
        <li><router-link to="/contact">Contact</router-link></li>
      </ul>
    </nav>
  </header>
</template>

<script>
export default {
  name: 'Header'
}
</script>

<style scoped>

  color: white;
  padding: 1rem;
}

  list-style: none;
  display: flex;
  justify-content: space-around;

  margin: 0 1rem;

</style>

  2. Importar y Usar el Componente Header en App.vue:

Enrutamiento

Para navegar entre diferentes páginas, necesitamos configurar Vue Router.

  1. Instalar Vue Router:

    npm install vue-router

  2. Configurar Vue Router en src/router/index.js:

    import Vue from 'vue'
import VueRouter from 'vue-router'
import Home from '../views/Home.vue'
import About from '../views/About.vue'
import Projects from '../views/Projects.vue'
import Contact from '../views/Contact.vue'

Vue.use(VueRouter)

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/projects', component: Projects },
  { path: '/contact', component: Contact }
]

const router = new VueRouter({
  mode: 'history',
  routes
})

export default router

  3. Importar y Usar el Router en src/main.js:

    import Vue from 'vue'
import App from './App.vue'
import router from './router'

Vue.config.productionTip = false

new Vue({
  router,
  render: h => h(App)
}).$mount('#app')
