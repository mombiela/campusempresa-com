El enrutamiento dinámico es una característica poderosa en Vue.js que te permite crear rutas que pueden cambiar basadas en la entrada del usuario u otros datos dinámicos. Esto es esencial para construir aplicaciones que necesitan manejar una variedad de caminos y parámetros.

Conceptos Clave

  • Segmentos Dinámicos: Partes de la URL que pueden cambiar, a menudo representadas con dos puntos (:) en la ruta.
  • Parámetros de Ruta: Variables que se pasan a través de la URL y pueden ser accedidas dentro del componente.
  • Rutas Anidadas: Rutas que están anidadas dentro de otras rutas, permitiendo estructuras de navegación más complejas y jerárquicas.
  • Navegación Programática: Usar la API de Vue Router para navegar a diferentes rutas programáticamente.

Configuración de Vue Router

Primero, asegúrate de tener Vue Router instalado en tu proyecto de Vue.js. Si no, puedes instalarlo usando npm:

npm install vue-router

Luego, configura tu router en src/router/index.js:

import Vue from 'vue';
import VueRouter from 'vue-router';
import Home from '../views/Home.vue';
import User from '../views/User.vue';

Vue.use(VueRouter);

const routes = [
  {
    path: '/',
    name: 'Home',
    component: Home
  },
  {
    path: '/user/:id',
    name: 'User',
    component: User
  }
];

const router = new VueRouter({
  mode: 'history',
  base: process.env.BASE_URL,
  routes
});

export default router;

Segmentos Dinámicos y Parámetros de Ruta

Los segmentos dinámicos en las rutas se definen usando dos puntos (:) seguidos del nombre del parámetro. Por ejemplo, /user/:id define una ruta con un segmento dinámico id.

Ejemplo

En el componente User.vue, puedes acceder al parámetro de ruta id usando this.$route.params.id:

<template>
  <div>
    <h1>Perfil de Usuario</h1>
    <p>ID de Usuario: {{ userId }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      userId: this.$route.params.id
    };
  },
  watch: {
    '$route.params.id'(newId) {
      this.userId = newId;
    }
  }
};
</script>

Rutas Anidadas

Las rutas anidadas te permiten crear estructuras de rutas más complejas. Por ejemplo, podrías tener una ruta de perfil de usuario con rutas anidadas para diferentes secciones del perfil.

Ejemplo

Define rutas anidadas en src/router/index.js:

const routes = [
  {
    path: '/user/:id',
    component: User,
    children: [
      {
        path: 'profile',
        component: UserProfile
      },
      {
        path: 'posts',
        component: UserPosts
      }
    ]
  }
];

En el componente User.vue, usa <router-view> para mostrar las rutas anidadas:

<template>
  <div>
    <h1>Perfil de Usuario</h1>
    <router-link :to="{ name: 'UserProfile', params: { id: userId } }">Perfil</router-link>
    <router-link :to="{ name: 'UserPosts', params: { id: userId } }">Publicaciones</router-link>
    <router-view></router-view>
  </div>
</template>

<script>
export default {
  data() {
    return {
      userId: this.$route.params.id
    };
  }
};
</script>

Navegación Programática

Puedes navegar a diferentes rutas programáticamente usando la API de Vue Router. Esto es útil para la navegación basada en acciones del usuario u otras condiciones dinámicas.

Ejemplo

En un método dentro de tu componente, usa this.$router.push para navegar a una ruta diferente:

<template>
  <div>
    <button @click="goToUserProfile">Ir al Perfil de Usuario</button>
  </div>
</template>

<script>
export default {
  methods: {
    goToUserProfile() {
      this.$router.push({ name: 'User', params: { id: 123 } });
    }
  }
};
</script>

Conclusión

El enrutamiento dinámico en Vue.js te permite crear estructuras de navegación flexibles y poderosas en tus aplicaciones. Al entender los segmentos dinámicos, los parámetros de ruta, las rutas anidadas y la navegación programática, puedes construir interfaces complejas y amigables para el usuario. Practica estos conceptos con diferentes escenarios para volverte competente en el enrutamiento dinámico con Vue.js.
