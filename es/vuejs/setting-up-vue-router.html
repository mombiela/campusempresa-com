Introducción

Vue Router es el enrutador oficial para Vue.js, que permite a los desarrolladores crear aplicaciones de una sola página con navegación y enrutamiento dinámico. Esta sección te guiará a través del proceso de configuración de Vue Router en tu proyecto Vue.js.

Instalación de Vue Router

Para comenzar a usar Vue Router, necesitas instalarlo en tu proyecto Vue.js. Puedes hacerlo usando npm o yarn.

Usando npm

npm install vue-router

Usando yarn

yarn add vue-router

Configuración Básica

Una vez que Vue Router está instalado, necesitas configurarlo en tu proyecto. Esto implica crear una instancia del enrutador y definir tus rutas.

Creando una Instancia del Enrutador

Primero, crea un nuevo archivo llamado router.js (o router/index.js si prefieres una estructura anidada) en tu directorio src.

// src/router.js
import Vue from 'vue';
import VueRouter from 'vue-router';
import Home from './components/Home.vue';
import About from './components/About.vue';

Vue.use(VueRouter);

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
];

const router = new VueRouter({
  routes
});

export default router;

Integrando el Enrutador con tu Instancia de Vue

A continuación, necesitas integrar el enrutador con tu instancia principal de Vue. Abre tu archivo main.js e importa el enrutador.

// src/main.js
import Vue from 'vue';
import App from './App.vue';
import router from './router';

new Vue({
  router,
  render: h => h(App)
}).$mount('#app');

Definiendo Rutas

Las rutas son el núcleo de Vue Router. Mapean URLs a componentes.

Definición Básica de Rutas

En el archivo router.js, defines rutas como objetos con propiedades path y component.

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About }
];

Rutas Nombradas

Las rutas nombradas facilitan el enlace a rutas de manera programática.

const routes = [
  { path: '/', component: Home, name: 'home' },
  { path: '/about', component: About, name: 'about' }
];

Rutas Dinámicas

Las rutas dinámicas te permiten capturar parámetros de la URL.

const routes = [
  { path: '/user/:id', component: User }
];

En el componente User, puedes acceder al parámetro de la ruta id usando this.$route.params.id.

Navegación

Vue Router proporciona varias formas de navegar entre rutas.

Usando ``

El componente <router-link> se usa para crear enlaces de navegación.

<template>
  <div>
    <router-link to="/">Inicio</router-link>
    <router-link to="/about">Acerca de</router-link>
  </div>
</template>

Navegación Programática

Puedes navegar de manera programática usando el método this.$router.push.

methods: {
  goToAbout() {
    this.$router.push('/about');
  }
}

Rutas Anidadas

Las rutas anidadas te permiten crear estructuras de rutas complejas.

const routes = [
  {
    path: '/user/:id',
    component: User,
    children: [
      {
        path: 'profile',
        component: UserProfile
      },
      {
        path: 'posts',
        component: UserPosts
      }
    ]
  }
];

Guardias de Ruta

Las guardias de ruta se usan para controlar el acceso a las rutas.

Guardias Globales

Las guardias globales se aplican a cada cambio de ruta.

router.beforeEach((to, from, next) => {
  if (to.path === '/about') {
    // Realizar alguna verificación
    next();
  } else {
    next();
  }
});

Guardias por Ruta

Las guardias por ruta se definen directamente en la configuración de la ruta.

const routes = [
  {
    path: '/about',
    component: About,
    beforeEnter: (to, from, next) => {
      // Realizar alguna verificación
      next();
    }
  }
];

Resumen

En esta sección, aprendiste cómo configurar Vue Router en un proyecto Vue.js. Cubrimos la instalación, configuración básica, definición de rutas, navegación, rutas anidadas y guardias de ruta. Con este conocimiento, ahora puedes crear aplicaciones dinámicas de una sola página con Vue.js y Vue Router.
