Renderizado condicional en Vue.js
Renderizado de Listas ►

El renderizado condicional es un concepto fundamental en Vue.js que te permite controlar dinámicamente la visibilidad y el renderizado de elementos en el DOM basado en ciertas condiciones. Este tema cubrirá desde las técnicas básicas hasta las avanzadas de renderizado condicional en Vue.js.

Conceptos básicos del renderizado condicional

Directiva v-if

La directiva v-if se utiliza para renderizar condicionalmente un bloque. El bloque solo se renderizará si la expresión de la directiva devuelve un valor verdadero.

<template>
  <div v-if="isVisible">
    Este contenido es visible si isVisible es verdadero.
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: true
    };
  }
};
</script>

Directiva v-else

La directiva v-else se puede usar para proporcionar un bloque "else" para v-if.

<template>
  <div>
    <div v-if="isVisible">
      Este contenido es visible si isVisible es verdadero.
    </div>
    <div v-else>
      Este contenido es visible si isVisible es falso.
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: false
    };
  }
};
</script>

Directiva v-else-if

La directiva v-else-if se puede usar para encadenar múltiples condiciones.

<template>
  <div>
    <div v-if="status === 'success'">
      ¡La operación fue exitosa!
    </div>
    <div v-else-if="status === 'error'">
      Hubo un error.
    </div>
    <div v-else>
      El estado es desconocido.
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      status: 'error'
    };
  }
};
</script>

Renderizado condicional avanzado

Directiva v-show

La directiva v-show alterna la visibilidad de un elemento cambiando su propiedad CSS display. A diferencia de v-if, el elemento siempre se renderiza y permanece en el DOM.

<template>
  <div v-show="isVisible">
    Este contenido siempre está en el DOM pero solo es visible si isVisible es verdadero.
  </div>
</template>

<script>
export default {
  data() {
    return {
      isVisible: true
    };
  }
};
</script>

v-if vs v-show

| Característica | v-if | v-show | |----------------|-------------------------------|-------------------------------| | Renderizado | Renderiza/elimina condicionalmente | Siempre renderiza, alterna display | | Rendimiento | Mayor costo para alternar | Menor costo para alternar | | Caso de uso | Elementos raramente alternados | Elementos frecuentemente alternados |

Grupos condicionales

Puedes usar un elemento <template> con v-if, v-else-if y v-else para renderizar condicionalmente múltiples elementos.

<template>
  <template v-if="status === 'success'">
    <p>¡La operación fue exitosa!</p>
    <button @click="retry">Reintentar</button>
  </template>
  <template v-else-if="status === 'error'">
    <p>Hubo un error.</p>
    <button @click="retry">Reintentar</button>
  </template>
  <template v-else>
    <p>El estado es desconocido.</p>
  </template>
</template>

<script>
export default {
  data() {
    return {
      status: 'unknown'
    };
  },
  methods: {
    retry() {
      // Lógica de reintento aquí
    }
  }
};
</script>

Conclusión

El renderizado condicional en Vue.js proporciona herramientas poderosas para controlar la visibilidad y el renderizado de elementos basados en condiciones dinámicas. Al entender y utilizar v-if, v-else, v-else-if y v-show, puedes crear interfaces de usuario más interactivas y responsivas. Recuerda elegir entre v-if y v-show basado en las implicaciones de rendimiento y la frecuencia de alternar la visibilidad del elemento.
