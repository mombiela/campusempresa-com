¿Qué es Axios?

Axios es una popular biblioteca de JavaScript utilizada para realizar solicitudes HTTP desde el navegador y Node.js. Está basada en promesas, lo que permite un código asincrónico más limpio y manejable. En el contexto de Vue.js, Axios se utiliza comúnmente para obtener datos de APIs y manejar solicitudes HTTP en una aplicación Vue.

Características Clave de Axios

  • Basado en Promesas: Simplifica el código asincrónico con async/await o .then() y .catch().
  • Interceptores: Te permite interceptar solicitudes o respuestas antes de que sean manejadas.
  • Transformación Automática de Datos JSON: Transforma automáticamente los datos JSON por ti.
  • Manejo de Errores: Proporciona una manera simple de manejar errores.
  • Cancelación de Solicitudes: Soporta la cancelación de solicitudes utilizando la API CancelToken.

Instalando Axios

Para usar Axios en un proyecto de Vue.js, primero necesitas instalarlo. Puedes hacerlo usando npm o yarn.

npm install axios
# o
yarn add axios

Usando Axios en un Componente Vue

Solicitud GET Básica

Aquí tienes cómo puedes hacer una solicitud GET básica para obtener datos de una API y mostrarlos en un componente Vue.

<template>
  <div>
    <h1>Datos Obtenidos</h1>
    <ul>
      <li v-for="item in items" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      items: []
    };
  },
  created() {
    axios.get('https://api.example.com/items')
      .then(response => {
        this.items = response.data;
      })
      .catch(error => {
        console.error('¡Hubo un error!', error);
      });
  }
};
</script>

Solicitud POST

Para enviar datos a una API, puedes usar una solicitud POST. Aquí tienes un ejemplo de cómo hacerlo en un componente Vue.

<template>
  <div>
    <h1>Agregar Nuevo Ítem</h1>
    <form @submit.prevent="addItem">
      <input v-model="newItem" placeholder="Ingresa el nombre del ítem" />
      <button type="submit">Agregar Ítem</button>
    </form>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      newItem: ''
    };
  },
  methods: {
    addItem() {
      axios.post('https://api.example.com/items', { name: this.newItem })
        .then(response => {
          console.log('Ítem agregado:', response.data);
          this.newItem = ''; // Limpiar el campo de entrada
        })
        .catch(error => {
          console.error('¡Hubo un error!', error);
        });
    }
  }
};
</script>

Manejo de Errores

El manejo de errores en Axios es sencillo. Puedes usar .catch() para manejar errores en tus solicitudes.

axios.get('https://api.example.com/items')
  .then(response => {
    this.items = response.data;
  })
  .catch(error => {
    console.error('¡Hubo un error!', error);
  });

Usando Interceptores de Axios

Los interceptores te permiten ejecutar tu código o modificar la solicitud/respuesta antes de que sea manejada por then o catch.

Interceptor de Solicitud

axios.interceptors.request.use(config => {
  // Hacer algo antes de que la solicitud sea enviada
  console.log('Interceptor de Solicitud:', config);
  return config;
}, error => {
  // Hacer algo con el error de la solicitud
  return Promise.reject(error);
});

Interceptor de Respuesta

axios.interceptors.response.use(response => {
  // Hacer algo con los datos de la respuesta
  console.log('Interceptor de Respuesta:', response);
  return response;
}, error => {
  // Hacer algo con el error de la respuesta
  return Promise.reject(error);
});

Cancelando Solicitudes

Puedes cancelar una solicitud de Axios utilizando la API CancelToken.

const CancelToken = axios.CancelToken;
let cancel;

axios.get('https://api.example.com/items', {
  cancelToken: new CancelToken(function executor(c) {
    cancel = c;
  })
}).catch(thrown => {
  if (axios.isCancel(thrown)) {
    console.log('Solicitud cancelada', thrown.message);
  } else {
    // manejar error
  }
});

// Cancelar la solicitud
cancel('Operación cancelada por el usuario.');

Conclusión

Axios es una biblioteca poderosa y flexible para manejar solicitudes HTTP en aplicaciones Vue.js. Simplifica el proceso de hacer solicitudes asincrónicas, manejar errores y gestionar transformaciones de solicitudes/respuestas. Al entender y utilizar Axios, puedes interactuar eficientemente con APIs y mejorar la funcionalidad de tus aplicaciones Vue.js.
