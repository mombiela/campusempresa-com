Introducción

En esta sección, te guiaremos a través del proceso de crear tu primera aplicación Vue.js. Vue.js es un framework progresivo de JavaScript utilizado para construir interfaces de usuario. Está diseñado para ser adoptado incrementalmente, lo que significa que puedes usar tanto o tan poco de Vue como necesites.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Conocimientos básicos de HTML, CSS y JavaScript.
  • Node.js y npm instalados en tu computadora.

Configurando el Entorno de Desarrollo

Instalando Vue CLI

Vue CLI (Interfaz de Línea de Comandos) es una herramienta poderosa para crear y gestionar proyectos Vue.js. Para instalar Vue CLI, abre tu terminal y ejecuta:

npm install -g @vue/cli

Creando un Nuevo Proyecto Vue

Una vez que Vue CLI esté instalado, puedes crear un nuevo proyecto Vue ejecutando:

vue create my-first-vue-app

Se te pedirá que elijas un preset. Para principiantes, se recomienda seleccionar el preset por defecto presionando Enter.

Navegando a Tu Directorio de Proyecto

Después de que el proyecto se haya creado, navega a tu directorio de proyecto:

cd my-first-vue-app

Ejecutando el Servidor de Desarrollo

Para iniciar el servidor de desarrollo, ejecuta:

npm run serve

Deberías ver una salida indicando que tu aplicación está corriendo en http://localhost:8080.

Entendiendo la Estructura del Proyecto

Un proyecto Vue típico creado con Vue CLI tiene la siguiente estructura:

my-first-vue-app/
├── node_modules/
├── public/
│   ├── index.html
├── src/
│   ├── assets/
│   ├── components/
│   ├── App.vue
│   ├── main.js
├── .gitignore
├── babel.config.js
├── package.json
├── README.md
  • public/index.html: El archivo HTML principal.
  • src/main.js: El punto de entrada de la aplicación.
  • src/App.vue: El componente raíz.
  • src/components/: Directorio para componentes Vue.

Creando Tu Primer Componente

Modificando el Componente App

Abre src/App.vue y reemplaza su contenido con:

<template>
  <div id="app">
    <h1>{{ message }}</h1>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: 'Hello, Vue!'
    };
  }
};
</script>

<style>
#app {
  font-family: Avenir, Helvetica, Arial, sans-serif;
  text-align: center;
  color: #2c3e50;
  margin-top: 60px;
}
</style>

Este código define un componente Vue simple que muestra un mensaje.

Creando un Nuevo Componente

Crea un nuevo archivo src/components/HelloWorld.vue y agrega el siguiente contenido:

<template>
  <div>
    <h2>{{ greeting }}</h2>
  </div>
</template>

<script>
export default {
  data() {
    return {
      greeting: '¡Bienvenido a tu primer componente Vue!'
    };
  }
};
</script>

<style scoped>
h2 {
  color: #42b983;
}
</style>

Usando el Nuevo Componente

Para usar el nuevo componente en App.vue, primero impórtalo y luego regístralo:

<template>
  <div id="app">
    <h1>{{ message }}</h1>
    <HelloWorld />
  </div>
</template>

<script>
import HelloWorld from './components/HelloWorld.vue';

export default {
  components: {
    HelloWorld
  },
  data() {
    return {
      message: 'Hello, Vue!'
    };
  }
};
</script>

<style>
#app {
  font-family: Avenir, Helvetica, Arial, sans-serif;
  text-align: center;
  color: #2c3e50;
  margin-top: 60px;
}
</style>

Conclusión

¡Felicidades! Has creado con éxito tu primera aplicación Vue.js. Aprendiste cómo configurar el entorno de desarrollo, crear un nuevo proyecto Vue, entender la estructura del proyecto, y crear y usar componentes Vue. Este conocimiento fundamental será crucial a medida que continúes explorando características más avanzadas de Vue.js.
