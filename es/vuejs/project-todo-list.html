Introducción

En este proyecto, construiremos una aplicación simple de Lista de Tareas usando Vue.js. Este proyecto te ayudará a entender los conceptos básicos de Vue.js, incluyendo la vinculación de datos, el manejo de eventos y la arquitectura basada en componentes.

Conceptos Clave

  • Instancia de Vue: La raíz de cada aplicación Vue.
  • Vinculación de Datos: Sincronización de datos entre el modelo y la vista.
  • Manejo de Eventos: Responder a las acciones del usuario.
  • Componentes: Instancias de Vue reutilizables con su propio ámbito.
  • Directivas: Tokens especiales en el marcado que le dicen a la biblioteca que haga algo con un elemento del DOM.

Configuración del Proyecto

  1. Instalar Vue CLI: 
    npm install -g @vue/cli
  2. Crear un Nuevo Proyecto: 
    vue create todo-list
  3. Navegar al Directorio del Proyecto: 
    cd todo-list
  4. Ejecutar el Servidor de Desarrollo: 
    npm run serve

Construyendo la Aplicación de Lista de Tareas

Creando la Instancia de Vue

Crea una nueva instancia de Vue en src/main.js:

import Vue from 'vue';
import App from './App.vue';

Vue.config.productionTip = false;

new Vue({
  render: h => h(App),
}).$mount('#app');

Diseñando la Estructura de la Aplicación

En src/App.vue, configura la estructura básica:

<template>
  <div id="app">
    <h1>Lista de Tareas</h1>
    <AddTodo @add-todo="addTodo"></AddTodo>
    <TodoList :todos="todos" @delete-todo="deleteTodo"></TodoList>
  </div>
</template>

<script>
import AddTodo from './components/AddTodo.vue';
import TodoList from './components/TodoList.vue';

export default {
  name: 'App',
  components: {
    AddTodo,
    TodoList
  },
  data() {
    return {
      todos: []
    };
  },
  methods: {
    addTodo(todo) {
      this.todos.push(todo);
    },
    deleteTodo(index) {
      this.todos.splice(index, 1);
    }
  }
};
</script>

<style>
/* Añadir algo de estilo básico */
</style>

Creando el Componente AddTodo

Crea un nuevo archivo src/components/AddTodo.vue:

<template>
  <div>
    <input v-model="newTodo" @keyup.enter="submitTodo" placeholder="Añadir una nueva tarea" />
    <button @click="submitTodo">Añadir</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      newTodo: ''
    };
  },
  methods: {
    submitTodo() {
      if (this.newTodo.trim()) {
        this.$emit('add-todo', this.newTodo.trim());
        this.newTodo = '';
      }
    }
  }
};
</script>

<style scoped>
/* Añadir algo de estilo básico */
</style>

Creando el Componente TodoList

Crea un nuevo archivo src/components/TodoList.vue:

<template>
  <ul>
    <li v-for="(todo, index) in todos" :key="index">
      {{ todo }}
      <button @click="$emit('delete-todo', index)">Eliminar</button>
    </li>
  </ul>
</template>

<script>
export default {
  props: ['todos']
};
</script>
