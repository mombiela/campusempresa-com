Introducción

En Vue.js, las propiedades computadas y los observadores son características poderosas que ayudan a gestionar y reaccionar a los cambios de datos de manera eficiente. Esta sección cubrirá los conceptos básicos de las propiedades computadas y los observadores, sus diferencias y cómo usarlos efectivamente en tus aplicaciones Vue.js.

Propiedades Computadas

¿Qué son las Propiedades Computadas?

  • Las propiedades computadas son propiedades que dependen de otras propiedades de datos.
  • Están en caché basadas en sus dependencias, lo que significa que solo se re-evaluarán cuando una de sus dependencias cambie.
  • Útiles para cálculos complejos o transformaciones de datos que necesitan ser reactivas.

Uso Básico

Las propiedades computadas se definen dentro de la opción computed en un componente Vue.

<template>
  <div>
    <p>Mensaje Original: {{ message }}</p>
    <p>Mensaje Invertido: {{ reversedMessage }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: '¡Hola Vue.js!'
    };
  },
  computed: {
    reversedMessage() {
      return this.message.split('').reverse().join('');
    }
  }
};
</script>

Conceptos Clave

  • Caché: Las propiedades computadas están en caché basadas en sus dependencias reactivas.
  • Seguimiento de Dependencias: Vue rastrea automáticamente las dependencias de las propiedades computadas.

Ejemplo: Propiedad Computada con Múltiples Dependencias

<template>
  <div>
    <p>Nombre Completo: {{ fullName }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      firstName: 'John',
      lastName: 'Doe'
    };
  },
  computed: {
    fullName() {
      return `${this.firstName} ${this.lastName}`;
    }
  }
};
</script>

Observadores

¿Qué son los Observadores?

  • Los observadores son funciones que observan cambios en una propiedad de datos específica y ejecutan una devolución de llamada cuando los datos cambian.
  • Útiles para realizar operaciones asincrónicas o lógica compleja en respuesta a cambios de datos.

Uso Básico

Los observadores se definen dentro de la opción watch en un componente Vue.

<template>
  <div>
    <p>Mensaje: {{ message }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: '¡Hola Vue.js!'
    };
  },
  watch: {
    message(newValue, oldValue) {
      console.log(`El mensaje cambió de ${oldValue} a ${newValue}`);
    }
  }
};
</script>

Conceptos Clave

  • Ejecución Inmediata: Los observadores pueden configurarse para ejecutarse inmediatamente al ser creados.
  • Observación Profunda: Los observadores pueden configurarse para observar propiedades anidadas.

Ejemplo: Observador con Operación Asincrónica

<template>
  <div>
    <p>Consulta de Búsqueda: {{ query }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      query: ''
    };
  },
  watch: {
    query(newQuery) {
      this.fetchResults(newQuery);
    }
  },
  methods: {
    fetchResults(query) {
      // Simular una llamada a la API
      setTimeout(() => {
        console.log(`Buscando resultados para ${query}`);
      }, 500);
    }
  }
};
</script>

Comparación: Propiedades Computadas vs. Observadores

| Característica | Propiedades Computadas | Observadores | |-----------------------|----------------------------------------------|-----------------------------------------------| | Caso de Uso | Transformación de datos, estado derivado | Operaciones asincrónicas, lógica compleja | | Caché | Sí | No | | Seguimiento de Dependencias | Automático | Manual | | Sintaxis | Definido en la opción computed | Definido en la opción watch | | Ejecución | Sincrónica | Puede ser asincrónica |

Conclusión

Las propiedades computadas y los observadores son herramientas esenciales en Vue.js para gestionar datos reactivos. Las propiedades computadas son ideales para transformaciones de datos y estado derivado, mientras que los observadores son más adecuados para manejar efectos secundarios y lógica compleja en respuesta a cambios de datos. Entender cuándo y cómo usar estas características te ayudará a construir aplicaciones Vue.js más eficientes y mantenibles.
