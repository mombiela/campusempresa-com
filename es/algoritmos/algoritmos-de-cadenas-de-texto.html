Introducción

Los algoritmos de cadenas de texto son fundamentales en la informática y tienen aplicaciones en diversas áreas como la búsqueda de información, el procesamiento de lenguaje natural y la bioinformática. Este tema cubre los conceptos clave y los algoritmos más utilizados para manipular y analizar cadenas de texto.

Conceptos Clave

  • Cadenas de Texto: Secuencias de caracteres que pueden representar palabras, frases o cualquier otro tipo de datos textuales.
  • Subcadena: Una porción de una cadena de texto.
  • Patrón: Una cadena de texto que se busca dentro de otra cadena.
  • Prefijo y Sufijo: Subcadenas que comienzan desde el inicio o terminan en el final de una cadena, respectivamente.

Algoritmos Comunes

Búsqueda de Subcadenas

Algoritmo de Fuerza Bruta

El algoritmo de fuerza bruta compara el patrón con todas las posibles subcadenas de la cadena principal.

def fuerza_bruta(texto, patron):
    n = len(texto)
    m = len(patron)
    for i in range(n - m + 1):
        j = 0
        while j < m and texto[i + j] == patron[j]:
            j += 1
        if j == m:
            return i
    return -1

texto = "hola mundo"
patron = "mundo"
print(fuerza_bruta(texto, patron))  # Salida: 5

Algoritmo de Knuth-Morris-Pratt (KMP)

El algoritmo KMP mejora la eficiencia al evitar comparaciones redundantes mediante el uso de una tabla de prefijos.

def construir_tabla_kmp(patron):
    m = len(patron)
    tabla = [0] * m
    j = 0
    for i in range(1, m):
        if patron[i] == patron[j]:
            j += 1
            tabla[i] = j
        else:
            if j != 0:
                j = tabla[j - 1]
                i -= 1
            else:
                tabla[i] = 0
    return tabla

def kmp(texto, patron):
    n = len(texto)
    m = len(patron)
    tabla = construir_tabla_kmp(patron)
    i = j = 0
    while i < n:
        if patron[j] == texto[i]:
            i += 1
            j += 1
        if j == m:
            return i - j
        elif i < n and patron[j] != texto[i]:
            if j != 0:
                j = tabla[j - 1]
            else:
                i += 1
    return -1

texto = "hola mundo"
patron = "mundo"
print(kmp(texto, patron))  # Salida: 5

Comparación de Algoritmos

| Algoritmo | Complejidad Temporal | Ventajas | Desventajas | |-------------------|----------------------|-------------------------------|------------------------------| | Fuerza Bruta | O(n * m) | Fácil de implementar | Ineficiente para cadenas largas| | Knuth-Morris-Pratt| O(n + m) | Eficiente para patrones largos| Más complejo de implementar |

Transformaciones de Cadenas

Algoritmo de Levenshtein

El algoritmo de Levenshtein calcula la distancia mínima de edición entre dos cadenas, es decir, el número mínimo de operaciones necesarias para transformar una cadena en otra.

def distancia_levenshtein(s1, s2):
    if len(s1) < len(s2):
        return distancia_levenshtein(s2, s1)

    if len(s2) == 0:
        return len(s1)

    anterior_fila = range(len(s2) + 1)
    for i, c1 in enumerate(s1):
        actual_fila = [i + 1]
        for j, c2 in enumerate(s2):
            insertar = anterior_fila[j + 1] + 1
            eliminar = actual_fila[j] + 1
            sustituir = anterior_fila[j] + (c1 != c2)
            actual_fila.append(min(insertar, eliminar, sustituir))
        anterior_fila = actual_fila

    return anterior_fila[-1]

s1 = "kitten"
s2 = "sitting"
print(distancia_levenshtein(s1, s2))  # Salida: 3

Conclusión

Los algoritmos de cadenas de texto son herramientas poderosas para la manipulación y análisis de datos textuales. Desde la búsqueda de subcadenas hasta la transformación de cadenas, estos algoritmos tienen aplicaciones prácticas en múltiples campos. Comprender y saber implementar estos algoritmos es esencial para cualquier profesional de TI que trabaje con datos textuales.
