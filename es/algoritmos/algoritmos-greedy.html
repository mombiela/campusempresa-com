Introducción a los Algoritmos Greedy

Los algoritmos Greedy (o voraces) son una clase de algoritmos que construyen soluciones a problemas mediante una serie de elecciones locales óptimas. La idea principal es tomar la mejor decisión posible en cada paso con la esperanza de encontrar una solución global óptima.

Características de los Algoritmos Greedy

  • Elección Greedy: En cada paso, se selecciona la opción que parece ser la mejor en ese momento.
  • Solución Parcial: La solución se construye paso a paso, añadiendo elementos a una solución parcial.
  • Irrevocabilidad: Una vez que se toma una decisión, no se revisa ni se cambia.

Ejemplos de Problemas Resueltos con Algoritmos Greedy

Problema del Cambio de Monedas

El objetivo es dar cambio utilizando la menor cantidad de monedas posible.

Ejemplo de Código en Python

def cambio_monedas(cantidad, monedas):
    resultado = []
    for moneda in sorted(monedas, reverse=True):
        while cantidad >= moneda:
            cantidad -= moneda
            resultado.append(moneda)
    return resultado

# Ejemplo de uso
cantidad = 63
monedas = [1, 5, 10, 25]
print(cambio_monedas(cantidad, monedas))  # Salida: [25, 25, 10, 1, 1, 1]

Problema de la Mochila Fraccionaria

El objetivo es maximizar el valor total de los objetos en la mochila sin exceder su capacidad.

Ejemplo de Código en Python

class Objeto:
    def __init__(self, peso, valor):
        self.peso = peso
        self.valor = valor
        self.valor_por_peso = valor / peso

def mochila_fraccionaria(capacidad, objetos):
    objetos.sort(key=lambda x: x.valor_por_peso, reverse=True)
    valor_total = 0
    for obj in objetos:
        if capacidad >= obj.peso:
            capacidad -= obj.peso
            valor_total += obj.valor
        else:
            valor_total += obj.valor_por_peso * capacidad
            break
    return valor_total

# Ejemplo de uso
objetos = [Objeto(10, 60), Objeto(20, 100), Objeto(30, 120)]
capacidad = 50
print(mochila_fraccionaria(capacidad, objetos))  # Salida: 240.0

Ventajas y Desventajas de los Algoritmos Greedy

Ventajas

  • Simplicidad: Son fáciles de implementar y entender.
  • Eficiencia: Suelen ser rápidos y tienen una complejidad computacional baja.

Desventajas

  • No siempre óptimos: No garantizan encontrar la solución óptima para todos los problemas.
  • Irrevocabilidad: Las decisiones no se pueden cambiar una vez tomadas, lo que puede llevar a soluciones subóptimas.

Conclusión

Los algoritmos Greedy son una herramienta poderosa y eficiente para resolver ciertos tipos de problemas. Aunque no siempre garantizan la solución óptima, su simplicidad y rapidez los hacen útiles en muchas situaciones prácticas. Es importante entender cuándo y cómo aplicarlos adecuadamente para aprovechar sus ventajas y mitigar sus desventajas.
