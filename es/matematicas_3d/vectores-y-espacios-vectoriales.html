Introducción

En esta sección, exploraremos los conceptos fundamentales de vectores y espacios vectoriales, esenciales para entender las matemáticas en 3D. Estos conceptos son la base para muchas aplicaciones en física, ingeniería, informática y otras disciplinas.

Vectores

Un vector es una entidad que tiene magnitud y dirección. En el contexto de las matemáticas 3D, los vectores se representan en un espacio tridimensional.

Representación de Vectores

  • Notación: Un vector en 3D se puede representar como \(\mathbf{v} = \begin{pmatrix} v_x \ v_y \ v_z \end{pmatrix}\), donde \(v_x\), \(v_y\) y \(v_z\) son las componentes del vector en las direcciones \(x\), \(y\) y \(z\) respectivamente.
  • Ejemplo: \(\mathbf{v} = \begin{pmatrix} 3 \ -2 \ 5 \end{pmatrix}\)

Operaciones con Vectores

  • Suma de Vectores: La suma de dos vectores \(\mathbf{u} = \begin{pmatrix} u_x \ u_y \ u_z \end{pmatrix}\) y \(\mathbf{v} = \begin{pmatrix} v_x \ v_y \ v_z \end{pmatrix}\) se define como: \[ \mathbf{u} + \mathbf{v} = \begin{pmatrix} u_x + v_x \ u_y + v_y \ u_z + v_z \end{pmatrix} \]
  • Producto Escalar: El producto escalar de dos vectores \(\mathbf{u}\) y \(\mathbf{v}\) se define como: \[ \mathbf{u} \cdot \mathbf{v} = u_x v_x + u_y v_y + u_z v_z \]
  • Producto Vectorial: El producto vectorial de dos vectores \(\mathbf{u}\) y \(\mathbf{v}\) se define como: \[ \mathbf{u} \times \mathbf{v} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \ u_x & u_y & u_z \ v_x & v_y & v_z \end{vmatrix} \]

Ejemplo de Código

import numpy as np

# Definir vectores
u = np.array([1, 2, 3])
v = np.array([4, 5, 6])

# Suma de vectores
suma = u + v
print("Suma de vectores:", suma)

# Producto escalar
producto_escalar = np.dot(u, v)
print("Producto escalar:", producto_escalar)

# Producto vectorial
producto_vectorial = np.cross(u, v)
print("Producto vectorial:", producto_vectorial)

Espacios Vectoriales

Un espacio vectorial es un conjunto de vectores que se pueden sumar entre sí y multiplicar por escalares, cumpliendo ciertas propiedades.

Definición y Propiedades

  • Definición: Un espacio vectorial \(V\) sobre un campo \(F\) es un conjunto de elementos (vectores) junto con dos operaciones (suma y multiplicación por un escalar) que satisfacen las siguientes propiedades:
    • Cerradura bajo la suma: Si \(\mathbf{u}, \mathbf{v} \in V\), entonces \(\mathbf{u} + \mathbf{v} \in V\).
    • Cerradura bajo la multiplicación por un escalar: Si \(\mathbf{u} \in V\) y \(c \in F\), entonces \(c\mathbf{u} \in V\).
    • Asociatividad de la suma: \((\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})\) para todos \(\mathbf{u}, \mathbf{v}, \mathbf{w} \in V\).
    • Conmutatividad de la suma: \(\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}\) para todos \(\mathbf{u}, \mathbf{v} \in V\).
    • Elemento neutro de la suma: Existe un vector \(\mathbf{0} \in V\) tal que \(\mathbf{u} + \mathbf{0} = \mathbf{u}\) para todo \(\mathbf{u} \in V\).
    • Elemento inverso de la suma: Para cada \(\mathbf{u} \in V\), existe un vector \(-\mathbf{u} \in V\) tal que \(\mathbf{u} + (-\mathbf{u}) = \mathbf{0}\).
    • Distributividad de la multiplicación por un escalar: \(c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}\) y \((c + d)\mathbf{u} = c\mathbf{u} + d\mathbf{u}\) para todos \(c, d \in F\) y \(\mathbf{u}, \mathbf{v} \in V\).
    • Compatibilidad de la multiplicación por un escalar: \((cd)\mathbf{u} = c(d\mathbf{u})\) para todos \(c, d \in F\) y \(\mathbf{u} \in V\).
    • Elemento neutro de la multiplicación por un escalar: \(1\mathbf{u} = \mathbf{u}\) para todo \(\mathbf{u} \in V\).

Ejemplo de Espacio Vectorial

  • Espacio Vectorial \(\mathbb{R}^3\): El conjunto de todos los vectores de la forma \(\begin{pmatrix} x \ y \ z \end{pmatrix}\) donde \(x, y, z \in \mathbb{R}\) es un espacio vectorial sobre el campo de los números reales \(\mathbb{R}\).

Ejemplo de Código

# Verificar propiedades de un espacio vectorial en Python

# Definir vectores
u = np.array([1, 2, 3])
v = np.array([4, 5, 6])
w = np.array([7, 8, 9])
c = 2
d = 3

# Cerradura bajo la suma
cerradura_suma = np.all((u + v) in np.array([u, v, w]))
print("Cerradura bajo la suma:", cerradura_suma)

# Cerradura bajo la multiplicación por un escalar
cerradura_escalar = np.all((c * u) in np.array([u, v, w]))
print("Cerradura bajo la multiplicación por un escalar:", cerradura_escalar)

# Asociatividad de la suma
asociatividad_suma = np.all((u + (v + w)) == ((u + v) + w))
print("Asociatividad de la suma:", asociatividad_suma)

# Conmutatividad de la suma
conmutatividad_suma = np.all((u + v) == (v + u))
print("Conmutatividad de la suma:", conmutatividad_suma)

Conclusión
