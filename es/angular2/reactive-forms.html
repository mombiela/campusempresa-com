Formularios Reactivos en Angular 2+
Validación de Formularios ►

Los Formularios Reactivos en Angular proporcionan un enfoque basado en modelos para manejar entradas de formularios cuyos valores cambian con el tiempo. Este enfoque es más poderoso y escalable en comparación con los formularios basados en plantillas, especialmente para formularios complejos.

Introducción a los Formularios Reactivos

  • Enfoque basado en modelos: Los formularios reactivos se construyen alrededor de flujos observables, lo que facilita la gestión de interacciones complejas del formulario.
  • FormControl, FormGroup y FormArray: Bloques de construcción fundamentales para crear formularios reactivos.
  • Validación: Validadores integrados y personalizados para controles de formulario.

Configuración de Formularios Reactivos

Para usar Formularios Reactivos, necesitas importar el ReactiveFormsModule en tu módulo de Angular.

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // otras importaciones
  ],
  // otras configuraciones
})
export class AppModule { }

Creación de un Formulario Reactivo Básico

Paso 1: Importar Clases Requeridas

import { Component } from '@angular/core';
import { FormControl, FormGroup } from '@angular/forms';

Paso 2: Definir el Modelo del Formulario

@Component({
  selector: 'app-basic-form',
  template: `
    <form [formGroup]="profileForm" (ngSubmit)="onSubmit()">
      <label>
        First Name:
        <input formControlName="firstName">
      </label>
      <label>
        Last Name:
        <input formControlName="lastName">
      </label>
      <button type="submit">Submit</button>
    </form>
  `
})
export class BasicFormComponent {
  profileForm = new FormGroup({
    firstName: new FormControl(''),
    lastName: new FormControl('')
  });

  onSubmit() {
    console.log(this.profileForm.value);
  }
}

Explicación

  • FormGroup: Representa el formulario en su totalidad.
  • FormControl: Representa campos individuales del formulario.
  • formControlName: Vincula el control del formulario al elemento de entrada.

Características Avanzadas del Formulario

FormGroups Anidados

Puedes anidar instancias de FormGroup para crear formularios complejos.

this.profileForm = new FormGroup({
  firstName: new FormControl(''),
  lastName: new FormControl(''),
  address: new FormGroup({
    street: new FormControl(''),
    city: new FormControl(''),
    state: new FormControl(''),
    zip: new FormControl('')
  })
});

FormArray

FormArray se utiliza para gestionar un array de controles de formulario.

this.profileForm = new FormGroup({
  firstName: new FormControl(''),
  lastName: new FormControl(''),
  aliases: new FormArray([
    new FormControl('')
  ])
});

get aliases() {
  return this.profileForm.get('aliases') as FormArray;
}

addAlias() {
  this.aliases.push(new FormControl(''));
}

Controles de Formulario Dinámicos

Puedes añadir o eliminar controles de formulario dinámicamente.

addAlias() {
  this.aliases.push(new FormControl(''));
}

removeAlias(index: number) {
  this.aliases.removeAt(index);
}

Validación

Validadores Integrados

Angular proporciona varios validadores integrados como Validators.required, Validators.minLength, etc.

import { Validators } from '@angular/forms';

this.profileForm = new FormGroup({
  firstName: new FormControl('', Validators.required),
  lastName: new FormControl('', Validators.minLength(2))
});

Validadores Personalizados

Puedes crear validadores personalizados para satisfacer tus necesidades específicas.

import { AbstractControl, ValidatorFn } from '@angular/forms';

export function forbiddenNameValidator(nameRe: RegExp): ValidatorFn {
  return (control: AbstractControl): {[key: string]: any} | null => {
    const forbidden = nameRe.test(control.value);
    return forbidden ? {'forbiddenName': {value: control.value}} : null;
  };
}

this.profileForm = new FormGroup({
  firstName: new FormControl('', [Validators.required, forbiddenNameValidator(/bob/i)]),
  lastName: new FormControl('')
});

Formularios Reactivos vs Formularios Basados en Plantillas

| Característica | Formularios Reactivos | Formularios Basados en Plantillas | |------------------------|-------------------------------------|-------------------------------------| | Configuración | Más compleja, requiere más configuración | Más simple, menos configuración | | Modelo del Formulario | Definido en la clase del componente | Definido en la plantilla | | Validación | Definida en la clase del componente | Definida en la plantilla | | Escalabilidad | Más escalable para formularios complejos | Menos escalable para formularios complejos | | Pruebas | Más fácil de probar | Más difícil de probar |

Conclusión

Los Formularios Reactivos en Angular proporcionan una forma robusta y escalable de manejar entradas y validaciones de formularios. Al aprovechar el poder de los observables, los Formularios Reactivos ofrecen un enfoque más flexible y comprobable en comparación con los formularios basados en plantillas. Ya sea que estés manejando formularios simples o formularios anidados complejos, los Formularios Reactivos pueden manejarlo todo de manera eficiente.
