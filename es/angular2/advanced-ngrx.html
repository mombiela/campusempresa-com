NgRx es una poderosa biblioteca de gestión de estado para aplicaciones Angular, que proporciona una forma de gestionar el estado utilizando el patrón Redux. Esta sección profundizará en conceptos y técnicas avanzadas para ayudarte a dominar NgRx en tus proyectos Angular.

Conceptos Clave

  • Selectores: Consultar y derivar el estado de manera eficiente.
  • Efectos: Manejar efectos secundarios como solicitudes HTTP.
  • Estado de Entidad: Gestionar colecciones de entidades.
  • Estado del Router: Integrar NgRx con Angular Router.
  • Pruebas de NgRx: Mejores prácticas para probar la gestión del estado.

Selectores

Los selectores son funciones puras utilizadas para obtener porciones del estado. Ayudan a optimizar el rendimiento memorizando el estado.

Ejemplo

import { createSelector, createFeatureSelector } from '@ngrx/store';
import { AppState } from '../state/app.state';
import { UserState } from '../state/user.state';

const selectUserState = createFeatureSelector<AppState, UserState>('user');

export const selectUser = createSelector(
  selectUserState,
  (state: UserState) => state.user
);

En este ejemplo:

  • createFeatureSelector selecciona la porción user del estado.
  • createSelector deriva los datos user del UserState.

Efectos

Los efectos manejan efectos secundarios, como operaciones asincrónicas. Escuchan acciones y realizan tareas como solicitudes HTTP.

Ejemplo

import { Injectable } from '@angular/core';
import { Actions, createEffect, ofType } from '@ngrx/effects';
import { of } from 'rxjs';
import { catchError, map, mergeMap } from 'rxjs/operators';
import { UserService } from '../services/user.service';
import * as UserActions from '../actions/user.actions';

@Injectable()
export class UserEffects {
  loadUsers$ = createEffect(() =>
    this.actions$.pipe(
      ofType(UserActions.loadUsers),
      mergeMap(() =>
        this.userService.getAll().pipe(
          map(users => UserActions.loadUsersSuccess({ users })),
          catchError(error => of(UserActions.loadUsersFailure({ error })))
        )
      )
    )
  );

  constructor(
    private actions$: Actions,
    private userService: UserService
  ) {}
}

En este ejemplo:

  • ofType filtra acciones del tipo loadUsers.
  • mergeMap realiza la solicitud HTTP.
  • map despacha loadUsersSuccess en caso de éxito.
  • catchError despacha loadUsersFailure en caso de error.

Estado de Entidad

NgRx Entity proporciona una forma de gestionar colecciones de entidades, reduciendo el código repetitivo.

Ejemplo

import { EntityState, EntityAdapter, createEntityAdapter } from '@ngrx/entity';
import { User } from '../models/user.model';

export interface UserState extends EntityState<User> {
  selectedUserId: number | null;
}

export const adapter: EntityAdapter<User> = createEntityAdapter<User>();

export const initialState: UserState = adapter.getInitialState({
  selectedUserId: null,
});

En este ejemplo:

  • EntityState y EntityAdapter se utilizan para gestionar una colección de entidades User.
  • createEntityAdapter crea un adaptador para User.
  • getInitialState inicializa el estado con una colección vacía y un selectedUserId.

Estado del Router

Integrar NgRx con Angular Router te permite gestionar el estado de las rutas dentro del store.

Ejemplo

import { routerReducer, RouterReducerState } from '@ngrx/router-store';

export interface AppState {
  router: RouterReducerState;
}

export const reducers = {
  router: routerReducer,
};

En este ejemplo:

  • routerReducer se añade al estado raíz.
  • RouterReducerState realiza un seguimiento del estado del router.

Pruebas de NgRx

Probar NgRx implica probar reductores, selectores y efectos para asegurarse de que funcionan como se espera.

Ejemplo

Probar un Reductor

import { userReducer, initialState } from '../reducers/user.reducer';
import * as UserActions from '../actions/user.actions';

describe('UserReducer', () => {
  it('debería devolver el estado inicial', () => {
    const action = {} as any;
    const state = userReducer(undefined, action);

    expect(state).toBe(initialState);
  });

  it('debería cargar usuarios', () => {
    const users = [{ id: 1, name: 'John' }];
    const action = UserActions.loadUsersSuccess({ users });
    const state = userReducer(initialState, action);

    expect(state.entities[1]).toEqual(users[0]);
  });
});

En este ejemplo:

  • Se prueba el estado inicial.
  • Se prueba la acción loadUsersSuccess para asegurarse de que actualiza el estado correctamente.

Conclusión

Dominar conceptos avanzados de NgRx como selectores, efectos, estado de entidad, estado del router y pruebas mejorará significativamente tu capacidad para gestionar el estado en aplicaciones Angular. Al comprender y aplicar estas técnicas, puedes crear aplicaciones más eficientes, escalables y mantenibles.
