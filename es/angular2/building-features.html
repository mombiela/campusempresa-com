Introducción

En esta sección, exploraremos cómo construir varias funcionalidades en Angular 2+. Cubriremos los conceptos esenciales, proporcionaremos ejemplos de código e incluiremos ejercicios para ayudarte a entender e implementar estas funcionalidades en tus aplicaciones.

Componentes

Los componentes son los bloques de construcción de las aplicaciones Angular. Controlan una porción de la pantalla llamada vista.

  • Creación de un Componente: Usa Angular CLI para generar un nuevo componente.

    ng generate component my-component

  • Decorador de Componente: El decorador @Component se usa para definir metadatos para el componente.

    import { Component } from '@angular/core';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent {
  title = 'My Angular Component';
}

  • Plantilla y Estilos: Define el HTML y CSS para el componente.

    <!-- my-component.component.html -->
<h1>{{ title }}</h1>

    /* my-component.component.css */
h1 {
  color: blue;
}

Servicios e Inyección de Dependencias

Los servicios se usan para compartir datos y funcionalidades entre componentes.

  • Creación de un Servicio: Usa Angular CLI para generar un nuevo servicio.

    ng generate service my-service

  • Decorador de Servicio: El decorador @Injectable se usa para definir un servicio.

    import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root',
})
export class MyService {
  getData() {
    return 'Service Data';
  }
}

  • Inyección de un Servicio: Usa la inyección de dependencias para usar el servicio en un componente.

    import { Component, OnInit } from '@angular/core';
import { MyService } from './my-service.service';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent implements OnInit {
  data: string;

  constructor(private myService: MyService) {}

  ngOnInit(): void {
    this.data = this.myService.getData();
  }
}

Enrutamiento

El enrutamiento te permite navegar entre diferentes vistas o componentes.

  • Configuración de Rutas: Define rutas en el app-routing.module.ts.

    import { NgModule } from '@angular/core';
import { RouterModule, Routes } from '@angular/router';
import { MyComponent } from './my-component/my-component.component';

const routes: Routes = [
  { path: 'my-component', component: MyComponent },
  { path: '', redirectTo: '/my-component', pathMatch: 'full' }
];

@NgModule({
  imports: [RouterModule.forRoot(routes)],
  exports: [RouterModule]
})
export class AppRoutingModule { }

  • Router Outlet: Usa <router-outlet> en la plantilla principal para mostrar componentes enrutados.

    <!-- app.component.html -->
<nav>
  <a routerLink="/my-component">My Component</a>
</nav>
<router-outlet></router-outlet>

Formularios

Angular proporciona dos enfoques para manejar la entrada del usuario a través de formularios: Formularios basados en plantillas y Formularios reactivos.

  • Formularios Basados en Plantillas: Usa directivas de Angular para crear formularios.

    <!-- my-component.component.html -->
<form #myForm="ngForm" (ngSubmit)="onSubmit(myForm)">
  <input name="name" ngModel required>
  <button type="submit">Submit</button>
</form>

    // my-component.component.ts
export class MyComponent {
  onSubmit(form: any): void {
    console.log('Form Data: ', form.value);
  }
}

  • Formularios Reactivos: Usa FormBuilder para crear formularios programáticamente.

    import { Component, OnInit } from '@angular/core';
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent implements OnInit {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {}

  ngOnInit(): void {
    this.myForm = this.fb.group({
      name: ['', Validators.required]
    });
  }

  onSubmit(): void {
    console.log('Form Data: ', this.myForm.value);
  }
}

    <!-- my-component.component.html -->
<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <input formControlName="name" required>
  <button type="submit">Submit</button>
</form>

Cliente HTTP

El módulo HttpClient de Angular te permite comunicarte con servicios backend.

  • Configuración de HttpClient: Importa HttpClientModule en tu módulo de la aplicación.

    import { HttpClientModule } from '@angular/common/http';

@NgModule({
  imports: [HttpClientModule],
  // other imports
})
export class AppModule { }

  • Realización de Solicitudes HTTP: Usa HttpClient para hacer solicitudes HTTP.

    import { HttpClient } from '@angular/common/http';
import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root',
})
export class MyService {
  private apiUrl = 'https://api.example.com/data';

  constructor(private http: HttpClient) {}

  getData() {
    return this.http.get(this.apiUrl);
  }
}

    import { Component, OnInit } from '@angular/core';
import { MyService } from './my-service.service';

@Component({
  selector: 'app-my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})
export class MyComponent implements OnInit {
  data: any;

  constructor(private myService: MyService) {}

  ngOnInit(): void {
    this.myService.getData().subscribe(response => {
      this.data = response;
    });
  }
}

Conclusión

Al entender y practicar estos conceptos, estarás bien equipado para desarrollar funcionalidades avanzadas en tus proyectos Angular.
