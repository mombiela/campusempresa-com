Introducción

En esta sección, cubriremos los pasos esenciales y las mejores prácticas para construir y desplegar aplicaciones Angular. Esto incluye configurar el entorno de desarrollo, construir la aplicación para producción y desplegarla en varias plataformas.

Configuración del Entorno de Desarrollo

Antes de comenzar a construir y desplegar aplicaciones Angular, necesitamos configurar nuestro entorno de desarrollo.

  • Node.js y npm: Angular requiere Node.js y npm (Node Package Manager) para gestionar dependencias.
  • Angular CLI: La Interfaz de Línea de Comandos de Angular (CLI) es una herramienta poderosa que ayuda a crear, construir y gestionar aplicaciones Angular.

Instalación de Node.js y npm

  1. Descarga e instala Node.js desde el sitio web oficial.
  2. Verifica la instalación ejecutando los siguientes comandos en tu terminal: 
    node -v
npm -v

Instalación de Angular CLI

  1. Instala Angular CLI globalmente usando npm: 
    npm install -g @angular/cli
  2. Verifica la instalación ejecutando: 
    ng version

Creación de una Aplicación Angular

Vamos a crear una nueva aplicación Angular usando Angular CLI.

  1. Abre tu terminal y ejecuta el siguiente comando: 
    ng new my-angular-app
  2. Navega al directorio del proyecto: 
    cd my-angular-app
  3. Sirve la aplicación localmente: 
    ng serve
  4. Abre tu navegador y navega a http://localhost:4200 para ver tu aplicación en funcionamiento.

Construcción de la Aplicación para Producción

Para desplegar una aplicación Angular, necesitamos construirla para producción. Este proceso optimiza la aplicación para el rendimiento y crea los archivos necesarios para el despliegue.

  1. Ejecuta el siguiente comando para construir la aplicación: 
    ng build --prod
  2. Los artefactos de la construcción se almacenarán en el directorio dist/.

Entendiendo la Salida de la Construcción

El directorio dist/ contiene los siguientes archivos:

  • index.html: El archivo HTML principal.
  • main.js: El paquete principal de JavaScript.
  • polyfills.js: Polyfills para compatibilidad con navegadores.
  • styles.css: Los estilos globales para la aplicación.
  • vendor.js: Bibliotecas y dependencias de terceros.

Despliegue de la Aplicación

Hay varias formas de desplegar una aplicación Angular. Cubriremos el despliegue en plataformas populares como GitHub Pages, Firebase Hosting y un servidor personalizado.

Despliegue en GitHub Pages

  1. Instala el desplegador de GitHub Pages de Angular CLI: 
    npm install -g angular-cli-ghpages
  2. Construye la aplicación: 
    ng build --prod --base-href "https://<nombre-de-usuario>.github.io/<nombre-del-repositorio>/"
  3. Despliega la aplicación: 
    ngh --dir=dist/<nombre-del-proyecto>

Despliegue en Firebase Hosting

  1. Instala Firebase CLI: 
    npm install -g firebase-tools
  2. Inicia sesión en Firebase: 
    firebase login
  3. Inicializa Firebase en tu proyecto: 
    firebase init
  4. Construye la aplicación: 
    ng build --prod
  5. Despliega la aplicación: 
    firebase deploy

Despliegue en un Servidor Personalizado

  1. Construye la aplicación: 
    ng build --prod
  2. Copia el contenido del directorio dist/ al directorio público de tu servidor.
  3. Configura tu servidor para servir el archivo index.html para todas las rutas.

Conclusión

En este tema, cubrimos los pasos esenciales para construir y desplegar aplicaciones Angular. Comenzamos configurando el entorno de desarrollo, creando una nueva aplicación Angular y construyéndola para producción. Finalmente, exploramos diferentes opciones de despliegue, incluyendo GitHub Pages, Firebase Hosting y servidores personalizados. Siguiendo estos pasos, puedes asegurarte de que tus aplicaciones Angular estén optimizadas y listas para el despliegue en producción.
