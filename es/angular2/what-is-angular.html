Angular es una plataforma y un framework para construir aplicaciones cliente de una sola página usando HTML y TypeScript. Desarrollado y mantenido por Google, Angular es una reescritura completa del mismo equipo que construyó AngularJS. Angular 2+ se refiere a todas las versiones de Angular a partir de la versión 2 en adelante.

Características Clave de Angular

  • Arquitectura Basada en Componentes: Las aplicaciones de Angular se construyen usando componentes, que son los bloques fundamentales de la interfaz de usuario.
  • TypeScript: Angular está escrito en TypeScript, que es un superconjunto de JavaScript que añade tipos estáticos.
  • Inyección de Dependencias: Angular tiene un poderoso sistema de inyección de dependencias que facilita la gestión e inyección de dependencias.
  • Programación Reactiva con RxJS: Angular usa RxJS para manejar operaciones asíncronas y programas basados en eventos.
  • Desarrollo Modular: Las aplicaciones de Angular son modulares, lo que permite a los desarrolladores dividir la aplicación en piezas más pequeñas y reutilizables.
  • Enlace de Datos Bidireccional: Angular soporta el enlace de datos bidireccional, lo que significa que cualquier cambio en el modelo se refleja en la vista y viceversa.
  • Enrutamiento: Angular proporciona un robusto mecanismo de enrutamiento para navegar entre diferentes vistas o componentes.

Conceptos Básicos en Angular

Componentes

Los componentes son los bloques de construcción de una aplicación Angular. Cada componente consta de:

  • Plantilla: Define la vista para el componente usando HTML.
  • Clase: Contiene la lógica y los datos para el componente, escrita en TypeScript.
  • Estilos: Define los estilos CSS para el componente.

Ejemplo de un componente simple en Angular:

import { Component } from '@angular/core';

@Component({
  selector: 'app-hello-world',
  template: `<h1>Hello, World!</h1>`,
  styles: [`h1 { font-family: Lato; }`]
})
export class HelloWorldComponent {
}

Módulos

Los módulos se usan para agrupar componentes, directivas, pipes y servicios relacionados. Cada aplicación Angular tiene al menos un módulo, el módulo raíz.

Ejemplo de un módulo básico en Angular:

import { NgModule } from '@angular/core';
import { BrowserModule } from '@angular/platform-browser';
import { AppComponent } from './app.component';
import { HelloWorldComponent } from './hello-world.component';

@NgModule({
  declarations: [
    AppComponent,
    HelloWorldComponent
  ],
  imports: [
    BrowserModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})
export class AppModule { }

Servicios e Inyección de Dependencias

Los servicios se usan para compartir datos y lógica entre múltiples componentes. El sistema de inyección de dependencias de Angular facilita la inyección de servicios en componentes.

Ejemplo de un servicio simple:

import { Injectable } from '@angular/core';

@Injectable({
  providedIn: 'root',
})
export class DataService {
  private data: string[] = [];

  addData(item: string) {
    this.data.push(item);
  }

  getData(): string[] {
    return this.data;
  }
}

Inyectando el servicio en un componente:

import { Component } from '@angular/core';
import { DataService } from './data.service';

@Component({
  selector: 'app-data',
  template: `
    <div>
      <input #newItem>
      <button (click)="addItem(newItem.value)">Add</button>
      <ul>
        <li *ngFor="let item of items">{{ item }}</li>
      </ul>
    </div>
  `,
})
export class DataComponent {
  items: string[] = [];

  constructor(private dataService: DataService) {}

  addItem(item: string) {
    this.dataService.addData(item);
    this.items = this.dataService.getData();
  }
}

Conclusión

Angular es un framework poderoso y versátil para construir aplicaciones web modernas. Su arquitectura basada en componentes, uso de TypeScript y características robustas como la inyección de dependencias y la programación reactiva lo hacen una elección popular entre los desarrolladores. Entender los conceptos básicos de Angular, como componentes, módulos y servicios, proporciona una base sólida para construir aplicaciones escalables y mantenibles. A medida que avances en este curso, obtendrás una comprensión más profunda de las características avanzadas de Angular y las mejores prácticas.
