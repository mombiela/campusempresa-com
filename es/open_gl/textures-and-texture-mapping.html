Introducción a las Texturas

Las texturas son imágenes aplicadas a las superficies de formas o modelos 3D para darles más detalle y realismo. En OpenGL, las texturas se pueden usar para agregar color, patrones u otros efectos visuales a los objetos.

Conceptos Clave

  • Imagen de Textura: El archivo de imagen (por ejemplo, PNG, JPEG) que se utilizará como textura.
  • Coordenadas de Textura: Coordenadas que mapean la imagen de textura a la geometría.
  • Filtrado de Textura: Técnicas para determinar cómo los píxeles de textura (texels) se mapean a los píxeles de la pantalla.
  • Mipmaps: Secuencias de imágenes pre-calculadas y optimizadas que acompañan a una textura principal, utilizadas para aumentar la velocidad de renderizado y reducir artefactos.

Cargando Texturas

Para usar texturas en OpenGL, primero necesitas cargar la imagen de textura en tu programa.

Ejemplo: Cargando una Textura

GLuint textura;
glGenTextures(1, &textura);
glBindTexture(GL_TEXTURE_2D, textura);

// Cargar imagen, crear textura y generar mipmaps
int ancho, alto, nrCanales;
unsigned char *data = stbi_load("ruta/a/tu/imagen.jpg", &ancho, &alto, &nrCanales, 0);
if (data)
{
    glTexImage2D(GL_TEXTURE_2D, 0, GL_RGB, ancho, alto, 0, GL_RGB, GL_UNSIGNED_BYTE, data);
    glGenerateMipmap(GL_TEXTURE_2D);
}
else
{
    std::cout << "Error al cargar la textura" << std::endl;
}
stbi_image_free(data);
  • stbi_load: Función de la biblioteca STB image para cargar imágenes.
  • glTexImage2D: Especifica una imagen de textura bidimensional.
  • glGenerateMipmap: Genera mipmaps para la textura especificada.

Coordenadas de Textura

Las coordenadas de textura van de (0,0) a (1,1), donde (0,0) es la esquina inferior izquierda y (1,1) es la esquina superior derecha de la textura.

Ejemplo: Aplicando Coordenadas de Textura

float vertices[] = {
    // posiciones          // coordenadas de textura
     0.5f,  0.5f, 0.0f,   1.0f, 1.0f, // arriba derecha
     0.5f, -0.5f, 0.0f,   1.0f, 0.0f, // abajo derecha
    -0.5f, -0.5f, 0.0f,   0.0f, 0.0f, // abajo izquierda
    -0.5f,  0.5f, 0.0f,   0.0f, 1.0f  // arriba izquierda 
};
  • Coordenadas de Textura: Los dos últimos valores en cada vértice especifican las coordenadas de textura.

Filtrado de Textura

El filtrado de textura determina cómo se muestrea una textura cuando se escala hacia arriba o hacia abajo.

Tipos de Filtrado de Textura

  • Filtrado de Vecino Más Cercano: Elige el texel más cercano.
  • Filtrado Lineal: Interpola entre múltiples texels.

Ejemplo: Configurando el Filtrado de Textura

glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);
glTexParameteri(GL_TEXTURE_2D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);
  • GL_TEXTURE_MIN_FILTER: Filtro usado cuando la textura se minimiza.
  • GL_TEXTURE_MAG_FILTER: Filtro usado cuando la textura se magnifica.

Mipmaps

Los mipmaps son una serie de texturas pre-calculadas, cada una de las cuales es una representación de resolución progresivamente más baja de la misma imagen.

Ejemplo: Generando Mipmaps

glGenerateMipmap(GL_TEXTURE_2D);
  • glGenerateMipmap: Genera automáticamente todos los mipmaps necesarios para la textura actualmente vinculada.

Técnicas Avanzadas de Mapeo de Texturas

A medida que avances, puedes explorar técnicas de mapeo de texturas más complejas como:

  • Multi-texturizado: Usar múltiples texturas en un solo objeto.
  • Mapeo de Normales: Agregar más detalle a las superficies simulando pequeños bultos y abolladuras.
  • Mapeo de Cubo: Usar una textura de cubo para crear reflejos.

Ejemplo: Multi-texturizado

glActiveTexture(GL_TEXTURE0);
glBindTexture(GL_TEXTURE_2D, textura1);
glUniform1i(glGetUniformLocation(shaderProgram, "textura1"), 0);

glActiveTexture(GL_TEXTURE1);
glBindTexture(GL_TEXTURE_2D, textura2);
glUniform1i(glGetUniformLocation(shaderProgram, "textura2"), 1);
  • glActiveTexture: Selecciona qué unidad de textura activar.
  • glUniform1i: Establece el muestreador de textura en la unidad de textura correcta.

Conclusión

Las texturas y el mapeo de texturas son conceptos fundamentales en OpenGL que mejoran significativamente la calidad visual de los objetos renderizados. Al entender cómo cargar texturas, aplicar coordenadas de textura y usar filtrado de texturas y mipmaps, puedes crear gráficos más realistas y visualmente atractivos. A medida que avances, explorar técnicas como el multi-texturizado y el mapeo de normales ampliará aún más tu capacidad para crear escenas complejas y detalladas.
