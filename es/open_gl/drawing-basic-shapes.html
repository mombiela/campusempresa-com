Introducción

En esta sección, exploraremos cómo dibujar formas básicas usando OpenGL. Esto es fundamental para cualquiera que comience con OpenGL, ya que sienta las bases para una programación gráfica más compleja. Cubriremos las siguientes formas:

  • Puntos
  • Líneas
  • Triángulos
  • Cuadriláteros (Quads)

Configuración de OpenGL

Antes de comenzar a dibujar formas, asegúrate de tener una configuración básica de OpenGL. Esto incluye inicializar el contexto de OpenGL, crear una ventana y configurar un bucle de renderizado. Aquí hay una configuración mínima usando GLFW y GLEW:

#include <GL/glew.h>
#include <GLFW/glfw3.h>

int main() {
    // Inicializar GLFW
    if (!glfwInit()) {
        return -1;
    }

    // Crear una ventana en modo ventana y su contexto OpenGL
    GLFWwindow* window = glfwCreateWindow(640, 480, "Configuración de OpenGL", NULL, NULL);
    if (!window) {
        glfwTerminate();
        return -1;
    }

    // Hacer el contexto de la ventana actual
    glfwMakeContextCurrent(window);

    // Inicializar GLEW
    if (glewInit() != GLEW_OK) {
        return -1;
    }

    // Bucle principal
    while (!glfwWindowShouldClose(window)) {
        // Renderizar aquí

        // Intercambiar los buffers delantero y trasero
        glfwSwapBuffers(window);

        // Sondear y procesar eventos
        glfwPollEvents();
    }

    glfwTerminate();
    return 0;
}

Dibujando Puntos

Los puntos son el primitivo más simple en OpenGL. Para dibujar un punto, necesitas especificar sus coordenadas.

Ejemplo

void drawPoint() {
    glBegin(GL_POINTS); // Comenzar a dibujar puntos
    glVertex2f(0.0f, 0.0f); // Especificar las coordenadas del punto
    glEnd(); // Terminar de dibujar puntos
}

Explicación

  • glBegin(GL_POINTS); le dice a OpenGL que estamos comenzando a dibujar puntos.
  • glVertex2f(0.0f, 0.0f); especifica las coordenadas del punto.
  • glEnd(); termina el proceso de dibujo.

Dibujando Líneas

Las líneas se definen por dos puntos. OpenGL proporciona varios tipos de líneas, pero comenzaremos con la línea básica.

Ejemplo

void drawLine() {
    glBegin(GL_LINES); // Comenzar a dibujar líneas
    glVertex2f(-0.5f, 0.0f); // Primer punto
    glVertex2f(0.5f, 0.0f); // Segundo punto
    glEnd(); // Terminar de dibujar líneas
}

Explicación

  • glBegin(GL_LINES); le dice a OpenGL que estamos comenzando a dibujar líneas.
  • glVertex2f(-0.5f, 0.0f); especifica el primer punto de la línea.
  • glVertex2f(0.5f, 0.0f); especifica el segundo punto de la línea.
  • glEnd(); termina el proceso de dibujo.

Dibujando Triángulos

Los triángulos son uno de los primitivos más importantes en OpenGL, ya que se utilizan para construir formas más complejas.

Ejemplo

void drawTriangle() {
    glBegin(GL_TRIANGLES); // Comenzar a dibujar triángulos
    glVertex2f(0.0f, 0.5f); // Primer vértice
    glVertex2f(-0.5f, -0.5f); // Segundo vértice
    glVertex2f(0.5f, -0.5f); // Tercer vértice
    glEnd(); // Terminar de dibujar triángulos
}

Explicación

  • glBegin(GL_TRIANGLES); le dice a OpenGL que estamos comenzando a dibujar triángulos.
  • glVertex2f(0.0f, 0.5f); especifica el primer vértice del triángulo.
  • glVertex2f(-0.5f, -0.5f); especifica el segundo vértice del triángulo.
  • glVertex2f(0.5f, -0.5f); especifica el tercer vértice del triángulo.
  • glEnd(); termina el proceso de dibujo.

Dibujando Cuadriláteros

Los cuadriláteros se definen por cuatro vértices. Sin embargo, OpenGL moderno ha deprecado los cuadriláteros en favor de usar dos triángulos para formar un cuadrilátero.

Ejemplo

void drawQuad() {
    glBegin(GL_QUADS); // Comenzar a dibujar cuadriláteros
    glVertex2f(-0.5f, 0.5f); // Primer vértice
    glVertex2f(0.5f, 0.5f); // Segundo vértice
    glVertex2f(0.5f, -0.5f); // Tercer vértice
    glVertex2f(-0.5f, -0.5f); // Cuarto vértice
    glEnd(); // Terminar de dibujar cuadriláteros
}

Explicación

  • glBegin(GL_QUADS); le dice a OpenGL que estamos comenzando a dibujar cuadriláteros.
  • glVertex2f(-0.5f, 0.5f); especifica el primer vértice del cuadrilátero.
  • glVertex2f(0.5f, 0.5f); especifica el segundo vértice del cuadrilátero.
  • glVertex2f(0.5f, -0.5f); especifica el tercer vértice del cuadrilátero.
  • glVertex2f(-0.5f, -0.5f); especifica el cuarto vértice del cuadrilátero.
  • glEnd(); termina el proceso de dibujo.

Conclusión

En esta sección, cubrimos lo básico de dibujar formas en OpenGL, incluyendo puntos, líneas, triángulos y cuadriláteros. Entender estos primitivos es crucial ya que forman la base para una programación gráfica más compleja. Practica dibujando estas formas y experimenta con diferentes coordenadas para familiarizarte con el sistema de coordenadas y el pipeline de renderizado de OpenGL.
