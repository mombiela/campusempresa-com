Introducción

Entender las coordenadas y transformaciones es fundamental para trabajar con OpenGL. Este tema cubrirá los conceptos básicos de los sistemas de coordenadas, transformaciones y cómo aplicarlas en OpenGL. Al final de esta sección, podrás manipular objetos en el espacio 3D de manera efectiva.

Sistemas de Coordenadas

En OpenGL, se utilizan varios sistemas de coordenadas para transformar vértices desde el espacio del objeto hasta el espacio de la pantalla.

  • Coordenadas del Objeto: Las coordenadas locales de un objeto.
  • Coordenadas del Mundo: Las coordenadas de un objeto en el espacio del mundo.
  • Coordenadas de Vista: Las coordenadas desde la perspectiva de la cámara.
  • Coordenadas de Recorte: Las coordenadas después de aplicar la matriz de proyección.
  • Coordenadas Normalizadas del Dispositivo (NDC): Las coordenadas después de la división de perspectiva.
  • Coordenadas de Ventana: Las coordenadas finales en la pantalla.

Ejemplo

// Vértice en coordenadas del objeto
vec3 objectVertex = vec3(1.0, 1.0, 1.0);

// La matriz de modelo transforma las coordenadas del objeto a coordenadas del mundo
mat4 model = mat4(1.0);
vec4 worldVertex = model * vec4(objectVertex, 1.0);

// La matriz de vista transforma las coordenadas del mundo a coordenadas de vista
mat4 view = lookAt(cameraPos, cameraTarget, upVector);
vec4 viewVertex = view * worldVertex;

// La matriz de proyección transforma las coordenadas de vista a coordenadas de recorte
mat4 projection = perspective(radians(45.0f), aspectRatio, 0.1f, 100.0f);
vec4 clipVertex = projection * viewVertex;

// División de perspectiva para obtener NDC
vec3 ndcVertex = clipVertex.xyz / clipVertex.w;

// Transformación del viewport para obtener coordenadas de ventana
vec2 windowVertex = (ndcVertex.xy + 1.0) / 2.0 * vec2(windowWidth, windowHeight);

Transformaciones

Las transformaciones se utilizan para mover, rotar y escalar objetos en OpenGL. Las transformaciones principales son:

  • Traslación: Mover un objeto en el espacio 3D.
  • Rotación: Rotar un objeto alrededor de un eje.
  • Escalado: Cambiar el tamaño de un objeto.

Traslación

La traslación mueve un objeto por un cierto desplazamiento.

mat4 translation = mat4(1.0);
translation = translate(translation, vec3(1.0, 2.0, 3.0));

Rotación

La rotación rota un objeto alrededor de un eje especificado por un cierto ángulo.

mat4 rotation = mat4(1.0);
rotation = rotate(rotation, radians(45.0f), vec3(0.0, 1.0, 0.0));

Escalado

El escalado cambia el tamaño de un objeto.

mat4 scaling = mat4(1.0);
scaling = scale(scaling, vec3(2.0, 2.0, 2.0));

Combinando Transformaciones

Las transformaciones se pueden combinar multiplicando sus matrices. El orden de la multiplicación es crucial.

Ejemplo

mat4 model = mat4(1.0);
model = translate(model, vec3(1.0, 2.0, 3.0)); // Traslación
model = rotate(model, radians(45.0f), vec3(0.0, 1.0, 0.0)); // Rotación
model = scale(model, vec3(2.0, 2.0, 2.0)); // Escalado

Ejemplo Práctico

Vamos a crear un ejemplo simple donde aplicamos estas transformaciones a un cubo.

Vertex Shader

#version 330 core
layout (location = 0) in vec3 aPos;

uniform mat4 model;
uniform mat4 view;
uniform mat4 projection;

void main()
{
    gl_Position = projection * view * model * vec4(aPos, 1.0);
}

Fragment Shader

#version 330 core
out vec4 FragColor;

void main()
{
    FragColor = vec4(1.0, 0.5, 0.2, 1.0);
}

Programa Principal

// Configurar datos de vértices (y buffer(s)) y configurar atributos de vértices
float vertices[] = {
    // posiciones          
    -0.5f, -0.5f, -0.5f,  
     0.5f, -0.5f, -0.5f,  
     0.5f,  0.5f, -0.5f,  
     0.5f,  0.5f, -0.5f,  
    -0.5f,  0.5f, -0.5f,  
    -0.5f, -0.5f, -0.5f,  

    // ... (otros vértices)
};

// Inicializar buffers y configurar atributos de vértices
unsigned int VBO, VAO;
glGenVertexArrays(1, &VAO);
glGenBuffers(1, &VBO);

glBindVertexArray(VAO);

glBindBuffer(GL_ARRAY_BUFFER, VBO);
glBufferData(GL_ARRAY_BUFFER, sizeof(vertices), vertices, GL_STATIC_DRAW);

glVertexAttribPointer(0, 3, GL_FLOAT, GL_FALSE, 3 * sizeof(float), (void*)0);
glEnableVertexAttribArray(0);

// Bucle principal de renderizado
while (!glfwWindowShouldClose(window))
{
    // Manejo de entrada
    processInput(window);

    // Renderizado
    glClear(GL_COLOR_BUFFER_BIT | GL_DEPTH_BUFFER_BIT);

    // Activar shader
    ourShader.use();

    // Crear transformaciones
    mat4 model = mat4(1.0f);
    model = translate(model, vec3(0.0f, 0.0f, -3.0f));
    model = rotate(model, (float)glfwGetTime(), vec3(0.5f, 1.0f, 0.0f));
    ourShader.setMat4("model", model);

    mat4 view = mat4(1.0f);
    view = translate(view, vec3(0.0f, 0.0f, -5.0f));
    ourShader.setMat4("view", view);

    mat4 projection = perspective(radians(45.0f), (float)SCR_WIDTH / (float)SCR_HEIGHT, 0.1f, 100.0f);
    ourShader.setMat4("projection", projection);

    // Renderizar contenedor
    glBindVertexArray(VAO);
    glDrawArrays(GL_TRIANGLES, 0, 36);

    // Intercambiar buffers y sondear eventos de IO
    glfwSwapBuffers(window);
    glfwPollEvents();
}

// Desalojar recursos
glDeleteVertexArrays(1, &VAO);
glDeleteBuffers(1, &VBO);

glfwTerminate();

Conclusión

Entender las coordenadas y transformaciones es esencial para la programación de gráficos 3D en OpenGL. Al dominar estos conceptos, puedes controlar la posición, orientación y escala de los objetos en tu escena. Practica combinando diferentes transformaciones para ver sus efectos y solidificar tu comprensión.
