Introducción a las Colecciones

Las colecciones en PL/SQL son tipos de datos compuestos que te permiten almacenar y manipular conjuntos de elementos. Son similares a los arrays en otros lenguajes de programación, pero ofrecen más flexibilidad y funcionalidad.

Tipos de Colecciones

PL/SQL soporta tres tipos principales de colecciones:

  • Arrays Asociativos (Tablas Indexadas)
  • Tablas Anidadas
  • VARRAYs (Arrays de Tamaño Variable)

Arrays Asociativos (Tablas Indexadas)

Los arrays asociativos, también conocidos como tablas indexadas, son colecciones que usan números arbitrarios y cadenas como claves.

Sintaxis y Ejemplo

DECLARE
  TYPE assoc_array IS TABLE OF VARCHAR2(100) INDEX BY PLS_INTEGER;
  my_array assoc_array;
BEGIN
  my_array(1) := 'Primer Elemento';
  my_array(2) := 'Segundo Elemento';
  DBMS_OUTPUT.PUT_LINE(my_array(1)); -- Salida: Primer Elemento
  DBMS_OUTPUT.PUT_LINE(my_array(2)); -- Salida: Segundo Elemento
END;

Conceptos Clave

  • Tamaño Dinámico: Los arrays asociativos pueden crecer dinámicamente.
  • Indexación: Pueden ser indexados por enteros o cadenas.
  • Dispersos: Los elementos no necesitan ser contiguos.

Tablas Anidadas

Las tablas anidadas son colecciones que pueden ser almacenadas en tablas de bases de datos y pueden ser manipuladas como conjuntos completos.

Sintaxis y Ejemplo

DECLARE
  TYPE nested_table IS TABLE OF VARCHAR2(100);
  my_table nested_table := nested_table('Elemento 1', 'Elemento 2', 'Elemento 3');
BEGIN
  FOR i IN 1..my_table.COUNT LOOP
    DBMS_OUTPUT.PUT_LINE(my_table(i));
  END LOOP;
END;

Conceptos Clave

  • Tamaño Dinámico: Las tablas anidadas pueden crecer dinámicamente.
  • Densas: Inicialmente densas pero pueden volverse dispersas después de eliminaciones.
  • Almacenamiento: Pueden ser almacenadas en columnas de bases de datos.

VARRAYs (Arrays de Tamaño Variable)

Los VARRAYs son colecciones con un tamaño máximo fijo, lo que los hace más eficientes para ciertas operaciones.

Sintaxis y Ejemplo

DECLARE
  TYPE varray_type IS VARRAY(5) OF VARCHAR2(100);
  my_varray varray_type := varray_type('Elemento 1', 'Elemento 2');
BEGIN
  FOR i IN 1..my_varray.COUNT LOOP
    DBMS_OUTPUT.PUT_LINE(my_varray(i));
  END LOOP;
END;

Conceptos Clave

  • Tamaño Fijo: El tamaño máximo se define en el momento de la declaración.
  • Densas: Siempre densas, sin huecos entre elementos.
  • Almacenamiento: Pueden ser almacenadas en columnas de bases de datos.

Comparación de Tipos de Colecciones

| Característica | Arrays Asociativos | Tablas Anidadas | VARRAYs | |-------------------|--------------------------|------------------------|-----------------------| | Indexación | Arbitraria (entero, cadena) | Secuencial (entero) | Secuencial (entero) | | Tamaño Dinámico | Sí | Sí | No (tamaño máximo fijo) | | Almacenamiento en BD | No | Sí | Sí | | Dispersión | Sí | Sí (después de eliminaciones) | No |

Uso Avanzado

Operaciones en Bloque

Las colecciones en PL/SQL soportan operaciones en bloque que pueden mejorar significativamente el rendimiento.

Ejemplo: Bulk Collect

DECLARE
  TYPE number_table IS TABLE OF NUMBER;
  my_numbers number_table;
BEGIN
  SELECT employee_id BULK COLLECT INTO my_numbers FROM employees;
  FOR i IN 1..my_numbers.COUNT LOOP
    DBMS_OUTPUT.PUT_LINE(my_numbers(i));
  END LOOP;
END;

Ejemplo: FORALL

DECLARE
  TYPE number_table IS TABLE OF NUMBER;
  my_numbers number_table := number_table(1, 2, 3, 4, 5);
BEGIN
  FORALL i IN 1..my_numbers.COUNT
    INSERT INTO my_table (column_name) VALUES (my_numbers(i));
END;

Métodos de Colección

PL/SQL proporciona varios métodos integrados para manipular colecciones:

  • COUNT: Devuelve el número de elementos.
  • EXISTS(n): Verifica si el elemento n existe.
  • FIRST: Devuelve el primer índice.
  • LAST: Devuelve el último índice.
  • PRIOR(n): Devuelve el índice anterior a n.
  • NEXT(n): Devuelve el índice siguiente a n.
  • EXTEND: Añade elementos a la colección.
  • TRIM: Elimina elementos de la colección.
  • DELETE: Elimina todos o elementos específicos.

Ejemplo: Uso de Métodos de Colección

DECLARE
  TYPE nested_table IS TABLE OF VARCHAR2(100);
  my_table nested_table := nested_table('Elemento 1', 'Elemento 2', 'Elemento 3');
BEGIN
  DBMS_OUTPUT.PUT_LINE('Count: ' || my_table.COUNT); -- Salida: Count: 3
  my_table.DELETE(2); -- Elimina el segundo elemento
  DBMS_OUTPUT.PUT_LINE('Count after delete: ' || my_table.COUNT); -- Salida: Count after delete: 2
  my_table.EXTEND(2); -- Añade dos elementos más
  DBMS_OUTPUT.PUT_LINE('Count after extend: ' || my_table.COUNT); -- Salida: Count after extend: 4
END;

Conclusión

Las colecciones en PL/SQL son herramientas poderosas que te permiten manejar conjuntos de datos de manera eficiente. Entender los diferentes tipos de colecciones—arrays asociativos, tablas anidadas y VARRAYs—junto con sus características y métodos únicos, te permitirá escribir código PL/SQL más flexible y eficiente. Utiliza operaciones en bloque y métodos de colección para optimizar tus tareas de procesamiento de datos.
