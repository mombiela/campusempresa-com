Introducción

En PL/SQL, las estructuras de control se utilizan para dictar el flujo de ejecución dentro de un programa. Una de las estructuras de control más fundamentales es la sentencia IF, que te permite ejecutar ciertas secciones de código basadas en condiciones específicas.

Conceptos Clave

  • Sentencia IF: Ejecuta un bloque de código si una condición especificada es verdadera.
  • Sentencia ELSIF: Proporciona condiciones adicionales si la condición inicial de IF es falsa.
  • Sentencia ELSE: Ejecuta un bloque de código si ninguna de las condiciones anteriores es verdadera.
  • Expresiones Booleanas: Condiciones utilizadas dentro de las sentencias IF que se evalúan como TRUE, FALSE o NULL.

Sentencia IF Básica

La forma más simple de una sentencia IF verifica una sola condición y ejecuta un bloque de código si la condición es verdadera.

Sintaxis

IF condition THEN
    -- sentencias a ejecutar si la condición es verdadera
END IF;

Ejemplo

DECLARE
    v_salary NUMBER := 5000;
BEGIN
    IF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('El salario es mayor que 3000');
    END IF;
END;

En este ejemplo, se imprimirá el mensaje "El salario es mayor que 3000" porque la condición v_salary > 3000 es verdadera.

Sentencia IF-ELSE

Una sentencia IF-ELSE te permite ejecutar un bloque de código si la condición es verdadera y otro bloque si la condición es falsa.

Sintaxis

IF condition THEN
    -- sentencias a ejecutar si la condición es verdadera
ELSE
    -- sentencias a ejecutar si la condición es falsa
END IF;

Ejemplo

DECLARE
    v_salary NUMBER := 2000;
BEGIN
    IF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('El salario es mayor que 3000');
    ELSE
        DBMS_OUTPUT.PUT_LINE('El salario es 3000 o menos');
    END IF;
END;

En este ejemplo, se imprimirá el mensaje "El salario es 3000 o menos" porque la condición v_salary > 3000 es falsa.

Sentencia IF-ELSIF-ELSE

La sentencia IF-ELSIF-ELSE te permite verificar múltiples condiciones secuencialmente. Si una condición es verdadera, se ejecuta su correspondiente bloque de código y se omiten las demás.

Sintaxis

IF condition1 THEN
    -- sentencias a ejecutar si condition1 es verdadera
ELSIF condition2 THEN
    -- sentencias a ejecutar si condition2 es verdadera
ELSE
    -- sentencias a ejecutar si ninguna de las condiciones anteriores es verdadera
END IF;

Ejemplo

DECLARE
    v_salary NUMBER := 4000;
BEGIN
    IF v_salary > 5000 THEN
        DBMS_OUTPUT.PUT_LINE('El salario es mayor que 5000');
    ELSIF v_salary > 3000 THEN
        DBMS_OUTPUT.PUT_LINE('El salario es mayor que 3000 pero menor o igual a 5000');
    ELSE
        DBMS_OUTPUT.PUT_LINE('El salario es 3000 o menos');
    END IF;
END;

En este ejemplo, se imprimirá el mensaje "El salario es mayor que 3000 pero menor o igual a 5000" porque la condición v_salary > 3000 es verdadera.

Sentencias IF Anidadas

Puedes anidar sentencias IF dentro de otras para crear una lógica condicional más compleja.

Ejemplo

DECLARE
    v_salary NUMBER := 4500;
    v_bonus NUMBER := 500;
BEGIN
    IF v_salary > 3000 THEN
        IF v_bonus > 0 THEN
            DBMS_OUTPUT.PUT_LINE('El salario es mayor que 3000 y el bono es positivo');
        ELSE
            DBMS_OUTPUT.PUT_LINE('El salario es mayor que 3000 pero no hay bono');
        END IF;
    ELSE
        DBMS_OUTPUT.PUT_LINE('El salario es 3000 o menos');
    END IF;
END;

En este ejemplo, se imprimirá el mensaje "El salario es mayor que 3000 y el bono es positivo" porque ambas condiciones v_salary > 3000 y v_bonus > 0 son verdaderas.

Conclusión

Las sentencias IF son una parte fundamental de la programación en PL/SQL, permitiéndote controlar el flujo de ejecución basado en condiciones. Al entender y usar las sentencias IF, IF-ELSE, IF-ELSIF-ELSE y las sentencias IF anidadas, puedes crear programas PL/SQL más dinámicos y receptivos. Practica escribiendo estas sentencias con diferentes condiciones para volverte competente en el control de la lógica de tu código PL/SQL.
