Introducción

En PL/SQL, las operaciones del Lenguaje de Manipulación de Datos (DML) son esenciales para gestionar y manipular datos dentro de las tablas de la base de datos. Las principales operaciones DML incluyen INSERT, UPDATE y DELETE. Esta sección cubrirá estas operaciones en detalle, proporcionando explicaciones claras, ejemplos y ejercicios para ayudarte a entender y aplicar estos conceptos de manera efectiva.

Operación INSERT

La declaración INSERT se utiliza para agregar nuevas filas a una tabla.

Sintaxis

INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);

Ejemplo

INSERT INTO employees (employee_id, first_name, last_name, email, hire_date, job_id)
VALUES (101, 'John', 'Doe', 'john.doe@example.com', '2023-01-15', 'IT_PROG');

Explicación

  • employees: El nombre de la tabla donde se insertarán los datos.
  • (employee_id, first_name, last_name, email, hire_date, job_id): Las columnas en la tabla donde se insertarán los valores.
  • VALUES (101, 'John', 'Doe', 'john.doe@example.com', '2023-01-15', 'IT_PROG'): Los valores que se insertarán en las columnas especificadas.

Ejercicio

  1. Inserta una nueva fila en la tabla departments con los siguientes valores:
    • department_id: 10
    • department_name: 'Finance'
    • manager_id: 205
    • location_id: 1700

Operación UPDATE

La declaración UPDATE se utiliza para modificar filas existentes en una tabla.

Sintaxis

UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

Ejemplo

UPDATE employees
SET email = 'john.doe@newexample.com', job_id = 'SA_REP'
WHERE employee_id = 101;

Explicación

  • employees: El nombre de la tabla donde se actualizarán los datos.
  • SET email = 'john.doe@newexample.com', job_id = 'SA_REP': Las columnas y sus nuevos valores.
  • WHERE employee_id = 101: La condición para identificar qué fila(s) actualizar.

Ejercicio

  1. Actualiza la tabla departments para cambiar el manager_id a 206 para el departamento con department_id 10.

Operación DELETE

La declaración DELETE se utiliza para eliminar filas existentes de una tabla.

Sintaxis

DELETE FROM table_name
WHERE condition;

Ejemplo

DELETE FROM employees
WHERE employee_id = 101;

Explicación

  • employees: El nombre de la tabla de la cual se eliminarán los datos.
  • WHERE employee_id = 101: La condición para identificar qué fila(s) eliminar.

Ejercicio

  1. Elimina la fila de la tabla departments donde el department_id es 10.

Resumen

En esta sección, cubrimos las operaciones básicas de DML en PL/SQL: INSERT, UPDATE y DELETE. Estas operaciones te permiten agregar, modificar y eliminar datos dentro de tus tablas de la base de datos. Entender y dominar estas operaciones es fundamental para gestionar datos de manera efectiva en cualquier aplicación de base de datos.

  • INSERT: Agrega nuevas filas a una tabla.
  • UPDATE: Modifica filas existentes en una tabla.
  • DELETE: Elimina filas existentes de una tabla.

Practica estas operaciones con los ejercicios proporcionados para reforzar tu comprensión y ganar confianza en el uso de declaraciones DML en PL/SQL.
