Introducción

En esta sección, exploraremos los conceptos fundamentales de bases de datos y tablas en SQL. Entender estos conceptos es crucial para cualquiera que desee trabajar con SQL, ya que forman la columna vertebral del almacenamiento y manipulación de datos.

¿Qué es una Base de Datos?

Una base de datos es una colección organizada de datos, generalmente almacenados y accesados electrónicamente desde un sistema informático. Las bases de datos se utilizan para almacenar, recuperar y gestionar datos de manera eficiente.

Conceptos Clave:

  • Sistema de Gestión de Bases de Datos (DBMS): Software que interactúa con el usuario, aplicaciones y la propia base de datos para capturar y analizar datos.
  • SQL (Lenguaje de Consulta Estructurado): Un lenguaje de programación estándar diseñado específicamente para gestionar y manipular bases de datos.

¿Qué es una Tabla?

Una tabla es una colección de datos relacionados mantenidos en un formato estructurado dentro de una base de datos. Consiste en filas y columnas.

Conceptos Clave:

  • Filas (Registros): Cada fila en una tabla representa un único elemento de datos estructurado implícitamente.
  • Columnas (Campos): Cada columna representa un atributo específico de los datos.

Ejemplo de Estructura de Tabla:

| ID | Nombre | Edad | Correo Electrónico | |-----|-------------|------|----------------------| | 1 | John Doe | 30 | john.doe@example.com | | 2 | Jane Smith | 25 | jane.smith@example.com|

Creando una Base de Datos

Para crear una base de datos, se utiliza la sentencia CREATE DATABASE.

Ejemplo:

CREATE DATABASE mi_base_de_datos;

Este comando crea una nueva base de datos llamada mi_base_de_datos.

Creando una Tabla

Para crear una tabla, se utiliza la sentencia CREATE TABLE.

Ejemplo:

CREATE TABLE usuarios (
    ID INT PRIMARY KEY,
    Nombre VARCHAR(100),
    Edad INT,
    Correo VARCHAR(100)
);

Este comando crea una tabla llamada usuarios con cuatro columnas: ID, Nombre, Edad y Correo.

Insertando Datos en una Tabla

Para insertar datos en una tabla, se utiliza la sentencia INSERT INTO.

Ejemplo:

INSERT INTO usuarios (ID, Nombre, Edad, Correo) VALUES (1, 'John Doe', 30, 'john.doe@example.com');
INSERT INTO usuarios (ID, Nombre, Edad, Correo) VALUES (2, 'Jane Smith', 25, 'jane.smith@example.com');

Estos comandos insertan dos registros en la tabla usuarios.

Consultando Datos de una Tabla

Para recuperar datos de una tabla, se utiliza la sentencia SELECT.

Ejemplo:

SELECT * FROM usuarios;

Este comando recupera todos los registros de la tabla usuarios.

Filtrando Datos:

Puedes filtrar los datos usando la cláusula WHERE.

Ejemplo:

SELECT * FROM usuarios WHERE Edad > 25;

Este comando recupera registros de la tabla usuarios donde la Edad es mayor a 25.

Actualizando Datos en una Tabla

Para actualizar datos existentes en una tabla, se utiliza la sentencia UPDATE.

Ejemplo:

UPDATE usuarios SET Correo = 'john.newemail@example.com' WHERE ID = 1;

Este comando actualiza el correo del usuario con ID 1.

Eliminando Datos de una Tabla

Para eliminar datos de una tabla, se utiliza la sentencia DELETE.

Ejemplo:

DELETE FROM usuarios WHERE ID = 2;

Este comando elimina el registro del usuario con ID 2.

Conclusión

Entender bases de datos y tablas es fundamental para trabajar con SQL. Las bases de datos almacenan colecciones de datos, y las tablas organizan estos datos en filas y columnas. Dominar las operaciones básicas como crear bases de datos y tablas, insertar, consultar, actualizar y eliminar datos proporcionará una base sólida para temas más avanzados de SQL.
