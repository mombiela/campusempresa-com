Introducción

Las funciones agregadas en SQL se utilizan para realizar cálculos en múltiples filas de una columna de una tabla y devolver un solo valor. Son esenciales para el análisis de datos y la elaboración de informes. Las funciones agregadas más comúnmente utilizadas son COUNT, SUM, AVG, MAX y MIN.

COUNT

La función COUNT devuelve el número de filas que coinciden con una condición especificada.

Sintaxis

SELECT COUNT(column_name)
FROM table_name
WHERE condition;

Ejemplo

SELECT COUNT(employee_id)
FROM employees
WHERE department = 'Sales';

Esta consulta cuenta el número de empleados en el departamento de 'Ventas'.

SUM

La función SUM devuelve la suma total de una columna numérica.

Sintaxis

SELECT SUM(column_name)
FROM table_name
WHERE condition;

Ejemplo

SELECT SUM(salary)
FROM employees
WHERE department = 'Sales';

Esta consulta calcula el salario total de todos los empleados en el departamento de 'Ventas'.

AVG

La función AVG devuelve el valor promedio de una columna numérica.

Sintaxis

SELECT AVG(column_name)
FROM table_name
WHERE condition;

Ejemplo

SELECT AVG(salary)
FROM employees
WHERE department = 'Sales';

Esta consulta calcula el salario promedio de los empleados en el departamento de 'Ventas'.

MAX

La función MAX devuelve el valor máximo en un conjunto de valores.

Sintaxis

SELECT MAX(column_name)
FROM table_name
WHERE condition;

Ejemplo

SELECT MAX(salary)
FROM employees
WHERE department = 'Sales';

Esta consulta encuentra el salario más alto en el departamento de 'Ventas'.

MIN

La función MIN devuelve el valor mínimo en un conjunto de valores.

Sintaxis

SELECT MIN(column_name)
FROM table_name
WHERE condition;

Ejemplo

SELECT MIN(salary)
FROM employees
WHERE department = 'Sales';

Esta consulta encuentra el salario más bajo en el departamento de 'Ventas'.

Combinando Funciones Agregadas

Puedes usar múltiples funciones agregadas en una sola consulta para obtener un resumen completo de tus datos.

Ejemplo

SELECT 
    COUNT(employee_id) AS total_employees,
    SUM(salary) AS total_salary,
    AVG(salary) AS average_salary,
    MAX(salary) AS highest_salary,
    MIN(salary) AS lowest_salary
FROM employees
WHERE department = 'Sales';

Esta consulta proporciona un resumen del número de empleados, salario total, salario promedio, salario más alto y salario más bajo en el departamento de 'Ventas'.

Agrupando Datos con Funciones Agregadas

Para realizar cálculos agregados en subconjuntos de datos, puedes usar la cláusula GROUP BY.

Ejemplo

SELECT 
    department,
    COUNT(employee_id) AS total_employees,
    SUM(salary) AS total_salary,
    AVG(salary) AS average_salary,
    MAX(salary) AS highest_salary,
    MIN(salary) AS lowest_salary
FROM employees
GROUP BY department;

Esta consulta proporciona un resumen de las estadísticas de empleados para cada departamento.

Conclusión

Las funciones agregadas son herramientas poderosas para resumir y analizar datos en SQL. Al dominar COUNT, SUM, AVG, MAX y MIN, puedes realizar una amplia gama de tareas de análisis de datos. Recuerda usar la cláusula GROUP BY cuando necesites agregar datos en diferentes categorías. Practica estas funciones con varios conjuntos de datos para volverte competente en el análisis de datos usando SQL.
