Introducción a SQL

SQL, o Structured Query Language, es un lenguaje de programación estándar diseñado específicamente para gestionar y manipular bases de datos relacionales. Se utiliza para realizar diversas operaciones en los datos almacenados en una base de datos, como consultar, actualizar, insertar y eliminar datos.

Conceptos Clave de SQL

  • Base de Datos: Una colección estructurada de datos almacenados electrónicamente en un sistema informático.
  • Tabla: Una colección de entradas de datos relacionadas que consiste en columnas y filas.
  • Consulta: Una solicitud de datos o información de una tabla de base de datos o combinación de tablas.
  • Esquema: La estructura que define la organización de los datos en una base de datos, incluyendo tablas, columnas y relaciones.
  • Clave Primaria: Un identificador único para un registro en una tabla.
  • Clave Foránea: Un campo en una tabla que identifica de manera única una fila de otra tabla.

Comandos Básicos de SQL

Los comandos SQL se categorizan en varios tipos, incluyendo Lenguaje de Consulta de Datos (DQL), Lenguaje de Definición de Datos (DDL), Lenguaje de Manipulación de Datos (DML) y Lenguaje de Control de Datos (DCL).

Lenguaje de Consulta de Datos (DQL)

  • SELECT: Se utiliza para recuperar datos de una base de datos.
SELECT columna1, columna2
FROM nombre_tabla;

Ejemplo:

SELECT first_name, last_name
FROM employees;

Esta consulta recupera los nombres y apellidos de todos los empleados de la tabla employees.

Lenguaje de Definición de Datos (DDL)

  • CREATE: Se utiliza para crear una nueva tabla o base de datos.
CREATE TABLE nombre_tabla (
    columna1 tipo_dato,
    columna2 tipo_dato,
    ...
);

Ejemplo:

CREATE TABLE employees (
    employee_id INT PRIMARY KEY,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    hire_date DATE
);

Este comando crea una nueva tabla llamada employees con las columnas especificadas.

  • ALTER: Se utiliza para modificar un objeto de base de datos existente, como una tabla.
ALTER TABLE nombre_tabla
ADD nombre_columna tipo_dato;

Ejemplo:

ALTER TABLE employees
ADD email VARCHAR(100);

Este comando añade una nueva columna email a la tabla employees.

  • DROP: Se utiliza para eliminar una tabla o base de datos.
DROP TABLE nombre_tabla;

Ejemplo:

DROP TABLE employees;

Este comando elimina la tabla employees de la base de datos.

Lenguaje de Manipulación de Datos (DML)

  • INSERT: Se utiliza para insertar nuevos datos en una tabla.
INSERT INTO nombre_tabla (columna1, columna2, ...)
VALUES (valor1, valor2, ...);

Ejemplo:

INSERT INTO employees (employee_id, first_name, last_name, hire_date)
VALUES (1, 'John', 'Doe', '2023-01-01');

Este comando inserta un nuevo registro de empleado en la tabla employees.

  • UPDATE: Se utiliza para actualizar datos existentes dentro de una tabla.
UPDATE nombre_tabla
SET columna1 = valor1, columna2 = valor2, ...
WHERE condición;

Ejemplo:

UPDATE employees
SET email = 'john.doe@example.com'
WHERE employee_id = 1;

Este comando actualiza la dirección de correo electrónico del empleado con employee_id 1.

  • DELETE: Se utiliza para eliminar datos de una tabla.
DELETE FROM nombre_tabla
WHERE condición;

Ejemplo:

DELETE FROM employees
WHERE employee_id = 1;

Este comando elimina el registro del empleado con employee_id 1 de la tabla employees.

Lenguaje de Control de Datos (DCL)

  • GRANT: Se utiliza para otorgar privilegios de acceso a los usuarios en la base de datos.
GRANT SELECT, INSERT ON nombre_tabla TO 'nombre_usuario';

Ejemplo:

GRANT SELECT, INSERT ON employees TO 'john_doe';

Este comando otorga privilegios de SELECT e INSERT en la tabla employees al usuario john_doe.

  • REVOKE: Se utiliza para retirar permisos otorgados a los usuarios.
REVOKE SELECT, INSERT ON nombre_tabla FROM 'nombre_usuario';

Ejemplo:

REVOKE SELECT, INSERT ON employees FROM 'john_doe';

Este comando revoca los privilegios de SELECT e INSERT en la tabla employees del usuario john_doe.

Conclusión

SQL es una herramienta esencial para cualquiera que trabaje con bases de datos relacionales. Permite a los usuarios gestionar y manipular datos de manera eficiente a través de una variedad de comandos. Entender los conceptos y comandos básicos de SQL es el primer paso hacia el dominio de la gestión de bases de datos y la adquisición de habilidades competentes en el manejo de datos. A medida que avances en este curso, aprenderás técnicas SQL más avanzadas y mejores prácticas para mejorar tus habilidades en bases de datos.
