Introducción

En este estudio de caso, exploraremos cómo se puede utilizar JCL (Job Control Language) para tareas de mantenimiento del sistema. El mantenimiento del sistema es crucial para asegurar que el entorno del mainframe funcione sin problemas y de manera eficiente. Cubriremos varias tareas de mantenimiento, incluyendo copias de seguridad, limpieza de datos y monitoreo del sistema.

Conceptos Clave

  • Mantenimiento del Sistema: El proceso de mantener un sistema informático funcionando eficientemente mediante la realización de verificaciones, actualizaciones y reparaciones regulares.
  • JCL: Un lenguaje de scripting utilizado en los sistemas operativos de mainframe de IBM para instruir al sistema sobre cómo ejecutar un trabajo por lotes o iniciar un subsistema.

Procedimientos de Copia de Seguridad

Hacer copias de seguridad de los datos es una tarea de mantenimiento crítica para prevenir la pérdida de datos. JCL se puede utilizar para automatizar los procesos de copia de seguridad.

Ejemplo: Trabajo de Copia de Seguridad

//BACKUP   JOB (ACCT),'BACKUP JOB',CLASS=A,MSGCLASS=X
//STEP1    EXEC PGM=ADRDSSU
//SYSPRINT DD SYSOUT=*
//SYSIN    DD *
  DUMP DATASET(INCLUDE(**))
  OUTDDNAME(TAPE1)
  COMPRESS
/*
//TAPE1    DD DSN=BACKUP.DATASET,DISP=(NEW,KEEP),UNIT=TAPE,
//         VOL=SER=000001,LABEL=(1,SL)
//
  • Explicación:
    • Declaración JOB: Define el nombre del trabajo y la información contable.
    • Declaración EXEC: Especifica el programa a ejecutar (ADRDSSU para la utilidad DFSMSdss).
    • Declaración SYSPRINT DD: Dirige los mensajes de salida a la salida del sistema.
    • Declaración SYSIN DD: Contiene las declaraciones de control para la utilidad DFSMSdss.
    • DUMP DATASET(INCLUDE())**: Especifica que todos los conjuntos de datos deben incluirse en el volcado.
    • OUTDDNAME(TAPE1): Nombra la declaración DD de salida.
    • COMPRESS: Indica que los datos deben comprimirse.
    • Declaración TAPE1 DD: Define el conjunto de datos de salida en cinta.

Limpieza de Datos

La limpieza regular de datos ayuda a mantener el rendimiento del sistema eliminando archivos y conjuntos de datos innecesarios.

Ejemplo: Trabajo de Limpieza de Datos

//CLEANUP  JOB (ACCT),'CLEANUP JOB',CLASS=A,MSGCLASS=X
//STEP1    EXEC PGM=IDCAMS
//SYSPRINT DD SYSOUT=*
//SYSIN    DD *
  DELETE USER.DATASET1
  DELETE USER.DATASET2
  DELETE USER.DATASET3
/*
//
  • Explicación:
    • Declaración JOB: Define el nombre del trabajo y la información contable.
    • Declaración EXEC: Especifica el programa a ejecutar (IDCAMS para los Servicios de Método de Acceso).
    • Declaración SYSPRINT DD: Dirige los mensajes de salida a la salida del sistema.
    • Declaración SYSIN DD: Contiene las declaraciones de control para IDCAMS.
    • Declaraciones DELETE: Especifica los conjuntos de datos a eliminar.

Monitoreo del Sistema

El monitoreo del sistema es esencial para detectar y resolver problemas antes de que afecten el rendimiento.

Ejemplo: Trabajo de Monitoreo del Sistema

//MONITOR  JOB (ACCT),'MONITOR JOB',CLASS=A,MSGCLASS=X
//STEP1    EXEC PGM=RMF
//SYSPRINT DD SYSOUT=*
//SYSIN    DD *
  REPORT TYPE(SYSTEM)
  INTERVAL(15)
/*
//
  • Explicación:
    • Declaración JOB: Define el nombre del trabajo y la información contable.
    • Declaración EXEC: Especifica el programa a ejecutar (RMF para la Instalación de Medición de Recursos).
    • Declaración SYSPRINT DD: Dirige los mensajes de salida a la salida del sistema.
    • Declaración SYSIN DD: Contiene las declaraciones de control para RMF.
    • REPORT TYPE(SYSTEM): Especifica el tipo de informe a generar.
    • INTERVAL(15): Establece el intervalo de informes a 15 minutos.

Conclusión

En este estudio de caso, hemos demostrado cómo se puede utilizar JCL para varias tareas de mantenimiento del sistema, como copias de seguridad, limpieza de datos y monitoreo del sistema. Al automatizar estas tareas con JCL, los administradores del sistema pueden asegurar que el entorno del mainframe se mantenga eficiente y confiable. El mantenimiento regular es esencial para prevenir problemas y mantener un rendimiento óptimo.
