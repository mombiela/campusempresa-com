Introducción

En el Lenguaje de Control de Trabajos (JCL), los datasets son cruciales para gestionar y procesar datos. Entender cómo definir y manipular los parámetros de los datasets es esencial para una ejecución eficiente de los trabajos. Esta sección cubrirá los conceptos básicos de los parámetros de los datasets, sus tipos y cómo usarlos en JCL.

Conceptos Clave

  • Dataset: Una colección de registros de datos almacenados en un disco o cinta.
  • Parámetros de Dataset: Atributos que definen las propiedades y el comportamiento de los datasets en JCL.

Tipos de Parámetros de Dataset

1. Nombre del Dataset (DSN)

  • Definición: El identificador único para un dataset.
  • Sintaxis: DSN=nombre.del.dataset
  • Ejemplo: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=SHR

2. Disposición (DISP)

  • Definición: Especifica el estado del dataset y qué hacer con él después de que el paso del trabajo se complete.
  • Sintaxis: DISP=(estado,terminación-normal,terminación-anormal)
  • Valores de Estado:
    • NEW: Crear un nuevo dataset.
    • OLD: Usar un dataset existente.
    • SHR: Compartir un dataset existente.
    • MOD: Modificar un dataset existente.
  • Ejemplo: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=(NEW,CATLG,DELETE)

3. Asignación de Espacio (SPACE)

  • Definición: Define la cantidad de espacio a asignar para un dataset.
  • Sintaxis: SPACE=(unidad,(primario,secundario,dir),RLSE)
  • Parámetros:
    • unidad: La unidad de asignación (e.g., CYL, TRK, BLK).
    • primario: Asignación de espacio primaria.
    • secundario: Asignación de espacio secundaria.
    • dir: Número de bloques de directorio (para datasets particionados).
    • RLSE: Liberar espacio no utilizado.
  • Ejemplo: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=(NEW,CATLG,DELETE),
//             SPACE=(CYL,(10,5),RLSE)

4. Bloque de Control de Datos (DCB)

  • Definición: Especifica las características físicas del dataset.
  • Sintaxis: DCB=(parámetro1,parámetro2,...)
  • Parámetros Comunes:
    • RECFM: Formato de registro (e.g., FB, VB).
    • LRECL: Longitud lógica del registro.
    • BLKSIZE: Tamaño del bloque.
  • Ejemplo: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=(NEW,CATLG,DELETE),
//             DCB=(RECFM=FB,LRECL=80,BLKSIZE=800)

5. Número de Serie del Volumen (VOL)

  • Definición: Especifica el número de serie del volumen donde reside el dataset.
  • Sintaxis: VOL=SER=volumen
  • Ejemplo: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=SHR,
//             VOL=SER=123456

6. Unidad (UNIT)

  • Definición: Especifica el tipo de dispositivo o grupo donde se almacena el dataset.
  • Sintaxis: UNIT=dispositivo
  • Ejemplo: 
    //MYJOB   JOB  ...
//STEP1   EXEC PGM=MYPROG
//DD1     DD   DSN=MY.DATASET,DISP=SHR,
//             UNIT=SYSDA

Comparación de Parámetros de Dataset

| Parámetro | Descripción | Ejemplo de Sintaxis | |-----------|-------------|---------------------| | DSN | Nombre del Dataset | DSN=MY.DATASET | | DISP | Disposición | DISP=(NEW,CATLG,DELETE) | | SPACE | Asignación de Espacio | SPACE=(CYL,(10,5),RLSE) | | DCB | Bloque de Control de Datos | DCB=(RECFM=FB,LRECL=80,BLKSIZE=800) | | VOL | Número de Serie del Volumen | VOL=SER=123456 | | UNIT | Tipo de Unidad | UNIT=SYSDA |

Conclusión

Entender los parámetros de dataset en JCL es fundamental para gestionar datasets de manera efectiva. Al dominar estos parámetros, puedes controlar cómo se crean, acceden y gestionan los datasets a lo largo de la ejecución de los trabajos. Practica usando estos parámetros en varias combinaciones para volverte competente en la gestión de datasets en JCL.
