Introducción

Bienvenido al curso AZ-104: Administrador de Microsoft Azure. Este curso está diseñado para llevarte de un nivel principiante a un nivel avanzado en la gestión y administración de recursos de Azure. Al final de este curso, estarás bien preparado para tomar el examen de certificación AZ-104 y sobresalir en tu rol como Administrador de Azure.

Esquema del Curso

  1. Introducción a Azure
  2. Gestión de Suscripciones y Recursos de Azure
  3. Implementación y Gestión de Almacenamiento
  4. Despliegue y Gestión de Máquinas Virtuales (VMs)
  5. Configuración y Gestión de Redes Virtuales
  6. Gestión de Identidades
  7. Monitoreo y Respaldo de Recursos de Azure

1. Introducción a Azure

¿Qué es Azure?

  • Azure es la plataforma de computación en la nube de Microsoft.
  • Proporciona una gama de servicios que incluyen computación, análisis, almacenamiento y redes.

Conceptos Clave

  • Grupos de Recursos: Contenedores lógicos para recursos desplegados en Azure.
  • Regiones: Áreas geográficas donde se encuentran los centros de datos de Azure.
  • Zonas de Disponibilidad: Ubicaciones físicamente separadas dentro de una región de Azure.

Ejemplo: Creación de un Grupo de Recursos

# Crear un grupo de recursos en la región Este de EE.UU.
az group create --name MyResourceGroup --location eastus

Explicación: Este comando crea un nuevo grupo de recursos llamado MyResourceGroup en la región eastus.

2. Gestión de Suscripciones y Recursos de Azure

Suscripciones de Azure

  • Una suscripción es un contenedor lógico utilizado para aprovisionar recursos en Azure.
  • Cada suscripción tiene una ID única y está vinculada a una cuenta de facturación.

Gestión de Recursos

  • Azure Resource Manager (ARM): El servicio de despliegue y gestión para Azure.
  • Etiquetas: Pares clave-valor que ayudan a organizar recursos.

Ejemplo: Aplicación de Etiquetas a un Recurso

# Aplicar una etiqueta a un recurso
az resource tag --tags Environment=Production --resource-id /subscriptions/{subscription-id}/resourceGroups/{resource-group}/providers/Microsoft.Compute/virtualMachines/{vm-name}

Explicación: Este comando aplica la etiqueta Environment=Production a una máquina virtual especificada.

3. Implementación y Gestión de Almacenamiento

Tipos de Cuentas de Almacenamiento

  • Blob Storage: Optimizado para almacenar grandes cantidades de datos no estructurados.
  • File Storage: Comparticiones de archivos gestionadas que pueden ser accedidas vía protocolo SMB.
  • Queue Storage: Proporciona colas de mensajes para grandes cargas de trabajo.
  • Table Storage: Almacenamiento NoSQL de clave-valor para desarrollo rápido.

Ejemplo: Creación de una Cuenta de Almacenamiento

# Crear una cuenta de almacenamiento
az storage account create --name mystorageaccount --resource-group MyResourceGroup --location eastus --sku Standard_LRS

Explicación: Este comando crea una nueva cuenta de almacenamiento llamada mystorageaccount en el grupo de recursos MyResourceGroup con el SKU Standard_LRS (Almacenamiento Localmente Redundante).

4. Despliegue y Gestión de Máquinas Virtuales (VMs)

Creación de VMs

  • Azure Marketplace: Ofrece una amplia gama de imágenes de VM preconfiguradas.
  • Imágenes Personalizadas: Crear VMs a partir de imágenes personalizadas.

Ejemplo: Creación de una Máquina Virtual

# Crear una máquina virtual
az vm create --resource-group MyResourceGroup --name MyVM --image UbuntuLTS --admin-username azureuser --generate-ssh-keys

Explicación: Este comando crea una nueva máquina virtual llamada MyVM en el grupo de recursos MyResourceGroup usando la imagen UbuntuLTS.

5. Configuración y Gestión de Redes Virtuales

Redes Virtuales (VNet)

  • Subredes: Segmentos dentro de una VNet.
  • Grupos de Seguridad de Red (NSGs): Controlan el tráfico entrante y saliente.

Ejemplo: Creación de una Red Virtual y Subred

# Crear una red virtual
az network vnet create --resource-group MyResourceGroup --name MyVNet --address-prefix 10.0.0.0/16

# Crear una subred
az network vnet subnet create --resource-group MyResourceGroup --vnet-name MyVNet --name MySubnet --address-prefix 10.0.1.0/24

Explicación: Estos comandos crean una red virtual llamada MyVNet con un espacio de direcciones de 10.0.0.0/16 y una subred llamada MySubnet con un prefijo de dirección de 10.0.1.0/24.

6. Gestión de Identidades

Azure Active Directory (AAD)

  • Usuarios y Grupos: Gestionar el acceso a recursos.
  • Roles y Asignaciones de Roles: Definir permisos.

Ejemplo: Creación de un Usuario en Azure AD

# Crear un usuario en Azure AD
az ad user create --display-name JohnDoe --user-principal-name johndoe@mydomain.com --password P@ssw0rd!

Explicación: Este comando crea un nuevo usuario llamado JohnDoe con el nombre principal de usuario johndoe@mydomain.com.

7. Monitoreo y Respaldo de Recursos de Azure

Monitoreo

  • Azure Monitor: Proporciona monitoreo de pila completa.
  • Log Analytics: Recoge y analiza datos de registros.

Ejemplo: Configuración de Azure Monitor

# Crear un espacio de trabajo de Log Analytics
az monitor log-analytics workspace create --resource-group MyResourceGroup --workspace-name MyWorkspace

# Habilitar monitoreo para una VM
az monitor diagnostic-settings create --resource /subscriptions/{subscription-id}/resourceGroups/{resource-group}/providers/Microsoft.Compute/virtualMachines/{vm-name} --workspace MyWorkspace --name MyDiagnosticSetting --metrics '[{"category": "AllMetrics", "enabled": true}]' --logs '[{"category": "AuditLogs", "enabled": true}]'

Explicación: Estos comandos crean un espacio de trabajo de Log Analytics llamado MyWorkspace y habilitan el monitoreo para una máquina virtual especificada.

Conclusión

En este curso, cubrimos los temas esenciales necesarios para convertirte en un Administrador de Azure competente. Desde entender los conceptos básicos de Azure hasta gestionar recursos complejos y soluciones de monitoreo, ahora tienes el conocimiento y las habilidades para gestionar entornos de Azure de manera efectiva. Continúa practicando y explorando los servicios de Azure para profundizar tu comprensión y prepararte para el examen de certificación AZ-104. ¡Buena suerte!
