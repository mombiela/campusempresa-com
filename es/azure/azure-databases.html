Introducción a las bases de datos de Azure

Azure ofrece una variedad de servicios de bases de datos que se adaptan a diferentes necesidades, desde bases de datos relacionales hasta bases de datos NoSQL. Esta sección proporcionará una visión general de los diferentes tipos de bases de datos disponibles en Azure y sus casos de uso.

  • Azure SQL Database: Una base de datos relacional totalmente administrada con inteligencia incorporada.
  • Azure Cosmos DB: Un servicio de base de datos multimodelo distribuido globalmente.
  • Azure Database for MySQL: Un servicio de base de datos MySQL administrado.
  • Azure Database for PostgreSQL: Un servicio de base de datos PostgreSQL administrado.
  • Azure Database for MariaDB: Un servicio de base de datos MariaDB administrado.

Azure SQL Database

Azure SQL Database es un servicio de base de datos relacional totalmente administrado que proporciona alta disponibilidad, escalabilidad y seguridad.

Características clave

  • Servicio administrado: Copias de seguridad automáticas, parches y actualizaciones.
  • Escalabilidad: Escalar hacia arriba o hacia abajo según tus necesidades.
  • Seguridad: Protección avanzada contra amenazas y cifrado.
  • Rendimiento: Inteligencia incorporada para la optimización del rendimiento.

Comenzando con Azure SQL Database

  1. Crear una base de datos SQL de Azure:

    az sql db create --resource-group myResourceGroup --server myServer --name myDatabase --service-objective S0

  2. Conectar a Azure SQL Database:

    import pyodbc

server = 'myServer.database.windows.net'
database = 'myDatabase'
username = 'myUsername'
password = 'myPassword'
driver= '{ODBC Driver 17 for SQL Server}'

cnxn = pyodbc.connect('DRIVER=' + driver + ';SERVER=' + server + ';PORT=1433;DATABASE=' + database + ';UID=' + username + ';PWD=' + password)
cursor = cnxn.cursor()

  3. Realizar operaciones CRUD:

    # Crear
cursor.execute("INSERT INTO myTable (name, age) VALUES (?, ?)", ('John Doe', 30))
cnxn.commit()

# Leer
cursor.execute("SELECT * FROM myTable")
for row in cursor:
    print(row)

# Actualizar
cursor.execute("UPDATE myTable SET age = ? WHERE name = ?", (31, 'John Doe'))
cnxn.commit()

# Eliminar
cursor.execute("DELETE FROM myTable WHERE name = ?", ('John Doe',))
cnxn.commit()

Azure Cosmos DB

Azure Cosmos DB es un servicio de base de datos multimodelo distribuido globalmente diseñado para alta disponibilidad y baja latencia.

Características clave

  • Distribución global: Distribuir datos en múltiples regiones.
  • Multimodelo: Soporta modelos de datos de documentos, clave-valor, grafos y familias de columnas.
  • Escalabilidad: Escalado automático de rendimiento y almacenamiento.
  • Niveles de consistencia: Elegir entre cinco niveles de consistencia para equilibrar rendimiento y consistencia.

Comenzando con Azure Cosmos DB

  1. Crear una cuenta de Azure Cosmos DB:

    az cosmosdb create --name myCosmosDBAccount --resource-group myResourceGroup --kind MongoDB

  2. Conectar a Azure Cosmos DB:

    from pymongo import MongoClient

client = MongoClient("mongodb://myCosmosDBAccount.documents.azure.com:10255/?ssl=true&replicaSet=globaldb", username="myUsername", password="myPassword")
db = client.myDatabase

  3. Realizar operaciones CRUD:

    # Crear
db.myCollection.insert_one({"name": "John Doe", "age": 30})

# Leer
for doc in db.myCollection.find():
    print(doc)

# Actualizar
db.myCollection.update_one({"name": "John Doe"}, {"$set": {"age": 31}})

# Eliminar
db.myCollection.delete_one({"name": "John Doe"})

Azure Database for MySQL

Azure Database for MySQL es un servicio de base de datos MySQL administrado que proporciona alta disponibilidad y seguridad.

Características clave

  • Servicio administrado: Copias de seguridad automáticas, parches y actualizaciones.
  • Escalabilidad: Escalar computación y almacenamiento de manera independiente.
  • Seguridad: Protección avanzada contra amenazas y cifrado.
  • Rendimiento: Monitoreo y ajuste de rendimiento incorporados.

Comenzando con Azure Database for MySQL

  1. Crear una base de datos MySQL de Azure:

    az mysql server create --resource-group myResourceGroup --name myMySQLServer --admin-user myAdmin --admin-password myPassword --sku-name GP_Gen5_2

  2. Conectar a Azure Database for MySQL:

    import mysql.connector

cnx = mysql.connector.connect(user='myAdmin', password='myPassword',
                              host='myMySQLServer.mysql.database.azure.com',
                              database='myDatabase')
cursor = cnx.cursor()

  3. Realizar operaciones CRUD:

    # Crear
cursor.execute("INSERT INTO myTable (name, age) VALUES (%s, %s)", ('John Doe', 30))
cnx.commit()

# Leer
cursor.execute("SELECT * FROM myTable")
for row in cursor:
    print(row)

# Actualizar
cursor.execute("UPDATE myTable SET age = %s WHERE name = %s", (31, 'John Doe'))
cnx.commit()

# Eliminar
cursor.execute("DELETE FROM myTable WHERE name = %s", ('John Doe',))
cnx.commit()

Conclusión

Azure ofrece una amplia gama de servicios de bases de datos que se adaptan a diversas necesidades, desde bases de datos relacionales como Azure SQL Database, MySQL, PostgreSQL y MariaDB hasta bases de datos NoSQL como Azure Cosmos DB. Cada servicio proporciona características y beneficios únicos, lo que facilita la elección de la base de datos adecuada para tu aplicación. Al comprender las características clave y comenzar con operaciones CRUD básicas, puedes aprovechar el poder de las bases de datos de Azure para construir aplicaciones escalables, seguras y de alto rendimiento.
