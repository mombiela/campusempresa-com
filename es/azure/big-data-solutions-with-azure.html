Configuración del clúster de HDInsight

  • Requisitos previos
  • Guía paso a paso para crear un clúster de HDInsight
  • Opciones de configuración
# Ejemplo: Creación de un clúster de HDInsight usando Azure CLI
az hdinsight create \
    --name myHDInsightCluster \
    --resource-group myResourceGroup \
    --cluster-type spark \
    --location eastus \
    --version 4.0 \
    --component-version spark=2.4 \
    --headnode-size Standard_D3_V2 \
    --workernode-size Standard_D3_V2 \
    --workernode-count 3

Trabajando con datos en HDInsight

  • Carga de datos en HDInsight
  • Procesamiento de datos con Apache Spark
  • Ejemplo: Programa de conteo de palabras en PySpark
# Ejemplo: Conteo de palabras en PySpark
from pyspark import SparkContext

sc = SparkContext("local", "WordCount")
text_file = sc.textFile("hdfs:///path/to/input.txt")
counts = text_file.flatMap(lambda line: line.split(" ")) \
                  .map(lambda word: (word, 1)) \
                  .reduceByKey(lambda a, b: a + b)
counts.saveAsTextFile("hdfs:///path/to/output")

Configuración de Azure Data Lake Storage

  • Requisitos previos
  • Guía paso a paso para crear una cuenta de Data Lake Storage
  • Opciones de configuración
# Ejemplo: Creación de una cuenta de Azure Data Lake Storage usando Azure CLI
az storage account create \
    --name mydatalakestorage \
    --resource-group myResourceGroup \
    --location eastus \
    --sku Standard_LRS \
    --kind StorageV2 \
    --hierarchical-namespace true

    --file-system myfilesystem \
    --path /path/to/uploadedfile.txt \
    --source /local/path/to/file.txt
