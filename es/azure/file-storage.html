Introducción al Almacenamiento de Archivos en Azure

El Almacenamiento de Archivos en Azure es un servicio de almacenamiento de archivos basado en la nube que te permite crear comparticiones de archivos en la nube. Está diseñado para ser altamente disponible, duradero y escalable, lo que lo convierte en una excelente opción para una variedad de aplicaciones, desde el simple intercambio de archivos hasta soluciones de almacenamiento a nivel empresarial complejas.

Conceptos Clave

  • Compartición de Archivos: Una compartición de archivos es un agrupamiento lógico de archivos que se puede acceder a través de la red.
  • Cuenta de Almacenamiento: Una cuenta de almacenamiento es un contenedor que contiene todos tus datos de Almacenamiento de Azure, incluidas las comparticiones de archivos.
  • Niveles de Acceso: El Almacenamiento de Archivos en Azure ofrece diferentes niveles de acceso (Hot, Cool y Archive) para optimizar los costos según la frecuencia con la que se accede a los datos.
  • Protocolo SMB: Server Message Block (SMB) es un protocolo de intercambio de archivos en red que permite a las aplicaciones leer y escribir archivos y solicitar servicios de programas de servidor en una red de computadoras.

Configuración del Almacenamiento de Archivos en Azure

Creación de una Cuenta de Almacenamiento

Para usar el Almacenamiento de Archivos en Azure, primero necesitas crear una cuenta de almacenamiento.

  1. Navegar al Portal de Azure: Ve al Portal de Azure.
  2. Crear una Cuenta de Almacenamiento:
    • Haz clic en "Crear un recurso" y selecciona "Cuenta de almacenamiento".
    • Completa los detalles requeridos como Suscripción, Grupo de Recursos, Nombre de la Cuenta de Almacenamiento y Ubicación.
    • Elige las opciones de Rendimiento (Estándar o Premium) y Replicación (LRS, GRS, etc.) según tus necesidades.
    • Haz clic en "Revisar + crear" y luego en "Crear".

Creación de una Compartición de Archivos

Una vez que tengas una cuenta de almacenamiento, puedes crear una compartición de archivos.

  1. Navegar a tu Cuenta de Almacenamiento: Ve a la cuenta de almacenamiento que creaste.
  2. Crear una Compartición de Archivos:
    • Haz clic en "Comparticiones de archivos" en la sección "Almacenamiento de datos".
    • Haz clic en "+ Compartición de archivos".
    • Ingresa un nombre para la compartición de archivos y especifica la cuota (en GB).
    • Haz clic en "Crear".

Acceso al Almacenamiento de Archivos en Azure

Montar la Compartición de Archivos en Windows

Puedes montar la compartición de archivos de Azure en una máquina con Windows usando el protocolo SMB.

  1. Obtener la Cadena de Conexión:

    • Navega a tu compartición de archivos en el Portal de Azure.
    • Haz clic en "Conectar" y selecciona "Windows".
    • Copia el script proporcionado.

  2. Ejecutar el Script:

    • Abre PowerShell en tu máquina con Windows.
    • Pega y ejecuta el script. Se verá algo así:
    $connectTestResult = Test-Path -Path "Z:"
if ($connectTestResult -eq $false) {
    New-PSDrive -Name Z -PSProvider FileSystem -Root "\\<storage_account_name>.file.core.windows.net\<file_share_name>" -Persist -Credential (Get-Credential)
}

Acceso a la Compartición de Archivos desde Linux

También puedes montar la compartición de archivos en una máquina con Linux usando el protocolo SMB.

  1. Instalar CIFS Utils:

    sudo apt-get update
sudo apt-get install cifs-utils

  2. Montar la Compartición de Archivos:

    sudo mount -t cifs //<storage_account_name>.file.core.windows.net/<file_share_name> /mnt/<mount_point> -o vers=3.0,username=<storage_account_name>,password=<storage_account_key>,dir_mode=0777,file_mode=0777

Funciones Avanzadas

Sincronización de Archivos de Azure

La Sincronización de Archivos de Azure te permite centralizar tus comparticiones de archivos en Azure Files, manteniendo la flexibilidad, el rendimiento y la compatibilidad de un servidor de archivos local.

  1. Instalar el Agente de Sincronización de Archivos de Azure: Descarga e instala el agente de Sincronización de Archivos de Azure en tu Windows Server.
  2. Registrar el Servidor: Usa el agente de Sincronización de Archivos de Azure para registrar tu servidor con tu suscripción de Azure.
  3. Crear un Grupo de Sincronización: En el Portal de Azure, crea un grupo de sincronización y agrega tu compartición de archivos y servidor registrado a él.

Control de Acceso y Seguridad

  • Integración con Azure Active Directory (AAD): Puedes integrar Azure Files con Azure AD para gestionar permisos de acceso.
  • Seguridad de Red: Usa reglas de red virtual (VNet) y firewall para restringir el acceso a tus comparticiones de archivos.

Conclusión

El Almacenamiento de Archivos en Azure proporciona una solución robusta, escalable y segura para almacenar y acceder a archivos en la nube. Al comprender los conceptos básicos de la creación y gestión de comparticiones de archivos, así como aprovechar las funciones avanzadas como la Sincronización de Archivos de Azure y el control de acceso, puedes usar eficazmente el Almacenamiento de Archivos en Azure para satisfacer tus necesidades de almacenamiento.
