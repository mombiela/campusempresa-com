Azure Automation
Servicio de Kubernetes de Azure (AKS) ►

Azure Automation es un servicio de automatización y configuración basado en la nube que proporciona una gestión coherente en tus entornos de Azure y no Azure. Te ayuda a automatizar tareas de TI frecuentes, que consumen mucho tiempo y son propensas a errores.

Introducción a Azure Automation

  • ¿Qué es Azure Automation?

    • Un servicio para automatizar tareas y orquestar flujos de trabajo.
    • Ayuda a gestionar entornos en la nube y locales.
    • Soporta la automatización de tareas repetitivas, la gestión de configuraciones y la gestión de actualizaciones.

  • Beneficios Clave:

    • Eficiencia: Automatiza tareas repetitivas para ahorrar tiempo.
    • Consistencia: Asegura que las tareas se realicen de la misma manera cada vez.
    • Ahorro de Costos: Reduce los costos operativos al automatizar procesos manuales.
    • Escalabilidad: Escala fácilmente las tareas de automatización a medida que tu entorno crece.

Empezando con Azure Automation

  • Creando una Cuenta de Automatización:

    1. Inicia sesión en el portal de Azure.
    2. Navega a "Crear un recurso" > "Automatización".
    3. Rellena los detalles requeridos (Nombre, Grupo de Recursos, Ubicación).
    4. Haz clic en "Crear".

  • Componentes de la Cuenta de Automatización:

    • Runbooks: Scripts que realizan tareas.
    • Assets: Recursos como variables, credenciales y conexiones.
    • Schedules: Definen cuándo deben ejecutarse los runbooks.
    • Modules: Bibliotecas de cmdlets de PowerShell.

Ejemplo: Creando un Runbook Simple

# Runbook de PowerShell de ejemplo
Write-Output "¡Hola, Azure Automation!"
  • Pasos para Crear y Ejecutar un Runbook:
    1. En tu cuenta de Automatización, ve a "Runbooks".
    2. Haz clic en "Agregar un runbook" > "Crear un nuevo runbook".
    3. Ingresa un nombre y selecciona "PowerShell" como el tipo de runbook.
    4. Pega el código de ejemplo anterior.
    5. Guarda y publica el runbook.
    6. Haz clic en "Iniciar" para ejecutar el runbook.

Conceptos Intermedios de Azure Automation

  • Uso de Variables y Credenciales:
    • Variables: Almacenan valores que pueden reutilizarse en runbooks.
    • Credenciales: Almacenan y usan de forma segura credenciales en runbooks.

Ejemplo: Usando Variables en un Runbook

# Recuperar el valor de una variable
$myVariable = Get-AutomationVariable -Name "MyVariable"
Write-Output "El valor de MyVariable es $myVariable"
  • Schedules y Webhooks:
    • Schedules: Automatizan la ejecución de runbooks en momentos específicos.
    • Webhooks: Activan runbooks a través de solicitudes HTTP.

Ejemplo: Creando un Schedule

  1. Navega a "Schedules" en tu cuenta de Automatización.
  2. Haz clic en "Agregar un schedule".
  3. Define el schedule (por ejemplo, diariamente a las 8 AM).
  4. Vincula el schedule a un runbook.

Técnicas Avanzadas de Azure Automation

  • Trabajadores de Runbook Híbridos:

    • Ejecutan runbooks en entornos locales o en otras nubes.
    • Útiles para tareas que requieren acceso a recursos locales.

  • Integración con Control de Código Fuente:

    • Integra con GitHub o Azure Repos para el control de versiones de runbooks.

  • Tipos Avanzados de Runbook:

    • Runbooks Gráficos: Crea runbooks usando una interfaz de arrastrar y soltar.
    • Runbooks de Python: Usa scripts de Python para tareas de automatización.

Ejemplo: Creando un Runbook Gráfico

  1. En tu cuenta de Automatización, ve a "Runbooks".
  2. Haz clic en "Agregar un runbook" > "Crear un nuevo runbook".
  3. Ingresa un nombre y selecciona "Gráfico" como el tipo de runbook.
  4. Usa el editor gráfico para diseñar tu flujo de trabajo.
  5. Guarda y publica el runbook.

Conclusión

Azure Automation es una herramienta poderosa para automatizar y gestionar tus entornos en la nube y locales. Al aprovechar los runbooks, schedules y otros componentes de la cuenta de automatización, puedes optimizar operaciones, reducir errores y ahorrar tiempo. Ya sea que estés comenzando o buscando implementar técnicas avanzadas de automatización, Azure Automation proporciona la flexibilidad y escalabilidad para satisfacer tus necesidades.
