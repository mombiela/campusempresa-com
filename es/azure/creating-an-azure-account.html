Creación de una Cuenta de Azure
Tour por el Portal de Azure ►

Crear una cuenta de Azure es el primer paso para aprovechar los servicios en la nube de Microsoft para tus aplicaciones y proyectos. Esta sección te guiará a través del proceso de configuración de una cuenta de Azure, asegurando que tengas la base necesaria para explorar y utilizar las amplias ofertas de Azure.

¿Por Qué Crear una Cuenta de Azure?

  • Acceso a Servicios en la Nube: Obtén acceso a una amplia gama de servicios en la nube, incluyendo computación, análisis, almacenamiento y redes.
  • Escalabilidad: Escala fácilmente tus aplicaciones y servicios según tus necesidades.
  • Gestión de Costos: Paga solo por lo que usas con los modelos de precios flexibles de Azure.
  • Seguridad: Benefíciate de las robustas medidas de seguridad y certificaciones de cumplimiento de Azure.

Requisitos Previos

Antes de crear una cuenta de Azure, asegúrate de tener lo siguiente:

  • Una dirección de correo electrónico válida.
  • Un número de teléfono para verificación.
  • Una tarjeta de crédito para verificación de identidad (no se te cobrará a menos que actualices a un plan de pago).

Pasos para Crear una Cuenta de Azure

  1. Visita el Sitio Web de Azure:

    • Ve al sitio web de Azure.

  2. Comienza Gratis:

    • Haz clic en el botón "Comienza gratis" en la página de inicio.

  3. Inicia Sesión o Crea una Cuenta de Microsoft:

    • Si ya tienes una cuenta de Microsoft, inicia sesión.
    • Si no, haz clic en "¡Crea una!" para configurar una nueva cuenta de Microsoft.

  4. Rellena Tus Datos:

    • Proporciona tu información personal, incluyendo nombre, correo electrónico y número de teléfono.
    • Verifica tu número de teléfono mediante un código enviado a tu teléfono.

  5. Verificación de Identidad:

    • Ingresa los detalles de tu tarjeta de crédito para la verificación de identidad. Nota que este es un procedimiento estándar y no se te cobrará a menos que explícitamente actualices a un plan de pago.

  6. Acepta los Términos:

    • Revisa y acepta el acuerdo de Microsoft Azure y la declaración de privacidad.

  7. Completa la Configuración:

    • Una vez que todos los detalles sean verificados, tu cuenta de Azure será creada y podrás comenzar a usar los servicios de Azure.

Beneficios de la Cuenta Gratuita de Azure

Cuando creas una nueva cuenta de Azure, obtienes acceso a varios beneficios:

  • Crédito de $200: Usa este crédito dentro de los primeros 30 días para explorar y probar cualquier servicio de Azure.
  • Servicios Gratuitos por 12 Meses: Accede a servicios populares de forma gratuita durante 12 meses, incluyendo:
    • 750 horas de Máquina Virtual Windows B1S.
    • 5 GB de Almacenamiento de Archivos LRS.
    • 250 GB de almacenamiento de Base de Datos SQL.
  • Servicios Siempre Gratuitos: Ciertos servicios son siempre gratuitos, tales como:
    • Servicio de Aplicaciones de Azure con 1 millón de solicitudes por mes.
    • Funciones de Azure con 1 millón de solicitudes por mes.

Ejemplo: Creación de una Máquina Virtual

Una vez que tu cuenta de Azure esté configurada, puedes comenzar a crear recursos. Aquí hay un ejemplo simple de cómo crear una Máquina Virtual (VM):

# Inicia sesión en tu cuenta de Azure
az login

# Crea un grupo de recursos
az group create --name myResourceGroup --location eastus

# Crea una máquina virtual
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys
  • az login: Comando para iniciar sesión en tu cuenta de Azure.
  • az group create: Crea un grupo de recursos llamado myResourceGroup en la región eastus.
  • az vm create: Crea una VM llamada myVM con la imagen de Ubuntu LTS, estableciendo azureuser como el nombre de usuario administrador y generando claves SSH para acceso seguro.

Conclusión

Crear una cuenta de Azure es un proceso sencillo que abre un mundo de posibilidades para la computación en la nube. Con tu nueva cuenta, puedes explorar la amplia gama de servicios de Azure, aprovechar los créditos gratuitos y comenzar a construir y desplegar tus aplicaciones en la nube. En las siguientes secciones, profundizaremos en servicios específicos de Azure y cómo utilizarlos eficazmente para satisfacer las necesidades de tu proyecto.
