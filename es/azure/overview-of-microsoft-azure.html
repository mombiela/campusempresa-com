Introducción a Microsoft Azure

Microsoft Azure es una plataforma y servicio de computación en la nube creada por Microsoft. Proporciona una gama de servicios en la nube, incluidos los de computación, análisis, almacenamiento y redes. Los usuarios pueden elegir entre estos servicios para desarrollar y escalar nuevas aplicaciones, o ejecutar aplicaciones existentes, en la nube pública.

Conceptos clave de Microsoft Azure

  • Computación en la nube: La entrega de servicios de computación a través de internet ("la nube") para ofrecer una innovación más rápida, recursos flexibles y economías de escala.
  • Portal de Azure: Una aplicación web que te permite gestionar tus recursos de Azure.
  • Grupos de recursos: Contenedores que contienen recursos relacionados para una solución de Azure.
  • Máquinas virtuales (VMs): Sistemas informáticos emulados que proporcionan la funcionalidad de una computadora física.
  • Almacenamiento de Azure: Almacenamiento en la nube escalable para datos, archivos y copias de seguridad.
  • Redes de Azure: Servicios que proporcionan capacidades de red para conectar y gestionar tus recursos de Azure.

Descripción general de los servicios de Azure

Servicios de computación

  • Máquinas virtuales: Recursos de computación escalables y bajo demanda.
  • Servicios de aplicaciones: Entorno de alojamiento gestionado para aplicaciones web.
  • Funciones de Azure: Servicio de computación sin servidor y basado en eventos.

Servicios de almacenamiento

  • Blob Storage: Almacenamiento de objetos para datos no estructurados.
  • File Storage: Comparticiones de archivos totalmente gestionadas en la nube.
  • Queue Storage: Servicio de mensajería para la comunicación entre componentes de aplicaciones.

Servicios de redes

  • Red virtual (VNet): Permite que muchos tipos de recursos de Azure se comuniquen de manera segura entre sí.
  • Balanceador de carga: Distribuye el tráfico de red entrante entre varios objetivos.
  • Puerta de enlace VPN: Establece conectividad segura entre instalaciones.

Servicios de bases de datos

  • Base de datos SQL de Azure: Servicio de base de datos relacional gestionado.
  • Cosmos DB: Servicio de base de datos multimodelo y distribuido globalmente.
  • Base de datos de Azure para MySQL/PostgreSQL: Servicios de bases de datos gestionados para MySQL y PostgreSQL.

Empezando con Azure

Creación de una cuenta de Azure

Para comenzar a usar Azure, necesitas crear una cuenta. Microsoft ofrece un nivel gratuito que incluye una cantidad limitada de servicios gratuitos durante 12 meses.

Navegando por el Portal de Azure

El Portal de Azure es tu interfaz principal para gestionar los recursos de Azure. Aquí tienes una guía rápida para empezar:

  1. Tablero: Tu página de inicio personalizable.
  2. Todos los servicios: Una lista de todos los servicios disponibles en Azure.
  3. Grupos de recursos: Gestiona tus grupos de recursos.
  4. Suscripciones: Visualiza y gestiona tus suscripciones de Azure.

Creación de una máquina virtual

Crear una VM es uno de los primeros pasos para obtener experiencia práctica con Azure. Aquí tienes un ejemplo simple usando el Portal de Azure:

  1. Navega al Portal de Azure.
  2. Haz clic en "Crear un recurso".
  3. Selecciona "Máquina virtual".
  4. Rellena los detalles requeridos:
    • Suscripción: Elige tu suscripción.
    • Grupo de recursos: Crea un nuevo grupo de recursos o selecciona uno existente.
    • Nombre de la VM: Nombra tu VM.
    • Región: Elige la región donde deseas desplegar tu VM.
    • Imagen: Selecciona la imagen del SO (por ejemplo, Windows Server, Ubuntu).
    • Tamaño: Elige el tamaño de la VM según tus necesidades.
  5. Configura los ajustes restantes y haz clic en "Revisar + crear".
  6. Revisa tus configuraciones y haz clic en "Crear".
# Ejemplo de creación de una VM usando Azure CLI
az vm create \
  --resource-group myResourceGroup \
  --name myVM \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys

Conclusión

Microsoft Azure es una plataforma en la nube integral que ofrece una amplia gama de servicios para satisfacer diversas necesidades de computación. Desde máquinas virtuales hasta bases de datos y redes, Azure proporciona las herramientas necesarias para construir, desplegar y gestionar aplicaciones en la nube. Comprender los conceptos básicos de Azure, como crear una cuenta, navegar por el portal y desplegar recursos, sienta las bases para temas y servicios más avanzados. A medida que avances en este curso, obtendrás conocimientos más profundos y experiencia práctica con las poderosas capacidades de Azure.
