Laboratorios prácticos y ejercicios

  • Creación de una máquina virtual

    az vm create 
        --resource-group myResourceGroup 
        --name myVM 
        --image UbuntuLTS 
        --admin-username azureuser 
        --generate-ssh-keys
    • Explicación: Este comando crea una nueva máquina virtual en Azure usando la CLI de Azure. Especifica el grupo de recursos, el nombre de la VM, el tipo de imagen, el nombre de usuario administrador y genera claves SSH para acceso seguro.

  • Configuración de una aplicación web

    az webapp up --name myUniqueAppName --resource-group myResourceGroup --plan myAppServicePlan
    • Explicación: Este comando despliega una aplicación web en Azure App Service. Especifica el nombre de la aplicación, el grupo de recursos y el plan de servicio de aplicaciones.

  • Configuración de una red virtual

    az network vnet create 
        --name myVnet 
        --resource-group myResourceGroup 
        --subnet-name mySubnet
    • Explicación: Este comando crea una red virtual con una subred especificada. Ayuda a configurar la infraestructura de red para tus recursos de Azure.

Conclusión

En este curso, cubrimos los aspectos fundamentales de Microsoft Azure, incluyendo sus servicios principales, herramientas de gestión, características de seguridad y modelos de precios. Al comprender estos conceptos básicos, ahora estás equipado para explorar temas más avanzados de Azure y comenzar a construir y gestionar tus soluciones en la nube de manera efectiva. Continúa practicando con laboratorios prácticos para solidificar tu conocimiento y ganar experiencia práctica.
