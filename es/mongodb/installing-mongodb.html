Introducción

MongoDB es una base de datos NoSQL popular conocida por su flexibilidad y escalabilidad. Esta sección te guiará a través del proceso de instalación de MongoDB en diferentes sistemas operativos. Al final de esta sección, tendrás MongoDB funcionando en tu máquina.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente:

  • Una computadora con conexión a internet.
  • Conocimientos básicos de uso de la línea de comandos o terminal.

Instalación de MongoDB en Windows

Paso 1: Descargar MongoDB

  1. Visita el Centro de Descargas de MongoDB.
  2. Selecciona la versión adecuada para tu sistema operativo Windows.
  3. Haz clic en el botón de descarga para obtener el instalador .msi.

Paso 2: Ejecutar el Instalador

  1. Localiza el archivo .msi descargado y haz doble clic para ejecutarlo.
  2. Sigue las indicaciones de instalación:
    • Elige el tipo de configuración "Completa".
    • Asegúrate de que "Instalar MongoDB como un Servicio" esté marcado.
    • Opcionalmente, puedes elegir instalar MongoDB Compass, una interfaz gráfica para MongoDB.

Paso 3: Verificar la Instalación

  1. Abre el Símbolo del sistema.
  2. Escribe mongo --version y presiona Enter.
  3. Deberías ver la información de la versión de MongoDB, confirmando la instalación.
C:\> mongo --version
MongoDB shell version v4.4.6

Instalación de MongoDB en macOS

Paso 1: Instalar Homebrew

Si no tienes Homebrew instalado, abre Terminal y ejecuta el siguiente comando:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Paso 2: Instalar MongoDB

  1. Abre Terminal.
  2. Ejecuta los siguientes comandos para agregar la fórmula de MongoDB e instalarla:
brew tap mongodb/brew
brew install mongodb-community@4.4

Paso 3: Iniciar MongoDB

  1. Para iniciar MongoDB, ejecuta:
brew services start mongodb/brew/mongodb-community
  1. Verifica la instalación ejecutando:
mongo --version

Deberías ver la información de la versión de MongoDB.

Instalación de MongoDB en Linux

Paso 1: Importar la Clave Pública

  1. Abre Terminal.
  2. Ejecuta el siguiente comando para importar la clave pública GPG de MongoDB:
wget -qO - https://www.mongodb.org/static/pgp/server-4.4.asc | sudo apt-key add -

Paso 2: Crear un Archivo de Lista para MongoDB

  1. Crea el archivo de lista usando el siguiente comando:
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/4.4 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-4.4.list

Paso 3: Instalar Paquetes de MongoDB

  1. Actualiza la lista de paquetes e instala MongoDB:
sudo apt-get update
sudo apt-get install -y mongodb-org

Paso 4: Iniciar MongoDB

  1. Inicia MongoDB usando el siguiente comando:
sudo systemctl start mongod
  1. Verifica la instalación ejecutando:
mongo --version

Deberías ver la información de la versión de MongoDB.

Conclusión

En esta sección, has aprendido cómo instalar MongoDB en Windows, macOS y Linux. Asegurarte de que MongoDB esté correctamente instalado es el primer paso para utilizar sus potentes características para la gestión de bases de datos. En las siguientes secciones, exploraremos cómo configurar y usar MongoDB para diversas aplicaciones.
