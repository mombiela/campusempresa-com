Introducción a PyMongo

PyMongo es una distribución de Python que contiene herramientas para trabajar con MongoDB, una base de datos NoSQL. Esta sección cubrirá los conceptos básicos de PyMongo, incluyendo la instalación y configuración inicial.

Instalación

Para instalar PyMongo, puedes usar pip, el instalador de paquetes de Python:

pip install pymongo

Conectarse a MongoDB

Para conectarse a un servidor MongoDB, necesitas crear una instancia de MongoClient:

from pymongo import MongoClient

client = MongoClient('mongodb://localhost:27017/')
  • MongoClient es el punto de entrada principal al servidor MongoDB.
  • La cadena de conexión 'mongodb://localhost:27017/' especifica que MongoDB se está ejecutando en la máquina local en el puerto predeterminado 27017.

Operaciones Básicas

Esta sección cubrirá las operaciones básicas CRUD (Crear, Leer, Actualizar, Eliminar) usando PyMongo.

Creando una Base de Datos y Colección

En MongoDB, las bases de datos contienen colecciones, y las colecciones contienen documentos.

db = client['mydatabase']
collection = db['mycollection']
  • db es una referencia a la base de datos llamada mydatabase.
  • collection es una referencia a la colección llamada mycollection.

Insertando Documentos

Puedes insertar documentos en una colección usando los métodos insert_one o insert_many.

document = {"name": "Alice", "age": 25, "city": "New York"}
collection.insert_one(document)
  • insert_one inserta un solo documento.
  • El documento es un diccionario de Python.

Consultando Documentos

Para recuperar documentos, usa el método find.

for doc in collection.find({"name": "Alice"}):
    print(doc)
  • find devuelve un cursor a los documentos que coinciden con la consulta.
  • La consulta es un diccionario que especifica los criterios.

Actualizando Documentos

Para actualizar documentos, usa los métodos update_one o update_many.

collection.update_one({"name": "Alice"}, {"$set": {"age": 26}})
  • El primer argumento es la consulta para hacer coincidir documentos.
  • El segundo argumento especifica la operación de actualización.

Eliminando Documentos

Para eliminar documentos, usa los métodos delete_one o delete_many.

collection.delete_one({"name": "Alice"})
  • El argumento es la consulta para hacer coincidir documentos a eliminar.

Operaciones Avanzadas

Esta sección cubrirá características más avanzadas de PyMongo, como indexación, agregación y transacciones.

Indexación

Los índices mejoran el rendimiento de las consultas.

collection.create_index([("name", pymongo.ASCENDING)])
  • create_index crea un índice en el campo especificado.
  • pymongo.ASCENDING especifica el orden de clasificación.

Agregación

Las operaciones de agregación procesan registros de datos y devuelven resultados calculados.

pipeline = [
    {"$match": {"city": "New York"}},
    {"$group": {"_id": "$city", "average_age": {"$avg": "$age"}}}
]
result = collection.aggregate(pipeline)
for doc in result:
    print(doc)
  • pipeline es una lista de etapas de agregación.
  • $match filtra documentos.
  • $group agrupa documentos por un campo especificado y realiza cálculos.

Transacciones

Las transacciones permiten que múltiples operaciones se ejecuten de manera todo o nada.

with client.start_session() as session:
    with session.start_transaction():
        collection.insert_one({"name": "Bob", "age": 30}, session=session)
        collection.update_one({"name": "Bob"}, {"$set": {"city": "San Francisco"}}, session=session)
  • start_session inicia una nueva sesión.
  • start_transaction inicia una nueva transacción dentro de la sesión.

Conclusión

PyMongo es una herramienta poderosa para interactuar con MongoDB desde Python. Soporta una amplia gama de operaciones, desde CRUD básico hasta características avanzadas como indexación, agregación y transacciones. Al dominar PyMongo, puedes gestionar y manipular eficientemente tus bases de datos MongoDB dentro de tus aplicaciones Python.
