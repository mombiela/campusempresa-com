Introducción a Big Data y MongoDB

  • Big Data: Se refiere a grandes volúmenes de datos que pueden ser estructurados, semi-estructurados o no estructurados. Se caracteriza por las 3 Vs: Volumen, Velocidad y Variedad.
  • MongoDB: Una base de datos NoSQL diseñada para manejar grandes volúmenes de datos. Almacena datos en documentos flexibles, similares a JSON.

Configuración de MongoDB

  • Instalación: Instrucciones para instalar MongoDB en varios sistemas operativos.
  • Comandos Básicos: Iniciar el servidor de MongoDB, conectarse a la shell de MongoDB y operaciones CRUD básicas.

Ejemplo: Operaciones CRUD Básicas

// Insertar un documento
db.users.insertOne({ name: "Alice", age: 25, city: "New York" });

// Leer un documento
db.users.find({ name: "Alice" });

// Actualizar un documento
db.users.updateOne({ name: "Alice" }, { $set: { age: 26 } });

// Eliminar un documento
db.users.deleteOne({ name: "Alice" });

Modelado de Datos en MongoDB

  • Diseño de Esquemas: Entender cómo diseñar esquemas para MongoDB.
  • Normalización vs. Desnormalización: Cuándo usar documentos embebidos vs. referencias.

Ejemplo: Documentos Embebidos vs. Referencias

// Documento Embebido
db.orders.insertOne({
  order_id: 1,
  product: "Laptop",
  customer: { name: "John Doe", address: "123 Main St" }
});

// Referencia
db.customers.insertOne({ customer_id: 1, name: "John Doe", address: "123 Main St" });
db.orders.insertOne({ order_id: 1, product: "Laptop", customer_id: 1 });

Indexación y Optimización del Rendimiento

  • Índices: Cómo crear y usar índices para mejorar el rendimiento de las consultas.
  • Optimización de Consultas: Técnicas para optimizar las consultas en MongoDB.

Ejemplo: Creación de Índices

// Crear un índice en el campo 'name'
db.users.createIndex({ name: 1 });

// Crear un índice compuesto en 'name' y 'age'
db.users.createIndex({ name: 1, age: -1 });

Marco de Agregación

  • Pipeline de Agregación: Usar etapas como $match, $group, $sort y $project para procesar datos.
  • Map-Reduce: Una alternativa al pipeline de agregación para procesamiento de datos complejo.

Ejemplo: Pipeline de Agregación

// Agregación para encontrar la edad promedio de los usuarios en cada ciudad
db.users.aggregate([
  { $group: { _id: "$city", averageAge: { $avg: "$age" } } }
]);

Manejo de Big Data con MongoDB

  • Sharding: Distribuir datos a través de múltiples servidores para manejar grandes conjuntos de datos.
  • Replicación: Asegurar la disponibilidad y redundancia de los datos.

Ejemplo: Sharding

// Habilitar sharding en la base de datos
sh.enableSharding("myDatabase");

// Shard una colección
sh.shardCollection("myDatabase.users", { _id: "hashed" });

Análisis en Tiempo Real

  • Change Streams: Procesamiento de datos en tiempo real usando change streams.
  • Integración con Herramientas de Big Data: Usar MongoDB con herramientas como Apache Spark para análisis en tiempo real.

Ejemplo: Change Streams

// Observar cambios en la colección 'users'
const changeStream = db.users.watch();
changeStream.on("change", (change) => {
  printjson(change);
});

Temas Avanzados

  • Consultas Geoespaciales: Trabajar con datos basados en ubicación.
  • Búsqueda de Texto: Implementar búsqueda de texto completo en MongoDB.
  • Datos de Series Temporales: Manejar y analizar datos de series temporales.

Ejemplo: Consulta Geoespacial

// Crear un índice geoespacial
db.places.createIndex({ location: "2dsphere" });

// Encontrar lugares dentro de 1 km de un punto
db.places.find({
  location: {
    $near: {
      $geometry: { type: "Point", coordinates: [ -73.9667, 40.78 ] },
      $maxDistance: 1000
    }
  }
});

Conclusión

En este curso, cubrimos lo esencial del uso de MongoDB para el Análisis de Big Data. Comenzamos con lo básico de la configuración de MongoDB y la realización de operaciones CRUD, luego avanzamos a temas más avanzados como el modelado de datos, la indexación y el marco de agregación. También exploramos cómo manejar grandes volúmenes de datos con sharding y replicación, y cómo realizar análisis en tiempo real. Finalmente, tocamos temas avanzados como consultas geoespaciales y búsqueda de texto.

Al final de este curso, deberías tener una comprensión sólida de cómo aprovechar MongoDB para el Análisis de Big Data, permitiéndote manejar grandes conjuntos de datos de manera eficiente y realizar análisis de datos complejos.
