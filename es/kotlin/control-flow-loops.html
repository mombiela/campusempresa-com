Introducción

Las declaraciones de control de flujo son esenciales en la programación ya que te permiten dictar el orden en el que se ejecutan las declaraciones. Los bucles son una parte fundamental del control de flujo, permitiéndote ejecutar un bloque de código múltiples veces. En Kotlin, hay varios tipos de bucles, cada uno adecuado para diferentes escenarios.

Tipos de Bucles en Kotlin

Bucle `for`

El bucle for en Kotlin se utiliza para iterar sobre un rango, colección o cualquier cosa que proporcione un iterador.

Sintaxis

for (item in collection) {
    // Código a ejecutar para cada elemento
}

Ejemplo

val numbers = listOf(1, 2, 3, 4, 5)
for (number in numbers) {
    println(number)
}
  • Explicación: Este bucle itera sobre cada elemento en la lista numbers y lo imprime.

Bucle `while`

El bucle while ejecuta un bloque de código mientras una condición especificada sea verdadera.

Sintaxis

while (condition) {
    // Código a ejecutar mientras la condición sea verdadera
}

Ejemplo

var i = 0
while (i < 5) {
    println(i)
    i++
}
  • Explicación: Este bucle imprime números del 0 al 4. El bucle continúa mientras i sea menor que 5.

Bucle `do-while`

El bucle do-while es similar al bucle while, pero garantiza que el bloque de código se ejecutará al menos una vez.

Sintaxis

do {
    // Código a ejecutar
} while (condition)

Ejemplo

var i = 0
do {
    println(i)
    i++
} while (i < 5)
  • Explicación: Este bucle también imprime números del 0 al 4. La diferencia clave es que la condición se verifica después de que se ejecuta el bloque de código.

Conceptos Avanzados de Bucles

Declaraciones de Control de Bucles

Kotlin proporciona varias declaraciones de control para gestionar la ejecución de bucles:

  • break: Termina el bucle inmediatamente.
  • continue: Omite la iteración actual y procede a la siguiente.
  • return: Sale de la función en la que se encuentra el bucle.

Ejemplo con break y continue

for (i in 1..10) {
    if (i == 5) break
    if (i % 2 == 0) continue
    println(i)
}
  • Explicación: Este bucle imprime números impares del 1 al 9. Se detiene completamente cuando i es igual a 5 debido a la declaración break. La declaración continue omite los números pares.

Bucles Anidados

Puedes colocar un bucle dentro de otro bucle. Esto es útil para trabajar con estructuras de datos multidimensionales.

Ejemplo

for (i in 1..3) {
    for (j in 1..3) {
        println("i = $i, j = $j")
    }
}
  • Explicación: Este bucle anidado imprime todas las combinaciones de i y j del 1 al 3.

Etiquetas de Bucles

Kotlin te permite etiquetar bucles y controlar su flujo utilizando estas etiquetas.

Ejemplo

outer@ for (i in 1..3) {
    for (j in 1..3) {
        if (i == 2 && j == 2) break@outer
        println("i = $i, j = $j")
    }
}
  • Explicación: La declaración break@outer termina el bucle exterior cuando i es igual a 2 y j es igual a 2.

Conclusión

Los bucles son una herramienta poderosa en Kotlin que te permiten ejecutar código múltiples veces de manera eficiente. Entender los diferentes tipos de bucles y cómo controlar su flujo es crucial para escribir programas efectivos en Kotlin. Practica usando bucles for, while y do-while, junto con declaraciones de control como break y continue, para volverte competente en la gestión del flujo de control en tus aplicaciones.
