En esta sección, te guiaremos a través del proceso de configuración de un proyecto en Kotlin. Este es un paso esencial antes de que puedas comenzar a codificar y construir aplicaciones. Cubriremos la instalación de las herramientas necesarias, la creación de un nuevo proyecto y la comprensión de la estructura del proyecto.

Herramientas y Configuración del Entorno

Antes de comenzar a codificar en Kotlin, necesitamos configurar nuestro entorno de desarrollo. Aquí están las herramientas clave que necesitarás:

  • Java Development Kit (JDK): Kotlin se ejecuta en la Java Virtual Machine (JVM), por lo que necesitas tener el JDK instalado.
  • Entorno de Desarrollo Integrado (IDE): Recomendamos usar IntelliJ IDEA, que tiene un excelente soporte para Kotlin.

Instalando el JDK

  1. Descargar el JDK: Visita la página de descarga de Oracle JDK o usa una alternativa de código abierto como AdoptOpenJDK.
  2. Instalar el JDK: Sigue las instrucciones de instalación para tu sistema operativo.
  3. Verificar la Instalación: Abre una terminal o símbolo del sistema y escribe: 
    java -version
    Deberías ver la versión del JDK que instalaste.

Instalando IntelliJ IDEA

  1. Descargar IntelliJ IDEA: Visita el sitio web de JetBrains y descarga la Edición Comunitaria (gratuita) o la Edición Ultimate (de pago).
  2. Instalar IntelliJ IDEA: Sigue las instrucciones de instalación para tu sistema operativo.
  3. Lanzar IntelliJ IDEA: Abre la aplicación después de la instalación.

Creando un Nuevo Proyecto en Kotlin

Una vez que hayas instalado las herramientas necesarias, puedes crear un nuevo proyecto en Kotlin en IntelliJ IDEA.

  1. Abrir IntelliJ IDEA: Lanza el IDE.

  2. Crear un Nuevo Proyecto:

    • Haz clic en New Project desde la pantalla de bienvenida.
    • Selecciona Kotlin de la lista de tipos de proyecto.
    • Elige JVM | IDEA como el tipo de proyecto.
    • Haz clic en Next.

  3. Configurar la Configuración del Proyecto:

    • Ingresa un nombre para el proyecto.
    • Elige una ubicación para guardar el proyecto.
    • Selecciona el JDK que instalaste anteriormente.
    • Haz clic en Finish.

  4. Estructura del Proyecto:

    • IntelliJ IDEA creará un proyecto con la siguiente estructura: 
      MyKotlinProject/
├── .idea/
├── src/
│   └── Main.kt
├── out/
└── build.gradle (si usas Gradle)

Comprendiendo la Estructura del Proyecto

Desglosemos los componentes clave de la estructura del proyecto:

  • .idea/: Contiene archivos de configuración de IntelliJ IDEA.
  • src/: Aquí es donde residirán tus archivos fuente de Kotlin.
    • Main.kt: El archivo principal de Kotlin donde puedes comenzar a escribir tu código.
  • out/: Este directorio contendrá los archivos de salida compilados.
  • build.gradle: Si estás usando Gradle como tu herramienta de construcción, este archivo contiene la configuración de la construcción.

Ejemplo: Hola Mundo en Kotlin

Escribamos un simple programa de "Hola, Mundo!" para asegurarnos de que todo esté configurado correctamente.

  1. Abrir Main.kt: Navega a src y abre Main.kt.
  2. Escribir el Código: 
    fun main() {
    println("¡Hola, Mundo!")
}
  3. Ejecutar el Programa:
    • Haz clic derecho en Main.kt.
    • Selecciona Run 'MainKt'.

Deberías ver la salida "¡Hola, Mundo!" en la ventana de Ejecución en la parte inferior del IDE.

Conclusión

En esta sección, cubrimos los pasos esenciales para configurar un proyecto en Kotlin, incluyendo la instalación del JDK y IntelliJ IDEA, la creación de un nuevo proyecto y la comprensión de la estructura del proyecto. También escribimos y ejecutamos un simple programa de "Hola, Mundo!" para verificar nuestra configuración. Con tu entorno de desarrollo listo, ahora puedes pasar a escribir programas más complejos en Kotlin.
