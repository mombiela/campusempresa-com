Introducción

En esta sección, cubriremos los conceptos básicos de variables y tipos de datos en Kotlin. Comprender estos conceptos es fundamental para cualquier programador de Kotlin, ya seas principiante o estés avanzando en tus habilidades.

Variables

Las variables son contenedores para almacenar valores de datos. En Kotlin, puedes declarar variables usando val o var.

  • val (valor): Variable inmutable, lo que significa que una vez asignado un valor, no puede ser cambiado.
  • var (variable): Variable mutable, lo que significa que su valor puede ser cambiado.

Declaración de Variables

Así es como puedes declarar variables en Kotlin:

val immutableVariable: String = "¡Hola, Kotlin!"
var mutableVariable: Int = 42
  • immutableVariable es un val, por lo que su valor no puede ser cambiado después de la inicialización.
  • mutableVariable es un var, por lo que su valor puede ser cambiado.

Inferencia de Tipos

Kotlin puede inferir el tipo de la variable a partir del valor asignado, por lo que puedes omitir la declaración del tipo:

val inferredString = "¡Hola, Kotlin!"
var inferredInt = 42

Tipos de Datos

Kotlin tiene un conjunto rico de tipos de datos. Aquí están los principales tipos de datos que encontrarás:

  • Tipos Numéricos: Byte, Short, Int, Long, Float, Double
  • Carácter: Char
  • Booleano: Boolean
  • Cadena: String
  • Arreglos: Array

Tipos Numéricos

Kotlin proporciona una variedad de tipos numéricos para manejar diferentes rangos y precisiones.

| Tipo | Tamaño (bits) | Valor Mínimo | Valor Máximo | |-------|---------------|-------------------------------|-------------------------------| | Byte | 8 | -128 | 127 | | Short | 16 | -32,768 | 32,767 | | Int | 32 | -2,147,483,648 | 2,147,483,647 | | Long | 64 | -9,223,372,036,854,775,808 | 9,223,372,036,854,775,807 | | Float | 32 | 1.4E-45 | 3.4028235E38 | | Double| 64 | 4.9E-324 | 1.7976931348623157E308 |

Tipos de Carácter y Booleano

  • Char: Representa un solo carácter.
  • Boolean: Representa un valor que puede ser true o false.
val charExample: Char = 'A'
val booleanExample: Boolean = true

Tipo Cadena

Las cadenas en Kotlin son secuencias de caracteres. Son inmutables.

val stringExample: String = "¡Hola, Kotlin!"

Arreglos

Los arreglos en Kotlin están representados por la clase Array. Puedes crear un arreglo usando la función arrayOf.

val intArray: Array<Int> = arrayOf(1, 2, 3, 4, 5)
val stringArray: Array<String> = arrayOf("Kotlin", "Java", "Swift")

Ejemplos y Ejercicios

Ejemplo 1: Declaración Básica de Variables

fun main() {
    val name: String = "John"
    var age: Int = 25

    println("Nombre: $name")
    println("Edad: $age")

    // Intentar cambiar el valor de un val causará un error en tiempo de compilación
    // name = "Doe" // Descomentar esta línea causará un error

    // Cambiar el valor de un var está permitido
    age = 26
    println("Edad Actualizada: $age")
}

Ejercicio 1: Declaración de Variables

  1. Declara una variable val para almacenar tu color favorito.
  2. Declara una variable var para almacenar tu edad actual.
  3. Imprime ambas variables.

Ejemplo 2: Uso de Diferentes Tipos de Datos

fun main() {
    val byteValue: Byte = 127
    val shortValue: Short = 32767
    val intValue: Int = 2147483647
    val longValue: Long = 9223372036854775807L
    val floatValue: Float = 3.14F
    val doubleValue: Double = 3.141592653589793

    println("Valor Byte: $byteValue")
    println("Valor Short: $shortValue")
    println("Valor Int: $intValue")
    println("Valor Long: $longValue")
    println("Valor Float: $floatValue")
    println("Valor Double: $doubleValue")
}

Ejercicio 2: Tipos de Datos

  1. Declara variables de cada tipo numérico y asígnales valores apropiados.
  2. Imprime cada variable.

Conclusión

Comprender las variables y los tipos de datos es crucial para programar en Kotlin. Al dominar estos conceptos básicos, estableces una base sólida para temas más avanzados. Recuerda practicar la declaración y el uso de diferentes tipos de variables para solidificar tu comprensión.
