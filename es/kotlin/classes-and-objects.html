Introducción a Clases y Objetos

En Kotlin, las clases y los objetos son bloques de construcción fundamentales de la programación orientada a objetos. Te permiten crear estructuras de código reutilizables y modelar entidades del mundo real.

Conceptos Clave

  • Clase: Un plano para crear objetos. Define propiedades y comportamientos.
  • Objeto: Una instancia de una clase. Representa una implementación específica del plano de la clase.
  • Constructor: Una función especial que inicializa un objeto.
  • Propiedades: Variables dentro de una clase que contienen datos.
  • Métodos: Funciones dentro de una clase que definen comportamientos.

Definiendo una Clase

Una clase en Kotlin se define usando la palabra clave class seguida del nombre de la clase.

class Person {
    var name: String = ""
    var age: Int = 0

    fun introduce() {
        println("Hi, my name is $name and I am $age years old.")
    }
}

Explicación

  • class Person { ... }: Define una clase llamada Person.
  • var name: String = "": Una propiedad name de tipo String con un valor predeterminado.
  • var age: Int = 0: Una propiedad age de tipo Int con un valor predeterminado.
  • fun introduce() { ... }: Un método introduce que imprime un mensaje.

Creando Objetos

Para crear un objeto de una clase, usa la palabra clave new seguida del nombre de la clase y paréntesis.

fun main() {
    val person = Person()
    person.name = "John"
    person.age = 30
    person.introduce()
}

Explicación

  • val person = Person(): Crea un objeto de la clase Person.
  • person.name = "John": Establece la propiedad name del objeto person.
  • person.age = 30: Establece la propiedad age del objeto person.
  • person.introduce(): Llama al método introduce en el objeto person.

Constructores

Kotlin proporciona un constructor primario y constructores secundarios para inicializar objetos.

Constructor Primario

El constructor primario es parte del encabezado de la clase.

class Person(val name: String, var age: Int) {
    fun introduce() {
        println("Hi, my name is $name and I am $age years old.")
    }
}

Explicación

  • class Person(val name: String, var age: Int) { ... }: Define un constructor primario con los parámetros name y age.

Usando el Constructor Primario

fun main() {
    val person = Person("John", 30)
    person.introduce()
}

Constructor Secundario

Una clase también puede tener constructores secundarios.

class Person {
    var name: String
    var age: Int

    constructor(name: String, age: Int) {
        this.name = name
        this.age = age
    }

    fun introduce() {
        println("Hi, my name is $name and I am $age years old.")
    }
}

Explicación

  • constructor(name: String, age: Int) { ... }: Define un constructor secundario.

Usando el Constructor Secundario

fun main() {
    val person = Person("John", 30)
    person.introduce()
}

Herencia

Kotlin soporta la herencia, permitiendo que una clase herede propiedades y métodos de otra clase.

Clase Base

open class Animal {
    open fun sound() {
        println("Animal makes a sound")
    }
}

Clase Derivada

class Dog : Animal() {
    override fun sound() {
        println("Dog barks")
    }
}

Explicación

  • open class Animal { ... }: Define una clase base Animal que puede ser heredada.
  • class Dog : Animal() { ... }: Define una clase derivada Dog que hereda de Animal.
  • override fun sound() { ... }: Sobrescribe el método sound en la clase Dog.

Usando Herencia

fun main() {
    val dog = Dog()
    dog.sound()  // Output: Dog barks
}

Clases de Datos

Kotlin proporciona un tipo especial de clase llamada clase de datos, que se usa para contener datos.

Definiendo una Clase de Datos

data class User(val name: String, val age: Int)

Explicación

  • data class User(val name: String, val age: Int): Define una clase de datos User con las propiedades name y age.

Usando una Clase de Datos

fun main() {
    val user = User("Alice", 25)
    println(user)  // Output: User(name=Alice, age=25)
}

Resumen

En este tema, cubrimos los conceptos básicos de clases y objetos en Kotlin, incluyendo:

  • Definir clases y crear objetos.
  • Usar constructores primarios y secundarios.
  • Implementar herencia.
  • Utilizar clases de datos para contener datos.

Entender estos conceptos es crucial para construir código robusto y reutilizable en Kotlin. Practica creando tus propias clases y objetos para solidificar tu comprensión.
