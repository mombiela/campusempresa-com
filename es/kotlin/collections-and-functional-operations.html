Introducción a las Colecciones

Kotlin proporciona un conjunto rico de tipos de colecciones que son tanto mutables como inmutables. Las colecciones se utilizan para almacenar grupos de objetos relacionados.

  • List: Una colección ordenada de elementos.
  • Set: Una colección de elementos únicos.
  • Map: Una colección de pares clave-valor.

List

Una List es una colección ordenada que permite elementos duplicados.

val fruits = listOf("Apple", "Banana", "Cherry")
println(fruits) // Output: [Apple, Banana, Cherry]

MutableList

Una MutableList es una List con métodos para agregar, eliminar y actualizar elementos.

val fruits = mutableListOf("Apple", "Banana")
fruits.add("Cherry")
println(fruits) // Output: [Apple, Banana, Cherry]

Set

Un Set es una colección que no contiene elementos duplicados.

val numbers = setOf(1, 2, 3, 3)
println(numbers) // Output: [1, 2, 3]

MutableSet

Un MutableSet es un Set con métodos para agregar, eliminar y actualizar elementos.

val numbers = mutableSetOf(1, 2, 3)
numbers.add(4)
println(numbers) // Output: [1, 2, 3, 4]

Map

Un Map es una colección de pares clave-valor.

val countryCodes = mapOf("US" to "Estados Unidos", "CA" to "Canadá")
println(countryCodes) // Output: {US=Estados Unidos, CA=Canadá}

MutableMap

Un MutableMap es un Map con métodos para agregar, eliminar y actualizar pares clave-valor.

val countryCodes = mutableMapOf("US" to "Estados Unidos")
countryCodes["CA"] = "Canadá"
println(countryCodes) // Output: {US=Estados Unidos, CA=Canadá}

Operaciones Funcionales en Colecciones

Las colecciones de Kotlin vienen con una variedad de operaciones funcionales que facilitan la manipulación de datos.

  • map: Transforma cada elemento en la colección.
  • filter: Filtra elementos basados en una condición.
  • reduce: Reduce la colección a un solo valor.
  • forEach: Realiza una acción para cada elemento.
  • sortedBy: Ordena la colección basada en una función selectora.

map

Transforma cada elemento en la colección.

val numbers = listOf(1, 2, 3)
val doubled = numbers.map { it * 2 }
println(doubled) // Output: [2, 4, 6]

filter

Filtra elementos basados en una condición.

val numbers = listOf(1, 2, 3, 4, 5)
val evenNumbers = numbers.filter { it % 2 == 0 }
println(evenNumbers) // Output: [2, 4]

reduce

Reduce la colección a un solo valor.

val numbers = listOf(1, 2, 3, 4)
val sum = numbers.reduce { acc, num -> acc + num }
println(sum) // Output: 10

forEach

Realiza una acción para cada elemento.

val fruits = listOf("Apple", "Banana", "Cherry")
fruits.forEach { println(it) }
// Output:
// Apple
// Banana
// Cherry

sortedBy

Ordena la colección basada en una función selectora.

val fruits = listOf("Banana", "Apple", "Cherry")
val sortedFruits = fruits.sortedBy { it }
println(sortedFruits) // Output: [Apple, Banana, Cherry]

Conclusión

El marco de colecciones de Kotlin proporciona una forma poderosa y flexible de trabajar con grupos de datos. Entender los diferentes tipos de colecciones y sus operaciones funcionales te permite escribir código más conciso y legible. Al dominar estos conceptos, puedes manipular datos de manera eficiente y realizar transformaciones complejas con facilidad.
