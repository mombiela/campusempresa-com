Introducción a los Condicionales

Los condicionales son fundamentales en la programación ya que te permiten ejecutar diferentes códigos basados en ciertas condiciones. En Kotlin, los condicionales se manejan principalmente usando las declaraciones if, else if y else, así como la expresión when.

Declaraciones If-Else

La declaración if es la forma más básica de condicional. Te permite ejecutar un bloque de código solo si una condición especificada es verdadera.

Sintaxis

if (condition) {
    // código a ejecutar si la condición es verdadera
} else {
    // código a ejecutar si la condición es falsa
}

Ejemplo

val number = 10

if (number > 0) {
    println("El número es positivo")
} else {
    println("El número no es positivo")
}

Explicación

  • if (number > 0): Verifica si number es mayor que 0.
  • println("El número es positivo"): Se ejecuta si la condición es verdadera.
  • else: Se ejecuta si la condición es falsa.
  • println("El número no es positivo"): Se ejecuta si la condición es falsa.

Declaraciones Else If

Cuando tienes múltiples condiciones para verificar, puedes usar else if.

Sintaxis

if (condition1) {
    // código a ejecutar si condition1 es verdadera
} else if (condition2) {
    // código a ejecutar si condition2 es verdadera
} else {
    // código a ejecutar si ninguna de las condiciones anteriores es verdadera
}

Ejemplo

val number = 0

if (number > 0) {
    println("El número es positivo")
} else if (number < 0) {
    println("El número es negativo")
} else {
    println("El número es cero")
}

Explicación

  • else if (number < 0): Verifica si number es menor que 0.
  • println("El número es negativo"): Se ejecuta si la segunda condición es verdadera.
  • else: Se ejecuta si ninguna de las condiciones es verdadera.
  • println("El número es cero"): Se ejecuta si ninguna de las condiciones es verdadera.

Expresión When

La expresión when en Kotlin es una alternativa poderosa a la cadena if-else if-else. Es similar a la declaración switch en otros lenguajes.

Sintaxis

when (variable) {
    value1 -> {
        // código a ejecutar si variable == value1
    }
    value2 -> {
        // código a ejecutar si variable == value2
    }
    else -> {
        // código a ejecutar si ninguna de las condiciones anteriores es verdadera
    }
}

Ejemplo

val dayOfWeek = 3

val dayName = when (dayOfWeek) {
    1 -> "Lunes"
    2 -> "Martes"
    3 -> "Miércoles"
    4 -> "Jueves"
    5 -> "Viernes"
    6 -> "Sábado"
    7 -> "Domingo"
    else -> "Día inválido"
}

println(dayName)

Explicación

  • when (dayOfWeek): Evalúa el valor de dayOfWeek.
  • 1 -> "Lunes": Se ejecuta si dayOfWeek es 1.
  • else -> "Día inválido": Se ejecuta si ninguna de las condiciones coincide.

Uso Avanzado de When

La expresión when también puede usarse sin un argumento, actuando como una cadena if-else avanzada.

Ejemplo

val number = -10

val result = when {
    number > 0 -> "Positivo"
    number < 0 -> "Negativo"
    else -> "Cero"
}

println(result)

Explicación

  • when {}: No se proporciona ningún argumento.
  • number > 0 -> "Positivo": Se ejecuta si number es mayor que 0.
  • number < 0 -> "Negativo": Se ejecuta si number es menor que 0.
  • else -> "Cero": Se ejecuta si ninguna de las condiciones coincide.

Conclusión

Los condicionales son esenciales para controlar el flujo de tus programas en Kotlin. Las declaraciones if-else son directas y útiles para condiciones simples, mientras que la expresión when ofrece una forma más legible y flexible de manejar múltiples condiciones. Entender y usar estos constructos de manera efectiva te permitirá escribir código más dinámico y receptivo.
