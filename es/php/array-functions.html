Introducción a los Arrays en PHP

Los arrays son uno de los tipos de datos más poderosos en PHP. Te permiten almacenar múltiples valores en una sola variable, los cuales pueden ser accedidos usando índices o claves.

Conceptos Clave

  • Arrays Indexados: Arrays con índices numéricos.
  • Arrays Asociativos: Arrays con claves nombradas.
  • Arrays Multidimensionales: Arrays que contienen uno o más arrays.

Ejemplo

// Array Indexado
$fruits = array("Apple", "Banana", "Cherry");

// Array Asociativo
$ages = array("Peter" => 35, "Ben" => 37, "Joe" => 43);

// Array Multidimensional
$contacts = array(
    array("name" => "Peter", "email" => "peter@example.com"),
    array("name" => "Ben", "email" => "ben@example.com")
);

Funciones Básicas de Array

`count()`

La función count() devuelve el número de elementos en un array.

$fruits = array("Apple", "Banana", "Cherry");
echo count($fruits); // Salida: 3

`array_push()`

La función array_push() añade uno o más elementos al final de un array.

$fruits = array("Apple", "Banana");
array_push($fruits, "Cherry", "Date");
print_r($fruits); // Salida: Array ( [0] => Apple [1] => Banana [2] => Cherry [3] => Date )

`array_pop()`

La función array_pop() elimina el último elemento de un array.

$fruits = array("Apple", "Banana", "Cherry");
array_pop($fruits);
print_r($fruits); // Salida: Array ( [0] => Apple [1] => Banana )

Funciones Intermedias de Array

`array_merge()`

La función array_merge() fusiona uno o más arrays en un solo array.

$array1 = array("color" => "red", 2, 4);
$array2 = array("a", "b", "color" => "green", "shape" => "trapezoid", 4);
$result = array_merge($array1, $array2);
print_r($result);
// Salida: Array ( [color] => green [0] => 2 [1] => 4 [2] => a [3] => b [shape] => trapezoid [4] => 4 )

`array_diff()`

La función array_diff() compara arrays y devuelve las diferencias.

$array1 = array("a" => "green", "red", "blue", "red");
$array2 = array("b" => "green", "yellow", "red");
$result = array_diff($array1, $array2);
print_r($result); // Salida: Array ( [1] => blue )

`array_keys()`

La función array_keys() devuelve todas las claves de un array.

$array = array("color" => "blue", "size" => "medium", "shape" => "circle");
$keys = array_keys($array);
print_r($keys); // Salida: Array ( [0] => color [1] => size [2] => shape )

Funciones Avanzadas de Array

`array_map()`

La función array_map() aplica una función de callback a los elementos de los arrays dados.

function square($n) {
    return $n * $n;
}
$numbers = array(1, 2, 3, 4, 5);
$squaredNumbers = array_map("square", $numbers);
print_r($squaredNumbers); // Salida: Array ( [0] => 1 [1] => 4 [2] => 9 [3] => 16 [4] => 25 )

`array_filter()`

La función array_filter() filtra elementos de un array usando una función de callback.

function isEven($n) {
    return $n % 2 == 0;
}
$numbers = array(1, 2, 3, 4, 5, 6);
$evenNumbers = array_filter($numbers, "isEven");
print_r($evenNumbers); // Salida: Array ( [1] => 2 [3] => 4 [5] => 6 )

`array_reduce()`

La función array_reduce() reduce un array a un solo valor usando una función de callback.

function sum($carry, $item) {
    $carry += $item;
    return $carry;
}
$numbers = array(1, 2, 3, 4, 5);
$total = array_reduce($numbers, "sum");
echo $total; // Salida: 15

Conclusión

PHP proporciona un conjunto rico de funciones para manipular arrays, desde operaciones básicas como contar elementos y añadir/eliminar ítems, hasta operaciones más avanzadas como mapear, filtrar y reducir arrays. Entender estas funciones mejorará significativamente tu capacidad para trabajar con arrays de manera efectiva en PHP. Practica usando estas funciones con diferentes tipos de arrays para volverte competente en el manejo de estructuras de datos de arrays.
