Introducción a Laravel

Laravel es un framework PHP poderoso y flexible diseñado para hacer el desarrollo web más fácil y agradable. Sigue el patrón arquitectónico Modelo-Vista-Controlador (MVC) y proporciona un conjunto rico de características para construir aplicaciones web modernas.

Conceptos Clave

  • Arquitectura MVC: Separa la lógica de la aplicación en tres componentes principales: Modelo, Vista y Controlador.
  • Enrutamiento: Define cómo la aplicación responde a varias solicitudes HTTP.
  • Eloquent ORM: Proporciona una forma simple y elegante de interactuar con la base de datos.
  • Motor de Plantillas Blade: Un motor de plantillas poderoso para crear vistas dinámicas.
  • Artisan CLI: Una interfaz de línea de comandos que proporciona numerosos comandos útiles para el desarrollo de aplicaciones.

Configuración de Laravel

Para empezar con Laravel, necesitas configurar tu entorno de desarrollo.

Requisitos Previos

  • PHP >= 7.3
  • Composer (Administrador de dependencias para PHP)
  • Un servidor web como Apache o Nginx
  • Una base de datos como MySQL o SQLite

Pasos de Instalación

  1. Instalar Composer: Si aún no has instalado Composer, descárgalo e instálalo desde getcomposer.org.
  2. Crear un Nuevo Proyecto Laravel: 
    composer create-project --prefer-dist laravel/laravel myLaravelApp
  3. Navegar al Directorio del Proyecto: 
    cd myLaravelApp
  4. Servir la Aplicación: 
    php artisan serve
    Este comando iniciará un servidor de desarrollo local en http://localhost:8000.

Estructura de Directorios de Laravel

Entender la estructura de directorios es crucial para un desarrollo eficiente.

  • app/: Contiene el código principal de tu aplicación.
  • bootstrap/: Contiene el script de arranque de la aplicación.
  • config/: Contiene archivos de configuración.
  • database/: Contiene migraciones de base de datos, fábricas de modelos y semillas.
  • public/: El punto de entrada para la aplicación, contiene el archivo index.php.
  • resources/: Contiene vistas, activos sin procesar (CSS, JS) y archivos de idioma.
  • routes/: Contiene todas las definiciones de rutas.
  • storage/: Contiene plantillas Blade compiladas, sesiones basadas en archivos, cachés de archivos y otros archivos generados por el framework.
  • tests/: Contiene pruebas automatizadas.
  • vendor/: Contiene dependencias de Composer.

Enrutamiento Básico

El enrutamiento en Laravel es sencillo y poderoso. Las rutas se definen en el archivo routes/web.php.

Ejemplo de Ruta

Route::get('/', function () {
    return view('welcome');
});

Esta ruta escucha solicitudes GET a la URL raíz (/) y devuelve la vista welcome.

Parámetros de Ruta

Route::get('/user/{id}', function ($id) {
    return 'Usuario '.$id;
});

Esta ruta captura un parámetro ({id}) de la URL y lo pasa al cierre.

Controladores

Los controladores son responsables de manejar las solicitudes entrantes y devolver respuestas.

Creación de un Controlador

Usa Artisan CLI para crear un controlador:

php artisan make:controller UserController

Este comando crea un nuevo archivo de controlador en el directorio app/Http/Controllers.

Método de Controlador de Ejemplo

namespace App\Http\Controllers;

use Illuminate\Http\Request;

class UserController extends Controller
{
    public function show($id)
    {
        return 'Usuario '.$id;
    }
}

Define una ruta que use este método del controlador:

Route::get('/user/{id}', [UserController::class, 'show']);

Eloquent ORM

Eloquent proporciona una sintaxis elegante para interactuar con la base de datos.

Definición de un Modelo

Crea un modelo usando Artisan:

php artisan make:model User

Este comando crea un nuevo archivo de modelo en el directorio app/Models.

Uso Básico de Eloquent

use App\Models\User;

// Recuperar todos los usuarios
$users = User::all();

// Recuperar un usuario por ID
$user = User::find(1);

// Crear un nuevo usuario
$user = new User;
$user->name = 'John Doe';
$user->email = 'john@example.com';
$user->save();

Motor de Plantillas Blade

Blade es el poderoso motor de plantillas de Laravel.

Sintaxis Básica de Blade

Crea un archivo de vista Blade en resources/views/welcome.blade.php:

<!DOCTYPE html>
<html>
<head>
    <title>Laravel</title>
</head>
<body>
    <h1>Hola, {{ $name }}</h1>
</body>
</html>

Pasa datos a la vista desde una ruta o controlador:

Route::get('/', function () {
    return view('welcome', ['name' => 'John Doe']);
});

Conclusión

Laravel es un framework robusto y flexible que simplifica el desarrollo web en PHP. Al entender sus conceptos básicos, configurar el entorno y explorar sus características como enrutamiento, controladores, Eloquent ORM y plantillas Blade, puedes construir aplicaciones web poderosas de manera eficiente. Continúa explorando la extensa documentación de Laravel y los recursos de la comunidad para profundizar tu conocimiento y habilidades.
