Introducción

En esta sección, proporcionaremos una descripción general del proyecto en el que trabajarás a lo largo de este curso de Webpack. También delinearemos los requisitos necesarios para completar con éxito el proyecto. Esto te ayudará a entender el alcance del proyecto y asegurarte de tener todo lo que necesitas antes de sumergirte en los detalles técnicos.

Descripción General del Proyecto

El proyecto para este curso es una aplicación web simple que evolucionará en complejidad a medida que avancemos de niveles principiantes a avanzados. Al final del curso, tendrás una aplicación web completamente optimizada y lista para producción.

Características Clave del Proyecto

  • Gestión de Activos Estáticos: Gestionar eficientemente archivos CSS, JavaScript e imágenes.
  • Agrupación de Módulos: Agrupar módulos de JavaScript para un mejor rendimiento.
  • División de Código: Dividir el código en fragmentos más pequeños para mejorar los tiempos de carga.
  • Servidor de Desarrollo: Configurar un servidor de desarrollo local con reemplazo de módulos en caliente.
  • Optimización para Producción: Optimizar la aplicación para producción con minificación y tree shaking.

Requisitos

Antes de comenzar el proyecto, asegúrate de tener los siguientes prerrequisitos y herramientas instaladas:

Prerrequisitos

  • Conocimientos Básicos de JavaScript: Comprensión de la sintaxis ES6+.
  • HTML y CSS Básicos: Familiaridad con la creación y el estilo de páginas web.
  • Node.js y npm: Asegúrate de tener Node.js y npm instalados en tu máquina. Puedes descargarlos desde el sitio oficial de Node.js.

Herramientas y Bibliotecas

  • Editor de Código: Un editor de código como Visual Studio Code, Sublime Text o Atom.
  • Navegador: Un navegador web moderno como Google Chrome o Firefox para pruebas.
  • Terminal: Acceso a una terminal o símbolo del sistema para ejecutar comandos.

Configuración Inicial

  1. Crear un Directorio de Proyecto: Crea un nuevo directorio para tu proyecto.

    mkdir webpack-project
cd webpack-project

  2. Inicializar npm: Inicializa un nuevo proyecto npm.

    npm init -y

    Esto creará un archivo package.json en tu directorio de proyecto.

  3. Instalar Webpack y Webpack CLI: Instala Webpack y su interfaz de línea de comandos.

    npm install --save-dev webpack webpack-cli

Estructura Básica del Proyecto

Aquí tienes una estructura básica del directorio del proyecto:

webpack-project/
├── src/
│   ├── index.js
│   └── style.css
├── dist/
├── package.json
└── webpack.config.js
  • src/: Este directorio contendrá tus archivos fuente.
    • index.js: El archivo principal de JavaScript.
    • style.css: El archivo principal de CSS.
  • dist/: Este directorio contendrá los archivos de salida agrupados.
  • package.json: Contiene los metadatos y dependencias del proyecto.
  • webpack.config.js: El archivo de configuración para Webpack.

Ejemplo de Archivo de Configuración

Aquí tienes un ejemplo básico de un archivo webpack.config.js:

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist')
  },
  module: {
    rules: [
      {
        test: /\.css$/,
        use: ['style-loader', 'css-loader']
      }
    ]
  }
};
  • entry: Especifica el punto de entrada de la aplicación.
  • output: Define el archivo y directorio de salida.
  • module: Contiene reglas sobre cómo deben tratarse los diferentes tipos de módulos.

Conclusión

En esta sección, cubrimos la descripción general del proyecto y los requisitos necesarios para comenzar. Asegúrate de tener todos los prerrequisitos y herramientas instaladas antes de pasar a las siguientes secciones. Con la configuración inicial completa, ahora estás listo para sumergirte en el mundo de Webpack y comenzar a construir tu aplicación web.
