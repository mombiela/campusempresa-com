Introducción

Webpack es un potente empaquetador de módulos para aplicaciones JavaScript. Toma módulos con dependencias y genera activos estáticos que representan esos módulos. Esta sección te guiará a través de la configuración de tu primer proyecto con Webpack desde cero.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente instalado:

  • Node.js (se recomienda la versión LTS)
  • npm (viene con Node.js)

Guía Paso a Paso

1. Inicializa Tu Proyecto

Primero, crea un nuevo directorio para tu proyecto y navega dentro de él:

mkdir my-webpack-project
cd my-webpack-project

Inicializa un nuevo proyecto Node.js:

npm init -y

Este comando crea un archivo package.json con configuraciones predeterminadas.

2. Instala Webpack y Webpack CLI

A continuación, instala Webpack y Webpack CLI como dependencias de desarrollo:

npm install --save-dev webpack webpack-cli

3. Crea la Estructura del Proyecto

Crea la siguiente estructura de directorios:

my-webpack-project/
├── src/
│   └── index.js
├── dist/
├── package.json
└── webpack.config.js

4. Configura Webpack

Crea un archivo webpack.config.js en la raíz de tu proyecto:

const path = require('path');

module.exports = {
  entry: './src/index.js',
  output: {
    filename: 'bundle.js',
    path: path.resolve(__dirname, 'dist'),
  },
  mode: 'development',
};
  • entry: El punto de entrada de tu aplicación.
  • output: Dónde Webpack debe generar los archivos empaquetados.
  • mode: Configurado en 'development' para facilitar la depuración.

5. Añade un Script a package.json

Añade un script de construcción a tu package.json:

"scripts": {
  "build": "webpack"
}

6. Crea un Archivo JavaScript Simple

Crea un archivo index.js simple en el directorio src:

console.log('Hello, Webpack!');

7. Construye el Proyecto

Ejecuta el script de construcción:

npm run build

Este comando generará un archivo bundle.js en el directorio dist.

8. Crea un Archivo HTML

Crea un archivo index.html en el directorio dist para incluir el JavaScript empaquetado:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Proyecto Webpack</title>
</head>
<body>
  <script src="bundle.js"></script>
</body>
</html>

9. Abre el Archivo HTML

Abre index.html en tu navegador para ver el resultado. Deberías ver "Hello, Webpack!" en la consola.

Resumen

En esta sección, aprendiste cómo configurar un proyecto básico con Webpack. Tú:

  • Inicializaste un proyecto Node.js.
  • Instalaste Webpack y Webpack CLI.
  • Creaste una estructura de proyecto.
  • Configuraste Webpack.
  • Construiste y serviste una aplicación JavaScript simple.

Esta configuración fundamental servirá como base para configuraciones y optimizaciones más avanzadas de Webpack en lecciones futuras.
