Introduction to Conversions in SEM

Conversions are specific actions that users take on your website that are valuable to your business, such as completing a purchase, filling out a contact form, or subscribing to a newsletter. Setting up conversions correctly is crucial to measure the success of your SEM (Search Engine Marketing) campaigns.

Types of Conversions

  • Sales Conversions: Purchases made on your website.
  • Lead Conversions: Completed contact forms, newsletter subscriptions, etc.
  • Engagement Conversions: Content downloads, video views, etc.

Tools for Conversion Tracking

  • Google Ads: Advertising platform that allows you to set up and track conversions.
  • Google Analytics: Web analytics tool that can be integrated with Google Ads for more detailed tracking.
  • Tag Manager: Tool that simplifies the management of tracking tags without needing to modify the website code.

Setting Up Conversions in Google Ads

Step 1: Access the Conversions Section

  1. Log in to your Google Ads account.
  2. Navigate to the "Tools & Settings" tab.
  3. Select "Conversions" under the "Measurement" section.

Step 2: Create a New Conversion Action

  1. Click the "+" button to create a new conversion.
  2. Select the type of conversion you want to track (website, app, phone call, etc.).

Step 3: Configure Conversion Details

  • Name: Assign a descriptive name to the conversion.
  • Value: Define if each conversion will have a specific monetary value.
  • Count: Decide whether to count all conversions or only one per click.
  • Conversion Window: Set the period during which conversions will be tracked after an ad click.

Step 4: Install the Conversion Tag

  1. Google Ads will generate a tracking code snippet.
  2. Insert this code on the pages of your website where you want to track conversions (e.g., the thank you page after a purchase).
<!-- Google Ads Conversion Tracking Code -->
<script>
  gtag('event', 'conversion', {
      'send_to': 'AW-CONVERSION_ID',
      'value': 1.0,
      'currency': 'USD'
  });
</script>

Step 5: Verify the Installation

  • Use Google's "Tag Assistant" tool to verify that the conversion code is working correctly.

Setting Up Conversions in Google Analytics

Step 1: Set Up Goals in Google Analytics

  1. Log in to your Google Analytics account.
  2. Navigate to the "Admin" tab.
  3. In the "View" column, select "Goals."

Step 2: Create a New Goal

  1. Click "New Goal."
  2. Select a template or create a custom goal.
  3. Configure the goal details, such as type (destination, duration, pages per session, event).

Step 3: Set Up Event Tracking

If you selected "Event" as the goal type, you will need to configure event categories, actions, and labels.

<!-- Google Analytics Event Tracking Code -->
<script>
  gtag('event', 'form_submission', {
      'event_category': 'Contact',
      'event_action': 'submit',
      'event_label': 'Contact Form'
  });
</script>

Step 4: Link Google Ads and Google Analytics

  1. In Google Analytics, navigate to "Admin."
  2. In the "Property" column, select "Google Ads Linking."
  3. Follow the instructions to link your Google Ads and Google Analytics accounts.

Using Google Tag Manager for Tag Management

Step 1: Create an Account and Container

  1. Log in to Google Tag Manager.
  2. Create a new account and container for your website.

Step 2: Add Conversion Tags

  1. In the container, select "Tags" and then "New."
  2. Set up a Google Ads or Google Analytics conversion tag.

Step 3: Configure Triggers

  1. Select "Triggers" and then "New."
  2. Configure the trigger to fire on the specific pages or events you want to track.

Step 4: Publish the Container

  1. Click "Submit" to publish the container with the new tags and triggers.

Conclusion

Setting up conversions is an essential step in any SEM strategy. It allows you to measure the performance of your campaigns and optimize your marketing efforts. By using tools like Google Ads, Google Analytics, and Google Tag Manager, you can track and analyze the valuable actions users take on your website, enabling you to make informed decisions and improve the return on investment of your advertising campaigns.
