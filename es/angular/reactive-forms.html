Introducción a los Formularios Reactivos

Los Formularios Reactivos en Angular proporcionan un enfoque basado en modelos para manejar entradas de formularios cuyos valores cambian con el tiempo. Este enfoque ofrece más control y flexibilidad en comparación con los formularios basados en plantillas.

  • Enfoque basado en modelos: Los formularios se crean y gestionan en la clase del componente.
  • Estructuras de datos inmutables: Cada cambio en el estado del formulario devuelve un nuevo estado.
  • Acceso sincrónico a los datos del formulario: Los datos del formulario están disponibles de inmediato.

Configuración de Formularios Reactivos

Para usar Formularios Reactivos, necesitas importar el ReactiveFormsModule en tu módulo de Angular.

import { ReactiveFormsModule } from '@angular/forms';

@NgModule({
  imports: [
    ReactiveFormsModule,
    // otras importaciones
  ],
  // otras configuraciones
})
export class AppModule { }

Creación de un Formulario Reactivo Básico

Paso 1: Importar Clases de Formularios

Importa las clases necesarias de @angular/forms.

import { FormGroup, FormControl } from '@angular/forms';

Paso 2: Definir el Modelo del Formulario

Crea un modelo de formulario en tu clase de componente.

export class MyComponent {
  myForm: FormGroup;

  constructor() {
    this.myForm = new FormGroup({
      name: new FormControl(''),
      email: new FormControl(''),
    });
  }
}

Paso 3: Vincular el Formulario en la Plantilla

Vincula el modelo del formulario a la plantilla usando directivas de Angular.

<form [formGroup]="myForm" (ngSubmit)="onSubmit()">
  <label for="name">Nombre:</label>
  <input id="name" formControlName="name">
  
  <label for="email">Correo Electrónico:</label>
  <input id="email" formControlName="email">
  
  <button type="submit">Enviar</button>
</form>

Paso 4: Manejar el Envío del Formulario

Maneja el envío del formulario en tu clase de componente.

export class MyComponent {
  myForm: FormGroup;

  constructor() {
    this.myForm = new FormGroup({
      name: new FormControl(''),
      email: new FormControl(''),
    });
  }

  onSubmit() {
    console.log(this.myForm.value);
  }
}

Controles de Formularios Avanzados

FormBuilder

FormBuilder es un servicio que proporciona métodos convenientes para crear controles de formularios.

import { FormBuilder, FormGroup } from '@angular/forms';

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: [''],
      email: [''],
    });
  }
}

FormArray

FormArray se usa para gestionar un array de controles de formularios.

import { FormBuilder, FormArray } from '@angular/forms';

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      users: this.fb.array([]),
    });
  }

  get users() {
    return this.myForm.get('users') as FormArray;
  }

  addUser() {
    this.users.push(this.fb.group({
      name: [''],
      email: [''],
    }));
  }
}

FormGroups Anidados

Puedes anidar FormGroups para crear formularios complejos.

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      userDetails: this.fb.group({
        name: [''],
        email: [''],
      }),
      address: this.fb.group({
        street: [''],
        city: [''],
      }),
    });
  }
}

Validación de Formularios

Validadores Incorporados

Angular proporciona varios validadores incorporados.

import { Validators } from '@angular/forms;

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: ['', Validators.required],
      email: ['', [Validators.required, Validators.email]],
    });
  }
}

Validadores Personalizados

Puedes crear validadores personalizados para lógica de validación más compleja.

import { AbstractControl, ValidationErrors } from '@angular/forms;

export function forbiddenNameValidator(control: AbstractControl): ValidationErrors | null {
  const forbidden = /admin/.test(control.value);
  return forbidden ? { forbiddenName: { value: control.value } } : null;
}

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({
      name: ['', [Validators.required, forbiddenNameValidator]],
      email: ['', [Validators.required, Validators.email]],
    });
  }
}

Formularios Dinámicos

Los formularios dinámicos te permiten crear formularios basados en datos en tiempo de ejecución.

export class MyComponent {
  myForm: FormGroup;

  constructor(private fb: FormBuilder) {
    this.myForm = this.fb.group({});
  }

  createForm(fields: any[]) {
    fields.forEach(field => {
      this.myForm.addControl(field.name, new FormControl(''));
    });
  }
}

Conclusión

Los Formularios Reactivos en Angular proporcionan una forma robusta y flexible de gestionar entradas y validaciones de formularios. Al aprovechar el poder de FormGroup, FormControl, FormArray y FormBuilder, puedes crear formularios complejos y dinámicos que son fáciles de mantener y extender. Entender estos conceptos te ayudará a construir aplicaciones Angular más eficientes y escalables.
