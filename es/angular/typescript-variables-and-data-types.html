Introducción

TypeScript es un superconjunto tipado estáticamente de JavaScript que agrega tipos opcionales, interfaces y otras características al lenguaje. Entender las variables y los tipos de datos en TypeScript es crucial para desarrollar aplicaciones robustas en Angular. Esta sección cubrirá los conceptos básicos de la declaración de variables y el uso de diferentes tipos de datos en TypeScript.

Variables en TypeScript

Declaración de Variables

En TypeScript, las variables pueden ser declaradas usando let, const y var. Cada una tiene su propio alcance y uso.

  • let: Declaración de variable con alcance de bloque.
  • const: Declaración de constante con alcance de bloque.
  • var: Declaración de variable con alcance de función (no recomendado para TypeScript moderno).

Ejemplos:

let message: string = "¡Hola, TypeScript!";
const pi: number = 3.14;
var globalVar: boolean = true;

Alcance de las Variables

  • Alcance de Bloque: Las variables declaradas con let y const están limitadas al bloque en el que se definen.
  • Alcance de Función: Las variables declaradas con var están limitadas a la función en la que se definen.

Ejemplo:

function exampleFunction() {
    if (true) {
        let blockScoped = "Estoy en el alcance de bloque";
        var functionScoped = "Estoy en el alcance de función";
    }
    console.log(functionScoped); // Funciona
    // console.log(blockScoped); // Error: blockScoped no está definido
}

Tipos de Datos en TypeScript

Tipos de Datos Básicos

TypeScript soporta varios tipos de datos básicos, incluyendo:

  • string
  • number
  • boolean
  • any
  • void
  • null
  • undefined

Ejemplos:

let name: string = "John Doe";
let age: number = 30;
let isStudent: boolean = true;
let anything: any = "Podría ser cualquier cosa";
let noReturnValue: void = undefined;
let empty: null = null;
let notAssigned: undefined = undefined;

Arreglos y Tuplas

  • Arreglos: Pueden ser declarados usando la sintaxis type[] o Array<type>.
  • Tuplas: Permiten expresar un arreglo con un número fijo de elementos cuyos tipos son conocidos.

Ejemplos:

let numbers: number[] = [1, 2, 3];
let strings: Array<string> = ["uno", "dos", "tres"];
let tuple: [string, number] = ["John Doe", 30];

Enums

Los enums permiten definir un conjunto de constantes con nombre. TypeScript proporciona enums tanto numéricos como basados en cadenas.

Ejemplo:

enum Color {
    Red,
    Green,
    Blue
}
let color: Color = Color.Green;

Interfaces

Las interfaces definen la estructura de un objeto. Se utilizan para la verificación de tipos y para asegurar que los objetos se adhieran a una forma específica.

Ejemplo:

interface Person {
    name: string;
    age: number;
    isStudent: boolean;
}

let person: Person = {
    name: "Jane Doe",
    age: 25,
    isStudent: false
};

Aserciones de Tipo

Las aserciones de tipo son una forma de decirle al compilador cuál es el tipo de una variable cuando sabes más sobre el tipo que TypeScript.

Ejemplo:

let someValue: any = "Esto es una cadena";
let strLength: number = (someValue as string).length;

Conclusión

Entender las variables y los tipos de datos en TypeScript es fundamental para desarrollar aplicaciones en Angular. Al dominar estos conceptos, puedes escribir código más robusto y seguro en términos de tipos. Esta sección cubrió la declaración de variables, alcances, tipos de datos básicos, arreglos, tuplas, enums, interfaces y aserciones de tipo. Practica estos conceptos con los ejemplos proporcionados para solidificar tu comprensión.
