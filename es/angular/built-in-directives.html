Introducción a las Directivas

Las directivas son una característica central de Angular que te permiten adjuntar comportamiento a elementos en el DOM. Pueden cambiar la apariencia o el comportamiento de un elemento, componente u otra directiva. Angular proporciona varias directivas integradas que puedes usar para gestionar formularios, listas, estilos y más.

Tipos de Directivas Integradas

Las directivas integradas de Angular se pueden categorizar en tres tipos principales:

  • Directivas de Atributo: Cambian la apariencia o el comportamiento de un elemento, componente u otra directiva.
  • Directivas Estructurales: Cambian el diseño del DOM añadiendo o eliminando elementos.
  • Directivas de Componente: Estas son directivas con una plantilla.

Directivas de Atributo

Las directivas de atributo cambian la apariencia o el comportamiento de un elemento. Aquí hay algunas directivas de atributo comúnmente usadas:

`ngClass`

La directiva ngClass te permite establecer dinámicamente clases CSS en un elemento.

Ejemplo:

<div [ngClass]="{'active': isActive, 'disabled': isDisabled}">
  Las clases de este div cambian según el estado del componente.
</div>

En el ejemplo anterior, la clase active se aplicará si isActive es true, y la clase disabled se aplicará si isDisabled es true.

`ngStyle`

La directiva ngStyle te permite establecer múltiples estilos en línea de forma dinámica.

Ejemplo:

<div [ngStyle]="{'color': textColor, 'font-size': fontSize + 'px'}">
  Los estilos de este div cambian según el estado del componente.
</div>

En este ejemplo, los estilos color y font-size se establecen dinámicamente según las propiedades textColor y fontSize del componente.

Directivas Estructurales

Las directivas estructurales cambian el diseño del DOM añadiendo o eliminando elementos. Aquí hay algunas directivas estructurales comúnmente usadas:

`ngIf`

La directiva ngIf incluye condicionalmente una plantilla basada en el valor de una expresión.

Ejemplo:

<div *ngIf="isVisible">
  Este div solo es visible si isVisible es true.
</div>

En este ejemplo, el div solo se incluirá en el DOM si isVisible es true.

`ngFor`

La directiva ngFor se usa para repetir una plantilla para cada elemento en una lista.

Ejemplo:

<ul>
  <li *ngFor="let item of items">
    {{ item }}
  </li>
</ul>

En este ejemplo, se crea un elemento <li> para cada elemento en el array items.

`ngSwitch`

La directiva ngSwitch se usa para incluir condicionalmente una de varias plantillas posibles, basada en el valor de una expresión.

Ejemplo:

<div [ngSwitch]="viewMode">
  <div *ngSwitchCase="'map'">Vista de Mapa</div>
  <div *ngSwitchCase="'list'">Vista de Lista</div>
  <div *ngSwitchDefault>Vista Predeterminada</div>
</div>

En este ejemplo, la plantilla que coincide con el valor de viewMode se incluye en el DOM.

Directivas de Componente

Las directivas de componente son esencialmente componentes de Angular. Tienen una plantilla, estilos y lógica encapsulada.

Ejemplo:

@Component({
  selector: 'app-hello',
  template: `<h1>¡Hola, {{name}}!</h1>`,
  styles: [`h1 { font-family: Lato; }`]
})
export class HelloComponent  {
  @Input() name: string;
}

En este ejemplo, HelloComponent es una directiva de componente que muestra un mensaje de saludo.

Conclusión

Las directivas integradas en Angular son herramientas poderosas que te permiten manipular el DOM de manera declarativa. Entender cómo usar directivas de atributo como ngClass y ngStyle, directivas estructurales como ngIf, ngFor y ngSwitch, y directivas de componente te permitirá crear aplicaciones web dinámicas y responsivas. Practica usando estas directivas en tus proyectos para obtener una mejor comprensión de sus capacidades y matices.
