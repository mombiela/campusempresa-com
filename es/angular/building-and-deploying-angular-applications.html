Introducción

En esta sección, cubriremos los pasos esenciales y las mejores prácticas para construir y desplegar aplicaciones Angular. Esto incluye configurar el entorno, construir la aplicación y desplegarla en varias plataformas.

Configuración del Entorno

Antes de que podamos construir y desplegar una aplicación Angular, necesitamos configurar nuestro entorno de desarrollo.

Requisitos Previos

  • Node.js y npm: Asegúrate de tener Node.js y npm instalados. Puedes descargarlos desde nodejs.org.
  • Angular CLI: Instala Angular CLI globalmente usando npm. 
    npm install -g @angular/cli

Creando un Nuevo Proyecto Angular

Usa Angular CLI para crear un nuevo proyecto.

ng new my-angular-app
  • Sigue las indicaciones para configurar tu proyecto. Puedes elegir enrutamiento y el formato de estilo (CSS, SCSS, etc.).

Ejecutando el Servidor de Desarrollo

Navega a tu directorio de proyecto y comienza el servidor de desarrollo.

cd my-angular-app
ng serve
  • Abre tu navegador y ve a http://localhost:4200 para ver tu aplicación en funcionamiento.

Construyendo la Aplicación

Una vez que tu aplicación esté lista, necesitas construirla para producción.

Construcción para Producción

Usa Angular CLI para crear una construcción de producción.

ng build --prod
  • Este comando creará una construcción optimizada de tu aplicación en el directorio dist/.

Configuración de la Construcción

Puedes personalizar el proceso de construcción usando el archivo angular.json. Aquí hay algunas configuraciones comunes:

  • Ruta de Salida: Cambia el directorio de salida.
  • Reemplazo de Archivos: Reemplaza archivos de entorno para diferentes configuraciones (por ejemplo, desarrollo vs. producción).
  • Optimización: Habilita o deshabilita optimizaciones como minificación y tree-shaking.

Desplegando la Aplicación

Hay varias formas de desplegar una aplicación Angular. Cubriremos el despliegue en plataformas populares como GitHub Pages, Firebase Hosting y AWS S3.

Desplegando en GitHub Pages

  1. Instala el desplegador de GitHub Pages de Angular CLI. 
    npm install -g angular-cli-ghpages
  2. Construye tu proyecto. 
    ng build --prod --base-href "https://<username>.github.io/<repository>/"
  3. Despliega en GitHub Pages. 
    ngh --dir=dist/<project-name>

Desplegando en Firebase Hosting

  1. Instala Firebase CLI. 
    npm install -g firebase-tools
  2. Inicia sesión en Firebase. 
    firebase login
  3. Inicializa Firebase en tu proyecto. 
    firebase init
    • Elige Hosting y sigue las indicaciones.
  4. Construye tu proyecto. 
    ng build --prod
  5. Despliega en Firebase. 
    firebase deploy

Desplegando en AWS S3

  1. Instala AWS CLI y configúralo con tus credenciales.
  2. Construye tu proyecto. 
    ng build --prod
  3. Sincroniza la salida de la construcción con tu bucket de S3. 
    aws s3 sync dist/<project-name> s3://<your-bucket-name> --acl public-read

Conclusión

Construir y desplegar aplicaciones Angular implica configurar el entorno de desarrollo, crear una construcción de producción y desplegarla en una plataforma de alojamiento. Siguiendo los pasos descritos en esta sección, puedes asegurarte de que tu aplicación Angular esté optimizada y accesible para los usuarios. Ya sea que elijas desplegar en GitHub Pages, Firebase Hosting o AWS S3, Angular CLI proporciona las herramientas necesarias para simplificar el proceso.
