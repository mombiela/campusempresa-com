Introducción a Angular Universal

Angular Universal es una tecnología que permite que las aplicaciones Angular se rendericen en el lado del servidor. Esto puede mejorar el rendimiento, SEO y la experiencia del usuario al entregar páginas completamente renderizadas al cliente.

Conceptos Clave

  • Renderizado del lado del servidor (SSR): El proceso de renderizar páginas web en el servidor en lugar de en el cliente.
  • Angular Universal: Una herramienta que habilita SSR para aplicaciones Angular.
  • Beneficios de SEO: Los motores de búsqueda pueden rastrear páginas completamente renderizadas, mejorando el posicionamiento en los motores de búsqueda.
  • Rendimiento: Tiempos de carga inicial más rápidos ya que el servidor envía una página completamente renderizada al cliente.

Configuración de Angular Universal

Para comenzar con Angular Universal, necesitas agregarlo a tu proyecto Angular existente.

Pasos para agregar Angular Universal

  1. Instalar Angular Universal:

    ng add @nguniversal/express-engine

    Este comando configurará tu aplicación Angular para usar Angular Universal con un servidor Express.

  2. Actualizar angular.json: Asegúrate de que las configuraciones de compilación y servidor estén correctamente configuradas en tu archivo angular.json.

  3. Modificar server.ts: El archivo server.ts es el punto de entrada para tu aplicación del lado del servidor. Debería verse algo así:

    import 'zone.js/dist/zone-node';
import { ngExpressEngine } from '@nguniversal/express-engine';
import * as express from 'express';
import { join } from 'path';

import { AppServerModule } from './src/main.server';
import { APP_BASE_HREF } from '@angular/common';
import { existsSync } from 'fs';

// La aplicación Express se exporta para que pueda ser utilizada por funciones sin servidor.
export function app() {
  const server = express();
  const distFolder = join(process.cwd(), 'dist/your-project-name/browser');
  const indexHtml = existsSync(join(distFolder, 'index.original.html')) ? 'index.original.html' : 'index';

  server.engine('html', ngExpressEngine({
    bootstrap: AppServerModule,
  }));

  server.set('view engine', 'html');
  server.set('views', distFolder);

  server.get('*.*', express.static(distFolder, {
    maxAge: '1y'
  }));

  server.get('*', (req, res) => {
    res.render(indexHtml, { req, providers: [{ provide: APP_BASE_HREF, useValue: req.baseUrl }] });
  });

  return server;
}

function run() {
  const port = process.env.PORT || 4000;

  const server = app();
  server.listen(port, () => {
    console.log(`Servidor Node Express escuchando en http://localhost:${port}`);
  });
}

declare const __non_webpack_require__: NodeRequire;

if (require.main === module) {
  run();
}

export * from './src/main.server';

  4. Actualizar main.server.ts: Asegúrate de que tu archivo main.server.ts inicie el módulo del servidor:

    import { enableProdMode } from '@angular/core';
import { environment } from './environments/environment';

if (environment.production) {
  enableProdMode();
}

export { AppServerModule } from './app/app.server.module';
export { renderModule, renderModuleFactory } from '@angular/platform-server';

Ejecutando la Aplicación

Para ejecutar tu aplicación Angular Universal, necesitas compilar tanto el código del cliente como el del servidor y luego servirlo.

Compilar y Servir

  1. Compilar el Cliente y el Servidor:

    npm run build:ssr

  2. Servir la Aplicación:

    npm run serve:ssr

Beneficios de Angular Universal

  • Mejor SEO: Los motores de búsqueda pueden indexar tu sitio de manera más efectiva.
  • Carga Inicial Más Rápida: Los usuarios reciben una página completamente renderizada desde el servidor, reduciendo el tiempo hasta la primera pintura significativa.
  • Mejor Experiencia de Usuario: Los usuarios pueden ver el contenido más rápido, incluso en redes lentas.

Conclusión

Angular Universal es una herramienta poderosa para mejorar el rendimiento y SEO de las aplicaciones Angular a través del renderizado del lado del servidor. Siguiendo los pasos de configuración y comprendiendo los conceptos clave, puedes aprovechar Angular Universal para ofrecer una mejor experiencia de usuario y mejorar la visibilidad de tu aplicación en los motores de búsqueda.
