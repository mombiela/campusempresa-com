Introducción

En Blender, dominar las transformaciones básicas—Mover, Rotar y Escalar—es esencial para cualquier artista 3D. Estas transformaciones te permiten manipular objetos en tu escena, posicionándolos precisamente donde los quieres. Esta sección cubrirá estas operaciones fundamentales, proporcionando explicaciones claras, ejemplos y ejercicios para ayudarte a entender y aplicar estas transformaciones de manera efectiva.

Mover (Trasladar)

Mover un objeto cambia su posición en el espacio 3D.

Conceptos Clave

  • Ejes: Los ejes X (rojo), Y (verde) y Z (azul) representan las dimensiones del espacio 3D.
  • Global vs. Local: Las transformaciones globales afectan al objeto en el espacio mundial, mientras que las transformaciones locales afectan al objeto en relación con su propia orientación.

Cómo Mover Objetos

  1. Usando el Gizmo:

    • Selecciona el objeto.
    • Haz clic y arrastra las flechas del gizmo para mover el objeto a lo largo del eje X, Y o Z.

  2. Usando Atajos de Teclado:

    • Presiona G para agarrar (mover) el objeto.
    • Presiona X, Y o Z inmediatamente después de G para restringir el movimiento al eje respectivo.
    • Mantén presionado Shift y presiona X, Y o Z para excluir un eje del movimiento.

Ejemplo

1. Selecciona el cubo.
2. Presiona `G` para mover el cubo.
3. Presiona `X` para restringir el movimiento al eje X.
4. Mueve el ratón para posicionar el cubo y haz clic para confirmar.

Ejercicio

  1. Crea un nuevo cubo.
  2. Mueve el cubo 2 unidades a lo largo del eje X.
  3. Mueve el cubo 3 unidades a lo largo del eje Y, pero no a lo largo del eje Z.

Rotar

Rotar un objeto cambia su orientación.

Conceptos Clave

  • Punto de Pivote: El punto alrededor del cual el objeto rota.
  • Ángulos de Euler: Método común para representar rotaciones usando tres ángulos (X, Y, Z).

Cómo Rotar Objetos

  1. Usando el Gizmo:

    • Selecciona el objeto.
    • Haz clic y arrastra los mangos circulares del gizmo para rotar el objeto alrededor del eje X, Y o Z.

  2. Usando Atajos de Teclado:

    • Presiona R para rotar el objeto.
    • Presiona X, Y o Z inmediatamente después de R para restringir la rotación al eje respectivo.
    • Presiona R dos veces para entrar en modo de rotación de bola de seguimiento.

Ejemplo

1. Selecciona el cubo.
2. Presiona `R` para rotar el cubo.
3. Presiona `Z` para restringir la rotación al eje Z.
4. Mueve el ratón para rotar el cubo y haz clic para confirmar.

Ejercicio

  1. Crea un nuevo cubo.
  2. Rota el cubo 45 grados alrededor del eje Y.
  3. Rota el cubo 30 grados alrededor del eje X.

Escalar

Escalar un objeto cambia su tamaño.

Conceptos Clave

  • Escalado Uniforme: Escalado igual en todas las direcciones.
  • Escalado No Uniforme: Escalado diferente a lo largo de diferentes ejes.

Cómo Escalar Objetos

  1. Usando el Gizmo:

    • Selecciona el objeto.
    • Haz clic y arrastra los mangos del gizmo para escalar el objeto a lo largo del eje X, Y o Z.

  2. Usando Atajos de Teclado:

    • Presiona S para escalar el objeto.
    • Presiona X, Y o Z inmediatamente después de S para restringir el escalado al eje respectivo.
    • Mantén presionado Shift y presiona X, Y o Z para excluir un eje del escalado.

Ejemplo

1. Selecciona el cubo.
2. Presiona `S` para escalar el cubo.
3. Presiona `Y` para restringir el escalado al eje Y.
4. Mueve el ratón para escalar el cubo y haz clic para confirmar.

Ejercicio

  1. Crea un nuevo cubo.
  2. Escala el cubo al doble de su tamaño a lo largo del eje Z.
  3. Escala el cubo a la mitad de su tamaño a lo largo del eje X.

Resumen

Entender y dominar las transformaciones básicas—Mover, Rotar y Escalar—es crucial para manipular objetos en Blender. Estas operaciones forman la base del modelado y la animación 3D. Practicando estas transformaciones, obtendrás un mejor control sobre tus escenas 3D y mejorarás tu eficiencia general en Blender. Recuerda usar el gizmo para una guía visual y los atajos de teclado para transformaciones rápidas.
