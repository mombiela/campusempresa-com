Introducción al Compositor

El Compositor en Blender es una herramienta poderosa para el post-procesamiento de tus renders. Te permite combinar múltiples imágenes, aplicar efectos y mejorar tu resultado final. Esta sección te guiará desde lo básico hasta técnicas avanzadas en el uso del Compositor.

Empezando con el Compositor

Habilitando el Compositor

Para empezar a usar el Compositor:

  1. Abre Blender y cambia al espacio de trabajo "Compositing".
  2. Marca la casilla "Use Nodes" en la ventana del Compositor.

Configuración Básica de Nodos

  • Render Layers Node: Este nodo es el punto de partida para tu imagen renderizada.
  • Composite Node: Este nodo es el punto final donde se produce la imagen final.
# Ejemplo de Configuración Básica de Nodos
import bpy

# Cambiar al espacio de trabajo de Compositing
bpy.context.area.ui_type = 'CompositorNodeTree'

# Habilitar Use Nodes
bpy.context.scene.use_nodes = True

# Obtener el árbol de nodos
tree = bpy.context.scene.node_tree

# Limpiar nodos predeterminados
for node in tree.nodes:
    tree.nodes.remove(node)

# Añadir nodo Render Layers
render_layers = tree.nodes.new(type='CompositorNodeRLayers')
render_layers.location = (0, 0)

# Añadir nodo Composite
composite = tree.nodes.new(type='CompositorNodeComposite')
composite.location = (400, 0)

# Enlazar nodos
tree.links.new(render_layers.outputs[0], composite.inputs[0])

Técnicas Intermedias de Composición

Añadiendo Efectos

  • Blur Node: Añade un efecto de desenfoque a tu imagen.
  • Color Balance Node: Ajusta el balance de color de tu imagen.
# Añadiendo un Efecto de Desenfoque
blur = tree.nodes.new(type='CompositorNodeBlur')
blur.location = (200, 0)
blur.size_x = 10
blur.size_y = 10

# Enlazar Render Layers a Blur, luego a Composite
tree.links.new(render_layers.outputs[0], blur.inputs[0])
tree.links.new(blur.outputs[0], composite.inputs[0])

Enmascarado

  • Mask Node: Te permite crear y usar máscaras para composición selectiva.
  • Mix Node: Combina dos imágenes usando una máscara.
# Usando una Máscara
mask = tree.nodes.new(type='CompositorNodeMask')
mask.location = (200, -200)
mask.name = "MyMask"

mix = tree.nodes.new(type='CompositorNodeMixRGB')
mix.location = (400, -200)
mix.blend_type = 'MIX'

# Enlazar nodos
tree.links.new(render_layers.outputs[0], mix.inputs[1])
tree.links.new(mask.outputs[0], mix.inputs[0])
tree.links.new(mix.outputs[0], composite.inputs[0])

Técnicas Avanzadas de Composición

Renderizado Multi-Pass

  • Render Passes: Separa diferentes aspectos del render (por ejemplo, Difuso, Brillante, Sombra) para tener más control en la composición.
# Habilitando Render Passes
bpy.context.scene.view_layers["View Layer"].use_pass_diffuse_color = True
bpy.context.scene.view_layers["View Layer"].use_pass_glossy_direct = True

# Accediendo a Render Passes en el Compositor
diffuse = render_layers.outputs['DiffCol']
glossy = render_layers.outputs['GlossDir']

# Ejemplo: Combinando Pases Difuso y Brillante
add = tree.nodes.new(type='CompositorNodeMixRGB')
add.location = (600, 0)
add.blend_type = 'ADD'

tree.links.new(diffuse, add.inputs[1])
tree.links.new(glossy, add.inputs[2])
tree.links.new(add.outputs[0], composite.inputs[0])

Efectos Avanzados

  • Lens Distortion Node: Simula efectos de distorsión de lente.
  • Glare Node: Añade efectos de resplandor a las áreas brillantes de la imagen.
# Añadiendo Distorsión de Lente
lens_distortion = tree.nodes.new(type='CompositorNodeLensdist')
lens_distortion.location = (200, 200)
lens_distortion.inputs['Distort'].default_value = 0.5

# Añadiendo Resplandor
glare = tree.nodes.new(type='CompositorNodeGlare')
glare.location = (400, 200)
glare.glare_type = 'FOG_GLOW'

# Enlazar nodos
tree.links.new(render_layers.outputs[0], lens_distortion.inputs[0])
tree.links.new(lens_distortion.outputs[0], glare.inputs[0])
tree.links.new(glare.outputs[0], composite.inputs[0])

Conclusión

El Compositor en Blender es una herramienta esencial para mejorar tus renders y lograr resultados de calidad profesional. Al entender y utilizar nodos, máscaras, pases de render y efectos avanzados, puedes mejorar significativamente la calidad visual de tus proyectos. Practica estas técnicas para volverte competente en composición y elevar tus habilidades en Blender al siguiente nivel.
