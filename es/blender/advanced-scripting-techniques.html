Introducción

En esta sección, profundizaremos en técnicas avanzadas de scripting en Blender usando Python. Esto se basará en el conocimiento fundamental que has adquirido en secciones anteriores e introducirá conceptos y métodos más complejos para mejorar tus capacidades de scripting.

Conceptos Clave

  • Operadores Personalizados: Crear y usar operadores personalizados para extender la funcionalidad de Blender.
  • Operadores Modales: Implementar operadores que se ejecutan continuamente e interactúan con la entrada del usuario.
  • Paneles de UI Personalizados: Diseñar e integrar paneles de UI personalizados dentro de la interfaz de Blender.
  • Gestión de Propiedades: Gestionar propiedades personalizadas y usarlas eficazmente en scripts.
  • Manipulación de Escenas: Técnicas avanzadas para manipular escenas, objetos y bloques de datos.
  • Complementos: Crear y gestionar complementos de Blender para distribución y reutilización.

Operadores Personalizados

Los operadores personalizados te permiten extender la funcionalidad de Blender creando nuevos comandos.

Ejemplo: Creando un Operador Personalizado Simple

import bpy

class SimpleOperator(bpy.types.Operator):
    bl_idname = "object.simple_operator"
    bl_label = "Simple Operator"
    
    def execute(self, context):
        self.report({'INFO'}, "Simple Operator Executed")
        return {'FINISHED'}

def register():
    bpy.utils.register_class(SimpleOperator)

def unregister():
    bpy.utils.unregister_class(SimpleOperator)

if __name__ == "__main__":
    register()
  • Explicación:
    • bl_idname: Identificador único para el operador.
    • bl_label: Nombre de visualización para el operador.
    • execute(): Método que se ejecuta cuando se llama al operador.

Operadores Modales

Los operadores modales se ejecutan continuamente y pueden interactuar con la entrada del usuario, lo que los hace adecuados para tareas que requieren interacción continua.

Ejemplo: Creando un Operador Modal

import bpy

class ModalOperator(bpy.types.Operator):
    bl_idname = "object.modal_operator"
    bl_label = "Modal Operator"
    
    def modal(self, context, event):
        if event.type in {'ESC'}:
            return {'CANCELLED'}
        return {'RUNNING_MODAL'}
    
    def invoke(self, context, event):
        context.window_manager.modal_handler_add(self)
        return {'RUNNING_MODAL'}

def register():
    bpy.utils.register_class(ModalOperator)

def unregister():
    bpy.utils.unregister_class(ModalOperator)

if __name__ == "__main__":
    register()
  • Explicación:
    • modal(): Método que maneja eventos.
    • invoke(): Método que inicia el operador modal.

Paneles de UI Personalizados

Los paneles de UI personalizados te permiten crear nuevos paneles en la interfaz de Blender para albergar tus herramientas y configuraciones.

Ejemplo: Creando un Panel de UI Personalizado

import bpy

class SimplePanel(bpy.types.Panel):
    bl_label = "Simple Panel"
    bl_idname = "OBJECT_PT_simple_panel"
    bl_space_type = 'VIEW_3D'
    bl_region_type = 'UI'
    bl_category = 'Tool'
    
    def draw(self, context):
        layout = self.layout
        layout.label(text="Hello, World!")

def register():
    bpy.utils.register_class(SimplePanel)

def unregister():
    bpy.utils.unregister_class(SimplePanel)

if __name__ == "__main__":
    register()
  • Explicación:
    • bl_space_type: Define dónde aparecerá el panel.
    • bl_region_type: Define la región dentro del espacio.
    • draw(): Método para definir el diseño del panel.

Gestión de Propiedades

Gestionar propiedades personalizadas es crucial para crear scripts dinámicos e interactivos.

Ejemplo: Añadiendo Propiedades Personalizadas

import bpy

bpy.types.Scene.my_custom_float = bpy.props.FloatProperty(name="My Float", default=1.0)

class CustomPropertyPanel(bpy.types.Panel):
    bl_label = "Custom Property Panel"
    bl_idname = "OBJECT_PT_custom_property_panel"
    bl_space_type = 'VIEW_3D'
    bl_region_type = 'UI'
    bl_category = 'Tool'
    
    def draw(self, context):
        layout = self.layout
        layout.prop(context.scene, "my_custom_float")

def register():
    bpy.utils.register_class(CustomPropertyPanel)

def unregister():
    bpy.utils.unregister_class(CustomPropertyPanel)
    del bpy.types.Scene.my_custom_float

if __name__ == "__main__":
    register()
  • Explicación:
    • FloatProperty: Añade una propiedad flotante a la escena.
    • layout.prop(): Añade la propiedad al panel.

Manipulación de Escenas

La manipulación avanzada de escenas implica modificar objetos, bloques de datos y otros elementos dentro de la escena de Blender.

Ejemplo: Duplicando Objetos

import bpy

class DuplicateObjectsOperator(bpy.types.Operator):
    bl_idname = "object.duplicate_objects"
    bl_label = "Duplicate Objects"
    
    def execute(self, context):
        for obj in context.selected_objects:
            new_obj = obj.copy()
            new_obj.data = obj.data.copy()
            context.collection.objects.link(new_obj)
        return {'FINISHED'}

def register():
    bpy.utils.register_class(DuplicateObjectsOperator)

def unregister():
    bpy.utils.unregister_class(DuplicateObjectsOperator)

if __name__ == "__main__":
    register()
  • Explicación:
    • obj.copy(): Duplica el objeto.
    • context.collection.objects.link(): Añade el objeto duplicado a la escena.

Complementos

Crear complementos te permite empaquetar tus scripts para distribución y reutilización.

Ejemplo: Estructura Básica de un Complemento

bl_info = {
    "name": "Simple Add-on",
    "blender": (2, 80, 0),
    "category": "Object",
}

import bpy

class SimpleAddonOperator(bpy.types.Operator):
    bl_idname = "object.simple_addon_operator"
    bl_label = "Simple Add-on Operator"
    
    def execute(self, context):
        self.report({'INFO'}, "Simple Add-on Operator Executed")
        return {'FINISHED'}

def menu_func(self, context):
    self.layout.operator(SimpleAddonOperator.bl_idname)

def register():
    bpy.utils.register_class(SimpleAddonOperator)
    bpy.types.VIEW3D_MT_object.append(menu_func)

def unregister():
    bpy.utils.unregister_class(SimpleAddonOperator)
    bpy.types.VIEW3D_MT_object.remove(menu_func)

if __name__ == "__main__":
    register()
  • Explicación:
    • bl_info: Metadatos para el complemento.
    • menu_func(): Añade el operador a un menú.
