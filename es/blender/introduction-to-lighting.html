La iluminación es un aspecto crucial del modelado y renderizado 3D en Blender. Puede afectar dramáticamente el ambiente, el realismo y la calidad general de tu escena. Esta sección te guiará a través de los conceptos básicos de la iluminación en Blender, desde entender los diferentes tipos de luces hasta técnicas avanzadas para crear configuraciones de iluminación realistas.

Tipos de Luces en Blender

Blender ofrece varios tipos de luces, cada una con propiedades y usos únicos. Entender estos tipos es el primer paso para dominar la iluminación en Blender.

  • Luz Puntual: Emite luz uniformemente en todas las direcciones desde un solo punto, similar a una bombilla.
  • Luz Solar: Simula la luz del sol, proporcionando rayos de luz paralelos, ideal para escenas al aire libre.
  • Luz Focal: Emite luz en forma de cono, útil para resaltar áreas específicas.
  • Luz de Área: Emite luz desde un área rectangular o cuadrada, proporcionando sombras suaves y una iluminación más realista.
  • Luz Hemisférica: Simula la luz ambiental del cielo, útil para escenas al aire libre.

Ejemplo: Añadir una Luz Puntual

Para añadir una Luz Puntual en Blender:

  1. Añadir Luz: Presiona Shift + A > Luz > Puntual.
  2. Posicionar Luz: Mueve la luz a la ubicación deseada usando la tecla G.
  3. Ajustar Propiedades: En el panel de Propiedades, ajusta la intensidad, el color y otros ajustes de la luz.
# Ejemplo de añadir una Luz Puntual usando scripting en Python en Blender
import bpy

# Crear una nueva luz puntual
light_data = bpy.data.lights.new(name="Nueva Luz Puntual", type='POINT')
light_object = bpy.data.objects.new(name="Nueva Luz Puntual", object_data=light_data)

# Vincular el objeto de luz a la escena
bpy.context.collection.objects.link(light_object)

# Establecer la ubicación de la luz
light_object.location = (5.0, 5.0, 5.0)

# Ajustar propiedades de la luz
light_data.energy = 1000  # Ajustar intensidad
light_data.color = (1.0, 0.8, 0.6)  # Ajustar color

Técnicas de Iluminación

Se pueden usar diferentes técnicas para lograr varios efectos de iluminación. Aquí hay algunas técnicas fundamentales:

  • Iluminación de Tres Puntos: Un método estándar que usa tres luces (Luz Principal, Luz de Relleno y Luz de Fondo) para iluminar un sujeto.
  • Iluminación Global: Simula la iluminación indirecta, haciendo que las escenas se vean más realistas.
  • Iluminación HDRI: Usa Imágenes de Alto Rango Dinámico para proporcionar una iluminación ambiental realista.

Configuración de Iluminación de Tres Puntos

  1. Luz Principal: La fuente de luz principal, usualmente posicionada en un ángulo hacia el sujeto.
  2. Luz de Relleno: Reduce las sombras creadas por la Luz Principal, colocada en el lado opuesto.
  3. Luz de Fondo: Añade profundidad iluminando el sujeto desde atrás.
# Ejemplo de configuración de Iluminación de Tres Puntos usando scripting en Python en Blender
import bpy

# Función para crear una luz
def create_light(name, light_type, location, energy):
    light_data = bpy.data.lights.new(name=name, type=light_type)
    light_object = bpy.data.objects.new(name=name, object_data=light_data)
    bpy.context.collection.objects.link(light_object)
    light_object.location = location
    light_data.energy = energy
    return light_object

# Luz Principal
create_light("Luz Principal", 'SPOT', (5.0, -5.0, 5.0), 1000)

# Luz de Relleno
create_light("Luz de Relleno", 'SPOT', (-5.0, -5.0, 5.0), 500)

# Luz de Fondo
create_light("Luz de Fondo", 'SPOT', (0.0, 5.0, 5.0), 750)

Técnicas Avanzadas de Iluminación

Para usuarios más avanzados, Blender ofrece herramientas y técnicas sofisticadas para crear una iluminación altamente realista:

  • Iluminación Volumétrica: Añade profundidad y atmósfera simulando la dispersión de la luz a través de un medio.
  • Sondas de Luz: Capturan y reemiten luz en una escena, útil para iluminación dinámica en entornos complejos.
  • Texturas de Luz Personalizadas: Usa texturas para crear patrones y efectos de luz personalizados.

Ejemplo: Iluminación Volumétrica

  1. Añadir Dispersión de Volumen: En el Editor de Sombras, añade un nodo de Dispersión de Volumen al material del Mundo u Objeto.
  2. Ajustar Densidad: Controla la densidad del volumen para lograr el efecto deseado.
# Ejemplo de configuración de Iluminación Volumétrica usando scripting en Python en Blender
import bpy

# Crear un nuevo material para iluminación volumétrica
material = bpy.data.materials.new(name="MaterialVolumétrico")
material.use_nodes = True
nodes = material.node_tree.nodes

# Limpiar nodos predeterminados
for node in nodes:
    nodes.remove(node)

# Añadir nodo de Dispersión de Volumen
volume_scatter = nodes.new(type='ShaderNodeVolumeScatter')
volume_scatter.location = (0, 0)
volume_scatter.inputs['Density'].default_value = 0.1

# Añadir nodo de Salida de Volumen
volume_output = nodes.new(type='ShaderNodeOutputMaterial')
volume_output.location = (200, 0)

# Vincular nodos
material.node_tree.links.new(volume_scatter.outputs['Volume'], volume_output.inputs['Volume'])

# Asignar material al mundo
bpy.context.scene.world.use_nodes = True
world_nodes = bpy.context.scene.world.node_tree.nodes
world_nodes.clear()
world_nodes.new(type='ShaderNodeBackground')
world_output = world_nodes.new(type='ShaderNodeOutputWorld')
world_nodes.new(type='ShaderNodeVolumeScatter')
bpy.context.scene.world.node_tree.links.new(world_nodes['Volume Scatter'].outputs['Volume'], world_output.inputs['Volume'])

Conclusión

La iluminación es una habilidad esencial para crear escenas 3D atractivas y realistas en Blender. Practica estas técnicas, experimenta con diferentes configuraciones y estarás en buen camino para dominar la iluminación en Blender.
