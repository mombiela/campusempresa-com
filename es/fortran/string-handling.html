Operaciones con Arreglos y Cadenas ►

El manejo de cadenas es un aspecto crucial de la programación en Fortran, especialmente cuando se trata de datos de texto. Este tema cubrirá los conceptos básicos del manejo de cadenas, técnicas de manipulación y operaciones avanzadas con cadenas en Fortran.

Introducción a las Cadenas en Fortran

En Fortran, las cadenas son secuencias de caracteres. Se utilizan típicamente para almacenar y manipular datos de texto.

Declaración de Cadenas

  • Las cadenas en Fortran se declaran usando la palabra clave CHARACTER.
  • Se puede especificar la longitud de la cadena.
CHARACTER(LEN=20) :: str
str = '¡Hola, Fortran!'

Operaciones Básicas con Cadenas

  • Asignación: Asignar un valor a una variable de cadena.
  • Concatenación: Combinar dos o más cadenas.
  • Subcadena: Extraer una parte de una cadena.
CHARACTER(LEN=20) :: str1, str2, result
str1 = 'Hola'
str2 = 'Mundo'
result = str1 // ' ' // str2  ! Concatenación
PRINT *, result  ! Salida: Hola Mundo

result = str1(1:4)  ! Subcadena
PRINT *, result  ! Salida: Hola

Técnicas de Manipulación de Cadenas

Longitud de la Cadena

  • La función LEN devuelve la longitud de una cadena.
CHARACTER(LEN=20) :: str
str = 'Fortran'
PRINT *, LEN(str)  ! Salida: 20 (incluyendo espacios finales)
PRINT *, LEN_TRIM(str)  ! Salida: 7 (excluyendo espacios finales)

Recortar Cadenas

  • La función TRIM elimina los espacios finales de una cadena.
CHARACTER(LEN=20) :: str
str = 'Fortran    '
PRINT *, TRIM(str)  ! Salida: 'Fortran'

Comparación de Cadenas

  • Las cadenas se pueden comparar usando operadores relacionales (==, /=, <, >, <=, >=).
CHARACTER(LEN=10) :: str1, str2
str1 = 'abc'
str2 = 'def'
IF (str1 < str2) THEN
    PRINT *, 'str1 es menor que str2'
ENDIF

Operaciones Avanzadas con Cadenas

Funciones Intrínsecas de Cadenas

Fortran proporciona varias funciones intrínsecas para operaciones avanzadas con cadenas.

  • INDEX: Encuentra la posición de una subcadena dentro de una cadena.
  • ADJUSTL: Justifica una cadena a la izquierda.
  • ADJUSTR: Justifica una cadena a la derecha.
CHARACTER(LEN=20) :: str
str = '¡Hola, Fortran!'
PRINT *, INDEX(str, 'Fortran')  ! Salida: 8

str = '   Fortran'
PRINT *, ADJUSTL(str)  ! Salida: 'Fortran   '
PRINT *, ADJUSTR(str)  ! Salida: '   Fortran'

Asignación Dinámica de Cadenas

  • Fortran 2003 introdujo la capacidad de asignar cadenas dinámicamente.
CHARACTER(:), ALLOCATABLE :: dyn_str
ALLOCATE(CHARACTER(LEN=30) :: dyn_str)
dyn_str = 'Asignación Dinámica de Cadenas'
PRINT *, dyn_str
DEALLOCATE(dyn_str)

Ejercicios

  1. Operaciones Básicas con Cadenas: Declara dos cadenas, concaténalas e imprime el resultado.
  2. Extracción de Subcadena: Extrae e imprime una subcadena de una cadena dada.
  3. Comparación de Cadenas: Escribe un programa para comparar dos cadenas e imprime cuál es mayor.
  4. Funciones Avanzadas: Usa la función INDEX para encontrar la posición de una subcadena dentro de una cadena.

Conclusión

El manejo de cadenas en Fortran implica varias operaciones como asignación, concatenación y extracción de subcadenas. Comprender estos conceptos básicos, junto con técnicas avanzadas como la asignación dinámica y las funciones intrínsecas, es esencial para una manipulación efectiva de datos de texto en Fortran. Practica estos conceptos a través de los ejercicios proporcionados para solidificar tu comprensión.
