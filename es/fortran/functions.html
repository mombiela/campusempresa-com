Introducción a las Funciones

Las funciones en Fortran son bloques esenciales que te permiten encapsular código para reutilizarlo, mejorar la legibilidad y gestionar la complejidad. En esta sección, cubriremos los conceptos básicos de la definición y uso de funciones en Fortran, progresando desde ejemplos simples hasta conceptos más avanzados.

Definición Básica de Función

Una función en Fortran es un bloque de código que toma argumentos de entrada, realiza una tarea específica y devuelve un valor. Aquí está la sintaxis básica para definir una función:

FUNCTION nombre_funcion(argumentos)
  ! Declaraciones
  IMPLICIT NONE
  ! Cuerpo de la función
  nombre_funcion = resultado
END FUNCTION nombre_funcion

Ejemplo: Función Simple

Comencemos con un ejemplo simple de una función que suma dos números:

PROGRAM main
  IMPLICIT NONE
  REAL :: resultado

  resultado = sumar(2.0, 3.0)
  PRINT *, "La suma es: ", resultado

CONTAINS

  FUNCTION sumar(a, b)
    REAL :: sumar
    REAL, INTENT(IN) :: a, b

    sumar = a + b
  END FUNCTION sumar

END PROGRAM main
  • Explicación:
    • PROGRAM main: El bloque principal del programa.
    • IMPLICIT NONE: Asegura que todas las variables deben ser declaradas explícitamente.
    • REAL :: resultado: Declara una variable real para almacenar el resultado.
    • resultado = sumar(2.0, 3.0): Llama a la función sumar con los argumentos 2.0 y 3.0.
    • CONTAINS: Indica que el siguiente bloque contiene procedimientos internos.
    • FUNCTION sumar(a, b): Define la función sumar.
    • REAL, INTENT(IN) :: a, b: Declara los argumentos de entrada a y b como números reales.
    • sumar = a + b: Calcula la suma y la asigna al nombre de la función.

Argumentos de Función y Tipos de Retorno

Las funciones pueden tomar varios tipos de argumentos y devolver diferentes tipos de valores. El tipo de retorno de una función se especifica en su declaración.

Ejemplo: Función con Argumentos Enteros

Aquí hay un ejemplo de una función que multiplica dos enteros:

PROGRAM main
  IMPLICIT NONE
  INTEGER :: resultado

  resultado = multiplicar(4, 5)
  PRINT *, "El producto es: ", resultado

CONTAINS

  FUNCTION multiplicar(a, b)
    INTEGER :: multiplicar
    INTEGER, INTENT(IN) :: a, b

    multiplicar = a * b
  END FUNCTION multiplicar

END PROGRAM main
  • Explicación:
    • INTEGER :: resultado: Declara una variable entera para almacenar el resultado.
    • resultado = multiplicar(4, 5): Llama a la función multiplicar con los argumentos 4 y 5.
    • FUNCTION multiplicar(a, b): Define la función multiplicar.
    • INTEGER, INTENT(IN) :: a, b: Declara los argumentos de entrada a y b como enteros.
    • multiplicar = a * b: Calcula el producto y lo asigna al nombre de la función.

Conceptos Avanzados de Función

Funciones Recursivas

Fortran soporta funciones recursivas, que son funciones que se llaman a sí mismas. Para definir una función recursiva, debes usar la palabra clave RECURSIVE.

Ejemplo: Función Recursiva de Factorial

Aquí hay un ejemplo de una función recursiva para calcular el factorial de un número:

PROGRAM main
  IMPLICIT NONE
  INTEGER :: resultado

  resultado = factorial(5)
  PRINT *, "El factorial es: ", resultado

CONTAINS

  RECURSIVE FUNCTION factorial(n) RESULT(fact)
    INTEGER :: fact
    INTEGER, INTENT(IN) :: n

    IF (n <= 1) THEN
      fact = 1
    ELSE
      fact = n * factorial(n - 1)
    END IF
  END FUNCTION factorial

END PROGRAM main
  • Explicación:
    • RECURSIVE FUNCTION factorial(n) RESULT(fact): Define una función recursiva factorial con resultado fact.
    • IF (n <= 1) THEN fact = 1 ELSE fact = n * factorial(n - 1) END IF: Implementa la lógica recursiva para calcular el factorial.

Funciones con Argumentos de Array

Las funciones también pueden tomar arrays como argumentos y devolver arrays.

Ejemplo: Función con Argumentos de Array

Aquí hay un ejemplo de una función que suma dos arrays elemento por elemento:

PROGRAM main
  IMPLICIT NONE
  REAL, DIMENSION(3) :: array1 = (/1.0, 2.0, 3.0/)
  REAL, DIMENSION(3) :: array2 = (/4.0, 5.0, 6.0/)
  REAL, DIMENSION(3) :: resultado

  resultado = sumar_arrays(array1, array2)
  PRINT *, "La suma de los arrays es: ", resultado

CONTAINS

  FUNCTION sumar_arrays(a, b) RESULT(suma)
    REAL, DIMENSION(3) :: suma
    REAL, DIMENSION(3), INTENT(IN) :: a, b

    suma = a + b
  END FUNCTION sumar_arrays

END PROGRAM main
  • Explicación:
    • REAL, DIMENSION(3) :: array1, array2, resultado: Declara arrays con tres elementos.
    • resultado = sumar_arrays(array1, array2): Llama a la función sumar_arrays con array1 y array2.
    • FUNCTION sumar_arrays(a, b) RESULT(suma): Define la función sumar_arrays con resultado suma.
    • REAL, DIMENSION(3), INTENT(IN) :: a, b: Declara los argumentos de entrada a y b como arrays.
    • suma = a + b: Suma los arrays elemento por elemento y asigna el resultado a suma.

Conclusión

Las funciones en Fortran son herramientas poderosas que te ayudan a organizar y reutilizar código de manera eficiente. Cubrimos los conceptos básicos de la definición de funciones, el uso de diferentes tipos de argumentos y conceptos avanzados como la recursión y los argumentos de array. Al dominar las funciones, puedes escribir programas en Fortran más modulares, legibles y mantenibles.
