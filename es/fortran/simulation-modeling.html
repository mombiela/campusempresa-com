Introducción a la Simulación y el Modelado

La simulación y el modelado son técnicas esenciales en la computación científica, permitiendo a los investigadores crear experimentos virtuales y predecir comportamientos de sistemas complejos. Fortran, con sus potentes capacidades de cálculo numérico, se utiliza ampliamente en estos campos.

Conceptos Clave

  • Modelado: Crear una representación matemática de un sistema del mundo real.
  • Simulación: Realizar experimentos en el modelo para estudiar su comportamiento bajo diversas condiciones.
  • Modelos Deterministas: Modelos donde el resultado se determina precisamente a través de relaciones conocidas.
  • Modelos Estocásticos: Modelos que incorporan aleatoriedad y comportamiento probabilístico.

Construyendo un Modelo Simple en Fortran

Ejemplo: Modelo de Crecimiento Poblacional

Un ejemplo común de un modelo simple es el crecimiento exponencial de una población.

Ejemplo de Código

program population_growth
    implicit none
    integer :: i, n
    real :: P0, r, P

    ! Población inicial
    P0 = 100.0
    ! Tasa de crecimiento
    r = 0.05
    ! Número de pasos de tiempo
    n = 10

    print *, 'Paso de Tiempo', 'Población'
    do i = 0, n
        P = P0 * exp(r * i)
        print *, i, P
    end do
end program population_growth

Explicación

  • P0: Población inicial.
  • r: Tasa de crecimiento.
  • n: Número de pasos de tiempo.
  • exp: Función intrínseca de Fortran para el exponencial.

Este programa calcula e imprime la población en cada paso de tiempo utilizando la fórmula de crecimiento exponencial \( P = P0 \times e^{rt} \).

Técnicas Avanzadas de Modelado

Ecuaciones Diferenciales

Muchos sistemas del mundo real se modelan utilizando ecuaciones diferenciales. Fortran puede resolver estas ecuaciones utilizando métodos numéricos como el método de Euler o los métodos de Runge-Kutta.

Ejemplo: Resolviendo una EDO usando el Método de Euler

Consideremos la ecuación diferencial \( \frac{dy}{dt} = -ky \).

Ejemplo de Código

program euler_method
    implicit none
    integer :: i, n
    real :: y, t, dt, k

    ! Condiciones iniciales
    y = 1.0
    t = 0.0
    dt = 0.1
    k = 0.5
    n = 100

    print *, 'Tiempo', 'y'
    do i = 1, n
        y = y - k * y * dt
        t = t + dt
        print *, t, y
    end do
end program euler_method

Explicación

  • y: Variable dependiente.
  • t: Variable independiente (tiempo).
  • dt: Tamaño del paso de tiempo.
  • k: Constante de tasa.

Este programa utiliza el método de Euler para resolver la ecuación diferencial e imprime el valor de \( y \) en cada paso de tiempo.

Modelado Estocástico

Simulaciones de Monte Carlo

Los métodos de Monte Carlo utilizan la aleatoriedad para resolver problemas que podrían ser deterministas en principio. Son particularmente útiles para sistemas con un gran número de grados de libertad acoplados.

Ejemplo: Estimando Pi

Un ejemplo clásico de simulación de Monte Carlo es estimar el valor de Pi colocando aleatoriamente puntos en un cuadrado unitario y contando cuántos caen dentro de un cuarto de círculo.

Ejemplo de Código

program monte_carlo_pi
    implicit none
    integer :: i, n, count
    real :: x, y, pi_estimate

    ! Número de puntos aleatorios
    n = 1000000
    count = 0

    call random_seed()

    do i = 1, n
        call random_number(x)
        call random_number(y)
        if (x**2 + y**2 <= 1.0) count = count + 1
    end do

    pi_estimate = 4.0 * count / n
    print *, 'Pi Estimado = ', pi_estimate
end program monte_carlo_pi

Explicación

  • random_seed: Inicializa el generador de números aleatorios.
  • random_number: Genera un número aleatorio entre 0 y 1.
  • count: Cuenta el número de puntos dentro del cuarto de círculo.

Este programa estima Pi generando puntos aleatorios y verificando si caen dentro del cuarto de círculo.

Conclusión

La simulación y el modelado son herramientas poderosas en la computación científica, y Fortran proporciona capacidades robustas para implementar estas técnicas. Desde modelos deterministas simples hasta simulaciones estocásticas complejas, la destreza numérica de Fortran lo convierte en una excelente opción para investigadores e ingenieros. Al dominar estos conceptos y técnicas, puedes modelar y simular eficazmente una amplia gama de sistemas del mundo real.
