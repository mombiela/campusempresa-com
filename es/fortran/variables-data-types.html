Introducción

Fortran, abreviatura de "Formula Translation", es uno de los lenguajes de programación de alto nivel más antiguos, utilizado principalmente para la computación numérica y científica. Entender las variables y los tipos de datos es fundamental para programar en Fortran. Esta sección cubrirá los conceptos básicos de la declaración de variables, la comprensión de los diferentes tipos de datos y su uso en Fortran.

Variables en Fortran

Las variables se utilizan para almacenar datos que pueden ser manipulados a lo largo de un programa. En Fortran, las variables deben ser declaradas antes de ser utilizadas.

Declaración de Variables

Para declarar una variable en Fortran, se especifica el tipo seguido del nombre de la variable. Opcionalmente, también se puede inicializar la variable en el momento de la declaración.

INTEGER :: edad
REAL :: altura = 5.9
CHARACTER(LEN=20) :: nombre = "John Doe"
  • INTEGER declara una variable entera.
  • REAL declara una variable de punto flotante.
  • CHARACTER declara una variable de cadena con una longitud especificada.

Reglas para Nombrar Variables

  • Los nombres de las variables deben comenzar con una letra.
  • Pueden contener letras, dígitos y guiones bajos.
  • No distinguen entre mayúsculas y minúsculas (por ejemplo, Edad y edad se consideran iguales).

Tipos de Datos en Fortran

Fortran admite varios tipos de datos, cada uno adecuado para diferentes tipos de datos. Comprender estos tipos es crucial para una programación eficiente.

Tipos de Datos Básicos

  1. INTEGER: Utilizado para números enteros. 
    INTEGER :: cuenta = 10
  2. REAL: Utilizado para números de punto flotante. 
    REAL :: temperatura = 36.6
  3. DOUBLE PRECISION: Utilizado para números de punto flotante de doble precisión. 
    DOUBLE PRECISION :: numeroGrande = 1.0D+300
  4. COMPLEX: Utilizado para números complejos. 
    COMPLEX :: z = (3.0, 4.0)
  5. CHARACTER: Utilizado para cadenas. 
    CHARACTER(LEN=10) :: saludo = "Hola"
  6. LOGICAL: Utilizado para valores booleanos. 
    LOGICAL :: esValido = .TRUE.

Tipos de Datos Derivados

Fortran permite la creación de tipos de datos personalizados utilizando la palabra clave TYPE. Esto es útil para agrupar variables relacionadas.

TYPE :: Persona
    CHARACTER(LEN=20) :: nombre
    INTEGER :: edad
    REAL :: altura
END TYPE Persona

TYPE(Persona) :: estudiante
estudiante%nombre = "Alice"
estudiante%edad = 21
estudiante%altura = 5.7

Conversión de Tipos

Fortran proporciona funciones intrínsecas para convertir entre diferentes tipos de datos.

  • INT(): Convierte a entero.
  • REAL(): Convierte a real.
  • DBLE(): Convierte a doble precisión.
  • CMPLX(): Convierte a complejo.
  • CHAR(): Convierte a carácter.

Ejemplo

INTEGER :: i = 5
REAL :: r
r = REAL(i)  ! Convierte entero a real

Arreglos

Los arreglos se utilizan para almacenar múltiples valores del mismo tipo. Pueden ser declarados con dimensiones especificadas.

Declaración de Arreglos

INTEGER, DIMENSION(5) :: numeros = (/1, 2, 3, 4, 5/)
REAL, DIMENSION(3, 3) :: matriz

Acceso a Elementos del Arreglo

Los elementos del arreglo se acceden utilizando índices.

numeros(1) = 10  ! Establece el primer elemento a 10
matriz(2, 3) = 5.5  ! Establece el elemento en la fila 2, columna 3 a 5.5

Resumen

En esta sección, cubrimos los conceptos básicos de las variables y los tipos de datos en Fortran. Aprendimos cómo declarar variables, entender diferentes tipos de datos y usar arreglos. Dominar estos conceptos es esencial para escribir programas eficientes y efectivos en Fortran. A medida que avances, encontrarás estructuras de datos y operaciones más complejas, pero una comprensión sólida de estos fundamentos proporcionará una base sólida para tu viaje de programación en Fortran.
