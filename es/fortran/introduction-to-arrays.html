Visión General

Los arrays son una estructura de datos fundamental en Fortran, que te permite almacenar y manipular colecciones de datos de manera eficiente. Esta sección te introducirá a los conceptos básicos de los arrays, incluyendo su declaración, inicialización y operaciones comunes.

¿Qué es un Array?

  • Un array es una colección de elementos, todos del mismo tipo, almacenados en ubicaciones de memoria contiguas.
  • Los arrays pueden ser unidimensionales (como una lista) o multidimensionales (como una matriz).

Declaración de Arrays

Para declarar un array en Fortran, especificas el tipo de elementos y las dimensiones del array.

Arrays Unidimensionales

INTEGER :: array1D(10)
REAL :: array1D_real(5)
  • INTEGER :: array1D(10) declara un array de enteros con 10 elementos.
  • REAL :: array1D_real(5) declara un array de números reales con 5 elementos.

Arrays Multidimensionales

INTEGER :: array2D(3, 3)
REAL :: array3D(2, 2, 2)
  • INTEGER :: array2D(3, 3) declara un array de enteros de 3x3.
  • REAL :: array3D(2, 2, 2) declara un array de números reales de 2x2x2.

Inicialización de Arrays

Los arrays pueden ser inicializados en el momento de la declaración o más tarde en el código.

Inicialización en la Declaración

INTEGER :: array1D(5) = (/1, 2, 3, 4, 5/)
REAL :: array2D(2, 2) = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/))
  • INTEGER :: array1D(5) = (/1, 2, 3, 4, 5/) inicializa un array 1D con valores del 1 al 5.
  • REAL :: array2D(2, 2) = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/)) inicializa un array de 2x2 con valores 1.0, 2.0, 3.0 y 4.0.

Inicialización Más Tarde en el Código

INTEGER :: array1D(5)
array1D = (/1, 2, 3, 4, 5/)

REAL :: array2D(2, 2)
array2D = RESHAPE((/1.0, 2.0, 3.0, 4.0/), (/2, 2/))

Acceso a Elementos del Array

Puedes acceder y modificar los elementos del array usando sus índices.

Acceso a Elementos

INTEGER :: value
value = array1D(3)  ! Accede al tercer elemento de array1D

Modificación de Elementos

array1D(3) = 10  ! Establece el tercer elemento de array1D a 10

Operaciones Comunes con Arrays

  • Recorrido del Array: Recorrer los elementos del array.
  • Segmentación del Array: Acceso a subarrays.
  • Funciones Intrínsecas del Array: Funciones integradas para la manipulación de arrays.

Recorrido del Array

INTEGER :: i
DO i = 1, 5
    PRINT *, array1D(i)
END DO

Segmentación del Array

INTEGER, DIMENSION(5) :: subArray
subArray = array1D(2:4)  ! Extrae los elementos del 2 al 4

Funciones Intrínsecas del Array

REAL :: sum
sum = SUM(array1D_real)  ! Calcula la suma de los elementos en array1D_real

Conclusión

Los arrays son una herramienta poderosa en Fortran para manejar colecciones de datos. Entender cómo declarar, inicializar y manipular arrays es esencial para una programación eficiente. Practica estos conceptos con los ejemplos y ejercicios proporcionados para solidificar tu comprensión.

Ejercicios

  1. Declara un array unidimensional de enteros con 7 elementos e inicialízalo con valores del 1 al 7.
  2. Crea un array real de 3x3 e inicialízalo con valores del 1.0 al 9.0.
  3. Escribe un bucle para imprimir todos los elementos del array de 3x3.
  4. Usa la segmentación de arrays para extraer un subarray del array unidimensional creado en el ejercicio 1.
  5. Calcula la suma de los elementos en el array de 3x3 usando una función intrínseca.
