Introducción a los Punteros

Los punteros son una característica poderosa en Fortran que te permite referenciar y manipular ubicaciones de memoria directamente. Esto puede ser particularmente útil para la asignación dinámica de memoria, la creación de estructuras de datos complejas y la optimización del rendimiento.

Conceptos Clave

  • Declaración de Punteros: Cómo declarar una variable puntero.
  • Asociación de Punteros: Asociar un puntero con un objetivo.
  • Operaciones con Punteros: Operaciones básicas que puedes realizar con punteros.
  • Asignación Dinámica de Memoria: Usar punteros para asignar y liberar memoria dinámicamente.
  • Arreglos de Punteros: Trabajar con arreglos de punteros.

Declaración de Punteros

Para declarar un puntero en Fortran, utilizas el atributo POINTER en la declaración del tipo.

INTEGER, POINTER :: ptr
REAL, POINTER :: rptr

En este ejemplo:

  • ptr es un puntero a un entero.
  • rptr es un puntero a un número real.

Asociación de Punteros

Los punteros necesitan ser asociados con un objetivo antes de que puedan ser utilizados. Esto se hace usando el operador =>.

INTEGER, TARGET :: target_var
INTEGER, POINTER :: ptr

ptr => target_var

En este ejemplo:

  • target_var es una variable entera que puede ser objetivo de punteros.
  • ptr está asociado con target_var.

Operaciones con Punteros

Una vez que un puntero está asociado con un objetivo, puedes realizar varias operaciones con él.

Ejemplo: Operaciones Básicas con Punteros

INTEGER, TARGET :: target_var = 10
INTEGER, POINTER :: ptr

ptr => target_var
PRINT *, "Valor de target_var: ", target_var
PRINT *, "Valor vía puntero: ", ptr

ptr = 20
PRINT *, "Nuevo valor de target_var: ", target_var

Salida:

Valor de target_var: 10
Valor vía puntero: 10
Nuevo valor de target_var: 20

Asignación Dinámica de Memoria

Los punteros son esenciales para la asignación dinámica de memoria en Fortran. Puedes asignar y liberar memoria usando las sentencias ALLOCATE y DEALLOCATE.

Ejemplo: Asignación Dinámica de Memoria

INTEGER, POINTER :: ptr
ALLOCATE(ptr)
ptr = 42
PRINT *, "Valor asignado dinámicamente: ", ptr
DEALLOCATE(ptr)

Salida:

Valor asignado dinámicamente: 42

Arreglos de Punteros

También puedes crear arreglos de punteros, lo cual puede ser útil para gestionar colecciones de memoria asignada dinámicamente.

Ejemplo: Arreglos de Punteros

INTEGER, POINTER :: ptr(:)
ALLOCATE(ptr(5))

ptr(1) = 1
ptr(2) = 2
ptr(3) = 3
ptr(4) = 4
ptr(5) = 5

PRINT *, "Valores del arreglo de punteros: ", ptr

DEALLOCATE(ptr)

Salida:

Valores del arreglo de punteros: 1 2 3 4 5

Resumen

Los punteros en Fortran proporcionan una manera de referenciar y manipular ubicaciones de memoria directamente, lo cual es útil para la asignación dinámica de memoria y la creación de estructuras de datos complejas. Los conceptos clave incluyen la declaración de punteros, la asociación, las operaciones básicas, la asignación dinámica de memoria y los arreglos de punteros. Entender estos conceptos te permitirá escribir programas en Fortran más flexibles y eficientes.
