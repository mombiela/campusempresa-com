La asignación dinámica de memoria es un concepto crucial en Fortran, especialmente para manejar estructuras de datos cuyo tamaño puede no ser conocido en tiempo de compilación. Este tema te guiará desde los conceptos básicos hasta los avanzados de la asignación dinámica de memoria en Fortran.

Introducción a la Asignación Dinámica de Memoria

La asignación dinámica de memoria te permite asignar memoria durante el tiempo de ejecución del programa. Esto es particularmente útil cuando se trabaja con matrices u otras estructuras de datos donde el tamaño puede variar.

Conceptos Clave

  • Asignación de Memoria Estática vs. Dinámica: La asignación de memoria estática ocurre en tiempo de compilación, mientras que la asignación de memoria dinámica ocurre en tiempo de ejecución.
  • Sentencias ALLOCATE y DEALLOCATE: Se utilizan para asignar y liberar memoria dinámicamente.
  • Punteros: Variables que almacenan la dirección de la memoria asignada dinámicamente.

Asignación Dinámica de Memoria Básica

Sentencias ALLOCATE y DEALLOCATE

La sentencia ALLOCATE se usa para asignar memoria para matrices u otras estructuras de datos, y la sentencia DEALLOCATE se usa para liberar esa memoria.

Ejemplo: Asignación y Liberación de una Matriz

PROGRAM DynamicArray
  IMPLICIT NONE
  INTEGER, ALLOCATABLE :: array(:)
  INTEGER :: n, i

  ! Preguntar al usuario por el tamaño de la matriz
  PRINT *, 'Ingrese el tamaño de la matriz:'
  READ *, n

  ! Asignar memoria para la matriz
  ALLOCATE(array(n))

  ! Inicializar la matriz
  DO i = 1, n
    array(i) = i * 2
  END DO

  ! Imprimir la matriz
  PRINT *, 'Elementos de la matriz:'
  PRINT *, array

  ! Liberar la memoria
  DEALLOCATE(array)
END PROGRAM DynamicArray

Manejo de Errores en la Asignación

Es importante manejar los errores que pueden ocurrir durante la asignación de memoria. Esto se puede hacer usando el especificador STAT.

Ejemplo: Manejo de Errores en la Asignación

PROGRAM AllocationErrorHandling
  IMPLICIT NONE
  INTEGER, ALLOCATABLE :: array(:)
  INTEGER :: n, stat

  ! Preguntar al usuario por el tamaño de la matriz
  PRINT *, 'Ingrese el tamaño de la matriz:'
  READ *, n

  ! Asignar memoria para la matriz con manejo de errores
  ALLOCATE(array(n), STAT=stat)
  IF (stat /= 0) THEN
    PRINT *, 'Error: No se pudo asignar memoria.'
    STOP
  END IF

  ! Liberar la memoria
  DEALLOCATE(array)
END PROGRAM AllocationErrorHandling

Asignación Dinámica de Memoria Avanzada

Uso de Punteros

Los punteros en Fortran se pueden usar para asignar memoria dinámicamente y referenciarla. Esto es útil para crear estructuras de datos complejas como listas enlazadas.

Ejemplo: Uso de Punteros para Asignación Dinámica

PROGRAM PointerExample
  IMPLICIT NONE
  INTEGER, POINTER :: pArray(:)
  INTEGER :: n, i

  ! Preguntar al usuario por el tamaño de la matriz
  PRINT *, 'Ingrese el tamaño de la matriz:'
  READ *, n

  ! Asignar memoria para la matriz usando punteros
  ALLOCATE(pArray(n))

  ! Inicializar la matriz
  DO i = 1, n
    pArray(i) = i * 3
  END DO

  ! Imprimir la matriz
  PRINT *, 'Elementos de la matriz con punteros:'
  PRINT *, pArray

  ! Liberar la memoria
  DEALLOCATE(pArray)
END PROGRAM PointerExample

Matrices Dinámicas en Tipos Derivados

Fortran permite matrices dinámicas dentro de tipos derivados, lo cual puede ser útil para crear estructuras de datos más complejas.

Ejemplo: Matrices Dinámicas en Tipos Derivados

MODULE DynamicModule
  TYPE :: DynamicType
    INTEGER, ALLOCATABLE :: array(:)
  END TYPE DynamicType
END MODULE DynamicModule

PROGRAM DerivedTypeExample
  USE DynamicModule
  IMPLICIT NONE
  TYPE(DynamicType) :: myType
  INTEGER :: n, i

  ! Preguntar al usuario por el tamaño de la matriz
  PRINT *, 'Ingrese el tamaño de la matriz:'
  READ *, n

  ! Asignar memoria para la matriz dentro del tipo derivado
  ALLOCATE(myType%array(n))

  ! Inicializar la matriz
  DO i = 1, n
    myType%array(i) = i * 4
  END DO

  ! Imprimir la matriz
  PRINT *, 'Elementos de la matriz en el tipo derivado:'
  PRINT *, myType%array

  ! Liberar la memoria
  DEALLOCATE(myType%array)
END PROGRAM DerivedTypeExample

Resumen

La asignación dinámica de memoria en Fortran es una característica poderosa que permite una gestión de memoria flexible y eficiente. Usando las sentencias ALLOCATE y DEALLOCATE, manejando errores, y utilizando punteros y tipos derivados, puedes crear estructuras de datos complejas y dinámicas. Entender estos conceptos es esencial para la programación avanzada en Fortran y puede mejorar significativamente el rendimiento y la flexibilidad de tus programas.
