Introducción

En Fortran, las estructuras de control son esenciales para dirigir el flujo de un programa. Una de las estructuras de control más fundamentales es la declaración if, que te permite ejecutar ciertas secciones de código basadas en condiciones específicas. Este tema cubrirá los conceptos básicos de las declaraciones if, incluyendo sintaxis, ejemplos y ejercicios para ayudarte a entender y aplicar esta estructura de control de manera efectiva.

Declaración If Básica

La forma más simple de una declaración if en Fortran verifica una condición y ejecuta un bloque de código si la condición es verdadera.

Sintaxis

if (condition) then
    ! Código a ejecutar si la condición es verdadera
end if

Ejemplo

program basic_if
    implicit none
    integer :: a

    a = 5

    if (a > 3) then
        print *, "a es mayor que 3"
    end if
end program basic_if

En este ejemplo, el programa verifica si a es mayor que 3. Dado que a es 5, la condición es verdadera y se imprime el mensaje "a es mayor que 3".

Declaración If-Else

A veces, necesitas ejecutar un bloque de código si una condición es verdadera y otro bloque si la condición es falsa. Aquí es donde entra la declaración if-else.

Sintaxis

if (condition) then
    ! Código a ejecutar si la condición es verdadera
else
    ! Código a ejecutar si la condición es falsa
end if

Ejemplo

program if_else
    implicit none
    integer :: a

    a = 2

    if (a > 3) then
        print *, "a es mayor que 3"
    else
        print *, "a no es mayor que 3"
    end if
end program if_else

En este ejemplo, el programa verifica si a es mayor que 3. Dado que a es 2, la condición es falsa y se imprime el mensaje "a no es mayor que 3".

Declaración If-Else If-Else

Para una toma de decisiones más compleja, puedes usar múltiples condiciones con else if.

Sintaxis

if (condition1) then
    ! Código a ejecutar si la condición1 es verdadera
else if (condition2) then
    ! Código a ejecutar si la condición2 es verdadera
else
    ! Código a ejecutar si ninguna de las condiciones anteriores es verdadera
end if

Ejemplo

program if_elseif_else
    implicit none
    integer :: a

    a = 5

    if (a > 10) then
        print *, "a es mayor que 10"
    else if (a > 3) then
        print *, "a es mayor que 3 pero menor o igual a 10"
    else
        print *, "a es 3 o menos"
    end if
end program if_elseif_else

En este ejemplo, el programa verifica múltiples condiciones. Dado que a es 5, la segunda condición es verdadera y se imprime el mensaje "a es mayor que 3 pero menor o igual a 10".

Declaraciones If Anidadas

También puedes anidar declaraciones if dentro de otras para manejar condiciones más complejas.

Ejemplo

program nested_if
    implicit none
    integer :: a, b

    a = 5
    b = 7

    if (a > 3) then
        if (b > 6) then
            print *, "a es mayor que 3 y b es mayor que 6"
        else
            print *, "a es mayor que 3 pero b no es mayor que 6"
        end if
    else
        print *, "a no es mayor que 3"
    end if
end program nested_if

En este ejemplo, el programa primero verifica si a es mayor que 3. Si es verdadero, luego verifica si b es mayor que 6. Dado que ambas condiciones son verdaderas, se imprime el mensaje "a es mayor que 3 y b es mayor que 6".

Ejercicios

  1. Escribe un programa que verifique si un número es positivo, negativo o cero e imprima un mensaje apropiado.
  2. Modifica el ejemplo if_elseif_else para incluir una condición que verifique si a es exactamente 5 e imprima un mensaje especial.
  3. Crea una declaración if anidada que verifique dos variables e imprima diferentes mensajes basados en sus valores.

Conclusión

Entender las declaraciones if en Fortran es crucial para controlar el flujo de tus programas. Al dominar las declaraciones básicas if, if-else, if-else if-else y las declaraciones if anidadas, puedes manejar una amplia gama de escenarios de toma de decisiones en tu código. Practica con los ejemplos y ejercicios proporcionados para solidificar tu comprensión y convertirte en un experto en el uso de declaraciones if en Fortran.
