Introducción

Fortran, abreviatura de "Formula Translation", es uno de los lenguajes de programación de alto nivel más antiguos, utilizado principalmente para la computación numérica y científica. Esta sección te guiará a través de la escritura de tu primer programa en Fortran, cubriendo los conceptos básicos de sintaxis, estructura y ejecución.

Conceptos Clave

  • Estructura del Programa Fortran: Entender el diseño básico de un programa Fortran.
  • Variables y Tipos de Datos: Introducción a la declaración y uso de variables.
  • Entrada y Salida Básica: Cómo leer desde y escribir en la consola.
  • Compilación y Ejecución: Pasos para compilar y ejecutar tu código Fortran.

Estructura del Programa Fortran

Un programa Fortran simple consta de las siguientes partes:

  1. Nombre del Programa: Declara el nombre del programa.
  2. Implicit None: Asegura que todas las variables deben ser declaradas explícitamente.
  3. Declaraciones de Variables: Declara las variables utilizadas en el programa.
  4. Sentencias Ejecutables: Contiene la lógica y las operaciones.
  5. End Program: Marca el final del programa.

Ejemplo

program HelloWorld
  implicit none
  ! Las declaraciones de variables van aquí

  ! Sentencias ejecutables
  print *, '¡Hola, Mundo!'

end program HelloWorld

Explicación

  • program HelloWorld: Define el inicio del programa llamado "HelloWorld".
  • implicit none: Asegura que todas las variables deben ser declaradas explícitamente, lo que ayuda a prevenir errores.
  • print *, '¡Hola, Mundo!': Imprime la cadena "¡Hola, Mundo!" en la consola.
  • end program HelloWorld: Marca el final del programa.

Variables y Tipos de Datos

Fortran soporta varios tipos de datos, incluyendo enteros, números reales y caracteres.

Ejemplo

program VariablesExample
  implicit none
  integer :: age
  real :: height
  character(len=20) :: name

  ! Asignando valores a las variables
  age = 25
  height = 5.9
  name = 'John Doe'

  ! Imprimiendo variables
  print *, 'Nombre:', name
  print *, 'Edad:', age
  print *, 'Altura:', height

end program VariablesExample

Explicación

  • integer :: age: Declara una variable entera llamada age.
  • real :: height: Declara una variable real (de punto flotante) llamada height.
  • character(len=20) :: name: Declara una variable de carácter llamada name con una longitud de 20.
  • Las sentencias print muestran los valores de las variables en la consola.

Entrada y Salida Básica

Fortran usa las sentencias read y print para operaciones de entrada y salida.

Ejemplo

program InputOutputExample
  implicit none
  integer :: age
  real :: height
  character(len=20) :: name

  ! Leyendo entrada del usuario
  print *, 'Introduce tu nombre:'
  read *, name
  print *, 'Introduce tu edad:'
  read *, age
  print *, 'Introduce tu altura:'
  read *, height

  ! Imprimiendo los valores de entrada
  print *, 'Nombre:', name
  print *, 'Edad:', age
  print *, 'Altura:', height

end program InputOutputExample

Explicación

  • read *, name: Lee una entrada de cadena del usuario y la almacena en la variable name.
  • read *, age: Lee una entrada entera del usuario y la almacena en la variable age.
  • read *, height: Lee una entrada de número real del usuario y la almacena en la variable height.

Compilación y Ejecución

Para compilar y ejecutar un programa Fortran, normalmente se usa un compilador Fortran como gfortran.

Pasos

  1. Escribir el código: Guarda tu código Fortran en un archivo con una extensión .f90 (por ejemplo, HelloWorld.f90).
  2. Compilar el código: Usa el comando gfortran HelloWorld.f90 -o HelloWorld para compilar el código.
  3. Ejecutar el ejecutable: Ejecuta el programa compilado con ./HelloWorld.

Ejemplo

$ gfortran HelloWorld.f90 -o HelloWorld
$ ./HelloWorld
¡Hola, Mundo!

Conclusión

En esta sección, aprendiste cómo escribir un programa básico en Fortran, declarar variables, realizar operaciones de entrada y salida, y compilar y ejecutar tu código. Estas habilidades fundamentales son esenciales a medida que avanzas hacia tareas de programación en Fortran más complejas. Practica escribiendo programas simples para reforzar estos conceptos y aumentar tu confianza en el uso de Fortran.
