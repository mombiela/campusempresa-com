Los módulos y paquetes son componentes esenciales en Fortran que ayudan a organizar el código, promover la reutilización y mantener un espacio de nombres limpio. Esta sección te guiará a través de los conceptos básicos de los módulos, cómo crearlos y usarlos, y cómo organizarlos en paquetes para proyectos más complejos.

Introducción a los Módulos

Los módulos en Fortran se utilizan para agrupar procedimientos y datos relacionados. Ayudan a encapsular datos y procedimientos, haciendo que el código sea más modular y fácil de gestionar.

Conceptos Clave

  • Definición de Módulo: Un módulo se define usando las sentencias MODULE y END MODULE.
  • Acceso Público y Privado: Por defecto, todas las entidades en un módulo son públicas, pero puedes restringir el acceso usando la palabra clave PRIVATE.
  • Sentencia Use: Para acceder al contenido de un módulo en otro programa o módulo, se emplea la sentencia USE.

Ejemplo: Módulo Básico

MODULE MathModule
  IMPLICIT NONE
  PRIVATE
  PUBLIC :: add, subtract

  CONTAINS

  FUNCTION add(a, b)
    INTEGER, INTENT(IN) :: a, b
    INTEGER :: add
    add = a + b
  END FUNCTION add

  FUNCTION subtract(a, b)
    INTEGER, INTENT(IN) :: a, b
    INTEGER :: subtract
    subtract = a - b
  END FUNCTION subtract

END MODULE MathModule

En este ejemplo:

  • El módulo MathModule contiene dos funciones públicas, add y subtract.
  • La palabra clave PRIVATE asegura que cualquier otra entidad en el módulo no sea accesible desde fuera de él.

Usando un Módulo

PROGRAM Main
  USE MathModule
  IMPLICIT NONE

  INTEGER :: result

  result = add(5, 3)
  PRINT *, "5 + 3 =", result

  result = subtract(5, 3)
  PRINT *, "5 - 3 =", result

END PROGRAM Main

Aquí, la sentencia USE MathModule hace que las funciones add y subtract estén disponibles en el programa Main.

Características Avanzadas del Módulo

Procedimientos del Módulo

Los módulos pueden contener procedimientos que se definen dentro del propio módulo. Estos se conocen como procedimientos del módulo.

MODULE AdvancedMathModule
  IMPLICIT NONE
  PUBLIC :: multiply

  CONTAINS

  FUNCTION multiply(a, b)
    INTEGER, INTENT(IN) :: a, b
    INTEGER :: multiply
    multiply = a * b
  END FUNCTION multiply

END MODULE AdvancedMathModule

Variables del Módulo

Los módulos también pueden contener variables que pueden ser compartidas entre procedimientos.

MODULE SharedDataModule
  IMPLICIT NONE
  INTEGER :: counter = 0
  PUBLIC :: counter

  CONTAINS

  SUBROUTINE increment_counter()
    counter = counter + 1
  END SUBROUTINE increment_counter

END MODULE SharedDataModule

Ejemplo: Usando Variables del Módulo

PROGRAM CounterProgram
  USE SharedDataModule
  IMPLICIT NONE

  CALL increment_counter()
  PRINT *, "Counter:", counter

  CALL increment_counter()
  PRINT *, "Counter:", counter

END PROGRAM CounterProgram

Paquetes en Fortran

Fortran no tiene un concepto incorporado de paquetes como algunos otros lenguajes de programación, pero puedes organizar tus módulos en directorios para crear una estructura similar a un paquete.

Organizando Módulos en Paquetes

  1. Estructura de Directorios: Crea un directorio para tu paquete y coloca los módulos relacionados dentro de él.
  2. Rutas de Inclusión: Usa opciones del compilador para especificar las rutas a tus archivos de módulo.

Ejemplo de Estructura de Directorios

my_project/
  src/
    math/
      MathModule.f90
      AdvancedMathModule.f90
    main/
      MainProgram.f90

Compilando con Paquetes

Al compilar, necesitas especificar las rutas a los archivos de módulo. Por ejemplo, usando gfortran:

gfortran -I./src/math -o main_program ./src/main/MainProgram.f90 ./src/math/MathModule.f90 ./src/math/AdvancedMathModule.f90

Conclusión

Los módulos y paquetes en Fortran son herramientas poderosas para organizar y gestionar tu código. Al encapsular procedimientos y datos relacionados, los módulos promueven la reutilización y el mantenimiento. Aunque Fortran no tiene un sistema de paquetes formal, puedes lograr una funcionalidad similar organizando tus módulos en directorios y gestionando las rutas de inclusión durante la compilación. Entender y usar efectivamente los módulos mejorará significativamente tus habilidades de programación en Fortran.
