Leer archivos es una habilidad fundamental en la programación en Fortran, lo que te permite manejar la entrada de datos de manera eficiente. Este tema te guiará desde los conceptos básicos hasta las técnicas avanzadas de lectura de archivos en Fortran.

Introducción al Manejo de Archivos en Fortran

Antes de profundizar en la lectura de archivos, es esencial entender algunos conceptos básicos del manejo de archivos en Fortran.

  • Números de Unidad de Archivo: Fortran usa números de unidad para referirse a los archivos. Estos son valores enteros que identifican de manera única cada archivo durante la ejecución del programa.
  • Sentencia OPEN: Esta sentencia se usa para abrir un archivo y asociarlo con un número de unidad.
  • Sentencia READ: Esta sentencia lee datos de un archivo.
  • Sentencia CLOSE: Esta sentencia cierra un archivo que ha sido abierto.

Lectura Básica de Archivos

Apertura de un Archivo

Para leer de un archivo, primero necesitas abrirlo usando la sentencia OPEN. Aquí tienes un ejemplo simple:

PROGRAM ReadFile
  INTEGER :: unit_number
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')
  ! Ahora el archivo 'data.txt' está asociado con el número de unidad 10
END PROGRAM ReadFile
  • unit=unit_number: Especifica el número de unidad.
  • file='data.txt': Especifica el nombre del archivo.
  • status='old': Indica que el archivo ya existe.

Lectura de Datos

Una vez que el archivo está abierto, puedes leer datos usando la sentencia READ.

PROGRAM ReadFile
  INTEGER :: unit_number, number
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')
  READ(unit_number, *) number
  PRINT *, 'Número leído del archivo:', number
  CLOSE(unit_number)
END PROGRAM ReadFile
  • READ(unit_number, *) number: Lee datos del archivo asociado con unit_number en la variable number.

Cierre de un Archivo

Después de leer los datos, es una buena práctica cerrar el archivo usando la sentencia CLOSE.

CLOSE(unit_number)

Lectura de Diferentes Tipos de Datos

Fortran te permite leer varios tipos de datos de un archivo, como enteros, números reales y cadenas.

Ejemplo: Lectura de Múltiples Tipos de Datos

PROGRAM ReadFile
  INTEGER :: unit_number, int_val
  REAL :: real_val
  CHARACTER(len=20) :: str_val

  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')
  READ(unit_number, *) int_val, real_val, str_val
  PRINT *, 'Entero:', int_val
  PRINT *, 'Real:', real_val
  PRINT *, 'Cadena:', str_val
  CLOSE(unit_number)
END PROGRAM ReadFile

Técnicas Avanzadas de Lectura de Archivos

Lectura de Datos en un Bucle

A menudo, necesitas leer múltiples líneas de datos. Esto se puede hacer usando un bucle.

PROGRAM ReadFile
  INTEGER :: unit_number, int_val, i
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')

  DO i = 1, 10
    READ(unit_number, *) int_val
    PRINT *, 'Valor leído:', int_val
  END DO

  CLOSE(unit_number)
END PROGRAM ReadFile

Manejo del Fin de Archivo

Para manejar la condición de fin de archivo (EOF), puedes usar el especificador IOSTAT.

PROGRAM ReadFile
  INTEGER :: unit_number, int_val, io_status
  unit_number = 10
  OPEN(unit=unit_number, file='data.txt', status='old')

  DO
    READ(unit_number, *, IOSTAT=io_status) int_val
    IF (io_status /= 0) EXIT
    PRINT *, 'Valor leído:', int_val
  END DO

  CLOSE(unit_number)
END PROGRAM ReadFile
  • IOSTAT=io_status: Almacena el estado de la operación de E/S. Un valor distinto de cero indica un error o EOF.

Resumen

En este tema, cubrimos las técnicas básicas y avanzadas de lectura de archivos en Fortran:

  • Apertura y cierre de archivos usando las sentencias OPEN y CLOSE.
  • Lectura de datos usando la sentencia READ.
  • Manejo de diferentes tipos de datos.
  • Lectura de datos en bucles y manejo de condiciones de EOF.

Entender estos conceptos te permitirá manejar de manera eficiente la entrada de datos en tus programas en Fortran. Practica estas técnicas con varios archivos de datos para volverte competente en el manejo de archivos.
