Introducción

En esta sección, revisaremos los conceptos y técnicas clave cubiertos a lo largo del curso. Esto ayudará a reforzar tu comprensión y asegurará que estés bien preparado para aplicar Fortran en escenarios del mundo real. También presentaremos algunos temas avanzados para darte una idea de lo que puedes explorar más adelante.

Revisión de Conceptos Clave

Sintaxis y Estructura Básica

  • Estructura del Programa: Entender la estructura básica de un programa en Fortran. 
    program HelloWorld
    print *, "¡Hola, Mundo!"
end program HelloWorld
  • Variables y Tipos de Datos: Declarar y usar variables de diferentes tipos de datos. 
    integer :: i
real :: x
character(len=20) :: name

Estructuras de Control

  • Sentencias Condicionales: Usar if, else if y else para la toma de decisiones. 
    if (x > 0) then
    print *, "x es positivo"
else if (x == 0) then
    print *, "x es cero"
else
    print *, "x es negativo"
end if
  • Bucles: Implementar bucles do para iteración. 
    do i = 1, 10
    print *, "i =", i
end do

Arreglos y Matrices

  • Declaración e Inicialización de Arreglos: Crear e inicializar arreglos. 
    integer, dimension(5) :: arr = (/1, 2, 3, 4, 5/)
  • Operaciones con Arreglos: Realizar operaciones en arreglos. 
    arr = arr * 2

Funciones y Subrutinas

  • Funciones: Definir y usar funciones. 
    function add(a, b)
    integer :: add
    integer, intent(in) :: a, b
    add = a + b
end function add
  • Subrutinas: Crear y llamar subrutinas. 
    subroutine print_message(msg)
    character(len=*) :: msg
    print *, msg
end subroutine print_message

Descripción de Temas Avanzados

Módulos

  • Creación de Módulos: Organizar el código en módulos reutilizables. 
    module math_operations
    contains
    function add(a, b)
        integer :: add
        integer, intent(in) :: a, b
        add = a + b
    end function add
end module math_operations
  • Uso de Módulos: Importar y usar módulos en programas. 
    program use_module
    use math_operations
    print *, add(3, 4)
end program use_module

Entrada/Salida de Archivos

  • Lectura de Archivos: Leer datos de archivos externos. 
    open(unit=10, file='data.txt', status='old', action='read')
read(10, *) x
close(10)
  • Escritura en Archivos: Escribir datos en archivos externos. 
    open(unit=20, file='output.txt', status='new', action='write')
write(20, *) x
close(20)

Programación Paralela

  • Introducción a Coarrays: Usar coarrays para programación paralela. 
    real, codimension[*] :: x
x = this_image()

Resumen

En esta sección de 'Presentación y Revisión', revisamos los conceptos fundamentales de Fortran, incluyendo sintaxis básica, estructuras de control, arreglos, funciones y subrutinas. También introdujimos temas avanzados como módulos, entrada/salida de archivos y programación paralela con coarrays. Esta revisión integral debería solidificar tu comprensión y prepararte para tareas de programación en Fortran más complejas. Continúa practicando y explorando estos conceptos para volverte competente en Fortran.
