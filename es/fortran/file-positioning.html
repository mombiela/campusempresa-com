El posicionamiento de archivos es un aspecto esencial del manejo de archivos en Fortran. Permite a los programadores controlar desde dónde se lee o escribe la información en un archivo. Este tema cubrirá los conceptos básicos del posicionamiento de archivos, incluyendo cómo mover el puntero del archivo, y proporcionará ejemplos y ejercicios para solidificar tu comprensión.

Conceptos Clave

  • Puntero de Archivo: Un marcador que indica la posición actual en un archivo donde ocurrirá la próxima operación de lectura o escritura.
  • Acceso Secuencial: Leer o escribir datos en una secuencia, desde el principio hasta el final del archivo.
  • Acceso Directo: Leer o escribir datos en posiciones específicas del archivo, permitiendo el acceso no secuencial.

Acceso Secuencial

En el acceso secuencial, el puntero del archivo se mueve automáticamente a medida que se leen o escriben datos. Esta es la forma más simple de posicionamiento de archivos.

Ejemplo: Lectura Secuencial

PROGRAM SequentialRead
  IMPLICIT NONE
  INTEGER :: i, io_status
  OPEN(UNIT=10, FILE='data.txt', STATUS='OLD', ACTION='READ')

  DO
    READ(10, *, IOSTAT=io_status) i
    IF (io_status /= 0) EXIT
    PRINT *, 'Read value:', i
  END DO

  CLOSE(10)
END PROGRAM SequentialRead

En este ejemplo:

  • El archivo data.txt se abre para lectura.
  • Un bucle lee enteros del archivo hasta que se alcanza el final.
  • El puntero del archivo se mueve automáticamente al siguiente entero después de cada operación de lectura.

Acceso Directo

El acceso directo te permite leer o escribir datos en posiciones específicas del archivo. Esto es útil para el acceso aleatorio a los datos.

Ejemplo: Escritura y Lectura de Acceso Directo

PROGRAM DirectAccess
  IMPLICIT NONE
  INTEGER :: i, rec_num
  INTEGER, DIMENSION(5) :: data = (/1, 2, 3, 4, 5/)
  INTEGER :: io_status

  ! Abrir archivo para acceso directo
  OPEN(UNIT=20, FILE='direct_data.dat', STATUS='NEW', ACCESS='DIRECT', RECL=4)

  ! Escribir datos en registros específicos
  DO rec_num = 1, 5
    WRITE(20, REC=rec_num, IOSTAT=io_status) data(rec_num)
    IF (io_status /= 0) THEN
      PRINT *, 'Error writing to record', rec_num
      STOP
    END IF
  END DO

  ! Leer datos de registros específicos
  DO rec_num = 1, 5
    READ(20, REC=rec_num, IOSTAT=io_status) i
    IF (io_status /= 0) THEN
      PRINT *, 'Error reading from record', rec_num
      STOP
    END IF
    PRINT *, 'Read value from record', rec_num, ':', i
  END DO

  CLOSE(20)
END PROGRAM DirectAccess

En este ejemplo:

  • El archivo direct_data.dat se abre para acceso directo con una longitud de registro de 4 bytes.
  • Los datos se escriben en registros específicos del archivo.
  • Luego, los datos se leen de esos registros, demostrando el acceso aleatorio.

Resumen

El posicionamiento de archivos en Fortran es una habilidad crucial para gestionar las operaciones de E/S de archivos de manera efectiva. Comprender tanto los métodos de acceso secuencial como directo permite una manipulación de datos flexible y eficiente. El acceso secuencial es sencillo y adecuado para operaciones de lectura/escritura simples, mientras que el acceso directo proporciona la capacidad de manipular datos en posiciones específicas, lo cual es útil para aplicaciones más complejas.

Al dominar estas técnicas, puedes manejar una amplia gama de escenarios de E/S de archivos en Fortran, mejorando tu capacidad para desarrollar programas robustos y eficientes.
