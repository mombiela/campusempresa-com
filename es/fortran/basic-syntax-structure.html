Introducción a Fortran

Fortran, abreviatura de "Formula Translation," es uno de los lenguajes de programación de alto nivel más antiguos, utilizado principalmente para la computación numérica y científica. Comprender su sintaxis y estructura básica es crucial para escribir programas eficientes y sin errores.

Conceptos Clave

  • Estructura del Programa: Los programas Fortran están estructurados en unidades como programas, subrutinas y funciones.
  • Comentarios: Líneas que no se ejecutan, utilizadas para documentación.
  • Variables y Tipos de Datos: Fortran soporta varios tipos de datos, incluyendo enteros, números reales y caracteres.
  • Operadores: Operadores aritméticos, relacionales y lógicos.
  • Estructuras de Control: Sentencias condicionales y bucles.
  • Entrada/Salida: Lectura y escritura en archivos o en la consola.

Estructura del Programa

Un programa básico en Fortran consiste en un nombre de programa, declaraciones, sentencias ejecutables y una sentencia de fin.

program HelloWorld
  implicit none
  print *, 'Hello, World!'
end program HelloWorld
  • program HelloWorld: Declara el nombre del programa.
  • implicit none: Asegura que todas las variables deben ser declaradas explícitamente.
  • print *, 'Hello, World!': Imprime texto en la consola.
  • end program HelloWorld: Marca el final del programa.

Comentarios

Los comentarios se utilizan para explicar el código y se ignoran durante la ejecución. En Fortran, los comentarios comienzan con un signo de exclamación (!).

! Este es un comentario
print *, 'Hello, World!'  ! Esto imprime Hello, World!

Variables y Tipos de Datos

Fortran soporta varios tipos de datos. Las variables deben ser declaradas antes de usarse.

Tipos de Datos Comunes

  • Integer: Números enteros.
  • Real: Números de punto flotante.
  • Character: Cadenas de texto.
  • Logical: Valores booleanos (verdadero o falso).
program VariablesExample
  implicit none
  integer :: i
  real :: x
  character(len=20) :: name
  logical :: flag

  i = 10
  x = 3.14
  name = 'Fortran'
  flag = .true.

  print *, 'Integer:', i
  print *, 'Real:', x
  print *, 'Character:', name
  print *, 'Logical:', flag
end program VariablesExample

Operadores

Fortran incluye varios operadores para realizar operaciones aritméticas, relacionales y lógicas.

Operadores Aritméticos

  • + Adición
  • - Sustracción
  • * Multiplicación
  • / División
  • ** Exponenciación

Operadores Relacionales

  • == Igual a
  • /= No igual a
  • > Mayor que
  • < Menor que
  • >= Mayor o igual que
  • <= Menor o igual que

Operadores Lógicos

  • .and. AND lógico
  • .or. OR lógico
  • .not. NOT lógico
program OperatorsExample
  implicit none
  integer :: a, b
  logical :: result

  a = 5
  b = 3

  print *, 'Addition:', a + b
  print *, 'Multiplication:', a * b
  print *, 'Exponentiation:', a ** b

  result = (a > b) .and. (a /= 0)
  print *, 'Logical AND result:', result
end program OperatorsExample

Estructuras de Control

Las estructuras de control permiten la ejecución condicional y la iteración.

Sentencias Condicionales

  • Sentencia if
  • Sentencia else if
  • Sentencia else
program ConditionalExample
  implicit none
  integer :: num

  num = 10

  if (num > 0) then
    print *, 'Number is positive'
  else if (num == 0) then
    print *, 'Number is zero'
  else
    print *, 'Number is negative'
  end if
end program ConditionalExample

Bucles

  • Bucle do
  • Bucle do while
program LoopExample
  implicit none
  integer :: i

  do i = 1, 5
    print *, 'Iteration:', i
  end do

  i = 1
  do while (i <= 5)
    print *, 'While loop iteration:', i
    i = i + 1
  end do
end program LoopExample

Entrada/Salida

Fortran proporciona formas sencillas de manejar operaciones de entrada y salida.

Lectura de Entrada

program InputExample
  implicit none
  integer :: num

  print *, 'Enter a number:'
  read *, num
  print *, 'You entered:', num
end program InputExample

Escritura de Salida

program OutputExample
  implicit none
  integer :: num

  num = 42
  print *, 'The answer is:', num
end program OutputExample

Conclusión

Comprender la sintaxis y estructura básica de Fortran es esencial para escribir programas efectivos. Esto incluye saber cómo estructurar un programa, declarar variables, usar operadores, controlar el flujo del programa y manejar la entrada y salida. Con estos fundamentos, puedes comenzar a explorar características y capacidades más avanzadas de Fortran.
