Introducción a la Computación Científica

La computación científica implica el uso de métodos computacionales y software para resolver problemas complejos científicos y de ingeniería. Fortran, abreviatura de "Formula Translation," es un lenguaje de programación de alto nivel que ha sido ampliamente utilizado en la computación científica debido a su eficiencia en el cálculo numérico y la manipulación de matrices.

Comenzando con Fortran

Antes de sumergirse en la computación científica, es esencial entender los conceptos básicos de Fortran.

Sintaxis y Estructura Básica

  • Estructura del Programa: Un programa en Fortran típicamente consiste en un nombre de programa, declaraciones, sentencias ejecutables y una sentencia de fin.
  • Variables y Tipos de Datos: Los tipos de datos comunes incluyen INTEGER, REAL, DOUBLE PRECISION, COMPLEX, LOGICAL y CHARACTER.
  • Estructuras de Control: Incluye sentencias IF, bucles DO y sentencias SELECT CASE.

Ejemplo: Programa Simple en Fortran

PROGRAM HelloWorld
  PRINT *, '¡Hola, Mundo!'
END PROGRAM HelloWorld

Compilación y Ejecución de Programas Fortran

  • Compilación: Utiliza un compilador de Fortran como gfortran para compilar tu código. 
    gfortran HelloWorld.f90 -o HelloWorld
  • Ejecución: Ejecuta el programa compilado. 
    ./HelloWorld

Métodos Numéricos en Fortran

Los métodos numéricos son algoritmos utilizados para resolver problemas matemáticos numéricamente.

Resolución de Ecuaciones Lineales

  • Eliminación de Gauss: Un método para resolver un sistema de ecuaciones lineales.
  • Descomposición LU: Descompone una matriz en una matriz triangular inferior (L) y una matriz triangular superior (U).

Ejemplo: Resolución de Ecuaciones Lineales usando Eliminación de Gauss

PROGRAM GaussElimination
  IMPLICIT NONE
  INTEGER, PARAMETER :: n = 3
  REAL :: A(n,n), b(n), x(n)
  INTEGER :: i, j, k

  ! Inicializar matriz A y vector b
  A = RESHAPE([2.0, -1.0, 1.0, 3.0, 3.0, 9.0, 3.0, 3.0, 5.0], SHAPE(A))
  b = [8.0, 0.0, 6.0]

  ! Eliminación hacia adelante
  DO k = 1, n-1
    DO i = k+1, n
      A(i,k:n) = A(i,k:n) - A(i,k)/A(k,k) * A(k,k:n)
      b(i) = b(i) - A(i,k)/A(k,k) * b(k)
    END DO
  END DO

  ! Sustitución hacia atrás
  x(n) = b(n) / A(n,n)
  DO i = n-1, 1, -1
    x(i) = (b(i) - SUM(A(i,i+1:n) * x(i+1:n))) / A(i,i)
  END DO

  PRINT *, 'Solución: ', x
END PROGRAM GaussElimination

Integración Numérica

  • Regla del Trapecio: Aproxima la integral de una función dividiendo el área bajo la curva en trapecios.
  • Regla de Simpson: Proporciona una aproximación más precisa utilizando segmentos parabólicos.

Ejemplo: Integración Numérica usando la Regla del Trapecio

PROGRAM TrapezoidalRule
  IMPLICIT NONE
  REAL :: a, b, h, integral
  INTEGER :: n, i
  REAL, EXTERNAL :: f

  a = 0.0
  b = 1.0
  n = 100
  h = (b - a) / n
  integral = 0.5 * (f(a) + f(b))

  DO i = 1, n-1
    integral = integral + f(a + i*h)
  END DO

  integral = integral * h
  PRINT *, 'Integral: ', integral
END PROGRAM TrapezoidalRule

REAL FUNCTION f(x)
  REAL, INTENT(IN) :: x
  f = x**2
END FUNCTION f

Temas Avanzados en Fortran

A medida que avances, encontrarás temas más complejos y especializados.

Computación Paralela con Fortran

  • OpenMP: Una API ampliamente utilizada para la programación paralela en entornos de memoria compartida.
  • MPI: Interfaz de Paso de Mensajes para la programación paralela en sistemas de memoria distribuida.

Ejemplo: Computación Paralela con OpenMP

PROGRAM ParallelSum
  USE omp_lib
  IMPLICIT NONE
  INTEGER :: i, n
  REAL :: sum
  REAL, ALLOCATABLE :: a(:)

  n = 1000000
  ALLOCATE(a(n))
  a = 1.0

  sum = 0.0
  !$OMP PARALLEL DO REDUCTION(+:sum)
  DO i = 1, n
    sum = sum + a(i)
  END DO
  !$OMP END PARALLEL DO

  PRINT *, 'Suma: ', sum
END PROGRAM ParallelSum

Bibliotecas de Fortran para Computación Científica

  • LAPACK: Paquete de Álgebra Lineal para resolver problemas de álgebra lineal.
  • BLAS: Subprogramas Básicos de Álgebra Lineal para realizar operaciones básicas de vectores y matrices.
  • FFTW: Transformada Rápida de Fourier en el Oeste para calcular transformadas de Fourier.

Conclusión

Fortran sigue siendo una herramienta poderosa para la computación científica debido a su eficiencia y extensas bibliotecas. Al dominar Fortran, puedes abordar una amplia gama de problemas numéricos y científicos, desde álgebra lineal básica hasta computación paralela avanzada. Este curso ha proporcionado una base en programación Fortran, métodos numéricos y temas avanzados, equipándote con las habilidades necesarias para la computación científica.
