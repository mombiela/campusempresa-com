Introducción

El estilo de código y la documentación son aspectos cruciales de la programación que aseguran la legibilidad, mantenibilidad y colaboración del código. En esta sección, cubriremos las mejores prácticas para escribir código Fortran limpio y bien documentado, adecuado tanto para principiantes como para programadores avanzados.

Estilo de Código

Indentación y Espaciado

  • Indentación Consistente: Usa una indentación consistente en todo tu código. La práctica común es usar 2 o 4 espacios por nivel de indentación.

  • Espaciado Alrededor de Operadores: Añade espacios alrededor de los operadores para mejorar la legibilidad.

    ! Buen ejemplo
a = b + c * d

! Mal ejemplo
a=b+c*d

Longitud de Línea

  • Límite de Longitud de Línea: Mantén las líneas dentro de 72 caracteres para el formato fijo y 132 caracteres para el formato libre para asegurar la legibilidad y compatibilidad con varios editores y herramientas.

Convenciones de Nombres

  • Variables y Funciones: Usa nombres significativos para variables y funciones. Usa letras minúsculas con guiones bajos para nombres de variables y camelCase para nombres de funciones.

    ! Buen ejemplo
integer :: total_sum
real :: averageValue

! Mal ejemplo
integer :: ts
real :: avgval

Comentarios

  • Comentarios en Línea: Usa comentarios en línea para explicar lógica compleja o pasos importantes en el código.

    ! Calcular el valor promedio
averageValue = total_sum / number_of_elements

  • Comentarios en Bloque: Usa comentarios en bloque para describir el propósito de una sección de código o una función.

    !--------------------------------------------------
! Esta función calcula el factorial de un número
! Entrada: entero n
! Salida: entero factorial
!--------------------------------------------------

Documentación

Encabezados de Archivo

  • Comentarios de Encabezado de Archivo: Incluye un comentario de encabezado al principio de cada archivo para describir el propósito del archivo, autor, fecha y cualquier nota relevante.

    !--------------------------------------------------
! Archivo: factorial.f90
! Propósito: Calcular el factorial de un número
! Autor: Jane Doe
! Fecha: 2023-10-01
!--------------------------------------------------

Documentación de Funciones y Subrutinas

  • Encabezados de Función: Documenta cada función y subrutina con un comentario de encabezado que describa su propósito, entradas, salidas y cualquier efecto secundario.

    !--------------------------------------------------
! Función: factorial
! Propósito: Calcular el factorial de un número
! Entrada: entero n
! Salida: entero factorial
!--------------------------------------------------
function factorial(n)
  integer, intent(in) :: n
  integer :: factorial
  ! Lógica de la función aquí
end function factorial

Documentación en Línea

  • Documentación en Línea: Usa comentarios dentro de la función o subrutina para explicar lógica compleja o pasos importantes.

    function factorial(n)
  integer, intent(in) :: n
  integer :: factorial
  integer :: i

  factorial = 1
  do i = 1, n
    factorial = factorial * i
    ! Multiplica factorial por el valor actual de i
  end do
end function factorial

Ejemplo: Código Fortran Bien Documentado

!--------------------------------------------------
! Archivo: example.f90
! Propósito: Demostrar código Fortran bien documentado
! Autor: John Smith
! Fecha: 2023-10-01
!--------------------------------------------------

program example
  implicit none
  integer :: num, result

  ! Solicitar entrada del usuario
  print *, 'Ingrese un número:'
  read *, num

  ! Calcular factorial
  result = factorial(num)

  ! Mostrar resultado
  print *, 'El factorial de', num, 'es', result
end program example

!--------------------------------------------------
! Función: factorial
! Propósito: Calcular el factorial de un número
! Entrada: entero n
! Salida: entero factorial
!--------------------------------------------------
function factorial(n)
  integer, intent(in) :: n
  integer :: factorial
  integer :: i

  factorial = 1
  do i = 1, n
    factorial = factorial * i
    ! Multiplica factorial por el valor actual de i
  end do
end function factorial

Conclusión

Adherirse a buenas prácticas de estilo de código y documentación en Fortran es esencial para crear código legible, mantenible y colaborativo. Siguiendo una indentación consistente, convenciones de nombres y una documentación exhaustiva, puedes asegurar que tu código sea fácil de entender y mantener, tanto para ti como para otros que puedan trabajar con tu código en el futuro.
