Introducción

NGINX es un servidor HTTP de alto rendimiento, proxy inverso y un servidor proxy IMAP/POP3. Esta sección te guiará a través del proceso de instalación de NGINX en varios sistemas operativos. Al final de esta sección, tendrás un servidor NGINX en funcionamiento en tu máquina.

Requisitos previos

Antes de instalar NGINX, asegúrate de tener lo siguiente:

  • Una máquina con un sistema operativo basado en Unix (Linux, macOS) o Windows.
  • Conocimientos básicos de la línea de comandos.
  • Acceso administrativo (root) a tu máquina.

Instalación de NGINX en Linux

Usando el gestor de paquetes (Ubuntu/Debian)

  1. Actualiza la lista de paquetes: 
    sudo apt update
  2. Instala NGINX: 
    sudo apt install nginx
  3. Inicia NGINX: 
    sudo systemctl start nginx
  4. Habilita NGINX para que se inicie al arrancar: 
    sudo systemctl enable nginx
  5. Verifica la instalación: Abre tu navegador web y navega a http://localhost. Deberías ver la página de bienvenida de NGINX.

Usando el gestor de paquetes (CentOS/RHEL)

  1. Instala el repositorio EPEL: 
    sudo yum install epel-release
  2. Instala NGINX: 
    sudo yum install nginx
  3. Inicia NGINX: 
    sudo systemctl start nginx
  4. Habilita NGINX para que se inicie al arrancar: 
    sudo systemctl enable nginx
  5. Verifica la instalación: Abre tu navegador web y navega a http://localhost. Deberías ver la página de bienvenida de NGINX.

Instalación de NGINX en macOS

Usando Homebrew

  1. Instala Homebrew (si no está ya instalado): 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instala NGINX: 
    brew install nginx
  3. Inicia NGINX: 
    sudo nginx
  4. Verifica la instalación: Abre tu navegador web y navega a http://localhost:8080. Deberías ver la página de bienvenida de NGINX.

Instalación de NGINX en Windows

Usando el binario de NGINX

  1. Descarga el binario de NGINX: Ve a la página de descargas de NGINX y descarga la versión estable para Windows.
  2. Extrae el archivo descargado: Extrae el contenido del archivo zip en un directorio de tu elección (por ejemplo, C:\nginx).
  3. Ejecuta NGINX: Abre el Símbolo del sistema y navega al directorio de NGINX: 
    cd C:\nginx
    Inicia NGINX: 
    start nginx
  4. Verifica la instalación: Abre tu navegador web y navega a http://localhost. Deberías ver la página de bienvenida de NGINX.

Pasos comunes después de la instalación

Comprobando el estado de NGINX

  • Linux: 
    sudo systemctl status nginx
  • macOS: 
    sudo nginx -t
  • Windows: Abre el Símbolo del sistema y navega al directorio de NGINX, luego ejecuta: 
    nginx -t

Deteniendo y reiniciando NGINX

  • Linux: 
    sudo systemctl stop nginx
sudo systemctl restart nginx
  • macOS: 
    sudo nginx -s stop
sudo nginx -s reload
  • Windows: Abre el Símbolo del sistema y navega al directorio de NGINX, luego ejecuta: 
    nginx -s stop
nginx -s reload

Conclusión

En esta sección, has aprendido cómo instalar NGINX en varios sistemas operativos, incluyendo Linux, macOS y Windows. También has aprendido cómo iniciar, detener y verificar el estado de tu servidor NGINX. Con NGINX instalado, ahora estás listo para configurarlo y usarlo para tus necesidades de servidor web.
