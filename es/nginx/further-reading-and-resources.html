Configuraciones de Ejemplo

  • Configuración Básica de Servidor HTTP:

    server {
    listen 80;
    server_name example.com;

    location / {
        root /usr/share/nginx/html;
        index index.html index.htm;
    }
}
    • Esta configuración establece un servidor HTTP básico que escucha en el puerto 80, sirviendo archivos desde /usr/share/nginx/html.

  • Configuración de Proxy Inverso:

    server {
    listen 80;
    server_name example.com;

    location / {
        proxy_pass http://localhost:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
    • Esta configuración establece NGINX como un proxy inverso, reenviando solicitudes a un servidor backend que se ejecuta en localhost:3000.
