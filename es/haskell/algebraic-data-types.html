Introducción a los Tipos de Datos Algebraicos (ADTs)

Los Tipos de Datos Algebraicos (ADTs) son un concepto fundamental en Haskell y en la programación funcional. Permiten definir tipos de datos personalizados que pueden representar una combinación de diferentes valores. Los ADTs se componen de dos tipos principales: tipos producto y tipos suma.

Conceptos Clave

  • Tipos Producto: Combinan múltiples valores en una sola estructura de datos (por ejemplo, tuplas, registros).
  • Tipos Suma: Representan un valor que puede ser uno de varios tipos diferentes (por ejemplo, Either, Maybe).

Tipos Producto

Los tipos producto se utilizan para agrupar múltiples valores juntos. En Haskell, los tipos producto más comunes son las tuplas y los registros.

Tuplas

Las tuplas son tipos producto simples que agrupan un número fijo de valores juntos.

-- Una tupla que contiene un Int y un String
exampleTuple :: (Int, String)
exampleTuple = (42, "Hola")

Registros

Los registros son tipos producto más descriptivos que utilizan campos con nombre.

-- Definiendo un tipo de registro para una Persona
data Person = Person {
    name :: String,
    age :: Int
}

-- Creando una instancia de Persona
examplePerson :: Person
examplePerson = Person { name = "Alicia", age = 30 }

Tipos Suma

Los tipos suma permiten que un valor sea uno de varios tipos diferentes. Se definen usando la palabra clave data con múltiples constructores.

Either

El tipo Either representa un valor que puede ser uno de dos tipos.

-- Either puede ser Left o Right
exampleEither :: Either String Int
exampleEither = Right 42

Maybe

El tipo Maybe representa un valor opcional que puede ser Just un valor o Nothing.

-- Maybe puede ser Just un valor o Nothing
exampleMaybe :: Maybe Int
exampleMaybe = Just 42

Definiendo ADTs Personalizados

Puedes definir tus propios ADTs usando la palabra clave data.

Ejemplo: Shape

Vamos a definir un ADT personalizado para formas geométricas.

-- Definiendo un tipo Shape con constructores Circle y Rectangle
data Shape = Circle Float
           | Rectangle Float Float

-- Función para calcular el área de una Shape
area :: Shape -> Float
area (Circle radius) = pi * radius * radius
area (Rectangle width height) = width * height

-- Ejemplos de uso del tipo Shape
exampleCircle :: Shape
exampleCircle = Circle 5.0

exampleRectangle :: Shape
exampleRectangle = Rectangle 4.0 6.0

Coincidencia de Patrones

La coincidencia de patrones es una característica poderosa en Haskell que permite descomponer ADTs y trabajar con sus componentes.

Ejemplo: Coincidencia de Patrones con Shape

Usando el tipo Shape definido anteriormente, podemos hacer coincidencia de patrones para manejar diferentes formas.

-- Función para describir una Shape
describeShape :: Shape -> String
describeShape (Circle radius) = "Un círculo con radio " ++ show radius
describeShape (Rectangle width height) = "Un rectángulo con ancho " ++ show width ++ " y alto " ++ show height

-- Ejemplos de uso de describeShape
exampleDescription1 :: String
exampleDescription1 = describeShape exampleCircle

exampleDescription2 :: String
exampleDescription2 = describeShape exampleRectangle

Temas Avanzados

A medida que te sientas más cómodo con los ADTs, puedes explorar temas más avanzados como:

  • Tipos de Datos Recursivos: ADTs que se referencian a sí mismos.
  • Tipos Parametrizados: ADTs que toman parámetros de tipo.
  • Clases de Tipo y ADTs: Usar clases de tipo con ADTs para polimorfismo.

Tipos de Datos Recursivos

Los tipos de datos recursivos se utilizan para definir estructuras como listas y árboles.

-- Definiendo un tipo de Lista recursivo
data List a = Empty
            | Cons a (List a)

-- Función para calcular la longitud de una Lista
listLength :: List a -> Int
listLength Empty = 0
listLength (Cons _ tail) = 1 + listLength(tail)

-- Ejemplo de uso del tipo List
exampleList :: List Int
exampleList = Cons 1 (Cons 2 (Cons 3 Empty))

Tipos Parametrizados

Los tipos parametrizados permiten que los ADTs sean más flexibles y reutilizables.

-- Definiendo un tipo Pair parametrizado
data Pair a b = Pair a b

-- Ejemplo de uso del tipo Pair
examplePair :: Pair Int String
examplePair = Pair 42 "Respuesta"

Conclusión

Los Tipos de Datos Algebraicos son una característica poderosa en Haskell que permite definir estructuras de datos complejas de una manera clara y concisa. Al comprender los tipos producto, los tipos suma y la coincidencia de patrones, puedes crear programas robustos y expresivos. A medida que avances, explorar los tipos recursivos y parametrizados mejorará aún más tu capacidad para modelar y resolver problemas de manera efectiva en Haskell.
