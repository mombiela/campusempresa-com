Data.Map y Data.Set en Haskell
Introducción a los Funtores ►

Introducción

En esta sección, exploraremos dos módulos esenciales en la biblioteca estándar de Haskell: Data.Map y Data.Set. Estos módulos proporcionan implementaciones eficientes de mapas (diccionarios) y conjuntos, respectivamente. Entender cómo usar estas estructuras de datos es crucial para escribir código Haskell eficiente e idiomático.

Data.Map

Conceptos Clave

  • Mapa: Una estructura de datos asociativa que mapea claves a valores.
  • Eficiencia: Data.Map se implementa utilizando árboles binarios balanceados, proporcionando una complejidad de tiempo logarítmica para la mayoría de las operaciones.
  • Inmutabilidad: Como todas las estructuras de datos de Haskell, los mapas son inmutables.

Operaciones Básicas

Aquí hay algunas operaciones básicas que puedes realizar con Data.Map:

  • Crear un Mapa: Usa fromList para crear un mapa a partir de una lista de pares clave-valor.
  • Insertar Elementos: Usa insert para agregar un par clave-valor a un mapa.
  • Buscar Valores: Usa lookup para recuperar un valor por su clave.
  • Eliminar Elementos: Usa delete para eliminar un par clave-valor de un mapa.

Ejemplos de Código

Creando y Manipulando un Mapa

import qualified Data.Map as Map

-- Crear un mapa a partir de una lista de pares clave-valor
let phoneBook = Map.fromList [("Alice", "1234"), ("Bob", "5678"), ("Charlie", "91011")]

-- Insertar una nueva entrada
let updatedPhoneBook = Map.insert "David" "1213" phoneBook

-- Buscar un valor
case Map.lookup "Alice" updatedPhoneBook of
    Just number -> putStrLn $ "El número de Alice es " ++ number
    Nothing -> putStrLn "Alice no está en la guía telefónica"

-- Eliminar una entrada
let finalPhoneBook = Map.delete "Bob" updatedPhoneBook

Operaciones Avanzadas

  • Unión e Intersección: Combina mapas usando union e intersection.
  • Map y Filter: Aplica funciones a todos los valores usando map y filtra entradas usando filterWithKey.

Ejemplo de Código Avanzado

import qualified Data.Map as Map

let map1 = Map.fromList [(1, "one"), (2, "two")]
let map2 = Map.fromList [(2, "dos"), (3, "three")]

-- Unión de dos mapas
let unionMap = Map.union map1 map2

-- Intersección de dos mapas
let intersectionMap = Map.intersection map1 map2

-- Mapear una función sobre todos los valores
let upperCaseMap = Map.map (map toUpper) map1

-- Filtrar entradas
let filteredMap = Map.filterWithKey (\k _ -> k > 1) map1

Data.Set

Conceptos Clave

  • Conjunto: Una colección de elementos únicos.
  • Eficiencia: Data.Set también se implementa utilizando árboles binarios balanceados, proporcionando una complejidad de tiempo logarítmica para la mayoría de las operaciones.
  • Inmutabilidad: Los conjuntos en Haskell son inmutables.

Operaciones Básicas

Aquí hay algunas operaciones básicas que puedes realizar con Data.Set:

  • Crear un Conjunto: Usa fromList para crear un conjunto a partir de una lista de elementos.
  • Insertar Elementos: Usa insert para agregar un elemento a un conjunto.
  • Comprobar Pertenencia: Usa member para verificar si un elemento está en un conjunto.
  • Eliminar Elementos: Usa delete para eliminar un elemento de un conjunto.

Ejemplos de Código

Creando y Manipulando un Conjunto

import qualified Data.Set as Set

-- Crear un conjunto a partir de una lista de elementos
let mySet = Set.fromList [1, 2, 3, 4]

-- Insertar un nuevo elemento
let updatedSet = Set.insert 5 mySet

-- Comprobar pertenencia
if Set.member 3 updatedSet
    then putStrLn "3 está en el conjunto"
    else putStrLn "3 no está en el conjunto"

-- Eliminar un elemento
let finalSet = Set.delete 2 updatedSet

Operaciones Avanzadas

  • Unión e Intersección: Combina conjuntos usando union e intersection.
  • Diferencia: Encuentra la diferencia entre dos conjuntos usando difference.
  • Map y Filter: Aplica funciones a todos los elementos usando map y filtra elementos usando filter.

Ejemplo de Código Avanzado

import qualified Data.Set as Set

let set1 = Set.fromList [1, 2, 3]
let set2 = Set.fromList [2, 3, 4]

-- Unión de dos conjuntos
let unionSet = Set.union set1 set2

-- Intersección de dos conjuntos
let intersectionSet = Set.intersection set1 set2

-- Diferencia entre dos conjuntos
let differenceSet = Set.difference set1 set2

-- Mapear una función sobre todos los elementos
let squaredSet = Set.map (^2) set1

-- Filtrar elementos
let filteredSet = Set.filter (>2) set1

Conclusión

En esta sección, hemos cubierto los conceptos básicos y algunas características avanzadas de Data.Map y Data.Set en Haskell. Estas estructuras de datos son herramientas poderosas para gestionar colecciones de datos de manera eficiente. Al entender cómo usar mapas y conjuntos, puedes escribir código Haskell más eficiente y expresivo. Practica usando estos módulos con diferentes tipos de datos para familiarizarte más con sus operaciones y capacidades.
