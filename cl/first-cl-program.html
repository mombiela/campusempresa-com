Introducción

Control Language (CL) es un lenguaje de scripting utilizado en los sistemas AS/400 de IBM (ahora conocidos como IBM i). Se utiliza principalmente para scripting y automatización a nivel de sistema. En esta sección, escribiremos nuestro primer programa CL, entenderemos su estructura y lo ejecutaremos.

Conceptos Clave

  • Programa CL: Una secuencia de comandos CL que realizan una tarea específica.
  • Archivo Físico de Fuente: Un archivo donde se almacena el código fuente CL.
  • Compilación: El proceso de convertir el código fuente CL en un objeto ejecutable.
  • Ejecución: Ejecutar el programa CL compilado.

Configuración del Entorno

Antes de escribir tu primer programa CL, asegúrate de tener acceso a un sistema IBM i y los permisos necesarios para crear y ejecutar programas CL.

Escribiendo Tu Primer Programa CL

  1. Crear un Archivo Físico de Fuente:

    • Un archivo físico de fuente es donde almacenarás tu código fuente CL.
    • Usa el siguiente comando para crear un archivo físico de fuente:
    CRTSRCPF FILE(QGPL/QCLSRC) RCDLEN(112)
    • Este comando crea un archivo físico de fuente llamado QCLSRC en la biblioteca QGPL con una longitud de registro de 112.

  2. Agregar un Miembro al Archivo Físico de Fuente:

    • Un miembro es un lugar dentro del archivo físico de fuente donde escribes tu programa CL.
    • Usa el siguiente comando para agregar un miembro:
    ADDPFM FILE(QGPL/QCLSRC) MBR(MYCLPGM) SRCTYPE(CL)
    • Este comando agrega un miembro llamado MYCLPGM al archivo QCLSRC con un tipo de fuente de CL.

  3. Editar el Miembro de Fuente:

    • Usa la Utilidad de Entrada de Fuente (SEU) o cualquier otro editor disponible en tu sistema para editar el miembro de fuente.
    • Usa el siguiente comando para comenzar a editar:
    STRSEU SRCFILE(QGPL/QCLSRC) SRCMBR(MYCLPGM)
    • Este comando abre el miembro MYCLPGM en el archivo QCLSRC para editar.

  4. Escribir el Programa CL:

    • Ingresa el siguiente código en el editor:
    PGM
DCL VAR(&MSG) TYPE(*CHAR) LEN(50)
CHGVAR VAR(&MSG) VALUE('¡Hola, Mundo!')
SNDPGMMSG MSG(&MSG)
ENDPGM
    • Explicación del código:
      • PGM: Indica el inicio del programa CL.
      • DCL VAR(&MSG) TYPE(*CHAR) LEN(50): Declara una variable &MSG de tipo carácter con una longitud de 50.
      • CHGVAR VAR(&MSG) VALUE('¡Hola, Mundo!'): Cambia el valor de &MSG a '¡Hola, Mundo!'.
      • SNDPGMMSG MSG(&MSG): Envía un mensaje de programa que contiene el valor de &MSG.
      • ENDPGM: Indica el final del programa CL.

Compilando el Programa CL

  • Después de escribir el programa CL, necesitas compilarlo para crear un objeto ejecutable.
  • Usa el siguiente comando para compilar el programa:
CRTCLPGM PGM(QGPL/MYCLPGM) SRCFILE(QGPL/QCLSRC) SRCMBR(MYCLPGM)
  • Este comando crea un objeto programa llamado MYCLPGM en la biblioteca QGPL usando el miembro de fuente MYCLPGM del archivo QCLSRC.

Ejecutando el Programa CL

  • Para ejecutar el programa CL compilado, usa el siguiente comando:
CALL PGM(QGPL/MYCLPGM)
  • Este comando ejecuta el programa MYCLPGM desde la biblioteca QGPL, y deberías ver el mensaje '¡Hola, Mundo!' mostrado.

Conclusión

En esta sección, has aprendido cómo crear, escribir, compilar y ejecutar un programa CL simple en un sistema IBM i. Este conocimiento fundamental es crucial a medida que avanzas hacia temas más avanzados en la programación CL. Practica escribiendo y ejecutando programas CL básicos para familiarizarte con la sintaxis y los comandos.
