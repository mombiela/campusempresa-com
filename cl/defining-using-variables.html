Definiendo y Usando Variables en CL (Lenguaje de Control)

Introducción

En esta sección, cubriremos los conceptos básicos de la definición y uso de variables en el Lenguaje de Control (CL). Las variables son esenciales para almacenar datos que pueden ser manipulados y utilizados a lo largo de tus programas CL. Comenzaremos con conceptos básicos y gradualmente avanzaremos a temas más avanzados.

¿Qué es una Variable?

  • Una variable es una ubicación de almacenamiento identificada por un nombre (o identificador) que contiene datos que pueden ser modificados durante la ejecución del programa.
  • Las variables en CL pueden almacenar diferentes tipos de datos, como valores numéricos, cadenas de texto y más.

Definiendo Variables

Para definir una variable en CL, se utiliza el comando DCL (Declarar). Aquí está la sintaxis básica:

DCL VAR(&NombreVariable) TYPE(*TipoDeDato) LEN(longitud)

Ejemplo:

DCL VAR(&NOMBRE) TYPE(*CHAR) LEN(20)
DCL VAR(&EDAD) TYPE(*DEC) LEN(3 0)
  • &NOMBRE es una variable de tipo carácter con una longitud de 20.
  • &EDAD es una variable decimal con una longitud de 3 dígitos y 0 decimales.

Conceptos Clave:

  • VAR: Especifica el nombre de la variable.
  • TYPE: Especifica el tipo de dato que la variable contendrá. Los tipos comunes incluyen:
    • *CHAR: Cadena de caracteres.
    • *DEC: Número decimal.
    • *INT: Número entero.
  • LEN: Especifica la longitud de la variable. Para tipos decimales, incluye el número total de dígitos y el número de decimales.

Asignando Valores a Variables

Una vez que una variable está definida, puedes asignarle valores usando el comando CHGVAR (Cambiar Variable).

Ejemplo:

CHGVAR VAR(&NOMBRE) VALUE('Juan Pérez')
CHGVAR VAR(&EDAD) VALUE(30)
  • &NOMBRE se le asigna el valor 'Juan Pérez'.
  • &EDAD se le asigna el valor 30.

Conceptos Clave:

  • CHGVAR: Comando usado para cambiar el valor de una variable.
  • VALUE: Especifica el nuevo valor a ser asignado a la variable.

Usando Variables en Expresiones

Las variables pueden ser usadas en expresiones para realizar cálculos o concatenar cadenas.

Ejemplo:

DCL VAR(&SALARIO) TYPE(*DEC) LEN(7 2)
DCL VAR(&BONO) TYPE(*DEC) LEN(7 2)
DCL VAR(&TOTAL) TYPE(*DEC) LEN(7 2)

CHGVAR VAR(&SALARIO) VALUE(50000.00)
CHGVAR VAR(&BONO) VALUE(5000.00)
CHGVAR VAR(&TOTAL) VALUE(&SALARIO + &BONO)
  • &TOTAL se calcula sumando &SALARIO y &BONO.

Sentencias Condicionales con Variables

Las variables son frecuentemente usadas en sentencias condicionales para controlar el flujo del programa.

Ejemplo:

DCL VAR(&EDAD) TYPE(*DEC) LEN(3 0)

CHGVAR VAR(&EDAD) VALUE(20)

IF COND(&EDAD *LT 18) THEN(DO)
    /* Código para menores de 18 */
    SNDPGMMSG MSG('Menor de 18')
ENDDO
ELSE DO
    /* Código para 18 y mayores */
    SNDPGMMSG MSG('18 o mayor')
ENDDO
  • La sentencia IF verifica si &EDAD es menor de 18 y ejecuta el bloque de código correspondiente.

Uso Avanzado de Variables

Arreglos

Los arreglos se usan para almacenar múltiples valores en una sola variable.

Ejemplo:

DCL VAR(&NOMBRES) TYPE(*CHAR) LEN(20) DIM(5)

CHGVAR VAR(&NOMBRES(1)) VALUE('Alicia')
CHGVAR VAR(&NOMBRES(2)) VALUE('Roberto')
CHGVAR VAR(&NOMBRES(3)) VALUE('Carlos')
CHGVAR VAR(&NOMBRES(4)) VALUE('Diana')
CHGVAR VAR(&NOMBRES(5)) VALUE('Eva')
  • &NOMBRES es un arreglo de 5 cadenas de caracteres, cada una con una longitud de 20.

Estructuras de Datos

Las estructuras de datos permiten agrupar variables relacionadas.

Ejemplo:

DCL VAR(&EMPLEADO) TYPE(*CHAR) LEN(50) 
DCL VAR(&IDEMP) TYPE(*CHAR) LEN(10) STG(*DEFINED) DEFVAR(&EMPLEADO 1)
DCL VAR(&NOMBREEMP) TYPE(*CHAR) LEN(40) STG(*DEFINED) DEFVAR(&EMPLEADO 11)

CHGVAR VAR(&IDEMP) VALUE('E12345')
CHGVAR VAR(&NOMBREEMP) VALUE('Juan Pérez')
  • &EMPLEADO es una estructura de datos que contiene &IDEMP y &NOMBREEMP.

Conclusión

En esta sección, cubrimos los conceptos básicos de la definición y uso de variables en CL, incluyendo:

  • Declarar variables con el comando DCL.
  • Asignar valores usando el comando CHGVAR.
  • Usar variables en expresiones y sentencias condicionales.
  • Uso avanzado con arreglos y estructuras de datos.

Entender estos conceptos es crucial para escribir programas CL efectivos. Practica creando y manipulando variables en tus propios scripts CL para reforzar estos conceptos.
