Introducción a los Constructos de Bucle

Los constructos de bucle son esenciales en la programación ya que permiten ejecutar un bloque de código múltiples veces. En CL, los constructos de bucle ayudan a automatizar tareas repetitivas, haciendo que tus scripts sean más eficientes y fáciles de gestionar.

Tipos de Constructos de Bucle en CL

En CL, hay principalmente dos tipos de constructos de bucle:

  • Bucle DO
  • Bucle DOWHILE

Bucle DO

El bucle DO se utiliza para ejecutar un bloque de código un número específico de veces. Es útil cuando sabes de antemano cuántas veces quieres ejecutar el bucle.

Sintaxis

DO var FROM start TO end BY step
    // Código a ejecutar
ENDDO

Ejemplo

DCL VAR(&I) TYPE(*INT) LEN(4)
DO VAR(&I) FROM(1) TO(5) BY(1)
    SNDPGMMSG MSG('Iteración ' *CAT &I)
ENDDO
  • Explicación:
    • DCL VAR(&I) TYPE(*INT) LEN(4): Declara una variable &I de tipo entero.
    • DO VAR(&I) FROM(1) TO(5) BY(1): Inicia un bucle con &I que va de 1 a 5, incrementando en 1.
    • SNDPGMMSG MSG('Iteración ' *CAT &I): Envía un mensaje de programa concatenando la cadena 'Iteración ' con el valor actual de &I.
    • ENDDO: Termina el bucle.

Bucle DOWHILE

El bucle DOWHILE ejecuta un bloque de código mientras una condición especificada sea verdadera. Es útil cuando el número de iteraciones no se conoce de antemano y depende de una condición.

Sintaxis

DOWHILE (condition)
    // Código a ejecutar
ENDDO

Ejemplo

DCL VAR(&I) TYPE(*INT) LEN(4) VALUE(1)
DOWHILE (&I <= 5)
    SNDPGMMSG MSG('Iteración ' *CAT &I)
    CHGVAR VAR(&I) VALUE(&I + 1)
ENDDO
  • Explicación:
    • DCL VAR(&I) TYPE(*INT) LEN(4) VALUE(1): Declara una variable &I de tipo entero y la inicializa a 1.
    • DOWHILE (&I <= 5): Inicia un bucle que continúa mientras &I sea menor o igual a 5.
    • SNDPGMMSG MSG('Iteración ' *CAT &I): Envía un mensaje de programa concatenando la cadena 'Iteración ' con el valor actual de &I.
    • CHGVAR VAR(&I) VALUE(&I + 1): Incrementa el valor de &I en 1.
    • ENDDO: Termina el bucle.

Técnicas Avanzadas de Bucle

Bucles Anidados

Puedes anidar bucles dentro de otros bucles para manejar escenarios más complejos.

Ejemplo

DCL VAR(&I) TYPE(*INT) LEN(4)
DCL VAR(&J) TYPE(*INT) LEN(4)
DO VAR(&I) FROM(1) TO(3) BY(1)
    DO VAR(&J) FROM(1) TO(2) BY(1)
        SNDPGMMSG MSG('Bucle exterior ' *CAT &I *CAT ' Bucle interior ' *CAT &J)
    ENDDO
ENDDO
  • Explicación:
    • El bucle exterior va de 1 a 3.
    • El bucle interior va de 1 a 2 para cada iteración del bucle exterior.
    • Se envía un mensaje para cada combinación de &I y &J.

Declaraciones de Control de Bucle

  • LEAVE: Sale del bucle inmediatamente.
  • ITERATE: Omite el código restante en la iteración actual y procede a la siguiente iteración.

Ejemplo con LEAVE e ITERATE

DCL VAR(&I) TYPE(*INT) LEN(4)
DO VAR(&I) FROM(1) TO(10) BY(1)
    IF COND(&I = 5) THEN(LEAVE)
    IF COND(&I MOD 2 = 0) THEN(ITERATE)
    SNDPGMMSG MSG('Iteración ' *CAT &I)
ENDDO
  • Explicación:
    • El bucle va de 1 a 10.
    • Si &I es igual a 5, la declaración LEAVE sale del bucle.
    • Si &I es par, la declaración ITERATE omite el código restante y procede a la siguiente iteración.
    • Se envía un mensaje solo para los valores impares de &I que no sean 5.

Conclusión

Los constructos de bucle en CL son herramientas poderosas que ayudan a automatizar tareas repetitivas y gestionar lógica compleja. Al entender y utilizar los bucles DO y DOWHILE, junto con técnicas avanzadas como bucles anidados y declaraciones de control, puedes escribir scripts CL más eficientes y efectivos. Practica estos constructos con varios ejemplos para obtener una comprensión más profunda y mejorar tus habilidades de programación en CL.
