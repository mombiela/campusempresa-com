Introducción a las Variables en CL

Las variables son fundamentales en cualquier lenguaje de programación, incluido el Lenguaje de Control (CL). Se utilizan para almacenar datos que pueden ser manipulados y recuperados a lo largo de la ejecución de un programa.

Conceptos Clave

  • Definición: Una variable es una ubicación de almacenamiento nombrada en la memoria que contiene un valor.
  • Tipos: CL soporta varios tipos de datos para variables, como carácter, decimal e entero.
  • Ámbito: Las variables pueden tener diferentes ámbitos, como local o global, determinando dónde pueden ser accedidas dentro del programa.

Declarando Variables

En CL, las variables se declaran usando el comando DCL (Declarar). La sintaxis para declarar una variable incluye especificar su nombre, tipo y longitud.

Sintaxis

DCL VAR(&NombreVariable) TYPE(*TipoDato) LEN(longitud)

Ejemplo

DCL VAR(&Nombre) TYPE(*CHAR) LEN(20)
DCL VAR(&Edad) TYPE(*DEC) LEN(3 0)
  • &Nombre es una variable de carácter con una longitud de 20.
  • &Edad es una variable decimal con una longitud de 3 dígitos y 0 decimales.

Asignando Valores a Variables

Los valores pueden ser asignados a variables usando el comando CHGVAR (Cambiar Variable).

Sintaxis

CHGVAR VAR(&NombreVariable) VALUE(expresión)

Ejemplo

CHGVAR VAR(&Nombre) VALUE('John Doe')
CHGVAR VAR(&Edad) VALUE(30)
  • &Nombre se le asigna el valor 'John Doe'.
  • &Edad se le asigna el valor 30.

Usando Variables en Expresiones

Las variables pueden ser usadas en expresiones para realizar cálculos o manipular datos.

Ejemplo

DCL VAR(&Total) TYPE(*DEC) LEN(5 2)
DCL VAR(&Precio) TYPE(*DEC) LEN(5 2)
DCL VAR(&Cantidad) TYPE(*DEC) LEN(3 0)

CHGVAR VAR(&Precio) VALUE(19.99)
CHGVAR VAR(&Cantidad) VALUE(3)
CHGVAR VAR(&Total) VALUE(&Precio * &Cantidad)
  • &Total se calcula multiplicando &Precio y &Cantidad.

Ámbito de las Variables

Las variables en CL pueden ser locales o globales. Las variables locales se declaran dentro de un procedimiento y solo son accesibles dentro de ese procedimiento. Las variables globales se declaran fuera de cualquier procedimiento y son accesibles a lo largo del programa.

Ejemplo de Variable Local

PGM
    DCL VAR(&VarLocal) TYPE(*CHAR) LEN(10)
    CHGVAR VAR(&VarLocal) VALUE('Local')
    /* &VarLocal solo es accesible dentro de este bloque PGM */
ENDPGM

Ejemplo de Variable Global

DCL VAR(&VarGlobal) TYPE(*CHAR) LEN(10)

PGM
    CHGVAR VAR(&VarGlobal) VALUE('Global')
    /* &VarGlobal es accesible a lo largo del programa */
ENDPGM

Resumen

Entender las variables en CL es crucial para una programación efectiva. Las variables almacenan datos que pueden ser manipulados y recuperados, y vienen en varios tipos y ámbitos. Declarar, asignar valores y usar variables en expresiones son habilidades fundamentales. Al dominar estos conceptos, puedes escribir programas CL más eficientes y legibles.

Puntos Clave

  • Las variables son ubicaciones de almacenamiento nombradas en la memoria.
  • Usa el comando DCL para declarar variables.
  • Usa el comando CHGVAR para asignar valores a variables.
  • Las variables pueden ser usadas en expresiones para cálculos.
  • Entiende el ámbito de las variables para gestionar su accesibilidad dentro de tus programas.

Al practicar estos conceptos, obtendrás una base sólida en el manejo de variables en CL, preparando el terreno para técnicas de programación más avanzadas.
