Creación de Gráficos de Torta

En esta sección, exploraremos cómo crear gráficos de torta utilizando D3.js, una poderosa biblioteca de JavaScript para producir visualizaciones de datos dinámicas e interactivas en navegadores web. Comenzaremos con lo básico y gradualmente avanzaremos hacia temas más avanzados.

Introducción a los Gráficos de Torta

Los gráficos de torta son gráficos circulares divididos en sectores, cada uno representando una proporción del total. Son útiles para mostrar datos que son parte de un todo, como porcentajes.

Conceptos Clave:

  • Generador de Arcos: Una función en D3.js que crea los datos de ruta SVG para los arcos.
  • Generador de Torta: Una función en D3.js que calcula los ángulos de inicio y fin para cada porción del gráfico de torta.
  • Elementos SVG: Elementos de Gráficos Vectoriales Escalables utilizados para dibujar el gráfico de torta.

Configuración de tu Entorno

Antes de comenzar a crear gráficos de torta, asegúrate de tener lo siguiente:

  • Un entendimiento básico de HTML, CSS y JavaScript.
  • La biblioteca D3.js incluida en tu proyecto.

Incluir D3.js

Puedes incluir D3.js en tu archivo HTML añadiendo la siguiente etiqueta de script:

<script src="https://d3js.org/d3.v7.min.js"></script>

Creación de un Gráfico de Torta Básico

Paso 1: Preparar los Datos

Primero, necesitamos preparar los datos que visualizaremos. Vamos a crear un conjunto de datos simple:

const data = [10, 20, 30, 40, 50];

Paso 2: Configurar el Contenedor SVG

A continuación, configuramos un contenedor SVG donde se dibujará el gráfico de torta:

const width = 400;
const height = 400;
const radius = Math.min(width, height) / 2;

const svg = d3.select("body")
  .append("svg")
  .attr("width", width)
  .attr("height", height)
  .append("g")
  .attr("transform", `translate(${width / 2}, ${height / 2})`);

Paso 3: Crear los Generadores de Torta y Arco

Necesitamos crear los generadores de torta y arco:

const pie = d3.pie();

const arc = d3.arc()
  .innerRadius(0)
  .outerRadius(radius);

Paso 4: Dibujar el Gráfico de Torta

Ahora, utilizamos los generadores de torta y arco para dibujar el gráfico de torta:

const arcs = svg.selectAll("arc")
  .data(pie(data))
  .enter()
  .append("g")
  .attr("class", "arc");

arcs.append("path")
  .attr("d", arc)
  .attr("fill", (d, i) => d3.schemeCategory10[i]);

Añadir Etiquetas al Gráfico de Torta

Para hacer el gráfico de torta más informativo, podemos añadir etiquetas a cada porción:

arcs.append("text")
  .attr("transform", d => `translate(${arc.centroid(d)})`)
  .attr("text-anchor", "middle")
  .text(d => d.data);

Personalizaciones Avanzadas

Añadir Interactividad

Podemos añadir interactividad al gráfico de torta, como resaltar una porción al pasar el ratón por encima:

arcs.on("mouseover", function(event, d) {
  d3.select(this).select("path").transition()
    .duration(200)
    .attr("d", d3.arc().innerRadius(0).outerRadius(radius + 10));
})
.on("mouseout", function(event, d) {
  d3.select(this).select("path").transition()
    .duration(200)
    .attr("d", arc);
});

Usar un Gráfico de Rosca

Un gráfico de rosca es una variación de un gráfico de torta con un agujero en el centro. Podemos crear un gráfico de rosca configurando el innerRadius del generador de arco:

const arc = d3.arc()
  .innerRadius(radius / 2)
  .outerRadius(radius);

Conclusión

En esta sección, cubrimos los conceptos básicos de la creación de gráficos de torta utilizando D3.js, desde la configuración del entorno hasta el dibujo del gráfico y la adición de etiquetas. También exploramos personalizaciones avanzadas como la interactividad y la creación de gráficos de rosca. Los gráficos de torta son una forma poderosa de visualizar relaciones de parte a todo en tus datos, y con D3.js, puedes crear gráficos altamente personalizables e interactivos.
