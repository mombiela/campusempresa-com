Introducción

Vincular datos a elementos es un concepto fundamental en D3.js, que te permite crear visualizaciones dinámicas e interactivas. Esta sección te guiará a través de los conceptos básicos de la vinculación de datos, desde ejemplos simples hasta técnicas más avanzadas.

Conceptos Clave

  • Vinculación de Datos: El proceso de conectar datos a elementos del DOM.
  • Entrar, Actualizar, Salir: Las tres fases principales de la vinculación de datos en D3.js.
  • Uniones de Datos: El mecanismo por el cual D3.js vincula datos a elementos.

Vinculación Básica de Datos

Selección de Elementos

Para vincular datos a elementos, primero necesitas seleccionar los elementos usando los selectores de D3.js.

// Seleccionar todos los elementos de párrafo
d3.selectAll("p");

Vinculación de Datos

Puedes vincular datos a los elementos seleccionados usando el método .data().

// Array de datos de ejemplo
const data = [10, 20, 30, 40, 50];

// Vincular datos a elementos de párrafo
d3.selectAll("p")
  .data(data)
  .text(d => d);

En este ejemplo:

  • d3.selectAll("p") selecciona todos los elementos de párrafo.
  • .data(data) vincula el array de datos a los elementos seleccionados.
  • .text(d => d) establece el contenido de texto de cada párrafo al valor de datos correspondiente.

Entrar, Actualizar, Salir

Selección de Entrada

La selección de entrada representa elementos que necesitan ser creados para coincidir con los datos.

const data = [10, 20, 30, 40, 50];

// Seleccionar y vincular datos
const p = d3.select("body").selectAll("p")
  .data(data);

// Selección de entrada
p.enter()
  .append("p")
  .text(d => d);

Selección de Actualización

La selección de actualización representa elementos que ya existen y necesitan ser actualizados.

const data = [10, 20, 30, 40, 50];

// Seleccionar y vincular datos
const p = d3.select("body").selectAll("p")
  .data(data);

// Selección de actualización
p.text(d => d);

Selección de Salida

La selección de salida representa elementos que necesitan ser eliminados porque ya no tienen datos correspondientes.

const data = [10, 20, 30];

// Seleccionar y vincular datos
const p = d3.select("body").selectAll("p")
  .data(data);

// Selección de salida
p.exit().remove();

Vinculación Avanzada de Datos

Funciones de Clave

Las funciones de clave ayudan a D3.js a identificar qué datos corresponden a qué elementos, especialmente útiles para actualizaciones dinámicas de datos.

const data = [
  { id: 1, value: 10 },
  { id: 2, value: 20 },
  { id: 3, value: 30 }
];

// Seleccionar y vincular datos con función de clave
const p = d3.select("body").selectAll("p")
  .data(data, d => d.id);

// Selección de entrada
p.enter()
  .append("p")
  .text(d => d.value);

// Selección de actualización
p.text(d => d.value);

// Selección de salida
p.exit().remove();

Fusión de Selecciones

Puedes fusionar las selecciones de entrada y actualización para aplicar operaciones comunes a ambas.

const data = [10, 20, 30, 40, 50];

// Seleccionar y vincular datos
const p = d3.select("body").selectAll("p")
  .data(data);

// Selección de entrada y actualización
p.enter()
  .append("p")
  .merge(p)
  .text(d => d);

Conclusión

Vincular datos a elementos es un concepto central en D3.js que te permite crear visualizaciones dinámicas e interactivas. Al comprender las selecciones de entrada, actualización y salida, así como usar funciones de clave y fusionar selecciones, puedes gestionar y actualizar eficazmente tus visualizaciones a medida que tus datos cambian. Practica estos conceptos con diferentes conjuntos de datos para volverte competente en la vinculación de datos en D3.js.
