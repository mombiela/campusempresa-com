Introducción

En esta sección, exploraremos los conceptos fundamentales de variables y constantes en Xcode usando Swift. Entender cómo declarar y usar variables y constantes es crucial para cualquier programador, ya que forman la base para almacenar y manipular datos en tus aplicaciones.

Variables

Las variables se utilizan para almacenar datos que pueden cambiar durante la ejecución de un programa. En Swift, las variables se declaran usando la palabra clave var.

Declaración de Variables

Para declarar una variable, usas la palabra clave var seguida del nombre de la variable y una anotación de tipo opcional.

var saludo: String = "¡Hola, mundo!"
  • var es la palabra clave usada para declarar una variable.
  • saludo es el nombre de la variable.
  • String es el tipo de la variable.
  • "¡Hola, mundo!" es el valor inicial asignado a la variable.

Inferencia de Tipo

Swift puede inferir el tipo de la variable basado en el valor inicial, por lo que puedes omitir la anotación de tipo.

var edad = 25

En este ejemplo, Swift infiere que edad es un Int porque el valor inicial es un entero.

Cambiando los Valores de las Variables

Puedes cambiar el valor de una variable después de que ha sido declarada.

var puntuación = 10
puntuación = 20

Constantes

Las constantes se utilizan para almacenar datos que no deben cambiar durante la ejecución de un programa. En Swift, las constantes se declaran usando la palabra clave let.

Declaración de Constantes

Para declarar una constante, usas la palabra clave let seguida del nombre de la constante y una anotación de tipo opcional.

let pi: Double = 3.14159
  • let es la palabra clave usada para declarar una constante.
  • pi es el nombre de la constante.
  • Double es el tipo de la constante.
  • 3.14159 es el valor inicial asignado a la constante.

Inferencia de Tipo

Al igual que con las variables, Swift puede inferir el tipo de la constante basado en el valor inicial.

let nombre = "Alicia"

En este ejemplo, Swift infiere que nombre es un String porque el valor inicial es una cadena de texto.

Intentando Cambiar los Valores de las Constantes

Si intentas cambiar el valor de una constante después de que ha sido declarada, Swift producirá un error en tiempo de compilación.

let añoDeNacimiento = 1990
// añoDeNacimiento = 1991 // Esto causará un error

Diferencias Clave Entre Variables y Constantes

| Característica | Variables (var) | Constantes (let) | |----------------|-------------------|--------------------| | Mutabilidad | Mutable | Inmutable | | Declaración | Palabra clave var| Palabra clave let| | Caso de Uso | Datos que cambian | Datos que permanecen constantes |

Mejores Prácticas

  • Usa constantes (let) por defecto a menos que sepas que un valor necesita cambiar.
  • Elige nombres descriptivos para tus variables y constantes para hacer tu código más legible.
  • Usa anotaciones de tipo cuando el tipo no sea claro por el contexto o cuando quieras ser explícito sobre el tipo.

Ejercicios

  1. Declara una variable llamada temperatura y asígnale el valor 72. Luego, cambia su valor a 68.
  2. Declara una constante llamada velocidadMáxima y asígnale el valor 120. Intenta cambiar su valor y observa el error.
  3. Declara una variable sin una anotación de tipo y asígnale un número de punto flotante. Deja que Swift infiera el tipo.

Conclusión

Entender las variables y constantes es esencial para gestionar datos en tus aplicaciones Swift. Las variables te permiten almacenar y modificar datos, mientras que las constantes aseguran que ciertos valores permanezcan sin cambios. Siguiendo las mejores prácticas y usando el tipo apropiado para tus datos, puedes escribir código más robusto y mantenible.
