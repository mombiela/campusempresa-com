Introducción a React

React es una biblioteca de JavaScript para construir interfaces de usuario, principalmente para aplicaciones de una sola página. Permite a los desarrolladores crear componentes de UI reutilizables, gestionar el estado de la aplicación de manera eficiente y manejar las interacciones del usuario sin problemas.

Conceptos Clave

  • Componentes: Los bloques de construcción de una aplicación React. Los componentes pueden ser funcionales o basados en clases.
  • JSX: Una extensión de sintaxis que permite escribir código similar a HTML dentro de JavaScript.
  • Estado: Un objeto que determina cómo se renderiza y se comporta un componente.
  • Props: Abreviatura de propiedades, las props son atributos de solo lectura utilizados para pasar datos de componentes padres a hijos.
  • Métodos del Ciclo de Vida: Métodos especiales en componentes de clase que permiten engancharse en diferentes etapas del ciclo de vida de un componente.

Configurando un Entorno de React

Para comenzar a construir aplicaciones React, necesitas configurar tu entorno de desarrollo.

Prerrequisitos

  • Node.js y npm (Node Package Manager) instalados en tu máquina.
  • Un editor de código, como Visual Studio Code.

Creando un Nuevo Proyecto de React

Usa la herramienta Create React App para configurar rápidamente un nuevo proyecto de React.

npx create-react-app my-react-app
cd my-react-app
npm start

Esto creará un nuevo proyecto de React y arrancará el servidor de desarrollo.

Entendiendo los Componentes

Los componentes son el núcleo de las aplicaciones React. Pueden ser funcionales o basados en clases.

Componentes Funcionales

Los componentes funcionales son simples funciones de JavaScript que devuelven JSX.

import React from 'react';

function Greeting() {
  return <h1>¡Hola, Mundo!</h1>;
}

export default Greeting;

Componentes de Clase

Los componentes de clase son clases ES6 que extienden React.Component y deben incluir un método render.

import React, { Component } from 'react';

class Greeting extends Component {
  render() {
    return <h1>¡Hola, Mundo!</h1>;
  }
}

export default Greeting;

JSX: JavaScript XML

JSX te permite escribir código similar a HTML dentro de JavaScript, facilitando la creación y visualización de componentes de UI.

Ejemplo de JSX

const element = <h1>¡Hola, Mundo!</h1>;

Incrustando Expresiones en JSX

Puedes incrustar expresiones de JavaScript dentro de JSX usando llaves {}.

const name = 'John';
const element = <h1>¡Hola, {name}!</h1>;

Estado y Props

El estado y las props son esenciales para gestionar datos y renderizar contenido dinámico en los componentes de React.

Estado

El estado es un objeto incorporado que almacena valores de propiedades que pertenecen a un componente. Cuando el objeto de estado cambia, el componente se vuelve a renderizar.

import React, { Component } from 'react';

class Counter extends Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }

  render() {
    return (
      <div>
        <p>Contador: {this.state.count}</p>
        <button onClick={() => this.setState({ count: this.state.count + 1 })}>
          Incrementar
        </button>
      </div>
    );
  }
}

export default Counter;

Props

Las props se utilizan para pasar datos de componentes padres a hijos.

function Welcome(props) {
  return <h1>Hola, {props.name}</h1>;
}

function App() {
  return <Welcome name="Sara" />;
}

export default App;

Ciclo de Vida del Componente

Los componentes de React tienen un ciclo de vida al que puedes engancharte usando métodos del ciclo de vida. Estos métodos solo están disponibles en componentes de clase.

Métodos Comunes del Ciclo de Vida

  • componentDidMount(): Llamado después de que el componente se renderiza.
  • componentDidUpdate(prevProps, prevState): Llamado después de que el componente se actualiza.
  • componentWillUnmount(): Llamado antes de que el componente se elimine del DOM.
import React, { Component } from 'react';

class Timer extends Component {
  componentDidMount() {
    this.timerID = setInterval(() => this.tick(), 1000);
  }

  componentWillUnmount() {
    clearInterval(this.timerID);
  }

  tick() {
    this.setState({
      date: new Date()
    });
  }

  render() {
    return (
      <div>
        <h1>{this.state.date.toLocaleTimeString()}</h1>
      </div>
    );
  }
}

export default Timer;

Conclusión

Entender los fundamentos de React es crucial para construir interfaces de usuario eficientes y escalables. Al dominar componentes, JSX, estado, props y métodos del ciclo de vida, puedes crear aplicaciones dinámicas e interactivas. A medida que progreses, estos conceptos fundamentales servirán como bloques de construcción para temas más avanzados en el desarrollo con React y React Native.
