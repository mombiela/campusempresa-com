Introducción a Redux

Redux es un contenedor de estado predecible para aplicaciones JavaScript, a menudo utilizado con bibliotecas como React para gestionar el estado de la aplicación. Te ayuda a escribir aplicaciones que se comportan de manera consistente, se ejecutan en diferentes entornos (cliente, servidor y nativo) y son fáciles de probar.

Conceptos Clave de Redux

  • Store: La única fuente de verdad que contiene el estado de tu aplicación.
  • Actions: Objetos simples de JavaScript que describen lo que ocurrió.
  • Reducers: Funciones que toman el estado actual y una acción, y devuelven un nuevo estado.
  • Dispatch: Una función para enviar acciones al store.
  • Selectors: Funciones para extraer y derivar datos del estado.

Configurando Redux en un Proyecto React Native

Instalando Dependencias

Para comenzar con Redux en un proyecto React Native, necesitas instalar los paquetes necesarios:

npm install redux react-redux

Creando el Store de Redux

Crea un archivo llamado store.js para configurar el store de Redux.

import { createStore } from 'redux';
import rootReducer from './reducers';

const store = createStore(rootReducer);

export default store;

Definiendo Acciones

Crea un archivo llamado actions.js para definir tus acciones.

// actions.js
export const INCREMENT = 'INCREMENT';
export const DECREMENT = 'DECREMENT';

export const increment = () => ({
  type: INCREMENT,
});

export const decrement = () => ({
  type: DECREMENT,
});

Creando Reducers

Crea un archivo llamado reducers.js para definir tus reducers.

// reducers.js
import { INCREMENT, DECREMENT } from './actions';

const initialState = {
  count: 0,
};

const counterReducer = (state = initialState, action) => {
  switch (action.type) {
    case INCREMENT:
      return { ...state, count: state.count + 1 };
    case DECREMENT:
      return { ...state, count: state.count - 1 };
    default:
      return state;
  }
};

export default counterReducer;

Combinando Reducers

Si tienes múltiples reducers, puedes combinarlos usando combineReducers.

// rootReducer.js
import { combineReducers } from 'redux';
import counterReducer from './counterReducer';

const rootReducer = combineReducers({
  counter: counterReducer,
});

export default rootReducer;

Conectando Redux a React Native

Envuelve tu componente principal de la aplicación con el Provider de react-redux y pásale el store.

// App.js
import React from 'react';
import { Provider } from 'react-redux';
import store from './store';
import Counter from './Counter';

const App = () => (
  <Provider store={store}>
    <Counter />
  </Provider>
);

export default App;

Usando Redux en Componentes

Conecta tus componentes al store de Redux usando la función connect de react-redux.

// Counter.js
import React from 'react';
import { View, Text, Button } from 'react-native';
import { connect } from 'react-redux';
import { increment, decrement } from './actions';

const Counter = ({ count, increment, decrement }) => (
  <View>
    <Text>{count}</Text>
    <Button title="Increment" onPress={increment} />
    <Button title="Decrement" onPress={decrement} />
  </View>
);

const mapStateToProps = (state) => ({
  count: state.counter.count,
});

const mapDispatchToProps = {
  increment,
  decrement,
};

export default connect(mapStateToProps, mapDispatchToProps)(Counter);

Conclusión

Integrar Redux con React Native implica configurar un store de Redux, definir acciones y reducers, y conectar tus componentes de React Native al store de Redux usando la función connect. Esta configuración ayuda a gestionar el estado de tu aplicación de una manera predecible y mantenible. Siguiendo estos pasos, puedes asegurarte de que tu aplicación React Native tenga una solución robusta para la gestión del estado.
