Introducción

En este curso, te guiaremos a través del proceso de creación de una aplicación del clima usando React Native. Este proyecto te ayudará a entender los fundamentos de React Native, la integración de API, la gestión del estado y más. Al final de este curso, tendrás una aplicación del clima completamente funcional que puede obtener y mostrar datos del clima desde una API externa.

Requisitos Previos

Antes de comenzar este curso, deberías tener:

  • Conocimientos básicos de JavaScript y ES6.
  • Familiaridad con conceptos de React como componentes, props y estado.
  • Node.js y npm instalados en tu máquina.

Configuración del Entorno de Desarrollo

Para comenzar con React Native, necesitas configurar tu entorno de desarrollo.

  1. Instalar Node.js y npm: Asegúrate de tener Node.js y npm instalados.
  2. Instalar React Native CLI: Ejecuta el siguiente comando para instalar React Native CLI globalmente. 
    npm install -g react-native-cli
  3. Crear un Nuevo Proyecto de React Native: Usa el CLI para crear un nuevo proyecto. 
    react-native init WeatherApp
  4. Ejecutar el Proyecto: Navega al directorio del proyecto y comienza el servidor de desarrollo. 
    cd WeatherApp
react-native run-android # Para Android
react-native run-ios # Para iOS

Construyendo la Interfaz de Usuario

En esta sección, construiremos la interfaz de usuario básica para nuestra aplicación del clima.

Creando la Pantalla Principal

  1. Crea un nuevo archivo MainScreen.js en el directorio src.

    import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

const MainScreen = () => {
    return (
        <View style={styles.container}>
            <Text style={styles.title}>Weather App</Text>
        </View>
    );
};

const styles = StyleSheet.create({
    container: {
        flex: 1,
        justifyContent: 'center',
        alignItems: 'center',
    },
    title: {
        fontSize: 24,
        fontWeight: 'bold',
    },
});

export default MainScreen;

  2. Actualiza App.js para usar el componente MainScreen.

    import React from 'react';
import MainScreen from './src/MainScreen';

const App = () => {
    return <MainScreen />;
};

export default App;

Obteniendo Datos del Clima desde una API

Para obtener datos del clima, usaremos la API de OpenWeatherMap.

  1. Regístrate para obtener una clave de API en OpenWeatherMap.
  2. Instala Axios para hacer solicitudes HTTP. 
    npm install axios

Obteniendo Datos

  1. Crea un nuevo archivo api.js en el directorio src para manejar las solicitudes de API.

    import axios from 'axios';

const API_KEY = 'YOUR_API_KEY';
const BASE_URL = 'https://api.openweathermap.org/data/2.5/weather';

export const fetchWeather = async (city) => {
    try {
        const response = await axios.get(`${BASE_URL}?q=${city}&appid=${API_KEY}`);
        return response.data;
    } catch (error) {
        console.error(error);
    }
};

  2. Actualiza MainScreen.js para obtener y mostrar datos del clima.

    import React, { useState, useEffect } from 'react';
import { View, Text, StyleSheet, TextInput, Button } from 'react-native';
import { fetchWeather } from './api';

const MainScreen = () => {
    const [city, setCity] = useState('');
    const [weather, setWeather] = useState(null);

    const getWeather = async () => {
        const data = await fetchWeather(city);
        setWeather(data);
    };

    return (
        <View style={styles.container}>
            <Text style={styles.title}>Weather App</Text>
            <TextInput
                style={styles.input}
                placeholder="Enter city"
                value={city}
                onChangeText={setCity}
            />
            <Button title="Get Weather" onPress={getWeather} />
            {weather && (
                <View style={styles.weatherContainer}>
                    <Text style={styles.weatherText}>Temperature: {weather.main.temp}°K</Text>
                    <Text style={styles.weatherText}>Condition: {weather.weather[0].description}</Text>
                </View>
            )}
        </View>
    );
};

const styles = StyleSheet.create({
    container: {
        flex: 1,
        justifyContent: 'center',
        alignItems: 'center',
    },
    title: {
        fontSize: 24,
        fontWeight: 'bold',
    },
    input: {
        height: 40,
        borderColor: 'gray',
        borderWidth: 1,
        marginBottom: 10,
        paddingHorizontal: 10,
        width: '80%',
    },
    weatherContainer: {
        marginTop: 20,
    },
    weatherText: {
        fontSize: 18,
    },
});

export default MainScreen;

Funcionalidades Avanzadas

Agregando Navegación

  1. Instala React Navigation: 
    npm install @react-navigation/native @react-navigation/stack
npm install react-native-screens react-native-safe-area-context
  2. Configura la Navegación en App.js: 
    import React from 'react';
import { NavigationContainer } from '@react-navigation/native';
import { createStackNavigator } from '@react-navigation/stack';
import MainScreen from './src/MainScreen';

const Stack = createStackNavigator();

const App = () => {
    return (
        <NavigationContainer>
            <Stack.Navigator>
                <Stack.Screen name="Main" component={MainScreen} />
            </Stack.Navigator>
        </NavigationContainer>
    );
};

export default App;

Agregando Clima Basado en Ubicación

  1. Instala React Native Geolocation: 
    npm install @react-native-community/geolocation
  2. Actualiza MainScreen.js para obtener el clima basado en la ubicación del usuario. 
    import React, { useState, useEffect } from 'react';
import { View, Text, StyleSheet, TextInput, Button } from 'react-native';
import Geolocation from '@react-native-community/geolocation';
import { fetchWeather } from './api';

const MainScreen = () => {
    const [city, setCity] = useState('');
    const [weather, setWeather] = useState(null);

    useEffect(() => {
        Geolocation.getCurrentPosition(
            async position => {
                const { latitude, longitude } = position.coords;
                const data = await fetchWeatherByCoords(latitude, longitude);
                setWeather(data);
            },
            error => console.error(error),
            { enableHighAccuracy: true, timeout: 20000, maximumAge: 1000 }
        );
    }, []);

    const getWeather = async () => {
        const data = await fetchWeather(city);
        setWeather(data);
    };

    return (
        <View style={styles.container}>
            <Text style={styles.title}>Weather App</Text>
            <TextInput
                style={styles.input}
                placeholder="Enter city"
                value={city}
                onChangeText={setCity}
            />
            <Button title="Get Weather" onPress={getWeather} />
            {weather && (
                <View style={styles.weatherContainer}>
                    <Text style={styles.weatherText}>Temperature: {weather.main.temp}°K</Text>
                    <Text style={styles.weatherText}>Condition: {weather.weather[0].description}</Text>
                </View>
            )}
        </View>
    );
};

const styles = StyleSheet.create({
    container: {
        flex: 1,
        justifyContent: 'center',
        alignItems: 'center',
    },
    title: {
        fontSize: 24,
        fontWeight: 'bold',
    },
    input: {
        height: 40,
        borderColor: 'gray',
        borderWidth: 1,
        marginBottom: 10,
        paddingHorizontal: 10,
        width: '80%',
    },
    weatherContainer: {
        marginTop: 20,
    },
    weatherText: {
        fontSize: 18,
    },
});

export default MainScreen;

Conclusión

En este curso, hemos construido una aplicación del clima usando React Native desde cero. Cubrimos la configuración del entorno de desarrollo, la construcción de la interfaz de usuario, la obtención de datos desde una API y la adición de funcionalidades avanzadas como la navegación y el clima basado en la ubicación. Este proyecto debería darte una base sólida en React Native y prepararte para proyectos más complejos. ¡Sigue experimentando y construyendo para mejorar tus habilidades aún más!
