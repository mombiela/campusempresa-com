Directivas de Vue.js
Vinculación de Datos ►

Las directivas de Vue.js son tokens especiales en el marcado que le dicen a la biblioteca que haga algo con un elemento del DOM. Están precedidas por v- para indicar que son atributos especiales proporcionados por Vue.js.

Directivas Clave

v-bind

  • Propósito: Vincular dinámicamente uno o más atributos, o una propiedad de componente, a una expresión.
  • Sintaxis: v-bind:attribute="expression"
<!-- Vinculando un atributo -->
<img v-bind:src="imageSrc" alt="Imagen Dinámica">

<!-- Forma abreviada -->
<img :src="imageSrc" alt="Imagen Dinámica">

v-model

  • Propósito: Crear una vinculación bidireccional en un elemento input, textarea o select.
  • Sintaxis: v-model="data"
<!-- Vinculación bidireccional con un elemento input -->
<input v-model="message" placeholder="Ingresa un mensaje">

<p>El mensaje es: {{ message }}</p>

v-if, v-else-if, v-else

  • Propósito: Renderizar condicionalmente elementos basados en la veracidad de una expresión.
  • Sintaxis: v-if="condition", v-else-if="condition", v-else
<!-- Renderizado condicional -->
<p v-if="seen">Ahora me ves</p>
<p v-else>Ahora no me ves</p>

v-show

  • Propósito: Alternar la visibilidad de un elemento basado en la veracidad de una expresión.
  • Sintaxis: v-show="condition"
<!-- Alternar visibilidad -->
<p v-show="isVisible">Soy visible</p>

v-for

  • Propósito: Renderizar una lista de elementos iterando sobre un array u objeto.
  • Sintaxis: v-for="item in items"
<!-- Iterando sobre un array -->
<ul>
  <li v-for="item in items" :key="item.id">{{ item.name }}</li>
</ul>

v-on

  • Propósito: Adjuntar oyentes de eventos a elementos.
  • Sintaxis: v-on:event="method"
<!-- Manejo de eventos -->
<button v-on:click="doSomething">Haz clic en mí</button>

<!-- Forma abreviada -->
<button @click="doSomething">Haz clic en mí</button>

v-pre

  • Propósito: Saltar la compilación para este elemento y todos sus hijos.
  • Sintaxis: v-pre
<!-- Saltando la compilación -->
<span v-pre>{{ esto no será compilado }}</span>

v-cloak

  • Propósito: Mantener el elemento y sus hijos ocultos hasta que la compilación de Vue esté completa.
  • Sintaxis: v-cloak
<!-- Previniendo el destello de contenido no compilado -->
<style>
[v-cloak] { display: none; }
</style>

<div v-cloak>
  {{ message }}
</div>

v-once

  • Propósito: Renderizar el elemento y componente una vez y omitir futuras actualizaciones.
  • Sintaxis: v-once
<!-- Renderizado una sola vez -->
<span v-once>Esto nunca cambiará: {{ message }}</span>

Directivas Personalizadas

Vue.js también te permite crear tus propias directivas personalizadas. Las directivas personalizadas pueden ser útiles para encapsular manipulaciones reutilizables del DOM.

Registrando una Directiva Personalizada

Vue.directive('focus', {
  // Cuando el elemento vinculado es insertado en el DOM...
  inserted: function (el) {
    // Enfocar el elemento
    el.focus()
  }
})

Usando una Directiva Personalizada

<!-- Aplicando la directiva personalizada -->
<input v-focus>

Resumen

Las directivas de Vue.js son herramientas poderosas que te permiten manipular el DOM de una manera declarativa. Cubren una amplia gama de funcionalidades, desde la vinculación de atributos y oyentes de eventos hasta el renderizado condicional y la renderización de listas. Entender y usar efectivamente estas directivas es crucial para construir aplicaciones dinámicas y responsivas con Vue.js. Las directivas personalizadas extienden aún más las capacidades de Vue, permitiéndote encapsular y reutilizar manipulaciones del DOM en toda tu aplicación.
