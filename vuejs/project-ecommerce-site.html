Introducción

En este proyecto, construiremos un sitio de comercio electrónico completamente funcional usando Vue.js. Este proyecto te ayudará a aplicar los conceptos aprendidos en lecciones anteriores y te introducirá a características más avanzadas de Vue.js. Al final de este proyecto, tendrás una comprensión sólida de cómo crear una aplicación web dinámica e interactiva.

Configuración del Proyecto

  • Instalar Vue CLI: Asegúrate de tener Vue CLI instalado. 
    npm install -g @vue/cli
  • Crear un nuevo proyecto: Usa Vue CLI para crear un nuevo proyecto. 
    vue create ecommerce-site
  • Navegar al directorio del proyecto: 
    cd ecommerce-site

Estructura del Proyecto

  • Componentes: Crea componentes reutilizables para diferentes partes del sitio.
  • Vistas: Configura diferentes vistas para la aplicación.
  • Enrutamiento: Implementa Vue Router para la navegación.
  • Gestión de Estado: Usa Vuex para gestionar el estado de la aplicación.

Componentes Clave

Componente de Encabezado

  • Propósito: Mostrar el logo del sitio, enlaces de navegación y un ícono de carrito de compras.
  • Ejemplo de Código: 
    <template>
  <header>
    <h1>Mi Sitio de Comercio Electrónico</h1>
    <nav>
      <router-link to="/">Inicio</router-link>
      <router-link to="/products">Productos</router-link>
      <router-link to="/cart">Carrito</router-link>
    </nav>
  </header>
</template>

<script>
export default {
  name: 'HeaderComponent'
}
</script>

<style scoped>
header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
  background-color: #333;
  color: #fff;
}
nav a {
  margin: 0 1rem;
  color: #fff;
  text-decoration: none;
}
</style>

Componente de Lista de Productos

  • Propósito: Mostrar una lista de productos obtenidos de una API.
  • Ejemplo de Código: 
    <template>
  <div>
    <h2>Productos</h2>
    <div v-if="loading">Cargando...</div>
    <div v-else>
      <div v-for="product in products" :key="product.id" class="product">
        <h3>{{ product.name }}</h3>
        <p>{{ product.description }}</p>
        <p>\${{ product.price }}</p>
        <button @click="addToCart(product)">Añadir al Carrito</button>
      </div>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      products: [],
      loading: true
    };
  },
  created() {
    fetch('https://api.example.com/products')
      .then(response => response.json())
      .then(data => {
        this.products = data;
        this.loading = false;
      });
  },
  methods: {
    addToCart(product) {
      this.$store.commit('addToCart', product);
    }
  }
}
</script>

<style scoped>
.product {
  border: 1px solid #ccc;
  padding: 1rem;
  margin: 1rem 0;
}
</style>

Enrutamiento

  • Configurar Vue Router: Instala y configura Vue Router. 
    npm install vue-router
  • Definir Rutas: Crea rutas para las páginas de Inicio, Productos y Carrito. 
    import Vue from 'vue';
import Router from 'vue-router';
import Home from './views/Home.vue';
import Products from './views/Products.vue';
import Cart from './views/Cart.vue';

Vue.use(Router);

export default new Router({
  routes: [
    { path: '/', component: Home },
    { path: '/products', component: Products },
    { path: '/cart', component: Cart }
  ]
});

Gestión de Estado

  • Configurar Vuex: Instala y configura Vuex para la gestión de estado. 
    npm install vuex
  • Definir Store: Crea una tienda Vuex para gestionar el estado del carrito. 
    import Vue from 'vue';
import Vuex from 'vuex';

Vue.use(Vuex);

export default new Vuex.Store({
  state: {
    cart: []
  },
  mutations: {
    addToCart(state, product) {
      state.cart.push(product);
    }
  }
});

Conclusión

En este proyecto, cubrimos la creación de un sitio de comercio electrónico usando Vue.js. Configuramos el proyecto, creamos componentes clave, implementamos enrutamiento y gestionamos el estado con Vuex. Este proyecto proporcionó una visión general comprensiva de cómo construir una aplicación web dinámica con Vue.js, preparándote para temas más avanzados y aplicaciones del mundo real.
