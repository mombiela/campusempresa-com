Proyecto Capstone

Introducción

El proyecto capstone es un ejercicio integral diseñado para consolidar tu aprendizaje y demostrar tu competencia en el uso de BigQuery. Este proyecto involucrará datos del mundo real y requerirá que apliques varios conceptos y técnicas cubiertos a lo largo del curso, desde consultas básicas hasta análisis de datos avanzados y optimización.

Descripción del Proyecto

En este proyecto capstone, tú:

  • Seleccionarás un Conjunto de Datos: Elige un conjunto de datos disponible públicamente o utiliza un conjunto de datos proporcionado por el curso.
  • Definirás Objetivos: Delinea claramente los objetivos de tu análisis.
  • Realizarás Exploración de Datos: Usa consultas SQL para explorar y comprender el conjunto de datos.
  • Limpieza y Transformación de Datos: Limpia y transforma los datos según sea necesario.
  • Análisis Avanzado: Aplica características avanzadas de BigQuery, como funciones de ventana, funciones definidas por el usuario y modelos de aprendizaje automático.
  • Visualización e Informes: Crea visualizaciones y genera informes para presentar tus hallazgos.

Guía Paso a Paso

1. Seleccionar un Conjunto de Datos

Elige un conjunto de datos que te interese. Algunas fuentes populares para conjuntos de datos incluyen:

  • Google Cloud Public Datasets
  • Kaggle Datasets
  • UCI Machine Learning Repository

2. Definir Objetivos

Define claramente lo que deseas lograr con tu análisis. Por ejemplo:

  • Identificar tendencias en los datos.
  • Predecir resultados futuros basados en datos históricos.
  • Encontrar correlaciones entre diferentes variables.

3. Realizar Exploración de Datos

Usa consultas SQL para explorar el conjunto de datos. Aquí hay algunas consultas de ejemplo:

-- Contar el número de filas en el conjunto de datos
SELECT COUNT(*) AS total_rows
FROM `project.dataset.table`;

-- Obtener un resumen de una columna específica
SELECT column_name, COUNT(*) AS count
FROM `project.dataset.table`
GROUP BY column_name
ORDER BY count DESC;

4. Limpieza y Transformación de Datos

Limpia y transforma los datos para hacerlos adecuados para el análisis. Esto podría implicar manejar valores faltantes, eliminar duplicados o crear nuevos campos calculados.

-- Eliminar duplicados
CREATE OR REPLACE TABLE `project.dataset.cleaned_table` AS
SELECT DISTINCT *
FROM `project.dataset.table`;

-- Manejar valores faltantes reemplazándolos con la media
CREATE OR REPLACE TABLE `project.dataset.cleaned_table` AS
SELECT *,
  IFNULL(numeric_column, (SELECT AVG(numeric_column) FROM `project.dataset.table`)) AS numeric_column
FROM `project.dataset.table`;

5. Análisis Avanzado

Aplica características avanzadas de BigQuery para obtener conocimientos más profundos.

Funciones de Ventana

-- Calcular el total acumulado de una columna numérica
SELECT column_name,
  SUM(numeric_column) OVER (ORDER BY date_column) AS running_total
FROM `project.dataset.table`;

Funciones Definidas por el Usuario

-- Crear una función definida por el usuario para categorizar datos
CREATE FUNCTION `project.dataset.categorize`(value FLOAT64) RETURNS STRING AS ((
  CASE
    WHEN value < 10 THEN 'Low'
    WHEN value BETWEEN 10 AND 20 THEN 'Medium'
    ELSE 'High'
  END
));

-- Usar la función en una consulta
SELECT column_name,
  `project.dataset.categorize`(numeric_column) AS category
FROM `project.dataset.table`;

Modelos de Aprendizaje Automático

-- Crear un modelo de regresión lineal
CREATE OR REPLACE MODEL `project.dataset.model`
OPTIONS(model_type='linear_reg') AS
SELECT
  feature1,
  feature2,
  target
FROM `project.dataset.table`;

-- Evaluar el modelo
SELECT
  *
FROM
  ML.EVALUATE(MODEL `project.dataset.model`, (
    SELECT
      feature1,
      feature2,
      target
    FROM `project.dataset.table`
  ));

6. Visualización e Informes

Crea visualizaciones y genera informes para presentar tus hallazgos. Puedes usar herramientas como Google Data Studio, Tableau, o incluso integrarte con bibliotecas de Python como Matplotlib o Seaborn.

-- Consulta de ejemplo para visualización
SELECT
  date_column,
  SUM(numeric_column) AS total_value
FROM `project.dataset.table`
GROUP BY date_column
ORDER BY date_column;

Conclusión

El proyecto capstone es una oportunidad para aplicar todo lo que has aprendido en un escenario práctico y real. Al completar este proyecto, tú:

  • Ganarás experiencia práctica con BigQuery.
  • Desarrollarás habilidades para resolver problemas al enfrentar desafíos de datos reales.
  • Crearás una pieza de portafolio que demuestre tu experiencia en BigQuery.

Recuerda documentar tu proceso, explicar tus decisiones y reflexionar sobre tus hallazgos. ¡Buena suerte!
