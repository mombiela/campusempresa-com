Introducción

El flujo de control es un concepto fundamental en la programación que te permite dictar el orden en que se ejecutan las declaraciones. En F#, el flujo de control se gestiona utilizando condicionales y bucles. Esta sección cubrirá los conceptos básicos de estos constructos, proporcionando ejemplos y ejercicios para ayudarte a entenderlos y aplicarlos de manera efectiva.

Condicionales

Declaraciones If-Else

La declaración if-else se utiliza para ejecutar código basado en una condición.

Sintaxis

if condition then
    // código a ejecutar si la condición es verdadera
else
    // código a ejecutar si la condición es falsa

Ejemplo

let number = 10

if number > 0 then
    printfn "El número es positivo."
else
    printfn "El número no es positivo."

Declaraciones If-Else Anidadas

Puedes anidar declaraciones if-else para manejar múltiples condiciones.

Ejemplo

let number = 0

if number > 0 then
    printfn "El número es positivo."
elif number < 0 then
    printfn "El número es negativo."
else
    printfn "El número es cero."

Coincidencia de Patrones

La coincidencia de patrones es una característica poderosa en F# que te permite hacer coincidir valores contra patrones y ejecutar código basado en la coincidencia.

Sintaxis

match expression with
| pattern1 -> // código a ejecutar si pattern1 coincide
| pattern2 -> // código a ejecutar si pattern2 coincide
| _ -> // código a ejecutar si ningún patrón coincide

Ejemplo

let number = 10

match number with
| n when n > 0 -> printfn "El número es positivo."
| n when n < 0 -> printfn "El número es negativo."
| _ -> printfn "El número es cero."

Bucles

Bucles For

El bucle for se utiliza para iterar sobre un rango de valores.

Sintaxis

for variable in start .. end do
    // código a ejecutar para cada iteración

Ejemplo

for i in 1 .. 5 do
    printfn "Iteración %d" i

Bucles While

El bucle while se utiliza para ejecutar código mientras una condición sea verdadera.

Sintaxis

while condition do
    // código a ejecutar mientras la condición sea verdadera

Ejemplo

let mutable count = 5

while count > 0 do
    printfn "El conteo es %d" count
    count <- count - 1

Funciones Recursivas

En F#, los bucles a menudo pueden ser reemplazados con funciones recursivas, que son funciones que se llaman a sí mismas.

Ejemplo

let rec factorial n =
    if n <= 1 then 1
    else n * factorial (n - 1)

printfn "El factorial de 5 es %d" (factorial 5)

Ejercicios

Ejercicio 1: Básico If-Else

Escribe una función que tome un entero y devuelva una cadena que indique si el número es positivo, negativo o cero.

Ejercicio 2: Coincidencia de Patrones

Escribe una función que tome un día de la semana (como una cadena) y devuelva si es un día laborable o un fin de semana.

Ejercicio 3: Bucle For

Escribe una función que imprima los primeros 10 números en la secuencia de Fibonacci usando un bucle for.

Ejercicio 4: Bucle While

Escribe una función que tome un entero e imprima todos los números pares desde 0 hasta ese entero usando un bucle while.

Ejercicio 5: Función Recursiva

Escribe una función recursiva para calcular el máximo común divisor (MCD) de dos números.

Conclusión

En esta sección, cubrimos los conceptos básicos del flujo de control en F#, incluyendo condicionales y bucles. Exploramos declaraciones if-else, coincidencia de patrones, bucles for, bucles while y funciones recursivas. Entender estos constructos es crucial para escribir código F# eficiente y efectivo. Asegúrate de practicar los ejercicios para solidificar tu comprensión.
