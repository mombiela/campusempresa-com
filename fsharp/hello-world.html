¡Bienvenido a tu viaje en la programación con F#! En esta sección, cubriremos lo básico para escribir tu primer programa en F#: el clásico ejemplo de "¡Hola, Mundo!". Esto te ayudará a familiarizarte con la sintaxis y estructura de F#.

Introducción a F#

F# es un lenguaje de programación funcional-primero que también soporta programación orientada a objetos e imperativa. Es parte del ecosistema .NET y es conocido por su sintaxis concisa y su potente inferencia de tipos.

Configurando Tu Entorno

Antes de escribir nuestro primer programa en F#, necesitamos configurar nuestro entorno de desarrollo.

Instalando el SDK de .NET

  1. Descarga e instala el SDK de .NET desde el sitio web oficial de .NET.
  2. Verifica la instalación abriendo una terminal o símbolo del sistema y escribiendo: 
    dotnet --version
    Deberías ver el número de versión del SDK instalado.

Configurando un Proyecto F#

  1. Abre una terminal o símbolo del sistema.
  2. Crea una nueva aplicación de consola ejecutando: 
    dotnet new console -lang "F#" -o HelloWorld
  3. Navega al directorio del proyecto: 
    cd HelloWorld

Escribiendo Tu Primer Programa en F#

Ahora que tenemos nuestro proyecto configurado, escribamos el programa "¡Hola, Mundo!".

Entendiendo la Estructura del Programa

Abre el archivo Program.fs en tu editor de texto o IDE favorito. Verás algo de código boilerplate que se ve así:

open System

[<EntryPoint>]
let main argv =
    printfn "Hello, World!"
    0 // return an integer exit code

Explicación del Código

  • open System: Esta línea importa el espacio de nombres System, que contiene clases fundamentales y clases base que definen tipos de datos de valor y referencia comúnmente usados.
  • [<EntryPoint>]: Este atributo marca la función main como el punto de entrada del programa.
  • let main argv =: Esto define la función main, que toma un arreglo de cadenas (argv) como argumentos.
  • printfn "Hello, World!": Esta línea imprime "¡Hola, Mundo!" en la consola. printfn es una función que formatea e imprime una cadena seguida de una nueva línea.
  • 0: Este es el valor de retorno de la función main, que es un código de salida entero. 0 típicamente significa que el programa se ha ejecutado exitosamente.

Ejecutando el Programa

Para ejecutar tu programa, usa el siguiente comando en tu terminal o símbolo del sistema:

dotnet run

Deberías ver la salida:

Hello, World!

Resumen

En esta sección, cubrimos lo siguiente:

  • Configurar el SDK de .NET y crear un nuevo proyecto F#.
  • Escribir y entender un simple programa de "¡Hola, Mundo!" en F#.
  • Ejecutar el programa para ver la salida.

¡Felicidades! Has dado tu primer paso en la programación con F#. En las próximas secciones, profundizaremos en la sintaxis de F#, tipos de datos y conceptos de programación funcional.
