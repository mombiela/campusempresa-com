Introducción a los Iteradores

Los iteradores son un concepto fundamental en C++ que te permiten recorrer los elementos de un contenedor (como arrays, vectores, listas, etc.) de manera consistente. Proporcionan una forma de acceder a los elementos de un contenedor sin exponer la estructura subyacente del mismo.

Conceptos Clave

  • Tipos de Iteradores: Entrada, Salida, Adelante, Bidireccional y Acceso Aleatorio.
  • Operaciones de Iteradores: Desreferenciación, Incremento, Decremento.
  • Categorías de Iteradores: Basadas en las capacidades y restricciones de los iteradores.

Uso Básico de Iteradores

¿Qué es un Iterador?

Un iterador es un objeto que apunta a un elemento dentro de un contenedor y puede ser usado para recorrer el contenedor.

Ejemplo: Uso Básico de Iteradores con `std::vector`

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    std::vector<int>::iterator it;

    for (it = numbers.begin(); it != numbers.end(); ++it) {
        std::cout << *it << " ";
    }
    return 0;
}
  • Explicación:
    • numbers.begin() devuelve un iterador al primer elemento.
    • numbers.end() devuelve un iterador a uno más allá del último elemento.
    • *it desreferencia el iterador para acceder al valor al que apunta.
    • ++it incrementa el iterador para apuntar al siguiente elemento.

Tipos de Iteradores

Iteradores de Entrada y Salida

  • Iteradores de Entrada: Pueden leer elementos de un contenedor.
  • Iteradores de Salida: Pueden escribir elementos en un contenedor.

Ejemplo: Iterador de Entrada

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    std::vector<int>::iterator it = numbers.begin();

    while (it != numbers.end()) {
        std::cout << *it << " ";
        ++it;
    }
    return 0;
}

Ejemplo: Iterador de Salida

#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> numbers(5);
    std::vector<int>::iterator it = numbers.begin();

    int value = 1;
    while (it != numbers.end()) {
        *it = value++;
        ++it;
    }

    for (int num : numbers) {
        std::cout << num << " ";
    }
    return 0;
}

Iteradores Adelante, Bidireccionales y de Acceso Aleatorio

  • Iteradores Adelante: Pueden leer y escribir elementos y moverse hacia adelante.
  • Iteradores Bidireccionales: Pueden moverse tanto hacia adelante como hacia atrás.
  • Iteradores de Acceso Aleatorio: Pueden moverse a cualquier elemento en tiempo constante.

Ejemplo: Iterador Bidireccional con `std::list`

#include <iostream>
#include <list>

int main() {
    std::list<int> numbers = {1, 2, 3, 4, 5};
    std::list<int>::iterator it = numbers.begin();

    // Mover hacia adelante
    ++it;
    std::cout << *it << " "; // Salida: 2

    // Mover hacia atrás
    --it;
    std::cout << *it << " "; // Salida: 1

    return 0;
}

Ejemplo: Iterador de Acceso Aleatorio con `std::vector`

#include <iostream>
#include <vector>

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    std::vector<int>::iterator it = numbers.begin();

    // Mover hacia adelante 3 posiciones
    it += 3;
    std::cout << *it << " "; // Salida: 4

    // Mover hacia atrás 2 posiciones
    it -= 2;
    std::cout << *it << " "; // Salida: 2

    return 0;
}

Técnicas Avanzadas de Iteradores

Uso de Traits de `std::iterator`

Los traits de std::iterator proporcionan una forma de definir iteradores personalizados heredando de std::iterator.

Ejemplo: Iterador Personalizado

#include <iostream>
#include <iterator>
#include <vector>

template <typename T>
class CustomIterator : public std::iterator<std::input_iterator_tag, T> {
    T* p;
public:
    CustomIterator(T* x) : p(x) {}
    CustomIterator(const CustomIterator& mit) : p(mit.p) {}
    CustomIterator& operator++() { ++p; return *this; }
    CustomIterator operator++(int) { CustomIterator tmp(*this); operator++(); return tmp; }
    bool operator==(const CustomIterator& rhs) const { return p == rhs.p; }
    bool operator!=(const CustomIterator& rhs) const { return p != rhs.p; }
    T& operator*() { return *p; }
};

int main() {
    std::vector<int> numbers = {1, 2, 3, 4, 5};
    CustomIterator<int> begin(numbers.data());
    CustomIterator<int> end(numbers.data() + numbers.size());

    while (begin != end) {
        std::cout << *begin << " ";
        ++begin;
    }
    return 0;
}

Conclusión

Los iteradores son una característica poderosa en C++ que proporcionan una forma uniforme de acceder a los elementos en un contenedor. Vienen en varios tipos y categorías, cada uno adecuado para diferentes tareas. Entender y usar los iteradores de manera efectiva puede mejorar enormemente tu capacidad para trabajar con colecciones en C++.
