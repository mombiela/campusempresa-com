Entrada y Salida en C++

Introducción

La entrada y salida (I/O) son conceptos fundamentales en cualquier lenguaje de programación. En C++, las operaciones de I/O se manejan principalmente usando flujos. Esta sección cubrirá los conceptos básicos de I/O en C++, incluyendo cómo leer desde el teclado y escribir en la consola, así como temas más avanzados como I/O de archivos y manipuladores de flujo personalizados.

I/O Básico en Consola

Lectura desde la Consola

Para leer la entrada desde la consola, C++ proporciona el objeto cin, que es parte de la biblioteca estándar.

Ejemplo:

#include <iostream>

int main() {
    int number;
    std::cout << "Introduce un número: ";
    std::cin >> number;
    std::cout << "Has introducido: " << number << std::endl;
    return 0;
}
  • std::cin se usa para leer la entrada desde la entrada estándar (teclado).
  • std::cout se usa para escribir la salida en la salida estándar (consola).
  • std::endl se usa para insertar un carácter de nueva línea y vaciar el flujo.

Escritura en la Consola

Para escribir la salida en la consola, C++ usa el objeto cout.

Ejemplo:

#include <iostream>

int main() {
    std::cout << "¡Hola, Mundo!" << std::endl;
    return 0;
}
  • << es el operador de inserción usado para enviar datos al flujo de salida.

I/O Avanzado en Consola

Formateo de la Salida

C++ proporciona varios manipuladores para formatear la salida.

Ejemplo:

#include <iostream>
#include <iomanip>

int main() {
    double pi = 3.14159;
    std::cout << "Por defecto: " << pi << std::endl;
    std::cout << "Fijo: " << std::fixed << pi << std::endl;
    std::cout << "Científico: " << std::scientific << pi << std::endl;
    std::cout << "Precisión: " << std::setprecision(2) << pi << std::endl;
    return 0;
}
  • std::fixed y std::scientific se usan para formatear números de punto flotante.
  • std::setprecision establece el número de dígitos a mostrar después del punto decimal.

I/O de Archivos

Lectura desde un Archivo

Para leer desde un archivo, C++ usa la clase ifstream.

Ejemplo:

#include <iostream>
#include <fstream>
#include <string>

int main() {
    std::ifstream inputFile("example.txt");
    std::string line;

    if (inputFile.is_open()) {
        while (getline(inputFile, line)) {
            std::cout << line << std::endl;
        }
        inputFile.close();
    } else {
        std::cout << "No se puede abrir el archivo" << std::endl;
    }

    return 0;
}
  • std::ifstream se usa para leer desde archivos.
  • getline lee una línea del archivo.

Escritura en un Archivo

Para escribir en un archivo, C++ usa la clase ofstream.

Ejemplo:

#include <iostream>
#include <fstream>

int main() {
    std::ofstream outputFile("example.txt");

    if (outputFile.is_open()) {
        outputFile << "Esta es una línea.\n";
        outputFile << "Esta es otra línea.\n";
        outputFile.close();
    } else {
        std::cout << "No se puede abrir el archivo" << std::endl;
    }

    return 0;
}
  • std::ofstream se usa para escribir en archivos.
  • El operador << se usa para enviar datos al archivo.

Manipuladores de Flujo Personalizados

C++ te permite crear manipuladores personalizados para formatear la salida de maneras específicas.

Ejemplo:

#include <iostream>
#include <iomanip>

std::ostream& customManipulator(std::ostream& os) {
    return os << std::setw(10) << std::setfill('*');
}

int main() {
    std::cout << customManipulator << 123 << std::endl;
    return 0;
}
  • std::setw establece el ancho del campo de salida.
  • std::setfill establece el carácter de relleno.

Conclusión

Entender la entrada y salida en C++ es crucial para una programación efectiva. Esta sección cubrió los conceptos básicos de I/O en consola, técnicas avanzadas de formateo, I/O de archivos y manipuladores de flujo personalizados. Dominar estos conceptos te permitirá manejar datos de manera eficiente y presentarlos de una manera amigable para el usuario.
