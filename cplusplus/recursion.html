Introducción a la Recursión

La recursión es un concepto fundamental en la informática y la programación donde una función se llama a sí misma directa o indirectamente. Es una herramienta poderosa para resolver problemas que pueden descomponerse en subproblemas más pequeños y similares.

Conceptos Clave

  • Caso Base: La condición bajo la cual la función recursiva deja de llamarse a sí misma.
  • Caso Recursivo: La parte de la función donde la función se llama a sí misma con argumentos modificados.
  • Desbordamiento de Pila: Un problema potencial donde demasiadas llamadas recursivas agotan la memoria de la pila del programa.

Ejemplo Básico de Recursión

Comencemos con un ejemplo simple: calcular el factorial de un número.

Función Factorial

El factorial de un número entero no negativo n es el producto de todos los enteros positivos menores o iguales a n. Se denota por n!.

Definición Matemática

  • Caso Base: 0! = 1
  • Caso Recursivo: n! = n * (n-1)!

Ejemplo de Código en C++

#include <iostream>

int factorial(int n) {
    if (n == 0) {
        return 1; // Caso base
    } else {
        return n * factorial(n - 1); // Caso recursivo
    }
}

int main() {
    int number = 5;
    std::cout << "Factorial de " << number << " es " << factorial(number) << std::endl;
    return 0;
}

Explicación:

  • La función factorial se llama a sí misma con n-1 hasta que n sea 0.
  • Cuando n es 0, la función devuelve 1, terminando la recursión.

Recursión Intermedia: Secuencia de Fibonacci

La secuencia de Fibonacci es otro ejemplo clásico de recursión. Cada número en la secuencia es la suma de los dos anteriores.

Función Fibonacci

Definición Matemática

  • Caso Base: F(0) = 0, F(1) = 1
  • Caso Recursivo: F(n) = F(n-1) + F(n-2)

Ejemplo de Código en C++

#include <iostream>

int fibonacci(int n) {
    if (n <= 1) {
        return n; // Casos base
    } else {
        return fibonacci(n - 1) + fibonacci(n - 2); // Caso recursivo
    }
}

int main() {
    int number = 10;
    std::cout << "Fibonacci de " << number << " es " << fibonacci(number) << std::endl;
    return 0;
}

Explicación:

  • La función fibonacci se llama a sí misma dos veces, reduciendo el tamaño del problema en 1 y 2 respectivamente.
  • Esto resulta en un número exponencial de llamadas, lo que puede ser ineficiente para n grandes.

Recursión Avanzada: Recursión de Cola

La recursión de cola es un caso especial de recursión donde la llamada recursiva es la última operación en la función. Puede ser optimizada por el compilador para evitar el desbordamiento de pila.

Factorial Recursivo de Cola

Ejemplo de Código en C++

#include <iostream>

int tailFactorial(int n, int accumulator = 1) {
    if (n == 0) {
        return accumulator; // Caso base
    } else {
        return tailFactorial(n - 1, n * accumulator); // Caso recursivo
    }
}

int main() {
    int number = 5;
    std::cout << "Factorial Recursivo de Cola de " << number << " es " << tailFactorial(number) << std::endl;
    return 0;
}

Explicación:

  • La función tailFactorial lleva un parámetro adicional accumulator para almacenar resultados intermedios.
  • La llamada recursiva es la última operación, permitiendo al compilador optimizarla.

Comparando Recursión e Iteración

La recursión y la iteración pueden resolver a menudo los mismos problemas, pero tienen diferentes compensaciones.

| Aspecto | Recursión | Iteración | |-----------------|------------------------------------------|-----------------------------------------| | Legibilidad | A menudo más legible y concisa | Puede ser menos intuitiva | | Uso de Memoria | Usa más memoria de pila | Usa menos memoria | | Rendimiento | Puede ser más lenta debido al overhead de llamadas a funciones | Generalmente más rápida | | Complejidad | Más fácil de implementar para problemas de divide y vencerás | Mejor para bucles e iteraciones simples |

Ejercicios

  1. Básico: Escribe una función recursiva para calcular la suma de los primeros n números naturales.
  2. Intermedio: Implementa una función recursiva para invertir una cadena.
  3. Avanzado: Escribe una función recursiva de cola para calcular el máximo común divisor (MCD) de dos números.

Conclusión

La recursión es una técnica poderosa en C++ que permite soluciones elegantes a problemas complejos. Entender los casos base y recursivos es crucial para escribir funciones recursivas correctas. Aunque la recursión puede ser menos eficiente que la iteración en algunos casos, es invaluable para problemas que se ajustan naturalmente a un patrón recursivo. La optimización de la recursión de cola puede ayudar a mitigar algunos de los problemas de rendimiento asociados con la recursión.
