Polimorfismo en C++
Encapsulación y Abstracción ►

El polimorfismo es uno de los conceptos fundamentales de la Programación Orientada a Objetos (POO) en C++. Permite que los objetos sean tratados como instancias de su clase padre en lugar de su clase real. Los dos tipos principales de polimorfismo en C++ son el polimorfismo en tiempo de compilación (o estático) y el polimorfismo en tiempo de ejecución (o dinámico).

Polimorfismo en Tiempo de Compilación

El polimorfismo en tiempo de compilación se logra mediante la sobrecarga de funciones y la sobrecarga de operadores.

Sobrecarga de Funciones

La sobrecarga de funciones permite que múltiples funciones tengan el mismo nombre pero diferentes parámetros.

#include <iostream>
using namespace std;

class Print {
public:
    void show(int i) {
        cout << "Entero: " << i << endl;
    }
    void show(double d) {
        cout << "Doble: " << d << endl;
    }
    void show(string s) {
        cout << "Cadena: " << s << endl;
    }
};

int main() {
    Print obj;
    obj.show(5);
    obj.show(5.5);
    obj.show("Hola");
    return 0;
}

En este ejemplo, la función show está sobrecargada para manejar diferentes tipos de parámetros.

Sobrecarga de Operadores

La sobrecarga de operadores permite redefinir la forma en que los operadores funcionan para tipos definidos por el usuario.

#include <iostream>
using namespace std;

class Complex {
private:
    float real;
    float imag;
public:
    Complex() : real(0), imag(0) {}
    Complex(float r, float i) : real(r), imag(i) {}

    Complex operator + (const Complex& obj) {
        Complex temp;
        temp.real = real + obj.real;
        temp.imag = imag + obj.imag;
        return temp;
    }

    void display() {
        cout << "Real: " << real << " Imaginario: " << imag << endl;
    }
};

int main() {
    Complex c1(3.3, 4.4), c2(1.1, 2.2);
    Complex c3 = c1 + c2;
    c3.display();
    return 0;
}

En este ejemplo, el operador + está sobrecargado para sumar dos objetos Complex.

Polimorfismo en Tiempo de Ejecución

El polimorfismo en tiempo de ejecución se logra mediante la herencia y las funciones virtuales.

Herencia

La herencia permite que una clase herede propiedades y comportamientos de otra clase.

#include <iostream>
using namespace std;

class Base {
public:
    void show() {
        cout << "Clase Base" << endl;
    }
};

class Derived : public Base {
public:
    void show() {
        cout << "Clase Derivada" << endl;
    }
};

int main() {
    Derived d;
    d.show();
    return 0;
}

En este ejemplo, la clase Derived hereda de la clase Base y sobrescribe la función show.

Funciones Virtuales

Las funciones virtuales permiten que las clases derivadas sobrescriban métodos en una clase base, habilitando el polimorfismo en tiempo de ejecución.

#include <iostream>
using namespace std;

class Base {
public:
    virtual void show() {
        cout << "Clase Base" << endl;
    }
};

class Derived : public Base {
public:
    void show() override {
        cout << "Clase Derivada" << endl;
    }
};

int main() {
    Base* b;
    Derived d;
    b = &d;
    b->show();
    return 0;
}

En este ejemplo, la función show en la clase Base se declara como virtual, permitiendo que la clase Derived la sobrescriba. Cuando se llama a la función show a través de un puntero de la clase base, se ejecuta la versión de la clase derivada.

Resumen

El polimorfismo en C++ es una característica poderosa que permite un código más flexible y mantenible. Se puede lograr mediante:

  • Polimorfismo en Tiempo de Compilación: Usando la sobrecarga de funciones y la sobrecarga de operadores.
  • Polimorfismo en Tiempo de Ejecución: Usando la herencia y las funciones virtuales.

Comprender y utilizar eficazmente el polimorfismo puede mejorar en gran medida tu capacidad para escribir programas en C++ robustos y escalables.
