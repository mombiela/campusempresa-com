Presentación del Proyecto

Introducción

En esta sección, presentaremos el proyecto que desarrollaremos a lo largo del curso de Fundamentos de la Programación. Este proyecto servirá como una aplicación práctica de los conceptos que aprenderemos, permitiéndote consolidar tus conocimientos y habilidades en programación.

Objetivos del Proyecto

  • Aplicar conceptos básicos de programación: Variables, estructuras de control, funciones, etc.
  • Desarrollar habilidades de resolución de problemas: Analizar y descomponer problemas complejos en partes manejables.
  • Fomentar buenas prácticas de codificación: Escribir código limpio, comentado y bien estructurado.
  • Introducir herramientas y entornos de desarrollo: Familiarizarse con IDEs, sistemas de control de versiones, etc.

Descripción del Proyecto

El proyecto consistirá en desarrollar una aplicación de gestión de tareas. Esta aplicación permitirá a los usuarios:

  • Crear tareas: Añadir nuevas tareas con detalles como título, descripción y fecha de vencimiento.
  • Listar tareas: Ver todas las tareas creadas, con opciones para filtrar y ordenar.
  • Actualizar tareas: Modificar los detalles de una tarea existente.
  • Eliminar tareas: Borrar tareas que ya no sean necesarias.

Requisitos del Proyecto

Funcionalidades Básicas

  • Interfaz de Usuario: Una interfaz simple que permita la interacción con las funcionalidades mencionadas.
  • Persistencia de Datos: Almacenar las tareas en un archivo o base de datos para que persistan entre sesiones.
  • Validación de Datos: Asegurarse de que los datos ingresados por el usuario sean válidos y coherentes.

Tecnologías y Herramientas

  • Lenguaje de Programación: Python
  • Entorno de Desarrollo: Visual Studio Code (VSCode)
  • Control de Versiones: Git y GitHub

Estructura del Proyecto

Módulos y Componentes

  • main.py: Punto de entrada de la aplicación.
  • task_manager.py: Módulo que contiene la lógica para gestionar las tareas.
  • ui.py: Módulo que maneja la interfaz de usuario.
  • storage.py: Módulo responsable de la persistencia de datos.

Ejemplo de Código

A continuación, se muestra un ejemplo de cómo podría lucir la estructura básica de task_manager.py:

# task_manager.py

tasks = []

def add_task(title, description, due_date):
    task = {
        'title': title,
        'description': description,
        'due_date': due_date,
        'completed': False
    }
    tasks.append(task)

def list_tasks():
    for task in tasks:
        print(f"Title: {task['title']}, Due Date: {task['due_date']}, Completed: {task['completed']}")

def update_task(index, title=None, description=None, due_date=None, completed=None):
    if index < len(tasks):
        if title:
            tasks[index]['title'] = title
        if description:
            tasks[index]['description'] = description
        if due_date:
            tasks[index]['due_date'] = due_date
        if completed is not None:
            tasks[index]['completed'] = completed

def delete_task(index):
    if index < len(tasks):
        tasks.pop(index)

Ejercicio Práctico

  1. Crear una Tarea: Implementa la función add_task en task_manager.py.
  2. Listar Tareas: Implementa la función list_tasks y prueba listando las tareas creadas.
  3. Actualizar una Tarea: Implementa la función update_task y prueba actualizando una tarea existente.
  4. Eliminar una Tarea: Implementa la función delete_task y prueba eliminando una tarea.

Conclusión

En esta sección, hemos presentado el proyecto que desarrollaremos durante el curso. A medida que avancemos en el curso, iremos construyendo y mejorando esta aplicación, integrando nuevas funcionalidades y optimizando el código. ¡Prepárate para aprender y disfrutar del proceso de creación de software!
