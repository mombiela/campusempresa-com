Introducción

Las estructuras de control condicionales son fundamentales en la programación, ya que permiten que el flujo de ejecución de un programa varíe en función de ciertas condiciones. Estas estructuras permiten a los programadores tomar decisiones dentro del código, ejecutando diferentes bloques de código según se cumplan o no ciertas condiciones.

Tipos de Estructuras Condicionales

En la mayoría de los lenguajes de programación, las estructuras de control condicionales más comunes son:

  • if
  • else
  • else if
  • switch (o case en algunos lenguajes)

`if`

La estructura if evalúa una condición y, si esta es verdadera, ejecuta un bloque de código.

# Ejemplo en Python
x = 10
if x > 5:
    print("x es mayor que 5")

`else`

La estructura else se utiliza junto con if para ejecutar un bloque de código alternativo si la condición del if es falsa.

# Ejemplo en Python
x = 3
if x > 5:
    print("x es mayor que 5")
else:
    print("x no es mayor que 5")

`else if` (o `elif` en Python)

La estructura else if permite evaluar múltiples condiciones secuencialmente. Si la primera condición es falsa, se evalúa la siguiente, y así sucesivamente.

# Ejemplo en Python
x = 7
if x > 10:
    print("x es mayor que 10")
elif x > 5:
    print("x es mayor que 5 pero menor o igual a 10")
else:
    print("x es 5 o menor")

`switch` (o `case`)

La estructura switch permite seleccionar entre múltiples bloques de código a ejecutar, basándose en el valor de una expresión.

// Ejemplo en Java
int day = 3;
switch (day) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    case 3:
        System.out.println("Miércoles");
        break;
    default:
        System.out.println("Otro día");
        break;
}

Comparación de Estructuras Condicionales

| Estructura | Uso Principal | Ventajas | Desventajas | |------------|---------------|----------|-------------| | if | Evaluar una sola condición | Simple y directo | Puede volverse complejo con múltiples condiciones | | else | Proveer una alternativa si if es falso | Fácil de leer | Depende de if | | else if | Evaluar múltiples condiciones secuenciales | Más control sobre el flujo | Puede ser difícil de mantener con muchas condiciones | | switch | Seleccionar entre múltiples opciones basadas en un valor | Claro para múltiples opciones | Menos flexible que if-else para condiciones complejas |

Ejercicios Prácticos

  1. Ejercicio 1: Condicional simple

    • Escribe un programa que verifique si un número es positivo, negativo o cero.
    # Ejercicio 1
num = int(input("Introduce un número: "))
if num > 0:
    print("El número es positivo")
elif num < 0:
    print("El número es negativo")
else:
    print("El número es cero")

  2. Ejercicio 2: Uso de switch

    • Escribe un programa que imprima el nombre del día de la semana basado en un número del 1 al 7.
    // Ejercicio 2
int day = 4; // Cambia este valor para probar diferentes días
switch (day) {
    case 1:
        System.out.println("Lunes");
        break;
    case 2:
        System.out.println("Martes");
        break;
    case 3:
        System.out.println("Miércoles");
        break;
    case 4:
        System.out.println("Jueves");
        break;
    case 5:
        System.out.println("Viernes");
        break;
    case 6:
        System.out.println("Sábado");
        break;
    case 7:
        System.out.println("Domingo");
        break;
    default:
        System.out.println("Número inválido");
        break;
}

Conclusión

Las estructuras de control condicionales son esenciales para la toma de decisiones en la programación. Comprender cómo y cuándo utilizarlas es crucial para escribir código eficiente y claro. A través de if, else, else if y switch, los programadores pueden dirigir el flujo de ejecución de sus programas de manera lógica y organizada. Practicar con ejemplos y ejercicios ayudará a consolidar estos conceptos y a aplicarlos correctamente en situaciones reales.
