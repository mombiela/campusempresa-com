Introducción

React Router es una biblioteca poderosa que permite la navegación entre vistas o componentes en una aplicación React. Te permite construir una aplicación de una sola página con navegación sin que la página se recargue a medida que el usuario navega. Esta sección te guiará a través de la configuración de React Router en tu proyecto React.

Instalando React Router

Para comenzar con React Router, necesitas instalar el paquete react-router-dom. Puedes hacerlo usando npm o yarn.

Usando npm

npm install react-router-dom

Usando yarn

yarn add react-router-dom

Configuración Básica

Una vez que hayas instalado react-router-dom, puedes comenzar a configurar tus rutas.

Importando Componentes Necesarios

Primero, importa los componentes necesarios de react-router-dom en tu archivo principal de la aplicación (usualmente App.js o index.js).

import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';

Creando Rutas Básicas

A continuación, envuelve tu aplicación con el componente Router y define tus rutas usando el componente Route. El componente Switch se usa para renderizar solo la primera ruta que coincida con la ubicación actual.

import React from 'react';
import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';
import Home from './components/Home';
import About from './components/About';
import Contact from './components/Contact';

function App() {
  return (
    <Router>
      <Switch>
        <Route exact path="/" component={Home} />
        <Route path="/about" component={About} />
        <Route path="/contact" component={Contact} />
      </Switch>
    </Router>
  );
}

export default App;

Explicación

  • Router: Envuelve toda la aplicación y habilita el enrutamiento.
  • Switch: Asegura que solo una ruta se renderice a la vez.
  • Route: Define una ruta y el componente que debe renderizarse cuando la ruta coincide.

Rutas Anidadas

React Router también soporta rutas anidadas, lo que te permite definir rutas dentro de otras rutas.

Ejemplo de Rutas Anidadas

import React from 'react';
import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';
import Dashboard from './components/Dashboard';
import Profile from './components/Profile';
import Settings from './components/Settings';

function App() {
  return (
    <Router>
      <Switch>
        <Route path="/dashboard" component={Dashboard} />
      </Switch>
    </Router>
  );
}

function Dashboard() {
  return (
    <div>
      <h2>Dashboard</h2>
      <Switch>
        <Route path="/dashboard/profile" component={Profile} />
        <Route path="/dashboard/settings" component={Settings} />
      </Switch>
    </div>
  );
}

export default App;

Explicación

  • Switch Anidado: El componente Switch dentro del componente Dashboard permite el enrutamiento anidado dentro del panel de control.

Enlaces de Navegación

Para navegar entre rutas, puedes usar el componente Link de react-router-dom.

Ejemplo de Enlaces de Navegación

import React from 'react';
import { BrowserRouter as Router, Route, Switch, Link } from 'react-router-dom';
import Home from './components/Home';
import About from './components/About';
import Contact from './components/Contact';

function App() {
  return (
    <Router>
      <nav>
        <ul>
          <li><Link to="/">Home</Link></li>
          <li><Link to="/about">About</Link></li>
          <li><Link to="/contact">Contact</Link></li>
        </ul>
      </nav>
      <Switch>
        <Route exact path="/" component={Home} />
        <Route path="/about" component={About} />
        <Route path="/contact" component={Contact} />
      </Switch>
    </Router>
  );
}

export default App;

Explicación

  • Link: El componente Link se usa para crear enlaces de navegación. Evita que la página se recargue y permite transiciones suaves entre rutas.

Conclusión

Configurar React Router es esencial para construir aplicaciones de una sola página con React. Siguiendo los pasos descritos en esta sección, puedes crear rutas básicas y anidadas, y navegar entre ellas usando el componente Link. Este conocimiento fundamental te permitirá construir estructuras de enrutamiento más complejas a medida que tu aplicación crezca.
