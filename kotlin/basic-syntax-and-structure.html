Introducción a Kotlin

Kotlin es un lenguaje de programación moderno, estáticamente tipado que se ejecuta en la Máquina Virtual de Java (JVM) y también puede ser compilado a JavaScript o código nativo. Está diseñado para ser completamente interoperable con Java y ofrece una sintaxis más concisa, lo que lo convierte en una opción popular para el desarrollo de Android y otras aplicaciones.

Conceptos Clave

  • Variables y Constantes
  • Tipos de Datos
  • Flujo de Control
  • Funciones
  • Clases y Objetos

Variables y Constantes

En Kotlin, puedes declarar variables usando var y constantes usando val.

Variables

var mutableVariable: Int = 10
mutableVariable = 20 // Esto está permitido
  • var se usa para declarar una variable mutable, lo que significa que su valor puede cambiar.

Constantes

val immutableVariable: Int = 10
immutableVariable = 20 // Esto causará un error de compilación
  • val se usa para declarar una variable de solo lectura, lo que significa que su valor no puede cambiar una vez asignado.

Tipos de Datos

Kotlin tiene un conjunto rico de tipos de datos integrados.

Tipos de Datos Básicos

  • Números: Int, Long, Float, Double
  • Caracteres: Char
  • Booleanos: Boolean
  • Cadenas: String

Ejemplo

val anInt: Int = 42
val aDouble: Double = 3.14
val aChar: Char = 'A'
val aBoolean: Boolean = true
val aString: String = "¡Hola, Kotlin!"

Flujo de Control

Kotlin proporciona varias formas de controlar el flujo de tu programa.

If-Else

val max = if (a > b) a else b
  • La declaración if puede ser usada como una expresión.

When

val result = when (x) {
    1 -> "Uno"
    2 -> "Dos"
    else -> "Desconocido"
}
  • when es una versión más poderosa de la declaración switch en otros lenguajes.

Bucles

Bucle For

for (i in 1..10) {
    println(i)
}
  • El bucle for itera sobre un rango.

Bucle While

var i = 0
while (i < 10) {
    println(i)
    i++
}
  • El bucle while continúa mientras la condición sea verdadera.

Funciones

Las funciones en Kotlin son ciudadanos de primera clase y pueden ser declaradas de varias maneras.

Función Básica

fun add(a: Int, b: Int): Int {
    return a + b
}
  • Las funciones se declaran usando la palabra clave fun.

Función de Expresión Única

fun add(a: Int, b: Int): Int = a + b
  • Las funciones con una sola expresión pueden ser escritas de manera más concisa.

Argumentos Predeterminados

fun greet(name: String = "Mundo") {
    println("¡Hola, $name!")
}
  • Las funciones pueden tener valores predeterminados para los parámetros.

Clases y Objetos

Kotlin soporta programación orientada a objetos con clases y objetos.

Clase Básica

class Person(val name: String, var age: Int)
  • Una clase puede tener propiedades declaradas en el constructor primario.

Creación de un Objeto

val person = Person("John", 30)
println(person.name) // John
println(person.age)  // 30
  • Los objetos se crean usando la palabra clave new.

Métodos en Clases

class Person(val name: String, var age: Int) {
    fun greet() {
        println("Hola, mi nombre es $name y tengo $age años.")
    }
}
  • Los métodos pueden ser definidos dentro de las clases.

Conclusión

Entender la sintaxis y estructura básica de Kotlin es crucial para escribir código efectivo y eficiente. Esto incluye saber cómo declarar variables y constantes, entender los tipos de datos, controlar el flujo de tu programa, definir funciones y crear clases y objetos. El dominio de estos conceptos fundamentales proporcionará una base sólida para temas de programación en Kotlin más avanzados.
