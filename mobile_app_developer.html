Diseña, construye y mantiene aplicaciones móviles para plataformas iOS y Android utilizando tecnologías nativas o multiplataforma.

Competencias técnicas

Apache Cordova
Framework de desarrollo móvil que permite crear aplicaciones multiplataforma utilizando tecnologías web estándar como HTML, CSS y JavaScript, empaquetándolas para su ejecución en dispositivos móviles.
Kotlin
Lenguaje de programación moderno y conciso para el desarrollo de aplicaciones Android, compatible con Java.
Xcode
Entorno de desarrollo integrado (IDE) utilizado para crear aplicaciones en iOS y macOS, desarrollado por Apple.
Firebase
Plataforma de desarrollo de aplicaciones móviles de Google que proporciona servicios backend como autenticación, almacenamiento en la nube, bases de datos en tiempo real y notificaciones push.
Flutter
Framework de desarrollo de aplicaciones multiplataforma creado por Google, que permite crear aplicaciones para iOS y Android con un solo código.
Java
Lenguaje de programación ampliamente utilizado para el desarrollo de aplicaciones empresariales y móviles.
Android Studio
Entorno de desarrollo integrado (IDE) oficial de Google para la creación de aplicaciones Android.
Ionic
Framework de desarrollo de aplicaciones móviles híbridas que utiliza tecnologías web como HTML, CSS y JavaScript, y permite el despliegue en múltiples plataformas como iOS y Android.
React Native
Framework basado en React para el desarrollo de aplicaciones móviles multiplataforma utilizando JavaScript.
Swift
Lenguaje de programación desarrollado por Apple para la creación de aplicaciones en iOS y macOS.

Conocimientos

Estructuras de Datos
Introducción a las estructuras de datos comunes como listas, pilas, colas, árboles y grafos, esenciales para programadores.
Algoritmos
Curso sobre análisis y diseño de algoritmos, que cubre la eficiencia de los algoritmos y cómo mejorar el rendimiento del código.
Patrones de Diseño
Curso sobre patrones de diseño de software que proporcionan soluciones reutilizables a problemas comunes de desarrollo.
REST API
Principios de diseño y desarrollo de APIs RESTful, utilizadas para la interacción entre aplicaciones.
Fundamentos de la Programación
Curso introductorio sobre conceptos básicos de la programación, como variables, estructuras de control, y algoritmos.
CI/CD
Integración y despliegue continuo (CI/CD), prácticas para automatizar la construcción, pruebas y entrega de software.

Competencias sociales

Creatividad
Capacidad para proponer ideas innovadoras y soluciones originales a los problemas.
Resolución de problemas
Capacidad para identificar problemas y desarrollar soluciones eficientes y efectivas.
Adaptabilidad
Capacidad para ajustarse rápidamente a nuevas circunstancias, tecnologías o métodos de trabajo.
