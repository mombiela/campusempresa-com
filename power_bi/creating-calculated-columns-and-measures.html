Introducción

Power BI es una poderosa herramienta de visualización de datos que permite a los usuarios crear informes y paneles informativos. Una de las características clave de Power BI es la capacidad de crear columnas calculadas y medidas, lo que permite un análisis de datos y una elaboración de informes más avanzados. Esta sección te guiará a través del proceso de creación de columnas calculadas y medidas, desde niveles principiantes hasta avanzados.

Columnas Calculadas

¿Qué son las Columnas Calculadas?

  • Definición: Las columnas calculadas son columnas que agregas a tu modelo de datos utilizando fórmulas DAX (Expresiones de Análisis de Datos). Se calculan fila por fila cuando los datos se cargan en el modelo.
  • Casos de Uso: Útiles para agregar nuevos campos de datos derivados de datos existentes, como concatenar nombres y apellidos, o calcular un nuevo campo de fecha.

Creación de una Columna Calculada

  1. Abre Power BI Desktop y carga tu conjunto de datos.
  2. Navega a la Vista de Datos haciendo clic en el ícono de Datos en la barra lateral izquierda.
  3. Selecciona la Tabla donde deseas agregar la columna calculada.
  4. Haz clic en 'Nueva Columna' en la pestaña de Modelado.

Ejemplo: Concatenar Nombres y Apellidos

FullName = [FirstName] & " " & [LastName]
  • Explicación: Esta fórmula DAX concatena las columnas FirstName y LastName con un espacio en el medio para crear una nueva columna FullName.

Ejemplo: Calcular la Edad a partir de la Fecha de Nacimiento

Age = DATEDIFF([Birthdate], TODAY(), YEAR)
  • Explicación: Esta fórmula calcula la edad encontrando la diferencia entre la Birthdate y la fecha de hoy en años.

Medidas

¿Qué son las Medidas?

  • Definición: Las medidas son cálculos utilizados en el análisis de datos que se crean utilizando fórmulas DAX. A diferencia de las columnas calculadas, las medidas se calculan sobre la marcha, según el contexto de los datos en tu informe.
  • Casos de Uso: Ideales para agregaciones, como sumas, promedios, mínimos, máximos, o cálculos más complejos como el crecimiento interanual.

Creación de una Medida

  1. Abre Power BI Desktop y carga tu conjunto de datos.
  2. Navega a la Vista de Informe haciendo clic en el ícono de Informe en la barra lateral izquierda.
  3. Selecciona la Tabla donde deseas agregar la medida.
  4. Haz clic en 'Nueva Medida' en la pestaña de Modelado.

Ejemplo: Ventas Totales

TotalSales = SUM(Sales[SalesAmount])
  • Explicación: Esta medida calcula las ventas totales sumando la columna SalesAmount en la tabla Sales.

Ejemplo: Promedio de Ventas por Cliente

AvgSalesPerCustomer = AVERAGE(Sales[SalesAmount])
  • Explicación: Esta medida calcula el monto promedio de ventas por cliente.

Ejemplo: Crecimiento de Ventas Interanual

YoY Sales Growth = 
VAR CurrentYearSales = CALCULATE(SUM(Sales[SalesAmount]), YEAR(Sales[OrderDate]) = YEAR(TODAY()))
VAR PreviousYearSales = CALCULATE(SUM(Sales[SalesAmount]), YEAR(Sales[OrderDate]) = YEAR(TODAY()) - 1)
RETURN
DIVIDE(CurrentYearSales - PreviousYearSales, PreviousYearSales, 0)
  • Explicación: Esta medida calcula el crecimiento de ventas interanual comparando las ventas del año actual con las del año anterior.

Diferencias Clave entre Columnas Calculadas y Medidas

| Característica | Columnas Calculadas | Medidas | |-----------------------|---------------------------------------------|-----------------------------------------------| | Tiempo de Cálculo | Calculado durante la carga de datos | Calculado sobre la marcha | | Almacenamiento | Almacenado en el modelo de datos | No almacenado, calculado según sea necesario | | Casos de Uso | Cálculos fila por fila | Agregaciones y cálculos basados en el contexto| | Impacto en el Rendimiento | Puede aumentar el tamaño del modelo | Generalmente más eficiente para grandes conjuntos de datos |

Conclusión

Entender cómo crear y usar columnas calculadas y medidas en Power BI es esencial para realizar análisis de datos avanzados y crear informes informativos. Las columnas calculadas son útiles para agregar nuevos campos derivados de datos existentes, mientras que las medidas son poderosas para agregaciones y cálculos basados en el contexto. Al dominar estas características, puedes desbloquear todo el potencial de Power BI y tomar decisiones basadas en datos con mayor confianza.
