Instalación de Power BI Desktop
Descripción general de la interfaz de Power BI ►

Power BI Desktop es una herramienta poderosa para el análisis y la visualización de datos. Esta sección te guiará a través del proceso de instalación de Power BI Desktop en tu computadora. Al final de esta sección, tendrás Power BI Desktop instalado y listo para usar.

Requisitos del Sistema

Antes de instalar Power BI Desktop, asegúrate de que tu sistema cumpla con los siguientes requisitos:

  • Sistema Operativo: Windows 10, Windows 8.1, Windows 8, Windows 7, Windows Server 2012 R2, Windows Server 2012 o Windows Server 2008 R2.
  • Memoria: Al menos 2 GB de RAM, aunque se recomiendan 4 GB o más.
  • Espacio en Disco: Al menos 1 GB de espacio disponible.
  • .NET Framework: .NET 4.5 o posterior.

Descargando Power BI Desktop

Para descargar Power BI Desktop, sigue estos pasos:

  1. Abre tu navegador web y ve a la página de descarga de Power BI Desktop.
  2. Haz clic en el botón Descargar Gratis.
  3. Serás redirigido a la Microsoft Store o a un enlace de descarga directa, dependiendo de tu navegador y configuración del sistema.

Instalando Power BI Desktop

Una vez que hayas descargado el instalador, sigue estos pasos para instalar Power BI Desktop:

  1. Localiza el archivo del instalador descargado (generalmente llamado PBIDesktopSetup.exe) y haz doble clic en él para iniciar el proceso de instalación.
  2. Sigue las instrucciones en pantalla:
    • Pantalla de Bienvenida: Haz clic en Siguiente.
    • Acuerdo de Licencia: Lee los términos, selecciona Acepto los términos del Acuerdo de Licencia y haz clic en Siguiente.
    • Carpeta de Instalación: Elige la carpeta de destino para la instalación o déjala como predeterminada, luego haz clic en Siguiente.
    • Listo para Instalar: Haz clic en Instalar para comenzar la instalación.
  3. Espera a que la instalación se complete. Esto puede tardar unos minutos.
  4. Una vez que la instalación esté completa, haz clic en Finalizar para salir del instalador.

Iniciando Power BI Desktop

Después de la instalación, puedes iniciar Power BI Desktop:

  1. Ve al menú de Inicio y busca Power BI Desktop.
  2. Haz clic en la aplicación Power BI Desktop para abrirla.
  3. La primera vez que abras Power BI Desktop, es posible que se te pida iniciar sesión con tu cuenta de Microsoft. Si no tienes una, puedes crearla de forma gratuita.

Resumen

En esta sección, aprendiste a:

  • Verificar que tu sistema cumpla con los requisitos necesarios para instalar Power BI Desktop.
  • Descargar el instalador de Power BI Desktop desde el sitio web oficial.
  • Instalar Power BI Desktop siguiendo las instrucciones paso a paso.
  • Iniciar Power BI Desktop e iniciar sesión con tu cuenta de Microsoft.

Con Power BI Desktop instalado, ahora estás listo para comenzar a explorar sus características y capacidades. En las próximas secciones, profundizaremos en los conceptos básicos de usar Power BI Desktop para crear potentes visualizaciones de datos e informes.
