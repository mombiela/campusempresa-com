Introducción

Las funciones de Inteligencia de Tiempo en Power BI son esenciales para analizar y manipular datos a lo largo del tiempo. Estas funciones te permiten crear cálculos que son conscientes del tiempo, como totales acumulados del año, promedios móviles y comparaciones en diferentes períodos.

Conceptos Clave

  • Tabla de Fechas: Una tabla que contiene un rango continuo de fechas, lo cual es crucial para los cálculos de inteligencia de tiempo.
  • DAX (Expresiones de Análisis de Datos): El lenguaje de fórmulas utilizado en Power BI para crear cálculos personalizados.

Configuración de una Tabla de Fechas

Antes de usar funciones de inteligencia de tiempo, necesitas una Tabla de Fechas. Esta tabla debe incluir columnas para el año, trimestre, mes, día y otros períodos de tiempo relevantes.

DateTable = 
ADDCOLUMNS (
    CALENDAR (DATE(2020, 1, 1), DATE(2023, 12, 31)),
    "Year", YEAR([Date]),
    "Month", MONTH([Date]),
    "Day", DAY([Date]),
    "Quarter", QUARTER([Date]),
    "YearMonth", FORMAT([Date], "YYYY-MM")
)

Funciones Comunes de Inteligencia de Tiempo

TOTALYTD

Calcula el total acumulado del año para una medida.

TotalSalesYTD = 
TOTALYTD (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

SAMEPERIODLASTYEAR

Devuelve una tabla que contiene una columna de fechas desplazadas un año hacia atrás en el tiempo.

SalesLastYear = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    SAMEPERIODLASTYEAR(DateTable[Date])
)

DATESINPERIOD

Devuelve una tabla que contiene una columna de fechas que comienza con una fecha de inicio especificada y continúa por el número especificado de intervalos.

SalesLast30Days = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    DATESINPERIOD(DateTable[Date], MAX(DateTable[Date]), -30, DAY)
)

PARALLELPERIOD

Devuelve una tabla que contiene una columna de fechas que son paralelas a las fechas en la columna especificada, desplazadas por el número especificado de intervalos.

SalesParallelPeriod = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    PARALLELPERIOD(DateTable[Date], -1, MONTH)
)

Funciones Avanzadas de Inteligencia de Tiempo

DATEADD

Desplaza un conjunto de fechas por un número especificado de intervalos.

SalesDateAdd = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    DATEADD(DateTable[Date], -1, YEAR)
)

CLOSINGBALANCEMONTH

Calcula el saldo de cierre para una medida al final del mes.

ClosingBalanceMonth = 
CLOSINGBALANCEMONTH (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

OPENINGBALANCEYEAR

Calcula el saldo de apertura para una medida al comienzo del año.

OpeningBalanceYear = 
OPENINGBALANCEYEAR (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

Ejemplos Prácticos

Comparación de Ventas Acumuladas del Año

Compara las ventas acumuladas del año del año actual con el año anterior.

YTD_Sales_CurrentYear = 
TOTALYTD (
    SUM(Sales[SalesAmount]),
    DateTable[Date]
)

YTD_Sales_PreviousYear = 
CALCULATE (
    TOTALYTD (
        SUM(Sales[SalesAmount]),
        DateTable[Date]
    ),
    SAMEPERIODLASTYEAR(DateTable[Date])
)

Ventas Acumuladas de los Últimos 12 Meses

Calcula el total acumulado de ventas de los últimos 12 meses.

Rolling12MonthSales = 
CALCULATE (
    SUM(Sales[SalesAmount]),
    DATESINPERIOD(DateTable[Date], MAX(DateTable[Date]), -12, MONTH)
)

Conclusión

Las funciones de Inteligencia de Tiempo en Power BI son herramientas poderosas para analizar datos a lo largo del tiempo. Al configurar una Tabla de Fechas adecuada y usar funciones como TOTALYTD, SAMEPERIODLASTYEAR y DATESINPERIOD, puedes crear cálculos perspicaces basados en el tiempo. A medida que avances, funciones como DATEADD, CLOSINGBALANCEMONTH y OPENINGBALANCEYEAR te ayudarán a realizar análisis más complejos basados en el tiempo. Entender y dominar estas funciones mejorará significativamente tus capacidades de análisis de datos en Power BI.
