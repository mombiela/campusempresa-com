Introducción a Power BI Mobile
Visualizando informes en dispositivos móviles ►

Descripción general

Power BI Mobile permite a los usuarios acceder e interactuar con sus informes y paneles de Power BI en dispositivos móviles. Esta sección cubrirá los conceptos básicos de Power BI Mobile, sus características y cómo comenzar a usarlo.

Conceptos clave

  • Aplicación móvil de Power BI: Una aplicación móvil disponible para dispositivos iOS, Android y Windows que permite a los usuarios ver e interactuar con informes y paneles de Power BI.
  • Informes y paneles móviles: Versiones optimizadas de informes y paneles de Power BI diseñadas para visualización móvil.
  • Interactividad: Características que permiten a los usuarios interactuar con los datos, como filtros, desgloses y gestos táctiles.

Comenzando con Power BI Mobile

Instalando la aplicación móvil de Power BI

  1. iOS: Descárgala desde la Apple App Store.
  2. Android: Descárgala desde la Google Play Store.
  3. Windows: Descárgala desde la Microsoft Store.

Iniciando sesión

  • Abre la aplicación móvil de Power BI.
  • Inicia sesión con las credenciales de tu cuenta de Power BI.

Navegando por la aplicación

  • Pantalla de inicio: Muestra informes y paneles recientes.
  • Favoritos: Accede rápidamente a tus informes y paneles favoritos.
  • Compartido conmigo: Ve informes y paneles compartidos por otros.
  • Espacios de trabajo: Accede a diferentes espacios de trabajo de los que formas parte.

Visualización e interacción con informes

Visualización de informes

  • Abre un informe desde la pantalla de inicio o el espacio de trabajo.
  • Desliza hacia la izquierda o derecha para navegar entre las páginas del informe.

Interacción con los datos

  • Filtros: Toca los íconos de filtro para aplicar filtros.
  • Desglose: Toca los puntos de datos para desglosar datos más detallados.
  • Gestos táctiles: Usa gestos de pellizco para hacer zoom y deslizar para navegar y explorar datos.

Ejemplo: Aplicar un filtro

1. Abre un informe.
2. Toca el ícono de filtro en la esquina superior derecha.
3. Selecciona los criterios del filtro (por ejemplo, Rango de fechas, Categoría de producto).
4. El informe se actualizará para reflejar el filtro seleccionado.

Personalización de informes móviles

Creación de informes optimizados para móviles

  • Usa Power BI Desktop para crear diseños optimizados para móviles.
  • Cambia a la vista de diseño móvil en Power BI Desktop.
  • Organiza los elementos visuales para que se ajusten a la pantalla móvil.

Ejemplo: Crear un diseño móvil

1. Abre tu informe en Power BI Desktop.
2. Haz clic en la pestaña "Vista".
3. Selecciona "Diseño para teléfono".
4. Arrastra y suelta los elementos visuales para organizarlos para la visualización móvil.
5. Guarda y publica el informe.

Notificaciones y alertas

  • Alertas de datos: Configura alertas para recibir notificaciones cuando los datos cambien.
  • Notificaciones push: Recibe notificaciones en tu dispositivo móvil para actualizaciones importantes.

Configuración de una alerta de datos

1. Abre un panel.
2. Toca un mosaico para el que deseas configurar una alerta.
3. Toca los puntos suspensivos (...) y selecciona "Administrar alertas".
4. Establece las condiciones para la alerta (por ejemplo, cuando las ventas superen una cierta cantidad).
5. Guarda la alerta.

Conclusión

Power BI Mobile proporciona una forma conveniente de acceder e interactuar con tus informes y paneles de Power BI en cualquier lugar. Al comprender cómo navegar por la aplicación, visualizar e interactuar con informes y personalizar diseños móviles, puedes aprovechar al máximo las capacidades móviles de Power BI. Ya sea que estés revisando datos sobre la marcha o configurando alertas para métricas críticas, Power BI Mobile te asegura estar conectado con tus datos en cualquier momento y lugar.
