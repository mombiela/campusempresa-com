Introducción

La limpieza y transformación de datos son pasos cruciales en el análisis y reporte de datos. En Power BI, estos procesos aseguran que tus datos sean precisos, consistentes y estén listos para el análisis. Esta sección te guiará desde las técnicas básicas hasta las avanzadas de limpieza y transformación de datos en Power BI.

Comenzando con el Editor de Power Query

El Editor de Power Query es la herramienta principal en Power BI para la limpieza y transformación de datos.

Conceptos Clave

  • Editor de Power Query: Una herramienta dentro de Power BI utilizada para conectar, combinar y refinar datos de diferentes fuentes.
  • Consulta: Un conjunto de pasos que Power BI utiliza para transformar datos.
  • Pasos Aplicados: Cada transformación aplicada a los datos se registra como un paso.

Ejemplo: Cargando Datos en el Editor de Power Query

1. Abre Power BI Desktop.
2. Haz clic en 'Inicio' > 'Obtener Datos'.
3. Selecciona una fuente de datos (por ejemplo, Excel, SQL Server).
4. Haz clic en 'Cargar' para importar los datos.
5. Haz clic en 'Transformar Datos' para abrir el Editor de Power Query.

Técnicas Básicas de Limpieza de Datos

Eliminando Duplicados

Los registros duplicados pueden sesgar tu análisis. Eliminarlos es esencial.

1. En el Editor de Power Query, selecciona la(s) columna(s) donde podrían existir duplicados.
2. Ve a 'Inicio' > 'Eliminar Filas' > 'Eliminar Duplicados'.

Manejo de Valores Faltantes

Los valores faltantes pueden ser reemplazados, eliminados o llenados según el contexto.

1. Selecciona la columna con valores faltantes.
2. Ve a 'Transformar' > 'Reemplazar Valores' para reemplazarlos con un valor específico.
3. Alternativamente, usa 'Eliminar Filas' para eliminar filas con valores faltantes.

Cambiando Tipos de Datos

Asegurarse de que cada columna tenga el tipo de datos correcto es crucial para un análisis preciso.

1. Selecciona la columna que deseas cambiar.
2. Ve a 'Transformar' > 'Tipo de Datos' y selecciona el tipo apropiado (por ejemplo, Texto, Número, Fecha).

Técnicas Intermedias de Transformación de Datos

Combinando Consultas

Combinar datos de múltiples fuentes puede proporcionar una vista más completa.

1. En el Editor de Power Query, ve a 'Inicio' > 'Combinar Consultas'.
2. Selecciona las consultas que deseas combinar.
3. Elige las columnas coincidentes y el tipo de unión (por ejemplo, Interna, Externa).
4. Haz clic en 'Aceptar' para combinar las consultas.

Pivoteando y Despivoteando Columnas

Estas técnicas ayudan a reestructurar tus datos para un mejor análisis.

  • Pivoteando: Convierte filas en columnas.
  • Despivoteando: Convierte columnas en filas.
1. Selecciona las columnas que deseas pivotear o despivotear.
2. Ve a 'Transformar' > 'Columna Pivote' o 'Columnas Despivote'.

Agrupando Datos

Agrupar datos te permite agregar valores basados en criterios específicos.

1. Selecciona la(s) columna(s) por las que deseas agrupar.
2. Ve a 'Transformar' > 'Agrupar Por'.
3. Elige la función de agregación (por ejemplo, Suma, Promedio).
4. Haz clic en 'Aceptar' para aplicar la agrupación.

Técnicas Avanzadas de Transformación de Datos

Usando el Lenguaje M

M es el lenguaje de fórmulas detrás de Power Query. Permite transformaciones más complejas.

1. En el Editor de Power Query, ve a 'Editor Avanzado'.
2. Escribe o modifica el código M para realizar transformaciones avanzadas.

Ejemplo:

let Source = Excel.Workbook(File.Contents("C:\Data\SalesData.xlsx"), null, true), Sales_Sheet = Source{[Name="Sales"]}[Content], ChangedType = Table.TransformColumnTypes(Sales_Sheet,{{"Date", type date}, {"Sales", Int64.Type}}) in ChangedType

Funciones Personalizadas


Crear funciones personalizadas puede ayudar a reutilizar lógica compleja en múltiples consultas.


1. Ve a 'Inicio' > 'Editor Avanzado'.
2. Define una función usando el lenguaje M.

Ejemplo:
let
AddTax = (price as number) as number =>
let
taxRate = 0.1,
finalPrice = price * (1 + taxRate)
in
finalPrice
in
AddTax


Parametrizando Consultas


Los parámetros permiten consultas dinámicas y reutilizables.


1. Ve a 'Inicio' > 'Administrar Parámetros' > 'Nuevo Parámetro'.
2. Define el parámetro (por ejemplo, nombre, tipo, valor predeterminado).
3. Usa el parámetro en tus consultas.

Ejemplo:
let
Source = Sql.Database("ServerName", "DatabaseName"),
FilteredData = Table.SelectRows(Source, each [Date] >= StartDate and [Date] <= EndDate)
in
FilteredData


Conclusión


La limpieza y transformación de datos son habilidades esenciales para cualquier usuario de Power BI. Al dominar estas técnicas, puedes asegurarte de que tus datos sean precisos, consistentes y estén listos para un análisis perspicaz. Desde tareas básicas como eliminar duplicados y manejar valores faltantes hasta técnicas avanzadas como usar el lenguaje M y funciones personalizadas, Power BI proporciona herramientas poderosas para transformar tus datos de manera efectiva.
