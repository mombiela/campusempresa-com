Introducción a Firebase e Ionic

  • Firebase: Una plataforma desarrollada por Google para crear aplicaciones móviles y web. Proporciona una variedad de herramientas y servicios para ayudar a los desarrolladores a construir aplicaciones de alta calidad, incluyendo bases de datos en tiempo real, autenticación y almacenamiento en la nube.
  • Ionic: Un popular framework de código abierto para construir aplicaciones móviles multiplataforma utilizando tecnologías web como HTML, CSS y JavaScript.

Configuración de Firebase en un Proyecto Ionic

Prerrequisitos

  • Node.js y npm instalados.
  • Ionic CLI instalado.
  • Un conocimiento básico de Ionic y Angular.

Creando un Nuevo Proyecto Ionic

ionic start myFirebaseApp blank
cd myFirebaseApp

Instalando Firebase y AngularFire

npm install firebase @angular/fire

Configurando Firebase

  1. Ve a la Consola de Firebase.
  2. Crea un nuevo proyecto.
  3. Añade una aplicación web a tu proyecto.
  4. Copia el objeto de configuración de Firebase.

Añadiendo la Configuración de Firebase a Ionic

En src/environments/environment.ts, añade tu configuración de Firebase:

export const environment = {
  production: false,
  firebaseConfig: {
    apiKey: "TU_API_KEY",
    authDomain: "TU_AUTH_DOMAIN",
    projectId: "TU_PROJECT_ID",
    storageBucket: "TU_STORAGE_BUCKET",
    messagingSenderId: "TU_MESSAGING_SENDER_ID",
    appId: "TU_APP_ID"
  }
};

Inicializando Firebase en el Módulo de la Aplicación

En src/app/app.module.ts, importa e inicializa Firebase:

import { AngularFireModule } from '@angular/fire';
import { AngularFireAuthModule } from '@angular/fire/auth';
import { AngularFirestoreModule } from '@angular/fire/firestore';
import { environment } from '../environments/environment';

@NgModule({
  declarations: [AppComponent],
  imports: [
    BrowserModule,
    IonicModule.forRoot(),
    AppRoutingModule,
    AngularFireModule.initializeApp(environment.firebaseConfig),
    AngularFireAuthModule,
    AngularFirestoreModule
  ],
  bootstrap: [AppComponent]
})
export class AppModule { }

Autenticación con Firebase

Configuración de la Autenticación

  1. En la Consola de Firebase, ve a la sección de Autenticación.
  2. Habilita los métodos de inicio de sesión que deseas usar (por ejemplo, Email/Contraseña, Google).

Implementando Autenticación con Email/Contraseña

Componente de Registro

import { Component } from '@angular/core';
import { AngularFireAuth } from '@angular/fire/auth';

@Component({
  selector: 'app-sign-up',
  template: `
    <form (submit)="signUp(email, password)">
      <input type="email" [(ngModel)]="email" placeholder="Email">
      <input type="password" [(ngModel)]="password" placeholder="Password">
      <button type="submit">Sign Up</button>
    </form>
  `
})
export class SignUpComponent {
  email: string;
  password: string;

  constructor(private afAuth: AngularFireAuth) {}

  signUp(email: string, password: string) {
    this.afAuth.createUserWithEmailAndPassword(email, password)
      .then(user => console.log(user))
      .catch(error => console.error(error));
  }
}

Componente de Inicio de Sesión

Firestore de Firebase

Configuración de Firestore

  1. En la Consola de Firebase, ve a la sección de Base de Datos de Firestore.
  2. Crea una base de datos.

Añadiendo Datos a Firestore

import { Component } from '@angular/core';
import { AngularFirestore } from '@angular/fire/firestore';

@Component({
  selector: 'app-add-data',
  template: `
    <form (submit)="addData()">
      <input type="text" [(ngModel)]="data" placeholder="Data">
      <button type="submit">Add Data</button>
    </form>
  `
})
export class AddDataComponent {
  data: string;

  constructor(private firestore: AngularFirestore) {}

  addData() {
    this.firestore.collection('items').add({ data: this.data })
      .then(() => console.log('Data added'))
      .catch(error => console.error(error));
  }
}

Leyendo Datos de Firestore

import { Component, OnInit } from '@angular/core';
import { AngularFirestore } from '@angular/fire/firestore';
import { Observable } from 'rxjs';

@Component({
  selector: 'app-read-data',
  template: `
    <ul>
      <li *ngFor="let item of items | async">{{ item.data }}</li>
    </ul>
  `
})
export class ReadDataComponent implements OnInit {
  items: Observable<any[]>;

  constructor(private firestore: AngularFirestore) {}

  ngOnInit() {
    this.items = this.firestore.collection('items').valueChanges();
  }
}

Almacenamiento de Firebase

Configuración del Almacenamiento

  1. En la Consola de Firebase, ve a la sección de Almacenamiento.
  2. Crea un bucket de almacenamiento.

Subiendo Archivos al Almacenamiento

import { Component } from '@angular/core';
import { AngularFireStorage } from '@angular/fire/storage';
import { finalize } from 'rxjs/operators';

@Component({
  selector: 'app-upload-file',
  template: `
    <input type="file" (change)="uploadFile($event)">
  `
})
export class UploadFileComponent {
  constructor(private storage: AngularFireStorage) {}

  uploadFile(event) {
    const file = event.target.files[0];
    const filePath = `uploads/${file.name}`;
    const fileRef = this.storage.ref(filePath);
    const task = this.storage.upload(filePath, file);

    task.snapshotChanges().pipe(
      finalize(() => {
        fileRef.getDownloadURL().subscribe(url => console.log('File URL:', url));
      })
    ).subscribe();
  }
}
