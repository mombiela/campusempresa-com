Aserciones Comunes en JUnit
Pruebas de Excepciones ►

Introducción

Las aserciones son una parte fundamental de las pruebas unitarias en JUnit. Te permiten verificar que tu código se comporte como se espera. En esta sección, cubriremos las aserciones más comunes utilizadas en JUnit, proporcionando explicaciones claras y ejemplos para ayudarte a entender cómo usarlas de manera efectiva.

Conceptos Clave

  • Aserciones: Declaraciones que verifican si una condición es verdadera. Si la condición es falsa, la aserción falla y el caso de prueba se marca como fallido.
  • JUnit: Un popular marco de pruebas para Java que proporciona varios métodos de aserción para facilitar las pruebas unitarias.

Aserciones Comunes

assertEquals

El método assertEquals verifica si dos valores son iguales. Es una de las aserciones más comúnmente utilizadas en JUnit.

import static org.junit.jupiter.api.Assertions.assertEquals;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testAddition() {
        int result = 2 + 3;
        assertEquals(5, result, "2 + 3 debería ser igual a 5");
    }
}

En este ejemplo, assertEquals verifica si el resultado de 2 + 3 es igual a 5. Si no lo es, la prueba fallará con el mensaje "2 + 3 debería ser igual a 5".

assertNotEquals

El método assertNotEquals verifica si dos valores no son iguales.

import static org.junit.jupiter.api.Assertions.assertNotEquals;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testSubtraction() {
        int result = 5 - 3;
        assertNotEquals(1, result, "5 - 3 no debería ser igual a 1");
    }
}

Aquí, assertNotEquals asegura que 5 - 3 no sea igual a 1. Si lo es, la prueba fallará.

assertTrue

El método assertTrue verifica si una condición es verdadera.

import static org.junit.jupiter.api.Assertions.assertTrue;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testIsPositive() {
        int number = 5;
        assertTrue(number > 0, "El número debería ser positivo");
    }
}

En este caso, assertTrue verifica que el número sea positivo. Si el número no es mayor que 0, la prueba fallará.

assertFalse

El método assertFalse verifica si una condición es falsa.

import static org.junit.jupiter.api.Assertions.assertFalse;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testIsNegative() {
        int number = -5;
        assertFalse(number >= 0, "El número debería ser negativo");
    }
}

Aquí, assertFalse asegura que el número no sea mayor o igual a 0. Si lo es, la prueba fallará.

assertNull

El método assertNull verifica si un objeto es nulo.

import static org.junit.jupiter.api.Assertions.assertNull;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testObjectIsNull() {
        Object obj = null;
        assertNull(obj, "El objeto debería ser nulo");
    }
}

En este ejemplo, assertNull verifica que el objeto sea nulo. Si no lo es, la prueba fallará.

assertNotNull

El método assertNotNull verifica si un objeto no es nulo.

import static org.junit.jupiter.api.Assertions.assertNotNull;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testObjectIsNotNull() {
        Object obj = new Object();
        assertNotNull(obj, "El objeto no debería ser nulo");
    }
}

Aquí, assertNotNull asegura que el objeto no sea nulo. Si es nulo, la prueba fallará.

assertArrayEquals

El método assertArrayEquals verifica si dos arreglos son iguales.

import static org.junit.jupiter.api.Assertions.assertArrayEquals;
import org.junit.jupiter.api.Test;

public class ExampleTest {

    @Test
    public void testArrayEquality() {
        int[] expected = {1, 2, 3};
        int[] actual = {1, 2, 3};
        assertArrayEquals(expected, actual, "Los arreglos deberían ser iguales");
    }
}

En este caso, assertArrayEquals verifica que los dos arreglos sean iguales. Si no lo son, la prueba fallará.

Resumen

Las aserciones son esenciales para verificar la corrección de tu código en las pruebas de JUnit. Las aserciones más comunes incluyen assertEquals, assertNotEquals, assertTrue, assertFalse, assertNull, assertNotNull y assertArrayEquals. Cada una de estas aserciones tiene un propósito específico y ayuda a asegurar que tu código se comporte como se espera. Al entender y usar estas aserciones de manera efectiva, puedes escribir pruebas más robustas y confiables.
