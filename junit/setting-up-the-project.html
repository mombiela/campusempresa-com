Configuración del Proyecto
Implementación de Pruebas ►

En esta sección, cubriremos cómo configurar un proyecto para usar JUnit, un popular marco de pruebas para Java. Esta configuración es crucial tanto para principiantes como para usuarios avanzados para asegurar un flujo de trabajo fluido al escribir y ejecutar pruebas.

Prerrequisitos

Antes de comenzar, asegúrate de tener lo siguiente instalado en tu máquina:

  • Java Development Kit (JDK) 8 o superior
  • Un Entorno de Desarrollo Integrado (IDE) como IntelliJ IDEA, Eclipse o NetBeans
  • Maven o Gradle (opcional pero recomendado para la gestión de dependencias)

Creación de un Nuevo Proyecto

Usando IntelliJ IDEA

  1. Abre IntelliJ IDEA y haz clic en New Project.
  2. Selecciona Java y haz clic en Next.
  3. Nombra tu proyecto y elige una ubicación para guardarlo.
  4. Selecciona el JDK que tienes instalado y haz clic en Next.
  5. Finaliza la configuración haciendo clic en Finish.

Usando Eclipse

  1. Abre Eclipse y ve a File > New > Java Project.
  2. Ingresa el nombre del proyecto y haz clic en Next.
  3. Configura el JRE (Entorno de Ejecución de Java) si es necesario.
  4. Haz clic en Finish para crear el proyecto.

Agregar JUnit a Tu Proyecto

Usando Maven

  1. Abre el archivo pom.xml en tu proyecto.

  2. Agrega la siguiente dependencia dentro de la etiqueta <dependencies>:

    <dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>

  3. Guarda el archivo pom.xml y deja que Maven descargue las dependencias.

Usando Gradle

  1. Abre el archivo build.gradle en tu proyecto.

  2. Agrega la siguiente dependencia dentro del bloque dependencies:

    testImplementation 'junit:junit:4.13.2'

  3. Guarda el archivo build.gradle y deja que Gradle descargue las dependencias.

Agregar Manualmente el JAR de JUnit

  1. Descarga el archivo JAR de JUnit desde el sitio web oficial de JUnit.
  2. Agrega el archivo JAR al classpath de tu proyecto:
    • En IntelliJ IDEA: Haz clic derecho en tu proyecto > Open Module Settings > Libraries > + > Agrega el archivo JAR.
    • En Eclipse: Haz clic derecho en tu proyecto > Build Path > Add External Archives > Agrega el archivo JAR.

Verificación de la Configuración

Para verificar que JUnit está correctamente configurado, crea un caso de prueba simple.

Ejemplo de Caso de Prueba

  1. Crea una nueva clase Java llamada Calculator:

    public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}

  2. Crea una clase de prueba llamada CalculatorTest:

    import org.junit.Test;
import static org.junit.Assert.assertEquals;

public class CalculatorTest {
    @Test
    public void testAdd() {
        Calculator calculator = new Calculator();
        int result = calculator.add(2, 3);
        assertEquals(5, result);
    }
}

  3. Ejecuta la prueba:

    • En IntelliJ IDEA: Haz clic derecho en la clase CalculatorTest > Run CalculatorTest.
    • En Eclipse: Haz clic derecho en la clase CalculatorTest > Run As > JUnit Test.

Si la prueba pasa, ¡tu configuración de JUnit está completa!

Conclusión

Configurar un proyecto para JUnit implica crear un nuevo proyecto Java, agregar JUnit como una dependencia y verificar la configuración con un caso de prueba simple. Ya sea que uses Maven, Gradle o agregues manualmente el archivo JAR, el proceso es sencillo y asegura que estés listo para escribir y ejecutar pruebas de manera eficiente. Esta configuración básica servirá como base para temas más avanzados en pruebas con JUnit.
