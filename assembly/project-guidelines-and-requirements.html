Introducción

En esta sección, delinearemos las directrices y requisitos para los proyectos que llevarás a cabo a lo largo de este curso de programación en Assembly. Estos proyectos están diseñados para reforzar los conceptos que aprendas en cada módulo, desde niveles principiantes hasta avanzados.

Directrices Generales

  • Entender el Enunciado del Problema: Lee y comprende cuidadosamente el enunciado del problema antes de comenzar a codificar.
  • Planificar tu Enfoque: Esboza tu enfoque y desglosa el problema en tareas más pequeñas y manejables.
  • Escribir Código Limpio: Asegúrate de que tu código esté bien comentado y siga las mejores prácticas para la legibilidad y el mantenimiento.
  • Probar Exhaustivamente: Prueba tu código con varios inputs para asegurarte de que maneja todos los casos extremos y funciona como se espera.
  • Entregar a Tiempo: Adhiérete a los plazos de entrega de los proyectos para mantenerte al día con el calendario del curso.

Requisitos del Proyecto por Nivel

Nivel Principiante

En el nivel principiante, los proyectos se centrarán en conceptos básicos del lenguaje Assembly y programas simples.

  • Sintaxis Básica e Instrucciones: Escribe programas que demuestren el uso de instrucciones básicas de Assembly como MOV, ADD, SUB y JMP.
  • Operaciones Aritméticas Simples: Implementa programas para realizar operaciones aritméticas básicas como suma, resta, multiplicación y división.
  • Flujo de Control: Crea programas que usen saltos condicionales (JE, JNE, JG, JL) para controlar el flujo de ejecución.

Proyecto de Ejemplo: Calculadora Simple

section .data
    num1 db 10
    num2 db 5
    result db 0

section .text
    global _start

_start:
    ; Cargar números en registros
    mov al, [num1]
    mov bl, [num2]

    ; Realizar la suma
    add al, bl
    mov [result], al

    ; Salir del programa
    mov eax, 1
    int 0x80
  • Explicación: Este programa carga dos números en registros, realiza una suma, almacena el resultado y luego sale.

Nivel Intermedio

Los proyectos intermedios involucrarán operaciones más complejas y el uso de funciones y bucles.

  • Funciones y Procedimientos: Escribe programas que usen funciones para modularizar el código.
  • Bucles: Implementa bucles usando LOOP, JMP y saltos condicionales para realizar tareas repetitivas.
  • Estructuras de Datos: Trabaja con arrays y cadenas en Assembly.

Proyecto de Ejemplo: Inversión de Cadenas

section .data
    str db 'Hello, World!', 0
    len equ $ - str

section .bss
    reversed resb len

section .text
    global _start

_start:
    ; Inicializar punteros
    mov esi, str
    mov edi, reversed
    add esi, len - 1

reverse_loop:
    ; Copiar carácter del final al inicio
    mov al, [esi]
    mov [edi], al
    dec esi
    inc edi
    cmp esi, str - 1
    jne reverse_loop

    ; Salir del programa
    mov eax, 1
    int 0x80
  • Explicación: Este programa invierte una cadena copiando caracteres desde el final de la cadena original hasta el inicio de una nueva cadena.

Nivel Avanzado

Los proyectos avanzados requerirán un entendimiento profundo del lenguaje Assembly y la programación a nivel de sistema.

  • Llamadas al Sistema: Usa llamadas al sistema para interactuar con el sistema operativo.
  • Optimización: Escribe código optimizado para aplicaciones críticas en rendimiento.
  • Algoritmos Complejos: Implementa algoritmos complejos como ordenamiento, búsqueda y encriptación.

Proyecto de Ejemplo: Ordenamiento Burbuja

section .data
    array db 5, 3, 8, 4, 2
    len equ $ - array

section .text
    global _start

_start:
    mov ecx, len

outer_loop:
    dec ecx
    mov esi, 0

inner_loop:
    mov al, [array + esi]
    mov bl, [array + esi + 1]
    cmp al, bl
    jle no_swap

    ; Intercambiar elementos
    mov [array + esi], bl
    mov [array + esi + 1], al

no_swap:
    inc esi
    cmp esi, ecx
    jl inner_loop

    cmp ecx, 1
    jg outer_loop

    ; Salir del programa
    mov eax, 1
    int 0x80
  • Explicación: Este programa ordena un array usando el algoritmo de Ordenamiento Burbuja intercambiando repetidamente elementos adyacentes si están en el orden incorrecto.

Conclusión

Siguiendo estas directrices y cumpliendo con los requisitos del proyecto, obtendrás una comprensión sólida de la programación en lenguaje Assembly. Cada proyecto está diseñado para construir sobre el anterior, aumentando gradualmente en complejidad y ayudándote a desarrollar las habilidades necesarias para escribir código Assembly eficiente y efectivo. Recuerda planificar tu enfoque, escribir código limpio y probar exhaustivamente para asegurar el éxito en tus proyectos.
