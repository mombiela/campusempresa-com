ResultSet y Manipulación de Datos en Java
Transacciones ►

Introducción

En esta sección, exploraremos cómo trabajar con ResultSet en Java para manipular datos recuperados de una base de datos. Cubriremos los conceptos básicos de ResultSet, cómo navegar a través de él y cómo actualizar datos usando el mismo. Este tema es crucial para cualquiera que desee construir aplicaciones Java robustas que interactúen con bases de datos.

Entendiendo ResultSet

Un ResultSet es una tabla de datos que representa el resultado de una consulta a una base de datos. Se genera al ejecutar una declaración que consulta la base de datos.

Conceptos Clave

  • Interfaz ResultSet: La interfaz ResultSet proporciona métodos para recuperar y manipular los datos en el conjunto de resultados.
  • Cursor: El ResultSet mantiene un cursor que apunta a su fila actual de datos.
  • Tipos de ResultSet:
    • TYPE_FORWARD_ONLY: El cursor solo puede moverse hacia adelante.
    • TYPE_SCROLL_INSENSITIVE: El cursor puede moverse tanto hacia adelante como hacia atrás, pero el conjunto de resultados no es sensible a los cambios realizados por otros.
    • TYPE_SCROLL_SENSITIVE: El cursor puede moverse tanto hacia adelante como hacia atrás, y el conjunto de resultados es sensible a los cambios realizados por otros.

Creando un ResultSet

Para crear un ResultSet, necesitas ejecutar una consulta usando un Statement o PreparedStatement.

Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/mydatabase", "user", "password");
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM employees");

Navegando ResultSet

Puedes navegar a través de las filas de un ResultSet usando varios métodos.

Métodos Clave

  • next(): Mueve el cursor a la siguiente fila.
  • previous(): Mueve el cursor a la fila anterior (solo para ResultSet desplazable).
  • first(): Mueve el cursor a la primera fila.
  • last(): Mueve el cursor a la última fila.
  • absolute(int row): Mueve el cursor al número de fila especificado.

Ejemplo

while (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");
    System.out.println("ID: " + id + ", Nombre: " + name);
}

Recuperando Datos de ResultSet

Puedes recuperar datos de la fila actual de un ResultSet usando métodos getter.

Métodos Clave

  • getInt(String columnLabel): Recupera un valor entero.
  • getString(String columnLabel): Recupera un valor de cadena.
  • getDouble(String columnLabel): Recupera un valor doble.
  • getDate(String columnLabel): Recupera un valor de fecha.

Ejemplo

if (rs.next()) {
    int id = rs.getInt("id");
    String name = rs.getString("name");
    double salary = rs.getDouble("salary");
    Date hireDate = rs.getDate("hire_date");
    System.out.println("ID: " + id + ", Nombre: " + name + ", Salario: " + salary + ", Fecha de Contratación: " + hireDate);
}

Actualizando Datos con ResultSet

Un ResultSet también puede ser usado para actualizar datos en la base de datos.

Conceptos Clave

  • Tipos de Concurrencia:
    • CONCUR_READ_ONLY: El ResultSet no puede ser actualizado.
    • CONCUR_UPDATABLE: El ResultSet puede ser actualizado.

Ejemplo

Para crear un ResultSet actualizable, necesitas especificar el tipo de concurrencia al crear el Statement.

Statement stmt = conn.createStatement(ResultSet.TYPE_SCROLL_INSENSITIVE, ResultSet.CONCUR_UPDATABLE);
ResultSet rs = stmt.executeQuery("SELECT * FROM employees");

// Moverse a la fila que deseas actualizar
if (rs.next()) {
    rs.updateString("name", "John Doe");
    rs.updateRow();
}

Insertando Datos

También puedes insertar nuevas filas en la base de datos usando ResultSet.

rs.moveToInsertRow();
rs.updateInt("id", 101);
rs.updateString("name", "Jane Doe");
rs.updateDouble("salary", 75000);
rs.updateDate("hire_date", java.sql.Date.valueOf("2023-01-01"));
rs.insertRow();

Eliminando Datos

Para eliminar una fila de la base de datos usando ResultSet:

if (rs.next()) {
    rs.deleteRow();
}

Conclusión

En esta sección, hemos cubierto los conceptos básicos de trabajar con ResultSet en Java. Exploramos cómo crear, navegar, recuperar, actualizar, insertar y eliminar datos usando ResultSet. Entender estos conceptos es esencial para una manipulación efectiva de datos en aplicaciones Java que interactúan con bases de datos. Practica estas técnicas para volverte competente en el manejo de operaciones de bases de datos en Java.
