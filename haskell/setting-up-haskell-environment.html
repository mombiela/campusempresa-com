Configuración del Entorno Haskell
Sintaxis Básica y Hola Mundo ►

Configurar tu entorno Haskell es el primer paso para aprender y desarrollar programas en Haskell. Esta sección te guiará a través del proceso de instalación y configuración de las herramientas necesarias para comenzar con Haskell.

Instalación de GHC (Glasgow Haskell Compiler)

El Glasgow Haskell Compiler (GHC) es el compilador de Haskell más utilizado. Es esencial para compilar y ejecutar programas en Haskell.

Pasos para Instalar GHC

  1. Descargar GHC:

    • Visita la página de descarga de GHC.
    • Elige la versión apropiada para tu sistema operativo (Windows, macOS, Linux).

  2. Instalar GHC:

    • Windows: Ejecuta el instalador y sigue las instrucciones en pantalla.
    • macOS: Usa Homebrew para instalar GHC ejecutando el comando: 
      brew install ghc
    • Linux: Usa tu gestor de paquetes. Por ejemplo, en Ubuntu: 
      sudo apt-get install ghc

  3. Verificar la Instalación:

    • Abre tu terminal o símbolo del sistema.
    • Ejecuta el comando: 
      ghc --version
    • Deberías ver la versión de GHC que instalaste.

Instalación de Stack

Stack es una herramienta de construcción para Haskell que simplifica el proceso de gestión de dependencias y construcción de proyectos.

Pasos para Instalar Stack

  1. Descargar Stack:

    • Visita la página de instalación de Stack.
    • Sigue las instrucciones para tu sistema operativo.

  2. Instalar Stack:

    • Windows: Ejecuta el instalador y sigue las instrucciones en pantalla.
    • macOS: Usa Homebrew para instalar Stack ejecutando el comando: 
      brew install haskell-stack
    • Linux: Usa tu gestor de paquetes. Por ejemplo, en Ubuntu: 
      sudo apt-get install haskell-stack

  3. Verificar la Instalación:

    • Abre tu terminal o símbolo del sistema.
    • Ejecuta el comando: 
      stack --version
    • Deberías ver la versión de Stack que instalaste.

Configuración de tu Primer Proyecto Haskell

Ahora que tienes GHC y Stack instalados, puedes configurar tu primer proyecto Haskell.

Pasos para Crear un Nuevo Proyecto

  1. Crear un Nuevo Proyecto:

    • Abre tu terminal o símbolo del sistema.
    • Ejecuta el comando: 
      stack new my-first-haskell-project
    • Este comando crea un nuevo directorio llamado my-first-haskell-project con una estructura básica de proyecto Haskell.

  2. Navegar al Directorio del Proyecto:

    • Ejecuta el comando: 
      cd my-first-haskell-project

  3. Construir el Proyecto:

    • Ejecuta el comando: 
      stack build
    • Este comando descarga las dependencias necesarias y compila el proyecto.

  4. Ejecutar el Proyecto:

    • Ejecuta el comando: 
      stack exec my-first-haskell-project-exe
    • Deberías ver la salida del programa Haskell por defecto.

Código de Ejemplo

Aquí está el código Haskell por defecto que viene con el nuevo proyecto:

module Main where

main :: IO ()
main = putStrLn "Hello, Haskell!"
  • Explicación:
    • module Main where: Define el módulo principal del programa.
    • main :: IO (): Declara el tipo de la función main, que realiza operaciones de entrada/salida.
    • main = putStrLn "Hello, Haskell!": La función main imprime "Hello, Haskell!" en la consola.

Conclusión

Configurar el entorno Haskell implica instalar GHC y Stack, que son herramientas esenciales para el desarrollo en Haskell. Siguiendo los pasos descritos en esta sección, ahora deberías tener un entorno Haskell funcional y un proyecto básico para comenzar a experimentar con el código Haskell. Esta configuración servirá como la base para tu viaje desde principiante hasta programador avanzado en Haskell.
