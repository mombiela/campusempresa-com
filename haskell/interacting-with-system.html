Interactuando con el Sistema
Manejo de Excepciones ►

En esta sección, exploraremos cómo los programas en Haskell pueden interactuar con el sistema. Esto incluye leer y escribir archivos, ejecutar comandos del sistema y manejar argumentos de línea de comandos. Estas capacidades son esenciales para crear aplicaciones del mundo real que necesiten interactuar con su entorno.

Entrada/Salida de Archivos

La entrada/salida de archivos (I/O) es un aspecto fundamental de la interacción con el sistema. Haskell proporciona varias funciones para leer y escribir archivos.

Leyendo de un Archivo

Para leer de un archivo, puedes usar la función readFile. Esta función toma una ruta de archivo como argumento y devuelve el contenido del archivo como una cadena.

import System.IO

main :: IO ()
main = do
    contents <- readFile "example.txt"
    putStrLn contents

Explicación:

  • readFile "example.txt" lee el contenido de example.txt.
  • contents <- readFile "example.txt" asigna el contenido del archivo a la variable contents.
  • putStrLn contents imprime el contenido en la consola.

Escribiendo en un Archivo

Para escribir en un archivo, puedes usar la función writeFile. Esta función toma una ruta de archivo y una cadena para escribir en el archivo.

import System.IO

main :: IO ()
main = do
    let content = "Hello, Haskell!"
    writeFile "output.txt" content

Explicación:

  • let content = "Hello, Haskell!" define la cadena a escribir.
  • writeFile "output.txt" content escribe la cadena en output.txt.

Añadiendo a un Archivo

Para añadir a un archivo, puedes usar la función appendFile. Esta función toma una ruta de archivo y una cadena para añadir al archivo.

import System.IO

main :: IO ()
main = do
    let additionalContent = "\nAppended text."
    appendFile "output.txt" additionalContent

Explicación:

  • let additionalContent = "\nAppended text." define la cadena a añadir.
  • appendFile "output.txt" additionalContent añade la cadena a output.txt.

Ejecutando Comandos del Sistema

Haskell puede ejecutar comandos del sistema usando el módulo System.Process. Este módulo proporciona funciones para ejecutar comandos externos y capturar su salida.

Ejecutando un Comando Simple

Para ejecutar un comando simple, puedes usar la función system.

import System.Process

main :: IO ()
main = do
    exitCode <- system "ls"
    print exitCode

Explicación:

  • system "ls" ejecuta el comando ls (listar contenido del directorio).
  • exitCode <- system "ls" captura el código de salida del comando.
  • print exitCode imprime el código de salida en la consola.

Capturando la Salida de un Comando

Para capturar la salida de un comando, puedes usar la función readProcess.

import System.Process

main :: IO ()
main = do
    output <- readProcess "echo" ["Hello, Haskell!"] ""
    putStrLn output

Explicación:

  • readProcess "echo" ["Hello, Haskell!"] "" ejecuta el comando echo con el argumento "Hello, Haskell!".
  • output <- readProcess "echo" ["Hello, Haskell!"] "" captura la salida del comando.
  • putStrLn output imprime la salida en la consola.

Manejo de Argumentos de Línea de Comandos

Haskell proporciona el módulo System.Environment para manejar argumentos de línea de comandos.

Accediendo a los Argumentos de Línea de Comandos

Para acceder a los argumentos de línea de comandos, puedes usar la función getArgs.

import System.Environment

main :: IO ()
main = do
    args <- getArgs
    print args

Explicación:

  • getArgs recupera la lista de argumentos de línea de comandos.
  • args <- getArgs asigna la lista de argumentos a la variable args.
  • print args imprime la lista de argumentos en la consola.

Usando Argumentos de Línea de Comandos

Puedes usar los argumentos de línea de comandos para personalizar el comportamiento de tu programa.

import System.Environment

main :: IO ()
main = do
    args <- getArgs
    case args of
        [fileName] -> do
            contents <- readFile fileName
            putStrLn contents
        _ -> putStrLn "Usage: runhaskell program.hs <filename>"

Explicación:

  • case args of hace un patrón de coincidencia con la lista de argumentos.
  • [fileName] -> do ... maneja el caso donde se proporciona exactamente un argumento (el nombre del archivo).
  • _ -> putStrLn "Usage: runhaskell program.hs <filename>" maneja cualquier otro caso, imprimiendo un mensaje de uso.

Resumen

En esta sección, cubrimos cómo interactuar con el sistema en Haskell. Aprendimos cómo:

  • Leer y escribir archivos usando readFile, writeFile y appendFile.
  • Ejecutar comandos del sistema usando system y readProcess.
  • Manejar argumentos de línea de comandos usando getArgs.

Estas capacidades permiten que los programas en Haskell interactúen con su entorno, haciéndolos más versátiles y útiles en aplicaciones del mundo real.
