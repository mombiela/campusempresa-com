Introducción a GHCi

GHCi (Glasgow Haskell Compiler interactive) es el entorno interactivo para Haskell. Te permite cargar y ejecutar código Haskell de manera interactiva, convirtiéndolo en una herramienta invaluable para aprender, depurar y experimentar con programas Haskell.

  • REPL: Read-Eval-Print Loop, un entorno de programación interactivo.
  • GHCi: El REPL para Haskell proporcionado por el Glasgow Haskell Compiler.

Iniciando GHCi

Para iniciar GHCi, abre tu terminal y escribe:

ghci

Deberías ver un prompt como este:

GHCi, version 8.10.4: https://www.haskell.org/ghc/  :? for help
Prelude>

Comandos Básicos

Aquí hay algunos comandos básicos que puedes usar en GHCi:

  • :load o :l: Cargar un archivo Haskell.
  • :reload o :r: Recargar el archivo actual.
  • :type o :t: Mostrar el tipo de una expresión.
  • :info o :i: Mostrar información sobre un nombre.
  • :quit o :q: Salir de GHCi.

Ejemplo de Uso

Prelude> :load MyModule.hs
[1 of 1] Compiling MyModule        ( MyModule.hs, interpreted )
Ok, one module loaded.

Prelude> :type 5
5 :: Num p => p

Prelude> :info Maybe
data Maybe a = Nothing | Just a
    -- Defined in ‘GHC.Base’
instance Eq a => Eq (Maybe a) -- Defined in ‘GHC.Base’
instance Monad Maybe -- Defined in ‘GHC.Base’
...

Evaluando Expresiones

Puedes evaluar expresiones Haskell directamente en GHCi:

Prelude> 2 + 3
5

Prelude> map (+1) [1, 2, 3]
[2, 3, 4]

Definiendo Funciones

Puedes definir funciones directamente en GHCi usando la palabra clave let:

Prelude> let square x = x * x
Prelude> square 5
25

Cargando y Recargando Módulos

Cuando trabajas con proyectos más grandes, a menudo cargarás y recargarás módulos:

Prelude> :load MyModule.hs
Prelude> :reload

Inspeccionando Tipos e Información

Entender los tipos y las funciones disponibles es crucial en Haskell:

Prelude> :type map
map :: (a -> b) -> [a] -> [b]

Prelude> :info map
map :: (a -> b) -> [a] -> [b] 	-- Defined in ‘GHC.Base’

Usando GHCi para Depuración

GHCi puede ser una poderosa herramienta de depuración. Puedes inspeccionar valores intermedios y probar pequeñas piezas de código de manera interactiva.

Ejemplo de Sesión de Depuración

Supongamos que tienes una función que no está funcionando como esperabas:

-- MyModule.hs
module MyModule where

factorial :: Integer -> Integer
factorial 0 = 1
factorial n = n * factorial (n - 1)

Puedes cargar este módulo y probar la función:

Prelude> :load MyModule.hs
Prelude MyModule> factorial 5
120

Si la función no funciona como esperabas, puedes inspeccionar valores intermedios:

Prelude MyModule> let factorial n = if n == 0 entonces 1 else n * factorial (n - 1)
Prelude MyModule> factorial 5
120

Conclusión

GHCi es una herramienta esencial para los programadores de Haskell, proporcionando un entorno interactivo para probar, depurar y explorar código Haskell. Al dominar los comandos y características de GHCi, puedes mejorar significativamente tu productividad y comprensión de Haskell.

  • Practica: Usa GHCi regularmente para probar pequeños fragmentos de código y depurar tus programas.
  • Explora: Usa comandos como :type y :info para aprender más sobre los tipos y funciones de Haskell.
  • Experimenta: No dudes en experimentar con nuevas ideas en GHCi antes de integrarlas en tu código principal.

Al incorporar GHCi en tu flujo de trabajo, encontrarás que aprender y trabajar con Haskell será una experiencia más interactiva y agradable.
