Introducción

¡Bienvenido al primer módulo de nuestro curso de Haskell! En esta sección, cubriremos la sintaxis básica de Haskell y te guiaremos para escribir tu primer programa "Hola Mundo". Al final de esta sección, tendrás una comprensión fundamental de la sintaxis de Haskell y cómo ejecutar un programa simple.

Conceptos Clave

  • Conceptos Básicos de Haskell: Comprender la estructura y sintaxis del código Haskell.
  • Comentarios: Cómo escribir comentarios en Haskell.
  • Programa Hola Mundo: Escribir y ejecutar tu primer programa en Haskell.

Conceptos Básicos de Haskell

Haskell es un lenguaje de programación puramente funcional con un sistema de tipos estático y fuerte. Aquí hay algunos aspectos fundamentales de la sintaxis de Haskell:

  • Módulos: El código Haskell se organiza en módulos.
  • Funciones: Las funciones son ciudadanos de primera clase en Haskell.
  • Tipos: Haskell tiene un sistema de tipos estático y fuerte.
  • Expresiones: Todo en Haskell es una expresión.

Ejemplo: Estructura Básica

-- Este es un comentario de una sola línea

{-
Este es un comentario de varias líneas
-}

module Main where

-- Definir una función
add :: Int -> Int -> Int
add x y = x + y

-- Función principal
main :: IO ()
main = do
    let result = add 5 3
    putStrLn ("El resultado es " ++ show result)

Explicación

  • Comentarios: Los comentarios de una sola línea comienzan con --, y los comentarios de varias líneas están encerrados en {- -}.
  • Declaración de Módulo: module Main where declara un módulo llamado Main.
  • Definición de Función: add :: Int -> Int -> Int define una función add que toma dos enteros y devuelve un entero.
  • Función Principal: main :: IO () es el punto de entrada del programa. La notación do se usa para secuenciar acciones de IO.

Escribiendo Tu Primer Programa "Hola Mundo"

Vamos a escribir un simple programa "Hola Mundo" en Haskell.

Ejemplo: Hola Mundo

module Main where

main :: IO ()
main = putStrLn "¡Hola, Mundo!"

Explicación

  • Declaración de Módulo: module Main where declara el módulo principal.
  • Función Principal: main :: IO () es la función principal que realiza acciones de IO.
  • Salida: putStrLn "¡Hola, Mundo!" imprime "¡Hola, Mundo!" en la consola.

Ejecutando el Programa

Para ejecutar tu programa en Haskell, sigue estos pasos:

  1. Guarda el código en un archivo llamado Main.hs.
  2. Abre una terminal y navega al directorio que contiene Main.hs.
  3. Compila el programa usando el compilador GHC: 
    ghc -o hola Main.hs
  4. Ejecuta el programa compilado: 
    ./hola

Conclusión

En esta sección, cubrimos la sintaxis básica de Haskell y escribimos un simple programa "Hola Mundo". Aprendiste sobre el sistema de módulos de Haskell, las definiciones de funciones y cómo realizar operaciones básicas de IO. Este conocimiento fundamental será esencial a medida que avancemos a temas más avanzados en Haskell. ¡Sigue practicando y pronto te sentirás cómodo con la sintaxis de Haskell y el paradigma de programación funcional!
