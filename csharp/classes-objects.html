Introducción

En C#, las clases y los objetos son conceptos fundamentales de la programación orientada a objetos (OOP). Comprender estos conceptos es crucial para escribir código eficiente y organizado. Esta sección te guiará desde los conceptos básicos hasta los temas avanzados relacionados con las clases y los objetos en C#.

¿Qué es una Clase?

Una clase es un plano para crear objetos. Define un tipo de dato al agrupar datos y métodos que trabajan sobre los datos en una sola unidad.

Conceptos Clave

  • Definición de Clase: Una clase se define usando la palabra clave class.
  • Miembros: Las clases pueden contener campos, propiedades, métodos y eventos.
  • Modificadores de Acceso: Controlan la visibilidad de los miembros de la clase (por ejemplo, public, private, protected).

Ejemplo

public class Person
{
    // Campos
    private string name;
    private int age;

    // Propiedades
    public string Name
    {
        get { return name; }
        set { name = value; }
    }

    public int Age
    {
        get { return age; }
        set { age = value; }
    }

    // Método
    public void DisplayInfo()
    {
        Console.WriteLine($"Name: {Name}, Age: {Age}");
    }
}

¿Qué es un Objeto?

Un objeto es una instancia de una clase. Cuando se define una clase, no se asigna memoria hasta que se crea un objeto de esa clase.

Conceptos Clave

  • Instanciación: Crear un objeto usando la palabra clave new.
  • Acceso a Miembros: Usar el operador punto (.) para acceder a campos, propiedades y métodos.

Ejemplo

// Creando un objeto de la clase Person
Person person1 = new Person();
person1.Name = "John";
person1.Age = 30;
person1.DisplayInfo(); // Salida: Name: John, Age: 30

Constructores

Los constructores son métodos especiales que se llaman cuando se instancia un objeto. Se utilizan para inicializar objetos.

Conceptos Clave

  • Constructor por Defecto: Un constructor sin parámetros.
  • Constructor Parametrizado: Un constructor que toma parámetros.
  • Sobrecarga de Constructores: Definir múltiples constructores con diferentes parámetros.

Ejemplo

public class Person
{
    // Campos
    private string name;
    private int age;

    // Constructor por Defecto
    public Person()
    {
        name = "Unknown";
        age = 0;
    }

    // Constructor Parametrizado
    public Person(string name, int age)
    {
        this.name = name;
        this.age = age;
    }

    // Método
    public void DisplayInfo()
    {
        Console.WriteLine($"Name: {name}, Age: {age}");
    }
}

// Usando Constructores
Person person1 = new Person(); // Constructor por defecto
Person person2 = new Person("Alice", 25); // Constructor parametrizado
person1.DisplayInfo(); // Salida: Name: Unknown, Age: 0
person2.DisplayInfo(); // Salida: Name: Alice, Age: 25

Herencia

La herencia permite que una clase herede campos y métodos de otra clase. La clase que hereda se llama clase derivada, y la clase de la que se hereda se llama clase base.

Conceptos Clave

  • Clase Base: La clase de la que se hereda.
  • Clase Derivada: La clase que hereda de la clase base.
  • Palabra Clave base: Se utiliza para acceder a los miembros de la clase base.

Ejemplo

// Clase Base
public class Animal
{
    public void Eat()
    {
        Console.WriteLine("Eating...");
    }
}

// Clase Derivada
public class Dog : Animal
{
    public void Bark()
    {
        Console.WriteLine("Barking...");
    }
}

// Usando Herencia
Dog dog = new Dog();
dog.Eat(); // Salida: Eating...
dog.Bark(); // Salida: Barking...

Polimorfismo

El polimorfismo permite que los métodos hagan cosas diferentes según el objeto sobre el que actúan. Se puede lograr mediante la sobrescritura de métodos y la sobrecarga de métodos.

Conceptos Clave

  • Sobrescritura de Métodos: Redefinir un método en una clase derivada.
  • Sobrecarga de Métodos: Definir múltiples métodos con el mismo nombre pero diferentes parámetros.

Ejemplo

// Clase Base
public class Animal
{
    public virtual void MakeSound()
    {
        Console.WriteLine("Animal sound");
    }
}

// Clase Derivada
public class Dog : Animal
{
    public override void MakeSound()
    {
        Console.WriteLine("Bark");
    }
}

// Usando Polimorfismo
Animal myAnimal = new Dog();
myAnimal.MakeSound(); // Salida: Bark

Encapsulación

La encapsulación es el concepto de envolver datos y métodos que operan sobre los datos dentro de una sola unidad (clase). Restringe el acceso directo a algunos de los componentes del objeto.

Conceptos Clave

  • Miembros Privados: Miembros que no pueden ser accedidos directamente desde fuera de la clase.
  • Métodos Públicos: Métodos que proporcionan acceso controlado a los miembros privados.

Ejemplo

public class Account
{
    private double balance;

    public void Deposit(double amount)
    {
        if (amount > 0)
        {
            balance += amount;
        }
    }

    public double GetBalance()
    {
        return balance;
    }
}

// Usando Encapsulación
Account myAccount = new Account();
myAccount.Deposit(100);
Console.WriteLine(myAccount.GetBalance()); // Salida: 100

Conclusión

Las clases y los objetos son los bloques de construcción de la programación orientada a objetos en C#. Comprender estos conceptos te permite crear código modular, reutilizable y mantenible. Al dominar las clases, objetos, herencia, polimorfismo y encapsulación, puedes aprovechar todo el poder de C# para construir aplicaciones robustas.
