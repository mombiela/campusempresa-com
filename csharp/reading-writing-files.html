Introducción

Leer y escribir archivos es una habilidad fundamental en la programación. En C#, el espacio de nombres System.IO proporciona clases y métodos para manejar operaciones de archivos. Este tema cubrirá los conceptos básicos de lectura y escritura de archivos, así como técnicas más avanzadas.

Conceptos Básicos de Manejo de Archivos

Conceptos Clave

  • Archivo: Representa un archivo en el sistema de archivos.
  • Stream: Una secuencia de bytes que se puede leer o escribir.
  • FileStream: Proporciona un flujo para operaciones de archivos.
  • StreamReader: Lee caracteres de un flujo de bytes.
  • StreamWriter: Escribe caracteres en un flujo de bytes.

Lectura de Archivos

Usando File.ReadAllText

La forma más simple de leer un archivo es usando el método File.ReadAllText, que lee todo el texto de un archivo en una sola cadena.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = File.ReadAllText(path);
        Console.WriteLine(content);
    }
}

Usando StreamReader

Para más control, puedes usar la clase StreamReader.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (StreamReader sr = new StreamReader(path))
        {
            string line;
            while ((line = sr.ReadLine()) != null)
            {
                Console.WriteLine(line);
            }
        }
    }
}

Escritura de Archivos

Usando File.WriteAllText

La forma más simple de escribir en un archivo es usando el método File.WriteAllText, que escribe una cadena en un archivo, creando el archivo si no existe.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        string content = "¡Hola, Mundo!";
        File.WriteAllText(path, content);
    }
}

Usando StreamWriter

Para más control, puedes usar la clase StreamWriter.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (StreamWriter sw = new StreamWriter(path))
        {
            sw.WriteLine("¡Hola, Mundo!");
            sw.WriteLine("Esta es una nueva línea.");
        }
    }
}

Manejo Avanzado de Archivos

Conceptos Clave

  • FileMode: Especifica cómo el sistema operativo debe abrir un archivo.
  • FileAccess: Define el nivel de acceso permitido a un archivo.
  • FileShare: Determina cómo el archivo será compartido por los procesos.

Usando `FileStream`

Para escenarios avanzados, puedes usar la clase FileStream, que proporciona más control sobre las operaciones de archivos.

using System;
using System.IO;

class Program
{
    static void Main()
    {
        string path = "example.txt";
        using (FileStream fs = new FileStream(path, FileMode.OpenOrCreate, FileAccess.ReadWrite))
        {
            byte[] info = new System.Text.UTF8Encoding(true).GetBytes("¡Hola, Mundo!");
            fs.Write(info, 0, info.Length);
        }
    }
}

Operaciones Asíncronas de Archivos

C# también soporta operaciones asíncronas de archivos, lo que puede mejorar el rendimiento en aplicaciones dependientes de I/O.

Lectura Asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = await File.ReadAllTextAsync(path);
        Console.WriteLine(content);
    }
}

Escritura Asíncrona

using System;
using System.IO;
using System.Threading.Tasks;

class Program
{
    static async Task Main()
    {
        string path = "example.txt";
        string content = "¡Hola, Mundo!";
        await File.WriteAllTextAsync(path, content);
    }
}

Resumen

En este tema, cubrimos las técnicas básicas y avanzadas para leer y escribir archivos en C#. Exploramos métodos simples como File.ReadAllText y File.WriteAllText, así como clases más avanzadas como StreamReader, StreamWriter y FileStream. Además, tocamos las operaciones asíncronas de archivos para manejar tareas dependientes de I/O de manera eficiente. Dominar estas técnicas te permitirá manejar operaciones de archivos de manera efectiva en tus aplicaciones C#.
