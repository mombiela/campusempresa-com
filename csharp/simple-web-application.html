Introducción

En este tema, te guiaremos a través de la construcción de una aplicación web simple utilizando C#. Esto cubrirá los conceptos básicos para principiantes y gradualmente se moverá hacia conceptos más avanzados. Al final de este tema, deberías tener una comprensión sólida de cómo crear, gestionar y desplegar una aplicación web utilizando C#.

Configurando Tu Entorno

Antes de comenzar a codificar, configuremos las herramientas y el entorno necesarios.

  • Instalar Visual Studio: Descarga e instala Visual Studio, que es un entorno de desarrollo integrado (IDE) para C#.
  • Instalar .NET SDK: Asegúrate de tener instalado el .NET SDK. Esto es necesario para construir y ejecutar aplicaciones .NET.

Creando un Nuevo Proyecto

  1. Abre Visual Studio.
  2. Selecciona "Crear un nuevo proyecto".
  3. Elige "Aplicación web ASP.NET Core" y haz clic en "Siguiente".
  4. Configura el nombre y la ubicación de tu proyecto, luego haz clic en "Crear".
  5. Selecciona "Aplicación web (Modelo-Vista-Controlador)" y haz clic en "Crear".

Entendiendo la Estructura del Proyecto

Una vez que el proyecto se crea, verás una variedad de archivos y carpetas. Aquí están los componentes clave:

  • Controladores: Manejan las solicitudes HTTP entrantes y envían respuestas de vuelta al cliente.
  • Vistas: Plantillas HTML que muestran datos al usuario.
  • Modelos: Representan los datos y la lógica de negocio.

Construyendo la Aplicación

Creando un Modelo

Primero, creemos un modelo simple. Los modelos representan los datos en tu aplicación.

public class Product
{
    public int Id { get; set; }
    public string Name { get; set; }
    public decimal Price { get; set; }
}

Creando un Controlador

A continuación, crea un controlador para manejar las solicitudes HTTP.

using Microsoft.AspNetCore.Mvc;
using System.Collections.Generic;

public class ProductController : Controller
{
    public IActionResult Index()
    {
        var products = new List<Product>
        {
            new Product { Id = 1, Name = "Laptop", Price = 999.99M },
            new Product { Id = 2, Name = "Smartphone", Price = 499.99M }
        };
        return View(products);
    }
}

Creando una Vista

Ahora, crea una vista para mostrar los productos.

  1. Haz clic derecho en la carpeta "Views", selecciona "Agregar" > "Nueva Carpeta", y nómbrala "Product".
  2. Haz clic derecho en la carpeta "Product", selecciona "Agregar" > "Nuevo Elemento", y elige "Vista Razor". Nómbrala "Index.cshtml".
@model IEnumerable<Product>

<!DOCTYPE html>
<html>
<head>
    <title>Products</title>
</head>
<body>
    <h1>Product List</h1>
    <table>
        <thead>
            <tr>
                <th>Id</th>
                <th>Name</th>
                <th>Price</th>
            </tr>
        </thead>
        <tbody>
            @foreach (var product in Model)
            {
                <tr>
                    <td>@product.Id</td>
                    <td>@product.Name</td>
                    <td>@product.Price</td>
                </tr>
            }
        </tbody>
    </table>
</body>
</html>

Ejecutando la Aplicación

Para ejecutar la aplicación, presiona F5 o haz clic en el botón "Iniciar Depuración" en Visual Studio. Esto construirá y lanzará tu aplicación web en un navegador.

Temas Avanzados

Agregando una Base de Datos

Para usuarios más avanzados, integrar una base de datos es esencial. Aquí hay una breve descripción:

  • Entity Framework Core: Usa EF Core para interactuar con la base de datos.
  • Migraciones: Gestiona los cambios en el esquema de la base de datos.

Autenticación y Autorización

Implementar seguridad es crucial para cualquier aplicación web.

  • Identity Framework: Usa ASP.NET Core Identity para la autenticación y autorización de usuarios.
  • Control de Acceso Basado en Roles: Define roles y restringe el acceso a ciertas partes de la aplicación.

Despliegue

Desplegar tu aplicación en un servidor o servicio en la nube es el paso final.

  • Azure: Despliega tu aplicación en Microsoft Azure.
  • Docker: Conteneriza tu aplicación usando Docker.

Conclusión

Construir una aplicación web simple en C# implica entender el patrón MVC, crear modelos, controladores y vistas, y ejecutar la aplicación. A medida que avances, puedes integrar bases de datos, implementar seguridad y desplegar tu aplicación. Este tema proporciona una base, y se te anima a explorar más y construir aplicaciones más complejas.
