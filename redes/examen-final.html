Examen Final

Introducción

El examen final del curso sobre Redes tiene como objetivo evaluar tu comprensión y habilidades en los conceptos clave que hemos cubierto a lo largo del curso. Este examen incluirá preguntas teóricas, ejercicios prácticos y problemas de análisis para asegurarnos de que puedas aplicar lo aprendido en situaciones reales.

Estructura del Examen

El examen se dividirá en las siguientes secciones:

  1. Preguntas Teóricas
  2. Configuración de Redes
  3. Análisis de Tráfico de Red
  4. Solución de Problemas de Red

Preguntas Teóricas

En esta sección, se evaluarán tus conocimientos sobre los conceptos fundamentales de redes. A continuación, se presentan algunos ejemplos de preguntas que podrías encontrar:

  • Define los siguientes términos:
    • IP Address
    • Subnet Mask
    • Gateway
    • DNS
  • Explica la diferencia entre TCP y UDP.
  • Describe el modelo OSI y sus capas.

Configuración de Redes

Esta sección evaluará tu capacidad para configurar y gestionar redes. Se te proporcionarán escenarios en los que deberás configurar dispositivos de red como routers y switches. Aquí tienes un ejemplo de ejercicio:

Ejemplo de Ejercicio

Configura una red simple con dos subredes utilizando un router y dos switches. Asegúrate de que las dos subredes puedan comunicarse entre sí.

Pasos:

  1. Asigna las siguientes direcciones IP a los dispositivos:
    • Subred 1: 192.168.1.0/24
    • Subred 2: 192.168.2.0/24
  2. Configura el router para enrutar el tráfico entre las dos subredes.
  3. Verifica la conectividad entre dispositivos de diferentes subredes.
Router(config)# interface g0/0
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# no shutdown

Router(config)# interface g0/1
Router(config-if)# ip address 192.168.2.1 255.255.255.0
Router(config-if)# no shutdown

Router(config)# ip route 192.168.2.0 255.255.255.0 g0/1
Router(config)# ip route 192.168.1.0 255.255.255.0 g0/0

Análisis de Tráfico de Red

En esta sección, se te pedirá que analices capturas de tráfico de red utilizando herramientas como Wireshark. Deberás identificar problemas y proponer soluciones.

Ejemplo de Ejercicio

Analiza la siguiente captura de tráfico y responde las preguntas:

  • ¿Cuál es la dirección IP de origen y destino del paquete número 10?
  • ¿Qué protocolo se está utilizando?
  • ¿Hay algún indicio de problemas de red en la captura?
No.     Time           Source                Destination           Protocol Length Info
10      0.000123       192.168.1.2           192.168.1.1           TCP      60     443 → 12345 [SYN] Seq=0 Win=65535 Len=0 MSS=1460

Solución de Problemas de Red

Esta sección evaluará tu capacidad para diagnosticar y resolver problemas de red. Se te presentarán escenarios con problemas específicos que deberás solucionar.

Ejemplo de Ejercicio

Un usuario reporta que no puede acceder a internet. Su dirección IP es 192.168.1.10 y su gateway es 192.168.1.1. Describe los pasos que seguirías para diagnosticar y resolver el problema.

Pasos:

  1. Verifica la conectividad del usuario con el gateway. 
    ping 192.168.1.1
  2. Comprueba la configuración de DNS. 
    nslookup www.google.com
  3. Revisa la configuración del router y asegúrate de que no haya reglas de firewall bloqueando el tráfico.

Conclusión

El examen final es una oportunidad para demostrar tu dominio de los conceptos y habilidades en redes que has adquirido durante el curso. Asegúrate de revisar todos los temas cubiertos y practica con ejercicios similares a los proporcionados. ¡Buena suerte!
