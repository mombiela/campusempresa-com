Introducción

En esta sección, exploraremos varios estudios de caso y proyectos que te ayudarán a aplicar los conceptos que has aprendido en Fortran. Estos ejemplos prácticos variarán desde niveles principiantes hasta avanzados, permitiéndote desarrollar tus habilidades progresivamente. Al final de esta sección, deberías ser capaz de abordar problemas del mundo real usando Fortran.

Proyectos de Nivel Principiante

1. Operaciones Matemáticas Simples

  • Objetivo: Implementar operaciones aritméticas básicas.
  • Conceptos Cubiertos: Variables, entrada/salida básica, operaciones aritméticas.

Código de Ejemplo:

program simple_math
  implicit none
  real :: a, b, sum, difference, product, quotient

  ! Ingresar valores
  print *, 'Ingrese el primer número:'
  read *, a
  print *, 'Ingrese el segundo número:'
  read *, b

  ! Realizar operaciones
  sum = a + b
  difference = a - b
  product = a * b
  quotient = a / b

  ! Mostrar resultados
  print *, 'Suma:', sum
  print *, 'Diferencia:', difference
  print *, 'Producto:', product
  print *, 'Cociente:', quotient
end program simple_math

2. Conversión de Temperatura

  • Objetivo: Convertir temperaturas entre Celsius y Fahrenheit.
  • Conceptos Cubiertos: Sentencias condicionales, funciones.

Código de Ejemplo:

program temp_conversion
  implicit none
  real :: celsius, fahrenheit

  ! Ingresar temperatura en Celsius
  print *, 'Ingrese la temperatura en Celsius:'
  read *, celsius

  ! Convertir a Fahrenheit
  fahrenheit = celsius * 9.0 / 5.0 + 32.0

  ! Mostrar resultado
  print *, 'Temperatura en Fahrenheit:', fahrenheit
end program temp_conversion

Proyectos de Nivel Intermedio

1. Multiplicación de Matrices

  • Objetivo: Multiplicar dos matrices.
  • Conceptos Cubiertos: Arreglos, bucles anidados.

Código de Ejemplo:

program matrix_multiplication
  implicit none
  integer, parameter :: n = 3
  real :: A(n, n), B(n, n), C(n, n)
  integer :: i, j, k

  ! Inicializar matrices A y B
  A = reshape([1, 2, 3, 4, 5, 6, 7, 8, 9], shape(A))
  B = reshape([9, 8, 7, 6, 5, 4, 3, 2, 1], shape(B))
  C = 0.0

  ! Multiplicación de matrices
  do i = 1, n
    do j = 1, n
      do k = 1, n
        C(i, j) = C(i, j) + A(i, k) * B(k, j)
      end do
    end do
  end do

  ! Mostrar resultado
  print *, 'Matriz Resultante C:'
  do i = 1, n
    print *, C(i, :)
  end do
end program matrix_multiplication

2. Integración Numérica

  • Objetivo: Implementar la regla del trapecio para la integración numérica.
  • Conceptos Cubiertos: Funciones, bucles, métodos numéricos.

Código de Ejemplo:

program trapezoidal_rule
  implicit none
  real :: a, b, h, integral
  integer :: n, i

  ! Definir la función a integrar
  contains
    real function f(x)
      real, intent(in) :: x
      f = x**2  ! Función de ejemplo: f(x) = x^2
    end function f

  ! Ingresar límites y número de intervalos
  print *, 'Ingrese el límite inferior a:'
  read *, a
  print *, 'Ingrese el límite superior b:'
  read *, b
  print *, 'Ingrese el número de intervalos n:'
  read *, n

  ! Calcular tamaño del paso
  h = (b - a) / n

  ! Aplicar la regla del trapecio
  integral = 0.5 * (f(a) + f(b))
  do i = 1, n-1
    integral = integral + f(a + i * h)
  end do
  integral = integral * h

  ! Mostrar resultado
  print *, 'Integral:', integral
end program trapezoidal_rule

Proyectos de Nivel Avanzado

1. Resolución de Sistemas Lineales

  • Objetivo: Resolver un sistema de ecuaciones lineales usando eliminación gaussiana.
  • Conceptos Cubiertos: Manipulaciones avanzadas de arreglos, álgebra lineal.

Código de Ejemplo:

program gaussian_elimination
  implicit none
  integer, parameter :: n = 3
  real :: A(n, n+1)
  integer :: i, j, k
  real :: factor

  ! Inicializar matriz aumentada A
  A = reshape([2.0, -1.0, 1.0, 8.0, -3.0, -1.0, 2.0, -11.0, -2.0, 1.0, 2.0, -3.0], shape(A))

  ! Eliminación gaussiana
  do k = 1, n-1
    do i = k+1, n
      factor = A(i, k) / A(k, k)
      do j = k, n+1
        A(i, j) = A(i, j) - factor * A(k, j)
      end do
    end do
  end do

  ! Sustitución hacia atrás
  do i = n, 1, -1
    A(i, n+1) = A(i, n+1) / A(i, i)
    A(i, i) = 1.0
    do j = i-1, 1, -1
      A(j, n+1) = A(j, n+1) - A(j, i) * A(i, n+1)
      A(j, i) = 0.0
    end do
  end do

  ! Mostrar resultado
  print *, 'Solución:'
  do i = 1, n
    print *, 'x', i, '=', A(i, n+1)
  end do
end program gaussian_elimination

2. Simulación de Sistemas Físicos

  • Objetivo: Simular el movimiento de un proyectil.
  • Conceptos Cubiertos: Ecuaciones diferenciales, simulaciones físicas.

Código de Ejemplo:

program projectile_motion
  implicit none
  real :: g, v0, theta, t, dt, x, y
  real, parameter :: pi = 3.141592653589793
  integer :: n, i

  ! Constantes
  g = 9.81  ! Aceleración debido a la gravedad (m/s^2)

  ! Ingresar condiciones iniciales
  print *, 'Ingrese la velocidad inicial (m/s):'
  read *, v0
  print *, 'Ingrese el ángulo de lanzamiento (grados):'
  read *, theta

  ! Convertir ángulo a radianes
  theta = theta * pi / 180.0

  ! Paso de tiempo y número de pasos
  dt = 0.01
  n = 100

  ! Inicializar posición y tiempo
  x = 0.0
  y = 0.0
  t = 0.0

  ! Simular movimiento
  do i = 1, n
    t = t + dt
    x = v0 * cos(theta) * t
    y = v0 * sin(theta) * t - 0.5 * g * t**2
    if (y < 0.0) exit
    print *, 'Tiempo:', t, 's, Posición: (', x, ',', y, ')'
  end do
end program projectile_motion
