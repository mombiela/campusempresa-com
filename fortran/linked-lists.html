Introducción a las listas enlazadas

Las listas enlazadas son una estructura de datos fundamental utilizada en ciencias de la computación para almacenar colecciones de elementos. A diferencia de los arreglos, las listas enlazadas consisten en nodos donde cada nodo contiene datos y una referencia (o enlace) al siguiente nodo en la secuencia. Esta estructura permite la inserción y eliminación eficiente de elementos.

Conceptos clave

  • Nodo: La unidad básica de una lista enlazada, que contiene datos y una referencia al siguiente nodo.
  • Cabeza: El primer nodo en una lista enlazada.
  • Cola: El último nodo en una lista enlazada, que apunta a null o nil.
  • Puntero: Una variable que contiene la dirección de memoria de otra variable, utilizada para enlazar nodos en una lista enlazada.

Creando una lista enlazada simple

Para crear una lista enlazada en Fortran, necesitamos definir una estructura de nodo e implementar operaciones básicas como la inserción y el recorrido.

Definiendo un nodo

En Fortran, podemos definir un nodo usando una construcción type.

type :: Node
    integer :: data
    type(Node), pointer :: next => null()
end type Node

Inicializando la lista enlazada

Comenzamos creando la cabeza de la lista enlazada.

program LinkedListExample
    implicit none
    type(Node), pointer :: head, newNode

    ! Inicializar el nodo cabeza
    allocate(head)
    head%data = 10
    head%next => null()

    ! Imprimir los datos del nodo cabeza
    print *, "Head node data: ", head%data
end program LinkedListExample

Insertando nodos

Podemos insertar nuevos nodos al principio, al final o en cualquier posición de la lista enlazada.

Insertando al principio

Para insertar un nodo al principio, necesitamos actualizar el puntero de la cabeza para que apunte al nuevo nodo.

subroutine insertAtBeginning(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: newNode

    allocate(newNode)
    newNode%data = value
    newNode%next => head
    head => newNode
end subroutine insertAtBeginning

Insertando al final

Para insertar un nodo al final, recorremos la lista para encontrar el último nodo y actualizamos su puntero next.

subroutine insertAtEnd(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: newNode, temp

    allocate(newNode)
    newNode%data = value
    newNode%next => null()

    if (.not. associated(head)) then
        head => newNode
    else
        temp => head
        do while (associated(temp%next))
            temp => temp%next
        end do
        temp%next => newNode
    end if
end subroutine insertAtEnd

Recorriendo la lista enlazada

El recorrido implica visitar cada nodo en la lista enlazada y realizar alguna operación, como imprimir los datos.

subroutine traverse(head)
    type(Node), pointer :: head
    type(Node), pointer :: temp

    temp => head
    do while (associated(temp))
        print *, "Node data: ", temp%data
        temp => temp%next
    end do
end subroutine traverse

Operaciones avanzadas

Las operaciones avanzadas en listas enlazadas incluyen eliminación, búsqueda y reversión de la lista.

Eliminando un nodo

Para eliminar un nodo, necesitamos actualizar los punteros para omitir el nodo que se va a eliminar.

subroutine deleteNode(head, value)
    type(Node), pointer :: head
    integer, intent(in) :: value
    type(Node), pointer :: temp, prev

    temp => head
    prev => null()

    do while (associated(temp))
        if (temp%data == value) then
            if (associated(prev)) then
                prev%next => temp%next
            else
                head => temp%next
            end if
            deallocate(temp)
            return
        end if
        prev => temp
        temp => temp%next
    end do
end subroutine deleteNode

Buscando un nodo

Para buscar un nodo, recorremos la lista y verificamos los datos de cada nodo.

function searchNode(head, value) result(found)
    type(Node), pointer :: head
    integer, intent(in) :: value
    logical :: found
    type(Node), pointer :: temp

    found = .false.
    temp => head
    do while (associated(temp))
        if (temp%data == value) then
            found = .true.
            return
        end if
        temp => temp%next
    end do
end function searchNode

Revirtiendo la lista enlazada

Revertir una lista enlazada implica cambiar la dirección de los punteros.

subroutine reverseList(head)
    type(Node), pointer :: head
    type(Node), pointer :: prev, current, next

    prev => null()
    current => head

    do while (associated(current))
        next => current%next
        current%next => prev
        prev => current
        current => next
    end do
    head => prev
end subroutine reverseList

Conclusión

Las listas enlazadas son una estructura de datos versátil y fundamental en la programación. En Fortran, las listas enlazadas se pueden implementar utilizando punteros y tipos personalizados. Entender cómo crear, manipular y recorrer listas enlazadas es esencial para resolver varios problemas computacionales de manera eficiente. Al dominar estos conceptos, puedes manejar estructuras de datos y algoritmos más complejos en tu viaje de programación.
