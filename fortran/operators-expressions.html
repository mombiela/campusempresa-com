Introducción

Los operadores y expresiones son componentes fundamentales en la programación en Fortran. Permiten realizar cálculos, tomar decisiones y manipular datos. Esta sección cubrirá los diversos tipos de operadores disponibles en Fortran, cómo usarlos y cómo construir expresiones.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos. Fortran soporta los siguientes operadores aritméticos:

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División
  • ** : Exponenciación

Ejemplos

PROGRAM ArithmeticExample
  IMPLICIT NONE
  INTEGER :: a, b, c
  REAL :: x, y, z

  ! Aritmética de enteros
  a = 10
  b = 3
  c = a + b  ! c = 13
  PRINT *, 'a + b = ', c

  c = a - b  ! c = 7
  PRINT *, 'a - b = ', c

  c = a * b  ! c = 30
  PRINT *, 'a * b = ', c

  c = a / b  ! c = 3 (división de enteros)
  PRINT *, 'a / b = ', c

  ! Aritmética de reales
  x = 10.0
  y = 3.0
  z = x / y  ! z = 3.3333333
  PRINT *, 'x / y = ', z

  z = x ** y  ! z = 1000.0
  PRINT *, 'x ** y = ', z
END PROGRAM ArithmeticExample

Operadores Relacionales

Los operadores relacionales se utilizan para comparar valores. Fortran soporta los siguientes operadores relacionales:

  • == : Igual a
  • /= : Diferente de
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual que
  • <= : Menor o igual que

Ejemplos

PROGRAM RelationalExample
  IMPLICIT NONE
  INTEGER :: a, b
  LOGICAL :: result

  a = 10
  b = 3

  result = (a == b)  ! result = .FALSE.
  PRINT *, 'a == b: ', result

  result = (a /= b)  ! result = .TRUE.
  PRINT *, 'a /= b: ', result

  result = (a > b)  ! result = .TRUE.
  PRINT *, 'a > b: ', result

  result = (a < b)  ! result = .FALSE.
  PRINT *, 'a < b: ', result

  result = (a >= b)  ! result = .TRUE.
  PRINT *, 'a >= b: ', result

  result = (a <= b)  ! result = .FALSE.
  PRINT *, 'a <= b: ', result
END PROGRAM RelationalExample

Operadores Lógicos

Los operadores lógicos se utilizan para realizar operaciones lógicas. Fortran soporta los siguientes operadores lógicos:

  • .AND. : AND lógico
  • .OR. : OR lógico
  • .NOT. : NOT lógico

Ejemplos

PROGRAM LogicalExample
  IMPLICIT NONE
  LOGICAL :: p, q, result

  p = .TRUE.
  q = .FALSE.

  result = p .AND. q  ! result = .FALSE.
  PRINT *, 'p .AND. q: ', result

  result = p .OR. q  ! result = .TRUE.
  PRINT *, 'p .OR. q: ', result

  result = .NOT. p  ! result = .FALSE.
  PRINT *, '.NOT. p: ', result
END PROGRAM LogicalExample

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables. El operador de asignación básico en Fortran es =.

Ejemplos

PROGRAM AssignmentExample
  IMPLICIT NONE
  INTEGER :: a
  REAL :: x

  a = 5  ! Asigna 5 a a
  PRINT *, 'a = ', a

  x = 3.14  ! Asigna 3.14 a x
  PRINT *, 'x = ', x
END PROGRAM AssignmentExample

Operadores Combinados

Fortran también soporta operadores combinados para operaciones aritméticas y asignación. Estos incluyen:

  • += : Asignación de adición
  • -= : Asignación de sustracción
  • *= : Asignación de multiplicación
  • /= : Asignación de división

Ejemplos

PROGRAM CombinedOperatorsExample
  IMPLICIT NONE
  INTEGER :: a

  a = 10
  a += 5  ! Equivalente a a = a + 5
  PRINT *, 'a += 5: ', a  ! a = 15

  a -= 3  ! Equivalente a a = a - 3
  PRINT *, 'a -= 3: ', a  ! a = 12

  a *= 2  ! Equivalente a a = a * 2
  PRINT *, 'a *= 2: ', a  ! a = 24

  a /= 4  ! Equivalente a a = a / 4
  PRINT *, 'a /= 4: ', a  ! a = 6
END PROGRAM CombinedOperatorsExample

Precedencia de Operadores

La precedencia de operadores determina el orden en que se realizan las operaciones en una expresión. La siguiente tabla resume la precedencia de los operadores en Fortran, de mayor a menor:

| Operador | Descripción | |----------------|-----------------------| | ** | Exponenciación | | *, / | Multiplicación, División | | +, - | Adición, Sustracción | | .NOT. | NOT lógico | | .AND. | AND lógico | | .OR. | OR lógico | | ==, /=, >, <, >=, <= | Operadores Relacionales |

Ejemplos

PROGRAM PrecedenceExample
  IMPLICIT NONE
  INTEGER :: a, b, c, result

  a = 2
  b = 3
  c = 4

  result = a + b * c  ! Multiplicación antes de adición
  PRINT *, 'a + b * c = ', result  ! result = 14

  result = (a + b) * c  ! Los paréntesis cambian el orden
  PRINT *, '(a + b) * c = ', result  ! result = 20

  result = a ** b + c  ! Exponenciación antes de adición
  PRINT *, 'a ** b + c = ', result  ! result = 12
END PROGRAM PrecedenceExample

Conclusión

Entender los operadores y expresiones es crucial para escribir programas efectivos en Fortran. Esta sección cubrió operadores aritméticos, relacionales, lógicos, de asignación y combinados, así como la precedencia de operadores. Al dominar estos conceptos, podrás realizar cálculos complejos y tomar decisiones informadas en tus programas en Fortran. Practica usando estos operadores en diferentes contextos para familiarizarte más con su uso y comportamiento.
