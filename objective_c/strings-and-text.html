Cadenas y Texto en Objective-C
Colecciones: Arrays, Diccionarios y Conjuntos ►

Introducción a las Cadenas

Las cadenas son secuencias de caracteres utilizadas para representar texto en programación. En Objective-C, las cadenas se gestionan comúnmente usando la clase NSString, que proporciona una amplia gama de métodos para crear, manipular y consultar datos de cadenas.

Conceptos Clave

  • NSString: La clase principal para la manipulación de cadenas.
  • NSMutableString: Una subclase de NSString que permite operaciones de cadena mutables.
  • Literales de Cadena: Cómo crear cadenas usando sintaxis literal.
  • Métodos de Cadena: Métodos comunes para la manipulación de cadenas.

Creación de Cadenas

Usando Literales de Cadena

Los literales de cadena son la forma más simple de crear cadenas en Objective-C. Están encerrados en comillas dobles y precedidos por un símbolo @.

NSString *greeting = @"¡Hola, Mundo!";

Usando Métodos de NSString

También puedes crear cadenas usando varios métodos de NSString.

NSString *greeting = [NSString stringWithFormat:@"¡Hola, %@!", @"Mundo"];

Métodos Comunes de NSString

Longitud de una Cadena

Para obtener la longitud de una cadena, usa el método length.

NSString *greeting = @"¡Hola, Mundo!";
NSUInteger length = [greeting length];
NSLog(@"Longitud: %lu", (unsigned long)length); // Salida: Longitud: 13

Comparando Cadenas

Para comparar cadenas, usa el método isEqualToString:.

NSString *string1 = @"Hola";
NSString *string2 = @"Hola";
BOOL areEqual = [string1 isEqualToString:string2];
NSLog(@"Son iguales: %d", areEqual); // Salida: Son iguales: 1

Concatenando Cadenas

Para concatenar cadenas, usa el método stringByAppendingString:.

NSString *greeting = @"Hola";
NSString *name = @"Mundo";
NSString *fullGreeting = [greeting stringByAppendingString:name];
NSLog(@"%@", fullGreeting); // Salida: HolaMundo

Cadenas Mutables

Introducción a NSMutableString

NSMutableString es una subclase de NSString que te permite modificar la cadena después de que ha sido creada.

NSMutableString *mutableGreeting = [NSMutableString stringWithString:@"Hola"];
[mutableGreeting appendString:@", Mundo!"];
NSLog(@"%@", mutableGreeting); // Salida: ¡Hola, Mundo!

Métodos Comunes de NSMutableString

  • appendString: Añade una cadena al final del receptor.
  • insertString:atIndex: Inserta una cadena en un índice especificado.
  • deleteCharactersInRange: Elimina caracteres en un rango especificado.
  • replaceCharactersInRange:withString: Reemplaza caracteres en un rango especificado con otra cadena.
NSMutableString *mutableString = [NSMutableString stringWithString:@"Hola"];
[mutableString appendString:@" Mundo"];
[mutableString insertString:@" Objective-C" atIndex:5];
[mutableString deleteCharactersInRange:NSMakeRange(5, 12)];
[mutableString replaceCharactersInRange:NSMakeRange(5, 5) withString:@" Universo"];
NSLog(@"%@", mutableString); // Salida: Hola Universo

Manipulación Avanzada de Cadenas

Subcadenas

Extraer subcadenas se puede hacer usando los métodos substringFromIndex:, substringToIndex: y substringWithRange:.

NSString *greeting = @"¡Hola, Mundo!";
NSString *substring1 = [greeting substringFromIndex:7]; // Salida: Mundo!
NSString *substring2 = [greeting substringToIndex:5]; // Salida: Hola
NSString *substring3 = [greeting substringWithRange:NSMakeRange(7, 5)]; // Salida: Mundo

Buscando dentro de Cadenas

Para buscar subcadenas, usa métodos como rangeOfString:.

NSString *greeting = @"¡Hola, Mundo!";
NSRange range = [greeting rangeOfString:@"Mundo"];
if (range.location != NSNotFound) {
    NSLog(@"Encontrado 'Mundo' en el índice %lu", (unsigned long)range.location); // Salida: Encontrado 'Mundo' en el índice 7
}

Formateando Cadenas

Usa stringWithFormat: para crear cadenas formateadas.

NSString *name = @"Juan";
int age = 30;
NSString *formattedString = [NSString stringWithFormat:@"%@ tiene %d años.", name, age];
NSLog(@"%@", formattedString); // Salida: Juan tiene 30 años.

Conclusión

Las cadenas son una parte fundamental de la programación en Objective-C, permitiéndote manejar y manipular texto de manera eficiente. Al entender y utilizar NSString y NSMutableString, puedes realizar una amplia gama de operaciones, desde la creación básica de cadenas hasta el procesamiento avanzado de texto. Practica estos conceptos con los ejemplos y ejercicios proporcionados para volverte competente en el manejo de cadenas en Objective-C.
