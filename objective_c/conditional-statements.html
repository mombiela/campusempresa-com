Introducción a las declaraciones condicionales

Las declaraciones condicionales son fundamentales en la programación, permitiéndote ejecutar diferentes bloques de código basados en ciertas condiciones. En Objective-C, las principales declaraciones condicionales incluyen if, else if, else y switch.

La declaración `if`

La declaración if evalúa una condición y ejecuta un bloque de código si la condición es verdadera.

Sintaxis

if (condition) {
    // Código a ejecutar si la condición es verdadera
}

Ejemplo

int number = 10;
if (number > 5) {
    NSLog(@"El número es mayor que 5");
}

En este ejemplo, se imprimirá el mensaje "El número es mayor que 5" porque la condición number > 5 es verdadera.

La declaración `else`

La declaración else sigue a una declaración if y ejecuta un bloque de código si la condición del if es falsa.

Sintaxis

if (condition) {
    // Código a ejecutar si la condición es verdadera
} else {
    // Código a ejecutar si la condición es falsa
}

Ejemplo

int number = 3;
if (number > 5) {
    NSLog(@"El número es mayor que 5");
} else {
    NSLog(@"El número no es mayor que 5");
}

En este ejemplo, se imprimirá el mensaje "El número no es mayor que 5" porque la condición number > 5 es falsa.

La declaración `else if`

La declaración else if te permite verificar múltiples condiciones secuencialmente.

Sintaxis

if (condition1) {
    // Código a ejecutar si la condición1 es verdadera
} else if (condition2) {
    // Código a ejecutar si la condición2 es verdadera
} else {
    // Código a ejecutar si ninguna de las condiciones anteriores es verdadera
}

Ejemplo

int number = 7;
if (number > 10) {
    NSLog(@"El número es mayor que 10");
} else if (number > 5) {
    NSLog(@"El número es mayor que 5 pero menor o igual a 10");
} else {
    NSLog(@"El número es 5 o menos");
}

En este ejemplo, se imprimirá el mensaje "El número es mayor que 5 pero menor o igual a 10" porque la condición number > 5 es verdadera.

La declaración `switch`

La declaración switch proporciona una forma de ejecutar un bloque de código entre muchos basándose en el valor de una variable.

Sintaxis

switch (variable) {
    case value1:
        // Código a ejecutar si variable == value1
        break;
    case value2:
        // Código a ejecutar si variable == value2
        break;
    // Más casos...
    default:
        // Código a ejecutar si ninguno de los casos anteriores es verdadero
        break;
}

Ejemplo

int day = 3;
switch (day) {
    case 1:
        NSLog(@"Lunes");
        break;
    case 2:
        NSLog(@"Martes");
        break;
    case 3:
        NSLog(@"Miércoles");
        break;
    case 4:
        NSLog(@"Jueves");
        break;
    case 5:
        NSLog(@"Viernes");
        break;
    case 6:
        NSLog(@"Sábado");
        break;
    case 7:
        NSLog(@"Domingo");
        break;
    default:
        NSLog(@"Día inválido");
        break;
}

En este ejemplo, se imprimirá el mensaje "Miércoles" porque el valor de day es 3.

Declaraciones condicionales anidadas

Puedes anidar declaraciones condicionales dentro de otras para manejar lógica más compleja.

Ejemplo

int number = 15;
if (number > 10) {
    if (number < 20) {
        NSLog(@"El número está entre 10 y 20");
    } else {
        NSLog(@"El número es 20 o más");
    }
} else {
    NSLog(@"El número es 10 o menos");
}

En este ejemplo, se imprimirá el mensaje "El número está entre 10 y 20" porque el número 15 satisface ambas condiciones.

Conclusión

Las declaraciones condicionales son esenciales para controlar el flujo de tu programa basándose en diferentes condiciones. Entender cómo usar las declaraciones if, else if, else y switch te permitirá escribir código más dinámico y receptivo. Practica usando estas declaraciones en varios escenarios para volverte competente en el manejo de lógica condicional en Objective-C.
