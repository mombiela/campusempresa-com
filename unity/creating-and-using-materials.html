Introducción

En Unity, los materiales definen cómo aparecen los objetos en la pantalla. Controlan el color, la textura y la reflectividad de los objetos, entre otras propiedades. Entender cómo crear y usar materiales es esencial para hacer que tu juego sea visualmente atractivo.

Creación de Materiales

Creación Básica de Materiales

  1. Abrir la Ventana del Proyecto: Navega a la ventana del Proyecto en Unity.

  2. Crear un Nuevo Material:

    • Haz clic derecho en la ventana del Proyecto.
    • Selecciona Create > Material.
    • Nombra tu nuevo material (por ejemplo, "MiMaterial").

  3. Asignar un Shader:

    • Por defecto, el nuevo material usa el Shader Estándar.
    • Puedes cambiar el shader seleccionando el material y eligiendo un shader diferente del menú desplegable Shader en el Inspector.

Configuración de Propiedades del Material

  • Color: Cambia el color base del material.
  • Textura: Añade una textura al material arrastrando un archivo de textura al espacio de Albedo.
  • Metálico y Suavidad: Ajusta estos deslizadores para cambiar la reflectividad y suavidad del material.
// Ejemplo: Creación y asignación de un material mediante script
using UnityEngine;

public class MaterialExample : MonoBehaviour
{
    void Start()
    {
        // Crear un nuevo material con un shader estándar
        Material myMaterial = new Material(Shader.Find("Standard"));
        
        // Establecer el color a rojo
        myMaterial.color = Color.red;
        
        // Asignar el material al renderer del objeto
        GetComponent<Renderer>().material = myMaterial;
    }
}

Uso de Materiales

Aplicación de Materiales a Objetos

  1. Arrastrar y Soltar:

    • Arrastra el material desde la ventana del Proyecto hasta el objeto en la ventana de Escena o Jerarquía.

  2. Mediante el Inspector:

    • Selecciona el objeto en la ventana de Escena o Jerarquía.
    • En el Inspector, encuentra el componente Mesh Renderer.
    • Arrastra el material al espacio Element 0 bajo Materiales.

Instancias de Material

  • Material Compartido: Los cambios en el material afectan a todos los objetos que lo usan.
  • Material de Instancia: Los cambios afectan solo a la instancia específica del material en el objeto.
// Ejemplo: Modificación de una instancia de material
using UnityEngine;

public class InstanceMaterialExample : MonoBehaviour
{
    void Start()
    {
        // Obtener el componente renderer
        Renderer renderer = GetComponent<Renderer>();
        
        // Crear una instancia del material
        Material instanceMaterial = renderer.material;
        
        // Cambiar el color del material de instancia
        instanceMaterial.color = Color.blue;
    }
}

Técnicas Avanzadas de Material

Uso de Shaders Personalizados

  • Shader Graph: Herramienta visual de Unity para crear shaders sin escribir código.
  • Código de Shader Personalizado: Escribe tus propios shaders usando HLSL.

Bloques de Propiedades de Material

  • Cambiar Propiedades de Material de Forma Eficiente: Usa MaterialPropertyBlock para cambiar propiedades sin crear nuevas instancias de material.
// Ejemplo: Uso de MaterialPropertyBlock
using UnityEngine;

public class MaterialPropertyBlockExample : MonoBehaviour
{
    void Start()
    {
        Renderer renderer = GetComponent<Renderer>();
        MaterialPropertyBlock block = new MaterialPropertyBlock();
        
        // Establecer una propiedad de color
        block.SetColor("_Color", Color.green);
        
        // Aplicar el bloque de propiedades al renderer
        renderer.SetPropertyBlock(block);
    }
}

Conclusión

Crear y usar materiales en Unity es una habilidad fundamental para el desarrollo de juegos. Al entender cómo crear materiales, configurar sus propiedades y aplicarlos a objetos, puedes mejorar significativamente la calidad visual de tus juegos. Técnicas avanzadas como shaders personalizados y MaterialPropertyBlocks permiten un control y optimización aún mayores. Practica estos conceptos para volverte competente en la gestión de materiales en Unity.
