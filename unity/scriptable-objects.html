Introducción a los Objetos Scriptables

Los Objetos Scriptables son una característica poderosa en Unity que te permite crear contenedores de datos que pueden ser utilizados para almacenar grandes cantidades de datos compartidos independientemente de las instancias de clase. Son particularmente útiles para gestionar datos del juego, configuraciones y ajustes.

Conceptos Clave

  • Contenedores de Datos: Los Objetos Scriptables se utilizan para almacenar datos que pueden ser compartidos entre múltiples instancias.
  • Creación de Activos: Los Objetos Scriptables se crean como activos en el Editor de Unity.
  • Serialización: Son serializados y guardados como activos, lo que los hace persistentes a través de las sesiones del juego.

Creando Objetos Scriptables

Guía Paso a Paso

  1. Crear una Clase de Objeto Scriptable:
    • Define una clase que herede de ScriptableObject.
    • Usa el atributo CreateAssetMenu para facilitar la creación de instancias desde el Editor de Unity.
using UnityEngine;

[CreateAssetMenu(fileName = "NewGameData", menuName = "Game Data", order = 51)]
public class GameData : ScriptableObject
{
    public string gameName;
    public int maxPlayers;
    public float gameDuration;
}
  1. Crear una Instancia en el Editor:
    • Haz clic derecho en la ventana del Proyecto.
    • Selecciona Create > Game Data (o como hayas nombrado tu Objeto Scriptable).
    • Rellena los campos de datos en el Inspector.

Ejemplo

// GameData.cs
using UnityEngine;

[CreateAssetMenu(fileName = "NewGameData", menuName = "Game Data", order = 51)]
public class GameData : ScriptableObject
{
    public string gameName = "My Game";
    public int maxPlayers = 4;
    public float gameDuration = 60.0f;
}

Usando Objetos Scriptables en Tu Juego

Accediendo a los Objetos Scriptables

  • Referencia en Scripts: Puedes referenciar Objetos Scriptables en tus scripts de MonoBehaviour para acceder a sus datos.
using UnityEngine;

public class GameManager : MonoBehaviour
{
    public GameData gameData;

    void Start()
    {
        Debug.Log("Game Name: " + gameData.gameName);
        Debug.Log("Max Players: " + gameData.maxPlayers);
        Debug.Log("Game Duration: " + gameData.gameDuration);
    }
}

Beneficios

  • Gestión Centralizada de Datos: Gestiona y actualiza fácilmente los datos del juego desde una única ubicación.
  • Reducción del Uso de Memoria: Los Objetos Scriptables se comparten entre instancias, reduciendo la sobrecarga de memoria.
  • Integración con el Inspector: Modifica los datos directamente en el Editor de Unity sin recompilar scripts.

Uso Avanzado

Creando Editores Personalizados

  • Inspector Personalizado: Mejora la experiencia del usuario creando inspectores personalizados para tus Objetos Scriptables.
using UnityEditor;
using UnityEngine;

[CustomEditor(typeof(GameData))]
public class GameDataEditor : Editor
{
    public override void OnInspectorGUI()
    {
        GameData gameData = (GameData)target;

        gameData.gameName = EditorGUILayout.TextField("Game Name", gameData.gameName);
        gameData.maxPlayers = EditorGUILayout.IntField("Max Players", gameData.maxPlayers);
        gameData.gameDuration = EditorGUILayout.FloatField("Game Duration", gameData.gameDuration);

        if (GUILayout.Button("Reset Data"))
        {
            gameData.gameName = "My Game";
            gameData.maxPlayers = 4;
            gameData.gameDuration = 60.0f;
        }

        EditorUtility.SetDirty(gameData);
    }
}

Usando Objetos Scriptables para Sistemas de Eventos

  • Gestión de Eventos: Usa Objetos Scriptables para gestionar eventos del juego y desacoplar el manejo de eventos de la lógica del juego.
using UnityEngine;

[CreateAssetMenu(fileName = "NewGameEvent", menuName = "Game Event", order = 52)]
public class GameEvent : ScriptableObject
{
    private readonly List<GameEventListener> eventListeners = new List<GameEventListener>();

    public void Raise()
    {
        for (int i = eventListeners.Count - 1; i >= 0; i--)
        {
            eventListeners[i].OnEventRaised();
        }
    }

    public void RegisterListener(GameEventListener listener)
    {
        if (!eventListeners.Contains(listener))
            eventListeners.Add(listener);
    }

    public void UnregisterListener(GameEventListener listener)
    {
        if (eventListeners.Contains(listener))
            eventListeners.Remove(listener);
    }
}

Conclusión

Los Objetos Scriptables son una forma versátil y eficiente de gestionar datos en Unity. Ayudan a organizar y centralizar datos, reducir el uso de memoria y mejorar la arquitectura general de tu juego. Al dominar los Objetos Scriptables, puedes crear proyectos de Unity más mantenibles y escalables.
