Introducción a la Física en Unity
Colisionadores y Triggers ►

La física en Unity es una característica poderosa que permite a los desarrolladores crear interacciones y comportamientos realistas en sus juegos. Este tema cubrirá los conceptos básicos de la física en Unity, incluyendo conceptos clave, componentes y ejemplos prácticos para ayudarte a comenzar.

Conceptos Clave

  • Motor de Física: Unity utiliza el motor NVIDIA PhysX para simular física. Este motor maneja los cálculos necesarios para movimientos realistas, colisiones y otras interacciones físicas.
  • Rigidbody: Un componente que permite que un GameObject sea afectado por fuerzas físicas como la gravedad y las colisiones.
  • Colliders: Componentes que definen la forma de un GameObject para propósitos de colisiones físicas.
  • Materiales de Física: Activos que definen cómo interactúan las superficies entre sí, incluyendo propiedades como fricción y rebote.
  • Fuerzas: Métodos para aplicar fuerzas físicas a los GameObjects, como la gravedad, impulsos y fuerzas continuas.

Rigidbody

El componente Rigidbody es esencial para cualquier GameObject que desees mover o que sea afectado por la física.

Añadiendo un Rigidbody

Para añadir un Rigidbody a un GameObject:

  1. Selecciona el GameObject en la Jerarquía.
  2. En el Inspector, haz clic en "Añadir Componente".
  3. Busca "Rigidbody" y añádelo.

Propiedades del Rigidbody

  • Masa: El peso del GameObject.
  • Arrastre: Resistencia al movimiento.
  • Arrastre Angular: Resistencia a la rotación.
  • Usar Gravedad: Si el GameObject es afectado por la gravedad.
  • Es Cinemático: Si es verdadero, el GameObject no es afectado por fuerzas físicas pero aún puede interactuar con otros objetos físicos.

Ejemplo: Añadiendo un Rigidbody a un Cubo

using UnityEngine;

public class AddRigidbody : MonoBehaviour
{
    void Start()
    {
        // Añadir un componente Rigidbody al GameObject al que este script está adjunto
        Rigidbody rb = gameObject.AddComponent<Rigidbody>();
        rb.mass = 2.0f; // Establecer la masa del Rigidbody
    }
}

Colliders

Los Colliders definen la forma de un GameObject para la detección de colisiones. Hay varios tipos de colliders:

  • Box Collider: Una forma de prisma rectangular.
  • Sphere Collider: Una forma esférica.
  • Capsule Collider: Una forma de cápsula.
  • Mesh Collider: Utiliza la malla del GameObject para la detección de colisiones.

Añadiendo un Collider

Para añadir un Collider a un GameObject:

  1. Selecciona el GameObject en la Jerarquía.
  2. En el Inspector, haz clic en "Añadir Componente".
  3. Busca el tipo de Collider deseado (por ejemplo, "Box Collider") y añádelo.

Ejemplo: Añadiendo un Box Collider a un Cubo

using UnityEngine;

public class AddBoxCollider : MonoBehaviour
{
    void Start()
    {
        // Añadir un componente Box Collider al GameObject al que este script está adjunto
        BoxCollider boxCollider = gameObject.AddComponent<BoxCollider>();
        boxCollider.size = new Vector3(1, 1, 1); // Establecer el tamaño del Box Collider
    }
}

Materiales de Física

Los Materiales de Física definen cómo interactúan las superficies entre sí. Tienen propiedades como fricción y rebote.

Creando un Material de Física

  1. En la ventana del Proyecto, haz clic derecho y selecciona "Crear" -> "Material de Física".
  2. Nombra el nuevo Material de Física.
  3. En el Inspector, establece las propiedades deseadas (por ejemplo, Fricción Dinámica, Fricción Estática, Rebote).

Aplicando un Material de Física

  1. Selecciona el GameObject con un Collider.
  2. En el componente Collider, encuentra la propiedad "Material".
  3. Arrastra el Material de Física desde la ventana del Proyecto a la propiedad "Material".

Ejemplo: Creando y Aplicando un Material de Rebote

using UnityEngine;

public class ApplyPhysicsMaterial : MonoBehaviour
{
    public PhysicsMaterial2D bouncyMaterial;

    void Start()
    {
        // Añadir un componente Box Collider al GameObject al que este script está adjunto
        BoxCollider2D boxCollider = gameObject.AddComponent<BoxCollider2D>();
        // Aplicar el material de rebote al Box Collider
        boxCollider.sharedMaterial = bouncyMaterial;
    }
}

Aplicando Fuerzas

Las fuerzas pueden ser aplicadas a los Rigidbodies para crear movimiento e interacciones.

Tipos de Fuerzas

  • AddForce: Aplica una fuerza al Rigidbody.
  • AddTorque: Aplica una fuerza rotacional.
  • AddExplosionForce: Aplica una fuerza como si fuera de una explosión.

Ejemplo: Aplicando una Fuerza a un Rigidbody

using UnityEngine;

public class ApplyForce : MonoBehaviour
{
    public float forceAmount = 10.0f;

    void Start()
    {
        Rigidbody rb = gameObject.GetComponent<Rigidbody>();
        if (rb != null)
        {
            // Aplicar una fuerza al Rigidbody
            rb.AddForce(Vector3.up * forceAmount, ForceMode.Impulse);
        }
    }
}

Conclusión

En este tema, cubrimos los conceptos básicos de la física en Unity, incluyendo conceptos clave como Rigidbodies, Colliders, Materiales de Física y la aplicación de fuerzas. Entender estos componentes y cómo interactúan es crucial para crear juegos realistas y atractivos. A medida que progreses, aprenderás técnicas y optimizaciones más avanzadas para aprovechar al máximo el motor de física de Unity.
