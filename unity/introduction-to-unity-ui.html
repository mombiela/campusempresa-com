Introducción a la UI de Unity
Creando y Personalizando Elementos de UI ►

Visión General

La UI de Unity es un sistema poderoso y flexible para crear interfaces de usuario en aplicaciones de Unity. Esta sección cubrirá los conceptos básicos de la UI de Unity, incluyendo cómo crear y manipular elementos de UI, gestionar diseños y manejar interacciones del usuario.

Conceptos Clave

  • Canvas: El componente raíz para todos los elementos de UI en Unity. Actúa como un contenedor para los elementos de UI y determina cómo se renderizan.
  • Elementos de UI: Bloques de construcción básicos para crear interfaces de usuario, como Texto, Imagen, Botón, Deslizador y más.
  • RectTransform: Un componente especial para elementos de UI que permite una posición y escalado flexibles.
  • Anclas y Pivotes: Herramientas para posicionar y alinear elementos de UI en relación con sus contenedores padres.
  • Sistema de Eventos: Gestiona eventos de entrada e interacciones con elementos de UI.

Creando una UI Básica

Configurando el Canvas

  1. Crear un Canvas: Haz clic derecho en la ventana de Jerarquía, ve a UI > Canvas.
  2. Escalador de Canvas: Ajusta el componente Escalador de Canvas para escalar la UI en función del tamaño de la pantalla u otros criterios.

Añadiendo Elementos de UI

  1. Texto: Haz clic derecho en el Canvas, ve a UI > Text. Personaliza el contenido y la apariencia del texto usando el componente Text.
  2. Botón: Haz clic derecho en el Canvas, ve a UI > Button. Personaliza la apariencia del botón y añade un evento OnClick para manejar las interacciones del usuario.

Ejemplo: Creando un Botón Simple

using UnityEngine;
using UnityEngine.UI;

public class ButtonExample : MonoBehaviour
{
    public Button myButton;

    void Start()
    {
        myButton.onClick.AddListener(OnButtonClick);
    }

    void OnButtonClick()
    {
        Debug.Log("¡Botón clickeado!");
    }
}
  • Explicación:
    • Adjunta este script a un GameObject en tu escena.
    • Asigna el componente Button al campo myButton en el Inspector.
    • Cuando se haga clic en el botón, se llamará al método OnButtonClick, registrando un mensaje en la consola.

Gestionando Diseños

Grupos de Diseño

  • Grupo de Diseño Horizontal: Organiza elementos hijos en una línea horizontal.
  • Grupo de Diseño Vertical: Organiza elementos hijos en una línea vertical.
  • Grupo de Diseño de Cuadrícula: Organiza elementos hijos en un formato de cuadrícula.

Ejemplo: Usando un Grupo de Diseño Vertical

  1. Crear un Panel: Haz clic derecho en el Canvas, ve a UI > Panel.
  2. Añadir Grupo de Diseño Vertical: Selecciona el Panel, ve a Añadir Componente > Vertical Layout Group.
  3. Añadir Elementos Hijos: Haz clic derecho en el Panel, ve a UI > Text o UI > Button para añadir elementos hijos. Se organizarán automáticamente de forma vertical.

Manejo de Interacciones del Usuario

Sistema de Eventos

  • Módulo de Entrada Independiente: Maneja la entrada del ratón, teclado y táctil.
  • Disparadores de Eventos: Te permite definir eventos personalizados para elementos de UI.

Ejemplo: Usando Disparadores de Eventos

  1. Añadir Disparador de Eventos: Selecciona un elemento de UI (por ejemplo, Botón), ve a Añadir Componente > Event Trigger.
  2. Añadir Tipo de Evento: Haz clic en Añadir Nuevo Tipo de Evento, selecciona PointerEnter.
  3. Asignar Método: Añade un método que se llamará cuando se dispare el evento.
using UnityEngine;
using UnityEngine.EventSystems;

public class EventTriggerExample : MonoBehaviour, IPointerEnterHandler
{
    public void OnPointerEnter(PointerEventData eventData)
    {
        Debug.Log("¡Puntero entrado!");
    }
}
  • Explicación:
    • Adjunta este script al elemento de UI.
    • Cuando el puntero entre en el elemento, se llamará al método OnPointerEnter, registrando un mensaje en la consola.

Conclusión

La UI de Unity proporciona un conjunto completo de herramientas para crear y gestionar interfaces de usuario en tus proyectos de Unity. Al entender los conceptos básicos de Canvas, elementos de UI, RectTransform y el Sistema de Eventos, puedes crear interfaces interactivas y visualmente atractivas. Practica creando diferentes diseños de UI y manejando varias interacciones del usuario para volverte competente en la UI de Unity.
