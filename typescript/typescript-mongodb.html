Introducción

En esta sección, exploraremos cómo usar TypeScript con MongoDB, una popular base de datos NoSQL. Cubriremos los conceptos básicos de la conexión a MongoDB, la realización de operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y el uso de TypeScript para garantizar la seguridad de tipos y mejorar la calidad del código.

Configuración del Entorno

Antes de comenzar a programar, necesitamos configurar nuestro entorno de desarrollo.

Requisitos Previos

  • Node.js instalado
  • TypeScript instalado
  • MongoDB instalado y en funcionamiento

Instalación de Paquetes Requeridos

Para interactuar con MongoDB en un proyecto TypeScript, necesitamos instalar los siguientes paquetes:

  • mongodb: Controlador de MongoDB para Node.js
  • @types/mongodb: Definiciones de tipos para el controlador de MongoDB
npm install mongodb @types/mongodb

Conectando a MongoDB

Para conectarnos a MongoDB, necesitamos crear una instancia de MongoClient y conectarla a nuestro servidor MongoDB.

Código de Ejemplo

import { MongoClient } from 'mongodb';

const uri = 'mongodb://localhost:27017';
const client = new MongoClient(uri);

async function connectToDatabase() {
    try {
        await client.connect();
        console.log('Conectado a MongoDB');
    } catch (error) {
        console.error('Error al conectar a MongoDB:', error);
    }
}

connectToDatabase();

Explicación

  • Importamos MongoClient del paquete mongodb.
  • Definimos la URI para nuestro servidor MongoDB.
  • Creamos una nueva instancia de MongoClient.
  • Definimos una función asíncrona connectToDatabase para conectarnos al servidor MongoDB.
  • Usamos await client.connect() para establecer la conexión.
  • Manejamos cualquier error de conexión usando un bloque try-catch.

Operaciones CRUD

Las operaciones CRUD son fundamentales al trabajar con bases de datos. Veamos cómo realizar estas operaciones usando TypeScript y MongoDB.

Crear

Para insertar documentos en una colección de MongoDB, usamos los métodos insertOne o insertMany.

async function createDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    const newDocument = { name: 'John Doe', age: 30 };

    try {
        const result = await collection.insertOne(newDocument);
        console.log('Documento insertado con _id:', result.insertedId);
    } catch (error) {
        console.error('Error al insertar documento:', error);
    }
}

createDocument();

Leer

Para leer documentos de una colección de MongoDB, usamos el método find.

async function readDocuments() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    try {
        const documents = await collection.find({}).toArray();
        console.log('Documentos:', documents);
    } catch (error) {
        console.error('Error al leer documentos:', error);
    }
}

readDocuments();

Actualizar

Para actualizar documentos en una colección de MongoDB, usamos los métodos updateOne o updateMany.

async function updateDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    const filter = { name: 'John Doe' };
    const update = { $set: { age: 31 } };

    try {
        const result = await collection.updateOne(filter, update);
        console.log('Documentos coincidentes:', result.matchedCount);
        console.log('Documentos modificados:', result.modifiedCount);
    } catch (error) {
        console.error('Error al actualizar documento:', error);
    }
}

updateDocument();

Eliminar

Para eliminar documentos de una colección de MongoDB, usamos los métodos deleteOne o deleteMany.

async function deleteDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection('mycollection');

    const filter = { name: 'John Doe' };

    try {
        const result = await collection.deleteOne(filter);
        console.log('Documentos eliminados:', result.deletedCount);
    } catch (error) {
        console.error('Error al eliminar documento:', error);
    }
}

deleteDocument();

Uso de Interfaces de TypeScript

Para garantizar la seguridad de tipos, podemos definir interfaces de TypeScript para nuestros documentos.

Ejemplo de Interfaz

interface User {
    name: string;
    age: number;
}

async function createTypedDocument() {
    const db = client.db('mydatabase');
    const collection = db.collection<User>('users');

    const newUser: User = { name: 'Jane Doe', age: 25 };

    try {
        const result = await collection.insertOne(newUser);
        console.log('Documento insertado con _id:', result.insertedId);
    } catch (error) {
        console.error('Error al insertar documento:', error);
    }
}

createTypedDocument();

Explicación

  • Definimos una interfaz User con las propiedades name y age.
  • Usamos la interfaz al definir el tipo de la colección: db.collection<User>('users').
  • Creamos un nuevo documento de usuario que se adhiere a la interfaz User.

Conclusión

En esta sección, cubrimos cómo usar TypeScript con MongoDB. Aprendimos cómo configurar el entorno, conectarnos a MongoDB y realizar operaciones CRUD. Además, vimos cómo usar interfaces de TypeScript para garantizar la seguridad de tipos. Al combinar TypeScript con MongoDB, podemos escribir código más robusto y mantenible.
