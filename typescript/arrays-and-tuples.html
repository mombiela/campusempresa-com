Arrays y Tuplas en TypeScript
Enums ►

Introducción

En esta sección, exploraremos arrays y tuplas en TypeScript. Los arrays y las tuplas son estructuras de datos fundamentales que te permiten almacenar colecciones de valores. Entender cómo usarlas efectivamente es crucial para cualquier programador de TypeScript.

Arrays

Los arrays se utilizan para almacenar múltiples valores en una sola variable. En TypeScript, los arrays pueden ser de un tipo específico o una unión de tipos.

Declaración de Arrays

Puedes declarar arrays en TypeScript usando dos sintaxis:

  1. Usando corchetes ([]):

    let numbers: number[] = [1, 2, 3, 4, 5];
let strings: string[] = ["manzana", "plátano", "cereza"];

  2. Usando el tipo genérico Array:

    let numbers: Array<number> = [1, 2, 3, 4, 5];
let strings: Array<string> = ["manzana", "plátano", "cereza"];

Acceso a Elementos del Array

Puedes acceder a los elementos del array usando su índice, comenzando desde 0:

let fruits: string[] = ["manzana", "plátano", "cereza"];
console.log(fruits[0]); // Salida: manzana
console.log(fruits[1]); // Salida: plátano

Métodos de Array

Los arrays en TypeScript vienen con una variedad de métodos incorporados. Aquí hay algunos de los más comúnmente usados:

  • push(): Añade uno o más elementos al final de un array.

    let numbers: number[] = [1, 2, 3];
numbers.push(4);
console.log(numbers); // Salida: [1, 2, 3, 4]

  • pop(): Elimina el último elemento de un array.

    let numbers: number[] = [1, 2, 3];
numbers.pop();
console.log(numbers); // Salida: [1, 2]

  • map(): Crea un nuevo array con los resultados de llamar a una función proporcionada en cada elemento.

    let numbers: number[] = [1, 2, 3];
let doubled = numbers.map(num => num * 2);
console.log(doubled); // Salida: [2, 4, 6]

  • filter(): Crea un nuevo array con todos los elementos que pasan la prueba implementada por la función proporcionada.

    let numbers: number[] = [1, 2, 3, 4, 5];
let even = numbers.filter(num => num % 2 === 0);
console.log(even); // Salida: [2, 4]

Tuplas

Las tuplas son un tipo especial de array donde se conoce el tipo de cada elemento. Te permiten expresar un array con un número fijo de elementos cuyos tipos son conocidos pero pueden ser diferentes.

Declaración de Tuplas

Puedes declarar tuplas especificando el tipo de cada elemento en el array:

let person: [string, number] = ["Alice", 30];

Acceso a Elementos de la Tupla

Puedes acceder a los elementos de la tupla al igual que con los arrays, usando su índice:

let person: [string, number] = ["Alice", 30];
console.log(person[0]); // Salida: Alice
console.log(person[1]); // Salida: 30

Métodos de Tupla

Las tuplas soportan los mismos métodos que los arrays, pero debes tener cuidado con los tipos:

let person: [string, number] = ["Alice", 30];
person.push("Ingeniera");
console.log(person); // Salida: ["Alice", 30, "Ingeniera"]

Nota: Añadir elementos más allá de los tipos declarados puede llevar a comportamientos inesperados.

Comparación: Arrays vs Tuplas

| Característica | Arrays | Tuplas | |------------------|---------------------------------|---------------------------------| | Longitud | Puede variar | Fija | | Tipos de Elementos| Homogéneos o Unión de Tipos | Heterogéneos | | Caso de Uso | Listas de elementos similares | Estructura fija con diferentes tipos |

Conclusión

Los arrays y las tuplas son estructuras de datos esenciales en TypeScript. Los arrays son flexibles y pueden contener múltiples valores del mismo tipo, mientras que las tuplas tienen una longitud fija y pueden contener valores de diferentes tipos. Entender cómo usar estas estructuras efectivamente te ayudará a escribir código TypeScript más robusto y seguro en cuanto a tipos.
