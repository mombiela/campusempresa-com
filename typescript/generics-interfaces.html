Introducción a los Genéricos

Los genéricos proporcionan una forma de crear componentes reutilizables. Permiten definir un componente que puede trabajar con una variedad de tipos de datos en lugar de uno solo. Esto es especialmente útil en TypeScript para crear código flexible y reutilizable.

¿Por Qué Usar Genéricos con Interfaces?

  • Reutilización: Crear interfaces que puedan manejar múltiples tipos de datos.
  • Seguridad de Tipos: Asegurar que los tipos sean consistentes en todo el código.
  • Flexibilidad: Permitir que funciones y clases operen sobre diferentes tipos de datos sin perder información de tipo.

Definiendo una Interfaz Genérica

Una interfaz genérica puede definirse usando la sintaxis <T>, donde T es un marcador de posición para el tipo que se especificará más adelante.

Ejemplo: Interfaz Genérica Básica

interface Box<T> {
    contents: T;
}

let stringBox: Box<string> = { contents: "¡Hola, Mundo!" };
let numberBox: Box<number> = { contents: 42 };

console.log(stringBox.contents); // Salida: ¡Hola, Mundo!
console.log(numberBox.contents); // Salida: 42
  • Explicación:
    • Box<T> es una interfaz genérica donde T puede ser cualquier tipo.
    • stringBox es una instancia de Box con T como string.
    • numberBox es una instancia de Box con T como number.

Uso de Múltiples Parámetros de Tipo

Puedes definir interfaces con múltiples parámetros de tipo para manejar escenarios más complejos.

Ejemplo: Interfaz Genérica con Múltiples Parámetros de Tipo

interface Pair<K, V> {
    key: K;
    value: V;
}

let numberStringPair: Pair<number, string> = { key: 1, value: "Uno" };
let stringBooleanPair: Pair<string, boolean> = { key: "isActive", value: true };

console.log(numberStringPair.key, numberStringPair.value); // Salida: 1 Uno
console.log(stringBooleanPair.key, stringBooleanPair.value); // Salida: isActive true
  • Explicación:
    • Pair<K, V> es una interfaz genérica con dos parámetros de tipo K y V.
    • numberStringPair es una instancia de Pair con K como number y V como string.
    • stringBooleanPair es una instancia de Pair con K como string y V como boolean.

Restricciones en los Genéricos

A veces, puedes querer restringir los tipos que pueden usarse con tu interfaz genérica. Esto puede hacerse usando restricciones.

Ejemplo: Interfaz Genérica con Restricciones

interface Lengthwise {
    length: number;
}

interface Collection<T extends Lengthwise> {
    items: T[];
    getLength(): number;
}

let stringCollection: Collection<string> = {
    items: ["Hola", "Mundo"],
    getLength: function() {
        return this.items.length;
    }
};

console.log(stringCollection.getLength()); // Salida: 2
  • Explicación:
    • Lengthwise es una interfaz que requiere una propiedad length.
    • Collection<T extends Lengthwise> es una interfaz genérica donde T debe tener una propiedad length.
    • stringCollection es una instancia de Collection con T como string.

Casos de Uso Prácticos

  • Estructuras de Datos: Implementación de pilas, colas y listas enlazadas.
  • Respuestas de API: Manejo de diferentes tipos de respuestas de API de manera segura.
  • Funciones Utilitarias: Creación de funciones que pueden operar sobre varios tipos de datos.

Ejemplo: Interfaz de Pila Genérica

interface Stack<T> {
    push(item: T): void;
    pop(): T | undefined;
    peek(): T | undefined;
    size(): number;
}

class ArrayStack<T> implements Stack<T> {
    private storage: T[] = [];

    push(item: T): void {
        this.storage.push(item);
    }

    pop(): T | undefined {
        return this.storage.pop();
    }

    peek(): T | undefined {
        return this.storage[this.storage.length - 1];
    }

    size(): number {
        return this.storage.length;
    }
}

let numberStack = new ArrayStack<number>();
numberStack.push(10);
numberStack.push(20);
console.log(numberStack.pop()); // Salida: 20
console.log(numberStack.size()); // Salida: 1
  • Explicación:
    • Stack<T> es una interfaz genérica para una estructura de datos de pila.
    • ArrayStack<T> es una clase que implementa la interfaz Stack<T> usando un array.
    • numberStack es una instancia de ArrayStack con T como number.

Conclusión

Usar genéricos con interfaces en TypeScript te permite crear código flexible, reutilizable y seguro. Al entender cómo definir y usar interfaces genéricas, puedes construir aplicaciones más robustas que puedan manejar una variedad de tipos de datos sin sacrificar la seguridad de tipos.
