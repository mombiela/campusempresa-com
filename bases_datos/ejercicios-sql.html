Introducción

En esta sección, nos enfocaremos en practicar y reforzar los conceptos de SQL aprendidos en las lecciones anteriores. Los ejercicios están diseñados para cubrir una variedad de consultas y operaciones comunes en bases de datos, desde las más básicas hasta las más avanzadas.

Ejercicios Básicos

Selección de Datos

  • Objetivo: Practicar la selección de datos de una tabla.
  • Ejercicio: Selecciona todos los registros de la tabla employees.
SELECT * FROM employees;
  • Explicación: Esta consulta selecciona todas las columnas y filas de la tabla employees.

Filtrado de Datos

  • Objetivo: Aprender a filtrar datos usando la cláusula WHERE.
  • Ejercicio: Selecciona todos los empleados cuyo salario sea mayor a 50,000.
SELECT * FROM employees
WHERE salary > 50000;
  • Explicación: La cláusula WHERE se usa para filtrar registros que cumplen con una condición específica.

Ejercicios Intermedios

Ordenación de Datos

  • Objetivo: Practicar la ordenación de resultados.
  • Ejercicio: Selecciona todos los empleados y ordénalos por apellido en orden ascendente.
SELECT * FROM employees
ORDER BY last_name ASC;
  • Explicación: La cláusula ORDER BY se usa para ordenar los resultados de una consulta en orden ascendente (ASC) o descendente (DESC).

Funciones de Agregación

  • Objetivo: Utilizar funciones de agregación para resumir datos.
  • Ejercicio: Calcula el salario promedio de todos los empleados.
SELECT AVG(salary) AS average_salary
FROM employees;
  • Explicación: La función AVG calcula el promedio de una columna numérica. AS se usa para renombrar la columna resultante.

Ejercicios Avanzados

Joins

  • Objetivo: Practicar la combinación de datos de múltiples tablas.
  • Ejercicio: Selecciona los nombres de los empleados y el nombre de sus departamentos.
SELECT employees.first_name, employees.last_name, departments.department_name
FROM employees
JOIN departments ON employees.department_id = departments.department_id;
  • Explicación: La cláusula JOIN se usa para combinar filas de dos o más tablas, basándose en una columna relacionada entre ellas.

Subconsultas

  • Objetivo: Utilizar subconsultas para realizar consultas más complejas.
  • Ejercicio: Selecciona los nombres de los empleados que tienen un salario superior al salario promedio de la empresa.
SELECT first_name, last_name
FROM employees
WHERE salary > (SELECT AVG(salary) FROM employees);
  • Explicación: La subconsulta (SELECT AVG(salary) FROM employees) calcula el salario promedio, y la consulta externa selecciona los empleados cuyo salario es mayor que este promedio.

Conclusión

Estos ejercicios cubren una variedad de operaciones y consultas SQL que son fundamentales para trabajar con bases de datos. Practicar estos ejercicios ayudará a consolidar tus conocimientos y habilidades en SQL, preparándote para manejar tareas más complejas en el futuro. Recuerda que la práctica constante es clave para dominar cualquier lenguaje de consulta.
