Introducción

En esta sección, aprenderás cómo configurar permisos y acceso en Google Data Studio. La correcta configuración de permisos es crucial para asegurar que solo las personas adecuadas tengan acceso a tus informes y datos. Esto no solo protege la información sensible, sino que también facilita la colaboración eficiente.

Tipos de permisos en Google Data Studio

Google Data Studio ofrece varios niveles de permisos que puedes asignar a los usuarios:

  • Propietario: Tiene control total sobre el informe o la fuente de datos. Puede compartir, editar y eliminar el recurso.
  • Editor: Puede modificar el informe o la fuente de datos, pero no puede eliminarlo ni cambiar la propiedad.
  • Visualizador: Solo puede ver el informe o la fuente de datos, sin capacidad para hacer modificaciones.

Configuración de permisos para informes

Asignar permisos a un informe

Para asignar permisos a un informe en Google Data Studio, sigue estos pasos:

  1. Abre el informe en Google Data Studio.
  2. Haz clic en el botón "Compartir" en la esquina superior derecha.
  3. En el cuadro de diálogo de compartir, ingresa las direcciones de correo electrónico de las personas a las que deseas otorgar acceso.
  4. Selecciona el nivel de permiso adecuado (Propietario, Editor, Visualizador).
  5. Haz clic en "Enviar".

Ejemplo de configuración de permisos

1. Abre el informe "Ventas Anuales 2023".
2. Haz clic en "Compartir".
3. Ingresa el correo electrónico "jane.doe@example.com".
4. Selecciona "Editor".
5. Haz clic en "Enviar".

Configuración de permisos para fuentes de datos

Asignar permisos a una fuente de datos

Para asignar permisos a una fuente de datos, sigue estos pasos:

  1. Abre la fuente de datos en Google Data Studio.
  2. Haz clic en el botón "Compartir" en la esquina superior derecha.
  3. En el cuadro de diálogo de compartir, ingresa las direcciones de correo electrónico de las personas a las que deseas otorgar acceso.
  4. Selecciona el nivel de permiso adecuado (Propietario, Editor, Visualizador).
  5. Haz clic en "Enviar".

Ejemplo de configuración de permisos para una fuente de datos

1. Abre la fuente de datos "Base de Datos de Clientes".
2. Haz clic en "Compartir".
3. Ingresa el correo electrónico "john.smith@example.com".
4. Selecciona "Visualizador".
5. Haz clic en "Enviar".

Buenas prácticas para la configuración de permisos

  • Principio de menor privilegio: Otorga solo los permisos necesarios para realizar las tareas asignadas.
  • Revisión periódica: Revisa y actualiza los permisos regularmente para asegurarte de que solo las personas adecuadas tengan acceso.
  • Documentación: Mantén un registro de quién tiene acceso a qué recursos y por qué.

Conclusión

La configuración adecuada de permisos y acceso en Google Data Studio es esencial para proteger tus datos y facilitar la colaboración. Asegúrate de entender los diferentes niveles de permisos y de seguir las mejores prácticas para mantener tus informes y fuentes de datos seguros y bien gestionados.
