Introducción

En este proyecto, aprenderás a desarrollar una aplicación de procesamiento de imágenes utilizando MATLAB. Este proyecto cubrirá varias técnicas de procesamiento de imágenes desde niveles básicos hasta avanzados, incluyendo la lectura y visualización de imágenes, mejora de imágenes, filtrado y detección de bordes.

Objetivos

  • Comprender los conceptos básicos del procesamiento de imágenes.
  • Aprender a leer, mostrar y guardar imágenes en MATLAB.
  • Aplicar técnicas de mejora de imágenes.
  • Implementar métodos de filtrado.
  • Realizar detección de bordes en imágenes.

Prerrequisitos

  • Comprensión básica de la sintaxis y funciones de MATLAB.
  • Familiaridad con operaciones matriciales.

Lectura y Visualización de Imágenes

Conceptos Clave

  • Leer imágenes usando imread.
  • Mostrar imágenes usando imshow.
  • Comprender los tipos de datos de imagen y los espacios de color.

Ejemplo

% Leer una imagen desde un archivo
image = imread('example.jpg');

% Mostrar la imagen
imshow(image);

% Obtener información de la imagen
info = imfinfo('example.jpg');
disp(info);

Explicación

  • imread('example.jpg') lee el archivo de imagen y lo almacena en la variable image.
  • imshow(image) muestra la imagen en una ventana de figura.
  • imfinfo('example.jpg') recupera información sobre el archivo de imagen, como sus dimensiones, tipo de color y formato de archivo.

Mejora de Imágenes

Conceptos Clave

  • Ajuste del brillo y contraste de la imagen.
  • Ecualización de histograma.
  • Afilado de imágenes.

Ejemplo

% Leer una imagen
image = imread('example.jpg');

% Convertir a escala de grises
grayImage = rgb2gray(image);

% Ajustar el brillo
brightImage = imadjust(grayImage, [], [], 1.2);

% Ecualización de histograma
equalizedImage = histeq(grayImage);

% Mostrar resultados
subplot(1, 3, 1), imshow(grayImage), title('Imagen Original en Escala de Grises');
subplot(1, 3, 2), imshow(brightImage), title('Imagen con Brillo Ajustado');
subplot(1, 3, 3), imshow(equalizedImage), title('Imagen con Ecualización de Histograma');

Explicación

  • rgb2gray(image) convierte la imagen RGB a escala de grises.
  • imadjust(grayImage, [], [], 1.2) ajusta el brillo de la imagen en escala de grises.
  • histeq(grayImage) realiza la ecualización del histograma para mejorar el contraste de la imagen.
  • Las funciones subplot e imshow se utilizan para mostrar múltiples imágenes en una sola ventana de figura.

Filtrado

Conceptos Clave

  • Filtros de suavizado (por ejemplo, filtro Gaussiano).
  • Filtros de afilado (por ejemplo, filtro Laplaciano).
  • Filtrado mediano.

Ejemplo

% Leer una imagen
image = imread('example.jpg');

% Convertir a escala de grises
grayImage = rgb2gray(image);

% Aplicar filtro Gaussiano
gaussianFiltered = imgaussfilt(grayImage, 2);

% Aplicar filtro Laplaciano
laplacianFiltered = imfilter(grayImage, fspecial('laplacian'));

% Aplicar filtro Mediano
medianFiltered = medfilt2(grayImage);

% Mostrar resultados
subplot(1, 3, 1), imshow(gaussianFiltered), title('Imagen Filtrada con Gaussiano');
subplot(1, 3, 2), imshow(laplacianFiltered), title('Imagen Filtrada con Laplaciano');
subplot(1, 3, 3), imshow(medianFiltered), title('Imagen Filtrada con Mediano');

Explicación

  • imgaussfilt(grayImage, 2) aplica un filtro Gaussiano con una desviación estándar de 2 a la imagen en escala de grises.
  • imfilter(grayImage, fspecial('laplacian')) aplica un filtro Laplaciano a la imagen en escala de grises.
  • medfilt2(grayImage) aplica un filtro mediano a la imagen en escala de grises.

Detección de Bordes

Conceptos Clave

  • Operador Sobel.
  • Detector de bordes Canny.

Ejemplo

% Leer una imagen
image = imread('example.jpg');

% Convertir a escala de grises
grayImage = rgb2gray(image);

% Aplicar detección de bordes Sobel
sobelEdges = edge(grayImage, 'sobel');

% Aplicar detección de bordes Canny
cannyEdges = edge(grayImage, 'canny');

% Mostrar resultados
subplot(1, 2, 1), imshow(sobelEdges), title('Detección de Bordes Sobel');
subplot(1, 2, 2), imshow(cannyEdges), title('Detección de Bordes Canny');

Explicación

  • edge(grayImage, 'sobel') detecta bordes en la imagen en escala de grises usando el operador Sobel.
  • edge(grayImage, 'canny') detecta bordes en la imagen en escala de grises usando el detector de bordes Canny.

Conclusión

En este proyecto, has aprendido a desarrollar una aplicación de procesamiento de imágenes en MATLAB. Has cubierto técnicas esenciales como la lectura y visualización de imágenes, mejora de imágenes, filtrado y detección de bordes. Estas habilidades son fundamentales para tareas y aplicaciones de procesamiento de imágenes más avanzadas. Continúa practicando y explorando funciones adicionales de MATLAB para mejorar aún más tus capacidades de procesamiento de imágenes.
