Introducción a los Gráficos 3D

Los gráficos 3D en MATLAB te permiten visualizar datos en tres dimensiones, proporcionando una comprensión más profunda de las relaciones y estructuras de los datos. Esta sección te guiará desde lo básico hasta técnicas avanzadas en gráficos 3D.

Gráficos 3D Básicos

Creación de un Gráfico 3D Simple

Para crear un gráfico 3D básico, puedes usar la función plot3. Esta función requiere tres vectores: X, Y y Z.

% Definir los datos
X = 1:10;
Y = rand(1, 10);
Z = rand(1, 10);

% Crear el gráfico 3D
figure;
plot3(X, Y, Z, '-o');
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');
title('Gráfico 3D Simple');
grid on;

Gráficos de Dispersión

Para puntos de datos dispersos, usa la función scatter3.

% Definir los datos
X = rand(1, 50);
Y = rand(1, 50);
Z = rand(1, 50);

% Crear el gráfico de dispersión 3D
figure;
scatter3(X, Y, Z, 'filled');
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');
title('Gráfico de Dispersión 3D');
grid on;

Gráficos de Superficie

Los gráficos de superficie son útiles para visualizar funciones de dos variables.

Creación de un Gráfico de Superficie

Usa la función surf para crear un gráfico de superficie.

% Definir los datos
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crear el gráfico de superficie
figure;
surf(X, Y, Z);
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');
title('Gráfico de Superficie');
colorbar;

Gráficos de Malla

Los gráficos de malla son similares a los gráficos de superficie pero muestran solo los bordes.

% Definir los datos
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crear el gráfico de malla
figure;
mesh(X, Y, Z);
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');
title('Gráfico de Malla');

Técnicas Avanzadas de Gráficos 3D

Gráficos de Contorno

Los gráficos de contorno se pueden agregar a los gráficos de superficie para mostrar curvas de nivel.

% Definir los datos
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crear el gráfico de superficie con contornos
figure;
surf(X, Y, Z);
hold on;
contour3(X, Y, Z, 20, 'k');
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');
title('Gráfico de Superficie con Contornos');
colorbar;

Personalización de Gráficos 3D

Puedes personalizar los gráficos 3D cambiando propiedades como el color, la iluminación y los ángulos de vista.

% Definir los datos
[X, Y] = meshgrid(-5:0.5:5, -5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));

% Crear el gráfico de superficie
figure;
surf(X, Y, Z);
xlabel('Eje X');
ylabel('Eje Y');
zlabel('Eje Z');
title('Gráfico de Superficie Personalizado');
colorbar;

% Personalizar el gráfico
shading interp; % Transiciones de color suaves
colormap jet; % Cambiar el mapa de colores
lighting phong; % Añadir iluminación
camlight headlight; % Añadir una fuente de luz
view([-30, 30]); % Cambiar el ángulo de vista

Conclusión

Los gráficos 3D en MATLAB proporcionan herramientas poderosas para visualizar y analizar datos en tres dimensiones. Comenzando con gráficos básicos como plot3 y scatter3, puedes avanzar a visualizaciones más complejas como gráficos de superficie y de malla. Las opciones avanzadas de personalización te permiten mejorar la legibilidad y la estética de tus gráficos. Al dominar estas técnicas, puedes comunicar eficazmente ideas complejas de datos.
