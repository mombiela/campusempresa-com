Introducción

En esta sección, exploraremos cómo importar datos desde archivos de Excel y archivos de texto en MATLAB. Esta es una habilidad crucial para el análisis de datos, ya que te permite trabajar con datos del mundo real almacenados en varios formatos.

Importación de Datos desde Archivos de Excel

Conceptos Clave

  • Formatos de Archivos de Excel: MATLAB soporta los formatos .xls, .xlsx, .xlsm, y .xlsb.
  • Funciones: readtable, xlsread, readmatrix, y readcell.

Uso de `readtable`

La función readtable es una de las más versátiles para importar datos desde archivos de Excel. Lee los datos en una tabla, que es un tipo de dato conveniente para manejar tipos de datos mixtos.

% Ejemplo: Importando datos desde un archivo de Excel
filename = 'data.xlsx';
dataTable = readtable(filename);
disp(dataTable);

Uso de `xlsread`

La función xlsread lee datos numéricos y datos de texto por separado. Es útil para versiones más antiguas de MATLAB.

% Ejemplo: Importando datos usando xlsread
filename = 'data.xlsx';
[num, txt, raw] = xlsread(filename);
disp(num);
disp(txt);
disp(raw);

Uso de `readmatrix`

La función readmatrix lee datos en una matriz, que es adecuada para datos puramente numéricos.

% Ejemplo: Importando datos numéricos usando readmatrix
filename = 'data.xlsx';
dataMatrix = readmatrix(filename);
disp(dataMatrix);

Uso de `readcell`

La función readcell lee datos en un array de celdas, que puede manejar tipos de datos mixtos.

% Ejemplo: Importando datos usando readcell
filename = 'data.xlsx';
dataCell = readcell(filename);
disp(dataCell);

Importación de Datos desde Archivos de Texto

Conceptos Clave

  • Formatos de Archivos de Texto: MATLAB soporta los formatos .txt, .csv, y otros archivos de texto delimitados.
  • Funciones: readtable, textscan, fscanf, y readmatrix.

Uso de `readtable`

La función readtable también puede ser usada para importar datos desde archivos de texto.

% Ejemplo: Importando datos desde un archivo CSV
filename = 'data.csv';
dataTable = readtable(filename);
disp(dataTable);

Uso de `textscan`

La función textscan proporciona más control sobre cómo se leen los datos, permitiéndote especificar delimitadores y formatos.

% Ejemplo: Importando datos usando textscan
filename = 'data.txt';
fileID = fopen(filename, 'r');
data = textscan(fileID, '%f %s %f', 'Delimiter', ',');
fclose(fileID);
disp(data);

Uso de `fscanf`

La función fscanf lee datos en un array, lo cual es útil para archivos de texto simples y estructurados.

% Ejemplo: Importando datos usando fscanf
filename = 'data.txt';
fileID = fopen(filename, 'r');
dataArray = fscanf(fileID, '%f');
fclose(fileID);
disp(dataArray);

Uso de `readmatrix`

La función readmatrix también puede ser usada para archivos de texto, particularmente para datos numéricos.

% Ejemplo: Importando datos numéricos desde un archivo CSV
filename = 'data.csv';
dataMatrix = readmatrix(filename);
disp(dataMatrix);

Conclusión

Importar datos desde archivos de Excel y texto en MATLAB es una habilidad fundamental para el análisis de datos. Usando funciones como readtable, xlsread, readmatrix, readcell, textscan, y fscanf, puedes traer eficientemente datos externos a MATLAB para su posterior procesamiento y análisis. Cada función tiene sus fortalezas y está adecuada para diferentes tipos de datos y formatos de archivo. Practica usando estas funciones con varios archivos de datos para volverte competente en la importación de datos.
