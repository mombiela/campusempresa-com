Resumen

MATLAB (Matrix Laboratory) es un lenguaje de alto rendimiento para la computación técnica. Integra computación, visualización y programación en un entorno fácil de usar donde los problemas y soluciones se expresan en notación matemática familiar.

Conceptos Clave

  • Interfaz de MATLAB: Comprender la disposición y los componentes del entorno de MATLAB.
  • Comandos Básicos: Aprender comandos esenciales para navegar y utilizar MATLAB.
  • Variables y Tipos de Datos: Introducción a las variables y los tipos de datos que MATLAB puede manejar.
  • Scripts y Funciones: Escribir y ejecutar scripts y funciones en MATLAB.

Interfaz de MATLAB

Cuando abres MATLAB, te encuentras con varios componentes clave:

  • Ventana de Comandos: El área principal donde puedes ingresar comandos y ver los resultados.
  • Espacio de Trabajo: Muestra las variables que has creado y sus valores.
  • Historial de Comandos: Mantiene un registro de los comandos que has ingresado.
  • Carpeta Actual: Muestra los archivos en el directorio actual.
  • Editor: Donde puedes escribir, editar y guardar scripts y funciones.

Ejemplo

% Esto es un comentario
disp('¡Hola, Mundo!') % Mostrar un mensaje

En la Ventana de Comandos, escribe el código anterior para mostrar "¡Hola, Mundo!".

Comandos Básicos

Aquí hay algunos comandos básicos para comenzar:

  • clc: Limpia la Ventana de Comandos.
  • clear: Elimina todas las variables del espacio de trabajo.
  • who: Lista todas las variables en el espacio de trabajo.
  • whos: Proporciona información detallada sobre las variables.

Ejemplo

a = 5; % Asigna el valor 5 a la variable a
b = 10; % Asigna el valor 10 a la variable b
sum = a + b; % Suma a y b, almacena el resultado en sum
disp(sum) % Muestra el valor de sum

Variables y Tipos de Datos

MATLAB soporta varios tipos de datos, incluyendo:

  • Tipos Numéricos: Enteros, números de punto flotante.
  • Caracteres y Cadenas: Datos de texto.
  • Tipos Lógicos: Valores booleanos (true o false).
  • Arreglos y Matrices: Fundamental en MATLAB, que significa Laboratorio de Matrices.

Ejemplo

x = 3.14; % Número de punto flotante
y = 'Hola'; % Cadena
z = true; % Valor lógico
A = [1, 2; 3, 4]; % Matriz 2x2

Scripts y Funciones

Los scripts y funciones son esenciales para automatizar tareas y organizar código.

  • Scripts: Archivos que contienen una secuencia de comandos de MATLAB.
  • Funciones: Archivos que aceptan argumentos de entrada y devuelven salidas.

Ejemplo de Script

Crea un archivo llamado myscript.m:

% myscript.m
a = 2;
b = 3;
c = a + b;
disp(c)

Ejecuta el script escribiendo myscript en la Ventana de Comandos.

Ejemplo de Función

Crea un archivo llamado myfunction.m:

function result = myfunction(x, y)
    result = x + y;
end

Llama a la función desde la Ventana de Comandos:

sum = myfunction(5, 7);
disp(sum)

Conclusión

En esta introducción, cubrimos los conceptos básicos del entorno de MATLAB, incluyendo la interfaz, comandos básicos, variables y tipos de datos, y cómo escribir scripts y funciones. Comprender estos fundamentos proporcionará una base sólida para temas más avanzados en la programación con MATLAB.
