Introducción

Leer y escribir archivos es una habilidad fundamental en MATLAB, esencial para el análisis de datos, guardar resultados y compartir información. Esta sección te guiará a través de los conceptos básicos de E/S (Entrada/Salida) de archivos en MATLAB, progresando desde técnicas para principiantes hasta técnicas avanzadas.

Operaciones Básicas de Archivos

Lectura de Archivos

MATLAB proporciona varias funciones para leer datos de archivos. Las funciones más comúnmente usadas son fopen, fclose, fread, fgets y fscanf.

Ejemplo: Lectura de un Archivo de Texto

% Abrir el archivo para lectura
fileID = fopen('example.txt', 'r');

% Verificar si el archivo se abrió correctamente
if fileID == -1
    error('No se pudo abrir el archivo.');
end

% Leer el archivo línea por línea
while ~feof(fileID)
    line = fgets(fileID);
    disp(line);
end

% Cerrar el archivo
fclose(fileID);
  • fopen: Abre el archivo.
  • fgets: Lee una línea del archivo.
  • feof: Verifica si se ha alcanzado el final del archivo.
  • fclose: Cierra el archivo.

Escritura de Archivos

Escribir datos en archivos es igualmente importante. Las funciones de MATLAB como fprintf, fwrite y fputc se usan comúnmente para este propósito.

Ejemplo: Escritura en un Archivo de Texto

% Abrir el archivo para escritura
fileID = fopen('output.txt', 'w');

% Verificar si el archivo se abrió correctamente
if fileID == -1
    error('No se pudo abrir el archivo.');
end

% Escribir datos en el archivo
fprintf(fileID, '¡Hola, Mundo!\n');
fprintf(fileID, 'Esto es una prueba.\n');

% Cerrar el archivo
fclose(fileID);
  • fprintf: Escribe datos formateados en el archivo.

Operaciones Intermedias de Archivos

Lectura y Escritura de Archivos Binarios

Los archivos binarios se usan para almacenamiento eficiente y acceso rápido. Las funciones de MATLAB fread y fwrite se usan para operaciones con archivos binarios.

Ejemplo: Escritura y Lectura de un Archivo Binario

% Crear algunos datos
data = rand(10, 1);

% Abrir el archivo para escritura en modo binario
fileID = fopen('data.bin', 'w');

% Escribir los datos en el archivo
fwrite(fileID, data, 'double');

% Cerrar el archivo
fclose(fileID);

% Abrir el archivo para lectura en modo binario
fileID = fopen('data.bin', 'r');

% Leer los datos del archivo
readData = fread(fileID, [10, 1], 'double');

% Cerrar el archivo
fclose(fileID);

% Mostrar los datos leídos
disp(readData);
  • fwrite: Escribe datos binarios en el archivo.
  • fread: Lee datos binarios del archivo.

Uso de `textscan` para Datos Estructurados

textscan es útil para leer datos estructurados de archivos de texto.

Ejemplo: Lectura de Datos Estructurados

% Abrir el archivo para lectura
fileID = fopen('data.txt', 'r');

% Definir el formato de los datos
formatSpec = '%f %s %d';

% Leer los datos
data = textscan(fileID, formatSpec);

% Cerrar el archivo
fclose(fileID);

% Mostrar los datos
disp(data);
  • textscan: Lee datos formateados de un archivo de texto.

Operaciones Avanzadas de Archivos

Trabajar con Archivos Grandes

Para archivos grandes, la lectura y escritura eficiente son cruciales. MATLAB proporciona funciones como matfile para manejar archivos MAT grandes.

Ejemplo: Uso de matfile para Datos Grandes

% Crear un objeto de archivo MAT
matObj = matfile('largeData.mat', 'Writable', true);

% Escribir datos grandes en el archivo
matObj.largeData = rand(1e6, 1);

% Leer una porción de los datos
partialData = matObj.largeData(1:1000);

% Mostrar los datos parciales
disp(partialData);
  • matfile: Proporciona acceso a variables en archivos MAT sin cargarlas en memoria.

Formatos de Archivo Personalizados

Crear y leer formatos de archivo personalizados se puede hacer usando funciones de E/S de archivos de bajo nivel.

Ejemplo: Formato de Archivo Binario Personalizado

% Definir datos personalizados
header = 'CustomHeader';
data = rand(10, 1);

% Abrir el archivo para escritura
fileID = fopen('custom.bin', 'w');

% Escribir el encabezado
fwrite(fileID, header, 'char');

% Escribir los datos
fwrite(fileID, data, 'double');

% Cerrar el archivo
fclose(fileID);

% Abrir el archivo para lectura
fileID = fopen('custom.bin', 'r');

% Leer el encabezado
readHeader = fread(fileID, length(header), 'char=>char')';

% Leer los datos
readData = fread(fileID, [10, 1], 'double');

% Cerrar el archivo
fclose(fileID);

% Mostrar los datos leídos
disp(readHeader);
disp(readData);
  • Los formatos de archivo personalizados se pueden gestionar combinando diferentes operaciones de lectura/escritura.

Conclusión

Leer y escribir archivos en MATLAB es una habilidad versátil que abarca desde operaciones simples con archivos de texto hasta el manejo de grandes conjuntos de datos y formatos de archivo personalizados. Al dominar estas técnicas, puedes gestionar eficientemente datos dentro de tus aplicaciones MATLAB, asegurando un código robusto y escalable.
