Introducción

Los arrays son estructuras de datos fundamentales en MATLAB, utilizadas para almacenar y manipular colecciones de datos. Esta sección te guiará a través de los conceptos básicos de la creación de arrays, el acceso a sus elementos y la realización de operaciones comunes sobre ellos. Al final de esta sección, te sentirás cómodo trabajando con arrays en MATLAB, desde la creación básica hasta técnicas avanzadas de manipulación.

Creación de Arrays

1. Creación de Vectores Fila y Columna

  • Vector Fila: Un vector fila es un array de 1xN, donde N es el número de elementos.
  • Vector Columna: Un vector columna es un array de Nx1.
% Creación de un vector fila
row_vector = [1, 2, 3, 4, 5];

% Creación de un vector columna
column_vector = [1; 2; 3; 4; 5];

2. Creación de Matrices

  • Matriz: Una matriz es un array 2D con M filas y N columnas.
% Creación de una matriz de 2x3
matrix = [1, 2, 3; 4, 5, 6];

3. Uso de Funciones Incorporadas

  • zeros: Crea un array de ceros.
  • ones: Crea un array de unos.
  • eye: Crea una matriz identidad.
  • rand: Crea un array con valores aleatorios entre 0 y 1.
% Creación de una matriz de 3x3 de ceros
zero_matrix = zeros(3, 3);

% Creación de una matriz de 2x4 de unos
one_matrix = ones(2, 4);

% Creación de una matriz identidad de 4x4
identity_matrix = eye(4);

% Creación de una matriz de 3x3 con valores aleatorios
random_matrix = rand(3, 3);

Acceso y Modificación de Elementos de un Array

1. Indexación

  • MATLAB usa indexación basada en 1.
  • Puedes acceder a los elementos usando paréntesis ().
% Acceso al elemento en la fila 2, columna 3
element = matrix(2, 3);

% Acceso a toda la segunda fila
second_row = matrix(2, :);

% Acceso a toda la tercera columna
third_column = matrix(:, 3);

2. Modificación de Elementos

  • Puedes modificar elementos asignando nuevos valores.
% Cambio del elemento en la fila 1, columna 2 a 10
matrix(1, 2) = 10;

% Cambio de toda la primera fila
matrix(1, :) = [7, 8, 9];

Operaciones Comunes con Arrays

1. Operaciones Aritméticas

  • Puedes realizar operaciones elemento a elemento usando .*, ./, y .^.
% Multiplicación elemento a elemento
result = matrix .* 2;

% División elemento a elemento
result = matrix ./ 2;

% Potencia elemento a elemento
result = matrix .^ 2;

2. Operaciones con Matrices

  • MATLAB soporta operaciones estándar con matrices como suma, resta, multiplicación y transposición.
% Suma de matrices
result = matrix + ones(2, 3);

% Multiplicación de matrices
result = matrix * eye(3);

% Transposición de matrices
transposed_matrix = matrix';

3. Cambio de Forma de Arrays

  • Puedes cambiar la forma de un array usando la función reshape.
% Cambio de forma de una matriz de 2x3 a una matriz de 3x2
reshaped_matrix = reshape(matrix, 3, 2);

Manipulación Avanzada de Arrays

1. Concatenación

  • Puedes concatenar arrays horizontal o verticalmente.
% Concatenación horizontal
h_concat = [matrix, ones(2, 3)];

% Concatenación vertical
v_concat = [matrix; ones(2, 3)];

2. Indexación Lógica

  • Usa condiciones lógicas para indexar y manipular arrays.
% Creación de un array lógico
logical_array = matrix > 3;

% Uso de indexación lógica para modificar elementos
matrix(matrix > 3) = 0;

3. Búsqueda de Elementos

  • Usa funciones como find para localizar elementos que cumplan una condición.
% Búsqueda de índices de elementos mayores que 3
indices = find(matrix > 3);

Conclusión

En esta sección, cubrimos los conceptos básicos de la creación y manipulación de arrays en MATLAB. Aprendiste a crear vectores y matrices, acceder y modificar sus elementos, realizar operaciones aritméticas y con matrices comunes, cambiar la forma de los arrays y usar técnicas avanzadas como la concatenación y la indexación lógica. Dominar estas habilidades te permitirá manejar datos de manera eficiente y realizar cálculos complejos en MATLAB.
