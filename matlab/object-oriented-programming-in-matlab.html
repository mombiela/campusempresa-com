Introducción a la Programación Orientada a Objetos (POO)

La Programación Orientada a Objetos (POO) es un paradigma de programación que utiliza "objetos" para diseñar aplicaciones y programas informáticos. MATLAB soporta POO, permitiéndote crear clases, objetos y usar herencia y polimorfismo.

Conceptos Clave de la POO

  • Clase: Un plano para crear objetos (una estructura de datos particular).
  • Objeto: Una instancia de una clase.
  • Propiedades: Variables que pertenecen a un objeto o clase.
  • Métodos: Funciones que pertenecen a un objeto o clase.
  • Herencia: Mecanismo para crear una nueva clase utilizando propiedades y métodos de una clase existente.
  • Polimorfismo: Capacidad de procesar objetos de manera diferente según su clase.

Creación de Clases y Objetos en MATLAB

Definiendo una Clase

En MATLAB, una clase se define usando la palabra clave classdef. A continuación se muestra un ejemplo de una definición de clase simple:

classdef Circle
    properties
        Radius
    end
    methods
        function obj = Circle(r)
            if nargin > 0
                obj.Radius = r;
            end
        end
        function area = getArea(obj)
            area = pi * obj.Radius^2;
        end
    end
end

Creando un Objeto

Para crear un objeto de la clase Circle, se utiliza el constructor de la clase:

c = Circle(5);
disp(c.Radius); % Salida: 5
disp(c.getArea()); % Salida: 78.5398

Propiedades y Métodos

Propiedades

Las propiedades son variables que pertenecen a una clase. Se definen dentro del bloque properties.

classdef Car
    properties
        Make
        Model
        Year
    end
end

Métodos

Los métodos son funciones que operan sobre objetos de la clase. Se definen dentro del bloque methods.

classdef Car
    properties
        Make
        Model
        Year
    end
    methods
        function obj = Car(make, model, year)
            obj.Make = make;
            obj.Model = model;
            obj.Year = year;
        end
        function age = getCarAge(obj)
            currentYear = year(datetime('now'));
            age = currentYear - obj.Year;
        end
    end
end

Ejemplo de Uso

myCar = Car('Toyota', 'Camry', 2015);
disp(myCar.Make); % Salida: Toyota
disp(myCar.getCarAge()); % Salida: (año actual - 2015)

Herencia

Definiendo una Subclase

La herencia permite que una nueva clase herede propiedades y métodos de una clase existente.

classdef ElectricCar < Car
    properties
        BatteryCapacity
    end
    methods
        function obj = ElectricCar(make, model, year, batteryCapacity)
            obj@Car(make, model, year); % Llamar al constructor de la superclase
            obj.BatteryCapacity = batteryCapacity;
        end
        function range = getRange(obj)
            range = obj.BatteryCapacity * 5; % Cálculo de ejemplo
        end
    end
end

Ejemplo de Uso

myElectricCar = ElectricCar('Tesla', 'Model S', 2020, 100);
disp(myElectricCar.Make); % Salida: Tesla
disp(myElectricCar.getRange()); % Salida: 500

Polimorfismo

Sobrescritura de Métodos

El polimorfismo en MATLAB se puede lograr mediante la sobrescritura de métodos, donde una subclase proporciona una implementación específica de un método que ya está definido en su superclase.

classdef SportsCar < Car
    methods
        function speed = getTopSpeed(obj)
            speed = 200; % Velocidad de ejemplo
        end
    end
end

classdef ElectricSportsCar < SportsCar
    methods
        function speed = getTopSpeed(obj)
            speed = 250; % Método sobrescrito
        end
    end
end

Ejemplo de Uso

mySportsCar = SportsCar('Ferrari', '488', 2019);
disp(mySportsCar.getTopSpeed()); % Salida: 200

myElectricSportsCar = ElectricSportsCar('Tesla', 'Roadster', 2022);
disp(myElectricSportsCar.getTopSpeed()); % Salida: 250

Resumen

En esta sección, exploramos los conceptos básicos de la Programación Orientada a Objetos en MATLAB. Cubrimos:

  • Los conceptos fundamentales de la POO.
  • Cómo definir clases y crear objetos.
  • El uso de propiedades y métodos.
  • Implementación de la herencia para crear subclases.
  • Lograr polimorfismo a través de la sobrescritura de métodos.

Entender estos conceptos te permitirá diseñar e implementar programas más complejos y modulares en MATLAB, aprovechando el poder de la POO para crear código reutilizable y mantenible.
