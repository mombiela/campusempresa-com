Introducción a ALGOL

ALGOL (Lenguaje Algorítmico) es una familia de lenguajes de programación imperativos desarrollados originalmente a mediados del siglo XX. Fue diseñado para ser un lenguaje universal para expresar algoritmos y ha influido en muchos lenguajes de programación modernos.

Comenzando con ALGOL

Antes de sumergirse en la implementación de algoritmos, es esencial entender los conceptos básicos de la sintaxis y estructura de ALGOL.

Sintaxis Básica

  • Estructura del Programa: Un programa en ALGOL típicamente consiste en una secuencia de declaraciones seguida de una secuencia de sentencias.
  • Variables: Las variables deben ser declaradas antes de ser usadas.
  • Estructuras de Control: ALGOL soporta estructuras de control estándar como bucles y condicionales.

Ejemplo:

begin
  integer i;
  for i := 1 step 1 until 10 do
    print(i);
end

Explicación:

  • begin y end marcan el inicio y el fin del programa.
  • integer i; declara una variable i de tipo entero.
  • El bucle for itera desde 1 hasta 10, imprimiendo cada valor de i.

Algoritmos Básicos

Búsqueda Secuencial

Un algoritmo simple para buscar un elemento en un array.

Ejemplo:

begin
  integer array[1:10];
  integer i, n, key, position;
  position := 0;
  key := 5;  // Elemento a buscar
  for i := 1 step 1 until 10 do
    if array[i] = key then
      position := i;
  if position = 0 then
    print("Elemento no encontrado")
  else
    print("Elemento encontrado en la posición: ", position);
end

Explicación:

  • Se declara un array de enteros.
  • El bucle for itera a través del array para encontrar la key.
  • Si se encuentra la key, se almacena su posición; de lo contrario, se imprime un mensaje.

Ordenamiento de Burbuja

Un algoritmo de ordenamiento básico que recorre repetidamente la lista, compara elementos adyacentes y los intercambia si están en el orden incorrecto.

Ejemplo:

begin
  integer array[1:5];
  integer i, j, temp;
  for i := 1 step 1 until 4 do
    for j := 1 step 1 until 5 - i do
      if array[j] > array[j + 1] then
        temp := array[j];
        array[j] := array[j + 1];
        array[j + 1] := temp;
end

Explicación:

  • El bucle exterior corre desde el primer hasta el penúltimo elemento.
  • El bucle interior compara elementos adyacentes y los intercambia si es necesario.
  • Este proceso se repite hasta que el array esté ordenado.

Algoritmos Intermedios

Búsqueda Binaria

Un algoritmo de búsqueda más eficiente para arrays ordenados.

Ejemplo:

begin
  integer array[1:10];
  integer low, high, mid, key;
  low := 1;
  high := 10;
  key := 5;  // Elemento a buscar
  while low <= high do
    mid := (low + high) div 2;
    if array[mid] = key then
      print("Elemento encontrado en la posición: ", mid);
      exit;
    else if array[mid] < key then
      low := mid + 1;
    else
      high := mid - 1;
  print("Elemento no encontrado");
end

Explicación:

  • low y high marcan el rango de búsqueda.
  • mid es el índice medio del rango actual.
  • El algoritmo reduce el rango de búsqueda basado en la comparación de key y array[mid].

Ordenamiento por Mezcla

Un algoritmo de divide y vencerás que divide el array en mitades, las ordena y luego las combina.

Ejemplo:

procedure mergeSort(array, left, right);
begin
  integer mid;
  if left < right then
    mid := (left + right) div 2;
    mergeSort(array, left, mid);
    mergeSort(array, mid + 1, right);
    merge(array, left, mid, right);
end

procedure merge(array, left, mid, right);
begin
  integer i, j, k;
  integer n1, n2;
  n1 := mid - left + 1;
  n2 := right - mid;
  integer L[1:n1], R[1:n2];
  for i := 1 step 1 until n1 do
    L[i] := array[left + i - 1];
  for j := 1 step 1 until n2 do
    R[j] := array[mid + j];
  i := 1; j := 1; k := left;
  while i <= n1 and j <= n2 do
    if L[i] <= R[j] then
      array[k] := L[i];
      i := i + 1;
    else
      array[k] := R[j];
      j := j + 1;
    k := k + 1;
  while i <= n1 do
    array[k] := L[i];
    i := i + 1;
    k := k + 1;
  while j <= n2 do
    array[k] := R[j];
    j := j + 1;
    k := k + 1;
end

begin
  integer array[1:10];
  mergeSort(array, 1, 10);
end

Explicación:

  • mergeSort divide recursivamente el array en mitades.
  • merge combina dos mitades ordenadas en un solo array ordenado.
  • Los arrays temporales L y R almacenan las mitades izquierda y derecha.

Algoritmos Avanzados

Programación Dinámica: Secuencia de Fibonacci

Un enfoque optimizado para calcular la secuencia de Fibonacci usando memoización.

Ejemplo:

integer array[1:100];

procedure fibonacci(n);
begin
  if n <= 1 then
    return n;
  if array[n] != 0 then
    return array[n];
  array[n] := fibonacci(n - 1) + fibonacci(n - 2);
  return array[n];
end

begin
  integer n;
  n := 10;  // Calcular el décimo número de Fibonacci
  print(fibonacci(n));
end

Explicación:

  • El procedimiento fibonacci usa memoización para almacenar valores previamente calculados.
  • Esto reduce la complejidad temporal de exponencial a lineal.

Algoritmos de Grafos: Algoritmo de Dijkstra

Un algoritmo para encontrar el camino más corto en un grafo.

Ejemplo:

procedure dijkstra(graph, source, n);
begin
  integer dist[1:n], visited[1:n], i, j, minDist, nextNode;
  for i := 1 step 1 until n do
    dist[i] := infinity;
    visited[i] := 0;
  dist[source] := 0;
  for i := 1 step 1 until n - 1 do
    minDist := infinity;
    for j := 1 step 1 until n do
      if dist[j] < minDist and visited[j] = 0 then
        minDist := dist[j];
        nextNode := j;
    visited[nextNode] := 1;
    for j := 1 step 1 until n do
      if visited[j] = 0 and dist[nextNode] + graph[nextNode][j] < dist[j] then
        dist[j] := dist[nextNode] + graph[nextNode][j];
  for i := 1 step 1 until n do
    print("Distancia desde la fuente al nodo ", i, ": ", dist[i]);
end

begin
  integer graph[1:5][1:5];
  dijkstra(graph, 1, 5);
end

Explicación:

  • El array dist almacena la distancia más corta desde la fuente a cada nodo.
  • El array visited lleva el registro de los nodos visitados.
  • El algoritmo selecciona iterativamente el nodo no visitado con la distancia más pequeña y actualiza las distancias de sus vecinos.
