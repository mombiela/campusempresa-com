Introducción

En esta sección, exploraremos los conceptos fundamentales de operadores y expresiones en ALGOL. Comprender estos conceptos es crucial para escribir programas en ALGOL de manera eficiente y efectiva. Cubriremos varios tipos de operadores, su precedencia y cómo construir expresiones.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar cálculos matemáticos. ALGOL soporta los siguientes operadores aritméticos:

  • + : Adición
  • - : Sustracción
  • * : Multiplicación
  • / : División
  • := : Asignación

Ejemplos

begin
  integer a, b, c;
  a := 10;
  b := 5;
  c := a + b;  ! c es ahora 15
  c := a - b;  ! c es ahora 5
  c := a * b;  ! c es ahora 50
  c := a / b;  ! c es ahora 2
end

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores. El resultado de una operación relacional es un valor booleano (true o false). ALGOL soporta los siguientes operadores relacionales:

  • = : Igual a
  • ≠ : No igual a
  • < : Menor que
  • <= : Menor o igual que
  • > : Mayor que
  • >= : Mayor o igual que

Ejemplos

begin
  integer x, y;
  x := 10;
  y := 5;
  if x = y then
    print("x es igual a y")
  else
    print("x no es igual a y");  ! Esto imprimirá "x no es igual a y"
end

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones booleanas. ALGOL soporta los siguientes operadores lógicos:

  • and : Y lógico
  • or : O lógico
  • not : NO lógico

Ejemplos

begin
  boolean a, b, c;
  a := true;
  b := false;
  c := a and b;  ! c es ahora false
  c := a or b;   ! c es ahora true
  c := not a;    ! c es ahora false
end

Precedencia de Operadores

La precedencia de operadores determina el orden en el que los operadores son evaluados en una expresión. La siguiente tabla resume la precedencia de los operadores en ALGOL, de mayor a menor:

| Nivel de Precedencia | Operadores | |----------------------|---------------------------| | 1 | not | | 2 | *, / | | 3 | +, - | | 4 | <, <=, >, >= | | 5 | =, ≠ | | 6 | and | | 7 | or | | 8 | := (asignación) |

Ejemplos

begin
  integer a, b, c, d;
  a := 10;
  b := 5;
  c := 2;
  d := a + b * c;  ! d es ahora 20 porque * tiene mayor precedencia que +
  d := (a + b) * c;  ! d es ahora 30 porque los paréntesis cambian la precedencia
end

Expresiones Compuestas

Las expresiones compuestas involucran múltiples operadores y operandos. Los paréntesis pueden ser utilizados para definir explícitamente el orden de evaluación.

Ejemplos

begin
  integer x, y, z;
  x := 10;
  y := 5;
  z := (x + y) * (x - y);  ! z es ahora 75
end

Conclusión

Comprender los operadores y las expresiones es fundamental para programar en ALGOL. Al dominar los operadores aritméticos, relacionales y lógicos, así como su precedencia, puedes escribir programas más complejos y eficientes. Practica construyendo y evaluando expresiones para volverte competente en el uso efectivo de estos operadores.
