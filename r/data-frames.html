Introducción a los Data Frames

Los data frames son una estructura de datos crucial en R, utilizada para almacenar datos tabulares. Son similares a las tablas en una base de datos o a las hojas de cálculo de Excel. Cada columna en un data frame puede contener diferentes tipos de datos (numéricos, caracteres, factores, etc.), lo que los hace versátiles para el análisis de datos.

Conceptos Clave

  • Estructura del Data Frame: Las filas representan observaciones y las columnas representan variables.
  • Creación de Data Frames: Usando la función data.frame().
  • Acceso a los Datos: Usando indexación, nombres de columnas y funciones como subset().

Creación de Data Frames

Para crear un data frame, puedes usar la función data.frame(). Aquí tienes un ejemplo simple:

# Creación de un data frame
df <- data.frame(
  Name = c("Alice", "Bob", "Charlie"),
  Age = c(25, 30, 35),
  Height = c(5.5, 6.0, 5.8)
)

# Mostrar el data frame
print(df)

Explicación

  • Name: Un vector de caracteres que representa nombres.
  • Age: Un vector numérico que representa edades.
  • Height: Un vector numérico que representa alturas.

Acceso a los Datos en Data Frames

Puedes acceder a los datos en un data frame usando varios métodos:

Por Índice

# Acceso a la primera fila
print(df[1, ])

# Acceso a la primera columna
print(df[, 1])

# Acceso a un elemento específico
print(df[1, 2])

Por Nombre de Columna

# Acceso a la columna 'Name'
print(df$Name)

# Acceso a la columna 'Age'
print(df[["Age"]])

Usando la Función `subset()`

# Subconjunto de datos donde Age es mayor que 25
subset_df <- subset(df, Age > 25)
print(subset_df)

Modificación de Data Frames

Puedes modificar los data frames añadiendo o eliminando columnas y filas.

Añadir Columnas

# Añadir una nueva columna 'Weight'
df$Weight <- c(120, 150, 140)
print(df)

Eliminar Columnas

# Eliminar la columna 'Height'
df$Height <- NULL
print(df)

Añadir Filas

# Añadir una nueva fila
new_row <- data.frame(Name = "David", Age = 40, Weight = 160)
df <- rbind(df, new_row)
print(df)

Eliminar Filas

# Eliminar la primera fila
df <- df[-1, ]
print(df)

Operaciones Avanzadas con Data Frames

A medida que te sientas más cómodo con los data frames, puedes realizar operaciones más complejas.

Fusionar Data Frames

Puedes fusionar dos data frames usando la función merge().

# Creación de otro data frame
df2 <- data.frame(
  Name = c("Alice", "Bob", "David"),
  Salary = c(50000, 60000, 70000)
)

# Fusionar data frames por 'Name'
merged_df <- merge(df, df2, por = "Name")
print(merged_df)

Aplicar Funciones a Data Frames

Puedes aplicar funciones a los data frames usando apply(), lapply() y sapply().

# Aplicar una función a cada columna
mean_values <- sapply(df[, -1], mean)
print(mean_values)

Resumen

Los data frames son una estructura de datos fundamental en R, esencial para la manipulación y el análisis de datos. Te permiten almacenar y operar sobre datos tabulares de manera eficiente. Al comprender cómo crear, acceder, modificar y realizar operaciones avanzadas en data frames, puedes manejar una amplia gama de tareas de análisis de datos en R.

Puntos Clave

  • Los data frames almacenan datos tabulares con filas y columnas.
  • Puedes crear data frames usando la función data.frame().
  • Accede a los datos usando indexación, nombres de columnas y la función subset().
  • Modifica los data frames añadiendo/eliminando columnas y filas.
  • Realiza operaciones avanzadas como fusionar y aplicar funciones a los data frames.

Al dominar estos conceptos, estarás bien equipado para manejar tareas de análisis de datos en R.
