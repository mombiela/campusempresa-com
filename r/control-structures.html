Estructuras de Control en R
Depuración y Manejo de Errores ►

Las estructuras de control son fundamentales en la programación ya que te permiten controlar el flujo de ejecución de tu código. En R, las estructuras de control incluyen condicionales, bucles y otros constructos que te permiten escribir programas más complejos y dinámicos. Esta sección cubrirá las estructuras de control esenciales en R, comenzando desde lo básico y avanzando hacia conceptos más avanzados.

Condicionales

Los condicionales te permiten ejecutar diferentes fragmentos de código basados en ciertas condiciones.

Declaración if

La declaración if se usa para ejecutar un bloque de código solo si una condición especificada es verdadera.

x <- 10

if (x > 5) {
  print("x es mayor que 5")
}

Declaración if-else

La declaración if-else proporciona un bloque de código alternativo para ejecutar si la condición es falsa.

x <- 3

if (x > 5) {
  print("x es mayor que 5")
} else {
  print("x no es mayor que 5")
}

Función ifelse

La función ifelse es una versión vectorizada de la declaración if-else, útil para aplicar lógica condicional a vectores.

x <- c(1, 2, 3, 4, 5)
resultado <- ifelse(x > 3, "Mayor", "No Mayor")
print(resultado)

Bucles

Los bucles te permiten ejecutar un bloque de código múltiples veces.

Bucle for

El bucle for se usa para iterar sobre una secuencia de elementos.

for (i en 1:5) {
  print(i)
}

Bucle while

El bucle while continúa ejecutando un bloque de código mientras una condición especificada sea verdadera.

x <- 1

while (x <= 5) {
  print(x)
  x <- x + 1
}

Bucle repeat

El bucle repeat ejecuta un bucle infinito hasta que se encuentra una declaración break.

x <- 1

repeat {
  print(x)
  x <- x + 1
  if (x > 5) {
    break
  }
}

Estructuras de Control Avanzadas

break y next

La declaración break se usa para salir de un bucle, mientras que la declaración next se usa para saltar la iteración actual y proceder a la siguiente.

for (i en 1:10) {
  if (i == 5) {
    break
  }
  print(i)
}

for (i en 1:10) {
  if (i %% 2 == 0) {
    next
  }
  print(i)
}

Declaración switch

La declaración switch te permite ejecutar uno de varios bloques de código basados en el valor de una expresión.

x <- "b"

resultado <- switch(x,
  "a" = "Opción A",
  "b" = "Opción B",
  "c" = "Opción C",
  "Opción Inválida"
)

print(resultado)

Resumen

Las estructuras de control son esenciales para gestionar el flujo de ejecución en tus programas en R. Al dominar los condicionales, bucles y estructuras de control avanzadas, puedes escribir código más eficiente y dinámico. Aquí tienes un resumen rápido:

  • Condicionales: Usa if, if-else y ifelse para ejecutar código basado en condiciones.
  • Bucles: Usa for, while y repeat para ejecutar código múltiples veces.
  • Estructuras de Control Avanzadas: Usa break, next y switch para un control de flujo más complejo.

Entender y usar efectivamente estas estructuras de control mejorará significativamente tu capacidad para escribir programas en R robustos y flexibles.
