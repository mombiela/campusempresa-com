Introducción

En esta sección, exploraremos las operaciones básicas en estructuras de datos en R. Entender estas operaciones es crucial para la manipulación y el análisis de datos. Cubriremos vectores, listas, matrices, marcos de datos y factores, proporcionando ejemplos y ejercicios para solidificar tu comprensión.

Vectores

Los vectores son el tipo más simple de estructura de datos en R. Son secuencias de elementos de datos del mismo tipo básico.

Creación de Vectores

  • Usa la función c() para crear un vector.
# Creando un vector numérico
numeric_vector <- c(1, 2, 3, 4, 5)
print(numeric_vector)

# Creando un vector de caracteres
char_vector <- c("a", "b", "c")
print(char_vector)

Operaciones Básicas

  • Accede a los elementos usando índices.
  • Realiza operaciones aritméticas.
# Accediendo a elementos
first_element <- numeric_vector[1]
print(first_element)

# Operaciones aritméticas
sum_vector <- numeric_vector + 2
print(sum_vector)

Listas

Las listas pueden contener elementos de diferentes tipos, incluyendo otras listas.

Creación de Listas

  • Usa la función list() para crear una lista.
# Creando una lista
my_list <- list(name = "John", age = 25, scores = c(90, 85, 88))
print(my_list)

Operaciones Básicas

  • Accede a los elementos usando $ o [[ ]].
# Accediendo a elementos
name <- my_list$name
print(name)

# Accediendo a elementos anidados
first_score <- my_list$scores[1]
print(first_score)

Matrices

Las matrices son estructuras de datos bidimensionales y homogéneas.

Creación de Matrices

  • Usa la función matrix() para crear una matriz.
# Creando una matriz
my_matrix <- matrix(1:9, nrow = 3, ncol = 3)
print(my_matrix)

Operaciones Básicas

  • Accede a los elementos usando índices de fila y columna.
  • Realiza operaciones con matrices.
# Accediendo a elementos
element <- my_matrix[2, 3]
print(element)

# Multiplicación de matrices
result_matrix <- my_matrix %*% my_matrix
print(result_matrix)

Marcos de Datos

Los marcos de datos son estructuras de datos bidimensionales y heterogéneas.

Creación de Marcos de Datos

  • Usa la función data.frame() para crear un marco de datos.
# Creando un marco de datos
my_data_frame <- data.frame(
  Name = c("John", "Jane", "Doe"),
  Age = c(25, 30, 22),
  Score = c(90, 85, 88)
)
print(my_data_frame)

Operaciones Básicas

  • Accede a los elementos usando $ o índices.
  • Añade nuevas columnas.
# Accediendo a elementos
age <- my_data_frame$Age
print(age)

# Añadiendo una nueva columna
my_data_frame$Gender <- c("M", "F", "M")
print(my_data_frame)

Factores

Los factores se usan para manejar datos categóricos.

Creación de Factores

  • Usa la función factor() para crear un factor.
# Creando un factor
gender <- factor(c("Male", "Female", "Female", "Male"))
print(gender)

Operaciones Básicas

  • Accede a los niveles y cambia los niveles.
# Accediendo a niveles
levels(gender)

# Cambiando niveles
levels(gender) <- c("M", "F")
print(gender)

Resumen

En esta sección, cubrimos las operaciones básicas en varias estructuras de datos en R, incluyendo vectores, listas, matrices, marcos de datos y factores. Estas operaciones forman la base para tareas más avanzadas de manipulación y análisis de datos. Practica estas operaciones con diferentes conjuntos de datos para volverte competente en el manejo de estructuras de datos en R.
