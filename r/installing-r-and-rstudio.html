Introducción

Antes de sumergirse en la programación con R, es esencial tener las herramientas adecuadas instaladas. Esta sección te guiará a través de la instalación de R y RStudio, que son los componentes de software principales que necesitarás.

Instalación de R

R es un entorno de software libre para computación estadística y gráficos. Sigue estos pasos para instalar R en tu sistema:

  1. Visita el Sitio Web de CRAN:

    • Ve a la Red Integral de Archivos de R (CRAN) en https://cran.r-project.org/.

  2. Elige Tu Sistema Operativo:

    • Selecciona tu sistema operativo (Windows, macOS o Linux).

  3. Descarga R:

    • Haz clic en el enlace apropiado para descargar el instalador de R para tu sistema operativo.

  4. Instala R:

    • Windows:
      • Ejecuta el archivo .exe descargado.
      • Sigue las instrucciones de instalación, aceptando la configuración predeterminada.
    • macOS:
      • Abre el archivo .pkg descargado.
      • Sigue las instrucciones de instalación.
    • Linux:
      • Abre una terminal.
      • Usa el gestor de paquetes para tu distribución (por ejemplo, apt para sistemas basados en Debian, yum para sistemas basados en Red Hat) para instalar R. Por ejemplo: 
        sudo apt-get update
sudo apt-get install r-base

Instalación de RStudio

RStudio es un entorno de desarrollo integrado (IDE) para R. Facilita la programación en R al proporcionar una interfaz fácil de usar.

  1. Visita el Sitio Web de RStudio:

    • Ve a https://rstudio.com/products/rstudio/download/.

  2. Descarga RStudio:

    • Elige la versión gratuita de RStudio Desktop.
    • Selecciona el instalador para tu sistema operativo y descárgalo.

  3. Instala RStudio:

    • Windows:
      • Ejecuta el archivo .exe descargado.
      • Sigue las instrucciones de instalación.
    • macOS:
      • Abre el archivo .dmg descargado.
      • Arrastra y suelta el icono de RStudio en la carpeta de Aplicaciones.
    • Linux:
      • Abre una terminal.
      • Usa el gestor de paquetes para tu distribución para instalar RStudio. Por ejemplo, en sistemas basados en Debian: 
        sudo apt-get install gdebi-core
sudo gdebi rstudio-x.yy.zzz-amd64.deb

Verificación de la Instalación

Después de instalar tanto R como RStudio, es importante verificar que todo esté configurado correctamente.

  1. Abre RStudio:

    • Inicia RStudio desde el menú de aplicaciones o el acceso directo en el escritorio.

  2. Verifica la Versión de R:

    • En la consola de RStudio, escribe el siguiente comando y presiona Enter: 
      version
    • Deberías ver una salida similar a esta: 
      platform       x86_64-w64-mingw32
arch           x86_64
os             mingw32
system         x86_64, mingw32
status
major          4
minor          1.2
year           2021
month          11
day            01
svn rev        80317
language       R
version.string R version 4.1.2 (2021-11-01)
nickname       Bird Hippie

  3. Ejecuta un Comando Simple en R:

    • Intenta ejecutar un comando simple para asegurarte de que R esté funcionando correctamente: 
      print("¡Hola, R!")
    • Deberías ver la salida: 
      [1] "¡Hola, R!"

Conclusión

Siguiendo estos pasos, deberías haber instalado con éxito R y RStudio en tu computadora. Estas herramientas proporcionarán un entorno robusto para tu viaje de programación en R, desde niveles principiantes hasta avanzados. Si encuentras algún problema durante la instalación, consulta la documentación oficial o busca ayuda en la comunidad de R.
