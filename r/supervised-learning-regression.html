Introducción a la Regresión

La regresión es un concepto fundamental en el aprendizaje supervisado, donde el objetivo es predecir una variable objetivo continua basada en una o más variables predictoras. En esta sección, cubriremos los conceptos básicos de la regresión, enfocándonos en cómo implementar e interpretar modelos de regresión en R.

Conceptos Clave

  • Variable Dependiente (Objetivo): La variable que estamos tratando de predecir.
  • Variables Independientes (Predictoras): Las variables utilizadas para hacer predicciones.
  • Regresión Lineal: Un método para modelar la relación entre la variable dependiente y una o más variables independientes ajustando una ecuación lineal.
  • Residuos: Las diferencias entre los valores observados y los valores predichos.

Regresión Lineal Simple

La regresión lineal simple implica predecir una variable dependiente usando una sola variable independiente.

Ejemplo

Comencemos con un ejemplo simple para entender cómo realizar una regresión lineal en R.

# Cargar la biblioteca necesaria
library(ggplot2)

# Datos de muestra
data <- data.frame(
  x = c(1, 2, 3, 4, 5),
  y = c(2, 4, 5, 4, 5)
)

# Graficar los datos
ggplot(data, aes(x = x, y = y)) +
  geom_point() +
  geom_smooth(method = "lm", se = FALSE) +
  labs(title = "Regresión Lineal Simple", x = "Variable Independiente (x)", y = "Variable Dependiente (y)")

Explicación

  • Preparación de Datos: Creamos un marco de datos con dos variables x y y.
  • Visualización: Usamos ggplot2 para graficar los puntos de datos y la línea de regresión lineal.

Ajuste del Modelo

# Ajustar el modelo lineal
model <- lm(y ~ x, data = data)

# Resumen del modelo
summary(model)

Explicación

  • Ajuste del Modelo: La función lm se utiliza para ajustar un modelo lineal.
  • Resumen del Modelo: La función summary proporciona información detallada sobre el modelo, incluyendo coeficientes, valor R-cuadrado y valores p.

Regresión Lineal Múltiple

La regresión lineal múltiple implica predecir una variable dependiente usando dos o más variables independientes.

Ejemplo

Extendamos nuestro ejemplo para incluir múltiples predictores.

# Datos de muestra
data <- data.frame(
  x1 = c(1, 2, 3, 4, 5),
  x2 = c(5, 4, 3, 2, 1),
  y = c(2, 3, 4, 5, 6)
)

# Ajustar el modelo lineal múltiple
model <- lm(y ~ x1 + x2, data = data)

# Resumen del modelo
summary(model)

Explicación

  • Preparación de Datos: Creamos un marco de datos con tres variables x1, x2 y y.
  • Ajuste del Modelo: La función lm se utiliza para ajustar un modelo de regresión lineal múltiple.
  • Resumen del Modelo: La función summary proporciona información detallada sobre el modelo.

Evaluación del Modelo

Evaluar el rendimiento de un modelo de regresión es crucial para entender su poder predictivo.

Métricas Clave

  • R-cuadrado: Indica la proporción de la varianza en la variable dependiente que es predecible a partir de las variables independientes.
  • Error Cuadrático Medio (MSE): El promedio de los cuadrados de los residuos.
  • Raíz del Error Cuadrático Medio (RMSE): La raíz cuadrada del MSE, proporcionando una medida de la magnitud promedio de los residuos.

Ejemplo

# Calcular predicciones
predictions <- predict(model, data)

# Calcular residuos
residuals <- data$y - predictions

# Calcular MSE y RMSE
mse <- mean(residuals^2)
rmse <- sqrt(mse)

# Imprimir las métricas
cat("MSE:", mse, "\n")
cat("RMSE:", rmse, "\n")

Explicación

  • Predicciones: La función predict se utiliza para generar predicciones a partir del modelo.
  • Residuos: Calculados como la diferencia entre los valores observados y los valores predichos.
  • MSE y RMSE: Calculados para evaluar el rendimiento del modelo.

Conclusión

En este tema, cubrimos los conceptos básicos del aprendizaje supervisado con un enfoque en la regresión. Exploramos la regresión lineal simple y múltiple, aprendimos cómo ajustar modelos en R y discutimos métricas clave para la evaluación del modelo. Al entender estos conceptos, puedes comenzar a construir y evaluar modelos de regresión para hacer predicciones basadas en datos continuos.
