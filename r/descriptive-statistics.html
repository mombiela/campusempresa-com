Estadísticas Descriptivas en R
Distribuciones de Probabilidad ►

Las estadísticas descriptivas son esenciales para resumir y entender las características principales de un conjunto de datos. En esta sección, cubriremos varias técnicas de estadísticas descriptivas usando R, desde niveles básicos hasta avanzados.

Introducción a las Estadísticas Descriptivas

Las estadísticas descriptivas proporcionan resúmenes simples sobre la muestra y las medidas. Estos resúmenes pueden ser cuantitativos (por ejemplo, media, mediana) o visuales (por ejemplo, gráficos, diagramas).

Conceptos Clave

  • Media: El promedio de un conjunto de números.
  • Mediana: El valor medio en un conjunto de números.
  • Moda: El valor que ocurre con mayor frecuencia en un conjunto de números.
  • Rango: La diferencia entre los valores más altos y más bajos.
  • Varianza: Una medida de cuánto difieren los valores en un conjunto de datos de la media.
  • Desviación Estándar: La raíz cuadrada de la varianza, que indica cuán dispersos están los valores.
  • Cuartiles: Valores que dividen un conjunto de datos en cuatro partes iguales.
  • Rango Intercuartílico (IQR): El rango entre el primer cuartil (Q1) y el tercer cuartil (Q3).

Estadísticas Descriptivas Básicas en R

Calcular Media, Mediana y Moda

# Datos de muestra
data <- c(10, 20, 20, 30, 40, 50, 60)

# Media
mean_value <- mean(data)
print(paste("Media:", mean_value))

# Mediana
median_value <- median(data)
print(paste("Mediana:", median_value))

# Moda (función personalizada ya que R no tiene una función de moda incorporada)
get_mode <- function(v) {
  uniqv <- unique(v)
  uniqv[which.max(tabulate(match(v, uniqv)))]
}
mode_value <- get_mode(data)
print(paste("Moda:", mode_value))

Calcular Rango, Varianza y Desviación Estándar

# Rango
range_value <- range(data)
print(paste("Rango:", range_value))

# Varianza
variance_value <- var(data)
print(paste("Varianza:", variance_value))

# Desviación Estándar
sd_value <- sd(data)
print(paste("Desviación Estándar:", sd_value))

Calcular Cuartiles y IQR

# Cuartiles
quartiles <- quantile(data)
print("Cuartiles:")
print(quartiles)

# Rango Intercuartílico (IQR)
iqr_value <- IQR(data)
print(paste("Rango Intercuartílico (IQR):", iqr_value))

Visualización de Estadísticas Descriptivas

Histogramas

# Histograma
hist(data, main="Histograma de Datos", xlab="Valores", ylab="Frecuencia", col="blue")

Diagramas de Caja

# Diagrama de Caja
boxplot(data, main="Diagrama de Caja de Datos", ylab="Valores", col="red")

Estadísticas Descriptivas Avanzadas en R

Estadísticas Resumidas con la Función `summary()`

# Estadísticas resumidas
summary_stats <- summary(data)
print("Estadísticas Resumidas:")
print(summary_stats)

Usando el Paquete `dplyr` para Estadísticas Descriptivas

# Instalar y cargar el paquete dplyr
install.packages("dplyr")
library(dplyr)

# Crear un marco de datos
df <- data.frame(values = data)

# Estadísticas resumidas usando dplyr
summary_dplyr <- df %>%
  summarise(
    mean = mean(values),
    median = median(values),
    mode = get_mode(values),
    variance = var(values),
    sd = sd(values),
    range = diff(range(values)),
    IQR = IQR(values)
  )
print("Estadísticas Resumidas usando dplyr:")
print(summary_dplyr)

Conclusión

Las estadísticas descriptivas son fundamentales para entender y resumir datos. R proporciona una variedad de funciones y paquetes para calcular y visualizar estas estadísticas. Al dominar estas técnicas, puedes obtener valiosos conocimientos sobre tus datos, lo cual es crucial para cualquier tarea de análisis de datos.

En esta sección, cubrimos:

  • Cálculos básicos de estadísticas descriptivas.
  • Técnicas de visualización usando histogramas y diagramas de caja.
  • Estadísticas resumidas avanzadas usando la función summary() y el paquete dplyr.

Con estas herramientas, estás bien equipado para realizar un análisis estadístico descriptivo completo en R.
