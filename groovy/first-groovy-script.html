Introducción

Groovy es un lenguaje de programación orientado a objetos para la plataforma Java. Es un lenguaje dinámico y poderoso con características similares a las de Python, Ruby y Smalltalk. Groovy se integra perfectamente con todos los objetos y bibliotecas existentes de Java, lo que lo convierte en una excelente opción para los desarrolladores de Java que buscan aumentar su productividad.

En esta sección, escribiremos nuestro primer script en Groovy. Esto te ayudará a familiarizarte con la sintaxis y estructura básica de Groovy.

Configuración del Entorno

Antes de escribir nuestro primer script, necesitamos configurar el entorno:

  1. Instalar Java Development Kit (JDK): Groovy se ejecuta en la plataforma Java, por lo que necesitas tener instalado el JDK.
  2. Instalar Groovy: Puedes descargar e instalar Groovy desde el sitio web oficial o usar un gestor de paquetes como SDKMAN.

Escribiendo Tu Primer Script

Comencemos con un simple script de "Hola, Mundo!" para familiarizarnos con la sintaxis de Groovy.

Hola, Mundo!

Crea un nuevo archivo llamado HelloWorld.groovy y añade el siguiente código:

println 'Hola, Mundo!'

Explicación

  • println: Este es un método de Groovy que imprime el mensaje especificado en la consola, seguido de un carácter de nueva línea.
  • 'Hola, Mundo!': Esto es un literal de cadena en Groovy.

Para ejecutar el script, abre tu terminal o símbolo del sistema, navega al directorio que contiene HelloWorld.groovy, y ejecuta el siguiente comando:

groovy HelloWorld.groovy

Deberías ver la salida:

Hola, Mundo!

Sintaxis y Estructura Básica

Exploremos algunas sintaxis y estructuras básicas de Groovy a través de ejemplos.

Variables

Groovy es de tipado dinámico, lo que significa que no tienes que declarar explícitamente el tipo de una variable.

def name = 'Groovy'
println name
  • def: Usado para declarar una variable.
  • name: El nombre de la variable.
  • 'Groovy': El valor asignado a la variable.

Tipos de Datos

Groovy soporta varios tipos de datos, incluyendo números, cadenas, listas y mapas.

def number = 42
def text = 'Groovy es genial'
def list = [1, 2, 3, 4]
def map = [name: 'Groovy', type: 'lenguaje']

Estructuras de Control

Groovy soporta estructuras de control estándar como if-else, bucles for y bucles while.

If-Else

def age = 18
if (age >= 18) {
    println 'Adulto'
} else {
    println 'Menor'
}

Bucle For

for (i in 1..5) {
    println i
}

Bucle While

def count = 0
while (count < 5) {
    println count
    count++
}

Ejercicios

Prueba los siguientes ejercicios para practicar la escritura de scripts en Groovy:

  1. Imprime Tu Nombre: Escribe un script que imprima tu nombre.
  2. Suma de Números: Escribe un script que calcule e imprima la suma de los números del 1 al 10.
  3. Manipulación de Listas: Crea una lista de tus frutas favoritas e imprime cada fruta usando un bucle.

Conclusión

En esta sección, escribimos nuestro primer script en Groovy y exploramos algunas sintaxis y estructuras básicas. Cubrimos variables, tipos de datos y estructuras de control. Estos fundamentos son esenciales a medida que avanzamos hacia temas más avanzados en Groovy.

Al practicar los ejercicios, ganarás confianza en la escritura de scripts simples en Groovy. En la siguiente sección, profundizaremos en las características de Groovy y exploraremos conceptos más complejos.
