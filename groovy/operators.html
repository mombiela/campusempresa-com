Introducción

Los operadores son símbolos especiales o palabras clave que se utilizan para realizar operaciones en operandos. En Groovy, los operadores son similares a los de Java, pero con algunas mejoras y características adicionales. Esta sección cubrirá los diversos tipos de operadores disponibles en Groovy, desde aritmética básica hasta operadores más avanzados.

Operadores Aritméticos

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

  • Adición (+)
  • Sustracción (-)
  • Multiplicación (*)
  • División (/)
  • Módulo (%)

Ejemplos

def a = 10
def b = 5

println a + b  // Salida: 15
println a - b  // Salida: 5
println a * b  // Salida: 50
println a / b  // Salida: 2
println a % b  // Salida: 0

Operadores Relacionales

Los operadores relacionales se utilizan para comparar dos valores.

  • Igual a (==)
  • No igual a (!=)
  • Mayor que (>)
  • Menor que (<)
  • Mayor o igual que (>=)
  • Menor o igual que (<=)

Ejemplos

def a = 10
def b = 5

println a == b  // Salida: false
println a != b  // Salida: true
println a > b   // Salida: true
println a < b   // Salida: false
println a >= b  // Salida: true
println a <= b  // Salida: false

Operadores Lógicos

Los operadores lógicos se utilizan para combinar múltiples expresiones booleanas.

  • Y (&&)
  • O (||)
  • NO (!)

Ejemplos

def a = true
def b = false

println a && b  // Salida: false
println a || b  // Salida: true
println !a      // Salida: false

Operadores a Nivel de Bits

Los operadores a nivel de bits se utilizan para realizar operaciones en bits individuales de valores enteros.

  • Y (&)
  • O (|)
  • XOR (^)
  • NO (~)
  • Desplazamiento a la izquierda (<<)
  • Desplazamiento a la derecha (>>)
  • Desplazamiento a la derecha sin signo (>>>)

Ejemplos

def a = 5  // Binario: 0101
def b = 3  // Binario: 0011

println a & b  // Salida: 1  (Binario: 0001)
println a | b  // Salida: 7  (Binario: 0111)
println a ^ b  // Salida: 6  (Binario: 0110)
println ~a     // Salida: -6 (Binario: 1010, complemento a dos)
println a << 1 // Salida: 10 (Binario: 1010)
println a >> 1 // Salida: 2  (Binario: 0010)
println a >>> 1 // Salida: 2 (Binario: 0010)

Operadores de Asignación

Los operadores de asignación se utilizan para asignar valores a variables.

  • Asignación simple (=)
  • Asignación de adición (+=)
  • Asignación de sustracción (-=)
  • Asignación de multiplicación (*=)
  • Asignación de división (/=)
  • Asignación de módulo (%=)

Ejemplos

def a = 10

a += 5  // Equivalente a a = a + 5
println a  // Salida: 15

a -= 3  // Equivalente a a = a - 3
println a  // Salida: 12

a *= 2  // Equivalente a a = a * 2
println a  // Salida: 24

a /= 4  // Equivalente a a = a / 4
println a  // Salida: 6

a %= 5  // Equivalente a a = a % 5
println a  // Salida: 1

Conclusión

Los operadores son fundamentales para realizar diversas operaciones en Groovy. Comprender cómo usar operadores aritméticos, relacionales, lógicos, a nivel de bits y de asignación te permitirá escribir código más eficiente y efectivo. Practica el uso de estos operadores en diferentes escenarios para volverte más competente en la programación con Groovy.
