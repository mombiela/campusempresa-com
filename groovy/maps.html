Mapas en Groovy
Rangos ►

Los mapas en Groovy son una estructura de datos poderosa y flexible que te permite almacenar pares clave-valor. Son similares a los diccionarios en Python o a los hash maps en Java. Esta sección te guiará desde los conceptos básicos de la creación y uso de mapas hasta técnicas más avanzadas y mejores prácticas.

Introducción a los Mapas

  • Definición: Un mapa es una colección de pares clave-valor donde cada clave es única.
  • Sintaxis: Los mapas en Groovy se definen usando la notación [:].

Creación de Mapas

Puedes crear un mapa en Groovy usando diferentes métodos:

// Usando la notación literal
def map = [key1: 'value1', key2: 'value2']

// Usando el constructor de mapas
def map = new HashMap()
map.put('key1', 'value1')
map.put('key2', 'value2')

Acceso a Elementos del Mapa

Puedes acceder a los elementos en un mapa usando la clave:

def map = [name: 'John', age: 30]

// Acceso a valores
println map['name']  // Salida: John
println map.age      // Salida: 30

Modificación de Mapas

Puedes agregar, actualizar o eliminar elementos en un mapa:

def map = [name: 'John', age: 30]

// Agregar un nuevo par clave-valor
map['city'] = 'New York'

// Actualizar un par clave-valor existente
map.age = 31

// Eliminar un par clave-valor
map.remove('name')

println map  // Salida: [age:31, city:New York]

Operaciones Avanzadas con Mapas

Iteración sobre Mapas

Puedes iterar sobre las claves, valores o entradas en un mapa:

def map = [name: 'John', age: 30, city: 'New York']

// Iterar sobre las claves
map.keySet().each { key ->
    println key
}

// Iterar sobre los valores
map.values().each { value ->
    println value
}

// Iterar sobre las entradas
map.each { key, value ->
    println "$key: $value"
}

Filtrado de Mapas

Puedes filtrar mapas basándote en ciertas condiciones:

def map = [name: 'John', age: 30, city: 'New York']

// Filtrar entradas donde el valor es una cadena
def filteredMap = map.findAll { key, value -> value instanceof String }

println filteredMap  // Salida: [name:John, city:New York]

Transformación de Mapas

Puedes transformar mapas usando el método collectEntries:

def map = [name: 'John', age: 30, city: 'New York']

// Transformar el mapa a claves en mayúsculas
def transformedMap = map.collectEntries { key, value ->
    [(key.toUpperCase()): value]
}

println transformedMap  // Salida: [NAME:John, AGE:30, CITY:New York]

Métodos Comunes de Mapas

Aquí hay algunos métodos comunes que puedes usar con mapas:

| Método | Descripción | Ejemplo de Uso | |-----------------|---------------------------------------------|----------------------------------------| | put | Agrega un par clave-valor al mapa | map.put('key', 'value') | | get | Recupera el valor para una clave dada | map.get('key') | | remove | Elimina un par clave-valor del mapa | map.remove('key') | | containsKey | Verifica si el mapa contiene una clave específica | map.containsKey('key') | | containsValue | Verifica si el mapa contiene un valor específico | map.containsValue('value') | | size | Devuelve el número de pares clave-valor | map.size() | | clear | Elimina todos los pares clave-valor del mapa | map.clear() |

Conclusión

Los mapas en Groovy son una estructura de datos versátil y esencial que te permite almacenar y manipular pares clave-valor de manera eficiente. Desde la creación y el acceso básicos hasta operaciones avanzadas como el filtrado y la transformación, Groovy proporciona un conjunto rico de características para trabajar con mapas. Comprender estos conceptos mejorará enormemente tu capacidad para gestionar y manipular datos en tus aplicaciones Groovy.
