Azure Cognitive Services
Soluciones de Big Data con Azure ►

Azure Cognitive Services es un conjunto de APIs, SDKs y servicios disponibles para desarrolladores para hacer que sus aplicaciones sean más inteligentes, atractivas y descubribles. Estos servicios permiten a los desarrolladores agregar fácilmente características cognitivas a sus aplicaciones sin tener habilidades directas en IA o ciencia de datos.

Introducción a Azure Cognitive Services

Azure Cognitive Services proporciona una gama de APIs que permiten a los desarrolladores integrar capacidades de IA en sus aplicaciones. Estos servicios se categorizan en cinco áreas principales:

  • Visión
  • Voz
  • Lenguaje
  • Decisión
  • Búsqueda

Servicios de Visión

Los servicios de visión permiten a las aplicaciones procesar imágenes y videos para extraer información, categorizar contenido y más.

Servicios Clave de Visión

  • Computer Vision: Extrae información de imágenes, como objetos, caras y texto.
  • Face API: Detecta y reconoce rostros humanos en imágenes.
  • Form Recognizer: Extrae texto, pares clave-valor y tablas de documentos.
  • Custom Vision: Te permite construir clasificadores de imágenes personalizados.

Ejemplo: Usando la API de Computer Vision

import requests

subscription_key = "your_subscription_key"
endpoint = "https://your_endpoint.cognitiveservices.azure.com/"

analyze_url = endpoint + "vision/v3.1/analyze"

# Establecer la URL de la imagen
image_url = "https://example.com/image.jpg"

headers = {'Ocp-Apim-Subscription-Key': subscription_key}
params = {'visualFeatures': 'Categories,Description,Color'}
data = {'url': image_url}

response = requests.post(analyze_url, headers=headers, params=params, json=data)
response.raise_for_status()

analysis = response.json()
print(analysis)

En este ejemplo, usamos la API de Computer Vision para analizar una imagen y extraer información como categorías, descripciones y colores.

Servicios de Voz

Los servicios de voz permiten a las aplicaciones convertir el lenguaje hablado en texto, texto en lenguaje hablado y más.

Servicios Clave de Voz

  • Speech to Text: Convierte el lenguaje hablado en texto.
  • Text to Speech: Convierte texto en lenguaje hablado.
  • Speech Translation: Traduce el lenguaje hablado en tiempo real.
  • Speaker Recognition: Identifica y verifica a los hablantes.

Ejemplo: Usando la API de Speech to Text

import azure.cognitiveservices.speech as speechsdk

speech_key = "your_speech_key"
service_region = "your_service_region"

speech_config = speechsdk.SpeechConfig(subscription=speech_key, region=service_region)
audio_config = speechsdk.AudioConfig(filename="path_to_audio_file.wav")

speech_recognizer = speechsdk.SpeechRecognizer(speech_config=speech_config, audio_config=audio_config)

result = speech_recognizer.recognize_once()
print(result.text)

En este ejemplo, usamos la API de Speech to Text para convertir un archivo de audio en texto.

Servicios de Lenguaje

Los servicios de lenguaje proporcionan capacidades para entender y procesar el lenguaje natural.

Servicios Clave de Lenguaje

  • Text Analytics: Extrae insights como sentimiento, frases clave, entidades nombradas y lenguaje del texto.
  • Translator: Traduce texto de un idioma a otro.
  • Language Understanding (LUIS): Construye comprensión del lenguaje natural en aplicaciones, bots y dispositivos IoT.
  • QnA Maker: Crea una capa de preguntas y respuestas conversacionales sobre tus datos.

Ejemplo: Usando la API de Text Analytics

from azure.ai.textanalytics import TextAnalyticsClient
from azure.core.credentials import AzureKeyCredential

endpoint = "https://your_endpoint.cognitiveservices.azure.com/"
key = "your_key"

text_analytics_client = TextAnalyticsClient(endpoint=endpoint, credential=AzureKeyCredential(key))

documents = ["I had a wonderful experience! The rooms were wonderful and the staff was helpful."]

response = text_analytics_client.analyze_sentiment(documents=documents)[0]
print("Sentiment:", response.sentiment)

En este ejemplo, usamos la API de Text Analytics para analizar el sentimiento de un texto dado.

Servicios de Decisión

Los servicios de decisión proporcionan APIs para tomar decisiones informadas basadas en datos.

Servicios Clave de Decisión

  • Personalizer: Proporciona recomendaciones personalizadas.
  • Content Moderator: Detecta contenido potencialmente ofensivo o no deseado.
  • Anomaly Detector: Detecta anomalías en datos de series temporales.

Ejemplo: Usando la API de Anomaly Detector

import requests

subscription_key = "your_subscription_key"
endpoint = "https://your_endpoint.cognitiveservices.azure.com/"

anomaly_url = endpoint + "anomalydetector/v1.0/timeseries/entire/detect"

headers = {'Ocp-Apim-Subscription-Key': subscription_key}
data = {
    "series": [
        {"timestamp": "2023-01-01T00:00:00Z", "value": 10},
        {"timestamp": "2023-01-02T00:00:00Z", "value": 20},
        # Agrega más puntos de datos
    ],
    "granularity": "daily"
}

response = requests.post(anomaly_url, headers=headers, json=data)
response.raise_for_status()

anomalies = response.json()
print(anomalies)

En este ejemplo, usamos la API de Anomaly Detector para detectar anomalías en un conjunto de datos de series temporales.

Servicios de Búsqueda

Los servicios de búsqueda proporcionan APIs para agregar capacidades de búsqueda a tus aplicaciones.

Servicios Clave de Búsqueda

  • Azure Cognitive Search: Proporciona una solución de búsqueda como servicio.
  • Bing Search APIs: Te permite integrar capacidades de búsqueda de Bing en tus aplicaciones.

Ejemplo: Usando Azure Cognitive Search

from azure.search.documents import SearchClient
from azure.core.credentials import AzureKeyCredential

endpoint = "https://your_search_service.search.windows.net"
index_name = "your_index_name"
key = "your_key"

search_client = SearchClient(endpoint=endpoint, index_name=index_name, credential=AzureKeyCredential(key))

results = search_client.search("search_term")
for result in results:
    print(result)

En este ejemplo, usamos Azure Cognitive Search para buscar un término dentro de un índice especificado.

Conclusión

Azure Cognitive Services proporciona un conjunto completo de APIs que permiten a los desarrolladores agregar características inteligentes a sus aplicaciones. Al aprovechar estos servicios, los desarrolladores pueden mejorar sus aplicaciones con capacidades como análisis de imágenes y videos, reconocimiento de voz, comprensión del lenguaje natural, toma de decisiones y búsqueda. Ya seas un principiante o un desarrollador avanzado, Azure Cognitive Services ofrece herramientas y recursos para ayudarte a construir aplicaciones más inteligentes.
