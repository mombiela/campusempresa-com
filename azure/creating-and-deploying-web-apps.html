Introducción

En este tema, exploraremos cómo crear y desplegar aplicaciones web utilizando Microsoft Azure. Esta guía está estructurada para llevarte desde los conceptos básicos hasta los avanzados, asegurando que tengas una comprensión completa del proceso.

Comenzando con Azure

Antes de sumergirnos en la creación de aplicaciones web, es esencial configurar tu entorno de Azure.

Configuración de una Cuenta de Azure

  • Registrarse: Ve al sitio web de Azure y regístrate para obtener una cuenta gratuita.
  • Portal de Azure: Familiarízate con el Portal de Azure, donde gestionarás tus recursos.

Instalación de Azure CLI

Azure CLI es una herramienta de línea de comandos que te permite gestionar recursos de Azure.

  • Instalación: Sigue la documentación oficial para instalar Azure CLI en tu sistema.
  • Inicio de Sesión: Usa el comando az login para autenticar tu cuenta.

Creación de una Aplicación Web

Azure proporciona varias formas de crear aplicaciones web. Nos enfocaremos en usar el Portal de Azure y Azure CLI.

Usando el Portal de Azure

  1. Navega al Portal de Azure.

  2. Crear un Recurso: Haz clic en "Crear un recurso" y selecciona "Aplicación Web".

  3. Configurar Ajustes Básicos:

    • Suscripción: Elige tu suscripción.
    • Grupo de Recursos: Crea un nuevo grupo de recursos o usa uno existente.
    • Nombre: Proporciona un nombre único para tu aplicación web.
    • Publicar: Elige "Código" o "Contenedor Docker".
    • Pila de Runtime: Selecciona la pila de runtime (por ejemplo, .NET, Node.js, Python).
    • Región: Elige una región cercana a tus usuarios.

  4. Revisar y Crear: Revisa tus configuraciones y haz clic en "Crear".

Usando Azure CLI

# Crear un grupo de recursos
az group create --name myResourceGroup --location eastus

# Crear un plan de App Service
az appservice plan create --name myAppServicePlan --resource-group myResourceGroup --sku FREE

# Crear una aplicación web
az webapp create --resource-group myResourceGroup --plan myAppServicePlan --name myUniqueAppName --runtime "NODE|14-lts"
  • Explicación:
    • az group create: Crea un grupo de recursos.
    • az appservice plan create: Crea un plan de App Service.
    • az webapp create: Crea la aplicación web con el runtime especificado.

Desplegando Tu Aplicación Web

Una vez que tu aplicación web esté creada, necesitas desplegar tu código en ella.

Usando Git

  1. Inicializar Repositorio Git:

    git init
git add .
git commit -m "Initial commit"

  2. Configurar Despliegue:

    az webapp deployment source config-local-git --name myUniqueAppName --resource-group myResourceGroup

  3. Enviar Código:

    git remote add azure <deployment-url>
git push azure master

Usando Azure DevOps

  1. Crear un Proyecto: En Azure DevOps, crea un nuevo proyecto.
  2. Configurar un Pipeline: Usa el servicio de Azure Pipelines para configurar un pipeline de CI/CD.
  3. Configurar Despliegue: Vincula tu proyecto de Azure DevOps a tu aplicación web de Azure para despliegue continuo.

Monitoreo y Escalado

Azure proporciona herramientas para monitorear y escalar tu aplicación web.

Monitoreo

  • Application Insights: Integra Application Insights para monitorear el rendimiento y diagnosticar problemas.
  • Logs: Usa Azure Monitor para ver logs y configurar alertas.

Escalado

  • Escalar Verticalmente: Aumenta los recursos de tu plan de App Service.
  • Escalar Horizontalmente: Aumenta el número de instancias que ejecutan tu aplicación web.

Temas Avanzados

Dominios Personalizados y SSL

  • Dominio Personalizado: Asocia un dominio personalizado a tu aplicación web.
  • Certificados SSL: Asegura tu aplicación web con certificados SSL.

Usando Docker

  • Soporte Docker: Despliega tu aplicación web usando contenedores Docker.
  • Registro de Contenedores de Azure: Almacena y gestiona imágenes Docker.

Serverless con Azure Functions

  • Azure Functions: Crea funciones serverless para manejar tareas específicas.
  • Integración: Integra Azure Functions con tu aplicación web para funcionalidad adicional.

Conclusión

Crear y desplegar aplicaciones web en Azure es una forma poderosa de aprovechar la infraestructura en la nube. Siguiendo esta guía, ahora deberías poder:

  • Configurar tu entorno de Azure.
  • Crear y desplegar aplicaciones web usando el Portal de Azure y Azure CLI.
  • Monitorear y escalar tus aplicaciones web.
  • Explorar temas avanzados como dominios personalizados, SSL, Docker y funciones serverless.

Con estas habilidades, puedes gestionar y desplegar eficientemente aplicaciones web en Azure, asegurando alta disponibilidad y rendimiento para tus usuarios.
