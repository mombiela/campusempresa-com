¿Qué son las Estructuras en C?

Las estructuras en C son tipos de datos definidos por el usuario que permiten combinar elementos de datos de diferentes tipos. Las estructuras se utilizan para representar un registro, por ejemplo, un registro de estudiante podría contener un nombre, edad y calificaciones.

Conceptos Clave

  • Definición: Una estructura se define usando la palabra clave struct.
  • Miembros: Las variables declaradas dentro de una estructura se llaman miembros.
  • Acceso a Miembros: Los miembros de una estructura se acceden usando el operador punto (.).
  • Inicialización: Las estructuras pueden ser inicializadas en el momento de la declaración.

Definiendo una Estructura

Para definir una estructura, se usa la palabra clave struct seguida del nombre de la estructura y los miembros de la estructura encerrados en llaves {}.

struct Estudiante {
    char nombre[50];
    int edad;
    float calificaciones;
};

En este ejemplo:

  • Estudiante es el nombre de la estructura.
  • nombre, edad y calificaciones son miembros de la estructura.

Declarando Variables de Estructura

Después de definir una estructura, puedes declarar variables de ese tipo.

struct Estudiante estudiante1, estudiante2;

Aquí, estudiante1 y estudiante2 son variables del tipo struct Estudiante.

Accediendo a los Miembros de la Estructura

Puedes acceder a los miembros de una estructura usando el operador punto (.).

estudiante1.edad = 20;
estudiante1.calificaciones = 85.5;
strcpy(estudiante1.nombre, "Juan Pérez");

En este ejemplo:

  • estudiante1.edad establece la edad de estudiante1 en 20.
  • estudiante1.calificaciones establece las calificaciones de estudiante1 en 85.5.
  • strcpy(estudiante1.nombre, "Juan Pérez") copia la cadena "Juan Pérez" en el miembro nombre de estudiante1.

Inicializando Estructuras

Las estructuras pueden ser inicializadas en el momento de la declaración.

struct Estudiante estudiante1 = {"Juan Pérez", 20, 85.5};

Esto inicializa estudiante1 con el nombre "Juan Pérez", edad 20 y calificaciones 85.5.

Programa de Ejemplo

Aquí hay un programa de ejemplo completo que demuestra el uso de estructuras.

#include <stdio.h>
#include <string.h>

// Definir la estructura
struct Estudiante {
    char nombre[50];
    int edad;
    float calificaciones;
};

int main() {
    // Declarar e inicializar variables de estructura
    struct Estudiante estudiante1 = {"Juan Pérez", 20, 85.5};
    struct Estudiante estudiante2;

    // Asignar valores a estudiante2
    strcpy(estudiante2.nombre, "Ana García");
    estudiante2.edad = 22;
    estudiante2.calificaciones = 90.0;

    // Imprimir detalles de estudiante1
    printf("Estudiante 1:\n");
    printf("Nombre: %s\n", estudiante1.nombre);
    printf("Edad: %d\n", estudiante1.edad);
    printf("Calificaciones: %.2f\n", estudiante1.calificaciones);

    // Imprimir detalles de estudiante2
    printf("Estudiante 2:\n");
    printf("Nombre: %s\n", estudiante2.nombre);
    printf("Edad: %d\n", estudiante2.edad);
    printf("Calificaciones: %.2f\n", estudiante2.calificaciones);

    return 0;
}

Conclusión

Las estructuras en C proporcionan una forma de agrupar elementos de datos relacionados de diferentes tipos en una sola unidad. Esto es particularmente útil para representar estructuras de datos complejas como registros. Al entender cómo definir, declarar, inicializar y acceder a estructuras, puedes gestionar y manipular datos relacionados de manera más efectiva en tus programas.
