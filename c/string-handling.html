Introducción

El manejo de cadenas es un aspecto fundamental de la programación en C. Las cadenas en C son arreglos de caracteres terminados por un carácter nulo ('\0'). Este tema cubrirá los conceptos básicos del manejo de cadenas, incluyendo declaración, inicialización, funciones de la biblioteca estándar y técnicas avanzadas de manipulación de cadenas.

1. Conceptos Básicos de las Cadenas en C

1.1 Declaración e Inicialización

  • Las cadenas se declaran como arreglos de caracteres.
  • Deben terminar con un carácter nulo ('\0').
char str1[20]; // Declaración de una cadena con un tamaño máximo de 20 caracteres
char str2[] = "¡Hola, Mundo!"; // Inicialización con un literal de cadena

1.2 Entrada y Salida de Cadenas

  • Usa scanf y printf para entrada y salida básicas.
  • gets y puts también pueden ser usados, pero gets es inseguro y debe ser evitado.
char name[50];
printf("Introduce tu nombre: ");
scanf("%s", name); // Nota: scanf deja de leer en el primer espacio en blanco
printf("¡Hola, %s!\n", name);

2. Funciones de la Biblioteca Estándar para Cadenas

2.1 `strlen` - Longitud de la Cadena

  • Devuelve la longitud de la cadena (excluyendo el carácter nulo).
#include <stdio.h>
#include <string.h>

char str[] = "Hola";
int length = strlen(str); // length = 4

2.2 `strcpy` - Copia de Cadenas

  • Copia una cadena a otra.
#include <stdio.h>
#include <string.h>

char src[] = "Hola";
char dest[20];
strcpy(dest, src); // dest ahora contiene "Hola"

2.3 `strcat` - Concatenación de Cadenas

  • Añade una cadena al final de otra.
#include <stdio.h>
#include <string.h>

char str1[20] = "Hola";
char str2[] = " Mundo";
strcat(str1, str2); // str1 ahora contiene "Hola Mundo"

2.4 `strcmp` - Comparación de Cadenas

  • Compara dos cadenas lexicográficamente.
#include <stdio.h>
#include <string.h>

char str1[] = "Hola";
char str2[] = "Mundo";
int result = strcmp(str1, str2); // result < 0 porque "Hola" es menor que "Mundo"

2.5 `strchr` y `strrchr` - Búsqueda de Caracteres

  • strchr encuentra la primera ocurrencia de un carácter.
  • strrchr encuentra la última ocurrencia de un carácter.
#include <stdio.h>
#include <string.h>

char str[] = "¡Hola, Mundo!";
char *first = strchr(str, 'o'); // first apunta al primer 'o' en "¡Hola, Mundo!"
char *last = strrchr(str, 'o'); // last apunta al último 'o' en "¡Hola, Mundo!"

3. Manipulación Avanzada de Cadenas

3.1 Cadenas Dinámicas

  • Usa punteros y asignación dinámica de memoria para cadenas cuyo tamaño no se conoce en tiempo de compilación.
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

char *dynamicStr = (char *)malloc(50 * sizeof(char));
if (dynamicStr != NULL) {
    strcpy(dynamicStr, "Cadena Dinámica");
    printf("%s\n", dynamicStr);
    free(dynamicStr); // Siempre libera la memoria asignada dinámicamente
}

3.2 Tokenización de Cadenas

  • Usa strtok para dividir una cadena en tokens basados en delimitadores.
#include <stdio.h>
#include <string.h>

char str[] = "¡Hola, Mundo! Bienvenido a la programación en C.";
char *token = strtok(str, " ,.!");

while (token != NULL) {
    printf("%s\n", token);
    token = strtok(NULL, " ,.!"); // Continúa tokenizando el resto de la cadena
}

Conclusión

El manejo de cadenas en C implica entender cómo las cadenas se representan como arreglos de caracteres y usar varias funciones de la biblioteca estándar para manipularlas. Desde operaciones básicas como copiar y concatenar hasta técnicas más avanzadas como la asignación dinámica y la tokenización, dominar el manejo de cadenas es esencial para una programación efectiva en C. Siempre recuerda manejar las cadenas con cuidado para evitar errores comunes como desbordamientos de búfer y fugas de memoria.
