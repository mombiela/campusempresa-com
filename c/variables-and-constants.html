Introducción

En la programación en C, las variables y constantes son conceptos fundamentales que necesitas entender para escribir código efectivo y eficiente. Esta sección cubrirá qué son las variables y constantes, cómo declararlas y usarlas, y las diferencias entre ellas.

Variables

¿Qué es una Variable?

  • Una variable es una ubicación de almacenamiento en memoria con un tipo específico que determina qué tipo de datos puede contener.
  • Las variables tienen nombres (identificadores) que se utilizan para acceder a sus valores almacenados.

Declaración de Variables

  • Para declarar una variable, necesitas especificar su tipo seguido de su nombre.
  • Sintaxis: tipo nombre_variable;
int edad;
float salario;
char inicial;

Inicialización de Variables

  • Puedes inicializar una variable en el momento de su declaración.
  • Sintaxis: tipo nombre_variable = valor;
int edad = 25;
float salario = 50000.50;
char inicial = 'A';

Uso de Variables

  • Las variables pueden ser usadas en expresiones y declaraciones.
  • Ejemplo:
#include <stdio.h>

int main() {
    int edad = 25;
    float salario = 50000.50;
    char inicial = 'A';

    printf("Edad: %d\n", edad);
    printf("Salario: %.2f\n", salario);
    printf("Inicial: %c\n", inicial);

    return 0;
}

Constantes

¿Qué es una Constante?

  • Una constante es un valor que no puede ser alterado por el programa durante su ejecución.
  • Las constantes se usan para definir valores fijos que no cambian.

Declaración de Constantes

  • En C, puedes declarar constantes usando la palabra clave const o la directiva del preprocesador #define.

Usando la Palabra Clave const

  • Sintaxis: const tipo nombre_constante = valor;
const int DIAS_EN_SEMANA = 7;
const float PI = 3.14159;
const char NUEVALINEA = '\n';

Usando la Directiva del Preprocesador #define

  • Sintaxis: #define NOMBRE_CONSTANTE valor
#define DIAS_EN_SEMANA 7
#define PI 3.14159
#define NUEVALINEA '\n'

Uso de Constantes

  • Las constantes pueden ser usadas en expresiones y declaraciones al igual que las variables.
  • Ejemplo:
#include <stdio.h>

#define PI 3.14159

int main() {
    const int DIAS_EN_SEMANA = 7;

    printf("Días en una semana: %d\n", DIAS_EN_SEMANA);
    printf("Valor de PI: %.5f\n", PI);

    return 0;
}

Diferencias Entre Variables y Constantes

| Característica | Variables | Constantes | |-----------------|---------------------------------|-----------------------------------| | Mutabilidad | Puede cambiar durante la ejecución del programa | No puede cambiar una vez definida | | Declaración | tipo nombre_variable; | const tipo nombre_constante = valor; o #define NOMBRE_CONSTANTE valor | | Uso | Usado para almacenar datos que pueden cambiar | Usado para almacenar valores fijos |

Conclusión

Entender las variables y constantes es crucial para gestionar datos en tus programas en C. Las variables te permiten almacenar y manipular datos que pueden cambiar, mientras que las constantes proporcionan una forma de definir valores que permanecen iguales a lo largo del programa. Al dominar estos conceptos, podrás escribir código más robusto y mantenible.
