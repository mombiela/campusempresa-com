Introducción

Las funciones matemáticas son esenciales en la programación para realizar varios cálculos. En C, la biblioteca estándar proporciona una amplia gama de funciones matemáticas que se pueden usar para realizar operaciones como trigonometría, exponenciación y logaritmos.

Conceptos Clave

  • Biblioteca Estándar: El archivo de encabezado math.h contiene las declaraciones de las funciones matemáticas.
  • Prototipos de Funciones: Entender los prototipos de estas funciones es crucial para su uso correcto.
  • Tipos de Retorno: La mayoría de las funciones matemáticas devuelven valores double.
  • Manejo de Errores: Algunas funciones pueden devolver valores especiales como NaN (Not a Number) o INF (Infinito) en caso de errores.

Funciones Matemáticas Comunes

Funciones Aritméticas Básicas

  • fabs(double x): Devuelve el valor absoluto de x.
  • fmod(double x, double y): Devuelve el resto de x dividido por y.
#include <stdio.h>
#include <math.h>

int main() {
    double x = -5.3;
    double y = 2.0;

    printf("El valor absoluto de %.2f es %.2f\n", x, fabs(x));
    printf("El resto de %.2f dividido por %.2f es %.2f\n", x, y, fmod(x, y));

    return 0;
}

Funciones Exponenciales y Logarítmicas

  • exp(double x): Devuelve e elevado a la potencia de x.
  • log(double x): Devuelve el logaritmo natural (base e) de x.
  • log10(double x): Devuelve el logaritmo en base 10 de x.
#include <stdio.h>
#include <math.h>

int main() {
    double x = 2.0;

    printf("e elevado a la potencia de %.2f es %.2f\n", x, exp(x));
    printf("El logaritmo natural de %.2f es %.2f\n", x, log(x));
    printf("El logaritmo en base 10 de %.2f es %.2f\n", x, log10(x));

    return 0;
}

Funciones de Potencia

  • pow(double base, double exponent): Devuelve base elevado a la potencia de exponent.
  • sqrt(double x): Devuelve la raíz cuadrada de x.
#include <stdio.h>
#include <math.h>

int main() {
    double base = 3.0;
    double exponent = 4.0;

    printf("%.2f elevado a la potencia de %.2f es %.2f\n", base, exponent, pow(base, exponent));
    printf("La raíz cuadrada de %.2f es %.2f\n", base, sqrt(base));

    return 0;
}

Funciones Trigonométricas

  • sin(double x): Devuelve el seno de x (x en radianes).
  • cos(double x): Devuelve el coseno de x (x en radianes).
  • tan(double x): Devuelve la tangente de x (x en radianes).
#include <stdio.h>
#include <math.h>

int main() {
    double angle = M_PI / 4; // 45 grados en radianes

    printf("El seno de %.2f radianes es %.2f\n", angle, sin(angle));
    printf("El coseno de %.2f radianes es %.2f\n", angle, cos(angle));
    printf("La tangente de %.2f radianes es %.2f\n", angle, tan(angle));

    return 0;
}

Funciones Hiperbólicas

  • sinh(double x): Devuelve el seno hiperbólico de x.
  • cosh(double x): Devuelve el coseno hiperbólico de x.
  • tanh(double x): Devuelve la tangente hiperbólica de x.
#include <stdio.h>
#include <math.h>

int main() {
    double x = 1.0;

    printf("El seno hiperbólico de %.2f es %.2f\n", x, sinh(x));
    printf("El coseno hiperbólico de %.2f es %.2f\n", x, cosh(x));
    printf("La tangente hiperbólica de %.2f es %.2f\n", x, tanh(x));

    return 0;
}

Manejo de Errores

  • Errores de Dominio: Ocurren cuando los valores de entrada están fuera del dominio sobre el cual la función está definida.
  • Errores de Rango: Ocurren cuando el resultado de una función no puede ser representado dentro del rango del tipo de retorno.

Ejemplo de Manejo de Errores

#include <stdio.h>
#include <math.h>
#include <errno.h>

int main() {
    errno = 0; // Restablecer el indicador de error global
    double result = sqrt(-1.0);

    if (errno == EDOM) {
        printf("Ocurrió un error de dominio\n");
    } else {
        printf("La raíz cuadrada es %.2f\n", result);
    }

    return 0;
}

Conclusión

Las funciones matemáticas en C son herramientas poderosas para realizar una amplia gama de cálculos. Al entender las funciones de la biblioteca estándar, sus prototipos y cómo manejar errores, puedes utilizar efectivamente estas funciones en tus programas. Practica usando estas funciones con diferentes entradas para obtener una comprensión más profunda y volverte competente en su uso.
